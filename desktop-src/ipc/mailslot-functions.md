# <a name="mailslot-functions"></a>Funciones de buzón de correo

Las siguientes funciones se pueden usar con los procesadores de elementos.



| Función                                   | Descripción                                                                  |
|--------------------------------------------|------------------------------------------------------------------------------|
| [**CreateMailslot**](/windows/desktop/api/Winbase/nf-winbase-createmailslota)   | Crea un buzón de correo con el nombre especificado.                                  |
| [**GetMailslotInfo**](/windows/desktop/api/Winbase/nf-winbase-getmailslotinfo) | Recupera información sobre el buzón de correo especificado.                          |
| [**SetMailslotInfo**](/windows/desktop/api/Winbase/nf-winbase-setmailslotinfo) | Establece el valor de tiempo de espera utilizado por el buzón de correo especificado para una operación de lectura. |
