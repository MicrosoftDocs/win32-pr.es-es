# <a name="the-word-balloon"></a>El globo de palabras

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Además de la salida de audio hablado, la interfaz de Microsoft Agent admite subtítulos textuales en forma de salida de texto en globos de palabras de estilo de animación. Las palabras aparecen en el globo mientras se pronuncian. El globo se oculta cuando se completa la salida hablada. La ventana Opciones avanzadas de caracteres proporciona opciones para deshabilitar la presentación del globo, así como atributos de control relacionados con su apariencia.
