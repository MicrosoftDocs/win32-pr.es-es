# <a name="about-remote-access-service"></a>Acerca del servicio de acceso remoto

El servicio de acceso remoto (RAS) proporciona funciones de acceso remoto a las aplicaciones cliente en equipos que ejecutan Windows.

Las aplicaciones RAS pueden realizar las siguientes tareas:

-   Mostrar cualquiera de los cuadros de diálogo comunes de RAS. Esto incluye el cuadro de diálogo principales **redes de acceso telefónico** , la hoja de propiedades **monitor de acceso telefónico a redes** y otros cuadros de diálogo para crear, editar, copiar o marcar una entrada de libreta de teléfonos.
-   Iniciar y finalizar una operación de conexión RAS mediante los cuadros de diálogo comunes o las funciones de marcado de bajo nivel.
-   Crear, editar o copiar entradas de libreta de teléfonos mediante los cuadros de diálogo comunes o las funciones de libreta de teléfonos de bajo nivel.
-   Trabajar con entradas en la base de datos de asignación de marcado automático de RAS. Esta base de datos asigna direcciones de red a la entrada de la libreta de teléfonos que puede establecer una conexión con la dirección.
-   Obtener información de RAS, incluida información acerca de las conexiones RAS existentes, información acerca de los dispositivos compatibles con RAS configurados en el equipo local y notificaciones cuando se inicia o finaliza una conexión RAS.

Windows NT 4,0 también proporciona compatibilidad con la administración del servidor RAS y con extensiones de terceros para la seguridad del servidor RAS y la administración de conexiones. Windows 95 no proporciona compatibilidad con el servidor RAS.
