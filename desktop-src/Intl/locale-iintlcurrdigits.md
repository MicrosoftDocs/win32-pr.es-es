# <a name="locale_iintlcurrdigits"></a>configuración regional \_ IINTLCURRDIGITS

\[La configuración regional \_ IINTLCURRDIGITS no se admite y puede modificarse o no estar disponible en el futuro. En su lugar, use la [configuración regional \_ ICURRDIGITS](locale-icurrdigits.md).\]

Número de dígitos fraccionarios para el formato monetario internacional. El número máximo de caracteres permitido para esta cadena es tres, incluido un carácter nulo de terminación. Por ejemplo, el número de dígitos fraccionarios para el formato Monetario Internacional de dólares estadounidenses es 2, como en $345,25.
