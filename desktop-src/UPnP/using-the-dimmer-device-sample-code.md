# <a name="using-the-dimmer-device-sample-code"></a>Usar el código de ejemplo de dispositivo de atenuación

El kit de desarrollo de software (SDK) de la plataforma incluye código de ejemplo de dispositivo atenuado. En este código de ejemplo se muestra un dispositivo basado en UPnP simple implementado mediante la API de host de dispositivo con tecnología UPnP.

El \_ Directorio de DCO DimmerService contiene la implementación para el archivo Device. dll. El directorio RegisterDevice contiene la implementación para registrar el archivo Device. dll con el servicio de registro de UPnP en el equipo. El directorio StepByStep contiene proyectos de VC 6.0 que corresponden a la "Guía paso a paso" incluida en el archivo de Readme.txt. La "Guía paso a paso" es para los desarrolladores que no tienen una experiencia previa de ATL/COM.
