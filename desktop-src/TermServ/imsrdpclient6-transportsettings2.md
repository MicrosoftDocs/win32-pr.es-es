# <a name="imsrdpclient6transportsettings2-property"></a>IMsRdpClient6:: TransportSettings2 (propiedad)

Recupera la interfaz [**IMsRdpClientTransportSettings2**](imsrdpclienttransportsettings2.md) .

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT get_TransportSettings2(
  [out] IMsRdpClientTransportSettings2 **ppXportSet2
);
```



## <a name="property-value"></a>Valor de propiedad
