# <a name="d1165-read-and-discard-map-options-are-incompatible"></a>D1165: las opciones de mapa de lectura y descartado no son compatibles

Las \_ opciones \_ de asignación D2D1 \_ leídas no son compatibles con \_ las opciones de asignación D2D1 \_ \_ descartadas



|             |         |
|-------------|---------|
| Nivel de error | Advertencia |



 

## <a name="possible-causes"></a>Causas posibles

Direct2D puede generar este mensaje si una aplicación llama a ID2D1Bitmap1:: Map con las opciones de \_ mapa D2D1 \_ \_ Read y D2D1 Map Options \_ \_ \_ discard Options.

## <a name="possible-fixes"></a>Posibles correcciones

Las opciones de lectura y descartado no coinciden entre sí porque permitir el acceso de lectura a datos sin definir tiene una utilidad limitada para una aplicación. Consulte D2D1 \_ map \_ Options para obtener una descripción de las opciones de asignación. Al llamar a ID2D1Bitmap1:: Map, tenga en cuenta qué tipo de acceso necesita la aplicación.
