# <a name="windows-app-certification-kit"></a>Kit para la certificación de aplicaciones en Windows

## <a name="platforms"></a>Plataformas

**Clientes-** Windows 8.1 Windows 7 \| Windows 8 \|  


## <a name="description"></a>Descripción

El kit de certificación de aplicaciones de Windows 3,1 se incluye con la Windows SDK para Windows 8.1. El kit se puede usar para certificar previamente aplicaciones de la tienda Windows para Windows 8 y Windows 8.1 antes de la incorporación, así como para los programas de certificación de aplicaciones de escritorio de Windows 7, Windows 8 y Windows 8.1 y Windows 8.1.

El kit de certificación de aplicaciones de Windows 3,1 se ha actualizado para ofrecer a los desarrolladores una experiencia de usuario sin problemas, incluida la ejecución de pruebas en paralelo para ahorrar tiempo global, la selección de pruebas selectivas y los informes mejorados, por nombrar algunas de las mejoras.

**Nota:** Cuando se le pida una actualización, actualice su versión del kit de certificación de aplicaciones de Windows para beneficiarse de las características más recientes incluidas en el kit. Cuando la aplicación se prueba con la versión más reciente del kit de certificación de aplicaciones de Windows, se garantiza un mejor cumplimiento de las directivas de la tienda Windows y los programas de certificación de escritorio.
