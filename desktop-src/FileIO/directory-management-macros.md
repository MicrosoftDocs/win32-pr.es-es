# <a name="directory-management-macros"></a>Macros de administración de directorios

Las macros siguientes se usan con puntos de reanción.

## <a name="in-this-section"></a>En esta sección



| Macro                                                                     | Descripción                                                                                                                             |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**IsReparseTagMicrosoft**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagmicrosoft)<br/>         | Determina si una etiqueta de punto de reantiqueta indica un punto de rean aproximado de Microsoft.<br/>                                                  |
| [**IsReparseTagNameSurrogate**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagnamesurrogate)<br/> | Determina si el punto de reantiquete asociado de una etiqueta es suplente para otra entidad con nombre (por ejemplo, una carpeta montada).<br/> |
