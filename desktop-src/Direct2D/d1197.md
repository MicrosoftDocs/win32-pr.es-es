# <a name="d1197-wrong-d3d-device"></a>D1197: Dispositivo D3D incorrecto

La superficie DXGI que se pasa a esta API es del dispositivo incorrecto.





 

## <a name="possible-causes"></a>Causas posibles

La superficie de DXGI que se pasa a esta API es del dispositivo incorrecto.

## <a name="possible-fixes"></a>Posibles correcciones

Compruebe que las superficies DXGI tienen el mismo dispositivo D3D que el contexto del dispositivo D2D.
