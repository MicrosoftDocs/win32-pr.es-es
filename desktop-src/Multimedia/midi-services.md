# <a name="midi-services"></a>Servicios MIDI

La mayoría de las aplicaciones podrán usar el secuenciador MIDI de MCI o los búferes de secuencia (y la función [**midiStreamOut**](/windows/win32/api/mmeapi/nf-mmeapi-midistreamout) ) para implementar toda la funcionalidad MIDI que necesitan. Los desarrolladores de MIDI graves, que producen herramientas de creación o secuenciación MIDI, pueden usar una combinación de las capacidades de Stream y los servicios MIDI, o bien usar solo los servicios MIDI. En los temas siguientes se proporciona información general acerca del uso de los servicios MIDI.

-   [Consulta de dispositivos MIDI](querying-midi-devices.md)
-   [Abrir y cerrar controladores de dispositivos](opening-and-closing-device-drivers.md)
-   [Asignación y preparación de bloques de datos MIDI](allocating-and-preparing-midi-data-blocks.md)
-   [Administrar bloques de datos MIDI](managing-midi-data-blocks.md)
-   [Solicitar formatos de hora](requesting-time-formats.md)
-   [Control de errores con funciones MIDI](handling-errors-with-midi-functions.md)
