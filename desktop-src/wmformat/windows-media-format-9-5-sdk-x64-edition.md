# <a name="windows-media-format-95-sdk-x64-edition"></a>Windows Media Format 9,5 SDK x64 Edition

Esta documentación se aplica al nuevo Windows Media Format 9,5 SDK x64 y a las versiones anteriores de 32 bits del SDK de Windows Media Format.

En la lista siguiente se resumen los cambios en el SDK para la versión basada en x64:

-   No se admite DRM. Los archivos relacionados con DRM (incluidos los ejemplos) se han quitado de la instalación del SDK.
-   Windows Media Administrador de dispositivos no forma parte de la instalación del SDK basado en x64.
-   Los ejemplos y encabezados que admiten las interfaces de códec de vídeo y Windows Media Audio no forman parte de la instalación del SDK basado en x64.
-   Los perfiles de sistema localizados no forman parte de la instalación del SDK basado en x64.

En las notas de la sección referencia de programación de esta documentación se llama a características que no se admiten en el SDK de Windows Media Format 9,5 x64 Edition.
