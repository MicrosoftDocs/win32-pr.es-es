# <a name="map-tag"></a>Etiqueta de mapa

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descripción**
</dt> <dd>

Mapas texto hablado en el texto que se muestra en el globo de palabras.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxis**
</dt> <dd>

\\**Map**="spokentext"="balloontext"\\



| Parte          | Descripción                                           |
|---------------|-------------------------------------------------------|
| *spokentext*  | Cadena que especifica el texto para la salida hablada.       |
| *balloontext* | Cadena que especifica el texto para la salida del globo de palabras. |



 

</dd> </dl>

### <a name="remarks"></a>Comentarios

Esta etiqueta le permite usar texto hablado diferente al que se muestra en el globo de palabras.
