# <a name="shader-model-51"></a>Modelo de sombreador 5,1

Esta sección contiene las páginas de referencia del modelo de sombreador HLSL 5,1, introducidos con D3D12.

El modelo de sombreador 5,1 es funcionalmente muy similar al [modelo de sombreador 5](d3d11-graphics-reference-sm5.md), el cambio principal es más flexible en la selección de recursos, ya que permite la indexación de matrices de descriptores dentro de un sombreador.



| Característica                   | Funcionalidad                                                                |
|---------------------------|---------------------------------------------------------------------------|
| Conjunto de instrucciones           | [**Funciones intrínsecas de HLSL**](dx-graphics-hlsl-intrinsic-functions.md)  |
| Sombreador de vértices máximo         | Sin restricción                                                            |
| Sombreador de píxeles máximo          | Sin restricción                                                            |
| Nuevos perfiles de sombreador agregados | CS \_ 5 \_ 1, DS \_ 5 \_ 1, GS \_ 5 \_ 1, HS \_ 5 \_ 1, PS \_ 5 \_ 1, vs \_ 5 \_ 1, rootsig \_ 1 \_ 0 |
