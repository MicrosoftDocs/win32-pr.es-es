# <a name="miscellaneous-helper-functions"></a>Funciones auxiliares misceláneas

A continuación se muestran varias funciones auxiliares.



| Función                                                   | Descripción                                                                                                   |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**EqualPins**](equalpins.md)                             | Comprueba si dos patillas están en el mismo objeto.                                                                    |
| [**Int64x32Div32**](int64x32div32.md)                     | Implementa la fórmula `((a*b)+rnd)/c` donde `a` es un valor de 64 bits y `b` , `c` y `rnd` son valores de 32 bits. |
| [**IsEqualObject**](isequalobject.md)                     | Comprueba si dos interfaces están en el mismo objeto.                                                              |
| [**llMulDiv**](llmuldiv.md)                               | Implementa la fórmula `((a*b)+rnd)/c` donde cada término es un valor de 64 bits.                                     |
| [**WaitDispatchingMessages**](waitdispatchingmessages.md) | Espera a que se señale un objeto mientras se envían mensajes de ventana.                                        |
