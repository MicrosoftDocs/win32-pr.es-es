# <a name="parameter-initializers"></a>Inicializadores de parámetros

Este tema no está actualizado. Para obtener la información más reciente, consulte la [especificación del esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Para obtener un resumen del tipo de elemento ParameterInit, consulte la sección [ParameterInit](parameterinit.md) .

Un tipo de elemento ParameterInit asigna un valor a un ParameterDef. Por lo tanto, ParameterDef se inicializa en PrintTicket. Para ver un ejemplo de un tipo de elemento ParameterInit, consulte el ejemplo de la [PrintTicket](printticket-example.md)del esquema de impresión.

Para obtener más información sobre la interacción entre los elementos ParameterDef y ParameterInit, consulte la sección [elementos ParameterDef y ParameterInit](./parameterdef-and-parameterinit-elements.md) .
