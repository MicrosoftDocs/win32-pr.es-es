# <a name="trust-posix-offset-attribute"></a>Trust-POSIX-atributo offset

El desplazamiento de la interfaz del sistema operativo portátil (POSIX) del dominio de confianza.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Trust-POSIX-desplazamiento                   |
| Nombre para mostrar de LDAP | trustPosixOffset                     |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | Cuando se crea una nueva confianza.         |
| Attribute-Id      | 1.2.840.113556.1.4.134               |
| System-ID-GUID    | bf967a5e-0de6-11d0-a285-00aa003049e2 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
