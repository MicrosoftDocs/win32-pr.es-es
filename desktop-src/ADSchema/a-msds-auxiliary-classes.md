# <a name="ms-ds-auxiliary-classes-attribute"></a>Atributo ms-DS-Auxiliary-Classes

Este atributo enumera las clases auxiliares que se han asociado dinámicamente a un objeto . Este atributo no está asociado a una clase. El sistema lo rellena automáticamente.



| Entrada | Valor |
|-------------------|-----------------------------------------------------------------|
| CN                | ms-DS-Auxiliary-Classes                                         |
| Ldap-Display-Name | msDS-Auxiliary-Classes                                          |
| Size              | \-                                                              |
| Privilegio actualizar  | El sistema establece este valor.                                |
| Frecuencia de actualización  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.4.1458                                         |
| System-Id-Guid    | c4af1073-ee50-4be0-b8c0-89a41fe99bios                            |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
