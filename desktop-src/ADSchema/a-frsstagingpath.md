# <a name="frs-staging-path-attribute"></a>FRS-staging-atributo de ruta de acceso

La ruta de acceso al área de ensayo de la replicación de archivos.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | FRS-staging-ruta de acceso                            |
| Nombre para mostrar de LDAP | fRSStagingPath                              |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.488                      |
| System-ID-GUID    | 1be8f175-a9ff-11d0-afe2-00c04fd930c9        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
