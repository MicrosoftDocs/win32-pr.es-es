# <a name="avencmpvaddseqendcode-property"></a>Propiedad AVEncMPVAddSeqEndCode

Especifica si el codificador agrega un código de fin de secuencia al final de la secuencia. Esta propiedad se aplica a los codificadores de vídeo MPEG.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncMPVAddSeqEndCode**

## <a name="remarks"></a>Observaciones

Si el valor es **Variant \_ true**, el codificador agrega un código de final de secuencia al final de la secuencia.
