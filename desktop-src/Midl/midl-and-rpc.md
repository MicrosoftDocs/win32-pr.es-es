# <a name="midl-and-rpc"></a>MIDL y RPC

Al desarrollar una aplicación RPC, se usa un editor de texto para crear un archivo IDL que define la interfaz entre el cliente y el servidor. El compilador MIDL se usa para generar archivos de código fuente que requieren los programas de cliente y servidor. En los temas siguientes se muestra información sobre el uso del compilador MIDL para aplicaciones RPC:

-   [Archivos generados para una interfaz RPC](files-generated-for-an-rpc-interface.md)
-   [Código auxiliar de destino para plataformas específicas de 32 o 64 bits](targeting-stubs-for-specific-32-bit-or-64-bit-platforms.md)

Para obtener más información, vea [Building RPC Applications](/windows/desktop/Rpc/building-rpc-applications).
