# <a name="neterrorlogwrite"></a>NetErrorLogWrite

La función **NetErrorLogWrite** está obsoleta. Se incluye por compatibilidad con las versiones de 16 bits de Windows. Otras aplicaciones deben usar el [registro de eventos](/windows/desktop/EventLog/event-logging).
