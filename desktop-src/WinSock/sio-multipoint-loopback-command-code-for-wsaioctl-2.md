# <a name="sio_multipoint_loopback-command-code-for-wsaioctl"></a>\_ \_ Código de comando SiO Multipoint loopback para WSAIoctl

Cuando \_ se usan sockets de hoja d en un plano de datos no raíz, es conveniente tener el tráfico que se envía de vuelta en el mismo Socket. El \_ \_ código de comando SiO Multipoint loopback para [**WSAIoctl**](/windows/desktop/api/Winsock2/nf-winsock2-wsaioctl) se usa para habilitar o deshabilitar el bucle invertido de tráfico multipoint.
