# <a name="wmtoolname"></a>WM/ToolName

El atributo **WM/ToolName** contiene el nombre de la aplicación utilizada para crear el archivo. También puede especificar la versión de la aplicación mediante [**WM/ToolVersion**](wm-toolversion.md).

## <a name="global-constant"></a>Constante global

g \_ wszWMToolName
