# <a name="idxtjpegget_borderwidth-method"></a>IDxtAsynceg::get \_ BorderWidth (método)

> [!Note]  
> \[Obsoleto. Esta API puede quitarse de futuras versiones de Windows.\]

 

El método recupera el ancho del borde sólido a lo `get_BorderWidth` largo de los bordes del patrón de borrado.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT get_BorderWidth(
  [out, retval] long *pVal
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*pVal* \[ out, retval\]
</dt> <dd>

Recibe el ancho del borde, en píxeles.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Si este método se realiza correctamente, devuelve **S \_ OK**. De lo contrario, devuelve un código de error **HRESULT.**
