# <a name="transformsatsource-policy"></a>TransformsAtSource Policy

Si esta directiva del sistema [por usuario](system-policy.md) está establecida en "1"; el instalador busca archivos de transformación en la raíz de los orígenes de red de la lista de origen del producto. De forma predeterminada, las transformaciones se almacenan en la carpeta Datos de aplicación del perfil de un usuario.

Windows El instalador interpreta que la directiva TransformsAtSource es la misma que la [directiva TransformsSecure](transformssecure-policy.md).

## <a name="registry-key"></a>Clave del Registro

**HKEY \_ Directivas \_ de** \\ **software de USUARIO** \\ **ACTUAL** \\ **Instalador de** \\ **Windows** \\ **Microsoft**

## <a name="data-type"></a>Tipo de datos

**REG \_ SZ**
