# <a name="rpc-ns-entry-flags-attribute"></a>atributo RPC-ns-Entry-Flags

Marca que indica que la entrada de RPC NS se creó explícitamente.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | RPC-ns-Entry-Flags                   |
| Nombre para mostrar de LDAP | rpcNsEntryFlags                      |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.754               |
| System-ID-GUID    | 80212841-4bdc-11d1-a9c4-0000f80367c1 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
