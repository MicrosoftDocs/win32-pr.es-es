# <a name="dcl_hs_fork_phase_instance_count-sm5---asm"></a>dcl \_ hs \_ fork \_ phase instance count \_ \_ (sm5 - asm)

Declare el recuento de instancias de la fase de bifurcación en una fase de bifurcación del sombreador de casco.



| dcl \_ hs \_ fork \_ phase instance count \_ \_ {1...max 32-bit UINT} |
|-------------------------------------------------------------|



 



| Elemento                                                   | Descripción                           |
|--------------------------------------------------------|---------------------------------------|
| <span id="N"></span><span id="n"></span>*N*<br/> | \[en \] El recuento de instancias.<br/> |

Esta instrucción se aplica a las siguientes fases del sombreador:



| Vértice | Casco | Domain | Geometría | Píxel | Compute |
|--------|------|--------|----------|-------|---------|
|        | X    |        |          |       |         |
