# <a name="setting-public-property-values-on-the-command-line"></a>Establecer valores de propiedades públicas en la línea de comandos

Para establecer una propiedad pública en un valor de cadena literal, incluya la cadena entre comillas.

PROPERTY = "cadena"

Las comillas solo son necesarias si la cadena contiene espacios. Para borrar una propiedad pública en una línea de comandos (si se establece en null), establezca el valor de la propiedad en una cadena vacía. En este caso, se requieren comillas.

PROPIEDAD = "".
