# <a name="wmwmshadowfilesourcefiletype-windows-media-format-11-sdk"></a>WM/WMShadowFileSourceFileType (SDK de Windows Media Format 11)

**WM/WMShadowFileSourceFileType** contiene el tipo de archivo del archivo original del que se deriva el archivo ASF.

## <a name="global-constant"></a>Constante global

g \_ wszWMWMShadowFileSourceFileType

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

El contenido que se encuentra en un formato de archivo distinto de ASF y está protegido con algún tipo de administración de derechos se puede convertir en un archivo de Windows Media protegido por DRM de Windows Media a través de un proceso llamado importación de licencias. Este tipo de archivo ASF debe contener este atributo, así como WM/WMShadowFileSourceDRMType, para realizar un seguimiento de la procedencia del contenido.
