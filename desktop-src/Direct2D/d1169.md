# <a name="d1169-discard-map-option-requires-dynamic-texture"></a>D1169: la opción de asignación descartada requiere textura dinámica

D2D1 \_ map \_ Options \_ discard solo es válido con un mapa de bits creado a partir de una textura dinámica.






 

## <a name="possible-causes"></a>Causas posibles

Direct2D puede generar este mensaje si una aplicación llama a ID2D1Bitmap1:: Map con el \_ \_ \_ atributo descartar opciones de asignación D2D1, pero el recurso de Direct3D subyacente para el mapa de bits no tiene atributos de uso de D3D11 \_ \_ dinámico.

## <a name="possible-fixes"></a>Posibles correcciones

Si una aplicación requiere acceso a un mapa de bits mediante ID2D1Bitmap1:: Map, asegúrese de que el recurso de Direct3D subyacente tiene el \_ atributo dinámico de uso de D3D11 \_ .
