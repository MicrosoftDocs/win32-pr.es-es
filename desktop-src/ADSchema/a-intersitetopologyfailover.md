# <a name="inter-site-topology-failover-attribute"></a>Inter-site-Topology: atributo de conmutación por error

Indica cuánto tiempo debe transcurrir desde la última conexión de mantenimiento para que el generador de topología entre sitios se considere inactivo.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Entre sitios: topología: conmutación por error         |
| Nombre para mostrar de LDAP | interSiteTopologyFailover            |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1248              |
| System-ID-GUID    | b7c69e60-2cc7-11d2-854e-00a0c983f608 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
