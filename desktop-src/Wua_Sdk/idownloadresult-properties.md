# <a name="idownloadresult-properties"></a>Propiedades de IDownloadResult

La interfaz [**IDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadresult) define las siguientes propiedades.



| Propiedad                                         | Descripción                                                                                                     |
|--------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**HResult**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-get_hresult)       | Obtiene el número de código de excepción si se produce un número de código de excepción durante una descarga.                         |
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-get_resultcode) | Obtiene una enumeración [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) que especifica el resultado de una descarga. |
