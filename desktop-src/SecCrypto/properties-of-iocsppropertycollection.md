# <a name="properties-of-iocsppropertycollection"></a>Propiedades de IOCSPPropertyCollection

La interfaz [**IOCSPPropertyCollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) define las siguientes propiedades.



| Propiedad                                                        | Descripción                                              |
|-----------------------------------------------------------------|----------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get__newenum)    | Obtiene un enumerador para un conjunto de propiedades.                   |
| [**Contabiliza**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_count)           | Obtiene el número de propiedades de un conjunto de propiedades.         |
| [**Elemento**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_item)             | Obtiene la propiedad identificada por el índice en un conjunto de propiedades. |
| [**ItemByName**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-get_itembyname) | Obtiene la propiedad identificada por el nombre en un conjunto de propiedades.  |
