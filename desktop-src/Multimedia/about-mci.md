# <a name="about-mci"></a>Acerca de MCI

MCI proporciona a las aplicaciones funciones independientes del dispositivo para controlar periféricos de audio y visuales. La aplicación puede usar MCI para controlar cualquier dispositivo multimedia compatible, incluidos los dispositivos de audio de onda de onda, los secuenciadores MIDI, los dispositivos de audio de CD y los dispositivos de vídeo digital (reproducción de vídeo).
