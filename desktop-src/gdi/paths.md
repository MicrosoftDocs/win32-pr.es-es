# <a name="paths-windows-gdi"></a>Rutas de acceso (Windows GDI)

Una *ruta de acceso* es una o más figuras (o formas) que se rellenan, se indican o se rellenan y se describen. Las rutas de acceso se usan para dibujar y pintar aplicaciones. Las aplicaciones de diseño asistido por PC (CAD) usan rutas de acceso para crear regiones de recorte únicas, dibujar contornos de formas irregulares y rellenar los interiores de formas irregulares. Una forma irregular es una forma formada por curvas de Bézier y líneas rectas. (Las formas normales incluyen puntos suspensivos, círculos, rectángulos y polígonos).
