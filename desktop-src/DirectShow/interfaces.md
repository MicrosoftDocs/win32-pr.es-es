# <a name="directshow-interfaces"></a>DirectShow interfaces

Este tema contiene información de referencia para las interfaces principales ® DirectShow® Microsoft. Para proporcionar una amplia compatibilidad con diversos tipos de hardware, formatos de archivo y operaciones de transmisión, implementa un gran número de interfaces. Algunas las usan las aplicaciones; Otros los usan internamente los filtros y otros DirectShow componentes.

El tema siguiente contiene una lista alfabética de todos los códigos DirectShow interfaces.

-   [Lista alfabética de DirectShow interfaces](alphabetical-list-of-directshow-interfaces.md)

Cualquier aplicación determinada usará solo un pequeño subconjunto de estas interfaces. En los temas siguientes se incluyen interfaces por categoría funcional. No todas las interfaces aparecen en estas páginas y algunas aparecen en más de una página.

-   [Interfaces para compilar gráficos de filtro](interfaces-for-building-filter-graphs.md)
-   [Interfaces para controlar un filtro Graph](interfaces-for-controlling-a-filter-graph.md)
-   [Interfaces de captura y representación de audio](audio-capture-and-rendering-interfaces.md)
-   [DVD Interfaces](dvd-interfaces.md)
-   [Interfaces de captura de vídeo](video-capture-interfaces.md)
-   [Interfaces de codificación y codificación de archivos](file-encoding-and-decoding-interfaces.md)
-   [Interfaces para la representación en vídeo y la superposición](interfaces-for-video-rendering-and-overlay.md)
-   [Interfaces de control de dispositivos externos](external-device-control-interfaces.md)
