# <a name="using-localized-searches"></a>Uso de búsquedas localizadas

Microsoft Windows Search admite la búsqueda completa de Unicode. Puede indizar documentos que usan Unicode y crear consultas de búsqueda que usan identificadores y literales Unicode. Sin embargo, la configuración de idioma de una aplicación o del sistema operativo puede tener efectos de gran alcance en el funcionamiento de la búsqueda de Windows.

Esta sección contiene los siguientes temas:

-   [Configuración regional del sistema y del documento](-search-sql-docandlocalesystemsettings.md)
-   [Especificar idiomas](-search-sql-specifyinglanguages.md)
