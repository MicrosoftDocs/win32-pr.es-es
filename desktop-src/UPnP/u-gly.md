# <a name="u-upnp-apis"></a>U (API de UPnP)

Contiene términos relacionados con UPnP que comienzan por la letra U.

<dl> <dt>

<span id="upnp.u_1_gly"></span><span id="UPNP.U_1_GLY"></span>**UDN**
</dt> <dd>

Representa el nombre de dispositivo único. Un UDN identifica de forma única un dispositivo; se basa en el UUID (identificador único universal).

</dd> <dt>

<span id="upnp.u_2_gly"></span><span id="UPNP.U_2_GLY"></span>**UTL**
</dt> <dd>

Representa el idioma de la plantilla UPnP. El lenguaje de la plantilla UPnP se usa para especificar descripciones de dispositivos o servicios.

</dd> </dl>
