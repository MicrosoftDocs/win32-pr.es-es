# <a name="ui_pkey_color"></a>\_Color de PKEY de IU \_

Identifica la \_ propiedad de color PKEY de la interfaz de usuario \_ .

```
propertyDescription
   name = UI_PKEY_Color
   shellPKey = UI_PKEY_Color
   formatID = 00000400-7363-696e-8441798acf5aebb7
   propID = 400
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Observaciones

\_ \_ Una aplicación usa el color PKEY de la interfaz de usuario para consultar el valor de color del control [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) .

El valor de propiedad es un valor de [COLORREF](../gdi/colorref.md) .

Se omite el byte de orden superior del valor de [COLORREF](../gdi/colorref.md) .
