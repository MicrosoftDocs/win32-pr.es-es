# <a name="d1229-d1229-immersive-call-to-getdesktopdpi"></a>D1229: D1229: llamada envolvente a GetDesktopDpi

Se llamó a ID2D1Factory:: GetDesktopDpi en una aplicación de la tienda Windows. Esto no es recomendable. En su lugar, se debe usar DisplayProperties:: LogicalDpi.






 

## <a name="possible-causes"></a>Causas posibles

Se llamó a ID2D1Factory:: GetDesktopDpi en una aplicación de la tienda Windows.

## <a name="possible-fixes"></a>Posibles correcciones

DisplayProperties:: LogicalDpi se debe usar para obtener el PPP en una aplicación de la tienda Windows.
