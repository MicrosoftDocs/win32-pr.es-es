# <a name="endif-sm4---asm"></a>endif (SM4-ASM)

Finaliza una instrucción [If](if--sm4---asm-.md) .



| endif |
|-------|



 

## <a name="remarks"></a>Observaciones

En el ejemplo siguiente se muestra cómo usar la instrucción endif.

``` syntax
                if     // any of the various forms of if* statements
                   ...
                else   // (optional)
                   ...
                endif
```

El formato de token contiene el desplazamiento de la instrucción **If** correspondiente en el sombreador como una comodidad.

Esta instrucción se aplica a las siguientes fases del sombreador:



| Sombreador de vértices | Sombreador de geometría | Sombreador de píxeles |
|---------------|-----------------|--------------|
| x             | x               | x            |
