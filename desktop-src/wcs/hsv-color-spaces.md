# <a name="hsv-color-spaces"></a>Espacios de color HSV

Los intérpretes suelen usar [espacios de colores](c.md) de matiz, saturación y valor (HSV). "Matiz" es lo que se suele considerar como color. Es el atributo de un color mediante el cual se le asigna un nombre como "rojo" o "azul". "Valor" es otra palabra para "luminosidad", el atributo de un color que parece equivalente a algún tono de gris entre blanco y negro. La saturación es una medida de cómo aparece un color diferente de un gris con la misma [luminosidad](b.md). La saturación cero indica que no hay matiz, solo la escala de grises. El espacio de colores HSV está normalizado.

![espacio de colores HSV](images/hsvline.png)

En la ilustración anterior se muestra un dibujo de línea del espacio HSV en forma de hexcone. Cada una de sus secciones transversales es un hexágono. En los vértices de cada sección transversal se encuentran los colores rojo, amarillo, verde, aguamarina, azul y magenta. Para especificar un color en el espacio HSV, se indica un ángulo de matiz, el nivel de croma y el nivel de luminosidad. Un ángulo de matiz de cero es rojo. El ángulo de matiz aumenta en sentido contrario a las agujas del reloj. Los colores complementarios están separados por 180.

Los espacios de colores HSV pueden ser dependientes del dispositivo o independientes del dispositivo.
