# <a name="avencvideopixelaspectratio-property"></a>Propiedad AVEncVideoPixelAspectRatio

Especifica la relación de aspecto de píxeles.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncVideoPixelAspectRatio**

## <a name="property-value"></a>Valor de propiedad

Los 16 bits superiores del valor contienen el ancho y los 16 bits inferiores contienen el alto.
