# <a name="iautomaticupdates-methods"></a>Métodos IAutomaticUpdates

La interfaz [**IAutomaticUpdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) define los siguientes métodos.



| Método                                                             | Descripción                                                         |
|--------------------------------------------------------------------|---------------------------------------------------------------------|
| [**DetectNow**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-detectnow)                   | Inicia la actualización automática Si aún no se ha iniciado.            |
| [**EnableService**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-enableservice)           | Habilita todos los componentes que Actualizaciones automáticas requiere.         |
| [**Pausar**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-pause)                           | Pausa las actualizaciones automáticas.                                          |
| [**Reanudar**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-resume)                         | Reinicia las actualizaciones automáticas si está en pausa.                        |
| [**ShowSettingsDialog**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-showsettingsdialog) | Muestra un cuadro de diálogo que contiene la configuración de Actualizaciones automáticas. |
