# <a name="d1152-command-list-closed-twice"></a>D1152: la lista de comandos se cerró dos veces

Se intentó cerrar una lista de comandos por segunda vez.






 

## <a name="possible-causes"></a>Causas posibles

Después de rellenar una lista de comandos, solo se debe cerrar una vez. Direct2D considera que las llamadas subsiguientes a ID2D1CommandList:: Close son un error.

## <a name="possible-fixes"></a>Posibles correcciones

Para evitar este mensaje, asegúrese de que una lista de comandos solo se cierra una vez. Si una aplicación usa las devoluciones de llamada o rutas de código recursivas para interactuar con las listas de comandos, asegúrese de que su comportamiento es correcto.
