# <a name="data-types-hlsl"></a>Tipos de datos (HLSL)

HLSL admite muchos tipos de datos intrínsecos diferentes. En esta tabla se muestran los tipos que se van a usar para definir variables de sombreador.



|                                                                                                                         |                                            |
|-------------------------------------------------------------------------------------------------------------------------|--------------------------------------------|
| Usar este tipo intrínseco                                                                                                 | Para definir esta variable de sombreador             |
| [Scala](dx-graphics-hlsl-scalar.md)                                                                                   | Escalar de un componente                       |
| [Vector](dx-graphics-hlsl-vector.md), [matriz](dx-graphics-hlsl-matrix.md)                                            | Vector o matriz de varios componentes        |
| [Muestra](dx-graphics-hlsl-sampler.md), [textura](dx-graphics-hlsl-texture.md) o [búfer](dx-graphics-hlsl-buffer.md)   | Muestra, textura u objeto de búfer         |
| [Struct](dx-graphics-hlsl-struct.md), [definido por el usuario](dx-graphics-hlsl-user-defined.md)                                | Estructura personalizada o typedef                |
| Array                                                                                   | Expresiones escalares literales declaradas que contienen la mayoría de los demás tipos                       |
| [Objeto de estado](dx-graphics-hlsl-state-object.md) | Representaciones de HLSL de objetos de estado |
