# <a name="values-for-guid_compartment_transitoryextension"></a>Valores para el \_ compartimiento de GUID \_ TRANSITORYEXTENSION

Los siguientes valores del compartimiento \_ de GUID \_ TRANSITORYEXTENSION se usan para controlar el comportamiento de la extensión transitoria.

<dl> <dt>

<span id="TF_TRANSITORYEXTENSION_NONE"></span><span id="tf_transitoryextension_none"></span>**TF \_ TRANSITORYEXTENSION \_ ninguno**
</dt> <dd>

Este valor detiene la extensión transitoria.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_FLOATING"></span><span id="tf_transitoryextension_floating"></span>**TF \_ TRANSITORYEXTENSION \_ flotante**
</dt> <dd>

Este valor inicia la extensión transitoria con la interfaz de usuario flotante.

</dd> <dt>

<span id="TF_TRANSITORYEXTENSION_ATSELECTION"></span><span id="tf_transitoryextension_atselection"></span>**TF \_ TRANSITORYEXTENSION \_ ATSELECTION**
</dt> <dd>

Este valor inicia la extensión transitoria con la interfaz de usuario emergente en la dirección IP o la selección del administrador de documentos primario.

</dd> </dl>
