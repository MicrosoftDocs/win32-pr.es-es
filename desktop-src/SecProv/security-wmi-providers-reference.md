# <a name="security-wmi-providers-reference"></a>Referencia de proveedores de WMI de seguridad

Las páginas de referencia de WMI de seguridad contienen descripciones detalladas del proveedor WMI del volumen cifrable y del proveedor WMI Módulo de plataforma segura (TPM).

Los proveedores de WMI de seguridad incluyen los siguientes proveedores.



| Sección                                                                        | Descripción                                                                                                                                  |
|--------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------|
| [Proveedor de cifrado de unidad BitLocker](bitlocker-drive-encryption-provider.md) | Documentación de la API para administrar Cifrado de unidad BitLocker (BDE) mediante la clase [**Win32 \_ EncryptableVolume**](win32-encryptablevolume.md) . |
| [Proveedor de Módulo de plataforma segura](trusted-platform-module-provider.md)       | Documentación de la API para administrar el Módulo de plataforma segura (TPM) mediante la clase [**Win32 \_ TPM**](win32-tpm.md) .                            |
