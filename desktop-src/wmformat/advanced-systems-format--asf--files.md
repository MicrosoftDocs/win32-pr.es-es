# <a name="advanced-systems-format-asf-files"></a>Archivos de formato de sistema avanzado (ASF)

Todos los objetos del SDK de Windows Media Format funcionan con archivos que se ajustan a la estructura de archivos Advanced Systems Format (ASF). En esta documentación se hace referencia a estos archivos como archivos ASF. Para obtener más información acerca de la estructura de los archivos ASF, consulte [información general del formato ASF](overview-of-the-asf-format.md).

A veces, el término *archivo de Windows Media* se usa para describir los archivos creados con este SDK u otras tecnologías de Windows Media. No es preciso llamar a un archivo ASF un archivo de Windows Media a menos que solo contenga contenido codificado mediante códecs de Windows Media. Para obtener más información, vea las instrucciones para el uso de las extensiones de archivo. WMA y. WMV en la sección [pautas de extensión de nombre de archivo](file-name-extension-guidelines.md) .
