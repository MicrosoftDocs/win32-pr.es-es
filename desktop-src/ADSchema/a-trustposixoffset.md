# <a name="trust-posix-offset-attribute"></a>Atributo Trust-Posix-Offset

Desplazamiento de la interfaz de sistema operativo portable (POSIX) para el dominio de confianza.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Trust-Posix-Offset                   |
| Ldap-Display-Name | trustPosixOffset                     |
| Size              | 4 bytes                              |
| Privilegio actualizar  | El sistema establece este valor.     |
| Frecuencia de actualización  | Cuando se crea una nueva confianza.         |
| Attribute-Id      | 1.2.840.113556.1.4.134               |
| System-Id-Guid    | bf967a5e-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeración**](s-enumeration.md) |
