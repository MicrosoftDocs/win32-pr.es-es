# <a name="teletex-terminal-identifier-attribute"></a>Atributo Teletex-Terminal-Identifier

Especifica el identificador de terminal teletex y, opcionalmente, los parámetros para un terminal de Teletex asociado a un objeto .



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Teletex-Terminal-Identifier                           |
| Ldap-Display-Name | teletexTerminalIdentifier                             |
| Size              | \-                                                    |
| Actualizar privilegios  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 2.5.4.22                                              |
| System-Id-Guid    | bf967a4a-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
