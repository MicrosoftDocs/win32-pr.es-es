# <a name="revokeinteractivesessionaccess-method-of-the-msvm_terminalservice-class"></a>Método RevokeInteractiveSessionAccess de la \_ clase TerminalService de MSVM

Revoca el acceso a la sesión interactiva de la máquina virtual a partir de la lista de confianzas especificada.

## <a name="syntax"></a>Sintaxis


```mof
uint32 RevokeInteractiveSessionAccess(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [in]  string                 Trustees[],
  [out] CIM_ConcreteJob    REF Job
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*ComputerSystem* \[ de\]
</dt> <dd>

Una referencia a una instancia de la clase de [**\_ ComputerSystem de CIM**](cim-computersystem.md) que representa la máquina virtual a la que se revocará el acceso.

</dd> <dt>

*Confianzas* \[ de\]
</dt> <dd>

Matriz de cadenas, cada una de las cuales identifica un administrador de confianza cuyos derechos de acceso se revocarán. Los identificadores de confianza deben especificarse en formato compatible con SAM de Windows o en formato de cadena de SID de Windows.

</dd> <dt>

*Trabajo* \[ de enuncia\]
</dt> <dd>

Si la operación se realiza de forma asincrónica, este método devolverá 4096 y este parámetro contendrá una referencia a un objeto derivado de [**\_ ConcreteJob CIM**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
