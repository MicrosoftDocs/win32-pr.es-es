# <a name="locale_iuseutf8legacyoemcp"></a>Configuración regional \_ IUSEUTF8LEGACYOEMCP

**A partir de la siguiente versión de Windows:** En su lugar, se recomienda que las aplicaciones usen Unicode. En los equipos que el usuario ha configurado para usar una página de códigos UTF-8, LOCALE_IUSEUTF8LEGACYOEMCP es la página de códigos OEM heredada que se habría usado en un equipo Windows heredado que no es UTF-8.
