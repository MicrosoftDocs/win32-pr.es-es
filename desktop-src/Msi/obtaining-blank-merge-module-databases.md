# <a name="obtaining-blank-merge-module-databases"></a>Obtener las bases de datos del módulo de combinación en blanco

Obtener una base de datos de módulo de mezcla en blanco. Puede usar el archivo Schema. MSM proporcionado con el SDK de Windows Installer como base de datos de inicio para el módulo de combinación. Para obtener más información, vea [componentes de Windows SDK para desarrolladores de Windows Installer](platform-sdk-components-for-windows-installer-developers.md).

Los desarrolladores deben crear módulos de combinación con el esquema de base de datos más simple que instala sus componentes. El uso de un esquema simple garantiza la mayor compatibilidad para el módulo de combinación. La combinación de un módulo de combinación en un paquete de instalación con un esquema de base de datos diferente normalmente produce conflictos de combinación.

Para obtener una lista completa de todas las tablas necesarias y opcionales en los módulos de combinación, vea [Merge Module Database tables](merge-module-database-tables.md).
