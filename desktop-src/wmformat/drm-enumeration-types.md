# <a name="microsoft-windows-media-drm-client-enumeration-types"></a>Tipos de enumeración de cliente DRM de Microsoft Windows Media

En la tabla siguiente se describen las enumeraciones admitidas por las API extendidas del cliente DRM de Microsoft Windows Media.



| Enumeración                                                                      | Descripción                                                                                                                                                   |
|----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**la \_ acción \_ DRM \_ permitió \_ resultados de la consulta**](drm-action-allowed-query-results.md) | Lo usa la interfaz [**IWMDRMLicenseQuery:: QueryActionAllowed**](iwmdrmlicensequery-queryactionallowed.md) para especificar la razón por la que no se permite una acción. |
| [**TIPO de los atributos de DRM \_ \_**](drm-attr-datatype.md)                                 | Define los tipos de datos utilizados para los atributos y propiedades de DRM.                                                                                                |
| [**\_tipo criptográfico de DRM \_**](drm-crypto-type.md)                                     | Define los tipos de algoritmos criptográficos que se pueden usar para cifrar el contenido.                                                                                |
| [**\_Estado http de DRM \_**](drmdrm-http-status.md)                                  | Define un intervalo de Estados HTTP para una solicitud DRM.                                                                                                             |
| [**\_Estado de individualización de DRM \_**](drmdrm-individualization-status.md)        | Define los Estados válidos para la individualización de DRM.                                                                                                           |
| [**\_categoría de \_ Estado de licencia de DRM \_**](drmdrm-license-state-category.md)           | Especifica el tipo de restricción de licencia que se describe mediante una estructura de [**datos de estado de licencia de DRM \_ \_ \_**](drmdrm-license-state-data.md) .                    |
| [**Estado de MSDRM \_**](msdrm-status.md)                                            | Define las condiciones de estado para el subsistema DRM.                                                                                                              |
| [**\_tipo de directiva WMDRMNET \_**](wmdrmnet-policy-type.md)                           | Enumera los tipos de directivas que están disponibles para las operaciones de Windows Media DRM para dispositivos de red.                                                          |
| [**derechos de WMT \_**](drm-wmt-rights.md)                                            | Define los derechos que se pueden especificar en una licencia de DRM.                                                                                                    |
