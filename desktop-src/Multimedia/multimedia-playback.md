# <a name="multimedia-playback"></a>Reproducción multimedia

La función [**MCIWndCreate**](/windows/desktop/api/Vfw/nf-vfw-mciwndcreatea) proporciona los medios para controlar una ventana de MCIWnd y el dispositivo asociado a él. En general, esta función registra la clase de ventana MCIWnd y crea una ventana de MCIWnd para usar los servicios de MCI. En esta sección se describe cómo usar **MCIWndCreate** para realizar las siguientes tareas:

-   [Agregar User-Controlled reproducción](adding-user-controlled-playback.md)
-   [Automatizar la reproducción](automating-playback.md)
-   [Usar estilos de ventana para cambiar la ventana de MCIWnd](using-window-styles-to-change-the-mciwnd-window.md)
-   [Permitir al usuario especificar dispositivos y archivos](allowing-the-user-to-specify-devices-and-files.md)
