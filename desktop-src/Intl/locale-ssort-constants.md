# <a name="locale_ssort_constants"></a>constantes de SSORT de configuración regional \_ \_

En este tema se definen las constantes de SSORT de configuración regional \_ \* utilizadas por NLS.



| Value               | Significado                                                                                                                                                                                                           |
|---------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| configuración regional \_ SSORTLOCALE | **Windows 7 y versiones posteriores:** Nombre de la configuración regional que se va a usar para la ordenación o el comportamiento de mayúsculas y minúsculas.                                                                                                                                |
| configuración regional \_ SSORTNAME   | **Windows Me/98, windows 2000:** Nombre localizado completo de la ordenación para el identificador de configuración regional especificado, en función del idioma del shell. Esta constante se utiliza para determinar el comportamiento de la grafía y la ordenación. |
