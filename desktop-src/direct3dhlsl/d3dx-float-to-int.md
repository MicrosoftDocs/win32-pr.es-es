# <a name="d3dx_float_to_int-function"></a>Función D3DX \_ FLOAT \_ to \_ INT

``` syntax
INT D3DX_FLOAT_to_INT(
   FLOAT _V,
   FLOAT _Scale
);
```

<dl> <dt>

</dd> <dt>
