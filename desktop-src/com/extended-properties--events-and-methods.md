# <a name="extended-properties-events-and-methods"></a>Propiedades extendidas, eventos y métodos

Los contenedores de controles ActiveX no son necesarios para admitir controles extendidos. Sin embargo, si el contenedor de control admite propiedades extendidas, debe admitir el siguiente conjunto mínimo:

-   Visible
-   Parent
-   Valor predeterminado
-   Cancelar

Actualmente, las propiedades extendidas, los eventos y los métodos no tienen DISPID estándar.
