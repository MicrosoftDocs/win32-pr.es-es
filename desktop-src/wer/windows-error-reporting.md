# <a name="windows-error-reporting"></a>Informe de errores de Windows

La característica informe de errores permite a los usuarios notificar a Microsoft los errores de aplicación, los errores de kernel, las aplicaciones que no responden y otros problemas específicos de la aplicación. Microsoft puede usar la característica de informes de errores para proporcionar a los clientes información, soluciones o actualizaciones de solución de problemas específicos. Los desarrolladores pueden utilizar esta infraestructura para recibir información que se puede usar para mejorar sus aplicaciones.

Los usuarios pueden habilitar los informes de errores a través de la interfaz de usuario de Windows. Pueden elegir notificar los errores de aplicaciones específicas. Los administradores pueden invalidar esta configuración mediante directiva de grupo.

Los desarrolladores pueden registrarse con el [programa de aplicación de escritorio de Windows](../appxpkg/windows-desktop-application-program.md) para obtener información sobre los problemas que experimentan los clientes con sus aplicaciones y ayudar a los clientes a corregir estos problemas. Los desarrolladores también pueden usar la [recuperación y el reinicio](/windows/desktop/Recovery/application-recovery-and-restart-portal) de la aplicación para asegurarse de que los clientes no pierdan los datos cuando se bloquee la aplicación y permitan a los usuarios volver rápidamente a sus tareas.
