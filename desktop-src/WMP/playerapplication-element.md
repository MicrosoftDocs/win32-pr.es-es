# <a name="playerapplication-element"></a>Elemento PLAYERAPPLICATION

Algunas máscaras pueden usar el elemento **PLAYERAPPLICATION** para tener acceso al objeto **PLAYERAPPLICATION** . Este elemento proporciona una funcionalidad similar al atributo global **playerApplication** . Este elemento solo lo usan las máscaras que se muestran en las aplicaciones de C++ que incrustan una instancia remota de Windows Media Player control.

El elemento **PLAYERAPPLICATION** admite los siguientes atributos.



| Atributo                                              | Descripción                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [hasDisplay](playerapplication-hasdisplay-scr.md)     | Recupera un valor que indica si el vídeo puede mostrarse a través del control remoto de Windows Media Player. |
| [playerDocked](playerapplication-playerdocked-scr.md) | Recupera un valor que indica si el Media Player de Windows está en un estado acoplado.                          |
