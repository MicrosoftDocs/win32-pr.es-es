# <a name="locale_skeyboardstoinstall"></a>configuración regional \_ SKEYBOARDSTOINSTALL

**Windows Vista y versiones posteriores:** Una lista delimitada por signos de punto y coma de teclados que pueden instalarse para la configuración regional y que Windows usa internamente. No hay ningún límite en el número de caracteres permitidos para esta cadena. Para recuperar el nombre del identificador de configuración regional de entrada activo (anteriormente denominado distribución del teclado), la aplicación puede llamar a la función [**GetKeyboardLayoutName**](/windows/win32/api/winuser/nf-winuser-getkeyboardlayoutnamea) .
