# <a name="extended-telephony-phone-functions"></a>Funciones de teléfono de telefonía extendidas

Los servicios de teléfono extendidos (o los servicios de teléfono Device-Specific) incluyen todas las extensiones del TAPI definido por el proveedor de servicios. TAPI define un mecanismo que permite a los proveedores de proveedores de servicios ampliar TAPI con extensiones específicas del dispositivo. TAPI solo define el mecanismo de extensión y, al hacerlo, se proporciona acceso a las extensiones específicas del dispositivo. TAPI no define su comportamiento, que está completamente definido por el proveedor de servicios.

TAPI consta de definiciones de constantes de datos de marcas de bits y escalares, estructuras de datos, funciones y mensajes. Se definen los procedimientos que permiten a un proveedor extender la mayoría de ellos, como se describe en los temas siguientes.

-   [Constantes de datos escalares](scalar-data-constants.md)
-   [Constantes de datos de marca de bits](bit-flag-data-constants.md)
-   [Funciones y mensajes](functions-and-messages.md)
