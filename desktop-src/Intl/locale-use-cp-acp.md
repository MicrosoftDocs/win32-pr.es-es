# <a name="locale_use_cp_acp"></a>configuración regional \_ use \_ CP \_ ACP

**Windows Me/98, windows 2000:** Página de códigos ANSI de Windows (ACP) predeterminada del sistema en lugar de la página de códigos de configuración regional utilizada para la traducción de cadenas. Vea [identificadores de páginas de códigos](code-page-identifiers.md) para obtener una lista de ANSI y otras páginas de códigos.
