# <a name="windows-touch-samples"></a>Ejemplos de Windows Touch

Describe la ruta de acceso a los ejemplos de Windows Touch y proporciona una breve introducción a los ejemplos.

Al instalar los [ejemplos táctiles de Windows 7](), puede elegir dónde están instalados los ejemplos. Por ejemplo, si va a instalar en la unidad C, el SDK se establecerá de forma predeterminada en "C: \\ archivos de programa \\ Microsoft SDKs \\ Windows \\ v 7.0 \\ Samples". Dentro del directorio de ejemplos está la carpeta Touch. Esta carpeta contiene todos los ejemplos de Windows Touch.

Los siguientes ejemplos de Windows Touch se incluyen con el [SDK de Windows 7]().

- [Ejemplo de manipulación e inercia](manipulation-and-inertia-sample.md)
- [Ejemplo de gesto de toque de Windows (MTGestures)](windows-touch-gesture-sample--mtgestures-.md)
- [Gestos táctiles de Windows en el ejemplo de C# (MTGesturesCS)](windows-touch-gestures-in-c--sample--mtgesturescs-.md)
- [Ejemplo de manipulación táctil de Windows (MTManipulation)](windows-touch-manipulation-sample--mtmanipulation-.md)
- [Ejemplo de Bloc Dete táctil de Windows (MTScratchpadWMTouch)](windows-touch-scratchpad-sample--mtscratchpadwmtouch-.md)
- [Ejemplo de Bloc de funciones de Windows Touch en C# (MTScratchpadWMTouchCS)](windows-touch-scratchpad-sample-in-c---mtscratchpadwmtouchcs-.md)
- [Bloc de entrada táctil de Windows con el ejemplo de lápiz óptico en tiempo real (MTScratchpadRTStylus)](windows-touch-scratchpad-using-the-real-time-stylus-sample--mtscratchpadrtstylus-.md)
- [Bloc de pantalla táctil de Windows mediante el lápiz en tiempo real en el ejemplo de C# (MTScratchpadRTStylusCS)](windows-touch-scratchpad-using-the-real-time-stylus-in-c--sample--mtscratchpadrtstyluscs-.md)
