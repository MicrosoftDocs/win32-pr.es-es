# <a name="d1238-source-transform-single-channel-srgb"></a>D1238: transformación de origen SRGB de canal único

ID2D1SourceTransform no permite combinar el \_ \_ sRGB de precisión del búfer \_ de D2D1 con la \_ profundidad 1 del canal de D2D1 \_ \_ .






 

## <a name="possible-causes"></a>Causas posibles

La aplicación especificó D2D1 de \_ precisión de búfer \_ \_ sRGB y D2D1 \_ \_ \_ de profundidad de canal 1 en un ID2D1SourceTransform.

## <a name="possible-fixes"></a>Posibles correcciones

\_ \_ \_ \_ \_ \_ En su lugar, se debe usar la profundidad de canal de D2D1 predeterminada o la profundidad de canal de D2D1 4.
