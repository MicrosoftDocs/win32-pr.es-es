# <a name="finding-the-server-program"></a>Buscar el programa de servidor

Una vez que el tiempo de ejecución de RPC del cliente se conecta a un sistema host de servidor solicitado en el identificador de enlace, la biblioteca en tiempo de ejecución de RPC del cliente encuentra el proceso de servidor. Para ello, consulta la asignación de punto de conexión en el sistema host del servidor. La asignación de punto de conexión contiene información sobre el punto de conexión al que escucha el servidor.
