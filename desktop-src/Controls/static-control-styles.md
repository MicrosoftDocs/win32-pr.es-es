# <a name="static-control-styles"></a>Estilos de control estáticos

Para crear un control estático mediante la función [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) o [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) , especifique la clase estática, las constantes de estilo de ventana apropiadas y una combinación de los siguientes estilos de control estáticos.

## <a name="remarks"></a>Observaciones

Para usar un estilo que trunque palabras o cadenas con puntos suspensivos para texto multilínea, [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) o [**DrawText**](/windows/desktop/api/winuser/nf-winuser-drawtext) se pueden usar en un control dibujado por el propietario con marcas de formato **DT \_ WORDBREAK** y **DT \_ ENDELLIPSIS** set.
