# <a name="adsi-error-codes"></a>Códigos de error ADSI

Todos los códigos de error ADSI se devuelven como valores **HRESULT** de com. Se pueden agrupar en los cuatro tipos siguientes:

-   [Códigos de error COM genéricos](generic-com-error-codes.md)
-   [Códigos de error ADSI genéricos](generic-adsi-error-codes.md)
-   [Códigos de error de Win32 para ADSI](win32-error-codes-for-adsi.md)
-   [Códigos de error LDAP para ADSI](ldap-error-codes-for-adsi.md)

Además, algunas interfaces proporcionan información adicional sobre el error, que se conoce como mensajes de error extendidos de ADSI, que se puede obtener mediante la función [**ADsGetLastError**](/windows/desktop/api/Adshlp/nf-adshlp-adsgetlasterror) .

Por último, el [código de ejemplo](code-example-for-working-with-adsi-error-messages.md) de C++ que se muestra en esta sección muestra cómo trabajar con los mensajes de error de ADSI.
