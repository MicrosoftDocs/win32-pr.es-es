# <a name="inkpicturemouseleave-event"></a>Evento InkPicture. MouseLeave

Se produce cuando el puntero del mouse sale del control [InkPicture](inkpicture-control-reference.md) .

## <a name="syntax"></a>Sintaxis


```C++
void MouseLeave();
```



## <a name="parameters"></a>Parámetros

Este evento no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este evento no devuelve un valor.

## <a name="remarks"></a>Observaciones

Este método de evento se define en la interfaz **\_ IInkPictureEvents** . La interfaz **\_ IInkPictureEvents** implementa la interfaz [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch) con un identificador de DISPID \_ IPEMouseLeave.
