# <a name="windows-7-developer-guide"></a>Guía del desarrollador de Windows 7

La creación de aplicaciones fáciles de usar, visualmente atractivas y ofrecer un alto rendimiento es un desafío al que se enfrentan los desarrolladores cada día. Las aplicaciones innovadoras pueden mejorar considerablemente la experiencia del usuario, lo que permite a las empresas diferenciar sus servicios y soluciones. Sin embargo, cada vez más se solicita a los desarrolladores más en menos tiempo, a la vez que optimizan los requisitos de potencia y rendimiento de sus aplicaciones.

La plataforma Windows 7 facilita a los desarrolladores la creación de aplicaciones atractivas y fáciles de usar, ya que proporciona herramientas conocidas y características de desarrollo enriquecidas que les permiten aprovechar las capacidades más recientes de los equipos.

Para obtener una versión descargable de la guía del desarrollador de Windows 7, visite la [Galería de código](/samples/browse/?redirectedfrom=MSDN-samples) de MSDN.

-   [Experiencias de aplicación más enriquecidas](richer-application-experiences.md)
-   [Lo mejor de Windows y la web](the-best-of-windows-and-the-web.md)
