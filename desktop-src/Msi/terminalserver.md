# <a name="terminalserver-property"></a>Propiedad TerminalServer

El instalador establece la **propiedad TerminalServer** cuando el sistema es un servidor con Terminal Services.

## <a name="default-value"></a>Valor predeterminado

Ninguno. La propiedad no está definida cuando no se ejecuta en Terminal Services.

## <a name="remarks"></a>Observaciones

La [**propiedad RemoteAdminTS**](remoteadmints.md) solo se puede establecer en Microsoft Windows 2000 o posterior. La **propiedad TerminalServer** se puede establecer en Windows 2000.
