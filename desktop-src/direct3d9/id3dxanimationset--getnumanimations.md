# <a name="id3dxanimationsetgetnumanimations-method"></a>ID3DXAnimationSet:: GetNumAnimations (método)

Obtiene el número de animaciones en el conjunto de animaciones.

## <a name="syntax"></a>Sintaxis


```C++
UINT GetNumAnimations();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Número de animaciones en el conjunto de animaciones.
