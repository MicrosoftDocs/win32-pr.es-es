# <a name="community-names"></a>Nombres de comunidad

Un nombre de comunidad identifica una colección de administradores y agentes SNMP. El uso de un nombre de comunidad proporciona seguridad primitiva y comprobación de contexto para agentes y administradores que reciben solicitudes e inician operaciones de captura. Un agente no aceptará una solicitud de un administrador fuera de la comunidad.

Tenga en cuenta que las recomendaciones de seguridad estándar recomiendan el uso de SNMP excepto en redes de confianza, ya que el protocolo, por diseño, proporciona una seguridad mínima. Para obtener más información, consulte Configuración de la [seguridad de SNMP](configuring-snmp-security.md).
