# <a name="for-more-information-wmp-sdk"></a>Para obtener más información (SDK de WMP)

-   Para obtener más información acerca del desarrollo de soluciones multimedia digitales, vea el [Centro para desarrolladores de Microsoft Windows Media](https://msdn.microsoft.com/windowsmedia/default.aspx).
-   Para obtener más información acerca de Windows Media, vea el [sitio web de Windows Media](https://support.microsoft.com/help/18612/windows-media-player).
-   Para obtener más información acerca de Windows Media Player, consulte la [Página Web](https://support.microsoft.com/help/18612/windows-media-player) de Windows Media Player en el sitio web de Windows Media.
-   Para descargar el Windows SDK, que incluye el SDK de Windows Media Player, vaya a la página [Windows SDK](https://msdn.microsoft.com/windows/bb980924.aspx) del centro para desarrolladores de Windows.
-   Para obtener más información acerca de la programación con tecnologías de Microsoft, incluidas las tecnologías de Windows Media y los kits de desarrollo de software, consulte [MSDN Library]().
-   La versión actual del SDK de Windows Media Player es la versión 12. Para ver los problemas relacionados con el SDK de Windows Media Player 11, consulte el [archivo Léame del SDK de windows media Player 11](/previous-versions/aa905044(v=msdn.10)).
