# <a name="the-performance-monitoring-provider"></a>El proveedor de supervisión de rendimiento

Los datos de las bibliotecas de rendimiento están disponibles como [clases de contador de rendimiento](/windows/desktop/CIMWin32Prov/performance-counter-classes) de WMI a través del [proveedor WMIPerfClass](wmiperfclass-provider.md) y el [proveedor WMIPerfInst](wmiperfinst-provider.md).
