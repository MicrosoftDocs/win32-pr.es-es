# <a name="nlm-structures"></a>Estructuras de NLM

Las siguientes estructuras admiten la API del administrador de lista de redes:



| Estructura                                                           | Descripción                                                          |
|---------------------------------------------------------------------|----------------------------------------------------------------------|
| [**\_información de perfil simulado de NLM \_ \_**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_simulated_profile_info) | Representa la información de perfil que se va a utilizar durante la simulación de costos.    |
| [**\_SOCKADDR NLM**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_sockaddr)                               | Representa la dirección de destino de IPv4/IPv6.                        |
| [**\_datos de uso de NLM \_**](/windows/desktop/api/Netlistmgr/ns-netlistmgr-nlm_usage_data)                          | Representa información que indica el uso de datos de un plan de datos. |
| [**\_Estado del \_ plan de datos de NLM \_**](https://www.bing.com/search?q=**NLM\_DATA\_PLAN\_STATUS**)             | Representa el estado actual de un plan de datos.                        |
