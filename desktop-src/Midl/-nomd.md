# <a name="nomd-switch"></a>Modificador /nomd

El **modificador /nomd** especifica que no se genera ningún archivo de metadatos (.winmd).

``` syntax
midlrt /nomd
```

## <a name="switch-options"></a>Cambiar opciones

Este modificador no tiene parámetros.

## <a name="remarks"></a>Observaciones

## <a name="examples"></a>Ejemplos

**midlrt /nomd**
