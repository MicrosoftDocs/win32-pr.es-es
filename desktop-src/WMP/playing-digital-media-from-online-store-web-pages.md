# <a name="playing-digital-media-from-online-store-web-pages"></a>Reproducir medios digitales desde páginas web de la tienda en línea

Puede reproducir contenido multimedia digital de las páginas web de la tienda en línea. Simplemente inserte el control Media Player de Windows tal y como se describe en [usar el control de windows media Player en una página web](using-the-windows-media-player-control-in-a-web-page.md). Al hacer esto en un panel de tareas de servicio, el control incrustado se conecta al programa de Media Player de Windows de forma similar al modo remoto (vea [la comunicación remota del control de windows media Player](remoting-the-windows-media-player-control.md)). Esto significa que los usuarios pueden usar los controles de transporte del reproductor en modo completo para controlar la reproducción de archivos multimedia digitales en la Página Web.

Se recomienda establecer **uiMode** en "none" o "invisible" al insertar el control Player en una página web del panel de tareas de servicio.
