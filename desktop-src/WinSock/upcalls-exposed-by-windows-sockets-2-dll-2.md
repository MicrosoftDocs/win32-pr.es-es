# <a name="upcalls-exposed-by-windows-sockets-2-dll"></a>Llamadas al método expuesto por Windows Sockets 2 DLL

En esta sección se enumeran las llamadas que los proveedores de servicios pueden convertir en el cliente de Windows Sockets. Los proveedores de servicios reciben una tabla de envío de llamadas como parámetro a la función [**WSPStartup**](/windows/desktop/api/Ws2spi/nf-ws2spi-wspstartup) y usan entradas en esta tabla para hacer las llamadas. Por lo tanto, un cliente no necesita exportar sus funciones WPU.

No es obligatorio que los proveedores utilicen todas estas llamadas. En la tabla siguiente se indican las llamadas que se deben usar y las que son opcionales.

| Función                                                               | Descripción                                                                                                              | Status                         | Significado                                                                                                                                                                          |
|------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|--------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WPUCloseEvent**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpucloseevent)                               | Cierra un identificador de objeto de evento abierto.                                                                                      | Opcional.                      | En su lugar, el proveedor puede usar una llamada de Windows adecuada.                                                                                                                     |
| [**WPUCloseSocketHandle**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpuclosesockethandle)                 | Cierra un identificador de socket asignado por el archivo DLL de Windows Sockets.                                                             | Obligatorio.                      | Ws2 \_32.dll necesita consultar o modificar la información de estado interna asociada con el identificador de socket.                                                                       |
| [**WPUCloseThread**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpuclosethread)                             | Cierra un identificador de subproceso para un subproceso de servicio interno.                                                                       |                                |                                                                                                                                                                                  |
| [**WPUCompleteOverlappedRequest**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpucompleteoverlappedrequest) | Proporciona una notificación de finalización de e/s superpuesta en la que el mecanismo de finalización es algo distinto de APC en modo de usuario.    |                                |                                                                                                                                                                                  |
| [**WPUCreateEvent**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpucreateevent)                             | Crea un nuevo objeto de evento.                                                                                              | Opcional.                      | En su lugar, el proveedor puede usar una llamada de Windows adecuada.                                                                                                                        |
| [**WPUCreateSocketHandle**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpucreatesockethandle)               | Crea un nuevo identificador de socket para los proveedores de nonIFS.                                                                        | Requerido para los proveedores de nonIFS. | Ws2 \_32.dll necesita consultar o modificar la información de estado interna asociada con el identificador de socket.                                                                       |
| [**WPUFDIsSet**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpufdisset)                                     | Comprueba la pertenencia del identificador de socket especificado.                                                                    | Opcional.                      | Esta es solo una función útil que sabe cómo profundizar en las estructuras de [**\_ conjuntos FD**](/windows/desktop/api/winsock/nf-winsock-fd_set) . Es posible que un proveedor tenga que examinar estas estructuras explícitamente de todos modos. |
| [**WPUGetProviderPath**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpugetproviderpath)                     | Recupera la ruta de acceso del archivo DLL para el proveedor especificado.                                                                       | Obligatorio.                      | Solo el \_32.dll Ws2 sabría dónde se ha instalado una capa de protocolo adyacente (potencialmente de otro proveedor).                                                           |
| [**WPUModifyIFSHandle**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpumodifyifshandle)                     | Recibe un identificador IFS (posiblemente) modificado de la DLL de Windows Sockets.                                                  | Requerido para los proveedores IFS.    | Ws2 \_32.dll necesita consultar o modificar la información de estado interna asociada con el identificador de socket.                                                                       |
| [**WPUPostMessage**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpupostmessage)                             | Realiza la función [**PostMessage**](/windows/win32/api/winuser/nf-winuser-postmessagea) estándar de forma que mantiene la compatibilidad con versiones anteriores. | Obligatorio.                      | Solo Windows 2000 y Windows NT. Windows 95 permite el envío de mensajes desde el modo kernel.                                                                                               |
| [**WPUQueryBlockingCallback**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpuqueryblockingcallback)         | Devuelve un puntero a la función de enlace de bloqueo de un subproceso.                                                                  | Obligatorio.                      | No hay ninguna funcionalidad de Windows correspondiente. Solo el \_32.dll Ws2 tiene la información para lograrlo.                                                                    |
| [**WPUQuerySocketHandleContext**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpuquerysockethandlecontext)   | Obtiene el valor de contexto de un socket (solo proveedores nonIFS).                                                                   | Requerido para los proveedores de nonIFS. | Ws2 \_32.dll necesita consultar o modificar la información de estado interna asociada con el identificador de socket.                                                                       |
| [**WPUQueueApc**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpuqueueapc)                                   | Pone en cola un APC de modo de usuario en el subproceso especificado.                                                                          | Opcional.                      | También se puede usar el [**QueueUserApc**](/windows/win32/api/processthreadsapi/nf-processthreadsapi-queueuserapc) .                                                                                                                      |
| [**WPUResetEvent**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpuresetevent)                               | Restablece un objeto de evento.                                                                                                  | Opcional.                      | En su lugar, el proveedor puede usar una llamada de Windows adecuada.                                                                                                                        |
| [**WPUSetEvent**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpusetevent)                                   | Establece un objeto de evento.                                                                                                    | Opcional.                      | En su lugar, el proveedor puede usar una llamada de Windows adecuada.                                                                                                                        |
