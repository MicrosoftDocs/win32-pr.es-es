# <a name="dvd-graph-builder"></a>Generador de gráficos de DVD

El generador de gráficos de DVD es un objeto auxiliar para crear gráficos de reproducción de DVD. Las aplicaciones pueden usar este componente para crear gráficos de filtros para la reproducción y la navegación de DVD. Cree este objeto llamando a **CoCreateInstance**.



|                  |                                              |
|------------------|----------------------------------------------|
| Identificador de clase | CLSID \_ DvdGraphBuilder                       |
| Interfaces       | [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) |
