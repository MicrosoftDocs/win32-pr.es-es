# <a name="providing-network-statistics-and-packet-information"></a>Proporcionar estadísticas de red e información de paquetes

El proveedor de paquetes de red (NPP) es un archivo DLL que se comunica con el controlador de Monitor de red y proporciona estadísticas de red y datos de paquetes.

Las formas más comunes de usar el NPP incluyen:

-   Estadísticas de red.
-   Fotogramas pasados a la aplicación en tiempo real.
-   Los paquetes accesibles solo después de que se haya detenido la captura.

Esta sección contiene los siguientes temas:

-   [Selección de una NIC](selecting-a-nic-using-getnppblobfromui.md)
-   [Usar GetNPPBlobTable](using-getnppblobtable.md)
