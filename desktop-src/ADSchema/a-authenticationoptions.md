# <a name="authentication-options-attribute"></a>Authentication-Options atributo

Opciones de autenticación usadas en ADSI para enlazar a objetos de servicios de directorio.



| Entrada | Value |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | Authentication-Options                                                                                                                                                                                                                                                       |
| Ldap-Display-Name | authenticationOptions                                                                                                                                                                                                                                                        |
| Size              | 4 bytes. Valores definidos en IADS.h: ADS \_ SECURE \_ AUTHENTICATION 0x1, ADS USE ENCRYPTION \_ \_ 0x2, ADS USE SSL \_ \_ 0x2, ADS \_ READONLY \_ SERVER 0x4, ADS PROMPT CREDENTIALS \_ \_ 0x8, ADS NO AUTHENTICATION \_ \_ 0x10, ADS FAST BIND \_ \_ 0x20, ADS USE SIGNING \_ \_ 0x40, ADS USE EL \_ \_ 0X80 |
| Privilegio actualizar  | \-                                                                                                                                                                                                                                                                           |
| Frecuencia de actualización  | \-                                                                                                                                                                                                                                                                           |
| Attribute-Id      | 1.2.840.113556.1.4.11                                                                                                                                                                                                                                                        |
| System-Id-Guid    | bf967928-0de6-11d0-a285-00aa003049e2                                                                                                                                                                                                                                         |
| Sintaxis            | [**Enumeración**](s-enumeration.md)                                                                                                                                                                                                                                         |
