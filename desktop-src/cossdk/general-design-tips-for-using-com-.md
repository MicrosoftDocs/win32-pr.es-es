# <a name="general-design-tips-for-using-com"></a>Sugerencias de diseño generales para el uso de COM+

La clave para un buen diseño es planear. Si no está familiarizado con el diseño de una aplicación de tres niveles, consulte la sección titulada [diseñar la aplicación com+ con UML](designing-the-com--application-using-uml.md).

La forma en que se crean los componentes COM que componen el nivel intermedio de la aplicación COM+ puede afectar drásticamente al rendimiento, la escalabilidad y la facilidad de implementación. En los temas siguientes se proporcionan consideraciones de diseño y sugerencias de implementación que debe conocer al diseñar componentes COM.



| Tema                                                                   | Descripción                                                                                                                |
|-------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------|
| [Diseño para la escalabilidad](designing-for-scalability.md)<br/>   | Sugiere formas de aumentar la escalabilidad de la aplicación COM+.<br/>                                             |
| [Diseño para disponibilidad](designing-for-availability.md)<br/> | Describe los servicios que puede usar para mejorar la disponibilidad de la funcionalidad de nivel intermedio en las aplicaciones COM+.<br/> |
| [Diseñar para la seguridad](designing-for-security.md)<br/>         | Describe las formas de configurar la seguridad de forma más eficaz para una aplicación COM+.<br/>                                      |
| [Diseñar para la implementación](designing-for-deployment.md)<br/>     | Describe formas de minimizar los problemas de implementación al diseñar una aplicación distribuida.<br/>                       |



 

Además, asegúrese de ver cómo [mejorar el rendimiento con la agrupación de objetos](improving-performance-with-object-pooling.md) para las formas de usar la agrupación de objetos de la manera más eficaz.
