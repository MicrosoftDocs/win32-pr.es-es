# <a name="pau-tag"></a>Etiqueta Pau

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Pausa la voz durante el número de milisegundos especificado.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintáctica**
</dt> <dd>

**\\Pau =***número***\\**



| Parte     | Descripción                          |
|----------|--------------------------------------|
| *número* | Número de milisegundos que se va a pausar. |



 

</dd> </dl>

### <a name="remarks"></a>Observaciones

Esta etiqueta solo se admite para la salida generada por TTS. El intervalo de valores para el parámetro puede variar en función del motor TTS instalado.
