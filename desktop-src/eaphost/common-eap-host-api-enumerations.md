# <a name="common-eaphost-api-enumerations"></a>Enumeraciones comunes de la API de EAPHost

Las enumeraciones siguientes son comunes a todas las tecnologías de API de EAPHost.



| Enumeración                                                                  | Descripción                                                                                   |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**\_tipo de atributo EAP \_**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_attribute_type)                           | Define el conjunto de posibles tipos de atributo EAP disponibles en una entidad de autenticación.        |
| [**\_tipo de \_ campo de entrada de configuración de EAP \_ \_**](/windows/desktop/api/eaptypes/ne-eaptypes-eap_config_input_field_type)       | Define un conjunto de posibles tipos de campo de entrada disponibles al consultar las credenciales del usuario. |
| [**\_tipo de \_ propiedad de método EAP \_**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_type)              | Define el conjunto de posibles propiedades de método EAP.                                            |
| [**\_tipo de \_ valor de propiedad de método EAP \_ \_**](/windows/desktop/api/EapTypes/ne-eaptypes-eap_method_property_value_type) | Define el conjunto de tipos de datos posibles para las propiedades del método.                             |
| [**EapCode**](/windows/win32/api/eapmethodtypes/ne-eapmethodtypes-eapcode)                                                   | Define el conjunto de tipos de paquetes EAP.                                                          |
