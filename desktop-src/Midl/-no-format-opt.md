# <a name="no_format_opt-switch"></a>\_modificador de formato/no \_

El modificador de **\_ formato \_ /no** cambia la manera en que MIDL optimiza el tipo y los descriptores de procedimiento.

``` syntax
midl /no_format_opt
```

## <a name="switch-options"></a>Opciones de conmutador

Este modificador no tiene parámetros.

## <a name="remarks"></a>Observaciones

De forma predeterminada, MIDL elimina los descriptores de tipos y procedimientos duplicados con el fin de reducir el tamaño del código auxiliar generado. El uso del modificador del **\_ \_ formato/no** desactiva este comportamiento de optimización.

## <a name="examples"></a>Ejemplos

**formato MIDL \_ /no \_ OPT. idl**
