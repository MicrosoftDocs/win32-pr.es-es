# <a name="wmdrmnet_revocation"></a>Revocación de \_ WMDRMNET

La **propiedad revocación \_ WMDRMNET** recupera la lista de revocación para Windows DRM 10 multimedia para dispositivos de red.

## <a name="global-constant"></a>Constante global

g \_ wszWMDRMNET \_ Revocation

## <a name="data-type"></a>Tipo de datos

**CADENA DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentarios

La lista de revocación contiene los certificados de los dispositivos que se han revocado.
