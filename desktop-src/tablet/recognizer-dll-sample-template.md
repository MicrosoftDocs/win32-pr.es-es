# <a name="recognizer-dll-sample-template"></a>Plantilla de ejemplo de DLL de reconocedor

El ejemplo de DLL de reconocedor contiene una plantilla para crear un archivo DLL de reconocedor. La plantilla contiene funciones para registrar y anular el registro del servidor, así como esqueletos para las funciones principales del reconocedor.

En esta plantilla se muestra cómo registrar un CLSID en el registro. También puede hacerlo al instalar el reconocedor con el programa de instalación. Asegúrese de crear su propio CLSID único para los lenguajes y no reutilice el CLSID que se muestra en el ejemplo.

De forma predeterminada, los ejemplos se instalan en el directorio Samples en el directorio en el que instaló el SDK.

Para obtener un ejemplo de cómo usar los reconocedores de la API de la plataforma de Tablet PC, consulte el [ejemplo de reconocimiento avanzado](advanced-recognition-sample.md).
