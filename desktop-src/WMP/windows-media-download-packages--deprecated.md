# <a name="windows-media-download-packages-deprecated"></a>Paquetes de descarga de Windows Media (en desuso)

En esta página se documenta una característica que puede no estar disponible en versiones futuras de Windows Media Player y el SDK de Windows Media Player.

Los paquetes de descarga de Windows Media combinan los bordes de Windows Media Player, la información de la lista de reproducción y el contenido multimedia en un único archivo descargable con la extensión de nombre de archivo. WMD. Un paquete de descarga de Windows Media podría incluir un álbum completo de vídeos musicales que también muestra publicidad en forma de marca gráfica y vínculos a un sitio web del distribuidor de música en línea.

Los usuarios pueden descargar un paquete de descarga de Windows Media desde un sitio web simplemente haciendo clic en un vínculo. Una vez que el paquete se descarga en el equipo del usuario, Windows Media Player extrae los archivos del paquete automáticamente y, a continuación, agrega la lista de reproducción empaquetada al cuadro desplegable listas de reproducción, agrega el contenido a la biblioteca, muestra la máscara de borde en el panel **reproducción en curso** de la Media Player de Windows en modo completo y, a continuación, reproduce el elemento inicial en la

Los paquetes de descarga de Windows Media ofrecen las siguientes ventajas:

-   Un solo clic descarga, extrae y cataloga varios archivos en el equipo del usuario.
-   El contenido se reproduce inmediatamente después de su descarga.
-   Los bordes personalizados pueden mostrar información de marca y publicidad.
-   Las listas de reproducción empaquetadas se catalogan en la biblioteca.
-   La redirección de sitios web puede producirse desde Media Player de Windows a un sitio relacionado.
-   Hay disponible una variedad de controles de borde interactivos.
-   Se admiten varios tipos de archivos de audio y vídeo para el empaquetado.

Todas estas ventajas se pueden usar conjuntamente. Por ejemplo, un único paquete de descarga de Windows Media podría ofrecer a los usuarios la oportunidad de ver las canciones a medida que se reproducen, mostrar un vídeo que acompaña a las canciones, ver información de publicidad sobre el distribuidor de registros, ver la portada del álbum, visitar un sitio web de ventilador y catalogar el contenido de la biblioteca.
