# <a name="the-listening-tip-window"></a>La ventana de la sugerencia de escucha

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Si la voz está habilitada, aparece una ventana de información sobre herramientas especial cuando el usuario presiona la tecla para iniciar la entrada de voz. La sugerencia de escucha muestra información contextual relacionada con el estado de entrada actual del agente de Microsoft. Si el reconocimiento de voz compatible no se ha instalado o se ha deshabilitado, la sugerencia de escucha no aparece.
