# <a name="mfpkey_fullframerate-property"></a>\_Propiedad FULLFRAMERATE de MFPKEY

Especifica si el endoder puede quitar fotogramas.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

**g \_ wszWMVCFullFrameRate**

## <a name="data-type"></a>Tipo de datos

**VT \_ I4**

## <a name="remarks"></a>Observaciones

Si establece esta propiedad en 1, el codificador no quitará fotogramas. Si establece esta propiedad en 0, el codificador podría quitar fotogramas.
