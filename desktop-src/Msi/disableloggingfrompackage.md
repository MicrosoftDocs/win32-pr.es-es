# <a name="disableloggingfrompackage"></a>DisableLoggingFromPackage

El valor de esta [Directiva del sistema](system-policy.md) por equipo se establece en "1" para deshabilitar el registro especificado para el paquete por la propiedad [**MsiLogging**](msilogging.md) para todos los usuarios del equipo.

La propiedad [**MsiLogging**](msilogging.md) y la Directiva DisableLoggingFromPackage requieren Windows Installer 4,0.

## <a name="registry-key"></a>Clave del Registro

**HKEY \_ \_** Directivas de software de equipo local \\  \\  \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Tipo de datos

**\_valor DWORD reg**
