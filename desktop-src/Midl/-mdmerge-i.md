# <a name="i-switch"></a>modificador/i

El modificador **/i** especifica el directorio que contiene los archivos de metadatos que se van a combinar.

``` syntax
mdmerge /i metadata_directory
```

## <a name="switch-options"></a>Opciones de conmutador

<dl> <dt>

*directorio de metadatos \_* 
</dt> <dd>

Especifica el directorio que contiene los archivos de metadatos que se van a combinar.

</dd> </dl>

## <a name="remarks"></a>Observaciones

## <a name="examples"></a>Ejemplos

**mdmerge/i directorio de metadatos \_**
