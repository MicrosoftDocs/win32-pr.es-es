# <a name="msmq-dependent-client-service-attribute"></a>Atributo MSMQ-dependent-Client-Service

Indica si este servidor proporciona servicios de clientes dependientes.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Servicio de cliente dependiente de MSMQ        |
| Nombre para mostrar de LDAP | mSMQDependentClientService           |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1239              |
| System-ID-GUID    | 2df90d83-009f-11d2-aa4c-00c04fd7d83a |
| Sintaxis            | [**Booleano**](s-boolean.md)         |
