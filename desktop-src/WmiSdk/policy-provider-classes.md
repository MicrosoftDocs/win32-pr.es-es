# <a name="policy-provider-classes"></a>Clases de proveedor de directivas

El proveedor de directivas WMI proporciona extensiones a la Directiva de grupo a través de clases, lo que permite refinar la aplicación de la Directiva. La Directiva se establece mediante el lenguaje de consulta de WMI (WQL) y reside en el Active Directory para facilitar la implementación.

En la tabla siguiente se enumeran las clases del proveedor de directivas WMI.



| Clase                                               | Descripción                                                       |
|-----------------------------------------------------|-------------------------------------------------------------------|
| [**Proveedores de MSFT \_**](/previous-versions/windows/desktop/wmisystemprov/msft-providers) | Expone la configuración relacionada con las instancias del proveedor.             |
| [**Regla de MSFT \_**](/previous-versions/windows/desktop/policmanprov/msft-rule)                | Define una regla única dentro de un ámbito de administración (SOM).         |
| [**MSFT \_ SomFilter**](/previous-versions/windows/desktop/policmanprov/msft-somfilter)      | Proporciona una lista de reglas que se evalúan en un equipo de destino. |
