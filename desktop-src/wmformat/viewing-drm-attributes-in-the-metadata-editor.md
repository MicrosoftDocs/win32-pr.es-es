# <a name="viewing-drm-attributes-in-the-metadata-editor"></a>Ver atributos DRM en el editor de metadatos

El editor de metadatos expone la interfaz IWMDRMEditor, que permite que las aplicaciones de edición examinen ciertos atributos de un encabezado DRM en un archivo ASF protegido y los atributos de la licencia de contenido, si hay alguno presente, incluso si la aplicación no tiene la biblioteca estática específica de DRM (wmstubdrm. lib) necesaria para las aplicaciones de reproductor y escritor DRM habilitadas totalmente.

> [!Note]  
> DRM no es compatible con la versión basada en x64 de este SDK.
