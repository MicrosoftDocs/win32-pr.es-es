# <a name="content-of-descriptive-properties"></a>Contenido de propiedades descriptivas

La interfaz [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) proporciona propiedades descriptivas, que describen diversos aspectos de un objeto. Algunas de estas propiedades son específicas del contenido; otras propiedades tienen contenido que se compone del texto descriptivo proporcionado por el servidor. El tipo de información de cada propiedad varía en función del objeto.

En los temas siguientes se describe la información que los clientes obtienen de estas propiedades. También proporcionan a los servidores sugerencias para elegir contenido.

-   [Propiedad Name](name-property.md)
-   [Role (propiedad)](role-property.md)
-   [Propiedad State](state-property.md)
-   [Value (propiedad)](value-property.md)
-   [Description (propiedad)](description-property.md)
-   [Propiedad Help](help-property.md)
-   [Propiedad HelpTopic](helptopic-property.md)
-   [Propiedad KeyboardShortcut](keyboardshortcut-property.md)
-   [Propiedad DefaultAction](defaultaction-property.md)

Al diseñar objetos accesibles, los desarrolladores de servidores también deben hacer referencia a los temas siguientes:

-   [Elección de las propiedades que se van a admitir](choosing-which-properties-to-support.md)
-   [Elección del contenido para propiedades descriptivas](choosing-the-content-for-descriptive-properties.md)

Para obtener información sobre los parámetros y los valores devueltos de estas propiedades, vea la sección [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) de la referencia de Microsoft Active Accessibility [C/C++](c-c---reference.md).
