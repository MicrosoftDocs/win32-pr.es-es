# <a name="itemthumbnail-property"></a>Propiedad Item. thumbnail

Ruta de acceso y nombre de archivo de la imagen en miniatura. Esta propiedad es **null** si el elemento no admite vistas en miniatura o si no se puede compilar una ruta de acceso. Solo lectura.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Item.Thumbnail
```



## <a name="property-value"></a>Valor de propiedad

Cadena que recibe la ruta de acceso y el nombre de archivo.
