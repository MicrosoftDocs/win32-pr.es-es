# <a name="broadcast-to-local-network"></a>Difusión a la red local

Se puede realizar una difusión en la red conectada localmente estableciendo **SA \_ netnum** en ceros binarios (0) y **SA \_ nodenum** en binarios (1). Esta difusión se puede enviar a la red principal del dispositivo o a todas las redes conectadas localmente a la opción del proveedor de servicios. Los enrutadores no propagan las difusiones a la red local.
