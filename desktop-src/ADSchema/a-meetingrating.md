# <a name="meetingrating-attribute"></a>atributo meetingRating

La clasificación de La plataforma para la selección de contenido de Internet de una reunión.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | meetingRating                               |
| Ldap-Display-Name | meetingRating                               |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.584                      |
| System-Id-Guid    | 11b6cc8d-48c4-11d1-a9c3-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
