> \[En desuso. Esta API se puede quitar de las versiones futuras de Windows.\]

 

Sin implementar.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT ReadXML(
   IUnknown *pTimeline,
   IUnknown *pXML
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*pTimeline* 
</dt> <dd>

Reservado.
