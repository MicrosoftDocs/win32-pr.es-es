# <a name="windows-accessibility"></a>Accesibilidad de Windows

En esta sección se incluye documentación para los desarrolladores de Windows que diseñan aplicaciones accesibles, desarrolladores de tecnología de asistencia, que compilan herramientas como lectores de pantalla y ampliadores e ingenieros de pruebas de software que crean scripts automatizados para probar aplicaciones de Windows.

Use los recursos que se proporcionan aquí para compilar aplicaciones de Windows accesibles que pueden usar tantas personas como sea posible, incluidas aquellas con discapacidades, preferencias personales y estilos de trabajo específicos.
