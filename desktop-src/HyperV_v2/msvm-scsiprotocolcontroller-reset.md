# <a name="reset-method-of-the-msvm_scsiprotocolcontroller-class"></a>Método Reset de la \_ clase MSVM SCSIProtocolController

Solicita un restablecimiento.

## <a name="syntax"></a>Sintaxis


```mof
uint32 Reset();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

<dl> <dt>
