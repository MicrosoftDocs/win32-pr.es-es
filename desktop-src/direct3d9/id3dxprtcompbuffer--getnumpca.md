# <a name="id3dxprtcompbuffergetnumpca-method"></a>ID3DXPRTCompBuffer:: GetNumPCA (método)

Recupera el número de vectores de base del análisis de componentes principales (PCA) que se van a usar en cada clúster.

## <a name="syntax"></a>Sintaxis


```C++
UINT GetNumPCA();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Devuelve el número de vectores de base del análisis de componentes principales (PCA) que se van a usar en cada clúster.
