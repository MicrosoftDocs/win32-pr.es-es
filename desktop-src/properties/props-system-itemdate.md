# <a name="systemitemdate"></a>System.ItemDate

Fecha principal de interés para un elemento. En el caso de las fotos, por ejemplo, esta propiedad se asigna a [System.Photo.DateTaken](./props-system-photo-datetaken.md).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.ItemDate
   shellPKey = PKEY_ItemDate
   formatID = F7DB74B4-4287-4103-AFBA-F1B13DCD75CF
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.ItemDate
   shellPKey = PKEY_ItemDate
   formatID = F7DB74B4-4287-4103-AFBA-F1B13DCD75CF
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
