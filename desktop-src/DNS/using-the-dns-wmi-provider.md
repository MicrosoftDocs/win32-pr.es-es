# <a name="using-the-dns-wmi-provider"></a>Usar el proveedor WMI de DNS

El proveedor WMI de DNS define [*clases CIM*](c-gly.md) de WMI que permiten que la aplicación controle un servidor DNS. En las secciones siguientes se proporciona información general que puede ayudarle a empezar a usar el proveedor WMI de DNS. Se proporcionan vínculos a scripts de ejemplo que muestran cómo hacer uso del proveedor WMI de DNS.

Para obtener más información acerca de las operaciones básicas de WMI, consulte Instrumental de administración de Windows (WMI) en el kit de desarrollo de software (SDK) de la plataforma.
