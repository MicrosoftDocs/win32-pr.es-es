# <a name="memory-classes"></a>Clases de memoria

Las clases de memoria modela la memoria de una máquina virtual.

Las siguientes son clases WMI de virtualización relacionadas con la memoria.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                      | Descripción                                                                                                         |
|--------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**Elemento \_ MsvmAllocatedFromNumaNode**](msvm-elementallocatedfromnumanode.md)<br/> | Asocia una instancia de un recurso asignado al nodo NUMA físico desde el que se asignó.<br/> |
| [**Memoria de \_ Msvm**](msvm-memory.md)<br/>                                             | Representa la memoria asignada actualmente a una máquina virtual.<br/>                                          |
| [**Msvm \_ MemorySettingData**](msvm-memorysettingdata.md)<br/>                       | Representa el estado configurado de la memoria de una máquina virtual.<br/>                                     |
| [**Msvm \_ NumaNode**](msvm-numanode.md)<br/>                                         | Representa un nodo de acceso a memoria no uniforme (NUMA) de un sistema virtual.<br/>                                  |
