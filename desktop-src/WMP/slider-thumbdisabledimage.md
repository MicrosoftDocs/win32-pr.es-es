# <a name="sliderthumbdisabledimage"></a>SLIDEr. thumbDisabledImage

El atributo **thumbDisabledImage** especifica o recupera la imagen Thumb del control deslizante que se utiliza cuando el control está deshabilitado.

``` syntax
        elementID.thumbDisabledImage
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de lectura/escritura que contiene el nombre de un archivo de imagen.

## <a name="remarks"></a>Observaciones

**ThumbDisabledImage** es opcional. Si no se especifica, se usa **thumbImage** en su lugar.

Los formatos admitidos son BMP, JPG, PNG y GIF (sin incluir los GIF animados).
