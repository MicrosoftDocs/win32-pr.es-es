# <a name="cbaselistgettailpositioni-method"></a>Método CBaseList.GetTailPositionI

El `GetTailPositionI` método recupera la posición del último elemento de la lista.

## <a name="syntax"></a>Sintaxis


```C++
POSITION GetTailPositionI();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve un valor POSITION o **NULL si** la lista está vacía.
