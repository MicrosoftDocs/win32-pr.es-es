# <a name="introduction-to-gdi"></a>Introducción a GDI+

Windows GDI+ es una interfaz de dispositivo gráfico que permite a los programadores escribir aplicaciones independientes del dispositivo. Los servicios de GDI+ se exponen a través de un conjunto de clases de C++.
