# <a name="side-by-side-assembly-api"></a>API de ensamblados en paralelo

La API de ensamblados en paralelo instala, quita o consulta ensamblados en paralelo desde el almacén de ensamblados en paralelo del sistema. También se denomina almacén de winSxS en Windows XP y el almacén de componentes de infraestructura de mantenimiento de componentes (CSI) a partir de Windows Vista y Windows Server 2008.

Las partes de la API de ensamblados en paralelo se describen en las siguientes secciones:

-   [Enumeraciones de ensamblados en paralelo](enumerations.md)
-   [Funciones de ensamblado en paralelo](side-by-side-assembly-functions.md)
-   [Interfaces de ensamblado en paralelo](side-by-side-assembly-interfaces.md)
-   [Estructuras de ensamblado en paralelo](structures.md)
