# <a name="the-scope-resolution-operator-in-c"></a>Operador de resolución de ámbito en C++

Dos puntos (::) se usan en C++ como un operador de *resolución de ámbito* . Este operador proporciona más libertad para asignar nombres a las variables, ya que permite distinguir entre las variables con el mismo nombre. Por ejemplo, mi *archivo:: Read* hace referencia al *método Read* *de la clase* de objetos YourFile, en lugar de a *:: Read*, que hace referencia a un método *Read* de la clase *YourFile* .
