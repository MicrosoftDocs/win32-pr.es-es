# <a name="systempropgroupdescription"></a>System. PropGroup. Description

El separador de grupos de propiedades utilizado en las listas de propiedades para separar las descripciones de otros tipos. No utilice esta propiedad para obtener o establecer valores. Esta propiedad está destinada únicamente como un marcador.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropGroup.Description
   shellPKey = PKEY_PropGroup_Description
   formatID = 8969B275-9475-4E00-A887-FF93B8B41E44
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
