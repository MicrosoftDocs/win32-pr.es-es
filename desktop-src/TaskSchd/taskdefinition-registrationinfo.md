# <a name="taskdefinitionregistrationinfo-property"></a>Propiedad TaskDefinition.RegistrationInfo

Para el scripting, obtiene o establece la información de registro que se usa para describir una tarea, como la descripción de la tarea, el autor de la tarea y la fecha en que se registra la tarea.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.RegistrationInfo As RegistrationInfo
```



## <a name="property-value"></a>Valor de propiedad

Información de registro que se usa para describir una tarea, como la descripción de la tarea, el autor de la tarea y la fecha en que se registra la tarea.
