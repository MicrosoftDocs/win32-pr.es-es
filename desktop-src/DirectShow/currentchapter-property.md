# <a name="currentchapter-property"></a>Propiedad CurrentChapter

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

La `CurrentChapter` propiedad recupera el número del capítulo que se está reproduciendo actualmente.

``` syntax
[ iCurChapter = ] MSWebDVD.CurrentChapter
```

## <a name="return-value"></a>Valor devuelto

Devuelve un valor entero que representa el capítulo actual del título actual.

## <a name="remarks"></a>Observaciones

Esta propiedad es de solo lectura y no tiene ningún valor predeterminado.
