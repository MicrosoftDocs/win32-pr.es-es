# <a name="legacy-accessibility-and-automation-technology---msaa-to-ui-automation"></a>Tecnología de automatización y accesibilidad heredada: MSAA en automatización de la interfaz de usuario

Windows accesibilidad y automatización constan de dos tecnologías: Microsoft Active Accessibility (MSAA) y Microsoft Automatización de la interfaz de usuario. MSAA es una tecnología heredada introducida con Windows 95, mientras que Automatización de la interfaz de usuario es una tecnología más reciente y más capaz que aborda las limitaciones de MSAA.

En esta sección se describen las principales diferencias entre MSAA y Automatización de la interfaz de usuario (y cómo realizar la transición a Automatización de la interfaz de usuario).
