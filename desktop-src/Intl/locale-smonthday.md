# <a name="locale_smonthday"></a>configuración regional \_ SMONTHDAY

**Windows 7 y versiones posteriores:** Cadena de formato para mostrar solo el mes y el día. El formato es similar al definido para la [configuración regional \_ SLONGDATE](locale-slongdate.md). Por ejemplo, si el patrón de mes/día es el nombre completo del mes seguido del número de día con ceros a la izquierda, como en "septiembre de 03", la cadena de formato es "MMMM dd". La cadena puede constar de una combinación de imágenes con formato de día y mes y cualquier cadena de caracteres entre comillas simples. Los caracteres entre comillas simples permanecen como se especifican, por ejemplo, "de" para español (España). Una configuración regional solo puede especificar un formato de mes/día.
