# <a name="msilogfilelocation-property"></a>Propiedad MsiLogFileLocation

El valor de la **propiedad MsiLogFileLocation** se establece en la ruta de acceso completa del archivo de registro, cuando el registro está habilitado. La **propiedad MsiLogFileLocation** se puede usar para mostrar el archivo de registro, o la ubicación del archivo de registro, en una interfaz de usuario al final de la instalación.
