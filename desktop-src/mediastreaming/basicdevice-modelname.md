# <a name="basicdevicemodelname-property"></a>Propiedad BasicDevice. ModelName

Obtiene el nombre de modelo del dispositivo.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT get_ModelName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Valor de propiedad

Puntero al nombre del modelo del dispositivo.
