# <a name="patchapi"></a>PatchAPI

En esta sección se documentan varias funciones que pertenecen a la interfaz PatchAPI. PatchAPI forma parte de la [API de compresión Delta](/previous-versions/bb417345(v=msdn.10)), que está diseñada para crear mediante programación diferencias (similares a revisiones o diferencias comprimidas) entre archivos.
