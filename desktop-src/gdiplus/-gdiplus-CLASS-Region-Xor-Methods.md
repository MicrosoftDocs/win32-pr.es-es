# <a name="regionxor-methods"></a>Region. XOR (métodos)

En este tema se enumeran los métodos XOR de la clase [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Para obtener una lista completa de los métodos de la clase **Region** , consulte [métodos de región](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                         | Descripción                                                                                                                                                                              |
|:---------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XOR (rectángulo&)**](/previous-versions//ms534787(v=vs.85))     | El método [**Region:: XOR**](/previous-versions//ms534787(v=vs.85)) actualiza esta región a las partes sin intersección de sí mismo y al interior del rectángulo especificado.<br/>  |
| [**XOR (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstrectf_))   | El método [**Region:: XOR**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstrectf_)) actualiza esta región a las partes sin intersección de sí mismo y al interior del rectángulo especificado.<br/> |
| [**XOR (región \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstregion))     | El método [**Region:: XOR**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstregion)) actualiza esta región a las partes de la no intersección de sí misma y a otra región.<br/>                         |
| [**XOR (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstgraphicspath)) | El método [**Region:: XOR**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstgraphicspath)) actualiza esta región a las partes sin intersección de sí mismo y al interior de la ruta de acceso especificada.<br/>            |
