# <a name="bluetooth"></a>Bluetooth

## <a name="purpose"></a>Propósito

Bluetooth es un protocolo estándar del sector que habilita la conectividad inalámbrica para equipos, dispositivos de mano, teléfonos móviles y otros dispositivos.

## <a name="where-applicable"></a>Donde sea aplicable

La interfaz de programación de aplicaciones de Bluetooth permite a los desarrolladores usar el conocimiento de programación de red existente para desarrollar o migrar aplicaciones rápidamente.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Bluetooth está diseñado para que lo usen los programadores de C/C++. Algunas características de Bluetooth están disponibles con Windows Sockets. Es necesario estar familiarizado con las redes de Microsoft Windows y la programación de Windows Sockets.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

La compatibilidad con Microsoft Bluetooth comienza con Windows XP con Service Pack 1 (SP1).

La compatibilidad con Bluetooth 2,1 se ofrece en Windows Vista SP2 y Windows 7.
