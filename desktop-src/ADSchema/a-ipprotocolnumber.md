# <a name="ipprotocolnumber-attribute"></a>atributo ipProtocolNumber

Contiene la sección del mapa de protocolos que almacena el número único que identifica el protocolo.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | ipProtocolNumber                     |
| Nombre para mostrar de LDAP | ipProtocolNumber                     |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.17                     |
| System-ID-GUID    | ebf5c6eb-0e2d-4415-9670-1081993b4211 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
