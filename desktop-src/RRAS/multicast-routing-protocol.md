# <a name="multicast-routing-protocol"></a>Protocolo de enrutamiento de multidifusión

Un protocolo de enrutamiento de multidifusión es un tipo de proveedor de servicios que funciona como cliente dentro del marco de la arquitectura del enrutador. La arquitectura de enrutamiento está diseñada para ampliarse mediante estos módulos de cliente de enrutador.

Un protocolo de enrutamiento de multidifusión administra la pertenencia a grupos y controla la ruta de acceso que los datos de multidifusión asumen sobre la red. Entre los ejemplos de protocolos de enrutamiento de multidifusión se incluyen: Multidifusión independiente del protocolo (PIM), Multidifusión open shortest path first (MOSPF) y Protocolo de enrutamiento de multidifusión de vectores de distancia (DVMRP). El Protocolo de administración de grupos de Internet (IGMP) es un protocolo de enrutamiento de multidifusión especial que actúa como intermediario entre hosts y enrutadores.
