# <a name="print-ticket-api-enumerations"></a>Print ticket API (enumeraciones)

La API de entradas de impresión usa las siguientes enumeraciones.

## <a name="in-this-section"></a>En esta sección



| Enumeración                                                   | Descripción                                                                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EDefaultDevmodeType**](/windows/win32/api/prntvpt/ne-prntvpt-edefaultdevmodetype)<br/> | Permite a los usuarios especificar qué [**DEVMODE**](/windows/win32/api/wingdi/ns-wingdi-devmodea) se va a usar como origen de los valores predeterminados cuando una solicitud de impresión no especifica todos los valores posibles.<br/> |
| [**EPrintTicketScope**](/windows/desktop/api/prntvpt/ne-prntvpt-eprintticketscope)<br/>     | Especifica el ámbito de una solicitud de impresión.<br/>                                                                                                                       |
