# <a name="ms-dfsr-stagingsizeinmb-attribute"></a>atributo MS-DFSR-StagingSizeInMb

Contiene el tamaño, en megabytes, del directorio de almacenamiento provisional.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-StagingSizeInMb              |
| Nombre para mostrar de LDAP | msDFSR-StagingSizeInMb               |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.6            |
| System-ID-GUID    | 250a8f20-f6fc-4559-ae65-e4b24c67aebe |
| Sintaxis            | [**Interval**](s-interval.md)       |
