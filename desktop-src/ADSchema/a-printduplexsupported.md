# <a name="print-duplex-supported-attribute"></a>Print-duplex-atributo compatible

Indica el tipo de compatibilidad dúplex que tiene una impresora.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | Impresión-dúplex: compatible                                |
| Nombre para mostrar de LDAP | printDuplexSupported                                  |
| Tamaño              | 4 bytes. Dúplex compatible = 2. Simplex solo = 1 o 0. |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1311                               |
| System-ID-GUID    | 281416cc-1968-11d0-a28f-00aa003049e2                  |
| Sintaxis            | [**Booleano**](s-boolean.md)                          |
