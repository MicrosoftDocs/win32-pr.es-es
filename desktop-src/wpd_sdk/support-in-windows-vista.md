# <a name="support-in-windows-7-and-windows-vista"></a>Compatibilidad con Windows 7 y Windows Vista

Para todas las SKU de Windows 7 y Windows Vista, excepto para las N, el tiempo de ejecución de WPD se incluye como parte del sistema operativo.

### <a name="suported-api-elements"></a>Elementos de la API de admite

En la tabla siguiente se identifican las características admitidas para Windows 7, Windows Vista y Windows Vista con la actualización de la plataforma.



| Característica                        | Windows 7 | Windows Vista | Windows Vista con la actualización de la plataforma |
|--------------------------------|-----------|---------------|----------------------------------------|
| MTP a través de USB                   | Sí       | Sí           | Sí                                    |
| MTP a través de IP                    | Sí       | Sí           | Sí                                    |
| MTP a través de Bluetooth             | Sí       | No            | Sí                                    |
| WPD y servicios de dispositivos MTP    | Sí       | No            | Sí                                    |
| Automatización de WPD                 | Sí       | No            | No                                     |
| Múltiples funciones/transporte múltiple | Sí       | No            | No                                     |
| Device Stage (puede estar en inglés)                   | Sí       | No            | No                                     |
| Plataforma de sincronización de dispositivos           | Sí       | No            | No                                     |



 

### <a name="the-platform-update-for-windows-vista"></a>La actualización de la plataforma para Windows Vista

La actualización de la plataforma para Windows Vista es compatible con las extensiones de Windows 7 para la plataforma de dispositivos portátiles de Windows (WPD).

### <a name="windows-vista-n-and-kn-versions-require-installation-of-windows-media-format-11-sdk"></a>Las versiones N y KN de Windows Vista requieren la instalación de Windows Media Format 11 SDK

-   En el caso de las versiones N y KN de Windows 7, debe instalar el SDK de Windows Media Format 11 para habilitar la funcionalidad de WPD.
-   En el caso de las versiones N y KN de Windows Vista, debe instalar el SDK de Windows Media Format 11 para habilitar la funcionalidad de WPD.
