# <a name="itssbservicenotification-methods"></a>Métodos ITsSbServiceNotification

Las [**interfaces ITsSbServiceNotification**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbservicenotification) exponen los métodos siguientes.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**Método NotifyServiceFailure**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicefailure)
</dt> <dd>

Notifica a los complementos registrados que el servicio Conexión a Escritorio remoto Broker (Agente de conexión a Escritorio remoto) se ha detenido.

</dd> <dt>

[**Método NotifyServiceSuccess**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbservicenotification-notifyservicesuccess)
</dt> <dd>

Notifica a los complementos registrados que se ha iniciado el servicio Agente de conexión a Escritorio remoto.

</dd> </dl>
