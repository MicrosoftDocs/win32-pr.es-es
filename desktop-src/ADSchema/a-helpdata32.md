# <a name="help-data32-attribute"></a>Help-Data32 atributo

Este atributo se usó para el formato de archivo de ayuda de Win32 Exchange 4.0. No se usa para ninguna otra versión de Exchange.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | Help-Data32                                           |
| Ldap-Display-Name | helpData32                                            |
| Size              | \-                                                    |
| Actualizar privilegios  | El sistema lo usa.                           |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.2.9                                  |
| System-Id-Guid    | 5fd424a8-1262-11d0-a060-00aa006c33ed                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
