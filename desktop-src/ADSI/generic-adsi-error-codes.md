# <a name="generic-adsi-error-codes"></a>Códigos de error ADSI genéricos

Cualquiera de los proveedores del sistema ADSI puede devolver estos tipos de códigos de error para representar los mensajes de error específicos de ADSI. Tienen el código de instalación 5 con el bit de gravedad establecido en **true** o **false**. Cuando el bit de gravedad se establece en **true**, se generan valores de error con el formato 0x80005xxx y mensajes de error con el formato **e \_ ADS \_ \***. Cuando el bit de gravedad se establece en **false**, se generan valores de error con el formato 0x00005xxx y los mensajes de error son del formulario **S \_ ADS \_ \*** .

En la tabla siguiente se enumeran los códigos de error ADSI genéricos definidos en Adserr. h.



| Value       | Código                                  | Descripción                                                           | Acción correctora                                                                                                                                                                                                                                                                                          |
|-------------|---------------------------------------|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0x00005011L | **S \_ ADS \_ ERRORSOCCURRED**            | Durante una consulta, se produjeron uno o más errores.                          | Compruebe que la preferencia de búsqueda se puede establecer legalmente y, si es así, que esté establecida correctamente.                                                                                                                                                                                                                  |
| 0x00005012L | **S \_ \_ más filas de \_ ADS**              | La operación de búsqueda ha alcanzado la última fila.                        | Continúe con el resto del programa.                                                                                                                                                                                                                                                                        |
| 0x00005013L | **\_ \_ más columnas de ADS \_**           | La operación de búsqueda ha alcanzado la última columna de la fila actual. | Pasar a la siguiente fila.                                                                                                                                                                                                                                                                                       |
| 0x80005000L | **E \_ ADS \_ \_ ruta de nombreruta incorrecta**             | Se pasó un nombreruta de ADSI no válido.                                  | Compruebe que el objeto existe en el servidor de directorio y compruebe si hay errores tipográficos en la ruta de acceso.                                                                                                                                                                                                        |
| 0x80005001L | **E \_ ADS \_ \_ objeto de dominio no válido \_**   | Se solicitó un objeto de dominio ADSI desconocido.                          | Compruebe la ruta de acceso del objeto de dominio.                                                                                                                                                                                                                                                                      |
| 0x80005002L | **E \_ ADS \_ \_ objeto de usuario no válido \_**     | Se solicitó un objeto de usuario ADSI desconocido.                            | Compruebe la existencia del objeto de usuario y compruebe si hay errores tipográficos en la ruta de acceso y los derechos de acceso de usuario.                                                                                                                                                                                                           |
| 0x80005003L | **E \_ ADS \_ \_ objeto de equipo no válido \_** | Se solicitó un objeto de equipo ADSI desconocido.                        | Compruebe la existencia del objeto de equipo, compruebe si hay errores ortográficos de la ruta de acceso y los derechos de acceso al equipo.                                                                                                                                                                                                   |
| 0x80005004L | **E \_ ADS \_ \_ objeto desconocido**           | Se solicitó un objeto ADSI desconocido.                                 | Compruebe el nombre de y los derechos de acceso al objeto.                                                                                                                                                                                                                                                    |
| 0x80005005L | **\_propiedad de \_ ADS \_ no \_ establecida**        | No se estableció la propiedad ADSI especificada.                              |                                                                                                                                                                                                                                                                                                            |
| 0x80005006L | **\_no se \_ admite la propiedad de ADS \_ \_**  | No se admite la propiedad ADSI especificada.                         | Compruebe que la propiedad correct esté establecida.                                                                                                                                                                                                                                                                   |
| 0x80005007L | **\_propiedad E \_ ADS \_ no válida**         | La propiedad ADSI especificada no es válida                                | Compruebe los parámetros pasados a la llamada al método.                                                                                                                                                                                                                                                           |
| 0x80005008L | **E \_ ADS \_ \_ parámetro incorrecto**            | Uno o varios parámetros de entrada no son válidos.                             |                                                                                                                                                                                                                                                                                                            |
| 0x80005009L | **\_objeto de anuncios E \_ \_ sin enlazar**           | El objeto ADSI especificado no está enlazado a un recurso remoto.          | Llame a [**GetInfo**](/windows/desktop/api/Iads/nf-iads-iads-getinfo) en un objeto recién creado después de llamar a [**SetInfo**](/windows/desktop/api/Iads/nf-iads-iads-setinfo) .                                                                                                                                                                                      |
| 0x8000500AL | **\_propiedad de \_ ADS \_ no \_ modificada**   | El objeto ADSI especificado no se ha modificado.                      |                                                                                                                                                                                                                                                                                                            |
| 0x8000500BL | **E \_ ADS \_ (propiedad \_ modificada)**        | Se ha modificado el objeto ADSI especificado.                          |                                                                                                                                                                                                                                                                                                            |
| 0x8000500CL | **E ADS no se pudo \_ \_ \_ convertir \_ DataType**   | El tipo de datos no se puede convertir en un tipo de datos de DS nativo ni desde él.      | Compruebe que se usa el tipo de datos correcto o que hay suficientes datos de esquema disponibles para realizar la conversión de tipos de datos.                                                                                                                                                                           |
| 0x8000500DL | **\_ \_ \_ no \_ se encontró la propiedad de ADS**      | No se puede encontrar la propiedad en la memoria caché.                            | Compruebe que se ha llamado a [**GetInfo**](/windows/desktop/api/Iads/nf-iads-iads-getinfo) implícita o explícitamente. Si el atributo es operativo, debe recuperarse explícitamente con [**GetInfoEx**](/windows/desktop/api/Iads/nf-iads-iads-getinfoex) en lugar de **GetInfo**. Si el problema persiste, la propiedad no se ha establecido en el servidor. |

| 0x80005014L | **E \_ ADS \_ filtro no válido \_**           | El filtro de búsqueda especificado no es válido.                               | Use el formato correcto del filtro aceptado por el servidor de directorio.                                                                                                                                                                                                                                     |
