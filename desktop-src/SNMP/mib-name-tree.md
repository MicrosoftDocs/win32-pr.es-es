# <a name="mib-name-tree"></a>Árbol de nombres de MIB

El espacio de nombres para los identificadores de objeto de MIB es jerárquico. Está estructurado para que a cada objeto administrable se le pueda asignar un nombre único global.

La autoridad de las partes del espacio de nombres se asigna a organizaciones individuales. Esto permite a las organizaciones asignar nombres sin consultar a una autoridad de Internet para cada asignación. Por ejemplo, el espacio de nombres asignado a Microsoft es 1.3.6.1.4.1.311, que se define en MSFT. Mib. Microsoft tiene la autoridad para asignar nombres a objetos en cualquier lugar por debajo de ese espacio de nombres.

El identificador de objeto de la jerarquía se escribe como una secuencia de subidentificadores que comienzan en la raíz y terminan en el objeto . Los subidentificadores se separan con un punto.
