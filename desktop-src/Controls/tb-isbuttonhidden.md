# <a name="tb_isbuttonhidden-message"></a>\_Mensaje ISBUTTONHIDDEN TB

Determina si el botón especificado de una barra de herramientas está oculto.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>

Identificador de comando del botón.

</dd> <dt>

*lParam* 
</dt> <dd>Debe ser cero.</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Devuelve un valor distinto de cero si el botón está oculto o cero en caso contrario.
