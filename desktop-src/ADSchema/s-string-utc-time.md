# <a name="stringutc-time-syntax"></a>Sintaxis de cadena (hora UTC)

Formato de cadena de tiempo definido por los estándares de ASN.1. Para más información, consulte ISO 8601 y X680. Use esta sintaxis para almacenar valores de tiempo en UTC-Time formato.



| Entrada | Valor |
|--------------|----------------------------------------------------------------------------|
| Nombre         | String(UTC-Time)                                                           |
| Identificador de sintaxis    | 2.5.5.11                                                                   |
| Id. de OM        | 23                                                                         |
| Tipo DE MAPI    | SYSTIME                                                                    |
| ADS Type     | ADSTYPE \_ UTC \_ TIME                                                         |
| Tipo de variante | VT \_ DATE                                                                   |
| Tipo sds     | [System.DateTime](/dotnet/api/system.datetime) |
