# <a name="frs-primary-member-attribute"></a>FRS: atributo de miembro principal

Referencia al miembro principal de un conjunto de réplicas.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | FRS-miembro principal                      |
| Nombre para mostrar de LDAP | fRSPrimaryMember                        |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.878                  |
| System-ID-GUID    | 2a132581-9373-11d1-aebc-0000f80367c1    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
