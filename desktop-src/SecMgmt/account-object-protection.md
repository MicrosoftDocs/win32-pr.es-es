# <a name="account-object-protection"></a>Protección de objetos de cuenta

Los objetos de [**cuenta**](account-object.md) están protegidos de la siguiente manera:

-   El grupo **mundial** tiene \_ ejecución genérica.
-   El **\_ administrador local** del grupo tiene acceso de eliminación, genérico \_ lectura, escritura genérica \_ , \_ ejecución genérica y escritura \_ DACL.
-   El **\_ administrador local** de grupo se asigna como propietario y grupo principal de objetos de cuenta.
