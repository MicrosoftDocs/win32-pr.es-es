# <a name="iaclcustommru-interface"></a>Interfaz IACLCustomMRU

Expone métodos que se usan para inicializar una lista de elementos usados recientemente (MRU) para un objeto de finalización automática.

## <a name="members"></a>Miembros

La interfaz **IACLCustomMRU** hereda de la interfaz [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) . **IACLCustomMRU** también tiene estos tipos de miembros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

La interfaz **IACLCustomMRU** tiene estos métodos.



| Método                                             | Descripción                                                             |
|:---------------------------------------------------|:------------------------------------------------------------------------|
| [**AddMRUString**](iaclcustommru-addmrustring.md) | Agrega una entrada a la lista MRU.<br/>                               |
| [**Inicialización**](iaclcustommru-initialize.md)     | Carga una lista de cadenas en la lista MRU del registro.<br/> |



 

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows XP \[\]<br/>          |
| Servidor mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows Server 2003 \[\]<br/> |
