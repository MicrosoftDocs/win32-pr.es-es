# <a name="d1256-invalid-analysis-effect-usage"></a>D1256: uso de efecto de análisis no válido

Se usó incorrectamente un efecto de análisis. Esto puede ocurrir cuando se usa un efecto de análisis como entrada para otro efecto o un pincel de imagen, se dibuja en una lista de comandos o se almacena en caché el resultado.
