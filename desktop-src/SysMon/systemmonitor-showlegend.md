# <a name="systemmonitorshowlegend-property"></a>Propiedad SystemMonitor. ShowLegend

Recupera o establece un valor que determina si se muestra la leyenda. La leyenda muestra los contadores y muestra el color que se usa para representar gráficamente cada contador.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
Property ShowLegend As Boolean
```



## <a name="property-value"></a>Valor de propiedad

True indica que la leyenda se muestra en el control; en caso contrario, false. El valor predeterminado de esta propiedad es true.
