# <a name="pointpoint-constructors"></a>Constructores Point.Point

En este tema se enumeran los constructores de la [**clase Point.**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) Para obtener una lista de clases completa, **vea Point Class**.

### <a name="overload-list"></a>Lista de sobrecarga



| Constructor                                                | Descripción                                                                                                                                                                      |
|:-----------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Point(Size&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inconstsize_))   | Crea un [**objeto Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) mediante un [**objeto Size**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) para inicializar los miembros de datos **X** e **Y.**<br/> |
| [**Point(Point&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inconstpoint_)) | Crea un nuevo [**objeto Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) y copia los miembros de datos de otro **objeto Point.**<br/>                                           |
| [**Point(INT,INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inint_inint))  | Crea un [**objeto Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) con dos enteros para inicializar los miembros de datos **X** **e Y.**<br/>                                       |
| [**Point()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point)             | Crea un [**objeto Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) e inicializa los miembros de datos **X** e **Y** en cero. Éste es el constructor predeterminado.<br/>               |
