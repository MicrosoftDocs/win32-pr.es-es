# <a name="companyname-property"></a>Propiedad COMPANYNAME

La propiedad **CompanyName** es la organización o la empresa del usuario que realiza la instalación.

## <a name="default-value"></a>Valor predeterminado

Se puede colocar un valor predeterminado en la [tabla de propiedades](property-table.md). Si no se establece la propiedad **CompanyName** , el instalador la establece automáticamente con los valores del registro.

## <a name="remarks"></a>Observaciones

Establezca la propiedad [**NoCompanyName**](nocompanyname.md) para suprimir la configuración automática de **CompanyName**.
