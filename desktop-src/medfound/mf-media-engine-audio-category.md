# <a name="mf_media_engine_audio_category-attribute"></a>MF \_ \_ atributo de \_ categoría de audio de motor multimedia \_

Especifica la categoría de la secuencia de audio.

## <a name="data-type"></a>Tipo de datos

**[**\_categoría de secuencia de audio \_**](/windows/win32/api/audiosessiontypes/ne-audiosessiontypes-audio_stream_category)**

## <a name="remarks"></a>Observaciones

El valor de este atributo es un miembro de la enumeración de la [**\_ \_ categoría secuencia de audio**](/windows/win32/api/audiosessiontypes/ne-audiosessiontypes-audio_stream_category) .

Este atributo se usa con el método [**IMFMediaEngineClassFactory:: CreateInstance**](/windows/desktop/api/mfmediaengine/nf-mfmediaengine-imfmediaengineclassfactory-createinstance) para inicializar el motor multimedia. El atributo es opcional.
