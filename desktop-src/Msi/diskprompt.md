# <a name="diskprompt-property"></a>Propiedad DiskPrompt

El valor de la propiedad **DiskPrompt** contiene una cadena que se muestra en un cuadro de diálogo que solicita un disco. Esta cadena debe incluir el nombre del producto y un marcador de posición para el texto que se imprime en la superficie de CD-ROM.

## <a name="remarks"></a>Observaciones

Anunciado como propiedad del producto.

Windows Installer en Windows Server 2003 o Windows XP. Consulte los [requisitos de Run-Time de Windows Installer](windows-installer-portal.md) para obtener información sobre la Service Pack mínima de Windows que requiere una versión Windows Installer.<br/> |
