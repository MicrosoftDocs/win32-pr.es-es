# <a name="ms-ds-external-store-attribute"></a>Atributo ms-DS-External-Store

Cadena que identifica la ubicación de un almacén externo, como una base de datos.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-External-Store                        |
| Ldap-Display-Name | msDS-ExternalStore                          |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1834                     |
| System-Id-Guid    | 604877cd-9cdb-47c7-b03d-3daadb044910        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
