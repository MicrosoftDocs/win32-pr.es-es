# <a name="ipconf-msp"></a>IPConf MSP

\[IPConf MSP no está disponible para su uso en Windows Vista, Windows Server 2008 y versiones posteriores del sistema operativo. La API de cliente RTC proporciona una funcionalidad similar.\]

El MSP de IPConf (Confmsp.dll), cuando se empareja con su TSP (IPconf.tsp), proporciona acceso de conferencia a redes de protocolo de Internet (IP), como Internet y muchas intranets corporativas. Además de los métodos MSPI estándar, este MSP expone interfaces que permiten el control sobre participantes individuales en una conferencia.

Consulte [Interfaces MSP de IPConf](ipconf-msp-interfaces.md) para obtener un resumen de las interfaces específicas del proveedor expuestas por este MSP.

El MSP de IPConf se ha mejorado en Windows 2000 SP1 para habilitar la comunicación entre los clientes H323 y SDP. Ahora expone además la interfaz [**IMulticastControl.**](imulticastcontrol.md)

El MSP de IPConf se instala con Windows 2000.
