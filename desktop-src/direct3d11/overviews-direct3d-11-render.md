# <a name="rendering-direct3d-11-graphics"></a>Representación (gráficos de Direct3D 11)

Esta sección contiene información sobre varias tecnologías de representación de Direct3D 11.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                            | Descripción                                                                                                                                                                                                                                                                                      |
|----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [MultiThreading](overviews-direct3d-11-render-multi-thread.md)<br/>       | Direct3D 11 implementa la compatibilidad con la creación y representación de objetos mediante varios subprocesos.<br/>                                                                                                                                                                                              |
| [Representación de varias pasadas](overviews-direct3d-11-render-multipass.md)<br/> | La representación de varias pasadas es un proceso en el que una aplicación atraviesa su gráfico de escenas varias veces para producir una salida que se represente en la pantalla. La representación de varias pasadas mejora el rendimiento porque divide escenas complejas en tareas que se pueden ejecutar simultáneamente. <br/> |
