# <a name="productid-property"></a>ProductID (propiedad)

La propiedad **ProductID** se establece en el **ProductID** completo después de una validación correcta.

Este valor lo muestra la interfaz de usuario y lo usa la [acción RegisterProduct](registerproduct-action.md).

Después de una validación correcta, esta propiedad se establece mediante la [Acción ValidateProductID](validateproductid-action.md) o [ControlEvent, ValidateProductID](validateproductid-controlevent.md).

Tenga en cuenta que la interfaz de usuario de ejemplo determinada proporcionada con el SDK como Uisample.msi requiere un valor para **ProductID**. Si utiliza esta interfaz de usuario, pero no desea usar **ProductID** para validar los identificadores de producto, establezca la propiedad **ProductID** en "none" en la [tabla de propiedades](property-table.md).
