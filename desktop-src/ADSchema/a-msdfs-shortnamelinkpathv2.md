# <a name="ms-dfs-short-name-link-path-v2-attribute"></a>Atributo ms-DFS-Short-Name-Link-Path-v2

Ruta de acceso del vínculo DFS de nombre corto en relación con el recurso compartido de destino raíz DFS (es decir, sin los componentes de nombre de espacio de nombres DFS o servidor/dominio). Use barras diagonales (/) en lugar de barras diagonales inversas ( ), para que las búsquedas LDAP se puedan realizar sin tener que \\ usar escapes.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DFS-Short-Name-Link-Path-v2              |
| Ldap-Display-Name | msDFS-ShortNameLinkPathv2                   |
| Size              | \-                                          |
| Actualizar privilegios  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2042                     |
| System-Id-Guid    | 2d7826f0-4cf7-42e9-a039-1110e0d9ca99        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
