# <a name="iinkanalysisrecognizersetashighestpriorityinkanalysisrecognizer-method"></a>Método IInkAnalysisRecognizer::SetAsHighestPriorityInkAnalysisRecognizer

Modifica este [**IInkAnalysisRecognizer**](iinkanalysisrecognizer.md) a la prioridad más alta en la lista interna de reconocedores de [**IInkAnalyzer.**](iinkanalyzer.md)

## <a name="syntax"></a>Sintaxis


```C++
HRESULT SetAsHighestPriorityInkAnalysisRecognizer();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Para obtener una descripción de los valores [devueltos, vea Clases e interfaces: análisis de entrada de lápiz.](classes-and-interfaces---ink-analysis.md)
