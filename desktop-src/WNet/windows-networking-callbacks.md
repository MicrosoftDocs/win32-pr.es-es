# <a name="windows-networking-callbacks"></a>Devoluciones de llamada de redes de Windows

Los siguientes prototipos de función se utilizan en las redes de Windows.



| Prototipo de función                                 | Descripción                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**PFNGETPROFILEPATH**](/previous-versions/windows/desktop/legacy/aa385358(v=vs.85))     | Prototipo para una función que obtiene una ruta de acceso de Perfil de usuario.                                                            |
| [**PFNPROCESSPOLICIES**](/previous-versions/windows/desktop/legacy/aa385360(v=vs.85))   | Prototipo de una función que devuelve la información de directiva de cuenta de usuario asociada a un proceso de Windows específico. |
| [**PFNRECONCILEPROFILE**](/previous-versions/windows/desktop/legacy/aa385364(v=vs.85)) | Prototipo de una función que reconcilia un perfil local con su perfil global correspondiente.                        |
