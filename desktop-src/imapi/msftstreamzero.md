# <a name="msftstreamzero-object"></a>Objeto MsftStreamZero

Use este objeto para crear un flujo de datos de solo lectura cuyos datos se inicializan en ceros. Debe llamar al **método SetSize** para establecer el tamaño solicitado de la secuencia.

## <a name="members"></a>Miembros

El **objeto MsftStreamZero** no define ningún miembro.
