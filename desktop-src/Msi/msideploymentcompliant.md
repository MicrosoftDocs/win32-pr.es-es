# <a name="msideploymentcompliant-property"></a>Propiedad MSIDEPLOYMENTCOMPLIANT

La propiedad **MSIDEPLOYMENTCOMPLIANT** se puede establecer para indicar al instalador que el paquete se ha creado y probado para cumplir con el [*control de cuentas de usuario*](u-gly.md) (UAC) en Windows Vista. Si no se establece esta propiedad, el instalador determinará si el paquete cumple con UAC en Windows Vista.

Para obtener más información acerca de UAC y Windows Installer, consulte [uso de Windows Installer con UAC](using-windows-installer-with-uac.md) y [directrices para paquetes](guidelines-for-packages.md).

**Windows Installer 3,1, Windows Installer 3,0 y Windows Installer 2,0:** Esta propiedad se omite. Esta propiedad solo se usa en Windows Installer 4,0 en Windows Vista.
