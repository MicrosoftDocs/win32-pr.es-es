# <a name="skin-files"></a>Archivos de máscara

Cuando se ve una máscara, solo se ve la superficie. Detrás de la superficie hay varios archivos que son la máscara completa. Técnicamente hablando, una máscara es un grupo de archivos, donde cada archivo contiene un tipo específico de información. Estos son los tipos de archivos que puede usar para crear una máscara completa.



| Tipo de archivo                                        | Descripción                                                                                                                                                               |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Archivo de definición de máscara](skin-definition-file.md) | Archivo maestro que define cómo se usarán los demás archivos. Se trata de un archivo de texto y tiene una extensión de nombre de archivo .wms.                                                    |
| [Archivos de arte](art-files.md)                       | Archivos de arte que contienen los elementos gráficos de la máscara. Estos incluyen archivos de mapa de bits, GIF, JPEG y PNG.                                                                 |
| [JScript Archivos](jscript-files.md)               | Si desea crear respuestas más complicadas a eventos, puede crear archivos de script mediante Microsoft JScript. Se trata de archivos de texto y tienen una .js de nombre de archivo. |
