# <a name="eject-method"></a>EJECT (método)

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

El `Eject` método expulsa o inserta un disco de o en la unidad de DVD.

``` syntax
        MSWebDVD.Eject()
```

## <a name="return-value"></a>Valor devuelto

No de devuelve ningún valor.

## <a name="remarks"></a>Observaciones

En algunas unidades de DVD, este método actúa como alternancia, expulsando e insertando en llamadas alternativas. Esto depende del hardware.
