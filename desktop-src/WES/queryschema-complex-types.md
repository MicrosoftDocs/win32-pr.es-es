# <a name="query-schema-complex-types"></a>Tipos complejos de esquema de consulta

A continuación se muestran los tipos complejos que define el esquema de consulta.



| Tipo complejo                                                   | Descripción                                                                                                                                                         |
|----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**QueryListType**](queryschema-querylisttype-complextype.md) | Define una lista de consultas que pueden contener un conjunto de selectores e supresors que se usan para incluir eventos en el conjunto de resultados o para excluir eventos del mismo. <br/> |
| [**QueryType**](queryschema-querytype-complextype.md)         | Define un conjunto de consultas de selector y suprimidor que se usan para incluir eventos en o excluir eventos del conjunto de resultados.<br/>                               |
