# <a name="structures-windows-property-system"></a>Estructuras (sistema de propiedades de Windows)

En esta sección se describen las estructuras del sistema de propiedades de Windows.



| Tema                                | Contenido                                                                                                                                    |
|--------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**PROPERTYKEY**](/windows/win32/api/wtypes/ns-wtypes-propertykey) | Especifica el identificador de FMTID/PID que identifica una propiedad mediante programación. Reemplaza [**SHCOLUMNID**](../shell/objects.md).<br/> |
| [**PROPPRG**](/windows/desktop/api/Shlobj_core/ns-shlobj_core-propprg)           | Esta estructura contiene información de un archivo. pif. [**PifMgr \_ GetProperties**](/windows/desktop/api/Shlobj_core/nf-shlobj_core-pifmgr_getproperties)lo usa.<br/>       |
