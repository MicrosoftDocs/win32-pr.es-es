# <a name="ms-ts-profile-path-attribute"></a>Atributo ms-TS-Profile-Path

Ruta de acceso del perfil de Terminal Services especifica una ruta de acceso de perfil itinerante o obligatoria que se usará cuando el usuario inicie sesión en el servidor de Terminal Server. La ruta de acceso del perfil tiene el siguiente formato de ruta de acceso de red: **\\\\** _ServerName_ *_\\_* _ProfilesFolderName_ *_\\_* _UserName_.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | ms-TS-Profile-Path                          |
| Ldap-Display-Name | msTSProfilePath                             |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1976                     |
| System-Id-Guid    | e65c30db-316c-4060-a3a0-387b083f09cd        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
