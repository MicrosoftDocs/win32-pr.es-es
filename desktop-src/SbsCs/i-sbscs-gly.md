# <a name="i-isolated-applications-and-side-by-side-assemblies"></a>I (aplicaciones aisladas y ensamblados en paralelo)

<dl> <dt>

<span id="_win32_isolated_application_gly"></span><span id="_WIN32_ISOLATED_APPLICATION_GLY"></span>**aplicación aislada**
</dt> <dd>

Aplicación que utiliza ensamblados en paralelo, pero no comparte sus ensamblados. Una aplicación aislada siempre va acompañada de un archivo de manifiesto de aplicación. El manifiesto especifica las versiones de los ensamblados en paralelo compartidos a los que la aplicación se enlaza en tiempo de ejecución y puede contener algunos metadatos para ensamblados en paralelo privados. El manifiesto contiene información almacenada tradicionalmente en el registro; una aplicación totalmente aislada puede ser independiente del registro.

</dd> </dl>
