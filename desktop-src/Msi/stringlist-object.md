# <a name="stringlist-object"></a>Objeto StringList

El objeto **StringList** es una colección de cadenas. El cliente debe comprobar que el objeto **StringList** existe y no está vacío antes de hacer referencia a sus propiedades.

## <a name="members"></a>Miembros

El objeto **StringList** tiene estos tipos de miembros:

-   [Propiedades](#properties)

### <a name="properties"></a>Propiedades

El objeto **StringList** tiene estas propiedades.



| Propiedad                                     | Descripción                                                          |
|:---------------------------------------------|:---------------------------------------------------------------------|
| [**Contabiliza**](stringlist-count.md)<br/> | Devuelve el número de elementos del objeto **StringList** .<br/> |
| [**Elemento**](stringlist-item.md)<br/>   | Devuelve una cadena en la colección de objetos **StringList** .<br/> |
