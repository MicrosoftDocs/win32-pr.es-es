# <a name="ms-dfsr-schedule-attribute"></a>atributo MS-DFSR-Schedule

Contiene la programación de replicación para el servicio de replicación de Sistema de archivos distribuido (DFS).



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | MS-DFSR-Schedule                                      |
| Nombre para mostrar de LDAP | msDFSR-programación                                       |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.14                            |
| System-ID-GUID    | 4699f15f-a71f-48e2-9ff5-5897c0759205                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
