# <a name="top-property-propertysheet-object"></a>Propiedad top (objeto hoja)

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Devuelve un valor entero que especifica el borde superior actual, en píxeles, de la ventana de la hoja de propiedades del agente de Microsoft.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintáctica**
</dt> <dd>

*agente * * *. PropertySheet.Top**

</dd> </dl>

## <a name="remarks"></a>Observaciones

El servidor muestra la ventana en función de la ubicación establecida por el usuario.
