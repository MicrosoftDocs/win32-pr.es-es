# <a name="art-files-mobile"></a>Archivos de arte (móvil)

Deberá crear archivos de arte para la máscara. Hay varios tipos de archivos de arte que puede crear y deben estar en un formato de archivo de arte específico.

En la tabla siguiente se describen siete tipos de archivos de arte que puede crear para la máscara.



| Tipo de archivo Art                            | Descripción                                                                                                |
|------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [Archivo en segundo plano](background-file.md)   | Obligatorio. El fondo es la imagen en la que se superponen todas las imágenes de botón                        |
| [Archivo deshabilitado](disabled-file.md)       | Obligatorio. Imágenes usadas para indicar que insertar botones no tendrá ningún efecto.                                |
| [Archivo pushed](pushed-file.md)           | Obligatorio. Define las imágenes que se muestran cuando el usuario inserta botones.                              |
| [Archivos de región](region-files.md)         | Define mapas de bits que usan áreas de color para representar las ubicaciones de respuesta de pulsación para los botones de tipo de llamada.      |
| [Super Files](super-files.md)           | Define las imágenes deshabilitadas para las barras de seguimiento.                                                                 |
| [Archivo SeekThumb](seekthumb-file.md)     | Define una imagen que se va a usar junto con una barra de seguimiento para indicar la posición actual en el elemento multimedia. |
| [Archivo VolumeThumb](volumethumb-file.md) | Define una imagen que se va a usar junto con una barra de seguimiento para indicar la posición del control de volumen.            |



 

En las secciones siguientes se proporciona más información sobre los archivos de arte.

-   [Formatos de archivo art](art-file-formats-mobile.md)
-   [Paletas de colores](color-palettes.md)
-   [Tamaños de imagen](image-sizes.md)
-   [Colocación de botones](button-placement.md)

Para obtener más información sobre los tipos de archivos de arte y cómo identificarlos en el archivo de definición de máscara, vea la sección [Mapas](bitmaps.md) de bits de la referencia de máscara.
