# <a name="allowincell-attribute"></a>Atributo AllowInCell

Determina si una forma se puede colocar en una tabla. Lectura/escritura **DvTriState**.

**Se aplica a**

[Forma](shape-element--vml.md)

**Sintaxis de etiquetas**

v:*element* o:allowincell="*expression*">

**Comentarios:**

Usado por Microsoft Word. El valor predeterminado es **False**.
