# <a name="endpoint-data-loss-prevention-structures"></a>Estructuras de prevención de pérdida de datos de punto de conexión

Estructuras usadas por la característica de prevención de pérdida de datos del punto de conexión.



| Estructuras                                                       | Descripción                                                           |
|-------------------------------------------------------------------|-----------------------------------------------------------------------|
| [DLP_APP_OP_ENLIGHTENED_LEVEL](endpointdlp-dlp_app_op_enlightened_level.md)                       | Asocia una acción de prevención de pérdida de datos (DLP) de punto de conexión a un nivel de cumplimiento.                                  |
| [DLP_DOCUMENT_INFO](endpointdlp-dlp_document_info.md)                       | Especifica información sobre un documento asociado a una operación DLP de punto de conexión.                                  |
| [DLP_POSTOP_STATUS](enpointdlp-dlp_postop_status.md)                         | Especifica información de estado sobre una operación DLP de punto de conexión.  |
| [DLP_PRINT_INFO](endpointdlp-dlp_print_info.md)                       | Especifica información sobre un documento asociado a una operación de impresión DLP de punto de conexión.                                  |
