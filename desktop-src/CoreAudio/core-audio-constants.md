# <a name="core-audio-constants"></a>Constantes de audio principales

En esta sección se describen las constantes que usan las API de audio principales en Windows Vista y versiones posteriores.

| Constante                                                                            | Descripción                                                                         |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**AUDCLNT \_ SESSIONFLAGS \_ XXX**](audclnt-sessionflags-xxx-constants.md)<br/> | Marcas para la sesión de audio que se abre durante la inicialización de la secuencia.<br/> |
| [**AUDCLNT \_ STREAMFLAGS \_ XXX**](audclnt-streamflags-xxx-constants.md)              | Marcas de inicialización para una secuencia de audio.                                           |
| [**ESTADO \_ DEL \_ DISPOSITIVO XXX**](device-state-xxx-constants.md)                            | Marcas de estado para un dispositivo de punto de conexión de audio.                                           |
| [**COMPATIBILIDAD CON \_ HARDWARE DE PUNTO DE CONEXIÓN \_ \_ XXX**](endpoint-hardware-support-xxx-constants.md) | Marcas de compatibilidad de hardware para un dispositivo de punto de conexión de audio.                                |
