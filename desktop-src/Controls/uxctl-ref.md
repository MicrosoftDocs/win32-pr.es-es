# <a name="visual-styles-reference"></a>Referencia de estilos visuales

En esta sección se describen los siguientes elementos de la API que se usan con [estilos visuales](themes-overview.md).

### <a name="functions"></a>Functions



| Tema                                                                                  | Contenido                                                                                                                                                                                                                           |
|----------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BeginBufferedAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedanimation)                               | Comienza una operación de animación almacenada en búfer. La animación se compone de una transición transversal entre el contenido de dos búferes durante un período de tiempo especificado. <br/>                                                                     |
| [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint)                                       | Comienza una operación de dibujo almacenada en búfer.<br/>                                                                                                                                                                                      |
| [**BeginPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginpanningfeedback)                                   | Notifica al sistema que debe enviar comentarios sobre una ventana de destino afectada por los gestos de movimiento panorámico. <br/>                                                                                                                               |
| [**BufferedPaintClear**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintclear)                                       | Borra un rectángulo especificado en el búfer a ARGB = {0,0,0,0} .<br/>                                                                                                                                                         |
| [**BufferedPaintInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintinit)                                         | Inicializa la pintura almacenada en búfer para el subproceso actual.<br/>                                                                                                                                                                    |
| [**BufferedPaintRenderAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintrenderanimation)                   | Pinta el siguiente fotograma de una animación de pintura almacenada en búfer.<br/>                                                                                                                                                                    |
| [**BufferedPaintSetAlpha**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintsetalpha)                                 | Establece el alfa en un valor especificado en un rectángulo determinado. El alfa controla la cantidad de transparencia aplicada al fusionar con el búfer en el contexto de dispositivo de destino (DC). <br/>                         |
| [**BufferedPaintStopAllAnimations**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintstopallanimations)               | Detiene todas las animaciones almacenadas en búfer para la ventana especificada.<br/>                                                                                                                                                                     |
| [**BufferedPaintUnInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintuninit)                                     | Cierra la pintura almacenada en búfer para el subproceso actual. Se le llama una vez para cada llamada a [**BufferedPaintInit**](/windows/desktop/api/Uxtheme/nf-uxtheme-bufferedpaintinit) después de que ya no se necesiten las llamadas a [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint) .<br/> |
| [**CloseThemeData**](/windows/desktop/api/Uxtheme/nf-uxtheme-closethemedata)                                               | Cierra el identificador de datos del tema.<br/>                                                                                                                                                                                           |
| [**DrawThemeBackground**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemebackground)                                     | Dibuja el borde y el relleno definidos por el estilo visual de la parte de control especificada.<br/>                                                                                                                                   |
| [**DrawThemeBackgroundEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemebackgroundex)                                 | Dibuja la imagen de fondo definida por el estilo visual para la parte de control especificada.<br/>                                                                                                                                  |
| [**DrawThemeEdge**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeedge)                                                 | Dibuja uno o más bordes definidos por el estilo visual de un rectángulo.<br/>                                                                                                                                                     |
| [**DrawThemeIcon**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeicon)                                                 | Dibuja una imagen a partir de una lista de imágenes con el efecto de icono definido por el estilo visual.<br/>                                                                                                                                     |
| [**DrawThemeParentBackground**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeparentbackground)                         | Dibuja la parte de un control primario que está incluida en un control secundario de mezcla alfa o parcialmente transparente.<br/>                                                                                                           |
| [**DrawThemeParentBackgroundEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemeparentbackgroundex)                     | Lo usan los controles secundarios parcialmente transparentes o con mezcla alfa para dibujar la parte de su elemento primario delante de la que aparecen. Envía un mensaje de WM \_ ERASEBKGND seguido de un \_ PRINTCLIENT de WM.<br/>                             |
| [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext)                                                 | Dibuja texto utilizando el color y la fuente definidos por el estilo visual.<br/>                                                                                                                                                        |
| [**DrawThemeTextEx**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetextex)                                             | Dibuja texto utilizando el color y la fuente definidos por el estilo visual. Extiende [**DrawThemeText**](/windows/desktop/api/Uxtheme/nf-uxtheme-drawthemetext) permitiendo opciones de formato de texto adicionales.<br/>                                                             |
| [**EnableThemeDialogTexture**](/windows/desktop/api/Uxtheme/nf-uxtheme-enablethemedialogtexture)                           | Habilita o deshabilita el estilo visual del fondo de una ventana de diálogo.<br/>                                                                                                                                                   |
| [**EnableTheming**](/windows/desktop/api/Uxtheme/nf-uxtheme-enabletheming)                                                 | Habilita o deshabilita los estilos visuales para el usuario actual en las sesiones actuales y posteriores.<br/>                                                                                                                               |
| [**EndBufferedAnimation**](/windows/desktop/api/Uxtheme/nf-uxtheme-endbufferedanimation)                                   | Representa el primer fotograma de una operación de animación almacenada en búfer e inicia el temporizador de animación.<br/>                                                                                                                               |
| [**EndBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-endbufferedpaint)                                           | Completa una operación de dibujo almacenada en búfer y libera el identificador de dibujo almacenado en búfer asociado.<br/>                                                                                                                                    |
| [**EndPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-endpanningfeedback)                                       | Finaliza cualquier animación existente que estaba en proceso o configurada por [**BeginPanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginpanningfeedback) y [**UpdatePanningFeedback**](/windows/desktop/api/Uxtheme/nf-uxtheme-updatepanningfeedback). <br/>                                    |
| [**GetBufferedPaintBits**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpaintbits)                                   | Recupera un puntero al mapa de bits del búfer si el búfer es un mapa de bits independiente del dispositivo (DIB).<br/>                                                                                                                            |
| [**GetBufferedPaintDC**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpaintdc)                                       | Obtiene el controlador de dominio de Paint. Es el mismo valor recuperado por [**BeginBufferedPaint**](/windows/desktop/api/Uxtheme/nf-uxtheme-beginbufferedpaint).<br/>                                                                                                                |
| [**GetBufferedPaintTargetDC**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpainttargetdc)                           | Recupera el controlador de dominio de destino. <br/>                                                                                                                                                                                               |
| [**GetBufferedPaintTargetRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getbufferedpainttargetrect)                       | Recupera el rectángulo de destino especificado por BeginBufferedPaint.<br/>                                                                                                                                                         |
| [**GetCurrentThemeName**](/windows/desktop/api/Uxtheme/nf-uxtheme-getcurrentthemename)                                     | Recupera el nombre del estilo visual actual y, opcionalmente, recupera el nombre y el nombre de la combinación de colores.<br/>                                                                                                           |
| [**GetThemeAppProperties**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeappproperties)                                 | Recupera las marcas de propiedades que controlan cómo se aplican los estilos visuales en la aplicación actual.<br/>                                                                                                                     |
| [**GetThemeBackgroundContentRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundcontentrect)                 | Recupera el tamaño del área de contenido para el fondo definido por el estilo visual.<br/>                                                                                                                                  |
| [**GetThemeBackgroundExtent**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundextent)                           | Calcula el tamaño y la ubicación del fondo, definido por el estilo visual, dado el área de contenido.<br/>                                                                                                                |
| [**GetThemeBackgroundRegion**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebackgroundregion)                           | Calcula la región para un fondo normal o parcialmente transparente que está limitado por un rectángulo especificado.<br/>                                                                                                         |
| [**GetThemeBitmap**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebitmap)                                               | Recupera el mapa de bits asociado a un tema, parte, estado y propiedad determinados.<br/>                                                                                                                                     |
| [**GetThemeBool**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemebool)                                                   | Recupera el valor de una propiedad **bool** de la sección SysMetrics de los datos del tema.<br/>                                                                                                                                   |
| [**GetThemeColor**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemecolor)                                                 | Recupera el valor de una propiedad de color.<br/>                                                                                                                                                                                |
| [**GetThemeDocumentationProperty**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemedocumentationproperty)                 | Recupera el valor de una propiedad de tema de la sección de documentación del archivo de tema especificado.<br/>                                                                                                                    |
| [**GetThemeEnumValue**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeenumvalue)                                         | Recupera el valor de una propiedad de tipo enumerado.<br/>                                                                                                                                                                     |
| [**GetThemeFilename**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemefilename)                                           | Recupera el valor de una propiedad FileName.<br/>                                                                                                                                                                             |
| [**GetThemeFont**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemefont)                                                   | Recupera el valor de una propiedad de fuente. <br/>                                                                                                                                                                                |
| [**GetThemeInt**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeint)                                                     | Recupera el valor de una propiedad **int** .<br/>                                                                                                                                                                             |
| [**GetThemeIntList**](/windows/desktop/api/UxTheme/nf-uxtheme-getthemeintlist)                                             | Recupera una lista de datos **int** de un estilo visual.<br/>                                                                                                                                                                   |
| [**GetThemeMargins**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthememargins)                                             | Recupera el valor de una propiedad [**Margins**](/windows/desktop/api/Uxtheme/ns-uxtheme-margins) .<br/>                                                                                                                                                           |
| [**GetThemeMetric**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthememetric)                                               | Recupera el valor de una propiedad de métrica.<br/>                                                                                                                                                                               |
| [**GetThemePartSize**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemepartsize)                                           | Calcula el tamaño original del elemento definido por un estilo visual.<br/>                                                                                                                                                     |
| [**GetThemePosition**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemeposition)                                           | Recupera el valor de una propiedad Position.<br/>                                                                                                                                                                             |
| [**GetThemePropertyOrigin**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemepropertyorigin)                               | Recupera la ubicación de la definición de la propiedad de tema para una propiedad. <br/>                                                                                                                                                |
| [**GetThemeRect**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemerect)                                                   | Recupera el valor de una propiedad [**Rect**](/previous-versions//dd162897(v=vs.85)) .<br/>                                                                                                                                                                 |
| [**GetThemeStream**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemestream)                                               | Recupera un flujo de datos correspondiente a un tema especificado, empezando por una parte, un estado y una propiedad especificados.<br/>                                                                                                        |
| [**GetThemeString**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemestring)                                               | Recupera el valor de una propiedad de cadena.<br/>                                                                                                                                                                               |

| [**GetThemeSysFont**](/windows/desktop/api/Uxtheme/nf-uxtheme-getthemesysfont)                                             | Recupera el [**LOGFONT**](/windows/win32/api/wingdi/ns-wingdi-logfonta) de una fuente del sistema.<br/>                                                                                                                                                              |
