# <a name="iupdateexceptioncollection-properties"></a>Propiedades de IUpdateExceptionCollection

La interfaz [**IUpdateExceptionCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexceptioncollection) define las siguientes propiedades.



| Propiedad                                                 | Descripción                                                                                                          |
|----------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexceptioncollection-get__newenum) | Obtiene una interfaz [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) que se utiliza para enumerar la colección. |
| [**Contabiliza**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexceptioncollection-get_count)        | Obtiene el número de elementos de la colección.                                                                       |
| [**Elemento**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexceptioncollection-get_item)          | Obtiene una interfaz [**IUpdateException**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexception) en la colección.                                    |
