# <a name="modifying-an-instance-property"></a>Modificar una propiedad de instancia

Después de crear o buscar una instancia de una clase, es probable que desee modificar la instancia. Por ejemplo, puede que desee cambiar el nombre del nombre del sistema almacenado en la instancia local de la clase [**de \_ ComputerSystem de Win32**](/windows/desktop/CIMWin32Prov/win32-computersystem) . La forma más común de actualizar una instancia es actualizar toda la instancia. Para obtener más información, vea [actualizar una instancia completa](updating-an-entire-instance.md). La segunda opción es actualizar solo parte de la instancia. Para obtener más información, vea [Actualizar parte de una instancia](updating-part-of-an-instance.md).

Para obtener más información sobre cómo trabajar con clases, vea [manipular información de clase e instancia](manipulating-class-and-instance-information.md).
