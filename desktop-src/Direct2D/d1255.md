# <a name="d1255-tile-opacity-metadata-conflict"></a>D1255: conflicto de metadatos de opacidad de mosaico

El rectángulo opaco calculado para un gráfico de efectos superpuesto a una región marcada como transparente. Esto puede deberse a un uso incorrecto de una transformación de ajuste de efectos o límites de Atlas.
