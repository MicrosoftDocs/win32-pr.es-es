# <a name="namedqueries-metadatatype-element"></a>Elemento namedQueries (MetadataType)

No se utiliza. Define una lista de consultas con nombre que consultan la cadena de mensaje de evento de un valor y realizan una acción especificada si se encuentran.

``` syntax
<xs:element name="namedQueries"
    type="NamedQueryType"
 />
```

El elemento **namedQueries** se define mediante el tipo complejo de [**MetadataType**](eventmanifestschema-metadatatype-complextype.md) .
