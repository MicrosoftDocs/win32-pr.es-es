# <a name="using-extensible-storage-engine"></a>Usar el motor de almacenamiento extensible


_**Se aplica a:** Windows | Windows Server_

## <a name="using-extensible-storage-engine"></a>Usar el motor de almacenamiento extensible

Esta sección contiene información general sobre cómo usar las siguientes partes de la API de ESE.

  - [Columnas](./columns.md)

  - [Indexación en la tabla](./indexing-in-the-table.md)

  - [Crear bases de datos](./creating-databases.md)

  - [Transactions](./transactions.md)

  - [Errores de ESE](./extensible-storage-engine-errors.md)

  - [Archivos de ESE](./extensible-storage-engine-files.md)

Los archivos de código fuente del programa deben incluir el archivo de encabezado esent. h para tener acceso a los prototipos de función y las definiciones de estructura de la API del motor de almacenamiento extensible. Los desarrolladores pueden utilizar el archivo de biblioteca esent. lib para compilar aplicaciones que usan la API del motor de almacenamiento extensible. En tiempo de ejecución, las aplicaciones se vinculan a la esent.dll.
