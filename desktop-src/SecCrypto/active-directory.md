# <a name="active-directory"></a>Active Directory

Servicios de Certificate Server admite el uso de Active Directory. En el caso de la entidad de [*certificación*](../secgloss/c-gly.md) (CA) empresarial, los servicios de Certificate Server requieren el uso de la Active Directory. La CA de empresa publica certificados de usuario y [*listas de revocación de certificados*](../secgloss/c-gly.md) (CRL) en el Active Directory. Una entidad de certificación independiente se puede administrar para realizar las mismas acciones siempre que el Active Directory esté disponible para la CA independiente y se use un módulo de directivas personalizado.
