# <a name="ipv6-connectivity-and-teredo"></a>Conectividad IPv6 y Teredo

La mayoría de las redes son solo IPv4 y la conectividad IPv6 de un extremo a otro no se puede obtener fácilmente. Además, la mayoría de los dispositivos NAT incluso limitan la conectividad IPv4 de un extremo a otro. Con Windows Vista, es posible la conectividad de un extremo a otro de IPv6 entre dos hosts cualquiera. Los desarrolladores ya no tienen que preocuparse por la conectividad de un extremo a otro, sino crear nuevos escenarios de aplicación que les permitan aprovechar esta conectividad.

El objetivo de Teredo, como tecnología de transición IPv6, es hacer que la conectividad IPv6 de un extremo a otro sea sencilla y automática. En muchos casos, es posible que los desarrolladores ni siquiera necesiten saber que existen tecnologías de transición como Teredo en Windows Vista. Windows Vista tiene la capacidad de usar Teredo en segundo plano como un método sin problemas de conectividad IPv6.
