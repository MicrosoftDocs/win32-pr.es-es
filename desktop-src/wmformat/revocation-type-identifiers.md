# <a name="revocation-type-identifiers"></a>Identificadores de tipo de revocación

Las constantes enumeradas en la tabla siguiente definen valores GUID que identifican distintos tipos de revocación.



| Constante                         | Descripción                                                                                         |
|----------------------------------|-----------------------------------------------------------------------------------------------------|
| \_aplicación de tipo REvocación de WMDRM \_ \_     | Identifica la información de revocación relativa a los certificados de aplicación.                           |
| \_dispositivo de tipo de REvocación de WMDRM \_ \_  | Identifica la información de revocación relativa a los certificados de dispositivo.                                |
| \_tipo de REvocación de WMDRM \_ \_ CARDEA  | Identifica la información de revocación relativa a DRM de Windows Media para los certificados de dispositivos de red. |
| \_tipo de REvocación de WMDRM \_ \_ REVINFO | No documentado en esta versión.                                                                     |
| \_tipo de REvocación de WMDRM \_ \_ GRL     | No documentado en esta versión.                                                                     |
| \_HDCP de tipo de REvocación de WMDRM \_ \_    | No documentado en esta versión.                                                                     |
