# <a name="wmoriginalreleaseyear"></a>WM/OriginalReleaseYear

El atributo **WM/OriginalReleaseYear** contiene el año en el que se liberó el contenido por primera vez.

## <a name="global-constant"></a>Constante global

g \_ wszWMOriginalReleaseYear

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

| Tipo de archivo | Valor de ejemplo |
|-----------|---------------|
