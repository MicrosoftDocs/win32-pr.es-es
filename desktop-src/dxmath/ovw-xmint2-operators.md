# <a name="xmint2-operators"></a>Operadores XMINT2

La estructura [**XMINT2**](/windows/win32/api/directxmath/ns-directxmath-xmint2) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                             | Descripción                                                                                                     |
|-----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| [**operador =**](/windows/win32/api/directxmath/nf-directxmath-xmint2-operator-assign)<br/> | Asigna los datos del componente de vector de una instancia de `XMINT2` a la instancia actual de `XMINT2` .<br/> |
