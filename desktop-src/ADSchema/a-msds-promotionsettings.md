# <a name="ms-ds-promotion-settings-attribute"></a>atributo MS-DS-promocionate-Settings

Para un equipo, contiene una cadena XML que se utilizará para la promoción de DSA delegada.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-DS-promoción-configuración                    |
| Nombre para mostrar de LDAP | msDS-PromotionSettings                      |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1962                     |
| System-ID-GUID    | c881b4e2-43c0-4ebe-b9bb-5250aa9b434c        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
