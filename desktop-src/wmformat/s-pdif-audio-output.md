# <a name="spdif-audio-output"></a>Salida de audio S/PDIF

Algunos dispositivos de audio usan el formato de interconexión digital de Samsung/Samsung (S/PDIF) para transportar datos de audio. Ahora puede descodificar audio en S/PDIF a partir de audio codificado con el códec Windows Media Professional.
