# <a name="meindividualizationcompleted-event"></a>Evento MEIndividualizationCompleted

Generado por el motor de directivas cuando se completa la individualización. Para obtener más información, consulte evento [MEIndividualizationStart](meindividualizationstart.md) .

## <a name="event-values"></a>Valores de evento

Los valores posibles recuperados de [**IMFMediaEvent:: GetValue**](/windows/desktop/api/mfobjects/nf-mfobjects-imfmediaevent-getvalue) son los siguientes.



| VARTYPE              | Descripción                           |
|----------------------|---------------------------------------|
| VT \_ vacío<br/> | Sin datos del evento.<br/> <br/> |
