# <a name="moduleadvtuisequence-table"></a>Tabla ModuleAdvtUISequence

No cree esta tabla. El Windows Installer no utiliza la [tabla AdvtUISequence](advtuisequence-table.md). La tabla AdvtUISequence no debe existir o debe dejarse en blanco en la base de datos de instalación. No cree tampoco la tabla ModuleAdvtUISequence.
