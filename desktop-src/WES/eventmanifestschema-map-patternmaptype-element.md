# <a name="map-patternmaptype-element"></a>Elemento Map (PatternMapType)

Define las expresiones regulares que se usan para buscar una cadena coincidente en la cadena de mensaje y modificarla.

``` syntax
<xs:element name="map"
    type="PatternMapValueType"
 />
```

El elemento de **mapa** se define mediante el tipo complejo de [**PatternMapType**](eventmanifestschema-patternmaptype-complextype.md) .
