# <a name="about-basic-debugging"></a>Acerca de la depuración básica

Las [funciones de depuración](debugging-functions.md) se pueden usar para crear un depurador básico controlado por eventos. *Controlado por eventos* significa que se notifica al depurador cada vez que se producen determinados eventos en el proceso que se está depurando. La notificación permite al depurador tomar las medidas adecuadas en respuesta a los eventos.

Para obtener información general sobre los términos de depuración, vea [Terminología de depuración.](debugging-terminology.md)

[La compatibilidad de depuración de las](debug-support-from-process-thread-and-exception-functions.md) funciones de proceso, subproceso y excepción describe las características específicas de la depuración de determinadas funciones de proceso, subproceso y control de excepciones.

[La creación de un depurador](creating-a-basic-debugger.md) básico describe el uso de las funciones de depuración básicas para crear un depurador simple. Estas funciones permiten a una aplicación esperar eventos de depuración, provocar excepciones de punto de interrupción, transferir el control de ejecución al depurador, y así sucesivamente.

[Eventos de depuración](debugging-events.md) describe el mecanismo de eventos de depuración. Los eventos de depuración hacen que el sistema notifique al depurador.
