# <a name="web-services-on-devices-constants"></a>Constantes de servicios web en dispositivos

Los servicios web en dispositivos usan las siguientes constantes.



| Constante                                                                                                                                                                                                                        | Descripción                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WSD_DEFAULT_HOSTING_ADDRESS"></span><span id="wsd_default_hosting_address"></span><dl> <dt>**DIRECCIÓN DE HOSPEDAJE \_ PREDETERMINADA DE WSD \_ \_**</dt> </dl>                       | Dirección predeterminada (en formato [UrlPrefix)](/windows/desktop/Http/urlprefix-strings) que usará un host de WSDAPI para escuchar solicitudes en el puerto 5357. <br/>                                                 |
| <span id="WSD_DEFAULT_SECURE_HOSTING_ADDRESS"></span><span id="wsd_default_secure_hosting_address"></span><dl> <dt>**DIRECCIÓN DE HOSPEDAJE \_ \_ SEGURO PREDETERMINADA \_ DE \_ WSD**</dt> </dl> | La dirección segura predeterminada (en formato [UrlPrefix)](/windows/desktop/Http/urlprefix-strings) que usará un host de WSDAPI para escuchar las solicitudes en el puerto 5358. <br/>                                          |
| <span id="WSD_DEFAULT_EVENTING_ADDRESS__"></span><span id="wsd_default_eventing_address__"></span><dl> <dt>**WSD \_ DIRECCIÓN \_ DE EVENTOS \_ PREDETERMINADA**</dt> </dl>               | Dirección predeterminada (en formato [UrlPrefix)](/windows/desktop/Http/urlprefix-strings) que usará un host WSDAPI para escuchar eventos en el puerto 5358 para hosts seguros y en el puerto 5357 para otros hosts. <br/> |
