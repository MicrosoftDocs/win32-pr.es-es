# <a name="recognizer-context"></a>Contexto del reconocedor

El objeto [**RecognizerContext**](inkrecognizercontext-class.md) define la configuración de un único contexto de reconocedor. Recibe la tinta para reconocer, define las restricciones en la entrada de lápiz y la salida de reconocimiento, y se puede consultar para obtener los resultados del reconocimiento.

Para obtener más información sobre los datos de tinta, vea [Ink Data](ink-data.md). Para obtener más información sobre la recopilación de entradas manuscritas, vea [recopilación de entradas manuscritas](ink-collection.md).

Las restricciones que se pueden establecer en la salida de entrada y reconocimiento de tinta del objeto [**RecognizerContext**](inkrecognizercontext-class.md) incluyen:

-   El idioma reconocido o devuelto
-   Diccionario que se usó
-   Factoids

Para obtener más información sobre las restricciones que puede establecer en el objeto [**RecognizerContext**](inkrecognizercontext-class.md) , consulte [diccionarios y Factoids](dictionaries-and-factoids.md).
