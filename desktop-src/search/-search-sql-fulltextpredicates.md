# <a name="full-text-predicates"></a>Predicados de Full-Text

El lenguaje de consulta de Microsoft Windows Search admite dos predicados de búsqueda de texto completo. El predicado CONTAINS realiza comparaciones en las columnas que contienen texto. La cláusula Contains puede buscar coincidencias con palabras o frases individuales, en función de la proximidad de los términos de búsqueda. En comparación, el predicado FREETEXT se ajusta para que coincida con el significado de las frases de búsqueda con respecto a las columnas de texto.

Esta sección contiene los siguientes temas:

-   [Predicado CONTAINS](-search-sql-contains.md)
-   [Predicado FREETEXT](-search-sql-freetext.md)
