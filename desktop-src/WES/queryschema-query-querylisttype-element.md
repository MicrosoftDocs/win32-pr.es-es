# <a name="query-querylisttype-element"></a>Elemento Query (QueryListType)

Define un conjunto de selectores y supresores que se usan para incluir eventos en o excluir eventos del conjunto de resultados.

``` syntax
<xs:element name="Query"
    type="QueryType"
 />
```

El tipo complejo **QueryListType** define el [**elemento Query.**](queryschema-querylisttype-complextype.md)
