# <a name="user-interface-plug-ins-programming-reference"></a>Referencia de programación de complementos de interfaz de usuario

El kit de desarrollo de software (SDK) de Microsoft Windows Media Player admite una interfaz para complementos de interfaz de usuario. En la sección siguiente se documenta esta información en detalle.



| Sección                                                  | Descripción                                                                                                   |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [WMPNotifyPluginAddRemove](/previous-versions/windows/desktop/api/wmpplug/nf-wmpplug-wmpnotifypluginaddremove) | Función independiente que se usa para notificar a Windows Media Player que se ha instalado o desinstalado un complemento. |
| [Interfaz IWMPPluginUI](/previous-versions/windows/desktop/api/wmpplug/nn-wmpplug-iwmppluginui)               | Interfaz para complementos de la interfaz de usuario.                                                                                  |
| [Marcas de registro](registration-flags.md)             | Una lista de marcas de configuración que afectan al comportamiento de los complementos de la interfaz de usuario.                                             |
