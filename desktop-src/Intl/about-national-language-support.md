# <a name="about-national-language-support"></a>Acerca de la compatibilidad con National Language

Las funciones de National Language support (NLS) permiten a las aplicaciones:

-   Establecer la configuración regional del usuario
-   Identificar el idioma en el que trabaja el usuario
-   Recupera las cadenas que representan las horas, fechas y otra información con el formato correcto para el idioma y la configuración regional especificados

NLS también incluye compatibilidad con las distribuciones del teclado y las fuentes específicas del idioma. Para obtener más información, vea [Administrador de métodos de entrada](input-method-manager.md) y [fuentes internacionales y presentación de texto](international-fonts-and-text-display.md).
