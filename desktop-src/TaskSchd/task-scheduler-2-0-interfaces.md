# <a name="task-scheduler-20-interfaces"></a>Programador de tareas 2.0 Interfaces

Las interfaces que se describen en los temas siguientes proporcionan acceso mediante programación a la funcionalidad que está disponible en el Programador de tareas que se usa en el sistema operativo Windows Vista.

Estos temas contienen una descripción de la interfaz, una lista de las propiedades y métodos definidos por la interfaz y comentarios sobre las circunstancias especiales que deben tenerse en cuenta al usar la interfaz.


Las siguientes interfaces se presentan en Programador de tareas 2.0.



| Interfaz                                                        | Descripción                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAction**](/windows/desktop/api/taskschd/nn-taskschd-iaction)                                       | Proporciona las propiedades comunes heredadas por todos los objetos de acción.                                                                                                                                                                                                                          |
| [**IActionCollection**](/windows/desktop/api/taskschd/nn-taskschd-iactioncollection)                   | Contiene las acciones que realiza la tarea. Sus métodos se pueden usar para agregar, quitar y recuperar las acciones de una tarea.                                                                                                                                                          |
| [**IBootTrigger**](/windows/desktop/api/taskschd/nn-taskschd-iboottrigger)                             | Representa un desencadenador que inicia una tarea cuando se inicia el sistema                                                                                                                                                                                                                                |
| [**IComHandlerAction**](/windows/desktop/api/taskschd/nn-taskschd-icomhandleraction)                   | Representa una acción que llama a un controlador.                                                                                                                                                                                                                                                        |
| [**IDailyTrigger**](/windows/desktop/api/taskschd/nn-taskschd-idailytrigger)                           | Representa un desencadenador que inicia una tarea en función de una programación diaria.                                                                                                                                                                                                                                |
| [**IEmailAction**](/windows/desktop/api/taskschd/nn-taskschd-iemailaction)                             | Representa una acción que envía un mensaje de correo electrónico.                                                                                                                                                                                                                                                 |
| [**IEventTrigger**](/windows/desktop/api/taskschd/nn-taskschd-ieventtrigger)                           | Representa un desencadenador que inicia una tarea cuando se produce un evento del sistema.                                                                                                                                                                                                                               |
| [**IExecAction**](/windows/desktop/api/taskschd/nn-taskschd-iexecaction)                               | Representa una acción que ejecuta una operación de línea de comandos.                                                                                                                                                                                                                                      |
| [**IIdleSettings**](/windows/desktop/api/taskschd/nn-taskschd-iidlesettings)                           | Especifica cómo el Programador de tareas realiza tareas cuando el equipo está en una condición de inactividad.                                                                                                                                                                                                        |
| [**IIdleTrigger**](/windows/win32/api/taskschd/nn-taskschd-iidletrigger)                             | Representa un desencadenador que inicia una tarea cuando el equipo entra en estado inactivo.                                                                                                                                                                                                                |
| [**ILogonTrigger**](/windows/desktop/api/taskschd/nn-taskschd-ilogontrigger)                           | Representa un desencadenador que inicia una tarea cuando un usuario inicia sesión.                                                                                                                                                                                                                                      |
| [**IMaintenanceSettings**](/windows/desktop/api/taskschd/nn-taskschd-imaintenancesettings)             | Proporciona la configuración que el Programador de tareas usa para realizar tareas durante el mantenimiento automático.                                                                                                                                                                                                  |
| [**IMonthlyDOWTrigger**](/windows/desktop/api/taskschd/nn-taskschd-imonthlydowtrigger)                 | Representa un desencadenador que inicia una tarea según una programación mensual del día de la semana.                                                                                                                                                                                                                        |
| [**IMonthlyTrigger**](/windows/desktop/api/taskschd/nn-taskschd-imonthlytrigger)                       | Representa un desencadenador que inicia una tarea en función de una programación mensual.                                                                                                                                                                                                                              |
| [**INetworkSettings**](/windows/desktop/api/taskschd/nn-taskschd-inetworksettings)                     | Proporciona la configuración que el Programador de tareas utiliza para obtener un perfil de red.                                                                                                                                                                                                           |
| [**IPrincipal**](/windows/desktop/api/taskschd/nn-taskschd-iprincipal)                                 | Proporciona las credenciales de seguridad para una entidad de seguridad.                                                                                                                                                                                                                                                |
| [**IPrincipal2**](/windows/desktop/api/taskschd/nn-taskschd-iprincipal2)                               | Proporciona credenciales de seguridad adicionales para una entidad de seguridad.                                                                                                                                                                                                                                         |
| [**IRegisteredTask**](/windows/desktop/api/taskschd/nn-taskschd-iregisteredtask)                       | Proporciona los métodos que se usan para ejecutar la tarea inmediatamente, obtener las instancias en ejecución de la tarea, obtener o establecer las credenciales que se usan para registrar la tarea y las propiedades que describen la tarea.                                                                                  |
| [**IRegisteredTaskCollection**](/windows/desktop/api/taskschd/nn-taskschd-iregisteredtaskcollection)   | Contiene todas las tareas registradas.                                                                                                                                                                                                                                                       |
| [**IRegistrationInfo**](/windows/desktop/api/taskschd/nn-taskschd-iregistrationinfo)                   | Proporciona la información administrativa que se puede usar para describir la tarea. Esta información incluye detalles como una descripción de la tarea, el autor de la tarea, la fecha en que se registra la tarea y el descriptor de seguridad de la tarea.                                             |
| [**IRegistrationTrigger**](/windows/desktop/api/taskschd/nn-taskschd-iregistrationtrigger)             | Representa un desencadenador que inicia una tarea cuando se registra la tarea.                                                                                                                                                                                                                              |
| [**IRepetitionPattern**](/windows/desktop/api/taskschd/nn-taskschd-irepetitionpattern)                 | Define la frecuencia con la que se ejecuta la tarea y cuánto tiempo se repite el patrón de repetición después de iniciar la tarea.                                                                                                                                                                                      |
| [**IRunningTask**](/windows/desktop/api/taskschd/nn-taskschd-irunningtask)                             | Proporciona los métodos para obtener información de una tarea en ejecución y controlarla.                                                                                                                                                                                                                          |
| [**IRunningTaskCollection**](/windows/desktop/api/taskschd/nn-taskschd-irunningtaskcollection)         | Proporciona una colección que se usa para controlar las tareas en ejecución.                                                                                                                                                                                                                                      |
| [**ISessionStateChangeTrigger**](/windows/desktop/api/taskschd/nn-taskschd-isessionstatechangetrigger) | Desencadena tareas para la conexión o desconexión de la consola, la conexión remota o la desconexión, o las notificaciones de bloqueo o desbloqueo de la estación de trabajo.                                                                                                                                                                      |
| [**IShowMessageAction**](/windows/desktop/api/taskschd/nn-taskschd-ishowmessageaction)                 | Representa una acción que muestra un cuadro de mensaje cuando se activa una tarea.                                                                                                                                                                                                                           |
| [**ITaskDefinition**](/windows/desktop/api/taskschd/nn-taskschd-itaskdefinition)                       | Define todos los componentes de una tarea, como la configuración de la tarea, los desencadenadores, las acciones y la información de registro.                                                                                                                                                                                 |
| [**ITaskFolder**](/windows/desktop/api/taskschd/nn-taskschd-itaskfolder)                               | Proporciona los métodos que se usan para registrar (crear) tareas en la carpeta, quitar tareas de la carpeta y crear o quitar subcarpetas de la carpeta.                                                                                                                                       |
| [**ITaskFolderCollection**](/windows/desktop/api/taskschd/nn-taskschd-itaskfoldercollection)           | Proporciona información y control para una colección de carpetas que contienen tareas.                                                                                                                                                                                                                  |
| [**ITaskHandler**](/windows/desktop/api/taskschd/nn-taskschd-itaskhandler)                             | Define los métodos a los que llama Programador de tareas servicio para administrar un controlador COM.                                                                                                                                                                                                        |
| [**ITaskHandlerStatus**](/windows/desktop/api/taskschd/nn-taskschd-itaskhandlerstatus)                 | Proporciona los métodos que usan los controladores COM para notificar al Programador de tareas sobre el estado del controlador.                                                                                                                                                                                  |
| [**ITaskNamedValuePair**](/windows/desktop/api/taskschd/nn-taskschd-itasknamedvaluepair)               | Crea un par nombre-valor en el que el nombre está asociado al valor.                                                                                                                                                                                                                         |
| [**ITaskNamedValueCollection**](/windows/desktop/api/taskschd/nn-taskschd-itasknamedvaluecollection)   | Contiene una colección de pares nombre-valor de interfaz [**ITaskNamedValuePair.**](/windows/desktop/api/taskschd/nn-taskschd-itasknamedvaluepair)                                                                                                                                                                                           |
| [**ITaskService**](/windows/desktop/api/taskschd/nn-taskschd-itaskservice)                             | Proporciona acceso al servicio Programador de tareas para administrar las tareas registradas.                                                                                                                                                                                                                      |
| [**ITaskSettings**](/windows/desktop/api/taskschd/nn-taskschd-itasksettings)                           | Proporciona la configuración que el Programador de tareas utiliza para realizar la tarea.                                                                                                                                                                                                                  |
| [**ITaskSettings2**](/windows/desktop/api/taskschd/nn-taskschd-itasksettings2)                         | Proporciona una configuración adicional que el Programador de tareas utiliza para realizar la tarea.                                                                                                                                                                                                           |
| [**ITaskSettings3**](/windows/desktop/api/taskschd/nn-taskschd-itasksettings3)                         | Proporciona la configuración extendida que el Programador de tareas usa para ejecutar la tarea.                                                                                                                                                                                                                      |
| [**ITaskVariables**](/windows/desktop/api/taskschd/nn-taskschd-itaskvariables)                         | Define variables de tarea que se pueden pasar como parámetros a controladores de tareas y ejecutables externos que inician las tareas. Los controladores de tareas que necesitan datos de entrada y salida en variables de trabajo deben realizar una interfaz de consulta en el puntero de servicios para [**ITaskVariables**](/windows/desktop/api/taskschd/nn-taskschd-itaskvariables). |
| [**ITimeTrigger**](/windows/desktop/api/taskschd/nn-taskschd-itimetrigger)                             | Representa un desencadenador que inicia una tarea cuando se activa el desencadenador.                                                                                                                                                                                                                            |
| [**ITrigger**](/windows/desktop/api/taskschd/nn-taskschd-itrigger)                                     | Proporciona las propiedades comunes heredadas por todas las interfaces de desencadenador.                                                                                                                                                                                                                      |
| [**ITriggerCollection**](/windows/desktop/api/taskschd/nn-taskschd-itriggercollection)                 | Proporciona los métodos que se usan para agregar, quitar y obtener los desencadenadores de una tarea.                                                                                                                                                                                                        |
| [**IWeeklyTrigger**](/windows/desktop/api/taskschd/nn-taskschd-iweeklytrigger)                         | Representa un desencadenador que inicia una tarea basada en una programación semanal.                                                                                                                                                                                                                               |
