# <a name="about-native-wifi"></a>Acerca de WiFi nativo

Las funciones de la interfaz de programación de aplicaciones (API) WiFi nativa tienen dos propósitos: administrar perfiles de red inalámbrica y administrar conexiones de red inalámbrica. El módulo de configuración automática (ACM) expone los elementos de la API. Los elementos de la API de conexión y desconexión expuestos se pueden usar para invalidar la lógica de configuración automática.
