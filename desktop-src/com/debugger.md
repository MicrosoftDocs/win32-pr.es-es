# <a name="debugger"></a>instantáneas

Almacena el nombre del depurador preferido que se va a utilizar con la depuración remota de COM a través de RPC.

## <a name="registry-entry"></a>Entrada del Registro

```
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\DebugObjectRPCEnabled\AeDebug
   Debugger = value
```

## <a name="remarks"></a>Observaciones

Este es un valor de **reg \_ SZ** .
