# <a name="em_getwordbreakprocex-message"></a>\_Mensaje GETWORDBREAKPROCEX em

Recupera la dirección del procedimiento extendido de separación de palabras que se ha registrado actualmente para un control Rich Edit.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>

No se utiliza; debe ser cero.

</dd> <dt>

*lParam* 
</dt> <dd>

No se utiliza; debe ser cero.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

El mensaje devuelve la dirección del procedimiento actual.
