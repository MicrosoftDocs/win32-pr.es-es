# <a name="specifying-other-search-options"></a>Especificar otras opciones de búsqueda

Active Directory Domain Services admiten la mayoría de las opciones de búsqueda proporcionadas por la tecnología de búsqueda. Para obtener más información sobre las opciones de búsqueda adicionales, consulte [especificar otras opciones de búsqueda con IDirectorySearch](/windows/desktop/ADSI/specifying-other-search-options-with-idirectorysearch).
