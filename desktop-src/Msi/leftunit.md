# <a name="leftunit-property"></a>LeftUnit, propiedad

Si se establece la propiedad **LeftUnit,** la unidad se coloca a la izquierda del número en lugar del lado derecho típico. Esto admite idiomas en los que este es el formato aceptado. La interfaz de usuario usa esta propiedad para mostrar el tamaño de los archivos y el espacio disponible en diferentes volúmenes.
