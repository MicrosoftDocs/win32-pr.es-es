# <a name="locale_idaylzero"></a>configuración regional \_ IDAYLZERO

Especificador para los ceros iniciales en campos de día solo para una fecha corta. El número máximo de caracteres permitido para esta cadena es dos, incluido un carácter nulo de terminación. Es preferible que la aplicación use la constante [ \_ SSHORTDATE de configuración regional](locale-sshortdate.md) en lugar de la configuración regional \_ IDAYLZERO.



| Value | Significado                   |
|-------|---------------------------|
| 0     | Sin ceros a la izquierda para días |
| 1     | Ceros a la izquierda para días    |
