# <a name="repl-uptodate-vector-attribute"></a>Atributo Repl-UpToDate-Vector

Realiza un seguimiento de la información de estado de replicación interna de una unidad de red completa. La información aquí se puede extraer en formato público a través de la API DsReplicaGetInfo(). Presente en todos los objetos raíz nc.



| Entrada | Valor |
|-------------------|--------------------------------------------------------------------------------|
| CN                | Repl-UpToDate-Vector                                                           |
| Ldap-Display-Name | replUpToDateVector                                                             |
| Size              | La longitud es proporcional al número de réplicas (pasadas y presentes) de la nc. |
| Privilegio actualizar  | El sistema establece este valor.                                               |
| Frecuencia de actualización  | Cambios en respuesta a los ciclos de replicación entrantes completados.                   |
| Attribute-Id      | 1.2.840.113556.1.4.4                                                           |
| System-Id-Guid    | bf967a16-0de6-11d0-a285-00aa003049e2                                           |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                          |
