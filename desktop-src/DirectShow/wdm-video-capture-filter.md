# <a name="wdm-video-capture-filter"></a>Filtro de captura de vídeo WDM

El filtro de captura de vídeo WDM controla los dispositivos de captura analógica que usan controladores Modelo de controlador de Windows (WDM).

Este filtro es realmente un complemento de KsProxy en modo kernel. Proporciona páginas de propiedades y las interfaces COM para los controladores WDM que controlan los dispositivos de captura analógicos (también denominados descodificadores de vídeo analógicos). Una aplicación puede tratarla simplemente como un filtro. Para agregar este filtro a un gráfico de filtros, use el [enumerador de dispositivos del sistema](system-device-enumerator.md). Devuelve un moniker único para cada dispositivo que utiliza este complemento. Para obtener más información, consulte [enumeración de dispositivos y filtros](enumerating-devices-and-filters.md) y [Cómo participan los dispositivos de hardware en el gráfico de filtro](how-hardware-devices-participate-in-the-filter-graph.md). Como con cualquier filtro basado en KsProxy, el nombre descriptivo del filtro dependerá del controlador subyacente.

No todos los dispositivos que se exponen a través del filtro de captura de vídeo WDM implementarán todas las interfaces que se enumeran a continuación. La aplicación puede usar **QueryInterface** para determinar qué interfaces son compatibles con un dispositivo determinado.
