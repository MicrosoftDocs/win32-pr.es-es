# <a name="eaphost-supplicant-callbacks"></a>Devoluciones de llamada de suplicante de EAPHost

Las funciones de devolución de llamada de la API suplicante de EAPHost son las siguientes.



| Devolución de llamada                                           | Descripción                                                                                                                                                                                                                                     |
|----------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**NotificationHandler**](/previous-versions/windows/desktop/api) | Un prototipo de devolución de llamada que notifica al solicitante que hay un cambio en el informe de mantenimiento (SoH) y se requiere la reautenticación de una conexión del sistema de [protección de acceso a redes](/windows/desktop/NAP/network-access-protection-start-page) (NAP). |
