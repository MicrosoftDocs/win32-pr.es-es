# <a name="restrictions-on-property-names"></a>Restricciones en los nombres de propiedad

Todos los nombres de propiedad deben seguir estas restricciones.

-   Un nombre de propiedad es un [identificador](identifier.md), que es una cadena de texto que debe comenzar con una letra o un carácter de subrayado. Los identificadores y los nombres de propiedad pueden contener letras, números, caracteres de subrayado o puntos. pero no puede comenzar con un número o un punto.
-   Los nombres de [propiedades públicas](public-properties.md) no pueden contener letras minúsculas.
-   Los nombres de [propiedad privados](private-properties.md) deben contener algunas letras minúsculas.
-   Los nombres de propiedad con el prefijo% representan las variables de entorno del usuario y del sistema. Nunca se escriben en la [tabla de propiedades](property-table.md). La configuración permanente de las variables de entorno solo se puede modificar mediante la [tabla de entorno](environment-table.md).
