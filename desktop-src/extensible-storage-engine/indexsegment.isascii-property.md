# <a name="indexsegmentisascii-property"></a>Propiedad IndexSegment.IsASCII

Obtiene un valor que indica si el segmento de índice está sobre una columna de texto ASCII. Este valor solo es significativo para los segmentos de columna de texto.

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop](./microsoft.isam.esent.interop-namespace.md)  
**Ensamblado:**  Microsoft.Isam.Esent.Interop (en Microsoft.Isam.Esent.Interop.dll)
