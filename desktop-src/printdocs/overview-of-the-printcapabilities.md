# <a name="overview-of-the-printcapabilities"></a>Información general de PrintCapabilities

Este tema no está actualizado. Para obtener la información más reciente, consulte la [especificación del esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

La PrintCapabilities describe las configuraciones o Estados disponibles en un dispositivo. Un dispositivo descrito por PrintCapabilities a menudo se puede colocar en una serie de configuraciones diferentes. En el caso de un dispositivo de impresión, los atributos de impresión típicos incluyen impresión dúplex, resolución y tamaño del medio. Si el dispositivo admite estos atributos, formarán parte de la configuración del dispositivo. El usuario selecciona una configuración determinada asignando un valor específico a cada uno de los atributos disponibles; por ejemplo, dúplex: OneSided, resolución: 600 ppp y MediaSize: legal.

La PrintCapabilities se basa en el marco de trabajo del esquema de impresión, que define los elementos que se pueden utilizar en la PrintCapabilities. Las palabras clave de esquema de impresión definen jerarquías de elementos de uso frecuente, o palabras clave, con el fin de promover la portabilidad de la información y la intención entre clientes individuales. Sin embargo, las palabras clave de esquema de impresión también permiten extensiones privadas para que PrintCapabilities pueda adaptarse a las necesidades específicas.
