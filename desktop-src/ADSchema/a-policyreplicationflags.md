# <a name="policy-replication-flags-attribute"></a>Atributo Policy-Replication-Flags

Determina qué propiedades LSA se replican en los clientes.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Policy-Replication-Flags             |
| Ldap-Display-Name | policyReplicationFlags               |
| Size              | 4 bytes                              |
| Actualizar privilegios  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.633               |
| System-Id-Guid    | 19405b96-3cfa-11d1-a9c0-0000f80367c1 |
| Syntax            | [**Enumeración**](s-enumeration.md) |
