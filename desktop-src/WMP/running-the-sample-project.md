# <a name="running-the-sample-project"></a>Ejecutar el proyecto de ejemplo

El componente COM de ejemplo muestra la funcionalidad proporcionada por los métodos de **IWMPSubscriptionServices** al mostrar un cuadro de diálogo cada vez que Windows Media Player llama a uno de los métodos. Para ver esto, primero debe agregar contenido a la biblioteca que contiene un atributo **ContentDistributor** que coincida con su servicio. Consulte [Agregar el atributo ContentDistributor](adding-the-contentdistributor-attribute.md). A continuación, use Windows Media Player 10 o posterior para abrir la tienda en línea, reproducir el contenido premium y copiar el contenido premium en un dispositivo. Windows Media Player creará una instancia del componente COM de ejemplo y llamará a los métodos cuando corresponda.
