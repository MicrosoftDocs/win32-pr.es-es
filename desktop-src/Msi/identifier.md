# <a name="identifier"></a>Identificador

El tipo de datos del identificador es una cadena de texto. Los identificadores pueden contener los caracteres ASCII A-Z (a-z), dígitos, caracteres de subrayado ( \_ ) o puntos (.). Sin embargo, cada identificador debe comenzar con una letra o un carácter de subrayado.
