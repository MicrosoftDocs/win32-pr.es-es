# <a name="compiling-the-sample-service-provider-using-nmake"></a>Compilar el proveedor de servicios de ejemplo mediante NMAKE

Para compilar el proveedor de servicios de ejemplo mediante NMAKE, abra una ventana de comandos de Visual Studio, vaya a <*ruta de instalación de SDK* > \\ WMFSDK95 \\ WMDM \\ mdsp y escriba `nmake` en el símbolo del sistema.
