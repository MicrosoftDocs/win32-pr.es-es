# <a name="terminal-type"></a>Tipo de terminal

El **tipo de terminal** identifica si un terminal es estático o dinámico. Por ejemplo, un módem es un terminal estático mientras que la ventana de vídeo de la pantalla de un equipo se crea cuando se necesita y se destruye cuando su utilidad es superior.

Este concepto se usa principalmente en aplicaciones TAPI 3.

**TAPI 3. x:** Vea [**ITTerminal:: get \_ TerminalType**](/windows/win32/api/tapi3if/nf-tapi3if-itterminal-get_terminaltype), [**\_ tipo de terminal**](/windows/desktop/api/Tapi3if/ne-tapi3if-terminal_type).
