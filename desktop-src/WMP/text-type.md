# <a name="text-type"></a>Tipo de texto

Debe definir el tipo de cuadro de texto que desea utilizar. En la tabla siguiente se muestran los siguientes tipos válidos.



| Tipo        | Descripción                                                                                                                                                                                                                                                                         |
|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Time        | Posición de reproducción actual en el elemento multimedia en minutos y segundos.                                                                                                                                                                                                                 |
| Lista de reproducción    | Nombre de la lista de reproducción actual.                                                                                                                                                                                                                                                       |
| Dar\#     | Número del elemento actual de la lista de reproducción.                                                                                                                                                                                                                                         |
| \#Pistas    | Número de elementos de la lista de reproducción actual.                                                                                                                                                                                                                                            |
| Title       | Título del elemento actual en la lista de reproducción actual. En determinadas condiciones, es posible que este tipo no se muestre correctamente. Como alternativa, use title como parte de una marquesina.                                                                                                                 |
| Autor      | Autor del elemento actual en la lista de reproducción actual. En determinadas condiciones, es posible que este tipo no se muestre correctamente. Como alternativa, use Author como parte de una marquesina.                                                                                                               |
| Copyright   | Datos de copyright del elemento actual en la lista de reproducción actual. En determinadas condiciones, es posible que este tipo no se muestre correctamente. Como alternativa, utilice los derechos de autor como parte de una marquesina.                                                                                                    |
| Nombre de archivo    | Nombre de archivo del elemento actual en la lista de reproducción actual.                                                                                                                                                                                                                              |
| FilenameExt | La extensión de nombre de archivo del elemento actual en la lista de reproducción actual. Los únicos valores admitidos son "WMA" y "WAV".                                                                                                                                                                 |
| Bitrate     | La velocidad de bits codificada (en kbps) del elemento multimedia actual.                                                                                                                                                                                                                           |
| Frecuencia   | La frecuencia muestreada (en KHz) del elemento multimedia actual.                                                                                                                                                                                                                           |
| Status\*    | El intérprete, el álbum, el género y la velocidad de bits del elemento de audio actual durante la reproducción, o la velocidad de bits y los subtítulos (si procede) del elemento de vídeo actual. Cuando no se reproduce un elemento multimedia, el tipo de texto de estado muestra el estado de reproducción actual (por ejemplo, detenido o en pausa). |



 

\*Disponible para Windows Media Player 10 Mobile o posterior.
