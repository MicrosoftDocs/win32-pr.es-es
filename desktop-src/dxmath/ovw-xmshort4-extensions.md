# <a name="xmshort4-extensions"></a>Extensiones de XMSHORT4

[**XMSHORT4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshort4) describe un vector 4D que consta de componentes de entero con signo de 16 bits.

> [!Note]  
> Las características que se enumeran aquí solo están disponibles cuando se desarrollan con C++.
