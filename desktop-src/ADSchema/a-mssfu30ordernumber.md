# <a name="mssfu-30-order-number-attribute"></a>Atributo msSFU-30-Order-Number

Contiene un valor que usa NIS para comprobar si el mapa ha cambiado. Cada vez que cambian los datos almacenados en el objeto [**msSFU-30-Domain-Info,**](c-mssfu30domaininfo.md) este valor se incrementa. Este valor se usa para realizar un seguimiento de los cambios de datos entre **las llamadas ypxfer.**



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | msSFU-30-Order-Number                       |
| Ldap-Display-Name | msSFU30OrderNumber                          |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.308                 |
| System-Id-Guid    | 02625f05-d1ee-4f9f-b366-55266becb95c        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
