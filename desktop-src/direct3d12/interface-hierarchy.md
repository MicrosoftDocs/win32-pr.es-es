# <a name="interface-hierarchy"></a>Jerarquía de la interfaz

El diagrama muestra la jerarquía de herencia de interfaz.

La mayoría de las aplicaciones solo usarán las interfaces de la parte superior de la jerarquía y no usarán directamente las interfaces heredadas. [**ID3D12Pageable,**](/windows/win32/api/d3d12/nn-d3d12-id3d12pageable) [**ID3D12DeviceChild,**](/windows/desktop/api/d3d12/nn-d3d12-id3d12devicechild) [**ID3D12Object**](/windows/desktop/api/d3d12/nn-d3d12-id3d12object) e [**ID3D12CommandList**](/windows/desktop/api/d3d12/nn-d3d12-id3d12commandlist).

Las nuevas interfaces se muestran en un cuadro verde.
