# <a name="registration-and-licensing"></a>Registro y licencias

Normalmente, un control se proporciona como un servidor en proceso (. DLL), aunque también puede ser un servidor local o remoto (. EXE).

Un control suele admitir el registro automático y crea un conjunto de entradas del registro cuando se crea una instancia. También se puede conceder una licencia a un control para ayudar a evitar el uso no autorizado. Vea [el modelo de objetos componentes](the-component-object-model.md) para obtener más información sobre el registro propio y las licencias.

Para obtener más información, vea la [información del registro de controles ActiveX](activex-controls-registry-information.md).
