# <a name="itemsplaylist"></a>ITEMSPLAYLIST

Se trata de un elemento **PLAYLIST** predefinido con los siguientes valores predeterminados.

``` syntax
backgroundColor="black"
columns="name=Name;Duration=Time"
columnsVisible="false"
dropDownVisible="false"
foregroundColor="white"
```

## <a name="remarks"></a>Comentarios

Esto creará una lista de **reproducción** simple que muestra los elementos de una lista de reproducción sin encabezados de lista desplegable o de columna visibles. Todas las propiedades de esta **lista** de reproducción se pueden invalidar especificándolos explícitamente.

|--------------------|----------------------------------------------|
