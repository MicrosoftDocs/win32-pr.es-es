# <a name="color-palettes-windows-media-player-sdk"></a>Paletas de colores (SDK de Windows Media Player)

Distintos dispositivos tienen diferentes paletas de colores. Probablemente querrá crear diferentes máscaras para cada paleta de colores. Se recomiendan las siguientes paletas de colores:

-   16 bits
-   8 bits
-   escala de grises de 2 bits
