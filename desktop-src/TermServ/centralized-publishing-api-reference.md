# <a name="remoteapp-and-desktop-connection-management-service-api-reference"></a>referencia Administración de conexiones de RemoteApp y Escritorio API de servicio

La API Administración de conexiones de RemoteApp y Escritorio Service admite una interfaz y una estructura que proporcionan información sobre los recursos disponibles para los usuarios de Conexiones de RemoteApp y Escritorio. La implementación de esta interfaz puede realizar un filtrado personalizado de los recursos y proporcionar compatibilidad con los tipos de archivo que no se admiten actualmente.
