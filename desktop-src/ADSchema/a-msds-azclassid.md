# <a name="ms-ds-az-class-id-attribute"></a>atributo MS-DS-AZ-Class-ID

IDENTIFICADOR de clase requerido por la interfaz de usuario de AzRoles en el objeto AzApplication.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-DS-AZ-Class-ID                           |
| Nombre para mostrar de LDAP | msDS-AzClassId                              |
| Tamaño              | 36 caracteres                               |
| Actualizar privilegio  | Administrador de AzRoles                               |
| Frecuencia de actualización  | Durante la inicialización o el cambio de directiva.     |
| Attribute-Id      | 1.2.840.113556.1.4.1816                     |
| System-ID-GUID    | 013a7277-5c2d-49ef-a7de-b765b36a3f6f        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
