# <a name="locale_snative-constants"></a>Constantes de SNATIVE de configuración regional \_ \*

En este tema se definen las constantes de SNATIVE de configuración regional \_ \* utilizadas por NLS para representar nombres de lenguajes nativos.



| Value                       | Significado                                                                                                                                                                                                                                                            |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| configuración regional \_ SNATIVECOUNTRYNAME  | **Windows 7 y versiones posteriores:** Nombre nativo del país o región, por ejemplo, España para España. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.                                                                 |
| configuración regional \_ SNATIVECTRYNAME     | En desuso para Windows 7 y versiones posteriores. Nombre nativo del país o región. Consulte Configuración regional \_ SNATIVECOUNTRYNAME.                                                                                                                                                             |
| configuración regional \_ SNATIVECURRNAME     | **Windows Me/98, windows 2000:** Nombre nativo de la moneda asociada a la configuración regional, en el idioma nativo de la configuración regional. No hay ningún límite en el número de caracteres permitidos para esta cadena.                                                          |
| configuración regional \_ SNATIVEDIGITS       | Equivalentes nativos de ASCII 0 a 9. El número máximo de caracteres permitido para esta cadena es once, incluido un carácter nulo de terminación. Por ejemplo, el árabe usa "012345 6789". Vea también [configuración regional \_ IDIGITSUBSTITUTION](locale-idigitsubstitution.md). |
| configuración regional \_ SNATIVEDISPLAYNAME  | **Windows 7 y versiones posteriores:** Nombre para mostrar de la configuración regional en su idioma nativo, por ejemplo, Deutsch (Deutschland) para la configuración regional alemán (Alemania). <br/>                                                                                                        |
| configuración regional \_ SNATIVELANGNAME     | En desuso para Windows 7 y versiones posteriores. Nombre nativo del idioma. Consulte Configuración regional \_ SNATIVELANGUAGENAME.                                                                                                                                                                  |
| configuración regional \_ SNATIVELANGUAGENAME | **Windows 7 y versiones posteriores:** Nombre nativo del idioma, por ejemplo, հայերեն para armenio (Armenia). El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.                                                         |
