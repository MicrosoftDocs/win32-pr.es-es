# <a name="rendering-and-layout-changes"></a>Representación y cambios de diseño

Windows Internet Explorer 8 interpreta de forma más estricta el marcado HTML que las versiones anteriores, y este cambio puede hacer que las páginas web se muestren incorrectamente. Por ejemplo, cuando un sitio muestra una lista o tabla correctamente en Windows Internet Explorer 7, podría aparecer roto en Internet Explorer 8.

Puede resolver estos tipos de problemas mediante la reducción de la página web para encontrar el CÓDIGO HTML con formato no formato. Si corrige la mayoría de los problemas de visualización, los cambios no crean problemas en versiones anteriores de Windows Internet Explorer porque las versiones anteriores simplemente omiten el HTML extraneous. Para obtener más información sobre estos cambios, vea la entrada de blog Compatibilidad del [sitio e IE8.](/archive/blogs/ie/site-compatibility-and-ie8)
