# <a name="kinds-of-arrays"></a>Tipos de matrices

MIDL proporciona la capacidad de especificar los siguientes tipos de matrices en la aplicación RPC:

-   [Matrices fijas](fixed-arrays.md)
-   [Matrices variables](varying-arrays.md)
-   [Matrices compatibles](conformant-arrays.md)

Los tres tipos de matriz se pueden usar como en los parámetros \[ [](/windows/desktop/Midl/in) \] , \[ [out](/windows/desktop/Midl/out-idl) \] o **\[ in,out. \]**
