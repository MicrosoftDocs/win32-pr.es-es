# <a name="ms-dfsr-rootpath-attribute"></a>atributo MS-DFSR-RootPath

Contiene la ruta de acceso completa del directorio raíz.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-DFSR-RootPath                            |
| Nombre para mostrar de LDAP | msDFSR-RootPath                             |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.3                   |
| System-ID-GUID    | d7d5e8c1-e61f-464f-9fcf-20bbe0a2ec54        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
