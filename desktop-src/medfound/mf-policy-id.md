# <a name="mf_policy_id-attribute"></a>Atributo MF \_ POLICY \_ ID

Identificador que se puede establecer en [UN ELEMENTO IMFOutputPolicy.](/windows/win32/api/mfidl/nn-mfidl-imfoutputpolicy)

## <a name="data-type"></a>Tipo de datos

**UINT32** (tratado como **BOOL)**

## <a name="remarks"></a>Observaciones

El valor se puede recibir del evento [multimedia MEPolicySet.](./mepolicyset.md) Se almacena como una carga **VT_UI4** tipo en el **evento MEPolicySet.**
