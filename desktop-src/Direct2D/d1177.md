# <a name="d1177-unsupported-wic-pixel-format"></a>D1177: formato de píxel WIC no admitido

El formato de píxel de WIC proporcionado no es compatible con Direct2D. En concreto, Direct2D no es compatible con los mapas de bits de WIC que usan Alpha recto a menos que la profundidad sea 8bpp.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación intentó interoperar un recurso de WIC cuyo formato no es compatible con Direct2D. Normalmente, estos formatos son los que tienen un alfa recto y cuya profundidad no es de 8 bits por plano.

## <a name="possible-fixes"></a>Posibles correcciones

Considere la posibilidad de crear el recurso de WIC con un formato alternativo para la interoperabilidad con Direct2D, o bien convertirlo.
