# <a name="interface-implementation-of-system-providers"></a>Implementación de la interfaz de los proveedores del sistema

ADSI incorpora varios proveedores de servicios de directorio, es decir, LDAP y Winnt. Los diferentes proveedores pueden optar por implementar o no una interfaz. Los diferentes proveedores de servicios de directorio admiten diferentes propiedades y métodos. Para obtener más información acerca de la compatibilidad específica con proveedores de servicios de directorio, consulte [ADSI System Providers](adsi-system-providers.md).
