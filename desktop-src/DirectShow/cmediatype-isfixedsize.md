# <a name="cmediatypeisfixedsize-method"></a>CMediaType. IsFixedSize, método

El `IsFixedSize` método determina si los ejemplos tienen un tamaño fijo o un tamaño variable.

## <a name="syntax"></a>Sintaxis


```C++
BOOL IsFixedSize() const;
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve el valor del miembro **bFixedSizeSamples** .
