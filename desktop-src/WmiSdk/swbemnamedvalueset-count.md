# <a name="swbemnamedvaluesetcount-property"></a>SWbemNamedValueSet. Count (propiedad)

Use la propiedad **Count** del objeto [**SWbemNamedValueSet**](swbemnamedvalueset.md) para determinar el número de elementos que hay en la colección. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemNamedValueSet.Count As Integer
```
