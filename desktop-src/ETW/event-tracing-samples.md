# <a name="event-tracing-samples"></a>Ejemplos de seguimiento de eventos

Platform Software Development Kit (SDK) contiene ejemplos completos de seguimiento de eventos. Estos ejemplos se encuentran en el directorio \\ Ejemplos de WinBase \\ EventTrace. La raíz de esta ruta de acceso es el directorio de instalación base del PSDK. Puede descargar el PSDK desde [https://www.microsoft.com/downloads/details.aspx?FamilyId=A55B6B43-E24F-4EA3-A93E-40C0EC4F68E5](https://msdn.microsoft.com/windowsserver/bb980924.aspx) .

En la tabla siguiente se describen los ejemplos incluidos en el directorio Ejemplos \\ de WinBase \\ EventTrace.



| Muestra   | Descripción                                                                                                                            |
|----------|----------------------------------------------------------------------------------------------------------------------------------------|
| TraceDmp | Consumidor de seguimiento de eventos. Descodifica los datos del evento mediante la información de formato obtenida de WMI y genera los datos en un .csv archivo. |
| TraceDp  | Usa la API del proveedor de seguimiento de eventos para proporcionar datos de seguimiento de eventos al registrador o a un consumidor.                                             |
| TraceLog | Usa la API del controlador de seguimiento de eventos para administrar las sesiones de registro.                                                                        |



 

Para obtener descripciones detalladas, consulte Readme.txt archivo incluido con estos ejemplos.
