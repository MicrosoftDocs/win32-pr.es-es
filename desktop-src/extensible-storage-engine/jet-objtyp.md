# <a name="jet_objtyp"></a>JET_OBJTYP


_**Se aplica a:** Windows | Windows Servidor_

## <a name="jet_objtyp"></a>JET_OBJTYP

El **JET_OBJTYP** de constantes representa el tipo de un objeto de base de datos. Actualmente, solo se admiten tablas.
