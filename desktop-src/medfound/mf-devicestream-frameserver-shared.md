# <a name="mf_devicestream_frameserver_shared-attribute"></a>\_ \_ Atributo compartido MF DEVICESTREAM FRAMESERVER \_

Este atributo, cuando se establece en un flujo, marca explícitamente la secuencia como compartida por el servidor de marco.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|-------------------------------------|------------------------------------------------------------------------------------|
| Cliente mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows 10, versión 1703 \[\]<br/>                          |
| Servidor mínimo compatible<br/> | No se admite ninguno<br/>                                                          |
| Encabezado<br/>                   | <dl> <dt>Mfidl. h</dt> </dl> |
