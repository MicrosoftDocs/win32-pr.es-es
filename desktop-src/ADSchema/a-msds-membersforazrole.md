# <a name="ms-ds-members-for-az-role-attribute"></a>atributo MS-DS-Members-for-AZ-role

Lista de grupos de aplicaciones miembro o usuarios vinculados a AZ-role.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-DS-Members-for-AZ-role               |
| Nombre para mostrar de LDAP | msDS-MembersForAzRole                   |
| Tamaño              | \-                                      |
| Actualizar privilegio  | Administrador de AzRoles                           |
| Frecuencia de actualización  | Durante la inicialización o el cambio de directiva. |
| Attribute-Id      | 1.2.840.113556.1.4.1806                 |
| System-ID-GUID    | cbf7e6cd-85a4-4314-8939-8bfe80597835    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
