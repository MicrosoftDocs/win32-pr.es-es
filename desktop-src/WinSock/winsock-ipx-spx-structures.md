# <a name="winsock-ipxspx-structures"></a>Estructuras de WINSOCK IPX/SPX

En la lista siguiente se proporcionan descripciones concisas de cada estructura de Winsock IPX/SPX. Para obtener información adicional sobre cualquier estructura, haga clic en el nombre de la estructura.



| Estructura IPX/SPX                                            | Descripción                                                          |
|--------------------------------------------------------------|----------------------------------------------------------------------|
| [**DATOS DE DIRECCIONES IPX \_ \_**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_address_data)             | Proporciona información sobre un adaptador específico al que está enlazado IPX. |
| [**IPX \_ NETNUM \_ DATA**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_netnum_data)               | Proporciona información sobre un número de red IPX especificado.           |
| [**DATOS \_ DE IPX SPXCONNSTATUS \_**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_spxconnstatus_data) | Proporciona información sobre un socket SPX conectado.                   |
