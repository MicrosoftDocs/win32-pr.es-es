# <a name="systemparameterseventlogginglevel-property"></a>Propiedad SystemParameters. EventLoggingLevel

Obtiene o establece el nivel de detalle de los mensajes de registro de eventos que el motor de base de datos emite al registro de eventos. Los números más altos producirán mensajes de registro de eventos más detallados.
