# <a name="authentication-services"></a>Servicios de autenticación

La interfaz del proveedor de compatibilidad de seguridad (SSPI) proporciona los servicios de autenticación subyacentes para RPC. Por lo tanto, cuando la aplicación especifica un servicio de autenticación, selecciona un SSP. Para obtener una lista de los SSP que admite actualmente S PPP, vea [Authentication-Service Constants](authentication-service-constants.md). Para obtener más información, vea Interfaz del proveedor de compatibilidad de [seguridad (SSPI).](security-support-provider-interface-sspi-.md)

> [!Note]  
> Los términos Proveedor de seguridad y Servicio de autenticación son intercambiables.
