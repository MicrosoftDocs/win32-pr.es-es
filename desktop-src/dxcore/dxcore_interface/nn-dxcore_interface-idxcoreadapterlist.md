# <a name="idxcoreadapterlist-interface"></a>Interfaz IDXCoreAdapterList

La interfaz **IDXCoreAdapterList**   implementa los métodos para recuperar los elementos de adaptador de una lista generada, así como los detalles de la lista. **IDXCoreAdapterList** hereda de la interfaz [IUnknown](/windows/win32/api/unknwn/nn-unknwn-iunknown) . Para obtener instrucciones de programación y ejemplos de código, vea [uso de DXCore para enumerar los adaptadores](../dxcore-enum-adapters.md).
