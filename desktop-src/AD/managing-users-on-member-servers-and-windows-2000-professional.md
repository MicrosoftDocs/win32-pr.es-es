# <a name="managing-users-on-member-servers-and-windows-2000-professional"></a>Administración de usuarios en servidores miembro y Windows 2000 Professional

En los servidores miembro y Windows 2000 Professional, hay una base de datos de seguridad local. Esa base de datos de seguridad local puede contener sus propios usuarios locales cuyo ámbito sea solo el equipo determinado en el que se crean. Al administrar estos tipos de usuarios en servidores miembro y estaciones de trabajo, se usa el proveedor de Winnt.

Cuando se administran usuarios en un dominio de Windows 2000 mediante ADSI, se usa el proveedor LDAP. Al administrar usuarios en servidores miembro y estaciones de trabajo, se usa el proveedor de Winnt.
