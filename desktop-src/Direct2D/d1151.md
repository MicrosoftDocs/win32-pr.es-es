# <a name="d1151-unsupported-array-selection"></a>D1151: selección de matriz no admitida

El IDXGISurface2 proporcionado pertenece a una matriz, pero los segmentos de matriz no se admiten en este nivel de dispositivo (D3D9).






 

## <a name="possible-causes"></a>Causas posibles

Interopped las texturas de Direct3D pueden ser matrices; Direct2D solo funcionará en un segmento de matriz a la vez. Los segmentos de matriz de la textura primaria no se verán afectados ni representarán en Direct2D. Sin embargo, los dispositivos de nivel de característica 9 no admiten la selección de segmentos de matriz y no pueden usar esta característica de interoperabilidad.

## <a name="possible-fixes"></a>Posibles correcciones

En el caso de las aplicaciones que se espera que se ejecuten correctamente en los dispositivos de nivel de característica 9, considere la posibilidad de modificar la lógica de representación para evitar la necesidad de interoperar de texturas de matriz. Si es posible, interopere un intermediario en su lugar. Para minimizar los efectos de solucionar esta limitación, la aplicación puede optar por implementar esto como una ruta de acceso de reserva específica de nivel de característica 9.
