# <a name="iwmpdvd-vb-and-c-interface"></a>Interfaz IWMPDVD (VB y C#)

La **interfaz IWMPDVD proporciona** propiedades y métodos para trabajar con DVD.

La **interfaz IWMPDVD** expone las siguientes propiedades.

## <a name="members"></a>Miembros

La **interfaz IWMPDVD (VB y C#)** tiene estos tipos de miembros:

-   [Métodos](#methods)
-   [Propiedades](#properties)

### <a name="methods"></a>Métodos

La **interfaz IWMPDVD (VB y C#)** tiene estos métodos.



| Método                                                         | Descripción                                                                                                     |
|:---------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------|
| [**Atrás**](wmplibiwmpdvd-iwmpdvd-back--vb-and-c.md)           | Cambia la presentación de un submenú a su menú primario.<br/>                                               |
| [**Reanudar**](wmplibiwmpdvd-iwmpdvd-resume--vb-and-c.md)       | Cambia al modo de reproducción desde el modo de menú y se reanuda en la misma posición que cuando se invocó el menú.<br/> |
| [**titleMenu**](wmplibiwmpdvd-iwmpdvd-titlemenu--vb-and-c.md) | Detiene la reproducción y muestra el menú de título.<br/>                                                          |
| [**topMenu**](wmplibiwmpdvd-iwmpdvd-topmenu--vb-and-c.md)     | Detiene la reproducción y muestra el menú raíz.<br/>                                                           |



 

### <a name="properties"></a>Propiedades

La **interfaz IWMPDVD (VB y C#)** tiene estas propiedades.



| Propiedad                                                            | Tipo de acceso          | Descripción                                                                                                                                                                          |
|:--------------------------------------------------------------------|:---------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Dominio**](wmplibiwmpdvd-iwmpdvd-domain--vb-and-c.md)<br/> | Solo lectura<br/> | Obtiene el dominio actual del DVD.<br/>                                                                                                                                       |
| [**Isavailable**](iwmpdvd-isavailable--vb-and-c.md)<br/>     | Solo lectura<br/> | Obtiene un valor que indica si un tipo especificado de información está disponible o si se puede realizar una acción especificada. En C#, este es el **método get \_ isAvailable.**<br/> |



 

Obtenga una **interfaz IWMPDVD** mediante la siguiente propiedad.



| Object                                                                   | Propiedad                                                   |
|--------------------------------------------------------------------------|------------------------------------------------------------|
| [Objeto AxWindowsMediaPlayer](axwindowsmediaplayer-object--vb-and-c.md) | [**Dvd**](axwmplib-axwindowsmediaplayer-dvd--vb-and-c.md) |
