# <a name="hyper-v-networking-api-reference"></a>Referencia de la API de red de Hyper-V

La arquitectura de red para virtualización modela la arquitectura de red física. Usa objetos de red estándar, como conmutadores, puertos de conmutador y adaptadores de red.

Las siguientes son clases WMI de virtualización relacionadas con las redes.
