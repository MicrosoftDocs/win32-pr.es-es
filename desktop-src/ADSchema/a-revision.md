# <a name="revision-attribute"></a>Atributo de revisión

Nivel de revisión de un descriptor de seguridad u otro cambio. Solo se usa en los objetos sam-server y ds-ui-settings.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Revisión                             |
| Ldap-Display-Name | revision                             |
| Size              | 4 bytes                              |
| Actualizar privilegios  | El sistema establece este valor.     |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.145               |
| System-Id-Guid    | bf967a21-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeración**](s-enumeration.md) |
