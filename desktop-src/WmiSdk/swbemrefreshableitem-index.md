# <a name="swbemrefreshableitemindex-property"></a>Propiedad SWbemRefreshableItem.Index

La **propiedad SWbemRefreshableItem.Index** representa el índice del elemento en su objeto [**SWbemRefresher**](swbemrefresher.md) primario.

Para obtener una explicación de esta sintaxis, vea [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Syntax


```VB
SWbemRefreshableItem.Index As Long
```
