# <a name="systemdevicesaepcontainercontainerid"></a>System.Devices.AepContainer.ContainerId

Identidad del contenedor de puntos de conexión de la Asociación de dispositivos

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.Devices.AepContainer.ContainerId
   shellPKey = PKEY_Devices_AepContainer_ContainerId
   formatID = 0BBA1EDE-7566-4F47-90EC-25FC567CED2A
   propID = 12
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Guid
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
