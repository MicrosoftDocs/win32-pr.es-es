# <a name="mscope-id-attribute"></a>Mscope-Id atributo)

Indica que hay un ámbito de multidifusión en el servidor DHCP especificado.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Mscope-Id                            |
| Nombre para mostrar de LDAP | mscopeId                             |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.716               |
| System-ID-GUID    | 963d2751-48be-11d1-a9c3-0000f80367c1 |
| Sintaxis            | [**String(IA5)**](s-string-ia5.md)  |
