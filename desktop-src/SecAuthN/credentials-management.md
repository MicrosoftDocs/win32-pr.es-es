# <a name="credentials-management"></a>Administración de credenciales

Los desarrolladores que escriben para Windows pueden usar la API de administración de credenciales, incluidas las funciones de la interfaz de usuario (IU) de administración de credenciales para obtener y administrar información de credenciales como nombres de usuario y contraseñas. Estas funciones solicitan la información de la cuenta de Windows que se va a usar en lugar de las credenciales establecidas durante el inicio de sesión. Estas solicitudes normalmente se producen cuando las credenciales de inicio de sesión no tienen permisos necesarios para la aplicación.

Las funciones de la interfaz de usuario de administración de credenciales proporcionan interfaces con la apariencia de la interfaz de usuario de Windows. Estas funciones incluyen opciones personalizables que agregan información del usuario al almacén de credenciales del usuario.

En los temas siguientes se proporciona más información acerca de la API de administración de credenciales:

-   [Tipos de credenciales](kinds-of-credentials.md)
-   [Formatos de nombre de usuario](user-name-formats.md)

Para obtener información sobre las funciones de la API de administración de credenciales, consulte [referencia de autenticación](authentication-reference.md).
