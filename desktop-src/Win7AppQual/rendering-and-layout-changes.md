# <a name="rendering-and-layout-changes"></a>Representación y cambios de diseño

Windows Internet Explorer 8 interpreta de forma más estricta el marcado HTML que las versiones anteriores, y este cambio puede hacer que las páginas web no se muestren correctamente. Por ejemplo, cuando un sitio muestra una lista o una tabla correctamente en Windows Internet Explorer 7, es posible que aparezca interrumpido en Internet Explorer 8.

Puede resolver estos tipos de problemas reduciendo la página web para encontrar el HTML con formato incorrecto. Si soluciona la mayoría de los problemas de visualización, los cambios no crean problemas en versiones anteriores de Windows Internet Explorer, ya que las versiones anteriores simplemente omiten el código HTML extraño. Para obtener más información sobre estos cambios, consulte la entrada de blog [compatibilidad del sitio y IE8](/archive/blogs/ie/site-compatibility-and-ie8) .
