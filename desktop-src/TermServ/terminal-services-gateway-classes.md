# <a name="remote-desktop-gateway-classes"></a>Escritorio remoto gateway

El proveedor WMI Escritorio remoto Gateway (Puerta de enlace de Escritorio remoto) proporciona las clases siguientes.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**TSGatewayConnection de Win32 \_**](win32-tsgatewayconnection.md)
</dt> <dd>

Representa una conexión desde un equipo cliente a un servidor de puerta de enlace de Escritorio remoto.

</dd> <dt>

[**Win32 \_ TSGatewayConnectionAuthorizationPolicy**](win32-tsgatewayconnectionauthorizationpolicy.md)
</dt> <dd>

Describe una directiva de Escritorio remoto de autorización de conexión (CAP de Escritorio remoto). Las CAP de Escritorio remoto se usan para determinar si un usuario puede conectarse al servidor de puerta de enlace de Escritorio remoto.

</dd> <dt>

[**Win32 \_ TSGatewayLoadBalancer**](win32-tsgatewayloadbalancer.md)
</dt> <dd>

Describe un conjunto de servidores de equilibrio de carga de puerta de enlace de Escritorio remoto. Se usan para equilibrar la carga de las conexiones de puerta de enlace de Escritorio remoto en varios servidores.

</dd> <dt>

[**TSGatewayRADIUSServer de Win32 \_**](win32-tsgatewayradiusserver.md)
</dt> <dd>

Describe un servidor Servicio de autenticación remota telefónica de usuario (RADIUS), que tiene un conjunto de directivas Servicios de Escritorio remoto autorización de conexión (CAP de Escritorio remoto).

</dd> <dt>

[**Win32 \_ TSGatewayResourceAuthorizationPolicy**](win32-tsgatewayresourceauthorizationpolicy.md)
</dt> <dd>

Describe una directiva Escritorio remoto autorización de recursos de escritorio remoto (RD RAP). Un RAP de Escritorio remoto se usa para decidir si un usuario está autorizado para conectarse a un recurso especificado a través de puerta de enlace de Escritorio remoto.

</dd> <dt>

[**TSGatewayResourceGroup de Win32 \_**](win32-tsgatewayresourcegroup.md)
</dt> <dd>

Describe un grupo de recursos, que asigna un conjunto de recursos (nombres de equipo) a una sola entidad.

</dd> <dt>

[**TSGatewayServer de Win32 \_**](win32-tsgatewayserver.md)
</dt> <dd>

Se usa para las operaciones específicas del servidor de puerta de enlace de Escritorio remoto.

</dd> <dt>

[**Win32 \_ TSGatewayServerSettings**](win32-tsgatewayserversettings.md)
</dt> <dd>

Proporciona métodos y propiedades para ver y configurar las opciones del servidor de puerta de enlace de Escritorio remoto.

</dd> </dl>
