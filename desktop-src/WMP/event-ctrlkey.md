# <a name="eventctrlkey"></a>evento. ctrlKey

El atributo **ctrlKey** recupera un valor que indica si la tecla Ctrl estaba inactiva cuando se produjo el evento.

``` syntax
event.ctrlKey
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **valor booleano** de solo lectura.



| Value | Descripción                                      |
|-------|--------------------------------------------------|
| true  | Indica que la tecla CTRL estaba en la posición hacia abajo. |
| false | Indica que la tecla CTRL estaba en la posición arriba.   |



 

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
