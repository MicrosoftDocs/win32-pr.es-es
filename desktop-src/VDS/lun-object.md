## <a name="lun-types"></a><span data-ttu-id="3705a-107">Tipos de LUN</span><span class="sxs-lookup"><span data-stu-id="3705a-107">LUN Types</span></span>

-   <span data-ttu-id="3705a-119">Los LUN reflejados son LUN tolerantes a errores que permiten la recuperación ante desastres mediante la duplicación de los datos en varios complejos LUN.</span><span class="sxs-lookup"><span data-stu-id="3705a-119">Mirrored LUNs are fault tolerant LUNs that provide for disaster recovery by duplicating the data to multiple LUN plexes.</span></span> <span data-ttu-id="3705a-120">Cada complejo de un LUN reflejado contiene una copia de los datos que se almacenan en el Plex original.</span><span class="sxs-lookup"><span data-stu-id="3705a-120">Each plex in a mirrored LUN contains a copy of the data that is stored on the original plex.</span></span> <span data-ttu-id="3705a-121">Cada uno de los Plex reside en una unidad independiente.</span><span class="sxs-lookup"><span data-stu-id="3705a-121">Each of the plexes resides on a separate drive.</span></span> <span data-ttu-id="3705a-122">Todos los datos que se escriben en un LUN reflejado se escriben simultáneamente en cada uno de sus complejos.</span><span class="sxs-lookup"><span data-stu-id="3705a-122">All data that is written to a mirrored LUN is written simultaneously to each of its plexes.</span></span> <span data-ttu-id="3705a-123">Si se produce un error en una de las unidades contribuyentes, el Plex de esa unidad deja de estar disponible, pero el sistema continúa funcionando con el Plex o complejos no afectados.</span><span class="sxs-lookup"><span data-stu-id="3705a-123">Should one of the contributing drives fail, the plex on that drive becomes unavailable, but the system continues to operate using the unaffected plex or plexes.</span></span> <span data-ttu-id="3705a-124">Un LUN reflejado puede tener cualquier número de Plex.</span><span class="sxs-lookup"><span data-stu-id="3705a-124">A mirrored LUN can have any number of plexes.</span></span>
-   <span data-ttu-id="3705a-125">Los LUN seccionados con paridad son LUN con tolerancia a errores que permiten la recuperación ante desastres mediante la división intermitente de datos de paridad en tres o más unidades.</span><span class="sxs-lookup"><span data-stu-id="3705a-125">Striped with parity LUNs are fault-tolerant LUNs that provide for disaster recovery by striping parity data intermittently across three or more drives.</span></span> <span data-ttu-id="3705a-126">Si se produce un error en una de las unidades contribuyentes, los datos perdidos se pueden volver a crear a partir de los datos y la paridad restantes.</span><span class="sxs-lookup"><span data-stu-id="3705a-126">Should one of the contributing drives fail, the lost data can be recreated from the remaining data and parity.</span></span>
