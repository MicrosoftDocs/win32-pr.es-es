# <a name="iupdatesearcher3-methods"></a>Métodos IUpdateSearcher3

La interfaz [**IUpdateSearcher3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher3) define el método siguiente:



| Método                                              | Descripción                                                                                                                                    |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SearchScope**](/previous-versions/windows/desktop/ee694850(v=vs.85)) | Obtiene y establece un valor de enumeración de [**SearchScope**](/windows/win32/api/wuapi/ne-wuapi-searchscope) que indica el ámbito de las actualizaciones devueltas por la búsqueda. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea las siguientes interfaces.

-   [**IUpdateSearcher**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher)
-   [**IUpdateSearcher2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher2)
