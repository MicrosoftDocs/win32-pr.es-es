# <a name="iinstallationresult-methods"></a>Métodos IInstallationResult

La interfaz [**IInstallationResult**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationresult) define el método siguiente.



| Método                                                         | Descripción                                                                                                                                        |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetUpdateResult**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationresult-getupdateresult) | Devuelve una interfaz [**IUpdateInstallationResult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstallationresult) que contiene los resultados de la instalación de una actualización especificada. |
