# <a name="the-word-balloon"></a>El globo de palabras

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Además de la salida de audio oral, la interfaz del agente de Microsoft admite el título de texto en forma de salida de texto en globos de palabras de estilo animado. Las palabras aparecen en el globo a medida que se pronuncian. El globo se oculta cuando se completa la salida de voz. La ventana Opciones de carácter avanzadas proporciona opciones para deshabilitar la presentación del globo, así como los atributos de control relacionados con su apariencia.
