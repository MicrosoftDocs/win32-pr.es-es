# <a name="wmcategory"></a>WM/Category

El **atributo WM/Category** contiene la categoría del contenido. No hay especificaciones para las categorías. Se recomienda a los usuarios que escriban sus propias categorías para organizar su contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMCategory

## <a name="data-type"></a>Tipo de datos

**CADENA DE \_ TIPO \_ WMT**

## <a name="example"></a>Ejemplo



| Tipo de archivo | Valor de ejemplo   |
|-----------|-----------------|
| Audio     | "Driving Música" |
| Vídeo     | "Vídeos de inicio"   |
