# <a name="ice33"></a>ICE33

ICE33 procesa las [](registry-table.md) entradas de la tabla del Registro y puede emitir una advertencia para cada entrada de tabla que registra clases, extensiones de nombre de archivo, progID, verbos de shell, appID de servidor remoto, tipos MIME o typelibs. Las advertencias están obsoletas y se pueden omitir.
