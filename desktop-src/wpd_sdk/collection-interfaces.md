# <a name="collection-interfaces"></a>Interfaces de colección

Las interfaces siguientes se usan para contener grupos de valores de varios tipos, incluidos los valores de **PROPERTYKEY** , los valores de **GUID** u otras interfaces de colección. Las aplicaciones cliente y los controladores usan estas interfaces para almacenar los datos de los parámetros de método.



| Interfaz                                                                            | Descripción                                                            |
|--------------------------------------------------------------------------------------|------------------------------------------------------------------------|
| [**IPortableDeviceKeyCollection**](iportabledevicekeycollection.md)                 | Contiene una colección de valores **PROPERTYKEY** .                          |
| [**IPortableDevicePropVariantCollection**](iportabledevicepropvariantcollection.md) | Contiene una colección de valores **PROPVARIANT** del mismo VARTYPE.      |
| [**IPortableDeviceValues**](iportabledevicevalues.md)                               | Contiene una colección de  / pares **PROPVARIANT** de PROPERTYKEY.           |
| [**IPortableDeviceValuesCollection**](iportabledevicevaluescollection.md)           | Contiene una colección de interfaces **IPortableDeviceValues** basadas en cero. |
