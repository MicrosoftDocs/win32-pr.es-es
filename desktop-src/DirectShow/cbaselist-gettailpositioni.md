# <a name="cbaselistgettailpositioni-method"></a>CBaseList. GetTailPositionI, método

El `GetTailPositionI` método recupera la posición del último elemento de la lista.

## <a name="syntax"></a>Sintaxis


```C++
POSITION GetTailPositionI();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve un valor de posición o **null** si la lista está vacía.
