# <a name="pens"></a>Lápices

Un lápiz es una herramienta de gráficos que una aplicación puede usar para dibujar líneas y curvas. Las aplicaciones de dibujo usan lápices para dibujar líneas de FreeHand, líneas rectas y curvas. Las aplicaciones de diseño asistido por PC (CAD) utilizan lápices para dibujar líneas visibles, líneas ocultas, líneas de sección, líneas de centro, etc. Las aplicaciones de procesamiento de texto y publicación de escritorio usan lápices para dibujar bordes y reglas. Las aplicaciones de hojas de cálculo usan lápices para designar tendencias en gráficos y en gráficos de barras y gráficos circulares.
