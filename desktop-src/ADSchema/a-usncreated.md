# <a name="usn-created-attribute"></a>USN-Created atributo)

Número de secuencia de actualización (USN) asignado durante la creación del objeto. Vea también, [**USN-Changed**](a-usnchanged.md).



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | USN-Created                          |
| Nombre para mostrar de LDAP | uSNCreated                           |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | Cuando se crea el objeto.          |
| Attribute-Id      | 1.2.840.113556.1.2.19                |
| System-ID-GUID    | bf967a70-0de6-11d0-a285-00aa003049e2 |
| Sintaxis            | [**Interval**](s-interval.md)       |
