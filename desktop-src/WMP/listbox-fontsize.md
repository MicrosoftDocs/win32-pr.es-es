# <a name="listboxfontsize"></a>LISTBOX.fontSize

El **atributo fontSize** especifica o recupera el tamaño de fuente del control de cuadro de lista.

``` syntax
        elementID.fontSize
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un número de lectura **y** escritura **(long)** que especifica el tamaño de fuente en puntos. El valor predeterminado es 10.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media para Windows XP o posterior<br/> |
