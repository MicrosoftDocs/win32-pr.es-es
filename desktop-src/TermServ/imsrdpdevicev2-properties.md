# <a name="imsrdpdevicev2-properties"></a>Propiedades de IMsRdpDeviceV2

La [**interfaz IMsRdpDeviceV2**](imsrdpdevicev2.md) expone las siguientes propiedades.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**CmClassGuid, propiedad**](imsrdpdevicev2-cmclassguid.md)
</dt> <dd>

Contiene el GUID de la clase de instalación de Configuration Manager para el dispositivo.

</dd> <dt>

[**CmDeviceInstance, propiedad**](imsrdpdevicev2-cmdeviceinstance.md)
</dt> <dd>

Contiene la instancia de dispositivo de Configuration Manager del dispositivo.

</dd> <dt>

[**Propiedad DeviceText**](imsrdpdevicev2-devicetext.md)
</dt> <dd>

Contiene el texto del dispositivo.

</dd> <dt>

[**Propiedad DriveLetterBitmap**](imsrdpdevicev2-driveletterbitmap.md)
</dt> <dd>

Contiene un campo de bits que representa un mapa de letras de unidad contenidas en el dispositivo.

</dd> <dt>

[**Propiedad IsCompositeDevice**](imsrdpdevicev2-iscompositedevice.md)
</dt> <dd>

Especifica si el dispositivo es un dispositivo compuesto.

</dd> <dt>

[**Propiedad IsOptionalDevice**](imsrdpdevicev2-isoptionaldevice.md)
</dt> <dd>

Especifica si el dispositivo es opcional para el redireccionamiento USB.

</dd> <dt>

[**Propiedad IsUSBDevice**](imsrdpdevicev2-isusbdevice.md)
</dt> <dd>

Especifica si el dispositivo es para el redireccionamiento USB.

</dd> </dl>
