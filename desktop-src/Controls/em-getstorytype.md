# <a name="em_getstorytype-message"></a>\_Mensaje GETSTORYTYPE em

Obtiene el tipo de caso.


```C++
#define EM_GETSTORYTYPE       (WM_USER + 290)
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>

Índice del caso.

</dd> <dt>

*lParam* 
</dt> <dd>

Sector debe ser 0.

</dd> </dl>
