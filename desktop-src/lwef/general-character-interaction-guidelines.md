# <a name="general-character-interaction-guidelines"></a>Instrucciones generales de interacción de caracteres

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Los principios del buen diseño de la interfaz de usuario también se aplican al diseño de una interfaz mediante Microsoft Agent. Debe comprender los usuarios y sus objetivos, así como seguir un buen proceso de diseño de la interfaz de usuario. Las directrices de la interfaz de Windows para el diseño de software proporcionan una excelente introducción a los principios de diseño generales y la metodología.

-   [Proporcionar los comentarios adecuados](provide-appropriate-feedback.md)
-   [Usar variación natural](use-natural-variation.md)
