# <a name="ms-dfsr-computerreference-attribute"></a>atributo MS-DFSR-ComputerReference

Contiene un vínculo hacia delante a un objeto de [**equipo**](c-computer.md) .



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-DFSR-ComputerReference               |
| Nombre para mostrar de LDAP | msDFSR-ComputerReference                |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.101             |
| System-ID-GUID    | 6c7b5785-3d21-41bf-8a8a-627941544d5a    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
