# <a name="d1184-createwicbitmaprendertarget-incompatible-with-hw-target-type"></a>D1184: CreateWicBitmapRenderTarget incompatible con el tipo de destino de HW

No es válido pasar el tipo de destino de representación de la marca D2D1 \_ \_ \_ \_ a CreateWicBitmapRenderTarget.






 

## <a name="possible-causes"></a>Causas posibles

Cuando una aplicación llamada ID2D1Factory:: CreateWicBitmapRenderTarget, el campo de tipo del \_ argumento de propiedades de destino de representación D2D1 \_ \_ era D2D1 \_ D2D1 representación de hardware de tipo de \_ \_ destino \_ \_ .

## <a name="possible-fixes"></a>Posibles correcciones

Compruebe que el tipo de destino que se usa al crear un destino de representación de WIC-interopped es SOFTWARE o predeterminado. Direct2D normalmente reserva la semántica de HARDWARE para describir la interoperabilidad de Direct3D.
