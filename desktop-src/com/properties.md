# <a name="exposing-properties-through-idispatch"></a>Exponer propiedades a través de IDispatch

Aunque la mayoría de los controles tienen propiedades, no se necesitan controles para exponer ninguna propiedad y, por lo tanto, el control no requiere **IDispatch**. Si el control tiene propiedades, no hay ningún requisito para las propiedades que debe exponer un control.
