# <a name="must-contain-attribute"></a>Must-Contain atributo)

Lista de atributos obligatorios para una clase. Estos atributos deben especificarse cuando se crea una instancia de la clase.



| Entrada | Value |
|-------------------|-----------------------------------------------------------------|
| CN                | Must-Contain                                                    |
| Nombre para mostrar de LDAP | mustContain                                                     |
| Tamaño              | \-                                                              |
| Actualizar privilegio  | Administrador de esquema                                            |
| Frecuencia de actualización  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.24                                           |
| System-ID-GUID    | bf9679d3-0de6-11d0-a285-00aa003049e2                            |
| Sintaxis            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
