# <a name="dsa-signature-attribute"></a>DSA-Signature atributo)

El DSA-Signature de un objeto es el identificador de invocación del último directorio en el que se va a modificar el objeto.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | DSA-Signature                                         |
| Nombre para mostrar de LDAP | dSASignature                                          |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | El sistema establece este valor.                      |
| Frecuencia de actualización  | Cada vez que se modifica un objeto.                       |
| Attribute-Id      | 1.2.840.113556.1.2.74                                 |
| System-ID-GUID    | 167757bc-47f3-11d1-a9c3-0000f80367c1                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
