# <a name="handling-text-related-events"></a>Controlar eventos relacionados con el texto

En este tema se describen los eventos generados por los controles basados en texto cuando se cambia, se selecciona o se anula la selección de contenido.


Los controles basados en texto generan un evento [**\_ \_ TextChangedEventId de texto de UIA**](uiauto-event-ids.md) cada vez que cambia el contenido textual. Generan el evento [**\_ \_ TextSelectionChangedEventId de texto de UIA**](uiauto-event-ids.md) siempre que se selecciona o se anula la selección de contenido textual. Para obtener más información, consulte [UI Automation Events Overview](uiauto-eventsoverview.md) y [suscribirse a eventos de automatización de la interfaz de usuario](uiauto-eventsforclients.md).
