# <a name="remote-access-request-for-comments"></a>Solicitud de acceso remoto para comentarios

Las solicitudes de comentarios (RFC) son una serie continua de documentos que describen las tecnologías relacionadas con Internet y Internet. Las RFC particulares se identifican por número. En la sección servicio de acceso remoto (RAS) del kit de desarrollo de software (SDK) de la plataforma se hace referencia a las siguientes RFC. Para obtener más información acerca de RFC y cómo buscar copias de una RFC determinada, vea [https://www.rfc-editor.org/rfc.html](https://www.rfc-editor.org/rfc.html) .

-   [RFC 791](https://www.ietf.org/rfc/rfc791.txt) : Protocolo de Internet, especificación del Protocolo de programas de Internet DARPA, 1981 de septiembre
-   [RFC 1171](https://www.ietf.org/rfc/rfc1171.txt) : el Protocolo punto a punto, julio 1990
-   [RFC 1172](https://www.ietf.org/rfc/rfc1172.txt) : opciones de configuración inicial de PPP, julio, 1990
-   [RFC 1570](https://www.ietf.org/rfc/rfc1570.txt) : Extensiones LCP de PPP, enero y 1994
-   [RFC 1962](https://www.ietf.org/rfc/rfc1962.txt) : el protocolo de control de compresión (CCP) de PPP, 1996 de junio
-   [RFC 1974](https://www.ietf.org/rfc/rfc1974.txt) : Protocolo de compresión de LZS de PPP Stac, 1996 de agosto
-   [RFC 1990](https://www.ietf.org/rfc/rfc1990.txt) : el protocolo PPP Multilink (MP), 1996 de agosto
-   [RFC 2118](https://www.ietf.org/rfc/rfc2118.txt) : Protocolo de compresión punto a punto de Microsoft (MPPC), 1997 de marzo
-   [RFC 2284](https://www.ietf.org/rfc/rfc2284.txt) : Protocolo de autenticación extensible PPP, 1998 de marzo
-   [RFC 2433](https://www.ietf.org/rfc/rfc2433.txt) : extensiones de Microsoft PPP CHAP, 1998 de octubre
-   [RFC 2484](https://www.ietf.org/rfc/rfc2484.txt) : opción de configuración de internacionalización de PPP LCP, 1999 de enero
-   [RFC 2548](https://www.ietf.org/rfc/rfc2848.txt) : atributos RADIUS específicos del proveedor de Microsoft
-   [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt) : servicio de autenticación remota telefónica de usuario (RADIUS), 2000 de junio
-   [RFC 2866](https://www.ietf.org/rfc/rfc2866.txt) – contabilidad de Radius, 2000 de junio
-   [RFC 2868](https://www.ietf.org/rfc/rfc2868.txt) : atributos RADIUS para la compatibilidad con el protocolo de túnel, 2000 de junio
-   [RFC 2869](https://www.ietf.org/rfc/rfc2869.txt) : extensiones de RADIUS, 2000 de junio
-   [Draft RADIUS MSCHAP ATTR 01](https://www.freeradius.org/rfc/draft-ietf-radius-mschap-attr-01.txt) : atributos RADIUS para la compatibilidad con MS-CHAP, noviembre, 1997
