# <a name="eaphost-peer-method-api-reference"></a>Referencia de API de método del mismo nivel de EAPHost

La documentación de la API del método del homólogo de EAPHost consta de las siguientes secciones.

-   [Funciones de configuración del método del mismo nivel de EAPHost](eaphost-peer-method-configuration-functions.md)
-   [Funciones Run-Time del método del mismo nivel](eaphost-peer-method-run-time-functions.md)
-   [Enumeraciones de métodos del mismo nivel de EAPHost](eap-host-peer-method-enumerations.md)
-   [Estructuras de método del mismo nivel de EAPHost](eap-host-peer-method-structures.md)

Tenga en cuenta que estos elementos se deben implementar en un archivo DLL compatible mediante el autor del método EAP del mismo nivel. El archivo DLL con la implementación de estos elementos de la API se carga y se llama desde EAPHost del lado cliente.
