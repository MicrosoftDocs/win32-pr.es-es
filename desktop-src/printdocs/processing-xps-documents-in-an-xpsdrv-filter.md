# <a name="processing-xps-documents-in-an-xpsdrv-filter"></a>Procesar documentos XPS en un filtro XPSDrv

El OM XPS se puede usar para tener acceso a los componentes de un documento XPS desde un filtro de controlador de impresora XPSDrv.

Para obtener más información acerca de los filtros XPS y los controladores de impresora XPSDrv, consulte [filtros XPS](/windows-hardware/drivers/print/xps-filters) en el kit de controladores de Windows.
