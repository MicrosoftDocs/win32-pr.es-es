# <a name="ms-dfsr-rdcminfilesizeinkb-attribute"></a>atributo MS-DFSR-RdcMinFileSizeInKb

Contiene el tamaño mínimo de archivo, en kilobytes, al que se va a aplicar RDC.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-RdcMinFileSizeInKb           |
| Nombre para mostrar de LDAP | msDFSR-RdcMinFileSizeInKb            |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.20           |
| System-ID-GUID    | f402a330-ace5-4dc1-8cc9-74d900bf8ae0 |
| Sintaxis            | [**Interval**](s-interval.md)       |
