# <a name="file-name"></a>Nombre de archivo

Este es el nombre del archivo que contiene la imagen. No se debe proporcionar la ruta de acceso y el archivo debe estar en el directorio de máscara. La extensión de nombre de archivo debe ser. bmp o. gif.
