# <a name="precomputed-radiance-transfer-functions"></a>Funciones de transferencia de radiancia precalutadas

D3DX proporciona las siguientes funciones de transferencia de radiancia precalutadas (PRT):

-   [**D3DXCreatePRTBuffer**](d3dxcreateprtbuffer.md)
-   [**D3DXCreatePRTBufferTex**](d3dxcreateprtbuffertex.md)
-   [**D3DXCreatePRTCompBuffer**](d3dxcreateprtcompbuffer.md)
-   [**D3DXCreatePRTEngine**](d3dxcreateprtengine.md)
-   [**D3DXCreateTextureGutterHelper**](d3dxcreatetexturegutterhelper.md)
-   [**D3DXLoadPRTBufferFromFile**](d3dxloadprtbufferfromfile.md)
-   [**D3DXLoadPRTCompBufferFromFile**](d3dxloadprtcompbufferfromfile.md)
-   [**D3DXSavePRTBufferToFile**](d3dxsaveprtbuffertofile.md)
-   [**D3DXSavePRTCompBufferToFile**](d3dxsaveprtcompbuffertofile.md)
-   [**D3DXSHPRTCompSplitMeshSC**](d3dxshprtcompsplitmeshsc.md)
-   [**D3DXSHPRTCompSuperCluster**](d3dxshprtcompsupercluster.md)

Para obtener más información sobre el uso de PRT, vea [Precomputed Radiance Transfer (Direct3D 9)](precomputed-radiance-transfer.md).

Las [funciones D3DXSHxxx](dx9-graphics-reference-d3dx-functions-math.md) también se usan para las operaciones de armónica esférica (SH) de PRT.
