# <a name="wireless-ad-hoc-enumerations"></a>Enumeraciones ad hoc inalámbricas

La interfaz ad hoc inalámbrica define y utiliza las enumeraciones siguientes.

-   [**\_Algoritmo de \_ autenticación ad hoc de DOT11 \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_auth_algorithm)
-   [**\_Algoritmo de \_ cifrado ad hoc de DOT11 \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_cipher_algorithm)
-   [**\_Motivo del \_ error de conexión ad hoc de \_ DOT11 \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_connect_fail_reason)
-   [**\_Estado de \_ conexión de red ad hoc de \_ DOT11 \_**](/windows/win32/api/adhoc/ne-adhoc-dot11_adhoc_network_connection_status)

> [!Note]  
> El modo ad hoc podría no estar disponible en versiones futuras de Windows. A partir de Windows 8.1 y Windows Server 2012 R2, utilice en su lugar [Wi-Fi Direct](about-the-wi-fi-direct-api.md) .
