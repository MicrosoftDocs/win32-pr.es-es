# <a name="wpd-constants"></a>Constantes de WPD

Dispositivos portátiles de Windows define los siguientes tipos constantes.



| Tipo constante                                              | Descripción                                                                                                                                                                                                                               |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Constantes generales](general-constants.md)                 | Uso general de constantes para dispositivos y aplicaciones.                                                                                                                                                                                       |
| [Constantes de error](error-constants.md)                     | Valores de error devueltos por los métodos.                                                                                                                                                                                                         |
| [Constantes de evento](event-constants.md)                     | Tipos de eventos definidos por dispositivos portátiles de Windows. Entre los tipos de eventos se incluyen el objeto agregado, el objeto quitado, el objeto actualizado, el objeto movido, el objeto copiado, la transferencia de objetos solicitada, el restablecimiento del dispositivo, la eliminación del dispositivo y las funcionalidades del dispositivo actualizadas. |
| [GUID de tipo de objeto](object-type-guids.md)                 | Tipos de objeto definidos por dispositivos portátiles de Windows. Los tipos de objeto incluyen objetos de audio, correo electrónico e imagen.                                                                                                                                  |
| [GUID de formato de objeto](object-format-guids.md)             | Tipos de formato de archivo definidos por dispositivos portátiles de Windows. Los tipos de formato incluyen WMA, WMV y JPEG.                                                                                                                                           |
| [GUID de tipo de recurso](resource-type-guids.md)             | Tipos de recursos definidos por dispositivos portátiles de Windows. Los tipos de recursos incluyen datos de imágenes de audio y miniatura.                                                                                                                                |
| [Propiedades y atributos](properties-and-attributes.md) | Propiedades y atributos definidos por dispositivos portátiles de Windows.                                                                                                                                                                            |
