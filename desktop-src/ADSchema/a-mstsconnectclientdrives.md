# <a name="ms-ts-connect-client-drives-attribute"></a>Atributo ms-TS-Conectar-Client-Drives

La sesión de Terminal Services Conectar cliente especifica si se debe volver a conectar a las unidades de cliente asignadas durante el inicio de sesión.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | ms-TS-Conectar-Client-Drives          |
| Ldap-Display-Name | msTSConnectClientDrives              |
| Size              | \-                                   |
| Actualizar privilegios  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1986              |
| System-Id-Guid    | 23572aaf-29dd-44ea-b0fa-7e8438b9a4a3 |
| Syntax            | [**Booleana**](s-boolean.md)         |
