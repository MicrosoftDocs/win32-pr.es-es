# <a name="developing-assistive-technology-for-windows"></a>Desarrollo de tecnología de asistencia para Windows

Los productos de tecnología de asistencia de terceros, como lectores de pantalla, lupa y hardware de especialización, son herramientas esenciales para muchos usuarios de Windows (y otros productos de Microsoft).

En esta introducción se describe cómo crear herramientas de tecnología de asistencia compatibles con los productos de Microsoft para personas con discapacidades o limitaciones de visión, aprendizaje, movilidad y lenguaje o comunicación.
