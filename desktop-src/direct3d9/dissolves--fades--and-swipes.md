# <a name="dissolves-fades-and-swipes-direct3d-9"></a><span data-ttu-id="7c9e9-103">Se resuelve, atenúa y desliza el dedo (Direct3D 9)</span><span class="sxs-lookup"><span data-stu-id="7c9e9-103">Dissolves, Fades, and Swipes (Direct3D 9)</span></span>

<span data-ttu-id="7c9e9-104">Cada vez más, las aplicaciones emplean efectos especiales que se suelen usar en películas y vídeo, como las desmedidas, los deslizamientos y los fundidos.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-104">Increasingly, applications employ special effects that are commonly used in movies and video, such as dissolves, swipes, and fades.</span></span>

<span data-ttu-id="7c9e9-105">En una resolución, una imagen se reemplaza gradualmente por otra en una secuencia de fotogramas suavizada.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-105">In a dissolve, one image is gradually replaced by another in a smooth sequence of frames.</span></span> <span data-ttu-id="7c9e9-106">Aunque Direct3D proporciona métodos para usar varias combinaciones de texturas para lograr el mismo efecto, las aplicaciones que usan el búfer de estarcido para las discapacidades pueden utilizar las capacidades de combinación de texturas para otros efectos mientras se resuelven.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-106">Although Direct3D provides methods of using multiple texture blending to achieve the same effect, applications that use the stencil buffer for dissolves can use texture-blending capabilities for other effects while they do a dissolve.</span></span>

<span data-ttu-id="7c9e9-107">Cuando la aplicación realiza una resolución, debe representar dos imágenes diferentes.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-107">When your application performs a dissolve, it must render two different images.</span></span> <span data-ttu-id="7c9e9-108">Usa el búfer de estarcido para controlar qué píxeles de cada imagen se dibujan en la superficie de destino de representación.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-108">It uses the stencil buffer to control which pixels from each image are drawn to the rendering target surface.</span></span> <span data-ttu-id="7c9e9-109">Puede definir una serie de máscaras de estarcido y copiarlas en el búfer de estarcido en fotogramas sucesivos.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-109">You can define a series of stencil masks and copy them into the stencil buffer on successive frames.</span></span> <span data-ttu-id="7c9e9-110">Como alternativa, puede definir una máscara de galería de símbolos base para el primer fotograma y modificarla incrementalmente.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-110">Alternately, you can define a base stencil mask for the first frame and alter it incrementally.</span></span>

<span data-ttu-id="7c9e9-111">Al principio de la resolución, la aplicación establece la función de estarcido y la máscara de la galería de símbolos de modo que la mayoría de los píxeles de la imagen inicial pasen la prueba de estarcido.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-111">At the beginning of the dissolve, your application sets the stencil function and stencil mask so that most of the pixels from the starting image pass the stencil test.</span></span> <span data-ttu-id="7c9e9-112">La mayoría de los píxeles de la imagen final deben tener un error en la prueba de estarcido.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-112">Most of the pixels from the ending image should fail the stencil test.</span></span> <span data-ttu-id="7c9e9-113">En fotogramas sucesivos, la máscara de la galería de símbolos se actualiza para que menos y menos de los píxeles de la imagen inicial pasen la prueba.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-113">On successive frames, the stencil mask is updated so that fewer and fewer of the pixels in the starting image pass the test.</span></span> <span data-ttu-id="7c9e9-114">A medida que los fotogramas progresan, se produce un error en la prueba con menos y menos píxeles de la imagen final.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-114">As the frames progress, fewer and fewer of the pixels in the ending image fail the test.</span></span> <span data-ttu-id="7c9e9-115">De esta manera, la aplicación puede realizar una resolución con cualquier patrón de resolución arbitrario.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-115">In this manner, your application can perform a dissolve using any arbitrary dissolve pattern.</span></span>

<span data-ttu-id="7c9e9-116">La atenuación o el fundido de salida es un caso especial de la solución.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-116">Fading in or fading out is a special case of dissolving.</span></span> <span data-ttu-id="7c9e9-117">Cuando se difumina, el búfer de estarcido se usa para disolver una imagen en blanco o negro en una representación de una escena 3D.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-117">When fading in, the stencil buffer is used to dissolve from a black or white image to a rendering of a 3D scene.</span></span> <span data-ttu-id="7c9e9-118">La atenuación es lo contrario, la aplicación comienza con una representación de una escena 3D y se resuelve en blanco o negro.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-118">Fading out is the opposite, your application starts with a rendering of a 3D scene and dissolves to black or white.</span></span> <span data-ttu-id="7c9e9-119">La atenuación se puede realizar con cualquier patrón arbitrario que desee emplear.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-119">The fade can be done using any arbitrary pattern you want to employ.</span></span>

<span data-ttu-id="7c9e9-120">Las aplicaciones Direct3D usan una técnica similar para los deslizamientos.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-120">Direct3D applications use a similar technique for swipes.</span></span> <span data-ttu-id="7c9e9-121">Por ejemplo, cuando una aplicación realiza un deslizamiento de izquierda a derecha, la imagen final parece deslizarse gradualmente en la parte superior de la imagen inicial de izquierda a derecha.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-121">For example, when an application performs a left-to-right swipe, the ending image appears to slide gradually on top of the starting image from left to right.</span></span> <span data-ttu-id="7c9e9-122">Al igual que en una solución, debe definir una serie de máscaras de galería de símbolos que se cargan en el búfer de estarcido en fotogramas sucesivos o modificar sucesivamente la máscara de la galería de símbolos de inicio.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-122">As in a dissolve, you must define a series of stencil masks that are loaded into the stencil buffer on successive frames, or successively modify the starting stencil mask.</span></span> <span data-ttu-id="7c9e9-123">Las máscaras de estarcido se usan para deshabilitar la escritura de píxeles de la imagen inicial y para permitir la escritura de píxeles desde la imagen final.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-123">The stencil masks are used to disable the writing of pixels from the starting image and to enable the writing of pixels from the ending image.</span></span>

<span data-ttu-id="7c9e9-124">Un deslizamiento es algo más complejo que resolver en que la aplicación debe leer píxeles de la imagen final en el orden inverso del deslizamiento.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-124">A swipe is somewhat more complex than a dissolve in that your application must read pixels from the ending image in the reverse order of the swipe.</span></span> <span data-ttu-id="7c9e9-125">Es decir, si el deslizamiento se desplaza de izquierda a derecha, la aplicación debe leer los píxeles de la imagen final de derecha a izquierda.</span><span class="sxs-lookup"><span data-stu-id="7c9e9-125">That is, if the swipe is moving from left to right, your application must read pixels from the ending image from right to left.</span></span>
