# <a name="itssbtaskplugin-methods"></a>Métodos ITsSbTaskPlugin

La interfaz [**ITsSbTaskPlugin**](/windows/desktop/api/sbtsv/nn-sbtsv-itssbtaskplugin) expone los métodos siguientes.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**Método InitializeTaskPlugin**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskplugin-initializetaskplugin)
</dt> <dd>

Inicializa una tarea que está en la cola de un complemento de agente de Conexión a Escritorio remoto.

</dd> <dt>

[**Método SetTaskQueue**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbtaskplugin-settaskqueue)
</dt> <dd>

Actualiza una tarea en la cola de un complemento de agente de Conexión a Escritorio remoto.

</dd> </dl>
