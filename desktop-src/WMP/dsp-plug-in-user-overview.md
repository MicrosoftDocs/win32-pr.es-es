# <a name="dsp-plug-in-user-overview"></a>Información general del usuario del complemento DSP

Para el usuario, los complementos DSP son características complementarias que cambian de algún modo la forma en que el usuario experimenta el audio o el vídeo que Windows Media Player está reproduciendo. El resultado de un complemento DSP podría ser algo obvio, como los colores de la imagen de vídeo que se invierten, por lo que se parece a un negativo fotográfico o algo muy sutil, como un efecto de reverberación ligera en el audio para agregar una sensación de spaciousness al sonido.

Los usuarios pueden instalar y desinstalar los complementos DSP **mediante la pestaña complementos** del cuadro de diálogo Opciones de Windows Media Player. Los usuarios pueden habilitar y deshabilitar los complementos instalados mediante las casillas proporcionadas **en la pestaña complementos** . los complementos DSP pueden implementar una página de propiedades, a la que los usuarios también pueden obtener acceso **desde la pestaña complementos** . Una página de propiedades proporciona al usuario una manera de cambiar la configuración que afecta a la forma en que funciona un complemento DSP. Por ejemplo, una página de propiedades podría incluir un control deslizante que permite al usuario cambiar el tiempo de caída de un efecto de reverberación.
