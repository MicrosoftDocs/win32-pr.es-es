# <a name="id3dxanimationcontrollergetmaxnumanimationoutputs-method"></a>Método ID3DXAnimationController::GetMaxNumAnimationOutputs

Obtiene el número máximo de salidas de animación que el controlador de animación puede admitir.

## <a name="syntax"></a>Sintaxis


```C++
UINT GetMaxNumAnimationOutputs();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

Número máximo de salidas de animación que el controlador puede administrar.
