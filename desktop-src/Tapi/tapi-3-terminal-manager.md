# <a name="tapi-3-terminal-manager"></a>TAPI 3 Terminal Manager

TapI 3 Terminal Manager se crea mediante Tapi3.dll a petición de un MSP y el MSP lo usa para tareas como la creación y eliminación de terminales dinámicos.

Los detalles de implementación precisos de los terminales dinámicos de un MSP están fuera del ámbito de esta documentación. Sin embargo, muchos autores pueden optar por usar algunas de las funcionalidades del Terminal de streaming multimedia (MST). Este terminal se basa en DirectShow y los filtros específicos de un protocolo determinado se pueden escribir y usar junto con el MST.
