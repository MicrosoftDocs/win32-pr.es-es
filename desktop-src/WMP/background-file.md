# <a name="background-file"></a>Archivo en segundo plano

Cada máscara necesita un archivo de fondo, que contiene la imagen que el usuario verá cuando se muestre la máscara por primera vez. Este archivo se define como el archivo de fondo en el archivo de definición de máscara.

La siguiente imagen es un archivo de fondo típico.

![archivo en segundo plano](images/cesdkbak.png)

Esta es la imagen de fondo que los usuarios verán al principio. Sin embargo, también verán las dos imágenes Thumb en las ubicaciones adecuadas en trackbars for Seek (anterior) y VOLUME (a continuación). Además, los botones deshabilitados se mostrarán en su lugar con sus imágenes deshabilitadas.
