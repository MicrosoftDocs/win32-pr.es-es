# <a name="about-parental-controls"></a>Acerca del control parental

La característica de controles parentales incluye lo siguiente:

-   Funcionalidad central de la interfaz de usuario integrada con el panel de control.
-   Implementaciones de restricciones seleccionadas.
-   Infraestructura en el sistema operativo que proporciona la configuración de las Directivas generales y las restricciones implementadas y que expone características de extensibilidad de destino.
