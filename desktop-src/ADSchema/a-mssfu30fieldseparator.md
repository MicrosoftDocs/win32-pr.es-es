Contiene el separador de campos de cada mapa de NIS.

| CN                | msSFU-30-separador de campos                    |
| Nombre para mostrar de LDAP | msSFU30FieldSeparator                       |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
