# <a name="bits-reference"></a>Referencia de BITS

La sección de referencia de Servicio de transferencia inteligente en segundo plano (BITS) contiene información detallada acerca de los siguientes elementos de la API:

-   [Enumeraciones](bits-enumerations.md)
-   [Interfaces](bits-interfaces.md)
-   [Valores devueltos](bits-return-values.md)
-   [Estructuras y uniones](bits-c---structures-and-unions.md)

Estos elementos de API se usan para crear aplicaciones que transfieren archivos entre un cliente y un servidor. Para ver el código de ejemplo que usa las interfaces de BITS, vea [ejemplos y herramientas de bits](bits-samples-and-tools.md).

Para obtener información general sobre BITS, vea la [Página de inicio de bits](background-intelligent-transfer-service-portal.md).
