| CN                | GPC-File-Sys-Path                           |
| Ldap-Display-Name | gPCFileSysPath                              |
| Size              | \-                                          |
| Actualizar privilegios  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.894                      |
