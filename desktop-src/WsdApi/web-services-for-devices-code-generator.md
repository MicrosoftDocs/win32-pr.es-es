# <a name="web-services-on-devices-code-generator"></a>Generador de código de servicios web en dispositivos

El generador de código de servicios web en dispositivos, WsdCodeGen, es una utilidad de línea de comandos que genera código de programa a partir de una descripción del servicio. La descripción del servicio se almacena en archivos WSDL o XSD. WsdCodeGen crea archivos de C++ y IDL (lenguaje de definición de interfaz). Los desarrolladores pueden usar WsdCodeGen para crear aplicaciones de WSDAPI sin preocuparse por el modo en que se calculan las referencias de los datos y se representan en la conexión.

WsdCodeGen se puede usar para generar código para un servicio, un cliente o ambos. La utilidad genera código auxiliar para servicios, código proxy para clientes y código de interfaz y tipo para ambos.

Esta herramienta está disponible en el kit de desarrollo de software (SDK) de Microsoft Windows y en el kit de controladores de Windows (WDK). Las herramientas del SDK se encuentran en el siguiente directorio: <Windows SDK Install Folder> \\ bin.

El Windows SDK incluye código de ejemplo generado por WsdCodeGen. Para obtener más información, vea [ejemplos de WSDAPI](wsdapi-samples.md).
