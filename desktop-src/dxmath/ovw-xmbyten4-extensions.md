# <a name="xmbyten4-extensions"></a>Extensiones de XMBYTEN4

[**XMBYTEN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmbyten4) describe un vector 3D para almacenar valores normalizados y firmados como enteros de 8 bits con signo (1 byte).

> [!Note]  
> Las características que se enumeran aquí solo están disponibles cuando se desarrollan con C++.
