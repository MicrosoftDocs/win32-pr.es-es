# <a name="xmfloat3se-operators"></a>Operadores XMFLOAT3SE

La [**estructura XMFLOAT3SE**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmfloat3se) expone los operadores siguientes.

> [!Note]  
> Los operadores enumerados aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                               | Descripción                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**operador uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmfloat3se-operatoruint32_t)<br/> | Devuelve una instancia de `uint32_t` que contiene los componentes de la instancia en un formato `XMFLOAT3SE` empaquetado.<br/> |
| [**operator =**](xmfloat3se-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMFLOAT3SE` .<br/>                                       |
