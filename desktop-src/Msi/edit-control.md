# <a name="edit-control-windows-installer"></a>Control de edición (Windows Installer)

El control de edición es un campo de edición que está asociado a una propiedad de cadena o valor entero. Escriba el nombre de la propiedad en la columna de propiedades de la [tabla de control](control-table.md).

## <a name="control-attributes"></a>Atributos de control

Puede usar los siguientes atributos con este control. Para cambiar el valor de un atributo mediante un evento, suscríbase el control a ControlEvent, en la [tabla EventMapping](eventmapping-table.md) y enumere el identificador del atributo en la columna de atributos. Escriba el identificador de ControlEvent, en la columna evento.

## <a name="remarks"></a>Observaciones

Este control se puede crear a partir de la clase EDIT mediante la función [**CreateWindowEx**](/windows/win32/api/winuser/nf-winuser-createwindowexa) . Tiene los estilos **WS \_ Border**, **WS \_ Child**, **WS \_ TABSTOP** y **WS \_ Group** .

La longitud del texto que se puede especificar se puede limitar colocando un número comprendido entre 0 y 2147483646 entre llaves al principio del campo de texto de la [tabla de control](control-table.md). Por ejemplo, si el campo de texto comienza con {80} , la longitud de la cadena se limita en 80 caracteres. Si no se proporciona ningún límite de este tipo en la tabla, o si se especifica 0, la longitud se establece en el máximo posible (2147483646 caracteres). Un valor negativo o no numérico generará un error.

Por compatibilidad con los lectores de pantalla, al crear un cuadro de diálogo con un control de edición como primer control activo, debe hacer que el campo de texto pertenezca al campo de edición del primer control activo en la [tabla de cuadro de diálogo](dialog-table.md). Dado que el texto estático no puede recibir el foco, cuando se crea el cuadro de diálogo, el campo de edición tendrá el foco inicialmente tal y como se pretende, pero esto garantizará que los lectores de pantalla muestren la información correcta.

La propiedad asociada al control de edición solo se establece cuando el control pierde el foco. Por lo tanto, debe hacer clic en el tabulador del control o seleccionar otro control para la propiedad que se va a actualizar.
