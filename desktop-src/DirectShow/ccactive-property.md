# <a name="ccactive-property"></a>CCActive( Propiedad)

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

La `CCActive` propiedad establece o recupera el estado actual de los subtítulos.

``` syntax
[ bCCActive = ] MSWebDVD.CCActive
```

## <a name="return-value"></a>Valor devuelto

Devuelve un valor booleano que indica si está activado el subtítulo.

## <a name="remarks"></a>Comentarios

Esta propiedad es de lectura y escritura con un valor predeterminado de false.
