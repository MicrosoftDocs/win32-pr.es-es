# <a name="capture-filters"></a>Filtros de captura

El SDK de Monitor de red incluye filtros de captura, con los que puede restringir el número de fotogramas que recibe una aplicación de proveedor de paquetes de red (NPP) durante su funcionamiento. Al especificar qué fotogramas del tráfico de red desea que examine un monitor personalizado o una aplicación NPP, puede mejorar la eficacia del procesamiento.

En esta sección se incluyen los siguientes temas:

-   [Acerca de los filtros de captura](about-capture-filters.md)
-   [**Estructura CAPTUREFILTER**](the-capturefilter-structure.md)
