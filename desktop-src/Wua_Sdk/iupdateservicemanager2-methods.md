# <a name="iupdateservicemanager2-methods"></a>Métodos IUpdateServiceManager2

La [**interfaz IUpdateServiceManager2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager2) define los métodos siguientes.



| Método                                                                                      | Descripción                                                                                                                                                                                |
|---------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AddService2**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-addservice2)                           | Registra un servicio con WUA sin necesidad de un archivo de archivador de autorización (.cab) y devuelve un puntero a una [**interfaz IUpdateServiceRegistration.**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicecollection) |
| [**QueryServiceRegistration**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicemanager2-queryserviceregistration) | Devuelve un puntero a una [**interfaz IUpdateServiceRegistration.**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateserviceregistration)                                                                                        |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz.

-   [**IUpdateServiceManager**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicemanager)
