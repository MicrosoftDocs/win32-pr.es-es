# <a name="http-version-20-features"></a>Características de la versión 2,0 de HTTP

La característica siguiente es compatible con la versión 2,0 de la API del servidor HTTP.

-   Objetos de configuración de grupo de URL y sesión de servidor
-   Propiedades de configuración expuestas en los objetos de configuración
-   Administración y configuración de la cola de solicitudes avanzada
-   SSL de modo kernel completo
-   Caché de respuesta de modo kernel
-   Autenticación del lado servidor
-   Control de versiones con ámbito de objeto
-   Registro del lado servidor
-   Cierre estable de la cola de solicitudes
-   Inicio de la demanda en la cola de solicitudes
-   Uso compartido de puertos
-   Seguimiento de eventos para el soporte técnico de Windows
-   Contadores de rendimiento
-   Compatibilidad con nombres de dominio internacionalizados (IDN)
-   Utilidad de configuración de net Shell
