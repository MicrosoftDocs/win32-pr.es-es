# <a name="print-number-up-attribute"></a>Print-Number-up (atributo)

El número de imágenes de página por hoja.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Imprimir-numeración                      |
| Nombre para mostrar de LDAP | printNumberUp                        |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.290               |
| System-ID-GUID    | 3bcbfcf4-4d3d-11d0-a1a6-00c04fd930c9 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
