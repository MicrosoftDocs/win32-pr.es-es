# <a name="configuring-snmp-security"></a>Configuración de la seguridad de SNMP

Las características de seguridad de SNMP permiten especificar las comunidades y hosts desde los que un equipo acepta solicitudes, así como el tipo de operaciones que se aceptan de los equipos que pertenecen a una comunidad. SNMP también permite especificar si se envía una captura de autenticación cuando una comunidad o un host no autorizado solicitan información.

Para obtener más información, consulte el Windows Resource Kit y la documentación en línea del servicio SNMP.

**Windows Server 2003:** De forma predeterminada, los nombres de comunidad no están configurados y el servicio SNMP está configurado para aceptar solicitudes SNMP solo desde el nombre de host "localhost".
