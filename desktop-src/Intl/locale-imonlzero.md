# <a name="locale_imonlzero"></a>configuración regional \_ IMONLZERO

Especificador para los ceros iniciales en los campos de mes solo para una fecha corta. Es preferible que la aplicación use la constante [ \_ SSHORTDATE de configuración regional](locale-sshortdate.md) en lugar de la configuración regional \_ IMONLZERO.



| Value | Significado                     |
|-------|-----------------------------|
| 0     | Sin ceros a la izquierda para los meses |
| 1     | Ceros a la izquierda para los meses    |
