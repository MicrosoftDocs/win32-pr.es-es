# <a name="navigation-for-type-2-online-stores"></a>Navegación para las tiendas en línea de tipo 2

Las páginas web hospedadas por Windows Media Player 10 o posterior, como las de los paneles de tareas servicio de tienda en línea, pueden usar el método [external. NavigateTaskPaneURL](external-navigatetaskpaneurl.md) para controlar la navegación dentro de Windows Media Player. La navegación puede producirse de las siguientes maneras:

-   Entre los paneles de tareas de servicio. Consulte [desplazarse entre los paneles de tareas de servicio](navigating-between-service-task-panes.md)
-   Desde características como la vista centro de información, las páginas web HTMLView y la página información del álbum hasta los paneles de tareas de servicio. Consulte [desplazarse de las características de Windows Media Player](navigating-from-windows-media-player-features.md).
-   Entre tiendas en línea. Consulte [navegación a otras tiendas en línea](navigating-to-other-online-stores.md).
