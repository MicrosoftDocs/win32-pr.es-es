La [*API de Resource Manager*](../secgloss/r-gly.md) proporciona la siguiente funcionalidad.



| Tema                                                                                               | Descripción                                                                                                                                                                                                                                                                 |
|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Funciones de consulta de bases de datos de tarjeta inteligente](smart-card-database-query-functions.md)<br/>           | Consultar la base de datos de tarjeta inteligente.<br/>                                                                                                                                                                                                                                   |
| [Funciones de administración de bases de datos de tarjeta inteligente](smart-card-database-management-functions.md)<br/> | Administrar la base de datos de tarjetas inteligentes.<br/>                                                                                                                                                                                                                                  |
| [Administrador de recursos funciones de contexto](resource-manager-context-functions.md)<br/>             | Administrar el contexto de las operaciones de base de datos del administrador de recursos.<br/>                                                                                                                                                                                                |
| [Administrador de recursos función de compatibilidad](resource-manager-support-function.md)<br/>               | Liberar memoria asignada.<br/>                                                                                                                                                                                                                                        |
| [Funciones de seguimiento de tarjetas inteligentes](smart-card-tracking-functions.md)<br/>                       | Realice un seguimiento de las tarjetas inteligentes en los lectores.<br/>                                                                                                                                                                                                                                |
| [Funciones de acceso a tarjetas inteligentes y lectores](smart-card-and-reader-access-functions.md)<br/>     | Conectarse y comunicarse con una tarjeta inteligente, incluida la transferencia de datos mediante [*T = 0*](../secgloss/t-gly.md), [*t = 1*](../secgloss/t-gly.md)y protocolos sin formato.<br/> |
