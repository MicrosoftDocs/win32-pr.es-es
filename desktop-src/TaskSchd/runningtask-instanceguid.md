# <a name="runningtaskinstanceguid-property"></a>Propiedad RunningTask. valor FileStream

En el caso de scripting, obtiene el identificador GUID para esta instancia de la tarea.

## <a name="syntax"></a>Sintaxis


```VB
RunningTask.InstanceGuid As string
```



## <a name="property-value"></a>Valor de propiedad

Identificador GUID para esta instancia de la tarea. El servicio Programador de tareas genera un identificador cada vez que se ejecuta la tarea.
