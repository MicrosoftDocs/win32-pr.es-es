# <a name="command-list"></a><span data-ttu-id="7c216-104">Lista de comandos</span><span class="sxs-lookup"><span data-stu-id="7c216-104">Command List</span></span>

<span data-ttu-id="7c216-107">Use una lista de comandos en los escenarios siguientes:</span><span class="sxs-lookup"><span data-stu-id="7c216-107">Use a command list in the following scenarios:</span></span>

-   <span data-ttu-id="7c216-108">Dentro de un solo fotograma, representa parte de la escena en un subproceso mientras se graba otra parte de la escena en un segundo subproceso.</span><span class="sxs-lookup"><span data-stu-id="7c216-108">Within a single frame, render part of the scene on one thread while recording another part of the scene on a second thread.</span></span> <span data-ttu-id="7c216-109">Al final del marco, vuelva a reproducir la lista de comandos grabada en el primer subproceso.</span><span class="sxs-lookup"><span data-stu-id="7c216-109">At the end of the frame, play back the recorded command list on the first thread.</span></span> <span data-ttu-id="7c216-110">Use este método para escalar tareas de representación complejas entre varios subprocesos o núcleos.</span><span class="sxs-lookup"><span data-stu-id="7c216-110">Use this approach to scale complex rendering tasks across multiple threads or cores.</span></span>
-   <span data-ttu-id="7c216-111">Grabe previamente una lista de comandos antes de que tenga que representarla (por ejemplo, mientras se carga un nivel) y reproducirla de forma eficaz más adelante en la escena.</span><span class="sxs-lookup"><span data-stu-id="7c216-111">Pre-record a command list before you need to render it (for example, while a level is loading) and efficiently play it back later in your scene.</span></span> <span data-ttu-id="7c216-112">Esta optimización funciona bien cuando es necesario representar algo a menudo.</span><span class="sxs-lookup"><span data-stu-id="7c216-112">This optimization works well when you need to render something often.</span></span>

<span data-ttu-id="7c216-113">Una lista de comandos es inmutable y está diseñada para grabarse y reproducirse durante una ejecución única de una aplicación.</span><span class="sxs-lookup"><span data-stu-id="7c216-113">A command list is immutable and is designed to be recorded and played back during a single execution of an application.</span></span> <span data-ttu-id="7c216-114">Una lista de comandos no se ha diseñado para que se registre previamente antes de la ejecución del juego y se cargue desde el medio, ya que no hay ninguna manera de conservar la lista.</span><span class="sxs-lookup"><span data-stu-id="7c216-114">A command list is not designed to be pre-recorded ahead of game execution and loaded from your media as there is no way to persist the list.</span></span>
