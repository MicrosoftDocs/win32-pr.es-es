# <a name="ms-ds-sitename-attribute"></a>atributo MS-DS-SiteName

Muestra el nombre del sitio que se corresponde con el controlador de dominio.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-DS-SiteName                              |
| Nombre para mostrar de LDAP | msDS-SiteName                               |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1961                     |
| System-ID-GUID    | 98a7f36d-3595-448a-9e6f-6b8965baed9c        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
