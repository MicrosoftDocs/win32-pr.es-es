# <a name="media-foundation-pipeline"></a>Media Foundation canalización

La *capa* de canalización Microsoft Media Foundation es la capa de la arquitectura que genera o procesa datos multimedia directamente.

La mayoría de las aplicaciones no llaman a métodos directamente en los componentes de canalización, aunque es posible hacerlo. Lea estos temas si va a escribir un componente de canalización personalizado.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                     | Descripción                                                                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Orígenes multimedia](media-sources.md)<br/>                             | Los orígenes multimedia generan datos multimedia, normalmente a partir de un archivo, un dispositivo de captura o una secuencia de red. <br/>                              |
| [Media Foundation transformaciones](media-foundation-transforms.md)<br/> | Media Foundation transformaciones (MTA) procesan datos multimedia. Por ejemplo, los códecs Media Foundation se implementan como MTA. <br/>   |
| [Receptores de medios](media-sinks.md)<br/>                                 | Los receptores multimedia consumen datos multimedia. Por ejemplo, un receptor multimedia podría escribir los datos en un archivo o enviar los datos a través de una red.<br/> |
