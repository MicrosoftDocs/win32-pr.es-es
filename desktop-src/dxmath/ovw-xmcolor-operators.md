# <a name="xmcolor-operators"></a>Operadores XMCOLOR

La estructura [**XMCOLOR**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmcolor) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                            | Descripción                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operador UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmcolor-operatoruint32_t)<br/> | Devuelve una instancia de `uint32_t` que contiene los componentes de la `XMCOLOR` instancia en un formato empaquetado. <br/> |
| [**operador =**](xmcolor-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMCOLOR` .<br/>                                        |
