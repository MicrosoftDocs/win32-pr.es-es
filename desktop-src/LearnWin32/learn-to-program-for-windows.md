# <a name="get-started-with-win32-and-c"></a>Introducción a Win32 y C++

El objetivo de esta serie de introducción es enseñarle a escribir un programa de escritorio en C++ mediante Win32 y las API de COM.

En el primer módulo, aprenderá paso a paso cómo crear y mostrar una ventana. Los módulos posteriores introducirán el modelo de objetos componentes (COM), los gráficos y el texto, y los datos proporcionados por el usuario.

En esta serie, se supone que tiene un buen conocimiento práctico de la programación de C++. No se presupone ninguna experiencia anterior con la programación de Windows. Si no está familiarizado con C++, puede encontrar material de aprendizaje en el [Centro para desarrolladores de Visual C++](https://msdn.microsoft.com/vstudio//default.aspx). (Es posible que este recurso no esté disponible en algunos idiomas y países).
