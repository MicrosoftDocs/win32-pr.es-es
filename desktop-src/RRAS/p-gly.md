<dl> <dt>

<span id="_rras_point_to_multipoint_interface_gly"></span><span id="_RRAS_POINT_TO_MULTIPOINT_INTERFACE_GLY"></span>**interfaz de punto a punto múltiple**
</dt> <dd>

La tecnología administrador de grupos de multidifusión usa este término para significar una interfaz que proporciona comunicación entre un único origen y un solo destino, como un vínculo PPP.

</dd> <dt>

<span id="_rras_point_to_point_interface_gly"></span><span id="_RRAS_POINT_TO_POINT_INTERFACE_GLY"></span>**interfaz de punto a punto**
</dt> <dd>

La tecnología administrador de grupos de multidifusión usa este término para significar una interfaz que proporciona comunicación entre un único host y varios destinos. Las interfaces de punto a punto se pueden pensar en como una colección de vínculos de punto a punto con una única terminación, como un vínculo de ATM.

</dd> </dl>
