# <a name="stream-attribute"></a>Stream (atributo)

> [!Note]  
> \[En desuso. Esta API se puede quitar de las versiones futuras de Windows.\]

 

El `stream` atributo especifica la secuencia que se va a usar en un archivo de origen de varias secuencias.

## <a name="possible-values"></a>Valores posibles

Debe ser un valor entero.
