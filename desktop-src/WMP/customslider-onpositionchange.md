# <a name="customslideronpositionchange"></a>CUSTOMSLIDER.onPositionChange

El controlador de eventos **onPositionChange** controla un evento que se produce cuando cambia la posición del control deslizante a medida que el usuario hace clic o arrastra.

``` syntax
onPositionChange
```

## <a name="remarks"></a>Observaciones

Si cambia la posición del control deslizante personalizado como resultado del atributo de **valor** que se está modificando en el script, este evento no se desencadena. Para dar cabida a esta posibilidad, implemente en su lugar el controlador de eventos **\_ onchange Value** .
