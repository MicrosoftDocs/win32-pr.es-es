# <a name="administrative-functions"></a>Funciones administrativas

Las siguientes funciones administrativas permiten a un proveedor de red realizar acciones específicas de la red para mostrar y manipular directorios específicos de la red, es decir, directorios asignados a protocolos no Windows red.



| Función                                         | Descripción                                                                                                                                                   |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**NPGetDirectoryType**](/windows/desktop/api/Npapi/nf-npapi-npgetdirectorytype) | Llamado por el Administrador de archivos para determinar el tipo de un directorio de red.                                                                                          |
| [**NPDirectoryNotify**](/windows/desktop/api/Npapi/nf-npapi-npdirectorynotify)   | Llamado por el Administrador de archivos para notificar al proveedor de red las operaciones de directorio. Esta función se puede usar para realizar un comportamiento especial para determinados directorios. |
