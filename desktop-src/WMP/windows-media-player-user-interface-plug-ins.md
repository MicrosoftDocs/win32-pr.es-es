# <a name="windows-media-player-user-interface-plug-ins"></a>Complementos de la interfaz de usuario de Windows Media Player

Microsoft Windows Media Player proporciona una arquitectura que permite al usuario instalar y activar programas de complementos que agregan la funcionalidad de la interfaz de usuario (IU) al modo completo del reproductor. Un complemento de IU típico podría permitir que un usuario compre el CD que contiene la pista que se está reproduciendo o tener acceso a información adicional sobre el artista. Esta sección del kit de desarrollo de software (SDK) de Windows Media Player proporciona la información de programación que necesita para crear su propio complemento de interfaz de usuario.

La documentación del complemento de la interfaz de usuario se divide en tres secciones:



| Sección                                                                                            | Descripción                                                                                                                                   |
|----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| [Acerca de los complementos de la interfaz de usuario](about-user-interface-plug-ins.md)                                 | Proporciona información general sobre la arquitectura utilizada para los complementos de la interfaz de usuario. Lea esta sección para obtener información sobre los conceptos generales implicados en esta tecnología. |
| [Guía de programación de complementos de la interfaz de usuario](user-interface-plug-ins-programming-guide.md)         | Explica lo que debe hacer para crear un complemento de la interfaz de usuario. Esta sección contiene código de ejemplo y procedimientos paso a paso.                          |
| [Referencia de programación de complementos de interfaz de usuario](user-interface-plug-ins-programming-reference.md) | Proporciona una referencia detallada para la interfaz COM y los métodos admitidos por el SDK de Windows Media Player para los complementos de la interfaz de usuario.                    |



 

> [!Note]  
> Windows Media Player 10 Mobile proporciona el mismo modelo de complemento que el Media Player de Windows de escritorio. Este modelo de complemento permite usar complementos de fondo para supervisar eventos, mostrar el estado de las propiedades, etc. En la guía de programación de esta sección se describe cómo crear un complemento de interfaz de usuario de fondo para Windows Media Player 10 Mobile mediante el Asistente para complementos de Windows Media Player de escritorio.
