# <a name="ms-sql-serviceaccount-attribute"></a>Atributo MS-SQL-ServiceAccount

Cadena definida por el usuario. El valor predeterminado se establece en ServiceAccount.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-ServiceAccount                       |
| Nombre para mostrar de LDAP | mS-SQL-ServiceAccount                       |
| Tamaño              | \-                                          |
| Actualizar privilegio  | Administrador de dominio                        |
| Frecuencia de actualización  | En el programa de instalación del sistema.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1369                     |
| System-ID-GUID    | 64933a3e-ccee-11d2-9993-0000f87a57d4        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
