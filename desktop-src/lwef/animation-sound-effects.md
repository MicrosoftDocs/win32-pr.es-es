# <a name="animation-sound-effects"></a>Efectos sonoros de animación

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

El agente de Microsoft también permite incluir efectos sonoros como parte de la animación de un carácter. Con el editor de caracteres del agente de Microsoft, puede especificar el nombre de archivo del sonido estándar de Windows (. WAV) para desencadenar en un fotograma determinado. Tenga en cuenta que el agente de Microsoft no combina efectos sonoros ni resultados de voz, por lo que la salida de voz no comienza hasta que se completa un efecto sonoro. Por lo tanto, evite cualquier efecto de sonido largo o en bucle como parte de la animación de un carácter.
