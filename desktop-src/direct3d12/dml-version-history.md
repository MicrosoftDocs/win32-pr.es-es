## <a name="directml-version-versus-feature-level"></a><span data-ttu-id="c2e8a-156">Versión de DirectML frente a nivel de característica</span><span class="sxs-lookup"><span data-stu-id="c2e8a-156">DirectML version versus feature level</span></span>

<span data-ttu-id="c2e8a-157">La versión de DirectML (por ejemplo, 1.0.0 o 1.4.0) describe una versión determinada de DirectML, incluido el archivo `DirectML.h` de encabezado y el `.lib` archivo asociados.</span><span class="sxs-lookup"><span data-stu-id="c2e8a-157">The DirectML version (for example, 1.0.0, or 1.4.0) describes a particular release of DirectML, including its associated `DirectML.h` header file and `.lib` file.</span></span>
