# <a name="partial-switch"></a>modificador/Partial

El modificador **/partial** especifica que los tipos sin resolver se resuelven en archivos de metadatos que se enumeran mediante el modificador [**\_ dir de/metadata**](-mdmerge-metadata-dir.md) .

``` syntax
mdmerge /partial 
```

## <a name="switch-options"></a>Opciones de conmutador

Este modificador no tiene parámetros.

## <a name="remarks"></a>Observaciones

## <a name="examples"></a>Ejemplos

**mdmerge /partial**
