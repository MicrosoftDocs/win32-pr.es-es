# <a name="swbemrefreshableitemrefresher-property"></a>Propiedad SWbemRefreshableItem. actualizador

La propiedad **SWbemRefreshableItem. refreshable** representa el objeto [**SWbemRefresher**](swbemrefresher.md) primario que contiene el objeto [**SWbemRefreshableItem**](swbemrefreshableitem.md) .

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemRefreshableItem.Refresher As Object
```
