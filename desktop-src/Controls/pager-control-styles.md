# <a name="pager-control-styles"></a>Estilos de control de buscapersonas

En esta sección se enumeran los estilos de ventana que se usan al crear controles de buscapersonas.



| Constante                                                                                                                                                         | Descripción                                                                                                                                                                                                                 |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="PGS_AUTOSCROLL"></span><span id="pgs_autoscroll"></span><dl> <dt>**\_desplazamiento automático de págs**</dt> </dl> | El control de paginación se desplazará cuando el usuario mantenga el mouse sobre uno de los botones de desplazamiento.<br/>                                                                                                                     |
| <span id="PGS_DRAGNDROP"></span><span id="pgs_dragndrop"></span><dl> <dt>**PÁGS \_ DRAGNDROP**</dt> </dl>    | La ventana contenida puede ser un destino de arrastrar y colocar. El control de paginación se desplazará automáticamente si se arrastra un elemento desde fuera del buscapersonas sobre uno de los botones de desplazamiento.<br/>                                     |
| <span id="PGS_HORZ"></span><span id="pgs_horz"></span><dl> <dt>**PÁGS \_ horizontal**</dt> </dl>                   | Crea un control de paginación que se puede desplazar horizontalmente. Este estilo y el estilo **págs \_ Vert** se excluyen mutuamente y no se pueden combinar.<br/>                                                                 |
| <span id="PGS_VERT"></span><span id="pgs_vert"></span><dl> <dt>**PÁGS \_ Vert**</dt> </dl>                   | Crea un control de paginación que se puede desplazar verticalmente. Esta es la dirección predeterminada si no se especifica ningún estilo de dirección. Este estilo y el estilo **págs \_ horizontal** se excluyen mutuamente y no se pueden combinar.<br/> |
