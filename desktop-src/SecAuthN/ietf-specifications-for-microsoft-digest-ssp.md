# <a name="ietf-specifications-for-microsoft-digest-ssp"></a>Especificaciones de IETF para Microsoft Digest SSP

Autenticación HTTP: la autenticación básica y de acceso implícita (RFC 2617) está disponible en [https://www.ietf.org/rfc/rfc2617.txt](https://www.ietf.org/rfc/rfc2617.txt) .

Usar Digest como un borrador de Internet de mecanismo SASL está disponible en [https://www.ietf.org/rfc/rfc2831.txt](https://www.ietf.org/rfc/rfc2831.txt) .

Para obtener información adicional sobre SASL, vea también la especificación SASL (RFC 2222) disponible en [https://www.ietf.org/rfc/rfc2222.txt](https://www.ietf.org/rfc/rfc2222.txt) .
