## <a name="remarks"></a>Comentarios

Este atributo puede ser cero o una combinación de uno o varios de los valores siguientes.



| Valor            | Descripción                                                                                                                                                                                                                                                                                                                                                               |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)   | Cree un índice para el atributo .                                                                                                                                                                                                                                                                                                                                        |
| 2 (0x00000002)   | Cree un índice para el atributo en cada contenedor.                                                                                                                                                                                                                                                                                                                      |
| 4 (0x00000004)   | Agregue este atributo al conjunto ambiguo de resolución de nombres (ANR). Esto se usa para ayudar a encontrar un objeto cuando solo se proporciona información parcial. Por ejemplo, si el filtro LDAP es (ANR=JEFF), la búsqueda buscará cada objeto en el que el nombre, el apellido, la dirección de correo electrónico u otro atributo ANR sea igual que JEFF. El bit 0 debe establecerse para que este índice suba efecto. |
| 8 (0x00000008)   | Conserve este atributo en el objeto de desecho para los objetos eliminados.                                                                                                                                                                                                                                                                                                      |
| 16 (0x00000010)  | Copie el valor de este atributo cuando se copie el objeto.                                                                                                                                                                                                                                                                                                              |
| 32 (0x00000020)  | Cree un índice de tupla para el atributo. Esto mejorará las búsquedas en las que el carácter comodín aparece al frente de la cadena de búsqueda. Por ejemplo, (sn= \* mith).                                                                                                                                                                                                                |
| 64(0x00000040)   | Se admite a partir de ADAM. Crea un índice para ayudar en gran medida al rendimiento de VLV en atributos arbitrarios.                                                                                                                                                                                                                                                                  |
| 128 (0x00000080) | Marque el atributo como confidencial. Se omite para los atributos de esquema base (systemFlags=0x10).                                                                                                                                                                                                                                                                                    |
| 64 (0x00000040)  | Se admite a partir Windows Server 2008. Cree un índice para mejorar el rendimiento de la búsqueda de VLV en este atributo.                                                                                                                                                                                                                                                        |
