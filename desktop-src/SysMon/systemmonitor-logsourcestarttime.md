# <a name="systemmonitorlogsourcestarttime-property"></a>Propiedad SystemMonitor. LogSourceStartTime

Recupera la marca de tiempo del valor del contador más antiguo de un contador de la colección de contadores que se registró en los archivos de registro.

## <a name="syntax"></a>Sintaxis


```VB
Property LogSourceStartTime As Date
```



## <a name="property-value"></a>Valor de propiedad

Marca de tiempo del valor del contador más antiguo de un contador de la colección de contadores que se registró en los archivos de registro.
