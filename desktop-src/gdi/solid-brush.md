# <a name="solid-brush"></a>Pincel sólido

Un *pincel sólido* es un pincel lógico que contiene 64 píxeles del mismo color. Una aplicación puede crear un pincel lógico sólido mediante una llamada a la función [**CreateSolidBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createsolidbrush) , que especifica el color del pincel necesario. Después de crear el pincel sólido, la aplicación puede seleccionarlo en su contexto de dispositivo y usarlo para pintar formas rellenas.
