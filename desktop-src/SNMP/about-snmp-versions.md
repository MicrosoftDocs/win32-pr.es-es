# <a name="about-snmp-versions"></a>Acerca de las versiones de SNMP

El marco de administración de redes estándar de Internet original, descrito en RFC 1155, 1157 y 1213, se denomina marco snmp versión 1 (SNMPv1). Las partes pertinentes del marco propuesto para la versión 2C del Protocolo simple de administración de redes (SNMPv2C) se describen en RFC de 1901 a 1908.

La API de WinSNMP admite la funcionalidad del protocolo SNMP descrita en las RFC de Internet pertinentes. WinSNMP no aplica ninguna restricción al uso de aplicaciones SNMPv1 o SNMPv2C por parte de WinSNMP.

Una entidad de administración puede admitir una versión diferente de SNMP que la que admite la aplicación WinSNMP. La implementación de Microsoft WinSNMP realiza las traducciones adecuadas de SNMPv1 a SNMPv2C de acuerdo con la RFC correspondiente.
