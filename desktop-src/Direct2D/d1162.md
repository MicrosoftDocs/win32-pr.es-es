# <a name="d1162-fillmesh-requires-aliased-rendering"></a>D1162: FillMesh requiere la representación con alias

FillMesh requiere que el modo antialias se establezca en el \_ modo ANTIALIAS D2D1 con \_ \_ alias.






 

## <a name="possible-causes"></a>Causas posibles

Cuando una aplicación llama a ID2D1RenderTarget:: FillMesh, el modo antialias de rendertarget implicado debe ser D2D1 con \_ alias del modo antialias \_ \_ . El comportamiento predeterminado de un destino de representación es usar el suavizado de contorno, por lo que es necesario llamar a ID2D1RenderTarget:: SetAntialiasMode antes de rellenar las mallas.

## <a name="possible-fixes"></a>Posibles correcciones

Use ID2D1RenderTarget:: SetAntialiasMode para asegurarse de que rendertarget usa la representación con alias al rellenar mallas. Si las operaciones subsiguientes requieren suavizado de contorno, vuelva a establecer el modo antialias del destino en el \_ modo D2D1 antialias \_ \_ por \_ primitiva. Asegúrese de que las llamadas a ID2D1RenderTarget:: SetAntialiasMode y ID2D1RenderTarget:: FillMesh se emitan en el mismo destino.
