# <a name="importinitialreplica-method-of-the-msvm_replicationservice-class"></a>Método ImportInitialReplica de la \_ clase ReplicationService de MSVM

Importa la replicación inicial de una máquina virtual.

## <a name="syntax"></a>Sintaxis


```mof
uint32 ImportInitialReplica(
  [in]  CIM_ComputerSystem REF ComputerSystem,
  [in]  string                 InitialReplicationImportLocation,
  [out] CIM_ConcreteJob    REF Job
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*ComputerSystem* \[ de\]
</dt> <dd>

Referencia a una instancia de un [**\_ ComputerSystem de CIM**](/windows/desktop/CIMWin32Prov/cim-computersystem) que representa la máquina virtual para la que se debe importar la replicación inicial.

</dd> <dt>

*InitialReplicationImportLocation* \[ de\]
</dt> <dd>

Ruta de acceso completa del directorio desde el que se va a importar la replicación inicial. No puede ser **null**.

</dd> <dt>

*Trabajo* \[ de enuncia\]
</dt> <dd>

Si la operación se realiza de forma asincrónica, este método devolverá 4096 y este parámetro contendrá una referencia a un objeto derivado de [**\_ ConcreteJob CIM**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
