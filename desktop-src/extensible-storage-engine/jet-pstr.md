# <a name="jet_pstr"></a>JET_PSTR


_**Se aplica a:** Windows | Windows Servidor_

## <a name="jet_pstr"></a>JET_PSTR

El JET_PSTR tipo de datos contiene una cadena ASCII terminada en NULL (char \* ).

**Windows Vista: JET_PSTR** se introdujo en Windows Vista.

```cpp
    typedef __nullterminated char *  JET_PSTR;
```

### <a name="data-types"></a>Tipo de datos

JET_PSTR

Cadena ASCII terminada en NULL \* (char).
