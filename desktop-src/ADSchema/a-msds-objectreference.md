# <a name="ms-ds-object-reference-attribute"></a>atributo MS-DS-Object-Reference

Vínculo al objeto que utiliza los datos almacenados en el objeto que contiene este atributo.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | Referencia de objetos MS-DS-Object                  |
| Nombre para mostrar de LDAP | msDS-ObjectReference                    |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1840                 |
| System-ID-GUID    | 638ec2e8-22e7-409c-85d2-11b21bee72de    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
