# <a name="wavestream"></a>Wavestream

Las aplicaciones usan las funciones de devolución de llamada de Wavestream durante la captura de streaming para procesar búferes de audio. La ventana captura llama a una función de devolución de llamada Wavestream justo antes de escribir cada búfer de audio en el disco.
