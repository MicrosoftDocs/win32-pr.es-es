# <a name="cim-classes-hyper-v-management"></a>Clases CIM (administración de Hyper-V)

Las clases CIM son clases base de las que derivan todos los objetos WMI. Cada proveedor puede usar potencialmente una versión ligeramente diferente de las clases CIM; por lo tanto, la mayoría de los proveedores de WMI de V2 contienen su propia documentación de la clase CIM.

Las siguientes clases CIM se encuentran en el \\ espacio de nombres de virtualización V2 de raíz \\ .
