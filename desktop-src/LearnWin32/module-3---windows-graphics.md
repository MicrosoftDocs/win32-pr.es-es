# <a name="module-3-windows-graphics"></a>Módulo 3. Gráficos de Windows

En el [módulo 1](your-first-windows-program.md) de esta serie se ha mostrado cómo crear una ventana en blanco. El [módulo 2](module-2--using-com-in-your-windows-program.md) llevó un ligero desvío a través del modelo de objetos componentes (com), que es la base de muchas de las API modernas de Windows. Ahora es el momento de agregar gráficos a la ventana en blanco que creamos en el módulo 1.

Este módulo se inicia con una introducción de alto nivel de la arquitectura de gráficos de Windows. A continuación, veremos Direct2D, una potente API de gráficos que se presentó en Windows 7.
