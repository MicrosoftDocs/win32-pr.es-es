# <a name="comments-summary-property"></a>Propiedad Resumen de comentarios

La **propiedad Resumen de comentarios** transmite el propósito general del paquete de instalación, la transformación o el paquete de revisión.

Un autor de un paquete de instalación, transformación o paquete de revisión debe establecer el valor de la propiedad **Resumen** de comentarios en uno de los valores siguientes:

<dl> "Esta base de datos del instalador contiene la lógica y los datos necesarios para instalar <*producto>".*  
"Esta transformación contiene la lógica y los datos necesarios para instalar <*producto>".*  
"Esta revisión contiene la lógica y los datos necesarios para instalar <*producto>".*  
</dl>

donde <*producto*> es el nombre del producto.
