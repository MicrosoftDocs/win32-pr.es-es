# <a name="d1163-print-bitmap-too-big"></a>D1163: imprimir mapa de bits demasiado grande

El mapa de bits es demasiado grande para imprimirlo. considere la posibilidad de dividirlo en mosaicos más pequeños.






 

## <a name="possible-causes"></a>Causas posibles

La impresión en Direct2D impone un límite en los tamaños de los mapas de bits imprimibles. Este límite es 65535 unidades dependientes del dispositivo. Si la aplicación incluye un mapa de bits que supera este tamaño en el contenido imprimible, Direct2D notificará un error y la capa de depuración generará este mensaje.

## <a name="possible-fixes"></a>Posibles correcciones

Asegúrese de que los mapas de bits utilizados en las operaciones de impresión no superen el límite de tamaño de la impresión de Direct2D. Considere la posibilidad de crear particiones del mapa de bits en varios mosaicos.
