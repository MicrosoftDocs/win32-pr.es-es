# <a name="adapters"></a>Adaptadores

Un adaptador representa el hardware que se conecta a un segmento de red determinado. Una tarjeta LAN enlazada es un ejemplo de un adaptador. De forma similar, un equipo con dos módems, cada uno capaz de conectarse a una red remota, tiene dos adaptadores, uno para representar cada módem.
