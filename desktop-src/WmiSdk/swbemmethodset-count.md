# <a name="swbemmethodsetcount-property"></a>SWbemMethodSet. Count (propiedad)

La propiedad **Count** del objeto [**SWbemMethodSet**](swbemmethodset.md) define el número de elementos de la colección. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemMethodSet.Count As Integer
```
