# <a name="instanceparameterswaypointlatency-property"></a>Propiedad InstanceParameters. WaypointLatency

Obtiene o establece el número de registros que esent aplazará los vaciados de la base de datos. Se puede usar para aumentar la capacidad de recuperación de la base de datos si los errores provocan la pérdida de los archivos de registro. Compatible con Windows 7 y up. Se omite en Windows XP, Windows Server 2003, Windows Vista y Windows Server 2008.

**Espacio de nombres:**  [Microsoft. ISAM. esent. Interop](./microsoft.isam.esent.interop-namespace.md)
