# <a name="windows-store-apps"></a>Aplicaciones de la Tienda Windows

## <a name="resources"></a>Recursos

Los recursos que se enumeran a continuación están diseñados para ayudarle a migrar aplicaciones integradas en Windows 8 a Windows 8.1.

-   [Actualización de la aplicación de Windows 8 a Windows 8.1](/previous-versions/windows/dn376326(v=win.10)) &mdash; Un documento técnico que contiene instrucciones y sugerencias sobre cómo migrar la aplicación de Windows 8 a Windows 8.1, de principio a fin. Incluye procedimientos recomendados, desusos e instrucciones para volver a enviar la aplicación a Windows Store.
-   [Nuevas API y características para desarrolladores](https://msdn.microsoft.com/library/windows/apps/dn751496.aspx) &mdash; Para conocer Windows 8.1 nuevas API y características.
-   [Redestinar la Windows 8 Store a Windows 8.1](/previous-versions/windows/apps/dn263114(v=win.10)) &mdash; Instrucciones paso a paso sobre cómo migrar la aplicación de Windows 8 a Windows 8.1.
-   [Ejemplos actualizados](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples) &mdash; Ejemplos de Windows 8.1 que muestran cómo usar nuevas API y características, y cómo mitigar los desusos.
-   [Programación de DirectX](https://msdn.microsoft.com/library/windows/apps/bg182880.aspx) &mdash; Nuevas características y mejoras para juegos y aplicaciones gráficas.
-   Para obtener información sobre los cambios de API en Windows 8.1 relacionados con la programación con JavaScript y Internet Explorer, consulte Cambios de API para [Windows 8.1 (aplicaciones](/previous-versions/windows/apps/dn263112(v=win.10))de Windows Store con JavaScript y HTML).
-   Para obtener información sobre los cambios de API en Windows 8.1 relacionados con la programación con C#/VB/C++ y XAML, vea Cambios de API para Windows 8.1 (aplicaciones de la Tienda Windows con [C#/VB/C++ y XAML).](/previous-versions/windows/apps/dn263110(v=win.10))
