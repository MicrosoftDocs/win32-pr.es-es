# <a name="wmasfsecurityobjectssize"></a>WM/ASFSecurityObjectsSize

El **atributo WM/ASFSecurityObjectsSize** contiene el tamaño, en bytes, de los objetos relacionados con DRM en el encabezado del archivo ASF.

## <a name="global-constant"></a>Constante global

g \_ wszWMASFSecurityObjectsSize

## <a name="data-type"></a>Tipo de datos

**TIPO WMT \_ \_ QWORD**

## <a name="remarks"></a>Comentarios

Este atributo es de solo lectura y se aplica a todo el archivo (secuencia 0).

Solo puede recuperar este atributo mediante los métodos de la [**interfaz IWMHeaderInfo3**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmheaderinfo3) del objeto del editor de metadatos.
