# <a name="about-system-restore"></a>Acerca de Restaurar sistema

Restaurar sistema supervisa los cambios del sistema y guarda el estado del sistema como un punto *de restauración*. Si se produce un problema del sistema como resultado de un cambio del sistema, el usuario puede devolver el sistema a un estado anterior mediante los datos de un punto de restauración.

Las aplicaciones y el sistema pueden crear puntos de restauración cuando se producen cambios en el sistema. Restaurar sistema también se puede configurar para crear puntos de restauración a intervalos regulares y los usuarios pueden crear puntos de restauración en cualquier momento. Restaurar sistema en Windows 7 comprueba cada día y crea un punto de restauración programado si no se ha creado ningún otro punto de restauración en los últimos 7 días. Restaurar sistema en Windows Vista crea un punto de restauración si no se crearon otros puntos de restauración en las últimas 24 horas. Restaurar sistema en Windows XP crea un punto de control cada 24 horas de tiempo absoluto.

Restaurar sistema restaura datos o documentos de usuario, por lo que no hará que los usuarios pierdan sus archivos, correo electrónico, historial de exploración o favoritos. Restaurar sistema también está disponible para los usuarios en el entorno de recuperación de Windows o en modo seguro, lo que les facilita la restauración de sus equipos a un estado antes de que se produjesen problemas.
