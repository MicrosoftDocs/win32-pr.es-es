# <a name="link-id-attribute"></a>Atributo Link-ID

Entero que indica que el atributo es un atributo vinculado. Un entero par es un vínculo hacia delante y un entero impar es un vínculo hacia atrás.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Id. de vínculo                              |
| Ldap-Display-Name | linkID                               |
| Size              | \-                                   |
| Actualizar privilegios  | Administrador de esquemas                 |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.50                |
| System-Id-Guid    | bf96799b-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Enumeración**](s-enumeration.md) |
