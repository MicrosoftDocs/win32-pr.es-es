# <a name="stringnt-sec-desc-syntax"></a>Sintaxis de String (NT-sec-DESC)

Cadena de octetos que contiene un descriptor de seguridad de Windows NT o Windows 2000.



| Entrada | Value |
|--------------|------------------------------------------------------------------------------------------------|
| Nombre         | String(NT-Sec-Desc)                                                                            |
| IDENTIFICADOR de sintaxis    | 2.5.5.15                                                                                       |
| IDENTIFICADOR DE OM        | 66                                                                                             |
| Tipo MAPI    | BINARY                                                                                         |
| Tipo ADS     | descriptor de seguridad de ADSTYPE \_ NT \_ \_                                                              |
| Tipo Variant | distribución de VT \_                                                                                   |
| Tipo de SDS     | Objeto COM que se puede convertir en un [**IADsSecurityDescriptor**](/windows/desktop/api/iads/nn-iads-iadssecuritydescriptor). |
