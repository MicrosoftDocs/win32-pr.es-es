# <a name="monitor-configuration-monitor-configuration"></a>Configuración de monitor (supervisar configuración)

Puede utilizar las funciones de configuración del monitor para obtener información de un monitor y cambiar la configuración de un monitor. Puede usar estas funciones para:

-   Cambiar la configuración de la geometría de un monitor, como el ancho y el alto de la pantalla.
-   Cambiar la configuración de la imagen, como el brillo y el contraste.
-   Restablecer la configuración de un monitor a sus valores predeterminados de fábrica.
-   Desmagnetización del monitor.

Internamente, las funciones de configuración del monitor usan la interfaz de comandos del canal de datos de pantalla (DDC/CI) para enviar comandos al monitor.

Las funciones de configuración del monitor se documentan en las siguientes secciones:

-   [Acerca de la configuración de monitor](about-monitor-configuration.md)
-   [Uso de la configuración de monitor](using-monitor-configuration.md)
-   [Referencia de configuración de monitor](monitor-configuration-reference.md)
