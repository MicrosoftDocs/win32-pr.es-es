# <a name="remoteadmints-property"></a>Propiedad RemoteAdminTS

El instalador establece la propiedad **RemoteAdminTS** cuando el sistema está configurado para la administración remota mediante Windows Terminal Services. Tenga en cuenta que, en este caso, el instalador no establece la propiedad [**TerminalServer**](terminalserver.md) .

## <a name="default-value"></a>Valor predeterminado

Ninguno. La propiedad no está definida cuando no se ejecuta en un servidor de administración remota con Terminal Services.
