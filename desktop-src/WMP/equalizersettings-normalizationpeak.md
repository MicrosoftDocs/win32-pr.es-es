# <a name="equalizersettingsnormalizationpeak"></a>EQUALIZERSETTINGS.normalizationPeak

El atributo **normalizationPeak** recupera el valor de normalización máximo.

``` syntax
        elementID.normalizationPeak
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **número** de solo lectura (**float**).

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------|
| Versión<br/> | Windows Media Player 9 series o posterior<br/> |
