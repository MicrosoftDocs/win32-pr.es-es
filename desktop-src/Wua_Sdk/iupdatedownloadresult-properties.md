# <a name="iupdatedownloadresult-properties"></a>Propiedades IUpdateDownloadResult

La [**interfaz IUpdateDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadresult) define las siguientes propiedades.



| Propiedad                                               | Descripción                                                                                                                          |
|--------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------|
| [**Hresult**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_hresult)       | Obtiene el valor **HRESULT de excepción,** si existe, que se genera durante una operación en una actualización.                                       |
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadresult-get_resultcode) | Obtiene un [**valor de enumeración OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) que especifica el resultado de una operación en una actualización. |
