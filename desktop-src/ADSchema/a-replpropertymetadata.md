# <a name="repl-property-meta-data-attribute"></a>Atributo REPL-Property-meta-data

Realiza un seguimiento de la información de estado de replicación interna para objetos de DS. La información aquí se puede extraer de forma pública a través de la API pública DsReplicaGetInfo (). Presente en todos los objetos de DS.



| Entrada | Value |
|-------------------|------------------------------------------------------------------------------|
| CN                | REPL-Property-meta-data                                                      |
| Nombre para mostrar de LDAP | replPropertyMetaData                                                         |
| Tamaño              | La longitud es proporcional al número de atributos replicables en el objeto. |
| Actualizar privilegio  | El sistema establece este valor.                                             |
| Frecuencia de actualización  | Cambios en respuesta a los cambios de replicación en el objeto.                |
| Attribute-Id      | 1.2.840.113556.1.4.3                                                         |
| System-ID-GUID    | 281416c0-1968-11d0-a28f-00aa003049e2                                         |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md)                        |
