# <a name="viewminwidth"></a>VER. minWidth

El atributo **minWidth** especifica o recupera el ancho mínimo, en píxeles, de la **vista** al cambiar el tamaño.

``` syntax
        elementID.minWidth
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **número** de lectura y escritura (**Long**) con un valor de cero o superior. Tiene un valor predeterminado de cero, lo que significa que no hay ninguna restricción en el ancho mínimo de la **vista**.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
