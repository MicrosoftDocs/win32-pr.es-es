# <a name="processor-classes"></a>Clases de procesador

Las clases de procesador modela un procesador para una máquina virtual.

Las siguientes son clases WMI de virtualización relacionadas con el procesador.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                      | Descripción                                                                               |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**Procesador \_ Msvm**](msvm-processor.md)<br/>                       | Representa el procesador virtual de una máquina virtual.<br/>                         |
| [**Msvm \_ ProcessorPool**](msvm-processorpool.md)<br/>               | Agrega los recursos de procesador que se pueden asignar a una máquina virtual.<br/> |
| [**Msvm \_ ProcessorSettingData**](msvm-processorsettingdata.md)<br/> | Representa la configuración del procesador virtual para una máquina virtual.<br/>               |
