# <a name="hardware-cryptography"></a>Criptografía de hardware

Certificate Services admite criptografía relacionada con hardware, como la emisión de certificados almacenados en [*tarjetas inteligentes.*](../secgloss/s-gly.md) La entidad [*de certificación*](../secgloss/c-gly.md) (CA) empresarial admite la emisión de certificados de tarjeta inteligente para el inicio de sesión de dominio en el sistema operativo. Además, tanto la CA empresarial como la ca independiente admiten la emisión de otros tipos de certificados de tarjeta inteligente.

Para más información, consulte la documentación del producto. Para obtener información para desarrolladores sobre las tarjetas inteligentes, vea [Autenticación con tarjeta inteligente.](../secauthn/smart-card-authentication.md) Para obtener información sobre el control que permite a un administrador solicitar certificados de tarjeta inteligente en nombre de los usuarios, vea [Control de inscripción de tarjetas inteligentes.](/previous-versions/windows/desktop/secsmart/smart-card-enrollment-control-usage-scenario)
