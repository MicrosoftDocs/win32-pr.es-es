# <a name="multimedia-streaming-sample-code"></a>Código de ejemplo de transmisión por secuencias multimedia

> [!Note]  
> Estas API están en desuso. Las aplicaciones deben usar el filtro de [**enganche de ejemplo**](sample-grabber-filter.md) o implementar un filtro personalizado para obtener datos de un gráfico de filtros de DirectShow.

 

En este artículo se proporciona código de ejemplo que implementa las interfaces de streaming multimedia. El código de ejemplo de streaming de vídeo muestra cómo leer un archivo y representarlo en una superficie principal de Microsoft® DirectDraw®. Por motivos de brevedad, este código no contiene ninguna comprobación de errores.

En el segundo ejemplo de código se muestra cómo usar las interfaces de streaming de audio para transmitir datos de audio.

Este artículo contiene las secciones siguientes.

-   [Código de ejemplo de streaming de vídeo](video-streaming-sample-code.md)
-   [Código de ejemplo de streaming de audio](audio-streaming-sample-code.md)
