# <a name="organization-of-the-documentation"></a>Organización de la documentación

El SDK Reproductor de Windows Media documenta una amplia variedad de tecnologías relacionadas con Reproductor de Windows Media. Para facilitar la búsqueda de la información que necesita, la documentación se divide en secciones que le ayudarán a encontrar rápidamente temas sobre una característica o tecnología determinadas.

Si procede, las secciones principales incluyen tres subsecciones: Acerca de, Guía de programación y Referencia. En la sección Acerca de se describe la característica o tecnología en particular para ayudarle a entender cómo puede usarla. La sección Guía de programación agrega tutoriales que le guiarán más a fondo en la tecnología con instrucciones paso a paso. En la sección Referencia se detallan las interfaces, objetos, métodos, propiedades, eventos, elementos y atributos que están disponibles para su uso.
