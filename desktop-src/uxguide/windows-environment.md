# <a name="windows-environment"></a>Entorno de Windows

> [!NOTE]
> Esta guía de diseño se ha creado para Windows 7 y no se ha actualizado para las versiones más recientes de Windows. Gran parte de la guía se sigue aplicando en principio, pero la presentación y los ejemplos no reflejan nuestra [Guía de diseño actual](/windows/uwp/design/).

El entorno de Windows es el área de trabajo en pantalla que proporciona Windows, análogo a un escritorio físico, y los puntos de extensión principales del sistema operativo. Obtenga información sobre cómo aprovechar el escritorio, la barra de tareas, el área de notificación, los paneles de control, la ayuda y el control de cuentas de usuario de la aplicación.

## <a name="in-this-section"></a>En esta sección



| Tema                                                   | Descripción                                                                                                                                                                                                               |
|---------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Dispositivo de escritorio](winenv-desktop.md)<br/>                | El escritorio es el área de trabajo del usuario para sus programas. No es una forma de promover el reconocimiento de su programa o su marca. ¡ No abuse! <br/>                                                                     |
| [Barra de tareas](winenv-taskbar.md)<br/>                | La barra de tareas es el punto de acceso para los programas que se muestran en el escritorio. Con las nuevas características de la barra de tareas de Windows 7, los usuarios pueden proporcionar comandos, tener acceso a los recursos y ver el estado del programa directamente desde la barra de tareas. <br/> |
| [Área de notificación](winenv-notification.md)<br/> | El área de notificación proporciona notificaciones y estado. Los programas bien diseñados usan el área de notificación de manera adecuada, sin ser molestas ni distraer. <br/>                                               |
| [Paneles de control](winenv-ctrl-panels.md)<br/>     | Use los elementos del panel de control para ayudar a los usuarios a configurar características de nivel de sistema y realizar tareas relacionadas. En su lugar, los programas que tienen una interfaz de usuario deben configurarse directamente desde su interfaz de usuario. <br/>                     |
| [Ayuda](winenv-help.md)<br/>                      | Use la ayuda como mecanismo secundario para ayudar a los usuarios a completar y comprender mejor las tareas que el mecanismo principal es la propia interfaz de usuario. Aplique estas directrices para que el contenido sea realmente útil y fácil de encontrar. <br/>    |
| [Control de cuentas de usuario](winenv-uac.md)<br/>       | Una experiencia de control de cuentas de usuario bien diseñada ayuda a evitar cambios no deseados en todo el sistema de una forma predecible y requiere un esfuerzo mínimo. <br/>                                                          |
