# <a name="update-assessor-enumerations"></a>Actualizar enumeraciones de evaluador

El proveedor de accessor de actualización usa las enumeraciones siguientes.



| Función                                                 | Descripción                                                                    |
|----------------------------------------------------------|--------------------------------------------------------------------------------|
| [**UpdateAssessmentStatus**](updateassessmentstatus.md) | Describe cómo está actualizado el sistema operativo de un dispositivo.                                |
| [**UpdateImpactLevel**](updateimpactlevel.md)           | Indica un impacto alto, medio o bajo de un dispositivo que ejecuta un sistema operativo no actualizado. |
