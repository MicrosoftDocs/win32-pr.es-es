# <a name="shellfolderitem-object"></a>Objeto ShellFolderItem

Extiende el objeto [**carpeta**](folderitem.md) . Además de las propiedades y los métodos admitidos por **carpeta**, **ShellFolderItem** tiene dos métodos adicionales.

## <a name="members"></a>Miembros

El objeto **ShellFolderItem** tiene estos tipos de miembros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

El objeto **ShellFolderItem** tiene estos métodos.



| Método                                                       | Descripción                                                                                                                                                                                         |
|:-------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ExtendedProperty**](shellfolderitem-extendedproperty.md) | Obtiene el valor de una propiedad del conjunto de propiedades de un elemento. La propiedad se puede especificar por nombre o por el identificador de formato del conjunto de propiedades (FMTID) y el identificador de propiedad (PID).<br/> |
| [**InvokeVerbEx**](invokeverbex.md)                         | Ejecuta un verbo en un elemento de Shell.<br/>                                                                                                                                                         |
