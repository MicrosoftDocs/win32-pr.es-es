# <a name="ms-wmi-targetclass-attribute"></a>atributo MS-WMI-TargetClass

Nombre de clase del objeto de directiva que se va a crear.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-TargetClass                          |
| Nombre para mostrar de LDAP | msWMI-TargetClass                           |
| Tamaño              | Menos de veinte caracteres.                |
| Actualizar privilegio  | Administrador de directiva de grupo                  |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1645                     |
| System-ID-GUID    | 95b6d8d6-c9e8-4661-a2bc-6a5cabc04c62        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
