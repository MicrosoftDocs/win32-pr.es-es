# <a name="system-flags-attribute"></a>System-Flags atributo)

Valor entero que contiene las marcas que definen las propiedades adicionales de la clase. Vea la sección Comentarios.



| Entrada | Value |
|-------------------|------------------------------------------------|
| CN                | System-Flags                                   |
| Nombre para mostrar de LDAP | systemFlags                                    |
| Tamaño              | \-                                             |
| Actualizar privilegio  | El administrador del esquema establece este valor. |
| Frecuencia de actualización  | Siempre que cambie el estado del objeto.      |
| Attribute-Id      | 1.2.840.113556.1.4.375                         |
| System-ID-GUID    | e0fa1e62-9b45-11d0-afdd-00c04fd930c9           |
| Sintaxis            | [**Enumeración**](s-enumeration.md)           |

## <a name="remarks"></a>Observaciones

Este atributo puede ser cero o una combinación de uno o varios de los valores siguientes.



| Value                   | Descripción                                                                                                                                                                                                                                                                                     |
|-------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)          | Cuando se aplica a un atributo, el atributo no se replicará. Cuando se aplica a un objeto de [**referencia cruzada**](c-crossref.md) , el contexto de nomenclatura está en NTDS.<br/>                                                                                                                          |
| 2 (0x00000002)          | Cuando se aplica a un atributo, el atributo se replica en el catálogo global. Cuando se aplica a un objeto de [**referencia cruzada**](c-crossref.md) , el contexto de nomenclatura es un dominio.<br/>                                                                                                       |
| 4 (0x00000004)          | Cuando se aplica a un atributo, se construye el atributo.                                                                                                                                                                                                                                     |
| 16 (0x00000010)         | Cuando se establece, indica que el objeto es un objeto de categoría 1. Un objeto de categoría 1 es una clase o un atributo que se incluye en el esquema base incluido con el sistema.                                                                                                                                |
| 33554432 (0x02000000)   | El objeto no se mueve al contenedor de objetos eliminados cuando se elimina. Se eliminará inmediatamente.                                                                                                                                                                                    |
| 67108864 (0x04000000)   | No se puede desplace el objeto.                                                                                                                                                                                                                                                                     |
| 134217728 (0x08000000)  | No se puede cambiar el nombre del objeto.                                                                                                                                                                                                                                                                   |
| 268435456 (0x10000000)  | En el caso de los objetos de la partición de configuración, si se establece esta marca, el objeto puede moverse con restricciones; de lo contrario, no se puede desplace el objeto. De forma predeterminada, esta marca no se establece en los nuevos objetos creados en la partición de configuración. Esta marca solo se puede establecer durante la creación del objeto. |
| 536870912 (0x20000000)  | En el caso de los objetos de la partición de configuración, si se establece esta marca, el objeto se puede desplace; de lo contrario, no se puede desplace el objeto. De forma predeterminada, esta marca no se establece en los nuevos objetos creados en la partición de configuración. Esta marca solo se puede establecer durante la creación del objeto.                   |
| 1073741824 (0x40000000) | En el caso de los objetos de la partición de configuración, si se establece esta marca, se puede cambiar el nombre del objeto; de lo contrario, no se puede cambiar el nombre del objeto. De forma predeterminada, esta marca no se establece en los nuevos objetos creados en la partición de configuración. Esta marca solo se puede establecer durante la creación del objeto.               |
| 2147483648 (0x80000000) | No se puede eliminar el objeto.                                                                                                                                                                                                                                                                   |
