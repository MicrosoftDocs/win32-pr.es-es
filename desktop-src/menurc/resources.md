# <a name="menus-and-other-resources"></a>Menús y otros recursos

Un *recurso* son datos binarios que puede agregar al archivo ejecutable de una Windows basada en aplicaciones. Un recurso puede ser estándar o definido. Los datos de un recurso estándar *describen* un icono, cursor, menú, cuadro de diálogo, mapa de bits, metarchivo mejorado, fuente, tabla de aceleradores, entrada de tabla de mensajes, entrada de tabla de cadenas o información de versión. Un *recurso definido por la aplicación*, también denominado recurso *personalizado*, contiene los datos que requiere una aplicación específica.
