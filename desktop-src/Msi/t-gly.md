<dl> <dt>

<span id="_msi_transaction_processing_gly"></span><span id="_MSI_TRANSACTION_PROCESSING_GLY"></span>**procesamiento de transacciones**
</dt> <dd>

Una o varias operaciones procesadas juntas como un único conjunto indivisible denominado transacción. Todas las operaciones constituyentes deben realizarse correctamente para que la transacción se haga correctamente; de lo contrario, todas las operaciones se revierte al estado original.

</dd> <dt>

<span id="_msi_transform_gly"></span><span id="_MSI_TRANSFORM_GLY"></span>**Transformar**
</dt> <dd>

Plantilla de las diferencias entre dos bases [*de datos del instalador*](i-gly.md) que se pueden aplicar para producir cambios similares en otras bases de datos. Por ejemplo, se puede usar una transformación de localización para cambiar el idioma de una aplicación. Para obtener más información, [vea Merges and Transforms](merges-and-transforms.md).

</dd> <dt>

<span id="_msi_transform_error_condition_flags_gly"></span><span id="_MSI_TRANSFORM_ERROR_CONDITION_FLAGS_GLY"></span>**marcas de condición de error de transformación**
</dt> <dd>

Conjunto de propiedades utilizadas para marcar las condiciones de error de una [*transformación.*](/windows) Para obtener más información, [**vea MsiCreateTransformSummaryInfo**](/windows/desktop/api/Msiquery/nf-msiquery-msicreatetransformsummaryinfoa).

</dd> <dt>

<span id="_msi_transform_validation_flags_gly"></span><span id="_MSI_TRANSFORM_VALIDATION_FLAGS_GLY"></span>**transformar marcas de validación**
</dt> <dd>

Conjunto de propiedades utilizadas para comprobar que la [*transformación*](/windows) se puede aplicar a la base de datos. Para obtener una lista de estas propiedades, [**vea MsiCreateTransformSummaryInfo**](/windows/desktop/api/Msiquery/nf-msiquery-msicreatetransformsummaryinfoa).

</dd> </dl>
