# <a name="asf-file-features"></a>Características de archivos ASF

El propósito principal del SDK de Windows Media Format es proporcionar compatibilidad para encapsular datos multimedia digitales en archivos de formato de sistema avanzado (ASF) y entregar los medios a una aplicación cliente. Los escenarios de entrega pueden variar ampliamente de una aplicación a otra, pero todos usan la estructura de archivos ASF entre la creación y la entrega. Los archivos ASF se ajustan a una estructura bien definida pero muy flexible. Para obtener más información acerca de la estructura de archivos ASF, consulte [información general del formato ASF](overview-of-the-asf-format.md).

En la especificación ASF se proporciona información detallada acerca de los datos de un archivo ASF, que puede descargar desde el [sitio web de Microsoft](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).

El SDK de Windows Media Format proporciona compatibilidad para las características de la especificación ASF principalmente a través del perfil que se usa para crear un archivo. Para obtener más información sobre los perfiles, vea [perfiles](profiles.md).
