# <a name="codecapi_avencvideocontenttype-property"></a>\_Propiedad AVEncVideoContentType de CODECAPI

Indica el tipo de contenido de vídeo de un codificador de vídeo.

## <a name="data-type"></a>Tipo de datos

**ULONG**

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncVideoContentType**

## <a name="property-value"></a>Valor de propiedad

El valor de esta propiedad es un miembro de la enumeración [**eAVEncVideoContentType**](/windows/win32/api/codecapi/ne-codecapi-eavencvideocontenttype) .

## <a name="remarks"></a>Observaciones

Esta propiedad es una sugerencia para que el codificador optimice su configuración de codificación.
