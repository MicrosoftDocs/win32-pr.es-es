# <a name="property-pages-and-property-sheets"></a>Páginas de propiedades y hojas de propiedades

Las páginas de propiedades OLE permiten a un objeto mostrar sus propiedades en un cuadro de diálogo con pestañas conocido como hoja de propiedades. Un usuario final puede ver y cambiar las propiedades del objeto. Un objeto puede mostrar sus páginas de propiedades independientemente de su cliente, o el cliente puede administrar la presentación de las páginas de propiedades de varios objetos contenidos en una sola hoja de propiedades. Las páginas de propiedades también proporcionan un medio para notificar a un cliente de los cambios en las propiedades de un objeto.

Cualquier objeto que desee proporcionar una interfaz de usuario para sus propiedades variables puede utilizar esta tecnología.

Para obtener más información, vea los temas siguientes:

-   [Enlace de datos a través de IPropertyNotifySink](data-binding-through-ipropertynotifysink.md)
