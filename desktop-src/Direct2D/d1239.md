# <a name="d1239-source-transform-bitmap-invalid-call"></a>D1239: llamada no válida de mapa de bits de transformación de origen

El mapa de bits pasado a ID2D1SourceTransform::D RAW no admite este método.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación llamó a un método no admitido en un mapa de bits pasado a ID2D1SourceTransform::D RAW.

## <a name="possible-fixes"></a>Posibles correcciones

Para actualizar este tipo de mapa de bits, use ID2D1Bitmap1:: Map.
