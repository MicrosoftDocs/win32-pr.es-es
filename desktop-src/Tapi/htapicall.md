# <a name="htapicall"></a>HTAPICALL

El **tipo de datos HTAPICALL** representa el identificador opaco TAPI para una estructura de datos de llamada. TAPI debe resolver un valor de este tipo en una referencia a la instancia de estructura de datos adecuada. El proveedor de servicios no debe intentar hacer referencia a través de esto como si fuera un puntero, hacer suposiciones sobre sus valores o interpretar su representación de cualquier manera que no sea pasar su valor a TAPI en los momentos adecuados.
