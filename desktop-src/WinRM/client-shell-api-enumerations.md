# <a name="client-shell-api-enumerations"></a>Enumeraciones de API de Shell de cliente

En la tabla siguiente se proporciona información general sobre las enumeraciones de la interfaz de programación de aplicaciones (API) de Shell de cliente de Administración remota de Windows (WinRM).



| Enumeraciones                                                 | Descripción                                                                                      |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**WSManAuthenticationFlags**](/windows/desktop/api/Wsman/ne-wsman-wsmanauthenticationflags) | Define el esquema de autenticación que se va a usar para la autenticación de servidor y proxy.               |
| [**WSManCallbackFlags**](/windows/desktop/api/Wsman/ne-wsman-wsmancallbackflags)             | Define un conjunto de marcas utilizadas por todas las funciones de devolución de llamada.                                           |
| [**WSManDataType**](/windows/desktop/api/Wsman/ne-wsman-wsmandatatype)                       | Especifica el tipo de datos actual de la Unión en la estructura de [**\_ datos de WSMAN**](/windows/desktop/api/Wsman/ns-wsman-wsman_data) . |
| [**WSManProxyAccessType**](/windows/desktop/api/Wsman/ne-wsman-wsmanproxyaccesstype)         | Define el tipo de acceso de proxy.                                                                   |
| [**WSManSessionOption**](/windows/desktop/api/Wsman/ne-wsman-wsmansessionoption)             | Define un conjunto de opciones extendidas para la sesión.                                               |
