# <a name="wmaudiofileurl"></a>WM/AudioFileURL

El **atributo WM/AudioFileURL** contiene la dirección de una página web oficial con información sobre el archivo en el que aparece este atributo. Por ejemplo, una canción podría tener un vínculo a la página del álbum en el sitio web del intérprete.

## <a name="global-constant"></a>Constante global

g \_ wszWMAudioFileURL

**CADENA DE TIPO WMT \_ \_**
