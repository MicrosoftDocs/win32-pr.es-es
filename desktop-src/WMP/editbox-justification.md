# <a name="editboxjustification"></a>EDITBOX. justificación

El atributo **justificación** especifica o recupera la alineación del texto dentro del control EditBox.

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de lectura/escritura que contiene uno de los valores siguientes.



| Value  | Descripción                                                        |
|--------|--------------------------------------------------------------------|
| Left   | Predeterminada. Alinea el texto en el borde izquierdo del control de cuadro de edición. |
| Right  | Alinea el texto con el borde derecho del control de cuadro de edición.         |
| Center | Alinea el texto con el centro horizontal del control del cuadro de edición.  |



 

## <a name="remarks"></a>Observaciones

Hay un margen de dos píxeles.
