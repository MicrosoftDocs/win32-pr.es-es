# <a name="x-property"></a>Propiedad x

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

La `x` propiedad establece o recupera la posición x de la esquina superior izquierda del rectángulo.

``` syntax
[ iXPos = ] DVDRect.x
```

## <a name="return-value"></a>Valor devuelto

Devuelve la posición X de la esquina superior izquierda del rectángulo como un entero.

## <a name="remarks"></a>Observaciones

Esta propiedad es de lectura/escritura y no tiene ningún valor predeterminado.
