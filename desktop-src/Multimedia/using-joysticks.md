# <a name="using-joysticks"></a>Usar joysticks

Esta sección contiene ejemplos que muestran cómo realizar las tareas siguientes:

-   [Obtención de las capacidades del controlador](getting-the-driver-capabilities.md)
-   [Captura de entradas de joystick](capturing-joystick-input.md)
-   [Procesamiento de mensajes de joystick](processing-joystick-messages.md)

Los ejemplos proceden de una sencilla aplicación de joystick que recupera información de posición y de estado de botón de los servicios de joystick, reproduce recursos de audio de forma de onda y pinta los agujeros de viñetas en la pantalla cuando un usuario presiona los botones del joystick.
