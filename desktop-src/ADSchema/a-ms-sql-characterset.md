# <a name="ms-sql-characterset-attribute"></a>Atributo MS-SQL-CharacterSet

Juego de caracteres de la instancia actual de SQL Server.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-SQL-CharacterSet                  |
| Nombre para mostrar de LDAP | mS-SQL-CharacterSet                  |
| Tamaño              | \-                                   |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | Al iniciar el sistema.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1370              |
| System-ID-GUID    | 696177a6-ccee-11d2-9993-0000f87a57d4 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
