|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**AUDCLNT \_ SESSIONFLAGS \_ XXX**](audclnt-sessionflags-xxx-constants.md)<br/> | Marcas de la sesión de audio que se abre durante la inicialización de la secuencia.<br/> |
| [**AUDCLNT \_ STREAMFLAGS \_ XXX**](audclnt-streamflags-xxx-constants.md)              | Marcas de inicialización de una secuencia de audio.                                           |
| [**Estado del dispositivo \_ \_ XXX**](device-state-xxx-constants.md)                            | Marcas de estado para un dispositivo de punto de conexión de audio.                                           |
| [**compatibilidad de hardware de punto de conexión \_ \_ \_ XXX**](endpoint-hardware-support-xxx-constants.md) | Marcas de compatibilidad de hardware para un dispositivo de punto de conexión de audio.                                |
