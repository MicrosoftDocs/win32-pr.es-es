# <a name="mf_mt_forward_custom_sei-attribute"></a>\_ \_ \_ Atributo Sei personalizado MF MT Forward \_

Especifica que el descodificador debe reenviar los tipos de unidad de información de mejora complementaria (SEI) en los ejemplos de salida.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

Si el descodificador analiza un SEI, no se reenviará.
