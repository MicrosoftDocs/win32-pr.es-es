# <a name="eventctrlkey"></a>event.ctrlKey

El **atributo ctrlKey** recupera un valor que indica si la tecla CTRL estaba presionada cuando se produjo el evento.

``` syntax
event.ctrlKey
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un valor booleano de **solo lectura.**



| Valor | Descripción                                      |
|-------|--------------------------------------------------|
| true  | Indica que la tecla CTRL estaba en la posición de abajo. |
| false | Indica que la tecla CTRL estaba en la posición de arriba.   |



 

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
