# <a name="msvm_systemcomponentsettingdata-class"></a>Clase \_ SystemComponentSettingData de Msvm

Clase base genérica para establecer clases de datos que representan los componentes de un sistema virtual.

La siguiente sintaxis se simplifica desde el código de Managed Object Format (MOF) e incluye todas las propiedades heredadas.

## <a name="syntax"></a>Sintaxis

``` syntax
[Abstract, AMENDMENT]
class Msvm_SystemComponentSettingData : CIM_SettingData
{
};
```

## <a name="members"></a>Miembros

La **clase \_ SystemComponentSettingData de Msvm** no define ningún miembro.
