# <a name="display-name-printable-attribute"></a>Display-Name-atributo imprimible

Nombre para mostrar imprimible de un objeto. El nombre para mostrar que se imprime suele ser la combinación del nombre del usuario, el inicial y el apellido.



| Entrada | Value |
|-------------------|----------------------------------------|
| CN                | Display-Name-printable                 |
| Nombre para mostrar de LDAP | displayNamePrintable                   |
| Tamaño              | \-                                     |
| Actualizar privilegio  | Administrador de dominio o propietario de la cuenta. |
| Frecuencia de actualización  | \-                                     |
| Attribute-Id      | 1.2.840.113556.1.2.353                 |
| System-ID-GUID    | bf967954-0de6-11d0-a285-00aa003049e2   |
| Sintaxis            | [**String(IA5)**](s-string-ia5.md)    |
