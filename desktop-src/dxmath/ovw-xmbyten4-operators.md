# <a name="xmbyten4-operators"></a>Operadores XMBYTEN4

La estructura [**XMBYTEN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmbyten4) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                               | Descripción                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operador =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmbyten4-operator-assign)<br/> | Asigna los datos del componente de vector de una instancia de `XMBYTEN4` a la instancia actual de `XMBYTEN4` . <br/> |
