# <a name="text-services-framework-return-values"></a>Valores devueltos del marco de trabajo de servicios de texto

Los métodos de interfaz del marco de trabajo de servicios de texto generan valores devueltos en el intervalo de 0x *hhhh* 0200 a 0x *hhhh* 0600, donde *hhhh* es un valor hexadecimal que depende del código de gravedad y el código de instalación de un valor devuelto específico.

El grupo valores devueltos en las siguientes categorías.

-   [Valores devueltos del servicio de texto](text-service-return-values.md)
-   [Valores devueltos por el administrador](manager-return-values.md)
-   [Valores devueltos del almacén de texto](text-store-return-values.md)
