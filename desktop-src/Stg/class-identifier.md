# <a name="class-identifier"></a>Identificador de clase

El identificador de clase (CLSID) es el de una clase que puede mostrar y/o proporcionar acceso mediante programación a los valores de propiedad. Si no hay ninguna clase de este tipo, se recomienda establecer este valor igual que el FMTID (consulte a continuación). Como alternativa, puede establecer este valor en ceros; sin embargo, el uso de FMTID permite más flexibilidad en el futuro.
