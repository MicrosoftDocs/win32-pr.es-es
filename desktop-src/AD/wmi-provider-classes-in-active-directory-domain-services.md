# <a name="wmi-provider-classes-in-active-directory-domain-services"></a>Clases de proveedor WMI en Active Directory Domain Services

Active Directory Domain Services tener las siguientes clases para su uso con el scripting de WMI:

-   [**MSAD \_ DomainController**](msad-domaincontroller.md)
-   [**MSAD \_ NamingContext**](msad-namingcontext.md)
-   [**MSAD \_ ReplCursor**](msad-replcursor.md)
-   [**MSAD \_ ReplNeighbor**](msad-replneighbor.md)
-   [**MSAD \_ ReplPendingOp**](msad-replpendingop.md)

El propio proveedor se representa mediante la clase [**ReplicationProvider1**](replicationprovider1.md) , que se instancia con el nombre "ReplProv1".
