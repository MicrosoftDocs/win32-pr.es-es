# <a name="xps-document-api"></a>XPS Document API

Document API de XPS es una API Windows que permite a los programas crear, leer, editar, guardar e imprimir documentos XPS. Document API de XPS se admite en Windows 7 y versiones posteriores de Windows, y se puede usar en programas en modo de usuario y controladores de impresora XPSDrv.

La API de documentos XPS admite XPS y OpenXPS, y está diseñada para programadores de C++ que escriben aplicaciones Windows nativas. Para obtener más información sobre la especificación de Open XPS, vea [Open XML Paper Specification (OpenXPS).](https://www.ecma-international.org/publications/standards/Ecma-388.htm)

[En el tema Sobre la API](about-xps-document-api.md) de documentos XPS se describe XPS OM, ya que se implementa mediante la API de documentos XPS.

En esta sección también se incluyen vínculos a aplicaciones de ejemplo que usan diferentes características de document API xps.
