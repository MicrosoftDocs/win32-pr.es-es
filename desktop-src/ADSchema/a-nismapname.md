# <a name="nismapname-attribute"></a>Atributo nisMapName

Contiene el nombre del mapa al que pertenece el objeto.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | nisMapName                           |
| Ldap-Display-Name | nisMapName                           |
| Size              | \-                                   |
| Privilegio actualizar  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.26                     |
| System-Id-Guid    | 969d3c79-0e9a-4d95-b0ac-bdde7ff8f3a1 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
