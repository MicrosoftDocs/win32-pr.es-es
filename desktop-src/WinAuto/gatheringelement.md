# <a name="gatheringelement"></a>GatheringElement

## <a name="text"></a>Texto

Este elemento se ha recopilado

## <a name="type"></a>Tipo

Advertencia

## <a name="description"></a>Descripción

Este mensaje no es una advertencia o un error y no requiere ninguna acción. Indica que la información del elemento se ha recopilado y almacenado en el archivo de registro.
