# <a name="imstscax-interface"></a>Interfaz IMsTscAx

Permite conectar y desconectar el control de cliente, crear objetos de canal virtual y enviar datos a través de un canal virtual. La interfaz también contiene métodos para recuperar y establecer propiedades relacionadas con el control de cliente. Estos métodos también están disponibles a través de [**la interfaz IMsRdpClient**](imsrdpclient-interface.md).

## <a name="members"></a>Miembros

La **interfaz IMsTscAx** hereda de la [**interfaz IDispatch.**](/windows/win32/api/oaidl/nn-oaidl-idispatch) **IMsTscAx** también tiene estos tipos de miembros:

-   [Métodos](#methods)
-   [Propiedades](#properties)

### <a name="methods"></a>Métodos

La **interfaz IMsTscAx** tiene estos métodos.



| Método                                                          | Descripción                                                                                                                                                                                           |
|:----------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Conectar**](imstscax-connect.md)                             | Inicia una conexión mediante las propiedades establecidas actualmente en el control .<br/>                                                                                                                  |
| [**CreateVirtualChannels**](imstscax-createvirtualchannels.md) | Crea un objeto de canal virtual del lado cliente para cada nombre de canal virtual especificado.<br/>                                                                                                      |
| [**Desconectar**](imstscax-disconnect.md)                       | Desconecta la conexión activa.<br/>                                                                                                                                                         |
| [**SendOnVirtualChannel**](imstscax-sendonvirtualchannel.md)   | Envía datos al servidor host de sesión de Escritorio remoto a través de un canal virtual que se creó anteriormente mediante el método [**IMsTscAx::CreateVirtualChannels.**](imstscax-createvirtualchannels.md)<br/> |
