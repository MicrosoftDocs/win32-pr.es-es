# <a name="common-hresult-values"></a>Valores HRESULT comunes

Los valores HRESULT siguientes son los más comunes. Se incluyen más valores en el archivo de encabezado Winerror. h.

Estos son los valores que aparecen ordenados alfabéticamente por nombre.



| Nombre            | Descripción                         | Value      |
|-----------------|-------------------------------------|------------|
| S \_ correcto           | Operación correcta                | 0x00000000 |
| E \_ anular        | Operación anulada                   | 0x80004004 |
| E \_ ACCESSDENIED | Error general de acceso denegado         | 0x80070005 |
| E \_ FAIL         | Error no especificado                 | 0x80004005 |
| \_identificador E       | Identificador no válido            | 0x80070006 |
| E \_ INVALIDARG   | Uno o varios argumentos no son válidos | 0x80070057 |
| E \_ NOinterface  | No se admite esta interfaz         | 0x80004002 |
| E \_ NOTIMPL      | No implementado                     | 0x80004001 |
| E \_ OUTOFMEMORY  | No se pudo asignar la memoria necesaria | 0x8007000E |
| \_puntero E      | Puntero no válido           | 0x80004003 |
| E \_ inesperado   | Error inesperado                  | 0x8000FFFF |



 

Estos son los valores que se muestran en orden numérico por valor.



| Value      | Nombre            | Descripción                         |
|------------|-----------------|-------------------------------------|
| 0x00000000 | S \_ correcto           | Operación correcta                |
| 0x80004001 | E \_ NOTIMPL      | No implementado                     |
| 0x80004002 | E \_ NOinterface  | No se admite esta interfaz         |
| 0x80004003 | \_puntero E      | Puntero no válido           |
| 0x80004004 | E \_ anular        | Operación anulada                   |
| 0x80004005 | E \_ FAIL         | Error no especificado                 |
| 0x8000FFFF | E \_ inesperado   | Error inesperado                  |
| 0x80070005 | E \_ ACCESSDENIED | Error general de acceso denegado         |
| 0x80070006 | \_identificador E       | Identificador no válido            |
| 0x8007000E | E \_ OUTOFMEMORY  | No se pudo asignar la memoria necesaria |
| 0x80070057 | E \_ INVALIDARG   | Uno o varios argumentos no son válidos |
