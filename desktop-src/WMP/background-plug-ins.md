# <a name="background-plug-ins"></a>Complementos de fondo

Los complementos de la interfaz de usuario de fondo no tienen interfaces de usuario, a menos que implementen páginas de propiedades. Cuando se está ejecutando uno o varios complementos de fondo, aparece un icono especial en la pantalla de Media Player de Windows junto a la hora de la pista en ejecución. Se puede hacer clic en este icono para mostrar las opciones de complementos de fondo en el cuadro de diálogo Opciones. Desde allí, el usuario puede optar por abrir la página de propiedades de cualquier complemento que admita uno.

Los complementos de fondo son útiles para proporcionar servicios automáticos que no requieren la intervención del usuario.
