# <a name="playlistdropdownlist"></a>Lista de reproducción. dropDownList

El atributo **dropDownList** especifica o recupera un valor que indica qué elementos se muestran en el cuadro de lista desplegable para una instancia determinada del elemento **playlist** .

``` syntax
        elementID.dropDownList
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de lectura/escritura que contiene uno de los valores siguientes.



| Value       | Descripción                                                                                      |
|-------------|--------------------------------------------------------------------------------------------------|
| showAlbums  | Muestra álbumes.                                                                                    |
| showAll     | Predeterminada. Muestra todos los elementos disponibles, como listas de reproducción de CD, listas de reproducción de usuario y valores preestablecidos de radio. |
| showCD      | Muestra la lista de reproducción del CD.                                                                           |
| showClips   | Muestra todos los clips.                                                                                 |
| showCurrent | Muestra la lista de reproducción actual y no guardada.                                                             |
| showLibrary | Muestra solo listas de reproducción de la biblioteca.                                                                    |
| showRadio   | Muestra los valores preestablecidos de radio.                                                                             |
| showQueries | Muestra las consultas.                                                                                   |
