# <a name="__systemclass-class"></a>\_\_Clase SystemClass

La clase de sistema abstracta **\_ \_ SystemClass** es la clase base de la que se derivan la mayoría de las clases del sistema. No se pueden crear instancias de esta clase.

La siguiente sintaxis se simplifica desde el código de Managed Object Format (MOF) e incluye todas las propiedades heredadas. Las propiedades se enumeran en orden alfabético, no en orden MOF.

## <a name="syntax"></a>Sintaxis

``` syntax
[abstract]
class __SystemClass
{
};
```

## <a name="members"></a>Miembros

La clase **\_ \_ SystemClass** no define ningún miembro.
