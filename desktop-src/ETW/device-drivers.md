# <a name="device-drivers"></a>Controladores de dispositivos

Esta documentación es para las aplicaciones de modo de usuario que desean usar ETW para el seguimiento de eventos. Esta documentación no es para los controladores de dispositivos que se ejecutan en modo kernel. Para aquellos que escriban controladores de dispositivos y aplicaciones de modo kernel que quieran usar ETW como depurador de bajo impacto para solucionar problemas relacionados con el tiempo, vea [seguimiento de eventos para Windows](/windows-hardware/drivers/devtest/event-tracing-for-windows--etw-), el [seguimiento de software WPP](/windows-hardware/drivers/devtest/wpp-software-tracing)y el [seguimiento de eventos WMI](/windows-hardware/drivers/kernel/wmi-event-tracing) en el kit de controladores de Windows (WDK) para obtener información sobre cómo instrumentar el controlador.
