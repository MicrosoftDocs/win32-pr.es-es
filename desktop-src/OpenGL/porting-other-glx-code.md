# <a name="porting-other-glx-code"></a>Trasladar otro código de GLX

Además de las funciones Xlib y GLX descritas en las secciones anteriores, el programa probablemente contiene algunas de las otras funciones GLX o Xlib que se enumeran en [traduciendo la biblioteca GLX](translating-the-glx-library.md). Vuelva a escribir el código de sistema de la ventana X en Windows, sustituyendo las funciones apropiadas.
