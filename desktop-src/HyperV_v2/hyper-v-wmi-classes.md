# <a name="hyper-v-wmi-classes"></a>Clases WMI de Hyper-V

Las siguientes son las clases WMI de Hyper-V.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                 | Descripción                                                                                                                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Clases CIM](cim-classes.md)<br/>                                             | Las clases CIM son clases base de las que derivan todos los objetos WMI.<br/>                                                                                                                                                                                                               |
| [Clases de BIOS](bios-classes.md)<br/>                                           | El BIOS virtual es una imagen de software que se carga en ram para configurar algunos de los aspectos básicos de y arrancar un sistema informático. Hay un elemento BIOS por sistema informático y ese elemento no se puede reemplazar ni quitar.<br/>                                                 |
| [Clases de entrada](input-classes.md)<br/>                                         | Estas clases representan los dispositivos de entrada del usuario. Una máquina virtual siempre tiene una instancia de cada clase asociada a ella.<br/>                                                                                                                                             |
| [Clases de Integration Services](integration-services-classes.md)<br/>           | Los servicios de integración son servicios de software que se ejecutan sobre el sistema operativo invitado dentro de una máquina virtual secundaria y como parte de la pila de virtualización del sistema operativo de administración para proporcionar algún nivel de integración con el sistema operativo de administración.<br/> |
| [Clases de memoria](memory-classes.md)<br/>                                       | Las clases de memoria modela la memoria de una máquina virtual.<br/>                                                                                                                                                                                                                       |
| [Clases de procesador](processor-classes.md)<br/>                                 | Las clases de procesador modela un procesador para una máquina virtual.<br/>                                                                                                                                                                                                                   |
| [Clases de registro de perfiles](profile-registration-classes.md)<br/>           | Clases WMI de virtualización relacionadas con el registro de perfiles.<br/>                                                                                                                                                                                                                      |
| [Escritorio remoto clases](remotefx-classes.md)<br/>                             | Las Escritorio remoto wmi de virtualización se usan para administrar Escritorio remoto y RemoteFX recursos y máquinas virtuales.<br/>                                                                                                                                                     |
| [Clases para la administración de recursos](resource-management-classes.md)<br/>             | Clases WMI de virtualización relacionadas con la administración de recursos.<br/>                                                                                                                                                                                                                       |
| [Clases de dispositivos serie](serial-devices-classes.md)<br/>                       | Los dispositivos serie de una máquina virtual constan de controladores serie y puertos serie. Cada máquina virtual tiene exactamente un controlador serie y cada controlador serie tiene exactamente dos puertos serie.<br/>                                                                         |
| [Clases de almacenamiento](storage-classes.md)<br/>                                     | Los objetos de almacenamiento se separan en tres tipos: controladores, unidades y medios.<br/>                                                                                                                                                                                              |
| [Clases de vídeo](video-classes.md)<br/>                                         | Todas las máquinas virtuales reflejan la presencia de un controlador de vídeo S3 emulado y un controlador de vídeo sintético acelerado.<br/>                                                                                                                                                     |
| [Clases de sistema virtual](virtual-system-classes.md)<br/>                       | Clases WMI de virtualización relacionadas con la máquina virtual.<br/>                                                                                                                                                                                                                       |
| [Clases de administración de sistemas virtuales](virtual-system-management-classes.md)<br/> | Clases WMI de virtualización relacionadas con la administración de máquinas virtuales.<br/>                                                                                                                                                                                                                |
