# <a name="sessionlanguage-property"></a>Propiedad Session. Language

La propiedad **Language** del objeto de [**sesión**](session-object.md) es una propiedad de solo lectura que representa el identificador de idioma numérico utilizado por la sesión de instalación actual.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Session.Language
```



## <a name="property-value"></a>Valor de propiedad
