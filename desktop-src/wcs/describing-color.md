# <a name="describing-color"></a>Descripción del color

El color puede describirse en términos de componentes. Los componentes que suelen usar los profesionales de la creación de imágenes son el *matiz*, *croma* y *luminosidad*. Un matiz es lo que normalmente se considera como un color. El matiz de un color corrige su posición en el espectro visible de la luz. El croma de un color es el modo en que es "puro" o "fuerte" un color. Se dice que el gris neutro tiene una saturación de cero. La luminosidad de un color se refiere a la intensidad de la luz que se refleja o transmite mediante una imagen.

Los términos *matiz*, *tono* y *sombra* también se usan en la documentación sobre imágenes de color. Un matiz de color se obtiene combinando su matiz con el blanco. Un tono de un color se crea mezclando un matiz con gris. Un tono de un color se realiza agregando negro a su matiz.
