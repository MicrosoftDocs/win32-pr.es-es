# <a name="systemdevicesaudiodevicerawprocessingsupported"></a>System.Devices.AudioDevice.RawProcessingSupported

Compatibilidad con el modo de procesamiento sin procesar para el dispositivo de audio. Si \_ el valor de Variant es true, el dispositivo admite el modo de procesamiento sin procesar.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1

```
propertyDescription
   name = System.Devices.AudioDevice.RawProcessingSupported
   shellPKey = PKEY_Devices_AudioDevice_RawProcessingSupported
   formatID = 8943B373-388C-4395-B557-BC6DBAFFAFDB
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
