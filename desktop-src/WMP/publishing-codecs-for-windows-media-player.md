# <a name="publishing-codecs-for-windows-media-player"></a>Publicar códecs para Windows Media Player

El sitio web de WMPlugins ya no se admite para publicar el códec y para que los usuarios de Windows Media Player puedan detectarlo. En su lugar, puede publicar el códec en un sitio web de su elección.
