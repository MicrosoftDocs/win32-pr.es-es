# <a name="extending-data-structures-and-types"></a>Extender estructuras y tipos de datos

Un intervalo de valores se reserva para alojar las futuras extensiones del conjunto de funciones TAPI básico y complementario. Para obtener información acerca de la cantidad por la que se puede extender una estructura de datos, vea [extensibilidad](extensibility.md). Para obtener una lista de las funciones utilizadas para ampliar telefonía, consulte [referencia de funciones rápidas de TAPI](./tapi-quick-function-reference.md).
