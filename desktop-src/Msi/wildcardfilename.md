# <a name="wildcardfilename"></a>WildCardFilename

El tipo de datos WildCardFilename es un [nombre de archivo](filename.md) que también puede contener los caracteres comodín "?" para cualquier carácter individual o " \* " para cero o más apariciones de cualquier carácter.
