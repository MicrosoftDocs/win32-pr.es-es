# <a name="creating-the-art"></a>Crear el arte

El arte de esta máscara se creó con Adobe Photoshop. Podrían haberse usado otros programas de edición de gráficos, como Jasc Paint Shop Pro o Microsoft Paint. Photoshop se eligió como ejemplo de una aplicación de edición de gráficos que admite capas, lo que resulta útil para crear varios archivos de mapa de bits que se alinean entre sí.

Los siguientes pasos se usaron para crear el arte. Se pueden usar muchos otros procedimientos gráficos y herramientas. se trata simplemente de una manera de hacerlo. Lo importante es tener en cuenta que desea acabar con los archivos de mapa de bits específicos que necesita para su máscara.

-   [Definir los rectángulos](define-the-rectangles.md)
-   [Crear el fondo](creating-the-background.md)
-   [Crear el archivo de región](creating-the-region-file.md)
-   [Crear el archivo insertado](creating-the-pushed-file.md)
-   [Crear el archivo deshabilitado](creating-the-disabled-file.md)
-   [Crear el súper archivo](creating-the-super-file.md)
