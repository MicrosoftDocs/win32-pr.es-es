# <a name="avencddsurround3dbattenuation-property"></a>Propiedad AVEncDDSurround3dBAttenuation

Especifica si los canales envolventes de una secuencia de audio Dolby Digital están atenuados 3 decibelios (dB) antes de la codificación. Esta propiedad se aplica a los codificadores de audio Dolby digital.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncDDSurround3dBAttenuation**
