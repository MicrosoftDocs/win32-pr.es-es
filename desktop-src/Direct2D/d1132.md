# <a name="d1132-dxgi-format-not-hwnd-or-dc-compatible"></a>D1132: formato de DXGI no HWND o compatible con DC

El formato de DXGI dado no es válido para este método de creación de destino de representación.



|             |         |
|-------------|---------|
| Nivel de error | Advertencia |



 

## <a name="possible-causes"></a>Causas posibles

La creación de HWND o DC Rendertargets mediante ID2D1Factory:: CreateHwndRenderTarget o ID2D1Factory:: CreateDCRenderTarget introduce restricciones adicionales en el formato de superficie del destino. En concreto, estos destinos deben ser de tipo DXGI \_ format \_ R8G8B8A8 \_ UNORM o dxgi \_ format \_ B8G8R8A8 \_ UNORM.

## <a name="possible-fixes"></a>Posibles correcciones

Para solucionar este problema, compruebe que el formato especificado para HWND o DC de la aplicación sea uno de los formatos mencionados anteriormente. Si la aplicación no requiere necesariamente estos tipos de destinos, considere la posibilidad de representar una textura en su lugar, las opciones de formato de la superficie para la compatibilidad como destino pueden ser menos restrictivas.
