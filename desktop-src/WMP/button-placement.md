# <a name="button-placement"></a>Ubicación de los botones

Aunque puede dibujar botones para que sean cualquier forma, se tratan internamente como si tuvieran un límite externo rectangular. Por esta razón, evite la superposición de coordenadas de botón porque pueden producirse resultados inesperados.
