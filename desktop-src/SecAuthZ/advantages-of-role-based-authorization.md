# <a name="advantages-of-role-based-authorization"></a>Ventajas de la autorización basada en roles

La API de Authorization Manager proporciona un modelo de autorización que tiene varias ventajas con respecto a la autorización de bajo nivel (basada en ACL):

-   Simplifica la administración del control de acceso.
-   Hace que la autorización esté disponible para los scripts y las aplicaciones desarrollados mediante el sistema de desarrollo Visual Basic Microsoft.
-   Proporciona un mecanismo para aplicar lógica de negocios en tiempo de ejecución al comprobar el acceso.
