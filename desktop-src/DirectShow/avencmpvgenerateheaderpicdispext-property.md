# <a name="avencmpvgenerateheaderpicdispext-property"></a>Propiedad AVEncMPVGenerateHeaderPicDispExt

Especifica si el codificador genera encabezados de extensión de presentación de imágenes. Esta propiedad se aplica a los codificadores de vídeo MPEG.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncMPVGenerateHeaderPicDispExt**

## <a name="remarks"></a>Observaciones

Si el valor es **Variant \_ true**, el codificador genera encabezados de extensión de presentación de imágenes.
