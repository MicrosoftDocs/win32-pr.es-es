# <a name="client-shell-api-additional-functions"></a>Funciones adicionales de la API de Shell de cliente

En la tabla siguiente se proporciona información general de las funciones no básicas de la interfaz de programación de aplicaciones (API) del Shell de cliente Windows Administración remota de Windows (WinRM).



| Funciones adicionales                                                   | Descripción                       |
|------------------------------------------------------------------------|-----------------------------------|
| [**WSManGetErrorMessage**](/windows/desktop/api/Wsman/nf-wsman-wsmangeterrormessage)                   | Obtiene mensajes de error.              |
| [**WSManGetSessionOptionAsDword**](/windows/desktop/api/Wsman/nf-wsman-wsmangetsessionoptionasdword)   | Obtiene las opciones de sesión como DWORD.  |
| [**WSManGetSessionOptionAsString**](/windows/desktop/api/Wsman/nf-wsman-wsmangetsessionoptionasstring) | Obtiene las opciones de sesión como una cadena. |
| [**WSManSetSessionOption**](/windows/desktop/api/Wsman/nf-wsman-wsmansetsessionoption)                 | Establece las opciones de sesión.             |
