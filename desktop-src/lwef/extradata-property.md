# <a name="extradata-property"></a>Propiedad ExtraData

\[Microsoft Agent está en desuso a partir de Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descripción**
</dt> <dd>

Devuelve una cadena que especifica datos adicionales almacenados como parte del carácter.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxis**
</dt> <dd>

*agente*. **Characters("**_CharacterID_*_"). ExtraData_*

</dd> </dl>

## <a name="remarks"></a>Comentarios

El valor predeterminado de la **propiedad ExtraData** para un carácter se define cuando el carácter se compila con el Editor de caracteres del Agente de Microsoft. No se puede cambiar ni especificar en tiempo de ejecución.

> [!Note]  
> El valor de la propiedad **ExtraData** es opcional y no se puede proporcionar para todos los caracteres.
