# <a name="systemdevicesdnssdinstancename"></a>System. Devices. dnssd. InstanceName

Parte de instancia del nombre de instancia de servicio DNS-SD. (por ejemplo, "mi servicio" en "servicio. \_ http. \_ TCP. local ")

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.Devices.Dnssd.InstanceName
   shellPKey = PKEY_Devices_Dnssd_InstanceName
   formatID = BF79C0AB-BB74-4CEE-B070-470B5AE202EA
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
