# <a name="about-cabinet-files"></a>Acerca de los archivos. cab

Un contenedor es un archivo único, normalmente con el sufijo. CAB, que almacena archivos comprimidos en una biblioteca de archivos. Un archivo comprimido se puede distribuir en varios archivos. cab. Durante la instalación, la aplicación de instalación descomprime los archivos almacenados en un archivo. cab y los copia en el sistema del usuario.

Si se usan archivos. cab, las funciones de instalación usan el nombre del archivo. cab como el archivo de etiqueta del disco de origen en el que reside.
