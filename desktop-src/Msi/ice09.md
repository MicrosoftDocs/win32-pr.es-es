# <a name="ice09"></a>ICE09

ICE09 valida que el bit permanente se establece para cada componente marcado para su instalación en el Carpetadelsistema. ICE09 publica una advertencia porque debe evitar la instalación de componentes del sistema no permanentes en Carpetadelsistema. Para evitar esta advertencia, establezca el bit permanente en la columna atributos de la [tabla de componentes](component-table.md) para cada componente marcado para su instalación en el carpetadelsistema. No se realiza ninguna validación si la base de datos no tiene una tabla de componentes.
