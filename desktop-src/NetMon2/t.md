# <a name="t-network-monitor"></a>T (Monitor de red)

<dl> <dt>

<span id="_netmon_total_statistics_gly"></span><span id="_NETMON_TOTAL_STATISTICS_GLY"></span>**Estadísticas totales**
</dt> <dd>

Estadísticas que generan los informes de tarjeta de interfaz de red (NIC) y Monitor de red. Por ejemplo, las estadísticas de los fotogramas de red incluyen el número de fotogramas que emite la NIC, el número de fotogramas que controla la NIC, el número de fotogramas que se pasan a través del filtro de captura y el número de fotogramas almacenados en el búfer o el archivo de captura. Para obtener una lista completa de estas estadísticas, vea la estructura **Statistics** .

</dd> </dl>
