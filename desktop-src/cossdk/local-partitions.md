# <a name="local-partitions"></a>Particiones locales

Una manera en que se pueden usar las particiones es localmente en un servidor de aplicaciones. En este escenario, un administrador puede crear una partición mediante la herramienta administrativa Servicios de componentes. Las particiones implementadas de forma local y no dentro de Active Directory solo son accesibles para los usuarios locales, no para los usuarios de otros equipos del dominio. Después de crear una partición y de instalar una aplicación COM+ en ella, el administrador puede asignarla como una partición predeterminada para uno o más usuarios de dicho servidor. Para asignar una partición predeterminada a un usuario, el administrador puede usar la herramienta administrativa Servicios de componentes.
