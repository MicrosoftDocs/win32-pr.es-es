# <a name="prefetch-debug-apis-for-windows-store-apps"></a>API de depuración de captura previa para Windows Store

Las SIGUIENTES API se usan para depurar y probar el comportamiento de captura previa de contenido en una aplicación de Windows Store que implementa la [**clase ContentPrefetcher.**](/uwp/api/Windows.Networking.BackgroundTransfer.ContentPrefetcher)



| Interfaz                                                              | Descripción                                                                                                                                                                      |
|------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IContentPrefetcherTaskTrigger**](/windows/desktop/api/IContentPrefetcherTaskTrigger/nn-icontentprefetchertasktrigger-icontentprefetchertasktrigger) | Use esta interfaz para comprobar que un paquete de aplicación Windows Store instalado está registrado para esta tarea en segundo plano y desencadenar manualmente sus operaciones de captura previa del contenido.<br/> |
