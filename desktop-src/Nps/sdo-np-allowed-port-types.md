# <a name="np-allowed-port-types"></a>Tipos de Puerto permitidos NP

Tipos de puertos permitidos para una conexión.

**Nombre LDAP:** msNPAllowedPortTypes

**ID:** 4104 = [ **tipo de puerto de atributo de IAS \_ \_ NP \_ permitido \_ \_**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintaxis:** Avanzó

**Valores  **

Definida por Internet Assigned Numbers Authority (IANA) en el documento de [tipos de RADIUS](https://www.iana.org/assignments/radius-types) basado en [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt).

-   Async (módem) (0)
-   Sincronizar (línea T1) (1)
-   Sincronización de ISDN (2)
-   ISDN Async V. 120 (3)
-   ISDN Async V. 110 (4)
-   Virtual (VPN) (5)
-   PIAFS (6)
-   Canal de borrado HDLC (7)
-   X. 25 (8)
-   X. 75 (9)
-   Fax G. 3 (10)
-   SDSL: DSL simétrico (11)
-   ADSL-CAP-modulación de fase de amplitud no portadora DSL asimétrica (12)
-   ADSL-DMT-multitono discreto de DSL asimétrico (13)
-   IDSL: línea de suscriptor digital ISDN (14)
-   Ethernet (15)
-   xDSL: línea de suscriptor digital de tipo desconocido (16)
-   Cable (17)
-   Inalámbrico-otro (18)
-   Inalámbrica-IEEE 802,11 (19)
-   Token ring (20)
-   FDDI (21)
