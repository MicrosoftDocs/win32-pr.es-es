# <a name="chstringformat-methods"></a>Métodos CHString::Format

\[La [**clase CHString**](chstring.md) forma parte del marco de proveedores WMI que ahora se considera en estado final y no habrá más desarrollos, mejoras o actualizaciones disponibles para problemas no relacionados con la seguridad que afecten a estas bibliotecas. Las [API de MI](/previous-versions/windows/desktop/wmi_v2/windows-management-infrastructure) deben usarse para todo el desarrollo nuevo.\]

El **método Format** da formato y almacena una serie de caracteres y valores en una cadena [**CHString.**](chstring.md)

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                              | Descripción                                                                                      |
|:----------------------------------------------------|:-------------------------------------------------------------------------------------------------|
| [**Format(UINT)**](/previous-versions/windows/desktop/legacy/aa385532(v=vs.85))       | Da formato a esta cadena CHString según el identificador de recurso especificado por **el UINT**.<br/> |
| [**Format(LPCWSTR)**](/windows/win32/api/chstring/nf-chstring-chstring-format(lpcwstr_---)) | Da formato a esta cadena CHString según el formato especificado por **LPCWSTR**.<br/>           |
