# <a name="ui_pkey_fontproperties"></a>FontProperties de PKEY de interfaz \_ \_ de usuario

Identifica la propiedad \_ FontProperties de PKEY de la \_ interfaz de usuario.

```
propertyDescription
   name = UI_PKEY_FontProperties
   shellPKey = UI_PKEY_FontProperties
   formatID = 00000300-7363-696e-8441798acf5aebb7
   propID = 300
   typeInfo
      type = IPropertyStore
```

## <a name="remarks"></a>Comentarios

Ui \_ PKEY \_ FontProperties es un almacén de propiedades utilizado por una aplicación para consultar todas las propiedades de fuente.
