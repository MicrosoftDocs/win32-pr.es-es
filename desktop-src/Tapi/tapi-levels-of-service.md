# <a name="tapi-levels-of-service"></a>Niveles de servicio de TAPI

Los servicios de telefonía TAPI se dividen en servicios de telefonía asistida y los servicios proporcionados por la API de telefonía completa. En general, la API de telefonía completa se usa para implementar aplicaciones telefónicas eficaces y la telefonía asistida se usa para agregar funcionalidad telefónica mínima pero útil a las aplicaciones que no son de telefonía.
