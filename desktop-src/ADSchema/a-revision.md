# <a name="revision-attribute"></a>Atributo de revisión

El nivel de revisión para un descriptor de seguridad u otro cambio. Solo se usa en los objetos Sam-Server y DS-UI-Settings.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Revisión                             |
| Nombre para mostrar de LDAP | revision                             |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.145               |
| System-ID-GUID    | bf967a21-0de6-11d0-a285-00aa003049e2 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
