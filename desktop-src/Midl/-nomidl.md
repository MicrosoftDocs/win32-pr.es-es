# <a name="nomidl-switch"></a>modificador/nomidl

El modificador **/nomidl** suprime la ejecución MIDL.EXE después de procesar el archivo Windows Runtime IDL.

``` syntax
midlrt /nomidl 
```

## <a name="switch-options"></a>Opciones de conmutador

Este modificador no tiene parámetros.

## <a name="remarks"></a>Observaciones

## <a name="examples"></a>Ejemplos

**midlrt/nomidl**
