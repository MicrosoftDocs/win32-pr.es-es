# <a name="ldap-syntax-object"></a>Objeto de sintaxis LDAP

El proveedor LDAP usa la siguiente asignación entre la sintaxis LDAP y la sintaxis ADSI.



| Sintaxis LDAP   | Sintaxis ADSI (Automation)            |
|---------------|-------------------------------------|
| ADsPath       | VT \_ BSTR                            |
| Boolean       | VT \_ BOOL                            |
| Contador       | VT \_ I4                              |
| EmailAddress  | VT \_ BSTR                            |
| FaxNumber     | VT \_ BSTR                            |
| Entero       | VT \_ I4                              |
| Intervalo      | VT \_ I4                              |
| List          | VT \_ VARIANT (VT \_ BSTR \| VT \_ ARRAY) |
| NetAddress    | VT \_ BSTR                            |
| OctetString   | VT \_ VARIANT                         |
| Ruta de acceso          | VT \_ BSTR                            |
| PhoneNumber   | VT \_ BSTR                            |
| PostalAddress | VT \_ BSTR                            |
| SmallInterval | VT \_ I4                              |
| String        | VT \_ BSTR                            |
| Time          | FECHA \_ DE VT                            |
