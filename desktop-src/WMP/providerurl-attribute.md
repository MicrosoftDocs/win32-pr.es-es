# <a name="providerurl-attribute"></a>Atributo ProviderURL

El atributo **ProviderURL** es la dirección de la Página principal del proveedor de los valores de atributo.

## <a name="applies-to"></a>Se aplica a

-   [Elementos de audio](audio-item-attributes.md)
-   [DVDs](dvd-attributes.md)

## <a name="remarks"></a>Observaciones

Este atributo solo se almacena en la base de datos (o caché) de la biblioteca.

Para determinar si puede cambiar el valor de este atributo, use el método [media. isReadOnlyItem](media-isreadonlyitem.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------|
| Versión<br/> | Windows Media Player 9 series o posterior<br/> |
