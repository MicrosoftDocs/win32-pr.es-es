# <a name="creating-a-basic-debugger"></a>Creación de un depurador básico

[En Depuración de un proceso en](debugging-a-running-process.md) ejecución se describe cómo asociar el depurador a un proceso en ejecución. [Al escribir el bucle main del depurador](writing-the-debugger-s-main-loop.md) se describen las funciones que se usan en el bucle principal de ejecución del depurador.

Aunque la mayoría de las funciones de depuración se usan para crear un depurador, varias funciones están diseñadas para su uso en el proceso que se depura. Para obtener más información, [vea Comunicación con el depurador](communicating-with-the-debugger.md).
