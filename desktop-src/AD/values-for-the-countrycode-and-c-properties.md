# <a name="values-for-the-countrycode-and-c-properties"></a>Valores de las propiedades countryCode y c

Los valores de las propiedades **CountryCode** y **c** se obtienen del estándar ISO 3166. Esta norma está disponible en el sitio web de [organización internacional de normalización](https://www.iso.org/) . El valor de la propiedad **CountryCode** es el código de país numérico de ISO 3166 y el valor de la propiedad **c** es la designación de país o región de dos letras de ISO 3166.
