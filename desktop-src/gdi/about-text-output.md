# <a name="about-text-output"></a>Acerca de la salida de texto

La salida de texto es el tipo más común de salida de gráficos que se encuentra en el área de cliente. lo usan las aplicaciones de maneras diferentes. Las aplicaciones de procesamiento de textos y de publicación de escritorio crean documentos con texto con formato. las aplicaciones de hoja de cálculo usan texto, números y símbolos para especificar fórmulas, columnas de etiqueta y valores de lista. las aplicaciones de base de datos crean registros y muestran consultas con texto, y las aplicaciones CAD usan texto para etiquetar objetos y mostrar las dimensiones.

Hay funciones para dar formato y dibujar texto en el área de cliente de una aplicación y en una página de papel de la impresora. Estas funciones se pueden dividir en dos categorías: las que dan formato al texto (o la preparan para la salida) y las que realmente dibujan el texto. Las funciones de formato alinean el texto, establecen el espaciado entre caracteres, establecen el texto y los colores de fondo del texto, y justifican el texto. Las funciones de dibujo dibujan caracteres individuales (o símbolos) o cadenas de texto completas.

Cuando se trabaja en Microsoft Windows, los saltos de línea se especifican con el par de retorno de carro/avance de línea ( \\ r \\ n).
