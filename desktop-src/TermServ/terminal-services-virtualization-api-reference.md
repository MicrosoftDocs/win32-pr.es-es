# <a name="remote-desktop-virtualization-api-reference"></a>Referencia de API de virtualización de Escritorio remoto

La API de virtualización Escritorio remoto admite interfaces y estructuras que puede usar para crear complementos personalizados que invalidan la lógica de redirección estándar del agente de Conexión a Escritorio remoto (agente de conexión a escritorio remoto). Los complementos personalizados pueden redirigir las conexiones a Escritorio remoto servidores host de sesión (host de sesión de escritorio remoto), así como máquinas virtuales, escritorios virtuales, servidores Blade y escritorios físicos.
