# <a name="monitor-configuration-enumeration-types"></a>Supervisar tipos de enumeración de configuración

Las funciones de configuración del monitor utilizan las siguientes enumeraciones.



| Tipo de enumeración                                                    | Descripción                                                                                      |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**\_temperatura de color de MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_color_temperature)              | Describe la temperatura de color de un monitor.                                                         |
| [**\_tipo de \_ tecnología de visualización MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_display_technology_type) | Identifica las tecnologías de visualización del monitor.                                                         |
| [**MC ( \_ tipo de unidad) \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_drive_type)                            | Especifica si se va a establecer u obtener una unidad roja, verde o azul del monitor.                           |
| [**\_tipo de ganancia MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_gain_type)                              | Especifica si se va a obtener o establecer una ganancia roja, verde o azul del monitor.                            |
| [**\_tipo de posición MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_position_type)                      | Especifica si se va a obtener o establecer la posición vertical u horizontal del área de presentación de un monitor. |
| [**\_tipo de tamaño MC \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_size_type)                              | Especifica si se va a obtener o establecer el ancho o el alto del área de presentación de un monitor.                 |
| [**\_tipo de \_ código MC VCP \_**](/windows/desktop/api/LowLevelMonitorConfigurationAPI/ne-lowlevelmonitorconfigurationapi-mc_vcp_code_type)                     | Describe un tipo de código del panel de control virtual (VCP).                                               |
