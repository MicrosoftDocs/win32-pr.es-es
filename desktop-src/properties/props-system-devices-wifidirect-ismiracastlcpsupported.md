# <a name="systemdeviceswifidirectismiracastlcpsupported"></a>System. Devices. WiFiDirect. IsMiracastLcpSupported

Indica si la protección de contenido de vínculo es compatible con el dispositivo Wi-Fi Direct si es compatible con Miracast

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.IsMiracastLcpSupported
   shellPKey = PKEY_Devices_WiFiDirect_IsMiracastLcpSupported
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 9
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
