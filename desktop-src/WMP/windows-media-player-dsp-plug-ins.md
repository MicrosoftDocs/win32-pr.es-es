# <a name="windows-media-player-dsp-plug-ins"></a>Complementos DSP de Windows Media Player

Microsoft Windows Media Player proporciona una arquitectura que permite al usuario instalar y activar programas de complementos que agregan funcionalidad de procesamiento de señal digital (DSP). Un complemento de DSP es un objeto Microsoft DirectX media (DMO) o una transformación de Media Foundation (MFT) que se conecta al reproductor mediante interfaces COM. Un complemento DSP típico puede ser un ecualizador de audio o un control de matiz de vídeo. Esta sección del SDK proporciona la información de programación que necesita para crear su propio complemento DSP.

La documentación del complemento de DSP está dividida en las tres secciones siguientes.



| Sección                                                                      | Descripción                                                                                                                                     |
|------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| [Acerca de los complementos DSP](about-dsp-plug-ins.md)                                 | Proporciona información general sobre la arquitectura utilizada para los complementos DSP. Lea esta sección para obtener información sobre los conceptos generales implicados en esta tecnología.  |
| [Guía de programación de complementos DSP](dsp-plug-ins-programming-guide.md)         | Explica lo que debe hacer para crear un complemento DSP. Esta sección contiene código de ejemplo y procedimientos paso a paso.                           |
| [Referencia de programación de complementos DSP](dsp-plug-ins-programming-reference.md) | Proporciona una referencia detallada de las interfaces, los métodos y los tipos enumerados COM admitidos por el SDK de Windows Media Player para los complementos de DSP. |
