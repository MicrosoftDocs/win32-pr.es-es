# <a name="wmoriginalreleaseyear"></a>WM/OriginalReleaseYear

El **atributo WM/OriginalReleaseYear** contiene el año durante el que se publicó por primera vez el contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMOriginalReleaseYear

## <a name="data-type"></a>Tipo de datos

**CADENA DE \_ TIPO \_ WMT**
