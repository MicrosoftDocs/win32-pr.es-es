# <a name="mft_enum_video_renderer_extension_profile-attribute"></a>\_Atributo de \_ \_ Perfil de extensión de representador de vídeo enum \_ de MFT \_

Contiene una lista de todas las entradas de **VideoRendererExtensionProfile** en la etiqueta **VideoRendererExtensionProfiles** de un archivo de manifiesto de aplicación para UWP. [**MFTEnumEx**](/windows/desktop/api/mfapi/nf-mfapi-mftenumex) lo almacena en el almacén de atributos del objeto [**IMFActivate**](/windows/desktop/api/mfobjects/nn-mfobjects-imfactivate) que **MFTEnumEx** crea para MFTS que tienen un manifiesto de UWP asociado que contiene la etiqueta **VideoRendererExtensionProfiles** .
