# <a name="vml-callout-element"></a>Elemento Callout de VML

En este tema se describe VML, una característica que está desusada en Windows Internet Explorer 9. Las páginas web y las aplicaciones que se basan en VML se deben migrar a SVG u otros estándares ampliamente admitidos.

> [!Note]  
> A partir del 2011 de diciembre, este tema se ha archivado. Como resultado, ya no se mantiene de forma activa. Para obtener más información, vea [contenido archivado](/previous-versions/windows/internet-explorer/ie-developer/). Para obtener información, recomendaciones e instrucciones sobre la versión actual de Windows Internet Explorer, consulte [Centro para desarrolladores de Internet Explorer](https://msdn.microsoft.com/ie/).

 

Define una llamada para una forma.

Los atributos siguientes modifican una llamada.



| Atributo                                                        | Descripción                                                                             |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [AccentBar](msdn-online-vml-accentbar-attribute.md)             | Determina si se utilizará una barra de énfasis con la llamada.                         |
| [Angulo](angle-attribute--callout--vml.md)                       | Define el ángulo que realiza la llamada con respecto al rectángulo de selección de la forma. |
| [Distancia](msdn-online-vml-distance-attribute.md)               | Define la distancia de colocación de una llamada.                                                 |
| [Omisiones](msdn-online-vml-drop-attribute.md)                       | Determina dónde se colocará la colocación de una llamada.                                  |
| [DropAuto](msdn-online-vml-dropauto-attribute.md)               | Determina si la llamada tendrá una eliminación automática.                             |
| [Total](ext-attribute--callout--vml.md)                           | Define cómo se procesa una llamada.                                                     |
| [Espacio](msdn-online-vml-gap-attribute.md)                         | Define la distancia de la llamada desde su rectángulo delimitador.                        |
| [Duración](msdn-online-vml-length-attribute.md)                   | Define la longitud de la llamada.                                                      |
| [LengthSpecified](msdn-online-vml-lengthspecified-attribute.md) | Determina si el atributo de **longitud** se utilizará para la llamada.               |
| [MinusX](msdn-online-vml-minusx-attribute.md)                   | Determina si la llamada se girará a lo largo del eje x.                              |
| [Menos](msdn-online-vml-minusy-attribute.md)                   | Determina si la llamada se girará a lo largo del eje y.                              |
| [Activado](on-attribute--callout--vml.md)                             | Determina si una forma es una llamada.                                                |
| [TextBorder](msdn-online-vml-textborder-attribute.md)           | Determina si una llamada tendrá un borde de texto.                                   |
| [Tipo](type-attribute--callout--vml.md)                         | Define el tipo de llamada.                                                            |



 

**Comentarios:**

Este elemento debe definirse dentro de un elemento de [forma](shape-element--vml.md) .

El elemento **Callout** es una extensión Microsoft Office a VML.
