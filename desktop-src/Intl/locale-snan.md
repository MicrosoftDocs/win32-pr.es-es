# <a name="locale_snan"></a>LOCALE \_ SNAN

**Windows Vista y versiones posteriores:** Valor de cadena para "No es un número", por ejemplo, "Nan" para la configuración regional de inglés (Estados Unidos). No hay ningún límite en el número de caracteres permitidos para esta cadena.
