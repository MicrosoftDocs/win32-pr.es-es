# <a name="ms-ds-az-operation-id-attribute"></a>atributo MS-DS-AZ-Operation-ID

Un identificador específico de la aplicación que hace que la operación sea exclusiva de la aplicación.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-DS-AZ-Operation-ID                   |
| Nombre para mostrar de LDAP | msDS-AzOperationID                      |
| Tamaño              | \-                                      |
| Actualizar privilegio  | Administrador de AzRoles                           |
| Frecuencia de actualización  | Durante la inicialización o el cambio de directiva. |
| Attribute-Id      | 1.2.840.113556.1.4.1800                 |
| System-ID-GUID    | a5f3b553-5d76-4cbe-ba3f-4312152cab18    |
| Sintaxis            | [**Enumeración**](s-enumeration.md)    |
