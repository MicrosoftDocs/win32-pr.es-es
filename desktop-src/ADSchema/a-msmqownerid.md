# <a name="msmq-owner-id-attribute"></a>Atributo MSMQ-Owner-ID

El atributo **MSMQ-Owner-ID** contiene información de modo mixto de MSMQ.

|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Owner-ID                                         |
| Nombre para mostrar de LDAP | mSMQOwnerID                                           |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.925                                |
