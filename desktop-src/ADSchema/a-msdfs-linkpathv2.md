# <a name="ms-dfs-link-path-v2-attribute"></a>Atributo ms-DFS-Link-Path-v2

Ruta de acceso del vínculo DFS relativa al recurso compartido de destino raíz DFS (es decir, sin los componentes de nombre de espacio de nombres DFS o servidor/dominio). Use barras diagonales (/) en lugar de barras diagonales inversas ( ), para que las búsquedas LDAP se puedan realizar sin tener que \\ usar escapes.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DFS-Link-Path-v2                         |
| Ldap-Display-Name | msDFS-LinkPathv2                            |
| Size              | \-                                          |
| Actualizar privilegios  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2039                     |
| System-Id-Guid    | 86b021f6-10ab-40a2-a252-1dc0cc3be6a9        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
