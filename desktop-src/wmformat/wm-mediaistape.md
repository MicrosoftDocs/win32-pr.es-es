# <a name="wmmediaistape"></a>WM/MediaIsTape

El atributo **WM/MediaIsTape** indica si el contenido se difundió desde una cinta grabada previamente.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaIsTape

## <a name="data-type"></a>Tipo de datos

**tipo de WMT \_ \_ bool**

## <a name="remarks"></a>Observaciones

Este atributo se utiliza para la televisión de difusión.
