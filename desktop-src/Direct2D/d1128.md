# <a name="d1128-dxgi-format-not-supported-in-any-api"></a>D1128: el formato de DXGI no se admite en ninguna API

El formato de DXGI que se pasa a esta API no es compatible con Direct2D.



|             |         |
|-------------|---------|
| Nivel de error | Advertencia |



 

## <a name="possible-causes"></a>Causas posibles

Algunos formatos de Surface no son compatibles con las operaciones de Direct2D, independientemente del modo en que se usan. Este mensaje indica que la aplicación debe elegir otro formato de superficie.

## <a name="possible-fixes"></a>Posibles correcciones

Vea [formatos de píxeles compatibles y modos alfa](supported-pixel-formats-and-alpha-modes.md) para obtener instrucciones sobre qué formatos se admiten y cuáles no. Si la aplicación debe representarse en un solo canal y está intentando usar un formato de solo canal rojo que no se admite, considere la posibilidad de cambiarla para que se represente en un formato solo de canal alfa.
