# <a name="taskfoldergetfolders-property"></a>Propiedad TaskFolder. GetFolders

En el caso de scripting, obtiene todas las subcarpetas de la carpeta.

## <a name="syntax"></a>Sintaxis


```VB
TaskFolder.GetFolders( _
  ByVal flags, _
  ByRef folders _
)
```
