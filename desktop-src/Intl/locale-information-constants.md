# <a name="locale-information-constants"></a>Constantes de información de configuración regional

En esta sección se definen las constantes que NLS usa para especificar partes concretas de información de configuración regional. Todos los nombres comienzan por \_ "LOCALE". Los nombres de constantes que representan valores enteros comienzan por "LOCALE \_ I". Todos los valores de estas constantes son cadenas Unicode terminadas en NULL. Las longitudes máximas de cadena incluyen el carácter nulo de terminación. Si no se indica ninguna longitud máxima para una constante, sus valores pueden variar de longitud.

## <a name="combining-constants"></a>Combinación de constantes

Las siguientes constantes se pueden combinar con otras constantes mediante el operador OR binario. El resto de las constantes son mutuamente excluyentes y, por tanto, no se pueden combinar entre sí.

-   CONFIGURACIÓN REGIONAL \_ NOUSEROVERRIDE
-   NÚMERO DE VALOR DEVUELTO \_ DE LA \_ CONFIGURACIÓN REGIONAL
-   CONFIGURACIÓN REGIONAL \_ USE \_ CP \_ ACP
