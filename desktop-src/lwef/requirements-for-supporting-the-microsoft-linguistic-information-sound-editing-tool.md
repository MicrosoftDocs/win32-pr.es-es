# <a name="requirements-for-supporting-the-microsoft-linguistic-information-sound-editing-tool"></a>Requisitos para admitir la herramienta de edición de sonido lingüístico de Microsoft

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

La herramienta de edición de sonido lingüístico de Microsoft usa un motor de reconocimiento de voz para generar saltos de palabras e información fonética para los archivos de sonido de Windows Wave (. wav) estándar. La versión 2,0 ahora admite el uso con otros motores de voz. Los proveedores que deseen admitir el editor de sonidos deben asegurarse de que sus motores sean totalmente compatibles con la especificación SAPI 4,0 para motores de gramática sin contexto y los siguientes requisitos.

-   [ISRResGraphEx y IAttributes](isrresgraphex-and-iattributes.md)
