# <a name="button-placement"></a>Colocación de botones

Aunque se pueden dibujar botones para que sean cualquier forma, se tratan internamente como si tengan un límite exterior rectangular. Por este motivo, evite la superposición de coordenadas de botón porque pueden producirse resultados inesperados.
