# <a name="unpublishfeatures-action"></a>Acción UnpublishFeatures

Cuando se desinstala un producto completo, la acción UnpublishFeatures quita la información de asignación de componentes de selección y estado de selección del registro del sistema. Esta acción consulta la [tabla FeatureComponents](featurecomponents-table.md).

## <a name="sequence-restrictions"></a>Restricciones de secuencia

No hay restricciones de secuencia.

## <a name="actiondata-messages"></a>Mensajes ActionData



| Campo | Descripción de los datos de acción             |
|-------|----------------------------------------|
| \[1\] | Identificador de una característica no anunciada. |
