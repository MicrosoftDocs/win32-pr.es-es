# <a name="ms-sql-lastbackupdate-attribute"></a>Atributo MS-SQL-LastBackupDate

Última fecha en la que se realizó la copia de seguridad de la base de datos.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-LastBackupDate                       |
| Nombre para mostrar de LDAP | mS-SQL-LastBackupDate                       |
| Tamaño              | \-                                          |
| Actualizar privilegio  | El sistema establece este valor.            |
| Frecuencia de actualización  | Cuando se realiza una copia de seguridad de la base de datos.             |
| Attribute-Id      | 1.2.840.113556.1.4.1398                     |
| System-ID-GUID    | f2b6abca-ccee-11d2-9993-0000f87a57d4        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
