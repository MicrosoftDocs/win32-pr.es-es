# <a name="custom-properties-events-and-control-patterns"></a>Propiedades, eventos y patrones de control personalizados

La API de automatización de la interfaz de usuario de Microsoft especifica un conjunto de propiedades, patrones de control y eventos predefinidos. Sin embargo, las aplicaciones no se limitan al uso de estas especificaciones predefinidas. Las características de extensibilidad de la automatización de la interfaz de usuario permiten a terceros incorporar propiedades, eventos y patrones de control personalizados, mutuamente acordados, para admitir nuevos elementos de interfaz de usuario y escenarios de aplicaciones. Los clientes y proveedores de automatización de la interfaz de usuario pueden empezar a usar las propiedades personalizadas, los eventos y los patrones de control inmediatamente, sin necesidad de que se actualice el marco de automatización de la interfaz de usuario principal.
