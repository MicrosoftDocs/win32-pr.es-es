# <a name="regions-windows-gdi"></a>Regiones (Windows GDI)

Una *región* es un rectángulo, un polígono o una elipse (o una combinación de dos o más de estas formas) que se puede rellenar, pintar, invertir, tramar y usar para realizar la prueba de posicionamiento (probar la ubicación del cursor).
