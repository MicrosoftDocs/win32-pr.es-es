# <a name="restorebookmark-method"></a>RestoreBookmark (método)

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

El método mueve el objeto MSWebDVD al punto del disco como se especifica en el marcador actual, con toda la configuración de audio, vídeo y `RestoreBookmark` subapicture restaurada. 

``` syntax
MSWebDVD.RestoreBookmark()
```

## <a name="return-value"></a>Valor devuelto

No de devuelve ningún valor.
