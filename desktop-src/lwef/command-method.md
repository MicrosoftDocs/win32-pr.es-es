# <a name="command-method"></a>Método de comando

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Devuelve un objeto [**Command**](/windows/desktop/lwef/the-command-object) en una colección [**Commands**](/windows/desktop/lwef/the-commands-collection-object) .

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintáctica**
</dt> <dd>

*agente ***. Caracteres ("*** CharacterID * *"). Commands. Command** "*Name*"

</dd> </dl>

## <a name="remarks"></a>Observaciones

Puede utilizar este método para tener acceso a las propiedades de un objeto [**Command**](/windows/desktop/lwef/the-command-object) .

> [!Note]  
> Este método puede ser necesario para algunos lenguajes de programación. No es necesario para VBScript o Visual Basic. Para obtener más información sobre cómo especificar métodos de **comando** , vea propiedades del objeto de comando.
