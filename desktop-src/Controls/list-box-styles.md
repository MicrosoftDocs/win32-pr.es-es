# <a name="list-box-styles"></a>Estilos de cuadro de lista

Para crear un cuadro de lista mediante la función [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) o [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) , use la clase ListBox, las constantes de estilo de ventana apropiadas y las siguientes constantes de estilo para definir el cuadro de lista. Una vez creado el control, estos estilos no se pueden modificar, salvo que se indique lo contrario.



| Constante                                                                                                                                                                              | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="LBS_COMBOBOX"></span><span id="lbs_combobox"></span><dl> <dt>**\_cuadro combinado lbs**</dt> </dl>                            | Notifica a un cuadro de lista que forma parte de un cuadro combinado. Esto permite la coordinación entre los dos controles para que presenten una interfaz de usuario unificada. El propio cuadro combinado debe establecer este estilo. Si el estilo se establece en cualquier cosa, excepto en el cuadro combinado, el cuadro de lista se considerará incorrectamente como un elemento secundario de un cuadro combinado y se producirá un error.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_DISABLENOSCROLL"></span><span id="lbs_disablenoscroll"></span><dl> <dt>**LIBRAS \_ DISABLENOSCROLL**</dt> </dl>       | Muestra una barra de desplazamiento horizontal o vertical deshabilitada cuando el cuadro de lista no contiene suficientes elementos para desplazarse. Si no especifica este estilo, la barra de desplazamiento se oculta cuando el cuadro de lista no contiene suficientes elementos. Este estilo se debe usar con el estilo [**WS \_ VSCROLL**](/windows/desktop/winmsg/window-styles) o [**WS \_ HSCROLL**](/windows/desktop/winmsg/window-styles) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_EXTENDEDSEL"></span><span id="lbs_extendedsel"></span><dl> <dt>**LIBRAS \_ EXTENDEDSEL**</dt> </dl>                   | Permite que se seleccionen varios elementos con la tecla Mayús y el mouse o las combinaciones de teclas especiales.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_HASSTRINGS"></span><span id="lbs_hasstrings"></span><dl> <dt>**LIBRAS \_ HASSTRINGS**</dt> </dl>                      | Especifica que un cuadro de lista contiene elementos que se componen de cadenas. El cuadro de lista mantiene la memoria y las direcciones para las cadenas, de modo que la aplicación pueda usar el mensaje [**lb \_ GETTEXT**](lb-gettext.md) para recuperar el texto de un elemento determinado. De forma predeterminada, todos los cuadros de lista excepto los cuadros de lista dibujados por el propietario tienen este estilo. Puede crear un cuadro de lista dibujado por el propietario con o sin este estilo.<br/> En el caso de los cuadros de lista dibujados por el propietario sin este estilo, el mensaje [**lb \_ GETTEXT**](lb-gettext.md) recupera el valor asociado a un elemento (los datos del elemento). <br/>                                                                                                                                                                                                                                                                                                                                    |
| <span id="LBS_MULTICOLUMN"></span><span id="lbs_multicolumn"></span><dl> <dt>**LIBRAS- \_ varias columnas**</dt> </dl>                   | Especifica un cuadro de lista de varias columnas que se desplaza horizontalmente. El cuadro de lista calcula automáticamente el ancho de las columnas, o una aplicación puede establecer el ancho mediante el mensaje [**lb \_ SETCOLUMNWIDTH**](lb-setcolumnwidth.md) . Si un cuadro de lista tiene el estilo- **\_ OWNERDRAWFIXED lb** , una aplicación puede establecer el ancho cuando el cuadro de lista envía el mensaje de [**\_ MEASUREITEM de WM**](wm-measureitem.md) . <br/> Un cuadro de lista con el estilo de **\_ varias columnas lbs** no se puede desplazar verticalmente; omite los mensajes [**\_ VSCROLL de WM**](wm-vscroll.md) que recibe. <br/> Los estilos **lbs \_ multicolumn** y **lb \_ OWNERDRAWVARIABLE** no se pueden combinar. Si se especifican ambos, se omite el **\_ OWNERDRAWVARIABLE de lbs** .<br/>                                                                                                                                                                                |
| <span id="LBS_MULTIPLESEL"></span><span id="lbs_multiplesel"></span><dl> <dt>**LIBRAS \_ MULTIPLESEL**</dt> </dl>                   | Activa o desactiva la selección de cadena cada vez que el usuario hace clic en una cadena del cuadro de lista o hace doble clic en ella. El usuario puede seleccionar cualquier número de cadenas.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NODATA"></span><span id="lbs_nodata"></span><dl> <dt>**LBS \_ NOdata**</dt> </dl>                                  | Especifica un cuadro de lista sin datos. Especifique este estilo cuando el recuento de elementos del cuadro de lista supere 1000. Un cuadro de lista sin datos también debe tener el estilo de la **lbs- \_ OWNERDRAWFIXED** , pero no debe tener el estilo de **\_ HASSTRINGS** **lbs o \_** lb. <br/> Un cuadro de lista sin datos es similar a un cuadro de lista dibujado por el propietario, salvo que no contiene datos de cadena o de mapa de bits para un elemento. Los comandos para agregar, insertar o eliminar un elemento siempre omiten los datos de elementos especificados; las solicitudes para buscar una cadena en el cuadro de lista siempre producen un error. El sistema envía el mensaje de Windows de [**WM \_**](wm-drawitem.md) a la ventana propietaria cuando se debe dibujar un elemento. El miembro **itemID** de la estructura [**drawitemstruct (**](/windows/win32/api/winuser/ns-winuser-drawitemstruct) pasada con el mensaje de **\_ DRAWITEM de WM** especifica el número de línea del elemento que se va a dibujar. Un cuadro de lista sin datos no envía un mensaje de [**WM \_ DELETEITEM**](wm-deleteitem.md) .<br/> |
| <span id="LBS_NOINTEGRALHEIGHT"></span><span id="lbs_nointegralheight"></span><dl> <dt>**LIBRAS \_ NOINTEGRALHEIGHT**</dt> </dl>    | Especifica que el tamaño del cuadro de lista es exactamente el tamaño especificado por la aplicación cuando creó el cuadro de lista. Normalmente, el sistema ajusta el tamaño de un cuadro de lista para que el cuadro de lista no muestre elementos parciales.<br/> En el caso de los cuadros de lista con el estilo **lb \_ OWNERDRAWVARIABLE** , siempre se aplica el estilo **lb \_ NOINTEGRALHEIGHT** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="LBS_NOREDRAW"></span><span id="lbs_noredraw"></span><dl> <dt>**LIBRAS \_ NOREDRAW**</dt> </dl>                            | Especifica que la apariencia del cuadro de lista no se actualiza cuando se realizan cambios.<br/> Para cambiar el estado de redibujo del control, use el mensaje de [**\_ SETREDRAW de WM**](/windows/desktop/gdi/wm-setredraw) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="LBS_NOSEL"></span><span id="lbs_nosel"></span><dl> <dt>**LIBRAS \_ morrol**</dt> </dl>                                     | Especifica que el cuadro de lista contiene elementos que se pueden ver pero que no se seleccionan. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NOTIFY"></span><span id="lbs_notify"></span><dl> <dt>**notificación de LBS \_**</dt> </dl>                                  | Hace que el cuadro de lista envíe un código de notificación a la ventana primaria siempre que el usuario haga clic en un elemento del cuadro de lista ([LBN \_ SELCHANGE](lbn-selchange.md)), haga doble clic en un elemento ([LBN \_ DBLCLK](lbn-dblclk.md)) o cancele la selección ([LBN \_ SELCANCEL](lbn-selcancel.md)). <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="LBS_OWNERDRAWFIXED"></span><span id="lbs_ownerdrawfixed"></span><dl> <dt>**+ \_ OWNERDRAWFIXED**</dt> </dl>          | Especifica que el propietario del cuadro de lista es responsable de dibujar su contenido y de que los elementos del cuadro de lista tienen el mismo alto. La ventana propietaria recibe un mensaje de [**\_ MEASUREITEM de WM**](wm-measureitem.md) cuando se crea el cuadro de lista y un mensaje de Windows de [**WM \_**](wm-drawitem.md) cuando cambia el aspecto visual del cuadro de lista.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="LBS_OWNERDRAWVARIABLE"></span><span id="lbs_ownerdrawvariable"></span><dl> <dt>**LIBRAS de \_ OWNERDRAWVARIABLE**</dt> </dl> | Especifica que el propietario del cuadro de lista es responsable de dibujar su contenido y de que los elementos del cuadro de lista tienen una altura variable. La ventana propietaria recibe un mensaje de [**\_ MEASUREITEM de WM**](wm-measureitem.md) para cada elemento del cuadro cuando se crea el cuadro de lista y un mensaje de Windows de [**WM \_**](wm-drawitem.md) cuando ha cambiado un aspecto visual del cuadro de lista.<br/> Este estilo hace que se habilite el estilo **lbs \_ NOINTEGRALHEIGHT** .<br/> Este estilo se omite si se especifica el estilo de **\_ varias columnas lbs** . <br/>                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="LBS_SORT"></span><span id="lbs_sort"></span><dl> <dt>**\_alfabetización**</dt> </dl>                                        | Ordena alfabéticamente las cadenas en el cuadro de lista.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="LBS_STANDARD"></span><span id="lbs_standard"></span><dl> <dt>**LIBRAS \_ estándar**</dt> </dl>                            | Ordena alfabéticamente las cadenas en el cuadro de lista. La ventana primaria recibe un código de notificación cada vez que el usuario hace clic en un elemento de cuadro de lista, hace doble clic en un elemento, o bien cancela la selección. El cuadro de lista tiene una barra de desplazamiento vertical y tiene bordes en todos los lados. Este estilo combina los **estilos \_ lbs Notify**, **lb \_ Sort**, [**WS \_ VSCROLL**](/windows/desktop/winmsg/window-styles)y [**WS \_ Border**](/windows/desktop/winmsg/window-styles) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="LBS_USETABSTOPS"></span><span id="lbs_usetabstops"></span><dl> <dt>**LIBRAS \_ USETABSTOPS**</dt> </dl>                   | Habilita un cuadro de lista para reconocer y expandir los caracteres de tabulación al dibujar sus cadenas. Puede usar el mensaje [**lb \_ SETTABSTOPS**](lb-settabstops.md) para especificar posiciones de tabulación. Las posiciones de tabulación predeterminadas son de 32 unidades de plantilla de cuadro de diálogo. Las unidades de plantilla de cuadro de diálogo son las unidades independientes del dispositivo que se usan en las plantillas de cuadro de diálogo. Para convertir las medidas de las unidades de plantilla de cuadro de diálogo en unidades de pantalla (píxeles), use la función [**MapDialogRect**](/windows/desktop/api/winuser/nf-winuser-mapdialogrect) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_WANTKEYBOARDINPUT"></span><span id="lbs_wantkeyboardinput"></span><dl> <dt>**LIBRAS \_ WANTKEYBOARDINPUT**</dt> </dl> | Especifica que el propietario del cuadro de lista recibe mensajes de [**WM \_ VKEYTOITEM**](wm-vkeytoitem.md) cada vez que el usuario presiona una tecla y el cuadro de lista tiene el foco de entrada. Esto permite que una aplicación realice procesamiento especial en la entrada del teclado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
