# <a name="avencaudiointervaltoskip-property"></a>AvEncAudioIntervalToSkip, propiedad

Especifica el número de muestras de audio que el codificador omitirá.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT64** (**VT \_ UI8**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncAudioIntervalToSkip**
