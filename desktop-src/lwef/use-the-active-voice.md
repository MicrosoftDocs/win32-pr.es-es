# <a name="use-the-active-voice"></a>Usar la voz activa

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Al utilizar la salida de voz para proporcionar información de directivas o para obtener una respuesta de usuario, use la voz activa y especifique claramente la acción esperada del usuario. En el ejemplo siguiente se muestran las diferencias:



|                                      |                               |
|--------------------------------------|-------------------------------|
| Permítanme repetir el número.           | Sin acción del usuario                |
| Se repetirá el número.         | Voz pasiva, sin intervención del usuario |
| Escuche mientras se repite el número. | Voz pasiva                 |
| Escuche la repetición.            | Mejor elección                   |



 

Además, construya la salida para desplegar la información de clave al final de la frase, tal como se muestra en los ejemplos siguientes:



|             |                                      |                                            |
|-------------|--------------------------------------|--------------------------------------------|
| En lugar de: | "¿Es tres el siguiente dígito?"           | Use: "¿es el siguiente dígito tres?"            |
| En lugar de: | "Haga clic en Aceptar para empezar".                 | Use: "para comenzar, haga clic en Aceptar".                 |
| En lugar de: | "Di ' listo ' para completar el pedido". | Use: "para completar el pedido, por ejemplo," listo ". |
