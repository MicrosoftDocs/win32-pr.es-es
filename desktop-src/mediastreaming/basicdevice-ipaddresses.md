# <a name="basicdeviceipaddresses-property"></a>Propiedad BasicDevice. IpAddresses

Obtiene un vector de direcciones IP.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT get_IpAddresses(
  [out] IVector< HSTRING > **value
);
```



## <a name="property-value"></a>Valor de propiedad

Colección enumerable de punteros a direcciones IP.
