# <a name="windows-media-metafile-elements-reference"></a>Referencia de elementos de metarchivo de Windows Media

Esta sección contiene documentación de referencia para todos los elementos de metarchivo de Windows Media para las aplicaciones del lado cliente. Las entradas de referencia incluyen definiciones de los elementos, sus atributos y valores, y condiciones especiales relacionadas con cada elemento.

Los siguientes elementos de metarchivo son compatibles con las aplicaciones del lado cliente.



| Elemento                                           | Descripción                                                                                                            |
|---------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [Descripción](abstract-element.md)                  | Contiene texto que describe el elemento **ASX**, **banner** o **entry** asociado.                                 |
| [ASX](asx-element.md)                            | Define un archivo como un metarchivo de Windows Media.                                                                            |
| [FRENTE](author-element.md)                      | Contiene el nombre del autor de un clip multimedia o un metarchivo de Windows Media.                                           |
| [CINTA](banner-element.md)                      | Especifica una dirección URL para un gráfico que aparece en el panel de información de Media Player de Windows.                               |
| [BÁSICA](base-element.md)                          | Especifica una cadena que se anexa al principio de las direcciones URL enviadas a Windows Media Player.                                 |
| [Comentarios](comments.md)                          | Especifica la sintaxis XML para los comentarios ( <!--...--> ).                                                            |
| [SSOFTWARE](copyright-element.md)                | Contiene la información de copyright de un elemento **ASX** o **entry** .                                                |
| [DURATION](duration-element.md)                  | Especifica la cantidad de tiempo que Windows Media Player representa una secuencia.                                                    |
| [ENDMARKER](endmarker-element.md)                | Especifica un marcador en el que Windows Media Player detiene la representación de la secuencia.                                           |
| [MOVIMIENTOS](entry-element.md)                        | Especifica la ruta de acceso de un clip multimedia.                                                                                   |
| [ENTRYREF](entryref-element.md)                  | Vínculos a los elementos de **entrada** de un metarchivo de Windows Media externo que tiene una extensión. ASX,. Wax,. wvx o. WMX. |
| [CESO](event-element.md)                        | Define un comportamiento o una acción realizados por Windows Media Player cuando recibe un comando de script con la etiqueta de un evento.      |
| [LOGURL](logurl-element.md)                      | Indica a Windows Media Player que envíe los datos de registro a la dirección URL especificada.                                            |
| [MOREINFO](moreinfo-element.md)                  | Especifica una dirección URL para un sitio web, una dirección de correo electrónico o un comando de script asociado a un programa, un clip o un banner.               |
| [PARAM](param-element.md)                        | Especifica el valor de un parámetro personalizado asociado a una lista de reproducción o un elemento de una lista de reproducción.                      |
| [PREVIEWDURATION](previewduration-element.md)    | Especifica la cantidad de tiempo que se reproduce un clip en el modo de vista previa.                                                         |
| [CLI](ref-element.md)                            | Especifica una dirección URL para un fragmento de contenido multimedia digital.                                                                  |
| [PUEDEN](repeat-element.md)                      | Especifica el número de veces que Windows Media Player repetirá uno o varios elementos de **entrada** o **ENTRYREF** .             |
| [MÁSCARA (desusado)](skin-element--deprecated.md) | Especifica una dirección URL a una máscara incrustada.                                                                                   |
| [STARTMARKER](startmarker-element.md)            | Especifica un marcador en el que Windows Media Player inicia la representación de la secuencia.                                          |
| [STARTTIME](starttime-element.md)                | Especifica la hora a la que Windows Media Player comenzará a representar la secuencia.                                        |
| [Titulo](title-element--metafile.md)              | Contiene el título de un elemento **ASX** o **entry** .                                                                 |
