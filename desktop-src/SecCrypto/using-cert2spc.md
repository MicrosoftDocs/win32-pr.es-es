# <a name="using-cert2spc"></a>Uso de Cert2SPC

El siguiente comando encapsula un certificado [*X.509,*](../secgloss/x-gly.md) *MyCert*.cer, en un certificado PKCS \# 7 Software Publisher [*Certificate*](../secgloss/s-gly.md) (SPC) de prueba denominado *MyCert*.spc. El SPC creado solo se usará con fines de prueba. Un SPC que se usa para firmar realmente el código que se va a distribuir al público debe obtenerse de GTE, VeriSign, Inc. u otra entidad de certificación (CA) [*de*](../secgloss/c-gly.md) confianza.

**Cert2SPC** *MyCert:.cer* *  *MyCert:.spc**
