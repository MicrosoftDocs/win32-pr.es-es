# <a name="keepmodeless-dialog-style-bit"></a>Bit de estilo del cuadro de diálogo KeepModeless

Normalmente, cuando no se establece este bit y se crea un cuadro de diálogo mediante una acción, se destruyen todos los demás cuadros de diálogo (normalmente no modales). Si se establece este bit, los otros cuadros de diálogo permanecen activos cuando se crea este cuadro de diálogo.

## <a name="value"></a>Value



| Decimal | Hexadecimal | Constante                              |
|---------|-------------|---------------------------------------|
| 16      | 0x00000010  | **msidbDialogAttributesKeepModeless** |
