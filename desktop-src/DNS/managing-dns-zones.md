# <a name="managing-dns-zones"></a>Administración de zonas DNS

Una zona DNS es una base de datos de entradas RR que corresponde a parte del espacio de nombres jerárquicos de DNS. Las zonas DNS se usan para delinear qué servidores DNS son autoritativo para resolver consultas de resolución de nombres para una sección determinada de la jerarquía DNS.

## <a name="administration-steps"></a>Pasos de administración

El proveedor WMI de DNS permite la administración de zonas DNS desde el propio servidor DNS autoritativo o desde hosts remotos. Los pasos generales necesarios para administrar una zona mediante el proveedor WMI de DNS son:

-   Conexión al proveedor WMI de DNS
-   Obtención de una instancia de zona
-   Enumeración o modificación de una propiedad de zona o mediante un método
