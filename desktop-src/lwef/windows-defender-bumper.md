# <a name="windows-defender"></a>Windows Defender

Windows Defender ayuda a proteger los equipos contra elementos emergentes, rendimiento lento y amenazas de seguridad causados por spyware y otro software no deseado. Las aplicaciones pueden consultar información como el estado del equipo, información de malware específica y el historial de malware del cliente, y pueden invocar exámenes a petición.
