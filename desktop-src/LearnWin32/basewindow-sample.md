# <a name="basewindow-sample"></a>Ejemplo de BaseWindow

En esta aplicación de ejemplo se muestra cómo pasar datos de estado de aplicación en el mensaje de [**\_ NCCREATE de WM**](/windows/desktop/winmsg/wm-nccreate) .

## <a name="description"></a>Descripción

La aplicación de ejemplo BaseWindow es una variación en el [ejemplo de Windows Hola mundo](windows-hello-world-sample.md). Usa el mensaje [**de \_ NCCREATE de WM**](/windows/desktop/winmsg/wm-nccreate) para pasar datos de aplicación al procedimiento de ventana. Este ejemplo se describe en el tema [administrar el estado](managing-application-state-.md)de la aplicación.

## <a name="downloading-the-sample"></a>Descargar el ejemplo

Este ejemplo está disponible [aquí](https://github.com/microsoft/Windows-classic-samples/tree/master/Samples/Win7Samples/begin/LearnWin32/BaseWindow).
