# <a name="cdrawimagem_sourcerect-member"></a>Miembro CDrawImage::m \_ SourceRect

La **variable miembro m \_ SourceRect** especifica el rectángulo de origen para dibujar. El valor inicial es un rectángulo vacío, que indica que se debe dibujar toda la imagen.

## <a name="syntax"></a>Sintaxis


```C++
RECT m_SourceRect;
```
