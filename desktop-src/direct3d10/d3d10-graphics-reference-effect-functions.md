# <a name="effect-functions-direct3d-10"></a>Funciones de efecto (Direct3D 10)

Esta sección contiene información acerca de las siguientes funciones de efecto:



| Functions                                                                      | Descripción                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**D3D10CompileEffectFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10compileeffectfrommemory)           | Compilar un efecto.                                                  |
| [**D3D10CreateEffectFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createeffectfrommemory)             | Cree un efecto.                                                   |
| [**D3D10CreateEffectPoolFromMemory**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createeffectpoolfrommemory)     | Cree un grupo de efectos.                                              |
| [**D3D10CreateStateBlock**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10createstateblock)                         | Cree un bloque de estado.                                               |
| [**D3D10DisassembleEffect**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10disassembleeffect)                       | Desensamblar un efecto compilado en las instrucciones de ensamblado de sombreador.    |
| [**D3D10StateBlockMaskDifference**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdifference)         | Combine dos máscaras de bloques de estado con una XOR bit a bit.                   |
| [**D3D10StateBlockMaskDisableAll**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdisableall)         | Deshabilitar la captura de estado.                                            |
| [**D3D10StateBlockMaskDisableCapture**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskdisablecapture) | Deshabilitar la captura de estado con una máscara de bloque de estado.                    |
| [**D3D10StateBlockMaskEnableAll**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskenableall)           | Habilite una máscara de bloque de estado para capturar y aplicar todas las variables de estado. |
| [**D3D10StateBlockMaskEnableCapture**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskenablecapture)   | Habilitación de un intervalo de valores de estado en una máscara de bloque de estado.               |
| [**D3D10StateBlockMaskGetSetting**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskgetsetting)         | Obtiene un elemento en una máscara de bloque de estado.                               |
| [**D3D10StateBlockMaskIntersect**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskintersect)           | Combine dos máscaras de bloques de estado con una operación and bit a bit.                   |
| [**D3D10StateBlockMaskUnion**](/windows/desktop/api/D3D10Effect/nf-d3d10effect-d3d10stateblockmaskunion)                   | Combine dos máscaras de bloques de estado con una operación OR bit a bit.                    |
