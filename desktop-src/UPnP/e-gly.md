<dl> <dt>

<span id="upnp.e_1_gly"></span><span id="UPNP.E_1_GLY"></span>**ceso**
</dt> <dd>

Un evento es una notificación de uno o más cambios en las variables de estado expuestas por un servicio.

</dd> <dt>

<span id="upnp.e_2_gly"></span><span id="UPNP.E_2_GLY"></span>**variable con eventos**
</dt> <dd>

Una variable con eventos es una variable de estado para un servicio que, cuando se cambia, desencadena un evento. Los puntos de control que se suscriben al origen de eventos de una variable reciben la notificación del cambio. Una variable de estado sin eventos nunca se incluye en una notificación de eventos.

</dd> </dl>
