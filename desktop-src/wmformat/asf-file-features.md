# <a name="asf-file-features"></a>Características del archivo ASF

El propósito principal del SDK de Windows Media Format es proporcionar compatibilidad para encapsular datos multimedia digitales en archivos de formato de sistemas avanzados (ASF) y entregar los medios a una aplicación cliente. Los escenarios de entrega pueden variar ampliamente de una aplicación a otra, pero todos usan la estructura de archivos ASF entre la creación y la entrega. Los archivos ASF se ajustan a una estructura bien definida pero muy flexible. Para obtener más información sobre la estructura de archivos de ASF, vea [Información general sobre el formato ASF](overview-of-the-asf-format.md).

En la especificación de ASF se proporciona información detallada sobre los datos de un archivo ASF, que puede descargar desde el sitio [web de Microsoft](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).

El SDK Windows Media Format proporciona compatibilidad con las características de la especificación de ASF principalmente a través del perfil que se usa para crear un archivo. Para obtener más información sobre los perfiles, vea [Perfiles](profiles.md).
