# <a name="initial-auth-incoming-attribute"></a>Initial-auth-entrante (atributo)

Contiene información sobre una solicitud de autenticación entrante inicial de un cliente a este servidor. Este servidor envía esta solicitud al servidor de autenticación del dominio.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Autenticación inicial-entrante                       |
| Nombre para mostrar de LDAP | initialAuthIncoming                         |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.539                      |
| System-ID-GUID    | 52458023-ca6a-11d0-AFFF-0000f80367c1        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
