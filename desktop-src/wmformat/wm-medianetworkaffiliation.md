# <a name="wmmedianetworkaffiliation"></a>WM/MediaNetworkAffiliation

El atributo **WM/MediaNetworkAffiliation** contiene la afiliación de red del canal que difunde el contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaNetworkAffiliation

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Este atributo se utiliza para la televisión de difusión.
