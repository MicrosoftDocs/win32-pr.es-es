# <a name="drm_v1licenseacqurl"></a>\_V1LICENSEACQURL DRM

El **atributo \_ V1LicenseAcqURL de DRM** contiene la dirección de un sitio web en el que un usuario puede obtener una licencia de DRM versión 1.

## <a name="global-constant"></a>Constante global

g \_ wszWMDRM \_ V1LicenseAcqURL

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Este atributo se puede establecer mediante [**IWMDRMWriter:: SetDRMAttribute**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) y se puede recuperar con [**IWMDRMReader:: GetDRMProperty**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmreader-getdrmproperty).
