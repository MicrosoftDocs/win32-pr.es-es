# <a name="pen-end-cap"></a>Extremo final del lápiz

El atributo de extremo final especifica la forma de un lápiz geométrico: redondo, cuadrado o plano. En la ilustración siguiente se muestran las líneas paralelas dibujadas con cada tipo de extremo.

![Ilustración que muestra tres líneas horizontales, cada una con un tipo diferente de extremo final](images/cspen-04.png)

Los extremos de redondeo y de cierre cuadrado se extienden más allá de los puntos inicial y final de una línea dibujada con un lápiz geométrico; el extremo sin formato no lo es.
