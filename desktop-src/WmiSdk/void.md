# <a name="void"></a>HUECO

VOID es un tipo de datos que representa la ausencia de un valor. Utilice VOID en MOF solo para indicar un método que no tiene ningún valor devuelto.

``` syntax
void Method2 ([in] sint32 InParam);
```
