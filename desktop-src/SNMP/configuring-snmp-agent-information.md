# <a name="configuring-snmp-agent-information"></a>Configuración de la información del agente SNMP

La información del agente SNMP permite especificar comentarios sobre el usuario y la ubicación física del equipo e indicar los tipos de servicio que se deben notificar. Los tipos de servicio que se pueden notifican se basan en la configuración del equipo.
