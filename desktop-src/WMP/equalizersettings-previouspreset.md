# <a name="equalizersettingspreviouspreset"></a>EQUALIZERSETTINGS.previousPreset

El **método previousPreset** aplica el valor preestablecido del igualador anterior.

``` syntax
        elementID.previousPreset()
```

## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.

## <a name="remarks"></a>Comentarios

Si el valor preestablecido actual es el primero disponible, el último valor preestablecido se hace actual.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
