# <a name="xmcolor-operators"></a>Operadores XMCOLOR

La [**estructura XMCOLOR**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmcolor) expone los operadores siguientes.

> [!Note]  
> Los operadores enumerados aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                            | Descripción                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operador uint32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmcolor-operatoruint32_t)<br/> | Devuelve una instancia de `uint32_t` que contiene los componentes de la instancia en un formato `XMCOLOR` empaquetado. <br/> |
| [**operator =**](xmcolor-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMCOLOR` .<br/>                                        |
