# <a name="d3dx_float4_to_r10g10b10a2_unorm-function"></a>D3DX \_ FLOAT4 \_ a \_ R10G10B10A2 \_ función UNORM

Vuelve a empaquetar el XMFLOAT4 determinado en un formato de DXGI \_ \_ R10G10B10A2 \_ UNORM.

## <a name="syntax"></a>Sintaxis

``` syntax
UINT D3DX_FLOAT4_to_R10G10B10A2_UNORM(
   hlsl_precise XMFLOAT4 unpackedInput
);
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

*unpackedInput* 
</dt> <dd>

</dd> </dl>
