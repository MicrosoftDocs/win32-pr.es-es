## <a name="remarks"></a>Observaciones

**XAUDIO2 \_ El procesador** es un tipo definido como un valor de 32 bits.

<div class="alert"><b>Nota:</b>  Si se establecen varios bits de procesadores al llamar a <a href="/windows/desktop/api/xaudio2/nf-xaudio2-xaudio2create">XAudio2Create</a>, el sistema creará un subproceso de trabajo independiente para cada procesador.</div>



```
typedef UINT32 XAUDIO2_PROCESSOR;
```
