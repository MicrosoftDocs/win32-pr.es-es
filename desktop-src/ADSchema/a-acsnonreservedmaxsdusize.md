# <a name="acs-non-reserved-max-sdu-size-attribute"></a>Atributo ACS-non-Reserved-Max-SDU-size

El atributo **ACS-non-Reserved-Max-SDU-size** solo es para uso interno. Basado en RFC2814.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | ACS-no reservado-Max-SDU-size        |
| Nombre para mostrar de LDAP | aCSNonReservedMaxSDUSize             |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1320              |
| System-ID-GUID    | aec2cfe3-3b90-11d2-90cc-00c04fd91ab1 |
| Sintaxis            | [**Interval**](s-interval.md)       |
