# <a name="avencddsurrounddownmixlevel-property"></a>Propiedad AVEncDDSurroundDownMixLevel

Especifica el nivel de downmix envolvente, en decibelios (dB), en una secuencia de audio Dolby digital. Esta propiedad se aplica a los codificadores de audio Dolby digital.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncDDSurroundDownMixLevel**
