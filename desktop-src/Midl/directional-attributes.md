# <a name="directional-attributes"></a>Atributos direccionales

Aplique atributos direccionales a parámetros para indicar la dirección en la que se pasan los datos. Ambos atributos se pueden aplicar a un único parámetro, pero al hacerlo aumenta la sobrecarga asociada a una llamada.



| Atributo              | Uso                                                                 |
|------------------------|-----------------------------------------------------------------------|
| [**de**](in.md)       | El parámetro se pasa desde el autor de la llamada a la función llamada. |
| [**enuncia**](out-idl.md) | El parámetro se pasa de la función llamada al llamador. |
