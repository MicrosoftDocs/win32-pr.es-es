# <a name="coordinate-systems-and-transformations"></a>Sistemas de coordenadas y transformaciones

Windows GDI+ proporciona una transformación universal y una transformación de página para que pueda transformar (girar, escalar, trasladar, etc.) los elementos que dibuje. Las dos transformaciones también permiten trabajar en varios sistemas de coordenadas.

-   [Representación matricial de transformaciones](-gdiplus-matrix-representation-of-transformations-about.md)
-   [Transformaciones globales y locales](-gdiplus-global-and-local-transformations-about.md)
