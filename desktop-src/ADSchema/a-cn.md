# <a name="common-name-attribute"></a>Common-Name atributo)

Nombre que representa un objeto. Se utiliza para realizar búsquedas.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Common-Name                                 |
| Nombre para mostrar de LDAP | cn                                          |
| Tamaño              | El tamaño medio es de 64 bytes.                   |
| Actualizar privilegio  | Administrador de dominio                        |
| Frecuencia de actualización  | Solo cuando se crea un nuevo objeto.          |
| Attribute-Id      | 2.5.4.3                                     |
| System-ID-GUID    | bf96793f-0de6-11d0-a285-00aa003049e2        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
