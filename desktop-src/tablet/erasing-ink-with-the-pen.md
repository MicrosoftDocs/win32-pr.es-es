# <a name="erasing-ink-with-the-pen"></a>Borrar la entrada manuscrita con el lápiz

Además de habilitar la escritura a mano y el dibujo, puede compilar la aplicación para admitir el uso del lápiz de Tablet PC para borrar la entrada manuscrita. Use el objeto [**InkOverlay**](inkoverlay-class.md) para hacerlo.

En las secciones siguientes se describe cómo borrar la entrada manuscrita con el lápiz de Tablet PC.

-   [Borrar mediante el objeto InkOverlay](erasing-by-using-the-inkoverlay-object.md)
-   [Borrar con el lápiz](erasing-by-using-the-pen.md)
