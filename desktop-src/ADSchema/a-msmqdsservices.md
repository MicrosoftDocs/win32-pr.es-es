# <a name="msmq-ds-services-attribute"></a>Atributo MSMQ-DS-Services

Indica si el MSMQ instalado en este equipo proporciona servicios de MSMQ DS.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Servicios de MSMQ-DS                     |
| Nombre para mostrar de LDAP | mSMQDsServices                       |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1228              |
| System-ID-GUID    | 2df90d78-009f-11d2-aa4c-00c04fd7d83a |
| Sintaxis            | [**Booleano**](s-boolean.md)         |
