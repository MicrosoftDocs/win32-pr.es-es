# <a name="wmcategory"></a>WM/categoría

El atributo **WM/categoría** contiene la categoría del contenido. No hay especificaciones para las categorías. Se debe animar a los usuarios a que escriban sus propias categorías para organizar el contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMCategory

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="example"></a>Ejemplo



| Tipo de archivo | Valor de ejemplo   |
|-----------|-----------------|
| Audio     | "Música en marcha" |
| Vídeo     | "Vídeos domésticos"   |
