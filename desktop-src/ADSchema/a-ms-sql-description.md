# <a name="ms-sql-description-attribute"></a>Atributo MS-SQL-Description

La descripción de la base de datos de SQL Server.



| Entrada | Value |
|-------------------|--------------------------------------------------|
| CN                | MS-SQL-Descripción                               |
| Nombre para mostrar de LDAP | mS-SQL-Descripción                               |
| Tamaño              | \-                                               |
| Actualizar privilegio  | Administrador de dominio                             |
| Frecuencia de actualización  | Cuando el usuario decide cambiar la descripción. |
| Attribute-Id      | 1.2.840.113556.1.4.1390                          |
| System-ID-GUID    | 8386603c-ccef-11d2-9993-0000f87a57d4             |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md)      |
