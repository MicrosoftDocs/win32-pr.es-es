# <a name="durationtext"></a>DURATIONTEXT

Se trata de un elemento de **texto** predefinido con los siguientes valores predeterminados.

``` syntax
value="wmpprop:player.currentMedia.DurationString"
tabstop="true"
justification="right"
```

## <a name="remarks"></a>Observaciones

Esto creará un elemento de **texto** que mostrará la duración del contenido multimedia que se está reproduciendo. Todas las propiedades de este elemento de **texto** se pueden invalidar mediante su especificación explícita.
