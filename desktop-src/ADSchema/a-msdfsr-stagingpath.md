# <a name="ms-dfsr-stagingpath-attribute"></a>atributo MS-DFSR-StagingPath

Contiene la ruta de acceso completa del directorio de almacenamiento provisional.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-DFSR-StagingPath                         |
| Nombre para mostrar de LDAP | msDFSR-StagingPath                          |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.5                   |
| System-ID-GUID    | 86b9a69e-f0a6-405d-99bb-77d977992c2a        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
