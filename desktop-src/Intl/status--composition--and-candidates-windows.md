composition, and candidates windows form the user interface for the IME.</span></span> <span data-ttu-id="5388a-105">La ventana de estado indica que el IME está abierto y proporciona al usuario los medios para establecer los modos de conversión.</span><span class="sxs-lookup"><span data-stu-id="5388a-105">The status window indicates that the IME is open and provides the user with the means to class="sxs-lookup"><span data-stu-id="5388a-107">The candidates window appears in conjunction with the composition window.</span></span> <span data-ttu-id="5388a-108">Contiene una lista de "candidatos" (caracteres alternativos) para el carácter o los caracteres seleccionados en la ventana de composición.</span><span class="sxs-lookup"><span data-stu-id="5388a-108">It contains a list of finalized and the window is closed.</span></span>

<span data-ttu-id="5388a-111">El IME envía los caracteres compuestos a la aplicación compatible con el IME en forma de mensajes de resultados/GCS de la composición del IME de [**WM \_ \_**](wm-ime-char.md) o de [**WM \_ \_ IME**](wm-ime-composition.md) \_ .</span><span class="sxs-lookup"><span data-stu-id="5388a-111">The IME sends the composed characters to the IME-aware application in the form of [**WM\_IME\_CHAR**](wm-ime-char.md) or [**WM\_IME\_COMPOSITION**](wm-ime-composition.md)/GCS\_RESULT messages.</span></span> <span data-ttu-id="5388a-112">Si la aplicación no procesa estos mensajes, la función [**DefWindowProc**](/windows/desktop/api/winuser/nf-winuser-defwindowproca) los convierte en uno o varios mensajes de tipo [**\_ Char de WM**](../inputdev/wm-char.md) .</span><span class="sxs-lookup"><span data-stu-id="5388a-112">If the application does not process these messages, the [**DefWindowProc**](/windows/desktop/api/winuser/nf-winuser-defwindowproca) function translates them into one or more [**WM\_CHAR**](../inputdev/wm-char.md) messages.</span></span>

<span data-ttu-id="5388a-113">De forma predeterminada, el sistema operativo crea y administra automáticamente ventanas de estado, composición y candidatos para los requisitos de entrada de texto.</span><span class="sxs-lookup"><span data-stu-id="5388a-113">By default, the operating system automatically creates and manages status, composition, and candidates windows for text input requirements.</span></span> <span data-ttu-id="5388a-114">Para muchas aplicaciones, este procesamiento predeterminado es suficiente.</span><span class="sxs-lookup"><span data-stu-id="5388a-114">For many applications, this default processing is sufficient.</span></span> <span data-ttu-id="5388a-115">Estas aplicaciones confían por completo en el sistema operativo para la compatibilidad con IME y se dice que no son compatibles con el IME porque no son conscientes de las muchas tareas que el sistema operativo lleva a cabo para administrar las ventanas del IME.</span><span class="sxs-lookup"><span data-stu-id="5388a-115">These applications rely entirely on the operating system for IME support and are said to be "IME-unaware" because they are unaware of the many tasks the operating system carries out to manage the IME windows.</span></span>

<span data-ttu-id="5388a-116">Por otro lado, una aplicación compatible con IME participa en la creación y la administración de las ventanas de IME.</span><span class="sxs-lookup"><span data-stu-id="5388a-116">An IME-aware application, on the other hand, participates in the creation and management of IME windows.</span></span> <span data-ttu-id="5388a-117">Dichas aplicaciones controlan la operación, la posición y la apariencia de las ventanas predeterminadas mediante el envío de mensajes a estas ventanas y la interceptación y el procesamiento de los mensajes de las ventanas.</span><span class="sxs-lookup"><span data-stu-id="5388a-117">Such applications control the operation, position, and appearance of the default windows by sending messages to these windows and by intercepting and processing messages from the windows.</span></span> <span data-ttu-id="5388a-118">En algunos casos, las aplicaciones crean sus propias ventanas de IME y proporcionan un procesamiento completo para sus ventanas de estado personalizado, composición y candidatos.</span><span class="sxs-lookup"><span data-stu-id="5388a-118">In some cases, applications create their own IME windows and provide complete processing for their custom status, composition, and candidates windows.</span></span>
