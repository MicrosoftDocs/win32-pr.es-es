# <a name="recordcleardata-method"></a>Método record. ClearData

El método **ClearData** del objeto [**Record**](record-object.md) borra los datos de todos los campos y los establece en NULL. Se liberan todos los objetos almacenados en los campos.

## <a name="syntax"></a>Sintaxis


```JScript
Record.ClearData()
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
