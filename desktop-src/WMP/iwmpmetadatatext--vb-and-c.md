# <a name="iwmpmetadatatext-vb-and-c-interface"></a>Interfaz IWMPMetadataText (VB y C#)

Proporciona propiedades para obtener información sobre los atributos de metadatos de texto complejos.

La interfaz **IWMPMetadataText** expone las siguientes propiedades.



| Propiedad                                                                         | Descripción                              |
|----------------------------------------------------------------------------------|------------------------------------------|
| [Descripción](wmplibiwmpmetadatatext-iwmpmetadatatext-description--vb-and-c.md) | Obtiene una descripción del texto de metadatos. |
| [text](wmplibiwmpmetadatatext-iwmpmetadatatext-text--vb-and-c.md)               | Obtiene el texto de los metadatos.                  |



 

Obtenga una interfaz **IWMPMetadataText** pasando un nombre de atributo como **WM/UserWebURL** al método siguiente y convirtiendo el objeto que se devuelve.



|                                        |                                                                                  |
|----------------------------------------|----------------------------------------------------------------------------------|
| Interfaz                              | Propiedad                                                                         |
| [IWMPMedia3](iwmpmedia3--vb-and-c.md) | [getItemInfoByType](wmplibiwmpmedia3-iwmpmedia3-getiteminfobytype--vb-and-c.md) |



 

## <a name="members"></a>Miembros

La interfaz **IWMPMetadataText (VB y C#)** no define ningún miembro.
