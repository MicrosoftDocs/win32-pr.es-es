# <a name="netboot-guid-attribute"></a>Netboot: atributo GUID

Arranque sin disco: el GUID en el panel de un equipo. Corresponde a la dirección MAC de la tarjeta de red del equipo.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | Netboot-GUID                                          |
| Nombre para mostrar de LDAP | netbootGUID                                           |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | El sistema establece este valor.                      |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.359                                |
| System-ID-GUID    | 3e978921-8c01-11d0-afda-00c04fd930c9                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
