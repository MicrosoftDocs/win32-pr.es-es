# <a name="geterror-method-of-the-msvm_virtualsystemreferencepointexportjob-class"></a>Método GetError de la \_ clase VirtualSystemReferencePointExportJob de MSVM

Recupera el error.

## <a name="syntax"></a>Sintaxis


```mof
uint32 GetError(
  [out] string Error
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*Error* \[ de enuncia\]
</dt> <dd>

Error recuperado.

</dd> </dl>
