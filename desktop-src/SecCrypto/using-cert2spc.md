# <a name="using-cert2spc"></a>Usar Cert2SPC

El comando siguiente incluye un certificado [*X. 509*](../secgloss/x-gly.md) , *mycert*. cer, en un certificado de editor de \# [*software*](../secgloss/s-gly.md) PKCS 7 (SPC) de prueba, denominado *mycert*. SPC. El SPC creado se va a usar solo con fines de prueba. Un SPC que se usa para firmar realmente el código que se va a distribuir al público debe obtenerse de GTE, VeriSign, Inc. u otra [*entidad de certificación*](../secgloss/c-gly.md) (CA) de confianza.
