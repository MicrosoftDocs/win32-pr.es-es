# <a name="avencmpvgenerateheaderseqdispext-property"></a>Propiedad AVEncMPVGenerateHeaderSeqDispExt

Especifica si el codificador genera encabezados de extensión de presentación de secuencias. Esta propiedad se aplica a los codificadores de vídeo MPEG.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncMPVGenerateHeaderSeqDispExt**

## <a name="remarks"></a>Observaciones

Si el valor es **Variant \_ true**, el codificador genera encabezados de extensión de presentación de secuencias.
