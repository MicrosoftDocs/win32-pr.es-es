# <a name="mssfu-30-order-number-attribute"></a>msSFU-30-Order-número atributo

Contiene un valor que usa NIS para comprobar si el mapa ha cambiado. Cada vez que los datos almacenados en el objeto [**msSFU-30-Domain-info**](c-mssfu30domaininfo.md) cambian, se incrementa este valor. Este valor se usa para realizar el seguimiento de los cambios de datos entre llamadas a **ypxfer** .



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | msSFU-30-número de orden                       |
| Nombre para mostrar de LDAP | msSFU30OrderNumber                          |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.308                 |
| System-ID-GUID    | 02625f05-d1ee-4f9f-b366-55266becb95c        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
