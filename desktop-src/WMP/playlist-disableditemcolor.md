# <a name="playlistdisableditemcolor"></a>Lista de reproducción. disabledItemColor

El atributo **disabledItemColor** especifica o recupera el color de una pista de CD deshabilitada o de contenido en línea cuando está sin conexión.

``` syntax
        elementID.disabledItemColor
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de lectura/escritura que contiene cualquier valor de color de Microsoft Internet Explorer. Tiene un valor predeterminado de "graytext".

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
