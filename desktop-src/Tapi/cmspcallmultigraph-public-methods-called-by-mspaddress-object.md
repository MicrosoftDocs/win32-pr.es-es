# <a name="cmspcallmultigraph-public-methods-called-by-mspaddress-object"></a>Métodos públicos CMSPCallMultiGraph a los que llama el objeto MSPAddress



| Métodos públicos CMSPCallMultiGraph               | Descripción                                                                                                                                |
|-------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| [**Init**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-init)         | Lo llama el objeto de dirección MSP (en el [**método CreateMSPCall**](/windows/desktop/api/msp/nf-msp-itmspaddress-createmspcall)) para inicializar el objeto de llamada MSP.    |
| [**Apagado**](/windows/desktop/api/Mspcall/nf-mspcall-cmspcallmultigraph-shutdown) | Lo llama el objeto de dirección MSP (en el método [**ShutdownMSPCall**](/windows/desktop/api/msp/nf-msp-itmspaddress-shutdownmspcall)) para apagar el objeto de llamada MSP. |
