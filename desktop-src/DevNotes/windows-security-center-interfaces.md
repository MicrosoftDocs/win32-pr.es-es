# <a name="windows-security-center-interfaces"></a>Seguridad de Windows Interfaces centrales

Las interfaces siguientes están disponibles para su uso con Seguridad de Windows Center.

## <a name="in-this-section"></a>En esta sección



| Tema                                                 | Descripción                                                                                                                      |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**IWscProduct**](/windows/desktop/api/Iwscapi/nn-iwscapi-iwscproduct)<br/>         | Proporciona métodos para obtener información del producto para que un proveedor individual interactúe con Seguridad de Windows Center.<br/> |
| [**IWSCProductList**](/windows/desktop/api/iwscapi/nn-iwscapi-iwscproductlist)<br/> | Proporciona métodos para recopilar información del producto para el tipo seleccionado de proveedores instalados en el equipo.<br/>         |
