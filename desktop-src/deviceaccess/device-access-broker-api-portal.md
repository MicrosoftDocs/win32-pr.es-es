# <a name="device-access-api"></a>API de acceso a dispositivos

## <a name="purpose"></a>Propósito

Puede usar la API de acceso de dispositivos para escribir aplicaciones de dispositivos de la tienda Windows para dispositivos especializados que usan controladores personalizados. La API proporciona métodos para enviar códigos de control para comunicarse con el controlador personalizado del dispositivo.

## <a name="developer-audience"></a>Audiencia de desarrolladores

La API de acceso a dispositivos está diseñada para los desarrolladores de fabricantes de hardware independientes (IHV) y OEM que están familiarizados con C++ y el modelo de objetos componentes (COM).
