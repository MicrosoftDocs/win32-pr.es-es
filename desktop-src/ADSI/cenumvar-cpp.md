# <a name="cenumvarcpp"></a>CENUMVAR. CPP

En el componente de proveedor de ejemplo, la implementación base para las clases derivadas de xxxEnumVariant se puede encontrar en cenumvar. cpp. En la tabla siguiente se enumeran los métodos asociados.



| Método                                          | Descripción                                                                           |
|-------------------------------------------------|---------------------------------------------------------------------------------------|
| **CSampleDSEnumVariant::CSampleDSEnumVariant**  | Constructor estándar.                                                                 |
| **CSampleDSEnumVariant:: ~ CSampleDSEnumVariant** | Destructor estándar.                                                                  |
| **CSampleDSEnumVariant:: QueryInterface**        | Implementación de [**IUnknown:: QueryInterface**](/windows/win32/api/unknwn/nf-unknwn-iunknown-queryinterface(q)) estándar. |
| **CSampleDSEnumVariant:: AddRef**                | Implementación [**IUnknown:: AddRef**](/windows/win32/api/unknwn/nf-unknwn-iunknown-addref) estándar.                 |
| **CSampleDSEnumVariant:: Release**               | Implementación estándar de [**IUnknown:: Release**](/windows/win32/api/unknwn/nf-unknwn-iunknown-release) .               |
| **CSampleDSEnumVariant:: Skip**                  | Implementación estándar de **IEnumXXXX:: Skip** .                                          |
| **CSampleDSEnumVariant:: RESET**                 | Implementación estándar de **IEnumXXXX:: RESET** .                                         |
| **CSampleDSEnumVariant:: Clone**                 | Implementación estándar de **IEnumXXXX:: Clone** .                                         |
