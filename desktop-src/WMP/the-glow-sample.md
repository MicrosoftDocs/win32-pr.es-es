# <a name="the-glow-sample"></a>El ejemplo de resplandor

En esta sección se describe cómo crear una visualización sencilla denominada Glow. Se usan tres valores preestablecidos: rojo, verde y azul. Se ejecutó el Asistente para visualización personalizada y se eligió el nombre Glow.

El resplandor simplemente cambia el color en el tiempo a la música. El resplandor parpadea porque la potencia de audio de la forma de onda atraviesa a veces cero en el momento de la instantánea. Este ejemplo se elige porque es uno de los ejemplos más sencillos posibles.

En las secciones siguientes se le guiará:

-   [Implementación de render](implementing-render-sample.md)
-   [Cambiar valores preestablecidos](changing-presets.md)
