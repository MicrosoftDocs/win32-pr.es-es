# <a name="programmenufolder-property"></a>Propiedad ProgramMenuFolder

El instalador establece la propiedad **ProgramMenuFolder** en la ruta de acceso completa de la carpeta de menú del programa para el usuario actual. Si existe un perfil "todos los usuarios" y se establece la propiedad [**AllUsers**](allusers.md) , esta propiedad se establece en la carpeta del perfil "todos los usuarios".
