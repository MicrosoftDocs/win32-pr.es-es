# <a name="iupdate2-methods"></a>Métodos IUpdate2

La [**interfaz IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2) define el método siguiente.



| Método                                      | Descripción                                                                                                            |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**CopyToCache**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-copytocache) | Copia los archivos de una actualización desde una ubicación de origen especificada a la caché de descarga Windows update Agent (WUA) interna. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz:

-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
