# <a name="winbio_orientation-constants"></a>Constantes DE ORIENTACIÓN DE WINBIO \_

Las constantes siguientes especifican las posibles orientaciones de la cámara que el componente del sensor especifica como obligatorias.



| Constante o valor                                                                                                                                                                                                                                                           | Descripción                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------|
| <span id="WINBIO_ORIENTATION_UNSPECIFIED"></span><span id="winbio_orientation_unspecified"></span><dl> <dt>**WINBIO \_ ORIENTATION \_ UNSPECIFIED**</dt> <dt>0</dt> </dl> | No se especifica una orientación obligatoria para la cámara.<br/> |
| <span id="WINBIO_ORIENTATION_LANDSCAPE"></span><span id="winbio_orientation_landscape"></span><dl> <dt>**WINBIO \_ ORIENTACIÓN \_ HORIZONTAL**</dt> <dt>1</dt> </dl>       | La orientación horizontal es necesaria para la cámara.<br/>    |
| <span id="WINBIO_ORIENTATION_PORTRAIT"></span><span id="winbio_orientation_portrait"></span><dl> <dt>**WINBIO \_ ORIENTACIÓN \_ VERTICAL**</dt> <dt>2</dt> </dl>          | La orientación vertical es necesaria para la cámara.<br/>     |
| <span id="WINBIO_ORIENTATION_ANY"></span><span id="winbio_orientation_any"></span><dl> <dt>**WINBIO \_ ORIENTATION \_ ANY**</dt> <dt>3</dt> </dl>                         | Se permite cualquier orientación para la cámara.<br/>             |
