# <a name="control-types-and-their-supported-control-patterns"></a><span data-ttu-id="38661-111">Tipos de control y sus patrones de control admitidos</span><span class="sxs-lookup"><span data-stu-id="38661-111">Control Types and Their Supported Control Patterns</span></span>

<span data-ttu-id="38661-112">Las aplicaciones cliente obtienen información sobre un control mediante la recuperación de propiedades generales a través de la interfaz [**IUIAutomationElement**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement) y la recuperación de propiedades específicas del tipo a través de las interfaces de patrón de control que admite el control.</span><span class="sxs-lookup"><span data-stu-id="38661-112">Client applications get information about a control by retrieving general properties through the [**IUIAutomationElement**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationelement) interface, and by retrieving type-specific properties through the control pattern interfaces that the control supports.</span></span> <span data-ttu-id="38661-113">En este tema se enumeran los tipos de control de automatización de la interfaz de usuario de Microsoft y se identifican los patrones de control que admite cada tipo.</span><span class="sxs-lookup"><span data-stu-id="38661-113">This topic lists the Microsoft UI Automation control types and identifies the control patterns that each type supports.</span></span>

<span data-ttu-id="38661-114">En la siguiente tabla se organizan los patrones de control en las siguientes categorías:</span><span class="sxs-lookup"><span data-stu-id="38661-114">The following table organizes the control patterns into the following categories:</span></span>

-   <span data-ttu-id="38661-115">Debe ser compatible con.</span><span class="sxs-lookup"><span data-stu-id="38661-115">Must support.</span></span> <span data-ttu-id="38661-116">El control debe admitir este patrón de control.</span><span class="sxs-lookup"><span data-stu-id="38661-116">The control must support this control pattern.</span></span>
-   <span data-ttu-id="38661-117">Compatibilidad condicional.</span><span class="sxs-lookup"><span data-stu-id="38661-117">Conditional support.</span></span> <span data-ttu-id="38661-118">El control puede admitir este patrón de control según el estado del control.</span><span class="sxs-lookup"><span data-stu-id="38661-118">The control may support this control pattern depending on the state of the control.</span></span>
-   <span data-ttu-id="38661-119">No admite.</span><span class="sxs-lookup"><span data-stu-id="38661-119">Does not support.</span></span> <span data-ttu-id="38661-120">El control no admite este patrón de control; sin embargo, una versión personalizada del control puede admitir este patrón de control.</span><span class="sxs-lookup"><span data-stu-id="38661-120">The control does not support this control pattern; however, a customized version of the control may support this control pattern.</span></span>

<span data-ttu-id="38661-121">Algunos controles tienen compatibilidad condicional con varios patrones de control, en función de la funcionalidad del control.</span><span class="sxs-lookup"><span data-stu-id="38661-121">Some controls have conditional support for several control patterns, depending on the functionality of the control.</span></span> <span data-ttu-id="38661-122">Por ejemplo, el control de elemento de menú tiene compatibilidad condicional con el patrón de control [Invoke](uiauto-implementinginvoke.md), [ExpandCollapse](uiauto-implementingexpandcollapse.md), [Toggle](uiauto-implementingtoggle.md)o [SelectionItem](uiauto-implementingselectionitem.md) , en función de su función en el control de menú.</span><span class="sxs-lookup"><span data-stu-id="38661-122">For example, the menu item control has conditional support for the [Invoke](uiauto-implementinginvoke.md), [ExpandCollapse](uiauto-implementingexpandcollapse.md), [Toggle](uiauto-implementingtoggle.md), or [SelectionItem](uiauto-implementingselectionitem.md) control pattern, depending on its function in the menu control.</span></span>
