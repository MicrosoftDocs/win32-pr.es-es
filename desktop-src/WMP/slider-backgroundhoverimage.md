# <a name="sliderbackgroundhoverimage"></a>SLIDER.backgroundHoverImage

El **atributo backgroundHoverImage** especifica o recupera la imagen de fondo del control deslizante que aparece al mantener el puntero sobre él con el mouse.

``` syntax
        elementID.backgroundHoverImage
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una cadena de **lectura** y escritura que contiene el nombre de un archivo de imagen.

## <a name="remarks"></a>Comentarios

Este atributo es opcional. Si no se especifica, se usará **backgroundImage.**

Los formatos admitidos son BMP, JPG, PNG y GIF (sin incluir GIF animados).
