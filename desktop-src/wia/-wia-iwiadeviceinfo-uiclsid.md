# <a name="deviceinfouiclsid-property"></a>Propiedad DeviceInfo.UIClsid

Recupera el identificador de clase de la interfaz de usuario proporcionada por el fabricante para este Windows de hardware de adquisición de imágenes (WIA). El valor es una representación de cadena de un GUID.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = DeviceInfo.UIClsid
```



## <a name="property-value"></a>Valor de propiedad

Cadena para recibir el identificador de clase.
