# <a name="testing-a-monitor"></a>Probar un monitor

Una vez instalado un monitor, puede probarlo con datos en directo de las operaciones de red, o puede probar un conjunto conocido de eventos capturando los datos con Monitor de red, guardando el archivo de captura y transmitiendo los datos localmente. (Para obtener más información sobre cómo transmitir un archivo de captura en una red, consulte Monitor de red ayuda en línea).
