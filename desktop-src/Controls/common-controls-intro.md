# <a name="about-common-controls"></a>Acerca de los controles comunes

En este tema se de abordan los controles comunes, un conjunto de ventanas que implementa la biblioteca de controles común, Comctl32.dll, que es un archivo DLL incluido con el sistema operativo Windows común. Al igual que otras ventanas de control, un control común es una ventana secundaria que una aplicación usa junto con otra ventana para habilitar la interacción con el usuario.
