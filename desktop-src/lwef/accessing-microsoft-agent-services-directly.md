# <a name="accessing-microsoft-agent-services-directly"></a>Acceso directo a los servicios de Microsoft Agent

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Si usa C, C++ o Java, puede tener acceso al servidor de Microsoft Agent directamente mediante sus interfaces ActiveX (OLE). Para obtener más información acerca de estas interfaces, consulte [Programming the Microsoft Agent Server Interface](programming-the-microsoft-agent-server-interface.md).
