# <a name="prevbutton"></a>PREVBUTTON

Se trata de un **botón** predefinido con los siguientes valores predeterminados.

``` syntax
onclick="jscript:player.controls.previous()"
upToolTip="Previous"
cursor="system"
enabled="wmpenabled:player.controls.previous"
```

## <a name="remarks"></a>Observaciones

Todas las propiedades de este botón se pueden invalidar si se especifican explícitamente.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
