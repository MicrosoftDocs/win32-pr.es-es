# <a name="eaphost-authenticator-method-enumerations"></a>Enumeraciones del método de autenticador de EAPHost

Las enumeraciones del método de autenticador de EAPHost son las siguientes.



| Enumeración                                                                                      | Descripción                                                                                                                                        |
|--------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**tiempo de espera de envío del \_ autenticador de EAP \_ \_**](/windows/desktop/api/eapAuthenticatorTypes/ne-eapauthenticatortypes-eap_authenticator_send_timeout)                      | Indica al sistema de información (IS) la cantidad de tiempo que hay que esperar a que el usuario escriba datos después de enviar el paquete. El valor de tiempo de espera se puede establecer en none. |
| [**\_acción de \_ respuesta del autenticador del método EAP \_ \_**](/windows/desktop/api/EapAuthenticatorActionDefine/ne-eapauthenticatoractiondefine-eap_method_authenticator_response_action) | Define las instrucciones de respuesta enviadas por el autenticador al método suplicante o EAP del mismo nivel.                                                  |
