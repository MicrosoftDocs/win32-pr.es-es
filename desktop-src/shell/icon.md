# <a name="customizing-icons"></a>Personalizar iconos

Windows proporciona iconos predeterminados para todos los elementos mostrados en el escritorio y en el explorador de Windows. A los dispositivos de almacenamiento masivo, como las unidades de disco, también se les asignan iconos predeterminados. Sin embargo, estos iconos suelen proporcionar poca información al usuario sobre el contenido del archivo o el programa que está asociado a él. Puede asignar un icono personalizado a una carpeta del sistema de archivos mediante la creación de un archivo de [Desktop.ini](how-to-customize-folders-with-desktop-ini.md) . Consulte los siguientes documentos para obtener información sobre cómo usar el registro para asociar iconos personalizados con tipos de archivo y Letras de unidad.

-   [Asignar un icono personalizado a un tipo de archivo](how-to-assign-a-custom-icon-to-a-file-type.md)
-   [Asignar un icono y una etiqueta personalizados a una letra de unidad](how-to-assign-a-custom-icon-and-label-to-a-drive-letter.md)
