# <a name="wic-structures"></a>Estructuras de WIC

Esta sección contiene información sobre las estructuras de Windows Imaging Component (WIC).

## <a name="in-this-section"></a>En esta sección



| Tema                                                                          | Descripción                                                                                                                   |
|--------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**WICBitmapPattern**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmappattern)<br/>             | Contiene miembros que identifican un patrón dentro de un archivo de imagen que se puede usar para identificar un formato determinado.<br/>   |
| [**WICBitmapPlane**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplane)<br/>                            | Especifica el formato de píxel, el búfer, el intervalo y el tamaño de un plano de componente para un formato de píxel plano.<br/>                |
| [**WICBitmapPlaneDescription**](/windows/desktop/api/Wincodec/ns-wincodec-wicbitmapplanedescription)<br/>      | Especifica el formato de píxel y el tamaño de un plano de componente.<br/>                                                          |
| [**WICDdsFormatInfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsformatinfo)<br/>                        | Especifica el [**\_ formato de DXGI**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) y la información de bloque de un formato DDS.<br/>                  |
| [**WICDdsParameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicddsparameters)<br/>                        | Especifica la dimensión de imagen DDS, el [**\_ formato de DXGI**](/windows/desktop/api/dxgiformat/ne-dxgiformat-dxgi_format) y el modo alfa de datos contenidos.<br/>  |
| [**WICImageParameters**](/windows/desktop/api/Wincodec/ns-wincodec-wicimageparameters)<br/>                    | Esto define los parámetros que se pueden usar para invalidar los parámetros predeterminados que se suelen usar al codificar una imagen. <br/> |
| [**WICJpegFrameHeader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegframeheader)<br/>                    | Representa un encabezado de marco JPEG.<br/>                                                                                    |
| [**WICJpegScanHeader**](/windows/desktop/api/wincodec/ns-wincodec-wicjpegscanheader)<br/>                      | Representa un encabezado de marco JPEG.<br/>                                                                                    |
| [**WICMetadataHeader**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadataheader)<br/>           | Representa el encabezado de metadatos.<br/>                                                                                        |
| [**WICMetadataPattern**](/windows/desktop/api/Wincodecsdk/ns-wincodecsdk-wicmetadatapattern)<br/>         | Representa un modelo de metadatos.<br/>                                                                                     |
| [**WICRawCapabilitiesInfo**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawcapabilitiesinfo)<br/> | Define las capacidades de códec sin formato.<br/>                                                                                     |
| [**WICRawToneCurve**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurve)<br/>               | Representa una curva de tono de imagen sin formato.<br/>                                                                                 |
| [**WICRawToneCurvePoint**](/windows/desktop/api/Wincodec/ns-wincodec-wicrawtonecurvepoint)<br/>     | Representa un punto de curva de tono de imagen sin formato.<br/>                                                                           |
| [**WICRect**](/windows/desktop/api/Wincodec/ns-wincodec-wicrect)<br/>                               | Representa un rectángulo para la API de WIC.<br/>                                                                                |
