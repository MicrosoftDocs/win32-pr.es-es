# <a name="bluetooth-and-getsockname"></a>Bluetooth y getsockname

Bluetooth usa la función [**getsockname**](/windows/desktop/api/winsock/nf-winsock-getsockname) para recuperar la dirección del dispositivo del servidor y el número de puerto asignados a un socket mediante una llamada anterior a la [**función bind.**](/windows/desktop/api/winsock/nf-winsock-bind)

Para obtener más información, [vea Windows Sockets](/windows/desktop/WinSock/windows-sockets-start-page-2).
