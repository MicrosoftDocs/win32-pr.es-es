# <a name="wmpeakbitrate"></a>WM/PeakBitrate

El **atributo WM/PeakBitrate** contiene la velocidad de bits máxima de una secuencia de velocidad de bits variable (VBR) en bits por segundo.

## <a name="global-constant"></a>Constante global

g \_ wszWMPeakBitrate

## <a name="data-type"></a>Tipo de datos

**DWORD \_ DE TIPO \_ WMT**
