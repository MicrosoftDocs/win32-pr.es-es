# <a name="from-entry-attribute"></a>From-Entry atributo

Se trata de un atributo construido que es **TRUE** si el objeto se puede escribir y **FALSE** si es de solo lectura, por ejemplo, una instancia de réplica de GC.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | From-Entry                           |
| Ldap-Display-Name | fromEntry                            |
| Size              | \-                                   |
| Actualizar privilegios  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.910               |
| System-Id-Guid    | 9a7ad949-ca53-11d1-bbd0-0080c76670c0 |
| Syntax            | [**Boolean**](s-boolean.md)         |
