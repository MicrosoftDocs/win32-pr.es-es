# <a name="ms-wmi-intflags4-attribute"></a>atributo MS-WMI-intFlags4

El atributo **MS-WMI-intFlags4** está reservado para uso interno.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-WMI-intFlags4                     |
| Nombre para mostrar de LDAP | msWMI-intFlags4                      |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1681              |
| System-ID-GUID    | bd74a7ac-c493-4c9c-bdfa-5c7b119ca6b2 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
