# <a name="creating-application-summary-property"></a>Creación de la propiedad Resumen de la aplicación

La **propiedad Creating Application Summary (Crear resumen** de aplicación) transmite qué aplicación creó la base de datos del instalador. En general, el valor de esta propiedad de resumen es el nombre del software que se usa para crear esta base de datos.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versión<br/> | Windows Instalador 5.0 en Windows Server 2012, Windows 8, Windows Server 2008 R2 o Windows 7. Windows Instalador 4.0 o Windows Installer 4.5 en Windows Server 2008 o Windows Vista. Windows Instalador en Windows Server 2003 o Windows XP<br/> |
