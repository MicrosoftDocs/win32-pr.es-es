# <a name="cb_getdroppedstate-message"></a>\_Mensaje GETDROPPEDSTATE CB

Determina si el cuadro de lista de un cuadro combinado está desplegado.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>

No se utiliza; debe ser cero.

</dd> <dt>

*lParam* 
</dt> <dd>

No se utiliza; debe ser cero.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Si el cuadro de lista está visible, el valor devuelto es **true**; de lo contrario, es **false**.
