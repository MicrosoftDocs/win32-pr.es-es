# <a name="sysmon-classes"></a>Clases SYSMON

Puede usar las siguientes clases para configurar el control Del Monitor de sistema (SYSMON).



| Nombre de la clase                             | Descripción                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**CounterItem**](counteritem.md)     | Proporciona información de propiedades de elemento de contador.                                                                  |
| [**Counters**](counters.md)           | Proporciona los métodos para administrar una colección de [**objetos CounterItem.**](counteritem.md)                |
| [**LogFileItem**](logfileitem.md)     | Proporciona una propiedad para recuperar la ruta de acceso de un único archivo de registro al que tiene acceso el control Monitor del sistema.     |
| [**LogFiles**](logfiles.md)           | Proporciona los métodos y propiedades para administrar una colección de [**objetos LogFileItem.**](logfileitem.md) |
| [**Systemmonitor**](systemmonitor.md) | Proporciona los métodos y propiedades utilizados para configurar el control Monitor de sistema.                            |
