# <a name="systemdevicesinlocalmachinecontainer"></a>System. Devices. InLocalMachineContainer

El dispositivo está en el contenedor de la máquina local.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.InLocalMachineContainer
   shellPKey = PKEY_Devices_InLocalMachineContainer
   formatID = 8C7ED206-3F8A-4827-B3AB-AE9E1FAEFC6C
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
