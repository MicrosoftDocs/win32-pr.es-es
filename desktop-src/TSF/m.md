# <a name="m-text-services-framework"></a>M (Text Services Framework)

<dl> <dt>

<span id="_tsf_modal_gly"></span><span id="_TSF_MODAL_GLY"></span>**Modal**
</dt> <dd>

Término que describe una ventana secundaria que restringe la interacción del usuario con otras ventanas. El usuario debe cerrar una ventana modal, como un cuadro de diálogo, para que la aplicación pueda continuar.
