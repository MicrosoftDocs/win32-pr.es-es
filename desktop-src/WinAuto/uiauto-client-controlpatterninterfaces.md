# <a name="control-pattern-interfaces-for-clients"></a>Interfaces de patrón de control para clientes

En esta sección se describen las interfaces de patrón de control de las aplicaciones cliente de automatización de la interfaz de usuario no administradas.

## <a name="in-this-section"></a>En esta sección



| Interfaz                                                                                                | Descripción                                                                                                                                                                                                                                                                                                                                             |
|----------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IUIAutomationAnnotationPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationannotationpattern)<br/>           | Proporciona acceso a las propiedades de una anotación en un documento.<br/>                                                                                                                                                                                                                                                                            |
| [**IUIAutomationCustomNavigationPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationcustomnavigationpattern)<br/>   | Expone un método para admitir el acceso de un cliente de automatización de la interfaz de usuario de Microsoft a los controles que admiten un orden de navegación personalizado.<br/>                                                                                                                                                                                                                   |
| [**IUIAutomationDockPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationdockpattern)<br/>                           | Proporciona acceso a un control que permite que los elementos secundarios se organicen horizontal y verticalmente, en relación unos con otros.<br/>                                                                                                                                                                                                                 |
| [**IUIAutomationDragPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationdragpattern)<br/>                           | Proporciona acceso a la información expuesta por un proveedor de automatización de la interfaz de usuario para un elemento que se puede arrastrar como parte de una operación de arrastrar y colocar.<br/>                                                                                                                                                                                                  |
| [**IUIAutomationDropTargetPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationdroptargetpattern)<br/>               | Proporciona acceso a la información de arrastrar y colocar expuesta por un proveedor de automatización de la interfaz de usuario para un elemento que puede ser el destino de colocación de una operación de arrastrar y colocar.<br/>                                                                                                                                                                                    |
| [**IUIAutomationExpandCollapsePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationexpandcollapsepattern)<br/>       | Proporciona acceso a un control que se puede expandir visualmente para mostrar el contenido y contraer para ocultar el contenido.<br/>                                                                                                                                                                                                                                      |
| [**IUIAutomationGridItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationgriditempattern)<br/>                   | Proporciona acceso a un control secundario en un contenedor de estilo de cuadrícula que admite la interfaz [**IUIAutomationGridPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationgridpattern) .<br/>                                                                                                                                                                                |
| [**IUIAutomationGridPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationgridpattern)<br/>                           | Proporciona acceso a un control que actúa como contenedor de una colección de controles secundarios que se organizan en un sistema de coordenadas lógico bidimensional que se puede atravesar por fila y columna.<br/>                                                                                                                                          |
| [**IUIAutomationInvokePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationinvokepattern)<br/>                       | Expone un método que permite a una aplicación cliente invocar la acción de un control (normalmente un botón).<br/>                                                                                                                                                                                                                                   |
| [**IUIAutomationItemContainerPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationitemcontainerpattern)<br/>         | Expone un método que recupera un elemento de un contenedor, como una lista virtual.<br/>                                                                                                                                                                                                                                                            |
| [**IUIAutomationLegacyIAccessiblePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationlegacyiaccessiblepattern)<br/> | Expone métodos y propiedades que permiten a los clientes de automatización de la interfaz de usuario recuperar información de la interfaz de usuario de servidores de Microsoft Active Accessibility (MSAA).<br/>                                                                                                                                                                                              |
| [**IUIAutomationMultipleViewPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationmultipleviewpattern)<br/>           | Proporciona acceso a un control que puede cambiar entre varias representaciones de la misma información o conjunto de controles secundarios.<br/>                                                                                                                                                                                                              |
| [**IUIAutomationObjectModelPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationobjectmodelpattern)<br/>             | Proporciona acceso al modelo de objetos subyacente implementado por un control o una aplicación. <br/>                                                                                                                                                                                                                                                     |
| [**IUIAutomationPatternHandler**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iuiautomationpatternhandler)<br/>                     | Devuelve un objeto contenedor de API de cliente y para anular la serialización de las solicitudes de propiedad y método en una instancia de proveedor real.<br/>                                                                                                                                                                                                                            |
| [**IUIAutomationPatternInstance**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iuiautomationpatterninstance)<br/>                   | Representa un objeto de patrón de control. El contenedor de la API de cliente usa esta interfaz para implementar todas las llamadas de método y propiedad en términos de los métodos [**GetProperty**](/windows/desktop/api/UIAutomationCore/nf-uiautomationcore-iuiautomationpatterninstance-getproperty) y [**CallMethod**](/windows/desktop/api/UIAutomationCore/nf-uiautomationcore-iuiautomationpatterninstance-callmethod) .<br/>                                          |
| [**IUIAutomationRangeValuePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationrangevaluepattern)<br/>               | Proporciona acceso a un control que presenta un intervalo de valores.<br/>                                                                                                                                                                                                                                                                                |
| [**IUIAutomationScrollItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationscrollitempattern)<br/>               | Expone un método que permite colocar un elemento en una vista desplazable en una parte visible de la vista.<br/>                                                                                                                                                                                                                                    |
| [**IUIAutomationScrollPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationscrollpattern)<br/>                       | Proporciona acceso a un control que actúa como contenedor desplazable para una colección de elementos secundarios.<br/>                                                                                                                                                                                                                                         |
| [**IUIAutomationSelectionItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionitempattern)<br/>         | Proporciona acceso a los elementos secundarios seleccionables de un control contenedor que admite [**IUIAutomationSelectionPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern).<br/>                                                                                                                                                                            |
| [**IUIAutomationSelectionPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern)<br/>                 | Proporciona acceso a un control que contiene elementos secundarios seleccionables. Los elementos secundarios de este elemento admiten [**IUIAutomationSelectionItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionitempattern).<br/>                                                                                                                                               |
| [**IUIAutomationSelectionPattern2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern2)<br/>               | Extiende la interfaz [**IUIAutomationSelectionPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationselectionpattern) para proporcionar información sobre los elementos seleccionados.<br/>                                                                                                                                                                                             |
| [**IUIAutomationSpreadsheetItemPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationspreadsheetitempattern)<br/> | Habilita a una aplicación cliente para que recupere información sobre un elemento (celda) en una hoja de cálculo.<br/>                                                                                                                                                                                                                                                  |
| [**IUIAutomationSpreadsheetPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationspreadsheetpattern)<br/>         | Permite a una aplicación cliente tener acceso a los elementos (celdas) de una hoja de cálculo.<br/>                                                                                                                                                                                                                                                                   |
| [**IUIAutomationStylesPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationstylespattern)<br/>                   | Habilita a los clientes de automatización de la interfaz de usuario para recuperar los estilos visuales asociados a un elemento de un documento.<br/>                                                                                                                                                                                                                                        |
| [**IUIAutomationSynchronizedInputPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationsynchronizedinputpattern)<br/> | Proporciona acceso a la entrada del teclado o del mouse de un control.<br/>                                                                                                                                                                                                                                                                                 |
| [**IUIAutomationTableItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtableitempattern)<br/>                 | Proporciona acceso a un elemento secundario de un contenedor que admite [**IUIAutomationTablePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtablepattern). <br/>                                                                                                                                                                                                      |
| [**IUIAutomationTablePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtablepattern)<br/>                         | Proporciona acceso a un control que actúa como contenedor para una colección de elementos secundarios. Los elementos secundarios de este elemento admiten [**IUIAutomationTableItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtableitempattern) y se organizan en un sistema de coordenadas lógico bidimensional que se puede atravesar por fila y columna.<br/>                       |
| [**IUIAutomationTextChildPattern**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationtextchildpattern)<br/>             | Proporciona acceso a un control basado en texto (o un objeto incrustado en el texto) que es un elemento secundario o descendiente de otro control basado en texto. <br/>                                                                                                                                                                                                           |
| [**IUIAutomationTextEditPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtexteditpattern)<br/>                   | Proporciona acceso a un control que modifica texto, por ejemplo, un control que realiza autocorrección o habilita la composición de entrada mediante un editor de métodos de entrada (IME).<br/>                                                                                                                                                                      |
| [**IUIAutomationTextPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern)<br/>                           | Proporciona acceso a un control que contiene texto.<br/>                                                                                                                                                                                                                                                                                             |
| [**IUIAutomationTextPattern2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern2)<br/>                         | Extiende la interfaz [**IUIAutomationTextPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern) . <br/>                                                                                                                                                                                                                                                  |
| [**IUIAutomationTextRange**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange)<br/>                               | Proporciona acceso a un intervalo de texto continuo en un contenedor que admite la interfaz [**IUIAutomationTextPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextpattern) . Las aplicaciones cliente pueden utilizar la interfaz [**IUIAutomationTextRange**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange) para seleccionar, comparar y recuperar objetos incrustados desde el intervalo de texto.<br/> |
| [**IUIAutomationTextRange2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange2)<br/>                             | Extiende la interfaz [**IUIAutomationTextRange**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange) para permitir a los clientes de automatización de la interfaz de usuario invocar menús contextuales mediante programación.<br/>                                                                                                                                                                              |
| [**IUIAutomationTextRange3**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange3)<br/>                             | Extiende la interfaz [**IUIAutomationTextRange2**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange2) para admitir un acceso más rápido a los datos de texto enriquecido subyacentes en un intervalo de texto.<br/>                                                                                                                                                                           |
| [**IUIAutomationTextRangeArray**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrangearray)<br/>                     | Representa una colección de objetos [**IUIAutomationTextRange**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtextrange) .<br/>                                                                                                                                                                                                                                          |
| [**IUIAutomationTogglePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtogglepattern)<br/>                       | Proporciona acceso a un control que puede recorrer en iteración un conjunto de Estados y mantener un estado una vez establecido.<br/>                                                                                                                                                                                                                                   |
| [**IUIAutomationTransformPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtransformpattern)<br/>                 | Proporciona acceso a un control que se puede desplazar, cambiar de tamaño o girar.<br/>                                                                                                                                                                                                                                                                         |
| [**IUIAutomationTransformPattern2**](/windows/desktop/api/uiautomationclient/nn-uiautomationclient-iuiautomationtransformpattern2)<br/>           | Extiende la interfaz [**IUIAutomationTransformPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationtransformpattern) para permitir a los clientes de automatización de la interfaz de usuario tener acceso mediante programación a la funcionalidad de zoom de la ventanilla de un control.<br/>                                                                                                                              |
| [**IUIAutomationValuePattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationvaluepattern)<br/>                         | Proporciona acceso a un control que contiene un valor que no abarca un intervalo y que se puede representar como una cadena.<br/>                                                                                                                                                                                                                       |
| [**IUIAutomationVirtualizedItemPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationvirtualizeditempattern)<br/>     | Representa un elemento virtualizado, que es un elemento representado por un elemento de automatización de marcadores de posición en el árbol de automatización de la interfaz de usuario.<br/>                                                                                                                                                                                                          |
| [**IUIAutomationWindowPattern**](/windows/desktop/api/UIAutomationClient/nn-uiautomationclient-iuiautomationwindowpattern)<br/>                       | Proporciona acceso a la funcionalidad fundamental de una ventana.<br/>                                                                                                                                                                                                                                                                                |
