# <a name="d1113-device-needs-bgra-flag"></a>D1113: el dispositivo necesita la marca BGRA

El dispositivo Direct3D no se creó con D3D10 \_ Create \_ Device \_ BGRA \_ support y, por lo tanto, no es compatible con Direct2D.



|             |         |
|-------------|---------|
| Nivel de error | Advertencia |



 

## <a name="possible-causes"></a>Causas posibles

El dispositivo Direct3D no se creó con D3D10 \_ Create \_ Device \_ BGRA \_ support.
