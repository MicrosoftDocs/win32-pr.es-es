# <a name="midl-and-com"></a>MIDL y COM

Todos los objetos COM implementan una o varias interfaces. Al crear un objeto COM personalizado, debe describir la interfaz o las interfaces de un archivo IDL. Los temas de esta sección presentan información sobre el uso de MIDL para definir interfaces COM:

-   [Archivos generados para una interfaz COM](files-generated-for-a-com-interface.md)
-   [Serialización de tipos de datos OLE](marshaling-ole-data-types.md)

Para obtener más información sobre cómo definir interfaces COM con MIDL, vea [Definición de interfaces COM](../com/defining-com-interfaces.md). Para obtener más información sobre la creación de archivos generados para varias [plataformas, vea Targeting Stubs for Specific 32-bit or 64-bit Platforms](targeting-stubs-for-specific-32-bit-or-64-bit-platforms.md)( Código auxiliar de destino para plataformas específicas de 32 o 64 bits).
