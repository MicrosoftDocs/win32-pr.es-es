# <a name="f16tof32-function"></a>Función f16tof32

Convierte el valor float16 almacenado en la mitad baja del uint en un valor float.

## <a name="syntax"></a>Sintaxis

``` syntax
float f16tof32(
  in uint value
);
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

*value* \[ En\]
</dt> <dd>

Tipo: **uint**

Valor de entrada.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Tipo: **float**

El valor convertido.

## <a name="remarks"></a>Comentarios

También están disponibles las siguientes versiones sobrecargadas:

``` syntax
float2 f16tof32(uint2 value);
float3 f16tof32(uint3 value);
float4 f16tof32(uint4 value);
```

### <a name="minimum-shader-model"></a>Modelo mínimo de sombreador

Esta función se admite en los siguientes modelos de sombreador.



| Modelo de sombreador                                                        | Compatible |
|---------------------------------------------------------------------|-----------|
| [Modelos de sombreador 4](dx-graphics-hlsl-sm4.md) y superiores | Sí       |



 

Aunque esta función está disponible en el modelo de sombreador [4](dx-graphics-hlsl-sm4.md) y en los modelos de sombreador posteriores, dado que se emula en 4.0 y 4.1, es menos importante en estos modelos de sombreador que en el modelo de sombreador [5.](d3d11-graphics-reference-sm5.md)

Esta función se admite en los siguientes tipos de sombreadores:



| Vértice | Casco | Domain | Geometría | Píxel | Proceso |
|--------|------|--------|----------|-------|---------|
| x      | x    | x      | x        | x     | x       |
