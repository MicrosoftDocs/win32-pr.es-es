# <a name="iwmdrmdecrypt-interface"></a>Interfaz IWMDRMDecrypt

La **interfaz IWMDRMDecrypt** proporciona un método para descifrar el contenido.

## <a name="members"></a>Miembros

La **interfaz IWMDRMDecrypt** hereda de la [**interfaz IUnknown.**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) **IWMDRMDecrypt** también tiene estos tipos de miembros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

La **interfaz IWMDRMDecrypt** tiene estos métodos.



| Método                                   | Descripción                                 |
|:-----------------------------------------|:--------------------------------------------|
| [**Descifrar**](iwmdrmdecrypt-decrypt.md) | Descifra un búfer de datos en su lugar.<br/> |
