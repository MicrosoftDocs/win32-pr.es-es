# <a name="d1178-custom-dash-style-requires-dash-array"></a>D1178: el estilo de guión personalizado requiere una matriz de guiones

La creación de un estilo de trazo con \_ el estilo de guión \_ personalizado requiere una matriz de guiones.






 

## <a name="possible-causes"></a>Causas posibles

Cuando la aplicación intentó crear un ID2D1StrokeStyle, usó el \_ atributo personalizado de estilo de guion D2D1, \_ \_ pero proporcionó una matriz de guiones null.

## <a name="possible-fixes"></a>Posibles correcciones

Si la aplicación requiere guiones personalizados, asegúrese de que se especifica una matriz de guiones válida. Vea CreateStrokeStyle para obtener un ejemplo. Si un estilo de guión integrado es aceptable, deje la matriz de guiones como NULL y use un especificador, como D2D1 \_ Dash \_ style \_ Dash.
