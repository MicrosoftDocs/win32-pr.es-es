# <a name="media-foundation-primitives"></a>Media Foundation primitivos

Media Foundation define varios tipos de objetos básicos que se usan en las MEDIA FOUNDATION api.



| Tema                                       | Descripción                                                                         |
|---------------------------------------------|-------------------------------------------------------------------------------------|
| [Atributos](attributes-and-properties.md) | Los atributos y las propiedades son pares clave-valor almacenados en un objeto .                  |
| [Tipos de medios](media-types.md)              | Un tipo de medio describe el formato de una secuencia multimedia digital.                        |
| [Búferes multimedia](media-buffers.md)          | Un búfer multimedia administra un bloque de memoria para que se pueda compartir entre objetos . |
| [Ejemplos de medios](media-samples.md)          | Un ejemplo multimedia es un objeto que contiene una lista de búferes multimedia.                  |
