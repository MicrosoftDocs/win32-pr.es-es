# <a name="about-system-restore"></a>Acerca de restaurar sistema

Restaurar sistema supervisa los cambios del sistema y guarda el estado del sistema como *punto de restauración*. Si un problema del sistema se desarrolla como resultado de un cambio del sistema, el usuario puede devolver el sistema a un estado anterior mediante los datos de un punto de restauración.

Las aplicaciones y el sistema pueden crear puntos de restauración cuando se producen cambios en el sistema. Restaurar sistema también puede configurarse para crear puntos de restauración a intervalos regulares y los usuarios pueden crear puntos de restauración en cualquier momento. Restaurar sistema en Windows 7 comprueba cada día y crea un punto de restauración programado si no se ha creado ningún otro punto de restauración en los últimos 7 días. Restaurar sistema en Windows Vista crea un punto de restauración si no se creó ningún otro punto de restauración en las últimas 24 horas. Restaurar sistema en Windows XP crea un punto de control cada 24 horas de tiempo absoluto.

Restaurar sistema no restaura los datos o documentos de usuario, por lo que no hará que los usuarios pierdan los archivos, el correo electrónico, el historial de exploración o los favoritos. Restaurar sistema también se pone a disposición de los usuarios en el entorno de recuperación de Windows o en el modo seguro, lo que facilita la restauración de los equipos en un estado anterior a los problemas.
