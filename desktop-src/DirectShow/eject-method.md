# <a name="eject-method"></a>Método de expulsión

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

El método expulsa o inserta un disco desde o `Eject` en la unidad de DVD.

``` syntax
        MSWebDVD.Eject()
```

## <a name="return-value"></a>Valor devuelto

No de devuelve ningún valor.

## <a name="remarks"></a>Comentarios

En algunas unidades de DVD, este método actúa como un botón de alternancia, expulsando e insertando en llamadas alternativas. Esto depende del hardware.
