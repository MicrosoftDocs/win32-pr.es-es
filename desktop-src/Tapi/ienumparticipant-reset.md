# <a name="ienumparticipantreset-method"></a>IEnumParticipant:: RESET (método)

\[El **restablecimiento** no está disponible para su uso en Windows Vista, windows Server 2008 y versiones posteriores del sistema operativo. La API de cliente de RTC proporciona una funcionalidad similar.\]

El método **RESET** se restablece al principio de la secuencia de enumeración. Este método está oculto en Visual Basic y en los lenguajes de scripting.
