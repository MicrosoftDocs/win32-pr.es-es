# <a name="ui_pkey_smallimage"></a>UI \_ PKEY \_ SmallImage

Identifica la propiedad SmallImage de PKEY de la interfaz de usuario \_ \_ .

```
propertyDescription
   name = UI_PKEY_SmallImage
   shellPKey = UI_PKEY_SmallImage
   formatID = 00000009-7363-696e-8441798acf5aebb7
   propID = 9
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Observaciones

\_ \_ Una aplicación usa la interfaz de usuario PKEY SmallImage para consultar la imagen pequeña asociada a un control de la cinta de opciones.

El valor de propiedad es un objeto [**IUIImage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) .
