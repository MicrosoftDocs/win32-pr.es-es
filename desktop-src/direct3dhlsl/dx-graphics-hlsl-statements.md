# <a name="statements"></a>Instrucciones

Las instrucciones HLSL son combinaciones de variables y funciones que son expresiones; Al igual que una frase, una instrucción HLSL requiere operadores que determinen cómo se evaluará una expresión. Puede crear instrucciones individuales en un sombreador o crear varias instrucciones en un grupo denominado bloque de instrucciones. Para crear instrucciones HLSL, debe conocer la sintaxis para escribir expresiones y los operadores que usará para combinar expresiones.

-   [Expresiones](dx-graphics-hlsl-expressions.md)
-   [Operadores](dx-graphics-hlsl-operators.md)
-   [Bloques de instrucciones](dx-graphics-hlsl-statement-blocks.md)
