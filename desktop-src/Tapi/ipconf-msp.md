# <a name="ipconf-msp"></a>IPConf MSP

\[IPConf MSP no está disponible para su uso en Windows Vista, Windows Server 2008 y las versiones posteriores del sistema operativo. La API de cliente de RTC proporciona una funcionalidad similar.\]

El IPConf MSP (Confmsp.dll), cuando se empareja con su TSP (IPconf. TSP), proporciona acceso de conferencia a redes de protocolo de Internet (IP), como Internet y muchas intranets corporativas. Además de los métodos estándar de MSPI, este MSP expone interfaces que permiten controlar a los participantes individuales de una conferencia.

Consulte [interfaces de IPCONF MSP](ipconf-msp-interfaces.md) para obtener un resumen de las interfaces específicas del proveedor que expone este MSP.

IPConf MSP se ha mejorado en Windows 2000 SP1 para habilitar la comunicación entre los clientes H323 y SDP. Ahora también expone la interfaz [**IMulticastControl**](imulticastcontrol.md) .

IPConf MSP se instala con Windows 2000.
