# <a name="mf_mt_video_3d-attribute"></a>Atributo \_ MF MT VIDEO \_ \_ 3D

Especifica si una secuencia de vídeo contiene contenido 3D.

## <a name="data-type"></a>Tipo de datos

**BOOL almacenado** como **UINT32**

## <a name="remarks"></a>Comentarios

Este atributo se aplica a los tipos de medios de vídeo. Si este atributo es **TRUE,** la secuencia de vídeo contiene contenido 3D. El valor predeterminado es **FALSE**.
