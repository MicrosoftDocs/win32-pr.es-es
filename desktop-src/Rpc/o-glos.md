<dl> <dt>

<span id="_rpc_odl_glos"></span><span id="_RPC_ODL_GLOS"></span>**Lenguaje de descripción de objetos (ODL)**
</dt> <dd>

Subconjunto de [*atributos MIDL,*](m-glos.md) palabras clave, instrucciones y directivas que se usan para definir bibliotecas de tipos para aplicaciones ole automation.

</dd> <dt>

<span id="_rpc_open_array_glos"></span><span id="_RPC_OPEN_ARRAY_GLOS"></span>**abrir matriz**
</dt> <dd>

Matriz que es compatible y variable; Es decir, el tamaño y el intervalo de elementos transmitidos se determinan en tiempo de ejecución mediante otros parámetros, estructuras o expresiones.

</dd> <dt>

<span id="_rpc_osf_glos"></span><span id="_RPC_OSF_GLOS"></span>**Open Software Foundation (OSF)**
</dt> <dd>

Consorcio de empresas, formado para definir el entorno informático distribuido (DCE).

</dd> <dt>

<span id="_rpc_orphaned_memory_glos"></span><span id="_RPC_ORPHANED_MEMORY_GLOS"></span>**memoria huérfana**
</dt> <dd>

Memoria en un cliente al que hace referencia previamente un parámetro de puntero que el servidor ha restablecido a null.

</dd> <dt>

<span id="_rpc_output_parameter_glos"></span><span id="_RPC_OUTPUT_PARAMETER_GLOS"></span>**parámetro de salida**
</dt> <dd>

Parámetro que contiene los datos que la aplicación de servidor transmite a la aplicación cliente. El \[ [atributo out](/windows/desktop/Midl/out-idl) \] designa un parámetro de salida.

</dd> </dl>
