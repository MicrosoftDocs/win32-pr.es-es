# <a name="using-the-auxiliary-library"></a>Uso de la biblioteca auxiliar

Con OpenGL, Silicon Graphics (SGI) creó la biblioteca auxiliar para escribir programas de ejemplo sencillos para la *Guía de programación de OpenGL*. El código fuente de la biblioteca auxiliar se suministra con el SDK de la plataforma de Microsoft, junto con los ejemplos de OpenGL. Para entender cómo se desarrolló la biblioteca auxiliar a partir de las funciones y rutinas de OpenGL, examine el código fuente. Puede usar funciones de la biblioteca auxiliar en sus propios programas. Para obtener una descripción de la biblioteca auxiliar, vea la *Guía de programación de OpenGL*.
