# <a name="regioncomplement-methods"></a>Métodos de region. Complement

En este tema se enumeran los métodos de complemento de la clase [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Para obtener una lista completa de los métodos de la clase **Region** , consulte [métodos de región](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                       | Descripción                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Complemento (Rect&)**](/previous-versions//ms534918(v=vs.85))     | El método [**Region:: Complement**](/previous-versions//ms534918(v=vs.85)) actualiza esta región a la parte del interior del rectángulo especificado que no forma una intersección con esta región.<br/>  |
| [**Complemento (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstrectf_))   | El método [**Region:: Complement**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstrectf_)) actualiza esta región a la parte del interior del rectángulo especificado que no forma una intersección con esta región.<br/> |
| [**Complemento (región \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstregion))     | El método [**Region:: Complement**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstregion)) actualiza esta región a la parte de otra región que no forma una intersección con esta región.<br/>                         |
| [**Complemento (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstgraphicspath)) | El método [**Region:: Complement**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-complement(inconstgraphicspath)) actualiza esta región a la parte del interior de la ruta de acceso especificada que no forma una intersección con esta región.<br/>            |
