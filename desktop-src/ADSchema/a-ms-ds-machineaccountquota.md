# <a name="ms-ds-machine-account-quota-attribute"></a>Atributo de cuota MS-DS-Machine-Account

El número de cuentas de equipo que un usuario puede crear en un dominio.



| Entrada | Value |
|-------------------|--------------------------------------------------|
| CN                | Cuota de la cuenta de MS-DS-Machine                      |
| Nombre para mostrar de LDAP | MS-DS-MachineAccountQuota                        |
| Tamaño              | 4 bytes                                          |
| Actualizar privilegio  | Administrador de dominio                             |
| Frecuencia de actualización  | Siempre que es necesario cambiar la cuota de un dominio. |
| Attribute-Id      | 1.2.840.113556.1.4.1411                          |
| System-ID-GUID    | d064fb68-1480-11d3-91c1-0000f87a57d4             |
| Sintaxis            | [**Enumeración**](s-enumeration.md)             |
