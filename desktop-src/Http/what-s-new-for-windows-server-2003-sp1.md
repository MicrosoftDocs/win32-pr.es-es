# <a name="whats-new-for-windows-server-2003-sp1"></a>Novedades de Windows Server 2003 SP1

Las siguientes características están disponibles para Windows Server 2003 con Service Pack 1 (SP1):

-   [Compatibilidad con SSL en modo kernel.](kernel-mode-ssl.md)
    -   Windows Server 2003 con SP1 agrega la opción de realizar operaciones SSL en modo kernel para mejorar el rendimiento.
-   Compatibilidad con la ejecución de aplicaciones de 32 bits en un sistema operativo de 64 bits (WOW64).
-   [Mejoras en el registro de errores HTTP.](error-logging-in-windows-server-2003-sp1.md)
    -   Compatibilidad con encabezados de estilo W3C.
    -   Campos de registro adicionales disponibles.
    -   Suversión del archivo de registro basado en fecha y hora.
-   [Excepciones de análisis http.](parsing-exceptions.md)
    -   Admite excepciones a HTTP/1.1 por compatibilidad con versiones anteriores.
-   [Mejoras del analizador HTTP.](parser-enhancements.md)
