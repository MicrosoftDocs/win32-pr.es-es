# <a name="print-separator-file-attribute"></a>Print-separator-(atributo de archivo)

La ruta de acceso del archivo de la página separador de impresora.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Print-separator-File                        |
| Nombre para mostrar de LDAP | printSeparatorFile                          |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.230                      |
| System-ID-GUID    | 281416c6-1968-11d0-a28f-00aa003049e2        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
