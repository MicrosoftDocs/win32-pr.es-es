# <a name="cdrawimagem_sourcerect-member"></a>Miembro SourceRect CDrawImage:: m \_

La variable miembro **m \_ SourceRect** especifica el rectángulo de origen del dibujo. El valor inicial es un rectángulo vacío, lo que indica que se debe dibujar toda la imagen.

## <a name="syntax"></a>Sintaxis


```C++
RECT m_SourceRect;
```
