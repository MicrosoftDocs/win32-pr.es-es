# <a name="wmcontentgroupdescription"></a>WM/ContentGroupDescription

El atributo **WM/ContentGroupDescription** contiene una descripción del grupo de contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMContentGroupDescription

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Los grupos de contenido son colecciones de medios a los que pueden pertenecer archivos individuales. Por ejemplo, un archivo de audio puede ser una canción que sea una pista de un disco compacto que forme parte de un conjunto de conversión boxing. De forma similar, un clip de vídeo puede formar parte de una serie.
