# <a name="display-area-plug-ins-deprecated"></a>Complementos de área de presentación (desusado)

En esta página se documenta una característica que puede no estar disponible en versiones futuras de Windows Media Player y el SDK de Windows Media Player.

Los complementos de la interfaz de usuario del área de presentación se muestran en el área principal del panel **reproducción en curso** , debajo del intérprete y seguimiento de la información y a la izquierda del área de la lista de reproducción. Solo se puede habilitar un complemento de área de presentación a la vez. Cuando un complemento de área de presentación está habilitado, el reproductor no puede entrar en modo de pantalla completa.

Los complementos de área de presentación son útiles para mostrar grandes cantidades de información. Por ejemplo, un complemento de área de presentación puede recuperar información sobre el elemento multimedia que se está reproduciendo y mostrar artículos, entrevistas y revisiones relacionados. Un complemento de área de presentación puede hospedar una ventana del explorador de Microsoft Internet Explorer para mostrar esta información mediante HTML.

> [!Note]  
> Una ventana del explorador hospedada puede incrustar la versión de ActiveX de Windows Media Player, pero no se recomienda porque el control incrustado no puede comunicarse con el reproductor independiente que lo contiene.

 

> [!Note]  
> Windows Media Player cierra cualquier complemento de área de presentación abierta cuando el usuario reproduce contenido que contiene vídeo.
