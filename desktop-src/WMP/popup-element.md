# <a name="popup-element"></a>Elemento POPUP

El **elemento POPUP** proporciona una manera de que los usuarios seleccionen elementos de una lista. Estos elementos se pueden especificar mediante el uso de **elementos ITEM** como elementos secundarios del **elemento POPUP.** El **elemento POPUP** es idéntico al elemento **LISTBOX,** salvo que el valor predeterminado del atributo **popUp** es true. Cuando **popUp** se establece en true, el control solo se muestra cuando se llama al método **show** y su ancho se establece en el ancho del contenido (se omite el atributo **de** ancho ambiente). El control se oculta automáticamente cuando el usuario selecciona un elemento de la lista o cuando se llama **al método dismiss.**

> [!Note]  
> Este elemento requiere Reproductor de Windows Media para Windows XP o posterior.
