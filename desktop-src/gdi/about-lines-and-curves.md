# <a name="about-lines-and-curves"></a>Acerca de las líneas y curvas

Muchos tipos de aplicaciones usan líneas y curvas para dibujar la salida de gráficos en dispositivos de trama. Las aplicaciones de diseño asistido por PC (CAD) y dibujo usan líneas y curvas para esquematear objetos, especificar los centros de objetos, las dimensiones de los objetos, y así sucesivamente. Las aplicaciones de hoja de cálculo usan líneas y curvas para dibujar cuadrículas, gráficos y gráficos. Las aplicaciones de procesamiento de palabras usan líneas para crear reglas y bordes en una página de texto.
