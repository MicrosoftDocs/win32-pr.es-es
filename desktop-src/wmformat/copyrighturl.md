# <a name="copyrighturl"></a>CopyrightURL

El atributo **CopyrightURL** contiene la dirección Web de una página de copyright asociada con la personalización de marca.

## <a name="global-constant"></a>Constante global

g \_ wszWMCopyrightURL

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Este atributo se aplica a todo el archivo.
