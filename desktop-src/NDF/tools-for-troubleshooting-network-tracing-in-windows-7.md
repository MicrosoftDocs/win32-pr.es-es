# <a name="tools-for-troubleshooting-using-network-tracing-in-windows-7"></a>Herramientas para solucionar problemas con el seguimiento de red en Windows 7

Hay una serie de herramientas que se pueden utilizar para ayudar a habilitar y administrar el seguimiento de la red y para ver los datos resultantes. Algunas de las herramientas que pueden ser útiles son Netsh.exe, Monitor de red, Visor de eventos, **netsh Trace Convert**, Tracerpt.exe y Xperf.exe.

En esta sección, veremos cómo usar Netsh.exe y Monitor de red para ayudarle a solucionar problemas de seguimiento de la red.
