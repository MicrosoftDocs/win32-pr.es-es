# <a name="d1200-pushlayer-with-multisampled-target"></a>D1200: PushLayer con destino multimuestreado

Pushlayer/Poplayer se usó con un destino de muestreo multimuestreado. La llamada se seguirá emitiendo, pero es posible que el resultado representado no represente el patrón de Multimuestra esperado.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación interopera una textura de Direct3D multimuestreada como un destino de Direct2D y se denomina PushLayer en el destino.

## <a name="possible-fixes"></a>Posibles correcciones

Evite las texturas multimuestreadas interoperables si es importante la funcionalidad de capas. El uso de capas o clips con estos tipos de recursos puede suponer una reducción del rendimiento o resultados inesperados. Si es posible, use un medio alternativo para controlar la opacidad o el enmascaramiento en lugar de las capas. O bien, use un intermediario en lugar de un recurso de muestreo multimuestreado. Si el recurso de muestreo múltiple está pensado para el contenido 2D de suavizado de contorno, considere la posibilidad de que el suavizado de contorno primitivo basado en Direct2d sea suficiente.
