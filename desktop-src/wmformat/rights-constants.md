# <a name="rights-constants"></a>Constantes de derechos

Las constantes que se enumeran en la tabla siguiente se usan para identificar las acciones DRM y generar listas de acciones.



| Constante                                        | Descripción                                                                                                                                                                                                                                                    |
|-------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| g \_ wszWMDRM \_ ACTIONLIST \_ etiqueta                    | Define el nombre del elemento XML para una lista de acciones.                                                                                                                                                                                                               |
| \_etiqueta de \_ acción g wszWMDRM \_                        | Define el nombre del elemento XML de una entrada de una lista de acciones.                                                                                                                                                                                                   |
| g \_ wszWMDRM \_ la \_ reproducción derecha                    | Define la cadena de la derecha para reproducir el contenido.                                                                                                                                                                                                              |
| g \_ wszWMDRM \_ \_ copia derecha                        | Define la cadena de la derecha para copiar contenido.                                                                                                                                                                                                              |
| g \_ wszWMDRM de la \_ lista de \_ reproducción derecha \_              | Define la cadena de la derecha para grabar contenido en CD como parte de una lista de reproducción.                                                                                                                                                                                  |
| g \_ wszWMDRM \_ derecha \_ crear \_ imagen en miniatura \_    | Define la cadena de la derecha para crear una imagen en miniatura a partir del contenido de vídeo.                                                                                                                                                                               |
| g \_ wszWMDRM \_ \_ copia derecha \_ a \_ CD                | Define la cadena de la derecha para copiar contenido en un CD. Las nuevas licencias no deben usar este derecho. En su lugar, todos los derechos que conceden el permiso para copiar contenido deben estar incluidos en el derecho de copia y en la lista de reproducción.                                     |
| g \_ wszWMDRM \_ la \_ copia derecha \_ en el dispositivo de \_ SDMI \_      | Define la cadena de la derecha para copiar contenido en un dispositivo que se ajusta a la iniciativa de música digital segura (SDMI). Las nuevas licencias no deben usar este derecho. En su lugar, todos los derechos que conceden el permiso para copiar contenido deben estar incluidos en el derecho de copia. |
| g \_ wszWMDRM \_ la \_ copia derecha \_ en un \_ dispositivo que no es de \_ SDMI \_ | Define la cadena de la derecha para copiar en un dispositivo que no se ajusta a la iniciativa de música digital segura (SDMI). Las nuevas licencias no deben usar este derecho. En su lugar, todos los derechos que conceden el permiso para copiar contenido deben estar incluidos en el derecho de copia. |
| copia de seguridad de g \_ wszWMDRM \_ derecha \_                      | Define la cadena de la derecha para hacer una copia de seguridad de la licencia.                                                                                                                                                                                                        |
| g \_ wszWMDRM de colaboración de la \_ derecha \_ \_         | Define la cadena de la derecha para reproducir contenido a través de una red como parte de una lista de reproducción compartida.                                                                                                                                                                  |
