# <a name="cabinet-api-structures"></a>Estructuras de API de archivo. cab

En esta sección se detallan las siguientes estructuras de API de archivo. cab:



| Estructura                                              | Descripción                                                                                     |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [**CABINETDLLVERSIONINFO**](cabinetdllversioninfo.md) | Contiene la información de versión de Cabinet.dll.<br/>                                    |
| [**CCAB**](/windows/desktop/api/Fci/ns-fci-ccab)                                   | Contiene información del archivo. cab.<br/>                                                        |
| [**Fun**](/windows/win32/api/fdi_fci_types/ns-fdi_fci_types-erf)                                     | Devuelve información de error de FCI/FDI. El autor de la llamada no debe modificar esta estructura.<br/> |
| [**FDICABINETINFO**](/windows/desktop/api/Fdi/ns-fdi-fdicabinetinfo)               | Contiene detalles sobre un archivo. cab determinado.<br/>                                    |
| [**FDINOTIFICATION**](/windows/desktop/api/Fdi/ns-fdi-fdinotification)             | Proporcione información a [**FNFDINOTIFY**](/windows/desktop/api/fdi/nf-fdi-fnfdinotify).<br/>                           |
| [**SESIÓN**](session.md)                             | Contiene información sobre la sesión actual.<br/>                                      |
