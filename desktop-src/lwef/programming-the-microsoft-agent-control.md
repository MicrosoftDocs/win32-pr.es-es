# <a name="programming-the-microsoft-agent-control"></a>Programación del control microsoft agent

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Aunque las aplicaciones pueden escribir directamente en los servicios de Microsoft Agent mediante sus interfaces de servidor de automatización, Microsoft Agent también incluye un control ActiveX (OLE). El control admite la programación sencilla mediante un lenguaje de scripting como Microsoft Visual Basic Scripting Edition (VBScript) u otros lenguajes que admiten la interfaz ActiveX control.
