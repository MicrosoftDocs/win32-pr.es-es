# <a name="mf_mse_activelist_callback-attribute"></a>\_Atributo de \_ devolución de llamada MF MSE ACTIVELIST \_

Contiene un puntero a la interfaz de devolución de llamada de la aplicación para la interfaz [**IMFBufferListNotify**](/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfbufferlistnotify) para la lista de búferes activos.

## <a name="data-type"></a>Tipo de datos

**IMFBufferListNotify \* *_ se almacena como _* IUnknown\***
