# <a name="boundingboxcontains-methods"></a>Métodos BoundingBox.Contains

Comprueba si boundingBox contiene un objeto especificado.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                               | Descripción                                                                                                                                |
|:-------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------|
| [**BoundingBox::Contains (XMVECTOR)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains)                   | Comprueba si el objeto BoundingBox contiene un punto especificado.<br/>                                                                   |
| [**BoundingBox::Contains (const BoundingBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(constboundingbox_))         | Comprueba si boundingBox contiene otro BoundingBox.<br/>                                                                     |
| [**BoundingBox::Contains (const BoundingSphere&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(constboundingsphere_))      | Comprueba si boundingBox contiene un objeto BoundingSphere especificado.<br/>                                                              |
| [**BoundingBox::Contains (const BoundingFrustum&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(constboundingfrustum_))     | Comprueba si [**boundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) contiene el [**valor de BoundingFrustum especificado.**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum)<br/>         |
| [**BoundingBox::Contains (XMVECTOR,XMVECTOR,XMVECTOR)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(fxmvector_fxmvector_fxmvector)) | Compruebe si boundingBox contiene un triángulo especificado.<br/>                                                                     |
| [**BoundingBox::Contains (const BoundingOrientedBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(constboundingorientedbox_)) | Comprueba si [**boundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) contiene el [**objeto BoundingOrientedBox especificado.**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox)<br/> |
