# <a name="cmspstream-type-definitions"></a>Definiciones de tipo CMSPStream

Las siguientes definiciones proporcionan un conjunto de Estados de secuencia que pueden ser útiles para la clase derivada, aunque su uso no es necesario. Las clases derivadas pueden extender el conjunto de Estados disponibles con facilidad mediante la definición de valores adicionales.

``` syntax
#define STRM_INITIAL            0x00000000
#define STRM_TERMINALSELECTED   0x00000001
#define STRM_CONFIGURED         0x00000002
#define STRM_RUNNING            0x00000004
#define STRM_PAUSED             0x00000008
#define STRM_STOPPED            0x00000010
```
