# <a name="custom-action-sources"></a>Orígenes de acciones personalizadas

Un origen de acción personalizado puede existir en varios formatos diferentes. En la tabla siguiente se enumeran los tipos de acciones personalizadas que pueden tener cada forma de origen. Consulte el tema sobre el tipo de acción personalizada básico para obtener más información.



| Origen de la acción personalizada                                           | Tipos de acción personalizados asociados                                                                                                                                                                                 |
|----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Acción personalizada almacenada en la tabla binaria.                      | [Acción personalizada tipo 1](custom-action-type-1.md), [acción personalizada tipo 2](custom-action-type-2.md), [acción personalizada tipo 5](custom-action-type-5.md), [acción personalizada tipo 6](custom-action-type-6.md)         |
| Acción personalizada copiada durante la instalación.                      | [Acción personalizada tipo 17](custom-action-type-17.md), [acción personalizada tipo 18](custom-action-type-18.md), [tipo acción personalizada 21](custom-action-type-21.md), [tipo de acción personalizada 22](custom-action-type-22.md) |
| Acción personalizada que hace referencia a un directorio.                         | [Tipo de acción personalizada 34](custom-action-type-34.md)                                                                                                                                                             |
| Acción personalizada que hace referencia a una propiedad.                          | [Tipo de acción personalizada 50](custom-action-type-50.md), [tipo de acción personalizada 53](custom-action-type-53.md), [tipo de acción personalizada 54](custom-action-type-54.md)                                                     |
| Acción personalizada almacenada como código de script literal en la tabla de base de datos. | [Tipo de acción personalizada 37](custom-action-type-37.md), [tipo de acción personalizada 38](custom-action-type-38.md)                                                                                                         |
| Acción personalizada que muestra un mensaje de error.                     | [Tipo de acción personalizada 19](custom-action-type-19.md)                                                                                                                                                             |



 

Vea [lista de Resumen de todos los tipos de acciones personalizadas](summary-list-of-all-custom-action-types.md) para obtener un resumen de todos los tipos de acciones personalizadas y cómo se especifican los orígenes en la [tabla CustomAction](customaction-table.md).
