<span data-ttu-id="0ba5a-118">Con un servidor de un solo subproceso, es más fácil coordinar las operaciones que afectan a varios clientes y es más fácil proteger los recursos compartidos frente al acceso simultáneo de varios clientes.</span><span class="sxs-lookup"><span data-stu-id="0ba5a-118">With a single-threaded server, it is easier to coordinate operations that affect multiple clients, and it is easier to protect shared resources from simultaneous access by multiple clients.</span></span> <span data-ttu-id="0ba5a-119">El desafío de un servidor de un solo subproceso es que requiere la coordinación de las operaciones superpuestas para asignar el tiempo de procesador para controlar las necesidades simultáneas de los clientes.</span><span class="sxs-lookup"><span data-stu-id="0ba5a-119">The challenge of a single-threaded server is that it requires coordination of overlapped operations to allocate processor time for handling the simultaneous needs of clients.</span></span>
