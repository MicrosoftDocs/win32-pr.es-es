# <a name="locale_senglish-constants"></a>Constantes de SENGLISH de configuración regional \_ \*

En este tema se definen las constantes de SENGLISH de configuración regional \_ \* utilizadas por NLS.



| Value                        | Significado                                                                                                                                                                                                                                                  |
|------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| configuración regional \_ SENGLISHCOUNTRYNAME  | **Windows 7 y versiones posteriores:** Nombre en inglés del país o región, por ejemplo, Alemania para Deutschland. Este nombre siempre está restringido a los caracteres que se pueden asignar al subconjunto de caracteres ASCII 127. <br/>                                      |
| configuración regional \_ SENGLISHDISPLAYNAME  | **Windows 7 y versiones posteriores:** Nombre para mostrar de la configuración regional en inglés. Normalmente, el nombre para mostrar está formado por el idioma y el país o región, por ejemplo, alemán (Alemania) para Deutsch (Deutschland). <br/>                                           |
| configuración regional \_ SENGLISHLANGUAGENAME | **Windows 7 y versiones posteriores:** Nombre en inglés del idioma en inglés, por ejemplo, alemán para Deutsch, del estándar ISO internacional 639. Este nombre siempre está restringido a los caracteres que se pueden asignar al subconjunto de caracteres ASCII 127. <br/> |
