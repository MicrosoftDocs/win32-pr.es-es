# <a name="odl-attribute"></a>odl (atributo)

MKTYPLIB necesitaba el atributo **\[ ODL \]** en las interfaces ODL. El compilador MIDL no requiere el atributo **\[ \] ODL** ; solo se reconoce por compatibilidad con archivos **ODL** antiguos.

> [!Note]  
> La herramienta Mktyplib.exe está obsoleta. En su lugar, utilice el compilador de MIDL.
