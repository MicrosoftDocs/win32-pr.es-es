# <a name="echo-sample-properties"></a>Eco de propiedades de ejemplo

El ejemplo echo expone dos propiedades: el tiempo de retraso y el nivel de efecto (mezcla húmeda). El valor del nivel de señal seco (mezcla seca) siempre se deriva del valor de la combinación húmeda. Debe modificar el código existente y agregar código nuevo para poder tener acceso a estas propiedades.

En las secciones siguientes se explica cómo modificar el código de las propiedades:

-   [Cómo funcionan las propiedades](how-properties-work.md)
-   [Variables para almacenar propiedades](variables-to-store-properties.md)
-   [Modificar la propiedad Scale](modifying-the-scale-property.md)
-   [Agregar la propiedad de combinación húmeda](adding-the-wet-mix-property.md)
