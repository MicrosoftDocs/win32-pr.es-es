# <a name="elements-of-the-sdo-api"></a>Elementos de la API SDO

La API SDO está diseñada en torno a un modelo de objetos que consta de objetos y colecciones de objetos. En el modelo de objetos SDO, los objetos y las colecciones se organizan en una jerarquía, es decir, los objetos y las colecciones contienen otros objetos y colecciones.

Los objetos se manipulan mediante la recuperación y el establecimiento de las propiedades del objeto.

En las secciones siguientes se describe cómo la API de SDO usa objetos, propiedades de objeto y colecciones:

-   [Objetos y propiedades](/windows/desktop/Nps/sdo-objects-and-properties)
-   [Jerarquía del modelo de objetos](/windows/desktop/Nps/sdo-object-model-hierarchy)
-   [Atributos](/windows/desktop/Nps/sdo-sdo-supported-attributes)
