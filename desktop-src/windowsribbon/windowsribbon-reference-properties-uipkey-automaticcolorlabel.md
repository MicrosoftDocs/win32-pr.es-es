# <a name="ui_pkey_automaticcolorlabel"></a>UI \_ PKEY \_ AutomaticColorLabel

Identifica la propiedad AutomaticColorLabel de PKEY de la interfaz de usuario \_ \_ .

```
propertyDescription
   name = UI_PKEY_AutomaticColorLabel
   shellPKey = UI_PKEY_AutomaticColorLabel
   formatID = 00000406-7363-696e-8441798acf5aebb7
   propID = 406
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Observaciones

\_ \_ Una aplicación usa la interfaz de usuario PKEY AutomaticColorLabel para consultar el valor de la etiqueta asociada al botón **automático** de una [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md). La aplicación usa este botón para especificar un color predeterminado según sea necesario.

\_La interfaz de usuario PKEY \_ AutomaticColorLabel solo es válida para [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) `ThemeColors` , donde o `StandardColors` se especifica como el valor del atributo **ColorTemplate** .
