# <a name="bda-media-types"></a>Tipos de medios BDA

## <a name="antenna"></a>Antena

Este tipo de medio se usa para la conexión de PIN entre el filtro de proveedor de red y el filtro de sintonizador.



|             |                                  |
|-------------|----------------------------------|
| Tipo principal  | KSDATAFORMAT \_ tipo de la \_ \_ antena BDA |
| Subtype     | MEDIASUBTYPE \_ ninguno               |
| Tipo de formato | FORMATO \_ ninguno                     |
