# <a name="modifydiskmergesettings-method-of-the-msvm_virtualsystemmanagementservice-class"></a>Método ModifyDiskMergeSettings de la clase Msvm \_ VirtualSystemManagementService

Modifica los datos de configuración de combinación de disco.

## <a name="syntax"></a>Sintaxis


```mof
uint32 ModifyDiskMergeSettings(
  [in]  string              SettingData,
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*SettingData* \[ En\]
</dt> <dd>

Instancia insertada de la [**clase \_ DiskMergeSettingData de Msvm**](msvm-diskmergesettingdata.md) que contiene datos de configuración modificados para la función de combinación de discos.

</dd> <dt>

*Trabajo* \[ out\]
</dt> <dd>

Si la operación se realiza de forma asincrónica, este método devolverá 4096 y este parámetro contendrá una referencia a un objeto derivado de [**\_ CIM ConcreteJob**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
