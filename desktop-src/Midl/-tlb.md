# <a name="tlb-switch"></a>conmutador/TLB

El modificador **/tlb** especifica un nombre de archivo para la biblioteca de tipos generada por el compilador de MIDL.

``` syntax
midl /tlb filename
```

## <a name="switch-options"></a>Opciones de conmutador

<dl> <dt>

*filename* 
</dt> <dd>

Especifica el nombre del archivo de la biblioteca de tipos de salida (TLB). De forma predeterminada, si no se usa el modificador **/tlb** , el archivo TLB tiene el mismo nombre que el archivo IDL, con la extensión. TLB.

</dd> </dl>

## <a name="examples"></a>Ejemplos

**MIDL/TLB NewName. tlb**
