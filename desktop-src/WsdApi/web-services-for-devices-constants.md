# <a name="web-services-on-devices-constants"></a>Constantes de servicios web en dispositivos

Los servicios web usan las siguientes constantes en los dispositivos.



| Constante                                                                                                                                                                                                                        | Descripción                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WSD_DEFAULT_HOSTING_ADDRESS"></span><span id="wsd_default_hosting_address"></span><dl> <dt>**\_dirección de \_ hospedaje \_ predeterminada de WSD**</dt> </dl>                       | La dirección predeterminada (en formato [UrlPrefix](/windows/desktop/Http/urlprefix-strings) ) que un host WSDAPI usará para escuchar las solicitudes en el puerto 5357. <br/>                                                 |
| <span id="WSD_DEFAULT_SECURE_HOSTING_ADDRESS"></span><span id="wsd_default_secure_hosting_address"></span><dl> <dt>**\_dirección de \_ \_ hospedaje segura predeterminada de \_ WSD**</dt> </dl> | La dirección segura predeterminada (en formato [UrlPrefix](/windows/desktop/Http/urlprefix-strings) ) que un host WSDAPI usará para escuchar las solicitudes en el puerto 5358. <br/>                                          |
| <span id="WSD_DEFAULT_EVENTING_ADDRESS__"></span><span id="wsd_default_eventing_address__"></span><dl> <dt>**WSD \_ de \_ \_ dirección de eventos predeterminada**</dt> </dl>               | La dirección predeterminada (en formato [UrlPrefix](/windows/desktop/Http/urlprefix-strings) ) que un host WSDAPI usará para escuchar eventos en el puerto 5358 para hosts seguros y en el puerto 5357 para otros hosts. <br/> |



## <a name="requirements"></a>Requisitos



| Requisito | Value |
|-------------------------------------|--------------------------------------------------------------------------------------|
| Cliente mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows Vista \[\]<br/>                                       |
| Servidor mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows Server 2008 \[\]<br/>                                 |
| Encabezado<br/>                   | <dl> <dt>Wsdutil. h</dt> </dl> |
