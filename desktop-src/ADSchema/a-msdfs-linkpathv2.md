# <a name="ms-dfs-link-path-v2-attribute"></a>atributo MS-DFS-Link-path-V2

Ruta de acceso de vínculo DFS relativa al recurso compartido de destino raíz DFS (es decir, sin los componentes servidor/dominio y nombre del espacio de nombres DFS). Utilice barras diagonales (/) en lugar de barras diagonales inversas ( \) , de modo que las búsquedas LDAP se puedan realizar sin tener que usar escapes.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-DFS-Link-path-V2                         |
| Nombre para mostrar de LDAP | msDFS-LinkPathv2                            |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2039                     |
| System-ID-GUID    | 86b021f6-10ab-40a2-a252-1dc0cc3be6a9        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
