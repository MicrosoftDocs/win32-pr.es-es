# <a name="msmq-label-ex-attribute"></a>Atributo MSMQ-Label-ex

Cadena que describe el tipo de servicio proporcionado por la cola.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MSMQ-etiqueta-ex                               |
| Nombre para mostrar de LDAP | mSMQLabelEx                                 |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1415                     |
| System-ID-GUID    | 4580ad25-d407-48d2-ad24-43e6e56793d7        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
