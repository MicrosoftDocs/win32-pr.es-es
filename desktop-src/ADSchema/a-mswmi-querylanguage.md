# <a name="ms-wmi-querylanguage-attribute"></a>atributo MS-WMI-QueryLanguage

Identifica un lenguaje de consulta WQL.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-QueryLanguage                        |
| Nombre para mostrar de LDAP | msWMI-QueryLanguage                         |
| Tamaño              | Menos de veinte caracteres.                |
| Actualizar privilegio  | Administrador de directiva de grupo                  |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1643                     |
| System-ID-GUID    | 7d3cfa98-c17b-4254-8bd7-4de9b932a345        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
