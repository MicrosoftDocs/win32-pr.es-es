# <a name="last-logoff-attribute"></a>Last-Logoff atributo)

Este atributo no se utiliza.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Last-Logoff                          |
| Nombre para mostrar de LDAP | lastLogoff                           |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | Cada vez que el usuario cierra la sesión.         |
| Attribute-Id      | 1.2.840.113556.1.4.51                |
| System-ID-GUID    | bf967996-0de6-11d0-a285-00aa003049e2 |
| Sintaxis            | [**Interval**](s-interval.md)       |
