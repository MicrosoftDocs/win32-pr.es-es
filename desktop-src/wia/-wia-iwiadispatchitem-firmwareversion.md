# <a name="itemfirmwareversion-property"></a>Propiedad Item. FirmwareVersion

Recupera la versión de firmware del dispositivo. Esta propiedad solo se aplica a los elementos de tipo Device (elementos raíz). Solo lectura.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Item.FirmwareVersion
```



## <a name="property-value"></a>Valor de propiedad

Cadena que recibe la versión de firmware.
