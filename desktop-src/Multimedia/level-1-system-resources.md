# <a name="level-1-system-resources"></a>Recursos del sistema de nivel 1

La configuración mínima de un equipo para satisfacer la especificación de nivel 1 incluye los siguientes elementos:

-   Un microprocesador 386SX con 2 megabytes (MB) de memoria de acceso aleatorio (RAM)
-   Una unidad de disquete de alta densidad de 3,5 pulgadas
-   Una unidad de disco duro con al menos 30 MB de espacio en disco
-   Monitor de color con una resolución de pantalla de 640 x 480 píxeles con 16 colores
-   Software del sistema que ofrece compatibilidad binaria con la versión del sistema operativo Microsoft Windows 3,0
