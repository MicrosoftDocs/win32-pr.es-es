# <a name="taskserviceconnecteddomain-property"></a>Propiedad TaskService. ConnectedDomain

En el caso de scripting, obtiene el nombre del dominio al que está conectado el equipo [**TargetServer**](taskservice-targetserver.md) .

## <a name="syntax"></a>Sintaxis


```VB
TaskService.ConnectedDomain As String
```



## <a name="property-value"></a>Valor de propiedad

Dominio al que está conectado el equipo [**TargetServer**](taskservice-targetserver.md) .
