# <a name="erroritemcustomurl"></a>ErrorItem.customUrl

La propiedad **customURL** recupera la dirección URL de un sitio web que muestra información específica sobre el error de descarga de códecs.

``` syntax
player.error.item(
        index
        ).customURL
      
```

## <a name="possible-values"></a>Valores posibles

Esta propiedad es una **cadena** de solo lectura.

## <a name="remarks"></a>Observaciones

**Windows Media Player 10 Mobile:** Esta propiedad siempre devuelve una cadena vacía.
