# <a name="component-wise-vector-functions"></a>Funciones de vector de modo de componente

Muestra las funciones de vector de modo de componente.

## <a name="in-this-section"></a>En esta sección



| Tema                                                             | Descripción                                                                                                                              |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVectorInsert**](/windows/win32/api/directxmath/nf-directxmath-xmvectorinsert)<br/>               | Gira un vector a la izquierda un número determinado de componentes de 32 bits e inserta los elementos seleccionados de ese resultado en otro vector.<br/> |
| [**XMVectorMergeXY**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergexy)<br/>             | Crea un nuevo vector combinando los componentes x e y de dos vectores.<br/>                                                      |
| [**XMVectorMergeZW**](/windows/win32/api/directxmath/nf-directxmath-xmvectormergezw)<br/>             | Crea un nuevo vector combinando los componentes z y w de dos vectores.<br/>                                                      |
| [**XMVectorPermute**](/windows/win32/api/directxmath/nf-directxmath-xmvectorpermute)<br/>             | Permute los componentes de dos vectores para crear un nuevo vector.<br/>                                                                |
| [**XMVectorRotateLeft**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateleft)<br/>       | Gira el vector a la izquierda un número determinado de elementos de 32 bits.<br/>                                                                 |
| [**XMVectorRotateRight**](/windows/win32/api/directxmath/nf-directxmath-xmvectorrotateright)<br/>     | Gira el vector a la derecha un número determinado de elementos de 32 bits.<br/>                                                                |
| [**XMVectorSelect**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect)<br/>               | Realiza una selección por componente entre dos vectores de entrada y devuelve el vector resultante.<br/>                                |
| [**XMVectorSelectControl**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselectcontrol)<br/> | Define un vector de control para su uso en [**XMVectorSelect**](/windows/win32/api/directxmath/nf-directxmath-xmvectorselect).<br/>                                                 |
| [**XMVectorShiftLeft**](/windows/win32/api/directxmath/nf-directxmath-xmvectorshiftleft)<br/>         | Desplaza un vector a la izquierda de un número determinado de elementos de 32 bits y rellena los elementos vacantes con elementos de un segundo vector.<br/>   |
| [**XMVectorSplatW**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatw)<br/>               | Replica el componente w de un vector en todos los componentes.<br/>                                                              |
| [**XMVectorSplatX**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatx)<br/>               | Replica el componente x de un vector en todos los componentes.<br/>                                                              |
| [**XMVectorSplatY**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplaty)<br/>               | Replica el componente y de un vector en todos los componentes.<br/>                                                              |
| [**XMVectorSplatZ**](/windows/win32/api/directxmath/nf-directxmath-xmvectorsplatz)<br/>               | Replica el componente z de un vector en todos los componentes.<br/>                                                              |
| [**XMVectorSwizzle**](/windows/win32/api/directxmath/nf-directxmath-xmvectorswizzle)<br/>             | Swizzles un vector.<br/>                                                                                                            |
