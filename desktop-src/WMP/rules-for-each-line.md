# <a name="rules-for-each-line"></a>Reglas para cada línea

Cada definición del archivo de definición de máscara debe estar en una línea independiente. Debe escribir texto en líneas mediante las siguientes reglas:

-   Cada línea debe terminar con un retorno de párrafo. Algunas líneas pueden ser más largas que la longitud de línea visible del editor de texto, pero no dividen las líneas. Es posible que desee desactivar el ajuste de línea para asegurarse de que las líneas no se interrumpan.
-   Use solo los caracteres visibles tal y como se define en la referencia de máscara.
-   Una línea está formada por uno o varios elementos, separados por espacios.
-   Puede especificar tantos espacios entre los elementos como desee para mejorar la legibilidad.
-   No use caracteres de tabulación en cualquier parte del archivo.
-   Se omiten las líneas que comienzan con dos barras diagonales consecutivas (//). Úselos para los comentarios. También se omiten las líneas en blanco.
-   Incluya espacios entre los corchetes y el nombre de la sección.
