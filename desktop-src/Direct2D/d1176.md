# <a name="d1176-negative-dpi"></a>D1176: PPP negativo

Es un error especificar un valor de PPP negativo al crear un rendertarget.






 

## <a name="possible-causes"></a>Causas posibles

Una aplicación intentó especificar un valor de PPP negativo, bien mediante el uso de ID2D1RenderTarget:: SetDpi o mediante la creación de un destino con una propiedad PPP negativa.

## <a name="possible-fixes"></a>Posibles correcciones

Elija valores válidos, no negativos para PPP al crear o modificar destinos de representación. Si un valor de PPP predeterminado es aceptable, abstenerse de llamar a SetDpi o especificar una opción de PPP al crear un destino. Especificar un valor de PPP de 0 equivale a utilizar un valor de PPP predeterminado.
