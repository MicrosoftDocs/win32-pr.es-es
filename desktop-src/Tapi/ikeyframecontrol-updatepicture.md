# <a name="ikeyframecontrolupdatepicture-method"></a>IKeyFrameControl:: UpdatePicture (método)

\[**UpdatePicture** no está disponible para su uso en Windows Vista, windows Server 2008 y versiones posteriores del sistema operativo. La API de cliente de RTC proporciona una funcionalidad similar.\]

Una aplicación llama al método **UpdatePicture** para solicitar al remitente de vídeo que actualice la imagen. Cada vez que se llama a este método, se envía una solicitud al remitente del vídeo. La secuencia debe estar activa.
