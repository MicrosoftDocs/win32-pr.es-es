|-|
| [**DXCoreAdapterMemoryBudget**](./dxcore_interface/ns-dxcore_interface-dxcoreadaptermemorybudget.md). Describe el presupuesto de memoria para un adaptador. |
| [**DXCoreAdapterMemoryBudgetNodeSegmentGroup**](./dxcore_interface/ns-dxcore_interface-dxcoreadaptermemorybudgetnodesegmentgroup.md). Describe un grupo de segmentos de memoria para un adaptador. |
