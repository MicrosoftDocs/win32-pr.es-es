# <a name="wmalbumartist"></a>WM/AlbumArtist

El atributo **WM/AlbumArtist** contiene el nombre del intérprete principal del álbum.

## <a name="global-constant"></a>Constante global

g \_ wszWMAlbumArtist

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

En el caso de los álbumes en los que las pistas pueden tener intérpretes distintos, como la distribución de álbumes o álbumes de colaboración, este atributo se puede usar para especificar el intérprete principal del álbum. Es preferible utilizar este atributo en lugar de usar un valor de autor de "varios intérpretes".
