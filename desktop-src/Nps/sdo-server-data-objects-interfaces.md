# <a name="server-data-objects-interfaces"></a>Interfaces de objetos de datos de servidor

La API de objetos de datos de servidor (SDO) expone las interfaces siguientes.




| Interfaz | Propósito | 
|-----------|---------|
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdo"><strong>ISdo</strong></a> | Manipular un objeto SDO.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdocollection"><strong>ISdoCollection</strong></a> | Manipular una colección de objetos SDO.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdodictionaryold"><strong>ISdoDictionaryOld</strong></a> | Manipular el diccionario de atributos.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdomachine"><strong>ISdoMachine</strong></a> | Administre el equipo SDO y recupere objetos SDO.<br /> | 
| <a href="/windows/desktop/api/sdoias/nn-sdoias-isdoservicecontrol"><strong>ISdoServiceControl</strong></a> | Administre el servicio que se administra a través de SDO, es decir, el Servicio de autenticación de Internet (IAS) o el Servidor de acceso remoto (RAS).<br /><blockquote>[!Note]<br />El nombre del Servicio de autenticación de Internet (IAS) se cambió a Servidor de directivas de red (NPS) a partir Windows Server 2008.</blockquote><br /> |
