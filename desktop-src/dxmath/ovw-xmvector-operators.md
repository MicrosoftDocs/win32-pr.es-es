# <a name="xmvector-operators"></a>Operadores XMVECTOR

La estructura [**XMVECTOR**](xmvector-data-type.md) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                    | Descripción                                                                                                                                 |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**operador + =**](/previous-versions/windows/desktop/legacy/ee421394(v=vs.85))<br/>  | Agrega un valor de punto flotante a una `XMVECTOR` instancia de y devuelve una referencia a la instancia actualizada. <br/>                         |
| [**operador-=**](/previous-versions/windows/desktop/legacy/ee421383(v=vs.85))<br/>    | Resta un valor de punto flotante de la instancia actual de `XMVECTOR` , y devuelve el resultado en la instancia actual actualizada. <br/> |
| [**operador \** de _](xmvector-operator-mul.md)<br/>    | Operadores de multiplicación<br/>                                                                                                         |
| [_ *( \* = operador* )*](xmvector-operator-muleq.md)<br/> | Operadores de asignación y multiplicación<br/>                                                                                              |
| [**Operator**](xmvector-operator-div.md)<br/>     | Operador de división.<br/>                                                                                                               |
| [**operador/=**](xmvector-operator-diveq.md)<br/>  | Operador de asignación y división.<br/>                                                                                                    |
| [**Operator**](xmvector-operator--.md)<br/>       | Operadores de resta y negación<br/>                                                                                               |
| [**operador +**](xmvector-operator-pls.md)<br/>     | Operadores de suma.<br/>                                                                                                              |
