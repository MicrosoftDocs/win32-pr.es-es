# <a name="formatting-text-windows-gdi"></a>Aplicar formato al texto (Windows GDI)

Las funciones de formato se pueden dividir en tres [](text-formatting-attributes.md) categorías: las que recuperan o establecen los atributos de formato de texto para un contexto de dispositivo, las que recuperan anchos de caracteres y las que recuperan anchos y [alturas](character-widths.md)de cadena.
