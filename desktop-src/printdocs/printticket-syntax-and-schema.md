# <a name="printticket-syntax-and-schema"></a>Esquema y sintaxis de PrintTicket

Este tema no está actualizado. Para obtener la información más reciente, consulte la [especificación del esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Elementos PrintTicket se expresan en la sintaxis del marco de esquema de impresión basado en XML. Por lo tanto, el lector debe revisar el [Resumen de los tipos de elemento](summary-of-element-types.md) para la sintaxis necesaria.

El esquema PrintTicket es muy similar al marco de esquema de impresión en cuanto a los tipos de elemento usados y en la estructura general, expresados por elementos primarios y secundarios. Tenga en cuenta que los elementos ParameterDef no se permiten en un PrintTicket y que los elementos ParameterInit solo se permiten en un PrintTicket.
