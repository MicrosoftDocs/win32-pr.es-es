# <a name="community-names"></a>Community Nombres

Un nombre de comunidad identifica una colección de agentes y administradores de SNMP. El uso de un nombre de comunidad proporciona seguridad primitiva y comprobación de contexto para agentes y administradores que reciben solicitudes e inician operaciones de captura. Un agente no aceptará una solicitud de un administrador fuera de la comunidad.

Tenga en cuenta que las recomendaciones de seguridad estándar recomiendan no usar SNMP excepto en redes de confianza, ya que el protocolo, por diseño, proporciona una seguridad mínima. Para obtener más información, vea [Configuring SNMP Security](configuring-snmp-security.md).
