# <a name="iupdateservice2-properties"></a>Propiedades de IUpdateService2

La interfaz **IUpdateService2** define la propiedad siguiente.



| Propiedad                                                         | Descripción                                                                                                                                                                         |
|------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IsDefaultAUService**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservice2-get_isdefaultauservice) | Obtiene un valor booleano que indica si el servicio está registrado con Actualizaciones automáticas y si el servicio está siendo utilizado por Actualizaciones automáticas como el servicio predeterminado. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz.

-   [**IUpdateService**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservice)
