# <a name="tree-view-control-msaa-ui-element-reference"></a>Control de vista de árbol (referencia de elementos de interfaz de usuario de MSAA)

> [!Note]  
> En este tema se describen los objetos de **control de vista de árbol** para fines de referencia de elementos de interfaz de usuario de MSAA. Cómo crear objetos de **control de vista de árbol** en varios marcos de interfaz de usuario no se describe aquí. Consulte la documentación de referencia de la API del marco de interfaz de usuario que está usando.

 

Un control de vista de árbol muestra una lista jerárquica de elementos con líneas para conectar los elementos relacionados en una jerarquía. Cada elemento se compone de una etiqueta y un mapa de bits opcional. El explorador de Windows utiliza un control de vista de árbol para mostrar los directorios.

El nombre de clase de ventana de un control de vista de árbol es WC \_ TREEVIEW, que se define como "SysTreeView" en commctrl. h.

|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

Dependiendo de la propiedad **State** de un elemento en la vista de árbol, **accDoDefaultAction** expande o contrae el elemento especificado. |

## <a name="notes"></a>Notas

La propiedad **accLocation** de los elementos de la vista de árbol devuelve la ubicación solo del texto del elemento. Las coordenadas devueltas por **accLocation** para los elementos de la vista de árbol no incluyen el icono situado a la izquierda del texto.
