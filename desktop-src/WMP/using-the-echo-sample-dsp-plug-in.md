# <a name="using-the-echo-sample-dsp-plug-in"></a>Usar el complemento DSP de ejemplo echo

Cuando cree una compilación sin errores de su complemento de Windows Media Player eco de DSP, puede usarlo en Windows Media Player para oír el efecto. Al iniciar Windows Media Player, el complemento debe estar habilitado de forma predeterminada. Si reproduce audio o vídeo con audio, debe oír un eco de un segundo en el sonido que se reproduce en el mismo volumen que el sonido original.

Puede experimentar con diferentes tiempos de retraso y niveles de efectos cambiando los valores de la página de propiedades del complemento eco. Pruebe el efecto en los distintos tipos de material de origen, como música o voz. Un tiempo de retardo de 200 ms con un nivel de efectos de 30 dará como resultado un eco muy espaciado que suena como el efecto que se usa en las grabaciones tempranas y las primeras. Los niveles de efectos mayores que 50 darán lugar a que la señal original sea inferior en el volumen que el eco, lo que se denomina efecto de eco previo.
