## <a name="print-schema-overview"></a><span data-ttu-id="bf039-109">Información general sobre el esquema de impresión</span><span class="sxs-lookup"><span data-stu-id="bf039-109">Print Schema Overview</span></span>

<span data-ttu-id="bf039-110">El esquema de impresión es un esquema basado en XML estructurado jerárquicamente que se utiliza para organizar y describir las propiedades de una impresora o un trabajo de impresión.</span><span class="sxs-lookup"><span data-stu-id="bf039-110">The Print Schema is a hierarchically structured, XML-based schema that is used to organize and describe the properties of a printer or print job.</span></span> <span data-ttu-id="bf039-111">El esquema de impresión incluye dos componentes: las palabras clave del esquema de impresión y el marco de trabajo del esquema de impresión.</span><span class="sxs-lookup"><span data-stu-id="bf039-111">The Print Schema includes two components: the Print Schema Keywords and the Print Schema Framework.</span></span> <span data-ttu-id="bf039-112">Las palabras clave del esquema de impresión son un conjunto de instancias de elementos que describen los atributos de la impresora y el intento de formato del trabajo de impresión.</span><span class="sxs-lookup"><span data-stu-id="bf039-112">The Print Schema Keywords are a set of element instances that describe printer attributes and print job formatting intent.</span></span> <span data-ttu-id="bf039-113">El marco de trabajo del esquema de impresión define una colección jerárquicamente estructurada de tipos de elementos XML y cómo estos tipos de elementos se pueden usar juntos.</span><span class="sxs-lookup"><span data-stu-id="bf039-113">The Print Schema Framework defines a hierarchically structured collection of XML element types and how these element types can be used together.</span></span>

<span data-ttu-id="bf039-114">Las tecnologías de esquema de impresión, denominadas *PrintCapabilities* y *PrintTicket*, se crean con las palabras clave del esquema de impresión tal y como se especifica en el marco de trabajo del esquema de impresión.</span><span class="sxs-lookup"><span data-stu-id="bf039-114">The Print Schema technologies, called *PrintCapabilities* and *PrintTicket*, are built using the Print Schema Keywords as specified by the Print Schema Framework.</span></span> <span data-ttu-id="bf039-115">La especificación del esquema de impresión admite extensiones de esquema por parte de terceros, de modo que los usuarios del esquema de impresión no se limiten a las instancias de propiedad, característica, opción o ParameterInit definidas por las palabras clave del esquema de impresión.</span><span class="sxs-lookup"><span data-stu-id="bf039-115">The Print Schema Specification supports schema extensions by third parties, so that Print Schema users are not restricted to the Property, Feature, Option, or ParameterInit instances that are defined by the Print Schema Keywords.</span></span> <span data-ttu-id="bf039-116">Las instancias de elemento de terceros se pueden agregar a instancias de elemento definidas por las palabras clave del esquema de impresión; sin embargo, las instancias de propiedad de terceros privadas deben pertenecer a un espacio de nombres que esté claramente asociado al tercero que creó el espacio de nombres.</span><span class="sxs-lookup"><span data-stu-id="bf039-116">Third-party element instances can be added to element instances that are defined by the Print Schema Keywords; however, private, third-party Property instances must belong to a namespace that is clearly associated with the third party that created the namespace.</span></span>
