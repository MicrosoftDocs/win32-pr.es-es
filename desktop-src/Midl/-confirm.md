# <a name="confirm-switch"></a>modificador/CONFIRM

El modificador **/CONFIRM** indica al compilador que muestre todas las opciones del compilador de MIDL sin procesar los archivos IDL y opcionales de entrada.

``` syntax
midl /confirm
```

## <a name="switch-options"></a>Opciones de conmutador

Este modificador no tiene parámetros.

## <a name="examples"></a>Ejemplos

**/CONFIRM de MIDL**

**MIDL/CONFIRM @response.rsp nombrearchivo. idl**
