# <a name="runningtaskcollection-object"></a>Objeto RunningTaskCollection

Objeto de scripting que proporciona una colección que se utiliza para controlar las tareas en ejecución.

## <a name="members"></a>Miembros

El objeto **RunningTaskCollection** tiene estos tipos de miembros:

-   [Propiedades](#properties)

### <a name="properties"></a>Propiedades

El objeto **RunningTaskCollection** tiene estas propiedades.



| Propiedad                                                | Descripción                                                    |
|:--------------------------------------------------------|:---------------------------------------------------------------|
| [**Contabiliza**](runningtaskcollection-count.md)<br/> | Obtiene el número de tareas en ejecución en la colección.<br/> |
| [**Elemento**](runningtaskcollection-item.md)<br/>   | Obtiene la tarea especificada de la colección. <br/>       |
