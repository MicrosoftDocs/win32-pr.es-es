# <a name="stationery-element"></a>Elemento Stationery

Contiene información sobre la papelería usada en la nota de diario, incluido el color de fondo o la imagen, la información del título y la información de diseño de línea, como reglas estrechas y reglas de margen.

## <a name="definition"></a>Definición

``` syntax
<xs:element name="Stationery" type="StationeryType" minOccurs="0" maxOccurs="unbounded" />
```
