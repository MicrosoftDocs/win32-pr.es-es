# <a name="search-size-limit"></a>Límite de tamaño de búsqueda

Para reducir los requisitos de memoria, un cliente puede centrarse en un pequeño número de objetos devueltos desde el servidor e ignorar el resto del conjunto de resultados. Para ello, el cliente especifica un límite de tamaño de búsqueda y otros criterios de búsqueda adecuados. Por ejemplo, si un directorio almacena las puntuaciones de prueba de un distrito educativo, puede consultar a los diez primeros alumnos con las puntuaciones de prueba más altas especificando un límite de tamaño de diez y un criterio de ordenación descendente.

Para obtener más información sobre el uso de la opción de límite de tamaño de búsqueda con una interfaz de búsqueda específica, vea:

-   [Límite de tamaño con IDirectorySearch](size-limit-with-idirectorysearch.md)
-   [Buscar con ActiveX objetos de datos](searching-with-activex-data-objects-ado.md)
-   [Buscar con OLE DB](searching-with-ole-db.md)
