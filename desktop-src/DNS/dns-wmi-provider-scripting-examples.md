# <a name="dns-wmi-provider-scripting-examples"></a>Ejemplos de scripting del proveedor WMI de DNS

En esta sección se proporcionan ejemplos de scripting que muestran cómo generar scripts de tareas administrativas mediante el proveedor WMI de DNS.

Se proporcionan los siguientes tipos de ejemplos de scripting.



| Sección                                                                                     | Ejemplos proporcionados                                                                                                                                                                          |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Administración de un servidor DNS](dns-wmi-provider-samples-managing-a-dns-server.md)                 | Proporciona ejemplos que permiten la administración de un servidor DNS, como la detención, el inicio o el reinicio de un servidor, la enumeración de zonas, la adición o eliminación de direcciones IP y otras.                |
| [Administración de registros de recursos DNS](dns-wmi-provider-samples-managing-dns-resource-records.md) | Muestra cómo conectarse al proveedor WMI de DNS y cómo realizar varias tareas asociadas a la administración de registros de recursos: agregar, eliminar, modificar o enumerar registros de recursos. |
| [Administrar Zonas DNS](dns-wmi-provider-samples-managing-dns-zones.md)                       | Los ejemplos aquí incluyen la creación, la modificación y la eliminación de una zona, así como la pausa, reanudación, actualización y otras tareas de zona que se plantean a los administradores de DNS.                                  |
