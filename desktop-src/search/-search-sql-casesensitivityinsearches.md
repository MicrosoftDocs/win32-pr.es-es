# <a name="case-sensitivity-in-searches"></a>Distinción de mayúsculas y minúsculas en búsquedas

Las consultas de Microsoft Windows Search no distinguen mayúsculas de minúsculas. Esto incluye las palabras clave de consulta (SELECT es idéntica a SELECT, SELECT y sELect) y los términos de búsqueda. Si un identificador o literal utiliza caracteres Unicode que tienen semántica de asignación de mayúsculas y minúsculas, el motor de búsqueda considera todos los casos equivalentes.
