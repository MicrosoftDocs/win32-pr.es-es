# <a name="dns-property-attribute"></a>DNS-Property atributo)

Se usa para almacenar la configuración binaria (propiedades) en objetos de zona DNS.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | DNS-Property                                          |
| Nombre para mostrar de LDAP | dNSProperty                                           |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | El sistema establece este valor.                      |
| Frecuencia de actualización  | Cada vez que se modifican los registros de la zona DNS.                     |
| Attribute-Id      | 1.2.840.113556.1.4.1306                               |
| System-ID-GUID    | 675a15fe-3b70-11d2-90cc-00c04fd91ab1                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
