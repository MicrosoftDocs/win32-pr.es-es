# <a name="txf-control-codes"></a>Códigos de control TxF

\[Microsoft recomienda encarecidamente que los desarrolladores usen medios alternativos para lograr las necesidades de las aplicaciones. Muchos escenarios para los que se desarrolló TxF se pueden lograr mediante técnicas más sencillas y más fáciles de usar. Además, es posible que TxF no esté disponible en versiones futuras de Microsoft Windows. Para obtener más información y alternativas a TxF, vea [alternativas al uso de NTFS transaccional](deprecation-of-txf.md).\]

NTFS transaccional (TxF) proporciona los siguientes códigos de control.

## <a name="in-this-section"></a>En esta sección



| Código de control                                                                                                 | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                             |
|--------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**FSCTL \_ TXFS \_ crear \_ miniversión**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_create_miniversion)<br/>                         | Crea una nueva [miniversión](glossary.md) para el archivo especificado. <br/> Miniversions permiten hacer referencia a una instantánea del archivo durante una transacción. Miniversions se descartan cuando una transacción se confirma o se revierte.<br/>                                                                                                                                                                      |
| [**\_información de \_ \_ metadatos de FSCTL TXFS Get \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_get_metadata_info)<br/>                          | Recupera los metadatos de NTFS (TxF) transaccionados para un archivo y el **GUID** de la transacción que ha bloqueado el archivo especificado (si el archivo está bloqueado). <br/>                                                                                                                                                                                                                                                                         |
| [**FSCTL \_ TXFS \_ obtener \_ versión de transacción \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_get_transacted_version)<br/>                | Devuelve una estructura de [**\_ \_ \_ versión de transacción de TXFS**](/windows/desktop/api/WinIoCtl/ns-winioctl-txfs_get_transacted_version) . La estructura identifica la versión confirmada más recientemente del archivo especificado, el número de versión del identificador. <br/>                                                                                                                                                                                                            |
| [**\_archivos de \_ \_ transacciones \_ bloqueadas \_ de la lista TXFS de FSCTL**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_list_transaction_locked_files)<br/> | Devuelve una lista de todos los archivos bloqueados actualmente por la transacción especificada. Si el valor devuelto es **error \_ más \_ datos**, devuelve la longitud del búfer necesario para contener la lista completa de archivos en el momento de esta llamada.<br/>                                                                                                                                                                                           |
| [**transacciones de la \_ lista de TXFS FSCTL \_ \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_list_transactions)<br/>                           | Devuelve una lista de todas las transacciones implicadas actualmente en el administrador de recursos especificado.<br/>                                                                                                                                                                                                                                                                                                                                 |
| [**FSCTL \_ TXFS \_ modificar \_ RM**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_modify_rm)<br/>                                           | Establece el modo de registro y la información de parámetros de registro para un administrador de recursos secundario (RM).<br/>                                                                                                                                                                                                                                                                                                                                       |
| [**\_información de \_ RM de consulta de TXFS de FSCTL \_ \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_query_rm_information)<br/>                    | Recupera información de un administrador de recursos (RM).<br/>                                                                                                                                                                                                                                                                                                                                                                           |
| [**FSCTL \_ TXFS \_ leer \_ información de copia de seguridad \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_read_backup_information)<br/>              | Devuelve información específica de NTFS transaccional (TxF) del archivo especificado.<br/>                                                                                                                                                                                                                                                                                                                                                |
| [**\_información de \_ punto de retorno de FSCTL TXFS \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_savepoint_information)<br/>                   | El código de control de la [**\_ información de punto de \_ retorno \_ de FSCTL TXFS**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_savepoint_information) controla, borra y revierte al punto de retorno especificado.<br/> Para realizar esta operación, llame a la función [**DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con los parámetros siguientes.<br/>                                                                                                                 |
| [**transacción de FSCTL \_ TXFS \_ \_ activa**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_transaction_active)<br/>                         | Devuelve un valor booleano que indica si hay transacciones activas en el volumen asociado cuando se tomó la instantánea. Esta llamada solo es válida para volúmenes de instantáneas de solo lectura.<br/>                                                                                                                                                                                                                                   |
| [**FSCTL \_ TXFS \_ escribir \_ información de copia de seguridad \_**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_write_backup_information)<br/>            | Escribe información específica de NTFS transaccional (TxF) en un archivo especificado. El miembro de **búfer** de la estructura de [**información de copia de \_ \_ seguridad \_ de escritura de TXFS**](/windows/desktop/api/WinIoCtl/ns-winioctl-txfs_write_backup_information) debe ser el miembro de **búfer** de la estructura de [**lectura de \_ \_ \_ información \_ de copia**](/windows/desktop/api/WinIoCtl/ns-winioctl-txfs_read_backup_information_out) de seguridad de TXFS devuelta por [**FSCTL \_ TXFS \_ leer \_ \_ información de copia de seguridad**](/windows/win32/api/winioctl/ni-winioctl-fsctl_txfs_read_backup_information).<br/> |
