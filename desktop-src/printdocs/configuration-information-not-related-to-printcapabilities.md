# <a name="configuration-information-not-related-to-printcapabilities"></a>Información de configuración no relacionada con PrintCapabilities

Este tema no está actualizado. Para obtener la información más reciente, consulte la [especificación del esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

Además de proporcionar selecciones para los atributos de dispositivo definidos en PrintCapabilities, el PrintTicket permite que los clientes agreguen información adicional en forma de elementos de propiedad. El esquema de impresión define un número de instancias de propiedad estándar, y el proveedor de la interfaz es libre de introducir también instancias de propiedades privadas. Por ejemplo, si los miembros de una organización envían la mayoría de los trabajos de impresión a una instalación de Batch central, pueden especificar para cada trabajo una instancia de propiedad privada que contenga una dirección de devolución. Esta propiedad puede facilitar la entrega del trabajo completado que se puede entregar al originador.
