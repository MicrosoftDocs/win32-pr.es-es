# <a name="imapiv1-support"></a>Compatibilidad con IMAPIv1

Image Mastering API versión 1 (IMAPIv1) permite que una aplicación se stage y queme una imagen de audio o datos simple en dispositivos CD-R y CD-RW. La API admite los formatos Joliet e ISO 9660 para discos de datos y audio de Redbook.

> [!Note]  
> IMAPI se ha rediseñado completamente para la versión 2.0. La mayor parte de la funcionalidad de la versión 1.0 está disponible en la versión 2.0. Se recomienda a los usuarios que escriban aplicaciones de dominio de imágenes o realicen el desarrollo de nuevos dispositivos y formatos para usar la [versión 2.0](using-imapi.md) en lugar de la versión 1.0. Si ejecuta una aplicación de la versión 1.0 en Windows Vista, las interfaces de la versión 1.0 convertirán esas llamadas en las llamadas de la versión 2.0 adecuadas para lograr una funcionalidad idéntica.

 

Para obtener más información sobre el uso de la versión 1.0 en Windows XP, consulte los temas siguientes:

-   [Acerca de Image Mastering API](about-the-image-mastering-api.md)
-   [Uso de Image Mastering API](using-the-image-mastering-api.md)
-   [Referencia de IMAPIv1](imapiv1-reference.md)
