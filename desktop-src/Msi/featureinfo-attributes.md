# <a name="featureinfoattributes-property"></a>Propiedad FeatureInfo. Attributes

La propiedad **attributes** es una propiedad de lectura y escritura que devuelve el valor de la característica en la columna Attributes de la [tabla de características](feature-table.md).

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = FeatureInfo.Attributes
FeatureInfo.Attributes = propVal 
```
