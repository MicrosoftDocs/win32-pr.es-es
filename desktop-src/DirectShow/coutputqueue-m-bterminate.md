# <a name="coutputqueuem_bterminate-member"></a>Miembro bTerminate COutputQueue:: m \_

Marca que especifica si el subproceso debe finalizar. Si **es true**, el subproceso finaliza.

## <a name="syntax"></a>Sintaxis


```C++
BOOL m_bTerminate;
```
