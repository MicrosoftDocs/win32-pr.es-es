# <a name="lineroammode_-constants"></a>Constantes LINEROAMMODE \_

Las constantes de marca de bits **LINEROAMMODE \_** describen el estado de itinerancia de un dispositivo de línea.

<dl> <dt>

<span id="LINEROAMMODE_HOME"></span><span id="lineroammode_home"></span>**LINEROAMMODE \_ HOME**
</dt> <dd> <dl> <dt>



La línea está conectada al nodo de red principal.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_ROAMA"></span><span id="lineroammode_roama"></span>**LINEROAMMODE \_ ROAMA**
</dt> <dd> <dl> <dt>



La línea está conectada al operador Roam-A y las llamadas se cobran en consecuencia.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_ROAMB"></span><span id="lineroammode_roamb"></span>**LINEROAMMODE \_ ROAMB**
</dt> <dd> <dl> <dt>



La línea está conectada al operador Roam-B y las llamadas se cobran en consecuencia.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_UNAVAIL"></span><span id="lineroammode_unavail"></span>**LINEROAMMODE \_ UNAVAIL**
</dt> <dd> <dl> <dt>



El modo de roam no está disponible y no se conocerá.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_UNKNOWN"></span><span id="lineroammode_unknown"></span>**LINEROAMMODE \_ UNKNOWN**
</dt> <dd> <dl> <dt>



El modo de roam es desconocido actualmente, pero puede conocerse más adelante.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Comentarios

Sin extensibilidad. Los 32 bits están reservados.
