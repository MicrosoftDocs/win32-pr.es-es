# <a name="error-handling-error-handling"></a>Control de errores (control de errores)

Las aplicaciones bien escritas incluyen código de control de errores que les permite recuperarse correctamente de errores inesperados. Cuando se produce un error, es posible que la aplicación tenga que solicitar la intervención del usuario o que pueda recuperarse por sí misma. En casos extremos, la aplicación puede cerrar o cerrar el sistema al usuario.

-   [Acerca del control de errores](about-error-handling.md)
-   [Uso del control de errores](using-error-handling.md)
-   [Referencia de control de errores](error-handling-reference.md)

Para obtener información sobre el control de excepciones, vea [Structured Exception Handling](structured-exception-handling.md).
