# <a name="snmp-compiler-error-messages"></a>Mensajes de error del compilador SNMP

En esta sección se describen las distintas condiciones de error que pueden producirse cuando el compilador del módulo de información del Protocolo simple de administración de redes (SNMP) analiza un módulo de la base de datos de información de administración (MIB).

Los mensajes de error tienen tres niveles de gravedad:

-   Errores graves
-   Advertencias
-   Mensajes de información

El nivel de gravedad se muestra en corchetes angulares al principio de la descripción del mensaje de error. A continuación se muestra una lista de errores del proveedor SNMP de WMI:

-   [Errores de 281 a 290](errors-281-through-290.md)
-   [Errores de 1001 a 1010](errors-1001-through-1010.md)
-   [Errores de 1011 a 1020](errors-1011-through-1020.md)
-   [Errores de 1021 a 1030](errors-1021-through-1030.md)
-   [Errores de 1031 a 1040](errors-1031-through-1040.md)
-   [Errores de 1041 a 1050](errors-1041-through-1050.md)
-   [Errores de 1051 a 1060](errors-1051-through-1060.md)
-   [Errores de 1061 a 1070](errors-1061-through-1070.md)
-   [Errores de 1071 a 1080](errors-1071-through-1080.md)
