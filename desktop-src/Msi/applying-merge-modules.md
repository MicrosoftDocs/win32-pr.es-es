# <a name="applying-merge-modules"></a>Aplicar módulos de combinación

Los módulos de combinación se deben combinar en un paquete de instalación mediante una herramienta de combinación. La mejor alternativa es obtener una herramienta de combinación distribuida de forma gratuita o comprar una de las herramientas de combinación para que esté disponible en proveedores de software independientes. Por ejemplo, puede usar la funcionalidad proporcionada por Mergemod.dll.

Los módulos de combinación en varios idiomas se pueden abrir en lenguajes diferentes y, a continuación, combinarlos en un destino para generar una versión de idioma diferente del paquete. Para más información, consulte:

-   [Usar la automatización para combinar un módulo de combinación en una base de datos](using-automation-to-merge-a-merge-module-into-a-database.md)
-   [Uso de la API para combinar un módulo de combinación en una base de datos](using-the-api-to-merge-a-merge-module-into-a-database.md)
-   [Abrir un módulo de combinación de Multiple-Language en un idioma específico](opening-a-multiple-language-merge-module-in-a-specific-language.md)
-   [Combinar un módulo de varios idiomas en el mismo paquete varias veces](merging-a-multiple-language-module-into-the-same-package-multiple-times.md)
-   [Conectar un módulo de combinación a varias características](connecting-a-merge-module-to-multiple-features.md)
