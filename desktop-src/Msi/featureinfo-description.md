# <a name="featureinfodescription-property"></a>FeatureInfo. Description (propiedad)

La propiedad **Description** es una propiedad de solo lectura que devuelve la descripción de la característica en la columna Descripción de la [tabla de características](feature-table.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = FeatureInfo.Description
```



## <a name="property-value"></a>Valor de propiedad
