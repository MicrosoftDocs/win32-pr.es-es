# <a name="ms-sql-contact-attribute"></a>Atributo MS-SQL-Contact

Cadena definida por el usuario. El valor predeterminado se establece en contacto.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Contacto de MS-SQL                              |
| Nombre para mostrar de LDAP | Contacto de mS-SQL                              |
| Tamaño              | \-                                          |
| Actualizar privilegio  | Administrador de dominio                        |
| Frecuencia de actualización  | En el programa de instalación del sistema.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1365                     |
| System-ID-GUID    | 4f6cbdd8-ccee-11d2-9993-0000f87a57d4        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
