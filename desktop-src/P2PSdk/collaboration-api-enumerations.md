# <a name="collaboration-api-enumerations"></a>Enumeraciones de API de colaboración

Las enumeraciones siguientes admiten las funciones de la infraestructura de colaboración del mismo nivel.



| Enumeración                                                                         | Descripción                                                                                                                                                                                                                                                                   |
|-------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_tipo de \_ registro de aplicación del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_application_registration_type) | Define el conjunto de marcas de registro de la aplicación del mismo nivel.                                                                                                                                                                                                                       |
| [**\_tipo de cambio del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_change_type)                                      | Define el conjunto de cambios que se pueden realizar en un objeto del mismo nivel, extremo o aplicación.                                                                                                                                                                                  |
| [**\_marcas de inicio de sesión del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_signin_flags)                                    | Define el conjunto de comportamientos de publicación de presencia del mismo nivel disponibles cuando el mismo nivel inicia sesión en una red de colaboración del mismo nivel.                                                                                                                                                      |
| [**\_permiso de inspección del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_watch_permission)                            | Define el conjunto de permisos que representan la presencia de un contacto en un elemento del mismo nivel de inspección.                                                                                                                                                                                        |
| [**\_tipo de \_ evento peer COLLAB \_**](/windows/desktop/api/P2P/ne-p2p-peer_collab_event_type)                         | Define el conjunto de eventos que se puede producir en un elemento del mismo nivel mediante la infraestructura de eventos de la red de colaboración del mismo nivel.                                                                                                                                                                |
| [**\_Estado de presencia del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_presence_status)                              | Define el conjunto de posibles valores de estado de presencia disponibles para un elemento del mismo nivel que participa en una red de colaboración del mismo nivel. Estos valores se pueden establecer mediante un punto de conexión de la red de colaboración del mismo nivel para indicar el nivel actual de participación del contacto del mismo nivel con sus monitores. |
| [**\_tipo de \_ respuesta de invitación del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_invitation_response_type)           | Define el conjunto de respuestas recibido a una invitación para unirse a una actividad de colaboración del mismo nivel.                                                                                                                                                                                 |
| [**\_ámbito de publicación del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_publication_scope)                          | Define el conjunto de ámbitos para la publicación de objetos o datos del mismo nivel.                                                                                                                                                                                                        |
