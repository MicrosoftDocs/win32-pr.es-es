# <a name="embedding-the-windows-media-player-control-in-a-web-page"></a>Incrustar el control Media Player de Windows en una página web

Al incrustar el control Media Player de Windows en una página web, puede adoptar un enfoque básico o un enfoque más elaborado.

En el enfoque básico, el control se usa simplemente para reproducir música en segundo plano o para agregar una pantalla de vídeo a la página.

En el enfoque más elaborado, puede hacer que la página sea interactiva agregando código de script para manipular el control a través del modelo de objetos del reproductor. Esto le permite proporcionar una interfaz de usuario de su propio diseño o responder dinámicamente a comandos o eventos de script, como transiciones de medios.

Para obtener más información, vea [usar el Control Media Player de Windows en una página web](using-the-windows-media-player-control-in-a-web-page.md).
