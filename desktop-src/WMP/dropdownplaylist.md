# <a name="dropdownplaylist"></a>DROPDOWNPLAYLIST

Este es un elemento de **lista de reproducción** predefinido con los siguientes valores predeterminados.

``` syntax
playlistItemsVisible="false"
```

## <a name="remarks"></a>Observaciones

Esto creará una **lista** desplegable simple sin elementos mostrados. Todas las propiedades de esta **lista de reproducción** se pueden invalidar mediante su especificación explícita.

| Requisito | Value |
|--------------------|----------------------------------------------|
| Versión<br/> | Windows Media Player 7,0 o posterior<br/> |
