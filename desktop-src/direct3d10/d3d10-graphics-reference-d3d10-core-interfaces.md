# <a name="direct3d-10-core-interfaces"></a>Interfaces de Direct3D 10 Core

Esta sección contiene información acerca de las siguientes interfaces principales:



| Interfaces                                                           | Descripción                                         |
|----------------------------------------------------------------------|-----------------------------------------------------|
| [**Interfaz ID3D10BlendState**](/windows/desktop/api/D3D10/nn-d3d10-id3d10blendstate)               | Obtiene acceso al estado de fusión de un dispositivo Direct3D 10,0. |
| [**Interfaz ID3D10BlendState1**](/windows/desktop/api/D3D10_1/nn-d3d10_1-id3d10blendstate1)             | Obtiene acceso al estado de fusión de un dispositivo Direct3D 10,1. |
| [**Interfaz ID3D10DepthStencilState**](/windows/desktop/api/D3D10/nn-d3d10-id3d10depthstencilstate) | Obtiene acceso al estado de la galería de símbolos de profundidad.                       |
| [**Interfaz ID3D10InputLayout**](/windows/win32/api/d3d10/nn-d3d10-id3d10inputlayout)             | Obtiene acceso a los datos de entrada de canalización de la memoria.           |
| [**Interfaz ID3D10RasterizerState**](/windows/desktop/api/D3D10/nn-d3d10-id3d10rasterizerstate)     | Obtiene acceso al estado del rasterizador.                          |
| [**Interfaz ID3D10SamplerState**](/windows/desktop/api/D3D10/nn-d3d10-id3d10samplerstate)           | Obtiene acceso al estado de muestra.                             |



 



| Interfaces                                                 | Descripción                                                                          |
|------------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**Interfaz ID3D10Asynchronous**](/windows/desktop/api/D3D10/nn-d3d10-id3d10asynchronous) | Recupera datos de la GPU de forma asincrónica.                                          |
| [**Interfaz ID3D10Blob**](/windows/desktop/api/D3DCommon/nn-d3dcommon-id3d10blob)                 | Devolver datos de la memoria.                                                             |
| [**Interfaz ID3D10Counter**](/windows/desktop/api/D3D10/nn-d3d10-id3d10counter)           | Mide el rendimiento de la GPU.                                                            |
| [**Interfaz ID3D10Debug**](/windows/desktop/api/D3D10SDKLayers/nn-d3d10sdklayers-id3d10debug)               | Habilita o deshabilita la capa de depuración.                                                    |
| [**Interfaz ID3D10Device**](/windows/desktop/api/D3D10/nn-d3d10-id3d10device)             | Representa un adaptador virtual para Direct3D 10,0.                                      |
| [**Interfaz ID3D10Device1**](/windows/desktop/api/D3D10_1/nn-d3d10_1-id3d10device1)           | Representa un adaptador virtual para Direct3D 10,1.                                      |
| [**Interfaz ID3D10DeviceChild**](/windows/desktop/api/D3D10/nn-d3d10-id3d10devicechild)   | Obtiene acceso a los datos usados por un dispositivo.                                                      |
| [**Interfaz ID3D10Include**](/previous-versions/windows/desktop/legacy/bb173775(v=vs.85))           | Proporciona métodos reemplazables por el usuario para administrar archivos de inclusión al cargar un efecto. |
| [**Interfaz ID3D10InfoQueue**](/windows/desktop/api/D3D10SDKLayers/nn-d3d10sdklayers-id3d10infoqueue)       | Almacena, recupera y filtra los mensajes de depuración.                                       |
| [**Interfaz ID3D10Multithread**](/windows/desktop/api/D3D10/nn-d3d10-id3d10multithread)   | Obtiene acceso a la configuración de multiproceso.                                                       |
| [**Interfaz ID3D10Predicate**](/windows/win32/api/d3d10/nn-d3d10-id3d10predicate)       | Determina si se debe procesar la geometría.                                     |
| [**Interfaz ID3D10Query**](/windows/desktop/api/D3D10/nn-d3d10-id3d10query)               | Consulta información de la GPU.                                                    |
| [**Interfaz ID3D10StateBlock**](/windows/desktop/api/d3d10effect/nn-d3d10effect-id3d10stateblock)     | Encapsula los Estados de representación.                                                          |
| [**Interfaz ID3D10SwitchToRef**](/windows/desktop/api/D3D10SDKLayers/nn-d3d10sdklayers-id3d10switchtoref)   | Cambia entre un dispositivo de hardware y software.                                     |
