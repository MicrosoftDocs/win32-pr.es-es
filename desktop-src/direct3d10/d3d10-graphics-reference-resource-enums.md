# <a name="resource-enumerations-direct3d-10-graphics"></a>Enumeraciones de recursos (gráficos de Direct3D 10)

Las enumeraciones siguientes se utilizan para especificar información sobre cómo se crean y se obtiene acceso a los recursos durante la representación.



| Enumeración                                                     | Descripción                                                                                                               |
|-----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**D3D10 \_ marca de enlace \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_bind_flag)                    | Identifica cómo usará un recurso la canalización y determina a qué fases de canalización se puede enlazar un recurso. |
| [**\_Marca de \_ acceso de CPU de D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_cpu_access_flag)       | Especifica los tipos de acceso de CPU permitidos para un recurso.                                                                 |
| [**D3D10 \_ \_ dimensión DSV**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_dsv_dimension)            | Especifica cómo obtener acceso a un recurso que se usa en una vista de galería de símbolos de profundidad.                                                  |
| [**\_Mapa D3D10**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_map)                                 | Identifica un recurso que se va a asignar para lectura y escritura de la CPU.                                                    |
| [**D3D10 \_ marca de mapa \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_map_flag)                      | Especifica cómo debe responder la CPU a los métodos de asignación cuando la GPU todavía no ha finalizado con el recurso.               |
| [**\_Dimensión de recursos D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_resource_dimension)  | Identifica el tipo de recurso que está en uso.                                                                           |
| [**\_Marca de recursos \_ varios \_ de D3D10**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_resource_misc_flag) | Identifica las opciones de creación de recursos menos comunes.                                                                         |
| [**D3D10 \_ dimensión de RTV \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_rtv_dimension)            | Especifica cómo obtener acceso a un recurso usado en una vista de destino de representación.                                                          |
| [**Uso de D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_usage)                             | Identifica cómo se espera que un recurso se lea y escriba en la CPU o la GPU.                                   |
