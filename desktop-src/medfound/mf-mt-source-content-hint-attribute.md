# <a name="mf_mt_source_content_hint-attribute"></a>Atributo MF \_ MT \_ SOURCE CONTENT \_ \_ HINT

Describe la relación de aspecto prevista para un tipo de medio de vídeo.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Comentarios

El valor de este atributo es miembro de la [**enumeración MFVideoSrcContentHintFlags.**](/windows/desktop/api/mfapi/ne-mfapi-mfvideosrccontenthintflags)

La constante GUID para este atributo se exporta desde mfuuid.lib.
