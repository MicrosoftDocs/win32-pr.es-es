# <a name="terminal-services-is-now-remote-desktop-services"></a>Terminal Services es ahora Servicios de Escritorio remoto

Se ha cambiado el nombre de Terminal Services Servicios de Escritorio remoto.

En Windows Server 2008 R2, se ha cambiado el nombre de todos los servicios de rol Servicios de Escritorio remoto. En la tabla siguiente se muestra el nombre anterior y el nuevo nombre de cada servicio de rol Servicios de Escritorio remoto.



| Nombre anterior                                                   | Nombre en Windows Server 2008 R2                                         |
|-----------------------------------------------------------------|------------------------------------------------------------------------|
| Terminal Services <br/>                                   | Servicios de Escritorio remoto<br/>                                     |
| Terminal Server<br/>                                      | Host de sesión de Escritorio remoto (Host de sesión de RD)<br/>               |
| Licencias de Terminal Services (licencias de TS)<br/>          | Administrador de licencias de Escritorio remoto (Administrador de licencias de RD)<br/>                     |
| Puerta de enlace de Terminal Services (Puerta de enlace de TS)<br/>               | Puerta de enlace de Escritorio remoto (Puerta de enlace de RD)<br/>                         |
| Agente de sesiones de Terminal Services (Agente de sesiones de TS)<br/> | Agente de Conexión a Escritorio remoto (agente de conexión a escritorio remoto)<br/>     |
| Terminal Services Web Access (TS Web Access)<br/>         | Escritorio remoto Web Access (acceso web de escritorio remoto)<br/>                  |
| Virtualización Terminal Services<br/>                     | Host de virtualización de Escritorio remoto (Host de virtualización de RD)<br/> |



 

Además, tenga en cuenta que un servidor que tiene instalado el servicio de rol host de sesión de escritorio remoto ahora se denomina servidor host de sesión de escritorio remoto, en lugar de un servidor de Terminal Server.
