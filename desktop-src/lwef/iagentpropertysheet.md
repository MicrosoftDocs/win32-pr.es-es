# <a name="iagentpropertysheet"></a>IAgentPropertySheet

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

**IAgentPropertySheet** define una interfaz que permite a las aplicaciones establecer y consultar las propiedades de la hoja de propiedades (ventana) de Microsoft Agent.

**Métodos en orden de Vtable**



| Métodos IAgentPropertySheet                             | Descripción                                                                                  |
|---------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [**GetVisible**](iagentpropertysheet--getvisible.md)   | Devuelve si la hoja de propiedades de Microsoft Agent está visible.                               |
| [**SetVisible**](iagentpropertysheet--setvisible.md)   | Establece la [**propiedad Visible**](visible-property.md) de la hoja de propiedades de Microsoft Agent. |
| [**GetPosition**](iagentpropertysheet--getposition.md) | Devuelve la posición de la hoja de propiedades de Microsoft Agent.                                  |
| [**GetSize**](iagentpropertysheet--getsize.md)         | Devuelve el tamaño de la hoja de propiedades de Microsoft Agent.                                      |
| [**GetPage**](iagentpropertysheet--getpage.md)         | Devuelve la página actual de la hoja de propiedades de Microsoft Agent.                             |
| [**SetPage**](iagentpropertysheet--setpage.md)         | Establece la página actual de la hoja de propiedades de Microsoft Agent.                                |
