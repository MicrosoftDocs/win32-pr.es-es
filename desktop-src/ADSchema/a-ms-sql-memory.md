# <a name="ms-sql-memory-attribute"></a>Atributo MS-SQL-Memory

Cantidad de memoria en el equipo.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-SQL-Memory                        |
| Ldap-Display-Name | mS-SQL-Memory                        |
| Size              | 8 bytes                              |
| Privilegio actualizar  | El sistema establece este valor.     |
| Frecuencia de actualización  | Al iniciar el sistema.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1367              |
| System-Id-Guid    | 5b5d448c-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Intervalo**](s-interval.md)       |
