# <a name="custom-action-in-script-execution-options"></a>Opciones de ejecución de la acción personalizada In-Script

Puede usar las siguientes marcas de opciones para especificar la ejecución en el script de las acciones personalizadas. Estas opciones copian el código de acción en el script de ejecución, reversión o confirmación. Para establecer una opción, agregue el valor de esta tabla al valor del campo Type de la [tabla CustomAction](customaction-table.md).

Tenga en cuenta que **msidbCustomActionTypeInScript** debe incluirse con cada una de estas opciones.
