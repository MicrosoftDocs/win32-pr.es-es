# <a name="default-values-in-the-spi"></a>Valores predeterminados en el SPI

Los detalles de implementación de Calidad de servicio (QoS) se proporcionan en una sección de QoS independiente y dedicada en el Kit de desarrollo de software (SDK) de Microsoft Windows. Consulte la sección [Calidad de servicio](/previous-versions/windows/desktop/qos/qos-start-page) del SDK de Windows para obtener información sobre las directrices de uso de QoS con respecto a los valores predeterminados.
