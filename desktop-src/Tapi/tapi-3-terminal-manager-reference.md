# <a name="tapi-3-terminal-manager-reference"></a>Referencia de TAPI 3 terminal Manager



| Interfaces del administrador de Terminal Server                      | Descripción                                                                                                                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ITTerminalControl**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalcontrol)   | Proporciona métodos para conectar y desconectar un terminal de un gráfico de filtros y ejecutar o detener un representador.                                                                                 |
| [**ITTerminalManager**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager)   | Proporciona métodos para crear terminales dinámicos.                                                                                                                                          |
| [**ITTerminalManager2**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager2) | Proporciona métodos para recuperar información sobre las clases de terminal conectables registradas en el sistema actual; deriva de la interfaz [**ITTerminalManager**](/windows/desktop/api/Termmgr/nn-termmgr-itterminalmanager) . |
