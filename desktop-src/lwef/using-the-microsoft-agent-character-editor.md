# <a name="using-the-microsoft-agent-character-editor"></a>Usar el editor de caracteres del agente de Microsoft

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

El editor de caracteres del agente de Microsoft le permite compilar animaciones de caracteres para su uso con Microsoft Agent. Puede definir animaciones importando imágenes de mapa de bits de Windows, estableciendo su duración y, opcionalmente, incluyendo bifurcaciones, efectos sonoros y superposiciones de hablar. Para obtener información sobre cómo diseñar animaciones de un carácter, vea [diseñar caracteres para el agente de Microsoft](designing-characters-for-microsoft-agent.md).

Con el editor de caracteres del agente puede definir el nombre y la descripción del carácter, así como las opciones de salida, como la conversión de texto a voz (TTS), la salida de voz sintetizada, la compatibilidad con el menú emergente y el diseño del globo de palabras del carácter.

En este documento se describe el editor de caracteres del agente de Microsoft para el agente de Microsoft que puede descargar de <https://www.microsoft.com/downloads/details.aspx?displaylang=en&FamilyID=c21be0ec-1712-427d-a43d-94c122ef8eb6> .
