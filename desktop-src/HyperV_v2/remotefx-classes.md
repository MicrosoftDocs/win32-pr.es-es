# <a name="remote-desktop-classes"></a>Clases de Escritorio remoto

Las clases WMI de virtualización Escritorio remoto se usan para administrar los recursos de Escritorio remoto y RemoteFX y las máquinas virtuales.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                                            | Descripción                                                                                                                                                  |
|------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ InteractiveSessionACE**](msvm-interactivesessionace.md)<br/>                                     | Representa una *entrada de control de acceso* (ACE) que determina el acceso a la sesión interactiva de una máquina virtual.<br/>                                |
| [**MSVM \_ Physical3dGraphicsProcessor**](msvm-physical3dgraphicsprocessor.md)<br/>                         | Describe la unidad de procesamiento de gráficos (GPU) física 3D.<br/>                                                                                        |
| [**MSVM \_ PhysicalGPUInfo**](msvm-physicalgpuinfo.md)<br/>                                                 | Contiene información sobre una unidad de procesamiento de gráficos (GPU) física de RemoteFX.<br/>                                                                    |
| [**MSVM \_ RdvComponent**](msvm-rdvcomponent.md)<br/>                                                       | Representa el estado del componente RDV, que es responsable de proporcionar un transporte para el elemento primario al invitado con fines de configuración.<br/> |
| [**MSVM \_ RdvComponentSettingData**](msvm-rdvcomponentsettingdata.md)<br/>                                 | Representa el estado configurado del componente de virtualización de Escritorio remoto (RDV). El estado predeterminado es habilitado.<br/>                               |
| [**MSVM \_ Synth3dVideoPool**](msvm-synth3dvideopool.md)<br/>                                               | Contiene información acerca de las unidades de procesamiento de gráficos (GPU) sintéticas de vídeo 3D disponibles en el sistema host.<br/>                                 |
| [**MSVM \_ Synthetic3DDisplayController**](msvm-synthetic3ddisplaycontroller.md)<br/>                       | Representa el controlador de pantalla sintético 3D que se asigna a una máquina virtual.<br/>                                                            |
| [**MSVM \_ Synthetic3DDisplayControllerSettingData**](msvm-synthetic3ddisplaycontrollersettingdata.md)<br/> | Representa la configuración de un controlador de pantalla sintético 3D para una máquina virtual.<br/>                                                                 |
| [**MSVM \_ Synthetic3DService**](msvm-synthetic3dservice.md)<br/>                                           | Describe el servicio de GPU 3D sintético.<br/>                                                                                                          |
| [**MSVM \_ Synthetic3DServiceSettingData**](msvm-synthetic3dservicesettingdata.md)<br/>                     | Representa la configuración para el servicio 3D sintético presente en un sistema host único.<br/>                                                            |
| [**MSVM \_ TerminalService**](msvm-terminalservice.md)<br/>                                                 | Administra todas las conexiones remotas de terminal a un host determinado.<br/>                                                                                     |
| [**MSVM \_ TerminalServiceSettingData**](msvm-terminalservicesettingdata.md)<br/>                           | Representa la configuración del equipo virtual Terminal Services en un host.<br/>                                                                     |
