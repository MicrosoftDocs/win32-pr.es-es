# <a name="p"></a>P

## <a name="primary-colors"></a>colores primarios

Los tres colores esenciales en un espacio de colores que se pueden mezclar para producir todos los demás colores.

## <a name="profile-connection-space-pcs"></a>Espacio de conexión de perfil (PC)

Espacio de colores independiente del dispositivo que se usa para las conversiones de color y la coincidencia de color.
