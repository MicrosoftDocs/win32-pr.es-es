# <a name="access-flags"></a>Marcas de acceso

Estos valores indican si una entrada de la lista de acceso describe los derechos permitidos o denegados.



| Constante o valor                                                                                                                                                                                                                                      | Descripción                                   |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------|
| <span id="ACTRL_ACCESS_ALLOWED"></span><span id="actrl_access_allowed"></span><dl> <dt>**ACTRL \_ ACCESO \_ permitido**</dt> <dt>0x00000000</dt> </dl> | Indica una entrada de acceso permitido.<br/> |
| <span id="ACTRL_ACCESS_DENIED"></span><span id="actrl_access_denied"></span><dl> <dt>**ACTRL \_ ACCESO \_ denegado**</dt> <dt>0x10000000</dt> </dl>    | Indica una entrada de acceso denegado.<br/>  |
