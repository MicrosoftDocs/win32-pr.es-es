# <a name="media-locator"></a>Localizador de medios

> [!Note]  
> \[En desuso. Esta API se puede quitar de las versiones futuras de Windows.\]

 

El objeto localizador de medios ubica los archivos asociados a los objetos de origen. Utilice este objeto para validar los orígenes de la escala de tiempo. Para crear este objeto, llame a **CoCreateInstance**. El identificador de clase es CLSID \_ MediaLocator.

El objeto localizador de medios expone la interfaz siguiente:

-   [**IMediaLocator**](imedialocator.md)
