# <a name="disk-management-control-codes"></a>Códigos de control de administración de discos

|-----------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

| [**RENDIMIENTO DEL \_ DISCO IOCTL \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_performance)<br/>                                     | Habilita los contadores de rendimiento que proporcionan información de rendimiento del disco.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                        |

| [**BLOQUES DE REASIGNACIÓN DE DISCO DE \_ \_ IOCTL, POR \_ \_ EJEMPLO**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_reassign_blocks_ex)<br/>                     | Dirige el dispositivo de disco para asignar uno o varios bloques a su grupo de bloques de reserva.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                     |

| [**INFORMACIÓN DE CACHÉ \_ DEL CONJUNTO DE DISCOS \_ \_ DE \_ IOCTL**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_set_cache_information)<br/>               | Establece los datos de configuración del disco.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**INFORMACIÓN DEL \_ CLÚSTER DE IOCTL DISK \_ SET \_ \_**](ioctl-disk-set-cluster-info.md)<br/>                         | Establece la información del clúster en un disco.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |

| [**COMPROBACIÓN DE DISCO DE IOCTL \_ \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_disk_verify)<br/>                                               | Comprueba la extensión especificada en un disco fijo.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**LÍMITE DE \_ ENERGÍA DEL DISPOSITIVO DE ALMACENAMIENTO \_ \_ DE \_ IOCTL**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_device_power_cap)<br/>                   | Windows aplicaciones pueden usar este código de control para especificar un nivel máximo de consumo de energía operativo para un dispositivo de almacenamiento. Lo mejor es que el sistema operativo realice la transición del dispositivo a un estado de energía que no supere el máximo dado. Sin embargo, esto depende de lo que admita el dispositivo. El máximo real puede ser menor o mayor que el máximo deseado.<br/> Para realizar esta operación, llame a la [**función DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con los parámetros siguientes.<br/>                        |
| [**ACTIVACIÓN DEL \_ FIRMWARE DE \_ ALMACENAMIENTO DE IOCTL \_**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_firmware_activate)<br/>                  | Windows aplicaciones pueden usar este código de control para activar una imagen de firmware en un dispositivo especificado.<br/> Para realizar esta operación, llame a la [**función DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con los parámetros siguientes.<br/>                                                                                                                                                                                                                                                                                          |
| [**DESCARGA DE \_ FIRMWARE DE \_ ALMACENAMIENTO DE IOCTL \_**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_firmware_download)<br/>                  | Windows aplicaciones pueden usar este código de control para descargar una imagen de firmware en el dispositivo de destino, pero no activarla. Si la imagen que se va a descargar es mayor que el tamaño máximo de transferencia de datos del controlador, se tendrá que llamar a esta IOCTL varias veces hasta que se descargue toda la imagen.<br/> Para realizar esta operación, llame a la [**función DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con los parámetros siguientes.<br/>                                                                                     |
| [**IOCTL \_ STORAGE \_ FIRMWARE \_ GET \_ INFO**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_firmware_get_info)<br/>                 | Windows aplicaciones pueden usar este código de control para consultar el dispositivo de almacenamiento para obtener información detallada del firmware. Una llamada correcta devolverá información sobre las revisiones de firmware, el estado de la actividad, así como los atributos de lectura y escritura de cada ranura. La cantidad de datos devueltos variará en función del protocolo de almacenamiento.<br/> Para realizar esta operación, llame a la [**función DeviceIoControl**](/windows/desktop/api/ioapiset/nf-ioapiset-deviceiocontrol) con los parámetros siguientes.<br/>                                                                         |
| [**COMANDO DEL \_ PROTOCOLO DE ALMACENAMIENTO DE \_ \_ IOCTL**](/windows/desktop/api/winioctl/ni-winioctl-ioctl_storage_protocol_command)<br/>                    | Windows aplicaciones pueden usar este código de control para devolver las propiedades de un adaptador o dispositivo de almacenamiento. La solicitud indica el tipo de información que se debe recuperar, como los datos de consulta de un dispositivo o las funcionalidades y limitaciones de un adaptador. <br/>                                                                                                                                                                                                                                                                               |
| [**PROPIEDAD DE \_ CONSULTA DE \_ ALMACENAMIENTO DE IOCTL \_**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_storage_query_property)<br/>                        | Windows aplicaciones pueden usar este código de control para devolver las propiedades de un adaptador o dispositivo de almacenamiento. La solicitud indica el tipo de información que se debe recuperar, como los datos de consulta de un dispositivo o las funcionalidades y limitaciones de un adaptador. [**IOCTL \_ STORAGE \_ QUERY \_ PROPERTY también**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_storage_query_property) se puede usar para determinar si el controlador de puerto admite una propiedad determinada o qué campos del descriptor de propiedad se pueden modificar con una solicitud de propiedad de cambio posterior.<br/> |
| [**UMBRAL DE \_ TEMPERATURA DEL CONJUNTO DE ALMACENAMIENTO \_ \_ DE \_ IOCTL**](/windows/desktop/api/WinIoctl/ni-winioctl-ioctl_storage_set_temperature_threshold)<br/> | Windows aplicaciones pueden usar este código de control para establecer el umbral de temperatura de un dispositivo (cuando es compatible con el dispositivo). <br/>                                                                                                                                                                                                                                                                                                                                                                                              |
