# <a name="teredo"></a>Teredo

## <a name="purpose"></a>Propósito

Teredo es una tecnología de transición IPv6 que proporciona asignación de direcciones y tunelización automática de host a host para el tráfico IPv6 de unidifusión cuando los hosts IPv6/IPv4 se encuentran detrás de uno o varios traductores de direcciones de red (NAT) IPv4. Para atravesar los NAT IPv4, los paquetes IPv6 se envían como mensajes del Protocolo de datagramas de usuario (UDP) IPv4.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Teredo está diseñado para su uso por los desarrolladores de C/C++ con experiencia de programación de red IPv6.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

La interfaz teredo es compatible principalmente con Windows Vista y Windows Server 2008. La funcionalidad limitada de la interfaz teredo compatible con Windows XP con Service Pack 2 (SP2) y Windows Server 2003 se detalla en Recepción de tráfico solicitado a través [de Teredo.](receiving-solicited-traffic-over-teredo.md)
