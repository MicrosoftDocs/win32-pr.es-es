# <a name="viewonload"></a>VER. onload

El controlador de eventos **OnLoad** controla un evento que se produce cuando se muestra la **vista** por primera vez.

``` syntax
onload
```

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
