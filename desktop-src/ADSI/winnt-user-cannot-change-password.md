# <a name="user-cannot-change-password-winnt-provider"></a>El usuario no puede cambiar la contraseña (proveedor WinNT)

La capacidad de un usuario de cambiar su contraseña es un permiso que se puede conceder o denegar. Para obtener más información sobre cómo leer y modificar este permiso mediante programación con el proveedor de WinNT, vea:

-   [Lectura de usuario no puede cambiar la contraseña (proveedor WinNT)](reading-user-cannot-change-password-winnt-provider.md)
-   [La modificación del usuario no puede cambiar la contraseña (proveedor WinNT)](modifying-user-cannot-change-password-winnt-provider.md)

El atributo **userFlags** del objeto User solo está disponible cuando se usa el proveedor Winnt.

También es posible leer y modificar este permiso mediante el proveedor LDAP. Para obtener más información, consulte el [usuario no puede cambiar la contraseña (proveedor LDAP)](user-cannot-change-password.md).
