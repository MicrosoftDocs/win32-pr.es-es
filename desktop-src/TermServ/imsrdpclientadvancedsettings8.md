# <a name="imsrdpclientadvancedsettings8-interface"></a>Interfaz IMsRdpClientAdvancedSettings8

Expone métodos y propiedades que administran la configuración avanzada del control Escritorio remoto ActiveX control.

Para obtener una instancia de esta interfaz, use la [**propiedad IMsRdpClient8::AdvancedSettings9.**](imsrdpclient8-advancedsettings9.md)

## <a name="members"></a>Miembros

La **interfaz IMsRdpClientAdvancedSettings8** hereda de [**IMsRdpClientAdvancedSettings7**](imsrdpclientadvancedsettings7.md). **IMsRdpClientAdvancedSettings8** también tiene estos tipos de miembros:

-   [Propiedades](#properties)

### <a name="properties"></a>Propiedades

La **interfaz IMsRdpClientAdvancedSettings8** tiene estas propiedades.



| Propiedad                                                                                  | Tipo de acceso           | Descripción                                                           |
|:------------------------------------------------------------------------------------------|:----------------------|:----------------------------------------------------------------------|
| [**BandwidthDetection**](imsrdpclientadvancedsettings8-bandwidthdetection.md)<br/> | Lectura/escritura<br/> | Especifica si los cambios de ancho de banda se detectan automáticamente.<br/> |
