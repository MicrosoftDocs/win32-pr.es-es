# <a name="developing-accessible-applications-for-windows"></a>Desarrollo de aplicaciones accesibles para Windows

Cada marco de aplicación de Windows admite numerosas características de accesibilidad que facilitan a los usuarios con discapacidades, preferencias personales o estilos de trabajo específicos el uso correcto de cualquier dispositivo de Windows que elijan.

En esta información general se describen las características de accesibilidad que se admiten en cada marco y cómo puede incorporarlas a medida que compila la aplicación.

- [Aplicaciones para UWP](/windows/uwp/design/accessibility/accessibility)
- [Aplicaciones de WinForms](/dotnet/framework/winforms/advanced/windows-forms-accessibility)
