# <a name="dsp-plug-in-user-overview"></a>Introducción al usuario del complemento DSP

Para el usuario, los complementos de DSP son características de complemento que cambian de algún modo la forma en que el usuario experimenta el audio o el vídeo que Reproductor de Windows Media está reproduciendo. El resultado de un complemento DSP podría ser algo obvio, como los colores de la imagen de vídeo que se invierten, por lo que parece un negativo de fotografía, o algo muy sutil, como un efecto de reverberación ligera en el audio para agregar una sensación de sensibilidad al sonido.

Los usuarios pueden instalar y desinstalar complementos DSP mediante la pestaña **Complementos** del cuadro de diálogo Opciones de Reproductor de Windows Media. A continuación, los usuarios pueden habilitar y deshabilitar los complementos instalados mediante las **casillas** que se proporcionan en la pestaña  Complementos. Los complementos de DSP pueden implementar una página de propiedades, a la que los usuarios también pueden acceder desde la pestaña Complementos. Una página de propiedades proporciona al usuario una manera de cambiar la configuración que afecta al funcionamiento de un complemento DSP. Por ejemplo, una página de propiedades podría incluir un control deslizante que permite al usuario cambiar el tiempo de decadencia de un efecto de reverberación.
