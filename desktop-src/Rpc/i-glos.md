# <a name="i-rpc"></a>I (RPC)

[A](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) [E](e-glos.md) [F](f-glos.md) G H I J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [o](o-glos.md) [p](p-glos.md) [Q](q.md) [R](r-glos.md) [s](s-glos.md) [T](t-glos.md) [U](u-glos.md) [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_idempotent_glos"></span><span id="_RPC_IDEMPOTENT_GLOS"></span>**altera**
</dt> <dd>

Llamada a procedimiento remoto que no cambia un estado y devuelve la misma información cada vez que se llama con los mismos parámetros de entrada.

</dd> <dt>

<span id="_rpc_input_parameter_glos"></span><span id="_RPC_INPUT_PARAMETER_GLOS"></span>**parámetro de entrada**
</dt> <dd>

Parámetro que contiene los datos que la aplicación cliente transmite a la aplicación de servidor. El atributo in designa un parámetro **de** entrada.

</dd> <dt>

<span id="_rpc_idl_glos"></span><span id="_RPC_IDL_GLOS"></span>**Lenguaje de definición de interfaz (IDL)**
</dt> <dd>

OSF-DCE: lenguaje estándar para especificar la interfaz para llamadas a procedimientos remotos. Vea también [*MIDL*](m-glos.md).

</dd> <dt>

<span id="_rpc_ip_glos"></span><span id="_RPC_IP_GLOS"></span>**Protocolo de Internet (IP)**
</dt> <dd>

Protocolo de comunicaciones de capa de red sin conexión. Vea también [*datagrama*](d-glos.md).

</dd> <dt>

<span id="_rpc_ipx_glos"></span><span id="_RPC_IPX_GLOS"></span>**Intercambio de paquetes de redes interconectadas (IPX)**
</dt> <dd>

Protocolo de comunicación de Novell NetWare que usa sockets de [*datagramas*](d-glos.md) para enrutar paquetes de información a través de redes de área local y redes de área extensa.

</dd> <dt>

<span id="_rpc_intranet_glos"></span><span id="_RPC_INTRANET_GLOS"></span>**interna**
</dt> <dd>

Red privada que utiliza productos y tecnologías de Internet (por ejemplo, servidores Web), pero no está disponible para los usuarios externos de Internet.

</dd> </dl>
