# <a name="mfsampleextension_multiplexed_manager-attribute"></a>MFSampleExtension \_ atributo de \_ Administrador multiplexado

Proporciona una instancia de [**IMFMuxStreamSampleManager**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmuxstreamsamplemanager) que se utiliza para tener acceso a la colección de ejemplos de las subsecuencias de un origen de medios multiplexado.

## <a name="data-type"></a>Tipo de datos

**[**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown)**

## <a name="remarks"></a>Observaciones

Pase este valor a [**IMFAttributes:: GetUnknown**](/windows/desktop/api/mfobjects/nf-mfobjects-imfattributes-getunknown) para obtener una instancia de [**IMFMuxStreamSampleManager**](/windows/desktop/api/mfobjects/nn-mfobjects-imfmuxstreamsamplemanager).
