<dl> <dt>

<span id="tapi2.analog_display_services_interface_adsi__tapgloss"></span><span id="TAPI2.ANALOG_DISPLAY_SERVICES_INTERFACE_ADSI__TAPGLOSS"></span>**Interfaz de servicios de pantalla analógica (ADSI)**
</dt> <dd>

Un estándar Bellcore que define un protocolo sobre el flujo de información entre un conmutador, un servidor, un sistema de correo de voz o una oficina de servicio, por ejemplo, y el teléfono, el equipo, el terminal de datos u otro dispositivo de comunicación de un suscriptor con una pantalla. ADSI permite mostrar una versión de texto de la conversación en la pantalla durante la conexión telefónica. Por ejemplo, el usuario marca en una cuenta de correo de voz y recibe un menú de audio para las funciones de correo de voz mientras la pantalla muestra el menú en forma de texto.

</dd> <dt>

<span id="tapi2.assisted_telephony_tapgloss"></span><span id="TAPI2.ASSISTED_TELEPHONY_TAPGLOSS"></span>**Telefonía asistida**
</dt> <dd>

Conjunto de funciones que no están dedicadas únicamente a la funcionalidad de telefonía, sino que están diseñadas para que el establecimiento de llamadas de voz y llamadas multimedia esté disponible para cualquier aplicación. Para obtener más información, consulte [telefonía asistida](./assisted-telephony-overview.md) en información general de telefonía. Consulte telefonía [*básica*](b-tapgloss.md), [*telefonía extendida*](e-tapgloss.md), [*telefonía adicional*](s-tapgloss.md).

</dd> <dt>

<span id="tapi2.asynchronous_tapgloss"></span><span id="TAPI2.ASYNCHRONOUS_TAPGLOSS"></span>**asincrónica**
</dt> <dd>

Método de transmisión de datos que permite enviar caracteres en intervalos irregulares en una línea, precediendo cada carácter con un bit de inicio y después con un bit de parada. Vea [*sincrónicos*](s-tapgloss.md).

</dd> </dl>
