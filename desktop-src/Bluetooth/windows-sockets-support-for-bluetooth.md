# <a name="windows-sockets-support-for-bluetooth"></a>Compatibilidad de Windows Sockets con Bluetooth

En esta sección se describe cómo usar la interfaz Windows Sockets con Bluetooth funcionalidad introducida en Windows Vista. Para obtener más información sobre las API que lo hacen posible, vea Programación Bluetooth con [sockets](bluetooth-programming-with-windows-sockets.md) de Windows y el ejemplo [de Bluetooth conexión.](https://github.com/microsoftarchive/msdn-code-gallery-microsoft/tree/master/Official%20Windows%20Platform%20Sample/Bluetooth%20connection%20sample)
