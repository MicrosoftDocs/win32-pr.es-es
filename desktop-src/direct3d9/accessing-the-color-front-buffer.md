# <a name="accessing-the-color-front-buffer-direct3d-9"></a>Obtener acceso al búfer frontal de color (Direct3D 9)

Se permite el acceso al búfer frontal a través del método [**IDirect3DDevice9:: GetFrontBufferData**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-getfrontbufferdata) . Este método es la única manera de obtener una captura de pantalla de una escena suavizada.
