# <a name="about-the-mru-source-list"></a>Acerca de la lista de orígenes MRU

La lista de orígenes usados más recientemente (MRU) sigue siendo residente en el sistema del usuario. Las funciones de configuración controlan internamente la creación de nuevas listas de origen y su ubicación en el equipo del usuario. Las funciones de instalación de usan esta lista para almacenar información sobre las rutas de acceso de origen usadas en las instalaciones anteriores.

Puede usar esta información al solicitar al usuario una ruta de acceso de origen. Por ejemplo, puede crear una lista desplegable de las conexiones de red usadas como rutas de acceso de origen en las instalaciones anteriores.

En función de los permisos, puede crear una lista de usuarios, una que sea específica de un usuario determinado o una lista de sistemas, una que sea la misma para todos los usuarios. Además de las listas de origen del sistema y del usuario, puede crear una lista de origen temporal que se descarte cuando se cierre la aplicación de instalación.
