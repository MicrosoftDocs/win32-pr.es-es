# <a name="disableflyweightpatching"></a>DisableFlyWeightPatching

Si esta Directiva del sistema por equipo está establecida en 1 (uno), todas las opciones de [optimización de revisión](patch-optimization.md) se desactivan durante la instalación.

Las opciones de [optimización de revisiones](patch-optimization.md) y la Directiva DisableFlyWeightPatching están disponibles a partir de Windows Installer versión 3,0.

## <a name="registry-key"></a>Clave del Registro

**HKEY \_ \_** Directivas de software de equipo local \\  \\  \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Tipo de datos

**\_valor DWORD reg**
