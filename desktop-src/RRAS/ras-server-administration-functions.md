# <a name="ras-server-administration-functions"></a>Funciones de administración del servidor RAS

En Windows NT 4,0, utilice las siguientes funciones para implementar la funcionalidad de administración del servidor RAS. Windows 95 no proporciona compatibilidad con el servidor RAS. Para Windows 2000, Windows Server 2003 y los sistemas operativos posteriores, use la [referencia de administración del servicio de acceso remoto](remote-access-service-administration-reference.md).

-   [**RasAdminFreeBuffer**](rasadminfreebuffer.md)
-   [**RasAdminGetErrorString**](rasadmingeterrorstring.md)
-   [**RasAdminGetUserAccountServer**](rasadmingetuseraccountserver.md)
-   [**RasAdminPortClearStatistics**](rasadminportclearstatistics.md)
-   [**RasAdminPortDisconnect**](rasadminportdisconnect.md)
-   [**RasAdminPortEnum**](rasadminportenum.md)
-   [**RasAdminPortGetInfo**](rasadminportgetinfo.md)
-   [**RasAdminServerGetInfo**](rasadminservergetinfo.md)
-   [**RasAdminUserGetInfo**](rasadminusergetinfo.md)
-   [**RasAdminUserSetInfo**](rasadminusersetinfo.md)
-   [Funciones DLL de administración de RAS](ras-administration-dll-functions.md)
-   [Funciones DLL de seguridad RAS](ras-security-dll-functions.md)
