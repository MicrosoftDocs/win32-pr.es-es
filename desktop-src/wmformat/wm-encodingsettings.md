# <a name="wmencodingsettings"></a>WM/EncodingSettings

El atributo **WM/EncodingSettings** contiene la configuración que se usa para codificar el contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMEncodingSettings

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

No hay ningún formato estándar para indicar la configuración de codificación. Sin embargo, es preferible describir la configuración de la codificación en inglés en lugar de incluir un volcado de la configuración.
