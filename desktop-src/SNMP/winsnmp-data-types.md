# <a name="winsnmp-data-types"></a>Tipos de datos WinSNMP

\[SNMP está disponible para su uso en los sistemas operativos especificados en la sección Requisitos. En versiones posteriores podría modificarse o no estar disponible. En su lugar, [Windows administración remota](/windows/desktop/WinRM/portal), que es la implementación de Microsoft de WS-Man.\]

Los tipos de datos estándar de WinSNMP se definen en WINSNMP. Archivo H.

Tenga en cuenta que WinSNMP especifica algunos parámetros con el tipo entero largo con signo, **smiINT**. Esto permite que WinSNMP cumpla los elementos de datos, especialmente los componentes de PDU, definidos en las RFC pertinentes.
