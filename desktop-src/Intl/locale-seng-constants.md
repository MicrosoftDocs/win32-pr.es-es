# <a name="locale_seng-constants"></a>Constantes de Seng de configuración regional \_ \*

En este tema se definen las constantes de Seng de configuración regional \_ \* utilizadas por NLS.



| Value                | Significado                                                                                                                                                                   |
|----------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| configuración regional \_ SENGCOUNTRY  | En desuso para Windows 7 y versiones posteriores. Nombre en inglés completo del país o región. Consulte [configuración regional \_ SENGLISHCOUNTRYNAME](locale-senglish-constants.md).                            |
| configuración regional \_ SENGCURRNAME | **Windows Me/98, windows 2000:** El nombre en inglés completo de la moneda asociada con la configuración regional. No hay ningún límite en el número de caracteres permitidos para esta cadena. |
| configuración regional \_ SENGLANGUAGE | En desuso para Windows 7 y versiones posteriores. Nombre en inglés completo del idioma del estándar ISO 639. Consulte [configuración regional \_ SENGLISHLANGUAGENAME](locale-senglish-constants.md).           |
