# <a name="bits-c-application-examples"></a>Ejemplos de aplicaciones de C++ de BITS

Los ejemplos de aplicación de Servicio de transferencia inteligente en segundo plano (BITS) de esta sección se escriben en C++. Muestran una serie de tareas que se pueden completar mediante las características de BITS. Cada aplicación se divide en una serie de pasos.

En la tabla siguiente se enumeran los ejemplos de C++ de esta sección.



| Ejemplo                                                                                                                                                            | Descripción                                                                                                                                                                                                                                                                                |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Ejemplo: clases comunes](common-classes.md)                                                                                                                      | Este ejemplo contiene clases comunes que se usan en los ejemplos de esta sección. Estas clases comunes incluyen una clase de devolución de llamada, una clase de error genérico y una clase auxiliar de adquisición de recursos para la función [CoInitializeEx](/windows/win32/api/combaseapi/nf-combaseapi-coinitializeex) de com. |
| [Ejemplo: especificar las credenciales de autenticación del servidor para un trabajo de transferencia de BITS](example-specifying-server-authentication-credentials-for-a-bits-transfer-job-.md) | En este ejemplo se obtienen el esquema de autorización y las credenciales del usuario, se crea un trabajo de transferencia de BITS, se establecen las credenciales para el trabajo y se supervisa el estado del trabajo.                                                                                                               |
| [Ejemplo: usar la codificación de autenticación SSPI con BITS.](example-using-sspi-authentication-encoding-with-bits.md)                                                 | En este ejemplo se muestra el uso de la autenticación de la interfaz del proveedor de compatibilidad con seguridad (SSPI) y BITS para obtener las credenciales de un usuario, codificar las credenciales y establecer las credenciales codificadas en un trabajo de transferencia de BITS.                                                                |
| [Ejemplo: agregar un token auxiliar a un trabajo de transferencia de BITS](example-adding-a-helper-token-to-a-bits-transfer-job-.md)                                                 | En este ejemplo se crea un trabajo de transferencia de BITS y se agrega un segundo conjunto de credenciales al trabajo.                                                                                                                                                                                                  |
