# <a name="working-with-sources"></a>Trabajar con orígenes

\[Esta API no se admite y puede modificarse o no estar disponible en el futuro.\]

Un objeto de origen representa una única secuencia de medios de un archivo de código fuente. Los orígenes proporcionan el material sin procesar de un proyecto. Todo lo demás de la escala de tiempo existe para proporcionar una estructura (pistas, composiciones y grupos) o para modificar los orígenes (efectos y transiciones).

En este artículo se describen diversos detalles relativos a los orígenes de los servicios de edición de® (DES) de Microsoft® DirectShow. Para obtener información sobre los pasos básicos necesarios para agregar un origen a la escala de tiempo, vea [crear una escala de tiempo](constructing-a-timeline.md).

Este artículo contiene las secciones siguientes.

-   [Establecer propiedades en orígenes](setting-properties-on-sources.md)
-   [Uso del detector de medios](using-the-media-detector.md)
-   [Usar el localizador de medios](using-the-media-locator.md)
-   [Otros objetos de origen](other-source-objects.md)
