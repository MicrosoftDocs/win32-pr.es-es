|-------------------|--------------------------------------|
| CN                | msSFU-30-NSMAP-campo-posición        |
| Nombre para mostrar de LDAP | msSFU30NSMAPFieldPosition            |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |

| System-ID-GUID    | 585c9d5e-f599-4f07-9cf9-4373af4b89d3 |
