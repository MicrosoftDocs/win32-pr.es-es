# <a name="shadowflag-attribute"></a>atributo shadowFlag

Contiene la sección de la asignación de instantáneas que se utiliza para almacenar el valor de la marca.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | shadowFlag                           |
| Nombre para mostrar de LDAP | shadowFlag                           |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.11                     |
| System-ID-GUID    | 8dfeb70d-c5db-46b6-b15e-a4389e6cee9b |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
