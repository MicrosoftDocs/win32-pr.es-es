# <a name="third-party-msp"></a>MSP de terceros

Normalmente, el hardware de comunicaciones especializadas llega con uno o varios profesionales y MSP específicos del dispositivo. La documentación del dispositivo debe indicar funcionalidades y proporcionar directrices de programación.
