# <a name="event-tracing-tools"></a>Herramientas de seguimiento de eventos

Las siguientes herramientas de seguimiento de eventos están disponibles para su uso.



| Herramienta     | Descripción                                                                                                                                                                                                                                                                                                                                |
|----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Logman   | Administra y programa colecciones de registros de seguimiento de eventos y contadores de rendimiento en sistemas locales y remotos. Puede usar esta herramienta para enumerar los proveedores que han publicado el diseño de sus datos [de evento](publishing-your-event-schema.md) en el espacio de \\ nombres wmi raíz. Para obtener más información sobre el uso de esta herramienta, **vea el Centro de ayuda y soporte técnico**. |
| Tracefmt | Da formato a la información de un archivo de registro de seguimiento en formato legible. Para obtener más información, consulte Herramientas para el seguimiento de software en Microsoft Windows Driver Development Kit (DDK). Solo lo usa el seguimiento de software de WPP.                                                                                                                                      |
| Registro de seguimiento | Controla una sesión de seguimiento de eventos. Para más información, consulte Herramientas para el seguimiento de software en el DDK.                                                                                                                                                                                                                                                   |
| Tracepdb | Crea un archivo de formato de mensaje de seguimiento que la herramienta Tracefmt usa para convertir los mensajes registrados en un formulario legible. Para más información, consulte Herramientas para el seguimiento de software en el DDK. Solo lo usa el seguimiento de software de WPP.                                                                                                                            |
| Tracerpt | Procesa registros de seguimiento de eventos o eventos en tiempo real de proveedores de seguimiento de eventos instrumentados y permite generar informes de análisis de seguimiento y archivos CSV (delimitados por comas) para los eventos generados. Para obtener más información, **vea el Centro de ayuda y soporte técnico**.                                                                                       |
