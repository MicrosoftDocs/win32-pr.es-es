# <a name="windows-automation-api-overview"></a>Windows Introducción a la API de Automation

En esta sección se proporciona información general de microsoft Windows Automation API 3.0, que incluye el Microsoft Active Accessibility heredado y la implementación de Windows de la especificación de microsoft Automatización de la interfaz de usuario.
