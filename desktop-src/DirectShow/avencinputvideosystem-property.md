# <a name="avencinputvideosystem-property"></a>Propiedad AVEncInputVideoSystem

Especifica el sistema de vídeo del contenido de origen.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncInputVideoSystem**

## <a name="property-value"></a>Valor de propiedad

El valor de esta propiedad es miembro de la [**enumeración eAVEncInputVideoSystem.**](/windows/desktop/api/codecapi/ne-codecapi-eavencinputvideosystem)
