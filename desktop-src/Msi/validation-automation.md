# <a name="validation-automation"></a>Automatización de la validación

Puede utilizar la Evalcom2.dll para implementar operaciones de validación para los paquetes de instalación y los módulos de combinación.

Las siguientes secciones de esta documentación contienen información sobre el uso de la validación de Windows Installer desde las herramientas de creación:

-   [Usar Evalcom2](using-evalcom2.md)
-   [Interfaces EvalCom2](evalcom2-interfaces.md)

Se recomienda instalar Evalcom2.dll mediante el Windows Installer. El paquete de Orca.msi proporcionado con el kit de desarrollo de software (SDK) de Microsoft Windows para Windows XP con Service Pack 2 (SP2) o superior instala Evalcom2.dll. El identificador de componente del componente que contiene la interfaz COM es {53315473-B8D6-470A-9951-D9AE5C11FC91}.

Para obtener información sobre cómo obtener el Windows SDK, vea [Windows SDK Components for Windows Installer Developers](platform-sdk-components-for-windows-installer-developers.md).
