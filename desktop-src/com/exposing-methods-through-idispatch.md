# <a name="exposing-methods-through-idispatch"></a>Exponer métodos a través de IDispatch

Aunque la mayoría de los controles exponen y admiten varios métodos, no es necesario que los controles expongan o admitan métodos, por lo que el control no requiere **IDispatch**. Si el control tiene algún método, no hay ningún requisito para los métodos que debe exponer un control.
