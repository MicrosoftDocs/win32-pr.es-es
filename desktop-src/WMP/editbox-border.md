# <a name="editboxborder"></a>EDITBOX. Border

El atributo **Border** especifica o recupera un valor que indica si el control de cuadro de edición tiene un borde. Solo se puede establecer en tiempo de diseño.

``` syntax
        elementID.border
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **valor booleano** especificado en tiempo de diseño y de solo lectura después. Tiene el valor predeterminado True.

## <a name="remarks"></a>Observaciones

El borde aparece como un borde hundido de dos píxeles alrededor del control cuadro de edición con los colores del sistema.
