# <a name="charsperline-property"></a>Propiedad CharsPerLine

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Devuelve los caracteres por línea admitidos por el globo de palabras para el carácter especificado.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintáctica**
</dt> <dd>

*agente*[**. Caracteres ("**](/windows/desktop/lwef/the-characters-object)* CharacterID ***").** Balloon. CharsPerLine

</dd> </dl>

## <a name="remarks"></a>Observaciones

Esta propiedad devuelve el número promedio de caracteres (letras) que se muestran en el globo de palabras como un valor entero largo. Puede establecer el valor mediante la propiedad [**Style**](style-property.md) .
