# <a name="enumerations-windows-animation-reference"></a>Enumeraciones (referencia de animación de Windows)

Documentación de referencia para las enumeraciones definidas por el administrador de animación de Windows.

| Tema                                                                                       | Descripción                                                                                                                                          |
|---------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_dependencias de animación de IU \_**](/windows/win32/api/uianimation/ne-uianimation-ui_animation_dependencies)<br/>                 | Define los aspectos de un interpolador que dependen de una entrada determinada.<br/>                                                                         |
| [**\_ \_ comportamiento inactivo de animación de interfaz de usuario \_**](/windows/win32/api/uianimation/ne-uianimation-ui_animation_idle_behavior)<br/>              | Define el comportamiento de un temporizador cuando el administrador de animaciones está inactivo.<br/>                                                                       |
| [**\_Estado del administrador de animaciones de IU \_ \_**](/windows/win32/api/uianimation/ne-uianimation-ui_animation_manager_status)<br/>            | Define el estado de actividad de un administrador de animaciones.<br/>                                                                                      |
| [**\_modo de animación de IU \_**](/windows/win32/api/uianimation/ne-uianimation-ui_animation_mode)<br/>                                 | Define los modos de animación.<br/>                                                                                                                  |
| [**\_efecto de \_ prioridad de animación de IU \_**](/windows/win32/api/uianimation/ne-uianimation-ui_animation_priority_effect)<br/>          | Define los posibles efectos en un guión gráfico si una comparación de prioridad devuelve false.<br/>                                                         |
| [**\_modo de \_ repetición de animación de IU \_**](/windows/win32/api/uianimation/ne-uianimation-ui_animation_repeat_mode)<br/>                  | Define el patrón de una iteración del bucle.<br/>                                                                                                 |

| [**\_pendiente de animación de IU \_**](/windows/win32/api/uianimation/ne-uianimation-ui_animation_slope)<br/>                               | Define las características de la pendiente de la animación.<br/>                                                                                                  |
| [**\_Estado de \_ guion gráfico de animación de IU \_**](/windows/win32/api/uianimation/ne-uianimation-ui_animation_storyboard_status)<br/>      | Define el estado de un guión gráfico.<br/>                                                                                                      |
| [**\_Estado de \_ cliente del temporizador de animación \_ de IU \_**](/windows/win32/api/uianimation/ne-uianimation-ui_animation_timer_client_status)<br/> | Define el estado de actividad para el cliente de un temporizador.<br/>                                                                                             |
