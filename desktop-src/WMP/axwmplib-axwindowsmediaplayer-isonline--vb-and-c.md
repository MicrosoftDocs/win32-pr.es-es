# <a name="axwindowsmediaplayerisonline-property"></a>Propiedad AxWindowsMediaPlayer.isOnline

La propiedad isOnline obtiene un valor que indica si el usuario está conectado a una red.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```CSharp
public System.Boolean isOnline {get;}
```


```VB

Public ReadOnly Property isOnline As System.Boolean
```





## <a name="property-value"></a>Valor de propiedad

Valor System.Boolean que indica si el usuario está conectado a una red.
