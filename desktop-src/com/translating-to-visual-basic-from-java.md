# <a name="translating-to-visual-basic-from-java"></a>Trasladar a Visual Basic desde Java

Visual Basic usa tipos de datos compatibles con la automatización como sus tipos de datos internos. Por lo tanto, los tipos de datos de Java que son compatibles con la automatización también son compatibles con Visual Basic. Es posible que los tipos de datos que no son compatibles con la automatización no se puedan convertir en Visual Basic.

Java y Visual Basic difieren ligeramente en cómo representan las propiedades. En Java, las propiedades se representan como un conjunto de funciones de descriptor de acceso, una que establece el valor de propiedad y otra que recupera el valor de propiedad. En Visual Basic, las propiedades se representan como un único elemento que se puede utilizar para recuperar o establecer el valor de la propiedad.
