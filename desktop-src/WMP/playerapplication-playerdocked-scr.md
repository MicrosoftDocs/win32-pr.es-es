# <a name="playerapplicationplayerdocked"></a>PLAYERAPPLICATION.playerDocked

El **atributo playerDocked** recupera un valor que indica si Reproductor de Windows Media está en estado acoplado.

``` syntax
        elementID.playerDocked
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un valor booleano de **solo lectura.**



| Valor | Descripción                       |
|-------|-----------------------------------|
| True  | Reproductor de Windows Media está acoplado.   |
| Falso | Reproductor de Windows Media está desacoplado. |



 

## <a name="remarks"></a>Comentarios

Este atributo solo se usa cuando se usa la comunicación remota Reproductor de Windows Media control .

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media serie 9 o posterior<br/> |
