# <a name="ui_pkey_decimalplaces"></a>PKEY de IU \_ \_

Identifica la propiedad PKEY de la interfaz de usuario \_ \_ .

```
propertyDescription
   name = UI_PKEY_DecimalPlaces
   shellPKey = UI_PKEY_DecimalPlaces
   formatID = 00000206-7363-696e-8441798acf5aebb7
   propID = 206
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Observaciones

\_La interfaz de usuario PKEY \_ DecimalPlaces se usa en una aplicación para consultar el número de dígitos que se muestran después del separador decimal en el campo de edición del control de [número](windowsribbon-controls-spinner.md) .
