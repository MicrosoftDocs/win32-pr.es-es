# <a name="imsrdpclientsecuredsettings2-interface"></a>Interfaz IMsRdpClientSecuredSettings2

Define propiedades adicionales del control ActiveX Escritorio remoto que están restringidas a zonas de seguridad de direcciones URL específicas de Internet Explorer.

## <a name="members"></a>Miembros

La interfaz **IMsRdpClientSecuredSettings2** hereda de [**IMsRdpClientSecuredSettings**](imsrdpclientsecuredsettings-interface.md). **IMsRdpClientSecuredSettings2** también tiene estos tipos de miembros:

-   [Propiedades](#properties)

### <a name="properties"></a>Propiedades

La interfaz **IMsRdpClientSecuredSettings2** tiene estas propiedades.



| Propiedad                                                   | Tipo de acceso           | Descripción                                                                                                          |
|:-----------------------------------------------------------|:----------------------|:---------------------------------------------------------------------------------------------------------------------|
| [**IMPRESO**](imsrdpclientsecuredsettings2-pcb.md)<br/> | Lectura/escritura<br/> | Especifica la configuración del BLOB de preconexión (PCB) que se va a usar antes de la conexión para la transmisión al servidor.<br/> |
