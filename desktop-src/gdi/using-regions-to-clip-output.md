# <a name="using-regions-to-clip-output"></a>Uso de regiones para recortar la salida

Esta sección contiene un solo ejemplo que muestra cómo puede usar las regiones para permitir al usuario definir cómo puede aparecer una parte del resultado del área cliente. Las regiones que se usan para este propósito se denominan regiones de recorte.

El ejemplo de esta sección se toma de una aplicación que permite a un usuario capturar todo el escritorio como un mapa de bits y, a continuación, aislar y guardar una parte de esta imagen como. Archivo BMP.

Al hacer clic en **definir región** de recorte en el menú de la aplicación, el usuario puede seleccionar una región de recorte haciendo clic en el botón primario del mouse y arrastrando el mouse. Cuando el usuario arrastra el mouse, la aplicación dibuja un rectángulo que corresponde a la nueva región de recorte.

Al hacer clic en **recortar** en el menú de la aplicación, el usuario puede volver a dibujar la parte aislada de la imagen dentro de los límites del rectángulo especificado.

En esta sección se proporciona información sobre los temas siguientes.

-   [Definir la región de recorte](defining-the-clipping-region.md)
-   [Salida de recorte](clipping-output.md)
