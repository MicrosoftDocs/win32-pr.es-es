# <a name="cbasevideorenderer-class"></a>Clase CBaseVideoRenderer

![jerarquía de clases cbasevideorenderer](images/rbase03.png)

La `CBaseVideoRenderer` clase base se utiliza para crear filtros de representador de vídeo.



| Miembros de datos protegidos                                                                 | Descripción                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| m \_ bDrawLateFrames                                                                     | Marca para indicar que no se van a quitar fotogramas. Solo depuración. Esto destruye la sincronización.                                                                                                                                          |
| m \_ bSupplierHandlingQuality                                                            | **True** indica que los mensajes de control de calidad se están controlando. Esto permite que el representador tenga que esperar hasta lo más tarde posible para quitar fotogramas y mostrar el fotograma siguiente muy pronto después de que el proveedor haya colocado un fotograma. |
| m \_ cFramesDrawn                                                                        | Número total de fotogramas que se han dibujado desde que se inició el streaming.                                                                                                                                                                 |
| m \_ cFramesDropped                                                                      | Fotogramas acumulados que se han quitado del representador desde que se inició el streaming. Los marcos también se pueden quitar de nivel superior sin que el representador los reconozca.                                                                         |
| m \_ idDecision                                                                          | Identificador MSR del \_ código de decisión de [**ShouldDrawSampleNow**](cbasevideorenderer-shoulddrawsamplenow.md).                                                                                                                              |
| m \_ idDuration                                                                          | \_Identificador MSR para la duración de un marco.                                                                                                                                                                                                 |
| m \_ idFrameAccuracy                                                                     | Identificador del registro de rendimiento para el tiempo en milisegundos que el marco estuvo atrasado.                                                                                                                                                     |
| m \_ idFrameAvg                                                                          | Identificador del registro de rendimiento para el tiempo medio de fotograma que se usa para la sincronización y el control de calidad.                                                                                                                          |
| m \_ idQualityRate                                                                       | \_Identificador MSR para la tasa de calidad solicitada.                                                                                                                                                                                              |
| m \_ idQualityTime                                                                       | \_Identificador MSR para el tiempo de calidad solicitado.                                                                                                                                                                                              |
| m \_ idRenderAvg                                                                         | Identificador del registro de rendimiento para el tiempo promedio de representación registrado.                                                                                                                                                                   |
| m \_ idSchLateTime                                                                       | \_Identificador MSR para el momento en que se programó el marco.                                                                                                                                                                                   |
| m \_ idSendQuality                                                                       | \_Identificador MSR para cronometrar las notificaciones (sin usar).                                                                                                                                                                                       |
| m \_ idTimeStamp                                                                         | \_Identificador MSR para una marca de tiempo de marco.                                                                                                                                                                                                      |
| m \_ idWait                                                                              | Identificador del registro de rendimiento para el tiempo de espera registrado (sin usar).                                                                                                                                                                      |
| m \_ idWaitReal                                                                          | Identificador del registro de rendimiento para el tiempo de espera real.                                                                                                                                                                                   |
| m \_ iSumFrameTime                                                                       | Suma de las horas de INTERMARCO; necesario para la página de propiedades.                                                                                                                                                                           |
| m \_ iSumSqAcc                                                                           | Suma de los cuadrados de las precisiones (en milisegundos) necesarias para la página de propiedades.                                                                                                                                                 |
| m \_ iSumSqFrameTime                                                                     | Suma de los cuadrados de las horas entre fotogramas; necesario para la página de propiedades.                                                                                                                                                                |
| m \_ iTotAcc                                                                             | Suma de las precisiones (en milisegundos) necesarias para la página de propiedades.                                                                                                                                                                |
| m \_ nNormal                                                                             | Número de fotogramas consecutivos dibujados en el momento programado. Un número negativo indica que el representador acaba de quitar un fotograma.                                                                                          |
| m \_ trDuration                                                                          | Duración del último fotograma (diferencia entre las horas de inicio y finalización).                                                                                                                                                             |
| m \_ trEarliness                                                                         | La primera vez que se permite que se reproduzca un fotograma cuando se acaba de quitar un fotograma.                                                                                                                                                        |
| m \_ trFrame                                                                             | Hora de registro más reciente entre fotogramas. Se utiliza en medidas estadísticas.                                                                                                                                                        |
| m \_ trFrameAvg                                                                          | Tiempo medio entre fotogramas en unidades de tiempo de referencia.                                                                                                                                                                                     |
| m \_ trLastDraw                                                                          | Hora del fotograma anterior. Se usa para las referencias de tiempo entre fotogramas.                                                                                                                                                                         |
| m \_ trLate                                                                              | Cantidad de tiempo que el fotograma actual estuvo atrasado. Se utiliza en medidas estadísticas.                                                                                                                                                    |
| m \_ trRenderAvg                                                                         | Tiempo que tardan los marcos en realizar la transferencia de bloque de bits.                                                                                                                                                                       |
| m \_ trRenderLast                                                                        | Hora de la última transferencia de bloque de bits de fotogramas.                                                                                                                                                                                          |
| m \_ trRenderStart                                                                       | Hora en que se inició la transferencia de bloque de bits. Se usa para obtener **m \_ trRenderLast**.                                                                                                                                                                |
| m \_ trThrottle                                                                          | Período que se va a insertar después de representar cada fotograma, normalmente se usa cuando se ha aumentado la calidad del audio y se debe reducir el rendimiento del vídeo para permitirlo.                                                                             |
| m \_ trWaitAvg                                                                           | Promedio de tiempo de espera en unidades de tiempo de referencia.                                                                                                                                                                                           |
| m \_ tStreamingStart                                                                     | Se utiliza para las estadísticas de la página de propiedades. Representa la hora de inicio del proceso de streaming actual o el proceso de streaming anterior, si no se está transmitiendo en este momento.                                                                          |
