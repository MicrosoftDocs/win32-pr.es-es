# <a name="iwebproxy-methods"></a>Métodos de IWebProxy

La interfaz [**IWebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) define los métodos siguientes.



| Método                                                                         | Descripción                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**PromptForCredentials**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentials)                 | Solicita al usuario la contraseña que se usará para la autenticación del proxy.  |
| [**PromptForCredentialsFromHwnd**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentialsfromhwnd) | Solicita al usuario la contraseña que se usará para la autenticación del proxy.  |
| [**SetPassword**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-setpassword)                                   | Establece la contraseña que se va a enviar al servidor proxy para la autenticación. |
