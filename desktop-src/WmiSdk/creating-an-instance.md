# <a name="creating-an-instance"></a>Crear una instancia

Una instancia es un caso de una clase que describe un objeto determinado de la empresa. Por ejemplo, si tiene dos unidades de CD-ROM en una plataforma de escritorio, puede utilizar dos instancias de la [**clase \_ CDROMDrive de Win32**](/windows/desktop/CIMWin32Prov/win32-cdromdrive) para describir ambas. Dado que una instancia representa una propiedad real de la empresa, lo más probable es que cree más instancias que las clases. Cada clase de la que se crea una instancia tiene una o más propiedades de clave. No se puede crear una instancia de con más de 256 claves.

Para obtener más información acerca de la creación de instancias, vea los temas siguientes:

-   [Crear y declarar una instancia con C++](creating-and-declaring-an-instance-using-c-.md)
-   [Crear una instancia mediante MOF](creating-an-instance-using-mof.md)
-   [Crear un objeto mediante VBScript](creating-an-object-using-vbscript.md)
