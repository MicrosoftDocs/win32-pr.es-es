# <a name="notifying-the-user"></a>Notificar al usuario

Para notificar al usuario que se ha producido algún tipo de error, muchas aplicaciones simplemente producen un sonido mediante la función [**MessageBeep**](/windows/desktop/api/WinUser/nf-winuser-messagebeep) o Flash en la ventana mediante la función [**FlashWindow**](/windows/desktop/api/Winuser/nf-winuser-flashwindow) o [**FlashWindowEx**](/windows/desktop/api/Winuser/nf-winuser-flashwindowex) . Una aplicación también puede utilizar estas funciones para llamar la atención sobre un error y, a continuación, mostrar un cuadro de mensaje o un mensaje de error que contenga los detalles del error.
