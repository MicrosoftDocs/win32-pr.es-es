# <a name="tspi_linedroponclose"></a>Línea \_ TSPIDropOnClose

La **función \_ lineDropOnClose de TSPI** está obsoleta. TAPI ya no quita automáticamente una llamada cuando el único propietario cierra su identificador al dispositivo de línea, con el fin de dar cabida al control distribuido de las llamadas entre varios equipos cliente en un entorno cliente/servidor.
