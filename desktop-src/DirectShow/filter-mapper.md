# <a name="filter-mapper"></a>Asignador de filtros

El asignador de filtros busca en el registro los filtros registrados. El administrador de gráficos de filtro utiliza este componente para generar gráficos de filtros. Las aplicaciones pueden utilizarlo para buscar filtros que coincidan con los criterios de búsqueda especificados. Cree este objeto llamando a **CoCreateInstance**.
