# <a name="developing-assistive-technology-for-windows"></a>Desarrollo de tecnología de asistencia para Windows

Los productos de tecnología de asistencia de terceros, como los lectores de pantalla, los ampliadores y el hardware de especial, son herramientas esenciales para muchos usuarios de Windows (y otros productos de Microsoft).

En esta información general se describe cómo crear herramientas de tecnología de asistencia que son compatibles con los productos de Microsoft para personas con visión, aprendizaje, Dexterity/Mobility y limitaciones de lenguaje/comunicación o limitaciones.
