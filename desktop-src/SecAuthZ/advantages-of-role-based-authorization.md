# <a name="advantages-of-role-based-authorization"></a>Ventajas de la autorización basada en roles

La API de administrador de autorización proporciona un modelo de autorización que tiene varias ventajas sobre la autorización de bajo nivel (basada en ACL):

-   Simplifica la administración del control de acceso.
-   Hace que la autorización esté disponible para los scripts y las aplicaciones desarrolladas con el sistema de desarrollo de Microsoft Visual Basic.
-   Proporciona un mecanismo para aplicar la lógica de negocios en tiempo de ejecución cuando se comprueba el acceso.
