# <a name="ipx-service-table-management"></a>Administración de tablas del servicio IPX

Un protocolo de enrutamiento IPX que se registra para el control de servicios mantiene una tabla de enlace de servicio según se define en las especificaciones del Protocolo de anuncios de servicio (SAP) de Novell. El protocolo de enrutamiento también proporciona acceso de administrador de enrutador a esta tabla a través de las siguientes funciones y estructuras.

-   [Funciones de administración de tablas de servicio](service-table-management-functions.md)
-   [Estructuras de administración de tablas de servicio](service-table-management-structures.md)

Vea la función [**RegisterProtocol**](/windows/desktop/api/Routprot/nc-routprot-pregister_protocol) para obtener más información sobre cómo se registra un protocolo de enrutamiento para el control de servicios.
