# <a name="storing-dynamic-data"></a>Almacenar datos dinámicos

La mayoría de los datos almacenados mediante Active Directory Domain Services es estático. Por ejemplo, los objetos de directorio que representan usuarios, grupos, impresoras, servicios de red, etc., suelen ser de larga duración y cambian con poca frecuencia.

En Windows Server 2003 y versiones posteriores de Windows, Active Directory Domain Services proporcionan compatibilidad con las siguientes características que permiten a las aplicaciones usar el directorio para almacenar datos dinámicos:

-   [Objetos dinámicos](dynamic-objects.md)
-   [Clases auxiliares dinámicas](dynamic-auxiliary-classes.md)
