# <a name="scheduling"></a>Scheduling

El programador del sistema controla la multitarea mediante la determinación de cuál de los subprocesos de la competencia recibe el siguiente intervalo de tiempo del procesador. El programador determina el subproceso que se ejecuta junto con las prioridades de programación.
