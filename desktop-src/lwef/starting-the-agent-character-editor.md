# <a name="starting-the-agent-character-editor"></a>Inicio del Editor de caracteres del agente

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Para ejecutar el Editor de caracteres del agente, elija la opción  **Editor** de caracteres del agente en el menú Inicio de la barra de tareas de Windows o haga doble clic en el icono Editor de caracteres de Microsoft **Agent** en el escritorio. Se abrirá la ventana del Editor, que muestra sus menús, una barra de herramientas con comandos usados con frecuencia, un árbol que enumera los componentes que forma la definición de un carácter y un conjunto de páginas con pestañas que cambian en función de la selección en el árbol de componentes.

![Captura de pantalla que muestra la página "Propiedades" del "Carácter - Editor de caracteres de Microsoft Agent".](images/f1chared.gif)

Para acceder a los campos de la ventana desde el teclado, puede usar la tecla TABy MAYÚS+TAB para navegar entre los controles, o bien usar la tecla de acceso (alt+ letra subrayada) para desplazarse a un control específico. Una vez que el editor finaliza el inicio, puede empezar a crear una nueva definición de caracteres o cargar una definición de caracteres existente.

La barra de estado muestra información sobre comandos o botones de la barra de herramientas al mover el puntero sobre ellos. También muestra información de resumen sobre los datos de animación de caracteres y la información de estado al compilar un carácter.
