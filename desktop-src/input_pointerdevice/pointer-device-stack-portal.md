# <a name="pointer-device-input-stack"></a>Pila de entrada de dispositivo de puntero

## <a name="purpose"></a>Propósito

En los temas de esta sección se proporciona información general sobre la pila de entrada del dispositivo de puntero de Windows. Las API de la [pila de entrada del dispositivo de puntero](/windows/win32/api/_input_pointerdevice/) proporcionan información acerca de los dispositivos de entrada conectados al sistema.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|---|---|
| [Referencia de la pila de entrada de dispositivo de puntero](unified-input-stack-reference.md)<br/> | En los temas de esta sección se proporcionan las especificaciones de referencia para la pila de entrada de dispositivo de puntero en Windows 8.<br/> |

## <a name="developer-audience"></a>Audiencia de desarrolladores

Los StackAPIs de entrada de dispositivo de puntero están diseñados para desarrolladores que requieren información sobre el dispositivo de puntero que se usa para interactuar con la aplicación.
