# <a name="about-the-winsnmp-api"></a>Acerca de la API de WinSNMP

\[SNMP está disponible para su uso en los sistemas operativos especificados en la sección Requisitos. En versiones posteriores podría modificarse o no estar disponible. En su lugar, [Windows administración remota](/windows/desktop/WinRM/portal), que es la implementación de Microsoft de WS-Man.\]

La API de WinSNMP es una interfaz para el desarrollo de aplicaciones de administración de red habilitadas para SNMP.

La API de WinSNMP proporciona las siguientes características:

-   Tecnología de habilitación de SNMP para aplicaciones de administración de redes
-   Compatibilidad con SNMP versión 1 (SNMPv1) y SNMP versión 2C (SNMPv2C)

Además de las operaciones del administrador de SNMP, la API de WinSNMP versión 2.0 también admite operaciones de agente SNMP.

La API winSNMP admite aplicaciones de 32 bits y se ejecuta en entornos de uno y varios subprocesos. La compatibilidad con la API de WinSNMP está disponible para las aplicaciones que se ejecutan en el Windows operativo de Windows 2000 o posterior.
