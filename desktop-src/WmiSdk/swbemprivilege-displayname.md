# <a name="swbemprivilegedisplayname-property"></a>Propiedad SWbemPrivilege. DisplayName

La propiedad **displayName** de un objeto [**SWbemPrivilege**](swbemprivilege.md) es una cadena que muestra una descripción de un objeto **SWbemPrivilege** . Por ejemplo, el privilegio que controla si un usuario puede ejecutar el método Shutdown de un objeto tiene la cadena "apagar el sistema" en la propiedad **SWbemPrivilege. DisplayName** . Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

```VB
SWbemPrivilege.DisplayName As 
```
