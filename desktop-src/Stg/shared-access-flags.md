# <a name="shared-access-flags"></a>Marcas de acceso compartido

Las marcas de acceso compartido especifican el grado de acceso que pueden tener otras aplicaciones a un objeto que la aplicación está abriendo. Puede denegar el acceso de lectura, denegar el acceso de escritura o denegar todo el acceso. También puede especificar explícitamente que no se deniega ningún tipo de acceso.
