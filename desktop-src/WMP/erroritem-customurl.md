# <a name="erroritemcustomurl"></a>ErrorItem.customUrl

La **propiedad customURL** recupera la dirección URL de un sitio web que muestra información específica sobre el error de descarga del códec.

``` syntax
player.error.item(
        index
        ).customURL
      
```

## <a name="possible-values"></a>Valores posibles

Esta propiedad es una cadena de solo **lectura.**

## <a name="remarks"></a>Comentarios

**Reproductor de Windows Media 10 Mobile:** Esta propiedad siempre devuelve una cadena vacía.
