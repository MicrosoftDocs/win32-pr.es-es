# <a name="d1174-pushlayer-to-command-list-will-ignore-layer-object"></a>D1174: PushLayer a la lista de comandos omitirá el objeto de capa

Al insertar una capa (no NULL) en una lista de comandos, se omitirá el objeto de capa.



|             |         |
|-------------|---------|
| Nivel de error | Advertencia |



 

## <a name="possible-causes"></a>Causas posibles

Cuando un comando PushLayer usa un objeto ID2D1Layer y el destino es una lista de comandos, Direct2D se comportará como si no se hubiera especificado ninguna capa y no impondrá límites fijas en la capa si la lista de comandos se representa posteriormente.

## <a name="possible-fixes"></a>Posibles correcciones

Las aplicaciones que prevén la transmisión de comandos a una lista de comandos deben considerar el uso de objetos de nivel nulo o tener en cuenta que la configuración de restricciones de tamaño especificada en el objeto ID2D1Layer no se respetará.
