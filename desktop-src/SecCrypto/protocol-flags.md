# <a name="protocol-flags"></a>Marcas de protocolo

A continuación se enumeran las constantes predefinidas para los protocolos de criptografía. Estos valores se definen en Wincrypt. h.



| Constante o valor                                                                                                                                                                                                                            | Descripción                                                           |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------|
| <span id="CRYPT_FLAG_IPSEC"></span><span id="crypt_flag_ipsec"></span><dl> <dt>**Cifrado \_ MARCAR \_ IPSec**</dt> <dt>0x0010</dt> </dl>       | Protocolo de seguridad de Internet (IPsec).<br/>               |
| <span id="CRYPT_FLAG_PCT1"></span><span id="crypt_flag_pct1"></span><dl> <dt>**Cifrado \_ MARCA \_ PCT1**</dt> <dt>0x0001</dt> </dl>          | Protocolo de transporte de comunicaciones privadas (PCT) versión 1.<br/> |
| <span id="CRYPT_FLAG_SIGNING"></span><span id="crypt_flag_signing"></span><dl> <dt>**Cifrado \_ 0x0020 de \_ firma de marca**</dt> <dt></dt> </dl> | Protocolo de firma.<br/>                                          |
| <span id="CRYPT_FLAG_SSL2"></span><span id="crypt_flag_ssl2"></span><dl> <dt>**Cifrado \_ MARCA \_ SSL2**</dt> <dt>0x0002</dt> </dl>          | Protocolo de capa de sockets seguros (SSL) versión 2.<br/>             |
| <span id="CRYPT_FLAG_SSL3"></span><span id="crypt_flag_ssl3"></span><dl> <dt>**Cifrado \_ MARCA \_ SSL3**</dt> <dt>0x0004</dt> </dl>          | Protocolo SSL versión 3.<br/>                                    |
| <span id="CRYPT_FLAG_TLS1"></span><span id="crypt_flag_tls1"></span><dl> <dt>**Cifrado \_ MARCA \_ TLS1**</dt> <dt>0x0008</dt> </dl>          | Protocolo de seguridad de la capa de transporte (TLS) versión 1.<br/>         |
