# <a name="frs-root-security-attribute"></a>Atributo FRS-root-Security

Descriptor de seguridad de la raíz del conjunto de réplicas para la replicación de archivos.



| Entrada | Value |
|-------------------|-----------------------------------------------------|
| CN                | FRS-raíz-seguridad                                   |
| Nombre para mostrar de LDAP | fRSRootSecurity                                     |
| Tamaño              | \-                                                  |
| Actualizar privilegio  | \-                                                  |
| Frecuencia de actualización  | \-                                                  |
| Attribute-Id      | 1.2.840.113556.1.4.535                              |
| System-ID-GUID    | 5245801f-ca6a-11d0-afff-0000f80367c1                |
| Sintaxis            | [**String(NT-Sec-Desc)**](s-string-nt-sec-desc.md) |
