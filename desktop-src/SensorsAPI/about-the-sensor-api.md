# <a name="about-the-sensor-api"></a>Acerca de la API de sensor

La API de sensor proporciona su funcionalidad a través de un conjunto de interfaces COM. La API de sensor expone algunas de estas interfaces, lo que significa que puede llamar a métodos en ellas. La API define otras interfaces para que pueda implementar los métodos que la API de sensor necesita para proporcionar notificaciones de eventos al programa.
