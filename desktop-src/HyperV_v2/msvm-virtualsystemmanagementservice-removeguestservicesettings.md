# <a name="removeguestservicesettings-method-of-the-msvm_virtualsystemmanagementservice-class"></a>Método RemoveGuestServiceSettings de la \_ clase VirtualSystemManagementService de MSVM

Quita la configuración del servicio de invitado de una configuración de sistema virtual.

Cuando se aplica a partes de una configuración de sistema virtual "actual", se puede modificar como un efecto secundario en los servicios invitados del sistema virtual activo.

## <a name="syntax"></a>Sintaxis


```mof
uint32 RemoveGuestServiceSettings(
  [in]  CIM_SettingData REF GuestServiceSettings[],
  [out] CIM_ConcreteJob REF Job
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*GuestServiceSettings* \[ de\]
</dt> <dd>

Referencia a una matriz de [**un \_ SettingData de CIM**](cim-settingdata.md) que describe la configuración del servicio de invitado que se va a quitar.

</dd> <dt>

*Trabajo* \[ de enuncia\]
</dt> <dd>

Si la operación se realiza de forma asincrónica, este método devolverá 4096 y este parámetro contendrá una referencia a un objeto derivado de [**\_ ConcreteJob CIM**](/previous-versions//cc136808(v=vs.85)).

</dd> </dl>
