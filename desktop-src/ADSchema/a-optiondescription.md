# <a name="option-description-attribute"></a>Option-Description atributo)

Este atributo contiene una descripción de una opción que se establece en el servidor DHCP.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Option-Description                          |
| Nombre para mostrar de LDAP | optionDescription                           |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.712                      |
| System-ID-GUID    | 963d274d-48be-11d1-a9c3-0000f80367c1        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
