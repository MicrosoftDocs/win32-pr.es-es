# <a name="locale_use_cp_acp"></a>CONFIGURACIÓN REGIONAL \_ USE \_ CP \_ ACP

**Windows Me/98, Windows 2000:** El sistema predeterminado Windows página de códigos ANSI (ACP) en lugar de la página de códigos de configuración regional que se usa para la traducción de cadenas. Vea [Identificadores de página de códigos](code-page-identifiers.md) para obtener una lista de ANSI y otras páginas de códigos.
