# <a name="shell-classes"></a>Clases de Shell

En esta sección se describen las clases del shell de Windows.

## <a name="in-this-section"></a>En esta sección



| Tema                                                               | Descripción                                                                                                                                                                                                                                                                                                                                                                                                  |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CCscSearchApiInterface**](/previous-versions/windows/desktop/legacy/cc448312(v=vs.85))<br/> | Expone métodos para interactuar con la biblioteca de API de búsqueda CSC (almacenamiento en caché del lado cliente, también conocido como Archivos sin conexión).<br/>                                                                                                                                                                                                                                                                               |
| [**CFolderItems**](class-cfolderitems-class.md)<br/>         | [**CFolderItems**](class-cfolderitems-class.md) es una colección de [**FolderItems**](folderitems.md). Implementa las interfaces siguientes: [**IPersistFolder**](/windows/desktop/api/shobjidl_core/nn-shobjidl_core-ipersistfolder), [**IObjectWithSite**](/windows/win32/api/ocidl/nn-ocidl-iobjectwithsite), [**FolderItems3**](folderitems3-object.md), [**IObjectSafety**](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa768224(v=vs.85)), [**IDispatch**](/windows/win32/api/oaidl/nn-oaidl-idispatch).<br/> |
| [**CFolderItemsFDF**](class-cfolderitemsfdf-class.md)<br/>   | [**CFolderItemsFDF**](class-cfolderitemsfdf-class.md) es una colección de [**FolderItems**](folderitems.md). Implementa la interfaz [**IInsertItem**](/windows/desktop/api/Shobjidl/nn-shobjidl-iinsertitem) .<br/>                                                                                                                                                                                                                     |
| [**CItemIDFactory**](/windows/desktop/api/shidfact/nl-shidfact-citemidfactory)<br/>                 | Expone métodos para interactuar con los orígenes de datos de Shell.<br/>                                                                                                                                                                                                                                                                                                                                          |
| [**VirtualDesktopManager**](virtualdesktopmanager.md)<br/>   | [**VirtualDesktopManager**](virtualdesktopmanager.md) implementa la interfaz [**IVirtualDesktopManager**](/windows/win32/api/shobjidl_core/nn-shobjidl_core-ivirtualdesktopmanager) .<br/>                                                                                                                                                                                                                                                         |
