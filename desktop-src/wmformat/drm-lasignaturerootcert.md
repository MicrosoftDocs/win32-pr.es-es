# <a name="drm_lasignaturerootcert"></a>DRM \_ LASignatureRootCert

El **atributo \_ DRM LASignatureRootCert** contiene el certificado raíz en la cadena de certificación que se usa para autenticar el certificado contenido en [**DRM \_ LASignatureLicSrvCert**](drm-lasignaturelicsrvcert.md).

## <a name="global-constant"></a>Constante global

g \_ wszWMDRM \_ LASignatureRootCert

## <a name="data-type"></a>Tipo de datos

**CADENA DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentarios

Esta propiedad se puede establecer con el [**método IWMDRMWriter::SetDRMAttribute.**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) No es accesible para el objeto de lector.
