# <a name="options-location-attribute"></a>Options-Location atributo

Para DHCP, la ubicación de opciones contiene el DN para sitios alternativos que contienen la información de opciones.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Options-Location                     |
| Ldap-Display-Name | optionsLocation                      |
| Size              | \-                                   |
| Actualizar privilegios  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.713               |
| System-Id-Guid    | 963d274e-48be-11d1-a9c3-0000f80367c1 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
