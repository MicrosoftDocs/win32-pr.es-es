|-------------------------------------------------------------------------|-----------------------------------------------------------------|
| [**XMVector4Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector4transform)<br/>             | Transforma un vector 4D mediante una matriz.<br/>                  |
