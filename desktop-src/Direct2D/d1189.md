# <a name="d1189-cpu-read-bitmap-option-was-used-incorrectly"></a>D1189: se usó incorrectamente la opción de mapa de bits de lectura de CPU

\_Opciones de mapa de bits D2D1 la \_ \_ \_ lectura de CPU debe combinarse con \_ las opciones de mapa de bits D2D1 \_ \_ no puede \_ dibujar






 

## <a name="possible-causes"></a>Causas posibles

La aplicación especificó una combinación incompatible de las \_ marcas de opciones de mapa de bits de D2D1 \_ .

## <a name="possible-fixes"></a>Posibles correcciones

Quite una de las marcas para crear un conjunto válido de opciones de mapa de bits de D2D1 \_ \_ .
