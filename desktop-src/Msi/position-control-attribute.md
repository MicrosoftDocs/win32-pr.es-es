# <a name="position-control-attribute"></a>Atributo de control de posición

Este atributo determina la posición del control en el cuadro de diálogo. Los cuatro campos del atributo son: izquierda, superior, ancho y alto. Escriba el ancho, el alto y las coordenadas de la esquina izquierda del control en las columnas width, height, X e Y de la tabla de [control](control-table.md). Use [unidades de instalador](installer-units.md) para las unidades de longitud y distancia.

## <a name="valid-controls"></a>Controles válidos

Todos los controles activos.

## <a name="associated-bit-flags"></a>Marcas de bits asociadas

Este atributo no utiliza marcas de bits.

## <a name="remarks"></a>Observaciones

Vea [atributos de control](control-attributes.md) y el control que debe crear en [controles](controls.md).

Una unidad de instalador es igual a una duodécima el alto del tamaño de fuente MS Sans Serif de 10 puntos.
