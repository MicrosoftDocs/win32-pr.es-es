# <a name="videotooltip"></a>VÍDEO. información sobre herramientas

El atributo **ToolTip** especifica o recupera el texto de información sobre herramientas para la ventana de vídeo.

``` syntax
        elementID.toolTip
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de lectura/escritura, que no debe superar los 1024 caracteres de longitud. No tiene valor predeterminado.

## <a name="remarks"></a>Observaciones

Cuando este atributo se establece en "" (cadena vacía), no se muestra ninguna información sobre herramientas.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
