# <a name="boundingorientedboxintersects-methods"></a>Métodos BoundingOrientedBox. Intersects

Comprueba el [**BoundingOrientedBox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) para la intersección con otro objeto.

| Método                                                                                                   | Descripción                                                                                                                                 |
|:---------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------|
| [**BoundingOrientedBox:: Intersects (XMVECTOR)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingorientedbox-intersects(fxmvector))                   | Comprueba el [**BoundingOrientedBox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) para la intersección con un plano.<br/>                                      |
| [**BoundingOrientedBox:: Intersects (const BoundingBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingorientedbox-intersects(constboundingbox_))         | Comprueba el [**BoundingOrientedBox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) para la intersección con un [**BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox).<br/>         |
| [**BoundingOrientedBox:: Intersects (const BoundingSphere&)**](/previous-versions/windows/desktop/legacy/hh855930(v=vs.85))      | Comprueba la intersección de [**BoundingOrientedBox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) con un [**BoundingSphere**](/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere).<br/>   |
| [**BoundingOrientedBox:: Intersects (const BoundingFrustum&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingorientedbox-intersects(constboundingfrustum_))     | Comprueba la intersección de [**BoundingOrientedBox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) con un [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/> |
| [**BoundingOrientedBox:: Intersects (XMVECTOR, XMVECTOR, Float&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingorientedbox-intersects(fxmvector_fxmvector_float_))   | Comprueba el [**BoundingOrientedBox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) para la intersección con un rayo.<br/>                                        |
| [**BoundingOrientedBox:: Intersects (XMVECTOR, XMVECTOR, XMVECTOR)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingorientedbox-intersects(fxmvector_fxmvector_fxmvector)) | Comprueba el [**BoundingOrientedBox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) para la intersección con un triángulo.<br/>                                   |
| [**BoundingOrientedBox:: Intersects (const BoundingOrientedBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingorientedbox-intersects(constboundingorientedbox_)) | Comprueba la intersección de [**BoundingOrientedBox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox) con un **BoundingOrientedBox**.<br/>                    |
