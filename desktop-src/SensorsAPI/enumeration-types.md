# <a name="enumeration-types"></a>Tipos de enumeración

La API de sensor define los siguientes tipos de enumeración.



| Enumeración                                                                | Descripción                                      |
|----------------------------------------------------------------------------|--------------------------------------------------|
| [**SensorConnectionType**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorconnectiontype) | Define los tipos de conexiones del dispositivo de sensor.      |
| [**SensorState**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorstate)                   | Define los posibles estados operativos de los sensores. |
