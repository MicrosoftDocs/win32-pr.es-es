# <a name="jet_unicodeindexdwmapflags-property"></a>JET_UNICODEINDEX.dwMapFlags, propiedad

Obtiene o establece las marcas que se usarán con LCMapString al normalizar datos Unicode.

Esta API no es conforme a CLS. 

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop](./microsoft.isam.esent.interop-namespace.md)
