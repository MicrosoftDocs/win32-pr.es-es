# <a name="application-inventory-classes"></a>Clases de inventario de aplicaciones

## <a name="purpose"></a>Propósito

Las clases WMI de Inventario de aplicaciones permiten la detección de las aplicaciones Win32 instaladas y Windows almacenar aplicaciones en un Windows aplicación.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                  | Descripción                                                                                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Win32 \_ InstalledWin32Program**](win32-installedwin32program.md)<br/>         | La [**clase Win32 \_ InstalledWin32Program**](win32-installedwin32program.md) representa una aplicación Win32 instalada.<br/>                                                                                                                            |
| [**Win32 \_ InstalledStoreProgram**](win32-installedstoreprogram.md)<br/>         | La [**clase \_ InstalledStoreProgram de Win32 representa**](win32-installedstoreprogram.md) una aplicación Windows store instalada.<br/>                                                                                                                    |
| [**Win32 \_ InstalledProgramFramework**](win32-installedprogramframework.md)<br/> | La [**clase \_ InstalledProgramFramework de Win32**](win32-installedprogramframework.md) representa un marco de tecnología en el que se compila o usa una instancia [**de \_ Win32 InstalledWin32Program**](win32-installedwin32program.md) en tiempo de ejecución.<br/> |



 

## <a name="developer-audience"></a>Audiencia de desarrolladores

Las clases WMI de Inventario de aplicaciones están diseñadas para su uso por los desarrolladores de soluciones de administración de Enterprise y herramientas de administración local o remota que necesitan generar un inventario de aplicaciones en un sistema.
