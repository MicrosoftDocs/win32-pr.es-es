# <a name="pending-parent-ca-attribute"></a>Atributo Pending-Parent-CA

Referencia a las entidades de certificación que emitieron los certificados pendientes para esta entidad de certificación.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | Pendiente-primario-CA                       |
| Nombre para mostrar de LDAP | pendingParentCA                         |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.695                  |
| System-ID-GUID    | 963d273e-48be-11d1-a9c3-0000f80367c1    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
