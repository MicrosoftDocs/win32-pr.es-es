# <a name="d1173-texture-cube-interop-unsupported"></a>D1173: Interoperabilidad del cubo de textura no admitida

Direct2D no puede usar texturas marcadas como D3D11RESOURCE \_ MISC \_ TEXTURECUBE.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación intentó interoperar una textura direct3D con la opción MISC TEXTURECUBE de D3D11RESOURCE \_ \_ especificada.

## <a name="possible-fixes"></a>Posibles correcciones

Evite interoperar texturas que tengan el especificador TEXTURECUBE. En su lugar, reorganize el diseño de representación de la aplicación o considere la posibilidad de interoperar un intermedio.
