# <a name="systemcontactaccountpicturelarge"></a>System. contact. AccountPictureLarge

Se trata de una secuencia que contiene los datos necesarios para representar la imagen de la cuenta grande de un contacto.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Contact.AccountPictureLarge
   shellPKey = PKEY_Contact_AccountPictureLarge
   formatID = 0B8BB018-2725-4B44-92BA-7933AEB2DDE7
   propID = 3
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Stream
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
