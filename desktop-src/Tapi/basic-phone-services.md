# <a name="basic-phone-services"></a>Basic Teléfono Services

Los servicios telefónicos básicos son el conjunto básico de servicios que admiten todos los dispositivos telefónicos. Dado que los servicios telefónicos básicos están definidos por el proveedor de servicios, hay mucha variedad en el conjunto principal.
