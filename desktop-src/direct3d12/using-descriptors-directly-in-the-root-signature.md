<span data-ttu-id="3b537-106">Un ejemplo de uso sería colocar en el diseño raíz una vista de búfer constante (CBV) que cambia por dibujo.</span><span class="sxs-lookup"><span data-stu-id="3b537-106">An example usage would be to place in the root layout a constant buffer view (CBV) that is changing per draw.</span></span> <span data-ttu-id="3b537-107">Esto es para que la aplicación no tenga que asignar el espacio del montón de descriptores por dibujo (y guarda apuntando a una tabla de descriptores en la nueva ubicación del montón de descriptores).</span><span class="sxs-lookup"><span data-stu-id="3b537-107">That's so that descriptor heap space doesn't have to be allocated by the application per draw (and saves pointing a descriptor table at the new location in the descriptor heap).</span></span> <span data-ttu-id="3b537-108">Al colocar algo en la firma raíz, la aplicación simplemente entrega la responsabilidad de control de versiones al controlador. pero esa es la infraestructura que los controladores ya tienen.</span><span class="sxs-lookup"><span data-stu-id="3b537-108">By putting something in the root signature, the application is merely handing the versioning responsibility to the driver; but that's infrastructure that drivers already have.</span></span>
