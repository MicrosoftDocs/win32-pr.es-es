# <a name="managing-dns-zones"></a>Administrar Zonas DNS

Una zona DNS es una base de datos de entradas de RR que corresponde a parte del espacio de nombres jerárquico de DNS. Las zonas DNS se utilizan para delimitar qué servidores DNS son autoritativos para resolver consultas de resolución de nombres para una sección determinada de la jerarquía de DNS.

## <a name="administration-steps"></a>Pasos de administración

El proveedor WMI de DNS permite la administración de zonas DNS desde el propio servidor DNS autoritativo o desde hosts remotos. Los pasos generales necesarios para administrar una zona mediante el proveedor WMI de DNS son los siguientes:

-   Conexión al proveedor WMI de DNS
-   Obtener una instancia de zona
-   Enumerar o modificar una propiedad de zona, o usar un método
