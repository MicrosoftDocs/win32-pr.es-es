# <a name="cooker_range"></a>INTERVALO DE LA \_ COCINA

La fórmula de tipo de contador COOKER RANGE muestra la diferencia entre los valores Mínimo y Máximo para un conjunto de observaciones sin procesar de una propiedad en una instancia \_ [**de Win32 \_ PerfRawData.**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) [](cooker-min.md) [](cooker-max.md) Este tipo de contador solo se define en WMI y no está disponible para las tecnologías de supervisión de rendimiento, como contadores de rendimiento [versión 6.0.](/windows/desktop/PerfCtrs/performance-counters-portal)

Para obtener más información sobre la fórmula de tipo de contador, vea [Tipos de contador](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Para obtener más información sobre los proveedores de alto rendimiento y el scripting, vea [Tipos de contadores de rendimiento wmi.](wmi-performance-counter-types.md)
