# <a name="starting-an-executable-weekly"></a>Iniciar una aplicación ejecutable semanalmente

La escritura de una tarea que inicia un archivo ejecutable semanalmente se realiza definiendo un desencadenador semanal y una acción ejecutable.

## <a name="weekly-triggers"></a>Desencadenadores semanales

Los desencadenadores semanales usan su límite de inicio para activar el desencadenador y especificar la hora del día en que se ejecuta la tarea. Cuando se activa el desencadenador, el desencadenador también especifica cuándo se ejecuta la tarea durante la semana y si la tarea se ejecuta cada semana, cada semana, cada tres semanas, etc.
