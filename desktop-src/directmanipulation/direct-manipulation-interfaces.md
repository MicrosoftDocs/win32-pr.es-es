# <a name="direct-manipulation-interfaces"></a>Interfaces de manipulación directa

Los temas contenidos en esta sección proporcionan las especificaciones de referencia para las interfaces [de manipulación](direct-manipulation-portal.md) directa.

> [!Note]  
> Al implementar un objeto [de manipulación](direct-manipulation-portal.md) directa, asegúrese de que la implementación [**de IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) admite el multithreading mediante el recuento de referencias seguras para subprocesos. Para obtener más información, [**vea InterlockedIncrement**](/windows/win32/api/winnt/nf-winnt-interlockedincrement) e [**InterlockedDecrement**](/windows/win32/api/winnt/nf-winnt-interlockeddecrement).

## <a name="in-this-section"></a>En esta sección

| Tema                                                                                                       | Descripción                                                                                                                                                                                                                                                                  |
|-------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IDirectManipulationAutoScrollBehavior**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationautoscrollbehavior)<br/>           | Representa el comportamiento de animación de desplazamiento automático del contenido a medida que se aproxima al límite de un eje o eje determinado.<br/>                                                                                                                                                   |
| [**IDirectManipulationCompositor**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationcompositor)<br/>                           | Representa un objeto compositor que asocia contenido manipulado a una superficie de dibujo, como el [**lienzo**](/uwp/api/Windows.UI.Xaml.Controls.Canvas).<br/> |
| [**IDirectManipulationCompositor2**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationcompositor2)<br/>                         | Representa un objeto compositor que asocia contenido manipulado con superficies de dibujo en varios procesos.<br/>                                                                                                                                               |
| [**IDirectManipulationContent**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationcontent)<br/>                                 | Encapsula el contenido dentro de una ventanilla, donde el contenido representa una superficie visual recortada dentro de la ventanilla.<br/>                                                                                                                                                    |
| [**IDirectManipulationDeferContactService**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationdefercontactservice)<br/>         | Representa un servicio para administrar asociaciones entre un contacto y una ventanilla.<br/>                                                                                                                                                                                  |
| [**IDirectManipulationDragDropBehavior**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationdragdropbehavior)<br/>               | Representa los comportamientos de las interacciones de arrastrar y colocar, que se desencadenan mediante gestos de deslizar o presionar y mantener presionados. <br/>                                                                                                                                              |
| [**IDirectManipulationDragDropEventHandler**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationdragdropeventhandler)<br/>       | Define métodos para controlar eventos de comportamiento de arrastrar y colocar.<br/>                                                                                                                                                                                                              |
| [**IDirectManipulationFrameInfoProvider**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationframeinfoprovider)<br/>             | Representa un objeto de mantenimiento del tiempo que mide la latencia de la infraestructura de composición utilizada por la aplicación y proporciona estos datos a [manipulación directa.](direct-manipulation-portal.md) <br/>                                                            |
| [**IDirectManipulationInteractionEventHandler**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationinteractioneventhandler)<br/> | Define métodos para controlar las interacciones cuando se detectan.<br/>                                                                                                                                                                                                    |
| [**IDirectManipulationManager**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager)<br/>                                 | Proporciona acceso a todas las características y API [de manipulación](direct-manipulation-portal.md) directa disponibles para la aplicación cliente.<br/>                                                                                                                           |
| [**IDirectManipulationManager2**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager2)<br/>                               | Extiende la interfaz [**IDirectManipulationManager**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager) que proporciona [](direct-manipulation-portal.md) acceso a todas las características y API de manipulación directa disponibles para la aplicación cliente. <br/>                              |
| [**IDirectManipulationManager3**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager3)<br/>                               | Extiende la interfaz [**IDirectManipulationManager2**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager2) que proporciona [](direct-manipulation-portal.md) acceso a todas las características y API de manipulación directa disponibles para la aplicación cliente. <br/>                            |
| [**IDirectManipulationPrimaryContent**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationprimarycontent)<br/>                   | Encapsula el contenido principal dentro de una ventanilla.<br/>                                                                                                                                                                                                               |
| [**IDirectManipulationUpdateHandler**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationupdatehandler)<br/>                     | Define métodos para controlar eventos de actualización de manipulación.<br/>                                                                                                                                                                                                          |

| [**IDirectManipulationViewport**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationviewport)<br/>                               | Define una región dentro de una ventana (denominada ventanilla) que puede recibir y procesar la entrada de las interacciones del usuario. <br/>                                                                                                                                   |

| [**IDirectManipulationViewportEventHandler**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationviewporteventhandler)<br/>       | Define métodos para controlar el estado y actualizar los eventos de la ventanilla.<br/>                                                                                                                                                                                           |
