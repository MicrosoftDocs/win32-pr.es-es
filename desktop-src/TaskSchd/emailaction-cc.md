# <a name="emailactioncc-property"></a>Propiedad EmailAction.Cc

\[Este objeto ya no se admite. Use IExecAction con el cmdlet [**send-MailMessage**](/powershell/module/microsoft.powershell.utility/send-mailmessage) de PowerShell como solución alternativa.\]

En el caso de scripting, obtiene o establece la dirección o direcciones de correo electrónico que desea CC en el mensaje de correo electrónico.

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Sintaxis


```VB
EmailAction.Cc As String
```



## <a name="property-value"></a>Valor de propiedad

La dirección o direcciones de correo electrónico que desea incluir en CC en el mensaje de correo electrónico.
