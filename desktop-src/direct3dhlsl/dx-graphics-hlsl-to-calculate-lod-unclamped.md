# <a name="calculatelevelofdetailunclamped-directx-hlsl-texture-object"></a>CalculateLevelOfDetailUnclamped (objeto de textura de HLSL de DirectX)

Calcula el LOD sin fijar el resultado.



| RET Object. CalculateLevelOfDetailUnclamped ( \_ Estado de muestra S, Float x); |
|--------------------------------------------------------------------------|



 

Esta función funciona de forma idéntica a [CalculateLevelOfDetail (objeto de textura de HLSL de DirectX)](dx-graphics-hlsl-to-calculate-lod.md), salvo que no hay ninguna compresión en el LOD calculado.
