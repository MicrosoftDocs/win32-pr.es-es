# <a name="locale_slist"></a>LOCALE \_ SLIST

Los caracteres que se usan para separar elementos de lista, por ejemplo, se usa una coma en muchas configuraciones regionales. El número máximo de caracteres permitido para esta cadena es cuatro, incluido un carácter nulo de terminación.
