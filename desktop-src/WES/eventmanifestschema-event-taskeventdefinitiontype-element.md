# <a name="event-taskeventdefinitiontype-element"></a>Elemento Event (TaskEventDefinitionType)

\[A partir del compilador de mensajes que se incluye con la versión de Windows 7 del Windows SDK, este elemento ya no está disponible.\]

Define un evento para un evento específico de tarea.

``` syntax
<xs:element name="event"
    type="EventDefinitionType"
 />
```

El elemento de **evento** se define mediante el tipo complejo de [**TaskEventDefinitionType**](eventmanifestschema-taskeventdefinitiontype-complextype.md) .
