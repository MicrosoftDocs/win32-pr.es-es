# <a name="monitor-configuration-monitor-configuration"></a>Supervisar configuración (configuración de monitor)

Puede usar las funciones de configuración de supervisión para obtener información de un monitor y cambiar la configuración de un monitor. Puede usar estas funciones para :

-   Cambie la configuración de geometría de un monitor, como el ancho y el alto de la pantalla.
-   Cambie la configuración de la imagen, como el brillo y el contraste.
-   Restablezca la configuración de un monitor a sus valores predeterminados de fábrica.
-   Desasoye el monitor.

Internamente, las funciones de configuración del monitor usan la interfaz de comandos del canal de datos para mostrar (DDC/CI) para enviar comandos al monitor.

Las funciones de configuración del monitor se documentan en las secciones siguientes:

-   [Acerca de la configuración del monitor](about-monitor-configuration.md)
-   [Uso de la configuración de Monitor](using-monitor-configuration.md)
-   [Referencia de configuración de supervisión](monitor-configuration-reference.md)
