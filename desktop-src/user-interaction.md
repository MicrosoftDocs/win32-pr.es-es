# <a name="user-interaction"></a>Interacción del usuario

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
| --- | --- |
| [Manipulación directa](./directmanipulation/direct-manipulation-portal.md)<br/> | Las API de [manipulación directa](directmanipulation/direct-manipulation-reference.md) se usan para procesar la entrada táctil en una región o un objeto, generar la salida que va a usar un motor de composición para aplicar transformaciones a la superficie de representación relacionada y optimizar la respuesta y reducir la latencia mediante la salida predictiva basada en el tiempo de representación del compositor.<br/> |
| [Entrada manuscrita](input_ink/input-ink-portal.md)<br/> | Obtenga información sobre las distintas API de entrada manuscrita de escritorio para aplicaciones Windows.<br/>  |
| [Configuración de comentarios de entrada](input_feedback/input-feedback-configuration-portal.md)<br/> | Las [API de configuración de comentarios de entrada](input_feedback/input-feedback-configuration-reference.md) le permiten configurar la información visual de las interacciones de los usuarios en función de la aplicación Windows o el marco de interfaz de usuario que está desarrollando. <br/> |
| [Identificación del origen de entrada](input_sourceid/input-source-identification-portal.md)<br/>   | Las API de [identificación del origen de entrada](input_sourceid/input-source-identification-reference.md) permiten a las aplicaciones Windows detectar de forma precisa y confiable un origen de entrada, identificar el tipo de entrada y proporcionar una experiencia de usuario más adecuada para el tipo de entrada. <br/> |
| [Contexto de interacción](input_intcontext/interaction-context-portal.md)<br/> | El [contexto de interacción](input_intcontext/interaction-context-reference.md) permite que las aplicaciones de Windows y los marcos de interfaz de usuario admitan varias interacciones simultáneas mediante la detección de gestos y el procesamiento de manipulación.<br/> |
| [Pila de entrada de dispositivo de puntero](input_pointerdevice/pointer-device-stack-portal.md)<br/> | En los temas de esta sección se proporciona información general sobre las API de referencia de la [pila de entrada del dispositivo de puntero](input_pointerdevice/unified-input-stack-reference.md) de Windows. La pila de entrada de dispositivo de puntero proporciona información acerca de los dispositivos de entrada conectados al sistema.<br/> |
| [Mensajes de entrada de puntero y notificaciones](inputmsg/messages-and-notifications-portal.md)<br/> | Las API de [mensajes de entrada de puntero](inputmsg/wmpointer-reference.md) exponen mensajes de entrada relacionados con el puntero, notificaciones y funciones asociadas en aplicaciones Windows. <br/> |
| [Entrada de controlador radial](input_radial/radialcontroller-portal.md)<br/> | Obtenga información sobre las distintas API de escritorio para los accesorios de la rueda de Windows, como el dial Surface (disponible para su compra en el [Microsoft Store](https://www.microsoft.com/store/d/Surface-Dial/925R551SKTGN?icid=Surface_Accessories_ModB_Surface_Dial_103116)). <br/> |
| [Prueba de posicionamiento táctil](input_touchhittest/touch-hit-testing-portal.md)<br/> | Use las API de [prueba de posicionamiento táctil](input_touchhittest/touchhittest-reference.md) para identificar mediante programación un destino de entrada determinando si una geometría o un punto se encuentra dentro del área de contenido de un elemento de la interfaz de usuario. <br/> |
| [Inyección táctil](input_touchinjection/touch-injection-portal.md)<br/> | Las API de [inyección táctil](input_touchinjection/touch-injection-reference.md) permiten a los desarrolladores de Windows simular mediante programación la entrada táctil.<br/> |
| [Características de interacción con el usuario heredadas](./legacy-user-interaction-features.md)<br/> | Características de interacción del usuario para Windows 7 y versiones anteriores. <br/> |
