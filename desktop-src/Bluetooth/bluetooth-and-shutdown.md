# <a name="bluetooth-and-shutdown"></a>Bluetooth y apagado

Bluetooth usa la función [**Shutdown**](/windows/desktop/api/winsock/nf-winsock-shutdown) para desconectarse de la radio remota. Bluetooth no tiene compatibilidad de nivel de protocolo para los mismos cierres o semicierres correctos que proporciona TCP/IP.
