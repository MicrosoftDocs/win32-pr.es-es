# <a name="w-telephony-api"></a>W (API de telefonía)

[A](a-tapgloss.md) [B](b-tapgloss.md) [C](c-tapgloss.md) [D](d-tapgloss.md) [E](e-tapgloss.md) F G [H](h-tapgloss.md) [I](i-tapgloss.md) J [K](k-tapgloss.md) [L](l-tapgloss.md) [M](m-tapgloss.md) N o [p](p-tapgloss.md) Q [R](r-tapgloss.md) [s](s-tapgloss.md) [T](t-tapgloss.md) [U](u-tapgloss.md) [V](v-tapgloss.md) W X Y Z

<dl> <dt>

<span id="tapi2.web_telephony_tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_TAPGLOSS"></span>**Telefonía Web**
</dt> <dd>

Plataforma de aplicaciones que admite sitios web de acceso dual (accesible por teléfono o explorador de Internet) y aplicaciones IVR basadas en HTML. La plataforma incluye un motor en tiempo de ejecución integrado en el servidor, una interfaz de administración basada en COM para instalar y configurar aplicaciones de telefonía Web, y acceso al objeto de llamada TAPI 3. x y a la API de scripting para implementar operaciones de telefonía.

</dd> <dt>

<span id="tapi2.web_telephony_engine_webte__tapgloss"></span><span id="TAPI2.WEB_TELEPHONY_ENGINE_WEBTE__TAPGLOSS"></span>**Motor de telefonía web (WTE)**
</dt> <dd>

Proporciona compatibilidad con aplicaciones IVR basadas en HTML y sitios web de acceso dual (un sitio web al que se puede tener acceso por teléfono, así como a través de un explorador gráfico). Consulte *telefonía web*.

</dd> <dt>

<span id="tapi2.windows_open_services_architecture_wosa__tapgloss"></span><span id="TAPI2.WINDOWS_OPEN_SERVICES_ARCHITECTURE_WOSA__TAPGLOSS"></span>**Arquitectura de servicios abiertos de Windows (WOSA)**
</dt> <dd>

Arquitectura de software que proporciona una única interfaz de nivel de sistema para conectar aplicaciones front-end con servicios back-end. La telefonía de Windows forma parte de WOSA.

</dd> </dl>
