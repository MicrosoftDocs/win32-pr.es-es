# <a name="byte-order-indicator"></a>Indicador de Byte-Order

El indicador de orden de bytes es un tipo de datos de **Word** y siempre debe contener el valor 0xFFFE. Es el mismo que el indicador de orden de bytes Unicode. Este valor siempre se escribe en el orden de bytes de Intel y, por lo tanto, aparece en el archivo o la secuencia como 0xFE, 0xFF.
