# <a name="bitmap-rotation"></a>Rotación de mapas de bits

Para copiar un mapa de bits en un paralelogramo; Use la función [**PlgBlt**](/windows/desktop/api/Wingdi/nf-wingdi-plgblt) , que realiza una transferencia de bloque de bits de un rectángulo en un contexto de dispositivo de origen a un paralelogramo en un contexto de dispositivo de destino. Para girar el mapa de bits, una aplicación debe proporcionar las coordenadas, en unidades universales, que se usarán para las esquinas del paralelogramo. (Para obtener más información sobre la rotación y las unidades universales, vea [espacios de coordenadas y transformaciones](coordinate-spaces-and-transformations.md)).
