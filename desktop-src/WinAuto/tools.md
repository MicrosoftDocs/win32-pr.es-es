# <a name="tools"></a>Herramientas

Puede usar las siguientes herramientas para probar implementaciones de Microsoft Active Accessibility. Para obtener más información acerca de las herramientas de pruebas, vea [herramientas de pruebas](testing-tools.md).



| Herramienta                                                     | Descripción                                                                                                                                                                           |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Accessible Event Watcher](accessible-event-watcher.md) | Registra [WinEvents](winevents-infrastructure.md) a medida que se producen dentro del sistema.                                                                                                         |
| [Inspeccionar](inspect-objects.md)                           | Identifica problemas con la implementación de propiedades y métodos en objetos accesibles de Microsoft Active Accessibility y en elementos de automatización de la automatización de la interfaz de usuario de Microsoft. |
