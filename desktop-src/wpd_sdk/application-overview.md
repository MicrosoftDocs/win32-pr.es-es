# <a name="application-overview"></a>Información general de la aplicación

Windows Dispositivos portátiles (WPD) proporciona una infraestructura que permite el acceso del equipo a dispositivos multifunción que almacenan varios contenido multimedia y no multimedia. Esta infraestructura permite dispositivos convergentes y se centra inicialmente en dispositivos de almacenamiento inteligente, como reproductores multimedia portátiles, cámaras digitales y teléfonos móviles.

WPD consta de una interfaz de controlador de dispositivo (DDI) y un conjunto de interfaces de programación de aplicaciones (API). En esta introducción se describe cómo puede aprovechar estas API en su Windows aplicación.

La infraestructura de WPD ofrece mecanismos de control y representación de dispositivos muy extensibles. Esto permite a los proveedores de soluciones verticales usar la API de WPD para crear experiencias de usuario mejoradas que están fuera del conjunto estandarizado de WPD. Algunos ejemplos de esto son el software proporcionado por el proveedor que realiza actualizaciones de firmware o diagnósticos de dispositivos (para soporte remoto).
