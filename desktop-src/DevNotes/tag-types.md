# <a name="tag-types"></a>Tipos TAG

Describe el tipo de datos del valor asociado a una ETIQUETA.



| Constante o valor                                                                                                                                                                                                                            | Descripción                                           |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| <span id="TAG_TYPE_NULL"></span><span id="tag_type_null"></span><dl> <dt>**TAG \_ Type \_ NULL**</dt> <dt>0x1000</dt> </dl>                | No hay ningún valor asociado a la ETIQUETA.<br/> |
| <span id="TAG_TYPE_BYTE"></span><span id="tag_type_byte"></span><dl> <dt>**TAG \_ Tipo \_ byte**</dt> <dt>0x2000</dt> </dl>                | Valor **BYTE.**<br/>                          |
| <span id="TAG_TYPE_WORD"></span><span id="tag_type_word"></span><dl> <dt>**TAG \_ ESCRIBA \_ WORD**</dt> <dt>0x3000</dt> </dl>                | Valor **de WORD.**<br/>                          |
| <span id="TAG_TYPE_DWORD"></span><span id="tag_type_dword"></span><dl> <dt>**TAG \_ Escriba \_ DWORD**</dt> <dt>0x4000</dt> </dl>             | Valor **DWORD.**<br/>                         |
| <span id="TAG_TYPE_QWORD"></span><span id="tag_type_qword"></span><dl> <dt>**TAG \_ Escriba \_ QWORD**</dt> <dt>0x5000</dt> </dl>             | Valor **ULONGLONG.**<br/>                     |
| <span id="TAG_TYPE_STRINGREF"></span><span id="tag_type_stringref"></span><dl> <dt>**TAG \_ ESCRIBA \_ STRINGREF**</dt> <dt>0x6000</dt> </dl> | Valor de cadena con token.<br/>                  |
| <span id="TAG_TYPE_LIST"></span><span id="tag_type_list"></span><dl> <dt>**TAG \_ Type \_ LIST**</dt> <dt>0x7000</dt> </dl>                | Lista de valores TAG.<br/>                      |
| <span id="TAG_TYPE_STRING"></span><span id="tag_type_string"></span><dl> <dt>**TAG \_ Type \_ STRING**</dt> <dt>0x8000</dt> </dl>          | Valor de cadena.<br/>                            |
| <span id="TAG_TYPE_BINARY"></span><span id="tag_type_binary"></span><dl> <dt>**TAG \_ Type \_ BINARY**</dt> <dt>0x9000</dt> </dl>          | Valor binario.<br/>                            |
