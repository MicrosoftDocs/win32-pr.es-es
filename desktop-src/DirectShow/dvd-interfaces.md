# <a name="dvd-interfaces"></a>Interfaces de DVD

Estas interfaces admiten la reproducción y la navegación de DVD.



| Interfaz                                    | Descripción                                                                                            |
|----------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**IAMDecoderCaps**](/windows/desktop/api/Strmif/nn-strmif-iamdecodercaps)     | Implementado por descodificadores para indicar varias funciones.                                              |
| [**IDvdCmd**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Bloquee el [filtro del navegador de DVD](dvd-navigator-filter.md) hasta que se inicie o se complete un comando.        |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Navegue y juegue DVD-Video títulos, incluidos los títulos de karaoke.                                          |
| [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Cree un gráfico de filtro para la reproducción DVD-Video del software y el hardware disponibles en el sistema.    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Consultar varios atributos de un disco DVD o el estado del explorador de DVD.                               |
| [**IDvdState**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Guarde el estado de la sesión del usuario en el disco, incluida la ubicación de reproducción, el nivel de parental y la región. |
