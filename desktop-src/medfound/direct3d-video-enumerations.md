# <a name="direct3d-9-video-enumerations"></a>Enumeraciones de vídeo de Direct3D 9

Describe las enumeraciones usadas por las interfaces de vídeo de Microsoft Direct3D 9.



| Enumeración                                                                                                                                                                                                                                                                                                                                           |
|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**D3DBUSTYPE**](d3dbustype.md)<br/> Especifica el tipo de bus de E/S utilizado por el adaptador de gráficos.<br/>                                                                                                                                                                                                                                    |
| [**D3DAUTHENTICATEDCHANNEL \_ PROCESSIDENTIFIERTYPE**](d3dauthenticatedchannel-processidentifiertype.md)<br/> Especifica el tipo de proceso que se identifica en la estructura [**D3DAUTHENTICATEDCHANNEL \_ QUERYRESTRICTEDSHAREDRESOURCEPROCESS \_ OUTPUT.**](d3dauthenticatedchannel-queryrestrictedsharedresourceprocess-output.md)<br/> |
| [**D3DAUTHENTICATEDCHANNELTYPE**](d3dauthenticatedchanneltype.md)<br/> Especifica el tipo de canal autenticado de Direct3D.<br/>                                                                                                                                                                                                        |
