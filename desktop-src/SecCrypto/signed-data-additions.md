# <a name="signed-data-additions"></a>Adiciones de datos firmados

Se han realizado los siguientes cambios en las capacidades de datos firmadas:

-   El [*contenido interno*](../secgloss/i-gly.md) que no es de datos se encapsula en una cadena de octeto.
-   El aprovisionamiento se realiza para diferentes versiones de mensajes.
-   Los firmantes se pueden identificar mediante el emisor y el número de serie, como en PKCS \# 7 1,5 o mediante el identificador de clave de sujeto.
-   Se permiten varios firmadores.
-   Se permiten las firmas RSA, DSA y ECDSA.
-   Los algoritmos basados en criptografía de curva elíptica (ECC) y AES requieren Windows Vista o posterior.
