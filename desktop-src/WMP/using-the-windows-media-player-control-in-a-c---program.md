# <a name="using-the-windows-media-player-control-in-a-c-program"></a>Usar el control Reproductor de Windows Media en un programa de C++

> [!Note]  
> El uso de C++ para insertar el control Reproductor de Windows Media es compatible con Reproductor de Windows Media serie 9 o posterior.

 

Hay varias maneras diferentes de usar el control Reproductor de Windows Media en un programa de C++. Puede crear una instancia del control en una aplicación de consola o puede insertar el control en una Windows aplicación. Además, puede implementar interfaces que le permitan ejecutar un control Player insertado en modo remoto. Puede personalizar la interfaz de usuario de un control incrustado aplicando un archivo de definición de máscara.

Esta información se describe en los temas siguientes.



| Tema                                                                                                                                      | Descripción                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Usar el control Reproductor de Windows Media en una aplicación de consola](using-the-windows-media-player-control-in-a-console-application.md)     | Describe una sencilla aplicación de consola de C++ que crea una instancia del control Reproductor de Windows Media para mostrar la versión.                                                       |
| [Hospedar el control Reproductor de Windows Media en una Windows aplicación](hosting-the-windows-media-player-control-in-a-windows-application.md) | Describe cómo usar la ventana de host de ActiveX ATL para insertar el control Reproductor de Windows Media en un Windows programa.                                                            |
| [Control remoto del Reproductor de Windows Media](remoting-the-windows-media-player-control.md)                                                 | Describe cómo insertar el control Reproductor de Windows Media en un programa de C++ en modo remoto, lo que permite a los usuarios desacoplar el control para cambiar al modo completo del Reproductor. |
| [Control de eventos en C++](handling-events-in-c.md)                                                                                         | Describe cómo recibir notificaciones de eventos de Reproductor de Windows Media.                                                                                                     |
| [Usar máscaras con el control Reproductor de Windows Media control](using-skins-with-the-windows-media-player-control.md)                                 | Describe cómo aplicar un archivo de máscara a un control Reproductor de Windows Media insertado en un programa de C++.                                                                             |



 

> [!Note]  
> Puede insertar el control Reproductor de Windows Media 10 Mobile en una Windows CE aplicación. Las técnicas que se usan para hacerlo son similares a las que se usan con el control de Reproductor de Windows Media escritorio. Sin embargo, hay diferencias entre ATL para Windows y ATL para Windows CE. En esta documentación se describen las diferencias entre estas implementaciones, si procede.
