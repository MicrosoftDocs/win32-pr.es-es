# <a name="rating"></a>Rating

El atributo rating contiene la clasificación del contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMRating

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Se trata de un atributo de nivel de archivo.

Las implementaciones individuales en archivos existentes usan distintas escalas y notaciones. Por este motivo, debe evitar el uso de este atributo; no hay ninguna estandarización desde la que desarrollar interfaces de usuario coherentes.
