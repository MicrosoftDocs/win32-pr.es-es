# <a name="color-functions"></a>Funciones de color

Las siguientes funciones se usan con color.



| Función                                                   | Descripción                                                                                                                                           |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AnimatePalette**](/windows/desktop/api/Wingdi/nf-wingdi-animatepalette)                   | Reemplaza las entradas de la paleta lógica especificada.                                                                                                    |
| [**CreateHalftonePalette**](/windows/desktop/api/Wingdi/nf-wingdi-createhalftonepalette)     | Crea una paleta de semitonos para el contexto de dispositivo (DC) especificado.                                                                                     |
| [**CreatePalette**](/windows/desktop/api/Wingdi/nf-wingdi-createpalette)                     | Crea una paleta lógica.                                                                                                                            |
| [**GetColorAdjustment**](/windows/desktop/api/Wingdi/nf-wingdi-getcoloradjustment)           | Recupera los valores de ajuste de color del controlador de dominio especificado.                                                                                           |
| [**GetNearestColor**](/windows/desktop/api/Wingdi/nf-wingdi-getnearestcolor)                 | Recupera un valor de color que identifica un color de la paleta del sistema que se mostrará cuando se use el valor de color especificado.                    |
| [**GetNearestPaletteIndex**](/windows/desktop/api/Wingdi/nf-wingdi-getnearestpaletteindex)   | Recupera el índice de la entrada en la paleta lógica especificada que coincide más estrechamente con un valor de color especificado.                                     |
| [**GetPaletteEntries**](/windows/desktop/api/Wingdi/nf-wingdi-getpaletteentries)             | Recupera un intervalo especificado de entradas de la paleta de la paleta lógica especificada.                                                                        |
| [**GetSystemPaletteEntries**](/windows/desktop/api/Wingdi/nf-wingdi-getsystempaletteentries) | Recupera un intervalo de entradas de la paleta de la paleta del sistema que está asociada al controlador de dominio especificado.                                                |
| [**GetSystemPaletteUse**](/windows/desktop/api/Wingdi/nf-wingdi-getsystempaletteuse)         | Recupera el estado actual de la paleta del sistema (física) del controlador de dominio especificado.                                                                    |
| [**RealizePalette**](/windows/desktop/api/Wingdi/nf-wingdi-realizepalette)                   | Asigna entradas de paleta de la paleta lógica actual a la paleta del sistema.                                                                          |
| [**ResizePalette**](/windows/desktop/api/Wingdi/nf-wingdi-resizepalette)                     | Aumenta o disminuye el tamaño de una paleta lógica según un valor específico.                                                                    |
| [**SelectPalette**](/windows/desktop/api/Wingdi/nf-wingdi-selectpalette)                     | Selecciona la paleta lógica especificada en un contexto de dispositivo.                                                                                          |
| [**SetColorAdjustment**](/windows/desktop/api/Wingdi/nf-wingdi-setcoloradjustment)           | Establece los valores de ajuste de color de un controlador de dominio usando los valores especificados.                                                                                 |
| [**SetPaletteEntries**](/windows/desktop/api/Wingdi/nf-wingdi-setpaletteentries)             | Establece los valores de color RGB (rojo, verde, azul) y los marcadores en un intervalo de entradas de una paleta lógica.                                                        |
| [**SetSystemPaletteUse**](/windows/desktop/api/Wingdi/nf-wingdi-setsystempaletteuse)         | Permite que una aplicación especifique si la paleta del sistema contiene 2 o 20 colores estáticos.                                                           |
| [**UnrealizeObject**](/windows/desktop/api/Wingdi/nf-wingdi-unrealizeobject)                 | Restablece el origen de un pincel o restablece una paleta lógica.                                                                                             |
| [**UpdateColors**](/windows/desktop/api/Wingdi/nf-wingdi-updatecolors)                       | Actualiza el área cliente del contexto de dispositivo especificado reasignando los colores actuales del área cliente a la paleta lógica actualmente realizada. |
