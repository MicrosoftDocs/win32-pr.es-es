# <a name="accessibility-structures"></a>Estructuras de accesibilidad

En esta sección se describen las estructuras que se usan para implementar Windows de accesibilidad.

## <a name="in-this-section"></a>En esta sección



| Estructura                                         | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ACCESSTIMEOUT**](/windows/win32/api/winuser/ns-winuser-accesstimeout)<br/> | Contiene información sobre el período de tiempo de espera asociado a las características de accesibilidad de Microsoft Win32. <br/> El período de tiempo de espera de accesibilidad es el período de tiempo que se debe pasar sin entrada de teclado y mouse antes de que el sistema operativo apague automáticamente las características de accesibilidad. El tiempo de espera de accesibilidad está diseñado para equipos que comparten varios usuarios, de modo que las opciones seleccionadas por un usuario no incomode a un usuario posterior.<br/> Las características de accesibilidad afectadas por el tiempo de espera son las características FilterKeys (SlowKeys, BounceKeys y RepeatKeys), MouseKeys, ToggleKeys y StickyKeys. El tiempo de espera de accesibilidad también afecta a la configuración del modo de contraste alto.<br/> |
| [**Filterkeys**](/windows/win32/api/winuser/ns-winuser-filterkeys)<br/>       | Contiene información sobre la característica de accesibilidad FilterKeys, que permite a un usuario con discapacidades establecer la velocidad de repetición del teclado (RepeatKeys), el retraso de aceptación (SlowKeys) y la tasa de rebotones (BounceKeys). <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**HIGHCONTRAST**](/windows/win32/api/winuser/ns-winuser-highcontrasta)<br/>   | Contiene información sobre la característica de accesibilidad de contraste alto.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**Mousekeys**](/windows/win32/api/winuser/ns-winuser-mousekeys)<br/>         | Contiene información sobre la característica de accesibilidad MouseKeys. Cuando la característica MouseKeys está activa, el usuario puede usar el teclado numérico para controlar el puntero del mouse y hacer clic, hacer doble clic, arrastrar y colocar. Al presionar NUMLOCK, el usuario puede alternar el teclado numérico entre el modo de control del mouse y el funcionamiento normal. <br/>                                                                                                                                                                                                                                                                                                                                                                                               |
| [**Serialkeys**](/windows/win32/api/winuser/ns-winuser-serialkeysa)<br/>       | Contiene información sobre la característica de accesibilidad SerialKeys, que interpreta los datos de una ayuda de comunicación conectada a un puerto serie como comandos que provocan que el sistema simule la entrada del teclado y del mouse. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**Soundsentry**](/windows/win32/api/winuser/ns-winuser-soundsentrya)<br/>     | Contiene información sobre la característica de accesibilidad SoundSentry. Cuando la característica SoundSentry está en, el equipo muestra una indicación visual solo cuando se genera un sonido.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**Stickykeys**](/windows/win32/api/winuser/ns-winuser-stickykeys)<br/>       | Contiene información sobre la característica de accesibilidad StickyKeys. Cuando la característica StickyKeys está activa, el usuario puede presionar una tecla modificadora (MAYÚS, CTRL o ALT) y, a continuación, otra tecla en secuencia en lugar de al mismo tiempo, para escribir caracteres desplazados (modificados) y otras combinaciones de teclas.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**Togglekeys**](/windows/win32/api/winuser/ns-winuser-togglekeys)<br/>       | Contiene información sobre la característica de accesibilidad ToggleKeys. Cuando la característica ToggleKeys está activa, el equipo emite un tono alto cada vez que el usuario activa la tecla CAPS LOCK, NUM LOCK o SCROLL LOCK, y un tono de tono bajo cada vez que el usuario desactiva una de esas claves. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
