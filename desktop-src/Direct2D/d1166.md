# <a name="d1166-discard-map-option-requires-write-map-option"></a>D1166: la opción de asignación descartar requiere la opción de asignación de escritura

\_ \_ Las opciones \_ de mapa de D2D1 descartadas se deben combinar con la escritura de opciones de mapa de D2D1 \_ \_ \_






 

## <a name="possible-causes"></a>Causas posibles

Direct2D puede generar este mensaje si una aplicación llama a ID2D1Bitmap1:: Map con \_ las opciones de asignación D2D1 \_ \_ discard, pero no D2D1 a \_ \_ las opciones de mapa \_ .

## <a name="possible-fixes"></a>Posibles correcciones

La opción discard requiere escritura para ser una operación útil para una aplicación. Consulte D2D1 \_ map \_ Options para obtener una descripción de las opciones de asignación. Al llamar a ID2D1Bitmap1:: Map, tenga en cuenta qué tipo de acceso necesita la aplicación.
