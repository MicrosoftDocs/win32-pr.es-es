# <a name="id2d1propertiesgetvaluesize-methods"></a>Métodos ID2D1Properties::GetValueSize

Obtiene el tamaño del valor de propiedad en bytes, utilizando el índice de propiedad.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                       | Descripción                                                                                                     |
|:-------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------|
| [**GetValueSize(U)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getvaluesize(u))     | Obtiene el tamaño del valor de propiedad en bytes, utilizando el índice de propiedad. Se trata de una sobrecarga de plantilla.<br/> |
| [**GetValueSize(UINT32)**](/windows/win32/api/d2d1_1/nf-d2d1_1-id2d1properties-getvaluesize(uint32)) | Obtiene el tamaño del valor de propiedad en bytes, utilizando el índice de propiedad.<br/>                              |
