# <a name="hyper-v-migration-api-reference"></a>Referencia de la API de migración de Hyper-V

La API de migración de Hyper-V define los siguientes elementos de programación.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                                                                | Descripción                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ MigrationJob**](msvm-migrationjob.md)<br/>                                                                           | Esta clase representa un trabajo de operación de migración creado para el almacenamiento o la migración del sistema virtual por el servicio de migración del sistema virtual.<br/>                                                 |
| [**MSVM \_ VirtualSystemMigrationCapabilities**](msvm-virtualsystemmigrationcapabilities.md)<br/>                               | Define los medios por los que un cliente puede detectar los métodos proporcionados por el servicio de migración y el intervalo válido de datos de configuración de migración del sistema virtual.<br/>                                |
| [**MSVM \_ VirtualSystemMigrationNetworkSettingData**](msvm-virtualsystemmigrationnetworksettingdata.md)<br/>                   | Representa la red en la que el servicio de migración del sistema virtual está escuchando la migración de sistema virtual entrante.<br/>                                                                 |
| [**MSVM \_ VirtualSystemMigrationService**](msvm-virtualsystemmigrationservice.md)<br/>                                         | Representa el servicio de migración del sistema virtual. Se usa para migrar un sistema virtual o para migrar el almacenamiento de un sistema virtual de una plataforma de virtualización a otra.<br/> |
| [**MSVM \_ VirtualSystemMigrationServiceSettingData**](msvm-virtualsystemmigrationservicesettingdata.md)<br/>                   | Representa la configuración para el servicio de migración del sistema virtual en un host.<br/>                                                                                                              |
| [**MSVM \_ VirtualSystemMigrationServiceSettingDataComponent**](msvm-virtualsystemmigrationservicesettingdatacomponent.md)<br/> | Asociación que se usa para representar la configuración de red de la migración del sistema virtual del servicio de migración del sistema virtual.<br/>                                                                      |
| [**MSVM \_ VirtualSystemMigrationSettingData**](msvm-virtualsystemmigrationsettingdata.md)<br/>                                 | Representa la configuración de migración para migrar un sistema virtual y el almacenamiento conectado a un sistema virtual.<br/>                                                                           |
