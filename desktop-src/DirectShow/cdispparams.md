# <a name="cdispparams-class"></a>CDispParams (clase)

![Jerarquía de clases cdispparams](images/cutil11.png)

La `CDispParams` clase implementa la estructura DIS **DISRAMS** que se usa en Automation como una clase base de C++. El **método IDispatch::Invoke** usa la estructura OLE **DIS DISRAMS** para contener los argumentos pasados a cualquier método o propiedad.
