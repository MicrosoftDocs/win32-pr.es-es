# <a name="using-lsa-policy"></a>Uso de la directiva LSA

En los temas siguientes se describe cómo usar las funciones de directiva LSA.

-   [El uso de cadenas Unicode LSA](using-lsa-unicode-strings.md) explica cómo convertir cadenas en el formato de estructura utilizado por algunas funciones de directiva LSA.
-   [Al abrir un identificador de objeto de](opening-a-policy-object-handle.md) directiva se describe cómo abrir un identificador para el objeto [**Policy**](policy-object.md) local. Muchas de las funciones de directiva de LSA requieren este identificador como parámetro de entrada.
-   [Administración de la información de](managing-policy-information.md) directiva detalla cómo establecer y consultar información de directiva global para el sistema local y el dominio.
-   [Recepción de eventos de cambio de](receiving-policy-change-events.md) directiva describe cómo crear y registrar un evento para recibir notificaciones cuando cambia la información de directiva.
-   [Administración de permisos de cuenta](managing-account-permissions.md) explica cómo crear, eliminar y enumerar cuentas y cómo agregar y quitar privilegios para esas cuentas.
-   [Administración de información de dominio](managing-trusted-domain-information.md) de confianza describe cómo crear, enumerar y eliminar dominios de confianza y cómo establecer y recuperar información de dominio de confianza.
-   [Traducción entre nombres y SID explica](translating-between-names-and-sids.md) cómo asignar SID a nombres de cuenta y viceversa.
-   [Asignación de identificadores posix](mapping-posix-identifiers.md) describe cómo asignar SID a identificadores Posix de 32 bits.
-   [Almacenamiento de datos privados](storing-private-data.md) explica cómo almacenar y recuperar cadenas de datos privadas.

Para obtener información sobre cómo usar el LSA para iniciar sesión y autenticar usuarios, vea [Autenticación de LSA.](/windows/desktop/SecAuthN/lsa-authentication)
