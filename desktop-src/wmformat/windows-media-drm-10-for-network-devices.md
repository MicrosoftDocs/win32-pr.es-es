# <a name="windows-media-drm-10-for-network-devices"></a>Windows Media DRM 10 para dispositivos de red

La aplicación puede transmitir contenido protegido a los receptores de medios digitales en una red mediante Windows Media DRM 10 para dispositivos de red. Esta característica es un protocolo para comunicarse de forma segura con los dispositivos de red.

El SDK de Windows Media Format es compatible con Windows Media DRM 10 para dispositivos de red al mantener una base de datos de dispositivos de red, validar la proximidad de la red de dichos dispositivos y convertir contenido protegido con DRM en un formato de transporte seguro.

> [!Note]  
> DRM no es compatible con la versión basada en x64 de este SDK.
