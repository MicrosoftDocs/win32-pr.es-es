# <a name="binding-and-handles"></a>Enlace y identificadores

En esta sección se explica cómo crear y usar enlaces y identificadores de enlace entre programas cliente y servidor. También describe los contextos de cliente/servidor y los identificadores de contexto. La información se divide en los temas siguientes:

-   [Identificadores de enlace](binding-handles.md)
-   [Identificadores de contexto](context-handles.md)

Además de los identificadores de enlace y contexto, Rpc de Microsoft también admite identificadores de serialización que se usan para codificar o descodificar datos. Se usan para la serialización en un equipo local y no implican el enlace remoto. Para obtener más información, vea [Servicios de serialización](serialization-services.md).
