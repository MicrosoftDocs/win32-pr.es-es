# <a name="router-management-mib-reference"></a>Referencia de la MIB de administración del enrutador

En esta documentación se describen los elementos de referencia que se usan en la API de base de datos de administración (MIB) específica de administración de enrutadores. Use estos elementos de referencia para consultar y establecer los valores de las variables MIB. Los administradores de enrutadores y los protocolos de enrutamiento que los servicios del administrador del enrutador exportan las variables MIB:

-   [Funciones MIB de administración del enrutador](mib-functions.md)
-   [Constantes de protocolo y transporte](transport-and-protocol-constants.md)

Las estructuras que definen variables MIB se documentan en el tema redes de [API base información de administración](/previous-versions/windows/desktop/mib/management-information-base-reference) independiente de la tecnología.
