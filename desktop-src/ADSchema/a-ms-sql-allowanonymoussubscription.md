# <a name="ms-sql-allowanonymoussubscription-attribute"></a>Atributo MS-SQL-AllowAnonymousSubscription

True si la publicación permite a los suscriptores anónimos suscribirse a ella.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-SQL-AllowAnonymousSubscription    |
| Nombre para mostrar de LDAP | mS-SQL-AllowAnonymousSubscription    |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | Cuando se configura la replicación.           |
| Attribute-Id      | 1.2.840.113556.1.4.1394              |
| System-ID-GUID    | db77be4a-ccee-11d2-9993-0000f87a57d4 |
| Sintaxis            | [**Booleano**](s-boolean.md)         |
