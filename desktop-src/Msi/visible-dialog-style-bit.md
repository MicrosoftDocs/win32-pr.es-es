# <a name="visible-dialog-style-bit"></a>Bit de estilo de cuadro de diálogo visible

Si se establece este bit, el cuadro de diálogo se crea originalmente como visible; de lo contrario, está oculto.

## <a name="value"></a>Value



| Decimal | Hexadecimal | Constante                         |
|---------|-------------|----------------------------------|
| 1       | 0x00000001  | **msidbDialogAttributesVisible** |
