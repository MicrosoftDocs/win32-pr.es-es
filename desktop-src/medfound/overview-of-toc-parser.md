# <a name="overview-of-table-of-contents-parser"></a>Información general sobre el analizador de tabla de contenido

El analizador de tabla de contenido (analizador de TDC) es un componente de Windows que permite almacenar información de índice avanzada en archivos multimedia. Ejemplos de información de índice avanzada son tablas de contenido, índices semánticos y paneles de casos.

Con el analizador de TDC, puede escribir software que permita a los usuarios buscar escenas de interés en un archivo de vídeo. Por ejemplo, puede proporcionar al usuario una tabla de contenido para que pueda identificar las secciones principales del vídeo y, a continuación, ver una sección concreta. También puede proporcionar al usuario un índice para que pueda encontrar una lista de escenas relacionadas con una palabra o frase en particular.

El generador de tabla de contenido (generador de TDC) es un componente de Windows que funciona junto con el analizador de TDC. El generador de TDC puede detectar escenas y clústeres de escenas en un archivo de vídeo y, a continuación, generar una tabla de contenido que puede usar como punto de partida del software.

En la lista siguiente se proporcionan algunos escenarios en los que se pueden usar los servicios proporcionados por el analizador de TDC o el generador de TDC.

-   Cree una tabla de contenido desde cero e insértela en un archivo de vídeo.
-   Leer una tabla de contenido que ya está incrustada en un archivo de vídeo.
-   Edite una tabla de contenido que ya está incrustada en un archivo de vídeo.
-   Utilice el generador de TDC para crear una tabla de contenido basada en escenas y clústeres que detecta en un archivo de vídeo. Edite la tabla de contenido generada e insértela en el archivo de vídeo.
