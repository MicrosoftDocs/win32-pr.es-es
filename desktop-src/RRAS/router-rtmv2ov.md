# <a name="router-rras"></a>Enrutador (RRAS)

Para los fines de esta documentación, un enrutador es un servidor Windows NT/Windows 2000 que ejecuta el servicio de enrutamiento y acceso remoto (RRAS) y el administrador de tablas de enrutamiento.
