# <a name="fatal-application-exit"></a>Salida irresíz de la aplicación

La **función FatalAppExit** muestra un cuadro de mensaje y finaliza la aplicación cuando el usuario cierra el cuadro de mensaje. Esta función solo debe usarse como último recurso, ya que es posible que no se libera la memoria o los archivos propiedad de la aplicación.
