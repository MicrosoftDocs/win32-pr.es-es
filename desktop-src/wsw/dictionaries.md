# <a name="dictionaries"></a>Diccionarios

Los diccionarios permiten la tokenización de datos de nodo comunes como nombres de elementos o atributos, espacios de nombres y texto común en un formato XML binario.


Un [**diccionario>**](/windows/desktop/api/WebServices/ns-webservices-ws_xml_dictionary) se puede especificar como parte del tipo [**de \_ \_ cadena XML de WS**](/windows/desktop/api/WebServices/ns-webservices-ws_xml_string) . Todas las cadenas de un solo **diccionario>** deben ser únicas.

Los IDENTIFICADOres de diccionario son compatibles con todas las codificaciones (no solo con el formato XML binario).

La siguiente función se usa con diccionarios:

-   [**WsGetDictionary**](/windows/desktop/api/WebServices/nf-webservices-wsgetdictionary)
