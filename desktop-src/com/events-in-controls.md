# <a name="events-in-controls"></a>Eventos en controles

Aunque la mayoría de los controles exponen y desencadenan varios eventos, no es necesario que los controles expongan o activen eventos y, por lo tanto, el control no requiere [**IConnectionPointContainer**](/windows/desktop/api/OCIdl/nn-ocidl-iconnectionpointcontainer). Si el control tiene algún evento, no hay ningún requisito para los eventos que debe exponer un control.
