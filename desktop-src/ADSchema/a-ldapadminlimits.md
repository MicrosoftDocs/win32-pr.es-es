# <a name="ldap-admin-limits-attribute"></a>Atributo LDAP-Admin-Limits

Contiene un conjunto de pares atributo-valor que definen los límites administrativos del servidor LDAP.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | LDAP-Admin-Limits                           |
| Ldap-Display-Name | lDAPAdminLimits                             |
| Size              | \-                                          |
| Actualizar privilegios  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.843                      |
| System-Id-Guid    | 7359a352-90f7-11d1-aebc-0000f80367c1        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
