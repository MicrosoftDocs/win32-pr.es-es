# <a name="last-known-parent-attribute"></a>Último atributo primario conocido

Nombre distintivo (DN) del último elemento primario conocido de un objeto huérfano.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | Último conocido-primario                       |
| Nombre para mostrar de LDAP | lastKnownParent                         |
| Tamaño              | \-                                      |
| Actualizar privilegio  | El sistema establece este valor.        |
| Frecuencia de actualización  | Cada vez que un objeto está huérfano.         |
| Attribute-Id      | 1.2.840.113556.1.4.781                  |
| System-ID-GUID    | 52ab8670-5709-11d1-a9c6-0000f80367c1    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
