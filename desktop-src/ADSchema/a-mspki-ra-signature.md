# <a name="ms-pki-ra-signature-attribute"></a>Atributo ms-PKI-RA-Signature

Especifica el número de firmas de entidad de registro de inscripción necesarias en una solicitud de inscripción.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-RA-Signature                                                                               |
| Ldap-Display-Name | msPKI-RA-Signature                                                                                |
| Size              | 4 bytes                                                                                           |
| Actualizar privilegios  | Administrador de dominio                                                                              |
| Frecuencia de actualización  | Cuando se edita, crea o clona el objeto de plantilla de certificado (ms-PKI-Certificate-Template). |
| Attribute-Id      | 1.2.840.113556.1.4.1429                                                                           |
| System-Id-Guid    | fe17e04b-937d-4f7e-8e0e-9292c8d5683e                                                              |
| Syntax            | [**Enumeración**](s-enumeration.md)                                                              |
