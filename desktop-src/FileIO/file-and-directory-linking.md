# <a name="file-and-directory-linking"></a>Vinculación de archivos y directorios

El sistema de archivos NTFS proporciona la capacidad de crear una representación del sistema de un archivo o directorio en una ubicación de la estructura de directorios diferente del objeto de archivo o directorio al que se está vinculando. Este proceso se denomina vinculación. Hay dos tipos de vínculos admitidos en el sistema de archivos NTFS: [vínculos duros y uniones](hard-links-and-junctions.md).

El sistema de archivos NTFS también proporciona el servicio [de seguimiento](distributed-link-tracking-and-object-identifiers.md)de vínculos distribuidos , que realiza un seguimiento automático de los vínculos a medida que se mueven.
