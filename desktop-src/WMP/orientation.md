# <a name="orientation"></a>Orientación

Algunos dispositivos basados en Windows Mobile se pueden usar en distintas orientaciones físicas. Las orientaciones disponibles incluyen:

-   **Vertical**. El dispositivo se mantiene verticalmente para que sea más alto que el ancho.
-   **Horizontal**. El dispositivo se mantiene de forma horizontal para que sea más ancho que el alto.
-   **Cuadrado**. Algunos dispositivos tienen una visualización en forma cuadrada, lo que significa que hay un único modo de orientación.

Cada máscara que cree debe especificar una sola orientación. Debe proporcionar una máscara independiente para cada orientación. El selector de máscara solo mostrará las máscaras que se crearon para la orientación y las dimensiones actuales de la pantalla del dispositivo.
