# <a name="virtual-system-management-service"></a>Servicio de administración de sistemas virtuales

El perfil del sistema virtual describe los objetos que forma una máquina virtual. Estos objetos incluyen el sistema base, los dispositivos que lo comprenden, la configuración del sistema y sus dispositivos y el servicio de administración que realiza operaciones en el sistema.
