# <a name="the-account-object-type"></a>El tipo de objeto de cuenta

Los objetos de [**cuenta**](account-object.md) se usan para indicar [*privilegios*](/windows/desktop/SecGloss/p-gly)o derechos de acceso al sistema que se van a bestowedr en un usuario o miembros de grupos o grupos locales.

Los objetos de [**cuenta**](account-object.md) residen dentro del objeto de [**Directiva**](policy-object.md) . Puede haber cero o más de estos objetos.

El objeto de [**cuenta**](account-object.md) no tiene ningún tipo de datos específico del objeto.
