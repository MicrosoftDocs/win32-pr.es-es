# <a name="services-services"></a>Servicios (servicios)

Una *aplicación de servicio* se ajusta a las reglas de la interfaz del administrador de control de servicios (SCM). Se puede iniciar automáticamente al arrancar el sistema, por un usuario a través del applet del panel de control de servicios o por una aplicación que utiliza las [funciones de servicio](service-functions.md)de. Los servicios se pueden ejecutar incluso cuando ningún usuario ha iniciado sesión en el sistema.

Un *servicio de controlador* se ajusta a los protocolos del controlador de dispositivo. Es similar a una aplicación de servicio, pero no interactúa con el SCM. Para simplificar, el término *servicio* hace referencia a una *aplicación de servicio* en esta información general.

Los desencadenadores ahora se pueden usar para controlar el inicio del servicio. Para obtener más información, consulte [configuración del servicio](service-configuration.md).
