# <a name="requirements-for-remote-desktop-web-connection"></a>Requisitos para Conexión web a Escritorio remoto

Conexión web a Escritorio remoto requiere dos sistemas configurados de la siguiente manera:

-   Un servidor host con Windows e IIS 4,0 o posterior.
-   Un cliente que puede descargar y ejecutar el Escritorio remoto control ActiveX. Cualquier cliente que ejecute Microsoft Internet Explorer 4,01 (preferiblemente Service Pack 2) o posterior puede ejecutar el Escritorio remoto control ActiveX.

El usuario que instala el software debe tener privilegios suficientes para instalar el software. Por ejemplo, en Windows XP, el usuario debe ser miembro de los grupos usuarios avanzados o administradores.
