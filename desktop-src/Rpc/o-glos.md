<dl> <dt>

<span id="_rpc_odl_glos"></span><span id="_RPC_ODL_GLOS"></span>**Lenguaje de descripción de objetos (ODL)**
</dt> <dd>

Subconjunto de atributos [*MIDL*](m-glos.md) , palabras clave, instrucciones y directivas que se usan para definir bibliotecas de tipos para aplicaciones de automatización OLE.

</dd> <dt>

<span id="_rpc_open_array_glos"></span><span id="_RPC_OPEN_ARRAY_GLOS"></span>**abrir matriz**
</dt> <dd>

Matriz que es compatible y varía; es decir, el tamaño y el intervalo de elementos transmitidos se determinan en tiempo de ejecución por otros parámetros, estructuras o expresiones.

</dd> <dt>

<span id="_rpc_osf_glos"></span><span id="_RPC_OSF_GLOS"></span>**Open Software Foundation (OSF)**
</dt> <dd>

Consorcio de compañías, formado para definir el entorno de computación distribuida (DCE).

</dd> <dt>

<span id="_rpc_orphaned_memory_glos"></span><span id="_RPC_ORPHANED_MEMORY_GLOS"></span>**memoria huérfana**
</dt> <dd>

Memoria de un cliente a la que se hace referencia anteriormente mediante un parámetro de puntero que el servidor ha restablecido a NULL.

</dd> <dt>

<span id="_rpc_output_parameter_glos"></span><span id="_RPC_OUTPUT_PARAMETER_GLOS"></span>**parámetro de salida**
</dt> <dd>

Parámetro que contiene los datos que la aplicación de servidor está transmitiendo a la aplicación cliente. El \[ atributo [out](/windows/desktop/Midl/out-idl) \] designa un parámetro de salida.

</dd> </dl>
