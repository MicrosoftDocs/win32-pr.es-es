# <a name="xmuint3-extensions"></a>Extensiones XMUINT3

[**XMUINT3**](/windows/win32/api/directxmath/ns-directxmath-xmuint3) describe una estructura que contiene un vector 3D que consta de tres valores enteros sin signo.

> [!Note]  
> Las características enumeradas aquí solo están disponibles al desarrollar con C++.
