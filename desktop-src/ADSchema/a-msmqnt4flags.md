# <a name="msmq-nt4-flags-attribute"></a>Atributo MSMQ-Nt4-Flags

El atributo **MSMQ-Nt4-Flags** contiene información de modo mixto de MSMQ.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MSMQ-Nt4-marcas                       |
| Nombre para mostrar de LDAP | mSMQNt4Flags                         |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.964               |
| System-ID-GUID    | eb38a158-d57f-11d1-90a2-00c04fd91ab1 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
