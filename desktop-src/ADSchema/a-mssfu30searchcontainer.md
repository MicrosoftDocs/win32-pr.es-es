Contiene el identificador de un objeto que especifica dónde se iniciará cada búsqueda.

|-------------------|---------------------------------------------|
| CN                | msSFU-30-Search-Container                   |
| Nombre para mostrar de LDAP | msSFU30SearchContainer                      |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
