# <a name="actionid-property"></a>Action.Id propiedad

Para el scripting, obtiene o establece el identificador de la acción.

## <a name="syntax"></a>Syntax


```VB
Action.Id As String
```



## <a name="property-value"></a>Valor de propiedad

Identificador definido por el usuario para la acción. Este identificador lo usa el Programador de tareas con fines de registro.

## <a name="remarks"></a>Comentarios

Para obtener información sobre cómo funcionan conjuntamente las acciones y las tareas, vea [Acciones de tarea](task-actions.md).
