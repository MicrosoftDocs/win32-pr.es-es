# <a name="d3d12_gpu_virtual_address"></a>\_ \_ Dirección virtual de GPU de D3D12 \_

Una D3D12_GPU_VIRTUAL_ADDRESS representa una dirección virtual en la memoria de GPU. El tipo de datos se define en d3d12. h como se indica a continuación:


```
typedef UINT64 D3D12_GPU_VIRTUAL_ADDRESS;
```
