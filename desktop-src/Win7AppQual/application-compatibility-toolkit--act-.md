# <a name="application-compatibility-toolkit-act"></a>Kit de herramientas de compatibilidad de aplicaciones (ACT)

> [!IMPORTANT]
> Ya no se admiten las versiones del kit de herramientas de compatibilidad de aplicaciones que se describen en este artículo. La última versión admitida se incluye en el [Kit de evaluación e implementación de Windows 10](/archive/blogs/yongrhee/download-windows-10-assessment-and-deployment-kit-adk) .

## <a name="affected-platforms"></a>Plataformas afectadas

**Clientes:** Windows XP Windows \| vista Windows \| 7  


**Servidores:** Windows Server 2003 \| Windows server 2008 \| windows Server 2008 R2  


## <a name="description"></a>Descripción

El kit de herramientas de compatibilidad de aplicaciones (ACT) de Microsoft es una herramienta de administración del ciclo de vida que ayuda a identificar y administrar toda la cartera de aplicaciones, lo que reduce el costo y el tiempo necesario para resolver problemas de compatibilidad de aplicaciones, y ayuda a implementar rápidamente las actualizaciones de Windows y Windows.

**Uso**

Con la ley, puede:

-   Analice su cartera de aplicaciones, sitios web y equipos
-   Evaluar implementaciones de sistema operativo, el impacto de las actualizaciones del sistema operativo y su compatibilidad con sitios web
-   Administrar de forma centralizada los evaluadores de compatibilidad y las opciones de configuración
-   Racionalizar y organizar aplicaciones, sitios web y equipos
-   Priorizar los esfuerzos de compatibilidad de aplicaciones con los informes filtrados
-   Agregue y administre problemas y soluciones para su entorno informático empresarial
-   Implementar mitigaciones automatizadas en problemas de compatibilidad conocidos
-   Enviar y recibir información de compatibilidad de Microsoft Compatibility Exchange
