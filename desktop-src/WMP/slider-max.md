# <a name="slidermax"></a>Control deslizante. Max

El atributo **Max** especifica o recupera el valor máximo del intervalo definido por el control deslizante.

``` syntax
        elementID.max
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **número** de lectura/escritura (**float**) con un valor predeterminado de 100.

## <a name="remarks"></a>Observaciones

El valor especificado para **Max** debe ser mayor que el de **min**.

Vea **CUSTOMSLIDER**. atributo [positionImage](customslider-positionimage.md) para un ejemplo que muestra cómo se utilizan los atributos del elemento **Slider** .
