# <a name="administrative-functions"></a>Funciones administrativas

Las siguientes funciones administrativas permiten que un proveedor de red realice acciones específicas de la red para mostrar y manipular directorios específicos de la red, es decir, directorios asignados a protocolos que no son de Windows.



| Función                                         | Descripción                                                                                                                                                   |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**NPGetDirectoryType**](/windows/desktop/api/Npapi/nf-npapi-npgetdirectorytype) | Lo llama el administrador de archivos para determinar el tipo de un directorio de red.                                                                                          |
| [**NPDirectoryNotify**](/windows/desktop/api/Npapi/nf-npapi-npdirectorynotify)   | Lo llama el administrador de archivos para notificar a la red el proveedor de operaciones de directorio. Esta función se puede utilizar para realizar un comportamiento especial en determinados directorios. |
