# <a name="swbemeventsourcesecurity_-property"></a>Propiedad SWbemEventSource. Security \_

La **propiedad \_ Security** del objeto [**SWbemEventSource**](swbemeventsource.md) obtiene o establece la configuración de seguridad de un objeto **SWbemEventSource** . Esta propiedad es un objeto [**SWbemSecurity**](swbemsecurity.md) .

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemEventSource.Security_ As Object
```
