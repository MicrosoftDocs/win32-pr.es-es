# <a name="sol_socket-socket-options"></a>\_Opciones de socket de socket sol

En las tablas siguientes se describen las opciones de socket de **\_ socket sol** . Consulte las páginas de referencia de [**la función**](/windows/desktop/api/winsock/nf-winsock-setsockopt) [**getsockopt**](/windows/desktop/api/winsock/nf-winsock-getsockopt) y el valor de My para obtener más información sobre cómo obtener y establecer las opciones de socket.

Para enumerar los protocolos y detectar las propiedades admitidas de cada protocolo instalado, use la función [**WSAEnumProtocols**](/windows/desktop/api/Winsock2/nf-winsock2-wsaenumprotocolsa), [**WSCEnumProtocols**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumprotocols)o [**WSCEnumProtocols32**](/windows/desktop/api/Ws2spi/nf-ws2spi-wscenumprotocols32) .

Algunas opciones de socket requieren más explicación de lo que pueden transmitir estas tablas; Estas opciones contienen vínculos a páginas adicionales.

> [!Note]  
> Todas las opciones de sockets de **\_ Sockets sol** se aplican igualmente a IPv4 e IPv6 (excepto para la \_ difusión, ya que la difusión no se implementa en IPv6).

## <a name="remarks"></a>Observaciones

Las opciones de socket de **\_ socket sol** se definen en varios archivos de encabezado de Winsock:

-   *WinSock2. h*
-   *Mswsock. h*
-   *Ws2def. h*

En el kit de desarrollo de software (SDK) de Microsoft Windows publicado para Windows Vista y versiones posteriores, la organización de los archivos de encabezado ha cambiado y el nivel de **\_ sockets de sol** se define en el archivo de encabezado *Ws2def. h* que se incluye automáticamente en el archivo de encabezado *WinSock2. h* . Algunas de las opciones de socket de **\_ Sockets sol** se definen en los archivos de encabezado *WinSock2. h* y *mswsock. h* . Las opciones restantes de socket de **\_ socket de sol** se definen en el archivo de encabezado *Ws2def. h* que se incluye automáticamente en el archivo de encabezado *WinSock2. h* . *Ws2def. h* nunca se debe usar directamente.

En el kit de desarrollo de software (SDK) de la plataforma publicado para Windows Server 2003 y Windows XP, el nivel de **\_ sockets de sol** se define en el archivo de encabezado *WinSock2. h* . Las opciones de socket de **\_ socket sol** se definen en los archivos de encabezado *WinSock2. h* y *mswsock. h* .
