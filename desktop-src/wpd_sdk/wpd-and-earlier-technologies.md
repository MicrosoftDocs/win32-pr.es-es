# <a name="wpd-and-earlier-technologies"></a>WPD y tecnologías anteriores

WPD contiene muchas mejoras para las cámaras digitales y los reproductores multimedia digitales portátiles, y es el método preferido para tener acceso a estos dispositivos en nuevas versiones de Windows. Los niveles de compatibilidad proporcionarán compatibilidad con las API de adquisición de imágenes de Windows (WIA) y Windows Media Administrador de dispositivos (WMDM) existentes en Windows. Estos niveles de compatibilidad garantizan que los dispositivos WPD se expongan de forma transparente a las aplicaciones WIA y WMDM existentes como dispositivos WIA o proveedores de servicios WMDM, respectivamente. Sin embargo, es posible que estas aplicaciones no tengan el mismo intervalo de capacidades que con la nueva API de WPD y los nuevos dispositivos basados en MTP.

El controlador MTP de WPD también admite dispositivos de protocolo de transferencia de imágenes (PTP). Los dispositivos PTP compatibles con WIA seguirán funcionando en las aplicaciones WIA a través del nivel de compatibilidad. Microsoft mantiene su responsabilidad para garantizar la compatibilidad de las cámaras digitales y de la aplicación en la próxima versión de Windows y estas nuevas tecnologías están diseñadas para complementar y mejorar la experiencia de cámara digital en Windows.
