# <a name="about-skins"></a>Acerca de las máscaras

Con Windows Media Player, los usuarios pueden elegir entre una variedad de máscaras estándar, cada una de las cuales proporciona una experiencia visual adicional que mejora la escucha y la visualización de los placer. Windows Media Player incluye varias máscaras entre las que elegir, pero es relativamente fácil crear y distribuir máscaras personalizadas.

Las máscaras son simplemente colecciones de uno o varios archivos de imagen de equipo, organizadas por un archivo de texto, que indica a Windows Media Player cómo usar estos archivos para mostrar una máscara. Puede usar diversos programas para modificar el arte existente o crear un arte sencillo. Por ejemplo, con las herramientas de dibujo y la imagen prediseñada suministradas con Microsoft PowerPoint, puede crear material gráfico arrastrando y colocando. Del mismo modo, no es necesario que sea programador crear el archivo de texto que se usa para definir una máscara individual.

Uno de los motivos por los que las máscaras son relativamente fáciles de crear es que puede examinar los elementos de arte individuales y el código fuente de cualquier aspecto y ver cómo se colocan cada máscara. Se proporcionan varios aspectos de ejemplo sencillos con este SDK, así como una referencia para los elementos y atributos de la máscara.
