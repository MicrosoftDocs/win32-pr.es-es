| Enumeración                                                       | Descripción                                                                                                                                                                            |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_Marca de canal de D3DX10 \_**](d3dx10-channel-flag.md)              | Estas marcas las utilizan las funciones que operan en uno o varios canales de una textura.                                                                                                  |
| [**Optimización de la CPU de D3DX \_ \_**](d3dx-cpu-optimization.md)          | Especifica que el conjunto de instrucciones D3DX está optimizado actualmente para.                                                                                                                         |

| [**\_Formato de \_ archivo de imagen D3DX10 \_**](d3dx10-image-file-format.md)   | Describe los formatos de archivo de imagen compatibles.                                                                                                                                            |
