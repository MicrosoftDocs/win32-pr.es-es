# <a name="win32_methodparameterclass-class"></a>Clase MethodParameterClass de Win32 \_

La clase [WMI](/windows/desktop/WmiSdk/retrieving-a-class) base **\_ abstracta MethodParameterClass de Win32** deriva cualquier clase definida como un contenedor de valores de parámetro que se pasarán a un método. Sin propiedades propias, esta clase actúa como nodo de clasificación. Un ejemplo similar es [**CIM \_ PhysicalElement.**](cim-physicalelement.md) La derivación **de \_ CIM PhysicalElement** indica que la clase describe una entidad física en lugar de lógica. En el caso de **\_ MethodParameterClass de Win32,** las clases derivadas de se crean específicamente para pasar parámetros a un método.

## <a name="syntax"></a>Sintaxis

## <a name="members"></a>Miembros

La **clase \_ MethodParameterClass de Win32** no define ningún miembro.
