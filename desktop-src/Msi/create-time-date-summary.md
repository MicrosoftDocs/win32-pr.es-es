# <a name="create-timedate-summary-property"></a>Crear la propiedad Resumen de fecha y hora

La **propiedad Resumen de fecha y** hora de creación transmite la hora y la fecha en que un autor creó el paquete de instalación, la transformación o el paquete de revisión.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versión<br/> | Windows Instalador 5.0 en Windows Server 2012, Windows 8, Windows Server 2008 R2 o Windows 7. Windows Instalador 4.0 o Windows Instalador 4.5 en Windows Server 2008 o Windows Vista. Windows Instalador en Windows Server 2003 o Windows XP<br/> |
