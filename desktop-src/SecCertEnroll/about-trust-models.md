# <a name="trust-models"></a>Modelos de confianza

A medida que aumenta el número de organizaciones que implementan la infraestructura de clave pública (PKI) y aumenta el número de usuarios finales y equipos de estas organizaciones, también aumentan los problemas asociados con la certificación de identidades y el establecimiento de confianza. Dos modelos de confianza principales, jerarquías de certificados y certificación cruzada, han evolucionado para ayudar a solucionar estos problemas. Inherente a ambos modelos es el concepto de una cadena de certificados. Para obtener más información, vea los temas siguientes:

-   [Jerarquía de certificados](about-certificate-hierarchy.md)
-   [Certificación cruzada](about-cross-certification.md)
-   [Cadena de certificados](about-certificate-chain.md)
