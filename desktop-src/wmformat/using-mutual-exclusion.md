# <a name="using-mutual-exclusion"></a>Uso de la exclusión mutua

Puede usar objetos de exclusión mutua para especificar grupos de secuencias, solo uno de los cuales se entregará a la vez. La exclusión mutua se puede usar para controlar diversos escenarios, tanto nativos para el SDK como personalizados.

En las secciones siguientes se describe cómo trabajar con diferentes tipos de exclusión mutua.



| Sección                                                                                  | Descripción                                                                                                       |
|------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------|
| [Uso de la exclusión mutua de velocidad de bits múltiple](using-multiple-bit-rate-mutual-exclusion.md) | Describe cómo usar la exclusión mutua para codificar una entrada en varias secuencias con diferentes configuraciones de velocidad de bits. |
| [Usar tipos de exclusión mutua personalizados](using-custom-mutual-exclusion-types.md)           | Describe la exclusión mutua personalizada.                                                                                |
