# <a name="pnrp-constants"></a>Constantes PNRP

La API del proveedor de espacios de nombres PNRP usa las siguientes constantes.



| Constante                                                 | Descripción                                                |
|----------------------------------------------------------|------------------------------------------------------------|
| [**MARCAS DE NUBE \_ PNRP \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_flags)           | Especifica la validez de un nombre de nube.                    |
| [**PNRP \_ CLOUD \_ STATE**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_cloud_state)           | Especifica los distintos estados en los que puede estar una nube PNRP.     |
| [**CRITERIOS DE RESOLUCIÓN DE PNRP \_ \_**](/windows/desktop/api/Pnrpdef/ne-pnrpdef-pnrp_resolve_criteria) | Especifica los criterios que pnrp usa para resolver las búsquedas. |
