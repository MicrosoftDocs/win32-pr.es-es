# <a name="taborderinquestion"></a>TabOrderInQuestion

## <a name="text"></a>Texto

Es posible que el orden de tabulación no siga las directrices estándar de Windows

## <a name="type"></a>Tipo

Informativo

## <a name="description"></a>Descripción

El uso de la navegación con el teclado estándar (TAB o Mayús + Tab) expone un orden de tabulación no estándar. Por ejemplo, según la configuración del sistema local, el orden de tabulación normalmente imita el orden de lectura.

Este problema causa problemas para las personas que usan un lector de pantalla y un teclado para la navegación porque los elementos de recorrido pueden ser errático e imprevisibles.
