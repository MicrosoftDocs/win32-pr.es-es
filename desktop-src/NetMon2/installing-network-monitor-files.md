# <a name="installing-network-monitor-files"></a>Instalación de archivos de Monitor de red

Para instalar y ejecutar la Monitor de red parte del kit de desarrollo de software (SDK) de la plataforma, siga este procedimiento.

**Para instalar y ejecutar Monitor de red archivos del SDK de la plataforma**

1.  Para instalar los archivos, durante la instalación del SDK de la plataforma, seleccione el Monitor de red documentos y el código de ejemplo.

    Una vez finalizada la instalación, el programa de instalación del SDK de la plataforma crea un archivo por lotes, Setenv.bat.

2.  Para inicializar las variables de entorno necesarias, ejecute Setenv.bat.
3.  Para compilar los ejemplos, ejecute **NMake**, que se incluye en Microsoft Visual Studio.
