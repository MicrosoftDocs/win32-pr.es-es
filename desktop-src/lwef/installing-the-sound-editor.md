# <a name="installing-the-sound-editor"></a>Instalar el editor de sonido

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

La configuración del sistema recomendada para usar el editor de sonidos es un equipo con un Pentium 166, al menos 48 MB de RAM y una tarjeta de sonido compatible con Windows. Si desea grabar la entrada oral con la herramienta, también necesitará un micrófono compatible.

Para instalar la herramienta de edición de sonido lingüístico de Microsoft, abra el archivo de instalación de extracción automática. Se instalarán automáticamente los archivos correspondientes en el sistema. Si descarga el editor de sonido desde el sitio web del agente de Microsoft, puede elegir instalar el editor después de descargarlo o guardarlo en el disco para abrirlo e instalarlo posteriormente. La herramienta de instalación de propondrá su instalación en el subdirectorio herramientas del agente de Microsoft. Se recomienda usar esta ubicación.

El motor de reconocimiento de voz de control y el comando de Microsoft (versión. 4,0) también debe instalarse antes de poder usar el editor de sonido. Normalmente se instala con el editor de sonido, pero si posteriormente se desinstaló, puede volver a instalarlo desde el SDK de la plataforma de Microsoft o desde el sitio web del agente de Microsoft en <https://www.microsoft.com/downloads/details.aspx?FamilyID=914fef6b-d764-4eca-88dc-d49d8912625c> . El editor de sonido solo puede generar información lingüística basada en el idioma compatible con el motor de voz. Para generar información para otros idiomas, debe instalarse un motor de reconocimiento de voz compatible para ese idioma. Póngase en contacto con el proveedor del motor de voz para determinar si admiten la herramienta de edición de sonido lingüístico de Microsoft.
