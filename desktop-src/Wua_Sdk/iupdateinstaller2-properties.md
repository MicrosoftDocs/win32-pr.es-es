# <a name="iupdateinstaller2-properties"></a>Propiedades de IUpdateInstaller2

La [**interfaz IUpdateInstaller2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller2) define la siguiente propiedad.



| Propiedad                                           | Descripción                                                                                                                       |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**ForceQuiet**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller2-get_forcequiet) | Obtiene y establece un valor booleano que indica si Windows Installer se ve obligado a instalar las actualizaciones sin interacción del usuario. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz.

-   [**IUpdateInstaller**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller)
