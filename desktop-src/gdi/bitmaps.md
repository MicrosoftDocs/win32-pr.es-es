# <a name="bitmaps-windows-gdi"></a>Mapas de bits (Windows GDI)

Un *mapa de* bits es un objeto gráfico que se usa para crear, manipular (escalar, desplazar, girar y pintar) y almacenar imágenes como archivos en un disco. En esta información general se describen las clases de mapa de bits y las operaciones de mapa de bits.
