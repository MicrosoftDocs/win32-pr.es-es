# <a name="cryptography-constants"></a>Constantes de criptografía

Las funciones de criptografía usan las siguientes constantes y se definen en Wincrypt.h. Las constantes de criptografía se clasifican según el uso que se muestra a continuación.



| Constante                                                                                                           | Descripción                                                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Constantes para CryptEncodeObject y CryptDecodeObject](constants-for-cryptencodeobject-and-cryptdecodeobject.md) | Constantes predefinidas que se usan con [*las operaciones*](../secgloss/a-gly.md) de codificación y descodificación de sintaxis abstracta One (ASN.1). |
| [Constantes para extensiones de Netscape](constants-for-netscape-extensions.md)                                         | Constantes predefinidas que se usan para las extensiones de certificado de Netscape. En esta sección se proporcionan detalles sobre el uso de estas extensiones con operaciones de codificación y descodificación de ASN.1.                                      |
| [Nombres de proveedor de servicios criptográficos](cryptographic-provider-names.md)                                                   | Constantes de cadena predefinidas para los nombres de proveedores de [*servicios criptográficos*](../secgloss/c-gly.md) comunes (CSP).   |
| [**MAXUIDLEN**](maxuidlen.md)                                                                                     | Constante numérica que especifica el número máximo de caracteres que algunos de los proveedores criptográficos de Microsoft usarán al obtener el identificador de usuario.                                                  |
| [**Constantes de almacenamiento en caché de clave privada**](private-key-caching-constants.md)                                             | Constantes que representan entradas del Registro que controlan el [*almacenamiento en*](../secgloss/p-gly.md) caché de claves privadas mediante un CSP.                                                |
| [Marcas de protocolo](protocol-flags.md)                                                                               | Constantes predefinidas para protocolos criptografía.                                                                                                                                                            |
