# <a name="multimedia-pc-specifications"></a>Especificaciones del equipo multimedia

El Consejo de marketing del equipo multimedia (MPC) ha desarrollado dos especificaciones para fomentar la adopción de capacidades multimedia. La especificación de nivel 1, desarrollada en 1990, proporciona una definición de línea de base de la informática multimedia en funcionalidad, componentes de hardware y componentes de software. La especificación de nivel 2, emitida en 1993, se basa en la primera especificación y se centra en las capacidades multimedia mejoradas desde que se emitió la primera especificación.

En este apéndice se resumen las especificaciones.

-   [Especificación de nivel 1](level-1-specification.md)
-   [Especificación de nivel 2](level-2-specification.md)
