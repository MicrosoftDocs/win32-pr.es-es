# <a name="binding-to-an-adsi-object"></a>Enlazar a un objeto ADSI

La conexión a un objeto en un servicio de directorio se conoce como enlace. El enlace a un objeto ADSI es el primer paso para comunicarse con el sistema de directorio subyacente. Un objeto debe estar enlazado a para navegar por su espacio de nombres, buscar datos, modificar datos o suplantar a un usuario. También es posible proporcionar características de enlace adicionales, como el nombre de usuario, la contraseña, el nombre del servidor, los métodos de cifrado y la autenticación segura. Las características de enlace adicionales reales que se pueden usar dependen del proveedor.
