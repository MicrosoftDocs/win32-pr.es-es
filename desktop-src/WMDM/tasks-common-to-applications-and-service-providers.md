# <a name="tasks-common-to-applications-and-service-providers"></a>Tareas comunes a las aplicaciones y proveedores de servicios

En esta sección se describen las siguientes tareas que se aplican tanto a aplicaciones como a proveedores de servicios basados en Windows Media Administrador de dispositivos.

-   [La compilación de los archivos IDL proporcionados con el SDK](compiling-the-idl-files-supplied-with-the-sdk.md) describe los archivos IDL que se usan para crear los archivos de encabezado, si decide no usar los archivos de encabezado proporcionados con el SDK.
-   La [habilitación del registro](enabling-logging.md) describe un objeto de registro proporcionado por Windows Media administrador de dispositivos para registrar datos de un objeto en ejecución.
-   El [uso de canales autenticados seguros](using-secure-authenticated-channels.md) describe los canales autenticados seguros, que se utilizan para el cifrado, el descifrado y la autenticación de componentes y mensajes. Cualquier elemento compilado en Windows Media Administrador de dispositivos debe ser capaz de realizar la autenticación de componentes al menos.
-   [Controlar el contenido protegido](handling-protected-content.md) describe el modo en que la aplicación puede controlar el contenido protegido por la administración de derechos digitales (DRM) de Windows Media.
-   [Obtener y establecer metadatos y atributos](getting-and-setting-metadata-and-attributes.md) describe cómo una aplicación puede recuperar y establecer atributos y metadatos para un dispositivo o almacenamiento.
