# <a name="xmmatrix-operators"></a>Operadores XMMATRIX

La [**estructura XMMATRIX**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) expone los operadores siguientes.

> [!Note]  
> Los operadores enumerados aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                    | Descripción                                                                                                                                                                                   |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Operador \***](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult)<br/>    | Realiza una multiplicación de matriz de la instancia actual de `XMMATRIX` por otra instancia de `XMMATRIX` .<br/>                                                                          |
| [**Operador \*=**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult-assign)<br/> | Realiza una multiplicación de matriz de la instancia actual de por otra instancia de y devuelve una referencia `XMMATRIX` `XMMATRIX` a la instancia actual, que se ha actualizado. <br/> |
| [**operator =**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-assign)<br/>      | Asigna los datos de matriz de una instancia de `XMMATRIX` a la instancia actual de y devuelve a la instancia `XMMATRIX` `reference` actual. <br/>                                    |
| [**operador ()**](xmmatrix-operator-parens.md)<br/> | Tiene acceso a elementos de matriz específicos a los que hace referencia fila y columna desde la instancia actual de `XMMATRIX` . <br/>                                                                           |
