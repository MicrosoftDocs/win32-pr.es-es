# <a name="managing-users-on-member-servers-and-windows-2000-professional"></a>Administración de usuarios en servidores miembros Windows 2000 Professional

En los servidores miembros y Windows 2000 Professional, hay una base de datos de seguridad local. Esa base de datos de seguridad local puede contener sus propios usuarios locales cuyo ámbito es solo el equipo en particular donde se crean. Al administrar estos tipos de usuarios en servidores miembro y estaciones de trabajo, se usa el proveedor WinNT.

Al administrar usuarios en un dominio Windows 2000 mediante ADSI, se usa el proveedor LDAP. Al administrar usuarios en servidores miembro y estaciones de trabajo, se usa el proveedor WinNT.
