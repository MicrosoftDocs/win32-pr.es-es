# <a name="d2d1_matrix_4x3_f"></a>D2D1 \_ Matrix \_ 4X3 \_ F

Representa una matriz de 4 por 3.


```C++
typedef D2D_MATRIX_4X3_F D2D1_MATRIX_4X3_F;
```



## <a name="remarks"></a>Observaciones

**D2D1 \_ MATRIX \_ 4X3** es un nombre nuevo para la estructura de la [**matriz de D2D \_ \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f) . Para obtener una lista de los campos que proporciona la matriz, vea [**D2D \_ Matrix \_ 4X3 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x3_f).
