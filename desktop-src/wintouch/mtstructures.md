# <a name="structures-windows-touch-gestures"></a>Estructuras (gestos táctiles de Windows)

Esta sección contiene estructuras para los gestos táctiles de Windows.

En esta sección se especifican las siguientes estructuras.



| Estructura                                          | Descripción                                                                              |
|----------------------------------------------------|------------------------------------------------------------------------------------------|
| [**GESTURECONFIG**](/windows/win32/api/winuser/ns-winuser-gestureconfig)             | Se usa para establecer la configuración de los mensajes de gestos habilitados o deshabilitados.      |
| [**GESTUREINFO**](/windows/win32/api/winuser/ns-winuser-gestureinfo)                 | Contiene información sobre un gesto.                                                    |
| [**GESTURENOTIFYSTRUCT**](/windows/win32/api/winuser/ns-winuser-gesturenotifystruct) | Contiene información sobre una notificación de [**\_ GESTURENOTIFY de WM**](wm-gesturenotify.md) . |
