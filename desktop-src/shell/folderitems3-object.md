# <a name="folderitems3-object"></a>Objeto FolderItems3

Extiende el objeto [**FolderItems2**](folderitems2-object.md) . Este objeto admite un método y una propiedad adicionales.

## <a name="members"></a>Miembros

El objeto **FolderItems3** tiene estos tipos de miembros:

-   [Métodos](#methods)
-   [Propiedades](#properties)

### <a name="methods"></a>Métodos

El objeto **FolderItems3** tiene estos métodos.



| Método                                | Descripción                                                       |
|:--------------------------------------|:------------------------------------------------------------------|
| [**Filter**](folderitems3-filter.md) | Establece un filtro comodín que se va a aplicar a los elementos devueltos.<br/> |



 

### <a name="properties"></a>Propiedades

El objeto **FolderItems3** tiene estas propiedades.



| Propiedad                                       | Tipo de acceso          | Descripción                                                       |
|:-----------------------------------------------|:---------------------|:------------------------------------------------------------------|
| [**Verbos**](folderitems3-verbs.md)<br/> | Solo lectura<br/> | Obtiene la lista de verbos comunes a todos los elementos de carpeta.<br/> |
