# <a name="item-properties"></a>Propiedades de los elementos

Las propiedades de los elementos son propiedades que controlan la función de un elemento de adquisición de imágenes de Windows (WIA). Se tiene acceso a estas propiedades a través de cualquier elemento de un árbol que representa un dispositivo. Hay tres tipos de propiedades de elemento: Common, Scanner y Camera.

Para obtener información de referencia sobre estas propiedades, vea:

-   [**Constantes comunes de propiedades de elementos WIA**](-wia-wiaitempropcommonitem.md)
-   [**Constantes de propiedad del elemento de WIA de cámara**](-wia-wiaitempropcameraitem.md)
-   [**Constantes de propiedad del elemento WIA del escáner**](-wia-wiaitempropscanneritem.md)
