# <a name="dialog-box-styles"></a>Estilos de cuadro de diálogo

En la tabla siguiente se enumeran los estilos de cuadro de diálogo que se pueden especificar al crear un cuadro de diálogo. Puede usar estos estilos en las llamadas a las funciones [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) y [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) , en el miembro **Style** de las estructuras [**DLGTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) y [**DLGTEMPLATEEX**](dlgtemplateex.md) , y en la instrucción de una definición de cuadro de diálogo en un archivo de recursos.



| Constante                                                                                                                                                               | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="DS_3DLOOK"></span><span id="ds_3dlook"></span><dl> <dt>**\_3DLOOK DS**</dt> </dl>                      | Obsoleto. El sistema aplica de forma automática los cuadros de diálogo de apariencia tridimensional creados por las aplicaciones.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_ABSALIGN"></span><span id="ds_absalign"></span><dl> <dt>**\_ABSALIGN DS**</dt> </dl>                | Indica que las coordenadas del cuadro de diálogo son coordenadas de la pantalla. Si no se especifica este estilo, las coordenadas son coordenadas de cliente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_CENTER"></span><span id="ds_center"></span><dl> <dt>**Centro de DS \_**</dt> </dl>                      | Centra el cuadro de diálogo en el área de trabajo del monitor que contiene la ventana propietaria. Si no se especifica ninguna ventana propietaria, el cuadro de diálogo se centra en el área de trabajo de un monitor determinado por el sistema. El área de trabajo es el área no oscurecida por la barra de tareas ni por ningún appbars.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_CENTERMOUSE"></span><span id="ds_centermouse"></span><dl> <dt>**\_CENTERMOUSE DS**</dt> </dl>       | Centra el cuadro de diálogo en el cursor del mouse.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_CONTEXTHELP"></span><span id="ds_contexthelp"></span><dl> <dt>**\_CONTEXTHELP DS**</dt> </dl>       | Incluye un signo de interrogación en la barra de título del cuadro de diálogo. Cuando el usuario hace clic en el signo de interrogación, el cursor se transforma en un signo de interrogación con un puntero. Si el usuario hace clic en un control en el cuadro de diálogo, el control recibe un mensaje de [**\_ ayuda de WM**](../shell/wm-help.md) . El control debe pasar el mensaje al procedimiento del cuadro de diálogo, que debe llamar a la función mediante el comando Help **\_ WM \_ Help** . La aplicación de ayuda muestra una ventana emergente que normalmente contiene ayuda para el control.<br/> Tenga en cuenta que **DS \_ CONTEXTHELP** es solo un marcador de posición. Cuando se crea el cuadro de diálogo, el sistema comprueba **\_ CONTEXTHELP de DS** y, si está ahí, agrega **WS \_ ex \_ CONTEXTHELP** al estilo extendido del cuadro de diálogo. **WS \_ EX \_ CONTEXTHELP** no se puede usar con los estilos **WS \_ MAXIMIZEBOX** o **WS \_ MINIMIZEBOX** .<br/>                                                                              |
| <span id="DS_CONTROL"></span><span id="ds_control"></span><dl> <dt>**CONTROL de DS \_**</dt> </dl>                   | Crea un cuadro de diálogo que funciona bien como una ventana secundaria de otro cuadro de diálogo, como una página en una hoja de propiedades. Este estilo permite al usuario desplazarse entre las ventanas de control de un cuadro de diálogo secundario, usar sus teclas de aceleración, etc.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="DS_FIXEDSYS"></span><span id="ds_fixedsys"></span><dl> <dt>**\_FIXEDSYS DS**</dt> </dl>                | Hace que el cuadro de diálogo Utilice **la \_ \_ fuente fija del sistema** en lugar de **la \_ fuente predeterminada del sistema**. Se trata de una fuente monoespaciada compatible con la fuente del sistema en versiones de 16 bits de Windows anteriores a 3,0.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="DS_LOCALEDIT"></span><span id="ds_localedit"></span><dl> <dt>**\_LOCALEDIT DS**</dt> </dl>             | Solo se aplica a las aplicaciones de 16 bits. Este estilo dirige los controles de edición del cuadro de diálogo para asignar memoria del segmento de datos de la aplicación. De lo contrario, los controles de edición asignan el almacenamiento de un objeto de memoria global.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_MODALFRAME"></span><span id="ds_modalframe"></span><dl> <dt>**\_MODALFRAME DS**</dt> </dl>          | Crea un cuadro de diálogo con un marco de cuadro de diálogo modal que se puede combinar con una barra de título y un menú de ventana especificando los estilos **WS \_ Caption** y **WS \_ SYSMENU** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="DS_NOFAILCREATE"></span><span id="ds_nofailcreate"></span><dl> <dt>**\_NOFAILCREATE DS**</dt> </dl>    | Crea el cuadro de diálogo incluso si se producen errores, por ejemplo, si no se puede crear una ventana secundaria o si el sistema no puede crear un segmento de datos Especial para un control de edición.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="DS_NOIDLEMSG"></span><span id="ds_noidlemsg"></span><dl> <dt>**\_NOIDLEMSG DS**</dt> </dl>             | Suprime los mensajes de [**WM \_ ENTERIDLE**](wm-enteridle.md) que el sistema enviará al propietario del cuadro de diálogo mientras se muestra el cuadro de diálogo.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="DS_SETFONT"></span><span id="ds_setfont"></span><dl> <dt>**SETFONT de DS \_**</dt> </dl>                   | Indica que el encabezado de la plantilla de cuadro de diálogo (estándar o extendido) contiene datos adicionales que especifican la fuente que se va a utilizar para el texto en el área de cliente y los controles del cuadro de diálogo. Si es posible, el sistema selecciona una fuente según los datos de fuente especificados. El sistema pasa un identificador a la fuente al cuadro de diálogo y a cada control enviándole el mensaje de [**WM \_ SETFONT**](/windows/desktop/winmsg/wm-setfont) . Para obtener descripciones del formato de estos datos de fuente, vea [**DLGTEMPLATE**](/windows/desktop/api/Winuser/ns-winuser-dlgtemplate) y [**DLGTEMPLATEEX**](dlgtemplateex.md).<br/> Si no se especifica **DS \_ SETFONT** ni **DS \_ SHELLFONT** , la plantilla de cuadro de diálogo no incluye los datos de la fuente.<br/>                                                                                                                                                                                                                                 |
| <span id="DS_SETFOREGROUND"></span><span id="ds_setforeground"></span><dl> <dt>**\_SETFOREGROUND DS**</dt> </dl> | Hace que el sistema utilice la función [**SetForegroundWindow**](/windows/desktop/api/winuser/nf-winuser-setforegroundwindow) para colocar el cuadro de diálogo en primer plano. Este estilo es útil en los cuadros de diálogo modales que requieren atención inmediata del usuario, independientemente de si la ventana propietaria es la ventana de primer plano.<br/> El sistema restringe los procesos que pueden establecer la ventana de primer plano. Para obtener más información, vea [ventanas de primer plano y de fondo](/windows/desktop/winmsg/window-features).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="DS_SHELLFONT"></span><span id="ds_shellfont"></span><dl> <dt>**\_SHELLFONT DS**</dt> </dl>             | Indica que el cuadro de diálogo debe utilizar la fuente del sistema. El miembro de tipo de **letra** de la plantilla de cuadro de diálogo extendido debe establecerse en MS Shell Dlg. De lo contrario, este estilo no tiene ningún efecto. También se recomienda usar el [recurso DIALOGEX](../menurc/dialogex-resource.md), en lugar del [recurso de cuadro de diálogo](../menurc/dialog-resource.md). Para obtener más información, vea [fuentes de cuadros de diálogo](about-dialog-boxes.md).<br/> El sistema selecciona una fuente con los datos de fuente especificados en los miembros de **puntuación**, **peso** y **cursiva** . El sistema pasa un identificador a la fuente al cuadro de diálogo y a cada control enviándole el mensaje de [**WM \_ SETFONT**](/windows/desktop/winmsg/wm-setfont) . Para obtener descripciones del formato de estos datos de fuente, vea [**DLGTEMPLATEEX**](dlgtemplateex.md). <br/> Si no se especifica **DS \_ SHELLFONT** ni **DS \_ SETFONT** , la plantilla de cuadro de diálogo extendido no incluye los datos de la fuente.<br/> |
| <span id="DS_SYSMODAL"></span><span id="ds_sysmodal"></span><dl> <dt>**\_SYSMODAL DS**</dt> </dl>                | Este estilo está obsoleto y se incluye por compatibilidad con las versiones de 16 bits de Windows. Si especifica este estilo, el sistema crea el cuadro de diálogo con el estilo **WS \_ ex de \_ nivel superior** . Este estilo no impide que el usuario tenga acceso a otras ventanas del escritorio.<br/> No combine este estilo con el estilo **de \_ control de DS** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
