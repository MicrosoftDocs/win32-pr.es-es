# <a name="using-oleview"></a>Usar OleView

Aunque el editor del registro muestra una vista no procesada del registro y es útil para aprender la organización del registro, el uso del editor del registro puede llevar mucho tiempo.

OleView proporciona una vista de nivel superior de la información contenida en el registro. En lugar de ver una larga lista de CLSID, las características de OleView controlan los controles de árbol con nombres descriptivos.
