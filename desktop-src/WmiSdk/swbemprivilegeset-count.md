# <a name="swbemprivilegesetcount-property"></a>Propiedad SWbemPrivilegeSet.Count

La **propiedad Count** del objeto [**SWbemPrivilege**](swbemprivilege.md) contiene el número de elementos de una colección [**SWbemPrivilegeSet.**](swbemprivilegeset.md) Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemPrivilegeSet.Count As Integer
```
