## <a name="tokenization-wordbreakers-and-language-resources"></a><span data-ttu-id="75b12-105">Tokenización, Wordbreakers y recursos de idioma</span><span class="sxs-lookup"><span data-stu-id="75b12-105">Tokenization, Wordbreakers, and Language Resources</span></span>

<span data-ttu-id="75b12-108">El lenguaje del texto indexado y la cadena de consulta se dividen en tokens.</span><span class="sxs-lookup"><span data-stu-id="75b12-108">Both the language of the indexed text and the query string are broken into tokens.</span></span> <span data-ttu-id="75b12-109">Dado que las reglas de tokenización varían según el lenguaje, hay wordbreakers independientes para cada idioma o familia de idiomas.</span><span class="sxs-lookup"><span data-stu-id="75b12-109">Because tokenization rules vary by language, there are separate wordbreakers for each language or family of languages.</span></span> <span data-ttu-id="75b12-110">Si hay una discrepancia entre el lenguaje de consulta y el lenguaje indizado, los resultados pueden ser imprevisibles.</span><span class="sxs-lookup"><span data-stu-id="75b12-110">If there is a mismatch between the query language and the indexed language, the results can be unpredictable.</span></span>

<span data-ttu-id="75b12-111">Windows Search incluye un conjunto bien definido de wordbreakers.</span><span class="sxs-lookup"><span data-stu-id="75b12-111">Windows Search ships with a well defined set of wordbreakers.</span></span> <span data-ttu-id="75b12-112">Los componentes del separador clásico y lingüísticos son compatibles con Windows Vista y versiones posteriores.</span><span class="sxs-lookup"><span data-stu-id="75b12-112">Classic wordbreaker and stemmer components are supported in Windows Vista and later.</span></span> <span data-ttu-id="75b12-113">Si no se puede determinar el idioma de un documento, Windows Search intenta detectar el idioma para identificar el separador más apropiado.</span><span class="sxs-lookup"><span data-stu-id="75b12-113">If the language of a document cannot be determined, Windows Search attempts to detect the language to identify the most appropriate wordbreaker.</span></span> <span data-ttu-id="75b12-114">Windows Search intenta detectar el idioma mediante una llamada a la función [**GetSystemPreferredUILanguages**](/windows/win32/api/winnls/nf-winnls-getsystempreferreduilanguages) para determinar el primer idioma de la interfaz de usuario múltiple (MUI) (que suele ser el idioma de la interfaz de usuario del sistema, a menos que se instalen paquetes de idioma MUI).</span><span class="sxs-lookup"><span data-stu-id="75b12-114">Windows Search attempts to detect the language by calling the [**GetSystemPreferredUILanguages**](/windows/win32/api/winnls/nf-winnls-getsystempreferreduilanguages) function to determine the first Multiple User Interface (MUI) language (which is typically the system UI language unless MUI language packs are installed).</span></span> <span data-ttu-id="75b12-115">Si la <span data-ttu-id="75b12-123">La información de configuración regional del contenido indizado se recupera del origen del contenido.</span><span class="sxs-lookup"><span data-stu-id="75b12-123">Locale information for indexed content is retrieved from the source of the content.</span></span> <span data-ttu-id="75b12-124">Si el implementador de origen no conoce la configuración regional del contenido indizado, debe establecer la configuración regional en [**\_ independiente de la configuración regional**](../intl/locale-neutral.md).</span><span class="sxs-lookup"><span data-stu-id="75b12-124">If the source implementer does not know the locale of the indexed content, it should set the locale to [**LOCALE\_NEUTRAL**](../intl/locale-neutral.md).</span></span>

<span data-ttu-id="75b12-125">Por ejemplo, si implementa un controlador de filtro (una implementación de la interfaz [**IFilter**](https://www.bing.com/search?q=**IFilter**) ), un controlador de propiedad o un controlador de protocolo, debe establecer la configuración regional del contenido indizado en [**\_ independiente**](../intl/locale-neutral.md) de la configuración regional a menos que tenga información de configuración regional específica y esté seguro de su precisión.</span><span class="sxs-lookup"><span data-stu-id="75b12-125">For example, if you implement a filter handler (an implementation of the [**IFilter**](https://www.bing.com/search?q=**IFilter**) interface), property handler, or protocol handler, you should set the locale for indexed content to [**LOCALE\_NEUTRAL**](../intl/locale-neutral.md) unless you have specific locale information and are confident of its accuracy.</span></span>

> [!TIP]
