# <a name="ms-sql-alias-attribute"></a>Atributo MS-SQL-alias

Alias que se utiliza para conectarse a la base de datos. El valor predeterminado se establece en alias.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Alias MS-SQL                                |
| Nombre para mostrar de LDAP | Alias mS-SQL                                |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1395                     |
| System-ID-GUID    | e0c6baae-ccee-11d2-9993-0000f87a57d4        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
