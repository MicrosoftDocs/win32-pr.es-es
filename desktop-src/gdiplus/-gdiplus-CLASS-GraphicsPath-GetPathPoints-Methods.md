# <a name="graphicspathgetpathpoints-methods"></a>Métodos GraphicsPath. GetPathPoints

En este tema se enumeran los métodos GetPathPoints de la clase [**GraphicsPath**](/windows/win32/api/gdipluspath/nl-gdipluspath-graphicspath) . Para obtener una lista completa de los métodos de la clase **GraphicsPath** , vea [**GraphicsPath**](/windows/win32/api/gdipluspath/nl-gdipluspath-graphicspath).

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                                    | Descripción                                                                                                                                                                                                                                                                           |
|:----------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetPathPoints (Point \* , int)**](/windows/win32/api/gdipluspath/nf-gdipluspath-graphicspath-getpathpoints(outpoint_inint))   | El método [**GraphicsPath:: GetPathPoints**](/windows/win32/api/gdipluspath/nf-gdipluspath-graphicspath-getpathpoints(outpoint_inint)) obtiene la matriz de puntos de esta ruta de acceso. La matriz contiene los extremos y los puntos de control de las líneas y las curvas spline de Bézier que se usan para dibujar el trazado.<br/>  |
| [**GetPathPoints (PointF \* , int)**](/previous-versions//ms535582(v=vs.85)) | El método [**GraphicsPath:: GetPathPoints**](/previous-versions//ms535582(v=vs.85)) obtiene la matriz de puntos de esta ruta de acceso. La matriz contiene los extremos y los puntos de control de las líneas y las curvas spline de Bézier que se usan para dibujar el trazado.<br/> |
