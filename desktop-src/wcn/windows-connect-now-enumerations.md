# <a name="windows-connect-now-enumerations"></a>Enumeraciones de Windows Connect Now

Windows Connect ahora define los siguientes tipos de enumeración.



| Enumeración                                                                                             | Descripción                                                                                                                   |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**\_tipo de atributo WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_attribute_type)                                                      | Define los diversos tipos de búfer de atributo definidos para Wi-Fi la instalación protegida.<br/>                                      |
| [**\_tipo de contraseña de WCN \_**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_password_type)                                                        | Define la autenticación que se usará en una sesión de WPS. <br/>                                                    |
| [**Estado de la sesión de WCN \_ \_**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_session_status)                                                      | Define el estado de una sesión de WPS.<br/>                                                                               |
| [**\_Estado de \_ Asociación de tipo de valor de WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_association_state)                        | Define los posibles estados de Asociación de una estación inalámbrica durante una solicitud de detección.<br/>                          |
| [**tipo de valor de WCN \_ \_ \_ booleano**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_boolean)                                             | Define los valores que se usan para representar las condiciones true/false que se encuentran durante la configuración del dispositivo y la asociación.<br/>            |
| [**tipo \_ de \_ autenticación de tipo de valor de \_ WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_authentication_type)                    | Define los métodos de autenticación admitidos por el inscrito (punto de acceso o estación).<br/>                            |
| [**\_métodos de \_ configuración del tipo de valor de WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_config_methods)                              | Define los métodos de configuración admitidos por el inscrito o registrador.<br/>                                          |
| [**\_error de \_ configuración del tipo de valor de WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_configuration_error)                    | Define los posibles valores de error devueltos a un dispositivo al intentar configurar y asociar a la WLAN.<br/> |
| [**tipo de valor de WCN \_ \_ \_ \_ ID. de contraseña de dispositivo \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_device_password_id)                     | Define los posibles valores de contraseña.<br/>                                                                              |
| [**tipo \_ de \_ cifrado de tipo de valor de \_ WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_encryption_type)                            | Define los tipos de cifrado de WLAN admitidos.<br/>                                                                       |
| [**tipo de valor de WCN- \_ \_ \_ bandas de RF \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_rf_bands)                                          | Define la banda de frecuencia de radio en la que un inscrito envía solicitudes de detección.<br/>                                    |
| [**\_versión de \_ tipo de valor de WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_version)                                             | Define la versión compatible de Wi-Fi la instalación protegida (WPS).<br/>                                                      |
| [**tipo de valor de WCN \_ \_ \_ Estado de \_ \_ \_ instalación protegida \_ de Wi-Fi**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_wi_fi_protected_setup_state) | Define valores que indican si se ha configurado un dispositivo.<br/>                                                            |
