# <a name="d1232-32bpp-rgba-causes-conversion"></a>D1232:32 bpp RGBA causa la conversión

Considere la posibilidad de usar BGR en lugar de RGB para formatos 32 bpp, de modo que el efecto BitmapSource pueda evitar conversiones de formato adicionales.
