# <a name="registeredtaskname-property"></a>RegisteredTask.Name propiedad

Para el scripting, obtiene el nombre de la tarea registrada.

## <a name="syntax"></a>Syntax


```VB
RegisteredTask.Name As String
```



## <a name="property-value"></a>Valor de propiedad

Nombre de la tarea registrada.
