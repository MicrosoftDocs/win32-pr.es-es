# <a name="custom-file-and-stream-handlers"></a>Controladores personalizados de archivos y secuencias

Los controladores de archivos y secuencias son controladores que proporcionan interfaces coherentes para una aplicación que controla los datos multimedia. Los controladores de archivos y secuencias incluidos en el sistema operativo usan datos de vídeo y de onda de audio almacenados en archivos de audio-vídeo intercalado (AVI) y de audio de onda.

Puede escribir Controladores para permitir que la aplicación escriba u obtenga acceso a los datos multimedia desde otro origen, como un archivo con un formato de propietario, un archivo AVI que se ha expandido para contener flujos de datos adicionales o un controlador que genera sus propios datos multimedia. Si tiene un formato de archivo personalizado para los datos AVI que quiere usar con las [funciones y macros de AVIFile](avifile-functions-and-macros.md), debe escribir un controlador personalizado.
