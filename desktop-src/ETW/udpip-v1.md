# <a name="udpip_v1-class"></a>UdpIp \_ V1 (clase)

Esta clase es la clase primaria para los eventos UDP/IP.

La sintaxis siguiente se simplifica a partir del código MOF.

## <a name="syntax"></a>Sintaxis

``` syntax
[Guid("{bf3a50c5-a9c9-4988-a005-2df0b7c80f80}")]
class UdpIp_V1 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Miembros

La **clase UdpIp \_ V1** no define ningún miembro.
