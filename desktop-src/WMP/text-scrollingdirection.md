# <a name="textscrollingdirection"></a>TEXT. scrollingDirection

El atributo **scrollingDirection** especifica o recupera la dirección del texto de desplazamiento.

``` syntax
        elementID.scrollingDirection
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de lectura/escritura.



| Value | Descripción                         |
|-------|-------------------------------------|
| Left  | Predeterminada. Desplazarse de derecha a izquierda. |
| Right | Desplazarse de izquierda a derecha.          |



 

## <a name="remarks"></a>Observaciones

Si el **desplazamiento** se establece en false, se omite **scrollingDirection** .

Vea el atributo [Value](text-value.md) para ver un ejemplo que muestra cómo se utilizan los atributos del elemento de **texto** .
