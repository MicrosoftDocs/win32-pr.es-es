# <a name="cfstr_ds_parenthwnd"></a>CFSTR \_ DS \_ PARENTHWND

<dl> <dt>

<span id="CFSTR_DS_PARENTHWND"></span><span id="cfstr_ds_parenthwnd"></span>**CFSTR \_ DS \_ PARENTHWND**
</dt> <dd> <dl> <dt>

"DsAdminParentHwndClipFormat"
</dt> <dt>



El formato del portapapeles de **CFSTR \_ DS \_ PARENTHWND** proporciona un identificador de memoria global (**HGLOBAL**) que contiene el identificador de ventana (**hWnd**) de la ventana principal de MMC.

> [!Note]  
> Este valor de formato del portapapeles no está definido en un archivo de encabezado publicado. Para usar este valor de formato del portapapeles, debe definirlo en el formato exacto que se muestra.
