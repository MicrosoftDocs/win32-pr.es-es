# <a name="iagentnotifysinkex"></a>IAgentNotifySinkEx

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

**IAgentNotifySinkEx** se deriva de la interfaz [**IAgentNotifySink**](https://www.bing.com/search?q=**IAgentNotifySink**) . Incluye todos los métodos **IAgentNotifySink** y proporciona acceso a funciones adicionales.

**Métodos en orden de Vtable**



| Métodos IAgentNotifySinkEx                                                   | Descripción                                                        |
|------------------------------------------------------------------------------|--------------------------------------------------------------------|
| [**HelpComplete**](iagentnotifysinkex--helpcomplete.md)                     | Se produce cuando el usuario selecciona un menú o el carácter en el modo de ayuda. |
| [**ListeningState**](iagentnotifysinkex--listeningstate.md)                 | Se produce cuando cambia el estado de escucha del carácter.               |
| [**DefaultCharacterChange**](iagentnotifysinkex--defaultcharacterchange.md) | Se produce cuando el usuario cambia el carácter predeterminado.                |
| [**AgentPropertyChange**](iagentnotifysinkex--agentpropertychange.md)       | Se produce cuando el usuario cambia la configuración de una propiedad del agente.            |
| [**ActiveClientChange**](iagentnotifysinkex--activeclientchange.md)         | Se produce cuando cambia el cliente activo de un carácter.              |
