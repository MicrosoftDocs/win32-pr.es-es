# <a name="cvideocaptureterminal"></a>CVideoCaptureTerminal

El terminal **CVideoCaptureTerminal** se deriva de [CSingleFilterTerminal](csinglefilterterminal.md) e implementa un terminal de captura de vídeo estático con un solo filtro DirectShow. No es compatible con determinadas interfaces de captura de vídeo avanzadas que solo son aplicables a la captura de vídeo WDM. La mayoría de los MSP no tendrán que invalidar la implementación de este terminal.
