# <a name="system-auxiliary-class-attribute"></a>Atributo System-Auxiliary-Class

Lista de clases auxiliares que el usuario no puede modificar.



| Entrada | Valor |
|-------------------|--------------------------------------------------------------------|
| CN                | Clase auxiliar del sistema                                             |
| Ldap-Display-Name | systemClassiliaryClass                                               |
| Size              | \-                                                                 |
| Privilegio actualizar  | Administrador de esquemas                                               |
| Frecuencia de actualización  | Cuando se crea la clase o se le agrega una nueva clase auxiliar. |
| Attribute-Id      | 1.2.840.113556.1.4.198                                             |
| System-Id-Guid    | bf967a43-0de6-11d0-a285-00aa003049e2                               |
| Syntax            | [**String(Object-Identifier)**](s-string-object-identifier.md)    |
