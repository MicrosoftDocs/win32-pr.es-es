# <a name="using-the-iis-ui-to-configure-the-virtual-directory"></a>Usar la interfaz de usuario de IIS para configurar el directorio virtual

Para usar la interfaz de usuario de IIS para configurar el directorio virtual para cargas de BITS, seleccione **propiedades** en el menú contextual del directorio virtual. Presione la pestaña **extensión bits** para tener acceso a la página de propiedades bits (debe estar instalada la extensión de servidor bits ISAPI). Cambie los valores de propiedad predeterminados según corresponda y habilite el directorio para las cargas. no se pueden cargar archivos hasta que el directorio esté habilitado. Para obtener más información sobre cómo establecer los valores de propiedad, vea la ayuda en pantalla.
