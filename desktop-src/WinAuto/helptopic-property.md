# <a name="helptopic-property"></a>Propiedad HelpTopic

La propiedad **helpTopic** obtiene los detalles que se usan con la función [WinHelp](/windows/win32/api/winuser/nf-winuser-winhelpa) de Microsoft Win32 para mostrar un tema de ayuda asociado al objeto. No todos los objetos admiten esta propiedad.

La propiedad **helpTopic** se recupera llamando a [**IAccessible:: get \_ accHelpTopic**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_acchelptopic).
