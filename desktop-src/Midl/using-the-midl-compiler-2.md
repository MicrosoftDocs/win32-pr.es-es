# <a name="using-the-midl-compiler"></a>Usar el compilador MIDL

En los temas siguientes se describen los requisitos del compilador de MIDL C y del preprocesador de C, las bibliotecas de vínculos que forman parte del producto llamada a procedimiento remoto (RPC) y los archivos que genera MIDL para las interfaces DCOM, las interfaces RPC y las interfaces de la biblioteca de tipos de automatización OLE.

-   [Invocar el compilador MIDL](invoking-the-midl-compiler.md)
-   [Archivos de respuesta](response-files.md)
-   [Requisitos del preprocesador de C para MIDL](c-preprocessor-requirements-for-midl.md)
-   [Consideraciones del compilador de C/C++](c-c-compiler-considerations.md)
-   [Usar la \_ \_ constante predefinida MIDL](using-the---midl-predefined-constant.md)
-   [MIDL y RPC](midl-and-rpc.md)
-   [MIDL y COM](midl-and-com.md)
-   [MIDL y ODL](midl-and-odl.md)

Para obtener más información acerca de los archivos que componen el producto RPC, vea [instalar el entorno de programación RPC](/windows/desktop/Rpc/installing-the-rpc-programming-environment).
