# <a name="system-restore-api"></a>API de restauración del sistema

La API de restauración del sistema permite que una aplicación administre los puntos de restauración. También hay una interfaz que admite scripts disponible a través de [instrumental de administración de Windows](/windows/desktop/WmiSdk/wmi-start-page) (WMI).

Para obtener más información acerca de la API de restauración del sistema, vea los temas siguientes:

-   [Llamando a SRSetRestorePoint](calling-srsetrestorepoint.md)
-   [Texto de descripción de punto de restauración](restore-point-description-text.md)
-   [Llamadas anidadas a SRSetRestorePoint](nested-calls-to-srsetrestorepoint.md)

Para obtener más información acerca de las clases WMI de restaurar sistema, vea [configurar Restaurar sistema](configuring-system-restore.md).
