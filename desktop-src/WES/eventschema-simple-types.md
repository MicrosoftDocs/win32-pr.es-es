# <a name="event-schema-simple-types"></a>Tipos simples de esquema de eventos

A continuación se enumeran los tipos simples que define el esquema de eventos.



| Tipo simple                                                 | Descripción                                                               |
|-------------------------------------------------------------|---------------------------------------------------------------------------|
| [**GUIDType**](eventschema-guidtype-simpletype.md)         | Define un tipo de identificador único global, en formato registro.<br/> |
| [**HexInt32Type**](eventschema-hexint32type-simpletype.md) | Define un tipo hexadecimal de 1 byte.<br/>                             |
| [**HexInt64Type**](eventschema-hexint64type-simpletype.md) | Define un tipo hexadecimal de 8 bytes.<br/>                            |
