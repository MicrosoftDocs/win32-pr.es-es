# <a name="sliderthumbdownimage"></a>SLIDEr. thumbDownImage

El atributo **thumbDownImage** especifica o recupera la imagen que representa el estado inactivo del control de posición.

``` syntax
        elementID.thumbDownImage
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de lectura/escritura que contiene el nombre de un archivo de imagen.

## <a name="remarks"></a>Observaciones

**ThumbDownImage** es opcional y es la imagen que se usa para representar el estado de inactividad de **thumbImage**. Si no se proporciona un **thumbDownImage** , el control deslizante usará **thumbImage**.

Los formatos admitidos son BMP, JPG, PNG y GIF (sin incluir los GIF animados).
