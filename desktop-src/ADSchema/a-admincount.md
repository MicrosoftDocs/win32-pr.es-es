# <a name="admin-count-attribute"></a>Admin-Count atributo)

Indica que el sistema ha cambiado las ACL de un objeto dado a un valor más seguro porque era miembro de uno de los grupos administrativos (directa o transitiva).



| Entrada | Value |
|-------------------|-----------------------------------------------------|
| CN                | Admin-Count                                         |
| Nombre para mostrar de LDAP | adminCount                                          |
| Tamaño              | 4 bytes                                             |
| Actualizar privilegio  | El sistema establece este valor.                    |
| Frecuencia de actualización  | Cuando se agrega un objeto a un grupo administrativo. |
| Attribute-Id      | 1.2.840.113556.1.4.150                              |
| System-ID-GUID    | bf967918-0de6-11d0-a285-00aa003049e2                |
| Sintaxis            | [**Enumeración**](s-enumeration.md)                |
