# <a name="directory-management-macros"></a>Macros de administración de directorios

Las macros siguientes se utilizan con puntos de reanálisis.

## <a name="in-this-section"></a>En esta sección



| Macro                                                                     | Descripción                                                                                                                             |
|---------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**IsReparseTagMicrosoft**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagmicrosoft)<br/>         | Determina si una etiqueta de punto de reanálisis indica un punto de reanálisis de Microsoft.<br/>                                                  |
| [**IsReparseTagNameSurrogate**](/windows/desktop/api/Winnt/nf-winnt-isreparsetagnamesurrogate)<br/> | Determina si el punto de reanálisis asociado de una etiqueta es un suplente para otra entidad con nombre (por ejemplo, una carpeta montada).<br/> |
