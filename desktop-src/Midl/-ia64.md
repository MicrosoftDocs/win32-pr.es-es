# <a name="ia64-switch"></a>modificador/ia64

El modificador **/ia64** indica al compilador de MIDL que genere archivos de código auxiliar, o un archivo de biblioteca de tipos, para un entorno de arquitectura Intel de 64 bits.

``` syntax
midl /ia64
```

## <a name="remarks"></a>Observaciones

> [!Note]  
> No es posible usar MIDL.exe para generar un TLB de 64 bits en Windows 2000.
