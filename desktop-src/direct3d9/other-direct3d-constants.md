## <a name="other-constants"></a>Otras constantes

En la tabla siguiente se enumeran las constantes que se usan internamente:



|                                       |                                                   |                                                                    |
|---------------------------------------|---------------------------------------------------|--------------------------------------------------------------------|
| \#define                              | Value                                             | Descripción                                                        |
| RENDERTARGETS de D3D como \_ máximo \_ simultáneo \_ | 4                                                 | Número máximo de Rendertargets.                               |
| D3DDMAPSAMPLER                        | 256                                               | Número máximo de ejemplos de mapa de desplazamiento.                    |
| D3DDP \_ MAXTEXCOORD                    | 8                                                 | Número máximo de coordenadas de textura.                         |
| MAXD3DDECLLENGTH                      | 64 (no incluye el elemento de vértice del marcador "final") | Número máximo de elementos en una declaración de vértice.                |
| MAXD3DDECLUSAGEINDEX                  | 15                                                | Índice máximo (0-15) que se puede usar en una declaración de vértice. |



 

Estas \# definiciones se declaran en d3d9types. h.
