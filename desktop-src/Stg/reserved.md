# <a name="reserved"></a>Reservado

El valor del tipo de datos **DWORD** se reserva para uso futuro. Los escritores de conjuntos de propiedades deben establecer este valor en 1; los lectores de conjuntos de propiedades deben asegurarse de que este valor sea al menos 1. Una excepción a esta directriz es que, para [los conjuntos de propiedades DocumentSummaryInformation y UserDefined](the-documentsummaryinformation-and-userdefined-property-sets.md), este valor puede ser 2.
