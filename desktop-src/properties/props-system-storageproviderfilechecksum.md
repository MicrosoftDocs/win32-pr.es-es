# <a name="systemstorageproviderfilechecksum"></a>System. StorageProviderFileChecksum

Suma de comprobación calculada por el proveedor de almacenamiento para el archivo. Los archivos con el mismo valor de suma de comprobación tendrán el mismo contenido.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.StorageProviderFileChecksum
   shellPKey = PKEY_StorageProviderFileChecksum
   formatID = B2F9B9D6-FEC4-4DD5-94D7-8957488C807B
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Buffer
      IsInnate = false
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
