# <a name="program-information-tables-group"></a>Grupo de tablas de información de programas

El grupo de tablas información del programa contiene información importante que se utiliza en toda la instalación.

-   La [tabla de propiedades](property-table.md) proporciona un medio para especificar todas las [propiedades](properties.md) de una instalación de.
-   La [tabla binaria](binary-table.md) contiene los datos binarios de elementos tales como mapas de bits, animaciones e iconos. La tabla binaria también se utiliza para almacenar datos para las acciones personalizadas. Esta tabla puede contener un mapa de bits para una cartelera, el icono del programa o el formulario ejecutable de una acción personalizada.
-   La [tabla de errores](error-table.md) se utiliza para buscar plantillas de formato de mensajes de error al procesar errores con un conjunto de códigos de error. Este es el caso habitual en el que no hay ningún conjunto de plantillas de formato. El instalador tiene su propio mecanismo de procesamiento de errores. Los errores se pasan como registros.
-   [Tabla de acceso directo](shortcut-table.md) la tabla de acceso directo contiene la información que la aplicación necesita para crear accesos directos en el equipo del usuario.
-   La [tabla ReserveCost](reservecost-table.md) contiene el espacio en disco necesario para que cada componente funcione correctamente.
