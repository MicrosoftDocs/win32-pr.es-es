# <a name="internet-protocol-version-6-ipv6"></a>Protocolo de Internet versión 6 (IPv6)

Es importante asegurarse de que las nuevas aplicaciones Winsock y las existentes sean totalmente compatibles con IPv6. La disponibilidad del espacio de direcciones IPv4 para las nuevas asignaciones de direcciones IPv4 se agotó para Asia y el Pacífico en 2011. Se espera que se agoten otras partes del mundo en unos años.

Un porcentaje creciente de nuevos sitios web y servicios está disponible mediante direcciones IPv6. Muchos usuarios de Internet de los mercados emergentes dependen de IPv6 para el acceso a Internet.

Microsoft tiene un gran compromiso de admitir IPv6. La compatibilidad completa con IPv6 se proporciona en Windows XP con Service Pack 1 (SP1) y versiones posteriores.

En este documento se proporciona información acerca de la compatibilidad de Winsock con IPv6:

-   [Usar IPv6](using-ipv6-2.md)
-   [Usar Internet Explorer para tener acceso a sitios web de IPv6](using-internet-explorer-to-access-ipv6-web-sites-2.md)
-   [Configuraciones recomendadas para IPv6](recommended-configurations-2.md)
-   [Temas adicionales de IPv6](additional-topics-2.md)

Para obtener más información sobre cómo agregar funcionalidad IPv6 a las aplicaciones de Windows Sockets, consulte [Guía de IPv6 para aplicaciones de Windows Sockets](ipv6-guide-for-windows-sockets-applications-2.md).

En TechNet en el [Protocolo de Internet versión 6 (IPv6) de Microsoft](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/dd379473(v=ws.10))se encuentra disponible una introducción al protocolo IPv6 junto con información general sobre la implementación y las tecnologías de transición de IPv6.
