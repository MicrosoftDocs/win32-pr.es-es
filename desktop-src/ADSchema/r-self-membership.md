# <a name="self-membership-validated-writes"></a>Self-Membership de escrituras validadas

Permiso de escritura validado para permitir la actualización de la pertenencia de un grupo en cuanto a la adición o eliminación de una cuenta propia.



| Entrada | Value |
|--------------|--------------------------------------|
| CN           | Self-Membership                      |
| Display-Name | Agregar o quitar self como miembro            |
| Rights-GUID  | bf9679c0-0de6-11d0-a285-00aa003049e2 |
