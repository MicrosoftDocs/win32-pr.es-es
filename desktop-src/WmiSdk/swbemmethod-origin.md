# <a name="swbemmethodorigin-property"></a>Propiedad SWbemMethod. Origin

La propiedad **Origin** del objeto [**SWbemMethod**](swbemmethod.md) devuelve el nombre de la clase en la que se presentó este método. En el caso de las clases con jerarquías de herencia profunda, a menudo es conveniente saber qué métodos se han declarado en cada clase. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemMethod.Origin As String
```
