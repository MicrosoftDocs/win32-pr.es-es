# <a name="windows-connect-now-structures"></a>Estructuras de Windows Connect Now

Las siguientes estructuras admiten las funciones de la API de Windows Connect Now.



| Estructura                                                                             | Descripción                                                                                                                                            |
|---------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**tipo de valor de WCN tipo de \_ \_ \_ \_ dispositivo primario \_**](/windows/win32/api/wcntypes/ns-wcntypes-wcn_value_type_primary_device_type) | Contiene datos enumerados que identifican el tipo de un dispositivo por categoría, subcategoría y un OUI específico del fabricante (identificador de la organización).<br/> |
| [**\_Especificaciones de \_ extensión de proveedor WCN \_**](/windows/win32/api/wcndevice/ns-wcndevice-wcn_vendor_extension_spec)                     | Contiene datos que definen una extensión de proveedor. <br/>                                                                                             |
