# <a name="wmalbumartist"></a>WM/AlbumArtist

El **atributo WM/AlbumArtist** contiene el nombre del intérprete principal del álbum.

## <a name="global-constant"></a>Constante global

g \_ wszWMAlbumArtist

## <a name="data-type"></a>Tipo de datos

**CADENA DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentarios

En el caso de los álbumes en los que las pistas pueden tener diferentes intérpretes, como álbumes de resalte o de colaboración, este atributo se puede usar para especificar el autor principal del álbum. Es preferible usar este atributo en lugar de usar un valor de autor de "varios autores".
