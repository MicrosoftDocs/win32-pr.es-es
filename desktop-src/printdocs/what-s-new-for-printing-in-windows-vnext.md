# <a name="whats-new-for-printing"></a>Novedades de la impresión

Windows 8 admite la impresión para aplicaciones de Windows Store mediante [JavaScript](/previous-versions/windows/apps/hh465225(v=win.10)) y HTML e impresión para aplicaciones de la Tienda Windows con [C#/VB/C++ y XAML.](/previous-versions/windows/apps/hh465196(v=win.10))

Windows 8 incluye una nueva API basada en COM que proporciona compatibilidad completa para la impresión [openXPS](app-support-for-openxps-printing.md) y acceso a partes de los nuevos controladores de impresora que Windows 8 admite. Para más información sobre la nueva API, consulte [Print Document Package API](tailored-app-printing-api.md).

El Windows de bandeja de entrada del controlador de impresión Windows 8 compatibilidad con un alto porcentaje de las impresoras populares.
