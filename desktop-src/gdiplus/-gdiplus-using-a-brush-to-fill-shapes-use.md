# <a name="using-a-brush-to-fill-shapes"></a>Utilizar un pincel para rellenar formas

Un objeto [**Brush**](/windows/desktop/api/gdiplusbrush/nl-gdiplusbrush-brush) de Windows GDI+ se usa para rellenar el interior de una forma cerrada. GDI+ define varios estilos de relleno: color sólido, patrón de trama, textura de imagen y degradado de color.

En los temas siguientes se trata el uso de los pinceles con más detalle:

-   [Rellenar una forma con un color sólido](-gdiplus-filling-a-shape-with-a-solid-color-use.md)
-   [Relleno de una forma con un patrón de trama](-gdiplus-filling-a-shape-with-a-hatch-pattern-use.md)
-   [Rellenar una forma con una textura de imagen](-gdiplus-filling-a-shape-with-an-image-texture-use.md)
-   [Colocar en mosaico una forma con una imagen](-gdiplus-tiling-a-shape-with-an-image-use.md)
-   [Rellenar una forma con un degradado de color](-gdiplus-filling-a-shape-with-a-color-gradient-use.md)
