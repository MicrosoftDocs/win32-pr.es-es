# <a name="d1172-invalid-to-unmap-nonmapped-bitmap"></a>D1172: no se pudo desasignar el mapa de bits no asignado

Se intentó anular la asignación de un mapa de bits que no está asignado.






 

## <a name="possible-causes"></a>Causas posibles

Direct2D puede generar este mensaje e informar de un error si una aplicación intenta anular la asignación de un mapa de bits que todavía no se ha asignado mediante ID2D1Bitmap1:: Map.

## <a name="possible-fixes"></a>Posibles correcciones

Del mismo modo que ID2D1RenderTarget:: EndDraw debe seguir las llamadas a ID2D1RenderTarget:: BeginDraw, asegúrese de que ID2D1Bitmap1:: desasignar sigue las llamadas a ID2D1Bitmap1:: Map. No llame a ID2D1Bitmap1:: desasignar varias veces para anular un mapa de bits.
