# <a name="relevant-rfcs"></a>RFC pertinentes

Los estándares TCP/IP se definen en Solicitudes de comentarios (RFC), que publica internet Engineering Task Force (IETF). En la tabla siguiente se enumeran las RFC que son relevantes para SNMP.



| Número RFC | Título                                                                                                           |
|------------|-----------------------------------------------------------------------------------------------------------------|
| 1155       | "Estructura e identificación de la información de administración para Internets basadas en TCP/IP". Define SMI. Mib.        |
| 1157       | "A Simple Network Management Protocol (SNMP)". Define el propio SNMP.                                          |
| 1213       | "Base de información de administración para la administración de redes de Internet basadas en TCP/IP: MIB-II". Define MIB \_ II.MIB. |
| 1901       | "Introduction to Community-based SNMPv2"                                                                        |
| 1902       | "Structure of Management Information for Version 2 of the Simple Network Management Protocol (SNMPv2)"          |
| 1903       | "Textual Conventions for Version 2 of the Simple Network Management Protocol (SNMPv2)"                          |
| 1904       | "Instrucciones de conformidad para la versión 2 del protocolo de administración de redes simple (SNMPv2)"                       |
| 1905       | "Operaciones de protocolo para la versión 2 del protocolo de administración de redes simple (SNMPv2)"                          |
| 1906       | "Asignaciones de transporte para la versión 2 del Protocolo de administración de redes simple (SNMPv2)"                           |
| 1907       | "Management Information Base for Version 2 of the Simple Network Management Protocol (SNMPv2)"                  |
| 1908       | "Coexistencia entre la versión 1 y la versión 2 del marco de administración de redes estándar de Internet"             |
| 2089       | "V2ToV1 Mapping SNMPv2 onto SNMPv1 within a bi-lingual SNMP agent" (Asignación de SNMPv2 a SNMPv1 dentro de un agente SNMP multilingüe)                                              |
