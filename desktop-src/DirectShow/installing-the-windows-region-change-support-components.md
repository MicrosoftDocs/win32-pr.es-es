# <a name="installing-the-windows-region-change-support-components"></a>Instalación de los componentes de Windows regiones de cambios

Los desarrolladores de aplicaciones pueden usar la compatibilidad de cambio de región de DVD integrada Windows para cambiar la región dentro de las aplicaciones de reproducción de DVD.

En Windows 2000 y versiones posteriores, el componente que proporciona compatibilidad con el cambio de región de DVD se encuentra en el archivo DLL storprop.dll (disponible en %windir% \\ system32). Este archivo DLL se instala de forma predeterminada y no se requieren pasos adicionales para asegurarse de que este archivo DLL está disponible.
