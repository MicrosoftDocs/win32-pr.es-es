# <a name="getpositioninformationoperation-class"></a>Clase GetPositionInformationOperation

Registra un controlador de eventos que se invoca cuando se completa la operación asincrónica iniciada por [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync) y proporciona un método que devuelve los resultados de la operación.

**GetPositionInformationOperation** tiene estos tipos de miembros:

-   [Métodos](#methods)
-   [Propiedades](#properties)

### <a name="methods"></a>Métodos

La clase **GetPositionInformationOperation** tiene estos métodos.



| Método                                                           | Descripción                                                                                                                                                 |
|:-----------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetResults**](getpositioninformationoperation-getresults.md) | Devuelve los resultados de la operación asincrónica iniciada por [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync). <br/> |



 

### <a name="properties"></a>Propiedades

La clase **GetPositionInformationOperation** tiene estas propiedades.



| Propiedad                                                                  | Tipo de acceso           | Descripción                                                                                                                                                                                          |
|:--------------------------------------------------------------------------|:----------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Completed**](getpositioninformationoperation-completed.md)<br/> | Lectura/escritura<br/> | Obtiene o establece un controlador de eventos que se invoca cuando se completa la operación asincrónica iniciada por [**GetPositionInformationAsync**](/previous-versions/windows/desktop/api/windows.media.streaming/nf-windows-media-streaming-imediarenderer-getpositioninformationasync) . <br/> |
