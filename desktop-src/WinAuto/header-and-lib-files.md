# <a name="header-and-lib-files"></a>Archivos de encabezado y lib

Los siguientes archivos de encabezado y biblioteca admiten el desarrollo de aplicaciones de Microsoft Active Accessibility.



| Archivo         | Descripción                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Winuser. h    | Archivo de encabezado de C y C++ para usar controles de Windows                                                                                                                             |
| Oleacc. h     | Archivo de encabezado de C y C++ que contiene las definiciones necesarias para clientes y servidores de Microsoft Active Accessibility                                                                    |
| Winver. h     | Archivo de encabezado de C y C++ para obtener información sobre la versión                                                                                                                                |
| Oleacc. lib   | Biblioteca de importación para Oleacc.dll                                                                                                                                                |
| Oleacc.dll   | Biblioteca de vínculos dinámicos que proporciona el tiempo de ejecución de Microsoft Active Accessibility, administra las solicitudes de los clientes de Microsoft Active Accessibility y los elementos de interfaz de usuario comunes de los servidores proxy |
| Oleaccrc.dll | Biblioteca de vínculos dinámicos de recursos multilingüe utilizada por Oleacc.dll                                                                                                                |
