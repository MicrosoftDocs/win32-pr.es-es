# <a name="dictionaries-and-factoids"></a>Diccionarios y Factoids

Un diccionario es un léxico interno que un reconocedor usa para determinar las coincidencias de palabras y alternativas. Un Factoid aumenta la precisión del reconocimiento seleccionando un contexto específico para un campo determinado.

Para obtener más información acerca de los diccionarios, consulte [uso de diccionarios de aplicación](using-application-dictionaries.md).

Para obtener más información sobre Factoids, consulte [usar el contexto para mejorar la precisión](using-context-to-improve-accuracy.md).
