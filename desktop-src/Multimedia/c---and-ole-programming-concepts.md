# <a name="c-and-ole-programming-concepts"></a>Conceptos de programación de C++ y OLE

Los controladores de archivos y secuencias incluidos con Windows usan un diseño orientado a objetos para promocionar una interfaz estándar y compartir la funcionalidad. Estos controladores se escriben en C++ y utilizan el modelo de objetos de componentes OLE.

Puede desarrollar controladores personalizados mediante los sistemas de desarrollo de C o C++; sin embargo, se recomienda encarecidamente el uso de C++, ya que proporciona un enfoque más sencillo y sencillo para implementar un controlador. Con C++, puede definir explícitamente los datos como objetos y puede asociar las funciones que manipulan los datos con las funciones miembro de un objeto.

En esta sección se identifican y resumen brevemente los conceptos importantes de C++ y el modelo de objetos de componentes OLE que se aplican al diseño y la implementación de controladores de archivos y secuencias. Hay muchos libros escritos sobre la programación de C++ a los que puede hacer referencia para obtener más información. Para obtener más información sobre OLE, vea la *Referencia del programador de OLE*.
