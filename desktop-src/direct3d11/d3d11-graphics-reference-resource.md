# <a name="resource-reference-direct3d-11-graphics"></a>Referencia de recursos (gráficos de Direct3D 11)

La API de Direct3D define varios elementos de API que le ayudarán a crear y administrar recursos.


## <a name="in-this-section"></a>En esta sección



| Tema                                                                              | Descripción                                                                                                                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interfaces de recursos](d3d11-graphics-reference-resource-interfaces.md)<br/> | Hay una serie de interfaces para los dos tipos básicos de recursos: búferes y texturas. También hay interfaces para las vistas de recursos. <br/> Una aplicación utiliza una vista para enlazar un recurso a una fase de canalización. La vista define cómo se puede tener acceso al recurso durante la representación. En esta sección se describen las interfaces de vista. <br/> |
| [Funciones de recursos](d3d11-graphics-reference-resource-functions.md)<br/>   | Esta sección contiene información sobre las funciones de recursos.<br/>                                                                                                                                                                                                                                                                                |
| [Estructuras de recursos](d3d11-graphics-reference-resource-structures.md)<br/> | Las estructuras se usan para crear y usar recursos.<br/>                                                                                                                                                                                                                                                                                               |
| [Enumeraciones de recursos](d3d11-graphics-reference-resource-enums.md)<br/>    | Las enumeraciones se utilizan para especificar información sobre cómo se crean y se obtiene acceso a los recursos durante la representación.<br/>                                                                                                                                                                                                                                    |
