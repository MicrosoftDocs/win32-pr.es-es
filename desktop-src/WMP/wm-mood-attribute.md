# <a name="wmmood-attribute"></a>Atributo WM/humor

El atributo **WM/humor** es un nombre de categoría para el ambiente del contenido.

## <a name="applies-to"></a>Se aplica a

-   [Elementos de audio](audio-item-attributes.md)
-   [Atributos de archivo de Windows Media de uso frecuente](commonly-used-windows-media-file-attributes.md)

## <a name="remarks"></a>Observaciones

Este atributo solo se almacena en el archivo multimedia digital.

El **ambiente** es un alias para este atributo.

La constante del SDK de Windows Media Format para este atributo es g \_ wszWMMood.

Para determinar si puede cambiar el valor de este atributo, use el método [media. isReadOnlyItem](media-isreadonlyitem.md) .
