# <a name="security-in-active-directory-domain-services"></a>Seguridad en Active Directory Domain Services

Active Directory Domain Services forman parte de la base de informática de confianza de Windows 2000 y son un participante completo en la infraestructura de seguridad de Windows 2000. Cada objeto de un servidor de Active Directory está protegido por su propio descriptor de seguridad. El sistema valida cualquier intento de obtener acceso a un objeto o atributo en un servidor de Active Directory comprobando los derechos de acceso concedidos por el descriptor de seguridad del objeto.

En los temas siguientes se tratan elementos importantes de seguridad en Active Directory Domain Services:

-   [Protección de objetos y atributos](object-and-attribute-protection.md)
-   [Delegación](delegation.md)
-   [Herencia](inheritance.md)

Para obtener más información y ejemplos de código, vea [controlar el acceso a objetos en Active Directory Domain Services](controlling-access-to-objects-in-active-directory-domain-services.md).
