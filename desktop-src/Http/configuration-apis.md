# <a name="configuration-apis"></a>API de configuración

Las API del servidor HTTP proporcionan funcionalidad para que los administradores del sistema establezcan, consulten y eliminen parámetros de configuración. Las API de configuración controlan la configuración de todas las aplicaciones que usan la API del servidor HTTP en el sistema.

Las secciones que se enumeran a continuación incluyen información sobre las opciones de configuración.

> [!Note]  
> Todos ellos son pasos de configuración opcionales que se aplicarán en función de los requisitos del sistema o de la aplicación.

 

[Lista de escucha](ip-listen-list.md) de IP: describe cómo configurar HTTP para que funcione sobre interfaces IP específicas.

[Certificados SSL:](ssl-certificates.md) detalla los procedimientos para configurar certificados con la API del servidor HTTP.

[Reservas, registros y](namespace-reservations-registrations-and-routing.md) enrutamiento de espacios de nombres: describe el proceso de reserva de partes del espacio de nombres para todas las aplicaciones de la máquina mediante la API del servidor HTTP. También se describe el proceso para registrar y enrutar solicitudes a aplicaciones específicas.

[HttpCfg.exe: ](httpcfg-exe.md) un programa de ejemplo que usa las API de configuración. Una versión compilada de esta herramienta también se incluye con el sistema operativo entre las herramientas de soporte técnico, junto con un archivo de ayuda que documenta cómo usarla.
