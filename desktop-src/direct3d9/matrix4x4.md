# <a name="matrix4x4"></a>Matrix4x4

Define una matriz de 4 x 4. Se usa como una matriz de transformación de fotogramas.

``` syntax
template Matrix4x4
{
    < F6F23F45-7686-11cf-8F52-0040333594A3 >
    array float matrix[16];
} 
```

Donde:

-   Matriz float \[ 16: \] matriz de 16 flotantes.
