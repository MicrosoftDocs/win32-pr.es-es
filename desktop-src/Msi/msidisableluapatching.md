# <a name="msidisableluapatching-property"></a>Propiedad MSIDISABLELUAPATCHING

Establezca la propiedad **MSIDISABLELUAPATCHING** en 1 en la línea de comandos o en la tabla de [propiedades](property-table.md) para evitar la aplicación de revisiones con privilegios mínimos en una aplicación. Para evitar la aplicación de revisiones con privilegios mínimos de todas las aplicaciones del equipo, establezca la directiva [DisableLUAPatching](disableluapatching.md) en 1. Para obtener información acerca de la revisión de la cuenta de usuario con privilegios mínimos, consulte [Patching control de cuentas de usuario (UAC)](user-account-control--uac--patching.md).

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versión<br/> | Windows Installer 5,0 en Windows Server 2012, Windows 8, Windows Server 2008 R2 o Windows 7. Windows Installer 4,0 o Windows Installer 4,5 en Windows Server 2008 o Windows Vista. Windows Installer 3,0 o posterior en Windows Server 2003 o Windows XP. Consulte los [requisitos de Run-Time de Windows Installer](windows-installer-portal.md) para obtener información sobre la Service Pack mínima de Windows que requiere una versión Windows Installer.<br/> |
