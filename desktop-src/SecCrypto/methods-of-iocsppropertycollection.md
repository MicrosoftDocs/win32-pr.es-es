# <a name="methods-of-iocsppropertycollection"></a>Métodos de IOCSPPropertyCollection

La interfaz [**IOCSPPropertyCollection**](/windows/desktop/api/Certadm/nn-certadm-iocsppropertycollection) define los métodos siguientes. Los métodos de acceso de propiedad no se muestran aquí. Para ver las propiedades de **IOCSPPropertyCollection,** vea [Propiedades de IOCSPPropertyCollection.](properties-of-iocsppropertycollection.md)



| Método                                                                                      | Descripción                                                                               |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**CreateProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-createproperty)                     | Crea una nueva propiedad y la agrega a un conjunto de propiedades.                                    |
| [**DeleteProperty**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-deleteproperty)                     | Quita una propiedad con nombre de un conjunto de propiedades.                                             |
| [**GetAllProperties**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-getallproperties)                 | Obtiene todas las propiedades de un conjunto de propiedades.                                                    |
| [**InitializeFromProperties**](/windows/desktop/api/Certadm/nf-certadm-iocsppropertycollection-initializefromproperties) | Crea un conjunto de propiedades a partir de las propiedades contenidas en una configuración de servidor existente. |
