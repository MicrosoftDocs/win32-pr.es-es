# <a name="no_cpp-nocpp"></a>/no \_ CPP,/nocpp

El modificador **/no \_ CPP** especifica que el compilador MIDL no llama al preprocesador de C para preprocesar el archivo IDL.

El modificador **/no \_ CPP** tiene prioridad sobre los modificadores [**/CPP \_ cmd**](-cpp-cmd.md) y [**/CPP \_ OPC**](-cpp-opt.md) .

## <a name="switch-options"></a>Opciones de conmutador

``` syntax
midl /no_cpp
midl /nocpp
```

## <a name="example"></a>Ejemplo

**MIDL/no \_ CPP nombreDeArchivo. idl**
