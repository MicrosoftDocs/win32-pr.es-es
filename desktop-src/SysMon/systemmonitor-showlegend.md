# <a name="systemmonitorshowlegend-property"></a>Propiedad SystemMonitor.ShowLegend

Recupera o establece un valor que determina si se muestra la leyenda. La leyenda enumera los contadores y muestra el color utilizado para representar cada contador.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
Property ShowLegend As Boolean
```



## <a name="property-value"></a>Valor de propiedad

True indica que la leyenda se muestra en el control ; de lo contrario, false. El valor predeterminado de esta propiedad es true.
