# <a name="mciwnd-window-class"></a>Clase Window de MCIWnd

MCIWnd es una clase de ventana para controlar dispositivos multimedia. Una biblioteca de funciones, mensajes y macros asociada a MCIWnd proporciona un método sencillo para agregar funciones de reproducción o grabación multimedia a las aplicaciones.
