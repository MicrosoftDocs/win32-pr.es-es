# <a name="updatestarted-property"></a>Propiedad UpdateStarted

El instalador establece la **propiedad UpdateStarted** cuando han comenzado los cambios en el sistema para esta instalación, incluida la reanudación de una instalación suspendida.

Las instrucciones de condición de interfaz de usuario usan este valor para seleccionar un cuadro de diálogo. Si se establece esta propiedad, se pregunta al usuario después de un error y la cancelación si desea restaurar o continuar más adelante.
