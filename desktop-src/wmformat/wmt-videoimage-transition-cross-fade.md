# <a name="wmt_videoimage_transition_cross_fade"></a>\_ \_ \_ difuminación cruzada de transición de imagen WMT \_

La transición de desvanecimiento cruzado no tiene ningún efecto especial; los miembros **fPrevBlendCoef** y **fCurrBlendCoef** de la estructura [**WMT \_ videoimage \_ SAMPLE2**](/previous-versions/windows/desktop/api/Wmsdkidl/ns-wmsdkidl-wmt_videoimage_sample2) determinan las características de la transición (disolver).

## <a name="parameters"></a>Parámetros

Esta transición no toma ningún parámetro.

## <a name="remarks"></a>Observaciones

Establecer esta transición es igual que establecer el miembro **dwEffectType** de la estructura **WMT \_ videoimage \_ SAMPLE2** en 0.
