# <a name="error-handling-error-handling"></a>Control de errores (control de errores)

Las aplicaciones bien escritas incluyen código de control de errores que les permite recuperarse correctamente frente a errores inesperados. Cuando se produce un error, la aplicación puede necesitar solicitar la intervención del usuario o puede recuperarse por sí misma. En casos extremos, la aplicación puede cerrar la sesión del usuario o apagar el sistema.

-   [Acerca del control de errores](about-error-handling.md)
-   [Usar el control de errores](using-error-handling.md)
-   [Referencia de control de errores](error-handling-reference.md)

Para obtener información sobre el control de excepciones, vea [control de excepciones estructurado](structured-exception-handling.md).
