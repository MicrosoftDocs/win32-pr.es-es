# <a name="visual-basic-sample-programs"></a>Visual Basic Programas de ejemplo

El libro *Advanced Microsoft Visual Basic 6.0, 2nd Edition* (Mandelbrot Set International, Ltd.) incluye un capítulo que describe Microsoft Active Accessibility. El capítulo proporciona código de ejemplo que muestra cómo usar elementos clave de la API Microsoft Active Accessibility API. (Microsoft Press, ISBN 1-57231-893-7) (Es posible que este libro no esté disponible en algunos idiomas y países).
