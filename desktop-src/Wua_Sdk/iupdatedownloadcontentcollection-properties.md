# <a name="iupdatedownloadcontentcollection-properties"></a>Propiedades de IUpdateDownloadContentCollection

La interfaz **IUpdateDownloadContentCollection** define las siguientes propiedades.



| Propiedad                                                       | Descripción                                                                                                                             |
|----------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontentcollection-get__newenum) | Obtiene una interfaz [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) que se utiliza para enumerar la colección.                    |
| [**Contabiliza**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontentcollection-get_count)        | Obtiene el número de elementos de la colección.                                                                                          |
| [**Elemento**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontentcollection-get_item)          | Obtiene el contenido de descarga de una actualización de una interfaz [**IUpdateDownloadContentCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadcontentcollection) . |
