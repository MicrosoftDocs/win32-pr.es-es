# <a name="wavdest-filter-sample"></a>Ejemplo de filtro WavDest

## <a name="description"></a>Descripción

El filtro WavDes escribe una secuencia de audio en un archivo WAV. Toma una sola secuencia de audio como entrada y su PIN de salida debe estar conectado al filtro del [escritor de archivos](file-writer-filter.md) . Este filtro de ejemplo se basa en la clase [**CTransformFilter**](ctransformfilter.md) .

## <a name="downloading-the-sample"></a>Descargar el ejemplo

Para descargar los ejemplos del SDK de DirectShow, instale la versión más reciente de la [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Este ejemplo se instala en la siguiente ruta de acceso: ejemplos *\[ raíz \] del SDK* \\ \\ filtros DirectShow de multimedia \\ \\ \\ WavDest.
