# <a name="callhub-object-interfaces"></a>Interfaces del objeto CallHub

El [objeto CallHub](callhub-object.md) expone métodos que recuperan información sobre los participantes en una llamada de varias entidades.

Las interfaces [**ITCallHubEvent**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhubevent) y [**IEnumCallHub**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumcallhub) no se exponen directamente en el objeto CallHub, pero están estrechamente relacionadas con ella y se enumeran aquí para facilitar la referencia.



| Interfaz                                | Descripción                                                           |
|------------------------------------------|-----------------------------------------------------------------------|
| [**ITCallHub**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhub)           | Proporciona métodos para recuperar información relativa a un objeto CallHub. |
| [**ITCallHubEvent**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhubevent) | Se usa para adquirir información relacionada con los eventos CallHub.                |
| [**IEnumCallHub**](/windows/desktop/api/tapi3if/nn-tapi3if-ienumcallhub)     | Enumera [**ITCallHub**](/windows/desktop/api/tapi3if/nn-tapi3if-itcallhub).                            |
