# <a name="oafilterstate"></a>OAFilterState

El `OAFilterState` tipo de datos define el estado del gráfico de filtro. El valor es un miembro de la enumeración de [**\_ Estado del filtro**](/windows/win32/api/strmif/ne-strmif-filter_state) . Este tipo de datos se define para los clientes de automatización OLE, como Microsoft Visual Basic 6,0.


```C++
typedef long OAFilterState;
```
