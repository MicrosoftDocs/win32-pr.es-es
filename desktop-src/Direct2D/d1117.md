# <a name="d1117-bitmap-is-not-a-target"></a>D1117: el mapa de bits no es un destino

Error de SetTarget porque el mapa de bits no tiene la opción de destino.






 

## <a name="possible-causes"></a>Causas posibles

Para establecer un ID2D1Bitmap como destino, debe crearse con la marca de destino de opciones de mapa de bits de D2D1 \_ \_ \_ . Esta marca está disponible como un campo de D2D1 \_ Bitmap \_ PROPERTIES1 usado con ID2D1DeviceContext:: CreateBitmap.

## <a name="possible-fixes"></a>Posibles correcciones

Asegúrese de que la aplicación crea su mapa de bits de destino mediante el destino de opciones de mapa de bits de D2D1 \_ \_ \_ . Si crea el mapa de bits de destino mediante ID2D1RenderTarget:: CreateBitmap, modifíquelo para usar ID2D1DeviceContext:: CreateBitmap. Cuando se especifica la marca de destino, Direct2D también comprueba que los atributos de Direct3D de la textura son compatibles con su uso como destino.
