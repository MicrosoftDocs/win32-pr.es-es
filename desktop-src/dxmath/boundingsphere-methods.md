# <a name="boundingsphere-methods"></a>Métodos BoundingSphere



| Método                                                                           | Descripción                                                                                                  |
|----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**Tuviera**](boundingsphere-contains.md)<br/>                           | Comprueba si el BoundingSphere contiene un objeto especificado.<br/>                                     |
| [**Forma intersección con**](boundingsphere-intersects.md)<br/>                       | Comprueba la intersección de BoundingSphere con un objeto.<br/>                                         |
| [**Transform**](boundingsphere-transform.md)<br/>                         | Transforma el BoundingSphere.<br/>                                                                    |
| [**ContainedBy**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>                     | Comprueba si el [**BoundingSphere**](/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere) está incluido en el frustum especificado.<br/> |
| [**CreateFromBoundingBox**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-createfromboundingbox(boundingsphere__constboundingorientedbox_))<br/> | Crea un BoundingSphere que contiene el BoundingBox especificado.<br/>                                    |
| [**CreateFromPoints**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-createfrompoints)<br/>           | Crea un nuevo BoundingSphere a partir de una lista de puntos.<br/>                                               |
| [**CreateMerged**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-createmerged)<br/>                   | Crea un BoundingSphere que contiene los dos objetos BoundingSphere especificados.<br/>                  |
| [**asignación de OP \_**](/windows/win32/api/directxcollision/nf-directxcollision-boundingsphere-operator-assign(boundingsphere__))<br/>                | Inicializa el BoundingSphere con valores de un BoundingSphere especificado.<br/>                       |
