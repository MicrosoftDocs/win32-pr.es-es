| Nombre                                                     | Descripción                                                                                                                                                                                                                                                                                                                                                                                           |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ActivateKeyboardLayout**](/windows/win32/api/winuser/nf-winuser-activatekeyboardlayout) | Establece el identificador de configuración regional de entrada (anteriormente denominado controlador de distribución de teclado) para el subproceso que realiza la llamada o el proceso actual. El identificador de configuración regional de entrada especifica una configuración regional, así como el diseño físico del teclado.<br/>                                                                                                                                                                |
| [**BlockInput**](/windows/win32/api/winuser/nf-winuser-blockinput)                         | Impide que los eventos de entrada de teclado y del mouse lleguen a aplicaciones. <br/>                                                                                                                                                                                                                                                                                                                        |

Cuando la entrada está deshabilitada, la ventana no recibe entradas, como los clics del mouse y las pulsaciones de teclas. Cuando se habilita la entrada, la ventana recibe todas las entradas.<br/>                                                                                                                                                     |
| [**GetActiveWindow**](/windows/win32/api/winuser/nf-winuser-getactivewindow)               | Recupera el identificador de ventana de la ventana activa asociada a la cola de mensajes del subproceso que realiza la llamada. <br/>                                                                                                                                                                                                                                                                                          |

| [**GetFocus**](/windows/win32/api/winuser/nf-winuser-getfocus)                             | Recupera el identificador de la ventana que tiene el foco de teclado, si la ventana está asociada a la cola de mensajes del subproceso de llamada. <br/>                                                                                                                                                                                                                                                          |
| [**GetKeyboardLayout**](/windows/win32/api/winuser/nf-winuser-getkeyboardlayout)           | Recupera el identificador de configuración regional de entrada activo (anteriormente denominado distribución del teclado) del subproceso especificado. Si el parámetro *idThread* es cero, se devuelve el identificador de configuración regional de entrada para el subproceso activo.<br/>                                                                                                                                                                           |
| [**GetKeyboardLayoutList**](/windows/win32/api/winuser/nf-winuser-getkeyboardlayoutlist)   | Recupera los identificadores de configuración regional de entrada (anteriormente denominados controladores de distribución de teclado) correspondientes al conjunto actual de configuraciones regionales de entrada en el sistema. La función copia los identificadores en el búfer especificado.<br/>                                                                                                                                                                             |
| [**GetKeyboardLayoutName**](/windows/win32/api/winuser/nf-winuser-getkeyboardlayoutnamea)   | Recupera el nombre del identificador de configuración regional de entrada activo (anteriormente denominado distribución del teclado). <br/>                                                                                                                                                                                                                                                                                           |
| [**GetKeyboardState**](/windows/win32/api/winuser/nf-winuser-getkeyboardstate)             | Copia el estado de las claves virtuales 256 en el búfer especificado. <br/>                                                                                                                                                                                                                                                                                                                        |
| [**GetKeyNameText**](/windows/win32/api/winuser/nf-winuser-getkeynametexta)                 | Recupera una cadena que representa el nombre de una clave. <br/>                                                                                                                                                                                                                                                                                                                                     |
| [**GetKeyState**](/windows/win32/api/winuser/nf-winuser-getkeystate)                       | Recupera el estado de la clave virtual especificada. El estado especifica si la tecla es hacia arriba, hacia abajo o activada (activada, desactivada y alternando cada vez que se presiona la tecla). <br/>                                                                                                                                                                                                                       |

| [**LoadKeyboardLayout**](/windows/win32/api/winuser/nf-winuser-loadkeyboardlayouta)         | Carga un nuevo identificador de configuración regional de entrada (anteriormente denominado distribución del teclado) en el sistema. Se pueden cargar varios identificadores de configuración regional de entrada a la vez, pero solo uno por proceso está activo cada vez. La carga de varios identificadores de configuración regional permite cambiar rápidamente entre ellos.<br/>                                                                                             |
| [**MapVirtualKey**](/windows/win32/api/winuser/nf-winuser-mapvirtualkeya)                   | Convierte (asigna) un código de tecla virtual en un código de recorrido o un valor de carácter, o traduce un código de recorrido en un código de tecla virtual. <br/> Para especificar un identificador para la distribución del teclado que se va a usar para traducir el código especificado, utilice la función [**MapVirtualKeyEx**](/windows/win32/api/winuser/nf-winuser-mapvirtualkeyexa) .<br/>                                                                                                |
| [**MapVirtualKeyEx**](/windows/win32/api/winuser/nf-winuser-mapvirtualkeyexa)               | Asigna un código de tecla virtual a un código de recorrido o un valor de carácter, o traduce un código de examen en un código de tecla virtual. La función traduce los códigos mediante el idioma de entrada y un identificador de configuración regional de entrada.<br/>                                                                                                                                                                                 |
| [**OemKeyScan**](/windows/win32/api/winuser/nf-winuser-oemkeyscan)                         | Asigna códigos de OEMASCII de código 0 a 0x0FF a los códigos de análisis de OEM y Estados de desplazamiento. La función proporciona información que permite a un programa enviar texto OEM a otro programa mediante la simulación de la entrada del teclado. <br/>                                                                                                                                                                                   |
| [**RegisterHotKey**](/windows/win32/api/winuser/nf-winuser-registerhotkey)                 | Define una tecla de acceso rápido para todo el sistema.<br/>                                                                                                                                                                                                                                                                                                                                                             |
| [**SendInput**](/windows/win32/api/winuser/nf-winuser-sendinput)                           | Sintetiza pulsaciones de teclas, movimientos del mouse y clics de botones.<br/>                                                                                                                                                                                                                                                                                                                                  |
| [**SetActiveWindow**](/windows/win32/api/winuser/nf-winuser-setactivewindow)               | Activa una ventana. La ventana se debe adjuntar a la cola de mensajes del subproceso de llamada. <br/>                                                                                                                                                                                                                                                                                                    |
| [**SetFocus**](/windows/win32/api/winuser/nf-winuser-setfocus)                             | Establece el foco de teclado en la ventana especificada. La ventana se debe adjuntar a la cola de mensajes del subproceso de llamada. <br/>                                                                                                                                                                                                                                                                       |
| [**SetKeyboardState**](/windows/win32/api/winuser/nf-winuser-setkeyboardstate)             | Copia una matriz de 256 bytes de Estados de tecla de teclado en la tabla de estado de entrada de teclado del subproceso que realiza la llamada. Se trata de la misma tabla a la que acceden las funciones [**GetKeyboardState**](/windows/win32/api/winuser/nf-winuser-getkeyboardstate) y [**GetKeyState**](/windows/win32/api/winuser/nf-winuser-getkeystate) . Los cambios realizados en esta tabla no afectan a la entrada del teclado en ningún otro subproceso. <br/>                                                                   |
| [**ToAscii**](/windows/win32/api/winuser/nf-winuser-toascii)                               | Convierte el código de tecla virtual especificado y el estado del teclado en el carácter o los caracteres correspondientes. La función traduce el código utilizando el idioma de entrada y la distribución de teclado física identificados por el controlador de distribución del teclado.<br/> Para especificar un identificador para la distribución del teclado que se va a usar para traducir el código especificado, utilice la función [**ToAsciiEx**](/windows/win32/api/winuser/nf-winuser-toasciiex) .<br/> |
| [**ToAsciiEx**](/windows/win32/api/winuser/nf-winuser-toasciiex)                           | Convierte el código de tecla virtual especificado y el estado del teclado en el carácter o los caracteres correspondientes. | [**ToUnicode**](/windows/win32/api/winuser/nf-winuser-tounicode)                           | Convierte el código de tecla virtual especificado y el estado del teclado en el carácter o caracteres Unicode correspondientes. <br/> Para especificar un identificador para la distribución del teclado que se va a usar para traducir el código especificado, utilice la función [**ToUnicodeEx**](/windows/win32/api/winuser/nf-winuser-tounicodeex) .<br/>                                                                                                                     |
| [**ToUnicodeEx**](/windows/win32/api/winuser/nf-winuser-tounicodeex)                       | Convierte el código de tecla virtual especificado y el estado del teclado en el carácter o caracteres Unicode correspondientes.<br/>                                                                                                                                                                                                                                                                         |

### <a name="notifications"></a>Notificaciones



| Nombre                                      | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
|-------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**activación de WM \_**](wm-activate.md)       | Se envía a la ventana que se está activando y la ventana que se está desactivando. Si las ventanas utilizan la misma cola de entrada, el mensaje se envía de forma sincrónica, primero en el procedimiento de ventana de la ventana de nivel superior que se va a desactivar y, a continuación, en el procedimiento de ventana de la ventana de nivel superior que se va a activar. Si Windows usa colas de entrada diferentes, el mensaje se envía de forma asincrónica, por lo que la ventana se activa inmediatamente. <br/>                                                                                                                               |
| [**APPCOMMAND de WM \_**](wm-appcommand.md)   | Notifica a una ventana que el usuario ha generado un evento de comando de aplicación, por ejemplo, haciendo clic en un botón de comando de la aplicación mediante el mouse o escribiendo una tecla de comando de la aplicación en el teclado.<br/>                                                                                                                                                                                                                                                                                                                                                       |
| [**carácter de WM \_**](wm-char.md)               | Se envía a la ventana con el foco de teclado cuando la función [**TranslateMessage**](/windows/desktop/api/winuser/nf-winuser-translatemessage) traduce un mensaje [**\_ KEYDOWN de WM**](wm-keydown.md) . El mensaje de caracteres de [**WM \_**](wm-char.md) contiene el código de carácter de la tecla que se presionó. <br/>                                                                                                                                                                                                                                                                             |
| [**DEADCHAR de WM \_**](wm-deadchar.md)       | Se envía a la ventana con el foco de teclado cuando la función [**TranslateMessage**](/windows/desktop/api/winuser/nf-winuser-translatemessage) traduce un mensaje de [**\_ KEYUP de WM**](wm-keyup.md) . [**WM \_ DEADCHAR**](wm-deadchar.md) especifica un código de carácter generado por una tecla muerta. Una tecla muerta es una tecla que genera un carácter, como diéresis (doble punto), que se combina con otro carácter para formar un carácter compuesto. Por ejemplo, el carácter diéresis-O () se genera escribiendo la tecla muerta para el carácter de diéresis y, a continuación, escribiendo la tecla O. <br/> |
| [**\_Hotkey (hotkey)**](wm-hotkey.md)           | Se envía cuando el usuario presiona una tecla de acceso rápido registrada por la función [**RegisterHotKey**](/windows/win32/api/winuser/nf-winuser-registerhotkey) . El mensaje se coloca en la parte superior de la cola de mensajes asociada al subproceso que registró la tecla de acceso rápido. <br/>                                                                                                                                                                                                                                                                                                                                 |
| [**KEYDOWN de WM \_**](wm-keydown.md)         | Se envía a la ventana con el foco de teclado cuando se presiona una tecla que no es del sistema. Una clave no del sistema es una tecla que se presiona cuando no se presiona la tecla ALT. <br/>                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**KEYUP de WM \_**](wm-keyup.md)             | Se envía a la ventana con el foco de teclado cuando se suelta una tecla que no es del sistema. Una clave no del sistema es una tecla que se presiona cuando no se presiona la tecla ALT o una tecla del teclado presionada cuando una ventana tiene el foco de teclado. <br/>                                                                                                                                                                                                                                                                                                                          |
| [**KILLFOCUS de WM \_**](wm-killfocus.md)     | Se envía a una ventana inmediatamente antes de que pierda el foco de teclado. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**SETFOCUS de WM \_**](wm-setfocus.md)       | Se envía a una ventana una vez que ha obtenido el foco de teclado. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| [**SYSDEADCHAR de WM \_**](wm-sysdeadchar.md) | Se envía a la ventana con el foco de teclado cuando la función [**TranslateMessage**](/windows/desktop/api/winuser/nf-winuser-translatemessage) traduce un mensaje de [**\_ SYSKEYDOWN de WM**](wm-syskeydown.md) . [**WM \_ SYSDEADCHAR**](wm-sysdeadchar.md) especifica el código de carácter de una clave muerta del sistema, es decir, una tecla muerta que se presiona mientras mantiene presionada la tecla Alt. <br/>                                                                                                                                                                                                        |
| [**SYSKEYDOWN de WM \_**](wm-syskeydown.md)   | Se envía a la ventana con el foco de teclado cuando el usuario presiona la tecla F10 (que activa la barra de menús) o mantiene presionada la tecla ALT y, a continuación, presiona otra tecla. También se produce cuando no hay ninguna ventana que tenga actualmente el foco de teclado; en este caso, el mensaje de [**\_ SYSKEYDOWN de WM**](wm-syskeydown.md) se envía a la ventana activa. La ventana que recibe el mensaje puede distinguir entre estos dos contextos comprobando el código de contexto en el parámetro *lParam* . <br/>                                                                             |
| [**SYSKEYUP de WM \_**](wm-syskeyup.md)       | Se envía a la ventana con el foco de teclado cuando el usuario suelta una tecla que se presionó mientras se mantenía presionada la tecla ALT. También se produce cuando no hay ninguna ventana que tenga actualmente el foco de teclado; en este caso, el mensaje de [**\_ SYSKEYUP de WM**](wm-syskeyup.md) se envía a la ventana activa. La ventana que recibe el mensaje puede distinguir entre estos dos contextos comprobando el código de contexto en el parámetro *lParam* . <br/>                                                                                                                           |
| [**UNICHAR de WM \_**](wm-unichar.md)         | Se envía a la ventana con el foco de teclado cuando la función [**TranslateMessage**](/windows/desktop/api/winuser/nf-winuser-translatemessage) traduce un mensaje [**\_ KEYDOWN de WM**](wm-keydown.md) . El mensaje de [**WM \_ UNICHAR**](wm-unichar.md) contiene el código de carácter de la tecla que se presionó.<br/>                                                                                                                                                                                                                                                                        |
