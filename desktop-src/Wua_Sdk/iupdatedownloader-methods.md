# <a name="iupdatedownloader-methods"></a>Métodos IUpdateDownloader

La [**interfaz IUpdateDownloader**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloader) define los métodos siguientes.



| Método                                                   | Descripción                                                                                |
|----------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [**BeginDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-begindownload) | Inicia una descarga asincrónica de los archivos de contenido asociados a las actualizaciones. |
| [**Descargar**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-download)           | Inicia una descarga sincrónica de los archivos de contenido asociados a las actualizaciones.   |
| [**EndDownload**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloader-enddownload)     | Completa una descarga asincrónica.                                                        |
