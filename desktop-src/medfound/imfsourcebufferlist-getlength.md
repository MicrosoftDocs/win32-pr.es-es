# <a name="imfsourcebufferlistgetlength-method"></a>IMFSourceBufferList:: GetLength (método)

Obtiene el número de objetos [**IMFSourceBuffer**](/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfsourcebuffer) de la lista.

## <a name="syntax"></a>Sintaxis


```C++
DWORD GetLength();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Número de búferes de origen de la lista.
