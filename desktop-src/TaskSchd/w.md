# <a name="w-task-scheduler"></a>W (Programador de tareas)

A B C D [E](e.md) F G H [I](i.md) J K L M N o [p](p.md) Q R [s](s.md) [T](t.md) U V W X Y Z

<dl> <dt>

<span id="_msb_work_items_gly"></span><span id="_MSB_WORK_ITEMS_GLY"></span>**elementos de trabajo**
</dt> <dd>

Un elemento que se puede programar mediante el servicio de Programador de tareas. Puede ser cualquier elemento que el servicio Programador de tareas se ejecute a la vez que se especifica en los desencadenadores del elemento.

Actualmente, las tareas son el único tipo de elemento de trabajo válido.

</dd> <dt>

<span id="_msb_working_directory_gly"></span><span id="_MSB_WORKING_DIRECTORY_GLY"></span>**directorio de trabajo**
</dt> <dd>

Directorio en el que Programador de tareas comenzará a ejecutar la tarea. Si no se especifica ningún directorio de trabajo, Programador de tareas ejecutará la tarea en el directorio% WINDIR% \\ system32.

</dd> </dl>
