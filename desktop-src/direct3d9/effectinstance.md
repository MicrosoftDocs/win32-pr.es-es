# <a name="effectinstance"></a>EffectInstance

Define una instancia de Effect.

``` syntax
template EffectInstance
{
    < E331F7E4-0559-4cc2-8E99-1CEC1657928F >
    STRING EffectFilename;
    [ ... ]
} 
```

Donde:

-   EffectFilename: el nombre de archivo del efecto.
-   \[ ... \] -Aquí puede usarse cualquier plantilla de archivo. x. Esto hace que la arquitectura sea extensible.
