# <a name="sb_issimple-message"></a>\_Mensaje ISSIMPLE de SB

Comprueba un control de barra de estado para determinar si está en modo simple.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>Debe ser cero.</dd> <dt>

*lParam* 
</dt> <dd>Debe ser cero.</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Devuelve un valor distinto de cero si el control de barra de estado está en modo simple o cero en caso contrario.
