# <a name="ws_metadata"></a>metadatos de WS \_

Tipo opaco que se usa para hacer referencia a un conjunto de documentos de [metadatos](metadata-import.md) .


```C++
typedef struct _WS_METADATA WS_METADATA;
```



## <a name="remarks"></a>Observaciones

Este objeto no es seguro para subprocesos. Para obtener más información, vea [seguridad para subprocesos](thread-safety.md).
