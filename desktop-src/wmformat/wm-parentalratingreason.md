# <a name="wmparentalratingreason"></a>WM/ParentalRatingReason

El atributo **WM/ParentalRatingReason** contiene los motivos de la clasificación parental asignada al contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMParentalRatingReason

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Este atributo se usa para difundir televisión y otro contenido de vídeo al que se le asigna una clasificación parental.
