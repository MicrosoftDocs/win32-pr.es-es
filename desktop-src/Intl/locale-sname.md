# <a name="locale_sname"></a>configuración regional \_ SNAME

**Windows Vista y versiones posteriores:** Nombre de la configuración regional, una etiqueta de varias partes para identificar de forma única la configuración regional. El número máximo de caracteres permitido para esta cadena es 85, incluido un carácter nulo de terminación. La etiqueta se basa en las convenciones de etiquetado de idioma de RFC 4646. El patrón que se va a usar se describe en [nombres de configuración regional](locale-names.md).
