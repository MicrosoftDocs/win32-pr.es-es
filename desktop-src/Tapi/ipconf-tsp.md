# <a name="ipconf-tsp"></a>IPConf TSP

\[IpConf TSP no está disponible para su uso en Windows Vista, Windows Server 2008 y versiones posteriores del sistema operativo. La API de cliente RTC proporciona una funcionalidad similar.\]

El TSP de IPConf (Ipconf.tsp), cuando se empareja con el MSP de [IPConf](./ipconf-msp.md) (Confmsp.dll), proporciona acceso de conferencia a redes de protocolo de Internet, como Internet y muchas intranets corporativas.

Este TSP admite un [**valor de tipo de**](./lineaddresstype--constants.md) dirección de LINEADDRESSTYPE \_ SDP.

El MSP de IPConf se ha mejorado en Windows 2000 SP1 para habilitar la comunicación entre los clientes H323 y SDP.

El TSP de IPConf se instala con Windows 2000 y versiones posteriores.
