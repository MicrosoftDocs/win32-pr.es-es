# <a name="tree-view-control-window-styles"></a>Tree-View estilos de la ventana control

En esta sección se enumeran los estilos de ventana que se usan al crear controles de vista de árbol.



| Constante                                                                                                                                                                        | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TVS_CHECKBOXES"></span><span id="tvs_checkboxes"></span><dl> <dt>**casillas de TELEVISORes \_**</dt> </dl>                | [Versión 4,70](common-control-versions.md). Habilita las casillas para los elementos de un control de vista de árbol. Solo se muestra una casilla si hay una imagen asociada al elemento. Cuando se establece en este estilo, el control usa eficazmente [**DrawFrameControl**](/windows/desktop/api/winuser/nf-winuser-drawframecontrol) para crear y establecer una lista de imágenes de estado que contiene dos imágenes. La imagen de estado 1 es la casilla desactivada y la imagen de estado 2 es la casilla activada. Si establece la imagen de estado en cero, la casilla se quitará de forma conjunta. Para obtener más información, vea [trabajar con índices de imagen de estado](tree-view-controls.md).<br/> [Versión 5,80](common-control-versions.md). Muestra una casilla incluso si no hay ninguna imagen asociada al elemento.<br/> Una vez que se crea un control de vista de árbol con este estilo, no se puede quitar el estilo. En su lugar, debe destruir el control y crear uno nuevo en su lugar. Al destruir el control de vista de árbol, no se destruye la lista de imágenes de estado de casilla. Debe destruirlo explícitamente. Obtenga el identificador de la lista de imágenes de estado enviando el control de vista de árbol a un mensaje [**TVM \_ GETIMAGELIST**](tvm-getimagelist.md) . A continuación, destruya la lista de imágenes con la anulación de [**ImageList \_**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_destroy).<br/> Si desea usar este estilo, debe establecer el estilo de las \_ casillas de los televisores con [**SetWindowLong**](/windows/desktop/api/winuser/nf-winuser-setwindowlonga) después de crear el control TreeView y antes de rellenar el árbol. De lo contrario, las casillas podrían aparecer desactivadas, en función de los problemas de temporización.<br/> |
| <span id="TVS_DISABLEDRAGDROP"></span><span id="tvs_disabledragdrop"></span><dl> <dt>**TELEVISORes \_ DISABLEDRAGDROP**</dt> </dl> | Impide que el control de vista de árbol envíe códigos de notificación de [TVN \_ BEGINDRAG](tvn-begindrag.md) . <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="TVS_EDITLABELS"></span><span id="tvs_editlabels"></span><dl> <dt>**TELEVISORes \_ EDITLABELS**</dt> </dl>                | Permite al usuario editar las etiquetas de los elementos de la vista de árbol. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| <span id="TVS_FULLROWSELECT"></span><span id="tvs_fullrowselect"></span><dl> <dt>**TELEVISORes \_ FULLROWSELECT**</dt> </dl>       | [Versión 4,71](common-control-versions.md). Habilita la selección de filas completas en la vista de árbol. La fila completa del elemento seleccionado se resalta y, al hacer clic en cualquier parte de la fila de un elemento, se selecciona. Este estilo no se puede usar junto con el **estilo \_ HASLINES de los televisores** . <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="TVS_HASBUTTONS"></span><span id="tvs_hasbuttons"></span><dl> <dt>**TELEVISORes \_ HASBUTTONS**</dt> </dl>                | Muestra los botones más (+) y menos (-) junto a los elementos primarios. El usuario hace clic en los botones para expandir o contraer la lista de elementos secundarios de un elemento primario. Para incluir botones con elementos en la raíz de la vista de árbol, \_ también se deben especificar los televisores LINESATROOT. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="TVS_HASLINES"></span><span id="tvs_haslines"></span><dl> <dt>**TELEVISORes \_ HASLINES**</dt> </dl>                      | Utiliza líneas para mostrar la jerarquía de elementos. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="TVS_INFOTIP"></span><span id="tvs_infotip"></span><dl> <dt>**recuadro \_ de TV**</dt> </dl>                         | [Versión 4,71](common-control-versions.md). Obtiene información sobre herramientas mediante el envío de la notificación de [ \_ GETINFOTIP de TVN](tvn-getinfotip.md) . <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="TVS_LINESATROOT"></span><span id="tvs_linesatroot"></span><dl> <dt>**TELEVISORes \_ LINESATROOT**</dt> </dl>             | Utiliza líneas para vincular elementos en la raíz del control de vista de árbol. Este valor se omite si \_ también se especifican los televisores HASLINES. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="TVS_NOHSCROLL"></span><span id="tvs_nohscroll"></span><dl> <dt>**TELEVISORes \_ NOHSCROLL**</dt> </dl>                   | [Versión 5,80](common-control-versions.md). Deshabilita el desplazamiento horizontal en el control. El control no mostrará ninguna barra de desplazamiento horizontal. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="TVS_NONEVENHEIGHT"></span><span id="tvs_nonevenheight"></span><dl> <dt>**TELEVISORes \_ NONEVENHEIGHT**</dt> </dl>       | [Versión 4,71](common-control-versions.md) Establece el alto de los elementos en un alto impar con el mensaje de [**\_ SETITEMHEIGHT TVM**](tvm-setitemheight.md) . De forma predeterminada, el alto de los elementos debe ser un valor par. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |

Deshabilita el desplazamiento horizontal y vertical en el control. El control no mostrará ninguna barra de desplazamiento. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="TVS_NOTOOLTIPS"></span><span id="tvs_notooltips"></span><dl> <dt>**\_información sobre herramientas de TV**</dt> </dl>                | [Versión 4,70](common-control-versions.md). Deshabilita la información sobre herramientas. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="TVS_RTLREADING"></span><span id="tvs_rtlreading"></span><dl> <dt>**TELEVISORes \_ RTLREADING**</dt> </dl>                | [Versión 4,70](common-control-versions.md). Hace que el texto se muestre de derecha a izquierda (RTL). Normalmente, Windows muestra texto de izquierda a derecha (LTR). Las ventanas se pueden *reflejar* para mostrar idiomas como el hebreo o el árabe que leen RTL. Normalmente, el texto de la vista de árbol se muestra en la misma dirección que el texto de la ventana primaria. Si \_ se establece TV RTLREADING, el texto de la vista de árbol lee en la dirección opuesta del texto de la ventana primaria. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="TVS_SHOWSELALWAYS"></span><span id="tvs_showselalways"></span><dl> <dt>**TELEVISORes \_ SHOWSELALWAYS**</dt> </dl>       | Hace que un elemento seleccionado permanezca seleccionado cuando el control de vista de árbol pierde el foco. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| <span id="TVS_SINGLEEXPAND"></span><span id="tvs_singleexpand"></span><dl> <dt>**TELEVISORes \_ SINGLEEXPAND**</dt> </dl>          | [Versión 4,71](common-control-versions.md). Hace que el elemento seleccionado se expanda y que el elemento que se va a anular la selección se contraiga al seleccionarlo en la vista de árbol. Si el mouse se usa para hacer un solo clic en el elemento seleccionado y dicho elemento está cerrado, se expandirá. Si el usuario mantiene presionada la tecla CTRL mientras selecciona un elemento, el elemento que se va a anular no se contraerá. <br/> [Versión 5,80](common-control-versions.md). Hace que el elemento seleccionado se expanda y que el elemento que se va a anular la selección se contraiga al seleccionarlo en la vista de árbol. Si el usuario mantiene presionada la tecla CTRL mientras selecciona un elemento, el elemento que se va a anular no se contraerá.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="TVS_TRACKSELECT"></span><span id="tvs_trackselect"></span><dl> <dt>**TELEVISORes \_ TRACKSELECT**</dt> </dl>             | [Versión 4,70](common-control-versions.md). Habilita el seguimiento activo en un control de vista de árbol. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
