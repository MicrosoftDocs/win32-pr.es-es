# <a name="installer-units"></a>Unidades de instalador

Una unidad de interfaz de usuario Windows Installer es aproximadamente igual a una duodécima (1/12) el alto del tamaño de fuente MS Sans Serif de 10 puntos. El tamaño relativo de los cuadros de diálogo, controles y fuentes puede cambiar en función del tamaño de fuente utilizado en la interfaz de usuario. Para garantizar la visualización correcta de la interfaz de usuario, los programadores de instalación deben probar siempre su aplicación con los tamaños de fuente reales.
