# <a name="structured-storage-client-sample-stoclien"></a>Ejemplo de cliente de almacenamiento estructurado (StoClien)

En el ejemplo **StoClien** se presenta una aplicación de dibujo simple. Esta aplicación permite al usuario usar un mouse o un dispositivo de tableta para realizar un dibujo de forma libre en la ventana del cliente. Se puede elegir el color y el ancho de la tinta electrónica y los dibujos se pueden guardar en los archivos.

Al compilar y ejecutar el ejemplo **StoClien** se proporciona la información necesaria para entender la estructura de cliente de almacenamiento estructurado, sus usos y funcionalidades.

En las secciones siguientes se proporcionan las instrucciones de ejemplo **StoClien** , la información de referencia y los vínculos al código de ejemplo **StoClien** .

-   [Operaciones de StoClien](stoclien-operations.md)
-   [Crear y ejecutar el ejemplo StoClien](create-and-run-stoclien-sample.md)
