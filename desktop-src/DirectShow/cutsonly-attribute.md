# <a name="cutsonly-attribute"></a>atributo cutsonly

> [!Note]  
> \[Obsoleto. Esta API puede quitarse de futuras versiones de Windows.\]

 

El `cutsonly` atributo especifica si una transición se representa como un corte. Si el valor es **TRUE,** la transición es un corte instantáneo. De lo contrario, la transición se representa normalmente. El valor predeterminado es **FALSE**.

## <a name="possible-values"></a>Valores posibles

Los valores siguientes se definen como TRUE: y, Y, t, T, 1. Los valores siguientes se definen como FALSE: n, N, f, F, 0 (cero).

## <a name="applies-to"></a>Se aplica a

[**Transición**](transition-element.md)
