# <a name="filesinuse-dialog"></a>Cuadro de diálogo FilesInUse

Un cuadro de diálogo FilesInUse alerta al usuario de los procesos que están ejecutando actualmente archivos que se deben sobrescribir o eliminar en la instalación. Esto ofrece al usuario la oportunidad de cerrar estos procesos y evitar tener que reiniciar el equipo para completar el reemplazo o la eliminación de estos archivos. Consulte [reinicios del sistema](system-reboots.md).

Este cuadro de diálogo se creará según sea necesario para la [acción InstallValidate](installvalidate-action.md).
