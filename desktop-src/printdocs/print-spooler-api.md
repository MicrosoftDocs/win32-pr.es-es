# <a name="print-spooler-api"></a>Print Spooler API

Print Spooler API proporciona una interfaz al colador de impresión para que las aplicaciones administren impresoras y trabajos de impresión.

La API Print Spooler la usa una aplicación como parte de su programación y no directamente por los usuarios finales.

Esta sección contiene información sobre los temas siguientes.



| Tema                                                                                             | Descripción                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Referencia de la API de Print Spooler](printing-and-print-spooler-reference.md)<br/>                | Información detallada sobre las funciones, estructuras y otros elementos de print Spooler API.<br/>                                                                                                           |
| [Referencia de notificación de impresión asincrónica](asynchronous-printing-notification.md)<br/> | Descripciones de las funciones, interfaces y enumeraciones que se usan en la comunicación asincrónica entre aplicaciones y componentes hospedados en la cola de impresión, como controladores de impresora y monitores de puerto.<br/> |
| [Referencia de instalación del controlador de impresora](printer-driver-installation-reference.md)<br/>     | Describe las funciones que instalan y configuran controladores de impresora en un equipo.<br/>                                                                                                                            |
