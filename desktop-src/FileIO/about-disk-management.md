# <a name="about-disk-management"></a>Acerca de administración de discos

El sistema de archivos proporciona una abstracción de las características físicas de los dispositivos de almacenamiento para que las aplicaciones puedan escribir y leer archivos. Sin embargo, el almacenamiento en una empresa se basa en gran medida en el concepto de discos.
