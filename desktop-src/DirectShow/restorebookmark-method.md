# <a name="restorebookmark-method"></a>Método RestoreBookmark

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

El `RestoreBookmark` método mueve el objeto **MSWebDVD** al punto del disco tal como se especifica en el marcador actual, con todos los valores de audio, vídeo y subimagen restaurados.

``` syntax
MSWebDVD.RestoreBookmark()
```

## <a name="return-value"></a>Valor devuelto

No de devuelve ningún valor.
