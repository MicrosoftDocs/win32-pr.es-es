| CN                | MS-DS-quota-usada                     |
| Nombre para mostrar de LDAP | msDS-QuotaUsed                       |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |

| Attribute-Id      | 1.2.840.113556.1.4.1849              |
