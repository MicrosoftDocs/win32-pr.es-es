# <a name="quality-of-service-in-the-windows-sockets-2-spi"></a>Calidad de servicio en Windows Sockets 2 SPI

Calidad de servicio (QoS) se implementa en Windows a través de varios componentes de QoS admitidos en Windows Server 2003, Windows XP y Windows 2000. Una explicación detallada de QoS y sus componentes se encuentra en una sección dedicada a la calidad del servicio, que se encuentra en el kit de desarrollo de software (SDK) de Microsoft Windows. Cada componente de QoS, y el rol que desempeña en la implementación de QoS, se explica en esta sección de QoS, con instrucciones adicionales que se proporcionan para la implementación de aplicaciones y servicios habilitados para QoS. Consulte la sección [calidad de servicio (QoS)](/previous-versions/windows/desktop/qos/qos-start-page) de la Windows SDK para obtener información más detallada.

En esta sección se describen las capacidades de calidad de servicio disponibles para los desarrolladores de Winsock. En la lista siguiente se describen los temas de esta sección:

-   [Modelo de uso](usage-model-2.md)
-   [Actualizaciones de QoS](qos-updates-2.md)
-   [Valores predeterminados en el SPI](default-values-in-the-spi-2.md)
-   [Plantillas de QoS en el SPI](qos-templates-in-the-spi-2.md)
