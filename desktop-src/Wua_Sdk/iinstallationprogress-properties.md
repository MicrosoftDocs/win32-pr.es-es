# <a name="iinstallationprogress-properties"></a>Propiedades de IInstallationProgress

La [**interfaz IInstallationProgress**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogress) define las siguientes propiedades.



| Propiedad                                                                                   | Descripción                                                                                                                                        |
|--------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CurrentUpdateIndex**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_currentupdateindex)                     | Obtiene un valor de índice de base cero que especifica la actualización que se está instalando o desinstalando actualmente cuando se han seleccionado varias actualizaciones. |
| [**CurrentUpdatePercentComplete**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_currentupdatepercentcomplete) | Obtiene hasta qué punto ha progresado el proceso de instalación o desinstalación de la actualización actual, como porcentaje.                                    |
| [**PercentComplete**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogress-get_percentcomplete)                           | Obtiene hasta qué punto ha progresado el proceso de instalación o desinstalación global, como porcentaje.                                                   |
