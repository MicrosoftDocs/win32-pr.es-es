# <a name="radial-controller-interfaces"></a>Interfaces de controlador radial

Los temas contenidos en esta sección proporcionan las especificaciones de referencia para las interfaces de [entrada del controlador radial](radialcontroller-portal.md) .

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|---|---|
| [**IRadialControllerConfigurationInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerconfigurationinterop)<br/> | Habilita la interoperabilidad con un objeto [**RadialControllerConfiguration**](/uwp/api/Windows.UI.Input.RadialControllerConfiguration) de plataforma universal de Windows (UWP) y proporciona acceso a los miembros de **RadialControllerConfiguration** para personalizar un menú [**RadialController**](/uwp/api/Windows.UI.Input.RadialController) .<br/> |
| [**IRadialControllerInterop**](/windows/win32/api/radialcontrollerinterop/nn-radialcontrollerinterop-iradialcontrollerinterop)<br/> | Habilita la interoperabilidad con un objeto [**RadialController**](/uwp/api/Windows.UI.Input.RadialController) de UWP y proporciona acceso a los miembros de **RadialController** para personalizar la experiencia de interacción.<br/> |
