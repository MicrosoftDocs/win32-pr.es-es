# <a name="d1145-print-control-dpi-incorrect"></a>D1145: PPP del control de impresión incorrecto

Impresión de Direct2D: \_ propiedades del control de impresión \_ . el valor de rasterDPI no es válido.






 

## <a name="possible-causes"></a>Causas posibles

Durante la creación de un control de impresión, este mensaje indica que la aplicación especificó un PPP no válido; en concreto, un DPI de menos de 1,0. Aunque Direct2D no considera que esto sea un error, la impresión en Direct2D se comportará como si la aplicación especificara un DPI de 150,0.

## <a name="possible-fixes"></a>Posibles correcciones

Para abordar este mensaje, use un PPP de 1,0 o superior mientras crea un control de impresión.
