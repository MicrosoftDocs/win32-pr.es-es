# <a name="encoding-windows-web-services"></a>Codificación (Windows Web Services)

La codificación es una transformación entre XML y un formato de conexión lineal. Lacoding es la transformación inversa.


Una codificación se identifica mediante una [**constante WS \_ ENCODING.**](/windows/desktop/api/WebServices/ne-webservices-ws_encoding)
