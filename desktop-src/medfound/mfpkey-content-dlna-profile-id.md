# <a name="mfpkey_content_dlna_profile_id-property"></a>Propiedad MFPKEY \_ Content \_ DLNA \_ Profile \_ ID

Contiene el identificador de perfil de Digital Living Network Alliance (DLNA) para un origen multimedia.



Tipo de datos

Tipo PROPVARIANT (vt)

Miembro de PROPVARIANT

**wchar \_ t\***

VT \_ LPWSTR

**pwszVal**



## <a name="remarks"></a>Comentarios

Un identificador de perfil DLNA es una cadena que identifica el perfil de codificación del contenido.

Los identificadores de perfil se describen en [dlna Networked Device Interoperability Guidelines (Directrices de](https://www.dlna.org/about/organization)interoperabilidad de dispositivos en red de DLNA). (Es posible que este recurso no esté disponible en algunos idiomas y países).
