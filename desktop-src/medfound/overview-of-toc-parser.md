# <a name="overview-of-table-of-contents-parser"></a><span data-ttu-id="e3284-103">Información general sobre el analizador de tabla de contenido</span><span class="sxs-lookup"><span data-stu-id="e3284-103">Overview of Table of Contents Parser</span></span>

<span data-ttu-id="e3284-104">El analizador de tabla de contenido (analizador de TDC) es un componente de Windows que permite almacenar información de índice avanzada en archivos multimedia.</span><span class="sxs-lookup"><span data-stu-id="e3284-104">Table of Contents Parser (TOC Parser) is a Windows component that enables you to store advanced index information in media files.</span></span> <span data-ttu-id="e3284-105">Ejemplos de información de índice avanzada son tablas de contenido, índices semánticos y paneles de casos.</span><span class="sxs-lookup"><span data-stu-id="e3284-105">Examples of advanced index information are tables of contents, semantic indices, and story boards.</span></span>

<span data-ttu-id="e3284-106">Con el analizador de TDC, puede escribir software que permita a los usuarios buscar escenas de interés en un archivo de vídeo.</span><span class="sxs-lookup"><span data-stu-id="e3284-106">With TOC Parser, you can write software that enables users to find scenes of interest in a video file.</span></span> <span data-ttu-id="e3284-107">Por ejemplo, puede proporcionar al usuario una tabla de contenido para que pueda identificar las secciones principales del vídeo y, a continuación, ver una sección concreta.</span><span class="sxs-lookup"><span data-stu-id="e3284-107">For example, you can provide the user with a table of contents so that he or she can identify the primary sections of the video and then view a particular section.</span></span> <span data-ttu-id="e3284-108">También puede proporcionar al usuario un índice para que pueda encontrar una lista de escenas relacionadas con una palabra o frase en particular.</span><span class="sxs-lookup"><span data-stu-id="e3284-108">You can also provide the user with an index so that he or she can find a list of scenes that are related to a particular word or phrase.</span></span>
