# <a name="systemdevicesaepservicebluetoothgattservicedevice"></a>System.Devices.AepService.Bluetooth.Service.Device

Establece la dirección del dispositivo Bluetooth que se consulta.

## <a name="windows-10-version-1511"></a>Windows 10, versión 1511

```
propertyDescription
   name = System.Devices.AepService.Bluetooth.GattService.Device
   shellPKey = PKEY_Devices_AepService_Bluetooth_GattService_Device
   formatID = 9744311E-7951-4B2E-B6F0-ECB293CAC119
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt64
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey.h.
