# <a name="predefined-language-bar-items"></a>Elementos de barra de idioma predefinidos

Los siguientes valores de [**GUID**](/windows/desktop/TSF/predefined-lang-bar-items) identifican los elementos de barra de lenguaje predefinidos. Estos valores se usan con [**GetItem**](/windows/desktop/api/Ctfutb/nf-ctfutb-itflangbaritemmgr-getitem) para obtener un elemento específico de la barra de idioma.

<dl> <dt>

<span id="_______________GUID_LBI_SAPILAYR_CFGMENUBUTTON_____________"></span><span id="_______________guid_lbi_sapilayr_cfgmenubutton_____________"></span> GUID \_ LBI \_ SAPILAYR \_ CFGMENUBUTTON 
</dt> <dd>

Elemento de menú de la barra de idioma de herramientas de voz.

</dd> <dt>

<span id="_______________GUID_TFCAT_TIP_HANDWRITING_____________"></span><span id="_______________guid_tfcat_tip_handwriting_____________"></span>\_escritura a \_ mano de sugerencia de GUID TFCAT \_ 
</dt> <dd>

Elemento de tipo de escritura a mano.

</dd> <dt>

<span id="_______________GUID_TFCAT_TIP_KEYBOARD_____________"></span><span id="_______________guid_tfcat_tip_keyboard_____________"></span>teclado de sugerencia de GUID \_ TFCAT \_ \_ 
</dt> <dd>

Elemento de tipo de teclado.

</dd> </dl>
