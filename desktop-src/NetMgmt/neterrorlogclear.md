# <a name="neterrorlogclear"></a>NetErrorLogClear

La función **NetErrorLogClear** está obsoleta. Se incluye por compatibilidad con las versiones de 16 bits de Windows. Otras aplicaciones deben usar el [registro de eventos](/windows/desktop/EventLog/event-logging).
