# <a name="www-page-other-attribute"></a>WWW-página-otro atributo

Una lista de páginas web alternativas.



| Entrada | Value |
|-------------------|-----------------------------------------------------------------------------|
| CN                | WWW-página-otro                                                              |
| Nombre para mostrar de LDAP | url                                                                         |
| Tamaño              | \-                                                                          |
| Actualizar privilegio  | Administrador de dominio o propietario de la cuenta.                                      |
| Frecuencia de actualización  | Cuando se crea el registro del usuario y cada vez que es necesario cambiar la Página Web. |
| Attribute-Id      | 1.2.840.113556.1.4.749                                                      |
| System-ID-GUID    | 9a9a0221-4a5b-11d1-a9c3-0000f80367c1                                        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md)                                 |
