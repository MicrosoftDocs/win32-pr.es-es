# <a name="capture-driver-capabilities"></a>Capacidades del controlador de captura

Puede recuperar las funcionalidades de hardware del controlador de captura conectado actualmente mediante el mensaje [**GET \_ \_ \_ \_ CAPS**](wm-cap-driver-get-caps.md) del CONTROLADOR DE WM CAP (o la macro [**capDriverGetCaps).**](/windows/desktop/api/Vfw/nf-vfw-capdrivergetcaps) Este mensaje devuelve las funciones del controlador de captura y el hardware subyacente en la [**estructura CAPDRIVERCAPS.**](/windows/win32/api/vfw/ns-vfw-capdrivercaps)
