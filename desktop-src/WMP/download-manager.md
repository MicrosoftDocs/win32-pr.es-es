# <a name="download-manager"></a>Administrador de descargas

El administrador de descargas de Microsoft Windows Media Player permite a las tiendas en línea de tipo 2 controlar el proceso de descarga de contenido desde un panel de tareas de la tienda Windows Media Player online. Un panel de tareas de la tienda en línea hospeda una página web que se muestra cuando el usuario elige usar una tienda en línea.

Esta introducción al administrador de descargas contiene las siguientes secciones.

|--------------------------------------------------------------------|-----------------------------------------------------------------|
| [Información general del administrador de descargas](download-manager-overview.md)         | Presenta el administrador de descargas y algunos escenarios de uso básico. |
| [Arquitectura de Download Manager](download-manager-architecture.md) | Proporciona un resumen del modelo de objetos del administrador de descargas.        |
