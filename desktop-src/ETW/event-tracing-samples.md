# <a name="event-tracing-samples"></a>Ejemplos de seguimiento de eventos

El kit de desarrollo de software (SDK) de plataforma contiene ejemplos completos de seguimiento de eventos. Estos ejemplos se encuentran en el directorio Samples \\ Winbase \\ eventtracer La raíz de esta ruta de acceso es el directorio de instalación base del PSDK. Puede descargar el PSDK desde [https://www.microsoft.com/downloads/details.aspx?FamilyId=A55B6B43-E24F-4EA3-A93E-40C0EC4F68E5](https://msdn.microsoft.com/windowsserver/bb980924.aspx) .

En la tabla siguiente se describen los ejemplos contenidos en el directorio Samples \\ Winbase \\ eventtracer.



| Muestra   | Descripción                                                                                                                            |
|----------|----------------------------------------------------------------------------------------------------------------------------------------|
| TraceDmp | Consumidor de seguimiento de eventos. Descodifica los datos de evento mediante la información de formato obtenida de WMI y genera los datos en un archivo. csv. |
| TraceDp  | Utiliza la API del proveedor de seguimiento de eventos para proporcionar datos de seguimiento de eventos al registrador o al consumidor.                                             |
| TraceLog | Utiliza la API del controlador de seguimiento de eventos para administrar las sesiones de registro.                                                                        |



 

Para obtener descripciones detalladas, vea el archivo Readme.txt incluido con estos ejemplos.
