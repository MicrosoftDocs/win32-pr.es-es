# <a name="ipropertybag-and-ipersistpropertybag"></a>IPropertyBag y IPersistPropertyBag

[**IPropertyBag**](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa768196(v=vs.85)) y [**IPersistPropertyBag**](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa768205(v=vs.85)) optimizan los mecanismos de guardar como texto y, por tanto, se recomiendan para los contenedores de controles ActiveX que implementan un mecanismo de texto guardar como. **IPropertyBag** lo implementa un contenedor y es aproximadamente análogo a [**IStream**](/windows/desktop/api/objidl/nn-objidl-istream). **IPersistPropertyBag** se implementa mediante controles y es aproximadamente análogo a [**IPersistStream**](/windows/desktop/api/ObjIdl/nn-objidl-ipersiststream).
