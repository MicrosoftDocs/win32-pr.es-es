# <a name="multicast-group"></a>Grupo de multidifusión

Un grupo de multidifusión es una dirección IP de clase D (es decir, en el intervalo de 224.0.0.0- 239.255.255.255). Los datos de multidifusión que se envían a una dirección de este intervalo no están destinados a un único destino. En su lugar, cualquier host que realiza una solicitud para unirse a un grupo específico puede recibir estos datos. El símbolo G o g hace referencia a un grupo de multidifusión.
