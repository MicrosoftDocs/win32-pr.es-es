# <a name="playlistindex-attribute"></a>Atributo PlaylistIndex

El **atributo PlaylistIndex** es el índice del elemento multimedia de su lista de reproducción primaria.

## <a name="applies-to"></a>Se aplica a

-   [Elementos de audio](audio-item-attributes.md)
-   [Pistas de CD](cd-track-attributes.md)
-   [Otros elementos](other-item-attributes.md)
-   [Elementos de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Comentarios

Este atributo es un atributo temporal. No se almacena en la biblioteca ni en el archivo multimedia digital.

Los elementos multimedia pueden ser elementos de lista de reproducción. Este atributo es el índice de base cero que representa la posición del elemento multimedia en la lista de reproducción. Si el elemento multimedia no forma parte de una lista de reproducción, el valor de este atributo es una cadena vacía ("").
