# <a name="importing-and-exporting-drm-protected-content"></a>Importación y exportación de contenido protegido con DRM

Uno de los roles principales del nuevo cliente DRM de Windows Media Extended API es habilitar la importación y exportación de contenido protegido con DRM en sistemas de administración de derechos de terceros.

El principal desafío de importar y exportar datos protegidos en la aplicación es compartir sólidamente ejemplos de datos ASF entre el sistema y el subsistema DRM. Cuando la aplicación controla ejemplos de datos, tiene que asegurarse de que se cumplen determinados estándares de solidez para evitar la posibilidad de que se eluda el mecanismo de protección. Estos estándares se describen en las reglas de solidez y cumplimiento con las que está de acuerdo al otorgar licencias de DRM.
