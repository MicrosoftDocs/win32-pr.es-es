# <a name="bluetooth-and-getpeername"></a>Bluetooth y getpeername

La [**función getpeername**](/windows/desktop/api/winsock/nf-winsock-getpeername) se usa para recuperar la Bluetooth del dispositivo del Bluetooth del mismo nivel. La **función getpeername** solo es válida para los sockets Bluetooth conexión.

Para obtener más información, [vea Windows Sockets](/windows/desktop/WinSock/windows-sockets-start-page-2).
