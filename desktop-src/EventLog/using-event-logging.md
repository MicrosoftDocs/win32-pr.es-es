# <a name="using-event-logging"></a>Uso del registro de eventos

Las aplicaciones pueden usar la API de registro de eventos para notificar y ver los eventos. Para más información sobre los eventos de informes, consulte [eventos de informes](reporting-an-event.md).

Para ver los eventos que se han escrito en los archivos de registro, consulte [consultar mensajes de origen de eventos](querying-for-event-source-messages.md) y [recibir notificaciones de eventos](receiving-event-notification.md). También puede usar el Visor de eventos para ver los eventos.
