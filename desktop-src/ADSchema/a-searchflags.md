## <a name="remarks"></a>Observaciones

Este atributo puede ser cero o una combinación de uno o varios de los valores siguientes.



| Value            | Descripción                                                                                                                                                                                                                                                                                                                                                               |
|------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 (0x00000001)   | Cree un índice para el atributo.                                                                                                                                                                                                                                                                                                                                        |
| 2 (0x00000002)   | Cree un índice para el atributo en cada contenedor.                                                                                                                                                                                                                                                                                                                      |
| 4 (0x00000004)   | Agregue este atributo al conjunto de resolución de nombres ambiguos (ANR). Se utiliza para ayudar a buscar un objeto cuando solo se proporciona información parcial. Por ejemplo, si el filtro LDAP es (ANR = JEFF), la búsqueda buscará todos los objetos en los que el nombre, el apellido, la dirección de correo electrónico u otro atributo ANR sea igual a Juan. El bit 0 debe estar establecido para que este índice surta efecto. |
| 8 (0x00000008)   | Conserva este atributo en el objeto de desecho para los objetos eliminados.                                                                                                                                                                                                                                                                                                      |
| 16 (0x00000010)  | Copie el valor de este atributo cuando se copie el objeto.                                                                                                                                                                                                                                                                                                              |
| 32 (0x00000020)  | Cree un índice de tupla para el atributo. Esto mejorará las búsquedas en las que el carácter comodín aparece al principio de la cadena de búsqueda. Por ejemplo, (SN = \* mith).                                                                                                                                                                                                                |
| 64 (0x00000040)   | Se admite a partir de ADAM. Crea un índice para ayudar considerablemente al rendimiento de VLV en atributos arbitrarios.                                                                                                                                                                                                                                                                  |
| 128 (0x00000080) | Marque el atributo como confidencial. Se omite para los atributos de esquema base (systemFlags = 0x10).                                                                                                                                                                                                                                                                                    |
| 64 (0x00000040)  | Se admite a partir de Windows Server 2008. Cree un índice para mejorar el rendimiento de las búsquedas de VLV en este atributo.                                                                                                                                                                                                                                                        |
