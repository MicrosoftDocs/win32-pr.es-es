# <a name="dns-resolvers"></a>Solucionadores de DNS

El [*solucionador*](r-gly.md) es un componente de software incluido en la pila de protocolo de control de transmisión/Protocolo de Internet (TCP/IP) de Windows 2000 que se comunica con los servidores DNS para crear y resolver consultas de resolución de nombres. El paquete también está disponible en cualquier cliente que no sea de Microsoft compatible con DNS. Existen resoluciones en clientes DNS y servidores DNS.
