# <a name="cdynamicoutputpinstopusingoutputpin-method"></a>CDynamicOutputPin. StopUsingOutputPin, método

El `StopUsingOutputPin` método libera el acceso al pin después de una operación de streaming.

## <a name="syntax"></a>Sintaxis


```C++
virtual void StopUsingOutputPin();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
