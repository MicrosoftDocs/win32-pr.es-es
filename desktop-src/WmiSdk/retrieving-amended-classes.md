# <a name="retrieving-amended-classes"></a>Recuperar clases modificadas

Puede recuperar y modificar una clase modificada especificando una configuración regional y estableciendo una marca que indica que desea información localizada. El procedimiento usado para recuperar definiciones de clases localizadas depende de cómo se conecte a WMI. Para obtener más información, vea Recuperar clases modificadas mediante [scripting API](retrieving-amended-classes-using-the-scripting-api-for-wmi.md) para WMI y Recuperar clases modificadas mediante la API COM para [WMI.](retrieving-amended-classes-using-the-com-api-for-wmi.md)
