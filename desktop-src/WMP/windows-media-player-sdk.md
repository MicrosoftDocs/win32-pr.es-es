# <a name="windows-media-player-sdk"></a>SDK de Media Player de Windows

En esta documentación se describe el kit de desarrollo de software (SDK) de Microsoft Windows Media Player. El SDK de Media Player de Windows es uno de los componentes de la Microsoft Windows SDK. Otros componentes multimedia incluyen el SDK de Microsoft Media Foundation, el SDK de Windows Media Format de Microsoft y el SDK de Microsoft Windows Media Services.

El SDK de Windows Media Player documenta las tecnologías de programación que se pueden usar para ampliar las capacidades de Windows Media Player y Windows Media Player Mobile. Estas tecnologías se documentan en las siguientes secciones:



| Sección                                                                      | Descripción                                                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Acerca del SDK de Windows Media Player](about-the-windows-media-player-sdk.md) | En esta sección se proporcionan detalles sobre cómo buscar información específica en el SDK. Incluye una sección sobre nuevas características e información sobre cómo usar los ejemplos incluidos con el SDK.                                                                                                                                     |
| [Modelo de objetos de Media Player de Windows](windows-media-player-object-model.md)   | Microsoft Windows Media Player control es un control ActiveX de Microsoft que se usa para agregar capacidades de reproducción de medios digitales a páginas Web. Proporciona una interfaz de programación para representar archivos multimedia digitales y secuencias.                                                                                                     |
| [Máscaras de Windows Media Player](windows-media-player-skins.md)                 | Las máscaras son una tecnología basada en XML que se usa para personalizar la interfaz de usuario de Windows Media Player. También puede usar las máscaras móviles de Windows Media Player para personalizar la interfaz de usuario de Windows Media Player Mobile.                                                                                                                |
| [Complementos de Media Player de Windows](windows-media-player-plug-ins.md)           | Los complementos son objetos que extienden la funcionalidad de Media Player de Windows de varias maneras. Entre los tipos de complementos se incluyen visualizaciones personalizadas, Complementos de la interfaz de usuario, Complementos DSP y complementos de representación. En esta sección también se describe la compatibilidad con complementos móviles de Windows Media Player.                                              |
| [Metaarchivos de Windows Media](windows-media-metafiles.md)                       | Los metaarchivos de Windows Media son documentos XML que proporcionan información sobre un elemento multimedia y su presentación. Los metaarchivos se pueden utilizar para organizar los elementos multimedia en listas de reproducción que pueden incluir funciones para la conmutación por secuencias sin problemas, la inserción de anuncios y otras características.                                                          |
| [Listas de reproducción de Windows Media](windows-media-playlists.md)                       | Las listas de reproducción son archivos que usan elementos XML para definir una lista de reproducción "inteligente" dinámica de elementos multimedia o un conjunto estático de elementos multimedia.                                                                                                                                                                                              |
| [Windows Media Player tiendas en línea](windows-media-player-online-stores.md) | Windows Media Player proporciona una funcionalidad que permite a los proveedores de contenido multimedia digital integrar sus servicios con Windows Media Player. La integración entre el reproductor y un almacén multimedia en línea permite al usuario buscar contenido, descargar y administrar archivos, reproducir contenido y copiar contenido en CDs o dispositivos. |
| [Reproductor de Windows Media](windows-media-player.md)                             | Algunas características del SDK se aplican a Windows Media Player, el control ActiveX de Windows Media Player y Windows Media Player Mobile. En esta sección se proporciona información acerca de estas características.                                                                                                                                        |
| [Glosario](glossary.md)                                                     | Esta sección contiene definiciones de términos que se usan en todo el SDK.                                                                                                                                                                                                                                                               |



 

> [!Note]  
> La instalación del Windows SDK no instala Windows Media Player ni Windows Media Player Mobile. Debe tener Windows Media Player instalado para usar el material de este SDK. Windows Media Player Mobile forma parte de Windows Mobile, que solo está disponible en Pocket PC o smartphone proporcionado por un fabricante de dispositivos o un operador de telefonía móvil.
