# <a name="__securityrelatedclass-class"></a>\_\_Clase SecurityRelatedClass

La clase de sistema abstracta **\_ \_ SecurityRelatedClass** actúa como clase primaria para todos los tipos de clases de seguridad. Nunca se crean instancias de esta clase y todas las clases de seguridad deben derivarse de esta clase base.

La siguiente sintaxis se simplifica desde el código de Managed Object Format (MOF) e incluye todas las propiedades heredadas. Las propiedades se enumeran en orden alfabético, no en orden MOF.

## <a name="syntax"></a>Sintaxis

``` syntax
[abstract]
class __SecurityRelatedClass
{
};
```

## <a name="members"></a>Miembros

La clase **\_ \_ SecurityRelatedClass** no define ningún miembro.
