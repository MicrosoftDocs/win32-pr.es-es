# <a name="audio-mixer-custom-controls"></a>Controles personalizados del mezclador de audio

Los controles personalizados son el más genérico de los controles de mezclador. Estos controles permiten que un controlador de mezclador defina las características del control y, implícitamente, la representación visual del control.
