# <a name="sample-trackbars-section"></a>Sección Trackbars de ejemplo

Las líneas siguientes muestran una sección trackbars típica de un archivo de definición de máscara:


```C++
[ Trackbars ]

//  <Type>      <Location>     <Dis Image Src>  <Thumb Image Src>  <Thumb Size>
//  ----------  ----------     ---------------  -----------------  ------------
    Seek      32,220,172,23  Disabled @ 32,27    SeekThumb.bmp    23,22

```



En el ejemplo anterior se define una barra de seguimiento que permite al usuario buscar a través de un elemento multimedia pulsando y arrastrando la imagen SeekThumb.
