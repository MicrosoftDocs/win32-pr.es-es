# <a name="actionid-property"></a>Propiedad Action.Id

En el caso de scripting, obtiene o establece el identificador de la acción.

## <a name="syntax"></a>Sintaxis


```VB
Action.Id As String
```



## <a name="property-value"></a>Valor de propiedad

Identificador definido por el usuario para la acción. El Programador de tareas usa este identificador para el registro.

## <a name="remarks"></a>Observaciones

Para obtener información sobre cómo funcionan conjuntamente las acciones y tareas, vea [acciones de tarea](task-actions.md).
