# <a name="ui_pkey_categoryid"></a>UI \_ PKEY \_ CategoryID

Identifica la \_ propiedad PKEY CategoryID de la interfaz de usuario \_ .

```
propertyDescription
   name = UI_PKEY_CategoryId
   shellPKey = UI_PKEY_CategoryId
   formatID = 00000103-7363-696e-8441798acf5aebb7
   propID = 103
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Observaciones

\_ \_ Una aplicación usa la interfaz de usuario PKEY CategoryID para consultar los identificadores de las categorías que se usan para agrupar elementos relacionados en un control de la galería.

El valor de la propiedad es el índice de base cero de un elemento de una colección de categorías.

En la captura de pantalla siguiente se muestra el uso de categorías en un menú de [**splitButton**](windowsribbon-element-splitbutton.md) donde las **formas de selección** tienen un identificador de 0 y **las opciones de selección** tienen un identificador de 1.
