# <a name="creating-an-instance-of-the-control"></a>Crear una instancia del control

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

El acceso al control del agente de Microsoft requiere que primero se cree una instancia del control. La forma más fácil de hacerlo es colocar una instancia del control en un formulario. Es posible que tenga que agregar el control al cuadro de herramientas antes de agregarlo al formulario. Dependiendo del lenguaje de programación, es posible que también pueda crear el control en tiempo de ejecución.
