# <a name="tb_getpressedimagelist-message"></a>\_Mensaje GETPRESSEDIMAGELIST TB

Obtiene la lista de imágenes que usa un control de barra de herramientas para mostrar los botones en estado presionado.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>

Debe ser cero.

</dd> <dt>

*lParam* 
</dt> <dd>

Debe ser cero.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Devuelve el identificador de la lista de imágenes o **null** si no se ha establecido ninguna lista de imágenes.
