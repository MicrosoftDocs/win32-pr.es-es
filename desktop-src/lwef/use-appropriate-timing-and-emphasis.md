# <a name="use-appropriate-timing-and-emphasis"></a>Usar el control de tiempo y el énfasis adecuados

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Al igual que todos los comentarios, la eficacia de la salida de voz depende de la temporización y el énfasis. Se puede comunicar una buena cantidad de información en el ritmo, el volumen y el tono utilizados cuando se habla algo. Si usa un motor de conversión de texto a voz como voz del carácter, la mayoría de los motores permiten establecer la velocidad, las pausas, el tono y el énfasis de palabras o frases. Puede usar estos atributos para indicar el interés y el conocimiento del carácter, así como dirigir la atención del usuario o indicar el intento del carácter. Para obtener más información sobre cómo establecer los atributos de voz, consulte [etiquetas de salida de voz del agente de Microsoft](microsoft-agent-speech-output-tags.md). Si usa archivos de sonido como salida del carácter, tenga en cuenta estos factores también en el audio grabado.
