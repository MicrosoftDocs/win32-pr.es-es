# <a name="tapi-service-providers"></a>Proveedores de servicios TAPI

Los proveedores de servicios TAPI son una capa de abstracción entre las aplicaciones TAPI y los protocolos de hardware y transporte subyacentes. Un [proveedor de servicios de](telephony-service-providers-start-page.md) telefonía (TSP) proporciona controles de llamada y un proveedor de servicios multimedia (MSP) proporciona control detallado sobre los medios de una llamada. [](./media-service-providers-start-page.md) Un par TSP/MSP puede implementar las funcionalidades necesarias para aplicaciones que van desde la telefonía básica hasta la conferencia multimedia de multidifusión IP.
