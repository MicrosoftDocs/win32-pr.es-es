# <a name="about-common-controls"></a>Acerca de los controles comunes

En este tema se describen los controles comunes, un conjunto de ventanas implementadas por la biblioteca de controles comunes, Comctl32.dll, que es un archivo DLL incluido en el sistema operativo Windows. Al igual que otras ventanas de control, un control común es una ventana secundaria que una aplicación usa junto con otra ventana para habilitar la interacción con el usuario.
