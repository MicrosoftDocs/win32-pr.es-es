# <a name="using-hardware-autoplay"></a>Uso de la reproducción automática de hardware

La reproducción automática es una nueva característica aprenderá de Windows XP que detecta contenido como imágenes, música o archivos de vídeo en medios extraíbles y dispositivos extraíbles. La reproducción automática inicia las aplicaciones automáticamente para reproducir o mostrar el contenido. Esto simplifica el uso de dispositivos periféricos especializados, como reproductores de MPEG-Layer 3 (MP3) y lectores de fotografías digitales. También facilita a los usuarios que no están familiarizados con el software necesario para tener acceso a varios tipos de contenido.

Se incluye una selección ampliada de iconos, incluidos los iconos de personalización de marca para los principales fabricantes, para ofrecer una mejor diferencia visual de las unidades y otros dispositivos conectados en **mi PC**. Los iconos de dispositivos con medios extraíbles, con la excepción de las unidades de disquete, se proporcionan en las versiones listo y no listo para indicar el estado actual del dispositivo. Esto confirma que un dispositivo conectado en caliente en el sistema está listo para su uso y ayuda a evitar los mensajes de "dispositivos no listos" de los dispositivos sin elementos multimedia insertados.

> [!Note]  
> La reproducción automática no debe confundirse con la ejecución automática. La ejecución automática permite que un disco compacto inicie automáticamente una función, como un programa de instalación o una reproducción de juegos, cuando se coloca en la unidad de CD-ROM. Esto se logra mediante el uso de un archivo Autorun. inf en el directorio raíz del CD. Para obtener más información, consulte [crear una aplicación de CD-ROM habilitada para Autorun](autoplay.md).
