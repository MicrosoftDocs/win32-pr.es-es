<span data-ttu-id="44996-104">El término *dispositivo de extremo* hace referencia a un dispositivo de hardware que se encuentra en un extremo de una ruta de acceso de datos que se origina o finaliza el dispositivo de extremo.</span><span class="sxs-lookup"><span data-stu-id="44996-106">The audio data that moves along the data path might traverse a number of software and hardware components during <span data-ttu-id="44996-123">Algunos dispositivos de extremo pueden conectarse permanentemente a un dispositivo de adaptador.</span><span class="sxs-lookup"><span data-stu-id="44996-123">Some endpoint devices might permanently connect to an adapter device.</span></span> <span data-ttu-id="44996-124">Por ejemplo, un equipo puede contener dispositivos internos como un reproductor de CD, un micrófono o altavoces que están integrados en el chasis del sistema.</span><span class="sxs-lookup"><span data-stu-id="44996-124">For example, a computer might contain internal devices such as a CD player, a microphone, or speakers that are integrated into the system chassis.</span></span> <span data-ttu-id="44996-125">Normalmente, el usuario no quita físicamente estos dispositivos de extremo.</span><span class="sxs-lookup"><span data-stu-id="44996-125">Typically, the user does not physically remove these endpoint devices.</span></span>

<span data-ttu-id="44996-126">Es posible que otros dispositivos de extremo se conecten a un adaptador de audio a través de conectores de audio.</span><span class="sxs-lookup"><span data-stu-id="44996-126">Other endpoint devices might connect to an audio adapter through audio jacks.</span></span> <span data-ttu-id="44996-127">El usuario conecta y desconecta estos dispositivos externos.</span><span class="sxs-lookup"><span data-stu-id="44996-127">The user plugs in and unplugs these external devices.</span></span> <span data-ttu-id="44996-128">Por ejemplo, un dispositivo de punto de conexión de audio, como un micrófono externo o auriculares, se encuentra en un extremo de un cable cuyo otro extremo se conecta a un conector en un dispositivo adaptador.</span><span class="sxs-lookup"><span data-stu-id="44996-128">For example, an audio endpoint device such as an external microphone or headphones lies at one end of a cable whose other end plugs into a jack on an adapter device.</span></span>

del chasis del equipo.</span><span class="sxs-lookup"><span data-stu-id="44996-137">Registering endpoint devices makes it easier for user-friendly applications to enable users to refer to the endpoint conector.</span><span class="sxs-lookup"><span data-stu-id="44996-138">The endpoint devices that are reported by the operating system faithfully track dynamic changes in the configuration of audio
