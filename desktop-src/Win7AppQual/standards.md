# <a name="standards"></a>Estándares

Windows Internet Explorer 8 contiene un nuevo motor de representación y diseño que implementa por completo CSS 2,1 e incluye compatibilidad adicional con HTML 4,0 (a diferencia de las versiones anteriores de Windows Internet Explorer). La compatibilidad más amplia de los estándares web en Internet Explorer 8 podría hacer que algunas aplicaciones web no funcionen correctamente porque no están escritas para Microsoft Internet Explorer 6 o de una manera no compatible con los estándares. Se puede producir un error en las aplicaciones de varias maneras, incluidas las imágenes o el texto que se muestran en diferentes ubicaciones o aplicaciones enteras que se representan de forma que la aplicación no se pueda usar.

Internet Explorer 8 también incluye un motor de JavaScript mejorado. Este nuevo motor de scripting mejora el rendimiento, admite nuevos métodos de forma nativa, proporciona una mayor seguridad y cambia algunas funciones para que funcionen de una manera más compatible con los estándares. El rendimiento mejorado no introduce problemas de compatibilidad de aplicaciones, pero otros cambios pueden hacer que algunas aplicaciones web dejen de funcionar según lo previsto. Para obtener más información sobre los problemas de compatibilidad relacionados con los cambios de JavaScript, vea [compatibilidad de sitios e Internet Explorer 8](/archive/blogs/ie/site-compatibility-and-ie8).
