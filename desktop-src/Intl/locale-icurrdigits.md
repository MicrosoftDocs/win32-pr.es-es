# <a name="locale_icurrdigits"></a>CONFIGURACIÓN REGIONAL \_ ICURRDIGITS

Número de dígitos fraccionarios para el formato monetario local. El número máximo de caracteres permitido para esta cadena es dos, incluidos los valores 0-9 y un carácter nulo final. Por ejemplo, 200,00 tiene un valor de 2 porque hay dos dígitos fraccionarios, mientras que 200 tiene un valor de 0 porque no se muestran dígitos fraccionarios.
