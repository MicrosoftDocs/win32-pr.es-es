# <a name="acs-max-aggregate-peak-rate-per-user-attribute"></a>ACS-Max-Aggregate-tasa máxima por atributo de usuario

La tasa máxima de picos que puede tener cualquier usuario para todos los flujos.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | ACS-Max-Aggregate-tasa máxima por usuario |
| Nombre para mostrar de LDAP | aCSMaxAggregatePeakRatePerUser       |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.897               |
| System-ID-GUID    | f072230c-aef5-11d1-bdcf-0000f80367c1 |
| Sintaxis            | [**Interval**](s-interval.md)       |
