# <a name="ms-dfsr-disablepacketprivacy-attribute"></a>atributo MS-DFSR-DisablePacketPrivacy

Deshabilite la privacidad de paquetes en una conexión.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-DisablePacketPrivacy         |
| Nombre para mostrar de LDAP | msDFSR-DisablePacketPrivacy          |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.32           |
| System-ID-GUID    | 6a84ede5-741e-43fd-9dd6-aa0f61578621 |
| Sintaxis            | [**Booleano**](s-boolean.md)         |
