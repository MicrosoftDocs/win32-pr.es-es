# <a name="configuring-snmp-security"></a>Configuración de la seguridad de SNMP

Las características de seguridad de SNMP permiten especificar las comunidades y los hosts de los que un equipo acepta solicitudes, así como el tipo de operaciones que se aceptan de los equipos que pertenecen a una comunidad. SNMP también permite especificar si se enviará una captura de autenticación cuando una comunidad o un host no autorizados solicite información.

Para obtener más información, consulte el kit de recursos de Windows y la documentación en línea del servicio SNMP.

**Windows Server 2003:** De forma predeterminada, los nombres de comunidad no están configurados y el servicio SNMP está configurado para aceptar solicitudes SNMP solo del nombre de host "localhost".
