# <a name="axwindowsmediaplayerclosedcaption-property"></a>Propiedad AxWindowsMediaPlayer. closedCaption

La propiedad closedCaption obtiene una interfaz **IWMPClosedCaption** para el elemento multimedia actual.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```CSharp
public IWMPClosedCaption closedCaption {get;}
```


```VB

Public ReadOnly Property closedCaption As IWMPClosedCaption
```





## <a name="property-value"></a>Valor de propiedad

WMPLIB. Interfaz IWMPClosedCaption.
