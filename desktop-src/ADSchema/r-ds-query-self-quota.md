# <a name="ds-query-self-quota-extended-right"></a>DS-Query: derecho extendido de cuota automática

Control derecho de acceso que permite a un usuario consultar las cuotas propias del usuario.



| Entrada | Value |
|--------------|--------------------------------------|
| CN           | DS-Query-auto-quota                  |
| Display-Name | Cuota de consulta automática                     |
| Rights-GUID  | 4ecc03fe-ffc0-4947-b630-eb672a8a9dbc |
