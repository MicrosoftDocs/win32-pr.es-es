# <a name="keepmodeless-dialog-style-bit"></a>Bit de estilo de diálogo KeepModeless

Normalmente, cuando no se establece este bit y se crea un cuadro de diálogo a través de DoAction, se destruyen todos los demás diálogos (normalmente modelados). Si se establece este bit, los demás diálogos permanecen activo cuando se crea este cuadro de diálogo.

## <a name="value"></a>Value



| Decimal | Hexadecimal | Constante                              |
|---------|-------------|---------------------------------------|
| 16      | 0x00000010  | **msidbDialogAttributesKeepModeless** |
