# <a name="communications-control-codes"></a>Códigos de control de comunicaciones

El siguiente código de control se usa con la entrada y salida del dispositivo.



| Código de control                                                        | Operación                                                                           |
|---------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**\_inserción de \_ LSRMST \_ serie ioctl**](/windows/desktop/api/WinIoCtl/ni-winioctl-ioctl_serial_lsrmst_insert) | Habilita o deshabilita la colocación de los datos de estado de línea y de módem en el flujo de datos. |
