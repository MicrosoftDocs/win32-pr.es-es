# <a name="byte-order-indicator"></a>Byte-Order indicador

El indicador de orden de bytes es un **tipo de** datos WORD y siempre debe contener el valor 0xFFFE. Esto es lo mismo que el indicador de orden de bytes Unicode. Este valor siempre se escribe en orden de bytes de Intel y, por tanto, aparece en el archivo o secuencia como 0xFE, 0xFF.
