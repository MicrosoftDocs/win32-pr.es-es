# <a name="text-editors"></a>Editores de texto

Debe usar un editor de texto que no incluya caracteres de formato. Se recomienda un editor de texto simple como el Bloc de notas, aunque los entornos de desarrollo como Microsoft Visual Studio ofrecen características avanzadas. No debe utilizar un editor como Microsoft Word, ya que el formato de archivo de Word predeterminado guarda códigos de formato adicionales que no se podrán utilizar. Si utiliza un procesador de textos como Word, debe recordar guardar el trabajo como texto sin formato cada vez que guarde el archivo.

Probablemente debe desactivar el ajuste de palabras si el editor de texto lo permite, y puede que desee activar la capacidad de ver todos los caracteres para asegurarse de que no agrega accidentalmente ninguna pestaña u otro carácter de control. Recuerde que, si se incluyen los caracteres de tabulación en el archivo de definición de máscara, se producirá un error en la máscara.
