# <a name="organizationalstatus-attribute"></a>atributo organizationalStatus

El tipo de atributo organizationalStatus especifica una categoría por la que a menudo se hace referencia a una persona en una organización.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | organizationalStatus                        |
| Nombre para mostrar de LDAP | organizationalStatus                        |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.45                  |
| System-ID-GUID    | 28596019-7349-4d2f-adff-5a629961f942        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
