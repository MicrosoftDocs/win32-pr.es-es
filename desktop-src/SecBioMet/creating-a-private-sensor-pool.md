# <a name="creating-a-private-sensor-pool"></a>Creación de un grupo de sensores privados

Un grupo de sensores privados es una colección de unidades biométricas reservadas para su uso exclusivo por parte de una aplicación cliente. Los grupos privados admiten métodos de autenticación propietarios y permiten que una aplicación cliente tenga acceso a una unidad biométrica mediante los comandos de control especificados por el proveedor.

Los temas siguientes contienen ejemplos de código que muestran cómo crear un grupo de sensores privados.
