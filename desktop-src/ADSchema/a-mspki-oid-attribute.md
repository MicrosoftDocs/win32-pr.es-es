# <a name="ms-pki-oid-attribute-attribute"></a>atributo MS-PKI-OID-Attribute

El atributo para el OID de empresa.



| Entrada | Value |
|-------------------|--------------------------------------------------------------------------------------------|
| CN                | Atributo MS-PKI-OID-                                                                       |
| Nombre para mostrar de LDAP | Atributo atributo mspki-OID                                                                        |
| Tamaño              | 4 bytes                                                                                    |
| Actualizar privilegio  | Administrador de empresa                                                                   |
| Frecuencia de actualización  | Siempre que se crea una nueva plantilla o se editan los atributos de las plantillas existentes. |
| Attribute-Id      | 1.2.840.113556.1.4.1671                                                                    |
| System-ID-GUID    | 8c9e1288-5028-4f4f-a704-76d026f246ef                                                       |
| Sintaxis            | [**Enumeración**](s-enumeration.md)                                                       |
