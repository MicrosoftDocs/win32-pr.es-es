# <a name="systemdeviceinterfaceprinterdriverdirectory"></a>System. DeviceInterface. PrinterDriverDirectory

Ubicación del directorio para el controlador de impresora.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.DeviceInterface.PrinterDriverDirectory
   shellPKey = PKEY_DeviceInterface_PrinterDriverDirectory
   formatID = 847C66DE-B8D6-4AF9-ABC3-6F4F926BC039
   propID = 14
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
