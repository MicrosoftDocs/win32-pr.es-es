# <a name="systemmonitorcounters-property"></a>SystemMonitor. counters (propiedad)

Recupera los **contadores** que contienen una colección de objetos de [**elementos de elemento**](counteritem.md) .

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
Property Counters As ICounters
```



## <a name="property-value"></a>Valor de propiedad

Objeto **counters** que contiene una colección de objetos de [**elementos de elemento**](counteritem.md) .

## <a name="remarks"></a>Observaciones

Esta es la propiedad predeterminada del objeto [**SystemMonitor**](systemmonitor.md) .
