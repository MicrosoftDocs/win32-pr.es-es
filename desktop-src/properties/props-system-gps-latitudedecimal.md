# <a name="systemgpslatitudedecimal"></a>System.GPS.LatitudeDecimal

Indica la latitud en función de la referencia de PKEY \_ GPS \_ LatitudeRef. Calculado a partir de PKEY \_ GPS \_ LatitudeNumerator y PKEY \_ GPS \_ LatitudeDenominator

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.GPS.LatitudeDecimal
   shellPKey = PKEY_GPS_LatitudeDecimal
   formatID = 0F55CDE2-4F49-450D-92C1-DCD16301B1B7
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
