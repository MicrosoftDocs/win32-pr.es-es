# <a name="canreclassify-element"></a>Elemento CanReClassify

Si **es true**, indica que se puede volver a analizar la tinta. Este elemento es **false** si el usuario ha establecido explícitamente el tipo de la entrada de lápiz mediante la conversión de escritura a mano en texto o la forma cambiar a características de Journal.

## <a name="definition"></a>Definición

``` syntax
<xs:element name="CanReclassify" type="xs:boolean" />
```
