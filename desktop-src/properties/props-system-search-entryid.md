# <a name="systemsearchentryid"></a>System. Search. EntryID

El ID. de entrada de un elemento dentro de un catálogo determinado en el índice de búsqueda de Windows.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.EntryID
   shellPKey = PKEY_Search_EntryID
   formatID = 49691C90-7E17-101A-A91C-08002B2ECDA9
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Int32
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

[System. Search. EntryID]() se usa en el SQL que se genera para consultar el índice. Este valor no se considera único a lo largo del tiempo porque se puede reciclar.

Los valores PKEY se definen en Propkey. h.
