# <a name="ms-authz-central-access-policy-id-attribute"></a>atributo MS-authz-central-Access-Policy-ID

En el caso de una directiva de acceso central, este atributo define un GUID que se puede utilizar para identificar el conjunto de directivas cuando se aplica a un recurso.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-authz-central-Access-Policy-ID    |
| Nombre para mostrar de LDAP | msAuthz-CentralAccessPolicyID        |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2154              |
| System-ID-GUID    | 62f29b60-be74-4630-9456-2f6691993a86 |
| Sintaxis            | [**Cadena (SID)**](s-string-sid.md)  |
