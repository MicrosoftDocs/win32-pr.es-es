# <a name="iagentaudiooutputproperties"></a>IAgentAudioOutputProperties

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

IAgentAudioOutputProperties proporciona acceso a las propiedades de salida de audio mantenidas por el servidor de agente de Microsoft. Estas funciones también están disponibles en IAgentAudioOutputProperties. Las propiedades son de solo lectura, pero el usuario puede cambiarlas en la hoja de propiedades de Microsoft Agent.

**Métodos en orden de Vtable**



| Métodos IAgentAudioOutputProperties                                               | Descripción                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------|
| [**GetEnabled**](iagentaudiooutputproperties--getenabled.md)                     | Devuelve si está habilitada la salida de audio.        |
| [**GetUsingSoundEffects**](iagentaudiooutputproperties--getusingsoundeffects.md) | Devuelve si está habilitada la salida de efecto sonoro. |
