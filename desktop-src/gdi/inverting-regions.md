# <a name="inverting-regions"></a>Invertir regiones

Una aplicación invierte los colores que aparecen dentro de una región mediante una llamada a la [**función InvertRgn.**](/windows/desktop/api/Wingdi/nf-wingdi-invertrgn) En las pantallas monocromáticas, **InvertRgn hace** que los píxeles de color blanco sea negro y negro. En las pantallas de color, esta inversión depende del tipo de tecnología que se usa para generar los colores de la pantalla.
