# <a name="appearstonotsupporttabbing"></a>AppearsToNotSupportTabbing

## <a name="text"></a>Texto

Parece que la aplicación no admite la tabulación.

## <a name="type"></a>Tipo

Error

## <a name="description"></a>Descripción

La aplicación no admite la navegación de teclado estándar (TAB o Mayús + Tab).

Este problema causa problemas de facilidad de uso para las personas que confían en un lector de pantalla o en el teclado para la navegación porque ninguno de los elementos de la aplicación es reconocible.

## <a name="possible-causes"></a>Causas posibles

-   El marco de interfaz de usuario de la aplicación no es compatible con MSAA.
-   MSAA no se implementa en ningún control de la aplicación.
