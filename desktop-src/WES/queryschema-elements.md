# <a name="query-schema-elements"></a>Elementos de esquema de consulta

Estos son los elementos que define el esquema de consulta. Esta sección contiene los nombres de los elementos que se usarían en la consulta; sin embargo, para obtener los detalles de cada elemento, vea el tipo complejo que contiene el elemento.



| Elemento                                                                  | Descripción                                                                                                                     |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**QueryList**](queryschema-querylist-element.md)                       | Nodo raíz de la consulta.<br/>                                                                                          |
| [**Query (QueryListType)**](queryschema-query-querylisttype-element.md) | Define un conjunto de selectores y supresores que se usan para incluir eventos en o excluir eventos del conjunto de resultados.<br/> |
| [**Select (QueryType)**](queryschema-select-querytype-element.md)       | Una consulta XPath que identifica los eventos que se incluirán en el conjunto de resultados de la consulta.<br/>                                        |
| [**Suprimir (QueryType)**](queryschema-suppress-querytype-element.md)   | Una consulta XPath que identifica los eventos que se excluirán del conjunto de resultados de la consulta.<br/>                                      |
