# <a name="d3dconvolutionmono"></a>D3DCONVOLUTIONMONO

Especifica las dimensiones máximas de los filtros de circunvolución de textura.



|                               |       |                                                            |
|-------------------------------|-------|------------------------------------------------------------|
| \#define                      | Value | Descripción                                                |
| \_MAXHEIGHT D3DCONVOLUTIONMONO | 7     | Alto máximo de un filtro de circunvolución de textura monocroma. |
| D3DCONVOLUTIONMONO \_ MAXWIDTH  | 7     | Ancho máximo de un filtro de circunvolución de textura monocroma.  |
