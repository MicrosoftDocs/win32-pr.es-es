# <a name="about-rectangles"></a>Acerca de los rectángulos

Los rectángulos se usan para la región de recorte de cursor, la parte no válida del área de cliente, un área para mostrar el texto con formato o el área de desplazamiento. Las aplicaciones también pueden usar rectángulos para rellenar, fotogramas o invertir una parte del área de cliente con un pincel determinado, y para recuperar las coordenadas de una ventana o del área de cliente de una ventana.

-   [Coordenadas del rectángulo](rectangle-coordinates.md)
-   [Operaciones de rectángulo](rectangle-operations.md)
