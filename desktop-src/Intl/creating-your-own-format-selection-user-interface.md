# <a name="international-text-display"></a>Presentación de texto internacional

Hay cuatro opciones posibles para mostrar texto internacional, que también conlleva la salida de scripts complejos:

-   Llamar a las API de texto de Windows
-   Crear instancias de controles de edición estándar de Windows
-   Crear instancias de controles Rich Edit
-   Llamar a Uniscribe

Vea [las opciones para mostrar texto](https://msdn.microsoft.com/globalization/mt662335) para la explicación y las ventajas de cada una de estas opciones. A continuación, depende de usted decidir cuál es la mejor opción para su aplicación, en función de la complejidad de la aplicación y sus características de diseño.
