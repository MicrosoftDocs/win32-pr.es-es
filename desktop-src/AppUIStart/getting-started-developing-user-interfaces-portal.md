# <a name="getting-started-developing-user-interfaces-for-windows-applications"></a>Introducción desarrollar interfaces de usuario para aplicaciones Windows

## <a name="purpose"></a>Propósito

En las secciones siguientes se ofrecen instrucciones generales para los programadores que diseñan, implementan y prueban la interfaz de usuario de una aplicación de Windows.

Además de los principios básicos de diseño de la interfaz de usuario, se proporcionan numerosas recomendaciones y sugerencias que ayudarán a los desarrolladores a proporcionar una experiencia de usuario tan sencilla, eficaz y agradable como sea posible.

> [!Note]  
> Estas instrucciones no pretenden ser completas y están sujetas a la funcionalidad y el ámbito específicos de una aplicación. Para obtener instrucciones más completas, consulte las directrices de interacción de la [experiencia del usuario de Windows](../uxguide/guidelines.md).
