# <a name="packaging-api"></a>API de empaquetado

Obtenga información sobre la API de empaquetado, que puede usar para crear, leer y escribir paquetes de aplicaciones. Cada paquete de aplicación contiene los archivos que constituyen una Windows aplicación y un archivo de manifiesto que describe el software que se va a Windows.
