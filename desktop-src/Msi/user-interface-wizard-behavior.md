# <a name="user-interface-wizard-behavior"></a>Comportamiento del Asistente para interfaz de usuario

Para crear una interfaz de usuario sencilla, los desarrolladores de paquetes de instalación pueden emplear un diseño de interfaz de usuario que se adhiere al comportamiento del Asistente para interfaz.

El término comportamiento del Asistente para interfaz de usuario significa que cada cuadro de diálogo de una secuencia contiene un botón **>>siguiente** , en el que el usuario hace clic para moverse al siguiente cuadro de diálogo después de escribir los datos o configurar la información en el cuadro de diálogo actual. Si el usuario decide volver y cambiar cualquier información introducida en un cuadro de diálogo anterior, cada cuadro de diálogo contiene un **<<botón anterior** en el que el usuario hace clic para volver atrás. Al final de la secuencia del asistente, el usuario hace clic en un botón **Finalizar** para comenzar el proceso de instalación.

La interfaz de usuario descrita en el ejemplo de interfaz de usuario se adhiere al comportamiento del Asistente para interfaz de usuario. Vea [un ejemplo de instalación](an-installation-example.md).
