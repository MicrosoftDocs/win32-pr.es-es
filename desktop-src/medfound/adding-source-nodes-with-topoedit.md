# <a name="adding-source-nodes-with-topoedit"></a>Agregar nodos de origen con TopoEdit

Un nodo de origen representa una secuencia en un archivo multimedia. Para crear un nodo de origen en TopoEdit, especifique un archivo multimedia. TopoEdit enumera las secuencias del archivo y crea los nodos de origen adecuados.

Para obtener información sobre cómo agregar nodos de origen mediante programación usando Media Foundation API, vea [crear nodos de origen](creating-source-nodes.md).

## <a name="to-add-a-source-node-to-a-topology"></a>Para agregar un nodo de origen a una topología

1.  En el menú **topología** , haga clic en **Agregar origen**.

    Se abre el cuadro de diálogo **Seleccionar origen de medios** .

2.  Navegue hasta la carpeta donde se encuentra el archivo multimedia.

3.  En el campo **nombre de archivo:** , escriba el nombre del archivo.

4.  Haga clic en **Abrir**.

TopoEdit crea nodos de origen para las secuencias. En el **Panel de topología** se muestra el nodo de origen contenido en un cuadro gris que muestra el nombre del archivo multimedia. El nodo de origen muestra el tipo de secuencia del nodo.
