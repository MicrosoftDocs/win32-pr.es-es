# <a name="last-update-sequence-attribute"></a>Atributo Last-Update-Sequence

Este atributo contiene el número de secuencia de actualización del último elemento del almacén de clases que se ha cambiado.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Última secuencia de actualización                        |
| Nombre para mostrar de LDAP | lastUpdateSequence                          |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.330                      |
| System-ID-GUID    | 7d6c0e9c-7e20-11d0-afd6-00c04fd930c9        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
