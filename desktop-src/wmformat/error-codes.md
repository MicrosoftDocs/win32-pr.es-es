# <a name="error-codes-windows-media-format-11-sdk"></a>Códigos de error (Windows Media Format 11 SDK)

Microsoft ha definido los siguientes códigos de error y de éxito para su uso en el SDK de Windows Media Format. Estos códigos son un subconjunto de una colección más grande que se puede ver en los archivos de encabezado asferr. h (códigos ASF) y NSError. h (otros códigos) que se incluyen en el SDK de Windows Media Format. Los códigos de error más comunes se pueden ver en Winerror. h.
