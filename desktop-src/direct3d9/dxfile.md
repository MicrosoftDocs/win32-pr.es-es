# <a name="dxfile-constants"></a>Constantes de DXFILE

Las marcas siguientes se usan para especificar en qué canales de una textura se va a operar. En desuso.

## <a name="dxfileformat"></a>DXFILEFORMAT



|                          |       |                  |
|--------------------------|-------|------------------|
| \#define                 | Value | Descripción      |
| \_binario DXFILEFORMAT     | 0     | Archivo binario.     |
| \_texto DXFILEFORMAT       | 1     | Archivo de texto.       |
| DXFILEFORMAT \_ comprimido | 2     | Archivo comprimido. |



 

Estas \# definiciones se declaran en Dxfile. h.

## <a name="dxfileloadoptions"></a>DXFILELOADOPTIONS



|                          |       |                              |
|--------------------------|-------|------------------------------|
| \#define                 | Value | Descripción                  |
| DXFILELOAD \_ FROMFILE     | 0x00L | Cargar un archivo de un archivo.     |
| DXFILELOAD \_ FROMRESOURCE | 0x01L | Cargar un archivo desde un recurso. |
| DXFILELOAD \_ FROMMEMORY   | 0x02L | Cargar un archivo de la memoria.     |
| DXFILELOAD \_ FROMSTREAM   | 0x04L | Carga de un archivo desde una secuencia.   |
| DXFILELOAD \_ FROMURL      | 0x08L | Cargar un archivo desde una dirección URL.      |



 

Estas \# definiciones se declaran en Dxfile. h.
