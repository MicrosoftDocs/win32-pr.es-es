# <a name="eventaltkey"></a>evento. altKey

El atributo **altKey** recupera un valor que indica si la tecla Alt estaba presionada cuando se produjo el evento.

``` syntax
event.altKey
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **valor booleano** de solo lectura.



| Value | Descripción                                     |
|-------|-------------------------------------------------|
| true  | Indica que la tecla ALT estaba en la posición hacia abajo. |
| false | Indica que la tecla ALT estaba en la posición arriba.   |



 

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
