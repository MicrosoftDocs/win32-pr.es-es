# <a name="regionxor-methods"></a>Métodos Region.Xor

En este tema se enumeran los métodos Xor de la [**clase Region.**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) Para obtener una lista completa de los métodos de la **clase Region,** vea [Métodos de región](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                         | Descripción                                                                                                                                                                              |
|:---------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Xor(Rect&)**](/previous-versions//ms534787(v=vs.85))     | El [**método Region::Xor**](/previous-versions//ms534787(v=vs.85)) actualiza esta región a las partes que no se interpretan de sí mismo y al interior del rectángulo especificado.<br/>  |
| [**Xor(RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstrectf_))   | El [**método Region::Xor**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstrectf_)) actualiza esta región a las partes que no se interpretan de sí mismo y al interior del rectángulo especificado.<br/> |
| [**Xor(Region \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstregion))     | El [**método Region::Xor**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstregion)) actualiza esta región a las partes que no se interpretan de sí misma y de otra región.<br/>                         |
| [**Xor(GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstgraphicspath)) | El [**método Region::Xor**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-xor(inconstgraphicspath)) actualiza esta región a las partes que no se interpretan de sí mismo y al interior de la ruta de acceso especificada.<br/>            |
