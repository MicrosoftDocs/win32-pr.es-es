# <a name="systemdevicesaepsignalstrength"></a>System.Devices.Aep.SignalStrength

Intensidad de señal del dispositivo. Solo se aplica a algunos protocolos.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.Devices.Aep.SignalStrength
   shellPKey = PKEY_Devices_Aep_SignalStrength
   formatID = A35996AB-11CF-4935-8B61-A6761081ECDF
   propID = 6
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Int32
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
