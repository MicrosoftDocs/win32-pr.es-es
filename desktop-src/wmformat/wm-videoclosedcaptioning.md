# <a name="wmvideoclosedcaptioning"></a>WM/VideoClosedCaptioning

El atributo **WM/VideoClosedCaptioning** indica si el contenido incluye subtítulos (CC).

## <a name="global-constant"></a>Constante global

g \_ wszWMVideoClosedCaptioning

## <a name="data-type"></a>Tipo de datos

**tipo de WMT \_ \_ bool**

## <a name="remarks"></a>Observaciones

Este atributo se usa para la televisión de difusión y otro contenido de vídeo.
