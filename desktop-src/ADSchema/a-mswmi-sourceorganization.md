# <a name="ms-wmi-sourceorganization-attribute"></a>atributo MS-WMI-SourceOrganization

La organización empresarial que creó inicialmente un objeto de directiva.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-SourceOrganization                   |
| Nombre para mostrar de LDAP | msWMI-SourceOrganization                    |
| Tamaño              | Menos de 50 caracteres.                 |
| Actualizar privilegio  | Administrador de directiva de grupo                  |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1644                     |
| System-ID-GUID    | 34f7ed6c-615d-418d-aa00-549a7d7be03e        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
