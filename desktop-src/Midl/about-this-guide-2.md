# <a name="about-this-guide"></a>Acerca de esta guía

Esta sección es una referencia para Lenguaje de definición de interfaz de Microsoft (MIDL) y una guía del usuario para el compilador MIDL, que genera archivos en lenguaje C a partir de archivos IDL.

En [la sección Using the MIDL Compiler (Usar](using-the-midl-compiler-2.md) el compilador midl) se enumeran los requisitos para el compilador de C y el preprocesador con los que MIDL debe interoperar. En esta sección también se describen los archivos que el compilador de MIDL genera para los códigos auxiliares RPC, los códigos auxiliares de DCOM y las bibliotecas de tipos de AUTOMATIZACIÓN OLE.

La sección Definiciones de interfaz y [bibliotecas](interface-definitions-and-type-libraries.md) de tipos proporciona información sobre cómo crear archivos de definición de interfaz (IDL) y configuración de aplicaciones (ACF) para MIDL, incluidas las páginas de referencia para los atributos IDL y ACF. También hay información sobre cómo importar archivos y bibliotecas de tipos, e importar archivos de encabezado del sistema.

La [sección midl Command-Line referencia](midl-command-line-reference.md) contiene información de referencia para cada modificador de línea de comandos y opción de modificador reconocido por el compilador de MIDL.

La [sección Referencia del lenguaje MIDL](midl-language-reference.md) contiene una entrada de referencia para cada palabra clave del Lenguaje de definición de interfaz de Microsoft.

En [la sección Errores y advertencias](midl-compiler-errors-and-warnings.md) del compilador de MIDL se enumeran los mensajes de error y advertencia que el compilador de MIDL puede generar.
