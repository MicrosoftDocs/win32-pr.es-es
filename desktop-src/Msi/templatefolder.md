# <a name="templatefolder-property"></a>Propiedad TemplateFolder

El instalador establece la propiedad **TemplateFolder** en la ruta de acceso completa a la carpeta de plantillas para el usuario actual.

Solo en Windows 2000, si se establece la propiedad [**AllUsers**](allusers.md) , esta propiedad señala a un directorio del sistema de archivos que contiene las plantillas que están disponibles para todos los usuarios.
