# <a name="basicdevicemanufacturername-property"></a>Propiedad BasicDevice. ManufacturerName

Obtiene el nombre del fabricante del dispositivo.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT get_ManufacturerName(
  [out] HSTRING *value
);
```



## <a name="property-value"></a>Valor de propiedad

Puntero al nombre del fabricante del dispositivo.
