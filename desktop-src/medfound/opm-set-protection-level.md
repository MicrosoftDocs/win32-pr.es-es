# <a name="opm_set_protection_level"></a>\_nivel de \_ protección de conjunto de OPM \_

Establece el nivel de protección de un mecanismo de protección de la salida.



| Requisito | Value |
|--------------|-----------------------------------------------------------------------------------------------------|
| GUID de comando | \_nivel de \_ protección de conjunto de OPM \_                                                                         |
| Datos de entrada   | Una estructura de [**parámetros de nivel de protección de \_ configuración \_ \_ \_ de OPM**](/windows/desktop/api/opmapi/ns-opmapi-opm_set_protection_level_parameters) |



 

## <a name="remarks"></a>Observaciones

Algunos conectores pueden admitir varios mecanismos de protección. Con ese tipo de conector, puede aplicar varios mecanismos de protección al mismo conector, con una configuración diferente para cada uno.
