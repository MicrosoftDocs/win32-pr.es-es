# <a name="monthlydowtriggerrunonlastweekofmonth-property"></a>Propiedad MonthlyDOWTrigger. RunOnLastWeekOfMonth

En el caso de scripting, obtiene o establece un valor booleano que indica que la tarea se ejecuta en la última semana del mes.

## <a name="syntax"></a>Sintaxis


```VB
MonthlyDOWTrigger.RunOnLastWeekOfMonth As Boolean
```



## <a name="property-value"></a>Valor de propiedad

True indica que la tarea se ejecuta en la última semana del mes; en caso contrario, false.
