# <a name="avencdddchighpassfilter-property"></a>Propiedad AVEncDDDCHighPassFilter

Especifica si un filtro de paso alto de bloqueo de DC se aplica a una secuencia de audio Dolby digital. Esta propiedad se aplica a los codificadores de audio Dolby digital.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncDDDCHighPassFilter**
