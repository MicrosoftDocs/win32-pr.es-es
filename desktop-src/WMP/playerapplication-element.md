# <a name="playerapplication-element"></a>ELEMENTO PLAYERAPPLICATION

Algunas **máscaras** pueden usar el elemento PLAYERAPPLICATION para acceder al **objeto PlayerApplication.** Este elemento proporciona una funcionalidad similar al atributo **global playerApplication.** Este elemento solo lo usan las máscaras que muestran las aplicaciones de C++ que insertan una instancia remota de Reproductor de Windows Media control.

El **elemento PLAYERAPPLICATION** admite los atributos siguientes.



| Atributo                                              | Descripción                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [hasDisplay](playerapplication-hasdisplay-scr.md)     | Recupera un valor que indica si el vídeo se puede mostrar a través del control Reproductor de Windows Media remoto. |
| [playerDocked](playerapplication-playerdocked-scr.md) | Recupera un valor que indica si Reproductor de Windows Media está en estado acoplado.                          |
