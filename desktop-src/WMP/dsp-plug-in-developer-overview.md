# <a name="dsp-plug-in-developer-overview"></a>Introducción al desarrollador del complemento DSP

Desde la perspectiva del desarrollador, los complementos de DSP son programas de software que reciben datos de audio o vídeo proporcionados por Reproductor de Windows Media justo antes de la fase de representación y, a continuación, procesan los datos y devuelven los datos a Reproductor de Windows Media para su representación.

En las secciones siguientes se proporciona información general sobre los Reproductor de Windows Media de DSP:

-   [Empaquetado del complemento DE DSP](dsp-plug-in-packaging.md)
-   [Conexión a Reproductor de Windows Media](connecting-to-windows-media-player.md)
-   [Entrada y salida de datos](data-input-and-output.md)
-   [Negociación de formato](format-negotiation.md)
-   [Proporcionar un Interfaz de usuario](providing-a-user-interface.md)
-   [Asistente para complementos DE DSP](dsp-plug-in-wizard.md)
-   [Interfaces necesarias](required-interfaces.md)
-   [Registro de complementos de DSP](registering-dsp-plug-ins.md)
-   [Actualización de los complementos de DSP existentes](updating-existing-dsp-plug-ins.md)
