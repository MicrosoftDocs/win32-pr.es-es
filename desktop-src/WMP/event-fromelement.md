# <a name="eventfromelement"></a>event.fromElement

El **atributo fromElement** recupera el elemento del que provenía el evento. Si ningún elemento pasa un evento, el valor es **NULL.**

``` syntax
event.fromElement
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un objeto de solo lectura.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
