# <a name="avdecvideopixelaspectratio-property"></a>Propiedad AVDecVideoPixelAspectRatio

Especifica la relación de aspecto de píxeles de la secuencia de vídeo descodificada.

Esta propiedad es de solo lectura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVDecVideoPixelAspectRatio**

## <a name="remarks"></a>Observaciones

Los 16 bits superiores del valor contienen el ancho y los 16 bits inferiores contienen el alto.
