# <a name="ipsec-policy-reference-attribute"></a>Atributo IPSec-Policy-Reference

Nombre distintivo de la Directiva relacionada con el protocolo de seguridad de Internet (IPsec).



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | IPSec-Directiva-referencia                  |
| Nombre para mostrar de LDAP | ipsecPolicyReference                    |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.517                  |
| System-ID-GUID    | b7b13118-b82e-11d0-afee-0000f80367c1    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
