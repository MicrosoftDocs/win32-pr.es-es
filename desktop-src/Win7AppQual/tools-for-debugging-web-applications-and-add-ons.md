# <a name="tools-for-debugging-web-applications-and-add-ons"></a>Herramientas para depurar aplicaciones web y Add-Ons

En lugar de [corregir problemas de compatibilidad en aplicaciones web y complementos](remediating-web-applications-and-add-ons.md), puede hacer que la aplicación web o el sitio web funcionen de forma nativa en el modo estándar de Windows Internet Explorer 8 escribiendo el código en los estándares web más recientes.

Para evaluar completamente los problemas de la aplicación o el sitio web, debe investigar la causa raíz del problema. Por ejemplo, el equipo de Internet Explorer ha determinado las causas raíz mediante el Herramientas de desarrollo y la reducción lógica de los posibles problemas. En las secciones siguientes se describen varias herramientas que ayudan a simplificar el proceso de depuración.



| Sección                                                                                                                    | Descripción                                                                                                                                                                         |
|----------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Microsoft Expression Web SuperPreview](microsoft-expression-web-superpreview.md)                                         | Herramienta de depuración visual independiente que agiliza y facilita la migración de los sitios de Microsoft Internet Explorer 6 a Windows Internet Explorer 7 o Internet Explorer 8. |
| [Herramienta de prueba de compatibilidad de Internet Explorer (IECTT)](inernet-explorer-compatibility-test-tool--iectt-.md)                  | Componente del kit de [herramientas de compatibilidad de aplicaciones de Microsoft](/windows-hardware/get-started/adk-install) que puede ayudarle a diagnosticar problemas en las aplicaciones Web.       |
| [Herramientas para desarrolladores de Internet Explorer 8](internet-explorer-8-developer-tools.md)                                             | Las herramientas de desarrollo que se incluyen en Internet Explorer 8 para la depuración.                                                                                                         |
| [Asistente para la compatibilidad de Internet Explorer 8 (Aggiorno)](inernet-explorer-8-compatibility-wizard--aggiorno-.md)              | Herramienta de terceros que actualiza automáticamente los sitios web para que se representen correctamente en Internet Explorer 8 sin interrumpir la compatibilidad en exploradores más antiguos.                                |
| [Herramienta de depurador web Fiddler](fiddler-web-debugger-tool.md)                                                                 | Herramienta de terceros para capturar el tráfico de red entre los equipos de Internet y de prueba.                                                                                           |
| [Prueba de compatibilidad de aplicaciones utilizando imágenes de Virtual PC](application-compatibility-testing-using-virtual-pc-images.md) | Imágenes de Virtual PC que simplifican las pruebas de compatibilidad.                                                                                                                              |
