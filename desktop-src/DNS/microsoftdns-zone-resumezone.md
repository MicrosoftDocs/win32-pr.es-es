# <a name="resumezone-method-of-the-microsoftdns_zone-class"></a>Método ResumeZone de la \_ clase de zona MicrosoftDNS

El método **ResumeZone** reanuda la zona DNS.

## <a name="syntax"></a>Sintaxis


```mof
void ResumeZone();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
