# <a name="disk-space-list"></a>Lista de Disk-Space

La API de instalación proporciona funcionalidad de lista de espacio en disco. Una lista de espacio en disco es una lista de operaciones de archivo que puede consultar para determinar la cantidad de espacio en disco necesario para esas operaciones.

En los temas siguientes se describen detalladamente las listas de espacio en disco, cómo calculan el espacio necesario y cómo utilizarlas en una aplicación de instalación:

-   [Acerca de la lista de Disk-Space](about-the-disk-space-list.md)
-   [Uso de la lista de Disk-Space](using-the-disk-space-list.md)
-   [Referencia de lista de espacio en disco](disk-space-list-reference.md)
