# <a name="system-device-enumerator"></a>Enumerador de dispositivos del sistema

El enumerador de dispositivos del sistema enumera los filtros y los dispositivos de hardware instalados en el sistema. Las aplicaciones pueden usar este componente para buscar filtros y dispositivos en una categoría determinada. Cree este objeto mediante una llamada **a CoCreateInstance**.



| Etiqueta | Value |
|------------------|------------------------------------------|
| Identificador de clase | CLSID \_ SystemDeviceEnum                  |
| Interfaces       | [**ICreateDevEnum**](/windows/desktop/api/Strmif/nn-strmif-icreatedevenum) |
