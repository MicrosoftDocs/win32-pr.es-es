# <a name="button-function"></a>Función Button

En la tabla siguiente se muestran los valores que son válidos para la función Button. No es necesario definir valores para funciones que no se vayan a usar en la máscara.



| Value           | Descripción                                                                                                                                                  |
|-----------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Reproducir            | Reproduce la selección actual de la lista de reproducción.                                                                                                                 |
| Pausar           | Pausa la reproducción del elemento multimedia actual.                                                                                                                   |
| Stop            | Detiene la reproducción del elemento multimedia actual.                                                                                                                    |
| Anterior            | Se desplaza al elemento anterior de la lista de reproducción.                                                                                                                  |
| Siguientes            | Se desplaza al siguiente elemento de la lista de reproducción.                                                                                                                      |
| Orden aleatorio         | Ordena aleatoriamente la lista de reproducción.                                                                                                                              |
| RepeatPlaylist  | Repite la lista de reproducción indefinidamente.                                                                                                                           |
| Silencio            | Silencia la salida de audio.                                                                                                                                   |
| PlayPause       | Alterna entre el estado de reproducción y el estado de pausa.                                                                                                          |
| Información            | Proporciona información sobre el elemento multimedia que se está reproduciendo. En el caso de las máscaras de Windows Media Player 10 Mobile o posterior, esta función no se usa.                     |
| FullScreen      | Cambia el tamaño del vídeo al tamaño máximo de la pantalla del dispositivo.                                                                                                 |
| WindowsMedia\*  | Abre Pocket Internet Explorer y carga el sitio web de Windowsmedia.com.                                                                                       |
| PlayPauseStop\* | Además de alternar entre el estado de reproducción y el estado de pausa, también permite a los usuarios detener una difusión en directo cuando no está disponible la funcionalidad de pausa. |
| VolumeUp\*      | Aumenta la salida de audio.                                                                                                                                  |
| VolumeDown\*    | Reduce la salida de audio.                                                                                                                                  |



 

\*Disponible para Windows Media Player 10 Mobile o posterior.
