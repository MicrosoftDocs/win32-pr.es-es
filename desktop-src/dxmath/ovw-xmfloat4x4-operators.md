# <a name="xmfloat4x4-operators"></a>Operadores XMFLOAT4X4

La estructura [**XMFLOAT4X4**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4x4) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                      | Descripción                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**operador =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat4x4-operator-assign)<br/>      | Asigna los datos del componente de vector de una instancia de `XMFLOAT4X4` a la instancia actual de `XMFLOAT4X4` . <br/> |
| [**operador ()**](xmfloat4x4-operator-parens.md)<br/> | Obtiene acceso a elementos de matriz concretos a los que hace referencia la fila y la columna de la instancia actual de `XMFLOAT4X4` . <br/>    |
