# <a name="print-spooler-api-messages"></a>Mensajes de la API del administrador de trabajos de impresión

La API del administrador de trabajos de impresión utiliza el siguiente mensaje.

## <a name="in-this-section"></a>En esta sección



| Message                                                  | Descripción                                                                                                                                                         |
|----------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SPOOLERSTATUS de WM \_**](wm-spoolerstatus.md)<br/> | El mensaje de [**\_ SPOOLERSTATUS de WM**](/windows/desktop/printdocs/wm-spoolerstatus) se envía desde el administrador de impresión siempre que se agrega o se quita un trabajo de la cola del administrador de impresión.<br/> |
