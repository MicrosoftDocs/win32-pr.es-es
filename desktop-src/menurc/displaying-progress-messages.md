# <a name="displaying-progress-messages"></a>Mostrar mensajes de progreso

De forma predeterminada, RC se compila sin mostrar mensajes de progreso. Sin embargo, puede especificar que RC muestre los mensajes de progreso. Para ello, use la opción **/v** .

En el ejemplo siguiente se hace que RC informe de su progreso al compilar el archivo de definición de recursos sample. RC y crea el archivo de recursos compilado sample. res:
