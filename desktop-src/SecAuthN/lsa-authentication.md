# <a name="lsa-authentication"></a>Autenticación LSA

La autenticación LSA describe las partes de la [*autoridad de seguridad local*](../secgloss/l-gly.md) (LSA) que las aplicaciones pueden usar para autenticar y registrar usuarios en el sistema local. También se describe cómo crear y llamar a paquetes de autenticación y paquetes de seguridad.

Las funciones de autenticación de LSA permiten escribir un paquete de autenticación, un paquete de subautenticación o un proveedor de compatibilidad de seguridad combinado/paquete de autenticación (SSP/AP).

En los temas siguientes se proporciona más información acerca de la autenticación LSA:

-   [Modelo de autenticación LSA](lsa-authentication-model.md)
-   [Sesiones de inicio de sesión LSA](lsa-logon-sessions.md)
-   [Autenticación de inicio de sesión de usuario LSA](lsa-user-logon-authentication.md)
-   [Paquetes de autenticación](authentication-packages.md)

Para obtener información sobre las funciones de autenticación LSA, vea [Authentication Reference](authentication-reference.md).

Para obtener información sobre las tecnologías relacionadas, vea [Directiva de LSA](../secmgmt/lsa-policy.md) y [proveedores de credenciales y Winlogon](winlogon-and-credential-providers.md).
