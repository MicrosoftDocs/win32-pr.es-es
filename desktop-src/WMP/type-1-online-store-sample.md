# <a name="type-1-online-store-sample"></a>Ejemplo de la tienda en línea de tipo 1

El SDK de Windows Media Player incluye un ejemplo de almacén en línea de tipo 1 que es compatible con Windows Media Player 11.

Para ver la tienda en línea de tipo 1 de ejemplo, configure el registro como se describe en [claves del registro y entradas para un almacén en línea de tipo 1](registry-keys-and-entries-for-a-type-1-online-store.md). La clave del almacén del tipo de ejemplo 1 es 2573.
