# <a name="frs-replica-set-guid-attribute"></a>Atributo FRS-Replica-Set-GUID

GUID que identifica un conjunto de réplicas de FRS.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | FRS-Replica-Set-GUID                                  |
| Nombre para mostrar de LDAP | fRSReplicaSetGUID                                     |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.533                                |
| System-ID-GUID    | 5245801a-ca6a-11d0-afff-0000f80367c1                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
