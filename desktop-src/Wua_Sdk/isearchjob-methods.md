# <a name="isearchjob-methods"></a>Métodos ISearchJob

La interfaz [**ISearchJob**](/windows/desktop/api/Wuapi/nn-wuapi-isearchjob) define los siguientes métodos.



| Método                                          | Descripción                                                                          |
|-------------------------------------------------|--------------------------------------------------------------------------------------|
| [**Reparación**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-cleanup)           | Espera a que se complete una operación asincrónica y, a continuación, libera todas las devoluciones de llamada. |
| [**RequestAbort**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-requestabort) | Realiza una solicitud para cancelar una búsqueda asincrónica.                                    |
