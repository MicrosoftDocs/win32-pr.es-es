# <a name="preprocessor-commands"></a>Comandos de preprocesador

Un comando de preprocesador es una línea de código que se coloca dentro de un archivo MOF que controla la forma en que el compilador MOF compila el código. Hay dos comandos de preprocesador que se usan para compilar archivos MOF:

-   [\#inclui](-include.md)
-   [\#pragma](-pragma.md)
