# <a name="wia-tutorial"></a>Tutorial de WIA

En este tutorial se proporcionan instrucciones paso a paso para usar la adquisición de imágenes de Windows (WIA) en aplicaciones reales.

En el tutorial se presenta el código de las aplicaciones de ejemplo incluidas en el SDK. Está pensado para demostrar cómo los desarrolladores pueden implementar la funcionalidad principal de WIA y no es un plano para crear una aplicación completa.

El tutorial se centra en las siguientes tareas:

-   [Crear un Administrador de dispositivos WIA](-wia-creating-a-wia-device-manager.md)
-   [Enumerar dispositivos del sistema](-wia-enumerating-system-devices.md)
-   [Lectura de las propiedades del dispositivo](-wia-reading-device-properties.md)
-   [Creación de un dispositivo](-wia-creating-a-device.md)
-   [Enumerar elementos](-wia-enumerating-items.md)
-   [Transferir datos de imagen](-wia-transferring-image-data.md)
-   [Registrar eventos](-wia-registering-for-events.md)
-   [Capturar una imagen de un vídeo de streaming](-wia-capturing-a-still-image-from-streaming-video.md)
