# <a name="n-network-monitor"></a>N (Monitor de red)

<dl> <dt>

<span id="_netmon_ndis_gly"></span><span id="_NETMON_NDIS_GLY"></span>**NDIS**
</dt> <dd>

Consulte Especificación de interfaz de controlador de red.

</dd> <dt>

<span id="_netmon_network_driver_interface_specification_gly"></span><span id="_NETMON_NETWORK_DRIVER_INTERFACE_SPECIFICATION_GLY"></span>**Especificación de interfaz de controlador de red (NDIS)**
</dt> <dd>

Especificación de la interfaz entre los controladores de dispositivo y una red. Todos los transportes llaman a la interfaz NDIS para tener acceso a tarjetas de interfaz de red y trabajar con ellas.

</dd> <dt>

<span id="_netmon_network_monitor_agent_gly"></span><span id="_NETMON_NETWORK_MONITOR_AGENT_GLY"></span>**Agente Monitor de red**
</dt> <dd>

Componente de Monitor de red. El agente permite a un equipo remoto capturar datos de la red. Cuando una instalación de Monitor de red se conecta de forma remota al agente de Monitor de red e inicia una captura, las estadísticas de la captura se transfieren a través de la red al equipo de administración. La transferencia continúa a intervalos especificados cuando se realiza la conexión.

</dd> <dt>

<span id="_netmon_network_packet_provider_gly"></span><span id="_NETMON_NETWORK_PACKET_PROVIDER_GLY"></span>**proveedor de paquetes de red (NPP)**
</dt> <dd>

El componente de Monitor de red que recopila el tráfico de red en fotogramas y, después, pasa los fotogramas a una aplicación de experto y NPP. Un NPP utiliza el Monitor de red controlador del sistema (Nmnt.sys) para obtener los fotogramas recopilados de la red y proporciona varias interfaces COM que pasan los fotogramas a una aplicación de experto, monitor y proveedor de paquetes de red (NPP) donde se pueden mostrar y analizar.

</dd> <dt>

<span id="_netmon_npp_gly"></span><span id="_NETMON_NPP_GLY"></span>**NPP**
</dt> <dd>

Consulte proveedor de paquetes de red.

</dd> <dt>

<span id="_netmon_npp_application_gly"></span><span id="_NETMON_NPP_APPLICATION_GLY"></span>**Aplicación NPP**
</dt> <dd>

Aplicación que omite la interfaz de usuario de Monitor de red y la herramienta de control de supervisión, y se conecta directamente al proveedor de paquetes de red (NPP). Una aplicación NPP puede conectarse al componente NPP con cualquiera de las cinco interfaces COM NPP.

</dd> <dt>

<span id="_netmon_npp_finder_gly"></span><span id="_NETMON_NPP_FINDER_GLY"></span>**Buscador de NPP**
</dt> <dd>

El componente de Monitor de red que se comunica con NPPs.

</dd> </dl>
