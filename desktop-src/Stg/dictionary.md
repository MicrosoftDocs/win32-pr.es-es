# <a name="dictionary"></a>Diccionario

[**IPropertyStorage:: WritePropertyNames**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writepropertynames) se implementa mediante el [Diccionario de identificador 0 de propiedad](/windows/desktop/Stg/reserved-property-identifiers). El identificador de propiedad 0 no es accesible a través de [**IPropertyStorage:: ReadMultiple**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-readmultiple) o [**IPropertyStorage:: WriteMultiple**](/windows/desktop/api/Propidl/nf-propidl-ipropertystorage-writemultiple).
