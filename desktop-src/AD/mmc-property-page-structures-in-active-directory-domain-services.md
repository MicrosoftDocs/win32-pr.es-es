# <a name="mmc-property-page-structures-in-active-directory-domain-services"></a>Estructuras de páginas de propiedades de MMC en Active Directory Domain Services

El complemento MMC de Active Directory Manager usa hojas de propiedades para mostrar los atributos de los objetos en Active Directory Domain Services. Las siguientes estructuras ayudan a proporcionar la infraestructura para controlar las operaciones realizadas en las páginas de estas hojas de propiedades:

-   [**ADSPROPERROR**](/windows/desktop/api/Adsprop/ns-adsprop-adsproperror)
-   [**ADSPROPINITPARAMS**](/windows/desktop/api/Adsprop/ns-adsprop-adspropinitparams)
