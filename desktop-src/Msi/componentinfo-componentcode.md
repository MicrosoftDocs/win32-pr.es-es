# <a name="componentinfocomponentcode-property"></a>ComponentInfo.ComponentCode, propiedad

Código de componente del componente en cuestión. Esta propiedad llama a la [**función MsiGetComponentPathEx.**](/windows/desktop/api/Msi/nf-msi-msigetcomponentpathexa)

**[Windows Installer 4.5 o anterior:](not-supported-in-windows-installer-4-5.md)** No se admite. Esta propiedad está disponible a partir de Windows Installer 5.0.
