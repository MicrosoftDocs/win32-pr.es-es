# <a name="common-name-attribute"></a>Common-Name atributo

Nombre que representa un objeto . Se usa para realizar búsquedas.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Common-Name                                 |
| Ldap-Display-Name | cn                                          |
| Size              | El tamaño medio es de 64 bytes.                   |
| Actualizar privilegios  | Administrador de dominio                        |
| Frecuencia de actualización  | Solo cuando se crea un nuevo objeto.          |
| Attribute-Id      | 2.5.4.3                                     |
| System-Id-Guid    | bf96793f-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
