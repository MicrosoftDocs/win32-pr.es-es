# <a name="using-disk-prompting-and-error-handling"></a>Uso de solicitudes de disco y control de errores

Los cuadros de diálogo creados por las funciones de instalación, [**SetupPromptForDisk**](/windows/desktop/api/Setupapi/nf-setupapi-setuppromptfordiska), [**SetupCopyError,**](/windows/desktop/api/Setupapi/nf-setupapi-setupcopyerrora) [**SetupDeleteError**](/windows/desktop/api/Setupapi/nf-setupapi-setupdeleteerrora)y [**SetupRenameError**](/windows/desktop/api/Setupapi/nf-setupapi-setuprenameerrora) pueden ser usados por una aplicación de instalación o por rutinas de devolución de llamada. En las secciones siguientes se describe el uso de las funciones de una aplicación y sus notificaciones correspondientes:

-   [Pedir un disco](prompting-for-a-disk.md)
-   [Tratamiento de errores](error-handling.md)
