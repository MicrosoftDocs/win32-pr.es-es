## <a name="remarks"></a>Observaciones

Los errores se representan con valores negativos y no se pueden combinar.

[Hay otros códigos de retorno \_ ](device-state-return-codes.md)incluidos en.

El código de instalación \_ FACD3D se utiliza para generar códigos de error, como en las siguientes macros.


```
// Direct3D Errors
#define D3DERR_UNSUPPORTEDFACTORVALUE  MAKE_D3DHRESULT(2079)
#define D3DERR_WASSTILLDRAWING         MAKE_D3DHRESULT(540)
    
#define _FACD3D                  0x876   // Direct3D facility code
#define MAKE_D3DHRESULT( code )  MAKE_HRESULT( 1, _FACD3D, code )
#define MAKE_D3DSTATUS( code )   MAKE_HRESULT( 0, _FACD3D, code )
```
