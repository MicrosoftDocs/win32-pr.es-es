# <a name="security-values"></a>Valores de seguridad

Hay varios valores que afectan a la seguridad de las aplicaciones COM, como el nivel de autenticación, el nivel de suplantación, la ocultación, la identidad de aplicación, la configuración de directivas de restricción de software y el seguimiento de referencias.

Para obtener más información, vea los temas siguientes:

-   [Nivel de autenticación](authentication-level.md) describe cómo ayudar a controlar la cantidad de protección para la comunicación entre objetos com.
-   [Delegación y suplantación](delegation-and-impersonation.md) explica cómo afecta la ocultación a la identidad presentada al servidor durante la suplantación y cómo concede al servidor la autoridad para que actúe en su nombre.
-   [Identidad de aplicación](application-identity.md) identifica y explica las identidades que puede usar la aplicación.
-   La [Directiva de restricción de software](software-restriction-policy.md) explica cómo ayudar a proteger el sistema frente a los peligros asociados con la ejecución de código desconocido.
-   [Seguimiento de referencias](reference-tracking.md) explica cómo puede asegurarse de que los objetos no se lancen demasiado pronto.
