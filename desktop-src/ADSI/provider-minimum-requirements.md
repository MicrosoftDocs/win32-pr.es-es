# <a name="provider-minimum-requirements"></a>Requisitos mínimos del proveedor

Para escribir un proveedor adsi, implemente las interfaces Active Directory servicio como se describe en las secciones siguientes.

-   [Implementación principal](core-implementation.md) enumera brevemente los elementos de software que debe admitir un proveedor adsi.
-   [La implementación opcional](optional-implementation.md) enumera brevemente los elementos de software que se recomienda encarecidamente admitir a un proveedor adsi.
-   [Propiedades personalizadas](custom-properties.md) son esas propiedades e interfaces expuestas por cada proveedor para proporcionar programación de las características nativas del servicio de directorio.
-   [Las interfaces duales](dual-interfaces.md) describen la convención adsi para proporcionar interfaces accesibles tanto para los controladores compatibles con Automation como para los que no son compatibles con Automation.
