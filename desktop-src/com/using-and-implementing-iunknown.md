# <a name="using-and-implementing-iunknown"></a>Usar e implementar IUnknown

COM proporciona un amplio conjunto de estándares para implementar y usar objetos y para la comunicación entre objetos. En los temas de esta sección se describe la información básica relacionada con la implementación de objetos. Para obtener información acerca de cómo interactúan los clientes y los servidores, consulte [com clients and Servers](com-clients-and-servers.md). Para obtener más información sobre los modelos de subprocesos y su implementación y uso, consulte [procesos, subprocesos y apartamentos](processes--threads--and-apartments.md).

Para obtener información detallada sobre el uso e implementación de [**IUnknown**](/windows/desktop/api/Unknwn/nn-unknwn-iunknown), vea los temas siguientes:

-   [QueryInterface: desplazarse por un objeto](queryinterface--navigating-in-an-object.md)
-   [Reglas para implementar QueryInterface](rules-for-implementing-queryinterface.md)
-   [Administrar la duración de los objetos mediante el recuento de referencias](managing-object-lifetimes-through-reference-counting.md)
