# <a name="wm_adsprop_notify_change-message"></a>MENSAJE \_ DE CAMBIO DE NOTIFICACIÓN DE ADSPROP \_ \_ DE WM

El objeto de notificación usa internamente el mensaje **WM \_ ADSPROP \_ NOTIFY \_ CHANGE.**


```C++
WM_ADSPROP_NOTIFY_CHANGE

    WPARAM wParam
    LPARAM lParam
    
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*Hwnd* 
</dt> <dd>

No se usa.

</dd> <dt>

*wParam* 
</dt> <dd>

No se usa.

</dd> <dt>

*lParam* 
</dt> <dd>

No se usa.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Este mensaje no tiene ningún valor devuelto.
