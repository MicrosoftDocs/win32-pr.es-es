# <a name="configuring-the-name-service-provider"></a>Configuración del proveedor de servicios de nombres

Si la aplicación distribuida registra su interfaz con un servicio de nombres, tanto el cliente como el servidor deben usar el mismo servicio de nombre. Rpc de Microsoft interopera con Microsoft Locator y cualquier proveedor de servicios de nombres (NSP) que se ajuste a la interfaz del servicio de nombres RPC (NSI) de Microsoft, por ejemplo, el Servicio de directorio de celdas de DCE al que se accede a través del demonio de servicio de nombres (NSID) de Digital Equipment Corporation. Microsoft Locator, que está diseñado para su uso Windows entornos, es el proveedor de servicios de nombres predeterminado.

En esta sección se presenta una explicación de la instalación y configuración de proveedores de servicios de nombres:

-   [Configuración del servicio de nombres para Windows 95](configuring-the-name-service-for-windows-95.md)
-   [Configuración del servicio de nombres para Windows 3.x o MS-DOS](configuring-the-name-service-for-windows-3-x-or-ms-dos.md)
