# <a name="understanding-the-application-compatibility-challenge"></a>Descripción del desafío de la compatibilidad de aplicaciones

Windows Internet Explorer 8 es el explorador predeterminado en el sistema operativo Windows 7. Internet Explorer 8 mejora muchas características, entre las que se incluye una arquitectura de plataforma mejorada que está diseñada para que la web sea más segura y más rápida para los usuarios y que sea más fácil de configurar y administrar para los profesionales de ti, así como para crear contenido innovador para los desarrolladores Web.

Algunas de estas características usan nuevos motores de representación y scripting, que son los elementos básicos que usa Windows Internet Explorer para mostrar contenido HTML y CSS y ejecutar scripts. Los cambios en la arquitectura y las mejoras de la plataforma presentan problemas de compatibilidad de aplicaciones que puede experimentar al migrar aplicaciones a Internet Explorer 8 desde un explorador anterior.

Internet Explorer 8 incluye alguna compatibilidad con versiones anteriores, pero hay algunos cambios (incluida la compatibilidad con las características de seguridad y compatibilidad con los estándares web) que pueden requerir la actualización de aplicaciones web compiladas para Microsoft Internet Explorer 6 o versiones anteriores.

Para obtener más información acerca de cómo probar y solucionar problemas de compatibilidad con Internet Explorer 8, consulte [abordar la compatibilidad de aplicaciones al migrar a Internet Explorer 8: información para profesionales de tecnologías de la información](/previous-versions/windows/it-pro/windows-7/ff955640(v=ws.10)).
