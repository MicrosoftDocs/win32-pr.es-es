# <a name="location-api-object-model-reference"></a>Referencia del modelo de objetos de la API de ubicación

\[El modelo de objetos de la API de ubicación está disponible para su uso en los sistemas operativos especificados en la sección Requisitos. En versiones posteriores podría modificarse o no estar disponible. En su lugar, para acceder a la ubicación desde un sitio web, use [la API de geolocalización de W3C](/previous-versions/windows/internet-explorer/ie-developer/samples/gg589513(v=vs.85)). Para acceder a la ubicación desde una aplicación de escritorio, use el [**Windows. Devices.Geolocation**](/uwp/api/Windows.Devices.Geolocation) API.\]

Location API proporciona objetos que puede usar en lenguajes de scripting, como Microsoft JScript. Puede usar estos objetos para agregar funcionalidad de ubicación a las páginas web. Estos objetos se documentan en las secciones siguientes.

-   [**LocationDisp.DispCivicAddressReport**](locationdisp-dispcivicaddressreport.md)
-   [**LocationDisp.CivicAddressReportFactory**](locationdisp-civicaddressreportfactory.md)
-   [**LocationDisp.DispLatLongReport**](locationdisp-displatlongreport.md)
-   [**LocationDisp.LatLongReportFactory**](locationdisp-latlongreportfactory.md)
-   [Eventos LocationDisp](locationdisp-events.md)
