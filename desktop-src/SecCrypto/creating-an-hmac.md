<span data-ttu-id="2898a-104">**Para calcular un HMAC**</span><span class="sxs-lookup"><span data-stu-id="2898a-104">**To compute an HMAC**</span></span>

2.  <span data-ttu-id="2898a-106">Cree un identificador para un [](../secgloss/h-gly.md)[*objeto hash*](../secgloss/h-gly.md) HMAC llamando a [**CryptCreateHash**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptcreatehash).</span><span class="sxs-lookup"><span data-stu-id="2898a-106">Create a handle to an [*HMAC*](../secgloss/h-gly.md)[*hash object*](../secgloss/h-gly.md) by calling [**CryptCreateHash**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptcreatehash).</span></span> <span data-ttu-id="2898a-107">Pase CALG \_ HMAC en el parámetro *algid* .</span><span class="sxs-lookup"><span data-stu-id="2898a-107">Pass CALG\_HMAC in the *Algid* parameter.</span></span> <span data-ttu-id="2898a-108">Pase el identificador de una [*clave simétrica*](../secgloss/s-gly.md) en el parámetro *hKey* .</span><span class="sxs-lookup"><span data-stu-id="2898a-108">Pass the handle of a [*symmetric key*](../secgloss/s-gly.md) in the *hKey* parameter.</span></span> <span data-ttu-id="2898a-109">Esta clave simétrica es la clave utilizada para calcular el HMAC.</span><span class="sxs-lookup"><span data-stu-id="2898a-109">This symmetric key is the key used to compute the HMAC.</span></span>
3.  <span data-ttu-id="2898a-110">Especifique el tipo de hash que se va a usar mediante una llamada a [**CryptSetHashParam**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptsethashparam) con el parámetro *dwParam* establecido en el valor HP \_ HMAC \_ info.</span><span class="sxs-lookup"><span data-stu-id="2898a-110">Specify the type of hash to be used by calling [**CryptSetHashParam**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptsethashparam) with the *dwParam* parameter set to the value HP\_HMAC\_INFO.</span></span> <span data-ttu-id="2898a-111">El parámetro *pbData* debe apuntar a una estructura [**de \_ información HMAC**](/windows/desktop/api/Wincrypt/ns-wincrypt-hmac_info) inicializada.</span><span class="sxs-lookup"><span data-stu-id="2898a-111">The *pbData* parameter must point to an initialized [**HMAC\_INFO**](/windows/desktop/api/Wincrypt/ns-wincrypt-hmac_info) structure.</span></span>
4.  <span data-ttu-id="2898a-112">Llame a [**CryptHashData**](/windows/desktop/api/Wincrypt/nf-wincrypt-crypthashdata) para empezar a calcular el HMAC de los datos.</span><span class="sxs-lookup"><span data-stu-id="2898a-112">Call [**CryptHashData**](/windows/desktop/api/Wincrypt/nf-wincrypt-crypthashdata) to begin computing the HMAC of the data.</span></span> <span data-ttu-id="2898a-113">La primera llamada a **CryptHashData** hace que el valor de clave se combine mediante el operador XOR con la cadena interna y los datos.</span><span class="sxs-lookup"><span data-stu-id="2898a-113">The first call to **CryptHashData** causes the key value to be combined using the XOR operator with the inner string and the data.</span></span> <span data-ttu-id="2898a-114">Se aplica un algoritmo hash al resultado de la operación XOR y, a continuación, se aplica un algoritmo hash a los datos de destino del HMAC (al que apunta el parámetro *pbData* que se pasa en la llamada a **CryptHashData**).</span><span class="sxs-lookup"><span data-stu-id="2898a-114">The result of the XOR operation is hashed, and then the target data for the HMAC (pointed to by the *pbData* parameter passed in the call to **CryptHashData**) is hashed.</span></span> <span data-ttu-id="2898a-115">En caso necesario, se pueden realizar llamadas subsiguientes a **CryptHashData** para finalizar el hash de los datos de destino.</span><span class="sxs-lookup"><span data-stu-id="2898a-115">If necessary, subsequent calls to **CryptHashData** may then be made to finish the hashing of the target data.</span></span>
5.  <span data-ttu-id="2898a-116">Llame a [**CryptGetHashParam**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptgethashparam) con el parámetro *DWPARAM* establecido en HP \_ HASHVAL.</span><span class="sxs-lookup"><span data-stu-id="2898a-116">Call [**CryptGetHashParam**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptgethashparam) with the *dwParam* parameter set to HP\_HASHVAL.</span></span> <span data-ttu-id="2898a-117">Esta llamada hace que se complete el hash interno y la cadena externa se combine con XOR con la clave.</span><span class="sxs-lookup"><span data-stu-id="2898a-117">This call causes the inner hash to be finished and the outer string to be combined using XOR with the key.</span></span> <span data-ttu-id="2898a-118">Se aplica un algoritmo hash al resultado de la operación XOR y, a continuación, se aplica un algoritmo hash al resultado del hash interno (completado en el paso anterior).</span><span class="sxs-lookup"><span data-stu-id="2898a-118">The result of the XOR operation is hashed, and then the result of the inner hash (completed in the previous step) is hashed.</span></span> <span data-ttu-id="2898a-119">Después, el hash externo se completa y se devuelve en el parámetro *pbData* y la longitud en el parámetro *dwDataLen* .</span><span class="sxs-lookup"><span data-stu-id="2898a-119">The outer hash is then finished and returned in the *pbData* parameter and the length in the *dwDataLen* parameter.</span></span>
