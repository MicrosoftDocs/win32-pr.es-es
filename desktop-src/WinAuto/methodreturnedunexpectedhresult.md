# <a name="methodreturnedunexpectedhresult"></a>MethodReturnedUnexpectedHResult

## <a name="text"></a>Texto

{0} devuelto, {1} pero el valor devuelto esperado debería haber sido {2}

## <a name="type"></a>Tipo

Advertencia

## <a name="description"></a>Descripción

Se devolvió un valor HRESULT inesperado al intentar recuperar todos los elementos secundarios del elemento.

## <a name="possible-causes"></a>Causas posibles

Una implementación de MSAA incorrecta o no válida.
