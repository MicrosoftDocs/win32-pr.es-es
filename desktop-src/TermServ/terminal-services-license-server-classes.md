# <a name="remote-desktop-license-server-classes"></a>Escritorio remoto clases del servidor de licencias

El Escritorio remoto WMI del servidor de licencias proporciona las clases siguientes.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**Win32 \_ TSIssuedLicense**](win32-tsissuedlicense.md)
</dt> <dd>

Describe las instancias de Servicios de Escritorio remoto licencias de acceso de cliente por dispositivo (CAL de RDS por dispositivo) que se emiten desde un servidor Escritorio remoto licencias.

</dd> <dt>

[**Win32 \_ TSLicenseKeyPack**](win32-tslicensekeypack.md)
</dt> <dd>

Proporciona métodos y propiedades para ver e instalar Servicios de Escritorio remoto de claves de licencia.

</dd> <dt>

[**TSLicenseReport de Win32 \_**](win32-tslicensereport.md)
</dt> <dd>

Proporciona instancias de informes de uso de Servicios de Escritorio remoto por usuario de licencia de acceso de cliente (CAL por usuario de RDS) que se generan en el servidor de licencias de Escritorio remoto y métodos para las operaciones de generación, captura y eliminación de informes de licencias.

</dd> <dt>

[**Win32 \_ TSLicenseReportEntry**](win32-tslicensereportentry.md)
</dt> <dd>

Proporciona detalles de la licencia de acceso de cliente Servicios de Escritorio remoto por usuario (CAL por usuario de RDS).

</dd> <dt>

[**Win32 \_ TSLicenseReportFailedPerUserEntry**](win32-tslicensereportfailedperuserentry.md)
</dt> <dd>

Proporciona detalles sobre el error Servicios de Escritorio remoto licencia de acceso de cliente por usuario (CAL por usuario de RDS).

</dd> <dt>

[**Win32 \_ TSLicenseReportFailedPerUserSummaryEntry**](win32-tslicensereportfailedperusersummaryentry.md)
</dt> <dd>

Proporciona detalles de los dominios en los que se ha fallado la emisión por usuario.

</dd> <dt>

[**Win32 \_ TSLicenseReportPerDeviceEntry**](win32-tslicensereportperdeviceentry.md)
</dt> <dd>

Proporciona detalles sobre el error Servicios de Escritorio remoto licencia de acceso de cliente por dispositivo (CAL de RDS por dispositivo).

</dd> <dt>

[**Win32 \_ TSLicenseReportSummaryEntry**](win32-tslicensereportsummaryentry.md)
</dt> <dd>

Proporciona un resumen de las licencias de acceso de cliente Servicios de Escritorio remoto por usuario (CAL de RDS por usuario) instaladas y emitidas.

</dd> <dt>

[**TSLicenseServer de Win32 \_**](win32-tslicenseserver.md)
</dt> <dd>

Proporciona métodos y propiedades para ver y configurar Escritorio remoto licencias (licencias de Escritorio remoto) en un servidor Escritorio remoto licencias.

</dd> </dl>
