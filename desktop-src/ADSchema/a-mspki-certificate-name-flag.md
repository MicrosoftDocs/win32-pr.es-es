# <a name="ms-pki-certificate-name-flag-attribute"></a>Atributo ms-PKI-Certificate-Name-Flag

Contiene las marcas relacionadas con la construcción del nombre del firmante en un certificado emitido.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Certificate-Name-Flag                                                                      |
| Ldap-Display-Name | msPKI-Certificate-Name-Flag                                                                       |
| Size              | 4 bytes                                                                                           |
| Privilegio actualizar  | Administrador de dominio                                                                              |
| Frecuencia de actualización  | Cuando se edita, crea o clona el objeto de plantilla de certificado (ms-PKI-Certificate-Template). |
| Attribute-Id      | 1.2.840.113556.1.4.1432                                                                           |
| System-Id-Guid    | ea1dddc4-60ff-416e-8cc0-17cee534bce7                                                              |
| Syntax            | [**Enumeración**](s-enumeration.md)                                                              |
