# <a name="swbemlasterrorproperties_-property"></a>Propiedad SWbemLastError. Properties \_

La **propiedad \_ Properties** del objeto [**SWbemLastError**](swbemlasterror.md) devuelve un objeto [**SWbemPropertySet**](swbempropertyset.md) que es una colección de las propiedades WMI de la clase o la instancia actual. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemLastError.Properties_ As Object
```
