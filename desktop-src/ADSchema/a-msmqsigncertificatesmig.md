# <a name="msmq-sign-certificates-mig-attribute"></a>MSMQ-Sign-Certificates-atributo MIG

En el modo mixto de MSMQ, el atributo contiene el valor anterior de mSMQSignCertificates. MSMQ admite la migración de MSMQ 1,0 DS a Windows 2000 DS y el modo mixto especifica un estado en el que algunos de los servidores de DS no se actualizaron a Windows 2000.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Sign-Certificates-MIG                            |
| Nombre para mostrar de LDAP | mSMQSignCertificatesMig                               |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.967                                |
| System-ID-GUID    | 3881b8ea-da3b-11d1-90a5-00c04fd91ab1                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
