# <a name="updating-directory-structure-for-an-upgrade"></a>Actualizar la estructura de directorios para una actualización

Use el editor de base de datos Orca que se proporciona con el SDK, u otro editor, para abrir la tabla [Directory](directory-table.md) en MNP2001.msi. La estructura de directorios de la actualización de ejemplo es la misma que la estructura de directorios del producto original y no se requiere ningún cambio en la tabla Directory original descrita en Especificación de la estructura [de directorios](specifying-directory-structure.md).
