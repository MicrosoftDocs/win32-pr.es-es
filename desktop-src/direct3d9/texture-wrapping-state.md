# <a name="texture-wrapping-state-direct3d-9"></a>Estado de ajuste de textura (Direct3D 9)

Los Estados de representación de D3DRS \_ WRAP0 a D3DRS \_ WRAP7 habilitan y deshabilitan el ajuste de u y v para varias texturas en cascada de múltiples texturas del dispositivo. Puede establecer estos Estados de representación en una combinación de las \_ marcas D3DWRAPCOORD 0, D3DWRAPCOORD \_ 1, D3DWRAPCOORD \_ 2 y D3DWRAPCOORD \_ 3 para habilitar el ajuste en la primera, segunda, tercera y cuarta dirección de la textura. Use un valor de cero para deshabilitar el ajuste por completo. El ajuste de textura está deshabilitado en todas las direcciones para todas las fases de textura de forma predeterminada.

Para obtener información general conceptual, vea [ajuste de textura (Direct3D 9)](texture-wrapping.md).
