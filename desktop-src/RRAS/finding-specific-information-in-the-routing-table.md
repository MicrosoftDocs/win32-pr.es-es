# <a name="finding-specific-information-in-the-routing-table"></a>Buscar información específica en la tabla de enrutamiento

Los clientes deben poder buscar información específica en la tabla de enrutamiento, en lugar de ser necesarios para enumerar todos los datos. La API RTMv2 proporciona la capacidad de buscar destinos, rutas y próximo saltos específicos en función de determinados criterios.
