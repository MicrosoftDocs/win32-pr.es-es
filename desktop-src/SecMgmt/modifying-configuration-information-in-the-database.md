# <a name="modifying-configuration-information-in-the-database"></a>Modificar la información de configuración en la base de datos

La extensión del complemento de datos adjuntos debe permitir a los usuarios cambiar la información de configuración a través de su nodo Analysis. Esto permite a los usuarios realizar cambios y volver a aplicar la configuración a lo largo del tiempo. La lógica para ello debe ser idéntica a la modificación de la información de los archivos de configuración. Los cambios realizados surtirán efecto en la configuración guardada en la base de datos de seguridad.
