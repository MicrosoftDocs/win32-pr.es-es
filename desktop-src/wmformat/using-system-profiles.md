# <a name="using-system-profiles"></a>Uso de perfiles del sistema

Los perfiles del sistema incluidos con Windows SDK de formato multimedia se diseñaron para cubrir las necesidades básicas de contenido multimedia digital compuesto por audio y vídeo. Debido a la gran variedad de características disponibles y al aumento relacionado de posibles escenarios de medios digitales, se han desfasado los perfiles del sistema. El conjunto más reciente de perfiles del sistema usa la versión 8 Windows códecs multimedia que se publicaron con Windows XP. A pesar de no admitir la versión más reciente de los códecs, los perfiles del sistema son un excelente punto de partida para crear perfiles que se adapten a sus necesidades.

En los temas siguientes se describe cómo usar perfiles del sistema como base para los archivos ASF.
