# <a name="dns-resolvers"></a>Solucionadores DNS

La [*resolución*](r-gly.md) es un componente de software incluido en la pila del Protocolo de control de transmisión de Windows 2000/Protocolo de Internet (TCP/IP) que se comunica con los servidores DNS para crear y resolver consultas de resolución de nombres. La agrupación también está disponible en cualquier cliente que no sea de Microsoft compatible con DNS. Existen resoluciones en clientes DNS y servidores DNS.
