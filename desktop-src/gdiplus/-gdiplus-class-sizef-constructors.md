# <a name="sizefsizef-constructors"></a>SizeF. SizeF (constructores)

En este tema se enumeran los constructores de la clase [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) . Para obtener una lista de clases completa, vea **SizeF (clase**).

### <a name="overload-list"></a>Lista de sobrecarga



| Constructor                                                          | Descripción                                                                                                                                             |
|:---------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SizeF (SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-sizef-sizef(inconstsizef_))            | Crea un objeto [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) e inicializa sus miembros copiando los miembros de otro objeto **SizeF** .<br/> |
| [**SizeF (REAL, REAL)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-sizef-sizef(inreal_inreal)) | Crea un objeto [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) e inicializa sus miembros de datos de **ancho** y **alto** .<br/>                      |
| [**SizeF ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-sizef-sizef)                       | Crea un objeto [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) e inicializa los miembros en cero. Éste es el constructor predeterminado.<br/>           |
