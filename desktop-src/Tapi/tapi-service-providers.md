# <a name="tapi-service-providers"></a>Proveedores de servicios TAPI

Los proveedores de servicios TAPI son una capa de abstracción entre las aplicaciones TAPI y el hardware y los protocolos de transporte subyacentes. Un [proveedor de servicios de telefonía](telephony-service-providers-start-page.md) (TSP) proporciona controles de llamada y un [proveedor de servicios multimedia](./media-service-providers-start-page.md) (MSP) proporciona un control detallado sobre los medios de una llamada. Un par de TSP/MSP puede implementar las capacidades necesarias para las aplicaciones que abarcan desde la telefonía básica hasta las conferencias multimedia de multidifusión IP.
