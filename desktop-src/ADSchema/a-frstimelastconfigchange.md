# <a name="frs-time-last-config-change-attribute"></a>FRS-Time-Last-config-Change (atributo)

| Entrada | Value |
|-------------------|---------------------------------------------------------------|
| CN                | FRS-Time-Last-config-Change                                   |
| Nombre para mostrar de LDAP | fRSTimeLastConfigChange                                       |
| Tamaño              | \-                                                            |
| Actualizar privilegio  | \-                                                            |
| Frecuencia de actualización  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.881                                        |
| System-ID-GUID    | 2a132584-9373-11d1-aebc-0000f80367c1                          |
| Sintaxis            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
