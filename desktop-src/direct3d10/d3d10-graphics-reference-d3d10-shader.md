# <a name="shader-reference-direct3d-10-graphics"></a>Referencia de sombreador (gráficos de Direct3D 10)

La API de Direct3D define varios elementos de API que le ayudarán a crear y administrar sombreadores programables.

-   [Interfaces de los sombreadores](d3d10-graphics-reference-d3d10-shader-interfaces.md)
-   [Funciones del sombreador](d3d10-graphics-reference-d3d10-shader-functions.md)
-   [Estructuras de los sombreadores](d3d10-graphics-reference-d3d10-shader-structures.md)
-   [Enumeraciones de los sombreadores](d3d10-graphics-reference-d3d10-shader-enums.md)
-   [Constantes de sombreador](d3d10-graphics-reference-d3d10-shader-constants.md)

Los sombreadores son programas ejecutables que controlan el sombreador de vértices, el sombreador de geometría y las [fases de canalización](d3d10-graphics-programming-guide-pipeline-stages.md)del sombreador de píxeles. Los sombreadores se programan exclusivamente mediante [HLSL](../direct3dhlsl/dx-graphics-hlsl-reference.md).
