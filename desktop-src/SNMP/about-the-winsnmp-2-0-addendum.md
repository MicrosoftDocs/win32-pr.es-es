# <a name="about-the-winsnmp-20-addendum"></a>Acerca del anexo WinSNMP 2,0

El anexo de la versión WinSNMP 2,0 es una actualización de la especificación de la API de administrador WinSNMP, versión 1.1 a. (En este momento, ninguna especificación define la versión 2,0 de la API WinSNMP). El anexo se desarrolló conjuntamente con la cooperación, la entrada y el soporte técnico de varias empresas, asociaciones y personas.

La especificación WinSNMP 1.1 a define la interfaz de programación de aplicaciones de administrador WinSNMP para desarrollar aplicaciones de administración de red basadas en SNMP. En el anexo de la versión WinSNMP 2,0, se cambia el nombre de la interfaz a la API WinSNMP. Esto se debe a que el WinSNMP 2,0 es compatible con las operaciones SNMP del agente y del administrador. Debe estar familiarizado con ambos documentos si está programando en el entorno WinSNMP.

La implementación de Microsoft WinSNMP es compatible con WinSNMP 2,0. Es compatible con todas las funciones y operaciones definidas tanto en la especificación WinSNMP 1.1 como en el anexo WinSNMP 2,0.
