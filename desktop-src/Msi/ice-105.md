# <a name="ice-105"></a>ICE 105

Valida el paquete con un conjunto de requisitos para la instalación en el contexto por usuario. ICE 105 también puede validar paquetes de doble finalidad. Un paquete de uso doble permite al usuario seleccionar si desea instalar la aplicación en el contexto por usuario o en el contexto por equipo. Para obtener información sobre el desarrollo de un paquete de doble propósito, vea creación de un [solo paquete](single-package-authoring.md).

ICE 105 realiza la siguiente validación del paquete.

-   Comprueba que la tabla [CustomAction](customaction-table.md) no contiene acciones personalizadas que se hayan marcado para ejecutarse con privilegios elevados. Para obtener más información sobre las acciones personalizadas elevadas, vea seguridad de la [acción personalizada](custom-action-security.md).
-   Comprueba que la tabla de [directorio](directory-table.md) no incluye ninguna de las siguientes propiedades de la carpeta del sistema.
    -   [**AdminToolsFolder**](admintoolsfolder.md)
    -   [**CommonAppDataFolder**](commonappdatafolder.md)
    -   [**FontsFolder**](fontsfolder.md)
    -   [**System16Folder**](system16folder.md)
    -   [**System64Folder**](system64folder.md)
    -   [**Carpetadelsistema**](systemfolder.md)
    -   [**TempFolder**](tempfolder.md)
    -   [**WindowsFolder**](windowsfolder.md)
    -   [**WindowsVolume**](windowsvolume.md)
-   Comprueba que el paquete no instala un ensamblado de Common Language Runtime en la caché de ensamblados global (GAC). Para obtener más información sobre cómo instalar ensamblados en la caché global de ensamblados, vea [Agregar ensamblados a un paquete](adding-assemblies-to-a-package.md) e [Instalar ensamblados de Common Language Runtime](installation-of-common-language-runtime-assemblies.md).
-   Comprueba la tabla [ODBCDataSource](odbcdatasource-table.md) para comprobar que el paquete no instala ningún origen de datos.
-   Comprueba la tabla [ServiceInstall](serviceinstall-table.md) para comprobar que el paquete no instala ningún servicio.
-   Comprueba que la tabla [del registro](registry-table.md) no escribe entradas en la clave del **\_ \_ equipo local HKEY** .

**[Windows Installer 4,5 o una versión anterior](not-supported-in-windows-installer-4-5.md):** No compatible. Esta ICE está disponible a partir de Windows Installer 5,0.
