# <a name="constants-structured-storage"></a>Constantes (almacenamiento estructurado)

Las constantes, en almacenamiento estructurado, son una secuencia de valores con nombre. Las constantes permiten describir objetos en lenguaje natural.

Las constantes de almacenamiento estructurado se describen en las siguientes secciones:

-   [Constantes de PROPSETFLAG](propsetflag-constants.md)
-   [Constantes de STGM](stgm-constants.md)
