# <a name="photo-attribute"></a>atributo Photo

Objeto codificado en el fax G3, tal como se explica en la Recomendación T. 4, con un contenedor ASN. 1 para que sea compatible con una BodyPart X. 400, tal y como se define en X. 420.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | álbum                                                 |
| Nombre para mostrar de LDAP | álbum                                                 |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 0.9.2342.19200300.100.1.7                             |
| System-ID-GUID    | 9c979768-ba1a-4c08-9632-c6a5c1ed649a                  |
