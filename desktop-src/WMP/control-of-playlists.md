# <a name="control-of-playlists"></a>Control de listas de reproducción

El cliente mantiene el control de las listas de reproducción. Windows Media Player analiza las listas de reproducción, que realiza acciones basadas en los elementos de la lista de reproducción. El cliente tiene el control de la experiencia de streaming.
