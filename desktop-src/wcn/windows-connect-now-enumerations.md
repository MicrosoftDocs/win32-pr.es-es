# <a name="windows-connect-now-enumerations"></a>Windows Connect Now Enumeraciones

Windows Connect Now define los siguientes tipos de enumeración.



| Enumeración                                                                                             | Descripción                                                                                                                   |
|---------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------|
| [**TIPO DE \_ ATRIBUTO WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_attribute_type)                                                      | Define los distintos tipos de búfer de atributo definidos para Wi-Fi programa de instalación protegida.<br/>                                      |
| [**TIPO DE CONTRASEÑA DE WCN \_ \_**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_password_type)                                                        | Define la autenticación que se usará en una sesión de WPS. <br/>                                                    |
| [**ESTADO DE SESIÓN DE WCN \_ \_**](/windows/win32/api/wcndevice/ne-wcndevice-wcn_session_status)                                                      | Define el estado de una sesión de WPS.<br/>                                                                               |
| [**ESTADO DE \_ ASOCIACIÓN \_ DEL TIPO DE \_ VALOR \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_association_state)                        | Define los posibles estados de asociación de una estación inalámbrica durante una solicitud de detección.<br/>                          |
| [**TIPO DE \_ VALOR WCN \_ \_ BOOLEAN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_boolean)                                             | Define los valores usados para representar las condiciones verdaderas o falsas detectadas durante la instalación y asociación del dispositivo.<br/>            |
| [**TIPO DE \_ AUTENTICACIÓN \_ DE TIPO DE \_ VALOR WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_authentication_type)                    | Define los métodos de autenticación admitidos por el inscrito (punto de acceso o estación).<br/>                            |
| [**MÉTODOS DE CONFIGURACIÓN \_ DEL TIPO DE VALOR \_ \_ WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_config_methods)                              | Define los métodos de configuración admitidos por el registrador o el inscritor.<br/>                                          |
| [**ERROR DE CONFIGURACIÓN \_ DEL TIPO DE VALOR \_ WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_configuration_error)                    | Define los posibles valores de error devueltos a un dispositivo al intentar configurar y asociar a la WLAN.<br/> |
| [**ID. DE \_ CONTRASEÑA DEL DISPOSITIVO DEL TIPO DE VALOR \_ \_ \_ WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_device_password_id)                     | Define los posibles valores de contraseña.<br/>                                                                              |
| [**TIPO DE \_ CIFRADO DE TIPO DE VALOR \_ \_ \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_encryption_type)                            | Define los tipos de cifrado WLAN admitidos.<br/>                                                                       |
| [**BANDAS DE RADIO DE \_ TIPO \_ DE VALOR WCN \_ \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_rf_bands)                                          | Define la banda de radiofrecuencia en la que un inscrito envía solicitudes de detección.<br/>                                    |
| [**VERSIÓN DEL TIPO \_ DE \_ VALOR WCN \_**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_version)                                             | Define la versión admitida de Wi-Fi protected setup (WPS).<br/>                                                      |
| [**ESTADO DE CONFIGURACIÓN \_ PROTEGIDA \_ POR \_ WI-FI DEL TIPO \_ \_ DE VALOR \_ \_ WCN**](/windows/win32/api/wcntypes/ne-wcntypes-wcn_value_type_wi_fi_protected_setup_state) | Define valores que indican si un dispositivo está configurado.<br/>                                                            |
