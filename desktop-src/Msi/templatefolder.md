# <a name="templatefolder-property"></a>Propiedad TemplateFolder

El instalador establece la **propiedad TemplateFolder** en la ruta de acceso completa a la carpeta Template del usuario actual.

Solo Windows 2000, si se establece la propiedad [**ALLUSERS,**](allusers.md) esta propiedad apunta a un directorio del sistema de archivos que contiene las plantillas que están disponibles para todos los usuarios.
