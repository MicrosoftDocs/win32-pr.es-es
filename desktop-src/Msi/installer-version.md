# <a name="installerversion-property"></a>Installer.Version, propiedad

La **propiedad Version** del objeto [**Installer**](installer-object.md) es una propiedad de solo lectura que es la representación de cadena de la versión actual de Windows Installer. La cadena se devuelve en el formato siguiente.

*principal.* *secundaria.* *compile*. *actualización*

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Installer.Version
```



## <a name="property-value"></a>Valor de propiedad
