# <a name="midl-structures"></a>Estructuras MIDL

La semántica normal de C se aplica a los campos de tipos base. Los campos de tipos más complejos, como punteros, matrices y otros tipos construidos, se pueden modificar mediante atributos de tipo o [**campo \_**](/windows/desktop/Rpc/field-attributes). Para obtener más información, vea [**struct**](struct.md).
