# <a name="featureinfotitle-property"></a>Propiedad FeatureInfo.Title

La **propiedad Title** es una propiedad de solo lectura que devuelve el título de la característica en la columna Title de la tabla [Feature](feature-table.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = FeatureInfo.Title
```
