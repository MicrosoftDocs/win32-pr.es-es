# <a name="locale_sshorttime"></a>LOCALE \_ SSHORTTIME

**Windows 7 y versiones posteriores:** Cadena de formato de poco tiempo para la configuración regional. Normalmente, los patrones se derivan quitando el valor "ss" (segundos) del patrón de formato de tiempo largo. Por ejemplo, si el formato de tiempo largo es "h:mm:ss tt", lo más probable es que el formato de hora corta sea "h:mm tt". Esta constante puede especificar varios formatos en una lista delimitada por punto y coma. Sin embargo, el formato de tiempo corto preferido debe ser el primer valor enumerado.
