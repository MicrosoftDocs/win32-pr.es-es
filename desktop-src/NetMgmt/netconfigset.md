# <a name="netconfigset"></a>NetConfigSet

La función **NetConfigSet** está obsoleta. Se incluye por compatibilidad con las versiones de 16 bits de Windows. Otras aplicaciones deben usar el [registro](/windows/desktop/SysInfo/registry).
