| Constante o valor                                                                                                                                                                                                                                                                                  | Descripción                                                                                                                                                                  |

| <span id="TF_FLOATINGLANGBAR_WNDTITLEW"></span><span id="tf_floatinglangbar_wndtitlew"></span><dl> <dt>**TF \_ FLOATINGLANGBAR \_ WNDTITLEW**</dt> <dt>LTF \_ FLOATINGLANGBAR \_ WndTitle</dt> </dl> | La cadena de caracteres Unicode L "TF \_ FloatingLangBar \_ WndTitle" para el título.<br/>                                                                                  |
