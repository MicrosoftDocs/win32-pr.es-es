# <a name="status-color"></a>Color de estado

Se debe especificar el valor de color del texto para mostrar el estado. El valor debe ser un número entero positivo separado por comas. Los valores representan la cantidad de rojo, verde y azul para un color de texto e oscilan entre 0 y 255.
