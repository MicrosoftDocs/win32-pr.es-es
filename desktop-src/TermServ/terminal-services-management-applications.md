# <a name="remote-desktop-services-management-applications"></a>Servicios de Escritorio remoto de administración de aplicaciones

Servicios de Escritorio remoto admite las siguientes aplicaciones de administración.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[Extensión ADSI para la configuración Servicios de Escritorio remoto usuario](adsi-extension-for-terminal-services-user-configuration.md)
</dt> <dd>

Una Active Directory interfaz de servicio (ADSI) es un objeto COM especial que permite la extensión de un objeto COM.

</dd> <dt>

[Servicios de Escritorio remoto permisos](terminal-services-permissions.md)
</dt> <dd>

Puede usar los permisos proporcionados para Servicios de Escritorio remoto controlar cómo los usuarios y grupos acceden al servidor.

</dd> </dl>

El [Servicios de Escritorio remoto WMI](terminal-services-wmi-provider.md) proporciona acceso WMI a la información que los administradores pueden configurar mediante el complemento Servicios de Escritorio remoto Configuration/Connections Microsoft Management Console (MMC).

La [extensión ADSI](adsi-extension-for-terminal-services-user-configuration.md) para Servicios de Escritorio remoto User Configuration amplía un objeto ADSI que permite a los administradores de dominio escribir scripts que automatizarán el mantenimiento de Servicios de Escritorio remoto propiedades de usuario específicas.

Para obtener una lista de los permisos que puede establecer y las tareas que permiten los permisos, [Servicios de Escritorio remoto permisos](terminal-services-permissions.md).

Para obtener más información, vea Servicios de Escritorio remoto [referencia del](terminal-services-wmi-provider-reference.md) proveedor WMI y Uso de la extensión [ADSI para Servicios de Escritorio remoto configuración de usuario](using-the-adsi-extension-for-terminal-services-user-configuration.md).
