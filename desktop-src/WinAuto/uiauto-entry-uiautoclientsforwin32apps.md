# <a name="ui-automation-clients"></a>Clientes de UI Automation

En esta sección se describen las interfaces y funciones de los clientes de automatización de la interfaz de usuario basados en COM no administrados.

> [!Note]  
> Las interfaces de cliente de automatización de la interfaz de usuario reemplazan por completo la funcionalidad proporcionada previamente por las funciones de patrón de control y las funciones de nodo de automatización de la interfaz de usuario, que están obsoletas y no deben usarse.
