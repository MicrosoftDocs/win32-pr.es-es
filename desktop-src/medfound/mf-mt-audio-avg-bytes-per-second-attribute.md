# <a name="mf_mt_audio_avg_bytes_per_second-attribute"></a>\_Atributo de \_ promedio de bytes de audio MF MT \_ \_ \_ por \_ segundo

Número promedio de bytes por segundo en un tipo de medio de audio.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

Este atributo corresponde al miembro **nAvgBytesPerSec** de la estructura [**WAVEFORMATEX**](/previous-versions/dd757713(v=vs.85)) .

La constante GUID para este atributo se exporta desde mfuuid. lib.
