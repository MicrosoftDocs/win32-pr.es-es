# <a name="installerversion-property"></a>Propiedad Installer. version

La propiedad **version** del objeto [**Installer**](installer-object.md) es una propiedad de solo lectura que es la representación de cadena de la versión actual de Windows Installer. La cadena se devuelve en el formato siguiente.

*principal*. *secundaria*. *compilación*. *actualización* de

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Installer.Version
```



## <a name="property-value"></a>Valor de propiedad

|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versión<br/> | Windows Installer 5,0 en Windows Server 2012, Windows 8, Windows Server 2008 R2 o Windows 7. Windows Installer 4,0 o Windows Installer 4,5 en Windows Server 2008 o Windows Vista. Windows Installer en Windows Server 2003 o Windows XP<br/> |
