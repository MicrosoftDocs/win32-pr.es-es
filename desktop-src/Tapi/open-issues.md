# <a name="open-issues"></a>Problemas abiertos

-   Antes de la versión final de Windows 2000, la ruta de cierre de la llamada puede cambiar ligeramente para facilitar el apagado de una llamada a una operación asincrónica (ayuda a protegerse contra interbloqueos para los MSP que admiten vídeo).
-   Los métodos y miembros de la clase de subproceso de trabajo, las clases base de terminal y los terminales estáticos derivados solo se documentan mediante los propios archivos de encabezado.
