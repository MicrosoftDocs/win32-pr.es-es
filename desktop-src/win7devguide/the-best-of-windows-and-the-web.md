# <a name="the-best-of-windows-and-the-web"></a>Lo mejor de Windows y la web

Con Windows 7, es fácil conectarse y permanecer conectado. Redes de Windows ofrece a los desarrolladores opciones para mejorar el almacenamiento en caché y el uso compartido de datos para mejorar el rendimiento de la red. Diagnósticos de red de Windows 7 proporciona a los desarrolladores información relevante para la supervisión de problemas de red. Si el objetivo es conectar dispositivos, conectarse a una red o simplemente entender el estado de la conexión, Windows 7 proporciona una plataforma mejorada.
