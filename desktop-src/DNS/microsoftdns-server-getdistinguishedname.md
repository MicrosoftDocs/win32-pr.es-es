# <a name="getdistinguishedname-method-of-the-microsoftdns_server-class"></a>Método GetDistinguishedName de la clase de servidor MicrosoftDNS \_

El **método GetDistinguishedName** recupera el nombre distintivo de DS para la zona.

## <a name="syntax"></a>Sintaxis


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve el nombre distintivo de DS para la zona.
