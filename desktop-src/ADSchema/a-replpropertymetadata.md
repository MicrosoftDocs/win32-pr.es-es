# <a name="repl-property-meta-data-attribute"></a>Atributo Repl-Property-Meta-Data

Realiza un seguimiento de la información de estado de replicación interna de los objetos DS. La información aquí se puede extraer en formato público a través de la API pública DsReplicaGetInfo(). Presente en todos los objetos DS.



| Entrada | Valor |
|-------------------|------------------------------------------------------------------------------|
| CN                | Repl-Property-Meta-Data                                                      |
| Ldap-Display-Name | replPropertyMetaData                                                         |
| Size              | La longitud es proporcional al número de atributos replicables en el objeto. |
| Actualizar privilegios  | El sistema establece este valor.                                             |
| Frecuencia de actualización  | Cambios en respuesta a cualquier cambio de replicación en el objeto .                |
| Attribute-Id      | 1.2.840.113556.1.4.3                                                         |
| System-Id-Guid    | 281416c0-1968-11d0-a28f-00aa003049e2                                         |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)                        |
