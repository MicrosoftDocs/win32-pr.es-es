| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MSMQ-Foreign                         |
| Ldap-Display-Name | mSMQForeign                          |
| Size              | \-                                   |
| Actualizar privilegios  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.934               |
| System-Id-Guid    | 9a0dc32f-c100-11d1-él5-0080c76670c0 |
