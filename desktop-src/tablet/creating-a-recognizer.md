# <a name="creating-a-recognizer"></a>Crear un reconocedor

Si decide hacerlo, la aplicación puede proporcionar sus propias implementaciones de reconocedor. En esta sección se describen los requisitos para crear un reconocedor de aplicación personalizado para la aplicación que se puede usar con la API de la plataforma de Tablet PC.

> [!Note]  
> Los reconocedores de aplicaciones personalizados no se usan en el panel de entrada de Tablet PC ni en Microsoft Windows Journal. Estos accesorios usan solo los reconocedores con los que se han probado.
