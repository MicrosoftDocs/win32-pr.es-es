# <a name="summaryinfopropertycount-property"></a>SummaryInfo.PropertyCount, propiedad

La **propiedad PropertyCount** del objeto [**SummaryInfo**](summaryinfo-object.md) es una propiedad de solo lectura que indica el número actual de valores de propiedad en el objeto de información de resumen. Tiene en cuenta las propiedades que se han agregado, eliminado o reemplazado.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = SummaryInfo.PropertyCount
```
