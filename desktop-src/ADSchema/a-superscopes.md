# <a name="super-scopes-attribute"></a>Super-Scopes atributo

Este atributo se usa para agrupar todos los distintos ámbitos usados en la clase DHCP en una sola entidad.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Super-Scopes                         |
| Ldap-Display-Name | Superámbitos                          |
| Size              | \-                                   |
| Privilegio actualizar  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.710               |
| System-Id-Guid    | 963d274b-48be-11d1-a9c3-0000f80367c1 |
| Syntax            | [**String(IA5)**](s-string-ia5.md)  |
