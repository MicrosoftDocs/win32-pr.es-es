# <a name="linetolllistoption_-constants"></a>Constantes de LINETOLLLISTOPTION \_

Las constantes de marcador de bits **LINETOLLLISTOPTION \_** describen las opciones para manipular una lista de peajes.

<dl> <dt>

<span id="LINETOLLLISTOPTION_ADD"></span><span id="linetolllistoption_add"></span>**LINETOLLLISTOPTION \_ Agregar**
</dt> <dd> <dl> <dt>



Se agregará un prefijo a la lista de peajes.


</dt> </dl> </dd> <dt>

<span id="LINETOLLLISTOPTION_REMOVE"></span><span id="linetolllistoption_remove"></span>**\_quitar LINETOLLLISTOPTION**
</dt> <dd> <dl> <dt>



Se debe quitar un prefijo de la lista de peajes.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Observaciones

Sin extensibilidad. Todos los 32 bits están reservados.
