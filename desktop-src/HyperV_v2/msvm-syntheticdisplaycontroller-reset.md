# <a name="reset-method-of-the-msvm_syntheticdisplaycontroller-class"></a>Método Reset de la \_ clase MSVM SyntheticDisplayController

Solicita un restablecimiento.

## <a name="syntax"></a>Sintaxis


```mof
uint32 Reset();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

Este método devuelve uno de los siguientes valores:

<dl> <dt>

**No compatible** (1)
