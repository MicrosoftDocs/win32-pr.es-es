## <a name="remarks"></a>Observaciones

> [!Note]  
> Comctl32.dll versión 6 no es redistribuible, pero se incluye en Windows. Para usar Comctl32.dll versión 6, especifíquelo en un manifiesto. Para obtener más información sobre los manifiestos, vea [habilitar estilos visuales](cookbook-overview.md).

 

Un elemento se vuelve más *rápido* cuando se mantiene el mouse sobre él. Si está habilitado el seguimiento activo, los elementos activos se resaltarán, pero el usuario todavía debe hacer clic en el elemento al menos una vez para seleccionarlo. La *activación* de un elemento hace que el control de vista de lista envíe una notificación [ \_ ITEMACTIVATE de LVN](lvn-itemactivate.md) . Se puede seleccionar un *elemento cuando se* encuentra en un estado en el que un solo clic lo seleccionará.

Si se establece el estilo **LVS \_ ex \_ ONECLICKACTIVATE** , solo se necesita un clic para seleccionar cualquier elemento, por lo que se pueden seleccionar todos los elementos. Si se establece el estilo **LVS \_ ex \_ TWOCLICKACTIVATE** , se necesitan dos clics para seleccionar un elemento. Con este estilo, el elemento se puede seleccionar solo después de haber hecho clic en él una vez.

Al establecer el estilo de **LVS \_ ex \_ UNDERLINEHOT** , se puede subrayar el texto de cualquier elemento activo que se pueda seleccionar. Al establecer el estilo de **LVS \_ ex \_ UNDERLINECOLD** , se subrayará el texto de los elementos que no están activos, pero que pueden estar seleccionados.
