# <a name="relation-to-other-services"></a>Relación con otros servicios

Otras partes del marco de seguridad de Internet de Microsoft usan el [*subsistema de tarjeta inteligente*](../secgloss/s-gly.md), tal como se muestra en la siguiente ilustración. (Debido a las restricciones de exportación de los Estados Unidos, un [*proveedor de servicios criptográficos*](../secgloss/c-gly.md) (CSP) que usa un SCSP para comunicar solicitudes relacionadas con el cifrado debe firmar y comprobar el SCSP).

![relación de Microsoft Internet Security Framework con el subsistema de tarjeta inteligente](images/smartovr3.png)

Para obtener más información sobre el rol del subsistema de tarjeta inteligente en Microsoft Internet Security Framework, consulte [https://www.microsoft.com/whdc/device/input/smartcard/default.mspx](https://www.microsoft.com/whdc/device/input/smartcard/default.mspx) .
