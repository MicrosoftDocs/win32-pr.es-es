# <a name="translating-to-vbscript"></a>Traducir a VBScript

VBScript es un subconjunto del lenguaje Microsoft Visual Basic para Aplicaciones. Dado que VBScript está diseñado para ser seguro para su uso en aplicaciones del lado cliente, como páginas web, no proporciona entradas y salidas de archivos ni acceso directo al sistema operativo subyacente. Esto impide que un script basado en Web elimine o modifique archivos del equipo. Para obtener una lista completa de las características de Visual Basic que no se admiten en VBScript, consulte la documentación de Visual Basic para Aplicaciones.

Además de las características de lenguaje que proporciona VBScript, el host de scripting también puede exponer un modelo de objetos de VBScript al que el script puede tener acceso. Por ejemplo, Internet Explorer expone un modelo de objetos que permite que los scripts interactúen con el explorador y lo controlen mediante programación. Para obtener más información sobre cómo trabajar con estos modelos de objetos, consulte la documentación del host de scripting.
