# <a name="mobile-computing-configurations-for-sens"></a>Configuraciones de informática móvil para SENS

Las funciones de conectividad y las notificaciones son útiles para los equipos configurados de la siguiente manera:

-   Un equipo móvil que se usa en una estación de acoplamiento en una red de ancho de banda alto que puede usar ocasionalmente una conexión de acceso telefónico.
-   Un equipo móvil que usa una conexión de acceso telefónico exclusivamente.
-   Un equipo de escritorio que usa una conexión de acceso telefónico exclusivamente.
-   Un equipo de escritorio conectado a una red de ancho de banda alto con problemas de latencia.

En cada una de estas configuraciones, una aplicación puede usar el ancho de banda de conexión y la información de latencia para optimizar dinámicamente sus operaciones para la disponibilidad de la red.
