# <a name="effectseffecttitle"></a>EFFECTS.effectTitle

El **método effectTitle** recupera el título de presentación de la visualización con el índice del Registro especificado.

``` syntax
        elementID.effectTitle(index)
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*Índice*
</dt> <dd>

**Number** (**long**) que contiene el índice del Registro de una visualización.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Este método devuelve una **cadena**.

## <a name="remarks"></a>Comentarios

Este método se usa para mostrar títulos de visualización en una interfaz de usuario.

| Requisito | Value |
|--------------------|---------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media serie 9 o posterior<br/> |
