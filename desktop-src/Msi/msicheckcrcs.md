# <a name="msicheckcrcs-property"></a>Propiedad MSICHECKCRCS

La comprobación de redundancia cíclica (CRC) está disponible con Windows Installer versión 2,0 y posteriores. El instalador realiza una CRC en archivos solo si se establece la propiedad **MSICHECKCRCS** . Para obtener más información, consulte [comprobación de CRC durante una instalación](crc-checking-during-an-installation.md).

Windows Installer en Windows Server 2003 o Windows XP. Consulte los [requisitos de Run-Time de Windows Installer](windows-installer-portal.md) para obtener información sobre la Service Pack mínima de Windows que requiere una versión Windows Installer.<br/> |
