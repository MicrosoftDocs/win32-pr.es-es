# <a name="wmasfpacketcount"></a>WM/ASFPacketCount

El atributo **WM/ASFPacketCount** contiene un recuento de los paquetes en la sección de datos del archivo ASF.

## <a name="global-constant"></a>Constante global

g \_ wszWMASFPacketCount

## <a name="data-type"></a>Tipo de datos

**tipo de WMT. \_ \_**

## <a name="remarks"></a>Observaciones

Este atributo es de solo lectura y se aplica a todo el archivo (secuencia 0).

Solo puede recuperar este atributo mediante los métodos de la interfaz [**IWMHeaderInfo3**](/previous-versions/windows/desktop/api/wmsdkidl/nn-wmsdkidl-iwmheaderinfo3) del objeto de editor de metadatos.
