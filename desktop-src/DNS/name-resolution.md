# <a name="name-resolution"></a>Resolución de nombres

El propósito del DNS es proporcionar la resolución de nombres: el proceso de comparar un nombre de host (por ejemplo, un nombre de equipo) con una lista de RRs e identificar la dirección IP correspondiente. La resolución de nombres inversa es el proceso de comparar una dirección IP con una lista de RRs e identificar el nombre de host correspondiente. Estos procesos de resolución se inician cuando un servidor DNS recibe una consulta.
