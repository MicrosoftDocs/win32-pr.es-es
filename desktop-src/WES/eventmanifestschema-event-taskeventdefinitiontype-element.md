# <a name="event-taskeventdefinitiontype-element"></a>elemento event (TaskEventDefinitionType)

\[A partir del compilador de mensajes que se incluye Windows versión 7 del SDK de Windows, este elemento ya no está disponible.\]

Define un evento para un evento específico de la tarea.

``` syntax
<xs:element name="event"
    type="EventDefinitionType"
 />
```

El **tipo** complejo [**TaskEventDefinitionType**](eventmanifestschema-taskeventdefinitiontype-complextype.md) define el elemento de evento.
