# <a name="mf_bytestream_content_type-attribute"></a>\_Atributo de \_ tipo de contenido MF BYTESTREAM \_

Especifica el tipo MIME de una secuencia de bytes.

## <a name="data-type"></a>Tipo de datos

Cadena de caracteres anchos

## <a name="remarks"></a>Observaciones

Para obtener el valor del atributo, consulte el objeto de flujo de bytes para la interfaz [**IMFAttributes**](/windows/desktop/api/mfobjects/nn-mfobjects-imfattributes) .

La constante GUID para este atributo se exporta desde mfuuid. lib.
