# <a name="error-handling-cryptography"></a>Control de errores (criptografía)

En todos los ejemplos de C que acompañan a la documentación de la función, se usa la función de ejemplo [**MyHandleError**](myhandleerror.md) para imprimir un mensaje de error y salir de un procedimiento. En situaciones reales, es posible que se requieran procedimientos de control de errores más extensos. Para obtener más información, vea [ejemplos de limitaciones de programas](example-program-limitations.md).
