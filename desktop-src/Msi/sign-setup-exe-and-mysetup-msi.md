# <a name="sign-setupexe-and-mysetupmsi"></a>Firmar Setup.exe y MySetup.msi

Antes de colocar Setup.exe y MySetup.msi en el servidor Web, debe firmar los archivos con el certificado digital y la clave privada, MyCert. cer y MyCert. PVK, mediante la utilidad de [SignTool](../seccrypto/signtool.md) . Para obtener más información acerca del uso de la utilidad de SignTool, consulte [referencia de las herramientas de CryptoAPI](../seccrypto/cryptoapi-tools-reference.md) en el kit de desarrollo de software (SDK) de Microsoft Windows.
