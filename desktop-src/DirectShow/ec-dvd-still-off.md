# <a name="ec_dvd_still_off"></a>DVD de EC \_ \_ sin \_ conexión

Señala el final de cualquier todavía (PGC, Cell o VOBU).

## <a name="parameters"></a>Parámetros

<dl> <dt>

<span id="lParam1"></span><span id="lparam1"></span><span id="LPARAM1"></span>*lParam1*
</dt> <dd>

Cero.

</dd> <dt>

<span id="lParam2"></span><span id="lparam2"></span><span id="LPARAM2"></span>*lParam2*
</dt> <dd>

Cero.

</dd> </dl>

## <a name="remarks"></a>Observaciones

Este evento indica que se ha liberado todavía cualquier actualmente activo.

Este evento se desencadena en todos los dominios.
