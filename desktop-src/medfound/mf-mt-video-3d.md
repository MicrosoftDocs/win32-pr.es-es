# <a name="mf_mt_video_3d-attribute"></a>\_ \_ Atributo 3D de vídeo MF MT \_

Especifica si una secuencia de vídeo contiene contenido 3D.

## <a name="data-type"></a>Tipo de datos

**Bool** almacenado como **UINT32**

## <a name="remarks"></a>Observaciones

Este atributo se aplica a los tipos de medios de vídeo. Si este atributo es **true**, el flujo de vídeo contiene contenido 3D. El valor predeterminado es **FALSE**.
