# <a name="adding-a-component-to-a-visual-basic-project"></a>Agregar un componente a un proyecto de Visual Basic

En el procedimiento siguiente se describe cómo agregar un objeto COM como componente a un proyecto Visual Basic. A continuación, la aplicación puede usar las clases de ese objeto.

Al agregar un control como componente, se exponen las propiedades y los métodos del extensor Visual Basic como si formaran parte del control. Por el contrario, cuando se agrega un objeto como referencia, solo se puede usar la biblioteca de tipos proporcionada por el control o la biblioteca de tipos "sin procesar".

**Para insertar un control en un proyecto de Visual Basic**

1.  En el menú **Proyecto**, haga clic en **Componentes**.
2.  Active la casilla situada junto al componente que desee agregar. Si el componente no aparece en la lista, busque el archivo. dll o. ocx mediante **examinar**.
3.  Haga clic en **OK**.

El componente ahora forma parte del proyecto y aparece en la barra de herramientas de objeto. La aplicación puede crear instancias del control arrastrando el control desde la barra de herramientas y colocándolo en un formulario o cuadro de diálogo.
