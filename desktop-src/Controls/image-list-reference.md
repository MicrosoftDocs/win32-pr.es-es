# <a name="image-lists"></a>Listas de imágenes

Esta sección contiene información sobre los elementos de programación utilizados con las listas de imágenes.

### <a name="overviews"></a>Temas de introducción



| Tema                          | Contenido                                                                                                            |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [Listas de imágenes](image-lists.md) | Una lista de imágenes es una colección de imágenes del mismo tamaño, a las que se puede hacer referencia a través de su índice.<br/> |
