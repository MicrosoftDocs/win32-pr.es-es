# <a name="mfpkey_dyn_bandtrunc_qfloor-property"></a>Propiedad MFPKEY \_ DYN \_ BANDTRUNC \_ METERLOOR

Especifica la calidad con la que se permite el ancho de banda máximo codificado. Con una calidad mayor o igual que el valor de esta propiedad, se permite el ancho de banda máximo codificado.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

Solo está disponible mediante [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo de datos

**VT \_ R4**
