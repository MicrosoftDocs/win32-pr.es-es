# <a name="id2d1devicecreateprintcontrol-methods"></a>Métodos ID2D1Device::CreatePrintControl

Crea un [**objeto ID2D1PrintControl**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1printcontrol) que convierte las primitivas [de Direct2D](/windows/desktop/direct2d/direct2d-portal) almacenadas en [**ID2D1CommandList**](/windows/win32/api/d2d1_1/nn-d2d1_1-id2d1commandlist) en una representación de página fija. A continuación, el subs sistema de impresión consume los primitivos.
