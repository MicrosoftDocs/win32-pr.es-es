# <a name="endloop-sm4---asm"></a>ENDLOOP (SM4-ASM)

Finaliza una instrucción de bucle.



| ENDLOOP |
|---------|



 

## <a name="remarks"></a>Observaciones

En el ejemplo siguiente se muestra cómo utilizar esta instrucción.

``` syntax
                loop
                    // example of termination condition
                    if_nz r0.x
                        break
                    endif
                    ...
                endloop
```

El formato de token contiene el desplazamiento de la instrucción de bucle correspondiente en el sombreador como una comodidad.

Esta instrucción se aplica a las siguientes fases del sombreador:



| Sombreador de vértices | Sombreador de geometría | Sombreador de píxeles |
|---------------|-----------------|--------------|
| x             | x               | x            |
