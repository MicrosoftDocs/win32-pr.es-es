# <a name="iwebproxy-methods"></a>Métodos IWebProxy

La [**interfaz IWebProxy**](/windows/desktop/api/Wuapi/nn-wuapi-iwebproxy) define los métodos siguientes.



| Método                                                                         | Descripción                                                         |
|--------------------------------------------------------------------------------|---------------------------------------------------------------------|
| [**PromptForCredentials**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentials)                 | Solicita al usuario la contraseña que se usará para la autenticación de proxy.  |
| [**PromptForCredentialsFromHwnd**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-promptforcredentialsfromhwnd) | Solicita al usuario la contraseña que se usará para la autenticación de proxy.  |
| [**SetPassword**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-setpassword)                                   | Establece la contraseña que se envía al servidor proxy para la autenticación. |
