# <a name="obsolete-wcs-functions"></a>Funciones de WCS obsoletas

En esta sección se delimitan las funciones que están obsoletas en WCS 1,0. Actualmente se conservan por razones de compatibilidad con versiones anteriores. Sin embargo, se quitarán de la API en versiones posteriores de WCS.



|                                            |                                              |
|--------------------------------------------|----------------------------------------------|
| [**UpdateICMRegKey**](/windows/desktop/api/Wingdi/nf-wingdi-updateicmregkeya) | *Obsoleto*: administra los perfiles de color y CMMS. |
