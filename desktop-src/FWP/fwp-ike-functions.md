# <a name="ikeauthip-functions"></a>Funciones IKE/AuthIP

Las funciones de Windows Filtering Platform (WFP) que se usan para interactuar con los módulos de claves(Internet Key Exchange (IKE), Internet Key Exchange v2 (IKEv2) y protocolo de Internet autenticado (AuthIP)) son las siguientes.

-   IxtGetStatistics:
    -   [**IxtGetStatistics0**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextgetstatistics0) (Windows Vista)
    -   [**IxtGetStatistics1**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextgetstatistics1) (Windows 7 y versiones posteriores)
-   [**IxtSaCreateEnumHandle0**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextsacreateenumhandle0)
-   [**IxtSaDbGetSecurityInfo0**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextsadbgetsecurityinfo0)
-   [**IxtSaDbSetSecurityInfo0**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextsadbsetsecurityinfo0)
-   [**IxtSaDeleteById0**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextsadeletebyid0)
-   [**IxtSaDestroyEnumHandle0**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextsadestroyenumhandle0)
-   IxtSaEnum:
    -   [**IxtSaEnum0**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextsaenum0) (Windows Vista)
    -   [**IxtSaEnum1**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextsaenum1) (Windows 7)
    -   [**IxtSaEnum2**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextsaenum2) (Windows 8)
-   IxtSaGetById:
    -   [**IxtSaGetById0**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextsagetbyid0) (Windows Vista)
    -   [**IxtSaGetById1**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextsagetbyid1) (Windows 7)
    -   [**IxtSaGetById2**](/windows/desktop/api/Fwpmu/nf-fwpmu-ikeextsagetbyid2) (Windows 8)
