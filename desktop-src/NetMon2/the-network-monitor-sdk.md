# <a name="network-monitor-introduction"></a>Introducción a Monitor de red

La parte Monitor de red del kit de desarrollo de software (SDK) de la plataforma amplía la funcionalidad de Monitor de red al ayudarle a crear expertos personalizados, analizadores de protocolos y proveedores de paquetes de red (NPPs). En esta sección se incluye la siguiente documentación:

-   Temas de programación necesarios para crear expertos y analizadores. Esta información también explica cómo escribir un filtro de captura y usar blobs y NPPs.
-   Información de referencia relacionada con la API sobre expertos, analizadores, blobs y NPPs.
-   Un glosario de términos usados en la documentación de Monitor de red.

Los subtemas siguientes se proporcionan para ayudar a obtener e instalar Monitor de red.

-   [Requisitos del sistema](system-requirements.md)
-   [Obtención de Monitor de red 2,1](obtaining-network-monitor-2-1.md)
-   [Instalación de archivos de Monitor de red](installing-network-monitor-files.md)
