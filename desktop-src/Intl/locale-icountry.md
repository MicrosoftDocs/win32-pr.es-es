# <a name="locale_idialingcode-and-locale_icountry"></a>Configuración regional \_ IDIALINGCODE y configuración regional \_ ICOUNTRY

Código de país o región, basado en códigos de teléfono internacionales, también denominados códigos de país o región de IBM. El número máximo de caracteres permitido para esta cadena es seis, incluido un carácter nulo de terminación.

Windows 10 agregó la configuración regional \_ IDIALINGCODE como sinónimo con nombre más preciso para la configuración regional \_ ICOUNTRY. Le recomendamos que use el nuevo nombre para mejorar la legibilidad del código.

> [!Note]  
> Los códigos de marcado internacionales no son exclusivos de un país determinado. Por ejemplo, varios países, incluidos el Estados Unidos y Canadá, usan "1" como código de marcado internacional.
