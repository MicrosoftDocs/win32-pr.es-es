# <a name="attribute-security-guid-attribute"></a>Atributo Attribute-Security-GUID

GUID que se va a usar para aplicar credenciales de seguridad a un conjunto de objetos.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Attribute-Security-GUID                               |
| Ldap-Display-Name | attributeSecurityGUID                                 |
| Size              | \-                                                    |
| Actualizar privilegios  | Administrador de esquemas                                  |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.149                                |
| System-Id-Guid    | bf967924-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
