# <a name="font-embedding-reference"></a>Referencia de incrustación de fuentes

La biblioteca de servicios de inserción de fuentes proporciona un mecanismo para agrupar fuentes TrueType y Microsoft OpenType en un documento o archivo. Normalmente, un documento que contiene fuentes incrustadas necesita esas fuentes para representar el documento en otro equipo. La inserción de una fuente garantiza que una fuente especificada en un archivo estará presente en el equipo que recibe el archivo. Sin embargo, algunas fuentes no se pueden mover a otros equipos debido a problemas de copyright que limitan la distribución.

Los siguientes elementos se usan con Font Embedding Services.

-   [Funciones de incrustación de fuentes](font-embedding-functions.md)
-   [Estructuras de incrustación de fuentes](font-embedding-services-structures.md)
-   [Mensajes de error de incrustación de fuentes](font-embedding-function-error-messages.md)
