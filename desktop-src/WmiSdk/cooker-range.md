# <a name="cooker_range"></a>\_intervalo Cooker

La \_ fórmula de tipo de contador de intervalo Cooker muestra la diferencia entre los valores [mínimo](cooker-min.md) y [máximo](cooker-max.md) de un conjunto de observaciones sin formato de una propiedad en una instancia de [**\_ PerfRawData de Win32**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) . Este tipo de contador solo se define en WMI y no está disponible para las tecnologías de supervisión de rendimiento, como los [contadores de rendimiento versión 6,0](/windows/desktop/PerfCtrs/performance-counters-portal).

Para obtener más información sobre la fórmula de tipo de contador, vea [tipos de contadores](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Para obtener más información sobre los proveedores y scripts de alto rendimiento, consulte [tipos de contadores de rendimiento de WMI](wmi-performance-counter-types.md).
