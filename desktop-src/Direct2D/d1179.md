# <a name="d1179-gradient-stop-collection-zero-stops"></a>D1179: puntos cero de la colección de delimitadores de degradado

La creación de un ID2D1GradientStopCollection requiere al menos un delimitador de degradado.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación ha llamado a ID2D1RenderTarget:: CreateGradientStopCollection, pero ha especificado un recuento de delimitadores de degradado de 0.

## <a name="possible-fixes"></a>Posibles correcciones

Al crear un ID2D1GradientStopCollection, la aplicación debe especificar al menos un delimitador de degradado con su color y posición. Vea ID2D1RenderTarget:: CreateGradientStopCollection para obtener un ejemplo.
