# <a name="previous-versions-of-windows-media-metafiles-deprecated"></a>Versiones anteriores de los metaarchivos de Windows Media (en desuso)

En este tema se documenta una característica que ya no se usa en la versión actual de los metaarchivos de Windows Media.

En esta sección se documenta una parte de la versión 2 de los metaarchivos de Windows Media. La versión 2 no es la versión actual de los metaarchivos de Windows Media, pero hay algunos casos en los que Windows Media Services proporciona a los clientes los metaarchivos que están en el formato de la versión 2. Los metaarchivos de Windows Media también se conocen como archivos de redireccionamiento avanzado de secuencias (ASX).

En esta sección se documentan los elementos siguientes.



| Elemento                                                   | Descripción                                                    |
|--------------------------------------------------------|----------------------------------------------------------------|
| [Elemento Reference](reference-element--deprecated.md) | Contiene una lista de direcciones URL de archivos multimedia.                        |
| [Parámetro MSWMExt](mswmext-parameter--deprecated.md) | Indica a un programa cliente el formato de un archivo al que se hace referencia. |
