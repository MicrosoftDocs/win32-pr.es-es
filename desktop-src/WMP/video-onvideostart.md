# <a name="videoonvideostart"></a>VÍDEO. onvideostart

El controlador de eventos **onvideostart** controla un evento que se produce cuando el vídeo se carga y comienza a representarse.

``` syntax

onvideostart 
```

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
