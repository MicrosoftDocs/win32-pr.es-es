# <a name="animationset"></a>AnimationSet

Contiene uno o varios objetos [**Animation**](animation.md) . Cada animación dentro de un conjunto de animaciones tiene la misma hora en un momento dado. Aumentar el tiempo del conjunto de animaciones aumenta el tiempo de todas las animaciones que contiene.

``` syntax
template AnimationSet
{
    < 3D82AB50-62DA-11cf-AB39-0020AF71E433 >
    [ Animation < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > ]
} 
```

Donde:

-   \[ Animation < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 > \] : plantilla de animación opcional.
