# <a name="configurableitemdescription-property"></a>ConfigurableItem. Description (propiedad)

La propiedad **Description** devuelve el valor de la columna Description de la [tabla ModuleConfiguration](moduleconfiguration-table.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = ConfigurableItem.Description
```
