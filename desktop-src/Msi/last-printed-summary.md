# <a name="last-printed-summary-property"></a>Última propiedad de Resumen impresa

En un paquete de instalación, la última propiedad de **Resumen impresa** se puede establecer en la fecha y hora durante una [instalación administrativa](administrative-installation.md) para registrar Cuándo se creó la imagen administrativa. En el caso de las instalaciones no administrativas, esta propiedad es la misma que la propiedad de [**Resumen de fecha y hora de creación**](create-time-date-summary.md) .

En una transformación, esta propiedad debe ser null.

En un paquete de revisión, esta propiedad debe ser null.
