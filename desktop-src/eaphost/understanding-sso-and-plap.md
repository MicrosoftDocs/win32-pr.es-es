# <a name="sso-and-plap"></a>SSO y PLAP

La API de Sign-On único (SSO) representa un conjunto de métodos que se usan para obtener credenciales específicas del método EAP para una cuenta de equipo o de usuario de red de un modo seguro sin tener que generar varias instancias de la interfaz de usuario.

> [!Note]  
> El término proveedor de acceso previo al inicio de sesión (PLAP) está obsoleto. Si se encuentra "proveedor de acceso previo al inicio de sesión" o "PLAP" en el código o en la documentación, reemplácelo por "Inicio de sesión único" o "SSO", respectivamente.
