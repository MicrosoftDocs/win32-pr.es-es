# <a name="ms-radius-framedipv6prefix-attribute"></a>atributo MS-RADIUS-FramedIpv6Prefix

Indica un prefijo IPv6 (y la ruta correspondiente) que se va a configurar para el usuario.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-RADIUS-FramedIpv6Prefix           |
| Nombre para mostrar de LDAP | msRADIUS-FramedIpv6Prefix            |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1915              |
| System-ID-GUID    | f63ed610-d67c-494d-87be-cd1e24359a38 |
| Sintaxis            | [**String(IA5)**](s-string-ia5.md)  |
