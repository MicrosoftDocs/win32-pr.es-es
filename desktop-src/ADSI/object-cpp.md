# <a name="objectcpp"></a>Objeto. CPP

En el componente proveedor de ejemplo, un ejemplo de código de listas de objetos, incluido el uso de un filtro, está en Object. cpp. En la tabla siguiente se enumeran los métodos admitidos.



| Object                               | Descripción                                 |
|--------------------------------------|---------------------------------------------|
| **ObjectTypeList::ObjectTypeList**   | Creador: cree una lista con el filtro.     |
| **ObjectTypeList:: ~ ObjectTypeList**  | Destructor estándar.                        |
| **ObjectTypeList::GetCurrentObject** | Recupera el objeto actual de la lista.  |
| **ObjectTypeList:: Next**             | Golpear el índice.                             |
| **ObjectTypeList:: RESET**            | Restablezca el índice.                            |
| **IsValidFilter**                    | Compruebe la integridad del filtro.          |
| **BuildDefaultObjectArray**          | Cree una matriz de objetos mediante el filtro. |
| **BuildObjectArray**                 | Cree una matriz de objetos.                      |
