# <a name="eaphost-authenticator-method-apis"></a>API del método de autenticador de EAPHost

La documentación de la API del método de autenticador de EAPHost consta de las siguientes secciones.

-   [Funciones del método de autenticador de EAPHost](eap-host-authenticator-method-functions.md)
-   [Enumeraciones del método de autenticador de EAPHost](eap-host-authenticator-method-enumerations.md)
-   [Estructuras de método de autenticador de EAPHost](eap-host-authenticator-method-structures.md)

Tenga en cuenta que estos elementos se deben implementar en un archivo DLL compatible mediante el autor del método de autenticación de EAP. El archivo DLL con la implementación de estos elementos de la API se carga y se llama desde EAPHost del servidor.
