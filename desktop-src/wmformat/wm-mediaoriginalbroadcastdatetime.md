# <a name="wmmediaoriginalbroadcastdatetime"></a>WM/MediaOriginalBroadcastDateTime

El atributo **WM/MediaOriginalBroadcastDateTime** contiene la fecha y la hora en que se difundió el contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaOriginalBroadcastDateTime

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Este atributo se usa para difundir televisión o radio.
