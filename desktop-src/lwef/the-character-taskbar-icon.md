# <a name="the-character-taskbar-icon"></a>El icono de la barra de tareas carácter

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Si se ha creado un carácter para incluir un icono, el icono aparece en el área de notificación de la barra de tareas cuando se ejecuta el agente de Microsoft. Este icono proporciona acceso al menú emergente del carácter, que proporciona acceso a los comandos globales del agente, como los que ocultan y muestran el carácter.

![área de notificación con el reloj y el icono](images/f1tbicon.gif)

Al mover el puntero sobre el icono de la barra de tareas, se muestra una ventana de sugerencia que refleja el nombre del carácter (en el idioma actual del sistema). Al hacer clic en el icono de la barra de tareas del carácter, se muestra el carácter. La acción asociada con el doble clic en el icono depende de la aplicación actual que controla ese carácter.

Al hacer clic con el botón secundario en el icono, se muestra un menú emergente. Cuando el carácter está visible, el menú emergente muestra los mismos comandos que los que se muestran al hacer clic con el botón secundario en el carácter. Si el carácter está oculto, solo aparece la ventana comandos de voz abrir (o cerrar) y mostrar comandos.
