# <a name="mf_devicestream_stream_category-attribute"></a>Atributo \_ MF DEVICESTREAM \_ STREAM \_ CATEGORY

Representa la categoría de secuencia.

## <a name="data-type"></a>Tipo de datos

**GUID**

## <a name="remarks"></a>Comentarios

Este atributo representa el GUID que identifica la categoría de secuencia definida en ksmedia.h.
