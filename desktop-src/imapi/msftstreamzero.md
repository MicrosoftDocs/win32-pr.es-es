# <a name="msftstreamzero-object"></a>Objeto MsftStreamZero

Utilice este objeto para crear un flujo de datos de solo lectura cuyos datos se inicializan en ceros. Debe llamar al método **setSize** para establecer el tamaño solicitado de la secuencia.

## <a name="members"></a>Miembros

El objeto **MsftStreamZero** no define ningún miembro.
