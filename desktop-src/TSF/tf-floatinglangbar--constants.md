# <a name="tf_floatinglangbar_-constants"></a>Constantes \_ FLOATINGLANGBAR \_ \* de TF

Define una cadena de \_ constantes DE TF FLOATINGLANGBAR \_ \* para el título de la ventana de la barra de lenguaje.



| Constante o valor                                                                                                                                                                                                                                                                                  | Descripción                                                                                                                                                                  |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="TF_FLOATINGLANGBAR_WNDTITLEW"></span><span id="tf_floatinglangbar_wndtitlew"></span><dl> <dt>**TF \_ FLOATINGLANGBAR \_ WNDTITLEW**</dt> <dt>LTF \_ FloatingLangBar \_ WndTitle</dt> </dl> | Cadena de caracteres Unicode L"TF \_ FloatingLangBar \_ WndTitle" para el título.<br/>                                                                                  |
| <span id="TF_FLOATINGLANGBAR_WNDTITLEA"></span><span id="tf_floatinglangbar_wndtitlea"></span><dl> <dt>**TF \_ FLOATINGLANGBAR \_ WNDTITLEA**</dt> <dt>TF \_ FloatingLangBar \_ WndTitle</dt> </dl>  | Cadena de caracteres ANSI "TF \_ FloatingLangBar \_ WndTitle" para el título.<br/>                                                                                      |
| <span id="TF_FLOATINGLANGBAR_WNDTITLE"></span><span id="tf_floatinglangbar_wndtitle"></span><dl> <dt>**TF \_ FLOATINGLANGBAR \_ WNDTITLE**</dt> <dt>TF \_ FLOATINGLANGBAR \_ WNDTITLEA</dt> </dl>    | Si se define Unicode, toma el valor de la constante \_ WNDTITLEW DE TF FLOATINGLANGBAR. \_ De lo contrario, toma el valor de la \_ constante \_ WNDTITLEA DE TF FLOATINGLANGBAR.<br/> |



## <a name="remarks"></a>Comentarios

Aunque hay tres constantes en este grupo, la principal es TF \_ FLOATINGLANGBAR WNDTITLE, que toma el valor ANSI o \_ Unicode, según corresponda.
