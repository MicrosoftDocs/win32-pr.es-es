# <a name="getnumberofchapters-method"></a>Método GetNumberOfChapters

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

El `GetNumberOfChapters` método recupera el número de capítulos del título especificado.

``` syntax
[iChapters = ] MSWebDVD.GetNumberOfChapters(iTitle)
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

<span id="iTitle"></span><span id="ititle"></span><span id="ITITLE"></span>*iTitle*
</dt> <dd>

Especifica el título como un entero.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Devuelve un valor entero comprendido entre 1 y 999 que indica el número de capítulos.
