# <a name="mf_devicestream_stream_category-attribute"></a>\_Atributo de \_ categoría de secuencia MF DEVICESTREAM \_

Representa la categoría de la secuencia.

## <a name="data-type"></a>Tipo de datos

**GUID**

## <a name="remarks"></a>Observaciones

Este atributo representa el GUID que identifica la categoría de secuencia definida en ksmedia. h.
