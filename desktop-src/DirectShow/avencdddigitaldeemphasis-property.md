# <a name="avencdddigitaldeemphasis-property"></a>Propiedad AVEncDDDigitalDeemphasis

Especifica si la desacentuación digital está habilitada en una secuencia de audio Dolby digital. Esta propiedad se aplica a los codificadores de audio Dolby digital.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncDDDigitalDeemphasis**
