# <a name="boundingbox-methods"></a>Métodos BoundingBox



| Método                                                              | Descripción                                                                                            |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Tuviera**](boundingbox-contains.md)<br/>                 | Comprueba si BoundingBox contiene un objeto especificado.<br/>                                  |
| [**CreateFromPoints**](boundingbox-createfrompoints.md)<br/> | Crea un BoundingBox a partir de puntos.<br/>                                                          |
| [**Forma intersección con**](boundingbox-intersects.md)<br/>             | Comprueba el BoundingBox para la intersección con otro objeto.<br/>                                 |
| [**Transform**](boundingbox-transform.md)<br/>               | Transforma el BoundingBox.<br/>                                                                 |
| [**ContainedBy**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>           | Comprueba si el [**BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) está incluido en el frustum especificado.<br/> |
| [**CreateFromSphere**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createfromsphere)<br/> | Crea un BoundingBox lo suficientemente grande como para contener el BoundingSphere especificado.<br/>               |
| [**CreateMerged**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createmerged)<br/>         | Crea un BoundingBox lo suficientemente grande como para contener dos instancias BoundBox especificadas.<br/>             |
| [**GetCorners**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-getcorners)<br/>             | Recupera las esquinas del BoundingBox.<br/>                                                   |
| [**asignación de OP \_**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-operator-assign)<br/>      | Copia los valores de otro BoundingBox.<br/>                                                     |
