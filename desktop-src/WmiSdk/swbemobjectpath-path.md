# <a name="swbemobjectpathpath-property"></a>SWbemObjectPath. Path (propiedad)

La propiedad **path** del objeto [**SWbemObjectPath**](swbemobjectpath.md) contiene la ruta de acceso absoluta. Es el mismo que la propiedad [ \_ \_ path](wmi-system-properties.md) de la API com. Esta es la propiedad predeterminada de este objeto.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemObjectPath.Path As String
```
