# <a name="example-code-for-using-idirectorysearch"></a>Código de ejemplo para usar IDirectorySearch

El kit de desarrollo de software (SDK) de la plataforma incluye un tema de ejemplo de código, DsSrch, que proporciona una sencilla aplicación de símbolo del sistema que usa la interfaz [**IDirectorySearch**](/windows/desktop/api/Iads/nn-iads-idirectorysearch) para consultar un servicio de directorio. El ejemplo de código completo se incluye en el SDK de la plataforma en el directorio Samples \\ NetDs \\ ADSI \\ Samples \\ General \\ DsSrch
