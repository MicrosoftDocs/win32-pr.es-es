# <a name="d1189-cpu-read-bitmap-option-was-used-incorrectly"></a>D1189: La opción de mapa de bits de lectura de CPU se usó incorrectamente

D2D1 \_ BITMAP OPTIONS CPU READ debe \_ \_ \_ combinarse con D2D1 \_ BITMAP OPTIONS CANNOT \_ \_ \_ DRAW






 

## <a name="possible-causes"></a>Causas posibles

La aplicación especificó una combinación incompatible de las marcas BITMAP OPTIONS de D2D1. \_ \_

## <a name="possible-fixes"></a>Posibles correcciones

Quite una de las marcas para crear un conjunto válido de OPCIONES DE MAPA DE BITS D2D1. \_ \_
