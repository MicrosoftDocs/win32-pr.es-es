# <a name="closed-captions-and-teletext"></a>Subtítulos y teletexto

Las señales de televisión análogas pueden llevar información adicional en el intervalo de espacio en blanco vertical (VBI), incluidos los subtítulos y el teletexto estándar mundial (WST). DirectShow proporciona un conjunto de filtros para la decodificación y visualización de texto CC y WST.

-   [Visualización de subtítulos](viewing-closed-captions.md)
