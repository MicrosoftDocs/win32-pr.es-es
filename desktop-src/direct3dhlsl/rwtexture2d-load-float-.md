# <a name="rwtexture2dloadint-function"></a>RWTexture2D:: Load (int) (función)

Lee los datos de textura.

## <a name="syntax"></a>Sintaxis


``` syntax
 Load(
  in int Location
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*Ubicación* \[ de de\]
</dt> <dd>

Tipo: **int**

Ubicación de la textura.

</dd> </dl>
