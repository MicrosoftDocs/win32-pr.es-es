# <a name="avencddrfpreemphasisfilter-property"></a>Propiedad AVEncDDRFPreEmphasisFilter

Especifica la configuración de protección de sobremodulación de radiofrecuencia en una secuencia de audio Dolby Digital. Esta propiedad se aplica a los codificadores de audio Dolby Digital.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncDDRFPreEmphasisFilter**
