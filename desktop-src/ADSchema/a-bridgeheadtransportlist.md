# <a name="bridgehead-transport-list-attribute"></a>Atributo de lista de transporte de cabeza de puente

Transportes para los que este servidor es un servidor cabeza de puente.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | Cabeza de puente-transporte-lista               |
| Nombre para mostrar de LDAP | bridgeheadTransportList                 |
| Tamaño              | \-                                      |
| Actualizar privilegio  | El sistema establece este valor.        |
| Frecuencia de actualización  | Cada vez que se configura un sitio.               |
| Attribute-Id      | 1.2.840.113556.1.4.819                  |
| System-ID-GUID    | d50c2cda-8951-11d1-aebc-0000f80367c1    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
