# <a name="properties-of-icertsrvsetup"></a>Propiedades de ICertSrvSetup

La interfaz [**ICertSrvSetup**](/windows/desktop/api/Casetup/nn-casetup-icertsrvsetup) define las siguientes propiedades.



| Propiedad                                             | Descripción                                                                                                                                                                                                  |
|------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAErrorId**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetup-get_caerrorid)         | Obtiene el identificador de la información de error adicional relacionada con una especificación de [*entidad de certificación*](../secgloss/c-gly.md) (CA) con errores. |
| [**CAErrorString**](/windows/desktop/api/Casetup/nf-casetup-icertsrvsetup-get_caerrorstring) | Obtiene los datos de cadena para la información de error adicional relacionada con una especificación de CA con error.                                                                                                                  |
