# <a name="client-context-reset"></a>Restablecimiento de contexto de cliente

Cuando el servidor deja de estar disponible, la aplicación cliente puede liberar sus datos de contexto mediante una llamada a la función RPC [**RpcSsDestroyClientContext**](/windows/desktop/api/Rpcndr/nf-rpcndr-rpcssdestroyclientcontext).
