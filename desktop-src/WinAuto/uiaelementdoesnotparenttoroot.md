# <a name="uiaelementdoesnotparenttoroot"></a>UiaElementDoesNotParentToRoot

## <a name="text"></a>Texto

La cadena primaria del elemento no va al elemento raíz

## <a name="type"></a>Tipo

Advertencia

## <a name="description"></a>Descripción

La navegación desde el elemento dado a la raíz del árbol de interfaz de usuario se ha roto.

Este problema puede provocar resultados impredecibles para una aplicación que intenta navegar por el árbol de interfaz de usuario.

## <a name="possible-causes"></a>Causas posibles

Implementación incorrecta de la jerarquía secundaria primaria del árbol de interfaz de usuario.
