# <a name="winlogon-notification-package-reference"></a>Referencia del paquete de notificación Winlogon

En los temas siguientes se proporciona información de referencia para los paquetes de notificación de [*Winlogon*](../secgloss/w-gly.md) :

-   [Entradas del registro](registry-entries.md) enumera los valores de clave y los valores que se deben escribir en el registro del sistema para registrar un paquete de notificación y sus funciones de controlador de eventos.
-   El [prototipo de función de controlador de eventos](event-handler-function-prototype.md) detalla el prototipo de las funciones de controlador de eventos. Cada función de controlador de eventos en el paquete de notificación debe coincidir con este prototipo.
