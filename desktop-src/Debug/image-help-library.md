# <a name="image-help-library"></a>Biblioteca de ayuda de imágenes

En esta información general se describe el conjunto de funciones proporcionado por el archivo DLL de ImageHlp. Estas funciones le permiten trabajar con una imagen ejecutable portable (PE).

-   [Acerca de ImageHlp](about-imagehlp.md)
-   [Referencia de ImageHlp](imagehlp-reference.md)

No se puede redistribuir el archivo DLL de ImageHlp que se incluye con el sistema operativo. Un subconjunto de las funciones se incluye en la biblioteca de [ayuda de depuración](debug-help-library.md), que se vuelve a atribuir.
