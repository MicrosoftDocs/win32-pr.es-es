# <a name="winsock-tcpip-structures"></a>Estructuras TCP/IP de Winsock

En la lista siguiente se proporcionan descripciones concisas de cada estructura de TCP/IP de Winsock. Para obtener información adicional sobre cualquier estructura, haga clic en el nombre de la estructura.



| Estructura TCP/IP                                 | Descripción                                                                                                                              |
|--------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**información de la interfaz \_**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info)      | Se usa con el comando ioctl para obtener información acerca de una dirección IP de interfaz.                                                         |
| [**información de la interfaz \_ \_ ex**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info_ex) | Se usa con el comando **SiO \_ Get \_ interface \_ List ioctl** para obtener información acerca de una dirección IP de interfaz, incluidas las direcciones IPv6. |
| [**sockaddr \_ gen**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-sockaddr_gen)            | Proporciona información de dirección de socket genérica. Se usa con la estructura de [**\_ información**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info) de la interfaz.                        |
