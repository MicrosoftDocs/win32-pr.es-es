# <a name="cbem_getcombocontrol-message"></a>CBEM \_ GETCOMBOCONTROL

Obtiene el identificador del control de cuadro combinado secundario.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>Debe ser cero.</dd> <dt>

*lParam* 
</dt> <dd>Debe ser cero.</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Devuelve el identificador del control de cuadro combinado dentro del control ComboBoxEx.
