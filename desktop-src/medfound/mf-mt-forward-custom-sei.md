# <a name="mf_mt_forward_custom_sei-attribute"></a>Atributo MF \_ MT \_ FORWARD CUSTOM \_ \_ SEI

Especifica que el descodificador debe reenviar los tipos de unidad de información de mejora complementaria (SEI) en los ejemplos de salida.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Comentarios

Si el descodificador analiza una SEI, no se reenviará.
