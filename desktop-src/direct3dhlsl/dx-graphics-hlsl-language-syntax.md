# <a name="language-syntax"></a>Sintaxis del lenguaje

Los sombreadores HLSL se componen de variables y funciones, que a su vez se componen de instrucciones. La sintaxis del lenguaje documenta cómo definir y declarar variables, agregar control de flujo para que los sombreadores puedan tomar decisiones en tiempo de ejecución basadas en variables y escribir funciones personalizadas.

-   [Funciones](dx-graphics-hlsl-functions.md)
