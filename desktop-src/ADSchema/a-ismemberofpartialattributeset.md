# <a name="is-member-of-partial-attribute-set-attribute"></a>Atributo is-member-of-Partial-Attribute-set

Si **es true**, este atributo se replica en el catálogo global.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Is-member-of-parcial-Attribute-set   |
| Nombre para mostrar de LDAP | isMemberOfPartialAttributeSet        |
| Tamaño              | \-                                   |
| Actualizar privilegio  | Administrador de esquema                 |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.639               |
| System-ID-GUID    | 19405b9d-3cfa-11d1-a9c0-0000f80367c1 |
| Sintaxis            | [**Booleano**](s-boolean.md)         |
