# <a name="memorycpp"></a>Memoria. CPP

En el componente de proveedor de ejemplo, un ejemplo de código que muestra la asignación y liberación de memoria se encuentra en Memory. cpp. En la tabla siguiente se enumeran las rutinas admitidas.



| Elemento                | Descripción                                                           |
|---------------------|-----------------------------------------------------------------------|
| **AllocProvMem**    | Asigna la memoria especificada.                                            |
| **FreeProvMem**     | Memoria libre indicada.                                                |
| **ReallocProvMem**  | Asigne memoria contigua.                                           |
| **AllocProvStr**    | Asigne una cadena LPWSTR.                                            |
| **FreeProvStr**     | Cadena libre si aún no se ha liberado.                                     |
| **ReallocProvStr**  | Asigne memoria contigua.                                           |
| **ProvAllocString** | Comprueba la cadena y el primer parámetro. Si es correcto, realiza la asignación. |
