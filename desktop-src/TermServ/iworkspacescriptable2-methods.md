# <a name="iworkspacescriptable2-methods"></a>Métodos IWorkspaceScriptable2

La [**interfaz IWorkspaceScriptable2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable2) admite los métodos siguientes.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**Método StartWorkspaceEx**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-startworkspaceex)
</dt> <dd>

Asocia las credenciales de usuario y los certificados a un identificador de conexión; también contiene elementos adicionales de seguridad e interfaz de usuario.

</dd> <dt>

[**Método ResourceDismissed**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-resourcedismissed)
</dt> <dd>

Alerta al usuario de que un recurso se ha deshabilitado o descartado de otro modo.

</dd> </dl>
