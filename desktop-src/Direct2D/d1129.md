# <a name="d1129-dxgi-format-incompatible-with-alpha-mode"></a>D1129: el formato de DXGI no es compatible con el modo alfa

La combinación del modo alfa y el formato de DXGI proporcionado no son compatibles entre sí.



|             |         |
|-------------|---------|
| Nivel de error | Advertencia |



 

## <a name="possible-causes"></a>Causas posibles

En el caso de las operaciones de Direct2D que usan un formato de superficie y un modo alfa, hay casos en los que el formato de superficie y el modo alfa no coinciden. Por ejemplo, un formato de superficie como DXGI \_ format \_ a8 \_ UNORM no tendría sentido usarse con ignore Alpha, ya que solo se representa en un canal alfa. Además, el alfa recto no es compatible con algunos \_ formatos de DXGI, pero el alfa premultiplicado es.

## <a name="possible-fixes"></a>Posibles correcciones

Vea [formatos de píxeles compatibles y modos alfa](supported-pixel-formats-and-alpha-modes.md) para obtener instrucciones sobre qué combinaciones de formato se admiten y cuáles no. Modifique la aplicación para usar una combinación de formato compatible.
