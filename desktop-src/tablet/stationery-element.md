# <a name="stationery-element"></a>Elemento de diseño de fondo

Contiene información sobre el diseño de fondo usado en la nota de Journal, incluidos el color de fondo o la imagen, la información de título y la información de diseño de línea, como reglas estrechas y reglas de margen.

## <a name="definition"></a>Definición

``` syntax
<xs:element name="Stationery" type="StationeryType" minOccurs="0" maxOccurs="unbounded" />
```

[**Title**](title-element.md)
