# <a name="strokebuilder-class"></a>Clase StrokeBuilder

Implementa la [**interfaz IStrokeBuilder**](/windows/desktop/api/RTSCom/nn-rtscom-istrokebuilder).

## <a name="com-implementation"></a>Implementación COM

Esta clase implementa la [**interfaz IStrokeBuilder**](/windows/desktop/api/RTSCom/nn-rtscom-istrokebuilder).

Se puede crear una instancia de esta clase llamando al método [**CoCreateInstance**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance) en C++.
