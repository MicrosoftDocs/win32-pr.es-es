# <a name="lineremovefromconf_-constants"></a>Constantes de LINEREMOVEFROMCONF \_

Las constantes escalares **LINEREMOVEFROMCONF \_** describen cómo se pueden quitar de una llamada de conferencia las entidades que participan en una llamada de conferencia.

<dl> <dt>

<span id="LINEREMOVEFROMCONF_ANY"></span><span id="lineremovefromconf_any"></span>**LINEREMOVEFROMCONF \_ any**
</dt> <dd> <dl> <dt>



Cualquier entidad participante puede quitarse de la llamada de conferencia.


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_LAST"></span><span id="lineremovefromconf_last"></span>**LINEREMOVEFROMCONF \_ última**
</dt> <dd> <dl> <dt>



Solo se puede quitar la entidad agregada más recientemente de la llamada de conferencia


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_NONE"></span><span id="lineremovefromconf_none"></span>**LINEREMOVEFROMCONF \_ ninguno**
</dt> <dd> <dl> <dt>



No se pueden quitar entidades de la llamada de conferencia.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Observaciones

Sin extensibilidad. Todos los 32 bits están reservados.
