# <a name="creating-a-default-printticket"></a>Crear un printticket predeterminado

Este tema no es actual. Para obtener la información más reciente, vea [La especificación de esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

La interfaz del proveedor PrintTicket contiene un método que devuelve un PrintTicket predeterminado. Además, la interfaz del proveedor PrintCapabilities tiene un método que devuelve un documento PrintCapabilities predeterminado; Es decir, una instantánea de PrintCapabilities creada mediante un PrintTicket predeterminado. El proceso siguiente debe generar automáticamente este valor predeterminado de PrintTicket.
