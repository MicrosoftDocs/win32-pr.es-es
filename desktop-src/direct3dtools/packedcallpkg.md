# <a name="span-idvspixenginepackedcallpkgspanpackedcallpkg-structure"></a><span id="vspixengine.packedcallpkg"></span>Estructura PackedCallPkg

Representa un paquete de cuatro llamadas.

## <a name="syntax"></a>Sintaxis


```C++
typedef struct PackedCallPkg {
  UINT pcp1;
  UINT pcp2;
  UINT pcp3;
  UINT pcp4;
} PackedCallPkg;
```

## <a name="members"></a>Miembros

**pcp1**  
La primera llamada en el paquete.

**pcp2**  
Segunda llamada del paquete.

**pcp3**  
Tercera llamada en el paquete.

**pcp4**  
Cuarta llamada del paquete.
