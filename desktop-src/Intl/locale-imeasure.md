# <a name="locale_imeasure"></a>configuración regional \_ IMEASURE

Sistema de medida. El número máximo de caracteres permitido para esta cadena es dos, incluido un carácter nulo de terminación. Este valor es 0 si se usa el sistema de métricas (Systéme International d'Units o S.I.), y 1 si se usa el sistema Estados Unidos.
