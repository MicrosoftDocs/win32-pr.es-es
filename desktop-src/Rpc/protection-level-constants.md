# <a name="protection-level-constants"></a>Constantes de nivel de protección

La especificación RPC de OSF/DCE ha cambiado el nombre de las constantes de nivel de autenticación. Ahora se denominan constantes de nivel de protección. Microsoft RPC es compatible con los nombres antiguos y nuevos. Para obtener más información, vea [constantes de nivel de autenticación](authentication-level-constants.md).
