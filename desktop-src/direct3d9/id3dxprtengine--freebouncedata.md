# <a name="id3dxprtenginefreebouncedata-method"></a>Método ID3DXPRTEngine::FreeBounceData

Libera la memoria usada para los datos temporales de simulación de luz desaforada.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT FreeBounceData();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

El valor devuelto es S \_ OK.
