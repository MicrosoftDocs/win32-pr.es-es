# <a name="prov_dh_schannel"></a>PROV \_ DH \_ Schannel

El tipo de proveedor de Schannel de PROV \_ DH admite los \_ protocolos Diffie-Hellman y [*Schannel*](../secgloss/s-gly.md) .

## <a name="algorithms-supported"></a>Algoritmos admitidos

Para obtener descripciones de cada uno de estos algoritmos, vea el glosario.



| Propósito      | Algoritmos admitidos                                                                                                                                   |
|--------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| Intercambio de claves | DH (efímero)                                                                                                                                         |
| Firma    | [*DSS*](../secgloss/d-gly.md)                                                                                            |
| Cifrado   | [*Des*](../secgloss/d-gly.md)[*triple des*](../secgloss/t-gly.md)<br/> |
| Aplicación de algoritmo hash      | [*MD5*](../secgloss/m-gly.md)[*Sha*](../secgloss/s-gly.md)<br/>                      |
