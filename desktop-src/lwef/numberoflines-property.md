# <a name="numberoflines-property"></a>Propiedad NumberOfLines

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Devuelve el número de líneas admitidas para el globo de palabras para el carácter especificado.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintáctica**
</dt> <dd>

*agente ***. Caracteres ("*** CharacterID * *"). Balloon. NumberOfLines**

</dd> </dl>

## <a name="remarks"></a>Observaciones

La propiedad **NumberOfLines** devuelve el número de líneas de texto como un valor entero largo.
