# <a name="using-sendto-while-connected"></a>Usar SendTo mientras está conectado

[**WSPSendTo**](/previous-versions/windows/desktop/legacy/ms742291(v=vs.85)) siempre proporcionará los datos a la dirección especificada, aunque se haya establecido un elemento del mismo nivel designado para el socket de envío en [**WSPConnect**](/previous-versions/windows/hardware/network/ff566275(v=vs.85)).
