# <a name="compiling-an-instrumentation-manifest"></a>Compilar un manifiesto de instrumentación

Después de [escribir el manifiesto](writing-an-instrumentation-manifest.md), use el compilador de mensajes para validar el manifiesto y generar los archivos de recursos y de encabezado que se incluyen en el proveedor. Para obtener información detallada sobre el uso del compilador, vea [**Message Compiler**](message-compiler--mc-exe-.md). Incluya el encabezado y los archivos de recursos generados por el compilador en el proyecto que contiene el resto del código fuente del proveedor. Para obtener información sobre cómo escribir los eventos definidos en el manifiesto, consulte desarrollo de [un proveedor](developing-a-provider.md).
