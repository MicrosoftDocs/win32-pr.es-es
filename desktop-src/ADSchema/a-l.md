# <a name="locality-name-attribute"></a>Locality-Name atributo)

Representa el nombre de una localidad, como una ciudad o una ciudad.



| Entrada | Value |
|-------------------|------------------------------------------------------------------------------|
| CN                | Locality-Name                                                                |
| Nombre para mostrar de LDAP | l                                                                            |
| Tamaño              | \-                                                                           |
| Actualizar privilegio  | Administrador de dominio o propietario de la cuenta.                                       |
| Frecuencia de actualización  | Cuando se crea el registro del usuario y cada vez que es necesario cambiar el emplazamiento. |
| Attribute-Id      | 2.5.4.7                                                                      |
| System-ID-GUID    | bf9679a2-0de6-11d0-a285-00aa003049e2                                         |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md)                                  |
