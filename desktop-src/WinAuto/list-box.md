# <a name="list-box-msaa-ui-element-reference"></a>Cuadro de lista (referencia de elementos de interfaz de usuario de MSAA)

> [!Note]  
> En este tema se describen los objetos de **cuadro de lista** para la referencia de elementos de interfaz de usuario de MSAA. La forma de crear objetos de **cuadro de lista** en varios marcos de interfaz de usuario no se describe aquí. Consulte la documentación de referencia de la API del marco de interfaz de usuario que está usando.

 

Los cuadros de lista muestran una lista en la que un usuario puede seleccionar uno o varios elementos. Un cuadro de lista se crea con una barra de desplazamiento vertical, una barra de control horizontal o ambos.

El nombre de clase de ventana de un cuadro de lista es "LISTBOX".

## <a name="iaccessible-methods"></a>Métodos IAccessible

Los cuadros de lista admiten los siguientes métodos [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) :



| Método                                                                    | Comentarios                                                                                                                                                                                                                                                                   |
|---------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**accDoDefaultAction**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) | El propio objeto de cuadro de lista no tiene una propiedad **DEFAULTACTION** ; sin embargo, [**accDoDefaultAction**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accdodefaultaction) hace doble clic en los elementos del cuadro de lista.                                                                                    |
| [**accHitTest**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest)                 | El método [**accHitTest**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acchittest) devuelve cero si el punto probado está en el cuadro de lista pero no en un elemento del cuadro de lista. Si el punto probado se encuentra en un elemento del cuadro de lista, **accHitTest** devuelve el índice del elemento del cuadro de lista. |
| [**accLocation**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-acclocation)               |                                                                                                                                                                                                                                                                            |
| [**accNavigate**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accnavigate)               |                                                                                                                                                                                                                                                                            |
| [**accSelect**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-accselect)                   |                                                                                                                                                                                                                                                                            |
