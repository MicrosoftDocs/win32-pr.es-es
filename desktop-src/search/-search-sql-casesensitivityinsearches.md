# <a name="case-sensitivity-in-searches"></a>Confidencialidad de mayúsculas y minúsculas en las búsquedas

Las consultas de Microsoft Windows Search no distinguen mayúsculas de minúsculas. Esto incluye palabras clave de consulta (SELECT es idéntica a Select, select y sELect) y términos de búsqueda. Si un identificador o literal usa caracteres Unicode que tienen semántica de asignación de mayúsculas y minúsculas, el motor de búsqueda considera todos los casos equivalentes.
