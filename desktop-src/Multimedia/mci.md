# <a name="mci"></a>Mci

La interfaz de control multimedia (MCI) proporciona comandos estándar para reproducir dispositivos multimedia y grabar archivos de recursos multimedia. Estos comandos son una interfaz genérica para casi todos los tipos de dispositivos multimedia.

-   [Uso de mensajes de comandos de MCI](using-mci-command-messages.md)
-   [Referencia de MCI](mci-reference.md)
