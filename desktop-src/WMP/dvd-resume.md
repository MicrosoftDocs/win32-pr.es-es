# <a name="dvdresume-method"></a>DVD. resume (método)

El método **resume** vuelve al modo de reproducción desde el modo de menú en la misma posición de título que cuando se invocó el menú.

## <a name="syntax"></a>Sintaxis


```JScript
DVD.resume()
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.

## <a name="remarks"></a>Observaciones

**Windows Media Player 10 Mobile:** Este método siempre se realiza correctamente, pero no realiza la operación deseada.
