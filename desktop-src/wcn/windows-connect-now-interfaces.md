# <a name="windows-connect-now-interfaces"></a>Interfaces de Windows Connect Now

La [API de Windows Connect Now](portal.md) (WCNAPI) define y emplea las siguientes interfaces.



| Interfaz                                      | Descripción                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IWCNConnectNotify**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcnconnectnotify) | Reciba una notificación de éxito o error cuando se complete la sesión de conexión de Windows Connect Now.<br/> |
| [**IWCNDevice**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcndevice)               | Configura el dispositivo e inicia la sesión.<br/>                                                |
