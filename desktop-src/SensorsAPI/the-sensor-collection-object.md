# <a name="the-sensor-collection-object"></a>Objeto de colección de sensores

El objeto de colección de sensores contiene un grupo de objetos de sensor.

En ocasiones, querrá trabajar con sensores como un grupo. Por ejemplo, puede que desee usar el administrador de sensores para detectar todos los sensores de movimiento disponibles. El objeto de colección sensor proporciona métodos para trabajar con un grupo de objetos de sensor. Puede usar la interfaz [**ISensorCollection**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection) para llamar a métodos que cuentan, agregan, quitan y recuperan sensores en la colección.
