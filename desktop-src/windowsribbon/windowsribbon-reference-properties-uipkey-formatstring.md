# <a name="ui_pkey_formatstring"></a>PKEY de IU \_ \_ FormatString

Identifica la propiedad PKEY de la interfaz de usuario \_ \_ .

```
propertyDescription
   name = UI_PKEY_FormatString
   shellPKey = UI_PKEY_FormatString
   formatID = 00000207-7363-696e-8441798acf5aebb7
   propID = 207
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Observaciones

Una aplicación usa la interfaz de usuario \_ PKEY \_ FormatString para consultar el formato (o la unidad de medida) del valor del control de [número](windowsribbon-controls-spinner.md) .
