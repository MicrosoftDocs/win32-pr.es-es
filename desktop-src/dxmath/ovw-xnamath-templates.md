# <a name="directxmath-library-template-functions"></a>Funciones de plantilla de la biblioteca de DirectXMath

Enumera las funciones de plantilla proporcionadas por DirectXMath.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                  | Descripción                                                                                                                                                                                                                               |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMMax**](xmmax-template.md)<br/>                             | Compara dos instancias de tipos de datos numéricos, o dos instancias de un objeto que admite una sobrecarga de <, y devuelve el mayor de las dos instancias. El tipo de datos de los argumentos y el valor devuelto es el mismo.<br/>  |
| [**XMMin**](xmmin-template.md)<br/>                             | Compara dos instancias de tipos de datos numéricos, o dos instancias de un objeto que admite una sobrecarga de <, y devuelve la menor de las dos instancias. El tipo de datos de los argumentos y el valor devuelto es el mismo.<br/> |
| [**XMVectorPermute**](xmvectorpermute-template.md)<br/>         | Permute los componentes de dos vectores para crear un nuevo vector.<br/>                                                                                                                                                                 |
| [**XMVectorRotateLeft**](xmvectorrotateleft-template.md)<br/>   | Gira el vector a la izquierda un número determinado de elementos de 32 bits.<br/>                                                                                                                                                                  |
| [**XMVectorRotateRight**](xmvectorrotateright-template.md)<br/> | Gira el vector a la derecha un número determinado de elementos de 32 bits.<br/>                                                                                                                                                                 |
| [**XMVectorShiftLeft**](xmvectorshiftleft-template.md)<br/>     | Desplaza un vector a la izquierda de un número determinado de elementos de 32 bits y rellena los elementos vacantes con elementos de un segundo vector.<br/>                                                                                                    |
| [**XMVectorSwizzle**](xmvectorswizzle-template.md)<br/>         | Swizzles un vector.<br/>                                                                                                                                                                                                             |
| [**XMVectorInsert**](xmvectorinsert-template.md)<br/>           | Gira un vector a la izquierda un número determinado de componentes de 32 bits e inserta los elementos seleccionados de ese resultado en otro vector.<br/>                                                                                                  |
