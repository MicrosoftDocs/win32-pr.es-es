# <a name="labeleduri-attribute"></a>atributo labeledURI

Identificador uniforme de recursos seguido de una etiqueta. La etiqueta se usa para describir el recurso al que apunta el URI y está pensada como un nombre para mostrar que es legible para el usuario.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | labeledURI                                  |
| Ldap-Display-Name | labeledURI                                  |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.3.6.1.4.1.250.1.57                        |
| System-Id-Guid    | c569bb46-c680-44bc-a273-e6c227d71b45        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
