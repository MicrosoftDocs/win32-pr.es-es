# <a name="user-parameters-attribute"></a>User-Parameters atributo

Parámetros del usuario. Apunta a una cadena Unicode que se reserva para su uso por parte de las aplicaciones. Esta cadena puede ser una cadena null o puede tener cualquier número de caracteres antes del carácter nulo de terminación. Los productos de Microsoft usan este miembro para almacenar datos de usuario específicos del programa individual.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | User-Parameters                             |
| Ldap-Display-Name | userParameters                              |
| Size              | \-                                          |
| Privilegio actualizar  | Modificado por las aplicaciones.                   |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.138                      |
| System-Id-Guid    | bf967a6d-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
