# <a name="starting-the-agent-character-editor"></a>Inicio del editor de caracteres del agente

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Para ejecutar el editor de caracteres del agente, elija la opción **Editor de caracteres del agente** en el menú **Inicio** de la barra de tareas de Windows o haga doble clic en el icono editor de **caracteres del agente** Microsoft en el escritorio. Se abrirá la ventana del editor, mostrando los menús, una barra de herramientas con comandos usados con frecuencia, un árbol que muestra los componentes que componen la definición de un carácter y un conjunto de páginas con pestañas que cambian en función de su selección en el árbol de componentes.

![Captura de pantalla que muestra la página ' propiedades ' del ' carácter-editor de caracteres del agente de Microsoft '.](images/f1chared.gif)

Para tener acceso a los campos de la ventana desde el teclado, puede usar la tecla TAB y MAYÚS + TAB para desplazarse por los controles, o bien usar la tecla de acceso (ALT +*letra subrayada*) para desplazarse a un control específico. Una vez que se completa el inicio, puede empezar a crear una nueva definición de caracteres o cargar una definición de caracteres existente.

La barra de estado muestra información sobre los comandos o los botones de la barra de herramientas cuando se mueve el puntero sobre ellos. También se muestra información de resumen sobre los datos de animación de caracteres y la información de estado al generar un carácter.
