# <a name="n-rpc"></a>N (RPC)

[A](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) [E](e-glos.md) [F](f-glos.md) G H [I](i-glos.md) J K [L](l-glos.md) [M](m-glos.md) N [o](o-glos.md) [p](p-glos.md) [Q](q.md) [R](r-glos.md) [s](s-glos.md) [T](t-glos.md) [U](u-glos.md) [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_name_service_glos"></span><span id="_RPC_NAME_SERVICE_GLOS"></span>**servicio de nombres**
</dt> <dd>

Servicio que asigna nombres a objetos y almacena los pares de nombre/objeto en una base de datos. Por ejemplo, el servicio de nombres RPC asigna un nombre lógico a un [*identificador de enlace*](b-glos.md) para que las aplicaciones cliente puedan hacer referencia a ese nombre lógico, en lugar de una secuencia de protocolo y una dirección de red. Vea también nombre Service-Interface daemon ([NSID)](/windows), servicio de directorio de cliente ([*CDs)*](c-glos.md), [*localizador*](l-glos.md).

</dd> <dt>

<span id="_rpc_nca_glos"></span><span id="_RPC_NCA_GLOS"></span>**Arquitectura de informática de red (NCA)**
</dt> <dd>

Colección de directrices para computación distribuida. Los protocolos de comunicación RPC siguen estas instrucciones.

</dd> <dt>

<span id="_rpc_nsi_glos"></span><span id="_RPC_NSI_GLOS"></span>**Independiente del servicio de nombres (NSI)**
</dt> <dd>

Estándar para las funciones de API que permiten a una aplicación distribuida tener acceso a los elementos de base de datos de servicio de nombres de RPC a través de varios proveedores de servicios de nombres, como OSF-DCE Servicio de directorio de celdas o el localizador de Microsoft. Vea también [– interface daemon](/windows) (NSID).

</dd> <dt>

<span id="_rpc_nsid_glos"></span><span id="_RPC_NSID_GLOS"></span>**servicio de nombres: demonio de interfaz (NSID)**
</dt> <dd>

Servicio que proporciona una interfaz entre el [*localizador*](l-glos.md) de Microsoft y las bases de datos de servicio de nombres de servicio de directorio de celdas OSF-DCE para las funciones de servicio de nombres de RPC.

</dd> <dt>

<span id="_rpc_named_pipe_glos"></span><span id="_RPC_NAMED_PIPE_GLOS"></span>**canalización con nombre**
</dt> <dd>

Protocolo orientado a la conexión, basado en bloques de mensajes del servidor (SMB) y [NetBIOS](/windows), que se usa para la comunicación entre un proceso de servidor y uno o más procesos de cliente.

</dd> <dt>

<span id="_rpc_netbeui_glos"></span><span id="_RPC_NETBEUI_GLOS"></span>**Interfaz de usuario extendida de NetBIOS (NetBEUI)**
</dt> <dd>

Protocolo de transporte nativo de LAN Manager y controlador de dispositivo de red. Vea también [NetBIOS](/windows).

</dd> <dt>

<span id="_rpc_netbios_glos"></span><span id="_RPC_NETBIOS_GLOS"></span>**Sistema básico de entrada y salida de red (NetBIOS)**
</dt> <dd>

Interfaz de software entre el sistema operativo Microsoft MS-DOS, el bus de e/s y una red de área local.

</dd> <dt>

<span id="_rpc_ndr_glos"></span><span id="_RPC_NDR_GLOS"></span>**Representación de datos de red (NDR)**
</dt> <dd>

Formato estándar usado durante la transmisión de red que es independiente del formato de tipo de datos en cualquier arquitectura de equipo determinada. Los datos transmitidos incluyen información que especifica su formato NDR.

</dd> <dt>

<span id="_rpc_network_address_glos"></span><span id="_RPC_NETWORK_ADDRESS_GLOS"></span>**Dirección de red**
</dt> <dd>

Dirección que identifica un servidor en una red.

</dd> <dt>

<span id="_rpc_nonencapsulated_union_glos"></span><span id="_RPC_NONENCAPSULATED_UNION_GLOS"></span>**Unión no encapsulada**
</dt> <dd>

[*Unión discriminada*](d-glos.md) que es menos restrictiva que una Unión encapsulada en que el discriminante y la Unión no están estrechamente enlazados. Si la Unión es un parámetro, discriminante es otro parámetro; Si la Unión es un campo de estructura, el discriminante es otro campo de estructura. El modificador de palabras clave IDL \[ [**\_ es**](/windows/desktop/Midl/switch-is) \] y \[ el [**\_ tipo de conmutador**](/windows/desktop/Midl/switch-type) \] identifica el discriminante y su tipo. Vea también [*Unión encapsulada*](e-glos.md).

</dd> <dt>

<span id="_rpc_nonidempotent_glos"></span><span id="_RPC_NONIDEMPOTENT_GLOS"></span>**nonidempotent**
</dt> <dd>

Indicador de que una llamada a procedimiento remoto no se puede ejecutar más de una vez porque devolverá un valor diferente o cambiará un estado.

</dd> </dl>
