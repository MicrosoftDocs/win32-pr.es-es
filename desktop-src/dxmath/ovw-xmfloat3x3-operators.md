# <a name="xmfloat3x3-operators"></a>Operadores XMFLOAT3X3

La [**estructura XMFLOAT3X3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3x3) expone los operadores siguientes.

> [!Note]  
> Los operadores enumerados aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                      | Descripción                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**operator =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat3x3-operator-assign)<br/>      | Asigna los datos del componente vectorial de una instancia de `XMFLOAT3X3` a la instancia actual de `XMFLOAT3X3` . <br/> |
| [**operador ()**](xmfloat3x3-operator-parens.md)<br/> | Tiene acceso a elementos de matriz específicos a los que hace referencia fila y columna desde la instancia actual de `XMFLOAT3X3` . <br/>    |
