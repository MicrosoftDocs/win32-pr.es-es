# <a name="installing-a-monitor-to-network-monitor"></a>Instalación de un monitor en Monitor de red

La instalación de un archivo DLL de monitor es un proceso sencillo. En primer lugar, copie el archivo DLL y el formulario de configuración HTML asociado al subdirectorio de monitores NPP (por ejemplo, C: \\ Windows \\ system32 \\ NPP \\ monitores). Cuando se copia en el directorio, el monitor se reconocerá y estará disponible la próxima vez que se active la herramienta de control de supervisión o el servicio de control de supervisión (MCSVC).

Por ejemplo, si la variable de entorno,% SystemRoot% es C: \\ Windows, el subdirectorio monitor es c: \\ Windows \\ system32 \\ NPP \\ monitores.
