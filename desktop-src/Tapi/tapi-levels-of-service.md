# <a name="tapi-levels-of-service"></a>Niveles de servicio de TAPI

Los servicios de telefonía TAPI se dividen en servicios de telefonía asistidos y los servicios que proporciona la API de telefonía completa. En general, la API de telefonía completa se usa para implementar aplicaciones de telefonía eficaces, y la telefonía asistida se usa para agregar una funcionalidad de telefonía mínima pero útil a las aplicaciones que no son de telefonía.
