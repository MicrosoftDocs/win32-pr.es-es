# <a name="xmfloat3x3-operators"></a>Operadores XMFLOAT3X3

La estructura [**XMFLOAT3X3**](/windows/win32/api/directxmath/ns-directxmath-xmfloat3x3) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                      | Descripción                                                                                                              |
|--------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**operador =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat3x3-operator-assign)<br/>      | Asigna los datos del componente de vector de una instancia de `XMFLOAT3X3` a la instancia actual de `XMFLOAT3X3` . <br/> |
| [**operador ()**](xmfloat3x3-operator-parens.md)<br/> | Obtiene acceso a elementos de matriz concretos a los que hace referencia la fila y la columna de la instancia actual de `XMFLOAT3X3` . <br/>    |
