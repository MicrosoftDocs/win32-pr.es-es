# <a name="xmbyten4-extensions"></a>Extensiones XMBYTEN4

[**XMBYTEN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmbyten4) describe un vector 3D para almacenar valores con signo y normalizados como enteros de 8 bits (1 byte) con signo.

> [!Note]  
> Las características enumeradas aquí solo están disponibles al desarrollar con C++.
