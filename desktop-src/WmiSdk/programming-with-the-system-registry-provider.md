# <a name="programming-with-the-system-registry-provider"></a>Programar con el proveedor del registro del sistema

En los temas siguientes se describe cómo las aplicaciones pueden utilizar el proveedor del registro del sistema para proporcionar datos para las clases que cree.



| Tema                                                                                                                      | Descripción                                                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Definir clases para el proveedor del registro del sistema](defining-classes-for-the-system-registry-provider.md)                 | Para obtener acceso al registro del sistema, debe crear clases de Windows que almacenen información del registro en el repositorio de Instrumental de administración de Windows (WMI).                                                                     |
| [Usar el proveedor del registro del sistema como proveedor de propiedades](using-the-system-registry-provider-as-a-property-provider.md) | El proveedor del registro del sistema requiere que use varios calificadores para ver las propiedades del registro.                                                                                                                             |
| [Descripción de un recurso para el registro](describing-a-resource-for-the-registry.md)                                       | Puede tener acceso a los datos del registro que describen un recurso del sistema. Para modificar la información, el proveedor del registro del sistema requiere una sintaxis específica.                                                                      |
| [Obtener acceso a un valor del registro sin nombre](accessing-an-unnamed-registry-value.md)                                             | Para obtener acceso a la mayoría de los valores del registro, cree una clase para recibir los valores en WMI o puede examinar propiedades específicas del registro. Sin embargo, el proveedor del registro del sistema requiere una sintaxis especial para ver los valores del registro sin nombre. |
