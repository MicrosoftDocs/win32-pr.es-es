# <a name="version-number-lo-attribute"></a>Atributo Version-Number-Lo

Número de versión secundaria de uso general.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Número de versión:Lo                    |
| Ldap-Display-Name | versionNumberLo                      |
| Size              | 4 bytes                              |
| Privilegio actualizar  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.329               |
| System-Id-Guid    | 7d6c0e9b-7e20-11d0-afd6-00c04fd930c9 |
| Syntax            | [**Enumeración**](s-enumeration.md) |
