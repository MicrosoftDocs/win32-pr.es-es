# <a name="object-model-reference-for-visual-basic-net-and-c"></a>Referencia del modelo de objetos para Visual Basic .NET y C #

> [!Note]  
> El uso del control ActiveX de Windows Media Player con Visual Basic .NET y C# solo es compatible con el SDK de Windows Media Player 9 series y versiones posteriores.

 

En esta sección de referencia se proporciona documentación para el objeto **AxWindowsMediaPlayer** y para las interfaces del control ActiveX de Windows Media Player. Esta funcionalidad está disponible cuando se usa el control ActiveX de Windows Media Player a través de la interoperabilidad COM en una solución de Visual Studio .NET 2002 o una versión posterior.

Para obtener una demostración del uso del control de Media Player de Windows con el .NET Framework, vea los ejemplos de Visual Basic y C# incluidos con este SDK.

En las secciones siguientes se documenta el Visual Basic objetos y las interfaces de .NET y C# proporcionados por el ensamblado de interoperabilidad COM que crea Visual Studio.

> [!Note]  
> La funcionalidad que requiere la comunicación remota del control de Media Player de Windows no se admite para la programación de .NET.
