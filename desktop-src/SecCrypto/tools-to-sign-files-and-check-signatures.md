# <a name="tools-to-sign-files-and-check-signatures"></a>Herramientas para firmar archivos y comprobar firmas

Las herramientas de CryptoAPI son las siguientes herramientas que puede utilizar para firmar archivos digitalmente, para confirmar firmas en archivos y para crear archivos de catálogo.



| Herramienta                     | Observaciones                                                                                                                                                                                                                                                                            |
|--------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [MakeCat](makecat.md)   | Crea un archivo de catálogo sin firmar. Posteriormente, este archivo de catálogo se usa para comparar los [*valores hash*](../secgloss/h-gly.md) del archivo de catálogo con los hash de archivos individuales, lo que ayuda a demostrar que los archivos originales no se pueden alterar. |
| [SetReg](setreg.md)     | Establece las claves del registro que controlan el proceso de comprobación de certificados.                                                                                                                                                                                                              |
| [SignTool](signtool.md) | Firma archivos digitalmente y confirma firmas digitales de archivos.                                                                                                                                                                                                                    |
