# <a name="locale_fontsignature"></a>LOCALE \_ FONTSIGNATURE

**Windows Me/98/95, Windows NT 4.0:** Patrón de bits específico que determina la relación entre la cobertura de caracteres necesaria para admitir la configuración regional y el contenido de la fuente. Tenga en cuenta que los datos FONTSIGNATURE de LOCALE tienen una \_ forma diferente de la del resto de la información de configuración regional. Toda la información de configuración regional se puede expresar en forma de cadena o como un número. Los datos \_ FONTSIGNATURE de LOCALE se recuperan en una [estructura LOCALIGNATURE.](/windows/win32/api/wingdi/ns-wingdi-localesignature)
