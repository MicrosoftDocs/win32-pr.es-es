# <a name="tapi-31-requirements"></a>Requisitos de TAPI 3.1

Estos son los requisitos del sistema operativo para TAPI 3.x:

-   TAPI versión 3.0 requiere Windows 2000 o posterior.
-   TAPI versión 3.1 requiere Windows operativos Server 2003 o Windows XP.

Tenga en cuenta [que el control de conferencias IP](about-rendezvous-ip-telephony-conferencing.md) que usa Rendezvous requiere acceso al menos a un servidor ILS o NTDS. Si hay un servidor NTDS en su lugar, se puede usar para buscar un servidor ILS. Si no existe ningún servidor NTDS, el usuario final debe conocer el nombre del servidor ILS.
