# <a name="about-colors"></a>Acerca de los colores

El color se puede usar para comunicar ideas, mostrar las relaciones entre los elementos y mejorar la apelación y la calidad de la salida. Windows permite a las aplicaciones detectar las capacidades de color de los dispositivos determinados y elegir entre los colores disponibles que mejor se adapten a sus necesidades.

En esta información general se proporciona información sobre los temas siguientes:

-   [Conceptos básicos de color](color-basics.md)
-   [Paletas de colores](color-palettes.md)

Aunque no se describe en esta información general, la coincidencia de color de imagen es una característica importante de la administración del color que ayuda a garantizar que las imágenes de color tienen el mismo aspecto, tanto si se muestran en pantalla como si se imprimen en papel. Para obtener más información, consulte [sistema de color de Windows](/previous-versions//dd372446(v=vs.85)).
