# <a name="creating-the-pushed-file"></a>Crear el archivo insertado

Puede tomar los elementos que ha creado y volver a usarlos para crear las demás imágenes que necesite. Al combinar las capas y colorear los resultados, puede crear un mapa de bits insertado que contenga las imágenes que desea que se vean cuando el usuario inserta un botón. Este archivo también contiene la imagen secundaria para el estado en pausa del botón PlayPause.

La siguiente imagen es el archivo insertado.

![archivo insertado](images/ceswmpus.png)

Los dos botones de la derecha son para el estado de pausa normal y el estado de pausa insertado del botón PlayPause. El color de los botones es amarillo, excepto el estado de pausa normal del botón PlayPause, como un buen contraste con el color verde que se usa para el estado normal de todos los botones.
