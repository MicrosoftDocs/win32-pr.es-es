# <a name="iagentex"></a>IAgentEx

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

**IAgentEx** deriva de la interfaz [**IAgent**](iagent.md) . Incluye todos los métodos **IAgent** y proporciona acceso a funciones adicionales.

**Métodos en orden de Vtable**



| Métodos IAgentEx                                                                   | Descripción                                              |
|------------------------------------------------------------------------------------|----------------------------------------------------------|
| [**ShowDefaultCharacterProperties**](iagentex--showdefaultcharacterproperties.md) | Muestra las propiedades de carácter predeterminadas.               |
| [**GetVersion**](iagentex--getversion.md)                                         | Devuelve el número de versión del agente de Microsoft (servidor). |
