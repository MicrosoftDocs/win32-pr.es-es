# <a name="restart-manager-callback-functions"></a>Funciones de devolución de llamada del Administrador de reinicio

La API de Restart Manager usa las funciones de devolución de llamada en la tabla siguiente.



| Función de devolución de llamada                                               | Descripción                                                                                                                                                            |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DEVOLUCIÓN DE \_ LLAMADA DE ESTADO DE ESCRITURA DE \_ \_ RM**](/windows/win32/api/restartmanager/nc-restartmanager-rm_write_status_callback) | La aplicación que inició la sesión de Restart Manager puede pasar un puntero a esta función a las funciones de Restart Manager para recibir un porcentaje de integridad. |
