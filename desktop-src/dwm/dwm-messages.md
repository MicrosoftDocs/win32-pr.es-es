# <a name="dwm-messages"></a>Mensajes DWM

Esta sección contiene información sobre los mensajes Administrador de ventanas de escritorio (DWM).

## <a name="in-this-section"></a>En esta sección




| Tema | Descripción | 
|-------|-------------|
| <a href="wm-dwmcolorizationcolorchanged.md"><strong>WM_DWMCOLORIZATIONCOLORCHANGED</strong></a><br /> | Informa a todas las ventanas de nivel superior de que el color de color ha cambiado.<br /> | 
| <a href="wm-dwmcompositionchanged.md"><strong>WM_DWMCOMPOSITIONCHANGED</strong></a><br /> | Informa a todas las ventanas de nivel superior de que la composición de DWM se ha habilitado o deshabilitado. <br /><blockquote>[!Note]<br />A Windows 8, la composición de DWM siempre está habilitada, por lo que este mensaje no se envía independientemente de los cambios en el modo de vídeo.</blockquote><br /> | 
| <a href="wm-dwmncrenderingchanged.md"><strong>WM_DWMNCRENDERINGCHANGED</strong></a><br /> | Se envía cuando ha cambiado la directiva de representación del área que no es de cliente.<br /> | 
| <a href="wm-dwmsendiconiclivepreviewbitmap.md"><strong>WM_DWMSENDICONICLIVEPREVIEWBITMAP</strong></a><br /> | Indica a una ventana que proporcione un mapa de bits estático que se usará como una vista previa en directo <em>(también</em> conocida como vista previa de <em>Peek)</em>de esa ventana.<br /> | 
| <a href="wm-dwmsendiconicthumbnail.md"><strong>WM_DWMSENDICONICTHUMBNAIL</strong></a><br /> | Indica a una ventana que proporcione un mapa de bits estático que se usará como representación en miniatura de esa ventana.<br /> | 
| <a href="wm-dwmwindowmaximizedchange.md"><strong>WM_DWMWINDOWMAXIMIZEDCHANGE</strong></a><br /> | Se envía cuando se maximiza una ventana compuesta de DWM.<br /> |
