| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-SPX                                  |
| Nombre para mostrar de LDAP | mS-SQL-SPX                                  |
| Tamaño              | \-                                          |
| Actualizar privilegio  | El sistema establece este valor.            |
| Frecuencia de actualización  | Al iniciar el sistema.                          |
| Attribute-Id      | 1.2.840.113556.1.4.1376                     |
| System-ID-GUID    | 86b08004-ccee-11d2-9993-0000f87a57d4        |
