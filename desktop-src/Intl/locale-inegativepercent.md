# <a name="locale_inegativepercent"></a>configuración regional \_ INEGATIVEPERCENT

**Windows 7 y versiones posteriores:** Patrón de formato de porcentaje negativo para la configuración regional. Solo se puede indicar un patrón. Si se usa más de un formato para la configuración regional, elija la opción preferida. Por ejemplo, si se muestra un porcentaje negativo "-9%" para "negative Nine Percent", la opción adecuada para esta constante es 0.



| Value | Formato                                                    |
|-------|-----------------------------------------------------------|
| 0     | Signo negativo, número, espacio, porcentaje; por ejemplo,-\# % |
| 1     | Signo negativo, número, porcentaje; por ejemplo,-\#%         |
| 2     | Signo negativo, porcentaje, número; por ejemplo,-%\#         |
| 3     | Porcentaje, signo negativo, número; por ejemplo,%-\#         |
| 4     | Porcentaje, número, signo negativo; por ejemplo,%\#-         |
| 5     | Número, signo negativo, porcentaje; por ejemplo, \#-%         |
| 6     | Número, porcentaje, signo negativo; por ejemplo, \#%-         |
| 7     | Signo negativo, porcentaje, espacio, número; por ejemplo,-% \# |
| 8     | Número, espacio, porcentaje, signo negativo; por ejemplo, \# %- |
| 9     | Porcentaje, espacio, número, signo negativo; por ejemplo,% \#- |
| 10    | Porcentaje, espacio, signo negativo, número; por ejemplo,%-\# |
| 11    | Número, signo negativo, espacio, porcentaje; por ejemplo, \# -% |
