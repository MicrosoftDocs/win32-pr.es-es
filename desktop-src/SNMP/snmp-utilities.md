# <a name="snmp-utilities"></a>Utilidades SNMP

En la tabla siguiente se enumeran las utilidades SNMP que están disponibles en el kit de recursos de Microsoft Windows.



| Nombre de archivo     | Descripción                                                                             |
|--------------|-----------------------------------------------------------------------------------------|
| EVNTCMD.EXE  | Una aplicación de línea de comandos para configurar el traductor de eventos SNMP                    |
| EVNTWIN.EXE  | Aplicación que proporciona una interfaz de usuario para configurar el traductor de eventos SNMP |
| MIBCC.EXE    | El compilador SNMP MIB                                                                   |
| SNMPUTIL.EXE | Una aplicación de consola de administrador SNMP de ejemplo                                               |



 

El proveedor SNMP del SDK de Instrumental de administración de Windows (WMI) permite a las aplicaciones cliente obtener acceso a información de SNMP estática y dinámica a través de WMI. Para obtener más información, consulte [proveedor SNMP](/windows/desktop/WmiSdk/snmp-provider) en la documentación del SDK de WMI.
