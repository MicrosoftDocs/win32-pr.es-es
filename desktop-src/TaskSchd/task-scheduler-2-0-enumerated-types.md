# <a name="task-scheduler-20-enumerated-types"></a>Programador de tareas tipos enumerados de 2.0

Describe los tipos enumerados que definen las constantes que usan las API Programador de tareas 2.0.


Los siguientes tipos enumerados se presentan en Programador de tareas 2.0.



| Enumeración                                                                  | Descripción                                                                                                      |
|------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**TIPO \_ DE ACCIÓN \_ TASK**](/windows/desktop/api/taskschd/ne-taskschd-task_action_type)                               | Define el tipo de acciones que puede realizar una tarea.                                                             |
| [**COMPATIBILIDAD DE \_ TAREAS**](/windows/desktop/api/taskschd/ne-taskschd-task_compatibility)                            | Define las versiones de Programador de tareas o el comando AT con el que la tarea es compatible.                      |
| [**CREACIÓN DE \_ TAREAS**](/windows/desktop/api/taskschd/ne-taskschd-task_creation)                                       | Define cómo el Programador de tareas crea, actualiza o deshabilita la tarea.                                   |
| [**MARCAS \_ DE \_ ENUMERACIÓN DE TAREAS**](/windows/desktop/api/taskschd/ne-taskschd-task_enum_flags)                                 | Define cómo el Programador de tareas enumera a través de tareas registradas.                                              |
| [**DIRECTIVA \_ DE INSTANCIAS DE \_ TAREA**](/windows/desktop/api/taskschd/ne-taskschd-task_instances_policy)                     | Define cómo el Programador de tareas controla las instancias existentes de la tarea cuando inicia una nueva instancia de la tarea. |
| [**TIPO DE \_ INICIO DE SESIÓN DE TAREA**](/windows/desktop/api/taskschd/ne-taskschd-task_logon_type)                                  | Define qué técnica de inicio de sesión es necesaria para ejecutar una tarea.                                                          |
| [**TASK \_ PROCESSTOKENSID \_ TYPE**](/windows/win32/api/taskschd/ne-taskschd-task_processtokensid_type)             | Define los tipos de SID de proceso que pueden usar las tareas.                                                      |
| [**MARCAS \_ DE EJECUCIÓN DE \_ TAREAS**](/windows/desktop/api/taskschd/ne-taskschd-task_run_flags)                                   | Define cómo se ejecuta una tarea.                                                                                       |
| [**TIPO \_ RUNLEVEL DE \_ TAREA**](/windows/win32/api/taskschd/ne-taskschd-task_runlevel_type)                           | Define marcas de elevación lua que especifican con qué nivel de privilegio se ejecutará la tarea.                         |
| [**TIPO DE \_ CAMBIO DE ESTADO DE SESIÓN DE \_ \_ \_ TAREA**](/windows/desktop/api/taskschd/ne-taskschd-task_session_state_change_type) | Define qué tipos de cambio de estado de sesión de Terminal Server puede usar para desencadenar una tarea para iniciarla.               |
| [**ESTADO DE \_ TAREA**](/windows/desktop/api/taskschd/ne-taskschd-task_state)                                            | Define los distintos estados en los que puede estar una tarea registrada.                                                   |
| [**TASK \_ TRIGGER \_ TYPE2**](/windows/desktop/api/taskschd/ne-taskschd-task_trigger_type2)                                  | Define el tipo de desencadenadores que pueden usar las tareas.                                                          |
