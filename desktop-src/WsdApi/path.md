# <a name="path-element"></a>elemento path

Especifica la ubicación y el nombre de archivo de un archivo WSDL. La ruta de acceso puede ser una ruta de acceso absoluta o relativa al archivo, pero no debe ser una dirección URL.

## <a name="usage"></a>Uso

``` syntax
<path/>
```

## <a name="attributes"></a>Atributos

No hay atributos.

## <a name="child-elements"></a>Elementos secundarios

No hay elementos secundarios.

## <a name="parent-elements"></a>Elementos primarios



| Elemento                         | Descripción                                                             |
|---------------------------------|-------------------------------------------------------------------------|
| [**mencionados**](wsdl.md)<br/> | Archivo WSDL para procesar la información de contrato.<br/> <br/> |



## <a name="examples"></a>Ejemplos

En el código XML siguiente se muestra un elemento **path** válido.

``` syntax
<path>"..\wsdls\example.wsdl"</path>
```
