# <a name="framework-security"></a>Seguridad del marco

El Windows Biometric Service permite a las aplicaciones cliente sin privilegios realizar operaciones biométricas sin proporcionarles acceso directo a muestras biométricas o hardware. El servicio crea un límite de seguridad entre los clientes y los componentes biométricos, como se muestra en la ilustración siguiente.
