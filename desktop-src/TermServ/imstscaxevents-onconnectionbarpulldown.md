# <a name="imstscaxeventsonconnectionbarpulldown-method"></a>IMsTscAxEvents:: OnConnectionBarPullDown (método)

Se llama cuando el usuario ha arrastrado hacia abajo en la barra de conexión.

## <a name="syntax"></a>Sintaxis


```C++
void OnConnectionBarPullDown();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
