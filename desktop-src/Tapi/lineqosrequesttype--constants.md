# <a name="lineqosrequesttype_-constants"></a>Constantes de LINEQOSREQUESTTYPE \_

Un TSP usa estas constantes para identificar el tipo de una solicitud QoS (calidad de servicio) que se realiza. Actualmente, solo se admite una solicitud de nivel de servicio. Vea [ \_ constantes LINEQOSSERVICELEVEL](lineqosservicelevel--constants.md).

<dl> <dt>

<span id="LINEQOSREQUESTTYPE_SERVICELEVEL"></span><span id="lineqosrequesttype_servicelevel"></span>**LINEQOSREQUESTTYPE \_ SERVICELEVEL**
</dt> <dd> <dl> <dt>

 0x00000001
</dt> <dt>



La solicitud QoS que se está realizando es para un nivel de servicio determinado.


</dt> </dl> </dd> </dl>
