# <a name="performance-counters"></a>Contadores de rendimiento

Los contadores de rendimiento de Windows proporcionan una capa de abstracción de alto nivel que proporciona una interfaz coherente para recopilar diversos tipos de datos del sistema, como la CPU, la memoria y el uso del disco. Los administradores del sistema suelen usar los contadores de rendimiento para supervisar los sistemas en busca de problemas de rendimiento o de comportamiento. A menudo, los desarrolladores de software usan contadores de rendimiento para examinar el uso de los recursos de sus programas.
