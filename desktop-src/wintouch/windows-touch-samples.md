# <a name="windows-touch-samples"></a>Windows Ejemplos táctiles

Describe la ruta de acceso al Windows touch y proporciona breves información general sobre los ejemplos.

Al instalar el Windows [7 Touch Samples](), puede elegir dónde están instalados los ejemplos. Por ejemplo, si va a instalar en la unidad C, el SDK tendrá como valor predeterminado "C: Archivos de programa SDK de \\ Microsoft Windows ejemplos de \\ \\ \\ v7.0". \\ Dentro del directorio Samples (Ejemplos), se encuentra la carpeta Touch. Esta carpeta contiene todos los ejemplos Windows Touch.

Los siguientes Windows touch se envían con el [SDK Windows 7.]()

- [Windows Ejemplo de gesto táctil (MTGestures)](windows-touch-gesture-sample--mtgestures-.md)

- [Windows Ejemplo de manipulación táctil (MTManipulation)](windows-touch-manipulation-sample--mtmanipulation-.md)
- [Windows Ejemplo de Touch Scratchpad (MTScratchpadWMTouch)](windows-touch-scratchpad-sample--mtscratchpadwmtouch-.md)
- [Windows Ejemplo de Scratchpad táctil en C# (MTScratchpadWMTouchCS)](windows-touch-scratchpad-sample-in-c---mtscratchpadwmtouchcs-.md)
- [Windows Touch Scratchpad mediante el ejemplo de lápiz óptico en tiempo real (MTScratchpadRTStylus)](windows-touch-scratchpad-using-the-real-time-stylus-sample--mtscratchpadrtstylus-.md)
- [Windows Touch Scratchpad con el lápiz óptico en tiempo real en el ejemplo de C# (MTScratchpadRTStylusCS)](windows-touch-scratchpad-using-the-real-time-stylus-in-c--sample--mtscratchpadrtstyluscs-.md)
