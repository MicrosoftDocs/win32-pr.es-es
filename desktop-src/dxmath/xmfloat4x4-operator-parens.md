# <a name="xmfloat4x4-operator--operators"></a>Operadores del operador XMFLOAT4X4 ()

Tiene acceso a elementos de matriz específicos a los que hace referencia fila y columna desde la instancia actual de `XMFLOAT4X4` .

Tiene acceso a elementos de matriz específicos a los que hace referencia la fila y la columna desde la instancia actual de [ **XMFLOAT4X4.**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4x4)

|----------|-------------|
