# <a name="idcompositionskewtransformsetcenterx-methods"></a>IDCompositionSkewTransform:: SetCenterX (métodos)

Cambia o anima el valor de la propiedad CenterX de una transformación de sesgo 2D. La propiedad CenterX especifica la coordenada x del punto sobre el que se realiza el sesgo.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                                       | Descripción                                            |
|:-------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcenterx(float))                                     | Cambia el valor de la propiedad CenterX.<br/>  |
| [**SetCenterX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionskewtransform-setcenterx(idcompositionanimation)) | Anima el valor de la propiedad CenterX.<br/> |
