# <a name="smtp-mail-address-attribute"></a>SMTP-correo-dirección (atributo)

Atributo de dirección de correo genérico. Se usa en el cuadro como atributo opcional de objetos de servidor, donde lo consume la replicación DS basada en correo (si los equipos están configurados).



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | SMTP-dirección de correo electrónico                           |
| Nombre para mostrar de LDAP | mailAddress                                 |
| Tamaño              | \-                                          |
| Actualizar privilegio  | Administrador de dominio                        |
| Frecuencia de actualización  | Casi nunca                                |
| Attribute-Id      | 1.2.840.113556.1.4.786                      |
| System-ID-GUID    | 26d9736f-6070-11d1-a9c6-0000f80367c1        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
