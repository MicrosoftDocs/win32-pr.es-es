# <a name="access-permission-flags"></a>Marcas de permiso de acceso

Las marcas de permiso de acceso especifican el tipo de acceso que una aplicación cliente tiene a un objeto abierto: lectura, escritura o lectura y escritura. El permiso de lectura permite a una aplicación leer el contenido de una secuencia, pero no escribir cambios en la secuencia. El permiso de escritura permite a una aplicación llamar a una función que confirma los cambios en el almacenamiento de un objeto, pero no leer el contenido de la secuencia. El permiso de lectura y escritura permite a una aplicación leer las secuencias del objeto o escribir en el almacenamiento del objeto.
