# <a name="show-servicestate"></a>show servicestate

Muestra una instantánea del servicio HTTP.

``` syntax
show servicestate [[view=]{session|requestq}] [[verbose=]{yes|no}]
 
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

<span id="__view___session_requestq__"></span><span id="__VIEW___SESSION_REQUESTQ__"></span>**\[\[View = \] {Session \| requestq}\]**
</dt> <dd>

Ver la instantánea del estado del servicio HTTP en función de la sesión del servidor o de las colas de solicitudes.

</dd> <dt>

<span id="__verbose___yes_no__"></span><span id="__VERBOSE___YES_NO__"></span>**\[\[verbose = \] {sí \| no}\]**
</dt> <dd>

Vea también la información detallada que muestra la información de propiedades.

</dd> </dl>

## <a name="examples"></a>Ejemplos

**Mostrar vista de servicestate = sesión**

**Show servicestate View = requestq verbose = Yes**
