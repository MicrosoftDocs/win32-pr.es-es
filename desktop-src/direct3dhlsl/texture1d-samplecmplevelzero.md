# <a name="texture1dsamplecmplevelzero-methods"></a>Texture1D:: SampleCmpLevelZero (métodos)

Muestrea un [**Texture1D**](sm5-object-texture1d.md) solo en el nivel 0 de mipmap y compara el resultado con un valor de comparación.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                               | Descripción                                                                                                                            |
|:-----------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmpLevelZero (S, Float, Float, int)**](dx-graphics-hlsl-to-samplecmplevelzero.md)              | Muestrea solo una textura en el nivel 0 del mipmap y compara el resultado con un valor de comparación.<br/>                                     |
| [**SampleCmpLevelZero (S, Float, Float, int, uint)**](t1d-samplecmplevelzero-s-float-float-int-uint-.md) | Muestrea solo una textura en el nivel 0 del mipmap y compara el resultado con un valor de comparación. Devuelve el estado de la operación.<br/> |
