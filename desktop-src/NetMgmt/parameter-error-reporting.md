# <a name="parameter-error-reporting"></a>Informes de errores de parámetros

Las funciones de administración de red Add y SetInfo devuelven un índice para un parámetro que está en error. El autor de la llamada puede pasar **un puntero NULL** para el parámetro *\_ parm err* que indica que la función no debe establecer el campo.
