# <a name="adding-deleting-and-replacing-resources"></a><span data-ttu-id="dce4d-103">Agregar, eliminar y reemplazar recursos</span><span class="sxs-lookup"><span data-stu-id="dce4d-103">Adding, Deleting, and Replacing Resources</span></span>

<span data-ttu-id="dce4d-104">Las aplicaciones deben agregar, eliminar o reemplazar recursos con frecuencia en archivos ejecutables.</span><span class="sxs-lookup"><span data-stu-id="dce4d-104">Applications must frequently add, delete, or replace resources in executable files.</span></span> <span data-ttu-id="dce4d-105">Se pueden usar dos métodos para llevar a cabo estas tareas.</span><span class="sxs-lookup"><span data-stu-id="dce4d-105">Two methods can be used to accomplish these tasks.</span></span> <span data-ttu-id="dce4d-106">La primera es editar el archivo de definición de recursos, volver a compilar los recursos y agregar los recursos recompilados al archivo ejecutable de la aplicación.</span><span class="sxs-lookup"><span data-stu-id="dce4d-106">The first is to edit the resource-definition file, recompile the resources, and add the recompiled resources to the application's executable file.</span></span> <span data-ttu-id="dce4d-107">El segundo método consiste en copiar los datos de recursos directamente en el archivo ejecutable de la aplicación.</span><span class="sxs-lookup"><span data-stu-id="dce4d-107">The second method is to copy the resource data directly into the application's executable file.</span></span>

<span data-ttu-id="dce4d-108">Por ejemplo, para localizar una aplicación en inglés para su uso en Noruega, puede que sea necesario reemplazar el cuadro de diálogo inglés por otro con noruego.</span><span class="sxs-lookup"><span data-stu-id="dce4d-108">For example, to localize an English-language application for use in Norway, it may be necessary to replace the English dialog box with one using Norwegian.</span></span> <span data-ttu-id="dce4d-109">Un programador crea un cuadro de diálogo adecuado mediante un editor de cuadros de diálogo o escribiendo una plantilla en el archivo de definición de recursos.</span><span class="sxs-lookup"><span data-stu-id="dce4d-109">A developer creates an appropriate dialog box by using a dialog box editor or by writing a template in the resource-definition file.</span></span> <span data-ttu-id="dce4d-110">A continuación, el desarrollador vuelve a compilar los recursos y agrega los nuevos recursos al archivo ejecutable de la aplicación.</span><span class="sxs-lookup"><span data-stu-id="dce4d-110">The developer then recompiles the resources and adds the new resources to the application's executable file.</span></span>

<span data-ttu-id="dce4d-111">Sin embargo, si existe un cuadro de diálogo adecuado en formato binario, el desarrollador puede copiar los datos directamente en el archivo ejecutable que se está localizando mediante las siguientes funciones.</span><span class="sxs-lookup"><span data-stu-id="dce4d-111">If an appropriate dialog box exists in binary form, however, the developer can copy the data directly to the executable file being localized by using the following functions.</span></span> <span data-ttu-id="dce4d-112">La función [**BeginUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-beginupdateresourcea) crea un identificador de actualización para el archivo ejecutable cuyos recursos se van a cambiar.</span><span class="sxs-lookup"><span data-stu-id="dce4d-112">The [**BeginUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-beginupdateresourcea) function creates an update handle to the executable file whose resources are to be changed.</span></span> <span data-ttu-id="dce4d-113">La función [**UpdateResource**](/windows/desktop/api/Winbase/nf-winbase-updateresourcea) usa este identificador para agregar, eliminar o reemplazar un recurso en el archivo ejecutable.</span><span class="sxs-lookup"><span data-stu-id="dce4d-113">The [**UpdateResource**](/windows/desktop/api/Winbase/nf-winbase-updateresourcea) function uses this handle to add, delete, or replace a resource in the executable file.</span></span> <span data-ttu-id="dce4d-114">La función [**EndUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-endupdateresourcea) cierra el identificador.</span><span class="sxs-lookup"><span data-stu-id="dce4d-114">The [**EndUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-endupdateresourcea) function closes the handle.</span></span>

<span data-ttu-id="dce4d-115">Una vez que [**BeginUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-beginupdateresourcea)crea un identificador de actualización para un archivo ejecutable, una aplicación puede usar [**UpdateResource**](/windows/desktop/api/Winbase/nf-winbase-updateresourcea) repetidamente para realizar cambios en los datos del recurso.</span><span class="sxs-lookup"><span data-stu-id="dce4d-115">After an update handle to an executable file is created by [**BeginUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-beginupdateresourcea), an application can use [**UpdateResource**](/windows/desktop/api/Winbase/nf-winbase-updateresourcea) repeatedly to make changes to the resource data.</span></span> <span data-ttu-id="dce4d-116">Cada llamada a **UpdateResource** contribuye a una lista interna de adiciones, eliminaciones y reemplazos, pero no escribe realmente los datos en el archivo ejecutable.</span><span class="sxs-lookup"><span data-stu-id="dce4d-116">Each call to **UpdateResource** contributes to an internal list of additions, deletions, and replacements but does not actually write the data to the executable file.</span></span> <span data-ttu-id="dce4d-117">Inmediatamente antes de cerrar el identificador de actualización, [**EndUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-endupdateresourcea) escribe los cambios acumulados en el archivo ejecutable.</span><span class="sxs-lookup"><span data-stu-id="dce4d-117">Immediately before closing the update handle, [**EndUpdateResource**](/windows/desktop/api/Winbase/nf-winbase-endupdateresourcea) writes the accumulated changes to the executable file.</span></span>

<span data-ttu-id="dce4d-118">A veces, una aplicación debe copiar recursos desde otro archivo.</span><span class="sxs-lookup"><span data-stu-id="dce4d-118">Sometimes, an application must copy resources from another file.</span></span> <span data-ttu-id="dce4d-119">[Actualizar recursos](using-resources.md) muestra un ejemplo de cómo obtener los datos de recursos y su tamaño de un archivo.</span><span class="sxs-lookup"><span data-stu-id="dce4d-119">[Updating Resources](using-resources.md) shows an example of obtaining the resource data and its size from a file.</span></span>
