# <a name="other-name-attribute"></a>Other-Name atributo)

Nombres adicionales para un usuario. Por ejemplo, segundo nombre, Patronymic, matronymic u otros.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Other-Name                                  |
| Nombre para mostrar de LDAP | middleName                                  |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 2.16.840.1.113730.3.1.34                    |
| System-ID-GUID    | bf9679f2-0de6-11d0-a285-00aa003049e2        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
