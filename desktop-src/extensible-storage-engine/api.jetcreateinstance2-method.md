# <a name="apijetcreateinstance2-method"></a>Método API. JetCreateInstance2

Asignar una nueva instancia del motor de base de datos para su uso en un único proceso, con un nombre para mostrar especificado.

**Espacio de nombres:**  [Microsoft. ISAM. esent. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Ensamblado:**  Microsoft. ISAM. esent. Interop (en Microsoft.Isam.Esent.Interop.dll)

``` vb

    <OutAttribute> ByRef instance As JET_INSTANCE, _

    grbit As CreateInstanceGrbit _
```

#### <a name="parameters"></a>Parámetros

  - instance  
    Tipo: [Microsoft.ISAM.esent.Interop.JET_INSTANCE](./jet-instance-structure.md)  
    
    Devuelve la instancia que se acaba de crear.

<!-- end list -->

  - name  
    Tipo: [System. String](/dotnet/api/system.string)  
    
    Especifica un identificador de cadena único para la instancia que se va a crear. Esta cadena debe ser única dentro de un proceso determinado que hospede el motor de base de datos.

<!-- end list -->

  - DisplayName  
    Tipo: [System. String](/dotnet/api/system.string)  
    
    Nombre para mostrar de la instancia de que se va a crear. Se utilizará en las entradas del registro de eventos.

<!-- end list -->

  - grbit  
    Tipo: [Microsoft. ISAM. esent. Interop. CreateInstanceGrbit](./createinstancegrbit-enumeration.md)  
    
    Opciones de creación.
