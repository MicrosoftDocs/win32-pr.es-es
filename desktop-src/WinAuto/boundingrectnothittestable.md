# <a name="boundingrectnothittestable"></a>BoundingRectNotHitTestable

## <a name="text"></a>Texto

El rectángulo indicado no parece contener un punto en el que este elemento se pueda recuperar con la prueba de posicionamiento.

## <a name="type"></a>Tipo

Advertencia

## <a name="description"></a>Descripción

La prueba de posicionamiento no puede obtener acceso al rectángulo delimitador del elemento. Por ejemplo, se encuentra fuera de las coordenadas físicas de la pantalla (o región visible) del escritorio.
