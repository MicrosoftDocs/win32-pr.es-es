# <a name="ui_pkey_decimalvalue"></a>UI \_ PKEY \_ DecimalValue

Identifica la propiedad \_ PKEY \_ DecimalValue de la interfaz de usuario.

```
propertyDescription
   name = UI_PKEY_DecimalValue
   shellPKey = UI_PKEY_DecimalValue
   formatID = 00000201-7363-696e-8441798acf5aebb7
   propID = 201
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Comentarios

La interfaz de usuario PKEY DecimalValue la usa una aplicación para consultar el valor en el \_ \_ campo de edición del control [Spinner.](windowsribbon-controls-spinner.md)
