# <a name="building-rpc-applications"></a>Compilar aplicaciones RPC

El procedimiento para crear una aplicación de llamada a procedimiento remoto (RPC) distribuida varía ligeramente en función de:

-   Plataforma del sistema operativo en la que se está desarrollando.
-   Plataforma de destino.
-   Versión del compilador de MIDL y C o C++ que se usa.
-   Las bibliotecas de API que se usan.

En esta sección se describe el proceso de creación de aplicaciones cliente/servidor con la llamada a procedimiento remoto de Microsoft. Presenta la información de los temas siguientes:

-   [Procedimiento de compilación general](general-build-procedure.md)
-   [Control de excepciones](exception-handling.md)
