# <a name="installing-your-skin"></a>Instalación de la máscara

Use el procedimiento siguiente para instalar la máscara:

1.  Copie todos los archivos de máscara en una nueva carpeta.
2.  Copie la carpeta que contiene los archivos de máscara en la carpeta Windows Media Player en el dispositivo portátil.

    Inicie Windows Media Player Mobile. Esto inicia el **selector de máscara**, que tiene dos botones de navegación que le permiten desplazarse por las máscaras disponibles y mostrará una vista previa de cada uno de ellos. El nombre que se muestra para la máscara será el nombre del archivo de definición de máscara.

3.  Puntee en **Aceptar** para seleccionar la máscara.
