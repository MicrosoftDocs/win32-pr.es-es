# <a name="dialect"></a>Dialect

Dos de las interfaces ADSI para buscar un directorio, ADO y OLE DB, usan una cadena de búsqueda para especificar los parámetros de la búsqueda. Los parámetros de búsqueda incluyen la base de la búsqueda, el filtro de búsqueda, los atributos que se recuperarán y la profundidad de la búsqueda.

Puede especificar la cadena de búsqueda en uno de estos dos dialectos:

-   [Dialecto LDAP](ldap-dialect.md)
-   [SQL dialecto](sql-dialect.md)
