# <a name="endif"></a>\#endif

La directiva **\# endif** marca el final de un bloque de compilación condicional definido por una directiva **\# ifdef** . Se requiere un **\# endif** para cada directiva **\# If**, **\# ifdef** o **\# ifndef** .

``` syntax
#endif
```

Esta Directiva no tiene parámetros.
