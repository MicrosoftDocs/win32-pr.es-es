| [**QuickAccessToolbar.ApplicationDefaults**](windowsribbon-element-quickaccesstoolbar-applicationdefaults.md) | Representa un contenedor para los comandos predeterminados en la [barra de herramientas de acceso rápido (Qat)](windowsribbon-controls-quickaccesstoolbar.md).<br/>                                                                                                                   |

ButtonItem**](windowsribbon-element-splitbutton-buttonitem.md)                                 | Representa un contenedor para un [botón](windowsribbon-controls-button.md) o [botón de alternancia](windowsribbon-controls-togglebutton.md) que expone el comando predeterminado de un [botón de expansión](windowsribbon-controls-splitbutton.md).<br/>                |

MenuGroups**](windowsribbon-element-splitbutton-menugroups.md)                                 | Representa un contenedor para un conjunto de elementos de menú desplegable en un control de [botón de expansión](windowsribbon-controls-splitbutton.md) estándar.<br/>                                                                                                            |
