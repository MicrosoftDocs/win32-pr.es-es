# <a name="monitoring"></a>Supervisión

La supervisión es una característica definida por el proveedor de servicios que detecta señales que indican otros medios. Por ejemplo, una aplicación puede reproducir un mensaje de voz "dejar un mensaje" saliente mientras la llamada entrante comienza a enviar un tono "llamando" y espera a un protocolo de enlace. Para no perder la llamada de fax, la aplicación local supervisa este tono mientras se reproduce el mensaje de voz.

La supervisión comprende la [supervisión multimedia](/previous-versions/windows/desktop/legacy/ms725242(v=vs.85)), la supervisión de los [dígitos](/previous-versions/windows/desktop/legacy/ms725185(v=vs.85))y la [supervisión de tonos](/previous-versions/windows/desktop/legacy/ms725520(v=vs.85)).
