# <a name="error-handling-cryptography"></a>Control de errores (criptografía)

En todos los ejemplos de C que acompañan a la documentación de cada función, la función de ejemplo [**MyHandleError**](myhandleerror.md) se usa para imprimir un mensaje de error y salir de un procedimiento. En situaciones reales, pueden ser necesarios procedimientos de control de errores más amplios. Para obtener más información, vea [Ejemplo de limitaciones del programa](example-program-limitations.md).
