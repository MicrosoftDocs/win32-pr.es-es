# <a name="the-microsoft-agent-user-interface"></a>La interfaz de usuario del agente de Microsoft

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Microsoft Agent permite que los sitios web y las aplicaciones convencionales incluyan un formulario mejorado de interacción con el usuario. Proporciona varios componentes de interfaz de usuario que permiten a los usuarios tener acceso al carácter e interactuar con él, conocer el estado del carácter y cambiar la configuración global que afecta a todos los caracteres.

En las secciones siguientes se describen estos elementos básicos de la interfaz de usuario del agente de Microsoft:

-   [La ventana de caracteres](the-character-window.md)
-   [El icono de la barra de tareas carácter](the-character-taskbar-icon.md)
-   [Ventana comandos de voz](the-voice-commands-window.md)
-   [El globo de palabras](the-word-balloon.md)
-   [La ventana de la sugerencia de escucha](the-listening-tip-window.md)
-   [La ventana Opciones de carácter avanzadas](the-advanced-character-options-window.md)
-   [La ventana Propiedades de carácter predeterminado](the-default-character-properties-window.md)
