# <a name="dvd_register"></a>registro de DVD \_

El tipo de datos del **\_ registro de DVD** contiene un valor para un registro de parámetros generales de DVD (GPRM) o un registro de parámetros del sistema (SPRM).


```C++
typedef WORD DVD_REGISTER;
```
