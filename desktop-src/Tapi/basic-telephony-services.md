# <a name="basic-telephony-services"></a>Servicios básicos de telefonía

Los servicios básicos de telefonía son un subconjunto mínimo de la especificación de telefonía. Dado que todos los proveedores de servicios deben admitir las funciones de los servicios de telefonía básica, las aplicaciones que usan solo estas funciones funcionarán con cualquier proveedor de servicios TAPI. La funcionalidad contenida en La telefonía básica se corresponde aproximadamente con las características de POTS.

Muchos programadores solo usarán los servicios que proporciona La telefonía básica. Otros, como los que escriben código para sistemas telefónicos DE UNIX, necesitarán las funciones de telefonía complementaria.

Para obtener una lista de las funciones de telefonía básica, vea [Referencia de función rápida tapi](./tapi-quick-function-reference.md).

Dado que no se supone que todos los proveedores de servicios ofrecen el control de los dispositivos telefónicos, los servicios de dispositivos telefónicos se consideran opcionales. Es decir, no forman parte de la telefonía básica. Para obtener una lista de servicios de dispositivos telefónicos, vea [Servicios de telefonía complementaria](supplementary-telephony-services.md); Para obtener más información sobre los dispositivos de teléfono, vea [Clases de dispositivos TAPI](./tapi-device-classes.md).
