# <a name="dependencymodule-property"></a>Dependency. Module (propiedad)

La propiedad **módulo** de solo lectura del [**objeto de dependencia**](dependency-object.md) devuelve el valor de moduleId del módulo requerido por la cadena actual en forma de **BSTR**. ModuleID es el mismo formato que se usa en la [tabla ModuleSignature](modulesignature-table.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Dependency.Module
```



## <a name="property-value"></a>Valor de propiedad

## <a name="c"></a>C++

Vea el método [**IMsmDependency \_ Get \_ Module**](/windows/win32/api/mergemod/nf-mergemod-imsmdependency-get_module) .
