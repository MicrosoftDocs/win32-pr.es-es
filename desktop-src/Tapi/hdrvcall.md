# <a name="hdrvcall"></a>HDRVCALL

El tipo de datos [**HDRVCALL**](hdrvline.md) representa el identificador opaco de un proveedor de servicios para una estructura de datos de llamada. El proveedor de servicios debe resolver un valor de este tipo en una referencia a la instancia de la estructura de datos adecuada.
