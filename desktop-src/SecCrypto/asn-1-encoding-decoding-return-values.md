# <a name="asn1-encodingdecoding-return-values"></a>Valores devueltos de codificación/descodificación ASN. 1

| Constante definida           | mensaje de texto                                      | Valor hexadecimal |
|----------------------------|---------------------------------------------------|-------------------|
| ERROR de CIFRAdo \_ E \_ ASN1 \_      | Base del valor de código de retorno/descodificación de certificado ASN. 1 | 0x80093100        |
| CIFRAr \_ E \_ \_ interno ASN1   | Error de codificación o descodificación de ASN. 1 interno             | 0x80093101        |
| CRYPT \_ E \_ ASN1 \_ denominadas EOD        | No se esperaba el final de los datos de ASN. 1                      | 0x80093102        |
| CIFRAdo \_ E \_ ASN1 \_ dañado    | Datos dañados ASN. 1                              | 0x80093103        |
| CIFRAr \_ E \_ ASN1 \_ grande      | Valor de ASN. 1 demasiado grande                             | 0x80093104        |
| CRYPT \_ E \_ ASN1 ( \_ restricción) | Se infringió la restricción ASN. 1                         | 0x80093105        |
| CIFRAr \_ E \_ \_ memoria ASN1     | ASN. 1 de memoria insuficiente                               | 0x80093106        |
| Desbordar el CIFRAdo \_ E \_ ASN1 \_   | Desbordamiento de búfer ASN. 1                             | 0x80093107        |
| CRYPT \_ E \_ ASN1 \_ BADPDU     | Función ASN. 1 no compatible con esta PDU         | 0x80093108        |
| CRYPT \_ E \_ ASN1 \_ BADARGS    | Argumentos incorrectos de ASN. 1 para la llamada de función              | 0x80093109        |
| CRYPT \_ E \_ ASN1 \_ BADREAL    | ASN. 1 valor real incorrecto                              | 0x8009310A        |
| CRYPT \_ E \_ ASN1 \_ BADTAG     | ASN. 1 valor de etiqueta incorrecta alcanzado                           | 0x8009310B        |

| CIFRAr \_ E \_ ASN1 ( \_ regla)       | Regla de codificación incorrecta ASN. 1                           | 0x8009310D        |
| CIFRAr \_ E \_ ASN1 \_ UTF8       | Un Unicode no válido de ASN. 1 (UTF8)                          | 0x8009310E        |
| CIFRAr \_ E \_ \_ tipo PDU ASN1 \_  | Tipo PDU incorrecto ASN. 1                                | 0x80093133        |
| CIFRAr \_ E \_ ASN1 \_ Nyi        | ASN. 1 aún no implementado                         | 0x80093134        |
| CIFRAdo \_ E \_ ASN1 \_ extendido   | ASN. 1 omitió extensiones desconocidas                  | 0x80093201        |
| CRYPT \_ E \_ ASN1 \_ NOEOD      | Se esperaba un fin de datos de ASN. 1.                        | 0x80093202        |
