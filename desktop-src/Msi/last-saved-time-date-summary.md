# <a name="last-saved-timedate-summary-property"></a>Propiedad de Resumen de fecha y hora de la última actualización

La propiedad de **Resumen de fecha y hora** de la última modificación transmite la última vez que se modificó el paquete de instalación, la transformación o el paquete de revisión.

Inicialmente, un autor debe establecer el valor de la propiedad de **Resumen de fecha y hora de la última vez** que se ha guardado en null para indicar que aún no se han realizado cambios en el paquete. Después, un autor debe actualizar la propiedad de **Resumen de fecha y hora** de la última vez que se guardó en la fecha y hora actuales del sistema cada vez que se guarda una base de datos de instalación modificada, una transformación o un paquete de revisión.
