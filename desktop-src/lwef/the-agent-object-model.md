# <a name="the-agent-object-model"></a>El modelo de objetos del agente

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

El modelo de objetos de agente de Microsoft consta de los siguientes objetos:

-   Request
-   Agente (control)
-   Caracteres (colección)
-   Carácter
-   Comandos (colección)
-   Get-Help
-   LDM
-   AnimationNames (colección)
-   SpeechInput
-   AudioOutput
-   CommandsWindow
-   Hoja

Estos objetos se organizan en la jerarquía siguiente. (La línea de puntos que sigue a un objeto indica que pueden existir varios objetos).
