# <a name="h-network-monitor"></a>H (Monitor de red)

<dl> <dt>

<span id="_netmon_handoff_set_gly"></span><span id="_NETMON_HANDOFF_SET_GLY"></span>**conjunto de entrega**
</dt> <dd>

Un conjunto de protocolos que sigue a un protocolo. Un analizador utiliza el conjunto de entrega de un protocolo si el analizador puede detectar el protocolo siguiente a partir de los datos de la captura. Vea también [ *siguiente conjunto*](f.md)

</dd> <dt>

<span id="_netmon_handoff_table_gly"></span><span id="_NETMON_HANDOFF_TABLE_GLY"></span>**tabla de entrega**
</dt> <dd>

Tabla que contiene el conjunto de entrega del analizador. Puede usar una tabla de entrega para tener acceso a la información del conjunto de entrega, sin tener acceso al archivo INI definido por el analizador. La información del conjunto de entrega se almacena en el archivo INI de cada analizador. Vea también *conjunto de entrega*

</dd> <dt>

<span id="_netmon_hex_pane_gly"></span><span id="_NETMON_HEX_PANE_GLY"></span>**panel Hex**
</dt> <dd>

Panel de la ventana del visor de tramas Monitor de red que muestra los bytes hexadecimales transmitidos en fotogramas que se incluyen en una captura. De forma predeterminada, el panel Hex aparece en la parte inferior de la ventana.

</dd> </dl>
