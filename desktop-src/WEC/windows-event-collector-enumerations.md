# <a name="windows-event-collector-enumerations"></a>Enumeraciones del recopilador de eventos de Windows

En la tabla siguiente se enumeran las enumeraciones del SDK del recopilador de eventos de Windows.



| Enumeración                                                                                               | Descripción                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**\_modo de \_ configuración de suscripción de EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_configuration_mode)                       | Especifica modos de configuración diferentes que cambian la configuración predeterminada de una suscripción.                                            |
| [**\_formato de \_ contenido de suscripción de EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_content_format)                               | Especifica cómo se representarán los eventos en el equipo que envía los eventos antes de que los eventos se envíen al equipo del recopilador de eventos. |
| [**\_tipo de \_ credenciales de suscripción de EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_credentials_type)                           | Especifica el tipo de credenciales que se van a usar al comunicarse con orígenes de eventos.                                                         |
| [**\_modo de \_ entrega de suscripción de EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_delivery_mode)                                 | Especifica cómo se entregan los eventos a través de una suscripción de eventos (mediante un modelo de inserción o de extracción).                                          |
| [**identificador de la propiedad de suscripción de EC \_ \_ \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_property_id)                                     | Define valores para identificar las propiedades de suscripción de eventos usadas para la configuración de la suscripción.                                           |
| [**estado \_ activo de estado de tiempo de ejecución de suscripción de EC \_ \_ \_ \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_active_status) | Especifica el estado de una suscripción o un origen de eventos con respecto a una suscripción.                                               |
| [**\_identificador de \_ información de estado de tiempo de ejecución de suscripción de \_ \_ EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_info_id)             | Especifica un valor que identifica una propiedad del estado de tiempo de ejecución de un origen de eventos o una suscripción.                                |
| [**\_tipo de suscripción de EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_type)                                                    | Especifica el tipo de suscripción que se va a usar (una suscripción iniciada o iniciada por el origen).                                     |
| [**\_tipo de variante EC \_**](/windows/desktop/api/Evcoll/ne-evcoll-ec_variant_type)                                                              | Define los valores que especifican los tipos de datos que se utilizan en las funciones del recopilador de eventos de Windows.                                  |
