# <a name="idxcoreadapterlistgetadaptercount-method"></a>IDXCoreAdapterList::GetAdapterCount (método)

Recupera el número de adaptadores de un objeto de lista de adaptadores DXCore. Para obtener instrucciones de programación y ejemplos de código, [vea Uso de DXCore para enumerar adaptadores.](../dxcore-enum-adapters.md)

## <a name="syntax"></a>Sintaxis

```cpp
virtual uint32_t STDMETHODCALLTYPE GetAdapterCount() = 0;
```

## <a name="returns"></a>Devoluciones

Tipo: **uint32_t**

Devuelve el número de elementos de adaptador de la lista.
