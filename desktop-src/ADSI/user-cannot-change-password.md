# <a name="user-cannot-change-password-ldap-provider"></a>El usuario no puede cambiar la contraseña (proveedor LDAP)

La capacidad de un usuario de cambiar su propia contraseña es un permiso que se puede conceder o denegar. Para obtener más información acerca de cómo leer y modificar este permiso mediante programación con el proveedor LDAP, consulte:

-   [Lectura de usuario no puede cambiar la contraseña (proveedor LDAP)](reading-user-cannot-change-password-ldap-provider.md)
-   [La modificación del usuario no puede cambiar la contraseña (proveedor LDAP)](modifying-user-cannot-change-password-ldap-provider.md)

Es más fácil leer y modificar la configuración del usuario no puede cambiar la contraseña mediante el proveedor de Winnt. Para obtener más información, vea el [usuario no puede cambiar la contraseña (proveedor WinNT)](winnt-user-cannot-change-password.md).
