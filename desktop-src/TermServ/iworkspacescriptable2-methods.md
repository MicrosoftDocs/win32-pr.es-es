# <a name="iworkspacescriptable2-methods"></a>Métodos IWorkspaceScriptable2

La interfaz [**IWorkspaceScriptable2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable2) admite los siguientes métodos.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**Método StartWorkspaceEx**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-startworkspaceex)
</dt> <dd>

Asocia las credenciales de usuario y los certificados con un identificador de conexión; también contiene elementos de interfaz de usuario y seguridad adicionales.

</dd> <dt>

[**Método ResourceDismissed**](/windows/desktop/api/workspaceruntime/nf-workspaceruntime-iworkspacescriptable2-resourcedismissed)
</dt> <dd>

Alerta al usuario de que un recurso se ha deshabilitado o se ha descartado de otro modo.

</dd> </dl>
