# <a name="bits-enumerations"></a>Enumeraciones de BITS

Las [interfaces de servicio de transferencia inteligente en segundo plano (bits)](bits-interfaces.md) utilizan las siguientes enumeraciones.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|-|-|
| [**BG_AUTH_SCHEME**](/windows/win32/api/bits1_5/ne-bits1_5-bg_auth_scheme) | Define constantes que especifican el esquema de autenticación que se va a utilizar cuando un servidor proxy o un servidor solicite autenticación de usuario. |
| [**BG_AUTH_TARGET**](/windows/win32/api/bits1_5/ne-bits1_5-bg_auth_target) | Define constantes que especifican si las credenciales se utilizan para las solicitudes de autenticación de usuario de servidor o proxy. |
| [**BG_CERT_STORE_LOCATION**](/windows/win32/api/bits2_5/ne-bits2_5-bg_cert_store_location) | Define constantes que especifican la ubicación del almacén de certificados. |
| [**BG_ERROR_CONTEXT**](/windows/win32/api/bits/ne-bits-bg_error_context) | Define constantes que especifican el contexto en el que se produjo el error. |
| [**BG_JOB_PRIORITY**](/windows/win32/api/bits/ne-bits-bg_job_priority) | Define constantes que especifican el nivel de prioridad de un trabajo.  |
| [**BG_JOB_PROXY_USAGE**](/windows/win32/api/bits/ne-bits-bg_job_proxy_usage) | Define constantes que especifican el proxy que se va a utilizar para las transferencias de archivos. Puede definir una configuración de proxy diferente para cada trabajo. |
| [**BG_JOB_STATE**](/windows/win32/api/bits/ne-bits-bg_job_state) | Define constantes que especifican los distintos Estados de un trabajo. |
| [**BG_JOB_TYPE**](/windows/win32/api/bits/ne-bits-bg_job_type) | Define constantes que especifican el tipo de trabajo de transferencia, como la descarga. |
| [**BITS_COST_STATE**](./bits-cost-state.md) | Define constantes que especifican el estado del costo de BITS. |
| [**BITS_FILE_PROPERTY_ID**](/windows/win32/api/bits5_0/ne-bits5_0-bits_file_property_id) | Define constantes que especifican los valores de identificador correspondientes a las propiedades de copia en segundo plano. Vea [IBackgroundCopyFile5:: GetProperty](../delivery_optimization/ibackgroundcopyfile5-getproperty.md) y [SetProperty](../delivery_optimization/ibackgroundcopyfile5-setproperty.md). |
| [**BITS_JOB_PROPERTY_ID**](/windows/win32/api/bits5_0/ne-bits5_0-bits_job_property_id) | Define constantes que especifican el identificador de la propiedad para el trabajo de BITS. |
| [**BITS_JOB_TRANSFER_POLICY**](/windows/win32/api/bits5_0/ne-bits5_0-bits_job_transfer_policy) | Define constantes que especifican valores de identificador correspondientes a las propiedades de BITS. |
