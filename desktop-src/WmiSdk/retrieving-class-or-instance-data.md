# <a name="retrieving-wmi-class-or-instance-data"></a>Recuperar datos de instancia o clase WMI

Los objetos de datos principales de WMI son clases e instancias. En la tabla siguiente se enumeran las distintas técnicas disponibles para recuperar datos de una clase o instancia de WMI.



| Tema                                                                   | Descripción                                                                       |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| [Recuperar una clase WMI](retrieving-a-class.md)                        | Recuperación de información sobre los parámetros generales de un objeto .                    |
| [Recuperar una instancia de WMI](retrieving-an-instance.md)                 | Recuperación de información que describe un objeto único e individual en la empresa. |
| [Recuperar parte de una instancia de WMI](retrieving-part-of-an-instance.md) | Recuperar una o varias partes de una instancia, pero no toda la instancia.         |
| [Acceso a un calificador WMI](accessing-a-qualifier.md)                  | Acceso a más información contenida en un calificador.                              |



 

Para obtener más información, vea [Manipulating Class and Instance Information](manipulating-class-and-instance-information.md).
