# <a name="ms-pki-template-minor-revision-attribute"></a>atributo MS-PKI-Template-MINOR-revision

Realiza un seguimiento de los atributos de la clase que cambia. Sin embargo, esto no desencadenará la inscripción automática.



| Entrada | Value |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-Template-MINOR-revision                                                                    |
| Nombre para mostrar de LDAP | Atributo mspki-Template-MINOR-revision                                                                     |
| Tamaño              | 4 bytes                                                                                           |
| Actualizar privilegio  | Administrador de dominio                                                                              |
| Frecuencia de actualización  | Cuando se edita, crea o clona el objeto plantilla de certificado (MS-PKI-Certificate-template). |
| Attribute-Id      | 1.2.840.113556.1.4.1435                                                                           |
| System-ID-GUID    | 13f5236c-1884-46b1-b5d0-484e38990d58                                                              |
| Sintaxis            | [**Enumeración**](s-enumeration.md)                                                              |
