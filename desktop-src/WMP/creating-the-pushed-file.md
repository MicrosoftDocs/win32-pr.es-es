# <a name="creating-the-pushed-file"></a>Crear el archivo inserdo

Puede tomar los elementos que ha creado y reutilizarlos para crear las demás imágenes que necesite. Al combinar capas y colorear los resultados, puede crear un mapa de bits pushed que contenga las imágenes que desea ver cuando el usuario presione un botón. Este archivo también contiene la imagen secundaria para el estado en pausa del botón PlayPause.

La siguiente imagen es el archivo pushed.

![archivo de inserción](images/ceswmpus.png)

Los dos botones de la derecha son para el estado normal en pausa y el estado en pausa presionado del botón PlayPause. El color de los botones es amarillo, excepto para el estado normal en pausa del botón PlayPause, como un buen contraste con el color verde que se usa para el estado normal de todos los botones.
