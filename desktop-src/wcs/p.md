# <a name="p"></a>P

## <a name="primary-colors"></a>colores primarios

Los tres colores esenciales de un espacio de colores que se pueden mezclar para generar todos los demás colores.

## <a name="profile-connection-space-pcs"></a>Espacio de conexión de perfil (PCS)

Espacio de color independiente del dispositivo que se usa para las conversiones de colores y la coincidencia de colores.
