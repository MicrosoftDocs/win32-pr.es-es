# <a name="createinkdivider-function"></a>CreateInkDivider función)

Crea una instancia de una implementación de la interfaz [**InkDivider**](inkdivider-class.md) y devuelve su identificador.

Esta función no está pensada para ser utilizada por el código de la aplicación.

## <a name="syntax"></a>Sintaxis


```C++
INT_PTR CreateInkDivider(void);
```



## <a name="parameters"></a>Parámetros

Esta función no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve un identificador para el objeto creado.

## <a name="remarks"></a>Observaciones

Esta clase hereda de la clase base [**InkDivider**](inkdivider-class.md) e implementa funciones virtuales.
