# <a name="resource-compiler"></a>compilador de recursos

El compilador de recursos de Microsoft Windows (RC) es una herramienta que se usa en la creación de aplicaciones basadas en Windows. En esta información general se describe cómo crear un archivo de definición de recursos (Script), cómo compilar los recursos de la aplicación y cómo agregar recursos compilados a la aplicación.

Esta herramienta está disponible en Visual Studio y en el kit de desarrollo de software (SDK) de Microsoft Windows.

En los temas siguientes se describe la sintaxis del archivo de recursos y la línea de comandos de RC:

-   [Acerca de los archivos de recursos](about-resource-files.md)
-   [Using RC (The RC Command Line)](using-rc-the-rc-command-line-.md) [Uso de RC (la línea de comandos de RC)]
