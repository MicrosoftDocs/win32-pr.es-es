# <a name="dds"></a>DDS

El formato de archivo DirectDraw Surface (.dds) se introdujo con DirectX 7 para almacenar texturas sin comprimir y comprimidas (DXTn). El formato de archivo admite mapas mip, mapas de cubos y mapas de volumen. El formato de archivo DDS es compatible con [DirectXTex,](https://github.com/Microsoft/DirectXTex) [DirectXTK,](https://github.com/Microsoft/DirectXTK)D3DX heredado y otras herramientas de DirectX. A partir de Direct3D 10, los archivos DDS admiten matrices de textura.

> [!Note]  
> La biblioteca de utilidades D3DX (D3DX 9, D3DX 10 y [D3DX 11)](/windows/desktop/direct3d11/d3d11-graphics-reference-d3dx11)está en desuso para Windows 8 y no es compatible con las aplicaciones de Windows Store. Se recomienda usar [DirectXTex,](https://github.com/Microsoft/DirectXTex) [DirectXTK](https://github.com/Microsoft/DirectXTK)o ambos.
