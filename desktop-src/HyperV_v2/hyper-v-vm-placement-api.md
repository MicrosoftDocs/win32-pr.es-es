# <a name="hyper-v-vm-placement-api"></a>API de selección de ubicación de máquinas virtuales de Hyper-V

La API de selección de ubicación de máquina virtual (VM) de Hyper-V contiene la información de compatibilidad de las máquinas virtuales o el sistema del equipo host.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                    | Descripción                                                                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ CompatibilityVector**](msvm-compatibilityvector.md)<br/> | Hace referencia a la información de compatibilidad de una máquina virtual (VM) (cuando se ejecuta en un equipo de máquina virtual) o un host (cuando se ejecuta en un sistema de equipo host).<br/> |
