# <a name="sysmodal-dialog-style-bit"></a>Bit de estilo de diálogo SysModal

Si se establece este bit de estilo, el cuadro de diálogo detendrá todas las demás aplicaciones y ninguna otra aplicación podrá centrarse. Este estado permanece hasta que se descarta el cuadro de diálogo SysModal.

## <a name="value"></a>Valor



| Decimal | Hexadecimal | Constante                          |
|---------|-------------|-----------------------------------|
| 8       | 0x00000008  | **msidbDialogAttributesSysModal** |
