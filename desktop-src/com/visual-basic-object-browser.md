# <a name="visual-basic-object-browser"></a>Visual Basic Examinador de objetos

Una vez que se ha agregado un objeto COM al proyecto como una referencia o componente, puede usar el Examinador de objetos Visual Basic para ver la información de la biblioteca de tipos para ese objeto en Visual Basic.

Para ver un objeto COM en el Examinador de objetos:

-   En el menú **Ver** , haga clic en **Examinador de objetos**.

En el lado izquierdo de la ventana de Examinador de objetos se muestran todas las clases de referencias y componentes agregados al proyecto actual. El lado derecho contiene los métodos, las propiedades y los eventos de esas clases.

Para filtrar las clases de objeto COM que se muestran, seleccione una biblioteca de objetos en la lista desplegable biblioteca de selección.

Para obtener más información, vea [examinar bibliotecas de tipos de componentes ActiveX](/previous-versions/visualstudio/visual-basic-6/aa240782(v=vs.60)) en la documentación de Visual Basic.

En algunos casos, el Examinador de objetos de Visual Basic parece no mostrar un tipo de valor devuelto para una función o propiedad. No aparece ningún tipo de valor devuelto cuando el tipo de valor devuelto es el tipo predeterminado, **variante**. Si el panel Descripción de la Examinador de objetos describe el elemento como una función o una propiedad y no se muestra un tipo de valor devuelto, la instrucción de tipo de valor devuelto es "As Variant".
