# <a name="vds-reference"></a>Referencia de VDS

\[A partir de Windows 8 y Windows Server 2012, la interfaz com de [servicio de disco virtual](virtual-disk-service-portal.md) se sustituye por la [API de administración de almacenamiento de Windows](/previous-versions/windows/desktop/stormgmt/windows-storage-management-api-portal).\]

En esta sección se describe la interfaz de programación de aplicaciones para el servicio de disco virtual (VDS).

Los desarrolladores de aplicaciones usan las interfaces, estructuras, enumeraciones y constantes simbólicas en esta API para consultar y configurar el almacenamiento, desde discos hasta matrices de almacenamiento. Para obtener una visión detallada de la relación entre los tipos de VDS, consulte el [modelo de objetos de VDS](vds-object-model.md).

Los fabricantes de almacenamiento implementan muchas de las interfaces definidas en esta sección.
