# <a name="onfocus"></a>onfocus

El controlador de eventos **onfocus** controla un evento que se produce cuando el elemento recibe el foco de teclado.

``` syntax
onfocus
```

## <a name="remarks"></a>Observaciones

El controlador de eventos **onfocus** no es aplicable al elemento **Effects** o al elemento **popup** .

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|-----------------------------------------------------|
| Versión<br/> | Windows Media Player versión 70 o posterior<br/> |
