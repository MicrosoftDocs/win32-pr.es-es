# <a name="playerdvd"></a>Player.dvd

La **propiedad dvd** recupera el objeto DVD.

## <a name="syntax"></a>Syntax

*player* . **dvd**

## <a name="possible-values"></a>Valores posibles

Esta propiedad es un objeto DVD de solo lectura.

|--------------------|------------------------------------------------------------------------------------|
