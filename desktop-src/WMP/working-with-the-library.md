# <a name="working-with-the-library"></a>Trabajar con la biblioteca

La biblioteca es una base de datos de información sobre el contenido multimedia que se almacena en el equipo del usuario o se ha reproducido allí. Parte de la información se muestra en la característica de biblioteca de Reproductor de Windows Media. Se puede acceder a un conjunto mayor de información mediante programación.

Los elementos multimedia se pueden organizar en listas de reproducción. Los usuarios o a través del código pueden crear listas de reproducción. El Reproductor de Windows Media de objetos proporciona objetos y métodos que puede usar para manipular elementos multimedia individuales y listas de reproducción.

En las secciones siguientes se explican las técnicas para trabajar con la biblioteca:

-   [Acceso a la biblioteca](library-access.md)
-   [Administración de elementos multimedia](managing-media-items.md)
-   [Atributos de elementos multimedia](media-item-attributes.md)
-   [Administración de listas de reproducción](managing-playlists.md)
