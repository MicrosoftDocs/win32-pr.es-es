# <a name="running-the-application"></a>Ejecutar la aplicación

Para ejecutar la aplicación en un solo equipo, abra dos ventanas de consola. En la primera ventana, escriba

**Hellos**

y, en la segunda ventana, escriba

**helloc**

Para experimentar con diferentes secuencias de protocolo, puntos de conexión y otras opciones, compile la aplicación hello de ejemplo a partir de los archivos de origen en el directorio RPC Hello del Kit de desarrollo de \\ software (SDK) de plataforma.

> [!Note]  
> **La restricción de interfaz RPC** puede hacer que esta aplicación presente un error de acceso denegado en Windows XP con Service Pack 2 (SP2) y versiones posteriores de Windows.  Para obtener más información, vea [Rpc Interface Restriction](/previous-versions/windows/it-pro/windows-xp/bb457156(v=technet.10)#EGAA).
