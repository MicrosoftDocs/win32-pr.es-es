# <a name="xaudio2-macros"></a>Macros de XAudio2

Esta sección contiene información sobre las macros que proporciona la API de Microsoft XAudio2.



| Macro                          | Descripción                                                                                                                                               |
|--------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XAPOAlloc**](/windows/desktop/api/XAPO/nf-xapo-xapoalloc) | Macro de asignación de memoria que usan los métodos de IXAPO que deben asignar estructuras de tamaño arbitrario que se devuelven posteriormente a la aplicación.<br/> |
| [**XAPOFree**](/windows/desktop/api/XAPO/nf-xapo-xapofree)   | Macro que se usa para liberar memoria asignada con la macro XAPOAlloc.<br/>                                                                                  |
