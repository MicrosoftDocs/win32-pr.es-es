# <a name="d"></a>D

## <a name="dac"></a>DAC

Convertidor digital a analógico.

## <a name="device-link-profile"></a>Perfil de vínculo de dispositivo

Archivo que contiene una concatenación de conversiones de color que se usan con frecuencia.

## <a name="device-profile"></a>Perfil de dispositivo

Un archivo que contiene información sobre cómo caracterizar las capacidades de color de un dispositivo. Los perfiles de dispositivo ICC convierten los colores de la gama de un dispositivo en los equipos. También contiene la información necesaria para la conversión de colores de los equipos a la gama del dispositivo. Los perfiles de dispositivo XML WCS proporcionan los datos de medición y las referencias algorítmicas para realizar la conversión entre el espacio de dispositivo nativo y CIEXYZ.

## <a name="dmp"></a>DMP

El formato de Perfil de modelo de dispositivo de color es un formato de archivo basado en XML que contiene las medidas necesarias para describir la relación entre los valores de espacio de colores del dispositivo nativo y los valores de CIEXYZ.
