# <a name="windows-7-and-windows-server-2008-r2-application-quality-cookbook"></a>Windows guía de calidad de aplicaciones de Windows Server 2008 R2 y 7 y 2008 R2

La guía de calidad de aplicaciones de Windows 7 y *Windows Server 2008 R2* familiariza a los desarrolladores de aplicaciones con cómo comprobar la compatibilidad de sus aplicaciones con el nuevo sistema operativo y proporciona información general sobre los pocos problemas conocidos de compatibilidad de aplicaciones de Windows 7 y Windows Server 2008 R2. También señala las diferencias de rendimiento, confiabilidad y facilidad de uso, y proporciona vínculos a documentos técnicos detallados y otras instrucciones para desarrolladores.

## <a name="overview"></a>Información general

Windows 7 y Windows Server 2008 R2 presentan la tecnología de sistema operativo y la plataforma de desarrollo de software más recientes para que las usen desarrolladores de aplicaciones y empresas de todo el mundo. Como parte de la mejora de la seguridad, la confiabilidad, el rendimiento y la experiencia del usuario de Windows, se han introducido muchas características nuevas, se han mejorado las características existentes y se han quitado algunas características.

Aunque Windows 7 y Windows Server 2008 R2 son altamente compatibles con la mayoría de sus aplicaciones respectivas escritas para Windows XP, Windows Server 2003, Windows Vista, Windows Server 2008, Windows Server 2008 R2 y sus Service Pack, algunas interrupciones de compatibilidad son inevitables debido a innovaciones, ajuste de seguridad y mayor confiabilidad. En general, la compatibilidad de Windows 7 y Windows Server 2008 R2 con aplicaciones existentes es alta.

Este documento se basa en los conceptos que se representan en la guía [Windows Vista y Windows Server 2008 Application Compatibility](/previous-versions/bb757005(v=msdn.10)). (Es posible que este recurso no esté disponible en algunos idiomas y países o regiones). Al igual que este documento, proporciona los medios para familiarizarse con cómo comprobar la compatibilidad de las aplicaciones con el nuevo sistema operativo y proporciona información general sobre los pocos problemas conocidos de incompatibilidad de aplicaciones en Windows 7 y Windows Server 2008 R2. Pero más allá de eso, también señala diferencias en cuanto al rendimiento, la confiabilidad y la facilidad de uso, y proporciona vínculos a documentos técnicos detallados y otras instrucciones para desarrolladores.

Además, Microsoft invierte en varias características y herramientas nuevas y mejoradas para permitirle crear aplicaciones de mayor calidad y solucionar problemas cuando las aplicaciones no funcionan correctamente en Windows 7 y Windows Server 2008 R2. Entre estas herramientas se incluyen instrucciones sobre cómo calificar las aplicaciones cliente y servidor para participar en el programa Windows logo.

Esta *guía contiene* más de tres docenas de temas divididos en tres secciones principales:

-   [Nuevas características y mejoras](new-features-and-enhancements.md)

Le invitamos a consultar estos temas para obtener información sobre cómo optimizar la aplicación para aprovechar las ventajas Windows 7.
