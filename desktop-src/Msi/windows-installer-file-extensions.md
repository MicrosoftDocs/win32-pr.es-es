# <a name="windows-installer-file-name-extensions"></a>Extensiones de nombre de archivo Windows Installer

Las siguientes extensiones de nombre de archivo se usan con el Windows Installer.



| Extensión | Descripción                                |
|-----------|--------------------------------------------|
| .msi      | Windows Installer base de datos.                |
| . MSM      | Módulo de combinación de Windows Installer.            |
| .msp      | Windows Installer revisión.                   |
| .mst      | Windows Installer transformación.               |
| . IDT      | Tabla de base de datos de Windows Installer exportada. |
| . Cub      | Módulo de validación.                         |
| . PCP      | Archivo de creación de revisión de Windows Installer      |
