# <a name="networksettingsname-property"></a>NetworkSettings.Name propiedad

Para el scripting, obtiene o establece el nombre de un perfil de red. El nombre se usa con fines para mostrar.

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Syntax


```VB
NetworkSettings.Name As String
```



## <a name="property-value"></a>Valor de propiedad

Nombre de un perfil de red.
