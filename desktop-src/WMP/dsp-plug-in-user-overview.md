# <a name="dsp-plug-in-user-overview"></a><span data-ttu-id="21bc8-109">Información general del usuario del complemento DSP</span><span class="sxs-lookup"><span data-stu-id="21bc8-109">DSP Plug-in User Overview</span></span>

<span data-ttu-id="21bc8-110">Para el usuario, los complementos DSP son características complementarias que cambian de algún modo la forma en que el usuario experimenta el audio o el vídeo que Windows Media Player está reproduciendo.</span><span class="sxs-lookup"><span data-stu-id="21bc8-110">To the user, DSP plug-ins are add-on features that somehow change the way the user experiences the audio or video that Windows Media Player is playing.</span></span> <span data-ttu-id="21bc8-111">El resultado de un complemento DSP podría ser algo obvio, como los colores de la imagen de vídeo que se invierten, por lo que se parece a un negativo fotográfico o algo muy sutil, como un efecto de reverberación ligera en el audio para agregar una sensación de spaciousness al sonido.</span><span class="sxs-lookup"><span data-stu-id="21bc8-111">The result of a DSP plug-in could be something obvious, like the colors in the video image being reversed so it looks like a photographic negative, or something very subtle, like a light reverb effect in the audio to add a feeling of spaciousness to the sound.</span></span>

<span data-ttu-id="21bc8-112">Los usuarios pueden instalar y desinstalar los complementos DSP **mediante la pestaña complementos** del cuadro de diálogo Opciones de Windows Media Player.</span><span class="sxs-lookup"><span data-stu-id="21bc8-112">Users can install and uninstall DSP plug-ins by using the **Plug-ins** tab of the Options dialog box in Windows Media Player.</span></span> <span data-ttu-id="21bc8-113">Los usuarios pueden habilitar y deshabilitar los complementos instalados mediante las casillas proporcionadas **en la pestaña complementos** . los complementos DSP pueden implementar una página de propiedades, a la que los usuarios también pueden obtener acceso **desde la pestaña complementos** . Una página de propiedades proporciona al usuario una manera de cambiar la configuración que afecta a la forma en que funciona un complemento DSP.</span><span class="sxs-lookup"><span data-stu-id="21bc8-113">Users can then enable and disable installed plug-ins using the checkboxes provided on the **Plug-ins** tab. DSP plug-ins may implement a property page, which users can also access from the **Plug-ins** tab. A property page provides the user with a way to change settings that affect the way a DSP plug-in works.</span></span> <span data-ttu-id="21bc8-114">Por ejemplo, una página de propiedades podría incluir un control deslizante que permite al usuario cambiar el tiempo de caída de un efecto de reverberación.</span><span class="sxs-lookup"><span data-stu-id="21bc8-114">For example, a property page might include a slider control that allows the user to change the decay time of a reverb effect.</span></span>
