# <a name="multiple-display-monitors"></a>Varios monitores de pantalla

Varios monitores de pantalla son un conjunto de características relacionadas que permiten a las aplicaciones hacer uso de varios dispositivos de pantalla al mismo tiempo. Hay dos maneras de usar varios monitores: como un escritorio grande o como un número de pantallas independientes. Cuando se usa como un escritorio grande, los monitores crean más espacio de pantalla para las aplicaciones. Esto resulta útil siempre que necesite maximizar el área de trabajo en pantalla y es especialmente útil para la publicación en el escritorio, el desarrollo web o la edición de vídeo. Cuando se usa como pantallas independientes, los monitores se pueden usar para reproducir juegos, depurar aplicaciones de pantalla completa, enseñar o realizar presentaciones.
