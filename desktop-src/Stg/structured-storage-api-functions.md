# <a name="structured-storage-api-functions"></a>Funciones de la API de almacenamiento estructurado

Algunas de las funciones de la API para el almacenamiento estructurado son funciones auxiliares que realizan una secuencia de llamadas a otras funciones de API y métodos de interfaz. Estas funciones auxiliares se pueden usar como cortes cortos.

Otras funciones de API proporcionan acceso a la implementación de COM de almacenamiento estructurado, archivos compuestos.

Además, otro conjunto de funciones de API le permite convertir objetos OLE 1 en almacenamiento estructurado. Puede utilizar estas funciones para determinar si una clase de objeto es de OLE 1 y para convertir objetos entre los formatos de almacenamiento COM de OLE 1 y actuales.

Por último, existen funciones de API para convertir y emular otros objetos. Estas funciones proporcionan servicios para que una aplicación de servidor trabaje con los datos de otra aplicación. Los datos se pueden convertir al formato nativo de la aplicación de servidor leyendo los datos desde su formato original, pero escribiendo en el formato nativo de la aplicación de objeto. O bien, los datos pueden permanecer en su formato original mientras la aplicación de servidor Lee y escribe los datos en su formato original.

Para obtener más información, consulte la siguiente sección [modos de acceso de almacenamiento estructurado](structured-storage-access-modes.md).
