<dl> <dt>

<span id="opengl_network"></span><span id="OPENGL_NETWORK"></span>**Storage**
</dt> <dd>

Conexión entre dos o más equipos que permite a cada uno de ellos transferir datos hacia y desde los demás.

</dd> <dt>

<span id="opengl_nonconvex"></span><span id="OPENGL_NONCONVEX"></span>**no convexa**
</dt> <dd>

Estado de un polígono en el que una línea del plano del polígono forma una intersección con el polígono más de dos veces.

</dd> <dt>

<span id="opengl_normal"></span><span id="OPENGL_NORMAL"></span>**sin**
</dt> <dd>

Ecuación de plano de tres componentes que define la orientación angular, pero no la posición, de un plano o una superficie.

</dd> <dt>

<span id="opengl_normalize"></span><span id="OPENGL_NORMALIZE"></span>**normalizar**
</dt> <dd>

Para dividir cada uno de los componentes de un normal por la raíz cuadrada de la suma de sus cuadrados. Después, si se considera que el normal es un vector desde el origen hasta el punto (NX ', NY ', NX '), este vector tiene la longitud de unidad:

<dl> <dd>NX ' = NX/factor</dd> <dd>NY ' = NY/factor</dd> <dd>NZ ' = NZ/factor</dd> </dl> </dd> <dt>

<span id="opengl_normal_vector"></span><span id="OPENGL_NORMAL_VECTOR"></span>**Vector normal**
</dt> <dd>

Vea *normal*.

</dd> <dt>

<span id="opengl_nurbs"></span><span id="OPENGL_NURBS"></span>**NURBS**
</dt> <dd>

Spline B-no uniforme. Una forma habitual de especificar curvas y superficies paramétricas.

</dd> </dl>
