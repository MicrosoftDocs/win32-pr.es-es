# <a name="directxmath-library-4d-vector-geometric-functions"></a>Funciones geométricas vectoriales 4D de la biblioteca DirectXMath

Enumera las funciones geométricas vectoriales 4D.

| Tema                                                                                 | Descripción                                                                   |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**XMVector4AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector4anglebetweennormals)<br/>       | Calcule el ángulo de base entre dos vectores 4D normalizados.<br/>        |
| [**XMVector4AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4anglebetweennormalsest)<br/> | Calcula el ángulo radiáno entre dos vectores 4D normalizados.<br/>      |
| [**XMVector4AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector4anglebetweenvectors)<br/>       | Calcule el ángulo de base entre dos vectores 4D.<br/>                   |
| [**XMVector4ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector4clamplength)<br/>                       | Fija la longitud de un vector 4D a un intervalo determinado.<br/>                 |
| [**XMVector4ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector4clamplengthv)<br/>                     | Fija la longitud de un vector 4D a un intervalo determinado.<br/>                 |
| [**XMVector4Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector4cross)<br/>                                   | Calcula el producto cruzado 4D.<br/>                                     |
| [**XMVector4Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector4dot)<br/>                                       | Calcula el producto de punto entre vectores 4D.<br/>                       |
| [**XMVector4InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector4inbounds)<br/>                             | Comprueba si los componentes de un vector 4D están dentro de los límites establecidos.<br/> |
| [**XMVector4Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector4length)<br/>                                 | Calcula la longitud de un vector 4D.<br/>                                |
| [**XMVector4LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lengthest)<br/>                           | Calcula la longitud de un vector 4D.<br/>                               |
| [**XMVector4LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lengthsq)<br/>                             | Calcula el cuadrado de la longitud de un vector 4D.<br/>                  |
| [**XMVector4Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector4normalize)<br/>                           | Devuelve la versión normalizada de un vector 4D.<br/>                     |
| [**XMVector4NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4normalizeest)<br/>                     | Calcula la versión normalizada de un vector 4D.<br/>                   |

| [**XMVector4ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4reciprocallengthest)<br/>       | Calcula el recíproco de la longitud de un vector 4D.<br/>             |
| [**XMVector4Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector4reflect)<br/>                               | Refleja un vector 4D de incidente en un vector normal 4D.<br/>          |
