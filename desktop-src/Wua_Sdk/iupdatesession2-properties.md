# <a name="iupdatesession2-properties"></a>Propiedades de IUpdateSession2

La [**interfaz IUpdateSession2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession2) define la siguiente propiedad.



| Propiedad                                         | Descripción                                                                   |
|--------------------------------------------------|-------------------------------------------------------------------------------|
| [**UserLocale**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesession2-get_userlocale) | Obtiene y establece la configuración regional preferida para la que se recupera la información de actualización. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz.

-   [**IUpdateSession**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesession)
