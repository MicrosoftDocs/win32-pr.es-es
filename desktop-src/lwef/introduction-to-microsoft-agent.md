# <a name="introduction-to-microsoft-agent"></a>Introducción a Microsoft Agent

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Microsoft Agent es un conjunto de servicios de software programables que admite la presentación de caracteres animados interactivos. Los desarrolladores pueden usar caracteres como asistentes interactivos para introducir, guiar, entretener o mejorar sus páginas web o aplicaciones, además del uso convencional de ventanas, menús y controles.

Microsoft Agent permite a los desarrolladores de software y a los autores web incorporar una nueva forma de interacción del usuario, conocida como interfaces conversacionales, que aprovecha aspectos naturales de la comunicación social humana. Además de la entrada del mouse y el teclado, Microsoft Agent incluye compatibilidad opcional con el reconocimiento de voz para que las aplicaciones puedan responder a comandos de voz. Los caracteres pueden responder mediante voz sintetizada, audio grabado o texto en un globo de palabras de animación.

El enfoque de interfaz conversacional facilitado por los servicios de Microsoft Agent no reemplaza el diseño de la interfaz gráfica de usuario (GUI) convencional. En su lugar, la interacción de caracteres se puede combinar fácilmente con los componentes de interfaz convencionales, como ventanas, menús y controles, para ampliar y mejorar la interfaz de la aplicación.

Las interfaces de programación de Microsoft Agent hacen que sea fácil animar un carácter para responder a la entrada del usuario. Los personajes animados aparecen en su propia ventana, lo que proporciona la máxima flexibilidad para ver dónde se pueden mostrar en la pantalla. Microsoft Agent incluye un control ActiveX que hace que sus servicios se puedan acceder a los lenguajes de programación que admiten ActiveX, incluidos los lenguajes de scripting web, como Visual Basic Scripting Edition (VBScript). Esto significa que la interacción de caracteres se puede programar incluso desde páginas HTML mediante la &lt; etiqueta &gt; OBJECT.
