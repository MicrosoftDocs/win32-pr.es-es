| CN                | CA-Connect                                  |
| Nombre para mostrar de LDAP | cAConnect                                   |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.687                      |
| System-ID-GUID    | 963d2735-48be-11d1-a9c3-0000f80367c1        |
