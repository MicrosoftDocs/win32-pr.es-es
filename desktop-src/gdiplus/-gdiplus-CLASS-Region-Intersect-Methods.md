# <a name="regionintersect-methods"></a>Region. Intersect (métodos)

En este tema se enumeran los métodos Intersect de la clase [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Para obtener una lista completa de los métodos de la clase **Region** , consulte [métodos de región](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                     | Descripción                                                                                                                                                                                     |
|:---------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Intersect (rectángulo&)**](/previous-versions//ms534804(v=vs.85))     | El método [**Region:: Intersect**](/previous-versions//ms534804(v=vs.85)) actualiza esta región a la parte de sí mismo que forma una intersección con el interior del rectángulo especificado.<br/>  |
| [**Intersect (RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstrectf_))   | El método [**Region:: Intersect**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstrectf_)) actualiza esta región a la parte de sí mismo que forma una intersección con el interior del rectángulo especificado.<br/> |
| [**Intersect (región \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstregion))     | El método [**Region:: Intersect**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstregion)) actualiza esta región a la parte de sí misma que forma una intersección con otra región.<br/>                         |
| [**Intersect (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstgraphicspath)) | El método [**Region:: Intersect**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-intersect(inconstgraphicspath)) actualiza esta región a la parte de sí mismo que forma una intersección con el interior de la ruta de acceso especificada.<br/>            |
