# <a name="com-clsid-attribute"></a>Atributo COM-CLSID

Este atributo contiene el GUID que está asociado a esta clase de objeto.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | COM-CLSID                                   |
| Nombre para mostrar de LDAP | comclsid                                    |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.249                      |
| System-ID-GUID    | 281416d9-1968-11d0-a28f-00aa003049e2        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
