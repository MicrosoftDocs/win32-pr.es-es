# <a name="media-streaming-structures"></a>Estructuras de streaming multimedia

La [API de streaming de multimedia](media-streaming-api-portal.md) proporciona las siguientes estructuras.

## <a name="in-this-section"></a>En esta sección



| Tema                                                           | Descripción                                                                                                                                        |
|-----------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PlaySpeed**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-playspeed)<br/>                       | Representa una velocidad de reproducción como un número racional. <br/>                                                                                      |
| [**PositionInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-positioninformation)<br/>   | Contiene los valores actuales de la información de posición de reproducción multimedia obtenida del DMR.<br/>                                               |
| [**RenderingParameters**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-renderingparameters)<br/>   | Contiene los valores actuales de los parámetros de representación en el DMR.<br/>                                                                         |
| [**TrackInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-trackinformation)<br/>         | Contiene el número de pista actual y la duración como parte de la [**PositionInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-positioninformation) obtenida a partir del DMR.<br/> |
| [**TransportInformation**](/previous-versions/windows/desktop/api/windows.media.streaming/ns-windows-media-streaming-transportinformation)<br/> | Contiene los valores actuales de la información relacionada con el transporte de reproducción multimedia obtenida del DMR.<br/>                                      |
