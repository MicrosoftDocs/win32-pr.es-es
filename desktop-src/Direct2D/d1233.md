# <a name="d1233-begindraw-invalid-from-previous-error"></a>D1233: BeginDraw no válido a partir del error anterior

Se llamó a BeginDraw, pero se omitirá porque el contexto del dispositivo está en un estado no válido.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación llamó a BeginDraw, pero el contexto del dispositivo estaba en un estado no válido de una operación anterior.

## <a name="possible-fixes"></a>Posibles correcciones

Asegúrese de que la aplicación responde correctamente a los errores devueltos por Flush o EndDraw, y vuelva a crear su dispositivo según sea necesario. Si un comando hizo que un contexto de dispositivo se encontrara en un estado no válido, el flush o EndDraw correspondiente devolverá un error.
