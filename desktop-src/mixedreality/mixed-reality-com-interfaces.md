# <a name="windows-mixed-reality-com-interfaces"></a>Interfaces COM de realidad mixta de Windows

Los temas contenidos en esta sección proporcionan las especificaciones de referencia para las interfaces COM de realidad mixta de Windows.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|-|-|
| [**ISpatialInteractionManagerInterop**](/windows/desktop/api/spatialinteractionmanagerinterop/nn-spatialinteractionmanagerinterop-ispatialinteractionmanagerinterop) | Habilita la interoperabilidad con un objeto [SpatialInteractionManager](/uwp/api/Windows.UI.Input.Spatial.SpatialInteractionManager) de plataforma universal de Windows (UWP) y proporciona acceso a los miembros de **SpatialInteractionManager** para tener acceso a los datos proporcionados por el usuario desde manos, controladores de movimiento y comandos de voz del sistema. |
| [**IHolographicSpaceInterop**](/windows/desktop/api/holographicspaceinterop/nn-holographicspaceinterop-iholographicspaceinterop) | Habilita la interoperabilidad con un objeto [HolographicSpace](/uwp/api/windows.graphics.holographic.holographicspace) de plataforma universal de Windows (UWP) y proporciona acceso a los miembros de **HolographicSpace** para representar una escena holográfica. |
