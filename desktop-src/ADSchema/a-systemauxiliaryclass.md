# <a name="system-auxiliary-class-attribute"></a>System-auxiliar-Class (atributo)

Una lista de las clases auxiliares que el usuario no puede modificar.



| Entrada | Value |
|-------------------|--------------------------------------------------------------------|
| CN                | System-auxiliar-clase                                             |
| Nombre para mostrar de LDAP | systemAuxiliaryClass                                               |
| Tamaño              | \-                                                                 |
| Actualizar privilegio  | Administrador de esquema                                               |
| Frecuencia de actualización  | Cuando se crea la clase o se le agrega una nueva clase auxiliar. |
| Attribute-Id      | 1.2.840.113556.1.4.198                                             |
| System-ID-GUID    | bf967a43-0de6-11d0-a285-00aa003049e2                               |
| Sintaxis            | [**String(Object-Identifier)**](s-string-object-identifier.md)    |
