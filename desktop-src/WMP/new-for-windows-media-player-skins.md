# <a name="new-for-windows-media-player-skins"></a>Novedades de las máscaras de Media Player de Windows

Windows Media Player 7,0 presentó la tecnología de máscara. Los elementos y atributos disponibles permanecen inalterados en Windows Media Player 7,1 y Windows Media Player para Windows XP. Windows Media Player 9 series incorporó nuevos elementos y atributos que puede usar. Windows Media Player 10 introduce un nuevo atributo.

Cada tema de la sección referencia de programación de la máscara incluye una sección de requisitos que enumera el requisito mínimo para el elemento, el método, el controlador de eventos o el atributo individual. En la siguiente lista también se detallan los nuevos elementos para versiones concretas de Windows Media Player.
