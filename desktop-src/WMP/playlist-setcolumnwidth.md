# <a name="playlistsetcolumnwidth"></a>Lista de reproducción. setColumnWidth

El método **setColumnWidth** especifica el ancho de columna y cambia el modo de cambio de tamaño de la columna a "Fixed".

``` syntax
        elementID.setColumnWidth(column, width)
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

<span id="column"></span><span id="COLUMN"></span>*artículo*
</dt> <dd>

**Número** (**largo**) que indica el índice de la columna que se va a cambiar.

</dd> <dt>

<span id="width"></span><span id="WIDTH"></span>*Ancho*
</dt> <dd>

**Número** (**largo**) que indica el nuevo ancho en píxeles.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
