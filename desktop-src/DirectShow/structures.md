# <a name="directshow-structures"></a>Estructuras de DirectShow

> [!Note]  
> \[En desuso. Esta API se puede quitar de las versiones futuras de Windows.\]

 

Los servicios de edición de DirectShow definen las siguientes estructuras.



| Estructura                             | Descripción                                          |
|---------------------------------------|------------------------------------------------------|
| [**parámetro DEXTERity \_**](dexter-param.md) | Especifica una propiedad en una transición o efecto.      |
| [**valor de DEXTERity \_**](dexter-value.md) | Especifica el valor de una propiedad.                   |
| [**SCompFmt0**](scompfmt0.md)        | Contiene información de formato para la recompresión inteligente. |
