# <a name="ipx-service-table-management"></a>Administración de tablas de servicio IPX

Un protocolo de enrutamiento IPX que se registra para el control de servicios mantiene una tabla de enlace de servicios tal y como se define en las especificaciones del Protocolo de publicidad de servicios IPX (SAP) de Ldap. El protocolo de enrutamiento también proporciona al administrador de enrutadores acceso a esta tabla a través de las siguientes funciones y estructuras.

-   [Funciones de administración de tablas de servicio](service-table-management-functions.md)
-   [Estructuras de administración de tablas de servicio](service-table-management-structures.md)

Consulte la [**función RegisterProtocol**](/windows/desktop/api/Routprot/nc-routprot-pregister_protocol) para obtener más información sobre cómo se registra un protocolo de enrutamiento para el control de servicios.
