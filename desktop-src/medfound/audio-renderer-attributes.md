# <a name="audio-renderer-attributes"></a>Atributos del representador de audio

Los siguientes atributos se pueden usar para configurar el representador de audio.



| Atributo                                                                                                     | Descripción                                                                                                 |
|---------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**\_ \_ \_ \_ ID. de extremo de atributo de representador de audio MF \_**](mf-audio-renderer-attribute-endpoint-id-attribute.md)     | Especifica el identificador del dispositivo de extremo de audio.                                                     |
| [**\_función de \_ extremo de \_ atributo \_ de representador de audio \_ MF**](mf-audio-renderer-attribute-endpoint-role-attribute.md) | Especifica el rol de punto de conexión de audio para el representador de audio.                                                   |
| [**\_etiquetas de \_ atributo de representador de audio MF \_ \_**](mf-audio-renderer-attribute-flags-attribute.md)                  | Contiene marcas para configurar el representador de audio.                                                             |
| [**\_ \_ \_ \_ ID. de sesión de atributo de representador de audio MF \_**](mf-audio-renderer-attribute-session-id-attribute.md)       | Especifica la clase de directiva de audio para el representador de audio.                                                    |
| [\_categoría de \_ flujo de \_ atributo \_ de representador de audio \_ MF](mf-audio-renderer-attribute-stream-category.md)           | Especifica la categoría de secuencia de audio para el [representador de audio de streaming](streaming-audio-renderer.md) (SAR). |
