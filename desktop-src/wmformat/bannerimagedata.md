# <a name="bannerimagedata"></a>BannerImageData

El atributo **BannerImageData** contiene una imagen de banner para la personalización de marca. El tipo de imagen que contiene este atributo se describe mediante el atributo [**BannerImageType**](bannerimagetype.md) .

## <a name="global-constant"></a>Constante global

g \_ wszWMBannerImageData

## <a name="data-type"></a>Tipo de datos

**tipo de WMT \_ \_ binario**

## <a name="remarks"></a>Observaciones

La imagen almacenada en este atributo puede estar en formato de mapa de bits, JPEG o GIF.
