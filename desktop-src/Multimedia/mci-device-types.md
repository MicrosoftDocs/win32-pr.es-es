# <a name="mci-device-types"></a>Tipos de dispositivos MCI

Los valores siguientes identifican los dispositivos de los mensajes y estructuras de MCI:



| Value                         | Significado                       |
|-------------------------------|-------------------------------|
| \_ \_ ID. de dispositivo MCI \_          | Cualquier dispositivo                    |
| \_animación DEVTYPE de MCI \_       | Animación: dispositivo de reproducción     |
| AUDIO de CD de MCI \_ DEVTYPE \_ \_       | Dispositivo de audio de CD               |
| \_ \_ archivos DAT de MCI DEVTYPE             | Dispositivo de cinta de audio digital     |
| \_ \_ vídeo digital de MCI DEVTYPE \_  | Dispositivo de reproducción de vídeo digital |
| MCI \_ DEVTYPE \_ otro           | Dispositivo no definido              |
| superposición de MCI \_ DEVTYPE \_         | Dispositivo superpuesto de vídeo          |
| \_escáner MCI DEVTYPE \_         | Dispositivo de escáner                |
| \_ \_ secuenciador MCI DEVTYPE       | Dispositivo MIDI Sequencer         |
| \_VCR DEVTYPE de MCI \_             | Grabadora de casete de vídeo       |
| videodisco de MCI \_ DEVTYPE \_       | Reproductor de videodisco              |
| \_audio de \_ onda \_ DEVTYPE de MCI | Onda: dispositivo de audio         |
