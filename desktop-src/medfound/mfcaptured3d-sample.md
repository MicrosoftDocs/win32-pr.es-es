# <a name="mfcaptured3d-sample"></a>Ejemplo MFCaptureD3D

Muestra cómo obtener una vista previa del vídeo desde una cámara de vídeo mediante Direct3D para representar el vídeo.

## <a name="apis-demonstrated"></a>API demostradas

En este ejemplo se muestran las siguientes API.

-   [**IMFMediaSource**](/windows/desktop/api/mfidl/nn-mfidl-imfmediasource)
-   [**IMFSourceReader**](/windows/desktop/api/mfreadwrite/nn-mfreadwrite-imfsourcereader)
-   [**MFEnumDeviceSources**](/windows/desktop/api/mfidl/nf-mfidl-mfenumdevicesources)

## <a name="requirements"></a>Requisitos



| Producto                                                        | Versión   |
|----------------------------------------------------------------|-----------|
| [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx) | Windows 7 |
