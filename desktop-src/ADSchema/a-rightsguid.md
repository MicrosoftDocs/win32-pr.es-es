# <a name="rights-guid-attribute"></a>Rights-Guid atributo)

GUID que se usa para representar un derecho extendido dentro de una entrada de control de acceso.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Rights-Guid                                 |
| Nombre para mostrar de LDAP | rightsGuid                                  |
| Tamaño              | 16 bytes                                    |
| Actualizar privilegio  | El sistema establece este valor.            |
| Frecuencia de actualización  | Cuando se crea un derecho extendido nuevo.       |
| Attribute-Id      | 1.2.840.113556.1.4.340                      |
| System-ID-GUID    | 8297931c-86d3-11d0-afda-00c04fd930c9        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
