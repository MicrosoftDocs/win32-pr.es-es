# <a name="ezrgb24-filter-sample"></a>Ejemplo de filtro EZRGB24

## <a name="description"></a>Descripción

El filtro EZRGB24 es un filtro de transformación de procesamiento de imágenes. Este filtro realiza una serie de efectos de vídeo individuales, incluidos los efectos rojo, verde, azul, oscuro, XOR, Desenfoque, gris y relieve.

En este ejemplo se muestra cómo usar varias clases base de DirectShow, como [**CTransformFilter**](ctransformfilter.md), [**CPersistStream**](cpersiststream.md)y [**CBasePropertyPage**](cbasepropertypage.md).

## <a name="downloading-the-sample"></a>Descargar el ejemplo

Para descargar los ejemplos del SDK de DirectShow, instale la versión más reciente de la [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Este ejemplo se instala en la siguiente ruta de acceso: ejemplos *\[ raíz \] del SDK* \\ \\ filtros DirectShow de multimedia \\ \\ \\ EZRGB24.
