# <a name="d1171-redundant-call-to-map-bitmap"></a>D1171: llamada redundante para asignar mapa de bits

Se intentó asignar un mapa de bits que ya está asignado.






 

## <a name="possible-causes"></a>Causas posibles

Direct2D puede generar este mensaje e informar de un error si una aplicación intenta asignar un mapa de bits que ya estaba asignado mediante ID2D1Bitmap1:: Map.

## <a name="possible-fixes"></a>Posibles correcciones

Del mismo modo que ID2D1RenderTarget:: EndDraw debe seguir las llamadas a ID2D1RenderTarget:: BeginDraw, asegúrese de que ID2D1Bitmap1:: desasignar sigue las llamadas a ID2D1Bitmap1:: Map. Si una aplicación usa las devoluciones de llamada o rutas de código recursivas para asignar mapas de bits, asegúrese de que su comportamiento es correcto.
