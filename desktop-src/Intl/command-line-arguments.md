# <a name="command-line-arguments"></a>Argumentos de la línea de comandos

Una aplicación puede utilizar la función de Win32 [GetCommandLine](/windows/win32/api/processenv/nf-processenv-getcommandlinea) para recuperar argumentos de la línea de comandos de Unicode. La aplicación debe llamar a la función de Win32 como una función Unicode.
