# <a name="assembling-primitives"></a>Ensamblar primitivos

Una vez realizados todos los cálculos necesarios, los vértices se ensamblan en puntos [primitivos, segmentos](primitives.md) de línea o polígonos junto con la información pertinente de la marca de borde, el color y la textura de cada vértice.
