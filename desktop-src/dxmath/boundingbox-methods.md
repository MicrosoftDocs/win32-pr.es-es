# <a name="boundingbox-methods"></a>Métodos BoundingBox



| Método                                                              | Descripción                                                                                            |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**Contiene**](boundingbox-contains.md)<br/>                 | Comprueba si boundingBox contiene un objeto especificado.<br/>                                  |
| [**CreateFromPoints**](boundingbox-createfrompoints.md)<br/> | Crea un BoundingBox a partir de puntos.<br/>                                                          |
| [**Intersecta**](boundingbox-intersects.md)<br/>             | Comprueba la intersección de BoundingBox con otro objeto .<br/>                                 |
| [**Transform**](boundingbox-transform.md)<br/>               | Transforma boundingBox.<br/>                                                                 |
| [**ContainedBy**](/windows/desktop/api/DirectXCollision/nf-directxcollision-boundingbox-containedby)<br/>           | Comprueba si [**boundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) está contenido en el frustum especificado.<br/> |
| [**CreateFromSphere**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createfromsphere)<br/> | Crea un Objeto BoundingBox lo suficientemente grande como para contener el objeto BoundingSphere especificado.<br/>               |
| [**CreateMerged**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-createmerged)<br/>         | Crea un Objeto BoundingBox lo suficientemente grande como para que contenga dos intances BoundBox especificados.<br/>             |
| [**GetCorners**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-getcorners)<br/>             | Recupera las esquinas de BoundingBox.<br/>                                                   |
| [**Op \_ Assignment**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-operator-assign)<br/>      | Copia valores de otro BoundingBox.<br/>                                                     |
