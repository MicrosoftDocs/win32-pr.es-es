# <a name="boundingboxcontains-methods"></a>BoundingBox. Contains (métodos)

Comprueba si BoundingBox contiene un objeto especificado.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                               | Descripción                                                                                                                                |
|:-------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------|
| [**BoundingBox:: Contains (XMVECTOR)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains)                   | Comprueba si el BoundingBox contiene un punto especificado.<br/>                                                                   |
| [**BoundingBox:: Contains (const BoundingBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(constboundingbox_))         | Comprueba si el BoundingBox contiene otro BoundingBox.<br/>                                                                     |
| [**BoundingBox:: Contains (const BoundingSphere&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(constboundingsphere_))      | Comprueba si BoundingBox contiene un BoundingSphere especificado.<br/>                                                              |
| [**BoundingBox:: Contains (const BoundingFrustum&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(constboundingfrustum_))     | Comprueba si [**BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) contiene el [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum)especificado.<br/>         |
| [**BoundingBox:: Contains (XMVECTOR, XMVECTOR, XMVECTOR)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(fxmvector_fxmvector_fxmvector)) | Comprueba si el BoundingBox contiene un triángulo especificado.<br/>                                                                     |
| [**BoundingBox:: Contains (const BoundingOrientedBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-contains(constboundingorientedbox_)) | Comprueba si [**BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) contiene el [**BoundingOrientedBox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox)especificado.<br/> |
