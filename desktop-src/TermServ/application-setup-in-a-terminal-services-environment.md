<span data-ttu-id="a4c70-109">Para evitar estos problemas, los procedimientos de configuración deben usar las siguientes directrices al instalar una aplicación en un servidor host de sesión de Escritorio remoto (host de sesión de escritorio remoto):</span><span class="sxs-lookup"><span data-stu-id="a4c70-109">To avoid these problems, setup procedures should use the following guidelines when installing an application on a Remote Desktop Session Host (RD Session Host) server:</span></span>
