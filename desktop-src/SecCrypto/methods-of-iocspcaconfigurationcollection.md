# <a name="methods-of-iocspcaconfigurationcollection"></a>Métodos de IOCSPCAConfigurationCollection

La interfaz [**IOCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) define los siguientes métodos. Los métodos de acceso de propiedad no se muestran aquí. Para ver las propiedades de **IOCSPCAConfigurationCollection**, consulte [propiedades de IOCSPCAConfigurationCollection](properties-of-iocspcaconfigurationcollection.md)



| Método                                                                                       | Descripción                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-createcaconfiguration) | Crea una nueva configuración de [*entidad de certificación*](../secgloss/c-gly.md) (CA) y la agrega al conjunto de configuración. |
| [**DeleteCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-deletecaconfiguration) | Quita una configuración de entidad de certificación con nombre del conjunto de configuración.                                                                                                                                   |
