# <a name="show-cachestate"></a>show cachestate

Enumera todos los recursos y sus propiedades asociadas que se almacenan en caché en la caché de respuesta HTTP o muestra un único recurso y sus propiedades asociadas.

``` syntax
show cachestate [[url=]string]
 
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

<span id="__url__string_"></span><span id="__URL__STRING_"></span>**\[\[URL = \] ***cadena***\]**
</dt> <dd>

Dirección URL completa. Si no se especifica, implica todas las direcciones URL. La dirección URL también puede ser un prefijo para las direcciones URL registradas.

</dd> </dl>

## <a name="examples"></a>Ejemplos

**Mostrar dirección URL de cachestate =https://www.contoso.com:80/myresource**
