# <a name="iamtimelineobjsetusername-method"></a>IamTimelineObj::SetUserName (método)

> [!Note]  
> \[Obsoleto. Esta API puede quitarse de futuras versiones de Windows.\]

 

El `SetUserName` método establece un nombre definido por la aplicación para el objeto .

## <a name="syntax"></a>Sintaxis


```C++
HRESULT SetUserName(
   BSTR newVal
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*newVal* 
</dt> <dd>

Cadena de caracteres anchos que contiene el nombre. Las cadenas de más de 256 caracteres podrían truncarse.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Si este método se realiza correctamente, devuelve **S \_ OK**. De lo contrario, devuelve un código de error **HRESULT.**
