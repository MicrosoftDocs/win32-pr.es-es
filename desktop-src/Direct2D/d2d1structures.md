# <a name="direct2d-helper-structures"></a>Estructuras auxiliares de Direct2D

El espacio de nombres D2D1 define las siguientes estructuras.



|                                                 |                                                                                                               |
|-------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| Nombre                                            | Descripción                                                                                                   |
| [**TypeTraits<Type>**](/windows/win32/api/d2d1_1helper/ns-d2d1_1helper-typetraits) | Contiene implementaciones de **Point**, **size** y **Rect** que almacenan sus datos utilizando el tipo especificado. |
