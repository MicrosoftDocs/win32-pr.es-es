# <a name="what-is-compatibility-view"></a>¿Qué es Vista de compatibilidad?

*Vista de compatibilidad* es una característica de Windows Internet Explorer 8 que permite al explorador representar una página web casi idéntica a la forma en que Windows Internet Explorer 7 la representaría.

En Internet Explorer 8, Vista de compatibilidad cambia la forma en que el explorador interpreta el código escrito en CSS, HTML y el Document Object Model (DOM) para intentar coincidir con Internet Explorer 7. Un sitio que un usuario ve en Internet Explorer 8 Vista de compatibilidad es casi idéntico a un sitio que el usuario ve en Internet Explorer 7. Sin embargo, Vista de compatibilidad no cambia la forma en que el explorador interpreta todo el código. Por ejemplo, los cambios en Internet Explorer 8 para el modo en que el explorador controla ActiveX, el analizador, AJAX, JavaScript, redes y seguridad podrían seguir causando problemas de compatibilidad. Vista de compatibilidad no cambia estos comportamientos.

En un entorno empresarial, algunas áreas tienen un riesgo menor de problemas de compatibilidad. Por ejemplo, los sitios web de la zona de intranet utilizan la Vista de compatibilidad de forma predeterminada. Las aplicaciones web cliente que se representan mediante el Control de explorador web o webOC (motor de representación de Internet Explorer) también tienen un riesgo bajo de problemas de compatibilidad porque Internet Explorer 8 tiene como valor predeterminado un modo de compatibilidad para WebOC. Sin embargo, es posible que los valores de configuración predeterminados Vista de compatibilidad no garanticen la compatibilidad completa. Para determinar si un sitio web o aplicación web es compatible con Internet Explorer 8, debe probar el sitio web o la aplicación web.

Para obtener más información sobre las diferencias entre Internet Explorer 8 Vista de compatibilidad y Internet Explorer 7, vea el blog Compatibilidad del sitio [Internet Explorer 8](/archive/blogs/ie/site-compatibility-and-ie8). Para obtener una lista de lo que debe comprobar al actualizar a Internet Explorer 8, vea el Internet Explorer [8 Readiness Toolkit](https://www.microsoft.com/windows/internet-explorer/readiness/developers.aspx).

Para obtener más información sobre Vista de compatibilidad, vea el [blog Internet Explorer Team .](/archive/blogs/ie/)
