# <a name="ws_group-and-ws_tabstop-flags-in-controls"></a>Marcas \_ TABSTOP de WS GROUP y \_ WS en controles

Un control no debe usar internamente las marcas WS GROUP y \_ WS TABSTOP; algunos contenedores dependen de estas marcas \_ para administrar el control de teclado.
