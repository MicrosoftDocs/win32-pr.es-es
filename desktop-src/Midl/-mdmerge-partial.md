# <a name="partial-switch"></a>/partial switch

El **modificador /partial** especifica que los tipos sin resolver se resuelven en los archivos de metadatos enumerados por el [**modificador /metadata \_ dir.**](-mdmerge-metadata-dir.md)

``` syntax
mdmerge /partial 
```

## <a name="switch-options"></a>Cambiar opciones

Este modificador no tiene parámetros.

## <a name="remarks"></a>Observaciones

## <a name="examples"></a>Ejemplos

**mdmerge /partial**
