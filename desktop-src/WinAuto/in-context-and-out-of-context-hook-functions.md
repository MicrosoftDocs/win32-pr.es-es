# <a name="in-context-and-out-of-context-hook-functions"></a>Funciones de enlace de In-Context y fuera de contexto

Cuando se registra una función de enlace con [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook), los clientes especifican si la función de enlace es *en contexto* o *fuera de contexto*. Estos términos describen la ubicación de memoria de la función de enlace en relación con el espacio de direcciones del servidor.
