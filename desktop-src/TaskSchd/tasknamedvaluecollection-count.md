# <a name="tasknamedvaluecollectioncount-property"></a>TaskNamedValueCollection. Count (propiedad)

En el caso de scripting, obtiene el número de pares nombre-valor de la colección.

## <a name="syntax"></a>Sintaxis


```VB
TaskNamedValueCollection.Count As long
```



## <a name="property-value"></a>Valor de propiedad

El número de pares nombre-valor de la colección.
