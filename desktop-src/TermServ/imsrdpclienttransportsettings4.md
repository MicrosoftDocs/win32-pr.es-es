# <a name="imsrdpclienttransportsettings4-interface"></a>Interfaz IMsRdpClientTransportSettings4

Define propiedades adicionales para el servidor de puerta de enlace de Escritorio remoto (puerta de enlace de escritorio remoto).

## <a name="members"></a>Miembros

La interfaz **IMsRdpClientTransportSettings4** hereda de [**IMsRdpClientTransportSettings3**](imsrdpclienttransportsettings3.md). **IMsRdpClientTransportSettings4** también tiene estos tipos de miembros:

-   [Propiedades](#properties)

### <a name="properties"></a>Propiedades

La interfaz **IMsRdpClientTransportSettings4** tiene estas propiedades.



| Propiedad                                                                                       | Tipo de acceso           | Descripción                                          |
|:-----------------------------------------------------------------------------------------------|:----------------------|:-----------------------------------------------------|
| [**GatewayBrokeringType**](imsrdpclienttransportsettings4-gatewaybrokeringtype.md)<br/> | Solo escritura<br/> | Obtiene y establece el tipo de agente de puerta de enlace.<br/> |
