# <a name="obtaining-a-root-oid-from-an-iso-name-registration-authority"></a>Obtención de un OID raíz a partir de una entidad de registro de nombres ISO

La mejor manera de obtener un identificador de objeto raíz (OID) es solicitar uno de una autoridad de registro de nombres ISO (International Standards Organization). Se trata de una acción única; Cuando se obtiene un OID raíz, el espacio OID que define es suyo y se puede administrar por sí mismo.

Normalmente, hay una cuota asociada al registro de un nombre de organización y la recepción de un OID raíz. Las tarifas y las directivas de registro varían según el país o la región. En el Estados Unidos, el NRA ISO es el American National Standards Institute (ANSI). Para obtener más información acerca del procedimiento de registro ANSI y la programación de tarifas, vea https://web.ansi.org . La ISO mantiene una lista de organizaciones miembro en https://www.iso.ch . Si está fuera del Estados Unidos, debe ponerse en contacto con la organización miembro ISO de su país o región para obtener información de registro de nombres.

Independientemente del origen que se utilice para obtener el OID, si tiene previsto extender el esquema de Active Directory y desea solicitar el logotipo certificado para Windows, debe registrar el OID en Microsoft. Para obtener más información acerca de cómo registrar el OID en Microsoft, consulte [obtener un identificador de objeto de Microsoft](obtaining-an-object-identifier-from-microsoft.md).
