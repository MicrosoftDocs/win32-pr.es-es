# <a name="windowsbuild-property"></a>Propiedad de WindowsBuild

El instalador establece la **propiedad WindowsBuild** en el número de compilación del sistema operativo. Para obtener más información, vea [Valores de propiedad del sistema operativo](operating-system-property-values.md).

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versión<br/> | Windows Instalador 5.0 en Windows Server 2012, Windows 8, Windows Server 2008 R2 o Windows 7. Windows Instalador 4.0 o Windows Installer 4.5 en Windows Server 2008 o Windows Vista. Windows Instalador en Windows Server 2003 o Windows XP Consulte los requisitos de Run-Time del instalador de [Windows](windows-installer-portal.md) para obtener información sobre el Service Pack de Windows mínimo que requiere una versión de Windows Installer.<br/> |
