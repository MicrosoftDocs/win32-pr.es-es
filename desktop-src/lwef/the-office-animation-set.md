# <a name="the-office-animation-set"></a>Conjunto de animaciones de Office

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

En la tabla siguiente se enumeran las animaciones definidas para el Microsoft Office caracteres 2000. Si piensa usar el carácter en Microsoft Office, debe admitir todas las animaciones de esta tabla. Además, puede agregar cualquier otra animación que desee, pero tenga en cuenta que Microsoft Office no las llamará. Las animaciones con asteriscos ( \* ) deben tener un bucle del 100%. Otras animaciones deben ser breves.
