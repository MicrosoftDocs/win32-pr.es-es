# <a name="instanceparametersversionstoretaskqueuemax-property"></a>Propiedad InstanceParameters.VersionStoreTaskQueueMax

Obtiene o establece el número de elementos de trabajo de limpieza en segundo plano que se pueden poner en cola en el grupo de subprocesos del motor de base de datos en cualquier momento.

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop](./microsoft.isam.esent.interop-namespace.md)
