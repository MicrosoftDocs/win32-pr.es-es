# <a name="mf_capture_engine_mediasource_config-attribute"></a>MF \_ ( \_ atributo de configuración de MEDIASOURCE del motor de captura) \_ \_

Contiene las propiedades de configuración para el origen de captura.

## <a name="data-type"></a>Tipo de datos

**IPropertyStore \* *_ se almacena como _* IUnknown\***

## <a name="remarks"></a>Observaciones

Actualmente, este atributo se omite. En el futuro, podría usarse para pasar propiedades de configuración al origen de medios.
