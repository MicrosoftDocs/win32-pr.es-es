# <a name="inheritance-ad-ds"></a>Herencia (AD DS)

La [*herencia*](/previous-versions/windows/desktop/legacy/ms681910(v=vs.85)) permite que una ACE determinada se propague desde el contenedor en el que se aplicó a todos los elementos secundarios del contenedor. La herencia se puede combinar con la delegación para conceder derechos administrativos a un subárbol completo del directorio en una sola operación.
