# <a name="directxmath-programming-guide"></a>Guía de programación de DirectXMath

DirectXMath proporciona una solución matemática optimizada para Windows.

## <a name="in-this-section"></a>En esta sección



| Tema                                                             | Descripción                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Introducción](pg-xnamath-getting-started.md)<br/>      | La biblioteca DirectXMath implementa una interfaz óptima y portátil para operaciones de álgebra aritméticas y lineales en vectores de punto flotante de precisión sencilla (2D, 3D y 4D) o matrices (3×3 y 4×4). <br/>                                                    |
| [Novedades](pg-xnamath-whatsnew.md)<br/>                  | La biblioteca DirectXMath se basa en la versión [2.04](https://walbourn.github.io/)de la biblioteca SIMD de C++ matemática de XNA. Aquí se describe cómo se diferencia DirectXMath de XNA Math y cómo difieren las versiones de DirectXMath. <br/> |
| [Migración de código](pg-xnamath-migration.md)<br/>             | En esta introducción se describen los cambios necesarios para migrar el código existente mediante la biblioteca matemática de XNA a la biblioteca DirectXMath.<br/>                                                                                                                                 |
| [Trabajo con D3DXMath](pg-xnamath-migration-d3dx.md)<br/> | D3DXMath es una biblioteca auxiliar matemática para aplicaciones Direct3D. <br/>                                                                                                                                                                                                |
| [Optimización de código](pg-xnamath-optimizing.md)<br/>         | En este tema se describen las consideraciones y estrategias de optimización con la biblioteca DirectXMath.<br/>                                                                                                                                                                |
| [Biblioteca interna](pg-xnamath-internals.md)<br/>          | En este tema se describe el diseño interno de la biblioteca DirectXMath.<br/>                                                                                                                                                                                         |
