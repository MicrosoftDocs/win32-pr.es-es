# <a name="packcpp"></a>Pack. Cpp

En el componente de proveedor de ejemplo, se encuentra un ejemplo de código de empaquetado y desempaquete de tipos de datos en variants en pack.cpp. Las funciones siguientes se usan cuando las propiedades se cargan en la memoria caché de propiedades y se quitan de ella:

-   **PackStringinVariant**
-   **DesempaquetarStringinVariant**
-   **PackLONGinVariant**
-   **DesempaquetarLONGfromVariant**
-   **PackDATEinVariant**
-   **DesempaquetarDATEinVariant**
-   **\_PackVARIANTVARIANTVARIant**
-   **\_DesempaquetarVARIANTFROMVARIant**
-   **PackVARIANTinVariant**
-   **DesempaquetarVARIANTfromVariant**
