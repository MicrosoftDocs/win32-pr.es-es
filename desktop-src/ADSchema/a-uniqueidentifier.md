# <a name="uniqueidentifier-attribute"></a>atributo uniqueIdentifier

El tipo de atributo uniqueIdentifier especifica un "identificador único" para un objeto representado en el directorio .



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Uniqueidentifier                            |
| Ldap-Display-Name | Uniqueidentifier                            |
| Size              | \-                                          |
| Actualizar privilegios  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.44                  |
| System-Id-Guid    | ba0184c7-38c5-4bed-a526-75421470580c        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
