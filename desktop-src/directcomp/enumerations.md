# <a name="directcomposition-enumerations"></a>Enumeraciones de DirectComposition

En esta sección se describen las enumeraciones proporcionadas por la API de Microsoft DirectComposition.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                                  | Descripción                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**visibilidad de la \_ superficie DCOMPOSITION \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_backface_visibility)<br/>              | Especifica la visibilidad del prefacio que se va a aplicar a un visual. <br/>                                                                                                                            |
| [**DCOMPOSITION \_ \_ modo de interpolación de mapa de bits \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_bitmap_interpolation_mode)<br/> | Especifica el modo de interpolación que se va a utilizar cuando un mapa de bits se compone de cualquier transformación en la que los píxeles del mapa de bits no se alinean exactamente uno a uno con píxeles en pantalla. <br/>         |
| [**DCOMPOSITION \_ modo de borde \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_border_mode)<br/>                              | Especifica el modo de borde que se va a utilizar al crear un mapa de bits o al aplicar un clip con cualquier transformación de modo que los bordes del mapa de bits o del clip no estén alineados por ejes con coordenadas de entero. <br/> |
| [**\_modo compuesto \_ DCOMPOSITION**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_composite_mode)<br/>                        | El modo que se va a usar para mezclar el contenido del mapa de bits de un visual con el destino de representación.<br/>                                                                                                          |
| [**\_modo de opacidad DCOMPOSITION \_**](/windows/desktop/api/dcomptypes/ne-dcomptypes-dcomposition_opacity_mode)<br/>                            | Especifica cómo se aplica el valor de opacidad efectiva de un elemento visual al contenido y los elementos secundarios de Visual s.<br/>                                                                                  |
| [**\_modo de transformación de DCOMPOSITION \_**](/previous-versions/windows/desktop/legacy/dn904487(v=vs.85))<br/>                        | Especifica el estilo de transformación que se va a aplicar a un visual. <br/>                                                                                                                                |
