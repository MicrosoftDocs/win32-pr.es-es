# <a name="d1196-a8-shared-bitmap"></a>D1196: mapa de bits compartido A8

\_ \_ \_ En esta API no se admite una superficie de DXGI con formato dxgi UNORM.



|             |         |
|-------------|---------|
| Nivel de error | Advertencia |



 

## <a name="possible-causes"></a>Causas posibles

\_El formato \_ de dxgi a8 \_ UNORM no se admite como formato de mapa de bits de interoperabilidad de dxgi, aunque se admita en otras API.

## <a name="possible-fixes"></a>Posibles correcciones

Cree un mapa de bits de interoperabilidad de DXGI con un formato diferente.
