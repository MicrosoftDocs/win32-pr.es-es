# <a name="generic-adsi-error-codes"></a>Códigos de error ADSI genéricos

Cualquiera de los proveedores del sistema ADSI puede devolver estos tipos de códigos de error para representar mensajes de error específicos de ADSI. Tienen el código de instalación 5 con el bit de gravedad establecido **en true** o **false.** Cuando el bit de gravedad se establece en **true,** se producirán valores de error con el formato 0x80005xxx y mensajes de error con el formato **E ADS \_ \_ \* *_. Cuando el*** bit de gravedad se establece en _ false , esto da como resultado valores de error con el formato 0x00005xxx y los mensajes de error son del formulario **\_ \_ \* ADS de S.**

En la tabla siguiente se enumeran los códigos de error ADSI genéricos definidos en Adserr.h.



| Valor       | Código                                  | Descripción                                                           | Acción correctora                                                                                                                                                                                                                                                                                          |
|-------------|---------------------------------------|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0x00005011L | **S \_ ADS \_ ERRORSOCCURRED**            | Durante una consulta, se produjeron uno o varios errores.                          | Compruebe que la preferencia de búsqueda se puede establecer legalmente y, si es así, que está establecida correctamente.                                                                                                                                                                                                                  |
| 0x00005012L | **S \_ ADS \_ NOMORE \_ ROWS**              | La operación de búsqueda ha alcanzado la última fila.                        | Pase al resto del programa.                                                                                                                                                                                                                                                                        |
| 0x00005013L | **S \_ ADS \_ NOMORE \_ COLUMNS**           | La operación de búsqueda ha alcanzado la última columna de la fila actual. | Pase a la siguiente fila.                                                                                                                                                                                                                                                                                       |
| 0x80005000L | **E \_ ADS \_ BAD \_ PATHNAME**             | Se pasó un nombre de ruta de acceso ADSI no válido.                                  | Compruebe que el objeto existe en el servidor de directorios y compruebe si hay errores tipográficos en la ruta de acceso.                                                                                                                                                                                                        |
| 0x80005001L | **E \_ ADS \_ INVALID \_ DOMAIN \_ OBJECT**   | Se solicitó un objeto de dominio ADSI desconocido.                          | Compruebe la ruta de acceso del objeto de dominio.                                                                                                                                                                                                                                                                      |
| 0x80005002L | **E \_ ADS \_ INVALID \_ USER \_ OBJECT**     | Se solicitó un objeto de usuario ADSI desconocido.                            | Compruebe la existencia del objeto de usuario y compruebe si hay errores tipográficos en la ruta de acceso y los derechos de acceso de usuario.                                                                                                                                                                                                           |
| 0x80005003L | **E \_ ADS \_ INVALID \_ COMPUTER \_ OBJECT** | Se solicitó un objeto de equipo ADSI desconocido.                        | Compruebe la existencia del objeto de equipo, compruebe si hay errores tipográficos en la ruta de acceso y los derechos de acceso del equipo.                                                                                                                                                                                                   |
| 0x80005004L | **E \_ ADS \_ UNKNOWN \_ OBJECT**           | Se solicitó un objeto ADSI desconocido.                                 | Compruebe el nombre y los derechos de acceso al objeto.                                                                                                                                                                                                                                                    |
| 0x80005005L | **E \_ ADS \_ PROPERTY \_ NOT \_ SET**        | No se estableció la propiedad ADSI especificada.                              |                                                                                                                                                                                                                                                                                                            |
| 0x80005006L | **NO \_ SE ADMITE LA PROPIEDAD E \_ \_ \_ ADS**  | No se admite la propiedad ADSI especificada.                         | Compruebe que está establecida la propiedad correcta.                                                                                                                                                                                                                                                                   |
| 0x80005007L | **E \_ ADS \_ PROPERTY \_ INVALID**         | La propiedad ADSI especificada no es válida                                | Compruebe los parámetros pasados a la llamada al método .                                                                                                                                                                                                                                                           |
| 0x80005008L | **E \_ ADS \_ BAD \_ PARAMETER**            | Uno o varios parámetros de entrada no son válidos.                             |                                                                                                                                                                                                                                                                                                            |
| 0x80005009L | **E \_ ADS \_ OBJECT \_ UNBOUND**           | El objeto ADSI especificado no está enlazado a un recurso remoto.          | Llame [**a GetInfo**](/windows/desktop/api/Iads/nf-iads-iads-getinfo) en un objeto recién creado después de llamar a [**SetInfo.**](/windows/desktop/api/Iads/nf-iads-iads-setinfo)                                                                                                                                                                                      |
| 0x8000500AL | **E \_ ADS \_ PROPERTY \_ NOT \_ MODIFIED**   | El objeto ADSI especificado no se ha modificado.                      |                                                                                                                                                                                                                                                                                                            |
| 0x8000500BL | **PROPIEDAD \_ DE ANUNCIOS DE E \_ \_ MODIFICADA**        | Se ha modificado el objeto ADSI especificado.                          |                                                                                                                                                                                                                                                                                                            |
| 0x8000500CL | **E \_ ADS \_ CANT \_ CONVERT \_ DATATYPE**   | El tipo de datos no se puede convertir a o desde un tipo de datos DS nativo.      | Compruebe que se usa el tipo de datos correcto o que hay suficientes datos de esquema disponibles para realizar la conversión de tipos de datos.                                                                                                                                                                           |
| 0x8000500DL | **E \_ ADS \_ PROPERTY \_ NOT \_ FOUND**      | La propiedad no se encuentra en la memoria caché.                            | Compruebe que se ha llamado a [**GetInfo**](/windows/desktop/api/Iads/nf-iads-iads-getinfo) de forma implícita o explícita. Si el atributo es un atributo operativo, debe recuperarse explícitamente con [**GetInfoEx**](/windows/desktop/api/Iads/nf-iads-iads-getinfoex) en lugar de **GetInfo**. Si el problema persiste, la propiedad no se ha establecido en el servidor. |
| 0x8000500EL | **E \_ ADS \_ OBJECT \_ EXISTS**            | El objeto ADSI existe.                                               | Use otro nombre para crear el objeto.                                                                                                                                                                                                                                                                 |
| 0x8000500FL | **E \_ ADS \_ SCHEMA \_ VIOLATION**         | El intento de acción infringe las reglas de esquema del servicio de directorio.     |                                                                                                                                                                                                                                                                                                            |
| 0x80005010L | **E \_ ADS \_ COLUMN \_ NOT \_ SET**          | No se estableció la columna especificada en adsi.                         |                                                                                                                                                                                                                                                                                                            |
| 0x80005014L | **FILTRO \_ NO VÁLIDO DE \_ ANUNCIOS E \_**           | El filtro de búsqueda especificado no es válido.                               | Use el formato correcto del filtro aceptado por el servidor de directorios.                                                                                                                                                                                                                                     |
