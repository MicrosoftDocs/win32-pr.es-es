# <a name="application-uses"></a>Usos de la aplicación

Servicios de Certificate Server emite certificados que se pueden utilizar en aplicaciones de infraestructura de clave pública (PKI). Por ejemplo, los certificados funcionan en las aplicaciones SSL, S/MIME e IPSEC. Los posibles escenarios en los que se pueden usar estos certificados son la firma digital y el cifrado de mensajes. Los servicios de Certificate Server proporcionan certificados, [*listas de revocación de certificados*](../secgloss/c-gly.md) (CRL) y puntos de distribución de CRL (CDP) que son utilizados por aplicaciones de Microsoft y que no son de Microsoft.
