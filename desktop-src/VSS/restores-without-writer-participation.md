<span data-ttu-id="c2174-104">La participación del escritor en una copia de seguridad de VSS está diseñada para permitir que las aplicaciones controlen qué y cómo se van a usar los datos de restauración.</span><span class="sxs-lookup"><span data-stu-id="c2174-104">Writer participation in a VSS backup is designed to allow applications to control what and how their restore data is to be used.</span></span>

<span data-ttu-id="c2174-107">Sin embargo, hay motivos por los que una aplicación de copia de seguridad podría querer o necesitar restaurar una copia de seguridad de VSS sin la participación del escritor:</span><span class="sxs-lookup"><span data-stu-id="c2174-107">However, there are reasons why a backup application might want or need to restore a VSS backup without writer participation:</span></span>

-   <span data-ttu-id="c2174-108">Las aplicaciones que no reconocen VSS administran los datos.</span><span class="sxs-lookup"><span data-stu-id="c2174-108">Data is managed by VSS-unaware applications.</span></span> <span data-ttu-id="c2174-109">Casi todos los sistemas tendrán algunas aplicaciones (editores de texto, lectores de correo, procesadores de textos, etc.) que no son conscientes de VSS.</span><span class="sxs-lookup"><span data-stu-id="c2174-109">Almost every system will have some applications—text editors, mail readers, word processors, and so forth—that are VSS unaware.</span></span> <span data-ttu-id="c2174-110">Estos datos no se pueden restaurar con la participación del escritor.</span><span class="sxs-lookup"><span data-stu-id="c2174-110">This data cannot be restored using writer participation.</span></span>

    <span data-ttu-id="c2174-111">Por lo general, este tipo de datos no es crítico para el sistema o el servicio, y la restauración no debe ser problemática o, al menos, no es problemático que durante una restauración convencional.</span><span class="sxs-lookup"><span data-stu-id="c2174-111">Generally, this type of data is not system- or service-critical, and restoring it should not be problematic, or at least no more problematic than during a conventional restore.</span></span>

    <span data-ttu-id="c2174-112">Al igual que con los preparativos de las restauraciones convencionales, si es posible, los operadores de restauración deben intentar suspender o finalizar dichas aplicaciones antes de iniciar una restauración de VSS.</span><span class="sxs-lookup"><span data-stu-id="c2174-112">As with preparations for conventional restores, if possible, restore operators should attempt to suspend or terminate such applications prior to starting a VSS restore.</span></span>

-   <span data-ttu-id="c2174-113">Escritores de VSS que faltan.</span><span class="sxs-lookup"><span data-stu-id="c2174-113">Missing VSS writers.</span></span> <span data-ttu-id="c2174-114">Esta situación puede ser bastante común al restaurar el estado de un sistema dañado.</span><span class="sxs-lookup"><span data-stu-id="c2174-114">This situation may be fairly common when restoring the state of a damaged system.</span></span> <span data-ttu-id="c2174-115">Una operación de copia de seguridad debe determinar si es conveniente restaurar los archivos para los escritores que faltan.</span><span class="sxs-lookup"><span data-stu-id="c2174-115">A backup operation must determine whether it is desirable to restore files for missing writers.</span></span> <span data-ttu-id="c2174-116">Si se desea la restauración, los archivos se pueden restaurar de la misma forma que una copia de seguridad convencional los restaurará.</span><span class="sxs-lookup"><span data-stu-id="c2174-116">If restoration is desirable, the files can be restored just as a conventional backup would restore them.</span></span>
-   <span data-ttu-id="c2174-117">Restauración privada de los datos de un escritor.</span><span class="sxs-lookup"><span data-stu-id="c2174-117">A private restore of a writer's data.</span></span> <span data-ttu-id="c2174-118">Un solicitante puede optar por restaurar los datos de un escritor en ejecución en alguna ubicación privada sin notificar al escritor.</span><span class="sxs-lookup"><span data-stu-id="c2174-118">A requester may choose to restore the data of a running writer to some private location without notifying the writer.</span></span> <span data-ttu-id="c2174-119">Un ejemplo de esto podría ser la restauración de los datos del escritor para admitir la comparación sin conexión.</span><span class="sxs-lookup"><span data-stu-id="c2174-119">An example of this might be restoration of the writer's data to support offline comparison.</span></span> <span data-ttu-id="c2174-120">En este tipo de situación, un solicitante no querrá usar la [*nueva ubicación de destino*](vssgloss-n.md) mientras realiza la restauración, ya que no quiere que el escritor tenga acceso a los datos.</span><span class="sxs-lookup"><span data-stu-id="c2174-120">In this sort of situation, a requester would not want to use the [*new target location*](vssgloss-n.md) while doing the restore, because it does not want the writer to access the data.</span></span>
-   <span data-ttu-id="c2174-121">Un escritor no desea estar implicado durante la restauración.</span><span class="sxs-lookup"><span data-stu-id="c2174-121">A writer does not want to be involved during restore.</span></span> <span data-ttu-id="c2174-122">Un escritor indica esto pasando VSS \_ WRE \_ nunca para el parámetro *writerRestore* de [**IVssCreateWriterMetadata:: SetRestoreMethod**](/windows/desktop/api/VsWriter/nf-vswriter-ivsscreatewritermetadata-setrestoremethod).</span><span class="sxs-lookup"><span data-stu-id="c2174-122">A writer indicates this by passing in VSS\_WRE\_NEVER for the *writerRestore* parameter of [**IVssCreateWriterMetadata::SetRestoreMethod**](/windows/desktop/api/VsWriter/nf-vswriter-ivsscreatewritermetadata-setrestoremethod).</span></span>
-   <span data-ttu-id="c2174-123">Un escritor requiere un método de restauración personalizado.</span><span class="sxs-lookup"><span data-stu-id="c2174-123">A writer requires a custom restore method.</span></span> <span data-ttu-id="c2174-124">Un escritor indica que requiere una restauración personalizada pasando VSS \_ RME \_ personalizado para el parámetro de *método* de [**IVssCreateWriterMetadata:: SetRestoreMethod**](/windows/desktop/api/VsWriter/nf-vswriter-ivsscreatewritermetadata-setrestoremethod).</span><span class="sxs-lookup"><span data-stu-id="c2174-124">A writer indicates that it requires a custom restore by passing in VSS\_RME\_CUSTOM for the *method* parameter of [**IVssCreateWriterMetadata::SetRestoreMethod**](/windows/desktop/api/VsWriter/nf-vswriter-ivsscreatewritermetadata-setrestoremethod).</span></span> <span data-ttu-id="c2174-125">En este caso, este escritor no debe estar implicado en el proceso de restauración a menos que la documentación de restauración personalizada para ese escritor indique lo contrario.</span><span class="sxs-lookup"><span data-stu-id="c2174-125">In this case, this writer should not be involved in the restore process unless the custom-restore documentation for that writer indicates otherwise.</span></span>

<span data-ttu-id="c2174-126">Un solicitante implica un escritor en el proceso de restauración mediante la especificación de uno de los componentes de ese escritor en una llamada a [**IVssBackupComponents:: SetSelectedForRestore**](/windows/desktop/api/VsBackup/nf-vsbackup-ivssbackupcomponents-setselectedforrestore).</span><span class="sxs-lookup"><span data-stu-id="c2174-126">A requester involves a writer in the restore process by specifying one of that writer's components in a call to [**IVssBackupComponents::SetSelectedForRestore**](/windows/desktop/api/VsBackup/nf-vsbackup-ivssbackupcomponents-setselectedforrestore).</span></span> <span data-ttu-id="c2174-127">Los datos de un escritor se pueden restaurar sin que intervenga el escritor simplemente si no se especifica ninguno de los componentes de ese escritor en una llamada a **IVssBackupComponents:: SetSelectedForRestore**.</span><span class="sxs-lookup"><span data-stu-id="c2174-127">A writer's data can be restored without involving the writer by simply not specifying any of that writer's components in a call to **IVssBackupComponents::SetSelectedForRestore**.</span></span> <span data-ttu-id="c2174-128">Si un escritor no espera ningún evento de restauración, en el proceso de restauración, el escritor puede provocar que se notifiquen falsos errores para ese escritor.</span><span class="sxs-lookup"><span data-stu-id="c2174-128">If a writer does not expect any restore events, involving that writer in the restore process can cause spurious errors to be reported for that writer.</span></span>
