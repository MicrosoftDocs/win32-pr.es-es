# <a name="g"></a>G

## <a name="gamma-correction"></a>corrección gamma

Las características nativas del tono físico de un dispositivo suelen ser diferentes en distintos tipos de dispositivos. El factor de corrección gamma compensa estas diferencias.

## <a name="gamma-curve"></a>curva gamma

Colección de correcciones gamma en toda la gama de colores. Cuando se trazan en un gráfico, forman una curva.

## <a name="gamma-ramp"></a>gamma ramp

Consulte curva gamma.

## <a name="gamut"></a>Gama

Conjunto de colores de un espacio de colores que un dispositivo puede generar.

## <a name="gmmp"></a>GMMP

El formato de perfil de modelo de mapa de gama de colores es un formato de archivo basado en XML que controla la asignación de colores entre las gamas de dispositivos de origen y destino.
