# <a name="accessing-active-directory-using-visual-basic"></a>Acceso a Active Directory mediante Visual Basic

Una de las características más eficaces que empezó a estar disponible con el sistema operativo Microsoft Windows 2000 es el servicio de directorio Active Directory Microsoft. Al iniciar sesión en un dominio Windows 2000, Active Directory se pone en acción; para buscar la impresora de color más cercana en el edificio, puede usar Active Directory. Se puede usar para una búsqueda de libreta de direcciones o una búsqueda de usuarios que trabajan en la creación de 40. Con Active Directory, puede usar una tarjeta inteligente para iniciar sesión en un Windows 2000. Incluso puede unir datos del software de base Microsoft SQL Server de datos y Active Directory.

Muchas otras tecnologías de Microsoft, como Microsoft Message Queue Server (MSMQ), COM (Almacén de clases), Seguridad del protocolo de Internet (IPsec), objetos directiva de grupo (GPO) y Exchange se integran con Active Directory.

En esta sección se describe cómo usar Active Directory Service Interfaces (ADSI) para acceder a Active Directory. Si usa un escenario para una empresa ficticia, Fabrikam Corporation, aprenderá a realizar algunas tareas administrativas básicas.

A medida que avance por este escenario, los ejemplos de código de cada sección se basa en sí mismos.

Por brevedad, todos los ejemplos de código se escriben con el sistema de desarrollo Visual Basic Microsoft. Para obtener más información sobre la conversión de C++, vea [Mapping ADSI Visual Basic Code to C++ Code](mapping-adsi-visual-basic-code-to-c-code.md).
