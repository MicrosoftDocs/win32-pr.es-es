# <a name="media-session-attributes"></a>Atributos de sesión multimedia

Los atributos siguientes se usan para configurar la sesión multimedia.



| Atributo                                                                                            | Descripción                                                                                                 |
|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**MF \_ SESSION \_ CONTENT \_ PROTECTION \_ MANAGER**](mf-session-content-protection-manager-attribute.md) | Proporciona una interfaz de devolución de llamada para que la aplicación reciba un objeto habilitador de contenido de la sesión PMP. |
| [**HORA \_ GLOBAL DE LA SESIÓN DE \_ \_ MF**](mf-session-global-time-attribute.md)                                | Especifica si las topologías tendrán una hora de inicio y de detenerse global.                                        |
| [**MF \_ SESSION \_ QUALITY \_ MANAGER**](mf-session-quality-manager-attribute.md)                        | Contiene el CLSID de un administrador de calidad para la sesión multimedia.                                              |
| [**MODO DE \_ ORIGEN REMOTO DE SESIÓN \_ \_ \_ MF**](mf-session-remote-source-mode-attribute.md)                 | Especifica que el origen de medios se está ejecutando en un proceso remoto.                                             |
| [**CONTEXTO \_ DEL SERVIDOR DE SESIÓN \_ \_ MF**](mf-session-server-context-attribute.md)                          | Permite que dos instancias de la sesión multimedia compartan el mismo proceso PMP.                                   |
| [**MF \_ SESSION \_ TOPOLOADER**](mf-session-topoloader-attribute.md)                                   | Contiene el CLSID de un cargador de topologías para la sesión multimedia.                                              |
