# <a name="sessiondatabase-property"></a>Propiedad Session. Database

La propiedad de **base de datos** del objeto de [**sesión**](session-object.md) es una propiedad de solo lectura que devuelve la base de datos de la sesión de instalación actual como un objeto de [**base de datos**](database-object.md) .

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Session.Database
```



## <a name="property-value"></a>Valor de propiedad
