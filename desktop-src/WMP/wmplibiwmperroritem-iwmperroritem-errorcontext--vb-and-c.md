# <a name="iwmperroritemerrorcontext-property"></a>IWMPErrorItem::errorContext, propiedad

La **propiedad errorContext** obtiene un valor que indica el contexto del error.

## <a name="syntax"></a>Syntax


```CSharp
public System.Object errorContext {get; set;}
```


```VB

Public ReadOnly Property errorContext As System.Object
```





## <a name="property-value"></a>Valor de propiedad

Objeto **System.Object que** es el contexto de error.

## <a name="remarks"></a>Comentarios

El contexto de error es información que Microsoft usa para proporcionar información adicional al personal de soporte técnico.
