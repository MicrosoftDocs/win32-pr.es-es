# <a name="ms-authz-effective-security-policy-attribute"></a>atributo MS-authz-Effective-Security-Policy

En el caso de una regla de acceso central, este atributo define el permiso que se aplica a los recursos de destino en la regla de acceso central.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-authz-Effective-Security-Policy          |
| Nombre para mostrar de LDAP | msAuthz-EffectiveSecurityPolicy             |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2150                     |
| System-ID-GUID    | 07831919-8f94-4fb6-8a42-91545dccdad3        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
