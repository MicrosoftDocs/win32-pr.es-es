# <a name="i-switch"></a>Modificador /i

El **modificador /i** especifica el directorio que contiene los archivos de metadatos que se combinarán.

``` syntax
mdmerge /i metadata_directory
```

## <a name="switch-options"></a>Cambiar opciones

<dl> <dt>

*directorio de \_ metadatos* 
</dt> <dd>

Especifica el directorio que contiene los archivos de metadatos que se combinarán.

</dd> </dl>
