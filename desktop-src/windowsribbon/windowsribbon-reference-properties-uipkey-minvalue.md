# <a name="ui_pkey_minvalue"></a>Interfaz de usuario \_ PKEY \_ MinValue

Identifica la \_ propiedad MinValue PKEY de la interfaz de usuario \_ .

```
propertyDescription
   name = UI_PKEY_MinValue
   shellPKey = UI_PKEY_MinValue
   formatID = 00000204-7363-696e-8441798acf5aebb7
   propID = 204
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Observaciones

\_ \_ Una aplicación usa la interfaz de usuario PKEY MinValue para consultar el valor mínimo del control de [número](windowsribbon-controls-spinner.md) .
