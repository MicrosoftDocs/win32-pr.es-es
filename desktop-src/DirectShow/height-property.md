# <a name="height-property-directshow"></a>Propiedad height (DirectShow)

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

La `Height` propiedad establece o recupera el alto del rectángulo.

``` syntax
[ iHeight = ] DVDRect.Height
```

## <a name="return-value"></a>Valor devuelto

Devuelve el alto, en píxeles, del rectángulo como un entero.

## <a name="remarks"></a>Observaciones

Esta propiedad es de lectura/escritura y no tiene ningún valor predeterminado.
