# <a name="last-backup-restoration-time-attribute"></a>Último: atributo de tiempo de restauración de copia de seguridad

Cuándo se produjo la última restauración del sistema.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Última copia de seguridad-tiempo de restauración         |
| Nombre para mostrar de LDAP | lastBackupRestorationTime            |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.519               |
| System-ID-GUID    | 1fbb0be8-ba63-11d0-afef-0000f80367c1 |
| Sintaxis            | [**Interval**](s-interval.md)       |
