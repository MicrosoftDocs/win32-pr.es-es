# <a name="required-libraries"></a>Bibliotecas necesarias

La vinculación de funciones CryptoAPI requiere la disponibilidad de crypt32. lib. Los programas que usan algunas de las funciones también pueden requerir advapi32. lib. Al utilizar el compilador de Visual C++, estos archivos se deben enumerar entre los módulos Object/Library.
