# <a name="obtaining-the-size-of-a-compressed-file"></a>Obtener el tamaño de un archivo comprimido

Use la [**función GetCompressedFileSize**](/windows/desktop/api/fileapi/nf-fileapi-getcompressedfilesizea) para obtener el tamaño comprimido de un archivo. Si el archivo está comprimido, su tamaño comprimido será menor que su tamaño sin comprimir. Use la [**función GetFileSize**](/windows/desktop/api/FileAPI/nf-fileapi-getfilesize) para determinar el tamaño sin comprimir de un archivo.
