# <a name="dmo-demo-sample"></a>Ejemplo de demostración de DMO

## <a name="description"></a>Descripción

En esta aplicación de ejemplo se muestra cómo utilizar [objetos multimedia de DirectX](directx-media-objects.md) (DMO) en una aplicación. Transmite los datos de audio de un archivo WAV a través de un efecto de audio DMO a un búfer de DirectSound.

## <a name="usage"></a>Uso

Haga clic en el botón **archivo de sonido** para cargar un archivo de sonido. Use el cuadro combinado desplegable para seleccionar un efecto de audio DMO. Haga clic en el botón **reproducir** para oír el archivo de sonido que se reproduce con el efecto de audio.

## <a name="downloading-the-sample"></a>Descargar el ejemplo

Para descargar los ejemplos del SDK de DirectShow, instale la versión más reciente de la [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Este ejemplo se instala en la siguiente ruta de acceso: ejemplos *\[ raíz \] del SDK* \\ \\ multimedia \\ DirectShow \\ DMO \\ DMODemo.
