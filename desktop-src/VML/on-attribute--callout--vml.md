# <a name="on-attribute-calloutvml"></a>Atributo On (callout)(VML)

En este tema se describe VML, una característica que está en desuso a partir Windows Internet Explorer 9. Las páginas web y las aplicaciones que se basan en VML deben migrarse a SVG u otros estándares ampliamente admitidos.

> A partir de diciembre de 2011, este tema se archivó. Como resultado, ya no se mantiene activamente. Determina si una forma es una llamada. Lectura/escritura **DvTriState**.

**Se aplica a**

[Llamada](msdn-online-vml-callout-element.md)

**Sintaxis de etiquetas**

<o: *element* on=" *expression* ">

**Comentarios:**

El valor predeterminado es **False**. Una forma es una llamada si el valor se establece en **True.**
