# <a name="windows-security-center-interfaces"></a>Interfaces de Security Center de Windows

Las siguientes interfaces están disponibles para su uso con Windows Security Center.

## <a name="in-this-section"></a>En esta sección



| Tema                                                 | Descripción                                                                                                                      |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| [**IWscProduct**](/windows/desktop/api/Iwscapi/nn-iwscapi-iwscproduct)<br/>         | Proporciona métodos para obtener información del producto para que un proveedor individual interactúe con Windows Security Center.<br/> |
| [**IWSCProductList**](/windows/desktop/api/iwscapi/nn-iwscapi-iwscproductlist)<br/> | Proporciona métodos para recopilar información de productos para el tipo seleccionado de proveedores instalados en el equipo.<br/>         |
