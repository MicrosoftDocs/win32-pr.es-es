# <a name="win32-switch"></a>Modificador /win32

El **modificador /win32** dirige al compilador MIDL a generar archivos de código auxiliar, o un archivo de biblioteca de tipos, para un entorno de 32 bits.

``` syntax
midl /win32
```

## <a name="switch-options"></a>Opciones de cambio

Este modificador no tiene parámetros.

## <a name="remarks"></a>Comentarios

El **modificador /win32** equivale a establecer el modificador [**/env**](-env.md) en win32.
