# <a name="libmaincpp"></a>LIBMAIN. CPP

En el componente de proveedor de ejemplo, un ejemplo de código del punto de entrada de DLL estándar para Adssmp.dll está en LibMain. cpp. En la tabla siguiente se enumeran las rutinas admitidas.



| Elemento                                  | Descripción                                                              |
|---------------------------------------|--------------------------------------------------------------------------|
| **DllGetClassObject**<br/>      | Punto de entrada de DLL estándar para buscar generadores de clases.<br/>        |
| **DllCanUnloadNow**<br/>        | Punto de entrada de DLL estándar para determinar si se puede descargar el archivo DLL.<br/> |
| **LibMain**<br/>                | Punto de entrada de inicialización de DLL estándar.<br/>                      |
| **IsCompatibleOleVersion**<br/> | Comprobar la compatibilidad con el tiempo de ejecución de OLE.<br/>                   |
| **DllMain**<br/>                | Punto de entrada para la mayoría de las versiones de Windows 2000 o Windows NT.<br/>     |
