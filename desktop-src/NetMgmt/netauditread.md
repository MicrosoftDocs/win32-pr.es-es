# <a name="netauditread"></a>NetAuditRead

La función **NetAuditRead** está obsoleta. Se incluye por compatibilidad con las versiones de 16 bits de Windows. Otras aplicaciones deben usar el [registro de eventos](/windows/desktop/EventLog/event-logging).
