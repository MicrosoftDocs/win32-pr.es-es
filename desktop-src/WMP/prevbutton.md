# <a name="prevbutton"></a>PREVBUTTON

Se trata de un **button** predefinido con los siguientes valores predeterminados.

``` syntax
onclick="jscript:player.controls.previous()"
upToolTip="Previous"
cursor="system"
enabled="wmpenabled:player.controls.previous"
```

## <a name="remarks"></a>Comentarios

Todas las propiedades de este BOTÓN se pueden invalidar especificándolos explícitamente.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
