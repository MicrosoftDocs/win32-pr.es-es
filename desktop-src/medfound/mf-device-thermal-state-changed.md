# <a name="mf_device_thermal_state_changed-attribute"></a>\_Atributo de \_ estado térmico del dispositivo MF \_ \_ modificado

Representa un evento que indica un cambio de estado térmico en el dispositivo.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

El valor de este atributo especifica el valor térmico del dispositivo en porcentaje.
