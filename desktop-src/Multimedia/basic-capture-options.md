# <a name="basic-capture-options"></a>Opciones de captura básicas

Al modificar uno o varios de los parámetros de captura definidos en la estructura [**CAPTUREPARMS**](/windows/win32/api/vfw/ns-vfw-captureparms) , puede realizar las siguientes tareas:

-   Cambiar la velocidad de captura de fotogramas.
-   Especificar el control del teclado o del mouse para finalizar una sesión de captura.
-   Especifique una duración para una sesión de captura.
