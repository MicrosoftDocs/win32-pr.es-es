# <a name="using-inf-files"></a>Uso de archivos INF

Después de crear un archivo INF para el proceso de instalación, las funciones de configuración pueden acceder a esa información para utilizarla al copiar, eliminar o cambiar el nombre de archivos y actualizar INIs y entradas del registro.

En los temas siguientes se describe cómo usar las funciones de configuración de para recuperar información de archivos INF y cómo realizar operaciones de instalación haciendo referencia a un archivo INF:

-   [Creación de un archivo INF](authoring-an-inf-file.md)
-   [Abrir y cerrar un archivo INF](opening-and-closing-an-inf-file.md)
-   [Recuperar información de un archivo INF](retrieving-information-from-an-inf-file.md)
-   [Instalación desde un archivo INF](installing-from-an-inf-file.md)
-   [Especificación de un descriptor de seguridad de un archivo INF](specifying-a-security-descriptor-from-an-inf-file.md)
