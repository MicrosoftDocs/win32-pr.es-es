# <a name="taskdefinitionsettings-property"></a>Propiedad TaskDefinition. Settings

En el caso de scripting, obtiene o establece la configuración que define cómo realiza la tarea el servicio de Programador de tareas.

## <a name="syntax"></a>Sintaxis


```VB
TaskDefinition.Settings As TaskSettings
```



## <a name="property-value"></a>Valor de propiedad

La configuración que define cómo realiza la tarea el servicio de Programador de tareas.
