# <a name="ttm_gettoolcount-message"></a>TTM \_ GETTOOLCOUNT

Recupera un recuento de las herramientas mantenidas por un control ToolTip.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>Debe ser cero.</dd> <dt>

*lParam* 
</dt> <dd>Debe ser cero.</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Devuelve un recuento de herramientas.

|-------------------------------------|---------------------------------------------------------------------------------------|
