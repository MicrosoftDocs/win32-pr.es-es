# <a name="iwmdrmencryptscatter-interface"></a>IWMDRMEncryptScatter (interfaz)

La **interfaz IWMDRMEncryptScatter** desenreda y cifra los datos.

## <a name="members"></a>Miembros

La **interfaz IWMDRMEncryptScatter** hereda de la [**interfaz IUnknown.**](/windows/desktop/api/unknwn/nn-unknwn-iunknown) **IWMDRMEncryptScatter** también tiene estos tipos de miembros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

La **interfaz IWMDRMEncryptScatter** tiene estos métodos.



| Método                                                                | Descripción                                                            |
|:----------------------------------------------------------------------|:-----------------------------------------------------------------------|
| [**EncryptScatter**](iwmdrmencryptscatter-encryptscatter.md)         | Desenlaza y cifra los datos.<br/>                              |
| [**InitEncryptScatter**](iwmdrmencryptscatter-initencryptscatter.md) | Inicializa la **interfaz IWMDRMEncryptScatter** para su uso.<br/> |
