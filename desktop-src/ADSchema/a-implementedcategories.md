Este atributo contiene una lista de los id. de categoría de componente que implementa este objeto.

| CN                | Implemented-Categories                                |
| Ldap-Display-Name | implementedCategories                                 |
| Size              | \-                                                    |
| Actualizar privilegios  | \-                                                    |
