# <a name="is-defunct-attribute"></a>Is-Defunct atributo)

Si es **true**, la clase o el atributo ya no se pueden usar. Puede que existan versiones anteriores de este objeto, pero no se pueden crear otras nuevas.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Is-Defunct                           |
| Nombre para mostrar de LDAP | isDefunct                            |
| Tamaño              | \-                                   |
| Actualizar privilegio  | Administrador de esquema                 |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.661               |
| System-ID-GUID    | 28630ebe-41d5-11d1-a9c1-0000f80367c1 |
| Sintaxis            | [**Booleano**](s-boolean.md)         |
