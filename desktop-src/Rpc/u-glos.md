# <a name="u-rpc"></a>U (RPC)

[A](a-glos.md) [B](b-glos.md) [C](c-glos.md) [D](d-glos.md) [E](e-glos.md) [F](f-glos.md) G H [I](i-glos.md) J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [o](o-glos.md) [p](p-glos.md) [Q](q.md) [R](r-glos.md) [s](s-glos.md) [T](t-glos.md) U [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_udp_glos"></span><span id="_RPC_UDP_GLOS"></span>**Protocolo de datagramas de usuario (UDP)**
</dt> <dd>

Transporte de red que usa sockets de [*datagrama*](d-glos.md) sin conexión y se superpone a la parte superior del Protocolo de Internet ([*IP)*](i-glos.md).

</dd> <dt>

<span id="_rpc_unbind_glos"></span><span id="_RPC_UNBIND_GLOS"></span>**Desenlazar**
</dt> <dd>

Proceso de finalización de la conexión lógica entre un cliente y un servidor.

</dd> <dt>

<span id="_rpc_unique_pointer_glos"></span><span id="_RPC_UNIQUE_POINTER_GLOS"></span>**puntero único**
</dt> <dd>

Puntero que puede ser null o señalar a datos existentes, cuyo valor puede cambiar durante una llamada a procedimiento remoto. Un puntero único no puede tener un alias. El \[ atributo [Unique](/windows/desktop/Midl/unique) \] designa un puntero único. Vea también [*puntero completo*](f-glos.md), [*puntero de referencia*](r-glos.md).

</dd> <dt>

<span id="_rpc_uuid_glos"></span><span id="_RPC_UUID_GLOS"></span>**Identificador único universal (UUID)**
</dt> <dd>

(o [**GUID**](/windows/win32/api/guiddef/ns-guiddef-guid)) valor de 128 bits utilizado en la comunicación entre procesos para identificar entidades como interfaces de cliente y de servidor, vectores de punto de entrada del administrador y objetos RPC. Vea también [uuidgen](/windows).

</dd> <dt>

<span id="_rpc_unmarshaling_glos"></span><span id="_RPC_UNMARSHALING_GLOS"></span>**resolución**
</dt> <dd>

Proceso de desempaquetar los parámetros que se han enviado a través de los límites del proceso.

</dd> <dt>

<span id="_rpc_uuidgen_glos"></span><span id="_RPC_UUIDGEN_GLOS"></span>**uuidgen**
</dt> <dd>

Programa de utilidad, que se incluye con el kit de desarrollo de software (SDK) de la plataforma, que usa un valor de tiempo y el identificador de la tarjeta de red de la máquina para generar los [UUID](/windows) que se garantiza que son únicos.

</dd> </dl>
