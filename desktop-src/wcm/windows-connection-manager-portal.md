# <a name="windows-connection-manager"></a>Administrador de conexiones de Windows

## <a name="purpose"></a>Propósito

Windows Connection Manager (WCM) permite la creación y configuración del software del administrador de conexiones.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Windows Connection Manager está diseñado para que lo usen los desarrolladores que usan software de desarrollo de C/C++.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

Windows Connection Manager API se admite en Windows 8 y Windows Server 2012.
