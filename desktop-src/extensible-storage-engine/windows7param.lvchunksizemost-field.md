# <a name="windows7paramlvchunksizemost-field"></a>Windows7Param. LVChunkSizeMost, campo

Este parámetro se usa para recuperar el tamaño del fragmento de los datos de valor largo (BLOB). Establecer y recuperar datos en múltiplos de este tamaño aumenta la eficacia.
