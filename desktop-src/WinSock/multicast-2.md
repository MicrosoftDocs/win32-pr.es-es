# <a name="multicast"></a>Multidifusión

Las funciones Multipoint de Winsock genéricas admiten la multidifusión IP. Sin embargo, los proveedores de transporte TCP/IP que admiten multidifusión también deben proporcionar el estilo de distribución de software Berkeley (BSD): compatibilidad con multidifusión admitiendo las opciones de multidifusión correspondientes. Esto simplifica la migración de aplicaciones de multidifusión existentes a Windows Sockets 2.
