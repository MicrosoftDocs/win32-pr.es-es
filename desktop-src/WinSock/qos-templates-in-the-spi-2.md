# <a name="qos-templates-in-the-spi"></a>Plantillas de QoS en el SPI

Los detalles de implementación de calidad de servicio (QoS) se proporcionan en una sección de QoS dedicada y independiente en el kit de desarrollo de software (SDK) de Microsoft Windows. Consulte la sección [Quality of Service](/previous-versions/windows/desktop/qos/qos-start-page) de la Windows SDK para obtener información sobre las directrices de uso de QoS, incluidas las plantillas QoS disponibles y sus valores [**FLOWSPEC**](/windows/win32/api/qos/ns-qos-flowspec) asociados.
