# <a name="using-installer-functions"></a>Usar las funciones del instalador

Para agregar Windows Installer funcionalidad a la aplicación, use las funciones del servicio del instalador. Dado que el instalador es una utilidad de instalación y un sistema de administración de componentes, una aplicación que usa el instalador debe incorporar las funciones del servicio instalador como parte del funcionamiento normal.

En los siguientes temas y procedimientos se explica cómo implementar la funcionalidad del instalador en la aplicación:

-   [Inicializar una aplicación](initializing-an-application.md)
-   [Obtención de información de la aplicación](getting-application-information.md)
-   [Solicitar una característica](requesting-a-feature.md)
-   [Instalación de una aplicación](installing-an-application.md)
-   [Reinstalación de una característica o aplicación](reinstalling-a-feature-or-application.md)
-   [Quitar una aplicación anunciada](removing-an-advertised-application.md)
-   [Trabajar con características y componentes](working-with-features-and-components.md)
-   [Instalación de un componente que falta](installing-a-missing-component.md)
-   [Productos y revisiones de inventario](inventory-products-and-patches-.md)
-   [Administrar orígenes de instalación](managing-installation-sources.md)
