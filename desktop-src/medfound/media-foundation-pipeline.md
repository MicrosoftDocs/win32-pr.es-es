# <a name="media-foundation-pipeline"></a>Canalización de Media Foundation

La capa de *canalización* en Microsoft Media Foundation es el nivel de la arquitectura que genera o procesa directamente los datos multimedia.

La mayoría de las aplicaciones no llaman a métodos directamente en los componentes de canalización, aunque es posible hacerlo. Lea estos temas si está escribiendo un componente de canalización personalizado.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                     | Descripción                                                                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| [Orígenes multimedia](media-sources.md)<br/>                             | Los orígenes multimedia generan datos multimedia, normalmente desde un archivo, un dispositivo de captura o un flujo de red. <br/>                              |
| [Media Foundation transformaciones](media-foundation-transforms.md)<br/> | Media Foundation transformaciones (MFTs) procesan datos multimedia. Por ejemplo, los códecs de Media Foundation se implementan como MFTs. <br/>   |
| [Receptores de medios](media-sinks.md)<br/>                                 | Los receptores de medios consumen datos multimedia. Por ejemplo, un receptor de medios podría escribir los datos en un archivo o enviar los datos a través de una red.<br/> |
