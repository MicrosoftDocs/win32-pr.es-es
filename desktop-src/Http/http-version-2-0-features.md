# <a name="http-version-20-features"></a>Características de http versión 2.0

La siguiente característica es compatible con la API del servidor HTTP versión 2.0.

-   Objetos de configuración de grupo de direcciones URL y sesión de servidor
-   Propiedades de configuración expuestas en los objetos de configuración
-   Configuración y administración avanzada de colas de solicitudes
-   SSL en modo kernel completo
-   Caché de respuestas en modo kernel
-   Autenticación del lado servidor
-   Control de versiones con ámbito de objeto
-   Registro del lado servidor
-   Cierre correcto de la cola de solicitudes
-   Inicio de la demanda en la cola de solicitudes
-   Uso compartido de puertos
-   Seguimiento de eventos para Windows compatibilidad
-   Contadores de rendimiento
-   Compatibilidad con el nombre de dominio internacionalizado (IDN)
-   Utilidad de configuración de Net Shell
