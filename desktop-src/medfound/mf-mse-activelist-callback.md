# <a name="mf_mse_activelist_callback-attribute"></a>Atributo \_ MSE \_ ACTIVELIST \_ CALLBACK de MF

Contiene un puntero a la interfaz de devolución de llamada de la aplicación para la [**interfazBUFFERBufferListNotify**](/windows/desktop/api/mfmediaengine/nn-mfmediaengine-imfbufferlistnotify) para la lista de búferes activos.

## <a name="data-type"></a>Tipo de datos

**BUFFERBufferListNotify \* *_ almacenado como _* IUnknown\***
