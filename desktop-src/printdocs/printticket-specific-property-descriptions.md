# <a name="printticket-specific-property-descriptions"></a>Descripciones de propiedades específicas de PrintTicket

Este tema no es actual. Para obtener la información más reciente, vea [La especificación del esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

En esta sección se describen las palabras clave públicas que pueden aparecer en un PrintTicket.

No es necesario que se admiten todas las palabras clave. Los implementadores de PrintTicket PUEDEN admitir la funcionalidad solo si es aplicable a su propia implementación específica.

## <a name="element-type-implementations-for-printticket-specific-property-descriptions"></a>Implementaciones de tipo de elemento para descripciones de propiedades específicas de PrintTicket

Es importante prestar atención a los elementos Primario y Secundario permitidos en el contenido estructural de cada palabra clave. Para obtener más información, consulte la [sección Resumen de tipos de elementos.](./summary-of-element-types.md)
