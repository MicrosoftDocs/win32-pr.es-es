# <a name="distributing-user-interface-components"></a>Distribución de componentes de la interfaz de usuario

Los componentes COM, las aplicaciones o los archivos usados por una extensión de la interfaz de usuario se pueden distribuir mediante la creación de un paquete de Windows Installer y el uso de una directiva de grupo para distribuir el paquete. Para obtener más información acerca de la creación del paquete del instalador, consulte [Windows Installer](/windows/desktop/Msi/windows-installer-portal). Para obtener más información acerca de las directivas de grupo, consulte [Directiva de grupo](/previous-versions/windows/desktop/Policy/group-policy-start-page).
