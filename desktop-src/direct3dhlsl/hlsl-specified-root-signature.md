# <a name="hlsl-specified-root-signature"></a>Firma raíz de HLSL especificada

Una [firma raíz](/windows/desktop/direct3d12/root-signatures) (una tabla de claves de recursos y otros elementos de D3D12) se puede especificar en HLSL como una cadena.

Consulte los documentos de D3D12 sobre cómo [especificar firmas de raíz en HLSL](/windows/desktop/direct3d12/specifying-root-signatures-in-hlsl) para obtener información completa y código de ejemplo.
