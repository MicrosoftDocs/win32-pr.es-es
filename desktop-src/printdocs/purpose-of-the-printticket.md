# <a name="purpose-of-the-printticket"></a>Propósito del objeto PrintTicket

Este tema no está actualizado. Para obtener la información más reciente, consulte la [especificación del esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

El propósito principal del PrintTicket es expresar las configuraciones de los dispositivos. A diferencia del documento de PrintCapabilities, que define todas las configuraciones posibles que el dispositivo puede asumir (el espacio de configuración del dispositivo), el propósito de PrintTicket es definir una configuración específica que se aplica al objeto de trabajo (en este caso, la página) que se está procesando. Para promover la portabilidad y la coherencia de las configuraciones, las palabras clave y la jerarquía específicas que se presentan en la sección Print Schema Keywords se usan en las tecnologías PrintCapabilities y PrintTicket. Además, el [marco de trabajo de esquemas de impresión](print-schema-framework.md) define las construcciones básicas de ambas tecnologías, para promover la comunicación no ambigua, abierta y extensible del formato y la PrintCapabilities de los trabajos.
