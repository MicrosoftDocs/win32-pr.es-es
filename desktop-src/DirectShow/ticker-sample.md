# <a name="ticker-sample"></a>Ejemplo de ticker

## <a name="description"></a>Descripción

En este ejemplo se usa el representador de mezcla de vídeo para combinar vídeo y texto. Usa la interfaz [**IVMRMixerBitmap9**](/previous-versions/windows/desktop/api/Vmr9/nn-vmr9-ivmrmixerbitmap9) para combinar texto en la parte inferior de la ventana de vídeo.

La aplicación crea un mapa de bits con una cadena de texto predeterminada y lo desplaza por la parte inferior de la pantalla. Para cambiar el texto, elija **Establecer cadena de texto...** en el **menú Ticer.** Para establecer la fuente, elija **Establecer fuente...** en el **menú Ticer.** Si selecciona Usar **imagen estática en** el menú **Ticer,** la aplicación cambia de una cadena de texto dinámico a una imagen de mapa de bits estática.
