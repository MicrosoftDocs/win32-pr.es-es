# <a name="legacy-accessibility-and-automation-technology---msaa-to-ui-automation"></a>Tecnología de automatización y accesibilidad heredada: MSAA en automatización de la interfaz de usuario

La automatización y la accesibilidad de Windows se componen de dos tecnologías: Microsoft Active Accessibility (MSAA) y automatización de la interfaz de usuario de Microsoft. MSAA es una tecnología heredada incorporada con Windows 95, mientras que la automatización de la interfaz de usuario es una tecnología más reciente y más compatible que aborda las limitaciones de MSAA.

En esta sección se describen las principales diferencias entre MSAA y la automatización de la interfaz de usuario (y cómo pasar a la automatización de la interfaz de usuario).
