# <a name="remote-desktop-services-audioendpoint-enumeration-types"></a>Servicios de Escritorio remoto tipos de enumeración AudioEndpoint

Los siguientes tipos de enumeración se usan con la API de AudioEndpoint.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**\_marcas de posición de AE \_**](/windows/desktop/api/Audioengineendpoint/ne-audioengineendpoint-ae_position_flags)
</dt> <dd>

Define las constantes de la estructura de la [**\_ \_ posición actual de AE**](/windows/desktop/api/Audioengineendpoint/ns-audioengineendpoint-ae_current_position) . Estas constantes describen el grado de validez de la posición actual.

</dd> <dt>

[**\_marcas de búfer de APO \_**](/windows/desktop/api/Audioapotypes/ne-audioapotypes-apo_buffer_flags)
</dt> <dd>

Define las marcas de validación de búfer para la estructura de [**\_ \_ propiedades de conexión de APO**](/windows/desktop/api/Audioapotypes/ns-audioapotypes-apo_connection_property) asociada a cada conexión de APO.

</dd> </dl>

## <a name="remarks"></a>Observaciones

La API de Servicios de Escritorio remoto AudioEndpoint es para su uso en escenarios de Escritorio remoto; no es para las aplicaciones cliente.
