# <a name="d-rpc"></a>D (RPC)

[A](a-glos.md) [B](b-glos.md) [C](c-glos.md) D [E](e-glos.md) [F](f-glos.md) G H [I](i-glos.md) J K [L](l-glos.md) [M](m-glos.md) [N](n-glos.md) [o](o-glos.md) [p](p-glos.md) [Q](q.md) [R](r-glos.md) [s](s-glos.md) [T](t-glos.md) [U](u-glos.md) [V](v-glos.md) [W](w-glos.md) X Y Z

<dl> <dt>

<span id="_rpc_datagram_glos"></span><span id="_RPC_DATAGRAM_GLOS"></span>**diagrama**
</dt> <dd>

Protocolo de comunicaciones o transporte en el que los paquetes de datos se enrutan de forma independiente. Pueden seguir rutas diferentes y llegar en un orden diferente al que se enviaron. [*UDP*](u-glos.md) e [*IPX*](i-glos.md) son ejemplos de protocolos de datagramas de nivel de transporte. Vea también [*orientado a la conexión*](c-glos.md).

</dd> <dt>

<span id="_rpc_discriminant_glos"></span><span id="_RPC_DISCRIMINANT_GLOS"></span>**discriminante**
</dt> <dd>

Variable que especifica los tipos de datos que se pueden almacenar en una Unión.

</dd> <dt>

<span id="_rpc_discriminated_union_glos"></span><span id="_RPC_DISCRIMINATED_UNION_GLOS"></span>**Unión discriminada**
</dt> <dd>

(o *registro de variante*) Unión que incluye un discriminador como parte de la estructura de datos para que el tipo de datos válido actualmente se transmita junto con la Unión. Vea también [*Unión encapsulada*](e-glos.md), [*Unión no encapsulada*](n-glos.md).

</dd> <dt>

<span id="_rpc_dce_glos"></span><span id="_RPC_DCE_GLOS"></span>**Entorno de computación distribuida (DCE)**
</dt> <dd>

Especificación de la base de software abierto para un conjunto de servicios integrados, incluidas las llamadas a procedimientos remotos, los sistemas de archivos distribuidos y los servicios de seguridad. El estándar RPC de OSF-DCE es la base de Microsoft RPC.

</dd> <dt>

<span id="_rpc_dynamic_endpoint_glos"></span><span id="_RPC_DYNAMIC_ENDPOINT_GLOS"></span>**punto de conexión dinámico**
</dt> <dd>

Específico de la red: dirección del servidor que se solicita y se asigna en tiempo de ejecución. Consulte también [*punto de conexión conocido*](w-glos.md).

</dd> <dt>

<span id="_rpc_dynamic_identity_tracking_glos"></span><span id="_RPC_DYNAMIC_IDENTITY_TRACKING_GLOS"></span>**seguimiento de identidad dinámica**
</dt> <dd>

Especificación de que la biblioteca en tiempo de ejecución de RPC utilizará las credenciales del subproceso que realiza la llamada, en lugar del identificador de enlace, para la autenticación cada vez que el cliente llame a un procedimiento remoto. Vea también [*seguimiento de identidad estático*](s-glos.md).

</dd> </dl>
