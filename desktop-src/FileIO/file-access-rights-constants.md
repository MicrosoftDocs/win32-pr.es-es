# <a name="file-access-rights-constants"></a>Constantes de derechos de acceso a archivos

Los derechos de acceso válidos para archivos y directorios incluyen los derechos de acceso estándar **DELETE**, **READ \_ CONTROL**, **WRITE \_ DAC,** **WRITE \_ OWNER** [y](/windows/desktop/SecAuthZ/standard-access-rights) **SYNCHRONIZE** . En la tabla siguiente se enumeran los derechos de acceso específicos de los archivos y directorios.

|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="FILE_ADD_FILE"></span><span id="file_add_file"></span><dl> <dt>**FILE \_ AGREGAR \_ ARCHIVO**</dt> <dt>2</dt> </dl>                                      | Para un directorio, el derecho para crear un archivo en el directorio.<br/>                                                                                                                                                                                                                                                                                                           |
| <span id="FILE_ADD_SUBDIRECTORY"></span><span id="file_add_subdirectory"></span><dl> <dt>**FILE \_ AGREGAR \_ SUBDIRECTORIO**</dt> <dt>4</dt> </dl>              | Para un directorio, el derecho para crear un subdirectorio.<br/>                                                                                                                                                                                                                                                                                                                    |
| <span id="FILE_ALL_ACCESS"></span><span id="file_all_access"></span><dl> <dt>**FILE \_ TODOS \_ LOS ACCESOS**</dt><dt></dt> </dl>                                 | Todos los derechos de acceso posibles para un archivo.<br/>                                                                                                                                                                                                                                                                                                                                  |
| <span id="FILE_APPEND_DATA"></span><span id="file_append_data"></span><dl> <dt>**FILE \_ APPEND \_ DATA**</dt> <dt>4</dt> </dl>                             | Para un objeto de archivo, derecho para anexar datos al archivo. (En el caso de los archivos locales, las operaciones de escritura no sobrescribirán los datos existentes si esta marca se especifica sin **FILE \_ WRITE \_ DATA).** Para un objeto de directorio, el derecho para crear un subdirectorio (**FILE \_ ADD \_ SUBDIRECTORY**).<br/>                                                                                        |

| <span id="FILE_DELETE_CHILD"></span><span id="file_delete_child"></span><dl> <dt>**FILE \_ DELETE \_ CHILD**</dt> <dt>64 (0x40)</dt> </dl>                  | Para un directorio, el derecho a eliminar un directorio y todos los archivos que contiene, incluidos los archivos de solo lectura.<br/>                                                                                                                                                                                                                                                              |

| <span id="FILE_READ_DATA"></span><span id="file_read_data"></span><dl> <dt>**FILE \_ READ \_ DATA**</dt> <dt>1</dt> </dl>                                   | Para un objeto de archivo, el derecho para leer los datos de archivo correspondientes.
