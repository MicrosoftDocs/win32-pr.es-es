|-------------------|--------------------------------------|
| CN                | Name-Service-Flags                   |
| Ldap-Display-Name | nameServiceFlags                     |
| Size              | 4 bytes                              |
| Privilegio actualizar  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.753               |
| System-Id-Guid    | 80212840-4bdc-11d1-a9c4-0000f80367c1 |
