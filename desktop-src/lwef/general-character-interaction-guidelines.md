# <a name="general-character-interaction-guidelines"></a>Directrices generales de interacción de caracteres

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Los principios de buen diseño de la interfaz de usuario también se aplican al diseño de una interfaz mediante Microsoft Agent. Debe comprender a los usuarios y sus objetivos, así como seguir un buen proceso de diseño de la interfaz de usuario. Las Windows interfaces de diseño de software proporcionan una excelente introducción a los principios generales de diseño y la metodología.

-   [Proporcionar los comentarios adecuados](provide-appropriate-feedback.md)
-   [Uso de variación natural](use-natural-variation.md)
