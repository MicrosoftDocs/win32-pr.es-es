# <a name="accessing-microsoft-agent-services-directly"></a>Acceso directo a los servicios de Microsoft Agent

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Si usa C, C++ o Java, puede acceder al servidor de Microsoft Agent directamente mediante sus interfaces ActiveX (OLE). Para obtener más información sobre estas interfaces, vea [Programming the Microsoft Agent Server Interface](programming-the-microsoft-agent-server-interface.md).
