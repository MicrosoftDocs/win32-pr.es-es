# <a name="understanding-internationalization"></a><span data-ttu-id="a67a9-103">Descripción de la internacionalización</span><span class="sxs-lookup"><span data-stu-id="a67a9-103">Understanding Internationalization</span></span>

<span data-ttu-id="a67a9-104">El desarrollo de productos de software internacionalizados no solo requiere la producción de código correcto, sino también un diseño que permita que el producto admita muchas configuraciones regionales.</span><span class="sxs-lookup"><span data-stu-id="a67a9-104">Developing internationalized software products requires not only the production of correct code, but also a design that permits the product to support many locales.</span></span> <span data-ttu-id="a67a9-105">Los desarrolladores y sus administradores deben tener en cuenta el nivel de esfuerzo y la atención de los detalles necesarios para crear aplicaciones de uso internacional, tanto si las entregan como archivos binarios únicos listos para usarse en muchos mercados diferentes como si son varios binarios que son específicos de una configuración regional.</span><span class="sxs-lookup"><span data-stu-id="a67a9-105">Developers and their managers should be aware of the level of effort and attention to detail required to create world-ready applications, whether they deliver them as single binaries ready for use in many different markets, or as multiple binaries that are each specific to one locale.</span></span> <span data-ttu-id="a67a9-106">Como desarrollador, asegúrese de que su administración entiende que la internacionalización implica algo más que el desarrollo de código.</span><span class="sxs-lookup"><span data-stu-id="a67a9-106">As a developer, make sure your management understands that internationalization involves more than just code development.</span></span> <span data-ttu-id="a67a9-107">El diseño para la internacionalización al principio del ciclo del producto le ahorra tiempo, dinero y esfuerzo.</span><span class="sxs-lookup"><span data-stu-id="a67a9-107">Designing for internationalization at the beginning of your product cycle saves you time, money, and effort.</span></span>

<span data-ttu-id="a67a9-108">**Internacionalización** es la creación de software que se adapta a las configuraciones regionales en todo el mundo.</span><span class="sxs-lookup"><span data-stu-id="a67a9-108">**Internationalization** is the creation of software that adapts to locales around the world.</span></span> <span data-ttu-id="a67a9-109">Una **configuración regional** es una colección de preferencias de usuario relacionadas con el lenguaje.</span><span class="sxs-lookup"><span data-stu-id="a67a9-109">A **locale** is a collection of language-related user preferences.</span></span> <span data-ttu-id="a67a9-110">La configuración regional se especifica mediante el emparejamiento de un idioma y una región geográfica.</span><span class="sxs-lookup"><span data-stu-id="a67a9-110">A locale is specified by the pairing of a language and a geographical region.</span></span> <span data-ttu-id="a67a9-111">Por ejemplo, Inglés (Estados Unidos) e inglés (Reino Unido) son dos configuraciones regionales diferentes, como inglés (Canadá) y francés (Canadá).</span><span class="sxs-lookup"><span data-stu-id="a67a9-111">For example, English (United States) and English (United Kingdom) are two different locales, as are English (Canada) and French (Canada).</span></span>

<span data-ttu-id="a67a9-112">El software internacionalizado tiene estos atributos:</span><span class="sxs-lookup"><span data-stu-id="a67a9-112">Internationalized software has these attributes:</span></span>

-   <span data-ttu-id="a67a9-113">**La preparación mundial** cubre el diseño y la implementación que separa el código independiente de la configuración regional de los recursos dependientes de la configuración regional y consta de dos áreas principales:</span><span class="sxs-lookup"><span data-stu-id="a67a9-113">**World-readiness** covers design and implementation that separate locale-independent code from locale-dependent resources, and comprises two major areas:</span></span>
    -   <span data-ttu-id="a67a9-114">La **globalización** es el proceso de desarrollar un núcleo de programa con características y diseño de código que son independientes del idioma o la configuración regional.</span><span class="sxs-lookup"><span data-stu-id="a67a9-114">**Globalization** is the process of developing a program core with features and code design that are independent of language or locale.</span></span> <span data-ttu-id="a67a9-115">En su lugar, el diseño es compatible con la entrada, la presentación y la salida de los scripts de lenguaje compatibles con Unicode, así como los datos relacionados con las configuraciones regionales específicas.</span><span class="sxs-lookup"><span data-stu-id="a67a9-115">Instead, the design supports the input, display, and output of Unicode-supported language scripts, as well as data related to specific locales.</span></span>
    -   <span data-ttu-id="a67a9-116">La **localizabilidad** es el diseño de la base de código de software y los recursos de modo que se pueda localizar un programa, como se define a continuación, en ediciones de idioma diferentes sin necesidad de realizar cambios en el código fuente.</span><span class="sxs-lookup"><span data-stu-id="a67a9-116">**Localizability** is the design of the software code base and resources such that a program can be localized, as defined below, into different language editions without any changes to the source code.</span></span> <span data-ttu-id="a67a9-117">Por ejemplo, los búferes de cadena en el código y los cuadros de texto de la interfaz de usuario deben ser lo suficientemente grandes como para admitir cadenas de texto más largas en idiomas como el alemán o el holandés.</span><span class="sxs-lookup"><span data-stu-id="a67a9-117">For example, string buffers in the code, and text boxes in the UI, must be large enough to accommodate longer text strings in languages such as German or Dutch.</span></span>
-   <span data-ttu-id="a67a9-118">**Localización** de Esto implica traducir y personalizar un producto para un mercado específico, y es principalmente la creación de archivos de recursos en lugar de escribir código fuente.</span><span class="sxs-lookup"><span data-stu-id="a67a9-118">**Localization** This involves translating and customizing a product for a specific market, and it is mainly about creating resource files rather than writing source code.</span></span>

<span data-ttu-id="a67a9-119">Por ejemplo, el uso de la compatibilidad con el idioma nacional (NLS) que proporciona la API Win32 de Microsoft es un proceso de desarrollo de software que admite la preparación mundial, mientras que la modificación de los elementos de la interfaz de usuario, la traducción de texto y la estandarización de la terminología son pasos de localización que suele realizar alguien que es un especialista en lenguaje, como un traductor.</span><span class="sxs-lookup"><span data-stu-id="a67a9-119">For example, using the National Language Support (NLS) supplied by the Microsoft Win32 API is a software development process that supports world-readiness, whereas modifying the UI elements, translating text, and standardizing terminology are localization steps usually performed by someone who is a language specialist, such as a translator.</span></span> <span data-ttu-id="a67a9-120">Incluso cuando el desarrollo de código enfatiza la preparación mundial, debe comprender la localización porque las decisiones de diseño afectan al trabajo del traductor.</span><span class="sxs-lookup"><span data-stu-id="a67a9-120">Even when your code development emphasizes world-readiness, you need to understand localization because your design decisions affect the work of the translator.</span></span>
