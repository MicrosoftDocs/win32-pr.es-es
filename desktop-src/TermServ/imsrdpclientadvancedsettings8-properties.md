# <a name="imsrdpclientadvancedsettings8-properties"></a>Propiedades de IMsRdpClientAdvancedSettings8

La interfaz [**IMsRdpClientAdvancedSettings8**](imsrdpclientadvancedsettings8.md) expone las siguientes propiedades.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**Propiedad BandwidthDetection**](imsrdpclientadvancedsettings8-bandwidthdetection.md)
</dt> <dd>

Especifica si los cambios de ancho de banda se detectan automáticamente.

</dd> <dt>

[**Propiedad ClientProtocolSpec**](imsrdpclientadvancedsettings8-clientprotocolspec.md)
</dt> <dd>

Especifica el protocolo de escritorio remoto utilizado entre el cliente y el servidor.

</dd> </dl>
