# <a name="about-joysticks"></a>Acerca de los joysticks

El joystick es un dispositivo de entrada auxiliar para aplicaciones que proporcionan alternativas al uso del teclado y el mouse. El joystick proporciona información posicional dentro de un sistema de coordenadas que tiene valores máximos y mínimos absolutos en cada eje de movimiento.

Los servicios de joystick se cargan cuando se inicia el sistema operativo. Los servicios de joystick pueden supervisar simultáneamente dos joysticks, cada uno con un movimiento de dos o tres ejes. Cada joystick puede tener hasta cuatro botones. Puede usar las funciones de joystick para determinar las capacidades de los joysticks y del controlador de joystick. Además, puede procesar la información posicional y de botón de un joystick consultando el joystick directamente o capturando el joystick y procesando los mensajes desde él. El último método es más sencillo porque la aplicación no tiene que consultar manualmente el joystick ni realizar un seguimiento del tiempo necesario para generar consultas a intervalos regulares.
