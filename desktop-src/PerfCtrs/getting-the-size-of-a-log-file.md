# <a name="getting-the-size-of-a-log-file"></a>Obtención del tamaño de un archivo de registro

[**PdhGetLogFileSize**](/windows/desktop/api/Pdh/nf-pdh-pdhgetlogfilesize) devuelve el tamaño, en bytes, del archivo de registro especificado. Utilice esta función cuando necesite asignar memoria para un búfer que recibirá el contenido del archivo de registro.
