# <a name="conversion-of-mts-packages-to-com-applications"></a>Conversión de paquetes MTS en aplicaciones COM+

COM+ se puede ver como una combinación de Microsoft Transaction Server (MTS) y COM, junto con la introducción de nuevas características.

En su mayor parte, los paquetes MTS se convierten en aplicaciones COM+ durante el procedimiento de instalación de Windows. Sin hacer nada más allá de la configuración típica, puede aprovechar todas las nuevas características de COM+. Para obtener una lista y una breve explicación de las mejoras más importantes de COM+, consulte [información general sobre la programación de com+](com--programming-overview.md).

Si desea obtener más información sobre la conversión automática de paquetes MTS en aplicaciones COM+ durante el proceso de instalación de Windows, así como sobre cómo solucionar los errores que puedan producirse durante este proceso, consulte [conversión automática desde mts](automatic-conversion-from-mts.md).

Si desea obtener información sobre cómo convertir manualmente paquetes MTS en aplicaciones COM+, consulte [conversión manual desde mts](manual-conversion-from-mts.md).

También debe tener en cuenta que la instalación de COM+ presenta varias funciones nuevas que los paquetes MTS pueden utilizar como aplicaciones COM+. Estos se explican en [problemas y resultados](com--conversion-results-and-issues.md)de la conversión de com+.
