# <a name="xmubyte4-operators"></a>Operadores XMUBYTE4

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                               | Descripción                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operador =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmubyte4-operator-assign)<br/> | Asigna los datos del componente de vector de una instancia de `XMUBYTE4` a la instancia actual de `XMUBYTE4` . <br/> |
