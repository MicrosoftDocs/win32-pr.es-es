# <a name="externaltask"></a>External. Task

> [!Note]  
> En este tema se describe la funcionalidad diseñada para su uso en tiendas en línea. No se admite el uso de esta funcionalidad fuera del contexto de una tienda en línea.

 

La propiedad **Task** recupera el nombre del panel de tareas actual.

## <a name="syntax"></a>Sintaxis

``` syntax
window.external.task
```

## <a name="possible-values"></a>Valores posibles

Esta propiedad es una **cadena** de solo lectura. Los valores posibles son examinar, grabar y sincronizar.
