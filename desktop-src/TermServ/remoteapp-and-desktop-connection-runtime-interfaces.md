# <a name="remoteapp-and-desktop-connection-runtime-interfaces"></a>Interfaces en tiempo de ejecución Conexión de RemoteApp y Escritorio

La API de tiempo de ejecución de Conexión de RemoteApp y Escritorio admite interfaces que permiten el desarrollo de clientes personalizados en Conexión de RemoteApp y Escritorio.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**IWorkspace**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspace)
</dt> <dd>

Expone métodos que proporcionan información sobre una conexión en Conexión de RemoteApp y Escritorio.

</dd> <dt>

[**IWorkspace2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspace2)
</dt> <dd>

Expone métodos adicionales que proporcionan información sobre una conexión en Conexión de RemoteApp y Escritorio.

</dd> <dt>

[**IWorkspace3**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspace3)
</dt> <dd>

Expone métodos que proporcionan información sobre una conexión en Conexión de RemoteApp y Escritorio y agregan la capacidad de recuperar o establecer un token de notificaciones.

</dd> <dt>

[**IWorkspaceClientExt**](/windows/desktop/api/Workspaceruntimeclientext/nn-workspaceruntimeclientext-iworkspaceclientext)
</dt> <dd>

Expone métodos que permiten al tiempo de ejecución desconectar un cliente personalizado en Conexión de RemoteApp y Escritorio.

</dd> <dt>

[**IWorkspaceRegistration**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspaceregistration)
</dt> <dd>

Expone métodos que agregan y quitan referencias a clientes personalizados en Conexión de RemoteApp y Escritorio.

</dd> <dt>

[**IWorkspaceRegistration2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspaceregistration2)
</dt> <dd>

Expone métodos que agregan y quitan referencias a clientes personalizados en Conexión de RemoteApp y Escritorio.

</dd> <dt>

[**IWorkspaceReportMessage**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacereportmessage)
</dt> <dd>

Expone métodos que admiten el control de mensajes de error para áreas de trabajo remotas.

</dd> <dt>

[**IWorkspaceResTypeRegistry**](/windows/desktop/api/Workspaceax/nn-workspaceax-iworkspacerestyperegistry)
</dt> <dd>

Expone métodos que permiten a un complemento administrar extensiones de nombre de archivo de terceros en Conexión de RemoteApp y Escritorio en tiempo de ejecución.

</dd> <dt>

[**IWorkspaceScriptable**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable)
</dt> <dd>

Expone métodos que administran Conexión de RemoteApp y Escritorio las credenciales y las conexiones.

</dd> <dt>

[**IWorkspaceScriptable2**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable2)
</dt> <dd>

Expone métodos que administran Conexión de RemoteApp y Escritorio las credenciales y las conexiones.

</dd> <dt>

[**IWorkspaceScriptable3**](/windows/desktop/api/workspaceruntime/nn-workspaceruntime-iworkspacescriptable3)
</dt> <dd>

Expone métodos que administran Conexión de RemoteApp y Escritorio las credenciales y las conexiones.

</dd> </dl>
