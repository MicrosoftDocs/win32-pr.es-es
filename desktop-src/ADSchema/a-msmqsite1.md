# <a name="msmq-site-1-attribute"></a>Atributo MSMQ-site-1

DN del primer sitio de un par que está conectado.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MSMQ-sitio-1                             |
| Nombre para mostrar de LDAP | mSMQSite1                               |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | Cada vez que se crea un MSMQ-sitio-línea. |
| Attribute-Id      | 1.2.840.113556.1.4.943                  |
| System-ID-GUID    | 9a0dc337-c100-11d1-bbc5-0080c76670c0    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
