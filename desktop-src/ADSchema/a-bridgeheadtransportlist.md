# <a name="bridgehead-transport-list-attribute"></a>Atributo Bridgehead-Transport-List

Transportes para los que este servidor es una punta de puente.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | Bridgehead-Transport-List               |
| Ldap-Display-Name | bridgeheadTransportList                 |
| Size              | \-                                      |
| Actualizar privilegios  | El sistema establece este valor.        |
| Frecuencia de actualización  | Cada vez que se configura un sitio.               |
| Attribute-Id      | 1.2.840.113556.1.4.819                  |
| System-Id-Guid    | d50c2cda-8951-11d1-aebc-0000f80367c1    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
