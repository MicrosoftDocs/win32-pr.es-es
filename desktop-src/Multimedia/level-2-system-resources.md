# <a name="level-2-system-resources"></a>Recursos del sistema de nivel 2

La configuración mínima de un equipo para satisfacer la especificación de nivel 2 incluye los siguientes elementos:

-   Un microprocesador 486SX de 25 MHz con 4 MB de RAM
-   Una unidad de disquete de alta densidad de 3,5 pulgadas
-   Una unidad de disco duro con al menos 160 MB de espacio en disco
-   Un teclado de clave 101 con un conector DIN estándar o uno que proporciona una funcionalidad idéntica mediante el uso de combinaciones de teclas
-   Un mouse de dos botones con un conector de serie o de bus
-   Un puerto MIDI (interfaz digital de instrumentos musicales) que incluye el dispositivo MIDI Out, MIDI in y MIDI through, y que tiene compatibilidad de interrupción para la transferencia de entrada y FIFO
-   Un puerto de joystick analógico o digital (juego) de estilo IBM
-   Monitor de color con una resolución de pantalla de 640 x 480 píxeles con colores de 65.536
-   Software del sistema que ofrece compatibilidad binaria con Windows 3,0 o Windows 3,1

Además, el objetivo de rendimiento recomendado para el adaptador de vídeo (VGA +) es de 1,2 millones píxeles por segundo, con un 40 por ciento o menos de ancho de banda de la CPU. Los mapas de bits independientes del dispositivo (DIB) que se usan para medir este objetivo de rendimiento tienen profundidades de color de 1, 4 y 8 bits, y pueden usar la codificación de longitud de ejecución o ser descodificados. Un segundo método para medir el rendimiento del vídeo es ofrecer imágenes de vídeo digital de 256, 320 por 240 en 15 fotogramas por segundo.
