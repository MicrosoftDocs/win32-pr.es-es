# <a name="wmmediaisdelay"></a>WM/MediaIsDelay

El atributo **WM/MediaIsDelay** indica si el contenido se difundió como retraso de cinta.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaIsDelay

## <a name="data-type"></a>Tipo de datos

**tipo de WMT \_ \_ bool**

## <a name="remarks"></a>Observaciones

Este atributo se utiliza para la televisión de difusión.
