# <a name="change-domain-master-extended-right"></a>Derecho extendido change-domain-master

|--------------|--------------------------------------|
| CN           | Change-Domain-Master                 |
| Display-Name | Cambiar maestro de dominio                 |
| Rights-GUID  | 014bf69c-7b3b-11d1-85f6-08002be74fab |
