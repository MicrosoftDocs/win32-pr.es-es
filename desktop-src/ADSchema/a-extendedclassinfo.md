# <a name="extended-class-info-attribute"></a>Atributo extendido-Class-info

Una propiedad con varios valores que contiene cadenas que representan información adicional para cada clase. Cada valor contiene los governsID, lDAPDisplayName y schemaIDGUID de la clase.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Extended-Class-info                         |
| Nombre para mostrar de LDAP | extendedClassInfo                           |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.908                      |
| System-ID-GUID    | 9a7ad948-ca53-11d1-bbd0-0080c76670c0        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
