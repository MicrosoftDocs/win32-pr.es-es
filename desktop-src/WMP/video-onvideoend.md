# <a name="videoonvideoend"></a>VIDEO.onvideoend

El **controlador de eventos onvideoend** controla un evento que tiene lugar cuando el vídeo deja de representarse y se descarga.

``` syntax

onvideoend 
```

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
