# <a name="comment-attribute"></a>Atributo comment

El atributo **comment** especifica un comentario definido por el usuario.

## <a name="applies-to"></a>Se aplica a

-   [Elementos de fotografía](photo-item-attributes.md)

## <a name="remarks"></a>Observaciones

Para determinar si puede cambiar el valor de este atributo, use el método [media. isReadOnlyItem](media-isreadonlyitem.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------|
| Versión<br/> | Windows Media Player 10 o posterior<br/> |
