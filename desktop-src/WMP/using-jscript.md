# <a name="using-jscript"></a>Uso de JScript

Reproductor de Windows Media requiere Microsoft Internet Explorer versión 4 o posterior. Al escribir código, no suponga que el equipo del usuario contendrá una versión de Microsoft JScript posterior a la 4.0.

En las secciones siguientes se proporcionan detalles sobre el uso de JScript en máscaras:
