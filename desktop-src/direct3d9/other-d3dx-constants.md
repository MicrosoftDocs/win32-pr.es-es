# <a name="other-d3dx-constants"></a>Otras constantes de D3DX

Otras constantes de D3DX son las siguientes:

## <a name="16-bit-floating-point-numbers"></a>Números de punto flotante de 16 bits



|                         |                                         |
|-------------------------|-----------------------------------------|
| \#define                | Descripción                             |
| 16F de D3DX \_ \_          | Número de dígitos decimales de precisión   |
| D3DX \_ 16F \_ épsilon      | Menor que 1,0 + épsilon! = 1,0 |
| \_16F \_ mant de \_ D3DX    | Número de bits en mantisa              |
| D3DX \_ 16F \_ Max          | Valor máximo                           |
| D3DX \_ 16F \_ Max \_ 10 \_ exp | Máximo exponente decimal                |
| D3DX \_ 16F \_ Max \_ exp     | Máximo exponente binario                 |
| D3DX \_ 16F \_ min          | Valor positivo mínimo                  |
| D3DX \_ 16F \_ min \_ 10 \_ exp | Mínimo exponente decimal                |
| D3DX \_ 16F \_ min \_ exp     | Mínimo exponente binario                 |
| Base de D3DX \_ 16F \_        | Base de exponente                          |
| 16F de D3DX \_ \_       | Adición de redondeo: cerca                 |
| D3DX \_ 1BYPI             | 1/pi                                    |
| D3DX \_ PI                | pi                                      |
| \_Valor Float predeterminado de D3DX \_    | Valor Float máximo                     |



 

Estas \# definiciones se declaran en d3dx9. h y d3dx9math. h.

## <a name="prt-constants"></a>Constantes de PRT



|                  |                                            |
|------------------|--------------------------------------------|
| \#define         | Descripción                                |
| D3DXSH \_ MINORDER | Orden mínimo permitido de la simulación.  |
| D3DXSH \_ MAXORDER | Orden máximo permitido de la simulación. |



 

Estas \# definiciones se declaran en d3dx9math. h. Para obtener más información sobre PRT, consulte [transferencia Radiance precalculada (Direct3D 9)](precomputed-radiance-transfer.md).

## <a name="texture-constants"></a>Constantes de textura



|                        |                                                                    |
|------------------------|--------------------------------------------------------------------|
| \#define               | Descripción                                                        |
| D3DFMT \_ desde \_ archivo     | Tome el formato exacto de un archivo.                               |
| \_Configuración predeterminada de D3DX          | Un valor predeterminado.                                                   |
| \_NONPOW2 predeterminado de D3DX \_ | No redondear números como el ancho o el alto a una potencia de dos. |
| D3DX \_ desde \_ archivo       | Tomar las dimensiones de textura exactamente desde un archivo.                   |



 

Estas \# definiciones se declaran en d3dx9. h.

## <a name="other-d3dx-constants"></a>Otras constantes de D3DX

Las \_ constantes de versión UNUSED16 y D3DX definidas en D3dx9mesh. h y D3dx9core. h se usan internamente. No utilice estas constantes.
