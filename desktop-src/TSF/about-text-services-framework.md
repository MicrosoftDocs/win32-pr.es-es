# <a name="about-text-services-framework"></a>Acerca del marco de trabajo de servicios de texto

Microsoft Windows Text Services Framework (TSF) es una interfaz de programación de aplicaciones (API) que permite la entrada de texto avanzada y independiente del origen. TSF expone su marco de trabajo a aplicaciones y servicios de texto a través de un conjunto de interfaces COM.
