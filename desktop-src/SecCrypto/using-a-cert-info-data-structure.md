# <a name="using-a-cert_info-data-structure"></a>Usar una estructura de datos de información de certificado \_

[**Certificado \_ de El contexto**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_context) y la [**\_ información de certificado**](/windows/desktop/api/Wincrypt/ns-wincrypt-cert_info) son las estructuras de datos principales que contienen la información de un certificado. Cuando se usa un certificado, es posible que deba descodificarse. Para obtener más información, vea [descodificar una \_ estructura de información de certificado](decoding-a-cert-info-structure.md). Se puede agregar información a una estructura de **\_ información de certificado** descodificada y, a continuación, se puede volver a codificar la estructura. Para obtener más información, consulte [codificación de una \_ estructura de información de certificado](encoding-a-cert-info-structure.md).
