# <a name="ws_heap"></a>\_montón WS

Tipo opaco que se usa para hacer referencia a un objeto [heap](heap.md) .


```C++
typedef struct _WS_HEAP WS_HEAP;
```



## <a name="remarks"></a>Observaciones

Este objeto no es seguro para subprocesos. Para obtener más información, vea [seguridad para subprocesos](thread-safety.md).
