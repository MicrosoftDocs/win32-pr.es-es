# <a name="spell-checking-api"></a>Revisión ortográfica de la API

La API de revisión ortográfica permite a los desarrolladores consumir la capacidad del corrector ortográfico para comprobar el texto, obtener sugerencias y mantener la configuración y los diccionarios de usuario. Los desarrolladores también pueden crear comprobadores ortográficos que estén integrados en el marco de revisión ortográfica como proveedores.

-   [Acerca de la API de revisión ortográfica](about-the-spell-checker-api.md)
-   [Referencia de la API de revisión ortográfica](spell-checker-api-reference.md)
