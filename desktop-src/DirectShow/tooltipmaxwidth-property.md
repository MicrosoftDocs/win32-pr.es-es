# <a name="tooltipmaxwidth-property"></a>Propiedad ToolTipMaxWidth

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

La `ToolTipMaxWidth` propiedad establece o recupera el ancho máximo para la información sobre herramientas asociada al objeto **MSWebDVD** .

``` syntax
[ iMaxWidth = ] MSWebDVD.ToolTipMaxWidth
```

## <a name="return-value"></a>Valor devuelto

Devuelve el ancho máximo de la información sobre herramientas de la aplicación como un entero.

## <a name="remarks"></a>Observaciones

Esta propiedad es de lectura/escritura con un valor predeterminado de 200 píxeles.
