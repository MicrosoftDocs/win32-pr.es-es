# <a name="hcryptprov_legacy"></a>HCRYPTPROV \_ heredado

El tipo de datos **\_ heredado HCRYPTPROV** se usa para reemplazar el tipo de datos [**HCRYPTPROV**](hcryptprov.md) en el que ya no se usa el tipo de datos **HCRYPTPROV** . Los parámetros de función y las estructuras de datos que son de tipo **HCRYPTPROV \_ Legacy** deben establecerse en **null**.


```C++
typedef ULONG_PTR HCRYPTPROV_LEGACY;
```
