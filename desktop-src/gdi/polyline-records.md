# <a name="polyline-records"></a>Registros de polilínea

Los registros de [polilínea](/windows/desktop/api/Wingdi/nf-wingdi-polyline) son una serie de puntos; las líneas dibujadas entre los puntos describen el contorno del carácter. Un registro de polilínea comienza con el último punto del registro anterior (o, para el primer registro del contorno, el punto inicial). Cada punto del registro se encuentra en el contorno del glifo y se puede conectar simplemente mediante líneas rectas.
