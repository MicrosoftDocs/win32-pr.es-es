# <a name="jet_indexid-structure"></a>Estructura de JET_INDEXID

Contiene un identificador de índice. Un ID. de índice es una sugerencia que se usa para acelerar la selección del índice actual mediante JetSetCurrentIndex. Resulta muy útil cuando hay un gran número de índices en una tabla. El identificador de índice se puede recuperar mediante JetGetIndexInfo o JetGetTableIndexInfo.
