# <a name="ms-ds-nc-repl-inbound-neighbors-attribute"></a>Atributo ms-DS-NC-Repl-Inbound-Neighbors

Asociados de replicación para esta partición. Este servidor obtiene los datos de replicación de estos otros servidores, que actúan como orígenes.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-NC-Repl-Inbound-Neighbors             |
| Ldap-Display-Name | msDS-NCReplInboundNeighbors                 |
| Size              | \-                                          |
| Actualizar privilegios  | El sistema establece este valor.            |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1705                     |
| System-Id-Guid    | 9edba85a-3e9e-431b-9b1a-a5b6e9eda796        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
