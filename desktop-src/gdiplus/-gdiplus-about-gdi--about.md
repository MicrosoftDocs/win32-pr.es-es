# <a name="about-gdi"></a>Acerca de GDI+

Windows GDI+ es la parte del sistema operativo Windows XP o del sistema operativo Windows Server 2003 que proporciona gráficos vectoriales bidimensionales, imágenes y tipografía. GDI+ mejora en Windows Interfaz de dispositivo gráfico (GDI) (la interfaz de dispositivo gráfico incluida en versiones anteriores de Windows) agregando nuevas características y optimizando las características existentes.

En los temas siguientes se proporciona información acerca de la API GDI+ con el lenguaje de programación C++.

-   [Novedades de GDI+](-gdiplus-what-s-new-in-gdi--about.md)
-   [Líneas, curvas y formas](-gdiplus-lines-curves-and-shapes-about.md)
-   [Imágenes, mapas de bits y metarchivos](-gdiplus-images-bitmaps-and-metafiles-about.md)
-   [Sistemas de coordenadas y transformaciones](-gdiplus-coordinate-systems-and-transformations-about.md)
-   [Contenedores de gráficos](-gdiplus-graphics-containers-about.md)
