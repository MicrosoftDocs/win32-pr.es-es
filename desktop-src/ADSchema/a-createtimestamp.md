# <a name="create-time-stamp-attribute"></a>Create-Time-Stamp (atributo)

Fecha en que se creó este objeto. Este valor se replica.



| Entrada | Value |
|-------------------|---------------------------------------------------------------|
| CN                | Creación: marca de tiempo                                             |
| Nombre para mostrar de LDAP | createTimeStamp                                               |
| Tamaño              | 8 bytes                                                       |
| Actualizar privilegio  | El sistema establece este valor.                              |
| Frecuencia de actualización  | Cuando se crea el objeto.                                   |
| Attribute-Id      | 2.5.18.1                                                      |
| System-ID-GUID    | 2df90d73-009f-11d2-aa4c-00c04fd7d83a                          |
| Sintaxis            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
