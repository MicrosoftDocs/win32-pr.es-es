# <a name="retrieving-object-properties-overview"></a>Información general sobre la recuperación de propiedades del objeto

Una aplicación puede recuperar propiedades para un solo objeto, para varios objetos identificados por identificadores de objeto o para varios objetos del mismo tipo. En el siguiente tema se describe la recuperación de un solo objeto. Consulte la sección [tareas avanzadas](advanced-tasks.md) para obtener una descripción de la recuperación de propiedades de varios objetos.
