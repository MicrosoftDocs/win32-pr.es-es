# <a name="inputtype-complex-type"></a>Tipo complejo InputType

Define un tipo de datos de entrada.

``` syntax
<xs:complexType name="InputType">
    <xs:sequence>
        <xs:element name="description"
            type="string"
         />
        <xs:element name="outType"
            type="OutputType"
            minOccurs="0"
            maxOccurs="unbounded"
         />
        <xs:any
            processContents="lax"
            namespace="##other"
         />
    </xs:sequence>
    <xs:attribute name="name"
        type="QName"
        use="required"
     />
    <xs:attribute name="value"
        type="string"
        use="required"
     />
    <xs:attribute name="symbol"
        type="CSymbolType"
        use="required"
     />
</xs:complexType>
```

## <a name="child-elements"></a>Elementos secundarios



| Elemento                                                                  | Tipo                                                             | Descripción                                                                                                                                                                                                                                                                                                                                                                                                   |
|--------------------------------------------------------------------------|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Descripción**](eventmanifestschema-description-inputtype-element.md) | string                                                           | Descripción del tipo de entrada.<br/>                                                                                                                                                                                                                                                                                                                                                                   |
| [**outType**](eventmanifestschema-outtype-inputtype-element.md)         | [**OutputType**](eventmanifestschema-outputtype-complextype.md) | Lista de tipos de datos de salida que determinan cómo se representan los datos. Debe especificar al menos un tipo de salida. Especifique una lista si el tipo de entrada se puede representar como tipos de salida diferentes. Por ejemplo, el tipo de entrada win:UInt16 se puede representar como un número corto sin signo, un número de puerto o un número hexadecimal. Uno de los tipos de salida que especifique debe tener el atributo predeterminado establecido en true.<br/> |



## <a name="attributes"></a>Atributos



| Nombre   | Tipo                                                              | Descripción                                                                                                                                                                                                                                              |
|--------|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| name   | **QName**                                                         | Nombre del tipo de entrada.<br/>                                                                                                                                                                                                                   |
| símbolo | [**CSymbolType**](eventmanifestschema-csymboltype-simpletype.md) | Símbolo que se usará para hacer referencia al tipo de entrada de la aplicación. El [**compilador de mensajes (MC.exe)**](message-compiler--mc-exe-.md) usa el símbolo para crear una constante para el tipo de entrada en el archivo de encabezado que genera el compilador.<br/> |
| value  | string                                                            | Identificador numérico que identifica de forma única el tipo de entrada en una lista de tipos de entrada que defina.<br/>                                                                                                                                        |
