# <a name="active-directory-schema-ad-schema"></a>Active Directory esquema (esquema de AD)

El esquema Active Directory microsoft contiene definiciones formales de todas las clases de objeto que se pueden crear en un Active Directory objeto. El esquema también contiene definiciones formales de cada atributo que puede existir en un Active Directory objeto . En esta sección se proporciona la referencia para cada objeto de esquema y se proporciona una breve explicación de los atributos, clases y otros objetos que Active Directory esquema.

> [!Note]  
> La siguiente documentación contiene la referencia de programación para Active Directory esquema. Si es un usuario final que intenta depurar un error de impresora, intente buscar en el sitio de [la comunidad de Microsoft](https://answers.microsoft.com). Si es un desarrollador que busca información general sobre el esquema Active Directory, consulte los temas de información general Active Directory [esquema.](/windows/desktop/AD/active-directory-schema) Si busca directrices de programación para actualizar o modificar el esquema, para obtener más información sobre cómo ampliar y personalizar el esquema, vea [Extender](/windows/desktop/AD/extending-the-schema)el esquema , así como muchos de los temas de las guías de programación [Active Directory Domain Services](/windows/desktop/AD/active-directory-domain-services) [y Active Directory Lightweight Directory Services.](/previous-versions/windows/desktop/adam/active-directory-lightweight-directory-services)

 

En cada uno de los temas de referencia, hay una sección para cada sistema operativo al que se aplica el tema. Actualmente se admiten los siguientes sistemas operativos. 

| Plataforma                                                      | Nombre en el tema                               |
|---------------------------------------------------------------|---------------------------------------------|
| Microsoft Windows Server 2003<br/>                      | Windows Server 2003<br/>              |
| Modo Active Directory aplicación de Microsoft (ADAM)<br/> | Adán<br/>                             |
| Microsoft Windows Server 2003 R2<br/>                   | Windows Server 2003 R2<br/>           |
| Microsoft Windows Server 2008<br/>                      | Microsoft Windows Server 2008<br/>    |
| Microsoft Windows Server 2008 R2<br/>                   | Microsoft Windows Server 2008 R2<br/> |
| Microsoft Windows Server 2012<br/>                      | Microsoft Windows Server 2012<br/>    |



 

Si un sistema operativo no aparece en el tema, el tema no se admite en ese sistema operativo. Por ejemplo, si un tema solo enumera Windows Server 2003 y ADAM, el tema no se aplica a Windows Server 2003 R2.
