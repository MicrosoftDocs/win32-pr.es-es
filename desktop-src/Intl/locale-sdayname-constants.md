# <a name="locale_sdayname-constants"></a>Constantes \_ LOCALE SDAYNAME \*

En este tema se definen las \_ constantes LOCALE SDAYNAME \* usadas por NLS.



| Value             | Significado                                                                                                                                 |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SDAYNAME1 | Nombre largo nativo para monday. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.    |
| LOCALE \_ SDAYNAME2 | Nombre largo nativo para el martes. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.   |
| LOCALE \_ SDAYNAME3 | Nombre largo nativo para el miércoles. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación. |
| LOCALE \_ SDAYNAME4 | Nombre largo nativo para el jueves. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.  |
| LOCALE \_ SDAYNAME5 | Nombre largo nativo para el viernes. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.    |
| LOCALE \_ SDAYNAME6 | Nombre largo nativo para el sábado. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.  |
| LOCALE \_ SDAYNAME7 | Nombre largo nativo para el domingo. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.    |
