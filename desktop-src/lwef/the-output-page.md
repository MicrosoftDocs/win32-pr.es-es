# <a name="the-output-page"></a>La página De salida

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Esta página incluye propiedades que controlan la salida de caracteres. Por ejemplo, el usuario puede determinar si se debe mostrar la salida en el globo de palabras, determinar cómo debe aparecer la salida del globo, reproducir la salida hablada como audio, reproducir efectos de sonido de caracteres, mostrar el mensaje de reinicio y ajustar la velocidad de habla.
