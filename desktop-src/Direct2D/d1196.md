# <a name="d1196-a8-shared-bitmap"></a>D1196: Mapa de bits compartido A8

No se admite una superficie DXGI de formato DXGI \_ FORMAT \_ A8 \_ UNORM en esta API.





 

## <a name="possible-causes"></a>Causas posibles

DXGI FORMAT A8 UNORM no se admite como formato de mapa de bits de interoperabilidad DXGI, incluso si se \_ \_ admite en \_ otras API.

## <a name="possible-fixes"></a>Posibles correcciones

Cree un mapa de bits de interoperabilidad DXGI de un formato diferente.
