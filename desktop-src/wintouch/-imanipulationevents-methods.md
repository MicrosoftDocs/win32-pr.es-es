# <a name="methods-_imanipulationevents"></a>Métodos (_IManipulationEvents)

Esta sección contiene los métodos para la [**\_ interfaz IManipulationEvents.**](/windows/win32/api/manipulations/nn-manipulations-_imanipulationevents)



| Método                                                                      | Descripción                                                   |
|-----------------------------------------------------------------------------|---------------------------------------------------------------|
| [**ManipulationStarted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationstarted)     | Controla el evento para cuando comienza la manipulación o la inercia.    |
| [**ManipulationDelta**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationdelta)         | Controla los eventos que se suceden cuando cambia un objeto manipulado. |
| [**ManipulationCompleted**](/windows/win32/api/manipulations/nf-manipulations-_imanipulationevents-manipulationcompleted) | Controla el evento cuando finaliza la manipulación o la inercia.      |
