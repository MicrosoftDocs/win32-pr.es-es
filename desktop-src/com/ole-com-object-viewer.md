# <a name="ole-com-object-viewer"></a>Visor de objetos OLE y COM

El visor de objetos OLE-COM, Oleview.exe, es una aplicación suministrada con Visual C++ que muestra los objetos COM instalados en el equipo y las interfaces que admiten. Puede usar este visor de objetos para ver las bibliotecas de tipos.

Para ver la biblioteca de tipos de un objeto COM:

1.  En el menú **archivo** del visor de objetos, haga clic en **Ver typelib**. Aparece un cuadro de diálogo **abrir** .
2.  Especifique el archivo de biblioteca de tipos que desea abrir y haga clic en **Aceptar**.

El visor de objetos muestra las interfaces del objeto.

Para obtener más información, vea [usar el visor de objetos OLE/com](/previous-versions/d0kh9f4c(v=vs.140)) en la documentación de Visual C++.
