# <a name="ms-sql-size-attribute"></a>Atributo size de MS-SQL

Tamaño, en kilobytes, de la base de datos.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-SQL-size                          |
| Nombre para mostrar de LDAP | mS-SQL-size                          |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | Cuando la base de datos cambia de tamaño.      |
| Attribute-Id      | 1.2.840.113556.1.4.1396              |
| System-ID-GUID    | e9098084-ccee-11d2-9993-0000f87a57d4 |
| Sintaxis            | [**Interval**](s-interval.md)       |
