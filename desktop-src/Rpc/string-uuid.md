# <a name="string-uuid"></a>UUID de cadena

Un UUID de cadena contiene la representación de la matriz de caracteres de un UUID. Un UUID de cadena se compone de varios campos de caracteres hexadecimales. Cada miembro tiene una longitud fija y los campos se separan mediante el carácter de guión. Por ejemplo:

`989C6E5C-2CC1-11CA-A044-08002B1BB4F5`

Al proporcionar un UUID de cadena como parámetro de entrada a una función en tiempo de ejecución de RPC, escriba los caracteres hexadecimales alfabéticos como caracteres en mayúsculas o minúsculas. Sin embargo, cuando una función en tiempo de ejecución devuelve un UUID de cadena, los caracteres hexadecimales siempre son minúsculas.
