# <a name="using-ascii-control-codes-0x000d-and-0x000a"></a>Uso de códigos de control ASCII 0x000D y 0x000A

El estándar Unicode no prescribe significados específicos para los códigos de control 0x000D (retorno de carro) y 0x000A (linefeed). No es necesario que estos códigos se utilicen en combinación. Si se usa individualmente, el código solo puede representarse a sí mismo o a ambos códigos juntos. La aplicación siempre debe determinar lo que representan estos códigos.
