# <a name="rendering-from-user-memory-pointers-direct3d-9"></a>Representación a partir de punteros de memoria de usuario (Direct3D 9)

Un conjunto secundario de interfaces de representación admite pasar datos de vértices e índices directamente desde punteros de memoria de usuario. Estas interfaces solo admiten un único flujo de datos de vértices. Para obtener más información, vea los siguientes temas de referencia.

-   [**IDirect3DDevice9::D rawPrimitiveUP**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-drawprimitiveup)
-   [**IDirect3DDevice9::D rawIndexedPrimitiveUP**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-drawindexedprimitiveup)

Estos métodos se representan con datos especificados por punteros de memoria de usuario, en lugar de búferes de vértices e índices.
