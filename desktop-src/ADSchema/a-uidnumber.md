# <a name="uidnumber-attribute"></a>Atributo uidNumber

Contiene un entero que identifica de forma única a un usuario en un dominio administrativo.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | uidNumber                            |
| Ldap-Display-Name | uidNumber                            |
| Size              | \-                                   |
| Actualizar privilegios  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.0                      |
| System-Id-Guid    | 850fcc8f-9c6b-47e1-b671-7c654be4d5b3 |
| Syntax            | [**Enumeración**](s-enumeration.md) |
