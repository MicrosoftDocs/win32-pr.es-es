# <a name="xtl-elements"></a>Elementos XTL

> [!Note]  
> \[En desuso. Esta API se puede quitar de las versiones futuras de Windows.\]

 

El formato XTL define los elementos siguientes.



| Elemento                                  | Descripción                                                                    |
|------------------------------------------|--------------------------------------------------------------------------------|
| [**Veamos**](at-element.md)                 | Especifica una configuración basada en el tiempo para un elemento **param** .                        |
| [**clip**](clip-element.md)             | Especifica un archivo de código fuente.                                                       |
| [**mixto**](composite-element.md)   | Define una composición.                                                         |
| [**realizado**](effect-element.md)         | Define un efecto.                                                             |
| [**agrupamiento**](group-element.md)           | Define un grupo (audio o vídeo).                                              |
| [**lineal**](linear-element.md)         | Especifica una configuración basada en el tiempo para un elemento **param** .                        |
| [**param**](param-element.md)           | Especifica el valor de una propiedad en una transición, un efecto u otro subobjeto. |
| [**marco**](timeline-element.md)     | Define la escala de tiempo.                                                          |
| [**track**](track-element.md)           | Define una pista.                                                               |
| [**transitorio**](transition-element.md) | Define una transición.                                                          |
