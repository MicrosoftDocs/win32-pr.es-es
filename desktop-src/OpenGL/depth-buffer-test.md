# <a name="depth-buffer-test"></a>Prueba de búfer de profundidad

La prueba de búfer de profundidad descarta un fragmento si se produce un error en una comparación de profundidad. [**glDepthFunc**](gldepthfunc.md) especifica la función de comparación. Si el estarcido está habilitado, el resultado de la comparación de profundidad también afecta al valor de actualización del búfer de estarcido.
