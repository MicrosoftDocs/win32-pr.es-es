# <a name="collaboration-api-functions"></a>Funciones de la API de colaboración

La infraestructura de colaboración del mismo nivel admite las siguientes funciones.



| Función                                                                                       | Descripción                                                                                                                                                                                     |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**PeerCollabAddContact**](/windows/desktop/api/P2P/nf-p2p-peercollabaddcontact)                                           | Agrega un contacto a la lista de contactos de un elemento del mismo nivel.                                                                                                                                                   |
| [**PeerCollabAsyncInviteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabasyncinvitecontact)                           | Envía una invitación a un contacto del mismo nivel de confianza para unirse a la actividad de colaboración del mismo nivel del remitente a través de una conexión segura.                                                                       |
| [**PeerCollabAsyncInviteEndpoint**](/windows/desktop/api/P2P/nf-p2p-peercollabasyncinviteendpoint)                         | Envía una invitación a un punto de conexión del mismo nivel especificado para unirse a la actividad de colaboración del mismo nivel del remitente. La disponibilidad de la respuesta a la invitación se actualiza a través de un evento asincrónico. |
| [**PeerCollabCancelInvitation**](/windows/desktop/api/P2P/nf-p2p-peercollabcancelinvitation)                               | Cancela una invitación enviada previamente por el autor de la llamada a un contacto.                                                                                                                               |
| [**PeerCollabCloseHandle**](/windows/desktop/api/P2P/nf-p2p-peercollabclosehandle)                                         | Cierra el identificador de una invitación de la actividad de colaboración del mismo nivel.                                                                                                                                  |
| [**PeerCollabDeleteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabdeletecontact)                                     | Elimina un contacto del elemento del mismo nivel actual.                                                                                                                                                        |
| [**PeerCollabDeleteEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabdeleteendpointdata)                           | Elimina los datos del punto de conexión del mismo nivel en el nodo del mismo nivel que realiza la llamada que coincide con los datos de extremo proporcionados.                                                                                                |
| [**PeerCollabDeleteObject**](/windows/desktop/api/P2P/nf-p2p-peercollabdeleteobject)                                       | Elimina un objeto del mismo nivel del extremo que realiza la llamada.                                                                                                                                                |
| [**PeerCollabEnumApplications**](/windows/desktop/api/P2P/nf-p2p-peercollabenumapplications)                               | Devuelve el identificador de una enumeración que contiene las funciones registradas en los puntos de conexión de un par específico.                                                                                |
| [**PeerCollabEnumApplicationRegistrationInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabenumapplicationregistrationinfo) | Obtiene el identificador de enumeración usado para recuperar la información de la aplicación del mismo nivel.                                                                                                                   |
| [**PeerCollabEnumContacts**](/windows/desktop/api/P2P/nf-p2p-peercollabenumcontacts)                                       | Devuelve un identificador a un conjunto enumerado que contiene todos los contactos de la red de colaboración del mismo nivel disponibles actualmente en el elemento del mismo nivel que realiza la llamada.                                                     |
| [**PeerCollabEnumEndpoints**](/windows/desktop/api/P2P/nf-p2p-peercollabenumendpoints)                                     | Devuelve el identificador de una enumeración que contiene los extremos asociados a un contacto del mismo nivel específico.                                                                                       |
| [**PeerCollabEnumObjects**](/windows/desktop/api/P2P/nf-p2p-peercollabenumobjects)                                         | Devuelve el identificador de una enumeración que contiene los objetos del mismo nivel asociados al extremo de un elemento del mismo nivel concreto.                                                                                 |
| [**PeerCollabEnumPeopleNearMe**](/windows/desktop/api/P2P/nf-p2p-peercollabenumpeoplenearme)                               | Devuelve un identificador a un conjunto enumerado que contiene todos los puntos de conexión de la red de colaboración del mismo nivel "People Near me" actualmente disponibles en la subred del elemento del mismo nivel que realiza la llamada.                     |
| [**PeerCollabExportContact**](/windows/desktop/api/P2P/nf-p2p-peercollabexportcontact)                                     | Exporta los datos de contacto asociados a un nombre de mismo nivel a un búfer de cadena de datos XML de contacto.                                                                                                       |
| [**PeerCollabGetAppLaunchInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetapplaunchinfo)                               | Obtiene la información de inicio de la aplicación del mismo nivel, incluido el nombre del contacto, el punto de conexión del mismo nivel y la solicitud de invitación.                                                                     |
| [**PeerCollabGetApplicationRegistrationInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetapplicationregistrationinfo)   | Obtiene la información específica de registro de la aplicación.                                                                                                                                          |
| [**PeerCollabGetContact**](/windows/desktop/api/P2P/nf-p2p-peercollabgetcontact)                                           | Obtiene la información de un contacto del mismo nivel específico dado el nombre del mismo nivel del contacto.                                                                                                         |
| [**PeerCollabGetEndpointName**](/windows/desktop/api/P2P/nf-p2p-peercollabgetendpointname)                                 | Recupera el nombre del punto de conexión actual del elemento del mismo nivel que realiza la llamada previamente establecido por una llamada a PeerCollabSetEndpointName.                                                                           |
| [**PeerCollabGetEventData**](/windows/desktop/api/P2P/nf-p2p-peercollabgeteventdata)                                       | Obtiene los datos asociados a un evento de colaboración del mismo nivel generado en el elemento del mismo nivel.                                                                                                                 |
| [**PeerCollabGetInvitationResponse**](/windows/desktop/api/P2P/nf-p2p-peercollabgetinvitationresponse)                     | Obtiene la respuesta del elemento del mismo nivel previamente invitado para unirse a una actividad de colaboración del mismo nivel.                                                                                                        |
| [**PeerCollabGetPresenceInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabgetpresenceinfo)                                 | Recupera la información de presencia para el punto de conexión asociado a un contacto específico.                                                                                                         |
| [**PeerCollabGetSigninOptions**](/windows/desktop/api/P2P/nf-p2p-peercollabgetsigninoptions)                               | Obtiene las opciones de presencia de red de colaboración del mismo nivel con sesión actual del mismo nivel.                                                                                                               |
| [**PeerCollabInviteContact**](/windows/desktop/api/P2P/nf-p2p-peercollabinvitecontact)                                     | Envía una invitación para unir una actividad de colaboración del mismo nivel a un contacto de confianza. Esta llamada es sincrónica y, si se realiza correctamente, obtiene una respuesta del contacto.                               |
| [**PeerCollabInviteEndpoint**](/windows/desktop/api/P2P/nf-p2p-peercollabinviteendpoint)                                   | Envía una invitación a un punto de conexión del mismo nivel especificado para unirse a la actividad de colaboración del mismo nivel del remitente. Esta llamada es sincrónica y, si se realiza correctamente, obtiene una respuesta del punto de conexión del mismo nivel.      |
| [**PeerCollabParseContact**](/windows/desktop/api/P2P/nf-p2p-peercollabparsecontact)                                       | Analiza un búfer de cadena Unicode que contiene datos XML de contacto en una estructura de datos de [**\_ contacto del mismo nivel**](/windows/desktop/api/P2P/ns-p2p-peer_contact) .                                                                         |
| [**PeerCollabQueryContactData**](/windows/desktop/api/P2P/nf-p2p-peercollabquerycontactdata)                               | Recupera la información de contacto para el punto de conexión del mismo nivel proporcionado.                                                                                                                               |
| [**PeerCollabRefreshEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabrefreshendpointdata)                         | Actualiza el nodo de llamada del mismo nivel con nuevos datos de extremo.                                                                                                                                           |
| [**PeerCollabRegisterApplication**](/windows/desktop/api/P2P/nf-p2p-peercollabregisterapplication)                         | Registra una aplicación en el equipo local para que se pueda iniciar en una actividad de colaboración del mismo nivel.                                                                                   |
| [**PeerCollabRegisterEvent**](/windows/desktop/api/P2P/nf-p2p-peercollabregisterevent)                                     | Registra una aplicación con la infraestructura de colaboración del mismo nivel para recibir devoluciones de llamada para eventos específicos de colaboración del mismo nivel.                                                                |
| [**PeerCollabSetEndpointName**](/windows/desktop/api/P2P/nf-p2p-peercollabsetendpointname)                                 | Establece el nombre del punto de conexión actual usado por la aplicación del mismo nivel.                                                                                                                             |
| [**PeerCollabSetObject**](/windows/desktop/api/P2P/nf-p2p-peercollabsetobject)                                             | Crea o actualiza un objeto de datos del mismo nivel usado en una red de colaboración del mismo nivel.                                                                                                                     |
| [**PeerCollabSetPresenceInfo**](/windows/desktop/api/P2P/nf-p2p-peercollabsetpresenceinfo)                                 | Actualiza la información de presencia del autor de la llamada a los contactos que lo ven.                                                                                                                          |
| [**PeerCollabSignIn**](/windows/desktop/api/P2P/nf-p2p-peercollabsignin)                                                   | Firma el elemento del mismo nivel en un proveedor de presencia de red hospedada en Internet (presencia sin servidor) o subred ("equipos a mi alrededor") de colaboración del mismo nivel.                                                          |
| [**PeerCollabSignOut**](/windows/desktop/api/P2P/nf-p2p-peercollabsignout)                                                 | Firma un elemento del mismo nivel de un tipo específico de proveedor de presencia de red de colaboración del mismo nivel.                                                                                                            |
| [**PeerCollabShutdown**](/windows/desktop/api/P2P/nf-p2p-peercollabshutdown)                                               | Apaga la infraestructura de colaboración del mismo nivel y libera todos los recursos asociados a ella.                                                                                                 |
| [**PeerCollabStartup**](/windows/desktop/api/P2P/nf-p2p-peercollabstartup)                                                 | Inicializa la infraestructura de colaboración del mismo nivel.                                                                                                                                              |
| [**PeerCollabSubscribeEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabsubscribeendpointdata)                     | Crea una suscripción a un punto de conexión disponible.                                                                                                                                                |
| [**PeerCollabUnregisterApplication**](/windows/desktop/api/P2P/nf-p2p-peercollabunregisterapplication)                     | Anula el registro de las aplicaciones específicas de un elemento del mismo nivel en el equipo local.                                                                                                                        |
| [**PeerCollabUnregisterEvent**](/windows/desktop/api/P2P/nf-p2p-peercollabunregisterevent)                                 | Anula el registro de una aplicación a partir de notificaciones de eventos de colaboración del mismo nivel concretos.                                                                                                          |
| [**PeerCollabUnsubscribeEndpointData**](/windows/desktop/api/P2P/nf-p2p-peercollabunsubscribeendpointdata)                 | Quita una suscripción a un punto de conexión creado con PeerCollabSubscribeEndpointData.                                                                                                             |
| [**PeerCollabUpdateContact**](/windows/desktop/api/P2P/nf-p2p-peercollabupdatecontact)                                     | Actualiza un elemento del mismo nivel que participa en una red de colaboración del mismo nivel con información nueva sobre un contacto del mismo nivel.                                                                                            |
