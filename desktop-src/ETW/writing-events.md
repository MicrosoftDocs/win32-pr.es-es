# <a name="writing-events"></a>Escribir eventos

El tipo de proveedor determina cómo se escriben los eventos en una sesión de seguimiento.

Para los [proveedores de MOF,](about-event-tracing.md) vea [Escribir eventos MOF (clásico).](tracing-events.md)

Para los [proveedores de WPP,](about-event-tracing.md) vea [Escribir eventos de WPP.](windows-software-trace-preprocessor.md)

Para [proveedores basados en](about-event-tracing.md) manifiestos, vea Escribir eventos [basados en manifiestos.](writing-manifest-based-events.md)

En el caso de los proveedores de [tracelogging,](about-event-tracing.md) [vea Using TraceLogging](../tracelogging/tracelogging-using-tracelogging.md).
