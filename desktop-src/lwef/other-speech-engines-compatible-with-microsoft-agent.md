# <a name="other-speech-engines-compatible-with-microsoft-agent"></a>Otros motores de voz compatibles con Microsoft Agent

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Además de los motores de Microsoft Speech que se proporcionan con y son compatibles con Microsoft Agent, varias empresas ofrecen también motores de voz compatibles con Microsoft Agent. En esta página se ofrece una breve descripción de estos motores que pueden estar disponibles en Inglés de EE. UU. y en otros idiomas. Puede encontrar información sobre la instalación y el acceso, así como la licencia y redistribución de los motores en sus sitios Web.

Al usar cualquier motor de voz de terceros, también debe instalar los archivos binarios de Microsoft SAPI 4.0 en tiempo de ejecución para que los motores se enumeren correctamente. Desde la página web puede incluir la siguiente etiqueta para desencadenar una descarga de la Speech API:

``` syntax
<OBJECT WIDTH=0 HEIGHT=0
  CLASSID="CLSID:0C7F3F20-8BAB-11d2-9432-00C04F8EF48F"
  CODEBASE="#VERSION=4,0,0,0">
</OBJECT>
```

Para obtener más información acerca de los archivos binarios en tiempo de ejecución de SAPI, consulte el [sitio web del grupo de Microsoft Speech](https://msdn.microsoft.com/library/ee705648.aspx). El agente de Microsoft también instala los archivos binarios en tiempo de ejecución de SAPI con el motor de reconocimiento de voz de Microsoft, el panel de control de voz y el editor de caracteres del agente.

Tenga en cuenta que estos vínculos apuntan a servidores que no están bajo el control de Microsoft. Lea la [declaración oficial](https://www.microsoft.com/isapi/gomscom.asp?TARGET=/Misc/NonMS.md) de Microsoft con respecto a otros servidores. Microsoft no respalda el contenido ni el software proporcionado en estos sitios Web. Todas las preguntas técnicas y de soporte técnico también deben dirigirse a los proveedores de los motores y no a Microsoft ni al equipo de Microsoft Agent.

**Digalo motor de texto a voz**

Digalo es un motor TTS asequible diseñado para usuarios finales y desarrolladores. El motor se ofrece en varios idiomas: francés, alemán, Portugués (Brasil), español, Ruso y Reino Unido y Inglés de EE. UU., con Italiano, Polaco y otros idiomas disponibles próximamente. También está disponible la información para desarrolladores, así como los detalles sobre el registro y los programas afiliados del usuario final.

**Motor de voz de Elan**

El motor de voz de Elan utiliza la tecnología de texto a voz de Elan y está disponible en siete idiomas: francés, alemán, Portugués (Brasil), español, Ruso, Reino Unido y Inglés de EE. UU.

**IBM ViaVoice de alta intensidad**

IBM ha desarrollado varios caracteres de Microsoft Agent para su uso con ViaVoice de alta intensidad. IBM ViaVoice es una tecnología de conversión de texto a voz (síntesis de voz) y está disponible en francés, alemán, Italiano, español, Inglés de Reino Unido y Inglés de EE. UU. Cuando se combina con el agente de Microsoft, puede crear aplicaciones vibrantes en muchos idiomas y ampliar sus oportunidades de ventas a nuevos mercados mundiales.
