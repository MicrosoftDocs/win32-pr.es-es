# <a name="control-pattern-interfaces-for-providers"></a>Interfaces de patrón de control para proveedores

En esta sección se describen las interfaces de *patrón de control* implementadas por los proveedores de automatización de la interfaz de usuario para exponer información sobre los controles en aplicaciones Windows no administradas.

## <a name="in-this-section"></a>En esta sección



| Interfaz                                                                          | Descripción                                                                                                                                                                                                                                                                                                         |
|------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAnnotationProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-iannotationprovider)<br/>           | Expone las propiedades de una anotación en un documento.<br/>                                                                                                                                                                                                                                                   |
| [**IDockProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idockprovider)<br/>                           | Proporciona acceso a un elemento de un contenedor de acoplamiento. <br/>                                                                                                                                                                                                                                                   |
| [**IDragProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idragprovider)<br/>                           | Habilita un elemento de automatización de la interfaz de usuario de Microsoft para describirse a sí mismo como un elemento que se puede arrastrar como parte de una operación de arrastrar y colocar.<br/>                                                                                                                                                                     |
| [**IDropTargetProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-idroptargetprovider)<br/>               | Habilita un elemento de automatización de la interfaz de usuario para describirse a sí mismo como un elemento que puede recibir una eliminación de un elemento arrastrado como parte de una operación de arrastrar y colocar de automatización de la interfaz de usuario.<br/>                                                                                                                                        |
| [**IExpandCollapseProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iexpandcollapseprovider)<br/>       | Proporciona acceso a un control que se expande visualmente para mostrar el contenido y se contrae para ocultar el contenido. <br/>                                                                                                                                                                                                   |
| [**IGridItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igriditemprovider)<br/>                   | Proporciona acceso a los controles secundarios individuales de contenedores que implementan [**IGridProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igridprovider).<br/>                                                                                                                                                                                 |
| [**IGridProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-igridprovider)<br/>                           | Proporciona acceso a los controles que actúan como contenedores para una colección de elementos secundarios organizados en un sistema de coordenadas lógico bidimensional que se puede atravesar (es decir, un cliente de automatización de la interfaz de usuario puede moverse a controles adyacentes) con el teclado.<br/>                                              |
| [**IInvokeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iinvokeprovider)<br/>                       | Proporciona acceso a los controles que inician o realizan una única acción inequívoca y no mantienen el estado cuando se activan.<br/>                                                                                                                                                                              |
| [**IItemContainerProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iitemcontainerprovider)<br/>         | Proporciona acceso a los controles que actúan como contenedores de otros controles, como una vista de lista virtual. <br/>                                                                                                                                                                                                      |
| [**ILegacyIAccessibleProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ilegacyiaccessibleprovider)<br/> | Permite a los clientes de automatización de la interfaz de usuario tener acceso a la implementación de [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) subyacente de los elementos de Microsoft Active Accessibility.<br/>                                                                                                                                                     |
| [**IMultipleViewProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-imultipleviewprovider)<br/>           | Proporciona acceso a los controles que proporcionan y pueden cambiar entre varias representaciones del mismo conjunto de información o controles secundarios.<br/>                                                                                                                                                     |
| [**IObjectModelProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-iobjectmodelprovider)<br/>         | Proporciona acceso al modelo de objetos subyacente implementado por un control o una aplicación. <br/>                                                                                                                                                                                                                 |
| [**IRangeValueProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irangevalueprovider)<br/>               | Proporciona acceso a los controles que se pueden establecer en un valor dentro de un intervalo. <br/>                                                                                                                                                                                                                                  |
| [**IScrollItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollitemprovider)<br/>               | Proporciona acceso a los controles secundarios individuales de contenedores que implementan [**IScrollProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollprovider). <br/>                                                                                                                                                                            |
| [**IScrollProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iscrollprovider)<br/>                       | Proporciona acceso a los controles que actúan como contenedores desplazables para una colección de objetos secundarios.<br/>                                                                                                                                                                                                         |
| [**ISelectionItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionitemprovider)<br/>         | Proporciona acceso a controles secundarios individuales seleccionables de contenedores que implementan [**ISelectionProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider). <br/>                                                                                                                                                          |
| [**ISelectionProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider)<br/>                 | Proporciona acceso a los controles que actúan como contenedores para una colección de elementos secundarios seleccionables individuales.<br/>                                                                                                                                                                                               |
| [**ISelectionProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionprovider2)<br/>               | Extiende la interfaz [**ISelectionItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iselectionitemprovider) para proporcionar información sobre los elementos seleccionados.<br/>                                                                                                                                                                       |
| [**ISpreadsheetItemProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-ispreadsheetitemprovider)<br/> | Proporciona acceso a información sobre un elemento (celda) en una hoja de cálculo. <br/>                                                                                                                                                                                                                                   |
| [**ISpreadsheetProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-ispreadsheetprovider)<br/>         | Proporciona acceso a los elementos (celdas) de una hoja de cálculo.<br/>                                                                                                                                                                                                                                                       |
| [**IStylesProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-istylesprovider)<br/>                   | Proporciona acceso a los estilos visuales asociados al contenido de un documento.<br/>                                                                                                                                                                                                                          |
| [**ISynchronizedInputProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-isynchronizedinputprovider)<br/> | Permite a las aplicaciones cliente de automatización de la interfaz de usuario dirigir la entrada del mouse o del teclado a un elemento específico de la interfaz de usuario. <br/>                                                                                                                                                                                               |
| [**ITableItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableitemprovider)<br/>                 | Proporciona acceso a los controles secundarios de los contenedores que implementan [**ITableProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableprovider). <br/>                                                                                                                                                                                         |
| [**ITableProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableprovider)<br/>                         | Proporciona acceso a los controles que actúan como contenedores para una colección de elementos secundarios. Los elementos secundarios de este elemento deben implementar [**ITableItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itableitemprovider) y organizarse en un sistema de coordenadas lógico bidimensional que se pueda atravesar con el teclado. <br/> |
| [**ITextChildProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itextchildprovider)<br/>            | Proporciona acceso a un control basado en texto (o un objeto incrustado en el texto) que es un elemento secundario o descendiente de otro control basado en texto. <br/>                                                                                                                                                                    |
| [**ITextEditProvider**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itexteditprovider)<br/>               | Extiende la interfaz [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) para permitir que los proveedores de automatización de la interfaz de usuario expongan acciones de edición de texto mediante programación.<br/>                                                                                                                                                          |
| [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider)<br/>                           | Proporciona acceso a los controles que contienen texto. <br/>                                                                                                                                                                                                                                                          |
| [**ITextProvider2**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itextprovider2)<br/>                     | Extiende la interfaz [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) para permitir a los proveedores de automatización de la interfaz de usuario exponer el contenido textual que es el destino de una anotación e información sobre un símbolo de intercalación que pertenece al proveedor.<br/>                                                                         |
| [**ITextRangeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider)<br/>                 | Proporciona acceso a un intervalo de texto continuo en un contenedor de texto que implementa [**ITextProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider) o [**ITextProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextprovider2).<br/>                                                                                                                       |
| [**ITextRangeProvider2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider2)<br/>               | Extiende la interfaz [**ITextRangeProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itextrangeprovider) para permitir a los proveedores de automatización de la interfaz de usuario invocar menús contextuales.<br/>                                                                                                                                                                 |
| [**IToggleProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itoggleprovider)<br/>                       | Proporciona acceso a los controles que pueden recorrer en iteración un conjunto de Estados y mantener un estado una vez establecido.<br/>                                                                                                                                                                                                 |
| [**ITransformProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itransformprovider)<br/>                 | Proporciona acceso a los controles que se pueden desplazar, cambiar de tamaño o girar dentro de un espacio bidimensional. <br/>                                                                                                                                                                                                  |
| [**ITransformProvider2**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-itransformprovider2)<br/>           | Extiende la interfaz [**ITransformProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-itransformprovider) para permitir que los proveedores de automatización de la interfaz de usuario expongan propiedades para admitir la funcionalidad de zoom de la ventanilla de un control. <br/>                                                                                                        |
| [**IValueProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ivalueprovider)<br/>                         | Proporciona acceso a los controles que tienen un valor intrínseco que no abarca un intervalo y que se puede representar como una cadena. <br/>                                                                                                                                                                           |
| [**IVirtualizedItemProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-ivirtualizeditemprovider)<br/>     | Proporciona acceso a los elementos virtualizados, que son elementos que se representan mediante elementos de automatización de marcadores de posición en el árbol de automatización de la interfaz de usuario.<br/>                                                                                                                                                                 |
| [**IWindowProvider**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iwindowprovider)<br/>                       | Proporciona acceso a la funcionalidad fundamental basada en ventanas de un control.<br/>                                                                                                                                                                                                                              |
