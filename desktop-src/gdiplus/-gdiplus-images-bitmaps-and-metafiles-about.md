# <a name="images-bitmaps-and-metafiles"></a>Imágenes, mapas de bits y metarchivos

Windows GDI+ proporciona la clase de [**imagen**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-image) para trabajar con imágenes de trama (mapas de bits) e imágenes vectoriales (metaarchivos). La clase [**Bitmap**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-bitmap) y la clase [**Metafile**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-metafile) heredan de la clase **Image** . La clase **Bitmap** amplía las capacidades de la clase **Image** proporcionando métodos adicionales para cargar, guardar y manipular imágenes de tramas. La clase **Metafile** amplía las capacidades de la clase **Image** proporcionando métodos adicionales para grabar y examinar imágenes vectoriales.
