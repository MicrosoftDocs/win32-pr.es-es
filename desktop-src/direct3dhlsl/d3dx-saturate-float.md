# <a name="d3dx_saturate_float-function"></a>D3DX \_ Saturate ( \_ función float)

Recupera el valor saturado del valor de tipo FLOAT dado.

## <a name="syntax"></a>Sintaxis

``` syntax
FLOAT D3DX_Saturate_FLOAT(
   FLOAT _V
);
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

*\_V* 
</dt> <dd>

Valor que se va a saturar.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Valor saturado.
