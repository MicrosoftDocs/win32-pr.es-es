# <a name="creating-the-background"></a>Crear el fondo

Ahora está listo para crear la imagen de fondo. Mantenga visible la capa del rectángulo, pero manténgala detrás de la capa en la que está dibujando y úsela como guía para crear las imágenes de la máscara. Querrá ensamblar cada elemento en una o varias capas para facilitar la copia en otro archivo posteriormente. Por ejemplo, el botón PlayPause tendrá que copiarse en el archivo en pausa, el archivo deshabilitado y el archivo de región. Cuando mueva la imagen al archivo de región, asegúrese de que la imagen que copia no tiene efectos de borde complejos, como sombras o biseles.

Cuando haya terminado de crear la imagen que desee, haga visible todas las capas que desee usar y oculte las otras (por ejemplo, boceto y rectángulo). A continuación, use el comando Guardar una copia para guardar la ilustración en un archivo. bmp.

El siguiente diagrama es la imagen de fondo.

![archivo en segundo plano](images/ceswmbak.png)

Los biseles, el relieve y las sombras se usaron para crear los efectos visuales de los distintos elementos. Tenga en cuenta que no hay ningún botón para la imagen de control de volumen. Se creará más adelante.

Photoshop tiene varios efectos visuales que se pueden aplicar a las imágenes, lo que facilita la creación de ilustraciones más interesantes. Por ejemplo, el botón PlayPause se creó con un círculo simple. A continuación se crea un bisel y un relieve con efecto de sombra. La flecha sólida era un triángulo simple creado con tres clics de la herramienta pluma. A continuación, se proporcionó una apariencia en relieve a la flecha con un efecto. Cuando se usa un programa como Photoshop, no es necesario poder dibujar. Solo puede crear imágenes simples y aplicar efectos. También puede importar sus propias fotografías o material gráfico digitalizado.
