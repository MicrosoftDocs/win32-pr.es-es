# <a name="keywords-channelpublishingtype-element"></a>Elemento keywords (ChannelPublishingType)

Máscara de bits que determina la categoría de eventos que se escriben en el canal.

``` syntax
<xs:element name="keywords"
    type="UInt64Type"
 />
```

El **elemento keywords** se define mediante el tipo complejo [**ChannelPublishingType.**](eventmanifestschema-channelpublishingtype-complextype.md)
