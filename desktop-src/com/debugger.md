# <a name="debugger"></a>instantáneas

Almacena el nombre del depurador preferido que se usará con la depuración COM remota a través de RPC.

## <a name="registry-entry"></a>Entrada del Registro

```
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\DebugObjectRPCEnabled\AeDebug
   Debugger = value
```

## <a name="remarks"></a>Comentarios

Se trata de **un valor \_ REG SZ.**
