# <a name="tablet-pc"></a>Tablet PC

Tablet PC es un equipo personal totalmente funcional orientado a las aplicaciones habilitadas para lápiz, habilitadas para escritura a mano y habilitadas para voz. La combinación de software y hardware en un Tablet PC permite estos métodos de interacción con el usuario y permite una experiencia informática enriquecida, interactiva y productiva para los usuarios.

La tecnología de Tablet PC de Windows permite la entrada y salida de datos de escritura a mano y voz en un Tablet PC, así como el intercambio de estos datos con otros equipos.

Windows Touch, introducido en Windows 7, habilita los gestos de multitoque y una experiencia táctil mejorada para los usuarios.
