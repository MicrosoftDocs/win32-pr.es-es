# <a name="vistaparamenableadvanced-field"></a>VistaParam. EnableAdvanced, campo

Este parámetro se usa para controlar el momento en que el motor de base de datos acepta o rechaza los cambios en un subconjunto de los parámetros del sistema. Este parámetro se usa junto con la [configuración](./vistaparam.configuration-field.md) para evitar que algunos parámetros del sistema se establezcan fuera de los valores predeterminados de la configuración seleccionada.

**Espacio de nombres:**  [Microsoft. ISAM. esent. Interop. vista](./microsoft.isam.esent.interop.vista-namespace.md)  
**Ensamblado:**  Microsoft. ISAM. esent. Interop (en Microsoft.Isam.Esent.Interop.dll)
