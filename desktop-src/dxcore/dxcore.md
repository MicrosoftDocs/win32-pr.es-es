# <a name="dxcore"></a>DXCore

DXCore es una API de enumeración de adaptadores para dispositivos de proceso y gráficos, por lo que algunas de sus funciones se superponen con las de [DXGI](../direct3ddxgi/dx-graphics-dxgi.md). Direct3D 12 usa DXCore.

Este conjunto de documentación contiene información sobre la programación con DXCore, así como la referencia de DXCore.

## <a name="requirements"></a>Requisitos

| &nbsp; | &nbsp; |
| ---- |:---- |
| **Entornos de tiempo de ejecución compatibles** | Windows/C++ |
| **Idiomas de programación recomendados** | C++ |
| **Cliente mínimo compatible** | Windows 10, versión 2004 (10,0; Compilación 19041) |
| **Header** | dxcore. h y dxcore_interface. h |
| **Library** | dxcore. lib |
| **DLL** | dxcore.dll |
