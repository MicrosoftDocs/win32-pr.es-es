# <a name="d1164-invalid-map-option"></a>D1164: opción de asignación no válida

D2D1 \_ \_ las opciones \_ de asignación None no es un parámetro válido para asignar






 

## <a name="possible-causes"></a>Causas posibles

Direct2D puede generar este mensaje si una aplicación llama a ID2D1Bitmap1:: Map con la \_ opción none de la asignación D2D1 \_ \_ .

## <a name="possible-fixes"></a>Posibles correcciones

Mientras se llama a ID2D1Bitmap1:: Map, utilice una opción de asignación válida. Consulte D2D1 \_ map \_ Options para obtener una descripción de las opciones de mapa; considere si la aplicación necesita acceso de lectura o escritura.
