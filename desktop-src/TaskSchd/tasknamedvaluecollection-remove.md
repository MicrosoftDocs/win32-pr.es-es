# <a name="tasknamedvaluecollectionremove-method"></a>TaskNamedValueCollection. Remove (método)

En el caso de scripting, quita un par nombre-valor seleccionado de la colección.

## <a name="syntax"></a>Sintaxis


```VB
TaskNamedValueCollection.Remove( _
  ByVal index _
)
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*Índice* \[ de de\]
</dt> <dd>

Índice del par de nombre y valor que se va a quitar.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
