# <a name="arrays-and-rpc"></a>Matrices y RPC

Los lenguajes de programación C y C++ proporcionan básicamente dos tipos de matrices: unidimensionales y multidimensionales. RPC permite a los desarrolladores especificar tipos de matriz adicionales mediante atributos MIDL para describir las características de las matrices en aplicaciones distribuidas.

En esta sección se describen los tipos de matrices disponibles en RPC. También se describen los atributos MIDL que los desarrolladores pueden usar para describir matrices unidimensionales y multidimensionales. Este material se presenta en los temas siguientes:

-   [Tipos de matrices](kinds-of-arrays.md)
-   [Atributos de matriz](array-attributes.md)
-   [Matrices multidimensionales](multidimensional-arrays.md)
