# <a name="text-services-framework-text-services-framework"></a>Marco de trabajo de servicios de texto (marco de trabajo de servicios de texto)

## <a name="purpose"></a>Propósito

Microsoft Windows Text Services Framework (TSF) es un servicio del sistema disponible como paquete redistribuible para Windows. TSF proporciona un marco de trabajo sencillo y escalable para la entrega de tecnologías avanzadas de entrada de texto y de lenguaje natural. TSF puede estar habilitado en las aplicaciones o como servicio de texto TSF. Un servicio de texto TSF proporciona compatibilidad multilingüe y proporciona servicios de texto como procesadores de teclado, reconocimiento de escritura a mano y reconocimiento de voz.

## <a name="where-applicable"></a>Donde sea aplicable

El marco de trabajo de servicios de texto es aplicable a equipos basados en Windows que usan servicios de texto y Windows XP o versiones posteriores del sistema operativo.

## <a name="developer-audience"></a>Audiencia de desarrolladores

El marco de trabajo de servicios de texto está diseñado para que lo usen los programadores del modelo de objetos componentes (COM) con los lenguajes de programación C/C++. Los programadores deben estar familiarizados con los servicios de texto para equipos basados en Windows. Se recomienda el conocimiento del reconocimiento de escritura a mano, el reconocimiento de voz y la programación para la compatibilidad multilingüe.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

Para el último redistribuible, descargue el SDK de la [plataforma Windows 10](https://developer.microsoft.com/windows/downloads/windows-10-sdk).

Para obtener más información sobre los requisitos de los elementos específicos de la API, consulte la sección de requisitos de la (/Windows/Win32/TSF/Text-Services-Framework-Reference) [documentación de referencia de TFS.

## <a name="additional-resources"></a>Recursos adicionales

Qué debe saber antes de leer esta guía

Para los fines de la ayuda del marco de trabajo de servicios de texto, el término "aplicación" hace referencia a una aplicación habilitada para TSF, el término "servicio de texto" hace referencia a un servicio de texto TSF y el término "administrador" hace referencia al administrador de TSF. Cada término se aplica como se indica a menos que se especifique lo contrario. Los proveedores de servicios de texto deben proporcionar firmas digitales con sus ejecutables binarios.
