# <a name="desktop-win32-code-samples"></a>Ejemplos de código win32 de escritorio

Consulte también Ejemplos [de código.](https://developer.microsoft.com/windows/samples/)

## <a name="find-a-sample-for-the-api-youre-interested-in"></a>Busque un ejemplo para la API que le interesa.

Puede usar Microsoft Visual Studio para buscar en el código fuente de un repositorio completo para ver si se está demostrando el uso de una API de Windows determinada.

* Clone cualquiera de los repositorios enumerados en las secciones siguientes (o descargue el archivo ZIP) en el sistema de archivos local.
* A **continuación, busque en** archivos Visual Studio. Establezca **Buscar en** la carpeta en la que clonó o descargó y busque el nombre de la API que le interesa. Si se **comprueba Match case (Coincidencia** mayúsculas y minúsculas), se dan mejores resultados.
* Tenga cuidado de comprobar **Match whole word** si se puede llamar a la API de varias formas, por &mdash; ejemplo, **CreateMutex,** **CreateMutexA** y **CreateMutexW.** En ese caso, busque **CreateMutex** con **Match whole word** unchecked (Coincidir palabra completa sin activar).

> [!NOTE]
> Puede instalar el [Visual Studio](https://visualstudio.microsoft.com/downloads/) sin gastos. Hay disponible una edición Community &mdash;es gratuita para estudiantes, colaboradores de código abierto y particulares—. Por supuesto, puede hacer lo mismo con cualquier repositorio de ejemplos.
