# <a name="about-multilingual-user-interface"></a>Acerca de la interfaz de usuario multilingüe

La interfaz de usuario multilingüe (MUI) es una tecnología que proporciona a los usuarios una interfaz de usuario localizada para las aplicaciones globalizadas y la administración de recursos de idioma de interfaz de usuario en el sistema operativo Windows. Se proporciona compatibilidad para agregar la funcionalidad de MUI a las aplicaciones globalizadas para que se ejecuten en Windows Vista y versiones posteriores, así como en muchos sistemas operativos anteriores a Windows Vista. Los modelos de localización y administración de recursos de MUI mejoran el desarrollo, la prueba y la compatibilidad con software de uso internacional.

Las características de la tecnología de MUI incluyen:

-   Administración de lenguaje simplificada para la localización de software. MUI permite cambiar la configuración de idioma para el sistema operativo a un idioma compatible según las preferencias del usuario.
-   Tecnología de recursos innovadora basada en la división del código de aplicación binario de archivos de recursos de lenguaje. Puede agregar recursos para un idioma adicional sin necesidad de volver a compilar o volver a vincular la aplicación. Los recursos localizados adicionales se convierten en complementos opcionales.
-   Complete la interfaz de programación de aplicaciones (API). Windows Vista y versiones posteriores exponen la API de MUI que puede usar para agregar la funcionalidad de MUI a las aplicaciones globalizadas. Las funciones de administración de idiomas permiten a las aplicaciones admitir una amplia variedad de idiomas de la interfaz de usuario. La API también incluye funciones de carga de recursos que permiten al cargador de recursos cargar recursos en Windows Vista y versiones posteriores, así como en los sistemas operativos anteriores a Windows Vista.
-   Un conjunto de herramientas de administración de paquetes de idioma que proporcionan flexibilidad en la administración de imágenes de implementación internacional. Estas herramientas no están destinadas a los desarrolladores y, por lo tanto, no se incluyen en el SDK. Puede encontrar más información en el kit de instalación automatizada de Windows disponible en TechNet.

La ventaja más visible de MUI es que varios usuarios pueden compartir la misma estación de trabajo y ver la interfaz de usuario en distintos idiomas. Las empresas y los fabricantes OEM se beneficiarán de la funcionalidad que tienen para implementar, admitir y mantener imágenes multilingües con una sola instalación. Pero es posible que la ventaja principal de MUI se encuentre en las eficiencias obtenidas al desarrollar, compilar y dar servicio a la aplicación. Puede enviar un binario de funcionalidad principal aplicable a todas las plataformas, independientemente del idioma de la interfaz de usuario, lo que reduce significativamente los esfuerzos de desarrollo y pruebas. Si tiene que emitir una actualización o un Service Pack, se aplicará a todos los idiomas admitidos sin ningún esfuerzo de ingeniería adicional. Una posterior compatibilidad con otros idiomas se convierte en un proyecto de localización en lugar de un proyecto de desarrollo de software completo.
