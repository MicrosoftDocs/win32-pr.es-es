# <a name="media-streaming-enumerations"></a>Enumeraciones de streaming multimedia

La [API de streaming de multimedia](media-streaming-api-portal.md) proporciona las siguientes enumeraciones.

## <a name="in-this-section"></a>En esta sección



| Tema                                                   | Descripción                                                                               |
|---------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**ConnectionStatus**](connectionstatus.md)<br/> | Representa el estado del dispositivo en la red que se ha detectado por última vez.<br/>                |
| [**DeviceTypes**](devicetypes.md)<br/>           | Describe los tipos de dispositivo DLNA que admite la API de streaming de multimedia.<br/> |
| [**TransportState**](transportstate.md)<br/>     | Define los Estados de transporte disponibles tal y como se define en las directrices de UPnP.<br/>      |
| [**TransportStatus**](transportstatus.md)<br/>   | Define el estado de transporte disponible tal y como se define en las directrices de UPnP.<br/>      |
