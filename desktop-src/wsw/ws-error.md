# <a name="ws_error"></a>ERROR de WS \_

Tipo opaco que se usa para hacer referencia a un objeto de [error](errors.md) .


```C++
typedef struct _WS_ERROR WS_ERROR;
```



## <a name="remarks"></a>Observaciones

Este objeto no es seguro para subprocesos. Para obtener más información, vea [seguridad para subprocesos](thread-safety.md).
