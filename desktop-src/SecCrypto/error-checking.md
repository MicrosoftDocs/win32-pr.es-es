# <a name="error-checking-cryptography"></a>Comprobación de errores (criptografía)

Para C++, el valor devuelto siempre es **HRESULT** e indica si la llamada al método se ha hecho correctamente o no. Si la llamada no se ha podido llamar, el valor devuelto indica por qué se ha fallado.
