# <a name="about-pens"></a>Acerca de los lápices

Hay dos tipos de plumilla: cosmética y geométrica. Un *lápiz cosmético* se utiliza con aplicaciones que requieren líneas de ancho fijo y líneas que se dibujan rápidamente. Una aplicación CAD, por ejemplo, usa un lápiz cosmético para generar líneas ocultas, de sección, de centro y de dimensiones entre .015 y .022 pulgadas de ancho, independientemente del factor de escala. Un *lápiz geométrico* se usa con aplicaciones que requieren líneas escalables, líneas con estilos de extremo o de combinación únicos, y líneas más anchas que un solo píxel. Por ejemplo, una aplicación de hoja de cálculo usa un lápiz geométrico para definir cada una de las barras de un gráfico de barras como una línea ancha.
