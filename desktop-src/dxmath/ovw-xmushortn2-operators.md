# <a name="xmushortn2-operators"></a>Operadores XMUSHORTN2

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección

|---------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**operador =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmushortn2-operator-assign)<br/> | Asigna los datos del componente de vector de una instancia de `XMUSHORTN2` a la instancia actual de `XMUSHORTN2` .<br/> |
