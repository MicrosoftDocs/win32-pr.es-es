# <a name="id3dxlinegetpattern-method"></a>ID3DXLine:: GetPattern (método)

Obtiene el patrón punteado de línea.

## <a name="syntax"></a>Sintaxis


```C++
DWORD GetPattern();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **DWORD**](../winprog/windows-data-types.md)**

Devuelve el patrón punteado de línea: 1 es opaco, 0 es transparente.
