# <a name="performance-windows-imaging-component"></a>Rendimiento (componente de Windows Imaging)

El objetivo de Microsoft es que el rendimiento del procesamiento sin procesar a través de la interfaz de Windows Imaging Component (WIC) sea equivalente a las herramientas y aplicaciones de procesamiento sin formato líderes del sector. Para proporcionar una guía de rendimiento, se realizó un conjunto de pruebas exhaustivas que comparan el rendimiento de las herramientas de flujo de trabajo sin procesar conocidas. Las pruebas incluyen las versiones de sistema operativo de Windows y Apple Macintosh de los productos de procesamiento sin procesar, archivos sin procesar de ejemplo de cámaras de Reflex de una sola lente (SLR) actuales y una serie de funciones que representan los pasos comunes del flujo de trabajo.

En función de estos resultados, Microsoft sugiere las siguientes recomendaciones a los asociados para el rendimiento del procesamiento sin procesar:

-   El rendimiento de línea base debe ser igual o superior a los tiempos de procesamiento de las aplicaciones y herramientas de procesamiento sin formato proporcionadas por el fabricante actual.
-   Los objetivos de rendimiento óptimos deben coincidir o superar la aplicación de mayor rendimiento sin sacrificar la calidad (en estas pruebas, la cámara RAW de Photoshop generalmente se realiza más rápido).
