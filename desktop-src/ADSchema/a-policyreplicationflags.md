# <a name="policy-replication-flags-attribute"></a>Atributo Policy-Replication-Flags

Determina qué propiedades de LSA se replican en los clientes.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Directiva-replicación-marcas             |
| Nombre para mostrar de LDAP | policyReplicationFlags               |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.633               |
| System-ID-GUID    | 19405b96-3cfa-11d1-a9c0-0000f80367c1 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
