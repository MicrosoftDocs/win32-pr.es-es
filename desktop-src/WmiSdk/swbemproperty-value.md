# <a name="swbempropertyvalue-property"></a>Propiedad SWbemProperty.Value

La **propiedad Value** del objeto [**SWbemProperty**](swbemproperty.md) define el valor variant de la propiedad WMI. Esta es la propiedad de automatización predeterminada de este objeto .

Para obtener una explicación de esta sintaxis, vea [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Syntax


```VB
SWbemProperty.Value As Variant
```
