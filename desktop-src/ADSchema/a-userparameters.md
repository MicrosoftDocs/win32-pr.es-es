# <a name="user-parameters-attribute"></a>User-Parameters atributo)

Parámetros del usuario. Apunta a una cadena Unicode que se reserva para su uso por parte de las aplicaciones. Esta cadena puede ser una cadena nula o puede tener cualquier número de caracteres antes del carácter nulo de terminación. Los productos de Microsoft usan este miembro para almacenar los datos de usuario específicos del programa individual.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | User-Parameters                             |
| Nombre para mostrar de LDAP | userParameters                              |
| Tamaño              | \-                                          |
| Actualizar privilegio  | Modificado por aplicaciones.                   |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.138                      |
| System-ID-GUID    | bf967a6d-0de6-11d0-a285-00aa003049e2        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
