# <a name="ice33"></a>ICE33

ICE33 procesa las entradas de la [tabla del registro](registry-table.md) y puede emitir una advertencia para cada entrada de la tabla que registra clases, extensiones de nombre de archivo, ProgID, verbos de Shell, AppIDs de servidor remoto, tipos MIME o bibliotecas de tipos. Las advertencias están obsoletas y se pueden omitir.
