# <a name="ms-ds-sitename-attribute"></a>Atributo ms-DS-SiteName

Enumera el nombre de sitio que corresponde al controlador de dominio.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | ms-DS-SiteName                              |
| Ldap-Display-Name | msDS-SiteName                               |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1961                     |
| System-Id-Guid    | 98a7f36d-3595-448a-9e6f-6b8965baed9c        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
