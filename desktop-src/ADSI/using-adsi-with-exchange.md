# <a name="using-adsi-with-exchange"></a>Usar ADSI con Exchange

Para Exchange 2000, la información sobre el uso de ADSI con Exchange se incluye en el SDK de Exchange 2000. Para obtener más información, vea [tareas de administración de ADSI](/previous-versions/office/developer/exchange-server-2003/aa125368(v=exchg.65)).

En esta sección se pueden encontrar las siguientes tareas:

-   Creación de una dirección URL de usuario
-   Crear rutas de acceso Active Directory
-   Enumerar servidores Exchange 2000 con ADSI
-   Manipular atributos de extensión con ADSI
-   Agregar o quitar un objeto de administrador con ADSI
-   Enumerar propiedades de objetos de Exchange con ADSI/ADO
-   Recuperar un nombre distintivo de Exchange heredado con ADSI
-   Buscar el nombre de la organización de Exchange mediante ADSI
-   Búsqueda de listas de direcciones de Exchange con ADSI
-   Crear una lista de distribución mediante CDOEXM y ADSI
-   Configuración de la restricción de mensajes en un servidor virtual SMTP mediante ADSI

En el caso de Exchange 5,5, la referencia ADSI y el uso de la información se pueden encontrar en la guía de [intercambio de ADSI](/previous-versions/office/developer/exchange-server-2007/aa579394(v=exchg.80)) .

En esta sección se pueden encontrar las siguientes tareas:

-   Ver y modificar el esquema de Exchange Server
-   Visualización de las propiedades sin procesar de un objeto de Exchange
-   Creación de un buzón de Exchange
-   Configuración del descriptor de seguridad en un buzón de Exchange
-   Manipular los descriptores de seguridad y los SID
-   Eliminar un objeto de buzón
-   Creación de un destinatario personalizado
-   Creación de un contenedor de destinatarios
-   Obtener el nombre de la organización y del sitio desde un servidor
-   Mostrar la versión de Exchange Server de todos los servidores de la organización
-   Búsqueda del servidor principal de un buzón
-   Recuperación de direcciones de correo electrónico
-   Obtener acceso a entradas ocultas o eliminadas
-   Recuperando los cambios realizados en el servicio de directorio
-   Crear una lista de distribución a partir de los resultados de una consulta
-   Obtener o modificar el tamaño del mensaje
-   Uso de códigos de error LDAP para solucionar problemas
