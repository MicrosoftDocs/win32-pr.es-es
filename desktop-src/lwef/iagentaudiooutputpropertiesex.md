# <a name="iagentaudiooutputpropertiesex"></a>IAgentAudioOutputPropertiesEx

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

IAgentAudioOutputPropertiesEx se deriva de la interfaz IAgentAudioOutputPropertiesEx. Incluye todos los métodos IAgentAudioOutputPropertiesEx y proporciona acceso a funciones adicionales.

**Métodos en orden de Vtable**



| Métodos IAgentAudioOutputPropertiesEx                         | Descripción                                     |
|---------------------------------------------------------------|-------------------------------------------------|
| [**GetStatus**](iagentaudiooutputpropertiesex--getstatus.md) | Devuelve el estado del canal de salida de audio. |
