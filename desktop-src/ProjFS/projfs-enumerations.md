# <a name="projfs-enumerations"></a>Enumeraciones de ProjFS

Las enumeraciones siguientes se declaran en projectedfslib. h.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|-|-|
| [**PRJ_CALLBACK_DATA_FLAGS**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_callback_data_flags) | Marcas que controlan lo que se devuelve en la enumeración. |
| [**PRJ_COMPLETE_COMMAND_TYPE**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_complete_command_type) | Especifica los tipos de comando. |
| [**PRJ_FILE_STATE**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_file_state) | El estado de un elemento. |
| [**PRJ_NOTIFICATION**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_notification) | Un valor de notificación especificado al enviar la notificación en una devolución de llamada. |
| [**PRJ_NOTIFY_TYPES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_notify_types) | Tipos de notificaciones que describen un cambio en el archivo o la carpeta. |
| [**PRJ_PLACEHOLDER_ID**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_placeholder_id) | Define la longitud de un identificador de marcador de posición. |
| [**PRJ_STARTVIRTUALIZING_FLAGS**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_startvirtualizing_flags) | Marcas que se van a proporcionar al iniciar una instancia de virtualización. |
| [**PRJ_UPDATE_FAILURE_CAUSES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_update_failure_causes) | Descripción de la razón por la que se produjo un error en una actualización. |
| [**PRJ_UPDATE_TYPES**](/windows/win32/api/projectedfslib/ne-projectedfslib-prj_update_types) | Marcas para especificar si se permitirán actualizaciones según el estado de un archivo o un directorio en el disco. |
