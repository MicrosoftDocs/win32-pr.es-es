# <a name="function-status"></a>Estado de la función

Las funciones de administración de red devuelven cero si se ejecutan correctamente; un código de retorno distinto de cero indica un error. Dado que las funciones de administración de red utilizan RPC, las definiciones de error incluyen códigos de error de RPC. Para obtener más información, consulte [códigos de error de administración de red](network-management-error-codes.md).
