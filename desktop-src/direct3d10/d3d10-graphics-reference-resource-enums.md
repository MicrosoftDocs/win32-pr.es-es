# <a name="resource-enumerations-direct3d-10-graphics"></a>Enumeraciones de recursos (gráficos de Direct3D 10)

Las enumeraciones siguientes se usan para especificar información sobre cómo se crean y se accede a los recursos durante la representación.



| Enumeración                                                     | Descripción                                                                                                               |
|-----------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------|
| [**D3D10 \_ BIND \_ FLAG**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_bind_flag)                    | Identifica cómo usará la canalización un recurso y determina a qué fases de canalización se puede enlazar un recurso. |
| [**MARCA DE ACCESO DE CPU D3D10 \_ \_ \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_cpu_access_flag)       | Especifica los tipos de acceso de CPU permitidos para un recurso.                                                                 |
| [**D3D10 \_ DSV \_ DIMENSION**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_dsv_dimension)            | Especifica cómo acceder a un recurso que se usa en una vista de galería de símbolos de profundidad.                                                  |
| [**MAPA D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_map)                                 | Identifica un recurso que se va a asignar para lectura y escritura por parte de la CPU.                                                    |
| [**D3D10 \_ MAP \_ FLAG**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_map_flag)                      | Especifica cómo debe responder la CPU a los métodos map cuando la GPU aún no ha terminado con el recurso.               |
| [**DIMENSIÓN DE RECURSOS D3D10 \_ \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_resource_dimension)  | Identifica el tipo de recurso que está en uso.                                                                           |
| [**MARCA MISC DEL RECURSO D3D10 \_ \_ \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_resource_misc_flag) | Identifica opciones de creación de recursos menos comunes.                                                                         |
| [**D3D10 \_ RTV \_ DIMENSION**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_rtv_dimension)            | Especifica cómo acceder a un recurso usado en una vista de destino de representación.                                                          |
| [**USO DE D3D10 \_**](/windows/desktop/api/D3D10/ne-d3d10-d3d10_usage)                             | Identifica cómo se espera que la CPU o la GPU lean y escriban un recurso.                                   |
