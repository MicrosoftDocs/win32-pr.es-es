# <a name="about-snmp-versions"></a>Acerca de las versiones de SNMP

El marco de administración de redes estándar de Internet estándar, que se describe en RFC 1155, 1157 y 1213, se denomina el marco de trabajo de SNMP versión 1 (SNMPv1). Las partes relevantes del marco propuesto para la versión 2C del Protocolo simple de administración de redes (SNMPv2C) se describen en RFC 1901 a 1908.

La API WinSNMP es compatible con la funcionalidad del protocolo SNMP que se describe en las RFC correspondientes de Internet. WinSNMP no coloca restricciones en el uso de SNMPv1 o SNMPv2C por parte de aplicaciones WinSNMP.

Una entidad de administración puede admitir una versión diferente de SNMP que la que admite la aplicación WinSNMP. La implementación de Microsoft WinSNMP realiza las traducciones adecuadas desde SNMPv1 hasta SNMPv2C de acuerdo con la RFC pertinente.
