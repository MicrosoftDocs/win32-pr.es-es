# <a name="d1247-dynamic-copy-from-memory"></a>D1247: copia dinámica de la memoria

CopyFromMemory no puede copiar memoria en un mapa de bits creado a partir de una textura dinámica.



|             |         |
|-------------|---------|
| Nivel de error | Advertencia |



 

## <a name="possible-causes"></a>Causas posibles

Se llamó a CopyFromMemory en un mapa de bits que se creó como D3D11 \_ Usage \_ Dynamic.

## <a name="possible-fixes"></a>Posibles correcciones

En su lugar, utilice Map y unmapping para este tipo de mapa de bits.
