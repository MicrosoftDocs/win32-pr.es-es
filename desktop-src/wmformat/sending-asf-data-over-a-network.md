# <a name="sending-asf-data-over-a-network"></a>Envío de datos ASF a través de una red

El SDK de Windows Media Format admite dos maneras diferentes de enviar datos ASF a través de una red:

-   La aplicación puede difundir los datos ASF mediante el protocolo HTTP.
-   La aplicación puede enviar los datos ASF a un punto de publicación en un servidor de Windows Media. A continuación, el servidor distribuye los datos.

En los temas siguientes se describen estas características:

-   [Difusión de datos de ASF](broadcasting-asf-data.md)
-   [Envío de datos ASF a un punto de publicación](sending-asf-data-to-a-publishing-point.md)
