# <a name="ms-sql-name-attribute"></a>Atributo de nombre de MS-SQL

Nombre de una instancia de SQL Server. El valor predeterminado es MSSQLSERVER.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Nombre MS-SQL                                 |
| Nombre para mostrar de LDAP | Nombre mS-SQL                                 |
| Tamaño              | \-                                          |
| Actualizar privilegio  | El sistema establece este valor.            |
| Frecuencia de actualización  | En el programa de instalación del sistema.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1363                     |
| System-ID-GUID    | 3532dfd8-ccee-11d2-9993-0000f87a57d4        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
