# <a name="span-idvspixenginepixenginetexturesliceindexspanpixenginetexturesliceindex-structure"></a><span id="vspixengine.pixenginetexturesliceindex"></span>Estructura PixEngineTextureSliceIndex

Representa el índice de un segmento de textura.

## <a name="syntax"></a>Sintaxis


```C++
} PixEngineTextureSliceIndex;
```

## <a name="members"></a>Miembros

**arrayIndex**  
Cuando la textura es una matriz de textura, el índice de la matriz del segmento.

**mipmapLevel**  
Nivel de mipmap del segmento.

**sampleIndex**  
Índice de ejemplo del segmento.

**zSlice**  
Profundidad (z) del segmento.
