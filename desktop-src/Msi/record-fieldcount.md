# <a name="recordfieldcount-property"></a>Propiedad Record.FieldCount

La **propiedad FieldCount** del [**objeto Record**](record-object.md) es una propiedad de solo lectura que devuelve el número de campos del registro. El acceso de lectura a campos más allá de este recuento devuelve valores NULL. Se produce un error en el acceso de escritura.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Record.FieldCount
```
