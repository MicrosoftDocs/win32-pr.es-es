# <a name="d1201-pushaxisalignedclip-with-antialiased-multisampled-target"></a>D1201: PushAxisAlignedClip con un destino de muestreo multialias

PushAxisAlignedClip se usó con un destino de muestreo multimuestreado. La llamada se seguirá emitiendo, pero es posible que se produzcan artefactos si los límites del rectángulo de recorte no están alineados por píxeles. Además, es posible que el resultado representado no represente el patrón de Multimuestra esperado.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación interopera una textura de Direct3D multimuestreada como un destino de Direct2D y se denomina PushAxisAlignedClip en el destino.

## <a name="possible-fixes"></a>Posibles correcciones

Evite las texturas multimuestreadas interoperables si es importante la funcionalidad de recorte. El uso de capas o clips con estos tipos de recursos puede suponer una reducción del rendimiento o resultados inesperados. Esto es especialmente cierto si los límites del clip no están alineados por píxeles. Si es posible, use un medio alternativo para enmascarar el contenido en lugar de un clip, como un tramo de Direct3D. O bien, use un intermediario en lugar de un recurso de muestreo multimuestreado. Si el recurso de muestreo múltiple está pensado para el contenido 2D de suavizado de contorno, considere la posibilidad de que el suavizado de contorno primitivo basado en Direct2d sea suficiente.
