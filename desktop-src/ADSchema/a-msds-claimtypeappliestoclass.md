# <a name="ms-ds-claim-type-applies-to-class-attribute"></a>atributo MS-DS-Claim-Type-appliesTo-to-Class

Para un objeto de tipo de notificación, este atributo vinculado apunta a las clases de entidad de seguridad de AD que se deben emitir para las notificaciones. (Por ejemplo, un vínculo a la clase de usuario).



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-DS-Claim-Type-se aplica a la clase       |
| Nombre para mostrar de LDAP | msDS-ClaimTypeAppliesToClass            |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2100                 |
| System-ID-GUID    | 6afb0e4c-d876-437c-aeb6-c3e41454c272    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
