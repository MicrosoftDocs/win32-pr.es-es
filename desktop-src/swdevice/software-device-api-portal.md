# <a name="software-device-api"></a>API de dispositivo de software

## <a name="purpose"></a>Propósito

Puede usar la API de dispositivo de software para crear un dispositivo PnP desde una aplicación. La API le permite enumerar el dispositivo como un elemento secundario de cualquier dispositivo primario existente. La API también le permite registrar interfaces de dispositivo en los dispositivos enumerados y establecer propiedades para los dispositivos e interfaces.

## <a name="developer-audience"></a>Audiencia de desarrolladores

La API de dispositivo de software está diseñada para que la usen los desarrolladores que quieran publicar la funcionalidad del dispositivo en el "directorio" pnP o para cargar un controlador de dispositivo.
