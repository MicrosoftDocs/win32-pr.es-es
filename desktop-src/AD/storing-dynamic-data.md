# <a name="storing-dynamic-data"></a>Almacenamiento de datos dinámicos

La mayoría de los datos almacenados Active Directory Domain Services datos son estáticos. Por ejemplo, los objetos de directorio que representan usuarios, grupos, impresoras, servicios de red, entre otros, suelen ser de larga duración y cambian con poca frecuencia.

En Windows Server 2003 y versiones posteriores de Windows, Active Directory Domain Services proporciona compatibilidad con las siguientes características que permiten a las aplicaciones usar el directorio para almacenar datos dinámicos:

-   [Objetos dinámicos](dynamic-objects.md)
-   [Clases auxiliares dinámicas](dynamic-auxiliary-classes.md)
