# <a name="mf_stf_version_date-attribute"></a>\_Atributo Date de la versión MF STF \_ \_

Este atributo almacena un [FILETIME](/windows/desktop/api/minwinbase/ns-minwinbase-filetime) que representa la marca de fecha y hora de un generador de transformaciones de sensor.

## <a name="data-type"></a>Tipo de datos

**UINT64**

| Requisito | Value |
|-------------------------------------|------------------------------------------------------------------------------------|
| Cliente mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows 10, versión 1703 \[\]<br/>                          |
| Servidor mínimo compatible<br/> | No se admite ninguno<br/>                                                          |
