# <a name="object-model-for-mergemoddll-version-20"></a>Modelo de objetos para Mergemod.dll versión 2,0

El modelo de objetos de Mergemod.dll versión 2,0 se organiza como se indica a continuación.

![modelo de objetos para mergemod.dll versión 2,0](images/mergobj2.png)

El [objeto Merge](merge-object.md) es el objeto principal del modelo. El [**objeto GetFiles**](getfiles-object.md) es un objeto secundario. Las dependencias son colecciones de [**objetos de dependencia**](dependency-object.md). Los errores son colecciones de [**objetos de error**](error-object.md). ConfigurableItems son colecciones de [**objetos ConfigurableItem**](configurableitem-object.md).
