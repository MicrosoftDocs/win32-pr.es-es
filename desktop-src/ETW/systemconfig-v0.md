# <a name="systemconfig_v0-class"></a>Clase SystemConfig \_ V0

Esta clase es la clase primaria para los eventos de configuración de hardware.

La sintaxis siguiente se simplifica a partir del código MOF.

## <a name="syntax"></a>Sintaxis

``` syntax
[Guid("{01853a65-418f-4f36-aefc-dc0f1d2fd235}"), EventVersion(0)]
class SystemConfig_V0 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Miembros

La **clase SystemConfig \_ V0** no define ningún miembro.

## <a name="remarks"></a>Comentarios

Para los eventos de configuración de hardware Windows XP, consulte la [**clase HWConfig.**](hwconfig.md)
