# <a name="sdk-versions"></a>Versiones del SDK

En esta documentación se trata el SDK de Windows Media Format 7 a través del SDK de Windows Media Format 11. Los métodos, estructuras, funciones y enumeraciones individuales incluyen los requisitos de versión en sus temas de referencia.

Windows Media Format 11 SDK solo se admite en Windows Vista y Windows XP.

El SDK de Windows Media Format 9,5 solo se admite en Windows XP. Si escribe una aplicación basada en el SDK de Windows Media Format 9,5, la aplicación no se ejecutará en ningún otro sistema operativo.

Algunas características del SDK de Windows Media Format 9 series también requieren Windows XP. En los temas de referencia individuales relacionados con esas características se llama a ese requisito.
