# <a name="id3d12heapgetdesc-method"></a>Método ID3D12Heap::GetDesc

Obtiene la descripción del montón.

## <a name="syntax"></a>Sintaxis


```C++
D3D12_HEAP_DESC GetDesc();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **D3D12 \_ HEAP \_ DESC**](/windows/desktop/api/d3d12/ns-d3d12-d3d12_heap_desc)**

Devuelve la [**estructura D3D12 \_ HEAP \_ DESC**](/windows/desktop/api/d3d12/ns-d3d12-d3d12_heap_desc) que describe el montón.
