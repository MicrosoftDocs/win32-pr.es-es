# <a name="locale_sopentypelanguagetag"></a>configuración regional \_ SOPENTYPELANGUAGETAG

**Windows 7 y versiones posteriores:** Etiqueta de idioma OpenType que se usa para recuperar las características tipográficas de una fuente de forma cultural apropiada. Para obtener más información, vea [**OPENTYPE ( \_ etiqueta**](opentype-tag.md)).
