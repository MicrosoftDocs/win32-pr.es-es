# <a name="aedebug"></a>AeDebug

**AEDebug** define las opciones de depuración remota para com.

## <a name="registry-key"></a>Clave del Registro

**HKEY \_ local \_ Machine \\ software \\ Microsoft \\ Windows NT \\ CurrentVersion \\ DebugObjectRPCEnabled \\ AEDebug**



| Valor del registro               | Descripción                                           |
|------------------------------|-------------------------------------------------------|
| [**Automático**](auto.md)         | Determina si el depurador se inicia automáticamente. |
| [**instantáneas**](debugger.md) | Almacena el nombre del depurador.                             |
