# <a name="hatch-brush"></a>Pincel de trama

GDI mantiene seis pinceles de trama lógica predefinidos. Los siguientes rectángulos se han pintado mediante los seis pinceles de trama predefinidos.

![Ilustración que muestra seis cuadros, uno rellenado por cada uno de los seis pinceles de trama](images/hatchbrush.png)

Una aplicación puede crear un pincel de trama mediante una llamada a la función [**CreateHatchBrush**](/windows/desktop/api/Wingdi/nf-wingdi-createhatchbrush) , especificando uno de los seis estilos de trama.
