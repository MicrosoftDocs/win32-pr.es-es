# <a name="ui_pkey_colormode"></a>UI \_ PKEY \_ ColorMode

Identifica la propiedad de la interfaz de usuario \_ PKEY \_ ColorMode.

```
propertyDescription
   name = UI_PKEY_ColorMode
   shellPKey = UI_PKEY_ColorMode
   formatID = 00000402-7363-696e-8441798acf5aebb7
   propID = 402
   typeInfo
      type = UI_SWATCHCOLORMODE
```

## <a name="remarks"></a>Observaciones

El valor de la propiedad es de la enumeración [**\_ SWATCHCOLORMODE**](/windows/desktop/api/Uiribbon/ne-uiribbon-ui_swatchcolormode) de la interfaz de usuario.
