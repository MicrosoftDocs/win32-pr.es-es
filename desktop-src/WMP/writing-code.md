# <a name="writing-code"></a>Escritura de código

La eficacia real de las máscaras proviene de poder responder a los eventos. Quiere que la máscara sea capaz de hacer algo cuando el usuario hace clic en un botón y también desea responder a los cambios que se producen en Windows Media Player, como el progreso del archivo multimedia digital o el flujo que se está reproduciendo.

En las secciones siguientes se describe la escritura de código para máscaras:



| Sección                                | Descripción                                 |
|----------------------------------------|---------------------------------------------|
| [Controlar eventos](handling-events.md) | Explica cómo controlar los eventos en el código de máscara. |
| [Usar JScript](using-jscript.md)     | Explica cómo escribir código de script en máscaras. |
