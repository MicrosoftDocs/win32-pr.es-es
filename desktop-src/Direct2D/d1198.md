# <a name="d1198-copy-to-read-only-bitmap"></a>D1198: copiar en Read-Only mapa de bits

Se ha intentado copiar datos en un mapa de bits de solo lectura o inmutable.






 

## <a name="possible-causes"></a>Causas posibles

Se ha intentado copiar datos en un mapa de bits de solo lectura o inmutable.

## <a name="possible-fixes"></a>Posibles correcciones

Compruebe que los mapas de bits que se escriben en no se crean como inmutables.
