# <a name="about-colors"></a>Acerca de los colores

El color se puede usar para comunicar ideas, mostrar relaciones entre elementos y mejorar el atractivo y la calidad de la salida. Windows permite a las aplicaciones detectar las funcionalidades de color de determinados dispositivos y elegir entre los colores disponibles los que mejor se adapten a sus necesidades.

Esta información general proporciona información sobre los temas siguientes:

-   [Conceptos básicos de color](color-basics.md)
-   [Paletas de colores](color-palettes.md)

Aunque no se describe en esta información general, la coincidencia de colores de imagen es una característica importante de la administración de colores que ayuda a garantizar que las imágenes de color tienen el mismo aspecto, tanto si se muestran en pantalla como si se imprimen en papel. Para obtener más información, [vea Windows Color System](/previous-versions//dd372446(v=vs.85)).
