# <a name="device-contexts"></a>Contextos de dispositivo

Un *contexto de dispositivo* es una estructura que define un conjunto de objetos gráficos y sus atributos asociados, así como los modos gráficos que afectan a la salida. Los *objetos gráficos* incluyen un lápiz para el dibujo de líneas, un pincel para dibujar y rellenar un mapa de bits para copiar o desplazar partes de la pantalla, una paleta para definir el conjunto de colores disponibles, una región para el recorte y otras operaciones, y una ruta de acceso para dibujar y dibujar operaciones. El resto de esta sección se divide en las tres áreas siguientes.

-   [Acerca de los contextos de dispositivo](about-device-contexts.md)
-   [Uso de las funciones de contexto de dispositivo](using-the-device-context-functions.md)
-   [Referencia de contexto de dispositivo](device-context-reference.md)
