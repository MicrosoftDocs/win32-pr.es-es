# <a name="acs-max-peak-bandwidth-attribute"></a>Atributo ACS-Max-pico-ancho de banda

El ancho de banda máximo que se puede reservar.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | ACS-Max-pico-ancho de banda               |
| Nombre para mostrar de LDAP | aCSMaxPeakBandwidth                  |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.767               |
| System-ID-GUID    | 7f561284-5301-11d1-a9c5-0000f80367c1 |
| Sintaxis            | [**Interval**](s-interval.md)       |
