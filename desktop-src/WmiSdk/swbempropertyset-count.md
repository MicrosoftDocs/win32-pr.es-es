# <a name="swbempropertysetcount-property"></a>Propiedad SWbemPropertySet.Count

Use la **propiedad de** automatización Count del objeto [**SWbemPropertySet**](swbempropertyset.md) para determinar cuántos elementos hay en la **colección SWbemPropertySet.** Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemPropertySet.Count As Integer
```
