# <a name="file-menu"></a>Menú Archivo

En el menú **archivo** se proporcionan los siguientes comandos:



<table>
<colgroup>
<col style="width: 50%" />
<col style="width: 50%" />
</colgroup>
<thead>
<tr class="header">
<th>Get-Help</th>
<th>Descripción</th>
</tr>
</thead>
<tbody>
<tr class="odd">
<td><strong>Nuevo...</strong></td>
<td>Borra la topología existente y quita los nodos de topología del panel de <strong>topología</strong>.<br/>
<ul>
<li>Si una topología está abierta actualmente, pide al usuario que guarde la topología. Para obtener información sobre cómo guardar la topología, vea <a href="saving-and-loading-topologies.md">Guardar y cargar topologías</a>.<br/></li>
<li>Crea una topología vacía. El usuario puede agregar nodos de topología a la nueva topología. Para obtener más información, vea <a href="building-topologies-by-using-topoedit.md">crear topologías con TopoEdit</a>.<br/></li>
</ul></td>
</tr>
<tr class="even">
<td><strong>Presentar archivo...</strong></td>
<td>Crea una topología de reproducción para un archivo multimedia. TopoEdit crea y conecta automáticamente los nodos en la topología. Para obtener más información, vea <a href="building-a-playback-topology-with-topoedit.md">crear una topología de reproducción con TopoEdit</a>.</td>
</tr>
<tr class="odd">
<td><strong>Representar transcodificación...</strong></td>
<td>Crea una topología de transcodificación. Para obtener más información, vea <a href="building-a-transcode-topology-with-topoedit.md">crear una topología de transcodificación con TopoEdit</a>.<br/>
<blockquote>
[!Note]<br />
Esta característica requiere Windows 7.
</blockquote>
<br/></td>
</tr>
<tr class="even">
<td><strong>Abrir...</strong></td>
<td>Carga una topología existente guardada como un archivo XML. Vea <a href="saving-and-loading-topologies.md">Guardar y cargar topologías</a>.</td>
</tr>
<tr class="odd">
<td><strong>Guardar...</strong></td>
<td>Guarda la topología abierta actualmente como un archivo XML. Vea <a href="saving-and-loading-topologies.md">Guardar y cargar topologías</a>.</td>
</tr>
<tr class="even">
<td><strong>Salir</strong></td>
<td>Cierra la aplicación TopoEdit.</td>
</tr>
</tbody>
</table>
