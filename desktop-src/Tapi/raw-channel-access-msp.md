# <a name="raw-channel-access-msp"></a>MSP de acceso al canal sin formato

El acceso al canal sin procesar MSP (Rcsmsp.dll) realiza el streaming a través de ksproxy a RCA, que permite el streaming a través de dispositivos de especificación de interfaz de controlador de red (NDIS) orientados a la conexión, como tarjetas ATM e ISDN. La ndptsp, que es el TSP proporcionado para los adaptadores de WAN NDIS, crea el MSP de RCA. Con la versión 2000 de Windows, solo se implementa el streaming de audio.

El MSP de acceso al canal sin formato se instala con Windows 2000.
