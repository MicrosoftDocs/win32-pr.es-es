# <a name="regpath"></a>RegPath

El tipo de datos RegPath es una cadena de texto que contiene una ruta de acceso del registro. Las rutas de acceso del registro pueden incluir propiedades, al igual que con el tipo de datos con [formato](formatted.md) . Un RegPath no puede comenzar ni terminar con una barra diagonal inversa ( \\ ). La clave de \[ \# archivo \] y la \[ clave \] de $Component se pueden incrustar en la ruta de acceso o precedidas de otros caracteres.
