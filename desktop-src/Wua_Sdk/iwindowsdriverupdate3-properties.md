# <a name="iwindowsdriverupdate3-properties"></a>Propiedades de IWindowsDriverUpdate3

La **interfaz IWindowsDriverUpdate3** define la siguiente propiedad.



| Propiedad                                               | Descripción                                                                                                             |
|--------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**BrowseOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate3-get_browseonly) | Obtiene un valor booleano que indica si una actualización solo se puede detectar explorando las actualizaciones disponibles. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
-   [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate2)
