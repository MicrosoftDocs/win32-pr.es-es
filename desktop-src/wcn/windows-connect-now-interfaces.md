# <a name="windows-connect-now-interfaces"></a>Windows Connect Now Interfaces

La [WINDOWS CONNECT NOW API](portal.md) (WCNAPI) define y utiliza las interfaces siguientes.



| Interfaz                                      | Descripción                                                                                                |
|------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IWCNConnectNotify**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcnconnectnotify) | Reciba una notificación de éxito o error cuando se complete Windows Connect Now sesión de conexión.<br/> |
| [**IWCNDevice**](/windows/desktop/api/WcnDevice/nn-wcndevice-iwcndevice)               | Configura el dispositivo e inicia la sesión.<br/>                                                |
