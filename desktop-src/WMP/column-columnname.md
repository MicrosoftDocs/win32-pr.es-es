# <a name="columncolumnname"></a>COLUMN.columnName

El **atributo columnName** especifica o recupera un nombre de columna en el control **PLAYLIST.**

``` syntax
        elementID.columnName
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una cadena de lectura y **escritura.**

## <a name="remarks"></a>Comentarios

El nombre de columna es el nombre descriptivo que aparece en el encabezado de la columna.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media serie 9 o posterior<br/> |
