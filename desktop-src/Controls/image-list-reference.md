# <a name="image-lists"></a>Listas de imágenes

Esta sección contiene información sobre los elementos de programación utilizados con listas de imágenes.

### <a name="overviews"></a>Temas de introducción



| Tema                          | Contenido                                                                                                            |
|--------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [Listas de imágenes](image-lists.md) | Una lista de imágenes es una colección de imágenes del mismo tamaño, a las que su índice puede hacer referencia a cada una de ellas.<br/> |

| [**ImageList \_ SetImageCount**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_setimagecount)           | Cambia el tamaño de una lista de imágenes existente. <br/>                                                                                                                                                                                                                                          |
| [**ImageList \_ SetOverlayImage**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_setoverlayimage)       | Agrega una imagen especificada a la lista de imágenes que se usarán como máscaras superpuestas. Una lista de imágenes puede tener hasta cuatro máscaras superpuestas en la versión 4.70 y anteriores y hasta 15 en la versión 4.71. La función asigna un índice de máscara de superposición a la imagen especificada. <br/>                   |
| [**ImageList \_ Write**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_write)                           | Escribe una lista de imágenes en una secuencia. <br/>                                                                                                                                                                                                                                        |
| [**ImageList \_ WriteEx**](/windows/desktop/api/Commctrl/nf-commctrl-imagelist_writeex)                       | Escribe una lista de imágenes en una secuencia. <br/>                                                                                                                                                                                                                                        |
