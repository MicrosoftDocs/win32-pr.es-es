# <a name="embedding-the-windows-media-player-control"></a>Incrustar el control Media Player de Windows

Además de incrustar el control de Media Player de Windows en una página web como se describe en la sección anterior, puede insertarlo en Microsoft Office documentos y en programas personalizados que cree mediante uno de varios lenguajes de programación, incluidos Microsoft Visual Basic, C++ y C#.

En las secciones siguientes se proporciona información acerca del uso del control en varios entornos de programación.



| Sección                                                                                                                                  | Descripción                                                                                             |
|------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------|
| [Incrustar el control Media Player de Windows en una página web](embedding-the-windows-media-player-control-in-a-web-page.md)                 | En esta sección se describe el uso del control en una página web.                                                  |
| [Incrustar el control Media Player de Windows en un documento de Office](embedding-the-windows-media-player-control-in-an-office-document.md) | En esta sección se describe el uso del control en un documento de Microsoft Office.                                |
| [Incrustar el control Media Player de Windows en un programa personalizado](embedding-the-windows-media-player-control-in-a-custom-program.md)     | En esta sección se describe el uso del control en una aplicación basada en Windows escrita en Visual Basic o en C++. |
