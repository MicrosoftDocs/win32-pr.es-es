# <a name="ms-ds-claim-type-applies-to-class-attribute"></a>Atributo ms-DS-Claim-Type-Applies-To-Class

Para un objeto de tipo de notificación, este atributo vinculado apunta a las clases de entidad de seguridad de AD para las que se deben emitir notificaciones. (Por ejemplo, un vínculo a la clase de usuario).
