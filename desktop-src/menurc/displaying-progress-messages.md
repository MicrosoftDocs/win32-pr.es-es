# <a name="displaying-progress-messages"></a>Mostrar mensajes de progreso

De forma predeterminada, RC se compila sin mostrar los mensajes de progreso. Sin embargo, puede especificar que RC muestre los mensajes de progreso. Para ello, use la **opción /v.**

En el ejemplo siguiente, RC informa de su progreso a medida que compila el archivo de definición de recursos Sample.rc y crea el archivo de recursos compilado Sample.res:

**rc /v sample.rc**
