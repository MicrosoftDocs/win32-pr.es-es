# <a name="about-gdi"></a>Acerca de GDI+

Windows GDI+ es la parte del sistema operativo Windows XP o del sistema operativo Windows Server 2003 que proporciona gráficos vectoriales bidimensionales, imágenes y tipografía. GDI+ mejora en Windows Interfaz de dispositivo gráfico (GDI) (la interfaz de dispositivo gráfico incluida con versiones anteriores de Windows) agregando nuevas características y optimizando las características existentes.
