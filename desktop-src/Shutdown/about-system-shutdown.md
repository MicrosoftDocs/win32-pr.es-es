# <a name="about-system-shutdown"></a>Acerca del cierre del sistema

El apagado del sistema trae el sistema a una condición en la que es seguro apagar el equipo. Todos los búferes del sistema de archivos se vacían en el disco y, a continuación, se muestra un cuadro de mensaje que informa al usuario de que el equipo puede desactivarse. También hay una opción de reinicio que reiniciará el equipo, en lugar de mostrar este cuadro de mensaje de apagado del sistema. Para obtener más información, consulte [apagar](shutting-down.md). Para obtener un ejemplo, vea [Cómo apagar el sistema](how-to-shut-down-the-system.md).

El cierre de sesión detiene todos los procesos asociados con el contexto de seguridad del proceso que llamó a la función Exit, registra el usuario actual en el sistema y muestra el cuadro de diálogo logon. Para obtener más información, consulte cerrar [sesión](logging-off.md). Para obtener un ejemplo, consulte [cómo cerrar la sesión del usuario actual](how-to-log-off-the-current-user.md).

El bloqueo de la estación de trabajo protege la pantalla del uso no autorizado siempre que salga del equipo. Para desbloquear la estación de trabajo, debe iniciar sesión. Para obtener un ejemplo, consulte [bloqueo de la estación de trabajo](how-to-lock-the-workstation.md).

El [rastreador de eventos de apagado](shutdown-event-tracker.md) permite al usuario o a la aplicación documentar el motivo de apagar o reiniciar el sistema.
