# <a name="trusted-execution"></a>Ejecución de confianza

Enclaves se usan para crear entornos de ejecución de confianza. Un enclave es una región aislada de código y datos dentro del espacio de direcciones para una aplicación. Solo el código que se ejecuta dentro de enclave puede tener acceso a los datos del mismo enclave.
