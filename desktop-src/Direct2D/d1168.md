# <a name="d1168-write-map-option-requires-cpu-writable-bitmap"></a>D1168: la opción de asignación de escritura requiere un mapa de bits grabable de CPU

D2D1 \_ \_ \_ la escritura de las opciones de asignación solo es válida en un mapa de bits grabable de CPU






 

## <a name="possible-causes"></a>Causas posibles

Direct2D puede generar este mensaje si una aplicación llama a ID2D1Bitmap1:: Map con el \_ atributo de escritura de las opciones de asignación D2D1 \_ \_ , pero el mapa de bits no se puede escribir en la CPU.

## <a name="possible-fixes"></a>Posibles correcciones

Si una aplicación necesita acceso de escritura mediante ID2D1Bitmap1:: Map, asegúrese de que el recurso de Direct3D subyacente tiene el \_ atributo de escritura de acceso de CPU D3D11 \_ \_ .
