# <a name="query-querylisttype-element"></a>Query (QueryListType), elemento

Define un conjunto de selectores y suprimidores que se usan para incluir eventos en o excluir eventos del conjunto de resultados.

``` syntax
<xs:element name="Query"
    type="QueryType"
 />
```

El tipo complejo [**QueryListType**](queryschema-querylisttype-complextype.md) define el elemento de **consulta** .
