# <a name="pki-max-issuing-depth-attribute"></a>PKI: atributo Max--Depth

La longitud máxima de la cadena de certificados que emite el certificado.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | PKI-Max-emisión-profundidad                |
| Nombre para mostrar de LDAP | pKIMaxIssuingDepth                   |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1329              |
| System-ID-GUID    | f0bfdefa-3b9d-11d2-90cc-00c04fd91ab1 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
