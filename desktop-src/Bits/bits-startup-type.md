# <a name="bits-startup-type"></a>Tipo de inicio de BITS

El **tipo de inicio** de bits es el inicio automático retrasado (si hay trabajos de bits activos) o el inicio de la demanda (si no hay ningún trabajo activo). Las versiones de Windows anteriores a la actualización de noviembre usaban solo el tipo de inicio de inicio automático retrasado; las versiones anteriores a vista usaban el tipo de inicio manual.

No debe establecer el **tipo de inicio** en deshabilitado. Deshabilitar BITS puede interrumpir las aplicaciones que dependen de BITS para transferir archivos.
