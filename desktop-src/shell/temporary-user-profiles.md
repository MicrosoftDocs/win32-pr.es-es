# <a name="temporary-user-profiles"></a>Perfiles de usuario temporales

Un perfil de usuario temporal se emite cada vez que una condición de error impide que el perfil del usuario se cargue. Los perfiles temporales se eliminan al final de cada sesión y los cambios realizados por el usuario en la configuración de escritorio y los archivos se pierden cuando el usuario cierra la sesión. Los perfiles temporales solo están disponibles en equipos que ejecutan Windows 2000 y versiones posteriores.
