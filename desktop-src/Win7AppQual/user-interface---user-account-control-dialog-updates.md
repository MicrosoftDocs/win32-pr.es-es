# <a name="user-interface---user-account-control-dialog-updates"></a>Interfaz de usuario: actualizaciones del cuadro de diálogo control de cuentas de usuario

## <a name="affected-platforms"></a>Plataformas afectadas

**Clientes** : Windows 7  
**Servidores** : Windows Server 2008 R2  









## <a name="feature-impact"></a>Impacto en las características

**Gravedad** baja  
**Frecuencia** : medio  











## <a name="description"></a>Descripción

En Windows 7, se han normalizado las opciones del cuadro de diálogo de UAC. Anteriormente, los usuarios tenían que seleccionar entre varias opciones, por ejemplo, "Continue", "Cancel", etc. Ahora todos los cuadros de diálogo proporcionan a los usuarios una simple "sí" o "no". El diseño del cuadro de diálogo ahora también muestra claramente el nombre del programa, el publicador y el origen.

## <a name="leveraging-feature-capabilities"></a>Aprovechar las capacidades de características

Los requisitos de desarrollo de aplicaciones en Windows 7 para la compatibilidad con UAC son los mismos que en Windows Vista. Las aplicaciones compatibles con Windows Vista interactuarán con UAC en Windows 7 sin ninguna modificación. Consulte los temas sobre el control de cuentas de usuario en la *Guía de compatibilidad de aplicaciones de Windows Vista* para obtener información acerca de cómo hacer que las aplicaciones de Windows XP funcionen correctamente en Windows 7. Aunque las mejoras de UAC para Windows 7 afectarán a la experiencia del usuario, no afectarán a la interfaz de la aplicación. Sin embargo, si hay contenido de ayuda vinculado a los cuadros de diálogo de UAC, puede que tenga que actualizar las capturas de pantallas.
