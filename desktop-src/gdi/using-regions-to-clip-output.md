# <a name="using-regions-to-clip-output"></a>Uso de regiones para recortar la salida

Esta sección contiene un único ejemplo que muestra cómo puede usar regiones para permitir que el usuario defina cómo puede aparecer una parte de la salida del área de cliente. Las regiones usadas para este propósito se denominan regiones de recorte.

El ejemplo de esta sección se toma de una aplicación que permite a un usuario capturar todo el escritorio como un mapa de bits y, a continuación, aislar y guardar una parte de esta imagen como un archivo .BMP datos.

Al hacer **clic en** Definir región de recorte en el menú de la aplicación, el usuario puede seleccionar una región de recorte haciendo clic en el botón izquierdo del mouse y arrastrando el mouse. A medida que el usuario arrastra el mouse, la aplicación dibuja un rectángulo que corresponde a la nueva región de recorte.

Al hacer **clic en** Clip (Recortar) en el menú de la aplicación, el usuario puede volver a dibujar la parte aislada de la imagen dentro de los límites del rectángulo especificado.

En esta sección se proporciona información sobre los temas siguientes.

-   [Definir la región de recorte](defining-the-clipping-region.md)
-   [Salida de recorte](clipping-output.md)
