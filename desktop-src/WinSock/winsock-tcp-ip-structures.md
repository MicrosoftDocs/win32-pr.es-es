# <a name="winsock-tcpip-structures"></a>Estructuras TCP/IP de Winsock

En la lista siguiente se proporcionan descripciones concisas de cada estructura TCP/IP de Winsock. Para obtener información adicional sobre cualquier estructura, haga clic en el nombre de la estructura.



| Estructura TCP/IP                                 | Descripción                                                                                                                              |
|--------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**INFORMACIÓN DE \_ INTERFAZ**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info)      | Se usa con el comando ioctl para obtener información sobre una dirección IP de interfaz.                                                         |
| [**INTERFACE \_ INFO \_ EX**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info_ex) | Se usa con **el comando SIO \_ GET INTERFACE LIST \_ \_ IOCTL** para obtener información sobre una dirección IP de interfaz, incluidas las direcciones IPv6. |
| [**sockaddr \_ gen**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-sockaddr_gen)            | Proporciona información de dirección de socket genérico. Se usa con la [**estructura \_ INTERFACE INFO.**](/windows/desktop/api/Ws2ipdef/ns-ws2ipdef-interface_info)                        |
