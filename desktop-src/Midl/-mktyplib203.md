# <a name="mktyplib203-switch"></a>modificador/mktyplib203

El modificador **/mktyplib203** obliga al compilador de MIDL a analizar el archivo de entrada.

``` syntax
midl /mktyplib203
```

## <a name="switch-options"></a>Opciones de conmutador

Este modificador no tiene parámetros.

## <a name="remarks"></a>Observaciones

Para usar el modificador **/mktyplib203** , el archivo de entrada debe seguir la sintaxis de ODL exactamente, lo que significa que no se pueden colocar instrucciones fuera del bloque de biblioteca. Ejemplos

**/mktyplib203 myoldodl. ODL de MIDL**

**MIDL/mktyplib203 oldhabit. idl**
