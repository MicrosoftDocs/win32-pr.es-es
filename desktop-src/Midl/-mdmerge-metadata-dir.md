# <a name="metadata_dir-switch-mdmerge"></a>modificador/metadata_dir (MDMERGE)

El modificador **\_ dir de/metadata** especifica el directorio que contiene los archivos de metadatos de plataforma.

``` syntax
mdmerge /metadata_dir metadata_directory
```

## <a name="switch-options"></a>Opciones de conmutador

<dl> <dt>

*directorio de metadatos \_* 
</dt> <dd>

Especifica el directorio que contiene los archivos de metadatos de plataforma.

</dd> </dl>

## <a name="remarks"></a>Observaciones

Use este modificador para especificar la ubicación del archivo de metadatos principal para Windows, que se denomina Windows. winmd.

## <a name="examples"></a>Ejemplos

**mdmerge/Metadata \_ dir dir1**
