# <a name="debugging-the-page"></a>Depurar la página

Cuando Windows Media Player hospeda la Página Web de la tienda en línea, deshabilita la posibilidad de abrir cuadros de diálogo desde la Página Web. Para depurar el código de la página web, en su lugar debe agregar código que imprima temporalmente el resultado en un elemento HTML de la página web, como un elemento DIV o TEXT.
