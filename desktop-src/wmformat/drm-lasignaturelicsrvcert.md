# <a name="drm_lasignaturelicsrvcert"></a>\_LASIGNATURELICSRVCERT DRM

El **atributo \_ LASignatureLicSrvCert de DRM** contiene el certificado que comprueba el certificado contenido en [**\_ LASignatureCert de DRM**](drm-lasignaturecert.md).

## <a name="global-constant"></a>Constante global

g \_ wszWMDRM \_ LASignatureLicSrvCert

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Esta propiedad se puede establecer con el método [**IWMDRMWriter:: SetDRMAttribute**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) . No es accesible para el objeto lector.
