# <a name="y-property"></a>y (Propiedad)

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

La propiedad establece o recupera la posición y de la esquina superior `y` izquierda del rectángulo.

``` syntax
[ iYPos = ] DVDRect.y
```

## <a name="return-value"></a>Valor devuelto

Devuelve la posición Y de la esquina superior izquierda del rectángulo como un entero.

## <a name="remarks"></a>Comentarios

Esta propiedad es de lectura y escritura sin ningún valor predeterminado.
