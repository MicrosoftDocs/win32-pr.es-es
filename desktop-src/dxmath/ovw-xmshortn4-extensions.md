# <a name="xmshortn4-extensions"></a>Extensiones de XMSHORTN4

[**XMSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn4) describe 4D vector para almacenar valores normalizados y firmados como enteros de 16 bits con signo, (tipo `int16_t` ).

> [!Note]  
> Las características que se enumeran aquí solo están disponibles cuando se desarrollan con C++.
