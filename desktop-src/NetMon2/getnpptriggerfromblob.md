# <a name="getnpptriggerfromblob-function"></a>GetNPPTriggerFromBlob función)

La función **GetNPPTriggerFromBlob** recupera el desencadenador del BLOB especificado.

## <a name="syntax"></a>Sintaxis


```C++
DWORD GetNPPTriggerFromBlob(
  _In_  HBLOB     hBlob,
  _Out_ LPTRIGGER pTrigger,
  _Out_ HBLOB     hErrorBlob
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*hBlob* \[ de\]
</dt> <dd>

Identificador del BLOB.

</dd> <dt>

*pTrigger* \[ enuncia\]
</dt> <dd>

Puntero al valor del desencadenador.

</dd> <dt>

*hErrorBlob* \[ enuncia\]
</dt> <dd>

Identificador de un BLOB de error que especifica dónde se produjo el error (si existe) en el BLOB original.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Si la función se realiza correctamente, el valor devuelto es NMERR \_ Success.

Si la función no es correcta, el valor devuelto es un valor de NMERR que indica el error.

## <a name="remarks"></a>Observaciones

La información del desencadenador se almacena en la categoría **desencadenador** del BLOB.
