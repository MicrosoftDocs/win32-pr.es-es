# <a name="other-login-workstations-attribute"></a>Atributo Other-Login-Workstations

No Windows estaciones de trabajo NT o LAN Manager desde las que un usuario puede iniciar sesión.

> [!Note]  
> Active Directory no usa ni rellena este campo.

 



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Otras estaciones de trabajo de inicio de sesión                    |
| Ldap-Display-Name | otherLoginWorkstations                      |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.91                       |
| System-Id-Guid    | bf9679f1-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
