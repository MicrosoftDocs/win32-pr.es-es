# <a name="qos-updates"></a>Actualizaciones de QoS

Los detalles de implementación de calidad de servicio (QoS) se proporcionan en una sección de QoS dedicada y independiente en el kit de desarrollo de software (SDK) de Microsoft Windows. Consulte la sección [Quality of Service (calidad de servicio](/previous-versions/windows/desktop/qos/qos-start-page) ) de la Windows SDK para obtener información sobre las actualizaciones de QoS.
