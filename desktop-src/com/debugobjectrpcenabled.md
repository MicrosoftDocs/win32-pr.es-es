# <a name="debugobjectrpcenabled"></a>DebugObjectRPCEnabled

**DebugObjectRPCEnabled** habilita la depuración remota de com a través de RPC. Si esta clave no está definida en el servidor, la depuración remota está deshabilitada.

## <a name="registry-key"></a>Clave del Registro

**HKEY \_ local \_ Machine \\ software \\ Microsoft \\ Windows NT \\ CurrentVersion \\ DebugObjectRPCEnabled**



| Clave del Registro               | Descripción                       |
|----------------------------|-----------------------------------|
| [**AeDebug**](aedebug.md) | Almacena opciones específicas del depurador. |
