# <a name="tasknamedvaluepairname-property"></a>TaskNamedValuePair.Name propiedad

Para el scripting, obtiene o establece el nombre asociado a un valor en un par nombre-valor.

## <a name="syntax"></a>Syntax


```VB
TaskNamedValuePair.Name As String
```



## <a name="property-value"></a>Valor de propiedad

Nombre asociado a un valor en un par nombre-valor.
