# <a name="em_getwordbreakprocex-message"></a>Mensaje \_ EM GETWORDBREAKPROCEX

Recupera la dirección del procedimiento de salto de palabras extendido registrado actualmente para un control de edición enriquecido.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>

No se usa; debe ser cero.

</dd> <dt>

*lParam* 
</dt> <dd>

No se usa; debe ser cero.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

El mensaje devuelve la dirección del procedimiento actual.
