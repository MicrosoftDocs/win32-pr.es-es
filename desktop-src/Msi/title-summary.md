# <a name="title-summary-property"></a>Propiedad de Resumen de título

La propiedad de **Resumen título** describe brevemente el tipo del paquete del instalador. Para esta propiedad se pueden usar frases como "base de datos de instalación" o "transformación" o "revisión".

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versión<br/> | Windows Installer 5,0 en Windows Server 2012, Windows 8, Windows Server 2008 R2 o Windows 7. Windows Installer 4,0 o Windows Installer 4,5 en Windows Server 2008 o Windows Vista. Windows Installer en Windows Server 2003 o Windows XP<br/> |
