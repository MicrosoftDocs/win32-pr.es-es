# <a name="supported-tasks-in-wpd-applications"></a>Tareas admitidas en las aplicaciones de WPD

Las interfaces de aplicación de WPD admiten las siguientes tareas:

-   Conexión a un dispositivo
-   Enumerar dispositivos conectados
-   Examinar las capacidades de un dispositivo
-   Explorar el contenido de un dispositivo
-   Transferir contenido a un dispositivo o crear contenido en él
-   Editar contenido en el dispositivo
-   Modificar objetos en un dispositivo
-   Controlar el dispositivo
-   Supervisar eventos de dispositivo

Para obtener más información acerca de cada tarea, así como los detalles sobre cómo programar una tarea determinada, consulte la [Guía de programación de dispositivos portátiles de Windows](programming-guide.md).
