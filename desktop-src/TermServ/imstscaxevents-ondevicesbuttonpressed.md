# <a name="imstscaxeventsondevicesbuttonpressed-method"></a>IMsTscAxEvents:: OnDevicesButtonPressed (método)

Se le llama cuando se ha presionado el botón dispositivos en la barra de conexión.

## <a name="syntax"></a>Sintaxis


```C++
void OnDevicesButtonPressed();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.

## <a name="remarks"></a>Observaciones

El botón dispositivos se muestra en la barra de conexión cuando la redirección USB está habilitada tanto en el cliente como en el servidor.
