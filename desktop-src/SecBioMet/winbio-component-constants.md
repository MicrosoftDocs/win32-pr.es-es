# <a name="winbio_component-constants"></a>Constantes de componentes de WINBIO \_

Se pueden usar las constantes siguientes al llamar a [**WinBioControlUnit**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunit) o [**WinBioControlUnitPrivileged**](/windows/desktop/api/Winbio/nf-winbio-winbiocontrolunitprivileged) para especificar el tipo de adaptador que se va a usar.



| Constante                                                                                                                                                                                        | Descripción                             |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------|
| <span id="WINBIO_COMPONENT_SENSOR"></span><span id="winbio_component_sensor"></span><dl> <dt>**\_sensor de componentes de WINBIO \_**</dt> </dl>    | Especifica un adaptador de sensor.<br/>  |
| <span id="WINBIO_COMPONENT_ENGINE"></span><span id="winbio_component_engine"></span><dl> <dt>**\_motor de componentes de WINBIO \_**</dt> </dl>    | Especifica un adaptador de motor.<br/> |
| <span id="WINBIO_COMPONENT_STORAGE"></span><span id="winbio_component_storage"></span><dl> <dt>**\_almacenamiento de componentes de WINBIO \_**</dt> </dl> | Especifica un adaptador de almacenamiento.<br/> |
