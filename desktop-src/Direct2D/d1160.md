# <a name="d1160-cycle-in-graph"></a>D1160: ciclo en el gráfico

Se intentó dibujar un gráfico de imagen con un ciclo en él.






 

## <a name="possible-causes"></a>Causas posibles

Direct2D permite a las aplicaciones construir gráficos de efectos para ofrecer flexibilidad en el comportamiento. Sin embargo, las aplicaciones no deben crear un gráfico de efectos que contenga un ciclo. Por ejemplo, considere la posibilidad de que un gráfico de efectos contenga los nodos a, B y C: no es válido para vincular a a B, a continuación, B a C y, a continuación, a C a a. Direct2D identificará los gráficos de efecto cíclico y notificará un error. el nivel de depuración generará este mensaje.

## <a name="possible-fixes"></a>Posibles correcciones

Para evitar este mensaje, no cree gráficos de efectos que tengan un ciclo. Los gráficos de efectos de diseño se incluyen de forma eficaz, lo que incluye al menos un nodo que actúa solo como salida.
