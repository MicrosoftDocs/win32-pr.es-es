# <a name="mf_mt_max_luminance_level-attribute"></a>Atributo de nivel de luminancia de MF \_ MT \_ máx. \_ \_

Especifica el nivel de luminancia máximo del contenido en Nits. Este atributo tiene la misma semántica que **MaxCLL** , tal y como se define en el estándar CEA-861,3.

## <a name="data-type"></a>Tipo de datos

**UINT32**
