# <a name="userclass-attribute"></a>userClass (atributo)

El tipo de atributo userClass especifica una categoría de usuario de equipo.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | userClass                                   |
| Nombre para mostrar de LDAP | userClass                                   |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.8                   |
| System-ID-GUID    | 11732a8a-e14d-4cc5-b92f-d93f51c6d8e4        |
