# <a name="retrieving-event-data"></a>Recuperación de datos de eventos

Para consumir datos específicos del evento, el consumidor debe conocer el formato de los datos del evento. Esto no es un problema si está consumiendo eventos de su propio proveedor porque conoce el formato de los datos. Sin embargo, para consumir un evento de cualquier proveedor, el proveedor debe publicar el diseño de los datos del evento. Para obtener información general sobre cómo funcionan los metadatos de eventos, así como cómo publicar los metadatos de eventos para que otros usuarios puedan usarlos, consulte Información general sobre metadatos [de eventos.](event-metadata-overview.md)

Para consumir eventos en Windows Vista que se publicaron mediante un manifiesto, MOF o archivos TMF, así como eventos TraceLogging, vea [Retrieving Event Data Using TDH](retrieving-event-data-using-tdh.md).

Para consumir eventos antes de Windows Vista que se publicaron mediante MOF, vea Recuperar datos [de eventos mediante MOF.](retrieving-event-data-using-mof.md)
