# <a name="systemphotosubjectdistance"></a>System.Photo.SubjectDistance

Distancia al sujeto en metros. Se calcula a partir de PKEY \_ Photo \_ SubjectDistanceNumerator y PKEY \_ Photo \_ SubjectDistanceDenominator.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Photo.SubjectDistance
   shellPKey = PKEY_Photo_SubjectDistance
   formatID = 14B81DA1-0135-4D31-96D9-6CBFC9671A99
   propID = 37382
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = Double
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
