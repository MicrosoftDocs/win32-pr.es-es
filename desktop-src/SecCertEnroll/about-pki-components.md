# <a name="pki-elements"></a>Elementos PKI

Una infraestructura de clave pública (PKI) típica de Microsoft incluye los siguientes elementos.

| Elemento                                                                     | Descripción                                                                                                                                                                                              |
|-----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Entidades de certificación](about-certification-authorities.md)<br/> | Proporcionar servicios que autentiquen la identidad de individuos, equipos y otras entidades de una red. Esto engloba las entidades de certificación raíz y las entidades subordinadas.<br/> |
| [Directorio de certificados](about-certificate-directory.md)<br/>         | Guarda las solicitudes de certificados y los certificados emitidos y revocados y las solicitudes de certificado.<br/>                                                                                                      |
| [Servidor de recuperación de claves](about-key-recovery-server.md)<br/>             | Guarda las claves privadas cifradas en la base de datos de certificados para su recuperación después de la pérdida.<br/>                                                                                                             |
