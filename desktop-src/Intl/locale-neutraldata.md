# <a name="locale_neutraldata"></a>configuración regional \_ NEUTRALDATA

**Windows 7 y versiones posteriores:** Datos de configuración regional neutros, es decir, datos definidos solo en el idioma. Los datos de país o región usan el valor predeterminado.
