# <a name="property-keys"></a>Claves de propiedad

Documentación de referencia de las claves de propiedad del marco de cinta de Windows, una colección de pares clave-valor de **PROPERTYKEY** y **PROPVARIANT** utilizadas para describir el estado de un comando de marco de cinta.

> [!Note]  
> Una clave debe ser única en la colección para cada comando.

 

## <a name="property-types"></a>Tipos de propiedades

[Propiedades de la colección](windowsribbon-reference-properties-collection.md)

Documentación de referencia sobre claves de propiedad para controles basados en la colección de marcos de cinta.

[Propiedades del selector de colores](windowsribbon-reference-properties-colorpicker.md)

Documentación de referencia sobre las claves de propiedad para el [selector de colores desplegables](windowsribbon-controls-dropdowncolorpicker.md)del marco de cinta.

[Propiedades de control de fuente](windowsribbon-reference-properties-fontcontrol.md)

Documentación de referencia sobre las claves de propiedad para el [control de fuente](windowsribbon-controls-fontcontrol.md)del marco de cinta.

[Propiedades de Framework](windowsribbon-reference-properties-framework.md)

Documentación de referencia sobre las claves de propiedades globales para el marco de la cinta de opciones.

[Propiedades de recursos](windowsribbon-reference-properties-resource.md)

Documentación de referencia sobre las claves de propiedad para los recursos del marco de cinta.

[Propiedades de la cinta](windowsribbon-reference-properties-ribbon.md)

Documentación de referencia sobre las claves de propiedad para la vista de la cinta de opciones del marco de cinta.

[Propiedades de estado](windowsribbon-reference-properties-state.md)

Documentación de referencia sobre las claves de propiedad para el estado o contexto del control de marco de cinta.
