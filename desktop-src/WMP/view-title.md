# <a name="viewtitle"></a>VER. título

El atributo **title** especifica o recupera el título de la **vista**.

``` syntax
        elementID.title
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** especificada en tiempo de diseño y de solo lectura después. Puede tener hasta 255 caracteres y no tiene ningún valor predeterminado.

## <a name="remarks"></a>Observaciones

El título especificado o recuperado es el nombre para mostrar del tema.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
