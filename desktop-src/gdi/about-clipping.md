# <a name="about-clipping"></a>Acerca del recorte

Las aplicaciones usan el recorte de varias maneras. Las aplicaciones de procesamiento de texto y de hoja de cálculo muestran entradas de teclado para evitar que aparezcan en los márgenes de una página o una hoja de cálculo. Las aplicaciones de diseño asistido por PC (CAD) y de dibujo de recortan la salida de gráficos para evitar que sobrescriban los bordes de un dibujo o una imagen.

Una *región de recorte* es una región con bordes que son líneas rectas o curvas. Un *trazado de clip* es una región con bordes que son líneas rectas, curvas de Bézier o combinaciones de ambos. Para obtener más información sobre las regiones, consulte [regiones](regions.md). Para obtener más información sobre las rutas de acceso, vea [rutas](paths.md)de acceso.

En esta información general se tratan los temas siguientes:

-   [Regiones de recorte](clipping-regions.md)
-   [Trazados de clip](clip-paths.md)
