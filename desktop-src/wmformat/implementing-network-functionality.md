# <a name="implementing-network-functionality"></a>Implementación de la funcionalidad de red

En esta sección se describe cómo usar las características de red disponibles a través del SDK de Windows Media Format. Una aplicación puede usar estas características para leer secuencias ASF de una red, difundir secuencias ASF a una red o enviar secuencias ASF a un punto de publicación en un servidor que ejecuta Windows Media Services.

En las secciones siguientes se describen las características de red de este SDK.



| Sección                                                                    | Descripción                                                                                                          |
|----------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [Información general de las interfaces de red](overview-of-networking-interfaces.md) | Denomina y describe cada interfaz que admite métodos de red.                                                 |
| [Lectura de datos ASF a través de una red](reading-asf-data-over-a-network.md)     | Describe cómo reproducir archivos desde un origen de red y cómo establecer la configuración de red en el objeto lector.    |
| [Envío de datos ASF a través de una red](sending-asf-data-over-a-network.md)     | Describe cómo difundir datos ASF a través de una red o enviar datos ASF a un punto de publicación en un servidor de Windows Media. |
| [Configuración de red predeterminada](default-networking-settings.md)             | Proporciona una referencia rápida para la configuración predeterminada que usa el SDK.                                                 |
