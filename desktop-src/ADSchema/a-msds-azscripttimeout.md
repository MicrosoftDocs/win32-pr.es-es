# <a name="ms-ds-az-script-timeout-attribute"></a>atributo MS-DS-AZ-script-timeout

Tiempo máximo, en milisegundos, que se esperará a que un script termine de auditar una directiva específica.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-DS-AZ-script-timeout                 |
| Nombre para mostrar de LDAP | msDS-AzScriptTimeout                    |
| Tamaño              | \-                                      |
| Actualizar privilegio  | Administrador de AzRoles                           |
| Frecuencia de actualización  | Durante la inicialización o el cambio de directiva. |
| Attribute-Id      | 1.2.840.113556.1.4.1797                 |
| System-ID-GUID    | 87d0fb41-2c8b-41f6-b972-11fdfd50d6b0    |
| Sintaxis            | [**Enumeración**](s-enumeration.md)    |
