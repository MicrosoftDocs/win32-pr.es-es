# <a name="rootdse-ad-schema"></a>RootDSE (esquema de AD)

En LDAP 3.0, rootDSE se define como la raíz del árbol de datos de directorio en un servidor de directorios. RootDSE no forma parte de ningún espacio de nombres. El propósito de rootDSE es proporcionar datos sobre el servidor de directorios. Para más información sobre rootDSE, consulte [Enlace sin servidor y RootDSE](/windows/desktop/AD/serverless-binding-and-rootdse) en la documentación Active Directory SDK.

rootDSE contiene los atributos siguientes. Todos los atributos son de un solo valor a menos que se indique lo contrario.



| Atributo                                    | Sintaxis                                                 | Descripción                                                                                                                                                                                                                                                                                                                                                                                                            |
|----------------------------------------------|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **configurationNamingContext**<br/>    | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del contenedor de configuración.<br/>                                                                                                                                                                                                                                                                                                                                            |
| **currentTime**<br/>                   | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene la hora actual establecida en este servidor de directorios en formato de hora universal coordinada.<br/>                                                                                                                                                                                                                                                                                                                |
| **defaultNamingContext**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del dominio del que es miembro este servidor de directorios.<br/>                                                                                                                                                                                                                                                                                                                  |
| **dnsHostName**<br/>                   | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene la dirección DNS de este servidor de directorios.<br/>                                                                                                                                                                                                                                                                                                                                                         |
| **domainControllerFunctionality**<br/> | [**String(Teletex)**](s-string-teletex.md)<br/> | Indica el nivel funcional del controlador de dominio. | **forestFunctionality**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Indica el nivel funcional del bosque. Contiene "TRUE" o "FALSE".<br/>                                                                                                                                                                                                                                                                                                                 |
| **ldapServiceName**<br/>               | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre de entidad de seguridad de servicio (SPN) para el servidor LDAP. Se usa para la autenticación mutua.<br/>                                                                                                                                                                                                                                                                                                              |
| **namingContexts**<br/>                | [**String(Teletex)**](s-string-teletex.md)<br/> | Atributo de varios valores que contiene los nombres distintivos de todos los contextos de nomenclatura almacenados en este servidor de directorios. | **rootDomainNamingContext**<br/>       | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del primer dominio del bosque que contiene el dominio del que es miembro este servidor de directorios.<br/>                                                                                                                                                                                                                                                                     |
| **schemaNamingContext**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del contenedor de esquema.<br/>                                                                                                                                                                                                                                                                                                                                                   |
| **serverName**<br/>                    | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del objeto de servidor para este servidor de directorio en el contenedor de configuración.<br/>                                                                                                                                                                                                                                                                                             |
| **subschemaSubentry**<br/>             | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del [**objeto subSchema.**](c-subschema.md) El **objeto subSchema** contiene propiedades que exponen los atributos admitidos (en la [**propiedad attributeTypes)**](a-attributetypes.md) y las clases (en la [**propiedad objectClasses).**](a-objectclasses.md)<br/> La **propiedad subschemaSubentry** y el subschema se definen en LDAP 3.0 (vea RFC 2251).<br/> |
| **supportedCapabilities**<br/>         | [**String(Teletex)**](s-string-teletex.md)<br/> | Atributo de varios valores que contiene las funcionalidades admitidas por este servidor de directorios.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedControl**<br/>              | [**String(Teletex)**](s-string-teletex.md)<br/> | Atributo de varios valores que contiene los ID de los controles de extensión admitidos por este servidor de directorios. Consulte la tabla siguiente para obtener una lista de los posibles OD de control.<br/>                                                                                                                                                                                                                                  |
| **supportedLDAPPolicies**<br/>         | [**String(Teletex)**](s-string-teletex.md)<br/> | Atributo de varios valores que contiene los nombres de las directivas de administración LDAP admitidas.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedLDAPVersion**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Atributo de varios valores que contiene las versiones LDAP (especificadas por el número de versión principal) compatibles con este servidor de directorios.<br/>                                                                                                                                                                                                                                                                         |
| **supportedSASLMechanisms**<br/>       | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene los mecanismos de seguridad admitidos para la negociación de SASL (consulte RFC LDAP). De forma predeterminada, se admite GSSAPI.<br/>                                                                                                                                                                                                                                                                                           |
