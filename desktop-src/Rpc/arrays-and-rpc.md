# <a name="arrays-and-rpc"></a>Matrices y RPC

Los lenguajes de programación C y C++ proporcionan esencialmente dos tipos de matrices: unidimensional y multidimensional. RPC permite a los desarrolladores especificar tipos de matriz adicionales mediante los atributos MIDL para describir las características de las matrices en las aplicaciones distribuidas.

En esta sección se describen los tipos de matrices disponibles en RPC. También se describen los atributos de MIDL que los desarrolladores pueden utilizar para describir Matrices unidimensionales y multidimensionales. Este material se presenta en los temas siguientes:

-   [Tipos de matrices](kinds-of-arrays.md)
-   [Atributos de matriz](array-attributes.md)
-   [Matrices multidimensionales](multidimensional-arrays.md)
