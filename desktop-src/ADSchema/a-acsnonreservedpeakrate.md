# <a name="acs-non-reserved-peak-rate-attribute"></a>Atributo acs-non-reserved-peak-rate

El **atributo ACS-Non-Reserved-Peak-Rate** es solo para uso interno. Basado en RFC2814.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Acs- Non-Reserved-Peak-Rate           |
| Ldap-Display-Name | aCSNonReservedPeakRate               |
| Size              | 8 bytes                              |
| Privilegio actualizar  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1318              |
| System-Id-Guid    | a331a73f-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**Intervalo**](s-interval.md)       |
