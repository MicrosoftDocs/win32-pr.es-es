# <a name="ms-com-objectid-attribute"></a>atributo MS-COM-ObjectId

GUID usado para identificar una partición COM+ o un PartitionSet COM+.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | MS-COM-ObjectId                                       |
| Nombre para mostrar de LDAP | msCOM-ObjectId                                        |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1428                               |
| System-ID-GUID    | 430f678b-889f-41f2-9843-203b5a65572f                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
