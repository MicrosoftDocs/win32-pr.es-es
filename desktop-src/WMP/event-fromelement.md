# <a name="eventfromelement"></a>evento. fromElement

El atributo **fromElement** recupera el elemento del que procede el evento. Si ningún elemento pasa un evento, el valor es **null**.

``` syntax
event.fromElement
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un objeto de solo lectura.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
