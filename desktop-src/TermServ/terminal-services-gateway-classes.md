# <a name="remote-desktop-gateway-classes"></a>Clases de puerta de enlace de Escritorio remoto

El proveedor WMI de puerta de enlace de Escritorio remoto (puerta de enlace de escritorio remoto) proporciona las siguientes clases.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**Win32 \_ TSGatewayConnection**](win32-tsgatewayconnection.md)
</dt> <dd>

Representa una conexión desde un equipo cliente a un servidor de puerta de enlace de escritorio remoto.

</dd> <dt>

[**Win32 \_ TSGatewayConnectionAuthorizationPolicy**](win32-tsgatewayconnectionauthorizationpolicy.md)
</dt> <dd>

Describe una Escritorio remoto Directiva de autorización de conexión (CAP de RD). Las Cap de RD se usan para determinar si un usuario puede conectarse al servidor de puerta de enlace de escritorio remoto.

</dd> <dt>

[**Win32 \_ TSGatewayLoadBalancer**](win32-tsgatewayloadbalancer.md)
</dt> <dd>

Describe un conjunto de servidores de equilibrio de carga de puerta de enlace de escritorio remoto. Se usan para equilibrar la carga de las conexiones de puerta de enlace de escritorio remoto entre varios servidores.

</dd> <dt>

[**Win32 \_ TSGatewayRADIUSServer**](win32-tsgatewayradiusserver.md)
</dt> <dd>

Describe un servidor Servicio de autenticación remota telefónica de usuario (RADIUS), que tiene un conjunto de Servicios de Escritorio remoto directivas de autorización de conexión (Cap de RD).

</dd> <dt>

[**Win32 \_ TSGatewayResourceAuthorizationPolicy**](win32-tsgatewayresourceauthorizationpolicy.md)
</dt> <dd>

Describe una Escritorio remoto Directiva de autorización de recursos (RAP de RD). Una RAP de RD se usa para decidir si un usuario está autorizado para conectarse a un recurso especificado a través de la puerta de enlace de escritorio remoto.

</dd> <dt>

[**Win32 \_ TSGatewayResourceGroup**](win32-tsgatewayresourcegroup.md)
</dt> <dd>

Describe un grupo de recursos, que asigna un conjunto de recursos (nombres de equipo) a una sola entidad.

</dd> <dt>

[**Win32 \_ TSGatewayServer**](win32-tsgatewayserver.md)
</dt> <dd>

Se usa para las operaciones específicas del servidor de puerta de enlace de escritorio remoto.

</dd> <dt>

[**Win32 \_ TSGatewayServerSettings**](win32-tsgatewayserversettings.md)
</dt> <dd>

Proporciona métodos y propiedades para ver y configurar las opciones del servidor de puerta de enlace de escritorio remoto.

</dd> </dl>
