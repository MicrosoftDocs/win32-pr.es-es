# <a name="obsolete-wcs-functions"></a>Funciones de WCS obsoletas

En esta sección se delinean las funciones que están obsoletas en WCS 1.0. Actualmente se mantienen por compatibilidad con versiones anteriores. Sin embargo, se quitarán de la API en versiones posteriores de WCS.



|     Función                               |      Descripción                             |
|--------------------------------------------|----------------------------------------------|
| [**UpdateICMRegKey**](/windows/desktop/api/Wingdi/nf-wingdi-updateicmregkeya) | *Obsoleto:* administra perfiles de color y CMM. |
