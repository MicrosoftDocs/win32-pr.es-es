# <a name="commands-design-basics"></a>Comandos (conceptos básicos del diseño)

> [!NOTE]
> Esta guía de diseño se ha creado para Windows 7 y no se ha actualizado para las versiones más recientes de Windows. Gran parte de la guía se sigue aplicando en principio, pero la presentación y los ejemplos no reflejan nuestra [Guía de diseño actual](/windows/uwp/design/).

Los comandos son acciones que los usuarios pueden llevar a cabo mientras usan la aplicación. Obtenga información sobre las directrices para agregar comandos a los menús, las cintas y las barras de herramientas de la aplicación.

## <a name="in-this-section"></a>En esta sección



| Tema                                   | Descripción                                                                                                                                                                                                                        |
|-----------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Menús](cmd-menus.md)<br/>       | Los menús son listas jerárquicas de comandos u opciones disponibles para los usuarios en el contexto actual.<br/>                                                                                                                          |
| [Cintas](cmd-ribbons.md)<br/>   | Las cintas de opciones son la manera moderna de ayudar a los usuarios a buscar, comprender y usar comandos de forma eficaz y directa con un número mínimo de clics, con menos necesidad de recurrir a la prueba y el error, y sin tener que hacer referencia a la ayuda.<br/> |
| [Barras de herramientas](cmd-toolbars.md)<br/> | Las barras de herramientas son una manera de agrupar comandos para obtener un acceso eficaz.<br/>                                                                                                                                                              |
