# <a name="d1203-copyfrombitmap-with-multisampled-source"></a>D1203: CopyFromBitmap con origen multimuestreo

CopyFromBitmap se usó con un mapa de bits de origen multimuestreo. La llamada todavía se emitirá, pero es posible que el resultado copiado no represente el patrón multimuestreo esperado.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación interoperaba una textura direct3D multimuestreo como un mapa de bits de Direct2D y se llamaba CopyFromBitmap en el mapa de bits.

## <a name="possible-fixes"></a>Posibles correcciones

Evite interoperar texturas multimuestreo como mapas de bits si la funcionalidad CopyFromBitmap es importante. Es posible que el resultado copiado no represente el patrón multimuestreo esperado.
