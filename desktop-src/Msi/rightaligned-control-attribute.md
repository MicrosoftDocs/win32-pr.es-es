# <a name="rightaligned-control-attribute"></a>Atributo de control rightaligned

Si se establece este bit de estilo, el texto del control se alinea a la derecha.

## <a name="valid-controls"></a>Controles válidos

[GroupBox](groupbox-control.md)

 

[ScrollableText](scrollabletext-control.md)

 

[Texto](text-control.md)

 

[VolumeCostList](volumecostlist-control.md)

 

[CheckBox](checkbox-control.md)

 

[ComboBox](combobox-control.md)

 

[DirectoryList](directorylist-control.md)

 

[DirectoryCombo](directorycombo-control.md)

 

[Editar](edit-control.md)

 

[PathEdit](pathedit-control.md)

 

[ListBox](listbox-control.md)

 

[ListView](listview-control.md)

 

[RadioButtonGroup](radiobuttongroup-control.md)

 

[SelectionTree](selectiontree-control.md)

 

[VolumeSelectCombo](volumeselectcombo-control.md)



| Decimal | Hexadecimal | Descripción                        |
|---------|-------------|------------------------------------|
| 64      | 0x00000040  | msidbControlAttributesRightAligned |



 

## <a name="remarks"></a>Comentarios

Para establecer este atributo en un control, incluya el bit RightAligned en la columna Attributes del registro del control en la [tabla Control](control-table.md).

Vea [Controles y atributos](control-attributes.md) de [control.](controls.md)
