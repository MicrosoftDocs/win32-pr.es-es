# <a name="event-reference-page"></a>Página Referencia de eventos

En esta sección se describe la página de referencia de eventos (ERP), un documento basado en HTML que puede escribir para mostrar los eventos que se producen durante el funcionamiento de un Monitor de red experto. Después de definir los requisitos de creación de expertos, puede usar los ERP para ayudar a Monitor de red solucionar problemas de red.
