# <a name="wmmediaispremiere"></a>WM/MediaIsPremiere

El atributo **WM/MediaIsPremiere** indica si el contenido es el primero de una serie.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaIsPremiere

## <a name="data-type"></a>Tipo de datos

**tipo de WMT \_ \_ bool**

## <a name="remarks"></a>Observaciones

Este atributo se utiliza para la televisión de difusión.
