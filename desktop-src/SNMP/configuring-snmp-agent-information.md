# <a name="configuring-snmp-agent-information"></a>Configuración de la información del agente SNMP

La información del agente SNMP le permite especificar comentarios sobre el usuario y la ubicación física del equipo, e indicar los tipos de servicio que se van a notificar. Los tipos de servicio a los que se puede informar se basan en la configuración del equipo.
