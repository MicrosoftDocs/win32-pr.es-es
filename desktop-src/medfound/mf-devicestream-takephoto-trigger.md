# <a name="mf_devicestream_takephoto_trigger-attribute"></a>Atributo TAKEPHOTO TRIGGER de MF \_ DEVICESTREAM \_ \_

Especifica si el desencadenador tomar foto está encapsulado en el origen del dispositivo.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Comentarios

Establezca este atributo en distinto de cero para encapsular el desencadenador take photo en el origen del dispositivo.
