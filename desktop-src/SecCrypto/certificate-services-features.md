# <a name="certificate-services-features"></a>Características de Servicios de certificados

Tiene una serie de características que hacen que sea útil para las organizaciones que no deciden confiar en entidades de certificación externas y que necesitan una herramienta flexible que se pueda adaptar a las necesidades de la organización.
