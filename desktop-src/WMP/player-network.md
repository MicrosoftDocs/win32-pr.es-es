# <a name="playernetwork"></a>Reproductor. red

La propiedad **Network** recupera el objeto de red.

## <a name="syntax"></a>Sintaxis

*reproductor* . **red** de

## <a name="possible-values"></a>Valores posibles

Esta propiedad es un objeto de red de solo lectura.
