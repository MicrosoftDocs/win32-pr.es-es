# <a name="d1232-32bpp-rgba-causes-conversion"></a>D1232: 32bpp RGBA provoca la conversión

Considere la posibilidad de usar BGR en lugar de RGB para formatos de 32bpp para que el efecto BitmapSource pueda evitar conversiones de formato adicionales.
