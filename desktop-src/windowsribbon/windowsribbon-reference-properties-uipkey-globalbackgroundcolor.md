# <a name="ui_pkey_globalbackgroundcolor"></a>UI \_ PKEY \_ GlobalBackgroundColor

Identifica la propiedad GlobalBackgroundColor de PKEY de la interfaz de usuario \_ \_ .

```
propertyDescription
   name = UI_PKEY_GlobalBackgroundColor
   shellPKey = UI_PKEY_GlobalBackgroundColor
   formatID = 00002000-7363-696e-8441798acf5aebb7
   propID = 2000
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Observaciones

\_ \_ Una aplicación usa la interfaz de usuario PKEY GlobalBackgroundColor para consultar el color de fondo global de una cinta de opciones personalizada.

**Introducido en Windows 8:** la  [interfaz de usuario \_ PKEY \_ GlobalHighlightColor](windowsribbon-reference-properties-uipkey-globalhighlightcolor.md) no se puede establecer independientemente de la interfaz de usuario \_ PKEY \_ GlobalBackgroundColor.
