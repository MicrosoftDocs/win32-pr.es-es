# <a name="active-directory-operational-attributes"></a>Active Directory atributos operativos

Los atributos operativos son propiedades que se almacenan y usan en el directorio para fines administrativos. Por ejemplo, podría mantener un atributo operativo para registrar la fecha y la hora en que se modificó otro atributo. Los atributos operativos se deben recuperar explícitamente con una llamada a [**GetInfoEx**](/windows/desktop/api/Iads/nf-iads-iads-getinfoex).
