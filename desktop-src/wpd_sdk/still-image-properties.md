# <a name="still-image-properties"></a>Propiedades de imagen fija

Los dispositivos portátiles de Windows admiten las siguientes propiedades de imagen fija.



| Propiedad                                                                                                                                                       | VarType        | Descripción                                                                                                                                                                                                                                                                                                                  |
|----------------------------------------------------------------------------------------------------------------------------------------------------------------|----------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **el \_ intérprete de imágenes de la imagen de WPD \_ \_**                                                                                                                                  | **VT \_ LPWStr** | Nombre del propietario o usuario del dispositivo.                                                                                                                                                                                                                                                                                    |
| <span id="wpd_still_image_burst_interval"></span><span id="WPD_STILL_IMAGE_BURST_INTERVAL"></span>**\_intervalo de \_ ráfaga de imagen \_ \_ de WPD**                       | **VT \_ UI4**    | En una captura de imagen en modo de ráfaga, el intervalo de tiempo entre las capturas de imagen, en milisegundos.                                                                                                                                                                                                                                       |
| <span id="wpd_still_image_burst_number"></span><span id="WPD_STILL_IMAGE_BURST_NUMBER"></span>**\_número de \_ ráfaga de imagen \_ \_ de WPD**                             | **VT \_ UI4**    | En una captura de imagen en modo de ráfaga, el número de imágenes que se van a capturar.                                                                                                                                                                                                                                                              |
| **\_retraso de \_ captura de imagen \_ \_ de WPD**                                                                                                                          | **VT \_ UI4**    | Especifica el tiempo de retardo entre el comando para capturar una imagen y la captura real, en milisegundos.                                                                                                                                                                                                                    |
| **\_formato de \_ captura de imagen \_ \_ de WPD**                                                                                                                         | **VT \_ CLSID**  | Formato que se va a usar para capturar la imagen. Se trata de uno de los [valores de GUID de formato](object-format-guids.md) definidos por dispositivos portátiles de Windows.                                                                                                                                                                        |
| <span id="wpd_still_image_capture_mode"></span><span id="WPD_STILL_IMAGE_CAPTURE_MODE"></span>**\_modo de \_ captura de imagen \_ \_ de WPD**                             | **VT \_ UI4**    | Un enumerador de [**\_ \_ modos de captura de WPD**](wpd-capture-modes.md) que especifica el modo de captura que se va a utilizar para capturar una imagen.                                                                                                                                                                                                    |
| **la resolución de captura de imagen de WPD \_ sigue siendo \_ \_ \_**                                                                                                                     | **VT \_ LPWStr** | Cadena que especifica el tamaño de la imagen que se va a capturar. El tamaño de la imagen se especifica en píxeles, con el formato "*ancho* x *alto*". Por ejemplo: "800x600".                                                                                                                                                                          |
| **\_configuración de \_ compresión de imagen \_ \_ de WPD**                                                                                                                    | **VT \_ UI8**    | Describe la configuración de compresión de imagen que se usa al capturar una imagen. Esto es específico del dispositivo.                                                                                                                                                                                                                     |
| **contraste de la imagen de WPD \_ \_ \_**                                                                                                                                | **VT \_ UI4**    | Número que especifica la configuración de contraste que se va a aplicar al capturar una imagen, siendo cero el menor contraste y números mayores, lo que indica un mayor contraste.                                                                                                                                                            |
| **el \_ \_ \_ zoom digital Image \_**                                                                                                                           | **VT \_ UI4**    | Número que especifica la cantidad de zoom digital que se va a utilizar al capturar una imagen. El número es 1 o superior, escalado en un factor de 10. Un valor de 1 (que representa 10) indica que no hay zoom digital; un valor de 2 (que representa 20) indica un zoom de 2x (1/4 de la imagen que ha aparecido un zoom de 1X se capturará).                  |
| **\_modo de \_ efecto de imagen \_ \_ de WPD**                                                                                                                            | **VT \_ UI4**    | Un enumerador de [**\_ \_ modos de efecto de WPD**](wpd-effect-modes.md) que especifica un efecto de imagen que se va a utilizar al capturar una imagen.                                                                                                                                                                                                   |
| **compensación de la \_ \_ tendencia de exposición de imagen \_ \_ \_ de WPD**                                                                                                            | **VT \_ UI4**    | Un número que especifica las unidades de APEX para ajustar la exposición. Las unidades se detienen x 1.000. Si se especifica cero, indicará el valor predeterminado de fábrica.                                                                                                                                                                                       |
| **\_Índice de \_ exposición de imagen \_ \_ de WPD**                                                                                                                         | **VT \_ UI4**    | Número que especifica la velocidad de la película ISO que se va a emular al capturar una imagen. Un valor de 0xFFFF indica que este valor debe establecerse automáticamente.                                                                                                                                                                      |
| **\_modo de \_ medición de exposición de imagen \_ de \_ WPD \_**                                                                                                                | **VT \_ UI4**    | Un enumerador de modos de medición de exposición de WPD que especifica el modo de medición que se va a usar al capturar una imagen. [**\_ \_ \_**](wpd-exposure-metering-modes.md)                                                                                                                                                                          |
| <span id="wpd_still_image_exposure_program_mode"></span><span id="WPD_STILL_IMAGE_EXPOSURE_PROGRAM_MODE"></span>**\_modo de \_ programa de exposición de imagen \_ \_ de WPD \_** | **VT \_ UI4**    | Un enumerador de [**\_ \_ \_ modos de exposición de WPD**](wpd-exposure-program-modes.md) que especifica el programa de exposición que se va a usar al capturar una imagen.                                                                                                                                                                         |
| **tiempo de exposición de la \_ \_ imagen de WPD \_ \_**                                                                                                                          | **VT \_ UI4**    | Velocidad del obturador que se va a usar al capturar una imagen, en segundos x 10.000.                                                                                                                                                                                                                                                       |
| **\_modo de \_ imagen \_ del \_ modo Flash de WPD**                                                                                                                             | **VT \_ UI4**    | Un enumerador de [**\_ \_ modos Flash de WPD**](wpd-flash-modes.md) que especifica el modo flash que se va a usar al capturar una imagen.                                                                                                                                                                                                      |
| **\_ \_ \_ FNUMBER Image**                                                                                                                                 | **VT \_ UI4**    | Un número que especifica la abertura de la lente al capturar una imagen. El número es el número de detención de f 100 (de modo que 14 indicaría f/1.4). Normalmente, esta propiedad solo es válida cuando el [modo de programa de exposición de imagen de WPD \_ sigue \_ \_ \_ \_ siendo](/windows) la prioridad manual o de apertura.<br/> |
| **la \_ \_ \_ \_ longitud focal de la imagen de WPD sigue siendo**                                                                                                                           | **VT \_ UI4**    | La longitud focal, en milímetros x 100, que se va a usar al capturar una imagen. Por ejemplo, 35mm sería 3500.                                                                                                                                                                                                                 |
| **la \_ \_ distancia del \_ foco \_ de la imagen de WPD**                                                                                                                         | **VT \_ UI4**    | Distancia de enfoque que debe usar un dispositivo, en milímetros. Un valor de 0xFFFF indica un foco infinito (más de 655 metros).                                                                                                                                                                                              |
| **\_modo de \_ medición de foco de imagen \_ de \_ WPD \_**                                                                                                                   | **VT \_ UI4**    | Un enumerador de modos de medición de foco de WPD que especifica qué modo de enfoque debe usar un dispositivo de captura de imagen al decidir cómo centrarse en un asunto. [**\_ \_ \_**](wpd-focus-metering-modes.md)                                                                                                                                     |
| **\_modo de \_ enfoque de imagen \_ \_ de WPD**                                                                                                                             | **VT \_ UI4**    | Un enumerador de [**\_ \_ modos de enfoque de WPD**](wpd-focus-modes.md) que especifica el modo de enfoque que se va a utilizar al capturar una imagen.                                                                                                                                                                                                      |
| **\_ganancia de \_ Image \_ RGB de imagen fija \_**                                                                                                                               | **VT \_ LPWStr** | Una cadena terminada en null que especifica la ganancia de RGB que se va a usar al capturar una imagen. La cadena es de tres números delimitados por dos puntos sin espacios intermedios, con el formato "*R*:*G*:*B*".                                                                                                                                    |
| **la nitidez de la imagen de WPD \_ sigue siendo \_ \_**                                                                                                                               | **VT \_ UI4**    | Número que especifica la nitidez que se va a usar al capturar una imagen, donde cero es el menos nítido y los números mayores indican un aumento de la nitidez.                                                                                                                                                                        |
| <span id="wpd_still_image_timelapse_interval"></span><span id="WPD_STILL_IMAGE_TIMELAPSE_INTERVAL"></span>**\_ \_ \_ intervalo TIMELAPSE de la imagen de WPD \_**           | **VT \_ UI4**    | Tiempo de retardo entre las capturas de imagen en una captura de imagen de lapso de tiempo, en milisegundos.                                                                                                                                                                                                                                        |
| <span id="wpd_still_image_timelapse_number"></span><span id="WPD_STILL_IMAGE_TIMELAPSE_NUMBER"></span>**número de TIMELAPSE de la \_ \_ imagen de WPD \_ \_**                 | **VT \_ UI4**    | El número de imágenes que se van a capturar en una captura de imagen del modo time-lapse.                                                                                                                                                                                                                                                          |
| **\_URL de \_ carga de imagen \_ \_ de WPD**                                                                                                                             | **VT \_ LPWStr** | Especifica una dirección URL en la que el dispositivo debe enviar la imagen después de capturarla.                                                                                                                                                                                                                                                 |
| **\_ \_ \_ equilibrio en blanco de la imagen de WPD \_**                                                                                                                          | **VT \_ UI4**    | Un valor de balance de blanco de WPD con valores enumerados que especifica la configuración de balance de blanco que se va a usar al capturar una imagen. [**\_ \_ \_**](wpd-white-balance-settings.md)                                                                                                                                                             |
