# <a name="swbemprivilegeisenabled-property"></a>Propiedad SWbemPrivilege. IsEnabled

La propiedad **IsEnabled** de un objeto [**SWbemPrivilege**](swbemprivilege.md) es un valor booleano que se usa para habilitar o deshabilitar este privilegio. Para obtener más información sobre las consecuencias de habilitar privilegios específicos, vea [**ejecutar con privilegios especiales**](swbemsecurity-privileges.md).

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemPrivilege.IsEnabled As Boolean
```
