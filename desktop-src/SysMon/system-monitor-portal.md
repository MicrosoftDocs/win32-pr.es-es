# <a name="system-monitor"></a>Monitor de sistema

## <a name="purpose"></a>Propósito

El monitor de sistema (SYSMON) es la interfaz de programación de aplicaciones (API) que se usa para configurar el control ActiveX del monitor de sistema de Microsoft. El control Monitor de sistema permite ver datos de contadores de rendimiento en tiempo real y registrados previamente, como la memoria, el disco y los datos de contadores del procesador.

## <a name="where-applicable"></a>Donde sea aplicable

Use la API SYSMON al agregar el control de supervisión del sistema a cualquier contenedor habilitado para ActiveX, como un documento de Microsoft Word, o una página web que vea Internet Explorer y desee modificar su comportamiento predeterminado. Por ejemplo, puede crear un monitor de rendimiento que solo muestre un conjunto predefinido de contadores del sistema o con configuraciones de pantalla predefinidas.

## <a name="developer-audience"></a>Audiencia de desarrolladores

SYSMON está diseñado para que lo usen C, C++, Visual Basic y los desarrolladores de scripting.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

SYSMON está disponible a partir de Microsoft Windows 2000.

Para obtener información sobre los requisitos de tiempo de ejecución para un elemento de programación determinado, vea la sección de requisitos de la página de referencia de ese elemento.
