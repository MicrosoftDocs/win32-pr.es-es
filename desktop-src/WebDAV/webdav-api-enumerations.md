|--------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AUTHNEXTSTEP**](/windows/desktop/api/davclnt/ne-davclnt-authnextstep) | Especifica la siguiente acción que el cliente WebDAV debe realizar después de una llamada correcta a la función de devolución de llamada [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) . |
