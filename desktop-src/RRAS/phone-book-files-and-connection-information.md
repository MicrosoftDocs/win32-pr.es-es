<span data-ttu-id="8f711-111">Para establecer una conexión sin usar una entrada de libreta de teléfonos, la llamada [**rasdial**](/windows/desktop/api/Ras/nf-ras-rasdiala) puede especificar una cadena vacía para el miembro **szEntryName** de la estructura [**RASDIALPARAMS**](/previous-versions/windows/desktop/legacy/aa377238(v=vs.85)) .</span><span class="sxs-lookup"><span data-stu-id="8f711-111">To make a connection without using a phone-book entry, the [**RasDial**](/windows/desktop/api/Ras/nf-ras-rasdiala) call can specify an empty string for the **szEntryName** member of the [**RASDIALPARAMS**](/previous-versions/windows/desktop/legacy/aa377238(v=vs.85)) structure.</span></span> <span data-ttu-id="8f711-112">El miembro **RASDIALPARAMS. szPhoneNumber** debe contener el número al que llamar.</span><span class="sxs-lookup"><span data-stu-id="8f711-112">The **RASDIALPARAMS.szPhoneNumber** member must contain the number to call.</span></span> <span data-ttu-id="8f711-113">En este caso, el administrador de conexiones de acceso remoto utiliza el primer puerto de módem disponible y los valores predeterminados para el resto de configuraciones.</span><span class="sxs-lookup"><span data-stu-id="8f711-113">In this case, the Remote Access Connection Manager uses the first available modem port and default values for all other settings.</span></span>
