# <a name="commonly-used-windows-media-file-attributes"></a>Atributos de archivo de Windows Media de uso frecuente

Los archivos de Windows Media pueden contener cualquier número de atributos, incluidos los atributos definidos por el usuario.

Si el archivo se ha agregado a la biblioteca, también pueden estar disponibles los atributos adicionales asociados al tipo de medio. Vea [elementos de audio](audio-item-attributes.md) o [elementos de vídeo](video-item-attributes.md) para los atributos adicionales.

Para obtener más información sobre los atributos de archivo de Windows Media, vea las [directrices de uso de metadatos de Windows Media](/previous-versions/ms867702(v=msdn.10)).

La lista siguiente contiene los atributos utilizados con frecuencia.

-   [Atributo Author](author-attribute.md)
-   [Atributo title](title-attribute.md)
-   [Atributo WM/AlbumArtist](wm-albumartist-attribute.md)
-   [Atributo WM/ContentDistributor](wm-contentdistributor-attribute.md)
-   [Atributo WM/Genre](wm-genre-attribute.md)
-   [Atributo WM/MCDI](wm-mcdi-attribute.md)
-   [Atributo WM/MediaClassPrimaryID](wm-mediaclassprimaryid-attribute.md)
-   [Atributo WM/MediaClassSecondaryID](wm-mediaclasssecondaryid-attribute.md)
-   [Atributo WM/proveedor](wm-provider-attribute.md)
-   [Atributo WM/TrackNumber](wm-tracknumber-attribute.md)
