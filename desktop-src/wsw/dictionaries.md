# <a name="dictionaries"></a>Diccionarios

Los diccionarios permiten la tokenización de datos de nodo comunes, como nombres de elemento o atributo, espacios de nombres y texto común en un formato XML binario.


Se [**puede>**](/windows/desktop/api/WebServices/ns-webservices-ws_xml_dictionary) diccionario como parte del tipo STRING XML [**\_ \_ de WS.**](/windows/desktop/api/WebServices/ns-webservices-ws_xml_string) Todas las cadenas dentro de un único **diccionario>** deben ser únicas.

Los identificadores de diccionario se admiten para todas las codificaciones (no solo el formato XML binario).

La siguiente función se usa con diccionarios:

-   [**WsGetDictionary**](/windows/desktop/api/WebServices/nf-webservices-wsgetdictionary)
