# <a name="line"></a><span data-ttu-id="8eae1-104">Línea</span><span class="sxs-lookup"><span data-stu-id="8eae1-104">Line</span></span>

<span data-ttu-id="8eae1-105">El concepto de línea ha evolucionado a lo largo del tiempo y se ha sustituido en parte por los conceptos de dirección y terminal.</span><span class="sxs-lookup"><span data-stu-id="8eae1-105">The line concept has evolved over time and been partly superseded by the Address and Terminal concepts.</span></span> <span data-ttu-id="8eae1-106">TAPI 3 no usa directamente el concepto de línea, pero TAPI 2 continúa incorporando este paradigma.</span><span class="sxs-lookup"><span data-stu-id="8eae1-106">TAPI 3 does not directly use the concept of line, but TAPI 2 continues to incorporate this paradigm.</span></span>

<span data-ttu-id="8eae1-107">Un *dispositivo de línea* es un dispositivo físico, como un panel de fax, un módem o una tarjeta ISDN (RDSI) que está conectada a una red.</span><span class="sxs-lookup"><span data-stu-id="8eae1-107">A *line device* is a physical device such as a fax board, a modem, or an ISDN card that is connected to a network.</span></span> <span data-ttu-id="8eae1-108">Es posible que el dispositivo no esté conectado físicamente al equipo en el que se ejecuta la aplicación TAPI, como un grupo de módems en un servidor.</span><span class="sxs-lookup"><span data-stu-id="8eae1-108">The device may not be physically connected to the computer on which the TAPI application is running, such as a modem pool on a server.</span></span> <span data-ttu-id="8eae1-109">Los dispositivos de línea admiten capacidades de comunicaciones al permitir que las aplicaciones envíen o reciban información de una red.</span><span class="sxs-lookup"><span data-stu-id="8eae1-109">Line devices support communications capabilities by allowing applications to send information to or receive information from a network.</span></span> <span data-ttu-id="8eae1-110">Un dispositivo de línea contiene un conjunto de uno o más canales homogéneos que se pueden utilizar para establecer llamadas.</span><span class="sxs-lookup"><span data-stu-id="8eae1-110">A line device contains a set of one or more homogeneous channels that can be used to establish calls.</span></span>

<span data-ttu-id="8eae1-111">En las aplicaciones TAPI 2. x, un dispositivo de línea es la representación lógica de un dispositivo telefónico físico.</span><span class="sxs-lookup"><span data-stu-id="8eae1-111">Within TAPI 2.x applications, a line device is the logical representation of a physical telephone device.</span></span> <span data-ttu-id="8eae1-112">Aunque "line" suele indicar algo con dos puntos de conexión, es posible abstraer un dispositivo de línea en un único punto porque TAPI solo lo ve como un punto de entrada a la línea que conduce al conmutador.</span><span class="sxs-lookup"><span data-stu-id="8eae1-112">Although "line" often connotes something with two endpoints, it is possible to abstract a line device to a single point because TAPI views it only as a point of entry to the line that leads to the switch.</span></span>

<span data-ttu-id="8eae1-114">Aunque las tres líneas de la ilustración anterior se componen de hardware diferente y se usan para distintas funciones, se abstraen en el mismo tipo de dispositivo y se rigen por las mismas reglas.</span><span class="sxs-lookup"><span data-stu-id="8eae1-114">Although the three lines in the preceding illustration are composed of different hardware and used for different functions, they are abstracted to the same device type and governed by the same rules.</span></span> <span data-ttu-id="8eae1-115">El teléfono no representa un dispositivo de teléfono, sino un dispositivo de línea usado para las llamadas de voz.</span><span class="sxs-lookup"><span data-stu-id="8eae1-115">The telephone represents not a phone device but a line device used for voice calls.</span></span> <span data-ttu-id="8eae1-116">Cuando se usa este dispositivo de línea para llamadas entrantes o salientes, la aplicación también necesita abrir y controlar una instancia de la clase de dispositivo de teléfono, que se describe en detalle en secciones posteriores.</span><span class="sxs-lookup"><span data-stu-id="8eae1-116">When using this line device for incoming or outgoing calls, the application would also need to open and control an instance of the phone-device class, which is described in detail in later sections.</span></span>
