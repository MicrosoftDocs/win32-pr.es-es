# <a name="listboxborder"></a>LISTBOX. Border

El atributo **Border** especifica o recupera un valor que indica si el control de cuadro de lista tiene un borde.

``` syntax
        elementID.border
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un valor **booleano** .



| Value | Descripción                                  |
|-------|----------------------------------------------|
| true  | Predeterminada. El control de cuadro de lista tiene un borde.  |
| false | El control de cuadro de lista no tiene borde. |



 

## <a name="remarks"></a>Observaciones

El borde aparece como un borde hundido de dos píxeles alrededor del control cuadro de lista con los colores del sistema.
