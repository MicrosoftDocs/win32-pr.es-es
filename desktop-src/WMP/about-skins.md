<span data-ttu-id="17146-107">Con Windows Media Player, los usuarios pueden elegir entre una variedad de máscaras estándar, cada una de las cuales proporciona una experiencia visual adicional que mejora la escucha y la visualización de los placer.</span><span class="sxs-lookup"><span data-stu-id="17146-107">With Windows Media Player, users can choose from a variety of standard skins, each one providing an additional visual experience that enhances listening and viewing pleasure.</span></span> <span data-ttu-id="17146-108">Windows Media Player incluye varias máscaras entre las que elegir, pero es relativamente fácil crear y distribuir máscaras personalizadas.</span><span class="sxs-lookup"><span data-stu-id="17146-108">Windows Media Player comes with several skins to choose from, but it is relatively easy to create and distribute custom skins.</span></span>

<span data-ttu-id="17146-109">Las máscaras son simplemente colecciones de uno o varios archivos de imagen de equipo, organizadas por un archivo de texto, que indica a Windows Media Player cómo usar estos archivos para mostrar una máscara.</span><span class="sxs-lookup"><span data-stu-id="17146-109">Skins are simply collections of one or more computer art files, organized by a text file, which tells Windows Media Player how to use these files to display a skin.</span></span> <span data-ttu-id="17146-110">Puede usar diversos programas para modificar el arte existente o crear un arte sencillo.</span><span class="sxs-lookup"><span data-stu-id="17146-110">You can use a variety of programs to modify existing art or create simple art of your own.</span></span> <span data-ttu-id="17146-111">Por ejemplo, con las herramientas de dibujo y la imagen prediseñada suministradas con Microsoft PowerPoint, puede crear material gráfico arrastrando y colocando.</span><span class="sxs-lookup"><span data-stu-id="17146-111">For example, using the drawing tools and clip art supplied with Microsoft PowerPoint, you can create artwork by just dragging and dropping.</span></span> <span data-ttu-id="17146-112">Del mismo modo, no es necesario que sea programador crear el archivo de texto que se usa para definir una máscara individual.</span><span class="sxs-lookup"><span data-stu-id="17146-112">Similarly, you do not need to be a programmer to create the text file used to define an individual skin.</span></span>

<span data-ttu-id="17146-113">Uno de los motivos por los que las máscaras son relativamente fáciles de crear es que puede examinar los elementos de arte individuales y el código fuente de cualquier aspecto y ver cómo se colocan cada máscara.</span><span class="sxs-lookup"><span data-stu-id="17146-113">One of the reasons that skins are relatively easy to create is that you can look at the individual pieces of art and the source code for any skin and see how each skin is put together.</span></span> <span data-ttu-id="17146-114">Se proporcionan varios aspectos de ejemplo sencillos con este SDK, así como una referencia para los elementos y atributos de la máscara.</span><span class="sxs-lookup"><span data-stu-id="17146-114">Several simple example skins are provided with this SDK, as well as a reference for skin elements and attributes.</span></span>
