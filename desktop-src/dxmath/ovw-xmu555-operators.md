# <a name="xmu555-operators"></a>Operadores XMU555

La [**estructura XMU555**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu555) expone los operadores siguientes.

> [!Note]  
> Los operadores enumerados aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                           | Descripción                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operador uint16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmu555-operatoruint16_t)<br/> | Devuelve una instancia de `uint16_t` que contiene los componentes de la instancia en un formato `XMU555` empaquetado. <br/> |
| [**operator =**](xmu555-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMU555` . <br/>                                       |
