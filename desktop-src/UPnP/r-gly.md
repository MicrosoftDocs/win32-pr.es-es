# <a name="r-upnp-apis"></a>R (API de UPnP)

Contiene términos relacionados con UPnP que comienzan por la letra R.

<dl> <dt>

<span id="upnp.r_1_gly"></span><span id="UPNP.R_1_GLY"></span>**directorio de recursos**
</dt> <dd>

El directorio de recursos es un directorio local del dispositivo donde se encuentran los archivos de icono.

</dd> <dt>

<span id="upnp.r_2_gly"></span><span id="UPNP.R_2_GLY"></span>**dispositivo raíz**
</dt> <dd>

Un dispositivo raíz es el dispositivo de nivel superior en un árbol de dispositivos. Por ejemplo, si tiene acceso a un dispositivo de combinación de VCR o TV denominado estación de visualización, hay tres dispositivos: la estación de visualización, el VCR y el televisor. La estación de visualización se considera el dispositivo raíz.

</dd> <dt>

<span id="upnp.r_3_gly"></span><span id="UPNP.R_3_GLY"></span>**dispositivo en ejecución**
</dt> <dd>

Un dispositivo en ejecución es un dispositivo que ya ha cargado su código antes del registro.

Se trata de dispositivos para los que no se conoce una descripción del servicio antes del registro.

Algunos ejemplos de estos dispositivos son los puentes, como un puente de SCP de UPnP. Este puente registra los dispositivos en ejecución con el host del dispositivo.

</dd> </dl>
