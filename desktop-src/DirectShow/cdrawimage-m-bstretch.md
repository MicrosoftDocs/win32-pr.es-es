# <a name="cdrawimagem_bstretch-member"></a>Miembro CDrawImage::m \_ bStretch

La `m_bStretch` variable miembro indica si la imagen de vídeo debe ajustarse a la ventana de destino. El valor es **FALSE si** el rectángulo de origen tiene el mismo tamaño que el rectángulo de destino o **TRUE** en caso contrario.

## <a name="syntax"></a>Sintaxis


```C++
BOOL m_bStretch;
```
