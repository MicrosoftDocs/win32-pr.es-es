# <a name="rebootprompt-property"></a>Propiedad REBOOTPROMPT

Si la propiedad **REBOOTPROMPT** se establece en suprimir (o simplemente S), cualquier reinicio realizado por el Windows Installer se produce automáticamente sin interacción del usuario. Si se establece esta propiedad, no se inicia un reinicio si no se necesita una, solo se suprime la presentación de los mensajes para los reinicios del usuario.

| Requisito | Value |
|--------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

Windows Installer en Windows Server 2003 o Windows XP. Consulte los [requisitos de Run-Time de Windows Installer](windows-installer-portal.md) para obtener información sobre la Service Pack mínima de Windows que requiere una versión Windows Installer.<br/> |
