# <a name="using-the-editor-with-a-different-speech-engine"></a>Usar el editor con un motor de voz diferente

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Aunque el editor de sonidos instala el motor de reconocimiento de voz de Microsoft (4,0), se puede usar con otro motor de voz si dicho motor es compatible con las interfaces requeridas documentadas en el documento de requisitos del motor de voz. Antes de intentar usar el editor con otro motor, confirme con el proveedor que cumplen estos requisitos.

Para usar el editor de sonidos con otro motor de voz, elija el comando motor de voz en el menú **edición** . Se muestra un cuadro de diálogo que muestra el motor actual en uso. Para elegir otro motor, muestre la lista de motores y, a continuación, haga clic en **Aceptar**. Si no aparecen otros motores, no se incluye ningún otro motor compatible.
