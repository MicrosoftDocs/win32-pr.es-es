# <a name="vml-h-element"></a>Elemento H de VML

En este tema se describe VML, una característica que está desusada en Windows Internet Explorer 9. Las páginas web y las aplicaciones que se basan en VML se deben migrar a SVG u otros estándares ampliamente admitidos.

> [!Note]  
> A partir del 2011 de diciembre, este tema se ha archivado. Como resultado, ya no se mantiene de forma activa. Para obtener más información, vea [contenido archivado](/previous-versions/windows/internet-explorer/ie-developer/). Para obtener información, recomendaciones e instrucciones sobre la versión actual de Windows Internet Explorer, consulte [Centro para desarrolladores de Internet Explorer](https://msdn.microsoft.com/ie/).

 

Define un identificador para una forma.

Los atributos siguientes modifican un identificador.



| Atributo                                                | Descripción                                                    |
|----------------------------------------------------------|----------------------------------------------------------------|
| [InvX](msdn-online-vml-invx-attribute.md)               | Determina si se invierte la posición *x* del identificador. |
| [InvY](msdn-online-vml-invy-attribute.md)               | Determina si se invierte la posición *y* del identificador. |
| [Map](msdn-online-vml-map-attribute.md)                 | Define el intervalo de asignación de un identificador.                         |
| [Polar](msdn-online-vml-polar-attribute.md)             | Define la posición central para los manipuladores polares.                 |
| [Position](position-attribute--h--vml.md)               | Especifica los  valores x *e y del* identificador.                |
| [RadiusRange](msdn-online-vml-radiusrange-attribute.md) | Define el intervalo de un identificador polar.                           |
| [Switch](msdn-online-vml-switch-attribute.md)           | Determina si las direcciones de identificador se intercambian.          |
| [XRange](msdn-online-vml-xrange-attribute.md)           | Define el intervalo *x* de un identificador.                             |
| [YRange](msdn-online-vml-yrange-attribute.md)           | Define el intervalo *y* de un identificador.                             |



 

**Comentarios:**

Este elemento debe definirse dentro de un elemento de [forma](shape-element--vml.md) .
