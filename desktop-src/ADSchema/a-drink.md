# <a name="drink-attribute"></a>Beba (atributo)

El tipo de atributo beba (bebida favorita) especifica la bebida favorita de un objeto (o persona).



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | beber                                       |
| Nombre para mostrar de LDAP | beber                                       |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.5                   |
| System-ID-GUID    | 1a1aa5b5-262e-4df6-af04-2cf6b0d80048        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
