# <a name="decoding-to-spdif-microsoft-media-foundation"></a>Descodificación a S/PDIF (Microsoft Media Foundation)

El identificador de clase para el objeto descodificador que transforma Windows Media Audio contenido de 9,1 Professional para su entrega a través del formato de interconexión digital de Sony/Philips (S/PDIF) es CLSID \_ CWMAudioSpdTxDMO. Este objeto se utiliza como lo haría con cualquier otro descodificador. Admite las mismas propiedades que el descodificador de Windows Media Audio 9. Los tipos de salida enumerados por este descodificador son todos formatos S/PDIF.

Para que la salida S/PDIF sea totalmente compatible, el equipo cliente debe tener instalado hardware especial.
