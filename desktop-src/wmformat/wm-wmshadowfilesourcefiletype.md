# <a name="wmwmshadowfilesourcefiletype-windows-media-format-11-sdk"></a>WM/WMShadowFileSourceFileType (Windows SDK de formato multimedia 11)

**WM/WMShadowFileSourceFileType** contiene el tipo de archivo del archivo original del que se deriva el archivo ASF.

## <a name="global-constant"></a>Constante global

g \_ wszWMWMShadowFileSourceFileType

## <a name="data-type"></a>Tipo de datos

**CADENA DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentarios

El contenido que existe en un formato de archivo distinto de ASF y que está protegido con algún tipo de administración de derechos se puede convertir en un archivo multimedia de Windows protegido por drm multimedia de Windows a través de un proceso denominado importación de licencias. Este tipo de archivo ASF debe contener este atributo, así como WM/WMShadowFileSourceDRMType para realizar un seguimiento de dónde provenía el contenido.
