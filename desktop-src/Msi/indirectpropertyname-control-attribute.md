# <a name="indirectpropertyname-control-attribute"></a>Atributo de control IndirectPropertyName

Este atributo es el nombre de una propiedad indirecta asociada al control. Si se establece el bit [indirecto](indirect-control-attribute.md) , el control muestra o cambia el valor de la propiedad que tiene este nombre y este nombre es el valor de la propiedad que aparece en la columna de propiedades de la [tabla de control](control-table.md).

En el caso de los controles inertes, este atributo devuelve NULL.

Todos los controles activos.

## <a name="associated-bit-flags"></a>Marcas de bits asociadas

Este atributo no utiliza marcas de bits.

Vea [atributos de control](control-attributes.md) y el control que debe crear en [controles](controls.md).
