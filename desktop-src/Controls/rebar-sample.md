# <a name="rebar-sample"></a>Ejemplo de rebar

En este tema se describe el ejemplo de código Rebar Sample. Contiene las secciones siguientes:

-   [Descripción](#description)
-   [Requisitos mínimos](#minimum-requirements)
-   [Descargar el ejemplo](#downloading-the-sample)
-   [Compilar el ejemplo](#building-the-sample)
-   [Temas relacionados](#related-topics)

## <a name="description"></a>Descripción

El ejemplo rebar muestra cómo implementar un control común de rebar simple en una aplicación. En este ejemplo se crea un control rebar que tiene dos bandas. Una contiene un cuadro combinado y la otra contiene un botón de inserción.

## <a name="minimum-requirements"></a>Requisitos mínimos



| Producto          | Versión                               |
|------------------|---------------------------------------|
| Archivo DLL              | comctl32.dll versión 4.71             |
| Sistema operativo | Windows 95 con Internet Explorer 4.0 |



 

## <a name="downloading-the-sample"></a>Descargar el ejemplo

El ejemplo rebar se instala como parte de [Windows Software Development Kit (SDK)](https://msdn.microsoft.com/windows/bb980924.aspx) y está disponible en la siguiente ubicación.



| Location    | Ruta de acceso o dirección URL                                                                                              |
|-------------|-------------------------------------------------------------------------------------------------------|
| Windows SDK | %Program Files% \\ Microsoft SDKs \\ Windows version number Samples \\ \[ \] \\ \\ winui controls common \\ \\ \\ rebar |



 

## <a name="building-the-sample"></a>Generar el ejemplo

Para compilar el ejemplo mediante el símbolo del sistema:

1.  Abra la ventana símbolo del sistema y vaya al directorio del proyecto.
2.  Escriba `msbuild [project file]`.

Para compilar el ejemplo mediante Visual Studio:

1.  Abra Windows Explorador de proyectos y vaya al directorio del proyecto.
2.  Haga doble clic en el icono del archivo .vcproj para abrir el proyecto en Visual Studio.
3.  En el **menú Compilar,** seleccione **Compilar solución** para compilar la solución.
