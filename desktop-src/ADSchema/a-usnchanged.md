# <a name="usn-changed-attribute"></a>USN-Changed atributo)

El número de secuencia de actualización (USN) asignado por el directorio local para el cambio más reciente, incluida la creación. Vea también [**USN-created**](a-usncreated.md).



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | USN-Changed                          |
| Nombre para mostrar de LDAP | uSNChanged                           |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | Cuando se cambia el objeto.          |
| Attribute-Id      | 1.2.840.113556.1.2.120               |
| System-ID-GUID    | bf967a6f-0de6-11d0-a285-00aa003049e2 |
| Sintaxis            | [**Interval**](s-interval.md)       |
