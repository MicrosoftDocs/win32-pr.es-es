# <a name="enumeration-types"></a>Tipos de enumeración

Sensor API define los siguientes tipos de enumeración.



| Enumeración                                                                | Descripción                                      |
|----------------------------------------------------------------------------|--------------------------------------------------|
| [**SensorConnectionType**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorconnectiontype) | Define los tipos de conexiones de dispositivo de sensor.      |
| [**SensorState**](/windows/win32/api/sensorsapi/ne-sensorsapi-sensorstate)                   | Define los posibles estados operativos de los sensores. |
