# <a name="account-object"></a>Objeto de cuenta

El objeto de **cuenta** se usa para asignar [*privilegios*](/windows/desktop/SecGloss/p-gly), acceso al sistema y cuotas especiales a usuarios individuales o a miembros de grupos o grupos locales.

La informaci√≥n almacenada en un objeto de **cuenta** incluye:

-   El SID de una cuenta de usuario, una cuenta de grupo o una cuenta de grupo local.
-   Una lista de privilegios que se asignan al usuario o a los miembros de un grupo. Si a un usuario se le asignan privilegios en varios objetos de **cuenta** , la lista incluye todos los privilegios asignados.
