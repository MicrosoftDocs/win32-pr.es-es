# <a name="sessiondatabase-property"></a>Propiedad Session.Database

La **propiedad Database** del objeto [**Session**](session-object.md) es una propiedad de solo lectura que devuelve la base de datos de la sesión de instalación actual como un [**objeto Database.**](database-object.md)

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Session.Database
```
