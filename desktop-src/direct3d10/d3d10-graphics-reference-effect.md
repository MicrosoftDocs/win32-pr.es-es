# <a name="effect-reference-direct3d-10-graphics"></a>Referencia de efectos (gráficos de Direct3D 10)

La API de Direct3D define varios elementos de API que le ayudarán a crear y administrar el sistema de efectos.

-   [Interfaces](d3d10-graphics-reference-effect-interfaces.md)
-   [Funciones](d3d10-graphics-reference-effect-functions.md)
-   [Estructuras](d3d10-graphics-reference-effect-structures.md)
-   [Constantes](d3d10-graphics-reference-effect-constants.md)

El sistema de efectos encapsula las asignaciones de estado en un efecto. Cada efecto es una colección de estado de la canalización que se puede dividir en las asignaciones de estado mediante bloques de estado, recursos y sombreadores. Dependiendo del tamaño de un efecto, puede optar por implementar uno en una cadena de texto ASCII o en un archivo de efectos (. FX). Para organizar la información de un efecto, vea el [formato del efecto](d3d10-effect-format.md).

Una vez que haya diseñado un efecto, use la API de efecto para establecer el estado en el dispositivo durante la representación. Además, el sistema de efectos es compatible con una serie de API de reflexión para obtener y establecer datos de efectos. Para obtener más información, vea [interfaces del sistema de efectos (Direct3D 10)](d3d10-graphics-programming-guide-effects-interfaces.md).
