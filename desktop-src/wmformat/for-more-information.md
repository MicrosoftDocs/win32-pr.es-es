# <a name="for-more-information-windows-media-format-sdk"></a>Más información (SDK de Windows Media Format)

-   Para obtener la información más reciente acerca de los problemas conocidos del SDK de Windows Media Format, vea las notas de la versión publicadas en el [sitio web de MSDN Library](/documentation/?url=%2flibrary%2fdnwmt%2fhtml%2fwmfsdk11_b2_release_notes_rzxb.asp%3fframe%3dtrue).
-   Para obtener más información sobre las características, los artículos y las respuestas a las preguntas más frecuentes sobre los medios de streaming en general y los SDK de Windows Media en concreto, vea el centro para desarrolladores de Windows Media en el [sitio web de Microsoft](https://msdn.microsoft.com/windowsmedia/default.aspx).
-   Para obtener más información acerca de cómo obtener una licencia para el SDK de Windows Media Format, vea la página de licencias de Windows Media en el [sitio web de Microsoft](https://msdn.microsoft.com/library/ms867178.aspx).
-   Para obtener más información acerca de los grupos de noticias, visite la página grupos de noticias y alias de correo electrónico de Windows Media en el [sitio web de Microsoft](https://windows.microsoft.com/windows/products/windows-media).
-   Para obtener descripciones detalladas de bajo nivel de los objetos de datos que componen archivos ASF, descargue la especificación ASF desde el [sitio web de Microsoft](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).
-   Para obtener información sobre las características, los artículos, los procedimientos recomendados y las respuestas a las preguntas más frecuentes sobre la administración de derechos digitales (DRM) y el administrador de derechos de Windows Media, consulte los artículos técnicos en la página de [Rights Management digital](/documentation/?url=%2flibrary%2fdnwmt%2fhtml%2fdigitalrightsmanagement.asp) en el sitio web de Microsoft.
