# <a name="task-scheduler-10-enumerated-types"></a>Programador de tareas tipos enumerados de 1.0

\[\[Esta API puede modificarse o no estar disponible en versiones posteriores del sistema operativo o del producto. Use las [interfaces Programador de tareas 2.0 o](task-scheduler-2-0-interfaces.md) Programador de tareas tipos enumerados [de 2.0](task-scheduler-2-0-enumerated-types.md) en su lugar. \]\]

Describe los tipos enumerados que definen las constantes que usan las PROGRAMADOR DE TAREAS API de Programador de tareas 1.0.


Los siguientes tipos enumerados se presentan en Programador de tareas 1.0.



| Nombre                                             | Descripción                                                                        |
|--------------------------------------------------|------------------------------------------------------------------------------------|
| [**TIPO DE \_ DESENCADENADOR \_ DE TAREA**](/windows/desktop/api/Mstask/ne-mstask-task_trigger_type) | Define los tipos de [*desencadenadores*](t.md) asociados a una tarea. |
| [**TASKPAGE**](/windows/desktop/api/Mstask/ne-mstask-taskpage)                     | Define el tipo de página de tareas que se va a recuperar.                                     |
