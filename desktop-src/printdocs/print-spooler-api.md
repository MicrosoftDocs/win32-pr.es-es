# <a name="print-spooler-api"></a>API del administrador de trabajos de impresión

La API del administrador de trabajos de impresión proporciona una interfaz al administrador de trabajos de impresión para que las aplicaciones administren impresoras y trabajos de impresión.

Una aplicación utiliza la API del administrador de trabajos de impresión como parte de su programación y no directamente por los usuarios finales.

Esta sección contiene información acerca de los temas siguientes.



| Tema                                                                                             | Descripción                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Referencia de la API del administrador de trabajos de impresión](printing-and-print-spooler-reference.md)<br/>                | Información detallada sobre las funciones, estructuras y otros elementos de la API del administrador de trabajos de impresión.<br/>                                                                                                           |
| [Referencia de notificación de impresión asincrónica](asynchronous-printing-notification.md)<br/> | Descripciones de las funciones, interfaces y enumeraciones que se utilizan en la comunicación asincrónica entre las aplicaciones y los componentes hospedados en la cola de impresión, como controladores de impresora y monitores de puerto.<br/> |
| [Referencia de instalación del controlador de impresora](printer-driver-installation-reference.md)<br/>     | Describe las funciones que instalan y configuran controladores de impresora en un equipo.<br/>                                                                                                                            |
