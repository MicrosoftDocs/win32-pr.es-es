# <a name="icm_setquality-message"></a>\_Mensaje SETQUALITY ICM

El **mensaje \_ SETQUALITY ICM** proporciona un controlador de compresión de vídeo con un nivel de calidad que se utiliza durante la compresión.


```C++
ICM_SETQUALITY 
wParam = (DWORD) (LPVOID) &dwICValue; 
lParam = 0; 
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

<span id="dwICValue"></span><span id="dwicvalue"></span><span id="DWICVALUE"></span>*dwICValue*
</dt> <dd>

Nuevo valor de calidad. Los valores de calidad oscilan entre 0 y 10.000.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Devuelve ICERR \_ OK si el controlador admite este mensaje o ICERR \_ no se admite en caso contrario.
