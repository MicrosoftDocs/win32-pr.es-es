# <a name="msvm_batterysettingdata-class"></a>Clase \_ BatterySettingData de Msvm

Representa la configuración de un dispositivo de batería virtual.

La siguiente sintaxis se simplifica desde el código de Managed Object Format (MOF) e incluye todas las propiedades heredadas.

## <a name="syntax"></a>Sintaxis

``` syntax
[Dynamic, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_BatterySettingData : CIM_ResourceAllocationSettingData
{
};
```

## <a name="members"></a>Miembros

La **clase \_ BatterySettingData de Msvm** no define ningún miembro.
