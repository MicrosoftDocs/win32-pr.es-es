# <a name="boolean-syntax"></a>Sintaxis booleana

Representa un valor booleano.



| Entrada | Value |
|--------------|--------------------------------------------------------------------------|
| Nombre         | Boolean                                                                  |
| IDENTIFICADOR de sintaxis    | 2.5.5.8                                                                  |
| IDENTIFICADOR DE OM        | 1                                                                        |
| Tipo MAPI    | BOOLEAN                                                                  |
| Tipo ADS     | ADSTYPE \_ booleano                                                         |
| Tipo Variant | VT \_ bool                                                                 |
| Tipo de SDS     | [System.Boolean](/dotnet/api/system.boolean) |



## <a name="remarks"></a>Observaciones

En el caso de las consultas que incluyen atributos de sintaxis booleana en un filtro, especifique "TRUE" o "FALSE" (por ejemplo, "myboolattr = TRUE").
