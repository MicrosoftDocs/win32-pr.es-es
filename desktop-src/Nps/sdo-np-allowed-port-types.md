# <a name="np-allowed-port-types"></a>NP-Allowed-Port-Types

Tipos de puerto permitidos para una conexión.

**Nombre LDAP:** msNPAllowedPortTypes

**IDENTIFICADOR:** 4104 = TIPOS DE PUERTO PERMITIDOS POR NP DEL ATRIBUTO [ **\_ \_ \_ \_ IAS \_**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintaxis:** Enumerador

**Valores: **

Definido por internet Assigned Numbers Authority (IANA) en el documento [RADIUS TYPES](https://www.iana.org/assignments/radius-types) basado en [RFC 2865](https://www.ietf.org/rfc/rfc2865.txt).

-   Async (Módem) (0)
-   Sync (línea T1) (1)
-   Sincronización de ISDN (2)
-   ISDN Async V.120 (3)
-   ISDN Async V.110 (4)
-   Virtual (VPN) (5)
-   PIAFS (6)
-   HDLC Clear Channel (7)
-   X.25 (8)
-   X.75 (9)
-   G.3 Fax (10)
-   SDSL: DSL simétrico (11)
-   ADSL-CAP: ensayo de fase de amplitud sin operador dsl asimétrico (12)
-   ADSL-DMT: dsl asimétrico discreto multitonal (13)
-   IDSL: línea de suscriptor digital de ISDN (14)
-   Ethernet (15)
-   xDSL: línea de suscriptor digital de tipo desconocido (16)
-   Cable (17)
-   Inalámbrico: otros (18)
-   Inalámbrico: IEEE 802.11 (19)
-   Token Ring (20)
-   FDDI (21)
