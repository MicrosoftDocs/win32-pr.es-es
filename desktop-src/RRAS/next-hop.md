# <a name="next-hop"></a>Próximo salto

Un próximo salto es el siguiente enrutador en la ruta de acceso hacia un destino. Los paquetes de un origen se reenvían a un destino de manera salto a salto.

La dirección del enrutador que es el próximo salto en una ruta se usa para identificar de forma única los vínculos en una interfaz de punto a Multipoint, donde todos los vínculos comparten el mismo índice de interfaz.
