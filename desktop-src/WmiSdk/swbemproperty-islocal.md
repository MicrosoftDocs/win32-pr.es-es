# <a name="swbempropertyislocal-property"></a>Propiedad SWbemProperty. IsLocal

La propiedad **IsLocal** del objeto [**SWbemProperty**](swbemproperty.md) es un valor booleano que se puede usar para determinar si esta propiedad es local. Un valor de **false** indica que esta propiedad se ha heredado de otra clase. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemProperty.IsLocal As Boolean
```
