# <a name="header-files"></a>Archivos de encabezado

Aplicaciones que usan:

-   Las funciones principales de OpenGL deben incluir el archivo de encabezado <\\ GL gl.h>.
-   La biblioteca de la utilidad OpenGL debe incluir el archivo de encabezado <GL \\ glu.h>.
-   La biblioteca auxiliar de la guía de programación de OpenGL debe incluir el archivo de encabezado <GL \\>.
-   Las funciones WGL deben incluir el archivo de encabezado windows.h.
-   Las Windows que admiten la implementación de OpenGL de Microsoft en Windows deben incluir el archivo de encabezado Windows.h.
