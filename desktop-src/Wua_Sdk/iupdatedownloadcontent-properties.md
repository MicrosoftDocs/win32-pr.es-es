# <a name="iupdatedownloadcontent-properties"></a>Propiedades de IUpdateDownloadContent

La interfaz [**IUpdateDownloadContent**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatecollection) define la propiedad siguiente.



| Propiedad                                                  | Descripción                                                                    |
|-----------------------------------------------------------|--------------------------------------------------------------------------------|
| [**DownloadUrl**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatedownloadcontent-get_downloadurl) | Obtiene la ubicación del contenido de descarga en el servidor que hospeda la actualización. |
