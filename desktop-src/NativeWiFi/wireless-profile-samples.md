# <a name="wireless-profile-samples"></a>Ejemplos de perfil inalámbrico

Puede Agregar un perfil inalámbrico al almacén de perfiles mediante programación llamando a [**WlanSetProfile**](/windows/desktop/api/wlanapi/nf-wlanapi-wlansetprofile). También puede Agregar un perfil al almacén de perfiles mediante la utilidad de línea de comandos **netsh** . Para obtener más información, consulte [comandos Netsh para red de área local inalámbrica (WLAN)](/previous-versions/windows/it-pro/windows-server-2008-R2-and-2008/cc755301(v=ws.10)).
