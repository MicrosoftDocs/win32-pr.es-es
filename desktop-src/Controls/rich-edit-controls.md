# <a name="rich-edit"></a>Edición enriquecida

Esta sección contiene información sobre los elementos de programación utilizados con controles Rich Edit. Un control Rich Edit permite al usuario escribir, editar, imprimir y guardar texto. El texto puede tener asignado un formato de carácter y de párrafo, y puede incluir objetos incrustados del modelo de objetos componentes (COM).

Dado que los controles Rich Edit admiten casi todos los mensajes y códigos de notificación utilizados con [los controles de edición](edit-controls.md)multilínea, las aplicaciones que ya usan controles de edición se pueden cambiar fácilmente para usar controles Rich Edit.
