# <a name="access-flags"></a>Marcas de acceso

Estos valores indican si una entrada de la lista de acceso describe los derechos que se permiten o se deniegan.



| Constante o valor                                                                                                                                                                                                                                      | Descripción                                   |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------|
| <span id="ACTRL_ACCESS_ALLOWED"></span><span id="actrl_access_allowed"></span><dl> <dt>**ACTRL \_ Acceso \_ permitido**</dt> <dt>0x00000000</dt> </dl> | Indica una entrada con acceso permitido.<br/> |
| <span id="ACTRL_ACCESS_DENIED"></span><span id="actrl_access_denied"></span><dl> <dt>**ACTRL \_ Acceso \_ denegado**</dt> <dt>0x10000000</dt> </dl>    | Indica una entrada de acceso denegado.<br/>  |
