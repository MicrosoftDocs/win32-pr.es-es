# <a name="about-pens"></a>Acerca de los lápices

Hay dos tipos de lápiz: geométrico y geométrico. Se *usa un lápiz* tortil con aplicaciones que requieren líneas de ancho fijo y líneas que se dibujan rápidamente. Una aplicación CAD, por ejemplo, usa un lápiz para generar líneas ocultas, de sección, central y de dimensión que están entre .015 y .022 pulgadas de ancho, independientemente del factor de escala. Un *lápiz geométrico* se usa con aplicaciones que requieren líneas escalables, líneas con estilos de extremo o combinación únicos y líneas que son más anchas que un solo píxel. Una aplicación de hoja de cálculo, por ejemplo, usa un lápiz geométrico para definir cada una de las barras de un gráfico de barras como una línea ancha.
