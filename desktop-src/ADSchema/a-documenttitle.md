# <a name="documenttitle-attribute"></a>atributo documentTitle

El tipo de atributo documentTitle especifica el título de un documento.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | documentTitle                               |
| Nombre para mostrar de LDAP | documentTitle                               |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.12                  |
| System-ID-GUID    | de265a9c-ff2c-47b9-91dc-6e6fe2c43062        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
