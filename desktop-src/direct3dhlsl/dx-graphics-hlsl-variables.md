# <a name="variables"></a>Variables

Las variables de HLSL son similares a las variables definidas en el lenguaje de programación C. De forma similar a C, las variables tienen algunas restricciones de nomenclatura, tienen propiedades de ámbito que dependen de dónde se declaran y pueden tener metadatos de usuario adjuntas. Al igual que C, hay varios tipos de datos estándar. A diferencia de C, también hay tipos de datos adicionales definidos por HLSL para ayudar a maximizar el rendimiento de los vectores de cuatro componentes que usan Math de matriz para operar en datos de gráficos 3D.
