# <a name="determine-the-installed-version-of-active-accessibility"></a>Determinar la versión instalada de Active Accessibility

Para averiguar qué versión de Microsoft Active Accessibility (MSAA) está instalada en el sistema, debe comprobar la versión del archivo de Oleacc.dll en la carpeta system32 del disco duro. En la tabla siguiente se asigna la versión de Microsoft Active Accessibility a la versión Oleacc.dll.

| Versión de Active Accessibility     | Versión de Oleacc.dll                                                                       |
|----------------------------------|------------------------------------------------------------------------------------------|
| 1,0                              | 4.01.0970.0                                                                              |
| 1.2                              | 4.02.2900.0                                                                              |
| 1.3                              | 4.02.3100.0                                                                              |
| 2.0                              | 5.0.0.0                                                                                  |
| 3,0 (API de automatización de Windows 3,0) | 7.0. x. y, donde *x* es el *número de compilación e y* es el número de QFE (Ingeniería de corrección rápida) |
