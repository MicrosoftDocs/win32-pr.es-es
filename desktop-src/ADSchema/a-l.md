# <a name="locality-name-attribute"></a>Locality-Name atributo

Representa el nombre de una localidad, como una ciudad o una ciudad.



| Entrada | Value |
|-------------------|------------------------------------------------------------------------------|
| CN                | Locality-Name                                                                |
| Ldap-Display-Name | l                                                                            |
| Size              | \-                                                                           |
| Actualizar privilegios  | Administrador de dominio o propietario de la cuenta.                                       |
| Frecuencia de actualización  | Cuando se crea el registro del usuario y cada vez que la localidad necesita cambiar. |
| Attribute-Id      | 2.5.4.7                                                                      |
| System-Id-Guid    | bf9679a2-0de6-11d0-a285-00aa003049e2                                         |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                  |
