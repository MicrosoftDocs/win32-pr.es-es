# <a name="interaction-context-structures"></a>Estructuras de contexto de interacción

En los temas de esta sección se proporcionan las especificaciones de referencia para las estructuras de [contexto de interacción](interaction-context-portal.md) .

| Tema | Descripción |
|---|---|
| [**\_parámetro de deslizamiento cruzado \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-cross_slide_parameter)<br/>                           | Define el umbral de deslizamiento transversal y su umbral de distancia.<br/>                                                                                                    |
| [**\_ \_ diapositiva cruzado de argumentos de interacción \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_cross_slide)<br/>  | Define el estado de la interacción entre diapositivas.<br/>                                                                                                                |
| [**\_manipulación de argumentos de interacción \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_manipulation)<br/> | Define el estado de una manipulación.<br/>                                                                                                                             |
| [**\_puntear argumentos de interacción \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_tap)<br/>                   | Define el estado de la interacción de TAP.<br/>                                                                                                                        |
| [**\_configuración de contexto de interacción \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_context_configuration)<br/>   | Define la configuración de un objeto de [contexto de interacción](interaction-context-portal.md) que habilita, deshabilita o modifica el comportamiento de una interacción.<br/> |
| [**\_resultado del contexto de interacción \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_context_output)<br/>                 | Define la salida del objeto de [contexto de interacción](interaction-context-portal.md) .<br/>                                                                          |
| [**transformación de manipulación \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-manipulation_transform)<br/>                          | Define los datos de transformación para una manipulación.<br/>                                                                                                              |
| [**velocidad de manipulación \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-manipulation_velocity)<br/>                            | Define los datos de velocidad de una manipulación.<br/>                                                                                                                     |
