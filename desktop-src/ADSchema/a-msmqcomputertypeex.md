# <a name="msmq-computer-type-ex-attribute"></a>Atributo MSMQ-Computer-Type-ex

Cadena que describe la versión del sistema operativo y la versión de MSMQ.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MSMQ-equipo-tipo-ex                       |
| Nombre para mostrar de LDAP | mSMQComputerTypeEx                          |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1417                     |
| System-ID-GUID    | 18120de8-f4c4-4341-bd95-32eb5bcf7c80        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
