# <a name="tb_isbuttonindeterminate-message"></a>\_Mensaje ISBUTTONINDETERMINATE TB

Determina si el botón especificado de una barra de herramientas es indeterminado.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>

Identificador de comando del botón.

</dd> <dt>

*lParam* 
</dt> <dd>Debe ser cero.</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Devuelve un valor distinto de cero si el botón es indeterminado o cero de lo contrario.
