# <a name="rpc_auth_identity_handle"></a>\_identificador de \_ identidad de autenticación RPC \_

El tipo de datos de **\_ identificador de \_ identidad \_ de autenticación RPC** declara un identificador que apunta a una estructura que contiene las credenciales de autenticación y autorización del cliente especificadas para llamadas a procedimientos remotos.


```C++
typedef void* RPC_AUTH_IDENTITY_HANDLE;
```
