# <a name="remote-tsp"></a>TSP remoto

El TSP remoto (Remotesp. TSP) proporciona acceso a los recursos de comunicaciones no locales del equipo actual, como una PBX. Cuando el TSP remoto está configurado para apuntar a un servidor determinado, los recursos aparecen en una aplicación local como si existieran en el equipo del usuario.

Tcmsetup es una utilidad de línea de comandos que permite la configuración de un TSP remoto. Para obtener más información acerca de esta utilidad, vea la ayuda de Windows o el kit de recursos de Windows.

El TSP remoto se instala con los sistemas operativos Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows me, Windows 98 y Windows 95.
