# <a name="sliderbackgroundhoverimage"></a>SLIDEr. backgroundHoverImage

El atributo **backgroundHoverImage** especifica o recupera la imagen de fondo del control deslizante que aparece al mantener el mouse sobre él.

``` syntax
        elementID.backgroundHoverImage
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de lectura/escritura que contiene el nombre de un archivo de imagen.

## <a name="remarks"></a>Observaciones

Este atributo es opcional. Si no se especifica, se utilizará la **BackgroundImage** .

Los formatos admitidos son BMP, JPG, PNG y GIF (sin incluir los GIF animados).
