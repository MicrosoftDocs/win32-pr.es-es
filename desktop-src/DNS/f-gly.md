# <a name="f-dns"></a>F (DNS)

A B [C](c-gly.md) D E F G [H I](h-gly.md) J K [L M](l-gly.md) N O P [P](p-gly.md) [Q R](r-gly.md) [S](s-gly.md) T U V W X Y [Z](z-gly.md)

<dl> <dt>

<span id="_dns_fault_tolerance_gly"></span><span id="_DNS_FAULT_TOLERANCE_GLY"></span>**tolerancia a errores**
</dt> <dd>

Método para garantizar que una red o un sistema informático sea resistente a errores de software o hardware.

</dd> <dt>

<span id="_dns_flat_name_space_gly"></span><span id="_DNS_FLAT_NAME_SPACE_GLY"></span>**espacio de nombre plano**
</dt> <dd>

Espacio de nombres sin particiones, en el que cada entrada debe ser única. Los espacios de nombres planos no se escalan bien, debido al requisito de que cada entrada sea única.

</dd> <dt>

<span id="_dns_forwarder_gly"></span><span id="_DNS_FORWARDER_GLY"></span>**Promotor**
</dt> <dd>

Un servidor DNS designado para controlar la comunicación con servidores DNS externos.

</dd> <dt>

<span id="_dns_fully_qualified_domain_name_fqdn__gly"></span><span id="_DNS_FULLY_QUALIFIED_DOMAIN_NAME_FQDN__GLY"></span>**Nombre de dominio completo (FQDN)**
</dt> <dd>

Nombre que identifica de forma única un host en la jerarquía DNS, de forma que un host denominado server1 en la jerarquía de productos de Microsoft puede tener un FQDN de server1.products.Microsoft.com.

</dd> </dl>
