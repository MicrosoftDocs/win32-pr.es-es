# <a name="getting-counter-information"></a>Obtener información de contador

Para recuperar información sobre un contador, como el tipo de contador, el texto de ayuda y la ruta de acceso, llame a la [**función PdhGetCounterInfo.**](/windows/desktop/api/Pdh/nf-pdh-pdhgetcounterinfoa) La función devuelve la información del contador como una [**estructura \_ PDH COUNTER \_ INFO.**](/windows/desktop/api/Pdh/ns-pdh-pdh_counter_info_a)
