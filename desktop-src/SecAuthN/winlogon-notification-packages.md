# <a name="winlogon-notification-packages"></a>Paquetes de notificación de Winlogon

Los paquetes de notificación de [*Winlogon*](../secgloss/w-gly.md) son archivos DLL que reciben y controlan eventos generados por Winlogon. Puede implementar este tipo de paquete de notificación para supervisar y responder a los eventos de Winlogon. Esto resulta útil para las aplicaciones que necesitan realizar procesamiento adicional durante el inicio o cierre de sesión, o para mantener la información de [*Estado*](../secgloss/s-gly.md) que se debe actualizar cuando se producen eventos de Winlogon.

Para obtener más información acerca de Winlogon y GINAs, consulte [Winlogon y Gina](winlogon-and-gina.md).
