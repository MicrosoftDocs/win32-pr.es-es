# <a name="input-services"></a>Servicios de entrada

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Una aplicación cliente proporciona la interfaz de usuario principal para la interacción con un carácter. Puede programar un carácter para responder a cualquier forma de entrada, desde clics en botones hasta texto con tipo. Además, Microsoft Agent proporciona eventos para que pueda programar lo que sucede cuando el usuario hace clic, hace doble clic o arrastra el carácter. El servidor pasa las coordenadas del puntero y cualquier estado de tecla modificadora para estos eventos.
