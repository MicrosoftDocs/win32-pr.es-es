# <a name="using-regions-gdi"></a>Usar regiones (GDI+)

La clase de [**región**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-region) GDI+ de Windows permite definir una forma personalizada. La forma se puede componer de líneas, polígonos y curvas.

Dos usos comunes de las regiones son la prueba de posicionamiento y el recorte. La prueba de posicionamiento determina si se hizo clic con el mouse en una determinada región de la pantalla. El recorte está restringiendo el dibujo a una región determinada.

En los temas siguientes se tratan las regiones con más detalle:

-   [Pruebas de posicionamiento con una región](-gdiplus-hit-testing-with-a-region-use.md)
-   [Recorte con una región](-gdiplus-clipping-with-a-region-use.md)
