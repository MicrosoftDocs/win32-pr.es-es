# <a name="using-native-wifi"></a>Usar Wi-Fi nativo

En esta sección se describen los procedimientos y las técnicas de programación empleadas con Wi-Fi nativa. Incluye técnicas básicas de programación WiFi nativa, como [ejemplo de API WiFi nativa](native-wifi-api-sample.md), así como técnicas avanzadas útiles para desarrolladores de NativeWifi experimentados.

Los temas siguientes se incluyen en esta sección:

-   [API de interfaz de usuario inalámbrica](wireless-user-interface-apis.md)
-   [Usar las funciones directas Wi-Fi](using-the-wi-fi-direct-api.md)
-   [Uso de redes hospedadas y conexión compartida a Internet](using-hosted-network-and-internet-connection-sharing.md)
-   [Ejemplo de API WiFi nativa](native-wifi-api-sample.md)
-   [Ejemplo de red hospedada inalámbrica](wireless-hosted-network-sample.md)
-   [Ejemplos de perfil con cable](wired-profile-samples.md)
-   [Ejemplos de perfil inalámbrico](wireless-profile-samples.md)

El tema API de la [interfaz de usuario inalámbrica](wireless-user-interface-apis.md) muestra código que se usa para iniciar interfaces de usuario de administración de redes inalámbricas.

En el tema [uso de las funciones directas Wi-Fi](using-the-wi-fi-direct-api.md) , se enumeran las funciones de Wi-Fi Direct y se explica cómo se usan las funciones.

En el tema [uso de redes hospedadas y conexión compartida a Internet](using-hosted-network-and-internet-connection-sharing.md) se describe cómo usar la red hospedada inalámbrica y conexión compartida a Internet (ICS).

El kit de desarrollo de software (SDK) de Microsoft Windows incluye dos ejemplos de código que muestran el uso de la API WiFi nativa. Además, hay perfiles de ejemplo disponibles que muestran el uso de varios esquemas de perfil Wi-Fi nativos.

Los temas siguientes contienen más información acerca de los ejemplos:

-   [Ejemplo de API WiFi nativa](native-wifi-api-sample.md)
-   [Ejemplo de red hospedada inalámbrica](wireless-hosted-network-sample.md)
-   [Ejemplos de perfil con cable](wired-profile-samples.md)
-   [Ejemplos de perfil inalámbrico](wireless-profile-samples.md)
