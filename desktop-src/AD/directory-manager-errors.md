# <a name="directory-manager-errors"></a>Errores del administrador de directorios

Este tema contiene los valores devueltos de los errores del administrador de directorios de las funciones de Active Directory Domain Services.



| Value                    | Descripción                                              |
|--------------------------|----------------------------------------------------------|
| **hrPMRecDeleted**       | El registro se ha eliminado.                                  |
| **hrRemainingVersions**  | Quedan trabajos inactivos.                            |
| **hrFLDKeyTooBig**       | La clave se truncó porque tiene más de 255 bytes. |
| **hrFLDTooManySegments** | Hay demasiados segmentos de clave.                         |
| **hrFLDNullKey**         | La clave es **null**.                                     |
