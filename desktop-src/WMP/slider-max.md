# <a name="slidermax"></a>SLIDER.max

El **atributo** max especifica o recupera el valor máximo del intervalo definido por el control deslizante.

``` syntax
        elementID.max
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un número de lectura y **escritura** (**float**) con un valor predeterminado de 100.

## <a name="remarks"></a>Comentarios

El valor especificado para **max** debe ser mayor que el de **min.**

Vea **CUSTOMSLIDER**. [Atributo positionImage](customslider-positionimage.md) para un ejemplo que ilustra cómo se usan los atributos del **elemento SLIDER.**
