# <a name="viewfocusobjectid"></a>VER. focusObjectID

El atributo **focusObjectID** especifica o recupera un valor que indica qué elemento tiene el foco de teclado. Solo se puede establecer en tiempo de ejecución.

``` syntax
        elementID.focusObjectID
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de lectura/escritura que especifica el **identificador** del elemento que tiene el foco. No tiene valor predeterminado.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
