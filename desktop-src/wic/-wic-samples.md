# <a name="wic-samples"></a>Ejemplos de WIC

Esta sección contiene los ejemplos de Windows Imaging Component (WIC).

WIC proporciona dos tipos de ejemplos:

-   Ejemplos: segmentos de código que usan varias características de WIC. El código fuente del ejemplo se entrega en el Windows SDK.
-   Ejemplos de código: ejemplos más cortos que muestran una característica específica. El código fuente del ejemplo no se proporciona con el SDK.
