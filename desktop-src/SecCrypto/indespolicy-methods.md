# <a name="indespolicy-methods"></a>Métodos Indespolicy interface

La interfaz [**indespolicy interface**](/windows/desktop/api/certpol/nn-certpol-indespolicy) admite los siguientes métodos.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                        | Descripción                                                                                       |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**Método GenerateChallenge**](/windows/desktop/api/certpol/nf-certpol-indespolicy-generatechallenge)<br/> | Realiza la decisión de la Directiva si se debe emitir una contraseña de desafío al cliente SCEP.<br/> |
| [**Initialize (método)**](/windows/desktop/api/certpol/nf-certpol-indespolicy-initialize)<br/>               | Inicializa el módulo de directivas de NDES.<br/>                                                    |
| [**NOTIFY (método)**](/windows/desktop/api/certpol/nf-certpol-indespolicy-notify)<br/>                       | Notifica al complemento del estado de la transacción de la solicitud de certificado SCEP.<br/>        |
| [**Uninitialize (método)**](/windows/desktop/api/certpol/nf-certpol-indespolicy-uninitialize)<br/>           | Desinicializa el módulo de directivas de NDES.<br/>                                                  |
| [**Método VerifyRequest**](/windows/desktop/api/certpol/nf-certpol-indespolicy-verifyrequest)<br/>         | Comprueba la solicitud de certificado NDES para su envío a la CA.<br/>                        |
