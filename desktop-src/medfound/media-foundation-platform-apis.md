# <a name="media-foundation-platform-apis"></a>Media Foundation PLATFORM API

La capa de plataforma de Media Foundation contiene primitivos y objetos auxiliares usados por las otras capas.

Esta sección contiene los siguientes temas:



| Tema                                                                           | Descripción                                                                                                                                                       |
|---------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Inicialización de la Media Foundation plataforma](initializing-media-foundation.md) | Cómo inicializar la plataforma Media Foundation datos.                                                                                                                  |
| [Media Foundation y COM](media-foundation-and-com.md)                        | Describe la interacción entre COM y Microsoft Media Foundation, y define algunos procedimientos recomendados que se deben usar al desarrollar Media Foundation componentes de complemento. |
| [Métodos de devolución de llamada asincrónicos](asynchronous-callback-methods.md)              | Cómo llamar a métodos asincrónicos y cómo implementar operaciones asincrónicas en Media Foundation.                                                                |
| [Colas de trabajo](work-queues.md)                                                  | Una cola de trabajo es una manera eficaz de realizar operaciones asincrónicas en otro subproceso.                                                                            |
| [Generadores de eventos multimedia](media-event-generators.md)                            | Cómo recibir eventos asincrónicos y cómo generar eventos en Media Foundation.                                                                                   |
| [Interfaces de servicio](service-interfaces.md)                                    | Una interfaz de servicio es una interfaz COM proporcionada por un objeto , pero expuesta a la aplicación a través de otro objeto.                                             |

| [Reloj de presentación](presentation-clock.md)                                    | El reloj de presentación genera la hora del reloj que se usa para controlar la reproducción, así como para secuencias de audio y vídeo sincrónicas.                                |
