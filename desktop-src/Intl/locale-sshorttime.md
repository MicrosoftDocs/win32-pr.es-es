# <a name="locale_sshorttime"></a>configuración regional \_ SSHORTTIME

**Windows 7 y versiones posteriores:** Cadena de formato de hora corta para la configuración regional. Los patrones suelen derivarse quitando el valor "SS" (segundos) del patrón de formato de hora larga. Por ejemplo, si el formato de hora larga es "h:mm: SS TT", lo más probable es que el formato de hora corta sea "h:mm TT". Esta constante puede especificar varios formatos en una lista delimitada por punto y coma. Sin embargo, el formato de hora corta preferida debe ser el primer valor que se muestra.
