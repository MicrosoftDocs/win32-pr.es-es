# <a name="directshow-editing-services"></a>DirectShow Servicios de edición

\[Esta API no se admite y puede modificarse o no estar disponible en el futuro.\]

DirectShow Editing Services (DES) es una interfaz de programación de aplicaciones (API) que simplifica en gran medida las tareas implicadas en la edición de vídeo. DES se basa en la arquitectura de DirectShow principal. Abstrae gran parte de la complejidad de DirectShow y proporciona un conjunto de interfaces diseñadas específicamente para manipular proyectos de edición de vídeo. Como desarrollador de aplicaciones, obtiene las ventajas de DirectShow dentro de un marco mucho más adecuado para crear aplicaciones de edición de vídeo.
