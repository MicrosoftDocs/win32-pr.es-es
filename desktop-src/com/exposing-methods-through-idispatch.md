# <a name="exposing-methods-through-idispatch"></a>Exposición de métodos a través de IDispatch

Aunque la mayoría de los controles exponen y admiten varios métodos, no son necesarios para exponer ni admitir ningún método y, por tanto, el control no requiere **IDispatch**. Si el control tiene algún método, no hay ningún requisito para los métodos que un control debe exponer.
