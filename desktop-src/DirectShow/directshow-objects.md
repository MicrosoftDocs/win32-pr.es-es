# <a name="directshow-objects"></a>Objetos de DirectShow

Este tema contiene entradas de referencia para los objetos COM principales incluidos con DirectShow, excepto los filtros. Para obtener una lista de los filtros incluidos con DirectShow, consulte [filtros de DirectShow](directshow-filters.md).



| Object                                                   | Descripción                                                      |
|----------------------------------------------------------|------------------------------------------------------------------|
| [Generador de gráficos de captura](capture-graph-builder.md)       | Compila gráficos de captura de vídeo.                                     |
| [Generador de gráficos de DVD](dvd-graph-builder.md)               | Crea gráficos de reproducción de DVD.                                      |
| [Filtrar el administrador de gráficos](filter-graph-manager.md)         | Crea y controla gráficos de filtros.                               |
| [Asignador de filtros](filter-mapper.md)                       | Busca en el registro los filtros registrados.                    |
| [Contenedor de propiedades de medios](media-property-bag.md)             | Establece o recupera fragmentos de información y DISP. en archivos AVI.             |
| [Asignador de memoria](memory-allocator.md)                 | Asigna búferes para los ejemplos de medios.                             |
| [Reloj de referencia del sistema](system-reference-clock.md)     | Implementa un reloj de referencia.                                    |
| [Enumerador de dispositivos del sistema](system-device-enumerator.md) | Enumera los filtros y los dispositivos de hardware instalados en el sistema. |
