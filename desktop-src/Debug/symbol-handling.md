# <a name="symbol-handling"></a>Control de símbolos

Las funciones de controlador de símbolos proporcionan a las aplicaciones acceso sencillo y portátil a la información de depuración simbólica de una imagen ejecutable. Estas funciones se utilizan exclusivamente para garantizar el acceso a la información simbólica. Esto es necesario porque estas funciones aíslan la aplicación del formato de símbolos.

Para obtener más información, vea los temas siguientes:

-   [Inicialización del controlador de símbolos](symbol-handler-initialization.md)
-   [Rutas de acceso de símbolos](symbol-paths.md)
-   [Carga de símbolos](symbol-loading.md)
-   [Carga de símbolos diferidos](deferred-symbol-loading.md)
-   [Nombres de símbolos representativos](decorated-symbol-names.md)
-   [Buscar símbolos](finding-symbols.md)
-   [Símbolos públicos, globales y locales](public--global--and-local-symbols.md)
-   [Limpieza del controlador de símbolos](symbol-handler-cleanup.md)
