# <a name="ivmvirtualpcsuggestedmaximummemorypervm-property"></a>IVMVirtualPC::SuggestedMaximumMemoryPerVM, propiedad

\[Windows El equipo virtual ya no está disponible para su uso a Windows 8. En su lugar, use [el proveedor WMI de Hyper-V (V2).](/windows/desktop/HyperV_v2/windows-virtualization-portal)\]

Recupera la cantidad máxima sugerida de memoria física por máquina virtual, en megabytes, para evitar condiciones de memoria baja en el host.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT get_SuggestedMaximumMemoryPerVM(
  [out, retval] long *megabytesOfMemory
);
```



## <a name="property-value"></a>Valor de propiedad

Cantidad máxima permitido sugerida, en megabytes, de memoria física por máquina virtual.
