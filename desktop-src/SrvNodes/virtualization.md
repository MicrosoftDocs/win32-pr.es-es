# <a name="virtualization"></a>Virtualización

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[Microsoft Virtual Server](/previous-versions/windows/desktop/msvs/microsoft-virtual-server-portal)
</dt> <dd>

Microsoft Virtual Server 2005 le permite crear máquinas virtuales independientes sobre el escritorio de Microsoft Windows, donde puede instalar prácticamente cualquier sistema operativo de arquitectura Intel. Cada máquina virtual emula un sistema de hardware completo, desde el procesador a la tarjeta de red, en un entorno de software independiente y aislado, lo que permite el funcionamiento simultáneo de sistemas incompatibles de otro modo.

</dd> <dt>

[Windows Virtual PC](/windows/desktop/VPC/virtual-pc-portal)
</dt> <dd>

Windows Virtual PC es la última tecnología de virtualización de Microsoft; permite ejecutar muchas aplicaciones de productividad en un entorno de Windows virtual, con un solo clic, directamente desde Windows 7. Puede crear máquinas virtuales independientes encima de su escritorio de Windows 7. Cada máquina virtual emula un sistema de hardware completo, desde el procesador a la tarjeta de red, en un entorno de software independiente y aislado, lo que permite el funcionamiento simultáneo de sistemas incompatibles de otro modo.

</dd> </dl>
