# <a name="introduction-windows-7-developer-guide"></a>Introducción (guía para desarrolladores de Windows 7)

El desarrollo de software nunca ha sido más emocionante. El mundo nunca ha estado más conectado, interactivo y móvil. La experiencia del usuario y el contenido nunca tenían mayor fidelidad. Los avances en conectividad, movilidad, interfaces naturales, gráficos y medios están habilitando nuevos escenarios y oportunidades para los desarrolladores. Al mismo tiempo, las expectativas nunca han sido más altas para el rendimiento, la flexibilidad y la interoperabilidad. Una plataforma de desarrollo de primera clase debe proporcionar una base flexible y completa para los desarrolladores y sus soluciones, lo que les permite crear aplicaciones que las configuran además de la competición a la vez que maximizan sus inversiones.

El sistema operativo Windows 7 es la plataforma esencial para los desarrolladores. Escuchando a nuestros clientes, mirando con anterioridad a los escenarios de mañana y a partir de las capacidades de Windows Vista, Microsoft ofrece a los desarrolladores una gran variedad de opciones y capacidades dentro de la plataforma para desarrolladores de Windows, a la vez que permite a los desarrolladores ofrecer soluciones creativa que son compatibles con móviles, conectadas, de alta fidelidad y proporcionan una experiencia de usuario muy intuitiva. Lo más importante, los aspectos básicos de la plataforma de desarrollo, como la seguridad, el rendimiento y la compatibilidad, son las principales prioridades de Windows 7.

En esta guía se resumen los avances clave del Desarrollador en cada una de las tres áreas siguientes:

[Base sólida](solid-foundation.md)

Excelentes experiencias de usuario comienzan con una base sólida. Windows 7 ofrece una plataforma de desarrollo sólida y herramientas innovadoras que proporcionan más opciones, mayor compatibilidad de aplicaciones, mejor rendimiento y compatibilidad con documentos sofisticada. Esto da como resultado un entorno sencillo y más confiable para las aplicaciones y un conocido y intuitivo para los usuarios.

Mantuvimos la interfaz de usuario coherente, predecible y fácil de usar, al mismo tiempo que agregamos compatibilidad con múltiples pantallas táctiles, desplazamiento y otras características de interacción intuitivas que aprovechan las capacidades más recientes de los equipos y dispositivos móviles. Con Windows 7, los desarrolladores tienen la capacidad de crear las experiencias de usuario adecuadas para sus aplicaciones. (Consulte [directrices de interacción](/windows/apps/desktop/)de la experiencia del usuario de Windows).

[Experiencias de aplicación más enriquecidas](richer-application-experiences.md)

Windows 7 permite a los desarrolladores crear aplicaciones hoy en día que se ejecutarán en los equipos del mañana. Software y dispositivos habilitados para el servicio, como teléfonos móviles, reproductores multimedia portátiles y cámaras digitales, conectividad constante a petición y aplicaciones avanzadas. Windows 7 ofrece una plataforma para ambos, lo que facilita a los desarrolladores aprovechar las ventajas de la funcionalidad y las características del hardware de próxima generación, a la vez que garantiza que los usuarios estén siempre activados y siempre conectados. (Consulte [centro de desarrollo de hardware de Windows](https://www.microsoft.com/whdc/default.mspx)).

[Lo mejor de Windows y la web](the-best-of-windows-and-the-web.md)

Con Windows 7, es fácil conectarse y permanecer conectado. Redes de Windows ofrece a los desarrolladores opciones para mejorar el almacenamiento en caché y el uso compartido de datos para mejorar el rendimiento de la red. Diagnósticos de red de Windows 7 proporciona a los desarrolladores información relevante para la supervisión de problemas de red. Windows Internet Explorer 8 es la plataforma esencial para crear experiencias Web rápidas y enriquecidas. Tanto si el objetivo es conectar dispositivos, conectarse a una red o conectarse a *Internet*, Windows 7 proporciona una plataforma mejorada. (Consulte [desarrollo web](https://msdn.microsoft.com/library/aa155073.aspx)).
