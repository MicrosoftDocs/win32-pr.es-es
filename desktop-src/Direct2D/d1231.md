# <a name="d1231-unknown-dxgi-format-support"></a>D1231: compatibilidad con formatos de DXGI desconocidos

Se realizó una llamada a IsDxgiFormatSupported con el \_ formato de DXGI \_ Unknown. Esta API devuelve false, aunque el \_ formato de DXGI \_ Unknown puede ser válido como entrada para algunas API.






 

## <a name="possible-causes"></a>Causas posibles

Se realizó una llamada a IsDxgiFormatSupported con el \_ formato de DXGI \_ Unknown.

## <a name="possible-fixes"></a>Posibles correcciones

Determinar si \_ el formato \_ de DXGI desconocido es una precisión de búfer válida para una API determinada; para ello, lea la documentación de esa API en lugar de intentar determinar esto mediante programación.
