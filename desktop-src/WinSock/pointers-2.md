# <a name="pointers-windows-sockets-2"></a>Punteros (Windows Sockets 2)

Todos los punteros usados por las aplicaciones con Windows Sockets deberían ser muy importantes, aunque esto solo es relevante para las aplicaciones de 16 bits y no tiene sentido en un 32 bits. Para facilitar esto, se proporcionan definiciones de tipos de datos como **LPHOSTENT** .
