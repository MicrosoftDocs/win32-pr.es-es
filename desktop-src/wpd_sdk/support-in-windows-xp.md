# <a name="support-in-windows-xp"></a>Compatibilidad con Windows XP

Para que la aplicación se ejecute en Windows XP, el usuario necesitará una copia del tiempo de ejecución de Windows Media Format en su equipo. Este tiempo de ejecución se puede redistribuir con la aplicación de WPD. (El tiempo de ejecución se encuentra en una carpeta de redistribuibles en el SDK de Windows Media Format).
