# <a name="meetingowner-attribute"></a>atributo meetingOwner

La persona responsable de la reunión.



| Entrada | Value |
|-------------------|----------------------------------------------------------------------------------|
| CN                | meetingOwner                                                                     |
| Ldap-Display-Name | meetingOwner                                                                     |
| Size              | \-                                                                               |
| Privilegio actualizar  | Cualquier persona puede actualizar este objeto en función de la seguridad del objeto que se va a crear. |
| Frecuencia de actualización  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.579                                                           |
| System-Id-Guid    | 11b6cc88-48c4-11d1-a9c3-0000f80367c1                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
