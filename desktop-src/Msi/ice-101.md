# <a name="ice-101"></a>ICE 101

Comprueba que ningún valor de la columna de características de la tabla de [características](feature-table.md) supera una longitud máxima de 38 caracteres.

**[Windows Installer 4,5 o una versión anterior](not-supported-in-windows-installer-4-5.md):** No compatible. Esta ICE está disponible a partir de Windows Installer 5,0.
