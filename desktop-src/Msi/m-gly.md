<dl> <dt>

<span id="_msi_managed_application_gly"></span><span id="_MSI_MANAGED_APPLICATION_GLY"></span>**aplicación administrada**
</dt> <dd>

Los privilegios del sistema, también conocidos como privilegios elevados, se usan para instalar la aplicación. Una aplicación se administra en un sistema si se trata de una instalación por equipo en Windows 2000, o si la aplicación se asigna o se publica mediante directiva de grupo en Windows 2000.

</dd> <dt>

<span id="_msi_merge_using_windows_installer_gly"></span><span id="_MSI_MERGE_USING_WINDOWS_INSTALLER_GLY"></span>**sin**
</dt> <dd>

Combina dos bases de datos de instalación para formar una sola base de datos. Para obtener más información, vea [combinar bases de datos](merging-databases.md).

</dd> <dt>

<span id="_msi_merge_module_gly"></span><span id="_MSI_MERGE_MODULE_GLY"></span>**módulo de combinación**
</dt> <dd>

Base de datos combinada con un paquete de instalación para ofrecer nuevos componentes a una aplicación. Los módulos de fusión mediante combinación no se pueden instalar solos. Para obtener más información, vea [módulos de combinación](merge-modules.md).

</dd> <dt>

<span id="_msi_microsoft_windows_installer_gly"></span><span id="_MSI_MICROSOFT_WINDOWS_INSTALLER_GLY"></span>**Microsoft Windows Installer**
</dt> <dd>

Servicio de instalador del lado cliente para administrar la instalación de aplicaciones en plataformas de 32 bits. La aplicación se debe encapsular en un [*paquete*](p-gly.md). Para obtener más información, vea [acerca de Windows Installer](about-windows-installer.md).

</dd> <dt>

<span id="_msi_.msi_file_gly"></span><span id="_MSI_.MSI_FILE_GLY"></span>**archivo. msi**
</dt> <dd>

Archivo de almacenamiento estructurado en COM que contiene las instrucciones y los datos necesarios para instalar una aplicación. Cada [*paquete*](p-gly.md) contiene al menos un archivo. msi. El archivo. msi contiene la [*base de datos del instalador*](i-gly.md), una [*secuencia de información de Resumen*](s-gly.md)y, posiblemente, una o más [*transformaciones*](t-gly.md) y [*archivos de código fuente internos*](i-gly.md). Para obtener más información, vea [Windows Installer extensiones de archivo](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_.msm_file_gly"></span><span id="_MSI_.MSM_FILE_GLY"></span>**archivo. MSM**
</dt> <dd>

[Módulo de combinación](merge-modules.md). Para obtener más información, vea [Windows Installer extensiones de archivo](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_.msp_file"></span><span id="_MSI_.MSP_FILE"></span>**archivo. MSP**
</dt> <dd>

[Paquete de revisión](patch-packages.md). Para obtener más información, vea [Windows Installer extensiones de archivo](windows-installer-file-extensions.md).

</dd> <dt>

<span id="_msi_.mst_file_gly"></span><span id="_MSI_.MST_FILE_GLY"></span>**archivo. MST**
</dt> <dd>

Paquete de transformación. Para obtener más información, vea [combinaciones y transformaciones](merges-and-transforms.md). Para obtener más información, vea [Windows Installer extensiones de archivo](windows-installer-file-extensions.md).

</dd> </dl>
