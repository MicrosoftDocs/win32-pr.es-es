# <a name="ui_pkey_decimalvalue"></a>UI \_ PKEY \_ DecimalValue

Identifica la propiedad DecimalValue de PKEY de la interfaz de usuario \_ \_ .

```
propertyDescription
   name = UI_PKEY_DecimalValue
   shellPKey = UI_PKEY_DecimalValue
   formatID = 00000201-7363-696e-8441798acf5aebb7
   propID = 201
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Observaciones

\_ \_ Una aplicación usa la interfaz de usuario PKEY DecimalValue para consultar el valor en el campo de edición del control de [número](windowsribbon-controls-spinner.md) .
