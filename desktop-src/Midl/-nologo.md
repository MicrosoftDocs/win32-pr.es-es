# <a name="nologo-switch"></a>/nologo (modificador)

El modificador **/nologo** deshabilita la presentación de la pancarta de copyright.

``` syntax
midl /nologo
```

## <a name="switch-options"></a>Opciones de conmutador

Este modificador no tiene parámetros.
