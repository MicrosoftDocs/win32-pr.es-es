# <a name="overview-of-the-printcapabilities"></a>Información general de PrintCapabilities

Este tema no es actual. Para obtener la información más reciente, vea [La especificación de esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

PrintCapabilities describe las configuraciones o estados disponibles en un dispositivo. Un dispositivo descrito por PrintCapabilities a menudo se puede colocar en una serie de configuraciones diferentes. En el caso de un dispositivo de impresión, los atributos de impresión típicos incluyen la impresión dúplex, la resolución y el tamaño del medio. Si el dispositivo admite estos atributos, forman parte de la configuración de ese dispositivo. El usuario selecciona una configuración determinada mediante la asignación de un valor específico a cada uno de los atributos disponibles; Por ejemplo, Dúplex: OneSided, Resolución: 600 ppp y MediaSize: Legal.

PrintCapabilities se basa en el marco de esquema de impresión, que define los elementos que se pueden usar en PrintCapabilities. Las palabras clave de esquema de impresión definen jerarquías de elementos de uso frecuente, o palabras clave, con el fin de promover la portabilidad de la información y la intención entre clientes individuales. Sin embargo, las palabras clave de esquema de impresión también permiten extensiones privadas para que PrintCapabilities se pueda personalizar para satisfacer necesidades específicas.
