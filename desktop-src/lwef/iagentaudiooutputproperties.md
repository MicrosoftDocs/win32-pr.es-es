# <a name="iagentaudiooutputproperties"></a>IAgentAudioOutputProperties

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

IAgentAudioOutputProperties proporciona acceso a las propiedades de salida de audio mantenidas por el servidor de Microsoft Agent. Estas funciones también están disponibles en IAgentAudioOutputProperties. Las propiedades son de solo lectura, pero el usuario puede cambiarlas en la hoja de propiedades de Microsoft Agent.

**Métodos en orden de Vtable**



| Métodos IAgentAudioOutputProperties                                               | Descripción                                     |
|-----------------------------------------------------------------------------------|-------------------------------------------------|
| [**GetEnabled**](iagentaudiooutputproperties--getenabled.md)                     | Devuelve si la salida de audio está habilitada.        |
| [**GetUsing SoundEffects**](iagentaudiooutputproperties--getusingsoundeffects.md) | Devuelve si la salida de efecto de sonido está habilitada. |
