# <a name="_alldiv-routine"></a>\_alldiv (rutina)

Divide dos **enteros LONGLONG.** Por ejemplo, para dividir dos valores int64, el compilador podría generar una llamada a \_ alldiv Routine.

## <a name="remarks"></a>Comentarios

\_alldiv Routine es una rutina auxiliar para el compilador de C. Si el compilador llama a \_ alldiv Routine depende completamente del conjunto de optimización.
