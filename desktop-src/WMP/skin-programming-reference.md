# <a name="skin-programming-reference"></a>Referencia de programación de máscara

La referencia de programación de la máscara documenta los siguientes elementos y sus atributos, métodos y eventos asociados.

Los elementos y atributos de esta sección requieren Windows Media Player 7,0 o posterior, a menos que se indique lo contrario.



| Elemento                                                  | Descripción                                                                         |
|----------------------------------------------------------|-------------------------------------------------------------------------------------|
| [Atributos de ambiente](ambient-attributes.md)             | Atributos que se aplican a todos los elementos de máscara con excepciones indicadas.                   |
| [Controladores de eventos de ambiente](ambient-event-handlers.md)     | Controladores de eventos que pueden ser implementados por la mayoría de los elementos de máscara.                       |
| [Atributos de evento de ambiente](ambient-event-attributes.md) | Atributos que detallan el estado de Windows Media Player cuando se desencadena un evento.      |
| [Automenú](automenu-element.md)                         | Proporciona una manera de mostrar el panel de acceso rápido en una máscara.                         |
| [BOTÓN](button-element.md)                             | Botón independiente.                                                                |
| [BUTTONELEMENT](buttonelement-element.md)               | Un botón dentro de un grupo de botones.                                                     |
| [BUTTONGROUP](buttongroup-element.md)                   | Grupo de elementos Button.                                                         |
| [COLUMN](column-element.md)                             | Representa una columna dentro de un control de lista de reproducción.                                      |
| [PERMITE](controls-element.md)                         | Proporciona acceso al objeto de [controles](controls-object.md) desde dentro de una máscara.   |
| [CUSTOMSLIDER](customslider-element.md)                 | Control deslizante personalizable.                                                      |
| [EDITBOX](editbox-element.md)                           | Proporciona una manera para que los usuarios escriban texto dentro de una máscara.                               |
| [EFECTOS](effects-element.md)                           | Un elemento que contiene y controla una colección de efectos.                      |
| [EQUALIZERSETTINGS](equalizersettings-element.md)       | Un elemento que permite la manipulación del ecualizador gráfico.                          |
| [MOVS](item-element.md)                                 | Representa un elemento en un cuadro de lista o control emergente.                                 |
| [IDENTIFICACIÓN](listbox-element.md)                           | Proporciona a los usuarios una manera de seleccionar elementos de una lista.                               |
| [ENCARGADO](player-element.md)                             | Proporciona acceso al objeto de [reproductor](player-object.md) desde dentro de una máscara.       |
| [AUTOMÁTICAS](playlist-element.md)                         | Un elemento para controlar la apariencia de una lista de reproducción dentro de una máscara.              |
| [EMERGENTE](popup-element.md)                               | Proporciona a los usuarios una manera de seleccionar elementos de una lista.                               |
| [PROGRESSBAR](progressbar-element.md)                   | Proporciona una manera de Mostrar información de progreso en un control horizontal o vertical. |
| [CONFIGURACIÓN](settings-element.md)                         | Proporciona acceso al objeto de [configuración](settings-object.md) desde dentro de una máscara.   |
| [HASTA](slider-element.md)                             | Control deslizante.                                                                   |
| [VISTA secundaria](subview-element.md)                           | Subsecciones de una vista que pueden moverse u ocultarse.                              |
| [NEGRITA](text-element.md)                                 | Un control que contiene texto.                                                          |
| [DICCIONARIOS](theme-element.md)                               | Elemento principal que identifica un archivo de máscara.                                           |
| [CÁMARA](video-element.md)                               | Un elemento para especificar la apariencia de una ventana de vídeo.                         |
| [Opciones de videoconfiguración](videosettings-element.md)               | Un elemento que permite el control de varias configuraciones de vídeo.                              |
| [VIEW](view-element.md)                                 | Especifica el aspecto de la interfaz de usuario (UI) para cada categoría de elementos multimedia.       |
| [Varios](miscellaneous.md)                       | Atributos especializados y otros temas variados para su uso en la creación de máscaras.    |
