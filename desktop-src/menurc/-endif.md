# <a name="endif"></a>\#endif

La **\# directiva endif** marca el final de un bloque de compilación condicional definido por una **\# directiva ifdef.** Se **\# necesita un endif** para cada **\# directiva ifdef** o **\# ifndef.** **\#**

``` syntax
#endif
```

Esta directiva no tiene parámetros.
