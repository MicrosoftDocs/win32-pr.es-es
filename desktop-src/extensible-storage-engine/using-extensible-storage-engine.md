# <a name="using-extensible-storage-engine"></a>Uso del motor Storage extensible


_**Se aplica a:** Windows | Windows Servidor_

## <a name="using-extensible-storage-engine"></a>Uso del motor Storage extensible

Esta sección contiene información general sobre cómo usar las siguientes partes de la API de ESE.

  - [Columnas](./columns.md)

  - [Indexación en la tabla](./indexing-in-the-table.md)

  - [Creación de bases de datos](./creating-databases.md)

  - [Transacciones](./transactions.md)

  - [Errores de ESE](./extensible-storage-engine-errors.md)

  - [Archivos ESE](./extensible-storage-engine-files.md)

Los archivos de origen del programa deben incluir el archivo de encabezado esent.h para tener acceso a prototipos de función y definiciones de estructura para extensible Storage Engine API. Los desarrolladores pueden usar el archivo de biblioteca esent.lib para compilar aplicaciones que usan extensible Storage Engine API. En tiempo de ejecución, las aplicaciones se vinculan al esent.dll.
