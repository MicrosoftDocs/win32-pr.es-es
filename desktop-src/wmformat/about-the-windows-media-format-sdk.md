# <a name="about-the-windows-media-format-sdk"></a>Acerca del SDK de Windows Media Format

El kit de desarrollo de software (SDK) de Windows Media® Format de Microsoft® permite a los desarrolladores crear aplicaciones que manipulen archivos multimedia digitales almacenados en archivos conforme a la estructura de archivos de Advanced Systems Format (ASF). La estructura ASF es un formato de contenedor versátil para medios de [*streaming*](wmformat-glossary.md) arbitrarios. Para obtener más información acerca de la estructura de los archivos ASF, consulte [información general del formato ASF](overview-of-the-asf-format.md).

Este SDK también proporciona compatibilidad para codificar y descodificar contenido de audio y vídeo mediante los códecs de Windows Media. Cada códec (compresor/descompresor) está optimizado para obtener la mejor calidad para un tipo de contenido específico. Para obtener más información acerca de los códecs admitidos, consulte [características de códec](codec-features.md).

Además de la compatibilidad con formatos de archivo generales y la integración de códecs, el SDK de Windows Media Format proporciona objetos para editar información descriptiva acerca de los medios digitales. La compatibilidad con la lectura de archivos y los metadatos también se incluye en archivos MP3, lo que permite crear aplicaciones más flexibles.
