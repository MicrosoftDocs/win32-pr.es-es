# <a name="frs-file-filter-attribute"></a>Atributo FRS-File-Filter

Lista de extensiones de nombre de archivo excluidas de la replicación de archivos.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | FRS-File-Filter                             |
| Ldap-Display-Name | fRSFileFilter                               |
| Size              | \-                                          |
| Privilegio actualizar  | Administrador de dominio                        |
| Frecuencia de actualización  | Cuando se configura la replicación.                  |
| Attribute-Id      | 1.2.840.113556.1.4.483                      |
| System-Id-Guid    | 1be8f170-a9ff-11d0-afe2-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
