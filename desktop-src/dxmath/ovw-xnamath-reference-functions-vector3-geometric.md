# <a name="directxmath-library-3d-vector-geometric-functions"></a>Función geométrica de vector 3D de biblioteca de DirectXMath

Muestra las funciones geométricas vectoriales 3D.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                 | Descripción                                                                                                                       |
|---------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormals)<br/>       | Calcula el ángulo en radianes entre dos vectores 3D normalizados.<br/>                                                           |
| [**XMVector3AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweennormalsest)<br/> | Calcula el ángulo en radianes entre dos vectores 3D normalizados.<br/>                                                          |
| [**XMVector3AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector3anglebetweenvectors)<br/>       | Calcula el ángulo en radianes entre dos vectores 3D.<br/>                                                                      |
| [**XMVector3ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplength)<br/>                       | Fija la longitud de un vector 3D a un intervalo determinado.<br/>                                                                     |
| [**XMVector3ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3clamplengthv)<br/>                     | Fija la longitud de un vector 3D a un intervalo determinado.<br/>                                                                     |
| [**XMVector3ComponentsFromNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3componentsfromnormal)<br/>     | Mediante el uso de un vector normal de referencia, divide un vector 3D en componentes que son paralelos y perpendiculares al normal.<br/> |
| [**XMVector3Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector3cross)<br/>                                   | Calcula el producto cruzado entre dos vectores 3D.<br/>                                                                     |
| [**XMVector3Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector3dot)<br/>                                       | Calcula el producto de punto entre vectores 3D.<br/>                                                                           |
| [**XMVector3InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inbounds)<br/>                             | Comprueba si los componentes de un vector 3D están dentro de los límites establecidos.<br/>                                                     |
| [**XMVector3Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector3length)<br/>                                 | Calcula la longitud de un vector 3D.<br/>                                                                                    |
| [**XMVector3LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lengthest)<br/>                           | Calcula la longitud de un vector 3D.<br/>                                                                                   |
| [**XMVector3LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector3lengthsq)<br/>                             | Calcula el cuadrado de la longitud de un vector 3D.<br/>                                                                      |
| [**XMVector3LinePointDistance**](/windows/win32/api/directxmath/nf-directxmath-xmvector3linepointdistance)<br/>           | Calcula la distancia mínima entre una línea y un punto.<br/>                                                              |
| [**XMVector3Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector3normalize)<br/>                           | Devuelve la versión normalizada de un vector 3D.<br/>                                                                         |
| [**XMVector3NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3normalizeest)<br/>                     | Calcula la versión normalizada de un vector 3D.<br/>                                                                       |
| [**XMVector3Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3orthogonal)<br/>                         | Calcula un vector perpendicular a un vector 3D.<br/>                                                                        |
| [**XMVector3ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reciprocallength)<br/>             | Calcula el recíproco de la longitud de un vector 3D.<br/>                                                                  |
| [**XMVector3ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reciprocallengthest)<br/>       | Calcula el recíproco de la longitud de un vector 3D.<br/>                                                                 |
| [**XMVector3Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector3reflect)<br/>                               | Refleja un vector 3D de incidente en un vector normal 3D.<br/>                                                              |
| [**XMVector3Refract**](/windows/win32/api/directxmath/nf-directxmath-xmvector3refract)<br/>                               | Refracts un vector 3D de incidente a través de un vector normal 3D.<br/>                                                              |
| [**XMVector3RefractV**](/windows/win32/api/directxmath/nf-directxmath-xmvector3refractv)<br/>                             | Refracts un vector 3D de incidente a través de un vector normal 3D.<br/>                                                              |
