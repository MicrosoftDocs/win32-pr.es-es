# <a name="skin-programming-reference"></a>Referencia de programación de máscaras

La Referencia de programación de máscaras documenta los siguientes elementos y sus atributos, métodos y eventos asociados.

Los elementos y atributos de esta sección requieren Reproductor de Windows Media 7.0 o posterior, a menos que se indique lo contrario.



| Elemento                                                  | Descripción                                                                         |
|----------------------------------------------------------|-------------------------------------------------------------------------------------|
| [Atributos ambientales](ambient-attributes.md)             | Atributos que se aplican a todos los elementos de máscara con excepciones anotadas.                   |
| [Controladores de eventos ambiente](ambient-event-handlers.md)     | Controladores de eventos que pueden implementar la mayoría de los elementos de máscara.                       |
| [Atributos de evento ambiente](ambient-event-attributes.md) | Atributos que detallan el estado de Reproductor de Windows Media cuando se desencadena un evento.      |
| [AUTOMENU](automenu-element.md)                         | Proporciona una manera de mostrar el Panel de acceso en una máscara.                         |
| [Botón](button-element.md)                             | Botón independiente.                                                                |
| [BUTTONELEMENT](buttonelement-element.md)               | Botón dentro de un grupo de botones.                                                     |
| [BUTTONGROUP](buttongroup-element.md)                   | Grupo de elementos de botón.                                                         |
| [COLUMN](column-element.md)                             | Representa una columna dentro de un control de lista de reproducción.                                      |
| [Controles](controls-element.md)                         | Proporciona acceso al [objeto Controls](controls-object.md) desde dentro de una máscara.   |
| [CUSTOMSLIDER](customslider-element.md)                 | Control deslizante personalizable.                                                      |
| [EDITBOX](editbox-element.md)                           | Proporciona una manera para que los usuarios escriban texto dentro de una máscara.                               |
| [Efectos](effects-element.md)                           | Elemento que contiene y controla una colección de efectos.                      |
| [EQUALIZERSETTINGS](equalizersettings-element.md)       | Elemento que permite la manipulación del igualador gráfico.                          |
| [Artículo](item-element.md)                                 | Representa un elemento en un cuadro de lista o un control emergente.                                 |
| [Listbox](listbox-element.md)                           | Proporciona una manera para que los usuarios seleccionen elementos de una lista.                               |
| [Jugador](player-element.md)                             | Proporciona acceso al [objeto Player](player-object.md) desde dentro de una máscara.       |
| [Reproducción](playlist-element.md)                         | Elemento para controlar la apariencia de una lista de reproducción dentro de una máscara.              |
| [Popup](popup-element.md)                               | Proporciona una manera para que los usuarios seleccionen elementos de una lista.                               |
| [Progressbar](progressbar-element.md)                   | Proporciona una manera de mostrar información de progreso en un control horizontal o vertical. |
| [CONFIGURACIÓN](settings-element.md)                         | Proporciona acceso al objeto [Configuración](settings-object.md) desde dentro de una máscara.   |
| [Deslizador](slider-element.md)                             | Control deslizante.                                                                   |
| [SUBVIEW](subview-element.md)                           | Subsecciones dentro de una vista que se pueden mover u ocultar.                              |
| [Texto](text-element.md)                                 | Control que contiene texto.                                                          |
| [Tema](theme-element.md)                               | Elemento principal que identifica un archivo de máscara.                                           |
| [Video](video-element.md)                               | Elemento para especificar la apariencia de una ventana de vídeo.                         |
| [VIDEOSETTINGS](videosettings-element.md)               | Elemento que permite el control de varias configuraciones de vídeo.                              |
| [VIEW](view-element.md)                                 | Especifica el aspecto de la interfaz de usuario (UI) para cada categoría de medios.       |
| [Varios](miscellaneous.md)                       | Atributos especializados y otros temas varios para su uso en la creación de máscaras.    |
