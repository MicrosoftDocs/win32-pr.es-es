# <a name="creating-the-art"></a>Creación del arte

El arte de esta máscara se creó con Adobe Photoshop. Otros programas de edición de gráficos, como Jasc Paint Shop Pro o Microsoft Paint, se podrían haber usado. Se eligió Photoshop como ejemplo de una aplicación de edición de gráficos que admite capas, que son útiles para crear varios archivos de mapa de bits que se alinearán entre sí.

Los pasos siguientes se usaron para crear el arte. Se pueden usar muchos otros procedimientos y herramientas de arte. esta es solo una manera de hacerlo. Lo importante que hay que tener en cuenta es que desea terminar con los archivos de mapa de bits específicos que necesita para la máscara.

-   [Crear un boceto](make-a-sketch.md)
-   [Definir los rectángulos](define-the-rectangles.md)
-   [Creación del fondo](creating-the-background.md)
-   [Creación del archivo de región](creating-the-region-file.md)
-   [Creación del archivo pushed](creating-the-pushed-file.md)
-   [Creación del archivo deshabilitado](creating-the-disabled-file.md)
-   [Creación del super archivo](creating-the-super-file.md)
-   [Creación del archivo VolumeThumb](creating-the-volumethumb-file.md)
