# <a name="viewminheight"></a>VER. minHeight

El atributo **MinHeight** especifica o recupera el alto mínimo en píxeles de la **vista** al cambiar el tamaño.

``` syntax
        elementID.minHeight
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **número** de lectura y escritura (**Long**) con un valor de cero o superior. Tiene un valor predeterminado de cero, lo que significa que no hay ninguna restricción en el alto mínimo de la **vista**.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
