# <a name="taskdefinitionregistrationinfo-property"></a>Propiedad TaskDefinition. RegistrationInfo

En el caso de scripting, obtiene o establece la información de registro que se utiliza para describir una tarea, como la descripción de la tarea, el autor de la tarea y la fecha en que se ha registrado la tarea.

## <a name="syntax"></a>Sintaxis


```VB
TaskDefinition.RegistrationInfo As RegistrationInfo
```



## <a name="property-value"></a>Valor de propiedad

La información de registro que se usa para describir una tarea, como la descripción de la tarea, el autor de la tarea y la fecha en que se ha registrado la tarea.
