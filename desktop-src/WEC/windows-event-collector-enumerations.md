# <a name="windows-event-collector-enumerations"></a>Windows Enumeraciones del recopilador de eventos

En la tabla siguiente se enumeran las enumeraciones del SDK Windows Recopilador de eventos.



| Enumeración                                                                                               | Descripción                                                                                                                             |
|-----------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| [**MODO DE \_ CONFIGURACIÓN DE SUSCRIPCIÓN DE \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_configuration_mode)                       | Especifica distintos modos de configuración que cambian la configuración predeterminada de una suscripción.                                            |
| [**FORMATO DE \_ CONTENIDO DE LA SUSCRIPCIÓN DE \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_content_format)                               | Especifica cómo se representarán los eventos en el equipo que envía los eventos antes de que se envíen al equipo del recopilador de eventos. |
| [**TIPO DE \_ CREDENCIALES DE SUSCRIPCIÓN DE \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_credentials_type)                           | Especifica el tipo de credenciales que se usarán al comunicarse con orígenes de eventos.                                                         |
| [**MODO DE \_ ENTREGA DE SUSCRIPCIONES DE \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_delivery_mode)                                 | Especifica cómo se entregan los eventos a través de una suscripción de eventos (mediante un modelo de inserción o extracción).                                          |
| [**IDENTIFICADOR DE \_ PROPIEDAD DE SUSCRIPCIÓN DE \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_property_id)                                     | Define valores para identificar las propiedades de suscripción de eventos utilizadas para la configuración de la suscripción.                                           |
| [**ESTADO ACTIVO \_ DEL ENTORNO DE EJECUCIÓN DE LA SUSCRIPCIÓN \_ \_ \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_active_status) | Especifica el estado de una suscripción o un origen de eventos con respecto a una suscripción.                                               |
| [**IDENTIFICADOR DE \_ INFORMACIÓN DE ESTADO DEL ENTORNO DE EJECUCIÓN DE SUSCRIPCIÓN DE \_ \_ \_ \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_runtime_status_info_id)             | Especifica un valor que identifica una propiedad del estado en tiempo de ejecución de un origen de eventos o una suscripción.                                |
| [**TIPO \_ DE SUSCRIPCIÓN DE \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_subscription_type)                                                    | Especifica el tipo de suscripción que se usará (una suscripción iniciada por el origen o iniciada por el recopilador).                                     |
| [**TIPO \_ DE VARIANTE DE \_ EC**](/windows/desktop/api/Evcoll/ne-evcoll-ec_variant_type)                                                              | Define los valores que especifican los tipos de datos que se usan en las Windows del recopilador de eventos.                                  |
