# <a name="locale_sshortestdayname-constants"></a>Constantes de SSHORTESTDAYNAME de configuración regional \_ \*

En este tema se definen las constantes de SSHORTESTDAYNAME de configuración regional \_ \* utilizadas por NLS.



| Value                     | Significado                                                                                                                                                                                                       |
|---------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Configuración regional \_ SSHORTESTDAYNAME1 | **Windows Vista y versiones posteriores:** Nombre nativo corto del primer día de la semana. Este nombre suele ser más corto que el [ \_ SABBREVDAYNAME1 de configuración regional](locale-sabbrev-constants.md)y es útil para los títulos de calendario.   |
| Configuración regional \_ SSHORTESTDAYNAME2 | **Windows Vista y versiones posteriores:** Nombre nativo corto del segundo día de la semana. Este nombre suele ser más corto que el [ \_ SABBREVDAYNAME2 de configuración regional](locale-sabbrev-constants.md)y es útil para los títulos de calendario.  |
| Configuración regional \_ SSHORTESTDAYNAME3 | **Windows Vista y versiones posteriores:** Nombre nativo corto del tercer día de la semana. Este nombre suele ser más corto que el [ \_ SABBREVDAYNAME3 de configuración regional](locale-sabbrev-constants.md)y es útil para los títulos de calendario.   |
| Configuración regional \_ SSHORTESTDAYNAME4 | **Windows Vista y versiones posteriores:** Nombre nativo corto del cuarto día de la semana. Este nombre suele ser más corto que el [ \_ SABBREVDAYNAME4 de configuración regional](locale-sabbrev-constants.md)y es útil para los títulos de calendario.  |
| Configuración regional \_ SSHORTESTDAYNAME5 | **Windows Vista y versiones posteriores:** Nombre nativo corto del quinto día de la semana. Este nombre suele ser más corto que el [ \_ SABBREVDAYNAME5 de configuración regional](locale-sabbrev-constants.md)y es útil para los títulos de calendario.   |
| Configuración regional \_ SSHORTESTDAYNAME6 | **Windows Vista y versiones posteriores:** Nombre nativo corto del sexto día de la semana. Este nombre suele ser más corto que el [ \_ SABBREVDAYNAME6 de configuración regional](locale-sabbrev-constants.md)y es útil para los títulos de calendario.   |
| Configuración regional \_ SSHORTESTDAYNAME7 | **Windows Vista y versiones posteriores:** Nombre nativo corto del séptimo día de la semana. Este nombre suele ser más corto que el [ \_ SABBREVDAYNAME7 de configuración regional](locale-sabbrev-constants.md)y es útil para los títulos de calendario. |
