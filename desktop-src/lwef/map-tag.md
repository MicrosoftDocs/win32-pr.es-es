# <a name="map-tag"></a>Etiqueta de mapa

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Asigna texto hablado al texto que se muestra en el globo de palabras.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintáctica**
</dt> <dd>

\\**Map**= "SpokenText" = "balloonText"\\



| Parte          | Descripción                                           |
|---------------|-------------------------------------------------------|
| *spokentext*  | Cadena que especifica el texto para la salida de voz.       |
| *balloontext* | Cadena que especifica el texto de la salida de globo de palabra. |



 

</dd> </dl>

### <a name="remarks"></a>Observaciones

Esta etiqueta permite utilizar texto hablado diferente al que se muestra en el globo de palabras.
