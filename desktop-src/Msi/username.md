# <a name="username-property"></a>Propiedad USERNAME

La propiedad **username** es un usuario que realiza la instalación.

## <a name="default-value"></a>Valor predeterminado

El valor predeterminado se puede colocar en la tabla de propiedades. Si no se establece la propiedad **username** , se establece automáticamente con los valores del registro.

## <a name="remarks"></a>Observaciones

Establezca la propiedad [**NoUserName**](nousername.md) para suprimir la configuración automática de **username**.
