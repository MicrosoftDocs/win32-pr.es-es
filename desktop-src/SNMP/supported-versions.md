# <a name="supported-versions"></a>Versiones admitidas

\[SNMP está disponible para su uso en los sistemas operativos especificados en la sección Requisitos. En versiones posteriores podría modificarse o no estar disponible. En su lugar, [Windows administración remota](/windows/desktop/WinRM/portal), que es la implementación de Microsoft de WS-Man.\]

Windows Vista y Windows Server 2008 admiten SNMPv1 y SNMPv2c. En este tema se proporciona información básica sobre cada una de estas versiones de SNMP.

El Protocolo de administración de redes simple versión 1 (SNMPv1) es la versión más antigua del protocolo. SNMPv1 se define mediante RFC 1157, RFC 1155 y RFC 1212. Para SNMPv1, la seguridad se basa en cadenas de la comunidad.

Rfc 1901, RFC 1905 y RFC 1906 definen el Protocolo de administración de redes simple versión 2c (SNMPv2c). SNMPv2c actualiza las operaciones de protocolo y los tipos de datos definidos en SNMPv2p. SNMPv2c, conocido como "SNMPv2 basado en cadenas de la comunidad", usa la seguridad basada en la comunidad SNMPv1.

SNMPv2x se admite en las plataformas de sistema operativo cliente y servidor de Windows 2000 y versiones posteriores, incluidos Windows XP y Windows Server 2003.
