# <a name="xmfloat4-operators"></a>Operadores XMFLOAT4

La estructura [**XMFLOAT4**](/windows/win32/api/directxmath/ns-directxmath-xmfloat4) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                               | Descripción                                                                                                         |
|-------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**operador =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat4-operator-assign(xmfloat4__))<br/> | Asigna los datos del componente de vector de una instancia de `XMFLOAT4` a la instancia actual de `XMFLOAT4` .<br/> |
