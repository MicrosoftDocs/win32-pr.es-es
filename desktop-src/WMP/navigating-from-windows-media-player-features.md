# <a name="navigating-from-windows-media-player-features"></a>Desplazarse de las características de Windows Media Player

Si proporciona una tienda de música en línea, puede realizar la navegación desde la vista del centro de información, la información del álbum y las páginas de HTMLView a los paneles de tareas de servicio. El proceso funciona exactamente como se describe en [desplazarse entre los paneles de tareas de servicio](navigating-between-service-task-panes.md). La diferencia es que puede desplazarse de una característica a un panel de tareas de servicio, pero no de un panel de tareas de servicio a una característica.
