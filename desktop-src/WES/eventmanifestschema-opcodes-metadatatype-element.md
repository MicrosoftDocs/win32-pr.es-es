# <a name="opcodes-metadatatype-element"></a>OpCodes (MetadataType), elemento

Define una lista de códigos de tareas que se pueden usar para agrupar eventos dentro de una tarea.

``` syntax
<xs:element name="opcodes"
    type="OpcodeListType"
 />
```

El elemento **OpCodes** se define mediante el tipo complejo [**MetadataType**](eventmanifestschema-metadatatype-complextype.md) .
