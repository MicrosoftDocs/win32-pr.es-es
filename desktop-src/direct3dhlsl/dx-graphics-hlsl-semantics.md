### <a name="direct3d-9-vpos-and-direct3d-10-sv_position"></a><span data-ttu-id="5ad6f-437">Posición de VPOS de Direct3D 9 y Direct3D 10 \_ SV</span><span class="sxs-lookup"><span data-stu-id="5ad6f-437">Direct3D 9 VPOS and Direct3D 10 SV\_Position</span></span>

<span data-ttu-id="5ad6f-438">La posición SV semántica D3D10 proporciona una funcionalidad similar a la semántica de VPOS 3 del modelo \_ de sombreador 3 de Direct3D 9.</span><span class="sxs-lookup"><span data-stu-id="5ad6f-438">The D3D10 semantic SV\_Position provides similar functionality to the Direct3D 9 shader model 3 VPOS semantic.</span></span> <span data-ttu-id="5ad6f-439">Por ejemplo, en Direct3D 9 se usa la sintaxis siguiente para un sombreador de píxeles mediante coordenadas de espacio de pantalla:</span><span class="sxs-lookup"><span data-stu-id="5ad6f-439">For instance, in Direct3D 9 the following syntax is used for a pixel shader using screen space coordinates:</span></span>

```HLSL
float4 psMainD3D9( float4 screenSpace : VPOS ) : COLOR
{
    // code here 
}
```

<span data-ttu-id="5ad6f-440">VPOS se agregó para la compatibilidad con el modelo de sombreador 3, para especificar coordenadas de espacio de pantalla, ya que la semántica POSITION estaba pensada para coordenadas de espacio de objetos.</span><span class="sxs-lookup"><span data-stu-id="5ad6f-440">VPOS was added for shader model 3 support, to specify screen space coordinates, since the POSITION semantic was intended for object-space coordinates.</span></span>

<span data-ttu-id="5ad6f-441">En Direct3D 10 y versiones posteriores, la semántica de la posición SV (cuando se usa en el contexto de un sombreador de píxeles) especifica las coordenadas del espacio de pantalla (desplazamiento por \_ 0,5).</span><span class="sxs-lookup"><span data-stu-id="5ad6f-441">In Direct3D 10 and later, the SV\_Position semantic (when used in the context of a pixel shader) specifies screen space coordinates (offset by 0.5).</span></span> <span data-ttu-id="5ad6f-442">Por lo tanto, el sombreador de Direct3D 9 sería aproximadamente equivalente (sin tener en cuenta el desplazamiento 0,5) a lo siguiente:</span><span class="sxs-lookup"><span data-stu-id="5ad6f-442">Therefore, the Direct3D 9 shader would be roughly equivalent (without accounting for the 0.5 offset) to the following:</span></span>

```HLSL
float4 psMainD3D10( float4 screenSpace : SV_Position ) : COLOR
{
    // code here
}
```

<span data-ttu-id="5ad6f-443">Al migrar de Direct3D 9 a Direct3D 10 y versiones posteriores, debe tener esto en cuenta al traducir los sombreadores.</span><span class="sxs-lookup"><span data-stu-id="5ad6f-443">When migrating from Direct3D 9 to Direct3D 10 and later, you will need to be aware of this when translating your shaders.</span></span>
