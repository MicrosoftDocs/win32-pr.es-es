# <a name="channel-com"></a>Canal (COM)

El canal tiene la responsabilidad de transmitir todos los mensajes entre el cliente y el objeto en el límite del proceso. El canal se ha diseñado para trabajar de forma transparente con tipos de canal diferentes, es compatible con la RPC estándar de OSF DCE y admite aplicaciones de un solo subproceso.
