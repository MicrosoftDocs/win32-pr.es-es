|-------------------|--------------------------------------|
| CN                | rpc-Ns-Entry-Flags                   |
| Ldap-Display-Name | rpcNsEntryFlags                      |
| Size              | 4 bytes                              |
| Actualizar privilegios  | \-                                   |
| Frecuencia de actualización  | \-                                   |
