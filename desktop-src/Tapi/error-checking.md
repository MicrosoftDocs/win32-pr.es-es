# <a name="error-checking"></a>Comprobación de errores

En el nivel de TAPI, una aplicación puede pasar varios parámetros diferentes, muchos de los cuales pueden no ser válidos. TAPI comprueba los parámetros y devuelve errores a la aplicación sin llamar al proveedor de servicios. Cada descripción de función en el nivel TSPI describe los errores de parámetro ya probados. No es necesario que el proveedor de servicios Repita estas pruebas, aunque debe realizar cualquier prueba de validez adicional adecuada para la función. En la tabla siguiente se enumeran los títulos y las descripciones de las pruebas de validez de parámetros comunes que aparecen en muchas funciones.



| Prueba de validez       | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|---------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Validez del puntero    | TAPI ya ha probado los punteros al almacenamiento de datos para asegurarse de que apuntan a la memoria legible o grabable del tamaño adecuado para la operación. Además, para las estructuras de datos de tamaño de variable que comienzan con un miembro de **dwTotalSize** , se ha verificado la estructura de datos para asegurarse de que el tamaño total indicado está disponible.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| Validez de tamaño fijo | En el caso de las estructuras de datos de tamaño variable, se ha comprobado que la estructura de datos tiene espacio para la parte de tamaño fijo de la estructura de datos y que **dwTotalSize** es suficiente para la parte fija.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Desplazamiento/tamaño con ceros  | Para las estructuras de datos de tamaño variable, el "... **Desplazamiento**"y"... **Tamaño**: los campos que corresponden a las partes que el proveedor de servicios establece se han preestablecido con cero valores antes de llamar al proveedor de servicios.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| Controlar la validez     | TAPI garantiza que los identificadores de línea, teléfono y llamada (de los tipos definidos [HDRVLINE](hdrvline.md), [HDRVPHONE](hdrvphone.md)y HDRVCALL) son válidos. Es decir, son valores que se han devuelto sin errores como identificadores en [**TSPI \_ lineOpen**](/windows/win32/api/tspi/nf-tspi-tspi_lineopen), [**TSPI \_ phoneOpen**](/windows/win32/api/tspi/nf-tspi-tspi_phoneopen), o uno de los siguientes que inicia la duración de un identificador de llamada: [**TSPI \_ lineMakeCall**](/windows/win32/api/tspi/nf-tspi-tspi_linemakecall)<br/> [**TSPI \_ lineCompleteTransfer**](/windows/win32/api/tspi/nf-tspi-tspi_linecompletetransfer)<br/> [**TSPI \_ lineForward**](/windows/win32/api/tspi/nf-tspi-tspi_lineforward)<br/> [**TSPI \_ linePickup**](/windows/win32/api/tspi/nf-tspi-tspi_linepickup)<br/> [**TSPI \_ linePrepareAddToConference**](/windows/win32/api/tspi/nf-tspi-tspi_lineprepareaddtoconference)<br/> [**TSPI \_ lineSetupConference**](/windows/win32/api/tspi/nf-tspi-tspi_linesetupconference)<br/> [**TSPI \_ lineSetupTransfer**](/windows/win32/api/tspi/nf-tspi-tspi_linesetuptransfer)<br/> [**TSPI \_ lineUnpark**](/windows/win32/api/tspi/nf-tspi-tspi_lineunpark)<br/> [**Línea \_ de Mensajes NEWCALL**](line-newcall.md)<br/> |
