# <a name="directxmath-library-types"></a>Tipos de biblioteca de DirectXMath

La biblioteca de DirectXMath proporciona una serie de estructuras y tipos definidos para encapsular los datos para admitir la facilidad de uso, la optimización y la portabilidad.

La lista siguiente incluye las estructuras que actualmente forman parte de la biblioteca de DirectXMath y están disponibles a través del encabezado DirectXMath. h.

## <a name="in-this-section"></a>En esta sección



| Tema                                                             | Descripción                                                                                                                                                                       |
|-------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MEDIO, tipo de datos**](half-data-type.md)<br/>               | Alias para **UInt16 \_ t** empaquetado con un número de punto flotante de 16 bits que consta de un bit de signo, un exponente sesgado de 5 bits y una mantisa de 10 bits.<br/>                         |
| [**XMVECTOR, tipo de datos**](xmvector-data-type.md)<br/>       | Tipo portable que se usa para representar un vector de componentes de punto flotante de 4 32 bits o enteros, cada uno alineado de manera óptima y asignado a un registro de vector de hardware.<br/>       |
| [**XMVECTORF32, tipo de datos**](xmvectorf32-data-type.md)<br/> | Un tipo opaco y portátil para admitir el uso de la sintaxis de inicializador de C/C++ para cargar valores de punto flotante en una instancia de tipo [**XMVECTOR**](xmvector-data-type.md) .<br/> |
| [**XMVECTORI32, tipo de datos**](xmvectori32-data-type.md)<br/> | Un tipo opaco y portátil para admitir el uso de la sintaxis de inicializador de C/C++ para cargar valores enteros en una instancia de tipo [**XMVECTOR**](xmvector-data-type.md) .<br/>        |
| [**XMVECTORU32, tipo de datos**](xmvectoru32-data-type.md)<br/> | Un tipo opaco y portátil para admitir el uso de la sintaxis de inicializador de C/C++ para cargar \_ valores UInt32 t en una instancia de tipo XMVECTOR.<br/>                                    |
| [**XMVECTORU8, tipo de datos**](xmvectoru8-data-type.md)<br/>   | Un tipo opaco y portátil para admitir el uso de la sintaxis de inicializador de C/C++ para cargar \_ valores de Uint8 t en una instancia de tipo XMVECTOR.<br/>                                     |
