# <a name="updating-directory-structure-for-an-upgrade"></a>Actualización de la estructura de directorios para una actualización

Use el editor de bases de datos orca que se proporciona con el SDK u otro editor para abrir la [tabla de directorio](directory-table.md) en MNP2001.msi. La estructura de directorios de la actualización de ejemplo es la misma que la estructura de directorios del producto original y no se requieren cambios en la tabla de directorios original descrita en [especificar la estructura de directorios](specifying-directory-structure.md).
