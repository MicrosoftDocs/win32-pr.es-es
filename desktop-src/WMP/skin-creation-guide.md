# <a name="skin-creation-guide"></a>Guía de creación de máscaras

Esta guía es una serie de explicaciones detalladas sobre cómo crear diferentes tipos de máscaras. Para obtener más información general sobre las máscaras, vea [Acerca de las máscaras.](about-skins.md) Para obtener detalles específicos sobre cada atributo, método y evento usados en máscaras, vea la [Referencia de programación de máscaras](skin-programming-reference.md). A medida que se involucra más en la programación de la máscara, es posible que desee leer la parte de este SDK que abarca Reproductor de Windows Media [object model](windows-media-player-object-model.md).

En esta guía, se proporcionan instrucciones para crear el arte para Adobe Photoshop 5.5, un popular programa de manipulación de arte. Las instrucciones específicas pueden ser diferentes si tiene un programa de arte similar, como Jasc Paint Shop Pro o Sonic Foundry Desa, pero los conceptos serán los mismos. Se eligió Photoshop por dos motivos: es un programa de arte popular para los intérpretes comerciales y funciona con capas. Como verá en los tutoriales, las capas son muy útiles para la creación de máscaras.

En esta guía se tratarán las tareas siguientes.



| Tarea                                                     | Descripción                                                                                     |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------------|
| [Creación de la primera máscara](building-your-first-skin.md) | Un recorrido por una máscara simple que se inicia y se detiene Reproductor de Windows Media.                     |
| [Agregar una lista de reproducción](adding-a-playlist.md)               | Cómo usar una lista de reproducción simple.                                                                   |
| [Elegir archivos](choosing-files.md)                     | Cómo seleccionar archivos con un cuadro de diálogo de archivo abierto.                                                 |
| [Adición de vídeo](adding-video.md)                         | Cómo colocar una ventana de vídeo en la máscara.                                                       |
| [Agregar visualizaciones](adding-visualizations.md)       | Cómo agregar visualizaciones.                                                                      |
| [Agregar un control deslizante](adding-a-slider.md)                   | Cómo usar un control deslizante para realizar un seguimiento del progreso del contenido multimedia... y dejar que el usuario lo cambie. |
| [Crear controles deslizantes personalizados](creating-custom-sliders.md)   | Cómo controlar el volumen con un control deslizante personalizado.                                                 |
| [Otras muestras de máscara](other-skin-samples.md)             | Consulte la sección de ejemplos del SDK.                                                             |
