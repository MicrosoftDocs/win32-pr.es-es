# <a name="the-voice-commands-window"></a>Ventana comandos de voz

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Si hay instalado un motor de voz compatible, Microsoft Agent proporciona una ventana especial denominada **ventana comandos de voz** que muestra los comandos habilitados para voz para el reconocimiento de voz. La **ventana comandos de voz** sirve como un símbolo visual de lo que se puede decir como entrada (los comandos no se pueden seleccionar con el mouse).

![comandos de voz (cuadro de diálogo)](images/f2voice.gif)

La ventana aparece cuando un usuario selecciona el comando **abrir la ventana comandos de voz** , ya sea hablando del comando o haciendo clic con el botón derecho en el carácter y eligiendo el comando en el menú emergente del carácter. Sin embargo, si el usuario deshabilita la entrada de voz, no se puede tener acceso a la **ventana comandos de voz** .

La **ventana comandos de voz** muestra los comandos habilitados para voz como un árbol. Si la aplicación de hospedaje actual proporciona comandos de voz, aparecen expandidos y en la parte superior de la ventana. Las entradas también aparecen para otras aplicaciones que utilizan el carácter. La ventana también incluye los comandos de voz globales que proporciona Microsoft Agent. Si la aplicación de hospedaje actual no tiene comandos de voz, los comandos de voz globales aparecen expandidos y en la parte superior de la ventana.

El usuario puede cambiar el tamaño de la **ventana comandos de voz** y moverla. El agente de Microsoft recuerda la última ubicación de la ventana y la vuelve a mostrar en esa ubicación si el usuario se cierra y vuelve a abrir la ventana. Si las entradas de la ventana superan el tamaño de presentación actual de la ventana, aparecen barras de desplazamiento.
