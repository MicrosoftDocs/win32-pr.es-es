# <a name="startmenufolder-property"></a>Propiedad StartMenuFolder

El instalador establece la propiedad **StartMenuFolder** en la ruta de acceso completa de la carpeta del menú Inicio. De forma predeterminada, esta propiedad se establece en la carpeta del usuario actual. Si existe un perfil "todos los usuarios" y se establece la propiedad [**AllUsers**](allusers.md) , esta propiedad se establece en la carpeta del perfil "todos los usuarios".
