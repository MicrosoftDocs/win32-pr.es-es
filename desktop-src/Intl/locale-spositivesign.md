# <a name="locale_spositivesign"></a>configuración regional \_ SPOSITIVESIGN

Valor de cadena localizada para el signo positivo de la configuración regional. Por ejemplo, la **configuración regional \_ SPOSITIVESIGN** se considera "+" para la configuración regional en inglés (Estados Unidos). Tenga en cuenta que la presencia de datos para esta cadena no significa que se utilice un signo positivo para dar formato a un número. Si los datos están en blanco o vacíos, se supone que se usa un "+". El número máximo de caracteres permitido para esta cadena es cinco, incluido un carácter nulo de terminación.
