# <a name="wia-property-constants"></a>Constantes de propiedad de WIA

Las constantes de propiedad de adquisición de imágenes de Windows (WIA) incluyen los siguientes grupos de constantes compatibles con Windows Millennium Edition (Windows me), Windows XP y versiones posteriores.

-   [**Constantes de propiedades de información del dispositivo**](-wia-wiadeviceinfoprop.md)
-   [**Constantes de propiedad de dispositivo comunes**](-wia-wiaitempropcommondevice.md)
-   [**Constantes de propiedades de dispositivo de escáner**](-wia-wiaitempropscannerdevice.md)
-   [**Constantes comunes de propiedades de elementos WIA**](-wia-wiaitempropcommonitem.md)
-   [**Constantes de propiedad del elemento WIA del escáner**](-wia-wiaitempropscanneritem.md)

Windows me, Windows XP y Windows Server 2003 admiten los siguientes grupos de constantes. pero no son compatibles con Windows Vista y versiones posteriores.

-   [**Constantes de propiedades de dispositivo de cámara**](-wia-wiaitempropcameradevice.md)
-   [**Constantes de propiedad del elemento de WIA de cámara**](-wia-wiaitempropcameraitem.md)

Windows me y Windows XP admiten el siguiente grupo de constantes. pero no es compatible con Windows Server 2003 y versiones posteriores.

-   [**Constantes de propiedad de dispositivo WIA de vídeo**](-wia-wiaitempropvideodevice.md)
