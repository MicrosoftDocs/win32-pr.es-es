# <a name="peer-distribution"></a>Distribución del mismo nivel

La API de distribución del mismo nivel, que admite la característica de la caché de rama en Windows 7 y Windows Server 2008 R2, ofrece un conjunto de API de plataforma que puede aumentar la capacidad de respuesta de la red de las aplicaciones centralizadas cuando se accede desde las oficinas remotas, lo que proporciona a los usuarios de esas oficinas la experiencia de trabajar en su red de área local. Las capacidades que ofrece este conjunto de API también pueden ayudar a reducir el uso general de la red de área extensa (WAN).

La implementación de estas características no requiere ninguna infraestructura preexistente. La realización de mejoras en el rendimiento de las redes remotas se realiza simplemente mediante la implementación de Windows 7 en los equipos cliente, la implementación de Windows Server 2008 R2 en los equipos servidor y la habilitación de la característica de caché de rama.

Las API de distribución del mismo nivel funcionan sin problemas junto con las tecnologías de seguridad de red, como SSL, firma SMB y IPsec de un extremo a otro. Puede usar esta API para reducir el uso del ancho de banda de red y mejorar el rendimiento de la aplicación incluso si el contenido está cifrado.
