# <a name="task-scheduler-reference"></a>Referencia de Programador de tareas

En esta sección se describen las API, los objetos de scripting y el esquema XML que proporciona el Programador de tareas.

Los temas de la API incluyen una descripción de la API, la sintaxis de declaración de la API, comentarios que describen las condiciones especiales de la API y un bloque de requisitos que describe qué plataforma, archivos de encabezado y bibliotecas son necesarios.

Los temas de objetos de scripting incluyen una descripción del objeto, la sintaxis de declaración del objeto, comentarios que describen condiciones especiales para el objeto, y un bloque de requisitos que describe qué bibliotecas de plataforma y tipo son necesarias.

Los temas del esquema XML incluyen una descripción de la información de elemento, primario, secundario y de atributo, así como comentarios que describen condiciones especiales.
