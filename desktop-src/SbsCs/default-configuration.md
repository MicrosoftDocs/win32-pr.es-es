# <a name="default-configuration"></a>Configuración predeterminada

Las versiones de ensamblado especificadas en [manifiestos de ensamblado](assembly-manifests.md) y [manifiestos de aplicación](application-manifests.md) conforman la configuración predeterminada que debe usar la aplicación cuando se ejecuta la aplicación. Las aplicaciones y los administradores pueden invalidar la configuración predeterminada del ensamblado instalando una configuración de [publicador](publisher-configuration.md) o una [configuración por aplicación](per-application-configuration.md) en Windows Server 2003.
