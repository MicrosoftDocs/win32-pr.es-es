# <a name="current-parent-ca-attribute"></a>Atributo Current-Parent-CA

Referencia a las entidades de certificación que emitieron los certificados actuales para una entidad de certificación.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | Current-Parent-CA                       |
| Ldap-Display-Name | currentParentCA                         |
| Size              | \-                                      |
| Actualizar privilegios  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.696                  |
| System-Id-Guid    | 963d273f-48be-11d1-a9c3-0000f80367c1    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
