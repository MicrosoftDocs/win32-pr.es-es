# <a name="memory-classes"></a>Clases de memoria

Las clases de memoria modelan la memoria de una máquina virtual.

A continuación se enumeran las clases WMI de virtualización relacionadas con la memoria.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                      | Descripción                                                                                                         |
|--------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ ElementAllocatedFromNumaNode**](msvm-elementallocatedfromnumanode.md)<br/> | Asocia una instancia de un recurso asignado al nodo NUMA físico desde el que se asignó.<br/> |
| [**Memoria de MSVM \_**](msvm-memory.md)<br/>                                             | Representa la memoria asignada actualmente a una máquina virtual.<br/>                                          |
| [**MSVM \_ MemorySettingData**](msvm-memorysettingdata.md)<br/>                       | Representa el estado configurado de la memoria de una máquina virtual.<br/>                                     |
| [**MSVM \_ NumaNode**](msvm-numanode.md)<br/>                                         | Representa un nodo de acceso a memoria no uniforme (NUMA) de un sistema virtual.<br/>                                  |
