# <a name="restart-manager-callback-functions"></a>Funciones de devolución de llamada del administrador de reinicio

La API del administrador de reinicio utiliza las funciones de devolución de llamada de la tabla siguiente.



| Función de devolución de llamada                                               | Descripción                                                                                                                                                            |
|-----------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_devolución de \_ llamada de estado de escritura de RM \_**](/windows/win32/api/restartmanager/nc-restartmanager-rm_write_status_callback) | La aplicación que inició la sesión del administrador de reinicio puede pasar un puntero a esta función a las funciones del administrador de reinicio para recibir un porcentaje de integridad. |
