# <a name="locale_sthousand"></a>LOCALE \_ STHOUSAND

Caracteres que se usan para separar grupos de dígitos a la izquierda del decimal. El número máximo de caracteres permitido para esta cadena es cuatro, incluido un carácter nulo de terminación. Normalmente, estos grupos representan miles. Sin embargo, dependiendo del valor especificado para [LOCALE \_ SGROUPING,](locale-sgrouping.md)pueden representar otra cosa.
