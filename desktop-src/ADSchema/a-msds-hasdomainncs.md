# <a name="ms-ds-has-domain-ncs-attribute"></a>atributo MS-DS-tiene-Domain-NC

Información de replicación del servicio de directorio que detalla los contextos de nomenclatura de dominio presentes en un servidor determinado.



| Entrada | Value |
|-------------------|-----------------------------------------------------|
| CN                | MS-DS-tiene-Domain-NC                                |
| Nombre para mostrar de LDAP | msDS-HasDomainNCs                                   |
| Tamaño              | \-                                                  |
| Actualizar privilegio  | El sistema establece este valor.                     |
| Frecuencia de actualización  | Está establecido por DCPromo y nunca se modificó después. |
| Attribute-Id      | 1.2.840.113556.1.4.1820                             |
| System-ID-GUID    | 6f17e347-a842-4498-b8b3-15e007da4fed                |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md)             |
