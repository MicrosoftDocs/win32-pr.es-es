# <a name="pidkey-property"></a>Propiedad PIDKEY

La propiedad **PIDKEY** contiene la parte del ID. de producto especificada por el usuario. La [Acción ValidateProductID](validateproductid-action.md) valida el ID. de producto especificado comparando la propiedad **PIDKEY** con la propiedad [**PIDTemplate**](pidtemplate.md) .

La propiedad **PIDKEY** se puede establecer en la línea de comandos o a través de la interfaz de usuario. Esta propiedad solo contiene la parte del ID. de producto especificada por el usuario.
