# <a name="mfpkey_videowindow-property"></a>MFPKEY \_ (propiedad de VideoWindow)

Especifica la cantidad de contenido, en milisegundos, que puede caber en el búfer del modelo.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

g \_ wszWMVCVideoWIndow

## <a name="data-type"></a>Tipo de datos

VT \_ I4

## <a name="default-value"></a>Valor predeterminado

3000

## <a name="remarks"></a>Observaciones

La ventana de búfer es uno de los parámetros del modelo de "depósito de fugas" que se usa en el control de velocidad de códec.
