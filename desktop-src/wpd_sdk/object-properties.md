# <a name="object-properties"></a>Propiedades de objeto

Los dispositivos portátiles de Windows admiten las siguientes propiedades de objeto.



| Propiedad                                                                                                                                    | VarType         | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|---------------------------------------------------------------------------------------------------------------------------------------------|-----------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ referencias inversas de objetos de WPD \_**                                                                                                           | **VT \_ desconocido** | Un [**IPortableDevicePropVariantCollection**](iportabledevicepropvariantcollection.md) de tipo VT \_ LPWStr que indica una lista de objectId. Cuando un objeto de contenedor hace referencia a un objeto, por ejemplo, un objeto de audio al que hace referencia una lista de reproducción, el objeto al que se hace referencia utiliza esta propiedad para volver a hacer referencia a su objeto contenedor. El objeto de audio puede volver a hacer referencia al objeto de lista de reproducción en este ejemplo.                                                                                                                   |
| **el \_ objeto WPD \_ puede \_ eliminar**                                                                                                                | **VT \_ bool**    | Valor booleano que especifica si se puede eliminar el objeto especificado.                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **\_identificador de \_ \_ objeto funcional \_ del contenedor de objetos de \_ WPD**                                                                                          | **VT \_ LPWStr**  | IDENTIFICADOR de objeto del objeto funcional más cercano que contiene este objeto. Por ejemplo, un archivo dentro de un objeto funcional de almacenamiento tendrá esta propiedad establecida en el identificador del objeto funcional de almacenamiento.                                                                                                                                                                                                                                                                                                                        |
| <span id="wpd_object_content_type"></span><span id="WPD_OBJECT_CONTENT_TYPE"></span>**\_tipo de \_ contenido del objeto WPD \_**                          | **VT \_ CLSID**   | Un **GUID** que identifica el tipo genérico de este objeto, por ejemplo, un documento o correo electrónico. Puede ser un [tipo de objeto definido por dispositivos portátiles de Windows](requirements-for-objects.md)o un tipo de contenido de controlador personalizado. El objeto de dispositivo es el único objeto que no notifica esta propiedad.                                                                                                                                                                                                                        |
| <span id="wpd_object_date_authored"></span><span id="WPD_OBJECT_DATE_AUTHORED"></span>**\_fecha del objeto WPD \_ \_ creado**                       | **fecha de VT \_**    | Valor que especifica la fecha y la hora en que se creó el contenido. Es posible que no coincida con la fecha de creación del archivo. Por ejemplo, un archivo de música tiene una fecha de creación de la grabación de la música, pero una fecha de creación en la que se creó realmente el archivo WMA en el dispositivo.                                                                                                                                                                                                                                |
| <span id="wpd_object_date_created"></span><span id="WPD_OBJECT_DATE_CREATED"></span>**\_fecha de objeto WPD \_ \_ creada**                          | **fecha de VT \_**    | Valor que especifica la fecha y hora en que se creó el objeto en el dispositivo.                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="wpd_object_date_modified"></span><span id="WPD_OBJECT_DATE_MODIFIED"></span>**\_fecha del objeto WPD \_ \_ modificada**                       | **fecha de VT \_**    | Valor que especifica la fecha y hora en que se modificó el objeto en el dispositivo.                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="wpd_object_format"></span><span id="WPD_OBJECT_FORMAT"></span>**\_formato de objeto WPD \_**                                             | **VT \_ CLSID**   | **GUID** que identifica el formato de los datos del objeto. Puede ser un [formato definido por dispositivos portátiles de Windows](object-format-guids.md) o un formato de controlador personalizado.                                                                                                                                                                                                                                                                                                                                                            |
| **el \_ objeto WPD \_ genera \_ miniaturas \_ desde el \_ recurso**                                                                                        | **VT \_ bool**    | Valor booleano que especifica si la imagen en miniatura de este objeto se debe crear a partir de los datos de recursos predeterminados. Esto proporciona una manera de que los objetos sin un recurso de miniatura ofrezcan una experiencia de exploración más agradable. El uso de esta marca puede afectar a la primera respuesta de pantalla, ya que la aplicación debe recuperar y calcular una imagen en miniatura desde el dispositivo. es más eficaz proporcionar un recurso de imagen en miniatura independiente, si es posible.<br/>                                                  |
| **nombre para mostrar de la ubicación de la \_ sugerencia de objeto WPD \_ \_ \_ \_**                                                                                              | **VT \_ LPWStr**  | Si el objeto especificado aparece como una ubicación de sugerencia, esta propiedad indica el nombre específico de la sugerencia que se va a mostrar en lugar del nombre de objeto. Los controladores pueden especificar sugerencias de ubicación para varios tipos de contenido. Pueden considerarse como accesos directos a objetos de carpeta de nivel superior que contienen objetos de los tipos especificados. Los clientes que utilizan estas sugerencias de ubicación pueden mostrar un nombre para estos accesos directos que difieren del nombre de objeto de la carpeta. Si esta propiedad no existe, normalmente se utiliza el **\_ \_ nombre del objeto WPD** . |
| <span id="wpd_object_id"></span><span id="WPD_OBJECT_ID"></span>**\_identificador de objeto de WPD \_**                                                         | **VT \_ LPWStr**  | IDENTIFICADOR de cadena que identifica de forma única el objeto en el dispositivo. Este identificador no debe almacenarse entre sesiones. Si esta propiedad es única y persistente, el controlador puede establecer el **\_ \_ \_ \_ identificador único persistente del objeto WPD** y el ID. de **\_ objeto \_ WPD** en el mismo valor.<br/>                                                                                                                                                                                                                                                 |
| <span id="wpd_object_is_drm_protected"></span><span id="WPD_OBJECT_IS_DRM_PROTECTED"></span>**el \_ objeto \_ WPD \_ está \_ protegido con DRM**             | **VT \_ bool**    | Valor booleano que especifica si los datos multimedia están protegidos con DRM. Si no está presente, se supone que es false.                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="wpd_object_ishidden"></span><span id="WPD_OBJECT_ISHIDDEN"></span>**\_objeto WPD \_ ISHIDDEN**                                       | **VT \_ bool**    | Valor booleano que especifica si se debe ocultar el objeto. Si no está presente, se supone que el objeto no está oculto.                                                                                                                                                                                                                                                                                                                                                                                                     |
| **el \_ objeto WPD \_ puede \_ eliminar**                                                                                                                | **VT \_ bool**    | Valor booleano que especifica si se puede eliminar el objeto.                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| <span id="wpd_object_issystem"></span><span id="WPD_OBJECT_ISSYSTEM"></span>**\_objeto WPD \_ ISSYSTEM**                                       | **VT \_ bool**    | Valor booleano que especifica si el objeto representa datos del sistema (por ejemplo, un archivo del sistema). Si no está presente, se supone que el objeto no es un objeto del sistema.                                                                                                                                                                                                                                                                                                                                                              |
| <span id="wpd_object_keywords"></span><span id="WPD_OBJECT_KEYWORDS"></span>**\_palabras clave del objeto WPD \_**                                       | **VT \_ LPWStr**  | Cadena que contiene una lista de palabras clave delimitadas por espacios asociadas a este objeto.                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| **\_ \_ configuración regional de idioma del objeto WPD \_**                                                                                                           | **VT \_ LPWStr**  | Una cadena que dentifies el lenguaje usado por un objeto determinado. Si hay varios idiomas incluidos en este objeto, debe identificar el idioma principal. Esta propiedad puede contener un código de idioma, tal como se define en ISO-639, por ejemplo: "en". También puede contener un código de país de idioma, que consta de un código de idioma de dos o tres caracteres, tal y como se define en el estándar ISO-639, seguido de un guion y seguido de un código de país tal como se define en ISO-3166, como: "en-US".                                |
| <span id="wpd_object_name"></span><span id="WPD_OBJECT_NAME"></span>**\_nombre del objeto WPD \_**                                                   | **VT \_ LPWStr**  | Nombre para mostrar del objeto.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="wpd_object_non_consumable"></span><span id="WPD_OBJECT_NON_CONSUMABLE"></span>**\_objeto WPD \_ no \_ consumible**                    | **VT \_ bool**    | Valor booleano que especifica si este objeto está pensado para ser entendido o simplemente almacenado por el dispositivo. Si esta propiedad no está presente, se supone que todos los datos están destinados al consumo.                                                                                                                                                                                                                                                                                                                            |
| <span id="wpd_object_original_file_name"></span><span id="WPD_OBJECT_ORIGINAL_FILE_NAME"></span>**\_nombre de \_ \_ archivo original del objeto \_ WPD**       | **VT \_ LPWStr**  | Un nombre de cadena para el archivo.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="wpd_object_parent_id"></span><span id="WPD_OBJECT_PARENT_ID"></span>**\_ \_ ID. primario del objeto WPD \_**                                   | **VT \_ LPWStr**  | IDENTIFICADOR de objeto del objeto primario. El único objeto que puede devolver una cadena vacía para este valor es el objeto de dispositivo raíz. Para modificar esta propiedad, llame a [**IPortableDevice:: SendCommand**](/windows/desktop/api/PortableDeviceApi/nf-portabledeviceapi-iportabledevice-sendcommand) (movimiento de almacenamiento de comandos de **WPD \_ \_ \_**).<br/>                                                                                                                                                                                                                                                    |
| <span id="wpd_object_persistent_unique_id"></span><span id="WPD_OBJECT_PERSISTENT_UNIQUE_ID"></span>**\_ \_ \_ identificador único persistente del objeto WPD \_** | **VT \_ LPWStr**  | IDENTIFICADOR de cadena que identifica de forma única el objeto en el dispositivo, similar **al \_ \_ identificador de objeto WPD**, pero debe almacenarse entre sesiones. Si el identificador de objeto de la **\_ \_ identificación** del objeto WPD es único y persistente, el controlador puede establecer el **\_ \_ \_ \_ identificador único persistente del objeto WPD** y el ID. de **\_ objeto \_ WPD** en el mismo valor.<br/>                                                                                                                                                                          |
| <span id="wpd_object_references"></span><span id="WPD_OBJECT_REFERENCES"></span>**\_referencias a objetos de WPD \_**                                 | **VT \_ desconocido** | Un [**IPortableDevicePropVariantCollection**](iportabledevicepropvariantcollection.md) que contiene una colección de \_ identificadores de objeto VT LPWStr que identifican los objetos a los que se hace referencia. Esto solo es necesario si el objeto es un objeto de referencia, como una carpeta o una lista de reproducción.                                                                                                                                                                                                                                                            |
| <span id="wpd_object_size"></span><span id="WPD_OBJECT_SIZE"></span>**\_tamaño del objeto WPD \_**                                                   | **VT \_ UI8**     | Tamaño de los datos de recursos de objeto.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="wpd_object_sync_id"></span><span id="WPD_OBJECT_SYNC_ID"></span>**\_identificador de \_ sincronización del objeto WPD \_**                                         | **VT \_ LPWStr**  | Cadena opaca creada por un cliente para conservar el estado entre sesiones sin conservar un catálogo de contenido de dispositivo conectado.                                                                                                                                                                                                                                                                                                                                                                                                |
