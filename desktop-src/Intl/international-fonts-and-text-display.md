# <a name="international-fonts-and-text-display"></a>Fuentes internacionales y presentación de texto

Las fuentes internacionales y la presentación de texto proporcionan compatibilidad con la plataforma para fuentes internacionales, texto internacional y tipografía precisa en la plataforma Windows.

Las fuentes y el texto internacionales se compilan en las tecnologías básicas de tipografía, texto y lectura avanzada de Windows para proporcionar fuentes y texto a usuarios mundiales de la plataforma Windows. La tecnología mejora los escenarios de presentación de fuentes y texto de un extremo a otro.

Se incluyen las siguientes tecnologías:

-   [Administración de fuentes internacionales](about-international-fonts-and-text.md). Presenta la funcionalidad básica de las fuentes internacionales.
-   [Usar MS Shell Dlg y MS Shell Dlg 2](using-ms-shell-dlg-and-ms-shell-dlg-2.md). Información general de las fuentes MS Shell Dlg y MS Shell Dlg 2 y la asignación de fuentes de MS Shell Dlg y MS Shell Dlg 2 en diferentes versiones de Windows.
-   [Enumeración y selección de fuentes internacionales](using-international-fonts-and-text.md). Explica las posibles maneras de enumerar fuentes internacionales para permitir que los usuarios seleccionen la fuente que deseen.
-   [Presentación de texto internacional](creating-your-own-format-selection-user-interface.md). Explica las posibles formas de mostrar texto internacional.
-   [Uniscribe](uniscribe.md). Permite un alto grado de control para la tipografía precisa y el procesamiento de scripts complejos.
