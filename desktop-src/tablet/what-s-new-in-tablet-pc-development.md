# <a name="whats-new-in-tablet-pc-development"></a>Novedades en el desarrollo de Tablet PC

En esta sección se describen las novedades del desarrollo tecnológico de Tablet PC.

## <a name="tablet-pc-development-kit-now-in-windows-sdk"></a>Kit de desarrollo de Tablet PC ahora en Windows SDK

A partir del muelle de 2006 y avanzar, el Windows SDK es el único mecanismo de distribución para la última tecnología de Tablet PC. Ya no hay un SDK de Tablet PC independiente. Los archivos binarios de la plataforma, los encabezados, las bibliotecas, la documentación y el código de ejemplo están disponibles en la Windows SDK. Esta distribución incluye los archivos binarios de la plataforma de Tablet PC 1,7 y los módulos de combinación redistribuibles para esos archivos binarios, así como documentación para los archivos binarios de la versión 1,7 y los nuevos archivos binarios de la plataforma de tecnología de Tablet PC (versión 6) de Windows Vista.

Consulte [el entorno de desarrollo](the-development-environment.md) para obtener más información acerca de cómo configurar el equipo para el desarrollo de Tablet PC.

## <a name="windows-server-2008-r2-now-supports-handwriting-recognition-and-custom-dictionaries"></a>Windows Server 2008 R2 admite ahora el reconocimiento de escritura a mano y los diccionarios personalizados

Windows Server 2008 R2 ahora puede realizar el reconocimiento de tinta y se han agregado características de diccionario personalizado para admitir nuevos escenarios de reconocimiento.

Consulte [reconocimiento de escritura a mano en Windows Server 2008 R2](handwriting-recognition-in-windows-server-2008-r2.md) y [creación de diccionarios personalizados para el reconocimiento de escritura a mano](creating-custom-dictionaries-for-handwriting-recognition-in-windows-7-and-windows-server-2008-r2.md) para obtener más información.

## <a name="math-input"></a>Entrada matemática

La API de entrada matemática ahora está disponible en una versión COM. Vea [Programming the Math INPUT control](programming-the-math-input-control.md) y [Math INPUT control Reference](math-input-control-reference.md) para obtener más información.

## <a name="ink-analysis"></a>Análisis de la entrada de lápiz

Se han creado nuevas características de análisis de tinta para la plataforma de tecnología de Tablet PC. Consulte [información general sobre análisis de tinta](ink-analysis-overview.md) para detalles conceptuales.

## <a name="stylus-input-api-com-version"></a>Versión de COM de la API de entrada del lápiz

La API de entrada del lápiz está ahora disponible en una versión COM. Consulte [referencia de RealTimeStylus](realtimestylus-reference.md) para la versión com de los temas de referencia de la API. Consulte [acceso y manipulación](accessing-and-manipulating-stylus-input.md) de la entrada del lápiz para obtener información conceptual sobre la API de entrada del lápiz.

## <a name="textinputpanel-class"></a>Clase TextInputPanel

La clase [TextInputPanel](/previous-versions/ms573640(v=vs.100)) está disponible como un objeto que se puede adjuntar, lo que le permite agregar funcionalidad del panel de entrada de Tablet PC a los controles existentes. La interfaz de usuario está asignada en gran medida por el idioma de entrada actual. Tiene la opción de elegir escritura a mano o teclado como el método de entrada predeterminado para el objeto TextInputPanel. El usuario final puede cambiar entre los métodos de entrada mediante el uso de botones en la interfaz de usuario.

## <a name="journal-reader-component"></a>Componente del lector del diario

El componente del lector del diario está disponible. Consulte [uso del componente de lector del diario](using-the-journal-reader-component.md) para ver el material de información general. Vea la clase [JournalReader](/previous-versions/ms583815(v=vs.100)) para las API administradas. Consulte [referencia de componentes del lector de Journal](journal-reader-component-reference.md) para la versión com de las API.
