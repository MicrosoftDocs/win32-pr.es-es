# <a name="default-queue-callback-routine"></a>Rutina de devolución de llamada de cola predeterminada

Incluida con las funciones de configuración de es una rutina de devolución de llamada predeterminada, [**SetupDefaultQueueCallback**](/windows/desktop/api/Setupapi/nf-setupapi-setupdefaultqueuecallbacka), que puede usar para procesar notificaciones devueltas por [**SetupCommitFileQueue**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea).

En las secciones siguientes se describe el formato de la rutina de devolución de llamada de cola predeterminada, utilizando la rutina de devolución de llamada de cola predeterminada con [**SetupCommitFileQueue**](/windows/desktop/api/Setupapi/nf-setupapi-setupcommitfilequeuea)y cómo crear una rutina de devolución de llamada de filtro que se basa en la funcionalidad proporcionada por la rutina de devolución de llamada de cola predeterminada.

-   [Acerca de la rutina de devolución de llamada de cola predeterminada](about-the-default-queue-callback-routine.md)
-   [Usar la rutina de devolución de llamada de cola predeterminada](using-the-default-queue-callback-routine.md)
-   [Referencia de rutina de devolución de llamada de cola predeterminada](default-queue-callback-routine-reference.md)
