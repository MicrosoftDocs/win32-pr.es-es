# <a name="contacting-support"></a>Ponerse en contacto con soporte técnico

Si experimenta un problema que no puede resolver, puede ponerse en contacto con el soporte técnico de Microsoft con la siguiente información:

-   Topografía de la aplicación donde se produjo el error, como una descripción de las aplicaciones, los componentes y las interfaces.
-   Registro de eventos de aplicación en todos los equipos.
-   Descripción de cómo reproducir el error, si es posible.
