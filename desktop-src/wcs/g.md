# <a name="g"></a>G

## <a name="gamma-correction"></a>corrección gamma

Las características de tono físico nativo de un dispositivo suelen ser diferentes en distintos tipos de dispositivos. El factor de corrección de gamma compensa estas diferencias.

## <a name="gamma-curve"></a>curva gamma

Colección de correcciones gamma sobre todo el intervalo de colores. Cuando se traza en un gráfico, forman una curva.

## <a name="gamma-ramp"></a>rampa de gamma

Vea curva gamma.

## <a name="gamut"></a>espectro

Conjunto de colores de un espacio de colores que puede producir un dispositivo.

## <a name="gmmp"></a>GMMP

El formato de perfil del modelo de mapa de la gama de colores es un formato de archivo basado en XML que controla la asignación de colores entre las gamas de dispositivos de origen y de destino.
