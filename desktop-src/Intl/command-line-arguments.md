# <a name="command-line-arguments"></a>Argumentos de la línea de comandos

Una aplicación puede usar la función [GetCommandLine](/windows/win32/api/processenv/nf-processenv-getcommandlinea) de Win32 para recuperar argumentos de línea de comandos Unicode. La aplicación debe llamar a la función Win32 como una función Unicode.
