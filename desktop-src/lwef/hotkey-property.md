# <a name="hotkey-property"></a>Propiedad HotKey

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descripción**
</dt> <dd>

Devuelve una cadena que especifica la configuración actual del usuario para la clave de escucha.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintaxis** 
</dt> <dd>

*agent!". SpeechInput.HotKey**

</dd> </dl>

## <a name="remarks"></a>Comentarios

La **propiedad Tecla de** acceso rápido refleja la configuración actual de la clave de escucha en la página Entrada de voz de la hoja de propiedades agente (Opciones avanzadas de caracteres). La propiedad es de solo lectura; solo el usuario puede cambiar la configuración.
