# <a name="the-teredo-security-model"></a>El modelo de seguridad de Teredo

La [tecnología Teredo,](about-teredo.md) cuando se combina con Seguridad de Windows Center [y Windows Firewall,](/previous-versions/windows/desktop/ics/windows-firewall-start-page)proporciona una seguridad mejorada al usuario final. Para Windows firewall, el tráfico IPv6 enviado y recibido a través de la interfaz teredo se procesa igual que el tráfico IPv6 nativo. La interfaz de Teredo se reconoce como otro mini puerto del sistema.

Para más información sobre el uso, consulte [Implementación del modelo de seguridad de Teredo.](implementing-the-teredo-security-model.md)
