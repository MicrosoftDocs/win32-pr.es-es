# <a name="monthlytriggerrunonlastdayofmonth-property"></a>MonthlyTrigger.RunOnLastDayOfMonth, propiedad

Para el scripting, obtiene o establece un valor booleano que indica que la tarea se ejecuta el último día del mes.

## <a name="syntax"></a>Sintaxis


```VB
MonthlyTrigger.RunOnLastDayOfMonth As Boolean
```



## <a name="property-value"></a>Valor de propiedad

True indica que la tarea se ejecuta el último día del mes, independientemente de la fecha real de ese día; de lo contrario, False.
