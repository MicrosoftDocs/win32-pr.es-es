# <a name="interface-implementation-of-system-providers"></a>Implementación de interfaz de proveedores de sistemas

ADSI incorpora varios proveedores de servicios de directorio, es decir, LDAP y WinNT. Los distintos proveedores pueden optar por implementar una interfaz o no implementarla. Los distintos proveedores de servicios de directorio admiten diferentes propiedades y métodos. Para obtener más información sobre la compatibilidad específica del proveedor de servicios de directorio, vea [Proveedores de sistemas ADSI.](adsi-system-providers.md)
