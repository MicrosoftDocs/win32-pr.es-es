# <a name="topoedit-main-window"></a>Ventana principal de TopoEdit

Después de iniciar TopoEdit, aparecerá la ventana que aparece en la captura de pantalla siguiente.

![captura de pantalla que muestra la ventana principal de topoedit](images/809bce6e-43b7-4399-a41b-deff894ba151.jpg)

Los elementos de la ventana principal de TopoEdit son:

-   **Panel de topología**. Aparece a la izquierda de la ventana. Esta área se usa para agregar nodos de topología, ver una topología y conectar los nodos de topología. Para obtener información sobre el uso de este panel, vea [crear topologías con TopoEdit](building-topologies-by-using-topoedit.md).
-   **Panel atributos**. Aparece a la derecha de la ventana. Este panel muestra los atributos del nodo de topología seleccionado actualmente. Para obtener información sobre los atributos que se muestran en este panel, vea [ver información](viewing-topology-information.md)de la topología.
-   Restringi. Vea [menús de TopoEdit](topoedit-menus.md).
-   Barra de herramientas. Vea la [barra de herramientas TopoEdit](topoedit-toolbar.md).
