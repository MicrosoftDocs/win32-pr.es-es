# <a name="video-capture-windows-multimedia"></a>Captura de vídeo (Windows multimedia)

Puede incorporar fácilmente capacidades de captura de vídeo en la aplicación mediante la clase de ventana AVICap. AVICap proporciona a las aplicaciones una sencilla interfaz basada en mensajes para tener acceso al hardware de adquisición de audio y vídeo, y para controlar el proceso de streaming de vídeo en disco.

-   [Uso de la captura de vídeo](using-video-capture.md)
-   [Referencia de captura de vídeo](video-capture-reference.md)
