# <a name="idl-attributes"></a>Atributos IDL

Los atributos son palabras clave que especifican las características de una interfaz y de los datos y métodos de esa interfaz. El conjunto de atributos que se aplica a una sola interfaz está entre corchetes (**\[ \]**). En los temas siguientes se presentan las categorías de atributos que se pueden utilizar en un archivo IDL.

-   [Atributos de campo](/windows/desktop/Rpc/field-attributes)
-   [Atributos de alias y serialización](aliasing-and-marshaling-attributes.md)
-   [Atributos asincrónicos](asynchronous-attributes.md)
-   [Atributos array y Sized-Pointer](array-and-sized-pointer-attributes.md)
-   [Atributos de tipo de datos](data-type-attributes.md)
-   [Atributos direccionales](directional-attributes.md)
-   [Atributos de llamada de función](function-call-attributes.md)
-   [Atributos de encabezado de interfaz](interface-header-attributes.md)
-   [Atributos de rendimiento](performance-attributes.md)
-   [Atributos de tipo de puntero](pointer-type-attributes.md)
-   [Atributos de estructura y de Unión](structure-and-union-attributes.md)
-   [Atributos de la biblioteca de tipos](type-library-attributes.md)
