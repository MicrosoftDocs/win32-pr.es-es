# <a name="registeredtaskcollectionitem-property"></a>RegisteredTaskCollection. Item (propiedad)

En el caso de scripting, obtiene la tarea registrada especificada de la colección.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
RegisteredTaskCollection.Item( _
  ByVal Index _
) As RegisteredTask
```



## <a name="property-value"></a>Valor de propiedad

Un objeto [**RegisteredTask**](registeredtask.md) que contiene la tarea registrada.
