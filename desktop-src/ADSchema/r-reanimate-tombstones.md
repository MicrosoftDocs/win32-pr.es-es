# <a name="reanimate-tombstones-extended-right"></a>Reanimate-Tombstones derecha extendida

Control derecho de acceso que permite restaurar los elementos de esquema eliminados.



| Entrada | Value |
|--------------|--------------------------------------|
| CN           | Reanimate-Tombstones                 |
| Display-Name | Reanimación de marcadores de exclusión                 |
| Rights-GUID  | 45ec5156-db7e-47bb-b53f-dbeb2d03c40f |
