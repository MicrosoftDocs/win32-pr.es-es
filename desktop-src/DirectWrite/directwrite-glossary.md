# <a name="directwrite-glossary"></a>DirectWrite Glosario

<dl> <dt>

<span id="directwrite.directwrite_glossary_bidi__unicode_feature_"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_BIDI__UNICODE_FEATURE_"></span>**BiDi (característica Unicode)**
</dt> <dd>

Se pronuncia "By-El-Insocente". Texto que contiene una combinación de palabras, de distintos idiomas, que se deben leer en direcciones diferentes. A menudo, se requiere un código especial para procesar. "El árabe y el hebreo son idiomas BiDi; el texto va de derecha a izquierda, pero los números van de izquierda a derecha".

</dd> <dt>

<span id="directwrite.directwrite_glossary_cleartype"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_CLEARTYPE"></span>**Cleartype**
</dt> <dd>

Tecnología de presentación de fuentes que mejora enormemente la resolución de presentación de las fuentes, de forma que las letras aparecen nítidas, no irregulares, en la pantalla. ClearType mejora en gran medida la legibilidad del texto en monitores LCD en color con una interfaz digital, como los de los equipos portátiles y las pantallas de escritorio planas de alta calidad.

</dd> <dt>

<span id="directwrite.directwrite_glossary_direct2d"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECT2D"></span>**Direct2D**
</dt> <dd>

UNA API de gráficos 2D acelerada por hardware, en modo inmediato que proporciona un alto rendimiento y una representación de alta calidad para geometría 2D, mapas de bits y texto. La API de Direct2D está diseñada para interoperar bien con el código existente que usa GDI, GDI+ o Direct3D.

</dd> <dt>

<span id="directwrite.directwrite_glossary_directwrite"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECTWRITE"></span>**DirectWrite**
</dt> <dd>

Una API de DirectX que proporciona una mejor representación de texto de alta calidad e interoperabilidad con GDI y Direct2D.

</dd> <dt>

<span id="directwrite.directwrite_glossary_directx"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECTX"></span>**Directx**
</dt> <dd>

Una extensión del sistema operativo Windows Microsoft. La tecnología DirectX ayuda a los juegos y otros programas a usar las funcionalidades multimedia avanzadas del hardware.

</dd> <dt>

<span id="directwrite.directwrite_glossary_gdi"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GDI"></span>**Gdi**
</dt> <dd>

Un programa ejecutable que procesa llamadas gráficas de función desde una aplicación basada en Windows y pasa esas llamadas al controlador de dispositivo adecuado, que realiza las funciones específicas del hardware que generan la salida. Al actuar como búfer entre aplicaciones y dispositivos de salida, GDI presenta una vista del mundo independiente del dispositivo para la aplicación mientras interactúa en un formato dependiente del dispositivo con el dispositivo.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH"></span>**Glifo**
</dt> <dd>

Representación física de un carácter en una fuente determinada. Los caracteres pueden tener muchos glifos, y cada fuente de un sistema podría definir un glifo diferente para ese carácter.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_composition"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_COMPOSITION"></span>**composición de glifos**
</dt> <dd>

Combinación de dos o más glifos en un único glifo.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_decomposition"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_DECOMPOSITION"></span>**descomposición de glifos**
</dt> <dd>

La división de un único glifo en varios glifos.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_run"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_RUN"></span>**glyph Run**
</dt> <dd>

Conjunto de glifos en un orden específico, con las mismas características de formato, como la cara de fuente, el tamaño, el peso y el estilo.

</dd> <dt>

<span id="directwrite.directwrite_glossary_hardware-accelerated_text"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_HARDWARE-ACCELERATED_TEXT"></span>**texto acelerado por hardware**
</dt> <dd>

Texto representado por una tecnología que usa la aceleración de hardware para mejorar el rendimiento de la representación.

</dd> <dt>

<span id="directwrite.directwrite_glossary_interoperability"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_INTEROPERABILITY"></span>**Interoperabilidad**
</dt> <dd>

La capacidad de dos o más API para trabajar con y transmitir información entre sí.

</dd> <dt>

<span id="directwrite.directwrite_glossary_kerning"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_KERNING"></span>**Kerning**
</dt> <dd>

Espaciado aplicado entre pares específicos de letras dentro de una palabra determinada.

</dd> <dt>

<span id="directwrite.directwrite_glossary_letterspacing"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_LETTERSPACING"></span>**letterspacing**
</dt> <dd>

Ajuste del espaciado entre dos caracteres para crear la apariencia del espaciado uniforme, ajustar el texto a un espacio determinado y ajustar los saltos de línea.

</dd> <dt>

<span id="directwrite.directwrite_glossary_ligature"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_LIGATURE"></span>**Ligadura**
</dt> <dd>

Dos o más caracteres combinados para representar un único carácter tipográfico. El script latino moderno usa algunos. Otros scripts usan muchas ligaduras que dependen de la fuente y el estilo. Algunos idiomas poseen ligaduras obligatorias, por ejemplo, árabe.

</dd> <dt>

<span id="directwrite.directwrite_glossary_smart_pointers"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_SMART_POINTERS"></span>**punteros inteligentes**
</dt> <dd>

Clase que encapsula punteros de interfaz COM que liberarán automáticamente el objeto de interfaz.

</dd> <dt>

<span id="directwrite.directwrite_glossary_swashes"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_SWASHES"></span>**swashes**
</dt> <dd>

Adición de anillos a un carácter que crea un glifo más estilístico.

</dd> <dt>

<span id="directwrite.directwrite_glossary_unicode"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_UNICODE"></span>**Unicode**
</dt> <dd>

Estándar de codificación de caracteres desarrollado por Unicode Consortium que representa casi todos los idiomas escritos del mundo. El repertorio de caracteres Unicode tiene varias formas de representación, incluidas UTF-8, UTF-16 y UTF-32. La mayoría de las interfaces de Windows utilizan el formato UTF-16. (GTMT tiene 20 definiciones).

</dd> <dt>

<span id="directwrite.directwrite_glossary_windows_presentation_foundation__wpf_"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_WINDOWS_PRESENTATION_FOUNDATION__WPF_"></span>**Windows Presentation Foundation (WPF)**
</dt> <dd>

Un marco de gui usado por .NET Micro Framework 3.0, Windows Vista y Microsoft Silverlight. Las clases de gui de .NET Micro Framework se basan de forma flexible en WPF.

</dd> </dl>
