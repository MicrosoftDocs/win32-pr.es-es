# <a name="d1209-default-dpi-assumed-due-to-failed-getresolution"></a>D1209: se ha asumido un valor de PPP predeterminado debido a un error de GetResolution

No se pudo ejecutar [**IWICBitmapSource:: GetResolution**](/windows/desktop/api/wincodec/nf-wincodec-iwicbitmapsource-getresolution) para el \[ *código de error* de origen del mapa de bits de entrada \] . Se supone que el valor predeterminado \[ *es PPP* \] .
