# <a name="wcs-10-availability"></a>Disponibilidad de WCS 1.0

ICM1 está disponible para Microsoft Windows 95. No está disponible para MS DOS, Windows 3. *x* o Windows NT 4.0 o anterior.

ICM2 está disponible para Microsoft Windows 98, Windows Millennium Edition (Me), Windows 2000 y Windows XP. No está disponible para MS DOS, Windows 3. *x*, Windows 95 o Windows NT 4.0 o anterior.

WCS 1.0 está disponible para Microsoft Windows Vista. No está disponible para Windows XP o versiones anteriores.
