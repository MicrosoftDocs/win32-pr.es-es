# <a name="direct3d-9-video-interfaces"></a>Interfaces de vídeo de Direct3D 9

Describe las interfaces de vídeo de Microsoft Direct3D 9.



| Interfaz                                                                | Descripción                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [**IDirect3D9ExOverlayExtension**](/windows/desktop/api/d3d9/nn-d3d9-idirect3d9exoverlayextension)     | Consulta las funcionalidades de hardware de superposición de un dispositivo Direct3D.                                            |
| [**IDirect3DAuthenticatedChannel9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dauthenticatedchannel9) | Proporciona un canal de comunicación al controlador de gráficos o al entorno de ejecución de Direct3D.                        |
| [**IDirect3DCryptoSession9**](/windows/desktop/api/d3d9/nn-d3d9-idirect3dcryptosession9)               | Representa una sesión criptográfica.                                                                        |
| [**IDirect3DDevice9Video**](/windows/desktop/api/d3d9/nn-d3d9-idirect3ddevice9video)                   | Permite que una aplicación use los servicios de protección de contenido y cifrado implementados por un controlador de gráficos. |
| [**IDirect3DDXVADevice9**](idirect3ddxvadevice9.md)                     | Representa un acelerador de hardware para DirectX Video Acceleration (DXVA) 1.0.                               |
| [**IDirect3DVideoDevice9**](idirect3dvideodevice9.md)                   | Habilita lacodización acelerada por hardware desde un dispositivo Direct3D mediante DXVA versión 1.0.                      |
