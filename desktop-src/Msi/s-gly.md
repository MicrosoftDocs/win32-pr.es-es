<span id="_msi_synchronous_execution_gly"></span><span id="_MSI_SYNCHRONOUS_EXECUTION_GLY"></span>**ejecución sincrónica**
</dt> <dd>

Para obtener más información, consulte [acciones personalizadas sincrónicas y asincrónicas](synchronous-and-asynchronous-custom-actions.md) .

</dd> <dt>

<span id="_msi_system_policy_using_windows_installer__gly"></span><span id="_MSI_SYSTEM_POLICY_USING_WINDOWS_INSTALLER__GLY"></span>**Directiva del sistema**
</dt> <dd>

Lo usan los administradores del sistema para controlar las configuraciones de usuarios y equipos desde una sola ubicación en una red. Las directivas del sistema propagan la configuración del registro a un gran número de equipos sin necesidad de que el Administrador tenga un conocimiento detallado del registro. La Directiva del sistema del instalador se puede configurar mediante el editor de directiva de grupo (GPE), incluido en Windows 2000. Para obtener más información, consulte [Directiva del sistema](system-policy.md).

</dd> </dl>
