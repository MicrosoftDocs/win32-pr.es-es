# <a name="confirm-switch"></a>/confirm switch

El **modificador /confirm** indica al compilador que muestre todas las opciones del compilador MIDL sin procesar el IDL de entrada y los archivos ACF opcionales.

``` syntax
midl /confirm
```

## <a name="switch-options"></a>Cambiar opciones

Este modificador no tiene parámetros.

## <a name="examples"></a>Ejemplos

**midl /confirm**

**midl /confirm @response.rsp filename.idl**
