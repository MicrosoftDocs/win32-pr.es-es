# <a name="privacy-statement-windows-media-device-manager-11-sdk"></a>Declaración de privacidad (SDK de Windows Media Administrador de dispositivos 11)

Los desarrolladores que crean aplicaciones de Windows Media Administrador de dispositivos deben comunicar claramente a sus usuarios cómo podría verse afectada su privacidad. Para ayudarle a crear una declaración de privacidad para el usuario final, Microsoft ha publicado directrices de privacidad en la [Página Web de privacidad de Windows Media](https://www.microsoft.com/windows/windowsmedia/privacy/10ssdk.aspx).
