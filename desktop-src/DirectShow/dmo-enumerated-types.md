# <a name="dmo-enumerated-types"></a>Tipos enumerados de DMO

En esta sección se describen los tipos enumerados de Microsoft DirectX media Object (DMO).



| Tipo enumerado                                                            | Descripción                                                                  |
|----------------------------------------------------------------------------|------------------------------------------------------------------------------|
| [**\_marcas de enumeración de DMO \_**](/previous-versions/windows/desktop/api/Dmoreg/ne-dmoreg-dmo_enum_flags)                                 | Define las marcas que especifican los criterios de búsqueda al enumerar DMOs.            |
| [**\_marcas de \_ búfer de datos de entrada de DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_input_data_buffer_flags)     | Define las marcas que describen un búfer de entrada.                                 |
| [**\_marcas de \_ información de flujo de entrada de DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_input_stream_info_flags)     | Define las marcas que describen un flujo de entrada.                                 |
| [**\_marcas de \_ búfer de datos de salida de DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_output_data_buffer_flags)   | Define las marcas que describen un búfer de salida.                                |
| [**\_marcas de \_ información de flujo de salida de DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_output_stream_info_flags)   | Define las marcas que describen un flujo de salida.                                |
| [**\_marcas de \_ salida de proceso de DMO \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_process_output_flags)            | Define las marcas que especifican las solicitudes de procesamiento de salida.                       |
| [**\_marcas de registro de DMO \_**](/previous-versions/windows/desktop/api/Dmoreg/ne-dmoreg-dmo_register_flags)                         | Define las marcas que especifican la información del registro para un DMO.                   |
| [**\_marcas de \_ tipo de conjunto de DMO \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_set_type_flags)                        | Define marcas para establecer el tipo de medio en una secuencia.                        |
| [**\_marcas de \_ flujo de salida de vídeo DMO \_ \_**](/previous-versions/windows/desktop/api/Mediaobj/ne-mediaobj-_dmo_video_output_stream_flags) | Define las marcas que describen las características solicitadas para las optimizaciones de vídeo.      |
| [**\_tipo de curva de MP \_**](/previous-versions/windows/desktop/api/Medparam/ne-medparam-mp_curve_type)                                   | Define la curva que sigue un parámetro multimedia dentro de un segmento de sobre. |
| [**tipo de MP \_**](/previous-versions/windows/desktop/api/Medparam/ne-medparam-mp_type)                                                | Especifica el tipo de datos de un parámetro.                                     |
