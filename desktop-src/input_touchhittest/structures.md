# <a name="touch-hit-testing-structures"></a>Estructuras de pruebas de impacto táctiles

En los temas de esta sección se proporcionan las especificaciones de referencia para las [estructuras de Prueba de impacto](touch-hit-testing-portal.md) táctil.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
| --- | --- |
| [**TOUCH_HIT_TESTING_INPUT**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_input)<br/> | Contiene información sobre el área de contacto táctil notificada por el digitalizador táctil.<br/> |
| [**TOUCH_HIT_TESTING_PROXIMITY_EVALUATION**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_proximity_evaluation)<br/> | Contiene la puntuación de la prueba de acceso que indica si el objeto es el destino probable del área de contacto táctil, en relación con otros objetos que intersecan con el área de contacto táctil.<br/> |
