# <a name="display-name-printable-attribute"></a>Atributo Display-Name-Printable

Nombre para mostrar imprimible de un objeto. El nombre para mostrar imprimible suele ser la combinación del nombre del usuario, la inicial intermedia y el apellido.



| Entrada | Valor |
|-------------------|----------------------------------------|
| CN                | Nombre para mostrar imprimible                 |
| Ldap-Display-Name | displayNamePrintable                   |
| Size              | \-                                     |
| Privilegio actualizar  | Administrador de dominio o propietario de la cuenta. |
| Frecuencia de actualización  | \-                                     |
| Attribute-Id      | 1.2.840.113556.1.2.353                 |
| System-Id-Guid    | bf967954-0de6-11d0-a285-00aa003049e2   |
| Syntax            | [**String(IA5)**](s-string-ia5.md)    |
