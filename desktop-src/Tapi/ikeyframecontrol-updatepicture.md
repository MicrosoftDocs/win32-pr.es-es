# <a name="ikeyframecontrolupdatepicture-method"></a>IKeyFrameControl::UpdatePicture (método)

\[**UpdatePicture** no está disponible para su uso en Windows Vista, Windows Server 2008 y versiones posteriores del sistema operativo. La API de cliente RTC proporciona una funcionalidad similar.\]

Una aplicación llama al **método UpdatePicture** para pedir al remitente del vídeo que actualice la imagen. Cada vez que se llama a este método, se envía una solicitud al remitente del vídeo. La secuencia debe estar activa.
