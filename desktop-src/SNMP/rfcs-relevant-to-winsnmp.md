# <a name="rfcs-relevant-to-winsnmp"></a>RFC relevantes para WinSNMP

Los estándares TCP/IP se definen en Solicitudes de comentarios (RFC), que publica el Grupo de trabajo de ingeniería de Internet (IETF). Las RFC que son relevantes para las características de WinSNMP se enumeran en la tabla siguiente.



| Número RFC | Título                                                                                                  |
|------------|--------------------------------------------------------------------------------------------------------|
| 1155       | "Structure and Identification of Management Information for TCP/IP-based Internets" (Estructura e identificación de la información de administración de Internet basada en TCP/IP)                    |
| 1157       | "A Simple Network Management Protocol (SNMP)"                                                          |
| 1213       | "Management Information Base for Network Management of TCP/IP-based Internets: MIB-II"                 |
| 1901       | "Introduction to Community-based SNMPv2"                                                               |
| 1902       | "Estructura de la información de administración para la versión 2 del Protocolo simple de administración de redes (SNMPv2)" |
| 1903       | "Textual Conventions for Version 2 of the Simple Network Management Protocol (SNMPv2)"                 |
| 1904       | "Instrucciones de conformidad para la versión 2 del Protocolo simple de administración de redes (SNMPv2)"              |
| 1905       | "Operaciones de protocolo para la versión 2 del Protocolo simple de administración de redes (SNMPv2)"                 |
| 1906       | "Asignaciones de transporte para la versión 2 del Protocolo simple de administración de redes (SNMPv2)"                  |
| 1907       | "Management Information Base for Version 2 of the Simple Network Management Protocol (SNMPv2)"         |
| 1908       | "Coexistencia entre la versión 1 y la versión 2 del marco de administración de redes estándar de Internet"    |
| 2089       | "V2ToV1 Mapping SNMPv2 onto SNMPv1 within a bi-lingual SNMP agent" (Asignación de SNMPv2 a SNMPv1 dentro de un agente SNMP bilingüe)                                     |
