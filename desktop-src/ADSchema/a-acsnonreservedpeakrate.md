# <a name="acs-non-reserved-peak-rate-attribute"></a>Atributo ACS-no reservado-tasa máxima

El atributo **ACS-non-Reserved-Rate-Rate** solo es para uso interno. Basado en RFC2814.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | ACS-no reservado-tasa máxima           |
| Nombre para mostrar de LDAP | aCSNonReservedPeakRate               |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1318              |
| System-ID-GUID    | a331a73f-3b90-11d2-90cc-00c04fd91ab1 |
| Sintaxis            | [**Interval**](s-interval.md)       |
