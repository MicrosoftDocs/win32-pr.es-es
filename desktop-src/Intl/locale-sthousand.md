# <a name="locale_sthousand"></a>configuración regional \_ STHOUSAND

Caracteres que se usan para separar grupos de dígitos a la izquierda del separador decimal. El número máximo de caracteres permitido para esta cadena es cuatro, incluido un carácter nulo de terminación. Normalmente, estos grupos representan miles. Sin embargo, en función del valor especificado para [ \_ SGROUPING de configuración regional](locale-sgrouping.md), pueden representar otra cosa.
