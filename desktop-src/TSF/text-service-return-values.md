# <a name="text-service-return-values"></a>Valores devueltos del servicio de texto

Un servicio de texto genera un valor devuelto de 0x *HHHH* 0600. En la tabla siguiente se muestra el valor devuelto del servicio de texto.



| Código o valor devuelto                                                                                                                                                                                                                          | Descripción                                                           |
|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------|
| <span id="TF_E_NOCONVERSION"></span><span id="tf_e_noconversion"></span><dl> <dt>**TF \_ E \_ noconversión**</dt> <dt>0x80040600</dt> </dl> | No hay ninguna conversión disponible para el intervalo de texto seleccionado.<br/> |
