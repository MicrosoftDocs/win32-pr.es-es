# <a name="alternate-images"></a>Imágenes alternativas

Las imágenes alternativas son imágenes que no se ven cuando se muestra inicialmente la máscara, pero se vuelven visibles en respuesta a los eventos del mouse. Por ejemplo, puede configurar la máscara de modo que cuando se mantenga el mouse sobre un área específica, la imagen de esa área cambie. Esto puede informar a los usuarios visualmente de que pueden hacer clic en un área para hacer algo. También puede usar una imagen alternativa cuando el usuario hace clic en un área para proporcionarle comentarios visuales en los que Windows Media Player ha recibido el clic.
