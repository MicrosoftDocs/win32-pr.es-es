# <a name="wpd-constants"></a>Constantes de WPD

Dispositivos portátiles de Windows define los siguientes tipos constantes.



| Tipo constante                                              | Descripción                                                                                                                                                                                                                               |
|------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Constantes generales](general-constants.md)                 | Constantes de uso general para dispositivos y aplicaciones.                                                                                                                                                                                       |
| [Constantes de error](error-constants.md)                     | Valores de error devueltos por los métodos.                                                                                                                                                                                                         |
| [Constantes de evento](event-constants.md)                     | Tipos de eventos definidos por dispositivos portátiles de Windows. Entre los tipos de evento se incluyen objeto agregado, objeto quitado, objeto actualizado, objeto trasladado, objeto copiado, transferencia de objetos solicitada, restablecimiento del dispositivo, dispositivo eliminado y capacidades del dispositivo actualizadas. |
| [GUID de tipo de objeto](object-type-guids.md)                 | Tipos de objetos definidos por dispositivos portátiles de Windows. Los tipos de objeto incluyen objetos de audio, correo electrónico e imagen.                                                                                                                                  |
| [GUID de formato de objeto](object-format-guids.md)             | Tipos de formato de archivo definidos por dispositivos portátiles de Windows. Los tipos de formato incluyen WMA, WMV y JPEG.                                                                                                                                           |
| [GUID de tipo de recurso](resource-type-guids.md)             | Tipos de recursos definidos por dispositivos portátiles de Windows. Los tipos de recursos incluyen datos de audio y de imagen en miniatura.                                                                                                                                |
| [Propiedades y atributos](properties-and-attributes.md) | Propiedades y atributos definidos por dispositivos portátiles de Windows.                                                                                                                                                                            |
