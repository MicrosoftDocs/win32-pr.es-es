# <a name="developing-accessible-ui-frameworks-for-windows"></a>Desarrollo de marcos de interfaz de usuario accesibles para Windows

Para que lo puedan usar tantas personas como sea posible, los marcos de interfaz de usuario compilados para la plataforma de Windows deben admitir características de accesibilidad que facilitan a los usuarios con discapacidades, preferencias personales o estilos de trabajo específicos usar sus dispositivos Windows correctamente.

En esta información general se describe cómo crear un marco de interfaz de usuario que admita características como el acceso mediante programación y la automatización, la navegación y el comando de teclado, las opciones de color y tema, y la personalización a través de la configuración de usuario.
