# <a name="ms-ds-cached-membership-attribute"></a>atributo MS-DS-Cached-Membership

El atributo **MS-DS-Cached-Membership** se usa en el administrador de cuentas de seguridad para la expansión de grupos durante la evaluación de tokens.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | MS-DS-en caché-pertenencia                               |
| Nombre para mostrar de LDAP | msDS: pertenencia a la caché                                |
| Tamaño              | BLOB binario de hasta 3 kilobytes.                     |
| Actualizar privilegio  | El sistema establece este valor.                      |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1441                               |
| System-ID-GUID    | 69cab008-cdd4-4bc9-bab8-0ff37efe1b20                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
