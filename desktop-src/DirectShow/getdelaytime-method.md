# <a name="getdelaytime-method"></a>Método GetDelayTime

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

El `GetDelayTime` método recupera el tiempo de retraso de la información sobre herramientas asociada al objeto **MSWebDVD** .

``` syntax
[ iDelay = ] MSWebDVD.GetDelayTime()
```

## <a name="return-value"></a>Valor devuelto

Devuelve un valor entero que indica el tiempo de retraso de la información sobre herramientas, en milisegundos.
