# <a name="winhttp-versions"></a>Versiones de WinHTTP

Los usuarios de los servicios HTTP de Microsoft Windows (WinHTTP) deben usar la versión más reciente de la tecnología, versión 5,1. Ya no se admite la versión 5,0.

## <a name="version-51"></a>Versión 5.1

WinHTTP 5,1 ofrece mejoras en la versión 5,0; para obtener más información acerca de las nuevas características, consulte [novedades de WinHTTP 5,1](what-s-new-in-winhttp-5-1.md).

Con la versión 5,1, WinHTTP es un componente del sistema operativo de los siguientes sistemas operativos:

-   Windows 2000, Service Pack 3 y versiones posteriores (excepto Datacenter Server)
-   Windows XP con Service Pack 1 (SP1) y versiones posteriores
-   Windows Server 2003 con Service Pack 1 (SP1) y versiones posteriores

En Windows Server 2003 con SP1, WinHTTP es un ensamblado en paralelo del sistema. Para obtener más información, vea [ensamblados en paralelo](/windows/desktop/SbsCs/about-side-by-side-assemblies-).

No se admite la redistribución de WinHTTP 5,1 fuera de estos sistemas operativos.

## <a name="version-50"></a>Versión 5.0

El redistribuible de WinHTTP 5,0, "winhttp5.dll", se ejecuta en Windows 2000 o posterior, o en Windows NT 4,0 con Microsoft Internet Explorer 5,01 o posterior.

> [!IMPORTANT]
> La descarga de WinHTTP 5,0 ya no está disponible. A partir del 1 de octubre de 2004, Microsoft ha quitado la descarga del SDK de WinHTTP 5,0 de MSDN y ha finalizado el soporte técnico para la versión 5,0.
