# <a name="ndf-helper-class-extension-examples"></a>Ejemplos de extensión de clase auxiliar NDF

Estos ejemplos solo se aplican cuando los desarrolladores creen que es necesario extender la funcionalidad de una clase auxiliar de Microsoft NDF que se declara como extensible. Por lo general, esto no es necesario, pero en algunos casos, las oportunidades de diagnóstico únicas se presentan a causa de los requisitos de hardware o software específicos del componente.

Los dos ejemplos siguientes están relacionados y el primero debe entenderse antes de analizar el segundo. El primero proporciona una implementación de una [extensión de clase auxiliar NDF](ndf-helper-class-example.md) mediante una clase denominada SimpleHelperFileClass.

La segunda [extensión de clase auxiliar NDF con la entrega](ndf-helper-class-example-with-handoff.md), proporciona un ejemplo de una extensión que no realiza tareas de diagnóstico específicas, sino que solo genera una hipótesis de mantenimiento bajo para SimpleHelperFileClass en el primer ejemplo.
