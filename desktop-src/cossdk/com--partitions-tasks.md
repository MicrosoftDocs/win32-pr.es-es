# <a name="com-partitions-tasks"></a>Tareas de particiones COM+

En los siguientes temas de esta sección se proporcionan instrucciones paso a paso para usar las particiones de COM+.

**Windows XP:** La capacidad de crear, configurar o delegar particiones COM+ no está disponible. La partición global es la única partición de COM+ disponible.

* * Windows 2000: * * el servicio de particiones COM+ no está disponible en Windows 2000.



| Tema                                                                                                         | Descripción                                                                                    |
|---------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [Crear y configurar particiones COM+](creating-and-configuring-com--partitions.md)<br/>           | Describe cómo crear y configurar una partición de COM+.<br/>                             |
| [Administrar particiones locales](managing-local-partitions.md)<br/>                                         | Describe cómo administrar particiones COM+ que no están dentro de Active Directory.<br/>       |
| [Administrar particiones en Active Directory](managing-partitions-within-active-directory.md)<br/>     | Describe cómo administrar las particiones COM+ que se especifican en Active Directory.<br/> |
| [Establecer derechos administrativos para una partición](setting-administrative-rights-for-a-partition.md)<br/> | Describe cómo establecer privilegios de seguridad para una partición de COM+.<br/>                      |
| [Agrupar aplicaciones en particiones](grouping-applications-into-partitions.md)<br/>                 | Describe cómo configurar aplicaciones COM+ para que pertenezcan a una partición COM+. <br/>        |
| [Recopilación de métricas de partición](collecting-partition-metrics.md)<br/>                                   | Describe cómo recopilar datos sobre las particiones de COM+.<br/>                                |
| [Configuración de la memoria caché de partición](configuring-the-partition-cache.md)<br/>                             | Describe cómo configurar la caché de particiones de COM+.<br/>                                |



 

> [!Note]  
> El servicio de particiones COM+ no está habilitado de forma predeterminada. Para usar el servicio de particiones de COM+, debe habilitarlo mediante la herramienta de administración de servicios de componentes o cambiando la propiedad PartitionsEnabled de la colección [**LocalComputer**](localcomputer.md) a true.
