# <a name="initialization-of-property-values"></a>Inicialización de valores de propiedad

Se recomienda que todas las propiedades que va a usar la instalación se escriban en la [tabla de propiedades](property-table.md) con un valor inicial. El instalador establece las propiedades en estos valores al iniciar la instalación. Propiedades para las que un espacio en blanco es un valor aceptable y no es necesario inicializar las propiedades integradas en el instalador.
