# <a name="embedding-the-windows-media-player-control-in-an-office-document"></a>Incrustar el control Media Player de Windows en un documento de Office

Insertar el control de Media Player de Windows en un documento de Office es una manera fácil de agregar contenido multimedia digital interactivo dinámico a un documento estático de otro tipo. Por ejemplo, puede crear una hoja de cálculo en Microsoft Excel e insertar un vídeo en el que se resuma un informe, o bien puede crear un documento de Microsoft Word e insertar una breve animación que ilustre un punto realizado en el texto. Si no le gusta la interfaz de usuario proporcionada por el control, puede usar Microsoft Visual Basic para Aplicaciones (VBA) para proporcionar una interfaz de usuario personalizada.

La incrustación del control de Media Player de Windows en una presentación de® de Microsoft PowerPoint le permite complementar los efectos dinámicos ya proporcionados por ese programa. Al guardar la presentación como una página web, el control Media Player de Windows se incrusta en el código HTML tal y como se describe en [usar el control Media Player de Windows en una página web](using-the-windows-media-player-control-in-a-web-page.md). Esto le permite agregar código de script adicional que puede haber desarrollado para otras páginas Web.

Otra manera de generar fácilmente páginas web que insertan el control de Media Player de Windows es con Microsoft FrontPage®. FrontPage le permite crear rápidamente una página, agregar el control de Windows Media Player desde un menú y configurarlo en un cuadro de diálogo de propiedades. De nuevo, el HTML resultante inserta el control Media Player de Windows tal y como se ha descrito anteriormente.
