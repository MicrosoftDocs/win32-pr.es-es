# <a name="pen-end-cap"></a>Pen End Cap

El atributo end cap especifica la forma de un lápiz geométrico: redondeado, cuadrado o plano. En la ilustración siguiente se muestran líneas paralelas dibujadas con cada tipo de extremo.

![ilustración que muestra tres líneas horizontales, cada una con un tipo diferente de extremo](images/cspen-04.png)

Los límites de extremo redondeado y cuadrado se extienden más allá de los puntos inicial y final de una línea dibujada con un lápiz geométrico; el extremo plano no lo hace.
