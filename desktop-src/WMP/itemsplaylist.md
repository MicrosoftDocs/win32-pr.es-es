# <a name="itemsplaylist"></a>ITEMSPLAYLIST

Este es un elemento de **lista de reproducción** predefinido con los siguientes valores predeterminados.

``` syntax
backgroundColor="black"
columns="name=Name;Duration=Time"
columnsVisible="false"
dropDownVisible="false"
foregroundColor="white"
```

## <a name="remarks"></a>Observaciones

Esto creará una **lista de reproducción** simple que muestra los elementos de una lista de reproducción sin los encabezados de columna o desplegables visibles. Todas las propiedades de esta **lista de reproducción** se pueden invalidar mediante su especificación explícita.
