# <a name="windows-server-2008-r2-logo-program-for-software"></a>Programa de logotipo de Windows Server 2008 R2 para software

## <a name="affected-platforms"></a>Plataformas afectadas

 **Servidores:** Windows Server 2008 R2  

### <a name="description"></a>Descripción

Se ha retirado el programa de logotipo de Windows Server 2008 R2.

Los envíos de prueba de Windows Server 2008 R2 2012 ya no se aceptan para las competencias Microsoft Partner Network (MPN) o el logotipo certificado para Windows Server.

Windows Server 2012 R2 es actualmente la plataforma preferida para el logotipo certificado para Windows Server. Para obtener más información, consulte el [programa de certificación del logotipo de aplicación de nube empresarial](https://azure.microsoft.com/overview/commercial-marketplace/) .
