# <a name="provider-minimum-requirements"></a>Requisitos mínimos del proveedor

Para escribir un proveedor ADSI, implemente las interfaces del servicio Active Directory como se describe en las secciones siguientes.

-   La [implementación de Core](core-implementation.md) enumera brevemente los elementos de software que un proveedor ADSI debe admitir.
-   En la [implementación opcional](optional-implementation.md) se enumeran brevemente los elementos de software que se recomienda que admita un proveedor ADSI.
-   [Las propiedades personalizadas](custom-properties.md) son las propiedades y las interfaces expuestas por cada proveedor para proporcionar capacidad de programación de las características nativas del servicio de directorio.
-   Las [interfaces duales](dual-interfaces.md) describen la Convención ADSI para proporcionar interfaces a las que se puede tener acceso tanto a los controladores de automatización como a los que no lo son.
