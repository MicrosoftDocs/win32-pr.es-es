# <a name="xps-document-api"></a>API de documento XPS

La API de documento XPS es una API nativa de Windows que permite a los programas crear, leer, editar, guardar e imprimir documentos XPS. La API de documentos XPS es compatible con Windows 7 y versiones posteriores de Windows, y se puede usar en programas de modo de usuario y controladores de impresora XPSDrv.

La API de documento XPS igualmente XPS y OpenXPS, y está diseñada para programadores de C++ que escriben aplicaciones Windows nativas. Para obtener más información sobre la especificación Open XPS, consulte [Open XML Paper Specification (OpenXPS)](https://www.ecma-international.org/publications/standards/Ecma-388.htm).
