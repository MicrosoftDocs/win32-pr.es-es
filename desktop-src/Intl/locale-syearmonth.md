# <a name="locale_syearmonth"></a>configuración regional \_ SYEARMONTH

**Windows Me/98, windows 2000:** Cadena de formato de mes y mes para la configuración regional. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación. Esta cadena muestra el formato adecuado para una cadena de fecha que solo contiene el año y el mes, mediante el uso de imágenes de formato, tal como se define en las [imágenes con formato de día, mes, año y era](day--month--year--and-era-format-pictures.md).
