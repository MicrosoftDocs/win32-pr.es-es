# <a name="pnrp-and-blob"></a>PNRP y BLOB

PNRP usa la estructura de [**blobs**](winsock-nsp-reference-links.md) para pasar datos a la estructura **WSAQUERYSET** durante las llamadas a varias funciones. Para su uso con PNRP, la estructura de **BLOB** apunta a una estructura [**PNRPINFO**](/windows/desktop/api/Pnrpns/ns-pnrpns-pnrpinfo_v1) o a una estructura [**PNRPCLOUDINFO**](/windows/desktop/api/Pnrpns/ns-pnrpns-pnrpcloudinfo) .
