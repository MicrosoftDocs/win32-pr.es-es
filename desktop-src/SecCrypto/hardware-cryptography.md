# <a name="hardware-cryptography"></a>Criptografía de hardware

Los servicios de Certificate Server admiten la criptografía relacionada con el hardware, como la emisión de certificados almacenados en [*tarjetas inteligentes*](../secgloss/s-gly.md). La entidad de [*certificación*](../secgloss/c-gly.md) (CA) empresarial admite la emisión de certificados de tarjeta inteligente para el inicio de sesión de dominio en el sistema operativo. Además, la CA de empresa y la CA independiente admiten la emisión de otros tipos de certificados de tarjeta inteligente.

Para obtener más información, consulte la documentación del producto. Para obtener información para desarrolladores sobre tarjetas inteligentes, consulte [autenticación mediante tarjeta inteligente](../secauthn/smart-card-authentication.md). Para obtener información sobre el control que permite a un administrador solicitar certificados de tarjeta inteligente en nombre de los usuarios, consulte [control de inscripción de tarjeta inteligente](/previous-versions/windows/desktop/secsmart/smart-card-enrollment-control-usage-scenario).
