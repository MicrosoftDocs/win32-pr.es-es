# <a name="performance-windows-imaging-component"></a>Rendimiento (Windows de creación de imágenes)

El objetivo de Microsoft es que el rendimiento del procesamiento RAW a través de la interfaz Windows Imaging Component (WIC) sea equivalente a las aplicaciones y herramientas de procesamiento RAW líderes del sector. Para proporcionar una guía de rendimiento, se realizó un conjunto de pruebas completas que comparan el rendimiento de las herramientas de flujo de trabajo RAW más populares. Las pruebas incluían versiones del sistema operativo Windows y Apple Macintosh de productos de procesamiento RAW, archivos RAW de ejemplo de cámaras reflex de una sola lente (SLR) actuales y una serie de funciones que representan pasos comunes del flujo de trabajo.

En función de estos resultados, Microsoft sugiere las siguientes recomendaciones a los asociados para el rendimiento del procesamiento RAW:

-   El rendimiento de línea de base debe ser igual o mejor que los tiempos de procesamiento de las herramientas y las aplicaciones de procesamiento RAW proporcionadas por el fabricante actuales.
-   Los objetivos de rendimiento óptimo deben coincidir o superar la aplicación de mayor rendimiento sin sacrificar la calidad (en estas pruebas, El raw de la cámara de Photoshop suele realizarse más rápido).
