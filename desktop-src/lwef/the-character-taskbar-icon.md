# <a name="the-character-taskbar-icon"></a>Icono de la barra de tareas Carácter

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Si se ha escrito un carácter para incluir un icono, el icono aparece en el área de notificación de la barra de tareas cuando se ejecuta Microsoft Agent. Este icono proporciona acceso al menú emergente del carácter, que proporciona acceso a los comandos globales del Agente, como los que ocultan y muestran el carácter.

![área de notificación con reloj e icono](images/f1tbicon.gif)

Al mover el puntero sobre el icono de la barra de tareas se muestra una ventana de sugerencias que refleja el nombre del carácter (en el idioma actual del sistema). Al hacer clic con un solo clic en el icono de la barra de tareas del carácter, se muestra el carácter. La acción asociada a hacer doble clic en el icono depende de la aplicación actual que controla ese carácter.

Al hacer clic con el botón derecho en el icono se muestra un menú emergente. Cuando el carácter está visible, el menú emergente muestra los mismos comandos que los que se muestran al hacer clic con el botón derecho en el carácter. Si el carácter está oculto, solo aparecen los comandos Abrir (o Cerrar) Comandos de voz y Mostrar.
