# <a name="maintaining-client-specific-lists"></a>Mantenimiento de Client-Specific listas de distribución

RTMv2 proporciona funciones que permiten a los clientes crear una lista privada de rutas almacenadas en la tabla de enrutamiento. El uso de esta lista puede ser más eficaz que enumerar las rutas de la tabla de enrutamiento.

Para obtener código de ejemplo que muestra cómo usar estas funciones, vea [Usar una Client-Specific de rutas.](use-a-client-specific-route-list.md)
