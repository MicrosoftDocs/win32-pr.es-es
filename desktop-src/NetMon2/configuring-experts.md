# <a name="configuring-experts"></a>Configuración de expertos

Para configurar expertos, use los cuadros de diálogo personalizados y la función [**configurar**](configure.md) . Al escribir expertos en el monitor de red, tiene la opción de elegir. Puede crear un conjunto de elementos de configuración predeterminados que se pueden usar cuando se ejecuta inicialmente el experto, o puede permitir que Monitor de red usuarios configuren el experto en tiempo de ejecución. Monitor de red almacena las configuraciones de cada experto. Como opción, los usuarios pueden restablecer la configuración de experto a la configuración predeterminada original. Tenga en cuenta que puede omitir el uso de la **configuración** mediante el desarrollo de un experto que pase una configuración fija a monitor de red.
