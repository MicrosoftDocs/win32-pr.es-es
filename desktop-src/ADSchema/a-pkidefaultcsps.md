# <a name="pki-default-csps-attribute"></a>PKI-default-atributo de CSP

La lista de proveedores de servicios de cifrado para la plantilla de certificado.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | PKI-predeterminado-CSP                            |
| Nombre para mostrar de LDAP | pKIDefaultCSPs                              |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1334                     |
| System-ID-GUID    | 1ef6336e-3b9e-11d2-90cc-00c04fd91ab1        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
