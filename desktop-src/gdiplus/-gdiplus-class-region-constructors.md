# <a name="regionregion-constructors"></a>Constructores Region.Region

En este tema se enumeran los constructores de la [**clase Region.**](/windows/desktop/api/gdiplusheaders/nl-gdiplusheaders-region) Para obtener una lista de clases completa, vea **Region Class**.

### <a name="overload-list"></a>Lista de sobrecarga



| Constructor                                                                 | Descripción                                                                                                                                                                                      |
|:----------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Región (HRGN)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-region(inhrgn))                  | Crea una región idéntica a la región especificada por un identificador para una región GDI.<br/>                                                                                       |
| [**Region(Rect&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-region(inconstrect_))            | Crea una región definida por un rectángulo.<br/>                                                                                                                                      |
| [**Region(RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-region(inconstrectf_))          | Crea una región definida por un rectángulo.<br/>                                                                                                                                      |
| [**Region(BYTE \* , INT)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-region(inconstbyte_inint)) | Crea una región definida por los datos obtenidos de otra región. <br/>                                                                                                               |
| [**Region(GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-region(inconstgraphicspath))        | Crea una región definida por una ruta de acceso (un objeto [**GraphicsPath)**](/windows/desktop/api/gdipluspath/nl-gdipluspath-graphicspath) y tiene un modo de relleno contenido en el **objeto GraphicsPath.**<br/> |
| [**Region()**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-region(constregion_))                           | Crea una región que es infinita. Éste es el constructor predeterminado. <br/>                                                                                                                  |
