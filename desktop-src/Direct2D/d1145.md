# <a name="d1145-print-control-dpi-incorrect"></a>D1145: PPP de control de impresión incorrecto

Impresión de Direct2D: el valor DE PRINT \_ CONTROL \_ PROPERTIES.rasterDPI no es válido.






 

## <a name="possible-causes"></a>Causas posibles

Durante la creación de un control de impresión, este mensaje indica que la aplicación especificó un PPP no válido; en concreto, un valor de PPP inferior a 1,0. Aunque Direct2D no considera que se trata de un error, La impresión de Direct2D se comportará como si la aplicación especificara un valor de PPP de 150,0.

## <a name="possible-fixes"></a>Posibles correcciones

Para solucionar este mensaje, use un valor de PPP de 1.0 o superior al crear un control de impresión.
