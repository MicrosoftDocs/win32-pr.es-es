# <a name="systemsearchcontents"></a>System. Search. Contents

Contenido del elemento. Esta propiedad solo es para restricciones de consulta; no se puede recuperar en el resultado de una consulta. El nombre descriptivo del servicio de indización es ' contents '.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Search.Contents
   shellPKey = PKEY_Search_Contents
   formatID = B725F130-47EF-101A-A5F1-02608C9EEBAC
   propID = 19
   SearchInfo
      InInvertedIndex = true
      IsColumn = false
   typeInfo
      type = String
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
