# <a name="wmdm-glossary"></a>Glosario de WMDM

Para buscar un término en el Glosario, haga clic en la letra del alfabeto que sea la primera letra del término que desea buscar.

También puede leer los términos del glosario en el texto de la ayuda haciendo clic en los vínculos de términos del glosario subrayados. Después de hacer clic en un vínculo de Glosario, el término y la definición del glosario aparecen en una ventana emergente. Para cerrar la ventana, haga clic en cualquier lugar de la pantalla.

<dl> <dt>

<span id="projectid114.active_template_library__atl_"></span><span id="PROJECTID114.ACTIVE_TEMPLATE_LIBRARY__ATL_"></span>**Active Template Library (ATL)**
</dt> <dd>

Colección de clases de plantilla para Microsoft Visual C++. Se utiliza para crear servidores COM al escribir un proveedor de servicios para Windows Media Administrador de dispositivos.

</dd> <dt>

<span id="projectid114.atl"></span><span id="PROJECTID114.ATL"></span>**ATL**
</dt> <dd>

Vea la definición de: [Active Template Library (ATL)](/windows)

</dd> <dt>

<span id="projectid114.gls_authentication"></span><span id="PROJECTID114.GLS_AUTHENTICATION"></span>**autenticación** de
</dt> <dd>

Proceso por el que se comprueba la identidad de un usuario, equipo, proceso u otra entidad validando las credenciales proporcionadas por la entidad. Las formas comunes de credenciales son las firmas digitales, tarjetas inteligentes, datos biométricos y una combinación de nombres de usuario y contraseña.

</dd> <dt>

<span id="projectid114.authorization"></span><span id="PROJECTID114.AUTHORIZATION"></span>**autorización** de
</dt> <dd>

Para conceder acceso a un recurso en un equipo o red, normalmente durante la autenticación de una entidad. Una entidad autenticada podría estar autorizada para distintos tipos de acceso o actividad.

</dd> <dt>

<span id="projectid114.bit_rate"></span><span id="PROJECTID114.BIT_RATE"></span>**velocidad de bits**
</dt> <dd>

Número de bits transferidos por unidad de tiempo, normalmente se expresa en bits por segundo.

</dd> <dt>

<span id="projectid114.buffer"></span><span id="PROJECTID114.BUFFER"></span>**búfer** de
</dt> <dd>

Área de la memoria reservada para almacenar datos temporalmente antes de que los utilice una aplicación o un dispositivo receptor. El almacenamiento en búfer protege contra la interrupción del flujo de datos.

</dd> <dt>

<span id="projectid114.challenge"></span><span id="PROJECTID114.CHALLENGE"></span>**desafío**
</dt> <dd>

Una solicitud de licencia. Un desafío contiene información sobre el equipo del consumidor, una lista de derechos solicitados y otra información sobre el contenido, incluido el encabezado de contenido y el ID. de clave. Este desafío también puede contener un desafío de la versión 1 en caso de que se necesite una licencia de la versión 1.

</dd> <dt>

<span id="projectid114.clear_content"></span><span id="PROJECTID114.CLEAR_CONTENT"></span>**borrar contenido**
</dt> <dd>

Un archivo de Windows Media que no está cifrado.

</dd> <dt>

<span id="projectid114.client"></span><span id="PROJECTID114.CLIENT"></span>**cliente** de
</dt> <dd>

Cualquier equipo o programa que se conecte a otro equipo o programa, o que solicite sus servicios. El cliente también puede hacer referencia al software que permite al equipo o programa establecer la conexión.

</dd> <dt>

<span id="projectid114.content"></span><span id="PROJECTID114.CONTENT"></span>**contenido** de
</dt> <dd>

Audio, vídeo, imágenes, texto o cualquier otra información que está contenida en un archivo multimedia digital o en una secuencia.

</dd> <dt>

<span id="projectid114.decrypt"></span><span id="PROJECTID114.DECRYPT"></span>**descifrar**
</dt> <dd>

Volver a convertir el contenido cifrado en su forma original.

</dd> <dt>

<span id="projectid114.digital_rights_management__drm_"></span><span id="PROJECTID114.DIGITAL_RIGHTS_MANAGEMENT__DRM_"></span>**Administración de derechos digitales (DRM)**
</dt> <dd>

Cualquier tecnología usada para proteger los intereses de los propietarios de contenido y servicios digitales (como los propietarios de los derechos de autor). Normalmente, los destinatarios o usuarios autorizados deben adquirir una licencia para poder usar el contenido.

</dd> <dt>

<span id="projectid114.drm"></span><span id="PROJECTID114.DRM"></span>**DRM**
</dt> <dd>

Consulte la definición de: [Administración de derechos digitales (DRM)](/windows)

</dd> <dt>

<span id="projectid114.encrypt"></span><span id="PROJECTID114.ENCRYPT"></span>**cifrar**
</dt> <dd>

Para disfrazar contenido mediante programación para ocultar su esencia.

</dd> <dt>

<span id="projectid114.firewall"></span><span id="PROJECTID114.FIREWALL"></span>**firewall** de
</dt> <dd>

Solución de seguridad que aísla una parte de una red de otra parte y que sólo deja pasar tráfico de red autorizado según las reglas de filtrado de tráfico.

</dd> <dt>

<span id="projectid114.key"></span><span id="PROJECTID114.KEY"></span>**clave** de
</dt> <dd>

Parte de los datos necesarios para desbloquear un archivo empaquetado de Windows Media. Esta clave se incluye en una licencia independiente.

</dd> <dt>

<span id="projectid114.license"></span><span id="PROJECTID114.LICENSE"></span>**licencia** de
</dt> <dd>

Datos adjuntos al contenido protegido que describe cómo se puede usar el contenido.

</dd> <dt>

<span id="projectid114.microsoft_secure_content_provider__scp_"></span><span id="PROJECTID114.MICROSOFT_SECURE_CONTENT_PROVIDER__SCP_"></span>**Proveedor de contenido seguro (SCP) de Microsoft**
</dt> <dd>

Complemento para Windows Media Administrador de dispositivos que proporciona servicios de administración de derechos digitales (DRM).

</dd> <dt>

<span id="projectid114.microsoft_windows_media_player"></span><span id="PROJECTID114.MICROSOFT_WINDOWS_MEDIA_PLAYER"></span>**Media Player de Microsoft Windows**
</dt> <dd>

Un cliente/control que recibe una secuencia de un servidor de Windows Media o de contenido local para la reproducción. Puede ejecutarse como un programa ejecutable de cliente independiente o estar incrustado en una página web, en un programa de C++ o en un programa de Microsoft Visual Basic que use el control ActiveX del cliente.

</dd> <dt>

<span id="projectid114.object"></span><span id="PROJECTID114.OBJECT"></span>**objeto** de
</dt> <dd>

Colección de propiedades y métodos que sirven para un fin determinado y que se tratan como una unidad.

</dd> <dt>

<span id="projectid114.player"></span><span id="PROJECTID114.PLAYER"></span>**reproductor** de
</dt> <dd>

Programa o control de cliente que recibe contenido multimedia digital transmitido desde un servidor o que se reproduce a partir de archivos locales. Windows Media Player es un ejemplo de un reproductor.

</dd> <dt>

<span id="projectid114.plug_in"></span><span id="PROJECTID114.PLUG_IN"></span>**complemento** de
</dt> <dd>

Componente de software auxiliar que amplía o mejora las características de otro software.

</dd> <dt>

<span id="projectid114.portable_device"></span><span id="PROJECTID114.PORTABLE_DEVICE"></span>**dispositivo portátil**
</dt> <dd>

Dispositivo móvil electrónico que puede intercambiar archivos u otros datos con un PC o un dispositivo. Pocket PC, reproductores de música digital portátiles y Smartphones son ejemplos de dispositivos portátiles.

</dd> <dt>

<span id="projectid114.portable_media_identifier"></span><span id="PROJECTID114.PORTABLE_MEDIA_IDENTIFIER"></span>**identificador multimedia portátil**
</dt> <dd>

Un identificador único para medios de almacenamiento extraíbles, como tarjetas CompactFlash. El identificador está codificado de forma rígida en el medio de almacenamiento y no es modificable.

</dd> <dt>

<span id="projectid114.protect"></span><span id="PROJECTID114.PROTECT"></span>**proteger**
</dt> <dd>

Para cifrar los archivos con una clave y agregar información como la dirección URL de adquisición de licencias.

</dd> <dt>

<span id="projectid114.sdmi"></span><span id="PROJECTID114.SDMI"></span>**SDMI**
</dt> <dd>

Véase la definición de: [iniciativa de música digital segura (SDMI)](/windows)

</dd> <dt>

<span id="projectid114.secure_digital_music_initiative__sdmi_"></span><span id="PROJECTID114.SECURE_DIGITAL_MUSIC_INITIATIVE__SDMI_"></span>**Iniciativa de música digital segura (SDMI)**
</dt> <dd>

Una organización que establece estándares para la música digital segura. Uno de los principales objetivos de SDMI es crear un marco para la reproducción, el almacenamiento y la distribución seguros de música digital.

</dd> <dt>

<span id="projectid114.windows_media_encoder"></span><span id="PROJECTID114.WINDOWS_MEDIA_ENCODER"></span>**Codificador de Windows Media**
</dt> <dd>

Tecnología que comprime audio y vídeo en directo o grabado en una secuencia de Windows Media, que puede distribuirse inmediatamente o guardarse como un archivo de Windows Media para su distribución posterior.

</dd> <dt>

<span id="projectid114.windows_media_file"></span><span id="PROJECTID114.WINDOWS_MEDIA_FILE"></span>**Archivo de Windows Media**
</dt> <dd>

Archivo que contiene audio, vídeo o datos de guión. El contenido del archivo está codificado con uno de los códecs de Windows Media.

</dd> <dt>

<span id="projectid114.windows_media_format"></span><span id="PROJECTID114.WINDOWS_MEDIA_FORMAT"></span>**Formato de Windows Media**
</dt> <dd>

Formato de un archivo multimedia digital o un flujo codificado con códecs de Windows Media.

</dd> <dt>

<span id="projectid114.windows_media_services"></span><span id="PROJECTID114.WINDOWS_MEDIA_SERVICES"></span>**Media Services de Windows**
</dt> <dd>

Un servicio que se usa para transmitir audio y vídeo digital a través de Internet o de una intranet.

</dd> </dl>
