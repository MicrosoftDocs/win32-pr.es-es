# <a name="assisted-telephony-functions"></a>Funciones de telefonía asistida

TAPI admite las siguientes funciones de [telefonía asistida](./assisted-telephony-overview.md) :

-   [**tapiGetLocationInfo**](/windows/desktop/api/Tapi/nf-tapi-tapigetlocationinfo)
-   [**tapiRequestDrop**](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop)
-   [**tapiRequestMakeCall**](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall)
-   [**tapiRequestMediaCall**](tapirequestmediacall.md)

Para una categorización de las funciones de TAPI por nivel de servicio y tarea, consulte [referencia de funciones rápidas de TAPI](tapi-quick-function-reference.md).
