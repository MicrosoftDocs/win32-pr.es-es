# <a name="the-agent-control"></a>Control de agente

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

La referencia al control de agente proporciona acceso a los eventos y la mayoría de los demás objetos que admite el agente de Microsoft. El control de agente también expone directamente su propio conjunto de propiedades.
