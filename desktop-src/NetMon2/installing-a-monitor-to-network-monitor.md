# <a name="installing-a-monitor-to-network-monitor"></a>Instalación de un monitor en Monitor de red

La instalación de un archivo DLL de supervisión es un proceso sencillo. En primer lugar, copie el archivo DLL y el formulario de configuración HTML asociado en el subdirectorio Monitores NPP (por ejemplo, C: \\ Windows \\ monitores \\ Npp system32). \\ Cuando se copia en el directorio, el monitor se reconocerá y estará disponible la próxima vez que se active la Herramienta de control de supervisión o el Servicio de control de supervisión (MCSVC).

Por ejemplo, si la variable de entorno %SystemRoot% es C: Windows, el subdirectorio de supervisión es \\ C: \\ Windows \\ Monitores Npp de System32. \\ \\
