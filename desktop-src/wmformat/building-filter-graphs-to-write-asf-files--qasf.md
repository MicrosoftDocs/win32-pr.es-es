# <a name="building-filter-graphs-to-write-asf-files-qasf"></a>Generar gráficos de filtro para escribir archivos ASF (QASF)

Las aplicaciones basadas en DirectShow normalmente usan uno de los tres tipos básicos de gráficos de filtros al crear contenido basado en Windows Media:

-   Filtre los gráficos para convertir o transcodificar contenido de otro formato en formato de Windows Media.
-   Filtre los gráficos para insertar contenido que no esté basado en Windows Media (formatos de secuencia nativo) en archivos ASF.
-   Filtre los gráficos para capturar datos activos y codificarlos inmediatamente en el formato de Windows Media antes de guardarlos en el disco.

Cada tipo de gráfico de filtro se describe con más detalle en las secciones siguientes.

| [Transcodificar archivos (QASF)](transcoding-files--qasf.md)                                                             | Describe cómo crear gráficos de filtros de transcodificación de archivos.                                               |
| [Insertar formatos de secuencias nativas en archivos ASF (QASF)](inserting-native-stream-formats-into-asf-files--qasf.md)   | Describe cómo colocar cualquier tipo de datos de audio o vídeo comprimidos o no comprimidos en un archivo ASF. |
| [Captura directa desde un dispositivo a un archivo ASF (QASF)](capturing-directly-from-a-device-to-an-asf-file--qasf.md) | Describe cómo crear gráficos de filtros de captura que se generan en un archivo ASF.                             |
