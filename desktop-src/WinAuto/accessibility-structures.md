# <a name="accessibility-structures"></a>Estructuras de accesibilidad

En esta sección se describen las estructuras utilizadas para implementar las características de accesibilidad de Windows.

## <a name="in-this-section"></a>En esta sección



| Estructura                                         | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**ACCESSTIMEOUT**](/windows/win32/api/winuser/ns-winuser-accesstimeout)<br/> | Contiene información sobre el período de tiempo de espera asociado a las características de accesibilidad de Microsoft Win32. <br/> El período de tiempo de espera de accesibilidad es la cantidad de tiempo que debe transcurrir sin la entrada del teclado y del mouse antes de que el sistema operativo desactive automáticamente las características de accesibilidad. El tiempo de espera de accesibilidad está diseñado para equipos compartidos por varios usuarios para que las opciones seleccionadas por un usuario no inpidan a los usuarios posteriores.<br/> Las características de accesibilidad afectadas por el tiempo de espera son las características de FilterKeys (SlowKeys, BounceKeys y RepeatKeys), MouseKeys, ToggleKeys y StickyKeys. El tiempo de espera de accesibilidad también afecta a la configuración del modo de contraste alto.<br/> |
| [**ACTIVAR**](/windows/win32/api/winuser/ns-winuser-filterkeys)<br/>       | Contiene información sobre la característica de accesibilidad de FilterKeys, que permite a un usuario con discapacidades establecer la frecuencia de repetición del teclado (RepeatKeys), el retraso de la aceptación (SlowKeys) y la tasa de rebote (BounceKeys). <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| [**ALTO**](/windows/win32/api/winuser/ns-winuser-highcontrasta)<br/>   | Contiene información sobre la característica de accesibilidad de contraste alto.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**MOUSEKEYS**](/windows/win32/api/winuser/ns-winuser-mousekeys)<br/>         | Contiene información sobre la característica de accesibilidad de MouseKeys. Cuando la característica MouseKeys está activa, el usuario puede utilizar el teclado numérico para controlar el puntero del mouse y hacer clic, hacer doble clic, arrastrar y colocar. Al presionar Bloq Num, el usuario puede alternar el teclado numérico entre el modo de control del mouse y el funcionamiento normal. <br/>                                                                                                                                                                                                                                                                                                                                                                                               |
| [**SERIALKEYS**](/windows/win32/api/winuser/ns-winuser-serialkeysa)<br/>       | Contiene información sobre la característica de accesibilidad SerialKeys, que interpreta los datos de una ayuda de comunicación conectada a un puerto serie como comandos que hacen que el sistema simule la entrada del teclado y del mouse. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**SOUNDSENTRY**](/windows/win32/api/winuser/ns-winuser-soundsentrya)<br/>     | Contiene información acerca de la característica de accesibilidad SoundSentry. Cuando la característica SoundSentry está activada, el equipo solo muestra una indicación visual cuando se genera un sonido.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**ACTIVAR**](/windows/win32/api/winuser/ns-winuser-stickykeys)<br/>       | Contiene información sobre la característica de accesibilidad de StickyKeys. Cuando la característica StickyKeys está activada, el usuario puede presionar una tecla modificadora (Mayús, CTRL o ALT) y, a continuación, otra tecla en secuencia, en lugar de al mismo tiempo, para escribir caracteres desplazados (modificados) y otras combinaciones de teclas.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**ACTIVAR**](/windows/win32/api/winuser/ns-winuser-togglekeys)<br/>       | Contiene información acerca de la característica de accesibilidad de ToggleKeys. Cuando la característica de ToggleKeys está activada, el equipo emite un tono de gran tono siempre que el usuario activa la tecla Bloq Mayús, BLOQ NUM o Bloq Despl y un tono de interrupción baja siempre que el usuario desactiva una de esas claves. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
