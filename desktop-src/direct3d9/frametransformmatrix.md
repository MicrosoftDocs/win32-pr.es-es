# <a name="frametransformmatrix"></a>FrameTransformMatrix

Define una transformación local para un marco (y todos sus objetos secundarios).

``` syntax
template FrameTransformMatrix
{
    < F6F23F41-7686-11cf-8F52-0040333594A3 >
    Matrix4x4 frameMatrix;
} 
```

Donde:

-   frameMatrix: plantilla Matrix4x4. Vea [**Matrix4x4**](matrix4x4.md).
