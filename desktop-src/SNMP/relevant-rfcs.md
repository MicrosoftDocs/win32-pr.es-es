# <a name="relevant-rfcs"></a>RFC pertinentes

Los estándares TCP/IP se definen en las solicitudes de comentarios (RFC), publicadas por Internet Engineering Task Force (IETF). En la tabla siguiente se enumeran las RFC que son relevantes para SNMP.



| Número RFC | Title                                                                                                           |
|------------|-----------------------------------------------------------------------------------------------------------------|
| 1155       | "Estructura e identificación de la información de administración para Internet basadas en TCP/IP". Define SMI. MIB.        |
| 1157       | "Protocolo simple de administración de redes (SNMP)". Define el propio SNMP.                                          |
| 1213       | "Base de información de administración para la administración de redes de Internet basadas en TCP/IP: MIB-II". Define MIB \_ II. MIB. |
| 1901       | "Introducción a SNMPv2 basado en la comunidad"                                                                        |
| 1902       | "Estructura de la información de administración para la versión 2 del Protocolo simple de administración de redes (SNMPv2)"          |
| 1903       | "Convenciones de texto para la versión 2 del Protocolo simple de administración de redes (SNMPv2)"                          |
| 1904       | "Instrucciones de conformidad para la versión 2 del Protocolo simple de administración de redes (SNMPv2)"                       |
| 1905       | "Operaciones de protocolo para la versión 2 del Protocolo simple de administración de redes (SNMPv2)"                          |
| 1906       | "Asignaciones de transporte para la versión 2 del Protocolo simple de administración de redes (SNMPv2)"                           |
| 1907       | "Base de información de administración para la versión 2 del Protocolo simple de administración de redes (SNMPv2)"                  |
| 1908       | "Coexistencia entre la versión 1 y la versión 2 del marco de administración de redes estándar de Internet"             |
| 2089       | "Asignación de V2ToV1 de SNMPv2 a SNMPv1 en un agente SNMP de dos idiomas"                                              |
