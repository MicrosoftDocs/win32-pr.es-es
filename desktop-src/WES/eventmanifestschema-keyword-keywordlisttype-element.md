# <a name="keyword-keywordlisttype-element"></a>palabra clave (KeywordListType) (elemento)

Define una palabra clave que identifica una categoría de eventos. Una palabra clave es una etiqueta que se adjunta a un evento para agrupar eventos en función de su uso.

``` syntax
<xs:element name="keyword"
    type="KeywordType"
 />
```

El elemento de **palabra clave** se define mediante el tipo complejo de [**KeywordListType**](eventmanifestschema-keywordlisttype-complextype.md) .
