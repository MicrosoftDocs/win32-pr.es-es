# <a name="itterminal-mspi"></a>ITTerminal (MSPI)

La [**interfaz ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) es la interfaz base para un [objeto terminal](terminal-object.md). Proporciona métodos para obtener información, como la clase terminal y los medios admitidos.

La [**interfaz ITTerminal**](/windows/win32/api/tapi3if/nn-tapi3if-itterminal) se implementa mediante un MSP y no está disponible si no hay ningún proveedor de servicios multimedia asociado a la dirección. Consulte **ITTerminal en la** sección Interfaz de MSP para obtener más información sobre esta interfaz.
