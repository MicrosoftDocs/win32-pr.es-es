# <a name="file-server-api-support"></a>Compatibilidad con la API del servidor de archivos

## <a name="platform"></a>Plataforma

**Servidores:** Windows Server 2012 

## <a name="description"></a>Descripción

La documentación sobre qué API se admiten para el protocolo Bloque de mensajes del servidor 2.2 (SMB 2.2) y los sistemas de archivos de volúmenes compartidos resistentes o de clúster (ReFS/CSVFS) estarán disponibles para los desarrolladores en el recurso que se muestra a continuación. La documentación se divide en estas categorías:

-   Funciones de administración de archivos
-   Funciones de administración de directorios
-   Funciones de administración de volúmenes
-   Funciones de seguridad
-   Códigos de compatibilidad de archivos y directorios
-   Códigos de control de volumen
-   Archivos asignados a memoria

## <a name="usage"></a>Uso

Algunas API no se admiten con todos los protocolos o en todos los escenarios, mientras que otras API han quedado en desuso en Windows Server 2012. Revise la lista de API que se admiten para garantizar la compatibilidad continua con las aplicaciones.
