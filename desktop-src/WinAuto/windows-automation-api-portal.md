# <a name="windows-accessibility-api-overview"></a>Información general de la API de accesibilidad de Windows

En esta sección se proporciona información general de alto nivel y referencia de API detallada para Microsoft Windows Automation API 3,0, que incluye la implementación de Windows de las [Especificaciones de automatización](./uiauto-specandcommunitypromise.md)de la interfaz de usuario de Microsoft y Microsoft Active Accessibility (característica heredada introducida en Windows 95 como complemento de plataforma).

Se describen las similitudes y diferencias entre Microsoft Active Accessibility y la automatización de la interfaz de usuario, los componentes y las características que permiten que las dos tecnologías funcionen juntas y se proporcionan directrices para elegir qué tecnología se debe implementar para su escenario específico.
