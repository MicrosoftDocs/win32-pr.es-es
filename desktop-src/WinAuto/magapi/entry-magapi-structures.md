# <a name="magnification-api-structures"></a>Estructuras de API de ampliación

Esta sección contiene información de referencia sobre las estructuras de la API de ampliación.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|:---|:---|
| [**MAGCOLOREFFECT**](/windows/win32/api/magnification/ns-magnification-magcoloreffect)<br/>  | Describe una matriz de transformación de color que un control ampliador utiliza para aplicar un efecto de color al contenido de la pantalla ampliada.<br/> |
| [***MAGIMAGEHEADER** _](/windows/win32/api/magnification/ns-magnification-magimageheader)<br/> | _ * En desuso en Windows 7 y versiones posteriores * *<br/>Describe un formato de imagen. |
| [**MAGTRANSFORM**](/windows/win32/api/magnification/ns-magnification-magtransform)<br/> | Describe una matriz de transformación que un control ampliador utiliza para ampliar el contenido de la pantalla.<br/> |
