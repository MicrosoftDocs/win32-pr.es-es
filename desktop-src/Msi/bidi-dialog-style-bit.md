# <a name="bidi-dialog-style-bit"></a>Bit de estilo de cuadro de diálogo BiDi

Se trata de una combinación del orden de lectura de derecha a izquierda [RTLRO](rtlro-dialog-style-bit.md), [RightAligned](rightaligned-dialog-style-bit.md)y los [bits de estilo del cuadro de diálogo LeftScroll](leftscroll-dialog-style-bit.md).

## <a name="value"></a>Value



| Decimal | Hexadecimal | Constante                                                                                                                                       |
|---------|-------------|------------------------------------------------------------------------------------------------------------------------------------------------|
| 896     | 0x00000380  | **msidbDialogAttributesBiDi**  =  **msidbDialogAttributesRTLRO** \| **msidbDialogAttributesRightAligned** \| **msidbDialogAttributesLeftScroll** |
