# <a name="authentication-services"></a>Servicios de autenticación

La interfaz del proveedor de compatibilidad para seguridad (SSPI) proporciona los servicios de autenticación subyacentes para RPC. Por lo tanto, cuando la aplicación especifica un servicio de autenticación, selecciona un SSP. Para obtener una lista de los SSP que SPPI admite actualmente, vea [Authentication-Service constantes](authentication-service-constants.md). Para obtener más información, vea [interfaz del proveedor de compatibilidad para seguridad (SSPI)](security-support-provider-interface-sspi-.md).

> [!Note]  
> Los términos proveedor de seguridad y servicio de autenticación son intercambiables.
