# <a name="winsock-enumerations"></a>Enumeraciones de Winsock

En la lista siguiente se proporcionan descripciones concisas de cada enumeración Winsock. Para obtener información adicional sobre cualquier enumeración, haga clic en el nombre de la enumeración.

| Enumeración | Descripción |
|-|-|
| [**\_Estado del \_ desencadenador de canal de control \_**](/windows/win32/api/mstcpip/ne-mstcpip-control_channel_trigger_status) | Especifica el estado de una consulta para la configuración de transporte de la **funcionalidad de \_ notificación en tiempo \_ \_ real** para un socket TCP que se usa con [**ControlChannelTrigger**](/uwp/api/Windows.Networking.Sockets.ControlChannelTrigger) para recibir notificaciones de red en segundo plano en una aplicación Windows Runtime. |
| [**eWINDOW \_ \_ método Advance**](/windows/win32/api/wsrm/ne-wsrm-ewindow_advance_method) | Especifica el modo Advance de ventana que se usa para la multidifusión confiable. |
| [**Garanti**](./guarantee-2.md) | Ya no se usa. |
| [**\_tipo de modo de MULTIdifusión \_**](/windows/win32/api/ws2ipdef/ne-ws2ipdef-multicast_mode_type) | Especifica el modo de filtro para las direcciones del grupo de multidifusión. |
| [**\_nivel de proveedor de nestado \_**](/windows/win32/api/nsemail/ne-nsemail-napi_provider_level) | Especifica el nivel de entidad de proveedor de un \_ proveedor de espacios de nombres de correo electrónico NS para un dominio determinado. |
| [**\_tipo de proveedor nestado \_**](/windows/win32/api/nsemail/ne-nsemail-napi_provider_type) | Especifica el tipo de hospedaje esperado para un proveedor de espacios de nombres. |
| [**\_tipo de \_ finalización de notificación de Rio \_**](/windows/win32/api/mswsock/ne-mswsock-rio_notification_completion_type) | Especifica el tipo de notificaciones de la cola de finalización que se va a usar con la función [**RIONotify**](/windows/win32/api/mswsock/nc-mswsock-lpfn_rionotify) al enviar o recibir datos con las extensiones de e/s registradas de Winsock. |
| [**\_Protocolo de seguridad de socket \_**](/windows/win32/api/mstcpip/ne-mstcpip-socket_security_protocol) | Indica el tipo de protocolo de seguridad que se va a utilizar en un socket. |
| [**\_tipo de uso de socket \_**](/windows/win32/api/Mstcpip/ne-mstcpip-socket_usage_type) | Se usa para especificar el tipo de uso para el socket. |
| [**TCPSTATE**](/windows/win32/api/mstcpip/ne-mstcpip-tcpstate) | Indica los Estados posibles de una conexión TCP (Protocolo de control de transmisión). |
| [**WSAECOMPARATOR**](/windows/win32/api/Winsock2/ne-winsock2-wsaecomparator) | Se usa para la semántica de comparación de la versión. |
| [**\_tipo de \_ información del proveedor de WSC \_**](/windows/win32/api/ws2spi/ne-ws2spi-wsc_provider_info_type) | Especifica la clase de información de un protocolo de servicio por niveles (LSP). |
