# <a name="cmspcallbase-mspi-methods-implemented"></a>Métodos CMSPCallBase MSPI implementados



| Métodos auxiliares de CMSPCallBase MSPI                             | Descripción                                                                                                                                                                           |
|--------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateStream (**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-createstream)         | Crea el flujo multimedia para una nueva llamada. Adquiere el bloqueo de la función.                                                                                                           |
| [**EnumerateStreams**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-enumeratestreams) | Enumera las secuencias disponibles actualmente. Adquiere el bloqueo de la función.                                                                                                            |
| [**obtener \_ secuencias**](/windows/win32/api/tapi3if/nf-tapi3if-itstreamcontrol-get_streams)          | Crea una colección de secuencias disponibles actualmente. Se proporciona para las aplicaciones cliente de automatización, como las escritas en Microsoft Visual Basic. Adquiere el bloqueo de la función. |
