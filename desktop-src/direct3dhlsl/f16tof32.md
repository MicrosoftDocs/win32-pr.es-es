# <a name="f16tof32-function"></a>f16tof32 función)

Convierte el float16 almacenado en la mitad inferior del uint en un valor float.

## <a name="syntax"></a>Sintaxis

``` syntax
float f16tof32(
  in uint value
);
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

*valor* \[ de de\]
</dt> <dd>

Tipo: **uint**

Valor de entrada.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Tipo: **float**

El valor convertido.

## <a name="remarks"></a>Observaciones

También están disponibles las siguientes versiones sobrecargadas:

``` syntax
float2 f16tof32(uint2 value);
float3 f16tof32(uint3 value);
float4 f16tof32(uint4 value);
```

### <a name="minimum-shader-model"></a>Modelo de sombreador mínimo

Esta función se admite en los siguientes modelos de sombreador.



| Modelo de sombreador                                                        | Compatible |
|---------------------------------------------------------------------|-----------|
| Modelos de sombreador [modelo 4](dx-graphics-hlsl-sm4.md) y versiones posteriores | sí       |



 

Aunque esta función está disponible en los modelos de sombreador [modelo 4](dx-graphics-hlsl-sm4.md) y superior, ya que se emula en 4,0 y 4,1, es menos perfomant en estos modelos de sombreador que en el modelo de [sombreador 5](d3d11-graphics-reference-sm5.md).

Esta función se admite en los siguientes tipos de sombreadores:



| Vértice | Casco | Dominio | Geometría | Píxel | Compute |
|--------|------|--------|----------|-------|---------|
| x      | x    | x      | x        | x     | x       |
