# <a name="finding-specific-information-in-the-routing-table"></a>Buscar información específica en la tabla de enrutamiento

Los clientes deben poder localizar información específica en la tabla de enrutamiento, en lugar de ser necesario para enumerar todos los datos. La API RTMv2 ofrece la posibilidad de buscar destinos, rutas y próximos saltos específicos en función de determinados criterios.
