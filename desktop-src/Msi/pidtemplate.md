# <a name="pidtemplate-property"></a>PidTemplate, propiedad

La **propiedad PIDTemplate** contiene la cadena que se usa como plantilla para la [**propiedad PIDKEY.**](pidkey.md) Para obtener la sintaxis usada en la plantilla, vea el tipo de control [MaskedEdit.](maskededit-control.md) [La acción ValidateProductID](validateproductid-action.md) usa este valor para validar el id. de producto.

Windows Instalador en Windows Server 2003 o Windows XP.
