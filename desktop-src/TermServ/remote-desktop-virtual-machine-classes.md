# <a name="remote-desktop-virtual-machine-classes"></a>Clases de máquinas virtuales Escritorio remoto

El proveedor WMI de máquinas virtuales Escritorio remoto proporciona las siguientes clases.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**Win32 \_ TSVirtualDesktopServerSettings**](win32-tsvirtualdesktopserversettings.md)
</dt> <dd>

Contiene información de configuración para un servidor de Host de virtualización de Escritorio remoto (host de virtualización de escritorio remoto).

</dd> <dt>

[**Win32 \_ TSVm**](win32-tsvm.md)
</dt> <dd>

Representa una máquina virtual Escritorio remoto.

</dd> <dt>

[**Win32 \_ TSVmMetadataAssociation**](win32-tsvmmetadataassociation.md)
</dt> <dd>

Representa una asociación entre un Escritorio remoto máquina virtual y sus propiedades de metadatos.

</dd> <dt>

[**Win32 \_ TSVmMetadataItem**](win32-tsvmmetadataitem.md)
</dt> <dd>

Representa un elemento de metadatos para una máquina virtual Escritorio remoto.

</dd> <dt>

[**Win32 \_ RdvhManagement**](win32-rdvhmanagement.md)
</dt> <dd>

Describe un servicio de administración de Escritorio remoto virtual (RDVH).

</dd> </dl>
