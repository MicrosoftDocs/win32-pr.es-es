# <a name="swbemrefreshercount-property"></a>SWbemRefresher. Count (propiedad)

La propiedad **Count** del objeto [**SWbemRefresher**](swbemrefresher.md) contiene el número de elementos de una colección [**SWbemRefresher**](swbemrefresher.md) . Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemRefresher.Count As Integer
```
