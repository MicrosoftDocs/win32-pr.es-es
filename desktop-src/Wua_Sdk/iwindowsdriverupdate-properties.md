# <a name="iwindowsdriverupdate-properties"></a>Propiedades de IWindowsDriverUpdate

La [**interfaz IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) define las siguientes propiedades.



| Propiedad                                                                                                    | Descripción                                                                                           |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------|
| [**DeviceProblemNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_deviceproblemnumber)[**Address**](/windows/desktop/api/Wuapi/nf-wuapi-iwebproxy-get_address) | Obtiene el número de problema del dispositivo que coincide con la actualización Windows controlador.                     |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_devicestatus)                                                   | Obtiene el estado del dispositivo que coincide con la actualización Windows controlador.                             |
| [**DriverClass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverclass)                                                     | Obtiene la clase del controlador Windows actualización.                                                          |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverhardwareid)                                           | Obtiene el identificador de hardware o el identificador compatible que la actualización Windows controlador debe coincidir para poder instalarse.    |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermanufacturer)                                       | Obtiene el nombre invariable del idioma del fabricante de la actualización Windows controlador.                    |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_drivermodel)                                                     | Obtiene el nombre del modelo invariable del idioma del dispositivo para el que se Windows la actualización del controlador. |
| [**DriverProvider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driverprovider)                                               | Obtiene el nombre invariable del idioma del proveedor de la Windows actualización del controlador.                        |
| [**DriverVerDate**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate-get_driververdate)                                                 | Obtiene la fecha de versión del controlador de Windows actualización del controlador.                                            |
