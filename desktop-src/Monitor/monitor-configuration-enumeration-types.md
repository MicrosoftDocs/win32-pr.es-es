# <a name="monitor-configuration-enumeration-types"></a>Supervisar tipos de enumeración de configuración

Las funciones de configuración del monitor usan las enumeraciones siguientes.



| Tipo de enumeración                                                    | Descripción                                                                                      |
|---------------------------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**MC \_ COLOR \_ TEMPERATURE**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_color_temperature)              | Describe la temperatura del color de un monitor.                                                         |
| [**TIPO \_ DE TECNOLOGÍA MC DISPLAY \_ \_**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_display_technology_type) | Identifica las tecnologías de visualización del monitor.                                                         |
| [**TIPO \_ DE UNIDAD \_ MC**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_drive_type)                            | Especifica si se debe establecer u obtener la unidad roja, verde o azul de un monitor.                           |
| [**TIPO \_ DE GANANCIA DE \_ MC**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_gain_type)                              | Especifica si se debe obtener o establecer la ganancia roja, verde o azul de un monitor.                            |
| [**TIPO \_ DE POSICIÓN DE \_ MC**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_position_type)                      | Especifica si se debe obtener o establecer la posición vertical u horizontal del área de presentación de un monitor. |
| [**TIPO \_ DE TAMAÑO DE \_ MC**](/windows/desktop/api/HighLevelMonitorConfigurationAPI/ne-highlevelmonitorconfigurationapi-mc_size_type)                              | Especifica si se debe obtener o establecer el ancho o alto del área de presentación de un monitor.                 |
| [**TIPO \_ DE CÓDIGO DE MC \_ \_ VCP**](/windows/desktop/api/LowLevelMonitorConfigurationAPI/ne-lowlevelmonitorconfigurationapi-mc_vcp_code_type)                     | Describe un tipo de código Panel de control virtual (VCP).                                               |
