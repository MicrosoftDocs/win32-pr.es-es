# <a name="messages-touch-input"></a>Mensajes (entrada táctil)

Esta sección contiene mensajes para la entrada multitáctil.

Los mensajes siguientes se usan para la entrada multitouch.



| Message                           | Descripción                                                                                                         |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**WM \_ TOUCH**](wm-touchdown.md) | Indica que uno o varios puntos táctiles, como un dedo o un lápiz, han tocado una superficie de digitalizador táctil. |
