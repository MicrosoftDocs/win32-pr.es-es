# <a name="prov_rsa_full"></a>aprovisionamiento \_ completo de RSA \_

El \_ tipo de \_ proveedor completo de RSA es compatible con las [*firmas digitales*](../secgloss/d-gly.md) y el cifrado de datos. Se considera un CSP de uso general. El algoritmo de clave pública RSA se utiliza para todas las operaciones de clave pública.

## <a name="algorithms-supported"></a>Algoritmos admitidos

Para obtener descripciones de cada uno de estos algoritmos, vea el glosario.



| Propósito      | Algoritmos admitidos                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Intercambio de claves | [*RSA*](../secgloss/r-gly.md)                                                                       |
| Firma    | RSA                                                                                                                               |
| Cifrado   | [*RC2*](../secgloss/r-gly.md)[*RC4*](../secgloss/r-gly.md)<br/> |
| Aplicación de algoritmo hash      | [*MD5*](../secgloss/m-gly.md)[*Sha*](../secgloss/s-gly.md)<br/> |



 

## <a name="related-documentation"></a>Documentación relacionada

Este tipo de proveedor está definido por Microsoft y RSA Data Security. Se describe en los siguientes documentos:

-   RSA Laboratories, estándares de criptografía de clave pública, RSA Data Security, noviembre de 1993.
