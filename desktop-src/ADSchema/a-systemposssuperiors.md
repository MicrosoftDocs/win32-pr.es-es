Lista de clases que pueden contener esta clase. Esta lista solo puede ser modificada por el sistema.

|-------------------|-----------------------------------------------------------------------------|
| CN                | System-Poss-superior                                                       |
| Nombre para mostrar de LDAP | systemPossSuperiors                                                         |
| Tamaño              | \-                                                                          |
| Actualizar privilegio  | Administrador de esquema                                                        |
| Frecuencia de actualización  | Cuando se crea la clase o se agrega un nuevo superior posible a la clase. |
| Attribute-Id      | 1.2.840.113556.1.4.195                                                      |
| System-ID-GUID    | bf967a47-0de6-11d0-a285-00aa003049e2                                        |
