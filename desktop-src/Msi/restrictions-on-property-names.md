# <a name="restrictions-on-property-names"></a>Restricciones en los nombres de propiedad

Todos los nombres de propiedad deben seguir estas restricciones.

-   Un nombre de propiedad es [un identificador](identifier.md), que es una cadena de texto que debe comenzar con una letra o un carácter de subrayado. Los identificadores y los nombres de propiedad pueden contener letras, números, caracteres de subrayado o puntos; pero no puede comenzar con un número o un punto.
-   [Los nombres de](public-properties.md) propiedad pública no pueden contener letras minúsculas.
-   [Los nombres de](private-properties.md) propiedad privada deben contener letras minúsculas.
-   Los nombres de propiedad con el prefijo % representan variables de entorno de usuario y sistema. Nunca se introducen en la [tabla Property](property-table.md). La configuración permanente de las variables de entorno solo se puede modificar mediante la [tabla de entorno](environment-table.md).
