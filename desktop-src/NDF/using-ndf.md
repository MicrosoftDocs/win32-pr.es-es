# <a name="using-ndf"></a>Uso de NDF

El marco de diagnóstico de redes (NDF) se incluye como parte de Windows Vista, Windows Server 2008 y versiones posteriores. Los desarrolladores de software pueden usar NDF para diagnosticar y reparar problemas de red que puedan estar causando un error en las aplicaciones. Los desarrolladores pueden usar las API proporcionadas por Microsoft que conectan sus aplicaciones a NDF. En algunos casos, los desarrolladores también pueden crear extensiones de clase auxiliar NDF para ayudar a diagnosticar problemas específicos de su software de red.
