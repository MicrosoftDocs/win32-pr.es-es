# <a name="xmfloat2-operators"></a>Operadores XMFLOAT2

La estructura [**XMFLOAT2**](/windows/win32/api/directxmath/ns-directxmath-xmfloat2) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                               | Descripción                                                                                                         |
|-------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**operador =**](/windows/win32/api/directxmath/nf-directxmath-xmfloat2-operator-assign)<br/> | Asigna los datos del componente de vector de una instancia de `XMFLOAT2` a la instancia actual de `XMFLOAT2` .<br/> |
