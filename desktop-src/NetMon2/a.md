# <a name="a-network-monitor"></a>A (Monitor de red)

<dl> <dt>

<span id="_netmon_address_gly"></span><span id="_NETMON_ADDRESS_GLY"></span>**Dirección**
</dt> <dd>

Identificador de un nodo de red que diferencia un nodo de otro en una red. También se conoce como *dirección de red* o *dirección Mac*. En Microsoft Systems Management Server (SMS), se usa una dirección para conectar un sitio y sistema de sitio. Un remitente usa una dirección específica para enviar instrucciones y datos a otros sitios.

</dd> <dt>

<span id="_netmon_address_pair_gly"></span><span id="_NETMON_ADDRESS_PAIR_GLY"></span>**par de direcciones**
</dt> <dd>

Direcciones de origen y de destino de una transmisión de datos. El destino puede ser una sola dirección, multidifusión o difusión. En los datos capturados, la dirección de origen es una dirección específica. En las especificaciones de filtro, la dirección de origen puede ser una dirección específica, o cualquier, pero no puede ser de difusión o multidifusión.

</dd> </dl>
