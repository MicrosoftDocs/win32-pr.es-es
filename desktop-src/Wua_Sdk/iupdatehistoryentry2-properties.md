# <a name="iupdatehistoryentry2-properties"></a>Propiedades de IUpdateHistoryEntry2

La interfaz [**IUpdateHistoryEntry2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatehistoryentry2) define la propiedad siguiente.



| Propiedad                                              | Descripción                                                            |
|-------------------------------------------------------|------------------------------------------------------------------------|
| [**Categorías**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatehistoryentry2-get_categories) | Obtiene una colección de las categorías de actualización a las que pertenece una actualización. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz:

-   [**IUpdateHistoryEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatehistoryentry)
