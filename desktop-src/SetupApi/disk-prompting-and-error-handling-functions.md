# <a name="disk-prompting-and-error-handling-functions"></a>Funciones de solicitud de disco y control de errores

Las siguientes funciones proporcionan la solicitud de disco y una interfaz de usuario de control de errores.



| Función                                         | Descripción                                                                            |
|--------------------------------------------------|----------------------------------------------------------------------------------------|
| [**SetupCopyError**](/windows/desktop/api/Setupapi/nf-setupapi-setupcopyerrora)         | Genera un cuadro de diálogo que informa al usuario de un error de copia.                          |
| [**SetupDeleteError**](/windows/desktop/api/Setupapi/nf-setupapi-setupdeleteerrora)     | Genera un cuadro de diálogo que informa al usuario de un error de eliminación.                      |
| [**SetupPromptForDisk**](/windows/desktop/api/Setupapi/nf-setupapi-setuppromptfordiska) | Genera un cuadro de diálogo que solicita al usuario un disco multimedia o la ubicación del archivo de origen. |
| [**SetupRenameError**](/windows/desktop/api/Setupapi/nf-setupapi-setuprenameerrora)     | Genera un cuadro de diálogo que informa al usuario de un error de cambio de nombre.                        |
