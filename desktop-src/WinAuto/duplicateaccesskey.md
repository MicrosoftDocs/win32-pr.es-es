# <a name="duplicateaccesskey"></a>DuplicateAccessKey

## <a name="text"></a>Texto

Hay un elemento de Automation relacionado que tiene la misma clave de acceso o tecla de aceleración.

## <a name="type"></a>Tipo

Error

## <a name="description"></a>Descripción

La clave de acceso/acelerador asignada al elemento también se asigna a otro elemento en el destino de comprobación.

Este problema provoca problemas para las personas que dependen del teclado para la navegación porque se podrían invocar varios elementos o centrarse en el ciclo con el mismo método abreviado o tecla de aceleración.
