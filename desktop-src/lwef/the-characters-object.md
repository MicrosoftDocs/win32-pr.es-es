# <a name="the-characters-object"></a>El objeto Characters

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

La aplicación cliente puede admitir uno o más caracteres. Además, puede compartir un carácter entre varias aplicaciones. El agente de Microsoft define el objeto de [**caracteres**](/windows/desktop/lwef/the-characters-object) como una colección de caracteres. Para tener acceso a un carácter, cargue los datos del carácter en la colección de [**caracteres**](/windows/desktop/lwef/the-characters-object) y especifique que el elemento de la colección use los métodos y las propiedades compatibles con ese carácter.

-   [Métodos del objeto Characters](characters-object-methods.md)
-   [Métodos del objeto Characters](character-object-methods.md)
-   [Propiedades del objeto Characters](character-object-properties.md)
