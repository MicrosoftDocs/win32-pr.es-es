# <a name="cooker_variance"></a>VARIANZA DE LA \_ COCINA

La fórmula de tipo de contador COOKER VARIANCE proporciona variabilidad que se usa para caracterizar la dispersión de un conjunto de observaciones sin procesar de una propiedad en una instancia \_ [**de \_ PerfRawData de Win32.**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) La varianza se calcula dividiendo la suma del cuadrado de la desviación de la media de cada contador por el número de contadores. En otras palabras, la varianza es el promedio de las desviaciones al cuadrado de la media de cada contador. Este tipo de contador solo se define en WMI y no está disponible para las tecnologías de supervisión de rendimiento, como contadores de rendimiento [versión 6.0.](/windows/desktop/PerfCtrs/performance-counters-portal)

Para obtener más información sobre la fórmula de tipo de contador, vea [Tipos de contador](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Para obtener más información sobre los proveedores de alto rendimiento y el scripting, vea [Tipos de contadores de rendimiento wmi.](wmi-performance-counter-types.md)
