# <a name="imfmediasourceextensiongetduration-method"></a>IMFMediaSourceExtension:: GetDuration (método)

Obtiene la duración del origen de los medios en unidades de 100-nanosegundos.

## <a name="syntax"></a>Sintaxis


```C++
double GetDuration();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

La duración del origen del medio en unidades de 100-nanosegundos.
