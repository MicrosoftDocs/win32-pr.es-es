# <a name="mssfu-30-key-attributes-attribute"></a>atributo msSFU-30-Key-Attributes

Contiene los nombres de los atributos que el servidor NIS usa como claves para un mapa de búsqueda.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | msSFU-30-clave-atributos                     |
| Nombre para mostrar de LDAP | msSFU30KeyAttributes                        |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.6.18.1.301                 |
| System-ID-GUID    | 32ecd698-ce9e-4894-A134-7ad76b082e83        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
