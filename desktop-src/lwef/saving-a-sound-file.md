# <a name="saving-a-sound-file"></a>Guardar un archivo de sonido

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Cuando esté listo para guardar el archivo de sonido, elija el comando **Guardar** en el menú **archivo** o en la barra de herramientas del editor. El editor muestra el cuadro de diálogo **Guardar como** y propone un nombre y un tipo de archivo predeterminado en función de si ha generado información lingüística para el archivo. Si guarda el archivo como un archivo de sonido (. wav), el editor solo guarda los datos de audio. Si guarda la información del archivo como un archivo de sonido lingüísticamente mejorado (. LWV), la información de la palabra y el fonema se incluye automáticamente como parte de un archivo de sonido modificado. Una vez que haya confirmado o editado el nombre, la ubicación, el tipo de archivo y el formato, elija el botón **Guardar** .

![Captura de pantalla que muestra el cuadro de diálogo ' guardar como ' con un ' nombre de archivo ', ' guardar como tipo ', ' formato ' y ' atributos ' seleccionados.](images/f7listsave.gif)

Si desea guardar un archivo de sonido con un nuevo nombre, una ubicación diferente o un formato diferente, elija el comando **Guardar como** en el menú **archivo** . Cuando aparezca el cuadro de diálogo **Guardar como** , escriba el nuevo nombre de archivo y haga clic en el botón **Guardar** .

También puede guardar una parte del archivo de sonido. Por ejemplo, puede que desee guardar el archivo sin un silencio excesivo al principio o al final. En la representación de audio, seleccione la parte del archivo que desea guardar y elija **Guardar selección como** en el menú **archivo** . El comando solo está habilitado cuando se tiene una selección en la **representación de audio**.
