# <a name="d3dx_saturate_float-function"></a>Función D3DX \_ Saturate \_ FLOAT

Recupera el valor saturado del FLOAT especificado.

## <a name="syntax"></a>Sintaxis

``` syntax
FLOAT D3DX_Saturate_FLOAT(
   FLOAT _V
);
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

*\_V* 
</dt> <dd>

Valor que se saturará.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Valor saturado.
