<span id="upnp.c_1_gly"></span><span id="UPNP.C_1_GLY"></span>**punto de control**
</dt> <dd>

Un punto de control es un controlador (por ejemplo, una aplicación) que recupera descripciones de dispositivos y servicios, envía acciones a servicios, consulta variables de estado de servicio y recibe eventos de servicios. Las aplicaciones que permiten a un usuario usar o ejecutar un dispositivo, como un dispositivo de reproductor de CD, se consideran puntos de control.
