# <a name="opm-enumerations"></a>Enumeraciones de OPM

El [Administrador de protección de salida](output-protection-manager.md) (OPM) utiliza las siguientes constantes y tipos de enumeración.



| Enumeración                                                                    | Descripción                                                                                                                                                                             |
|--------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Marcas de tipo de bus OPM**](opm-bus-type-flags.md)                               | Especifica el tipo de bus de e/s utilizado por el adaptador de gráficos.                                                                                                                             |
| [**Marcas de tipo de conector de OPM**](opm-connector-type-flags.md)                   | Especifica el tipo de conector físico para una salida de vídeo.                                                                                                                            |
| [**Marcas de tipo de protección OPM**](opm-protection-type-flags.md)                 | Especifica los mecanismos de protección de salida para el administrador de protección de salida (OPM).                                                                                                             |
| [**Marcas de estado de OPM**](opm-status-flags.md)                                   | Especifica el estado de una sesión de OPM.                                                                                                                                                 |
| [**nivel de protección de OPM \_ ACP \_ \_**](/windows/desktop/api/opmapi/ne-opmapi-opm_acp_protection_level)                | Especifica el nivel de protección para la protección contra copia analógica (ACP).                                                                                                                        |
| [**\_nivel de \_ protección de DPCP de OPM \_**](/windows/desktop/api/opmapi/ne-opmapi-opm_dpcp_protection_level)              | Especifica el nivel de protección de DisplayPort Content Protection (DPCP).                                                                                                               |
| [**\_nivel de \_ protección de HDCP de OPM \_**](/windows/desktop/api/opmapi/ne-opmapi-opm_hdcp_protection_level)              | Especifica el nivel de protección de High-Bandwidth Content Protection digital (HDCP).                                                                                                    |
| [**Relación de aspecto de imagen de OPM \_ \_ \_ \_ EN300294**](/windows/desktop/api/opmapi/ne-opmapi-opm_image_aspect_ratio_en300294) | Especifica la relación de aspecto de ETSI EN 300 294.                                                                                                                                         |
| [**\_semántica de \_ salida de vídeo OPM \_**](/windows/desktop/api/opmapi/ne-opmapi-opm_video_output_semantics)            | Especifica si la interfaz [**IOPMVideoOutput**](/windows/desktop/api/opmapi/nn-opmapi-iopmvideooutput) usará la semántica de administrador de protección de salida (OPM) o la semántica del Protocolo de protección de la salida (COPP). |
