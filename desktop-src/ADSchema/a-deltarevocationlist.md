# <a name="delta-revocation-list-attribute"></a>Atributo de lista de revocación Delta

Lista de certificados que se han revocado desde la última actualización diferencial.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | Delta-revocación-lista                                 |
| Nombre para mostrar de LDAP | deltaRevocationList                                   |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 2.5.4.53                                              |
| System-ID-GUID    | 167757b5-47f3-11d1-a9c3-0000f80367c1                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
