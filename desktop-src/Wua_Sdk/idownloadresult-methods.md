# <a name="idownloadresult-methods"></a>Métodos IDownloadResult

La interfaz [**IDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-idownloadresult) define el método siguiente.



| Método                                                     | Descripción                                                                                                                              |
|------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetUpdateResult**](/windows/desktop/api/Wuapi/nf-wuapi-idownloadresult-getupdateresult) | Devuelve una interfaz [**IUpdateDownloadResult**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatedownloadresult) que contiene información de descarga para la actualización especificada. |
