# <a name="about-the-control-point-api"></a>Acerca de la API de punto de control

La tecnología UPnP permite redes dinámicas de dispositivos inteligentes, equipos y otros puntos de control. La red dinámica está disponible en una gran variedad de medios físicos. La implementación de Microsoft de la API de punto de control con tecnología UPnP está formada por un conjunto de objetos COM que se usan para escribir aplicaciones y páginas web que buscan y controlan dispositivos basados en UPnP. Por ejemplo, un VCR compatible con UPnP puede controlarse desde una página web diseñada específicamente para controlarlo. Para ampliar este concepto, los desarrolladores pueden controlar muchos dispositivos desde una ubicación central, remota o del mismo nivel.

Las aplicaciones creadas con la API de punto de control con tecnología UPnP van desde scripts incrustados en páginas HTML hasta programas de C++ completos. La API de punto de control con tecnología UPnP es compatible con el sistema de desarrollo de Microsoft Visual Basic, Microsoft Visual Basic Scripting Edition (VBScript) y C++.

En las secciones siguientes se explica, en general, cómo funciona la API de punto de control con tecnología UPnP:

-   [Aspectos básicos de los puntos de control](control-point-basics.md)
-   [Búsqueda de dispositivos](finding-devices.md)
-   [Describir dispositivos](describing-devices.md)
-   [Control de dispositivos](controlling-devices.md)

Todos los ejemplos de la API de punto de control con la documentación de tecnología UPnP hacen referencia a dispositivos ficticios, tipos de dispositivo y cadenas.
