# <a name="iwmpclosedcaption-vb-and-c-interface"></a>Interfaz IWMPClosedCaption (VB y C#)

Proporciona una manera de incluir subtítulos con un archivo multimedia digital. El texto de subtítulos está en un archivo de intercambio multimedia accesible sincronizado (SAMI).

## <a name="members"></a>Miembros

La **interfaz IWMPClosedCaption (VB y C#)** no define ningún miembro.

La **interfaz IWMPClosedCaption** expone las siguientes propiedades.



| Propiedad                                                                             | Descripción                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [captioningId](wmplibiwmpclosedcaption-iwmpclosedcaption-captioningid--vb-and-c.md) | Obtiene o establece el nombre del elemento HTML que muestra los subtítulos.                       |
| [SAMIFileName](wmplibiwmpclosedcaption-iwmpclosedcaption-samifilename--vb-and-c.md) | Obtiene o establece el nombre del archivo que contiene la información necesaria para los subtítulos. |
| [SAMILang](wmplibiwmpclosedcaption-iwmpclosedcaption-samilang--vb-and-c.md)         | Obtiene o establece el idioma que se muestra para los subtítulos.                                 |
| [SAMIStyle](wmplibiwmpclosedcaption-iwmpclosedcaption-samistyle--vb-and-c.md)       | Obtiene o establece el estilo de subtítulos.                                                  |



 

Obtenga una **interfaz IWMPClosedCaption** mediante la siguiente propiedad.



| Object                                                            | Propiedad                                                                   |
|-------------------------------------------------------------------|----------------------------------------------------------------------------|
| [AxWindowsMediaPlayer](axwindowsmediaplayer-object--vb-and-c.md) | [closedCaption](axwmplib-axwindowsmediaplayer-closedcaption--vb-and-c.md) |
