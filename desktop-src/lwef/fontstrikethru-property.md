# <a name="fontstrikethru-property"></a>Propiedad FontStrikeThru

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descripción**
</dt> <dd>

Devuelve el estilo de fuente que se muestra actualmente en la ventana de globo de palabras para el carácter especificado.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxis**
</dt> <dd>

*agent***. Caracteres ("**_CharacterID_*_"). Balloon.FontStrikeThru_*



| Valor     | Descripción                                             |
|-----------|---------------------------------------------------------|
| **True**  | La fuente del globo usa el efecto de tachado.         |
| **False** | La fuente del globo no usa el efecto de tachado. |



 

</dd> </dl>

## <a name="remarks"></a>Comentarios

El valor predeterminado de la configuración de fuente del globo de palabras de un carácter se establece en el Editor de caracteres de Microsoft Agent. Además, el usuario puede invalidar la configuración de fuente de todos los caracteres de la hoja de propiedades de Microsoft Agent.
