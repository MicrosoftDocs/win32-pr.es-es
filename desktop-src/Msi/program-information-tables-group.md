# <a name="program-information-tables-group"></a>Grupo de tablas de información de programa

El grupo Tablas de información del programa contiene información importante que se usa a lo largo de la instalación.

-   La [tabla Property](property-table.md) proporciona un medio para especificar todas las [propiedades](properties.md) de una instalación.
-   La [tabla Binary](binary-table.md) contiene los datos binarios de elementos como mapas de bits, animaciones e iconos. La tabla binaria también se usa para almacenar datos para acciones personalizadas. Esta tabla puede contener un mapa de bits para un cuadro, el icono del programa o el formato ejecutable de una acción personalizada.
-   La [tabla Error se](error-table.md) usa para buscar plantillas de formato de mensajes de error al procesar errores con un conjunto de código de error. Esto es para el caso habitual en el que no hay ningún conjunto de plantillas de formato. El instalador tiene su propio mecanismo de procesamiento de errores. Los errores se pasan como registros.
-   [La tabla de](shortcut-table.md) métodos abreviados contiene la información que la aplicación necesita para crear accesos directos en el equipo del usuario.
-   La [tabla ReserveCost contiene](reservecost-table.md) el espacio en disco necesario para que cada componente funcione correctamente.
