# <a name="prov_rsa_sig"></a>\_firma RSA \_ SIG

El \_ \_ tipo de proveedor de RSA SIG de aprovisionamiento es un subconjunto de la RSA de provisión \_ \_ completa. Solo admite las funciones y algoritmos necesarios para los [*hashes*](../secgloss/h-gly.md) y las [*firmas digitales*](../secgloss/d-gly.md).

## <a name="algorithms-supported"></a>Algoritmos admitidos

Para obtener descripciones de cada uno de estos algoritmos, vea el glosario.



| Propósito      | Algoritmos admitidos                                                                                                              |
|--------------|-----------------------------------------------------------------------------------------------------------------------------------|
| Intercambio de claves | None                                                                                                                              |
| Firma    | [*RSA*](../secgloss/r-gly.md)                                                                       |
| Cifrado   | Ninguno                                                                                                                              |
| Aplicación de algoritmo hash      | [*MD5*](../secgloss/m-gly.md)[*Sha*](../secgloss/s-gly.md)<br/> |
