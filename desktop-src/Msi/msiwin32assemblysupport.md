# <a name="msiwin32assemblysupport-property"></a>Propiedad MsiWin32AssemblySupport

La **propiedad MsiWin32AssemblySupport** indica si el equipo admite ensamblados Win32. En los sistemas que admiten ensamblados Win32, el instalador establece el valor de **MsiWin32AssemblySupport** en la versión de archivo de sxs.dll. El instalador no establece esta propiedad si el sistema operativo no admite ensamblados Win32. Para obtener más información, vea [Ensamblados](assemblies.md).
