# <a name="international-isdn-number-attribute"></a>Atributo International-ISDN-Number

Especifica un número ISDN internacional asociado a un objeto .



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | International-ISDN-Number                   |
| Ldap-Display-Name | internationalISDNNumber                     |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 2.5.4.25                                    |
| System-Id-Guid    | bf96798d-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Numeric)**](s-string-numeric.md) |
