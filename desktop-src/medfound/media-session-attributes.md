# <a name="media-session-attributes"></a>Atributos de la sesión multimedia

Los atributos siguientes se usan para configurar la sesión multimedia.



| Atributo                                                                                            | Descripción                                                                                                 |
|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------|
| [**\_Administrador de \_ protección de contenido de sesión MF \_ \_**](mf-session-content-protection-manager-attribute.md) | Proporciona una interfaz de devolución de llamada para que la aplicación reciba un objeto de habilitador de contenido de la sesión de PMP. |
| [**\_ \_ hora global de sesión MF \_**](mf-session-global-time-attribute.md)                                | Especifica si las topologías tendrán una hora de inicio y de finalización global.                                        |
| [**\_Administrador de \_ calidad de sesión MF \_**](mf-session-quality-manager-attribute.md)                        | Contiene el CLSID de un administrador de calidad para la sesión multimedia.                                              |
| [**\_modo de \_ \_ origen remoto de sesión \_ MF**](mf-session-remote-source-mode-attribute.md)                 | Especifica que el origen multimedia se está ejecutando en un proceso remoto.                                             |
| [**\_contexto de \_ servidor de sesión MF \_**](mf-session-server-context-attribute.md)                          | Permite que dos instancias de la sesión multimedia compartan el mismo proceso de PMP.                                   |
| [**\_TOPOLOADER de sesión MF \_**](mf-session-topoloader-attribute.md)                                   | Contiene el CLSID de un cargador de topología para la sesión multimedia.                                              |
