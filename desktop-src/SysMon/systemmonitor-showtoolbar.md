# <a name="systemmonitorshowtoolbar-property"></a>Propiedad SystemMonitor. ShowToolbar

Recupera o establece un valor que determina si la barra de herramientas se muestra en el control.

## <a name="syntax"></a>Sintaxis


```VB
Property ShowToolbar As Boolean
```



## <a name="property-value"></a>Valor de propiedad

True indica que la barra de herramientas se muestra en el control; en caso contrario, false. El valor predeterminado de esta propiedad es true.

## <a name="remarks"></a>Observaciones

La barra de herramientas permite al usuario agregar o quitar contadores, elegir el tipo de gráfico que se usa para mostrar los datos del contador y modificar otras propiedades del control.
