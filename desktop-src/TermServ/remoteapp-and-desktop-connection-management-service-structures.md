# <a name="remoteapp-and-desktop-connection-management-service-structures"></a>Administración de conexiones de RemoteApp y Escritorio estructuras de servicio

La API del servicio Administración de conexiones de RemoteApp y Escritorio proporciona las siguientes estructuras.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**pluginResource**](/windows/win32/api/tspubplugincom/ns-tspubplugincom-pluginresource)
</dt> <dd>

Contiene información acerca de un recurso que se puede asignar a los usuarios en Conexión de RemoteApp y Escritorio.

</dd> <dt>

[**pluginResource2**](/windows/win32/api/tspubplugin2com/ns-tspubplugin2com-pluginresource2)
</dt> <dd>

Contiene información adicional acerca de un recurso que se puede asignar a los usuarios en Conexión de RemoteApp y Escritorio.

</dd> <dt>

[**pluginResource2FileAssociation**](/windows/win32/api/tspubplugin2com/ns-tspubplugin2com-pluginresource2fileassociation)
</dt> <dd>

Contiene información sobre una asociación de archivo en Conexión de RemoteApp y Escritorio.

</dd> </dl>
