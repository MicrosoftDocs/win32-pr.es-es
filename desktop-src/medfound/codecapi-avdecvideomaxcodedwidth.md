# <a name="codecapi_avdecvideomaxcodedwidth-property"></a>\_Propiedad AVDecVideoMaxCodedWidth de CODECAPI

Establece el ancho de imagen máximo que un descodificador de vídeo aceptará como tipo de entrada.

## <a name="data-type"></a>Tipo de datos

**Long** (**VT \_ I4**)

## <a name="property-guid"></a>GUID de propiedad

CODECAPI \_ AVDecVideoMaxCodedWidth
