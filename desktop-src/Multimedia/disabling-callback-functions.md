# <a name="disabling-callback-functions"></a>Deshabilitar funciones de devolución de llamada

Puede deshabilitar de forma permanente o temporal cualquiera de las funciones de devolución de llamada si especifica **null** en lugar de la función de devolución de llamada al enviar el mensaje adecuado para establecer una función de devolución de llamada.
