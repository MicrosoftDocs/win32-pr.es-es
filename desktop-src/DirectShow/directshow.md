# <a name="directshow"></a>DirectShow

La interfaz de programación de aplicaciones (API) de Microsoft DirectShow es una arquitectura de streaming multimedia para Microsoft Windows. Con DirectShow, las aplicaciones pueden realizar capturas o reproducción de vídeo y audio de alta calidad.

Los encabezados de DirectShow, las bibliotecas, las herramientas de SDK y los ejemplos están disponibles en la [Windows SDK](https://msdn.microsoft.com/windows/aa904949.aspx).

> [!Note]  
> Las versiones anteriores del SDK de DirectShow se incluían en el SDK de DirectX. La última versión del SDK de DirectX para incluir DirectShow era actualización del SDK de DirectX 9,0: (2005 de febrero). Después de esta versión, DirectShow se ha migrado a la Windows SDK. Para obtener la versión más reciente de los encabezados, las bibliotecas y los ejemplos de DirectShow, descargue el Windows SDK.
