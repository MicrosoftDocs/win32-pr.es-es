# <a name="resource-manager-context-functions"></a>Administrador de recursos funciones de contexto

Las siguientes funciones establecen y liberan el [*contexto de Resource Manager*](../secgloss/r-gly.md) que usan las funciones de administración de bases de datos y consultas de base de datos.



| Tema                                                  | Descripción                                                                                                                                          |
|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SCardEstablishContext**](/windows/desktop/api/Winscard/nf-winscard-scardestablishcontext) | Establece un contexto para tener acceso a la base de datos de la [*tarjeta inteligente*](../secgloss/s-gly.md). |
| [**SCardReleaseContext**](/windows/desktop/api/Winscard/nf-winscard-scardreleasecontext)     | Cierra un contexto establecido.                                                                                                                       |
