# <a name="using-the-windows-media-player-control-with-firefox"></a>Uso del control Reproductor de Windows Media con Firefox

Microsoft proporciona un complemento que le permite insertar el control Reproductor de Windows Media ActiveX en una página web para que un explorador Firefox lo muestre correctamente. El complemento, que debe instalarse en el equipo cliente, se puede descargar desde el sitio [web de Microsoft Port25](https://www.mozilla.org/).

Para obtener información sobre el explorador Firefox, consulte el sitio [web de Firefox.](https://www.mozilla.org/)

Para obtener más información sobre el Reproductor de Windows Media con Firefox, consulte la Reproductor de Windows Media del sitio [web de mozilla Firefox.](http://kb.mozillazine.org/Windows_Media_Player)

En los temas siguientes se describe cómo insertar el control Reproductor de Windows Media en una página web que se puede mostrar correctamente tanto Internet Explorer como Firefox.
