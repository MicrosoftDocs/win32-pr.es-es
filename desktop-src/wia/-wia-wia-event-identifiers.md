# <a name="wia-event-identifiers"></a>Identificadores de eventos WIA

Los identificadores de eventos WIA son identificadores únicos globales (GUID) definidos por el sistema de adquisición de imágenes de Windows (WIA).

Una aplicación WIA puede suscribirse a estos eventos no persistentes mediante [**IwiaDevMgr2:: RegisterEventCallbackInterface**](-wia-iwiadevmgr2-registereventcallbackinterface.md) para recibir notificaciones de cambio de estado de dispositivo del dispositivo WIA al que está conectada la sesión de aplicación WIA actual. El controlador mini de WIA debe asegurarse de que \_ \_ \_ el estado de tratamiento de documentos de DPS de WIA \_ esté actualizado correctamente antes de que el controlador señale a uno de estos eventos.

Los siguientes son GUID de eventos WIA válidos: 

| GUID                             | Significado                                                                                                                                                                                                                                                                                                                               |
|----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_ \_ \_ e/s de evento de WIA           | La aplicación intenta cancelar una transferencia de datos.                                                                                                                                                                                                                                                                                   |
| \_cubierta de eventos WIA \_ \_ cerrada        | Todas las cubiertas del escáner están cerradas. Si hay varias cubiertas de escáner abiertas, el minicontrolador WIA debe señalar este evento solo cuando se cerró la última portada. Esta característica está disponible a partir de Windows 8.                                                                                                                      |
| apertura de la cubierta de eventos de WIA \_ \_ \_          | El operador abrió una cubierta del escáner. El dispositivo no puede examinar en este momento. El minicontrolador WIA debe señalar este evento solo una vez si se abren varias cubiertas del escáner en sucesión. Esta característica está disponible a partir de Windows 8.                                                                                        |
| \_dispositivo de eventos WIA \_ \_ conectado    | Un dispositivo de hardware WIA se conectó al equipo del usuario o se inició.                                                                                                                                                                                                                                                                |
| \_dispositivo de eventos WIA \_ \_ desconectado | Un dispositivo de hardware WIA se desconectó del equipo del usuario o se detuvo.                                                                                                                                                                                                                                                           |
| \_dispositivo de eventos WIA \_ \_ no \_ preparado   | El dispositivo entró en estado no listo. La aplicación debe leer el \_ \_ \_ \_ Estado de tratamiento de documentos de WIA DPS para obtener más información. Esta característica está disponible a partir de Windows 8.                                                                                                                                                         |
| \_dispositivo de eventos WIA \_ \_ preparado        | El dispositivo entró en estado listo. Se borró el estado anterior no preparado señalado con el \_ dispositivo de eventos WIA \_ \_ no \_ está listo. Esta característica está disponible a partir de Windows 8.                                                                                                                                                             |
| se \_ \_ vació el alimentador de eventos WIA \_      | El operador quitó todos los documentos de la bandeja de entrada del alimentador y el alimentador está vacío. El dispositivo no puede examinar desde el alimentador en este momento. Importante: el minicontrolador no debe señalar este evento cuando el alimentador se queda vacío como resultado de un trabajo de digitalización ejecutado por una aplicación. Esta característica está disponible a partir de Windows 8. |
| se \_ \_ cargó el alimentador de eventos WIA \_       | El operador colocó los documentos en la bandeja de entrada del alimentador. El alimentador se carga con papel. En este momento, la aplicación puede iniciar un trabajo de digitalización desde el alimentador. Esta característica está disponible a partir de Windows 8.                                                                                                                     |
| \_tapa plana de eventos de WIA \_ \_ \_ cerrada | El operador ha cerrado la tapa del plano. En este momento, la aplicación puede iniciar un trabajo de digitalización desde el escáner. Esta característica está disponible a partir de Windows 8.                                                                                                                                                                 |
| \_tapa plana de eventos de WIA \_ \_ \_ abierta   | El operador abrió la tapa del plano. El dispositivo no puede escanear desde el escáner en este momento. Esta característica está disponible a partir de Windows 8.                                                                                                                                                                                  |
| \_elemento de evento WIA \_ \_ creado        | Se ha agregado un elemento al árbol de objetos [**IWiaItem**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) o [**IWiaItem2**](-wia-iwiaitem2.md) para un dispositivo WIA.                                                                                                                                                                                                   |
| \_elemento de evento WIA \_ \_ eliminado        | Se eliminó un elemento del árbol de objetos [**IWiaItem**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) o [**IWiaItem2**](-wia-iwiaitem2.md) para un dispositivo WIA.                                                                                                                                                                                               |
| \_ \_ reanudación de energía de evento de WIA \_        | Se ha restaurado la alimentación en el dispositivo o equipo.                                                                                                                                                                                                                                                                                    |
| \_suspensión de \_ energía de eventos WIA \_       | La energía se suspendió en el dispositivo o equipo.                                                                                                                                                                                                                                                                                   |
| \_imagen de \_ \_ correo electrónico de examen de eventos WIA \_   | Se inició un examen con la intención de enviar por correo electrónico la imagen resultante.                                                                                                                                                                                                                                                                   |
| \_imagen de \_ fax de examen de eventos WIA \_ \_     | Se inició un examen con la intención de enviar el fax a la imagen resultante.                                                                                                                                                                                                                                                                      |
| \_imagen de \_ película de examen de eventos WIA \_ \_    | Se examinó una imagen desde un adaptador de película.                                                                                                                                                                                                                                                                                             |
| \_imagen de \_ examen de eventos WIA \_          | Se examinó una imagen.                                                                                                                                                                                                                                                                                                                 |
| Examen de eventos de WIA \_ \_ \_ IMAGE2         | Se examinó una imagen. Este evento permite que los escáneres con más de un botón de exploración dirijan los botones a diferentes aplicaciones.                                                                                                                                                                                                      |
| Examen de eventos de WIA \_ \_ \_ IMAGE3         | Se examinó una imagen. Este evento permite que los escáneres con más de un botón de exploración dirijan los botones a diferentes aplicaciones.                                                                                                                                                                                                      |
| Examen de eventos de WIA \_ \_ \_ archivo image4         | Se examinó una imagen. Este evento permite que los escáneres con más de un botón de exploración dirijan los botones a diferentes aplicaciones.                                                                                                                                                                                                      |
| \_ \_ imagen OCR de examen de eventos WIA \_ \_     | Se inició un examen con la intención de convertir la imagen resultante con reconocimiento óptico de caracteres (OCR).                                                                                                                                                                                                                         |
| \_imagen de \_ impresión de análisis de eventos WIA \_ \_   | Se inició un examen con la intención de imprimir la imagen resultante.                                                                                                                                                                                                                                                                    |
| \_ \_ proxy STI de evento WIA \_           | Sólo para uso interno. No debe usarse.                                                                                                                                                                                                                                                                                           |
| \_almacenamiento de eventos WIA \_ \_ creado     | Se ha agregado al dispositivo una unidad de almacenamiento, como una tarjeta flash. La aplicación debe volver a crear el árbol de elementos que representa este dispositivo.                                                                                                                                                                                            |
| \_almacenamiento de eventos WIA \_ \_ eliminado     | Se quitó del dispositivo una unidad de almacenamiento, como una tarjeta flash. La aplicación debe volver a crear el árbol de elementos que representa este dispositivo.                                                                                                                                                                                        |
| \_árbol de eventos WIA \_ \_ actualizado        | Se actualizó el árbol de elementos WIA.                                                                                                                                                                                                                                                                                                        |
| \_volumen de eventos WIA \_ \_ insertado     | Un dispositivo de almacenamiento, como una tarjeta flash, se insertó en el dispositivo.                                                                                                                                                                                                                                                                 |
