<span data-ttu-id="23d93-144">**El uso adecuado de puntos suspensivos** es importante para indicar que los usuarios pueden tomar más decisiones antes de realizar la acción o incluso cancelar la acción por completo.</span><span class="sxs-lookup"><span data-stu-id="23d93-144">**Proper use of ellipses is important to indicate that users can make further choices before performing the action, or even cancel the action entirely**.</span></span> <span data-ttu-id="23d93-145">La indicación visual que ofrece un botón de puntos suspensivos permite a los usuarios explorar el software sin miedo.</span><span class="sxs-lookup"><span data-stu-id="23d93-145">The visual cue offered by an ellipsis allows users to explore your software without fear.</span></span>

<span data-ttu-id="23d93-146">**Esto no significa que deba** usar puntos suspensivos cada vez que una acción muestre otra ventana solo cuando se requiera información adicional para realizar la acción.</span><span class="sxs-lookup"><span data-stu-id="23d93-146">**This doesn't mean you should use an ellipsis whenever an action displays another window** only when additional information is required to perform the action.</span></span> <span data-ttu-id="23d93-147">Por lo tanto, cualquier botón de comando cuyo verbo implícito es "mostrar otra **ventana"** no toma puntos suspensivos, como con los comandos Acerca de, Avanzado, Ayuda (o cualquier otro comando que se vincule a un tema de Ayuda), Opciones, Propiedades o Configuración.</span><span class="sxs-lookup"><span data-stu-id="23d93-147">Consequently, **any command button whose implicit verb is to "show another window" doesn't take an ellipsis**, such as with the commands About, Advanced, Help (or any other command linking to a Help topic), Options, Properties, or Settings.</span></span>

<span data-ttu-id="23d93-148">Por lo general, los puntos suspensivos se usan en las interfaces de usuario para indicar que están incompletos.</span><span class="sxs-lookup"><span data-stu-id="23d93-148">Generally, ellipses are used in user interfaces to indicate incompleteness.</span></span> <span data-ttu-id="23d93-149">Los comandos que muestran que otras ventanas no están incompletas deben mostrar otra ventana y no se necesita información adicional para realizar su acción.</span><span class="sxs-lookup"><span data-stu-id="23d93-149">Commands that show other windows aren't incomplete they must display another window and additional information isn't needed to perform their action.</span></span> <span data-ttu-id="23d93-150">Este enfoque elimina el desorden de pantalla en situaciones en las que los puntos suspensivos tienen poco valor.</span><span class="sxs-lookup"><span data-stu-id="23d93-150">This approach eliminates screen clutter in situations where ellipses have little value.</span></span>

<span data-ttu-id="23d93-151">**Nota:** Al determinar si un botón de comando necesita puntos suspensivos, no use la necesidad de elevar los [privilegios](winenv-uac.md) como un factor.</span><span class="sxs-lookup"><span data-stu-id="23d93-151">**Note:** When determining if a command button needs an ellipsis, don't use the need to [elevate privileges](winenv-uac.md) as a factor.</span></span> <span data-ttu-id="23d93-152">La elevación no es información necesaria para realizar un comando (en su lugar, es para el permiso) y la necesidad de elevarse se indica con el escudo de seguridad.</span><span class="sxs-lookup"><span data-stu-id="23d93-152">Elevation isn't information needed to perform a command (rather, it's for permission) and the need to elevate is indicated with the security shield.</span></span>

<span data-ttu-id="23d93-153">**Si solo hace una cosa...** Use una etiqueta concisa, específica y autoexplicativa que describa claramente la acción que realiza el botón de comando y use puntos suspensivos cuando corresponda.</span><span class="sxs-lookup"><span data-stu-id="23d93-153">**If you do only one thing...** Use a concise, specific, self-explanatory label that clearly describes the action that the command button performs, and use an ellipsis when appropriate.</span></span>
