# <a name="ms-dns-maintain-trust-anchor-attribute"></a>atributo MS-DNS-maintain-Trust-Anchor

Atributo que se usa para definir el tipo de anclaje de veracidad que se va a publicar automáticamente en el almacén de anclaje de confianza para todo el bosque cuando la zona DNS está firmada.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-DNS-maintain-Trust-Anchor         |
| Nombre para mostrar de LDAP | msDN: MaintainTrustAnchor            |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2133              |
| System-ID-GUID    | 0dc063c1-52d9-4456-9e15-9c2434aafd94 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
