# <a name="ui-automation-fundamentals"></a>Fundamentos de UI Automation

Microsoft UI Automation permite que las aplicaciones de tecnología de asistencia y las herramientas de pruebas automatizadas interactúen con los controles de interfaz de usuario de otras aplicaciones. En esta sección se explican los conceptos fundamentales en los que se basa la automatización de la interfaz de usuario.

La API de automatización de la interfaz de usuario se encuentra en dos partes. Una parte la usan las aplicaciones del proveedor de automatización de la *interfaz de usuario* y la otra la usan las aplicaciones cliente de automatización de la interfaz de *usuario* . La API de proveedor de permite a los desarrolladores de controles personalizados de Microsoft Win32 y otros marcos de control exponer esos controles a la automatización de la interfaz de usuario y hacerlos visibles en las aplicaciones cliente. La API de cliente permite a las aplicaciones interactuar con los controles de otras aplicaciones y recuperar información sobre ellos.
