# <a name="functions-programming-reference"></a>Funciones (Referencia de programación)

Funciones XInput disponibles.

## <a name="in-this-section"></a>En esta sección

| Tema                                                                                 | Descripción                                                                                                                                             |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XInputEnable**](/windows/desktop/api/XInput/nf-xinput-xinputenable)<br/>                                       | Establece el estado de informes de XInput.<br/>                                                                                                          |
| [**XInputGetAudioDeviceIds**](/windows/desktop/api/XInput/nf-xinput-xinputgetaudiodeviceids)<br/>                 | Recupera los IDs de dispositivo de audio de representación de sonido y captura de sonido asociados al casco conectado al controlador especificado.<br/> |
| [**XInputGetBatteryInformation**](/windows/desktop/api/XInput/nf-xinput-xinputgetbatteryinformation)<br/>         | Recupera el tipo de batería y el estado de carga de un controlador inalámbrico.<br/>                                                                       |
| [**XInputGetCapabilities**](/windows/desktop/api/XInput/nf-xinput-xinputgetcapabilities)<br/>                     | Recupera las funcionalidades y características de un controlador conectado.<br/>                                                                           |
| [**XInputGetDSoundAudioDeviceGuids**](/windows/desktop/api/XInput/nf-xinput-xinputgetdsoundaudiodeviceguids)<br/> | Obtiene los GUID de dispositivo de representación de sonido y de captura de sonido asociados al casco conectado al controlador especificado.<br/>          |
| [**XInputGetKeystroke**](/windows/desktop/api/XInput/nf-xinput-xinputgetkeystroke)<br/>                           | Recupera un evento de entrada del controlador para juegos.<br/>                                                                                                             |
| [**XInputGetState**](/windows/desktop/api/XInput/nf-xinput-xinputgetstate)<br/>                                   | Recupera el estado actual del controlador especificado.<br/>                                                                                     |
| [**XInputSetState**](/windows/desktop/api/XInput/nf-xinput-xinputsetstate)<br/>                                   | Envía datos a un controlador conectado. Esta función se usa para activar la función de vibración de un controlador.<br/>                              |
