# <a name="microsoft-rsaschannel-cryptographic-provider"></a>Proveedor de servicios criptográficos de Microsoft RSA/Schannel

El proveedor de servicios criptográficos de Microsoft [*RSA*](../secgloss/r-gly.md) / [*Schannel*](../secgloss/s-gly.md) admite el hash, la firma de datos y la comprobación de firmas. El identificador de algoritmo CALG \_ SSL3 \_ SHAMD5 se usa para la autenticación de cliente SSL 3,0 y TLS 1,0. Este CSP admite la derivación de claves para los protocolos SSL2, PCT1, SSL3 y TLS1. El [*hash*](../secgloss/h-gly.md) consta de una concatenación de un hash MD5 con un hash Sha y se firma con una [*clave privada*](../secgloss/p-gly.md)RSA. Se puede exportar a otros países o regiones.

|----------------|----------------------------------|
| Tipo de proveedor: | **aprovisionamiento de \_ RSA \_ Schannel**          |
| Nombre del proveedor: | **Microsoft \_ Def \_ RSA \_ Schannel \_ Prov** |



 

Para obtener más información sobre los proveedores RSA/Schannel, consulte [funciones de CSP](cryptography-functions.md).
