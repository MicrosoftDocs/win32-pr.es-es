# <a name="properties-of-iscrdenr"></a>Propiedades de ISCrdEnr

En la tabla siguiente se enumeran las propiedades introducidas por ISCrdEnr.



| Propiedad                              | Descripción                                                                                                                                                                                              |
|---------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CSPCount**](iscrdenr-cspcount.md) | Especifica el número de [*proveedores de servicios criptográficos*](../secgloss/c-gly.md) (CSP). Esta propiedad es de solo lectura. |
| [**CSPName**](iscrdenr-cspname.md)   | Nombre del CSP. Esta propiedad es de lectura y escritura.                                                                                                                                                            |
