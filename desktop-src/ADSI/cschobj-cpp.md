# <a name="cschobjcpp"></a>CSCHOBJ. CPP

En el componente de proveedor de ejemplo, un ejemplo de código que administra la duración de los objetos de esquema se encuentra en cschobj. cpp. En la tabla siguiente se enumeran los métodos admitidos.



| Método                                                   | Descripción                                    |
|----------------------------------------------------------|------------------------------------------------|
| **CSampleDSSchema:: CreateSchema**                        | Constructor estándar.                          |
| **CSampleDSSchema:: ~ CSampleDSSchema**                    | Destructor estándar.                           |
| Métodos [**IADs**](/windows/desktop/api/Iads/nn-iads-iads) estándar.                   |                                                |
| Métodos [**IADsContainer**](/windows/desktop/api/Iads/nn-iads-iadscontainer) estándar. |                                                |
| **CSampleDSSchema:: QueryInterface**                      | Compruebe el identificador de interfaz especificado en este objeto.  |
| **CSampleDSSchema::AllocateSchema**                      | Cree un objeto de esquema y cargue sus datos de tipo. |
