# <a name="iupdate-methods"></a>Métodos IUpdate

La interfaz [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate) define los siguientes métodos.



| Método                                         | Descripción                                                                                      |
|------------------------------------------------|--------------------------------------------------------------------------------------------------|
| [**AcceptEula**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate-accepteula)       | Acepta los términos de licencia del software de Microsoft asociados a Windows Update.<br/> |
| [**CopyFromCache**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate-copyfromcache) | Copia el contenido de una actualización en una ruta de acceso especificada.<br/>                                 |
