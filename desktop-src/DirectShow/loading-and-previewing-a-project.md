# <a name="loading-and-previewing-a-project"></a>Cargar y obtener una vista previa de un proyecto

\[Esta API no se admite y puede modificarse o no estar disponible en el futuro.\]

En este tema se describe cómo cargar y obtener una vista previa de un archivo de proyecto con los [servicios de edición de DirectShow](directshow-editing-services.md) (des). Presenta una aplicación de ejemplo que carga un archivo de proyecto XML en una escala de tiempo vacía y representa la escala de tiempo para la vista previa. Puede modificar la aplicación de ejemplo para que funcione con cualquier archivo de proyecto XML de ejemplo incluido en este SDK.

Este artículo contiene los siguientes temas.

-   [Cargar un archivo de proyecto](loading-a-project-file.md)
-   [Obtener una vista previa del proyecto](previewing-the-project.md)
-   [Obtener una vista previa de un proyecto: código de ejemplo](previewing-a-project--example-code.md)
