# <a name="using-wia-video"></a>Uso de vídeo WIA

WIA proporciona la interfaz [**IWiaVideo**](/windows/desktop/api/Wiavideo/nn-wiavideo-iwiavideo) para permitir que las aplicaciones muestren vídeo en streaming en una ventana, capturar imágenes fijas de la secuencia de vídeo y guardarlas como archivos JPEG.

En las siguientes secciones se explica cómo usar las capacidades de vídeo de adquisición de imágenes de Windows (WIA):

-   [Obtener una vista previa del vídeo](-wia-previewing-video.md)
-   [Capturar una imagen de un flujo de vídeo](-wia-capturing-a-still-image-from-a-video-stream.md)

> [!Note]  
> WIA no admite dispositivos de vídeo en Windows Server 2003, Windows Vista o posterior. En el caso de las versiones de Windows, use [DirectShow](/previous-versions//ms783323(v=vs.85)) para adquirir imágenes del vídeo.
