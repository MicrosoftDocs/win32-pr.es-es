# <a name="multicast-group"></a>Grupo de multidifusión

Un grupo de multidifusión es una dirección IP de clase D (es decir, en el intervalo de 224.0.0.0 a 239.255.255.255). Los datos de multidifusión que se envían a una dirección de este intervalo no están destinados a un único destino. En su lugar, estos datos pueden ser recibidos por cualquier host que realice una solicitud para unirse a un grupo específico. Se hace referencia a un grupo de multidifusión mediante el símbolo G o g.
