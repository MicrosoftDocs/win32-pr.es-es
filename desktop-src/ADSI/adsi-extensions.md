# <a name="adsi-extensions"></a>Extensiones ADSI

Una extensión ADSI es un objeto COM especial que permite a un desarrollador ampliar un objeto ADSI. Cada extensión está asociada a una clase especificada en el directorio . Con este modelo de extensión, los desarrolladores pueden agregar métodos para dar un significado más dinámico a un objeto. En un modelo de programación de directorios tradicional, se tiene acceso a un objeto obteniendo y estableciendo los atributos del objeto. Con las extensiones ADSI, puede agregar más características a un objeto de directorio.
