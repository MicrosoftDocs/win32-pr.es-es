# <a name="controlling-object-access-in-active-directory-domain-services"></a>Controlar el acceso a objetos en Active Directory Domain Services

Cada Active Directory objeto de servicio de directorio está protegido por la seguridad de Windows 2000. Esta protección de seguridad controla las operaciones que cada entidad de seguridad puede realizar en el directorio.
