# <a name="folderitemverb-object"></a>Objeto FolderItemVerb

Representa un único Verbo disponible para un elemento. Este objeto contiene propiedades y métodos que permiten recuperar información sobre el verbo.

## <a name="members"></a>Miembros

El objeto **FolderItemVerb** tiene estos tipos de miembros:

-   [Métodos](#methods)
-   [Propiedades](#properties)

### <a name="methods"></a>Métodos

El objeto **FolderItemVerb** tiene estos métodos.



| Método                              | Descripción                                                                                  |
|:------------------------------------|:---------------------------------------------------------------------------------------------|
| [**DoIt**](folderitemverb-doit.md) | Ejecuta un verbo en el [**carpeta**](folderitem.md) asociado con el verbo.<br/> |



 

### <a name="properties"></a>Propiedades

El objeto **FolderItemVerb** tiene estas propiedades.



| Propiedad                                                     | Tipo de acceso          | Descripción                          |
|:-------------------------------------------------------------|:---------------------|:-------------------------------------|
| [**Application**](folderitemverb-application.md)<br/> | Solo lectura<br/> | Sin implementar.<br/>          |
| [**Nombre**](folderitemverb-name.md)<br/>               | Solo lectura<br/> | Contiene el nombre del verbo.<br/> |
| [**Parent**](folderitemverb-parent.md)<br/>           | Solo lectura<br/> | Sin implementar.<br/>          |
