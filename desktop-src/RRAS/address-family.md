# <a name="address-family"></a>Familia de direcciones

Las familias de direcciones incluyen intercambio de paquetes de Internet (IPX), protocolo de Internet versión 4 (IPv4) y Protocolo de Internet versión 6 (IPv6). También se puede hacer referencia a las familias de direcciones como familias de protocolos. Estas familias de direcciones se definen en [RFC 1700](routing-protocols-request-for-comments.md).
