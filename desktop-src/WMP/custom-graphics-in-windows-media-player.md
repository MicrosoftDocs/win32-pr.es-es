# <a name="custom-graphics-in-windows-media-player"></a>Gráficos personalizados en Windows Media Player

Con el scripting de metarchivo de Windows Media, puede agregar elementos de gráficos a Windows Media Player a medida que se reproduce la secuencia. Hay un tipo básico de gráficos que puede Agregar.



| Tipo   | Descripción                                                                                                                                                                                                                        |
|--------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CINTA | Con el elemento **banner** puede colocar una imagen de banner (32 píxeles de alto por 194 píxeles de ancho) en Windows Media Player en la parte inferior del área de presentación de vídeo. Después, puede usar un elemento secundario para agregar un hipervínculo al banner. |
