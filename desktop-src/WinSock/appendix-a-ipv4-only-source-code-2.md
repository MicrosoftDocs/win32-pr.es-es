# <a name="appendix-a-ipv4-only-source-code"></a>Apéndice A: código fuente solo IPv4

Este apéndice proporciona el código fuente de los ejemplos del kit de desarrollo de software (SDK) de la plataforma Simplec. c y simple. c, que se usan en esta guía de IPv6 como ejemplos:

-   [Código de cliente solo IPv4](ipv4-only-client-code-2.md)
-   [Código de servidor solo IPv4](ipv4-only-server-code-2.md)

> [!WARNING]
>
> Este código se incluye en esta guía para contrastar el código independiente de la versión de IP que se proporciona en el Apéndice B. Este código de ejemplo se proporciona con fines de comparación y para proporcionar una referencia clara y disponible al leer esta guía de IPv6. En el [Apéndice B: código fuente independiente de la versión IP](appendix-b-ip-version-agnostic-source-code-2.md)se proporciona una versión independiente de la versión de IP de este código fuente.

 

> [!Note]  
> Al comparar el código fuente del Apéndice A (solo IPv4) y el Apéndice B (independiente de la versión IP), se obtiene una idea de los cambios necesarios para modificar la aplicación de Windows Sockets para agregar compatibilidad con IPv6.
