La compañía de Fabrikam actualizó su dominio de Windows NT 4,0 a Windows 2000. Durante la instalación, se pedirá a Joe Worden, al administrador de red, que proporcione un nombre DNS de dominio. Joe con el tipo "fabrikam.com". Los usuarios, grupos y equipos existentes se migran a Active Directory en el nuevo dominio: fabrikam.com.
