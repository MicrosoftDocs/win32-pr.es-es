# <a name="using-more-than-one-context-handle-per-call"></a>Usar más de un identificador de contexto por llamada

La capacidad de usar matrices de identificadores de contexto como parámetros está disponible en Windows Vista y sistemas operativos posteriores. Sin embargo, esta característica debe usarse con cuidado dentro de una aplicación. Por ejemplo, en una situación en la que se usa un identificador de contexto en varias llamadas, resulta cada vez más difícil realizar un seguimiento de su uso.
