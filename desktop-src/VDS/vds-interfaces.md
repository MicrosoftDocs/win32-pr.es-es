# <a name="vds-interfaces"></a>Interfaces VDS

\[A partir de Windows 8 y Windows Server 2012, la interfaz com de [servicio de disco virtual](virtual-disk-service-portal.md) se sustituye por la [API de administración de almacenamiento de Windows](/previous-versions/windows/desktop/stormgmt/windows-storage-management-api-portal).\]

Los objetos de servicio de disco virtual (VDS) proporcionan interfaces que exponen métodos para consultar, configurar y mantener dispositivos de almacenamiento.



| Interfaz                                                            | Descripción                                                                                                                                                                                           |
|----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IEnumVdsObject**](/windows/desktop/api/Vds/nn-vds-ienumvdsobject)                             | Enumera a través de un conjunto de objetos VDS de un tipo determinado.                                                                                                                                              |
| [**IVdsAdmin**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdsadmin)                                       | Registra un proveedor con VDS.                                                                                                                                                                        |
| [**IVdsAdvancedDisk**](/windows/desktop/api/Vds/nn-vds-ivdsadvanceddisk)                         | Crea y elimina particiones y modifica los atributos de la partición.                                                                                                                                    |
| [**IVdsAdvancedDisk2**](/windows/desktop/api/Vds/nn-vds-ivdsadvanceddisk2)                       | Proporciona un método para cambiar los tipos de partición.                                                                                                                                                          |
| [**IVdsAdviseSink**](/windows/desktop/api/Vds/nn-vds-ivdsadvisesink)                             | Recibe notificaciones de VDS.                                                                                                                                                                           |
| [**IVdsAsync**](/windows/desktop/api/Vds/nn-vds-ivdsasync)                                       | Administra operaciones asincrónicas.                                                                                                                                                                      |
| [**IVdsController**](/windows/desktop/api/Vds/nn-vds-ivdscontroller)                             | Expone métodos para realizar operaciones de consulta y configuración en un controlador.                                                                                                                    |
| [**IVdsControllerControllerPort**](/windows/desktop/api/Vds/nn-vds-ivdscontrollercontrollerport) | Proporciona un método para enumerar los puertos de controlador de una clase que implementa la interfaz [**IVdsController**](/windows/desktop/api/Vds/nn-vds-ivdscontroller) .                                                                      |
| [**IVdsControllerPort**](/windows/desktop/api/Vds/nn-vds-ivdscontrollerport)                     | Proporciona métodos para realizar operaciones de consulta y configuración en un puerto de controlador.                                                                                                              |
| [**IVdsCreatePartitionEx**](/windows/desktop/api/Vds/nn-vds-ivdscreatepartitionex)               | Crea una partición en un disco básico.                                                                                                                                                                  |
| [**IVdsDisk**](/windows/desktop/api/Vds/nn-vds-ivdsdisk)                                         | Consulta y configura discos básicos y dinámicos.                                                                                                                                                       |
| [**IVdsDisk2**](/windows/desktop/api/Vds/nn-vds-ivdsdisk2)                                       | Proporciona un método para establecer el modo SAN de un disco en sin conexión o en línea.                                                                                                                                 |
| [**IVdsDiskPartitionMF**](/windows/desktop/api/Vds/nn-vds-ivdsdiskpartitionmf)                   | Proporciona métodos para realizar operaciones de administración del sistema de archivos en particiones.                                                                                                                          |
| [**IVdsDiskPartitionMF2**](/windows/desktop/api/Vds/nn-vds-ivdsdiskpartitionmf2)                 | Proporciona un método para dar formato a una partición con opciones de formato adicionales.                                                                                                                           |
| [**IVdsDrive**](/windows/desktop/api/Vds/nn-vds-ivdsdrive)                                       | Proporciona métodos para realizar operaciones de consulta y configuración en una unidad.                                                                                                                        |
| [**IVdsDrive2**](/windows/desktop/api/Vds/nn-vds-ivdsdrive2)                                     | Proporciona un método para consultar las propiedades de una unidad.                                                                                                                                             |
| [**IVdsHbaPort**](/windows/desktop/api/Vds/nn-vds-ivdshbaport)                                   | Proporciona métodos para consultar e interactuar con los puertos HBA en el sistema local.                                                                                                                            |
| [**IVdsHwProvider**](/windows/desktop/api/Vds/nn-vds-ivdshwprovider)                             | Proporciona métodos para realizar operaciones de consulta, reenumeración y actualización en un proveedor de hardware.                                                                                                  |
| [**IVdsHwProviderPrivate**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdshwproviderprivate)               | Proporciona un método que permite a VDS determinar si el proveedor de hardware posee un LUN especificado.                                                                                                   |
| [**IVdsHwProviderPrivateMpio**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdshwproviderprivatempio)       | Proporciona un método que establece el estado de las rutas de acceso que proceden de un puerto HBA determinado al proveedor.                                                                                               |
| [**IVdsHwProviderStoragePools**](/windows/desktop/api/Vds/nn-vds-ivdshwproviderstoragepools)     | Proporciona métodos para crear LUN en un [bloque de almacenamiento](storage-pool-object.md) y enumerar los grupos de almacenamiento administrados por un proveedor de hardware.                                                          |
| [**IVdsHwProviderType**](/windows/desktop/api/Vds/nn-vds-ivdshwprovidertype)                     | Proporciona un método para recuperar el tipo de proveedor de hardware.                                                                                                                                          |
| [**IVdsHwProviderType2**](/windows/desktop/api/Vds/nn-vds-ivdshwprovidertype2)                   | Esta interfaz no está implementada. Use [**IVdsHwProviderType**](/windows/desktop/api/Vds/nn-vds-ivdshwprovidertype) en su lugar.                                                                                                      |
| [**IVdsIscsiInitiatorAdapter**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiinitiatoradapter)       | Proporciona métodos para consultar e interactuar con los adaptadores de iniciador iSCSI en el sistema local.                                                                                                             |
| [**IVdsIscsiInitiatorPortal**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiinitiatorportal)         | Proporciona métodos para consultar e interactuar con portales del iniciador iSCSI en el sistema local.                                                                                                              |
| [**IVdsIscsiPortal**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiportal)                           | Proporciona métodos para realizar operaciones de consulta y configuración en un portal de iSCSI.                                                                                                                |
| [**IVdsIscsiPortalGroup**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiportalgroup)                 | Proporciona métodos para realizar servicios de consulta y configuración en un grupo de portales iSCSI.                                                                                                            |
| [**IVdsIscsiPortalLocal**](/windows/desktop/api/Vds/nn-vds-ivdsiscsiportallocal)                 | Proporciona métodos para establecer claves de IPSEC precompartidas específicas del iniciador local en un portal de iSCSI.                                                                                                       |
| [**IVdsIscsiTarget**](/windows/desktop/api/Vds/nn-vds-ivdsiscsitarget)                           | Proporciona métodos para realizar operaciones de consulta y configuración en un destino iSCSI.                                                                                                                |
| [**IVdsLun**](/windows/desktop/api/Vds/nn-vds-ivdslun)                                           | Proporciona métodos para realizar operaciones de consulta y configuración en un LUN (número de unidad lógica).                                                                                                    |
| [**IVdsLun2**](/windows/desktop/api/Vds/nn-vds-ivdslun2)                                         | Proporciona métodos para aplicar y consultar Sugerencias de número de unidad lógica (LUN).                                                                                                                           |
| [**IVdsLunControllerPorts**](/windows/desktop/api/Vds/nn-vds-ivdsluncontrollerports)             | Proporciona métodos para realizar operaciones de configuración de puerto de controlador en un LUN.                                                                                                                    |
| [**IVdsLunIscsi**](/windows/desktop/api/Vds/nn-vds-ivdsluniscsi)                                 | Proporciona métodos para realizar operaciones de consulta y configuración en un LUN iSCSI.                                                                                                                   |
| [**IVdsLunMpio**](/windows/desktop/api/Vds/nn-vds-ivdslunmpio)                                   | Proporciona métodos para realizar operaciones de consulta y configuración en un LUN con extensiones MPIO.                                                                                                     |
| [**IVdsLunNaming**](/windows/desktop/api/Vds/nn-vds-ivdslunnaming)                               | Proporciona un método para asignar nombres a los LUN de una clase que implementa la interfaz [**IVdsLun**](/windows/desktop/api/Vds/nn-vds-ivdslun) .                                                                                                     |
| [**IVdsLunNumber**](/windows/desktop/api/Vds/nn-vds-ivdslunnumber)                               | Proporciona un método para consultar el número de LUN de un LUN.                                                                                                                                                  |
| [**IVdsLunPlex**](/windows/desktop/api/Vds/nn-vds-ivdslunplex)                                   | Proporciona métodos para realizar operaciones de consulta y configuración en un complejo LUN.                                                                                                                     |
| [**IVdsMaintenance**](/windows/desktop/api/Vds/nn-vds-ivdsmaintenance)                           | Proporciona métodos para realizar operaciones de mantenimiento en un subsistema, un controlador o una unidad.                                                                                                          |
| [**IVdsOpenVDisk**](/windows/desktop/api/Vds/nn-vds-ivdsopenvdisk)                               | Define métodos para administrar un disco virtual.                                                                                                                                                          |
| [**IVdsPack**](/windows/desktop/api/Vds/nn-vds-ivdspack)                                         | Consulta y configura paquetes que contienen discos y crea volúmenes.                                                                                                                                   |
| [**IVdsPack2**](/windows/desktop/api/Vds/nn-vds-ivdspack2)                                       | Proporciona un método para crear volúmenes alineados en un paquete.                                                                                                                                                |
| [**IVdsProvider**](/windows/desktop/api/Vds/nn-vds-ivdsprovider)                                 | Devuelve las propiedades de un proveedor de hardware o software.                                                                                                                                                 |
| [**IVdsProviderPrivate**](/windows/desktop/api/VdsHwPrv/nn-vdshwprv-ivdsproviderprivate)                   | Proporciona métodos para permitir que VDS realice operaciones misceláneas en objetos de proveedor.                                                                                                               |
| [**IVdsProviderSupport**](/windows/desktop/api/Vds/nn-vds-ivdsprovidersupport)                   | Proporciona un método para indicar qué versiones de las interfaces de VDS son compatibles con el proveedor.                                                                                                      |
| [**IVdsRemovable**](/windows/desktop/api/Vds/nn-vds-ivdsremovable)                               | Consulta y expulsa discos extraíbles, como CD-ROM.                                                                                                                                                  |
| [**IVdsService**](/windows/desktop/api/Vds/nn-vds-ivdsservice)                                   | Proporciona métodos de nivel de servicio para trabajar con VDS.                                                                                                                                                  |
| [**IVdsServiceHba**](/windows/desktop/api/Vds/nn-vds-ivdsservicehba)                             | Proporciona un método para consultar los puertos HBA en el sistema local.                                                                                                                                             |
| [**IVdsServiceIscsi**](/windows/desktop/api/Vds/nn-vds-ivdsserviceiscsi)                         | Proporciona métodos para interactuar con el servicio del iniciador local.                                                                                                                                       |
| [**IVdsServiceLoader**](/windows/desktop/api/Vds/nn-vds-ivdsserviceloader)                       | Inicia VDS.                                                                                                                                                                                         |
| [**IVdsServiceUninstallDisk**](/windows/desktop/api/Vds/nn-vds-ivdsserviceuninstalldisk)         | Proporciona métodos para desinstalar discos básicos y dinámicos.                                                                                                                                                |
| [**IVdsStoragePool**](/windows/desktop/api/Vds/nn-vds-ivdsstoragepool)                           | Proporciona métodos para consultar información y enumerar los objetos relacionados de un [grupo de almacenamiento](storage-pool-object.md).                                                                                    |
| [**IVdsSubSystem**](/windows/desktop/api/Vds/nn-vds-ivdssubsystem)                               | Proporciona métodos para realizar operaciones de consulta y configuración en un subsistema.                                                                                                                    |
| [**IVdsSubSystem2**](/windows/desktop/api/Vds/nn-vds-ivdssubsystem2)                             | Proporciona métodos para realizar operaciones de consulta y configuración en un subsistema mediante las estructuras PROP2 de [**VDS \_ HINTS2**](/windows/desktop/api/Vds/ns-vds-vds_hints2) y [**VDS \_ \_ Subsystem \_**](/windows/desktop/api/Vds/ns-vds-vds_sub_system_prop2) . |
| [**IVdsSubSystemImportTarget**](/windows/desktop/api/Vds/nn-vds-ivdssubsystemimporttarget)       | Proporciona métodos para consultar y configurar el destino de importación de VSS predeterminado para el subsistema.                                                                                                              |
| [**IVdsSubSystemInterconnect**](/windows/desktop/api/Vds/nn-vds-ivdssubsysteminterconnect)       | Proporciona un método para consultar los tipos de interconexión admitidos por un subsistema.                                                                                                                  |
| [**IVdsSubSystemIscsi**](/windows/desktop/api/Vds/nn-vds-ivdssubsystemiscsi)                     | Proporciona métodos para consultar y configurar destinos iSCSI y portales en un subsistema.                                                                                                                     |
| [**IVdsSubSystemNaming**](/windows/desktop/api/Vds/nn-vds-ivdssubsystemnaming)                   | Proporciona un método para asignar nombres a los subsistemas de una clase que implementa la interfaz [**IVdsSubSystem**](/windows/desktop/api/Vds/nn-vds-ivdssubsystem) .                                                                                   |
| [**IVdsSwProvider**](/windows/desktop/api/Vds/nn-vds-ivdsswprovider)                             | Realiza operaciones del proveedor de software.                                                                                                                                                                |
| [**IVdsVDisk**](/windows/desktop/api/Vds/nn-vds-ivdsvdisk)                                       | Define métodos para administrar un disco virtual.                                                                                                                                                          |
| [**IVdsVdProvider**](/windows/desktop/api/Vds/nn-vds-ivdsvdprovider)                             | Define métodos para crear y administrar discos virtuales.                                                                                                                                              |
| [**IVdsVolume**](/windows/desktop/api/Vds/nn-vds-ivdsvolume)                                     | Crea y elimina los Plex de volumen y modifica los atributos de volumen.                                                                                                                                    |
| [**IVdsVolume2**](/windows/desktop/api/Vds/nn-vds-ivdsvolume2)                                   | Proporciona un método para devolver información de propiedades de volumen, incluidos los GUID de volumen.                                                                                                              |
| [**IVdsVolumeMF**](/windows/desktop/api/Vds/nn-vds-ivdsvolumemf)                                 | Realiza las operaciones de ruta de acceso y del sistema de archivos en objetos de volumen.                                                                                                                                    |
| [**IVdsVolumeMF2**](/windows/desktop/api/Vds/nn-vds-ivdsvolumemf2)                               | Proporciona métodos para realizar operaciones adicionales de administración del sistema de archivos en el objeto de volumen.                                                                                                        |
| [**IVdsVolumeMF3**](/windows/desktop/api/Vds/nn-vds-ivdsvolumemf3)                               | Proporciona métodos para realizar operaciones adicionales de administración del sistema de archivos en el objeto de volumen.                                                                                                        |
