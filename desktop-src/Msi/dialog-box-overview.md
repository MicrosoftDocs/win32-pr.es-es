# <a name="dialog-box-overview"></a><span data-ttu-id="01864-103">Información general del cuadro de diálogo</span><span class="sxs-lookup"><span data-stu-id="01864-103">Dialog Box Overview</span></span>

<span data-ttu-id="01864-104">El proceso de creación de un cuadro de diálogo en Windows Installer es similar al proceso de creación de un cuadro de diálogo mediante programación con una plantilla de cuadro de diálogo de la API de Microsoft Windows.</span><span class="sxs-lookup"><span data-stu-id="01864-104">The process of creating a dialog box in Windows Installer is similar to the process of creating a dialog box programmatically using a Microsoft Windows API dialog box template.</span></span> <span data-ttu-id="01864-105">Mientras que una plantilla de cuadro de diálogo de Windows se almacena en una cadena de caracteres terminada en null, Windows Installer almacena los parámetros del cuadro de diálogo en la tabla de diálogo.</span><span class="sxs-lookup"><span data-stu-id="01864-105">While a Windows dialog box template is stored in a null terminated–character string, Windows Installer stores the dialog box parameters in the Dialog table.</span></span> <span data-ttu-id="01864-106">La tabla del cuadro de diálogo contiene una columna de atributos que es análoga a los estilos de ventana de la API de la interfaz de usuario de Microsoft Windows.</span><span class="sxs-lookup"><span data-stu-id="01864-106">The Dialog table contains an attributes column that is analogous to Window styles in the Microsoft Windows user interface API.</span></span> <span data-ttu-id="01864-107">Sin embargo, el número de bits de estilo de cuadro de diálogo de Windows Installer es un conjunto reducido y especializado.</span><span class="sxs-lookup"><span data-stu-id="01864-107">However, the number of dialog style bits in Windows Installer is a reduced and specialized set.</span></span>

<span data-ttu-id="01864-108">Para crear físicamente el cuadro de diálogo mediante la API de la interfaz de usuario de Windows, se llama a [**DialogBox**](/windows/win32/api/winuser/nf-winuser-dialogboxa) y pasa un puntero a la plantilla.</span><span class="sxs-lookup"><span data-stu-id="01864-108">To physically create the dialog box using the Windows user interface API, [**DialogBox**](/windows/win32/api/winuser/nf-winuser-dialogboxa) is called and passes a pointer to the template.</span></span> <span data-ttu-id="01864-109">En Windows Installer, el cuadro de diálogo se crea durante la ejecución de una de las tres tablas de secuencia de interfaz de usuario.</span><span class="sxs-lookup"><span data-stu-id="01864-109">In Windows Installer, the dialog box is created during execution of one of the three UI sequence tables.</span></span>

<span data-ttu-id="01864-110">Windows Installer no contiene una interfaz de usuario predeterminada que los autores de paquetes pueden emplear para sus paquetes.</span><span class="sxs-lookup"><span data-stu-id="01864-110">Windows Installer does not contain a default UI that package authors can utilize for their packages.</span></span> <span data-ttu-id="01864-111">Contiene un conjunto limitado de cuadros de diálogo predeterminados que muestran los mensajes de error y de información, pero solo se muestran si Windows Installer consulta las tablas de la secuencia de cuadros de diálogo y de la interfaz de usuario y detecta que no hay cuadros de diálogo personalizados disponibles.</span><span class="sxs-lookup"><span data-stu-id="01864-111">It does contain a limited set of default dialog boxes that display error and information messages, but these are displayed only if Windows Installer queries the Dialog and UI Sequence tables and finds there are no custom dialog boxes available.</span></span>

<span data-ttu-id="01864-112">El SDK del instalador incluye una base de datos mínima denominada Uisample.msi que contiene tablas de secuencia de ejecución y IU llenas y una interfaz de usuario completa.</span><span class="sxs-lookup"><span data-stu-id="01864-112">The installer SDK includes a minimal database named Uisample.msi that contains populated UI and execution sequence tables and a complete UI.</span></span> <span data-ttu-id="01864-113">Esta base de datos se personaliza y combina con facilidad en cualquier paquete.</span><span class="sxs-lookup"><span data-stu-id="01864-113">This database is easily customized and merged on to any package.</span></span>

<span data-ttu-id="01864-114">Algunas acciones estándar e internas Windows Installer condiciones de error devuelven un conjunto específico de datos de interfaz de usuario y, por tanto, requieren un cuadro de diálogo con un conjunto específico de controles para acomodar esos datos.</span><span class="sxs-lookup"><span data-stu-id="01864-114">Some standard actions and internal Windows Installer error conditions return a specific set of UI data and therefore require a dialog box with a specific set of controls to accommodate that data.</span></span> <span data-ttu-id="01864-115">Windows Installer comprueba dos ubicaciones independientes para los identificadores de estos cuadros de diálogo.</span><span class="sxs-lookup"><span data-stu-id="01864-115">Windows Installer checks two separate locations for the identifiers for these dialog boxes.</span></span>

-   <span data-ttu-id="01864-116">Windows Installer contiene un conjunto de nombres de cuadros de diálogo incrustados.</span><span class="sxs-lookup"><span data-stu-id="01864-116">Windows Installer contains a set of embedded dialog box names.</span></span> <span data-ttu-id="01864-117">La acción personalizada consulta en la tabla de diálogo los nombres reservados.</span><span class="sxs-lookup"><span data-stu-id="01864-117">The custom action queries the Dialog table for these reserved names.</span></span>
-   <span data-ttu-id="01864-118">En cada una de las tres tablas de secuencias de la interfaz de usuario, los nombres de cuadro de diálogo con un número de secuencia de-1,-2 o-3 se muestran como respuesta a la salida, salida solicitada por el usuario y mensajes de error irrecuperables de Windows Installer.</span><span class="sxs-lookup"><span data-stu-id="01864-118">In each of the three UI sequence tables, dialog names with a sequence number of -1,-2, or -3 are displayed in response to exit, user requested exit, and fatal error messages from Windows Installer.</span></span>

<span data-ttu-id="01864-119">En los [cuadros de diálogo](dialog-boxes.md)hay disponible una lista completa de los nombres de los cuadros de diálogo de clave principal reservados.</span><span class="sxs-lookup"><span data-stu-id="01864-119">A complete list of reserved primary key dialog box names is available in [dialog boxes](dialog-boxes.md).</span></span> <span data-ttu-id="01864-120">Tenga en cuenta que el nombre del cuadro de diálogo de la clave principal solo está reservado por el instalador y no hay ninguna implementación específica del cuadro de diálogo en Windows Installer.</span><span class="sxs-lookup"><span data-stu-id="01864-120">Note that the primary key dialog box name is only reserved by the installer and there is no specific implementation of the dialog box within Windows Installer.</span></span> <span data-ttu-id="01864-121">Los autores de paquetes deben rellenar todos los campos de la base de datos que especifiquen los atributos y controles de estos cuadros de diálogo reservados.</span><span class="sxs-lookup"><span data-stu-id="01864-121">Package authors must still populate all the fields in the database that specify the attributes and controls of these reserved dialog boxes.</span></span>
