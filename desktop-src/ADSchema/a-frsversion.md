| CN                | FRS-Version                                 |
| Ldap-Display-Name | fRSVersion                                  |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |

| System-Id-Guid    | 2a132585-9373-11d1-aebc-0000f80367c1        |
