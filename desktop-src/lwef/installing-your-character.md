# <a name="installing-your-character"></a>Instalación del carácter

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Para que el carácter se muestre en la galería de asistentes, instálelo en la \\ carpeta Microsoft Office \\ Actors dentro de la carpeta de datos de la aplicación del usuario. La carpeta de datos de la aplicación se puede encontrar en el directorio de Windows o en el directorio de perfiles de usuario dentro del directorio de Windows. Por ejemplo, en Windows 2000, puede instalar el carácter en "C: \\ Documents and Settings \\ nombreiniciosesión \\ Application Data \\ Microsoft \\ Office \\ Actors".
