# <a name="systempropgroupmusic"></a>System. PropGroup. Music

El separador de grupos de propiedades usado en las listas de propiedades para separar los archivos de música de otros tipos. No utilice esta propiedad para obtener o establecer valores. Está pensado únicamente como un marcador.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropGroup.Music
   shellPKey = PKEY_PropGroup_Music
   formatID = 68DD6094-7216-40F1-A029-43FE7127043F
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Null
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
