# <a name="remoteapp-and-desktop-connection-management-service-interfaces"></a>Interfaces de servicio Administración de conexiones de RemoteApp y Escritorio

La API del servicio Administración de conexiones de RemoteApp y Escritorio proporciona las siguientes interfaces.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**ItsPubPlugin**](/windows/desktop/api/tspubplugincom/nn-tspubplugincom-itspubplugin)
</dt> <dd>

Expone propiedades y métodos que proporcionan información sobre los recursos disponibles para los usuarios de conexiones de RemoteApp y escritorio.

</dd> <dt>

[**ItsPubPlugin2**](/windows/desktop/api/tspubplugin2com/nn-tspubplugin2com-itspubplugin2)
</dt> <dd>

Especifica métodos que proporcionan información sobre los recursos disponibles para los usuarios de conexiones de RemoteApp y escritorio.

</dd> </dl>
