# <a name="windows-system-assessment-tool-provider"></a>Windows Proveedor de herramientas de evaluación del sistema

\[WinSAT puede modificarse o no estar disponible para las versiones después de Windows 8.1.\]

## <a name="purpose"></a>Propósito

La Windows System Assessment Tool (WinSAT) expone una serie de clases que evalúan las características y funcionalidades de rendimiento de un equipo. Los desarrolladores pueden usar esta API para desarrollar software que pueda acceder a la información de rendimiento y funcionalidad de un equipo para determinar la configuración óptima de la aplicación en función de las funcionalidades de rendimiento de ese equipo.

Los usuarios pueden usar los resultados de una evaluación formal para determinar qué escenarios y aplicaciones se ejecutarán bien en un equipo. Por ejemplo, si un paquete de software contiene una clasificación en su empaquetado, un usuario puede usar la clasificación para determinar si el software se ejecutará bien en el equipo.

## <a name="developer-audience"></a>Audiencia de desarrolladores

WinSAT está diseñado para C, C++ y Visual Basic desarrolladores y aquellos que escriben scripts.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

WinSAT está disponible en equipos cliente a partir del Windows operativo Vista.
