# <a name="avencddlorocentermixlvl_x10-property"></a>AvEncDDLoRoCenterMixLvl \_ x10, propiedad

Especifica el desplazamiento de nivel que se aplica al canal central para la mezclación lo/ro. Esta propiedad se aplica a los codificadores de audio Dolby Digital.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncDDLoRoCenterMixLvl \_ x10**

## <a name="property-value"></a>Valor de propiedad

El valor se especifica como decibelios (dB) x 10.
