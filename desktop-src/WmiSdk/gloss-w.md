# <a name="w-wmi"></a>W (WMI)

[A](gloss-a.md) B [C](gloss-c.md) [D](gloss-d.md) [E](gloss-e.md) [F](gloss-f.md) G [H](gloss-h.md) [I](gloss-i.md) J [K](gloss-k.md) [L M](gloss-l.md) [N](gloss-m.md) [O](gloss-n.md) [](gloss-o.md) [P P](gloss-p.md) [Q](gloss-q.md) [R](gloss-r.md) [S](gloss-s.md) [T](gloss-t.md) U V W X Y Z

<dl> <dt>

<span id="wmi.gloss_win32_schema"></span><span id="WMI.GLOSS_WIN32_SCHEMA"></span>**Esquema de Win32**
</dt> <dd>

Una extensión de Microsoft al esquema [*CIM*](gloss-c.md) que contiene [](gloss-m.md) definiciones de clases e instancias para representar objetos administrados que existen en un entorno Windows normal.

</dd> <dt>

<span id="wmi.gloss_wmi_class"></span><span id="WMI.GLOSS_WMI_CLASS"></span>**Clase WMI**
</dt> <dd>

Plantilla para un tipo de [*objeto administrado.*](gloss-m.md) Las clases WMI definen las unidades básicas de administración.

</dd> <dt>

<span id="wmi.gloss_wmi_class_provider"></span><span id="WMI.GLOSS_WMI_CLASS_PROVIDER"></span>**Proveedor de clases WMI**
</dt> <dd>

Un servidor COM que proporciona definiciones de clase. Los proveedores de clases implementan métodos en [**la interfaz IWbemServices**](/windows/desktop/api/WbemCli/nn-wbemcli-iwbemservices) para admitir la recuperación, modificación, eliminación, enumeración y procesamiento de consultas de datos.

</dd> <dt>

<span id="wmi.gloss_wmi_control"></span><span id="WMI.GLOSS_WMI_CONTROL"></span>**WMI Control**
</dt> <dd>

Complemento Microsoft Management Console (MMC) que permite a un usuario configurar wmi en un equipo local o remoto.

</dd> <dt>

<span id="wmi.gloss_wmi_for_windows_driver_model"></span><span id="WMI.GLOSS_WMI_FOR_WINDOWS_DRIVER_MODEL"></span>**WMI para Windows driver model (WDM)**
</dt> <dd>

Componente de bajo nivel que proporciona acceso entre wmi y dispositivos de hardware.

</dd> <dt>

<span id="wmi.gloss_wmi_odbc_adapter"></span><span id="WMI.GLOSS_WMI_ODBC_ADAPTER"></span>**Adaptador ODBC de WMI**
</dt> <dd>

Interfaz que permite a una aplicación obtener información de Windows Management mediante *lenguaje de consulta de WMI*. Esto proporciona una API estándar que permite a las aplicaciones basadas en ODBC usar los datos del [*repositorio CIM*](gloss-c.md) como si fuera una base de datos.

</dd> <dt>

<span id="wmi.gloss_wmi_query_language"></span><span id="WMI.GLOSS_WMI_QUERY_LANGUAGE"></span>**lenguaje de consulta de WMI (WQL)**
</dt> <dd>

Subconjunto del lenguaje de consulta estructurado (SQL) con extensiones para admitir la notificación de eventos WMI y otras características específicas de WMI.

</dd> <dt>

<span id="wmi.gloss_wmi_method"></span><span id="WMI.GLOSS_WMI_METHOD"></span>**Método WMI**
</dt> <dd>

Método implementado por la [API COM de](com-api-for-wmi.md) WMI o scripting [API](scripting-api-for-wmi.md). Dos ejemplos son el método COM [**IWbemServices::ExecMethod**](/windows/desktop/api/WbemCli/nf-wbemcli-iwbemservices-execmethod) y el método de API de scripting similar [**SWbemServices.ExecMethod**](swbemservices-execmethod.md).

</dd> <dt>

<span id="wmi.gloss_wmi_repository"></span><span id="WMI.GLOSS_WMI_REPOSITORY"></span>**Repositorio WMI**
</dt> <dd>

Un área de almacenamiento central administrada por el [*Administrador de objetos CIM*](gloss-c.md).

</dd> <dt>

<span id="wmi.gloss_wql"></span><span id="WMI.GLOSS_WQL"></span>**WQL**
</dt> <dd>

Consulte *lenguaje de consulta de WMI (WQL).*

</dd> </dl>
