# <a name="capture-windows"></a>Capturar ventanas

Las ventanas de captura son conceptualmente similares a los controles estándar (como botones, cuadros de lista o barras de desplazamiento). Normalmente, las ventanas de captura usan los \_ estilos de ventana WS Child y WS \_ visible.
