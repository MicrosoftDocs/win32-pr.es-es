# <a name="wmtoolversion"></a>WM/ToolVersion

El atributo **WM/ToolVersion** contiene el número de versión de la aplicación utilizada para crear el archivo. Si usa este atributo, también debe incluir el nombre de la aplicación mediante [**WM/ToolName**](wm-toolname.md).

## <a name="global-constant"></a>Constante global

g \_ wszWMToolVersion
