# <a name="fhconfigmgr-class"></a>Clase DesconfigMgr

Representa la Historial de archivos del usuario en el que se crea el objeto **DesconfigMgr.** Todas las operaciones de configuración se realizan mediante una llamada a los métodos de [**la interfaz IFhConfigMgr.**](/windows/desktop/api/Fhcfg/nn-fhcfg-ifhconfigmgr)

## <a name="when-to-implement"></a>Cuándo implementar

La API Historial de archivos implementa esta clase. Para crear una instancia de esta clase, use la [**función CoCreateInstance.**](/windows/win32/api/combaseapi/nf-combaseapi-cocreateinstance)
