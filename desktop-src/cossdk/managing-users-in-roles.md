# <a name="managing-users-in-roles"></a>Administración de usuarios en roles

La mayoría de las tareas de administración para configurar la seguridad de aplicaciones COM+ implican el rellenado de roles predefinidos con usuarios, tanto al instalar aplicaciones como de manera continua. Para obtener información introductoria sobre cómo funcionan estas características de seguridad, consulte Configuración de la [seguridad de Role-Based](configuring-role-based-security.md).

En esta sección, se proporcionan instrucciones paso a paso para las siguientes tareas:

-   [Habilitar la comprobación de la autorización de Role-Based](enabling-role-based-authorization-checking.md)
-   [Asignación de una cuenta de usuario o un grupo a un rol](assigning-a-user-account-or-group-to-a-role.md)
-   [Quitar una cuenta de usuario o un grupo de un rol](removing-a-user-account-or-group-from-a-role.md)
-   [Mover una cuenta de usuario o un grupo a un rol diferente](moving-a-user-account-or-group-to-a-different-role.md)
