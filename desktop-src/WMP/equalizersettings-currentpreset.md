# <a name="equalizersettingscurrentpreset"></a>EQUALIZERSETTINGS.currentPreset

El atributo **currentPreset** especifica o recupera el índice del valor preestablecido actual.

``` syntax
        elementID.currentPreset
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **número** de lectura y escritura (**Long**).

## <a name="remarks"></a>Observaciones

Si no se especifica este atributo, se conservará el valor anterior.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
