# <a name="xmfloat3pk-operators"></a>Operadores XMFLOAT3PK

La estructura [**XMFLOAT3PK**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmfloat3pk) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                               | Descripción                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**operador UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmfloat3pk-operatoruint32_t)<br/> | Devuelve una instancia de `uint32_t` que contiene los componentes de la `XMFLOAT3PK` instancia en un formato empaquetado.<br/> |
| [**operador =**](xmfloat3pk-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMFLOAT3PK` .<br/>                                       |
