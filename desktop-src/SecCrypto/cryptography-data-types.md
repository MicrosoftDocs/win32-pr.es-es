# <a name="cryptography-data-types"></a>Tipos de datos de criptografía

Las funciones de criptografía, las interfaces y los objetos usan los siguientes tipos de datos.



| Tipo de datos                                                                      | Descripción                                                                                                                                                                                      |
|--------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Id. de ALG \_**](alg-id.md)                                                      | Especifica identificadores de algoritmo.                                                                                                                                                                 |
| [**RESPUESTA \_ \_ OCSP DEL SERVIDOR HCERT \_**](hcert-server-ocsp-response.md)            | Representa un identificador para una respuesta OCSP asociada a una cadena de certificados de servidor.                                                                                                              |
| [**HCRYPTHASH**](hcrypthash.md)                                               | Especifica los identificadores de un [*objeto hash*](../secgloss/h-gly.md).                                                                                      |
| [**HCRYPTKEY**](hcryptkey.md)                                                 | Especifica los identificadores de las claves criptográficas.                                                                                                                                                         |
| [**HCRYPTOIDFUNCADDR**](hcryptoidfuncaddr.md)                                 | Representa un identificador para una función que se puede instalar mediante un identificador [*de objeto*](../secgloss/o-gly.md) (OID).                 |
| [**HCRYPTOIDFUNCSET**](hcryptoidfuncset.md)                                   | Representa un identificador para un conjunto de funciones que se pueden instalar mediante un OID.                                                                                                                 |
| [**HCRYPTPROV \_ HEREDADO**](hcryptprov-legacy.md)                                | Se usa para reemplazar el tipo de datos [**HCRYPTPROV**](hcryptprov.md) donde ya no se usa el tipo de datos **HCRYPTPROV.**                                                                             |
| [**IDENTIFICADOR DE CLAVE HCRYPTPROV \_ \_ O NCRYPT \_ \_**](hcryptprov-or-ncrypt-key-handle.md) | Especifica un identificador para un proveedor de servicios [*criptográficos*](../secgloss/c-gly.md) CryptoAPI (CSP) o CSP de CNG. |
| [**HCRYPTPROV**](hcryptprov.md)                                               | Especifica un identificador para un CSP de CryptoAPI.                                                                                                                                                           |
| [**IDENTIFICADOR \_ KEYSVCC**](keysvcc-handle.md)                                      | Especifica los identificadores del servicio de claves.                                                                                                                                                            |
