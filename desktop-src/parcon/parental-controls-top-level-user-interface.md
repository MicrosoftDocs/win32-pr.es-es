# <a name="parental-controls-top-level-user-interface"></a>Control parental Top-Level interfaz de usuario

El vínculo fuerte entre las cuentas de seguridad de la familia y de Windows ha impulsado una categoría combinada que aparece como **cuentas de usuario y seguridad** de la familia en el panel de control para las SKU orientadas a clientes de Windows Vista.

> [!Note]  
> La categoría aparecerá como **cuentas de usuario** cuando se combine un equipo con la funcionalidad de unión a dominio.

 

Si aún no se ha configurado una cuenta de usuario estándar para el usuario controlado, un vínculo en el panel de control proporciona un flujo de trabajo simple para agregar una nueva cuenta controlada de forma jerárquica.

Después de crear una cuenta, el panel de control de controles parentales expone la funcionalidad orientada al usuario de nivel superior para el usuario controlado. Elemento

-   Restricciones generales de controles parentales en o desactivar los botones de radio.
-   Informes de actividades independientes activar o desactivar botones de radio control.
-   Una sección de configuración con el vínculo restricciones web implementadas por Microsoft y que se muestra condicionalmente y los tipos de restricciones principales no implementados por Microsoft que se muestran.
-   Un área de configuración más que aparece si las extensiones de la interfaz de usuario están registradas por Microsoft o por terceros.
-   Un área de Resumen de estado que muestra el nombre de usuario y el icono, el vínculo ver informes de actividad y el estado de la configuración del control parental. Si el valor es on, la pantalla incluye el nivel de configuración de filtro de contenido web (si el filtro de Microsoft está activo) o el nombre de filtro, si se reemplaza por una extensión de terceros, y el estado de la configuración sin conexión para el usuario.

El botón de radio general de habilitar controles parentales está establecido inicialmente en estado desactivado. Cuando está activado por un administrador, la característica informes de actividades también estará activada de forma predeterminada, pero puede desactivarse de forma independiente. Al igual que con casi todas las opciones de control parental, la configuración puede realizarse mediante programación a través de las API disponibles.
