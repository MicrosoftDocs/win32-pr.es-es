# <a name="locale_snan"></a>configuración regional \_ SNAN

**Windows Vista y versiones posteriores:** Valor de cadena para "no es un número", por ejemplo, "Nan" para la configuración regional en inglés (Estados Unidos). No hay ningún límite en el número de caracteres permitidos para esta cadena.
