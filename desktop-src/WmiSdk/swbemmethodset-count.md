# <a name="swbemmethodsetcount-property"></a>Propiedad SWbemMethodSet.Count

La **propiedad Count** del objeto [**SWbemMethodSet**](swbemmethodset.md) define el número de elementos de la colección. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemMethodSet.Count As Integer
```
