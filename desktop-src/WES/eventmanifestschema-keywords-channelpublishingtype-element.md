# <a name="keywords-channelpublishingtype-element"></a>Keywords (ChannelPublishingType), elemento

Máscara de máscara que determina la categoría de eventos que se escriben en el canal.

``` syntax
<xs:element name="keywords"
    type="UInt64Type"
 />
```

El elemento **Keywords** se define mediante el tipo complejo [**ChannelPublishingType**](eventmanifestschema-channelpublishingtype-complextype.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows Vista \[\]<br/>       |
| Servidor mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows Server 2008 \[\]<br/> |
