# <a name="address-family"></a>Familia de direcciones

Las familias de direcciones incluyen Internet Packet Exchange (IPX), Internet Protocol Version 4 (IPv4) y Internet Protocol Version 6 (IPv6). Las familias de direcciones también se pueden denominar familias de protocolos. Estas familias de direcciones se definen en [RFC 1700.](routing-protocols-request-for-comments.md)
