# <a name="authoring-merge-module-summary-information-streams"></a>Crear secuencias de información de resumen del módulo de combinación

Al crear un módulo de combinación, es necesario que la secuencia de información de Resumen incluya estas propiedades de información de Resumen necesarias.

Cada módulo de combinación debe tener una secuencia de información de resumen del módulo de combinación con las siguientes propiedades de información de Resumen:

-   [**Resumen de plantillas**](template-summary.md)
-   [**Resumen de número de revisión**](revision-number-summary.md)
-   [**Resumen de recuento de páginas**](page-count-summary.md)
-   [**Resumen de recuento de palabras**](word-count-summary.md)

Para obtener una lista completa de las propiedades de información de resumen en los módulos de combinación, vea [referencia de flujo de información de resumen del módulo de combinación](merge-module-summary-information-stream-reference.md).
