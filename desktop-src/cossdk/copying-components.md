# <a name="copying-components"></a>Copiar componentes

Puede copiar un componente de una aplicación COM+ a otra. Después de copiar un componente en otra aplicación COM+, puede configurar este componente de forma diferente que el componente original.

**Para copiar un componente de una aplicación COM+ a otra**

1.  En el panel de detalles de la herramienta administrativa Servicios de componentes, haga clic con el botón derecho en el componente que desea copiar y, a continuación, haga clic **en Copiar**.

2.  En el **cuadro de** diálogo  Copiar componente, en el panel Destino, seleccione la aplicación COM+ en la que se copiará el componente.

3.  Escriba el nuevo ProgID para el componente copiado.

    > [!Note]  
    > El ProgID del componente original se muestra como referencia y no se puede cambiar.

     

4.  Escriba el nuevo CLSID para el componente copiado. COM+ muestra un CLSID generado automáticamente. En la mayoría de los casos, esto será suficiente para identificar de forma única el componente copiado. Escriba un nuevo CLSID solo si desea cambiar el proporcionado por COM+.

5.  Haga clic en **Aceptar**.
