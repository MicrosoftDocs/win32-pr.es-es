# <a name="header-files"></a>Archivos de encabezado

Aplicaciones que usan:

-   Las funciones básicas de OpenGL deben incluir el archivo de encabezado <GL \\ GL. h>.
-   La biblioteca de la utilidad OpenGL debe incluir el archivo de encabezado <GL \\ Glu. h>.
-   La biblioteca auxiliar de la guía de programación de OpenGL debe incluir el archivo de encabezado <GL \\ Glaux. h>.
-   Las funciones WGL deben incluir el archivo de encabezado Windows. h.
-   Las funciones de Windows que admiten la implementación de OpenGL en Windows de Microsoft deben incluir el archivo de encabezado Windows. h.
