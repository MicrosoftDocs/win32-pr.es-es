# <a name="default-values-in-the-spi"></a>Valores predeterminados en el SPI

Los detalles de implementación de calidad de servicio (QoS) se proporcionan en una sección de QoS dedicada y independiente en el kit de desarrollo de software (SDK) de Microsoft Windows. Consulte la sección [calidad de servicio](/previous-versions/windows/desktop/qos/qos-start-page) del Windows SDK para obtener información sobre las directrices de uso de QoS relativas a los valores predeterminados.
