# <a name="function-status"></a>Estado de la función

Las funciones de administración de red devuelven cero si se ejecuta correctamente; Un código de retorno distinto de cero indica un error. Dado que las funciones de administración de red usan RPC, las definiciones de error incluyen códigos de error rpc. Para obtener más información, vea [Códigos de error de administración de redes.](network-management-error-codes.md)
