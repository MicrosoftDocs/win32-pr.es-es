<span data-ttu-id="b59e9-112">El SDK de Windows Media Format proporciona métodos por los que una aplicación puede interactuar con el sitio web de un emisor de licencias para revocar una licencia de contenido.</span><span class="sxs-lookup"><span data-stu-id="b59e9-112">The Windows Media Format SDK provides methods by which an application can interact with a license issuer's Web site to revoke a content license.</span></span>
