# <a name="windows-design-basics"></a>Windows (conceptos básicos del diseño)

> [!NOTE]
> Esta guía de diseño se ha creado para Windows 7 y no se ha actualizado para las versiones más recientes de Windows. Gran parte de la guía se sigue aplicando en principio, pero la presentación y los ejemplos no reflejan nuestra [Guía de diseño actual](/windows/uwp/design/).

Windows son las principales "lienzos" o superficies de interfaz de usuario de la aplicación de escritorio, incluidas las ventanas principales y los elementos emergentes, cuadros de diálogo y asistentes. Siga estas instrucciones a la hora de decidir qué superficie usar y cuál es la mejor manera de usarlas.
