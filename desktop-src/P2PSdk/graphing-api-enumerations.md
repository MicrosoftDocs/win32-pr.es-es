# <a name="graphing-api-constants"></a>Constantes de la API de gráficos

La API de gráficos usa las siguientes constantes:



| Constante                                                          | Descripción                                                       |
|-------------------------------------------------------------------|-------------------------------------------------------------------|
| [**\_tipo de \_ evento del grafo del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_event_type)         | Especifica los tipos de evento del mismo nivel para los que se va a notificar la aplicación. |
| [**\_marcas de \_ propiedad del grafo del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_property_flags) | Especifica las propiedades de un grafo del mismo nivel.                             |
| [**\_ámbito del grafo del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_scope)                    | Especifica el ámbito de red de un grafo del mismo nivel.                      |
| [**\_marcas de \_ Estado del grafo del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_graph_status_flags)     | Especifica el estado actual de un nodo dentro del grafo del mismo nivel.     |
| [**\_tipo de \_ cambio de nodo del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_node_change_type)         | Especifica los tipos de cambios de estado del grafo del nodo del mismo nivel.            |
