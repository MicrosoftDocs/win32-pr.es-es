# <a name="assembling-primitives"></a>Ensamblar primitivos

Una vez realizados todos los cálculos necesarios, los vértices se ensamblan en puntos [primitivos](primitives.md) , segmentos de línea o polígonos junto con la marca de borde, el color y la información de textura correspondientes para cada vértice.
