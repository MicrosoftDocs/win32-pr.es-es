<dl> <dt>

<span id="opengl_network"></span><span id="OPENGL_NETWORK"></span>**Red**
</dt> <dd>

Una conexión entre dos o más equipos que permite a cada uno transferir datos hacia y desde los demás.

</dd> <dt>

<span id="opengl_nonconvex"></span><span id="OPENGL_NONCONVEX"></span>**nonconvex**
</dt> <dd>

Estado de un polígono en el que una línea del plano del polígono forma una intersección con el polígono más de dos veces.

</dd> <dt>

<span id="opengl_normal"></span><span id="OPENGL_NORMAL"></span>**Normal**
</dt> <dd>

Ecuación de plano de tres componentes que define la orientación angular, pero no la posición, de un plano o superficie.

</dd> <dt>

<span id="opengl_normalize"></span><span id="OPENGL_NORMALIZE"></span>**Normalizar**
</dt> <dd>

Para dividir cada uno de los componentes de un normal por la raíz cuadrada de la suma de sus cuadrados. A continuación, si lo normal se piensa como un vector desde el origen hasta el punto (nx', ny', nx'), este vector tiene longitud de unidad:

<dl> <dd>nx' = nx/factor</dd> <dd>ny' = ny/factor</dd> <dd>nz' = nz/factor</dd> </dl> </dd> <dt>

<span id="opengl_normal_vector"></span><span id="OPENGL_NORMAL_VECTOR"></span>**vector normal**
</dt> <dd>

Vea *normal*.

</dd> <dt>

<span id="opengl_nurbs"></span><span id="OPENGL_NURBS"></span>**Nurbs**
</dt> <dd>

Spline B no uniforme lógica. Una manera común de especificar curvas y superficies paramétricas.

</dd> </dl>
