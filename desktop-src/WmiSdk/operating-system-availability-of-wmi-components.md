# <a name="operating-system-availability-of-wmi-components"></a>Disponibilidad del sistema operativo de los componentes WMI

WMI se instala de forma predeterminada en todas las Windows de escritorio y servidor. Sin embargo, es posible que determinados proveedores WMI se instalen o no, dependiendo de la versión y configuración del sistema operativo. Por ejemplo, el proveedor SNMP no está habilitado de forma predeterminada y el proveedor del instalador de Windows (proveedor MSI) no está instalado de forma predeterminada en sistemas operativos de 64 bits. Para obtener más información sobre la disponibilidad de un proveedor específico, vea [Proveedores WMI](/windows/desktop/SrvNodes/wmi-mi-omi-providers).
