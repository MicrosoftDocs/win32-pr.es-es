# <a name="removal-of-microsoft-agent"></a>Eliminación de Microsoft Agent

## <a name="platform"></a>Plataforma

**Clientes** : Windows 7  
**Servidores** : Windows Server 2008 R2  









## <a name="feature-impact"></a>Impacto en las características

 **Gravedad** : medio  
**Frecuencia** : baja  


## <a name="description"></a>Descripción

Microsoft ha decidido interrumpir el desarrollo de las tecnologías de Microsoft Agent. A partir de Windows 7, el agente de Microsoft no se incluirá ni admitirá en versiones futuras del sistema operativo Microsoft Windows. Se recomienda a los desarrolladores y redistribuidores de aplicaciones de agente de Microsoft evaluar sus actividades a la luz de esta decisión.

## <a name="manifestation"></a>Manifestación

-   Microsoft no ofrecerá licencias ni renovaciones nuevas
-   Microsoft no actualizará los componentes de Microsoft Agent, entre los que se incluyen:
    -   Biblioteca en tiempo de ejecución de "msagent.exe" y bibliotecas
    -   Los caracteres Merlin, genio, Peedy y Robbo
    -   Editor de caracteres del agente
    -   Todo el software, las herramientas y la documentación relacionados con Microsoft Agent
-   Windows 7 y versiones posteriores no admiten el desarrollo de aplicaciones o caracteres del agente de Microsoft

## <a name="mitigation"></a>Mitigación

Los desarrolladores pueden seguir usando los siguientes recursos para las versiones anteriores de Windows:

-   [Componentes principales de Microsoft Agent (solo para su uso en Microsoft Windows 95, Windows 98 y Windows NT 4,0)](https://www.microsoft.com/downloads/details.aspx?familyid=E11BF712-7862-45BA-826D-44AE3A11836F)
-   [Archivo de caracteres Merlin](https://www.microsoft.com/downloads/details.aspx?FamilyID=fee1dadd-2f23-41d0-8a81-2affd74c0aa5)
-   [Archivo de caracteres Peedy](https://www.microsoft.com/downloads/details.aspx?FamilyID=bd3c4655-79e4-4791-ab9d-abc7bbd133ef)
-   [Archivo de caracteres de genio](https://www.microsoft.com/downloads/details.aspx?FamilyID=da86ba4e-bc2d-4c1d-b5a0-3183fe206414)
-   [Archivo de caracteres Robby](https://www.microsoft.com/downloads/details.aspx?FamilyID=fa36d1d5-d828-494a-ad0a-7b571db5bd2e)
