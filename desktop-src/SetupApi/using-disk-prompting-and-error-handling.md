# <a name="using-disk-prompting-and-error-handling"></a>Usar la solicitud de disco y el control de errores

Los cuadros de diálogo creados por las funciones de configuración, [**SetupPromptForDisk**](/windows/desktop/api/Setupapi/nf-setupapi-setuppromptfordiska), [**SetupCopyError**](/windows/desktop/api/Setupapi/nf-setupapi-setupcopyerrora), [**SetupDeleteError**](/windows/desktop/api/Setupapi/nf-setupapi-setupdeleteerrora)y [**SetupRenameError**](/windows/desktop/api/Setupapi/nf-setupapi-setuprenameerrora) se pueden usar en una aplicación de instalación o mediante rutinas de devolución de llamada. En las secciones siguientes se describe el uso de las funciones de una aplicación y sus notificaciones correspondientes:

-   [Solicitar un disco](prompting-for-a-disk.md)
-   [Tratamiento de errores](error-handling.md)
