# <a name="clsid-attribute"></a>Atributo CLSID

> [!Note]  
> \[En desuso. Esta API se puede quitar de las versiones futuras de Windows.\]

 

El `clsid` atributo especifica un identificador de clase (CLSID) para un objeto.

## <a name="possible-values"></a>Valores posibles

Debe ser una cadena con el formato {00000000-0000-0000-0000-000000000000} .

## <a name="applies-to"></a>Se aplica a

[**en**](at-element.md), [**efecto**](effect-element.md), [**transición**](transition-element.md)
