# <a name="ms-com-partitionlink-attribute"></a>atributo MS-COM-PartitionLink

Un vínculo que se usa para asociar un PartitionSet COM+ con sus particiones COM+ constituyentes.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-COM-PartitionLink                    |
| Nombre para mostrar de LDAP | msCOM-PartitionLink                     |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1423                 |
| System-ID-GUID    | 09abac62-043f-4702-AC2B-6ca15eee5754    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
