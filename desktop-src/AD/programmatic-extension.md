# <a name="programmatic-extension"></a>Extensión mediante programación

La ventaja de un archivo LDIF es que el administrador puede examinar su función. Sin embargo, el esquema también se puede extender mediante programación. Las extensiones mediante programación tienen varios fundamentos, pero una aplicación es opaca. el administrador debe confiar en la documentación incluida con la aplicación de instalación. El uso de extensiones de esquema mediante programación puede ser una barrera para la implementación de las aplicaciones que las usan. Una extensión mediante programación es invariable; es un ejecutable Windows NT. El archivo binario no se puede manipular, a diferencia de un archivo LDIF o .csv, que se puede editar de forma involuntaria o malintencionada.

Las ventajas de un archivo LDIF incluyen:

-   Las aplicaciones pueden detectar y recuperarse de errores y proporcionar comentarios de los usuarios.
-   Las aplicaciones pueden controlar Unicode sin recurrir a la codificación Base64.
-   Las aplicaciones pueden aprovechar Windows api de instalación del instalador.
-   Las aplicaciones se pueden firmar para establecer la autenticidad.
