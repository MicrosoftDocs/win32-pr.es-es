# <a name="performance-dimensions"></a>Dimensiones de rendimiento

El rendimiento de la aplicación es distinto en función de la perspectiva. Para un usuario, una aplicación ideal ofrecer un alto rendimiento y capacidad de respuesta. Para un administrador, una aplicación ideal consumiría la menor cantidad de recursos de red. El rendimiento también difiere en función del tipo de aplicación. El rendimiento de una aplicación de transferencia de archivos masiva puede tener implicaciones diferentes que el rendimiento de una aplicación transaccional.

En las secciones siguientes se tratan las distintas dimensiones de rendimiento y los tipos de aplicación, y cómo afectan al rendimiento los entornos de red.

-   [Aplicaciones transaccionales frente a streaming](transactional-versus-streaming-applications-2.md)
-   [Entornos de red diferentes](different-network-environments-2.md)
