# <a name="shell-macros"></a>Macros de Shell

En esta sección se describen las macros de la utilidad Shell de Windows.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                  | Descripción                                                                                                                                                                                                                                                     |
|------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**DEFINIR \_ PROPERTYKEY**](/windows/desktop/api/Propkeydef/nf-propkeydef-define_propertykey)<br/>           | Se usa para empaquetar un identificador de formato (FMTID) y un identificador de propiedad (PID) en una estructura [**PROPERTYKEY**](/windows/win32/api/wtypes/ns-wtypes-propertykey) que representa una clave de propiedad.<br/>                                                                                    |
| [**IID \_ PPV \_ args**](/windows/win32/api/combaseapi/nf-combaseapi-iid_ppv_args)<br/>                      | Se utiliza para recuperar un puntero de interfaz, proporcionando el valor de IID de la interfaz solicitada automáticamente según el tipo de puntero de interfaz utilizado. Esto evita un error de codificación común al comprobar el tipo del valor que se pasa en tiempo de compilación.<br/> |
| [**IsEqualPropertyKey**](/windows/desktop/api/Propkeydef/nf-propkeydef-isequalpropertykey)<br/>            | Compara los miembros de dos estructuras [**PROPERTYKEY**](/windows/win32/api/wtypes/ns-wtypes-propertykey) y devuelve si son iguales.<br/>                                                                                                                                 |
| [**MAKEDLLVERULL**](/windows/desktop/api/Shlwapi/nf-shlwapi-makedllverull)<br/>                      | Se usa para empaquetar la información de versión de la DLL en un valor de ULONGLONG.<br/>                                                                                                                                                                                         |
| [**NetAddr \_ DisplayErrorTip**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_displayerrortip)<br/> | Muestra un mensaje de error en el globo de sugerencias asociado con el control de dirección de red.<br/>                                                                                                                                                            |
| [**NetAddr \_ GetAddress**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_getaddress)<br/>           | Indica si una dirección de red se ajusta a un tipo y formato especificados.<br/>                                                                                                                                                                         |
| [**NetAddr \_ GetAllowType**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_getallowtype)<br/>       | Recupera los tipos de dirección de red que un control de dirección de red especificado acepta.<br/>                                                                                                                                                                |
| [**NetAddr \_ SetAllowType**](/windows/desktop/api/Shellapi/nf-shellapi-netaddr_setallowtype)<br/>       | Establece los tipos de dirección de red que acepta un control de dirección de red especificado.<br/>                                                                                                                                                                     |
