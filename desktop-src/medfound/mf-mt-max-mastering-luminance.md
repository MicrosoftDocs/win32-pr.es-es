# <a name="mf_mt_max_mastering_luminance-attribute"></a>MF \_ MT \_ Max \_ Mastering \_ luminancia (atributo)

Especifica la luminancia máxima de la pantalla en la que se creó el contenido, en Nits. Este atributo tiene la misma semántica que la **\_ \_ \_ luminancia de maestro de visualización máxima** , tal y como se define en el estándar CEA-861,3.

## <a name="data-type"></a>Tipo de datos

**UINT32**
