# <a name="dns-wmi-provider-scripting-examples"></a>Ejemplos de scripting del proveedor WMI de DNS

En esta sección se proporcionan ejemplos de scripting que muestran cómo crear scripts de tareas administrativas mediante el proveedor WMI de DNS.

Se proporcionan los siguientes tipos de ejemplos de scripting.



| Sección                                                                                     | Ejemplos proporcionados                                                                                                                                                                          |
|---------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Administración de un servidor DNS](dns-wmi-provider-samples-managing-a-dns-server.md)                 | Proporciona ejemplos que permiten la administración de un servidor DNS, como detener, iniciar o reiniciar un servidor, enumerar zonas, agregar o eliminar direcciones IP, etc.                |
| [Administración de registros de recursos DNS](dns-wmi-provider-samples-managing-dns-resource-records.md) | Muestra cómo conectarse al proveedor WMI de DNS y cómo realizar varias tareas asociadas a la administración de registros de recursos: agregar, eliminar, modificar o enumerar registros de recursos. |
| [Administración de zonas DNS](dns-wmi-provider-samples-managing-dns-zones.md)                       | Estos ejemplos incluyen la creación, modificación y eliminación de una zona, así como pausar, reanudar, actualizar y otras tareas de zona que se planteen a los administradores de DNS.                                  |
