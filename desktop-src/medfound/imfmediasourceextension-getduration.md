# <a name="imfmediasourceextensiongetduration-method"></a>MÉTODO IMFMediaSourceExtension::GetDuration

Obtiene la duración del origen multimedia en unidades de 100 nanosegundos.

## <a name="syntax"></a>Sintaxis


```C++
double GetDuration();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Duración del origen multimedia en unidades de 100 nanosegundos.
