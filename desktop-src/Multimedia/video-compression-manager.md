# <a name="video-compression-manager"></a>Administrador de compresión de vídeo

El administrador de compresión de vídeo (VCM) proporciona acceso a la interfaz utilizada por los compresores instalables para administrar los datos en tiempo real. Las aplicaciones pueden usar compresores instalables para realizar las siguientes tareas:

-   Comprimir y descomprimir los datos de vídeo.
-   Envíe datos de vídeo comprimidos del representador y colóquelos en la pantalla.
-   Comprimir, descomprimir o dibujar datos con representadores definidos por la aplicación.
-   Usar representadores para controlar texto y datos personalizados.
