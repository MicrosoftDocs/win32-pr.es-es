# <a name="opengl-on-windows"></a>OpenGL en Windows

La implementación de OpenGL en el sistema operativo Microsoft Windows es una implementación de la interfaz de software de gráficos OpenGL tridimensionales (3D) estándar del sector con la que los programadores crean imágenes de color 3D aún y animadas de alta calidad. En esta información general se describe la implementación de Windows de OpenGL.
