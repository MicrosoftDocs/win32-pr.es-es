# <a name="d1149-print-detected-an-invalid-unicode-character"></a>D1149: Print detectó un carácter Unicode no válido

Impresión de Direct2D: se detectó un carácter Unicode no válido en la descripción de ejecución del glifo de DWRITE \_ \_ \_ . cadena. Se ha reemplazado por Space (0x20).






 

## <a name="possible-causes"></a>Causas posibles

La impresión en Direct2D tiene en cuenta que los caracteres no válidos que se indican en la [documentación de Unicode](https://unicode.org/faq/utf_bom.html#utf16-7), así como ciertos caracteres ASCII fuera del intervalo visible, no se pueden imprimir. El uso de estos caracteres no produce un error. En su lugar, la impresión en Direct2D sustituirá estos caracteres por espacios.

## <a name="possible-fixes"></a>Posibles correcciones

Para evitar esta notificación, no use caracteres Unicode con formato incorrecto o ASCII no imprimible mientras dibuja texto para la impresión.
