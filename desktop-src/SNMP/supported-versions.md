# <a name="supported-versions"></a>Versiones admitidas

\[SNMP está disponible para su uso en los sistemas operativos especificados en la sección de requisitos. En versiones posteriores podría modificarse o no estar disponible. En su lugar, use [administración remota de Windows](/windows/desktop/WinRM/portal), que es la implementación de Microsoft de WS-MAN.\]

Windows Vista y Windows Server 2008 admiten SNMPv1 y SNMPv2c. En este tema se proporciona información básica acerca de cada una de estas versiones de SNMP.

La versión 1 del Protocolo simple de administración de redes (SNMPv1) es la versión más antigua del protocolo. SNMPv1 se define mediante RFC 1157, RFC 1155 y RFC 1212. Para SNMPv1, la seguridad se basa en las cadenas de la comunidad.

La versión 2C del Protocolo simple de administración de redes (SNMPv2c) se define en RFC 1901, RFC 1905 y RFC 1906. SNMPv2c actualiza las operaciones de protocolo y los tipos de datos que se definieron en SNMPv2p. SNMPv2c, denominado "SNMPv2 basado en cadena de comunidad", usa la seguridad basada en la comunidad de SNMPv1.

SNMPv2x es compatible con las plataformas de sistema operativo cliente y servidor de Windows 2000 y versiones posteriores, como Windows XP y Windows Server 2003.
