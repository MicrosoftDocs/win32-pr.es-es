# <a name="mf_user_data_payload-attribute"></a>\_Atributo de \_ carga de datos de usuario MF \_

Establece si se debe incluir una carga de datos de usuario con el ejemplo de salida.

## <a name="data-type"></a>Tipo de datos

**BLOB**

## <a name="remarks"></a>Observaciones

Los datos de usuario pueden contener datos como datos de título de cierre o datos de barras.

Los datos de usuario se pasan sin ninguna descodificación.
