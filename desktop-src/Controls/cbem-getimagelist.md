# <a name="cbem_getimagelist-message"></a>Mensaje \_ GETIMAGELIST de CBEM

Obtiene el identificador de una lista de imágenes asignada a un control ComboBoxEx.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>Debe ser cero.</dd> <dt>

*lParam* 
</dt> <dd>Debe ser cero.</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Devuelve el identificador a la lista de imágenes asignada al control si se realiza correctamente o **NULL** en caso contrario.
