# <a name="inf-file-structures"></a>Estructuras de archivo INF

Las estructuras siguientes se usan con archivos INF.



| Estructura                                              | Descripción                        |
|--------------------------------------------------------|------------------------------------|
| [**INFCONTEXT**](/windows/desktop/api/Setupapi/ns-setupapi-infcontext)                   | Contexto posicional en un archivo INF. |
| [**INFORMACIÓN \_ DE SP INF \_**](/windows/desktop/api/Setupapi/ns-setupapi-sp_inf_information) | Información sobre el archivo INF.    |



 

> [!Note]  
> Las funciones de instalación usan internamente la estructura INFCONTEXT y las aplicaciones de instalación no deben modificar ni hacer referencia a la misma. Se incluye aquí solo con fines informativos.
