# <a name="effect-reference-direct3d-10-graphics"></a>Referencia de efecto (gráficos de Direct3D 10)

La API de Direct3D define varios elementos de API para ayudarle a crear y administrar el sistema de efectos.

-   [Interfaces](d3d10-graphics-reference-effect-interfaces.md)
-   [Funciones](d3d10-graphics-reference-effect-functions.md)
-   [Estructuras](d3d10-graphics-reference-effect-structures.md)
-   [Constantes](d3d10-graphics-reference-effect-constants.md)

El sistema de efectos encapsula las asignaciones de estado en un efecto. Cada efecto es una colección de estado de canalización que se puede dividir en asignaciones de estado mediante bloques de estado, recursos y sombreadores. Dependiendo del tamaño de un efecto, puede optar por implementar uno en una cadena de texto ASCII o en un archivo de efecto (.fx). Para organizar la información en un efecto, vea el [formato de efecto](d3d10-effect-format.md).

Una vez diseñado un efecto, use la API de efecto para establecer el estado en el dispositivo durante la representación. Además, el sistema de efectos admite una serie de API de reflexión para obtener y establecer datos de efecto. Para obtener más información, [vea Effect System Interfaces (Direct3D 10)](d3d10-graphics-programming-guide-effects-interfaces.md).
