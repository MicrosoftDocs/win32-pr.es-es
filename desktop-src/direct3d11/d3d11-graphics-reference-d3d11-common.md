# <a name="common-version-reference"></a>Referencia de la versión común

La API de Direct3D define varios elementos de API comunes a Direct3D 12, Direct3D 11, Direct3D 10 y Direct3D 10,1. Puede usar estos elementos de la API en el código para cualquiera de estas versiones de Direct3D. Estos elementos de la API se conocen como versiones neutras.
