# <a name="encoding-windows-web-services"></a>Codificación (servicios Web de Windows)

La codificación es una transformación entre XML y un formato de conexión lineal. La descodificación es la transformación inversa.


Una codificación se identifica mediante una constante [**de \_ codificación de WS**](/windows/desktop/api/WebServices/ne-webservices-ws_encoding) .
