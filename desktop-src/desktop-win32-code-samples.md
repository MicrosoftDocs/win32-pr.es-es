# <a name="desktop-win32-code-samples"></a>Ejemplos de código Win32 de escritorio

Vea también [ejemplos de código](https://developer.microsoft.com/windows/samples/).

## <a name="find-a-sample-for-the-api-youre-interested-in"></a>Busque un ejemplo de la API que le interese

Puede usar Microsoft Visual Studio para buscar el código fuente de un repositorio completo y ver si se muestra el uso de una API de Windows determinada.

* Clone cualquiera de las repositorios enumeradas en las secciones siguientes (o descargue el archivo ZIP) en el sistema de archivos local.
* A continuación, **Busque en archivos** en Visual Studio. Establezca **Buscar en** la carpeta clonada o descargada en y busque el nombre de la API que le interesa. La comprobación de **coincidencia de mayúsculas y minúsculas** proporciona mejores resultados.
* Tenga cuidado al comprobar la **palabra completa de coincidencia** si se puede llamar a la API en varios formatos &mdash; , por ejemplo, **CreateMutex**, **CreateMutexA** y **CreateMutexW**. En ese caso, busque **CreateMutex** con **palabra completa coincidir** desactivada.

> [!NOTE]
> Puede instalar [Visual Studio](https://visualstudio.microsoft.com/downloads/) sin gastos. Hay disponible una edición Community &mdash;es gratuita para estudiantes, colaboradores de código abierto y particulares—. Por supuesto, puede hacer lo mismo con cualquier repositorio de ejemplos.

## <a name="the-windows-classic-samples-repo"></a>Repositorio de ejemplos de Windows clásico

El repositorio principal que contiene aplicaciones de ejemplo de Windows con la API de Win32 es el repositorio de [ejemplos de Windows clásico](https://github.com/Microsoft/Windows-classic-samples) .
