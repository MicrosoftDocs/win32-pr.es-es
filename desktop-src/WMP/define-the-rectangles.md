# <a name="define-the-rectangles"></a>Definir los rectángulos

Dado que no se pueden superponer los elementos de la interfaz de usuario, resulta útil definir los rectángulos en los que se dibujará cada elemento. Además, la creación de rectángulos resultará útil cuando tenga que anotar las coordenadas.

Active la cuadrícula de Photoshop con líneas de cuadrícula cada diez píxeles. Active ajustar a la cuadrícula para facilitar el dibujo de rectángulos.

Cree una nueva capa y dibuje rectángulos. Al tener la capa de esbozo debajo, conoce la ubicación general en la que desea que se encuentre el elemento de la interfaz de usuario, pero ahora puede ajustar la ubicación exacta. Siempre puede cambiar cosas más adelante, pero le ayudará a obtener el cierre como ya puede. En la ilustración siguiente se muestran varios rectángulos en una capa.

![capa de rectángulo](images/ceswmrek.png)

Ahora que se han definido los rectángulos, puede desactivar la capa de esbozo.
