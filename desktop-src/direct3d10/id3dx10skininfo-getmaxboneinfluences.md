# <a name="id3dx10skininfogetmaxboneinfluences-method"></a>ID3DX10SkinInfo:: GetMaxBoneInfluences (método)

Obtiene el número de vértices a los que puede afectar máximamente el hueso.

## <a name="syntax"></a>Sintaxis


```C++
UINT GetMaxBoneInfluences();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

El número de vértices a los que puede influir el hueso.
