# <a name="attribute-reference"></a>Referencia de atributo

La referencia de atributo de Windows Media Player documenta los atributos de metadatos que son de interés para los desarrolladores de Windows Media Player SDK. En la tabla siguiente se enumeran estos atributos en varias categorías.



| Category                                                                                       | Descripción                                                                                                              |
|------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [Referencia alfabética](alphabetical-attribute-reference.md)                                 | Proporciona una lista alfabética de atributos predefinidos.                                                                  |
| [Elementos de audio](audio-item-attributes.md)                                                       | Muestra los atributos que están disponibles para un elemento de audio de la biblioteca.                                                    |
| [Listas de reproducción de CD](cd-playlist-attributes.md)                                                     | Muestra los atributos que están disponibles para un CD.                                                                            |
| [Pistas de CD](cd-track-attributes.md)                                                           | Muestra los atributos que están disponibles para una pista de CD.                                                                      |
| [Atributos de archivo de Windows Media de uso frecuente](commonly-used-windows-media-file-attributes.md) | Muestra los atributos que están disponibles para un archivo multimedia digital.                                                            |
| [DVDs](dvd-attributes.md)                                                                     | Muestra los atributos que están disponibles para un DVD.                                                                           |
| [Archivos de música](music-file-attributes.md)                                                       | Muestra los atributos que están disponibles para un archivo de música que no está en la biblioteca.                                         |
| [Otros elementos](other-item-attributes.md)                                                       | Muestra los atributos que están disponibles para los elementos de la biblioteca que no son elementos de audio, vídeo, lista de reproducción, foto o radio. |
| [Elementos de fotografía](photo-item-attributes.md)                                                       | Muestra los atributos que están disponibles para un elemento de fotografía en la biblioteca.                                                     |
| [Reproducción](playlist-attributes-ref.md)                                                       | Muestra los atributos que están disponibles para las listas de reproducción recuperadas como elementos multimedia de la biblioteca.                             |
| [Elementos de radio](radio-item-attributes.md)                                                       | Muestra los atributos que están disponibles para un elemento de radio en la biblioteca.                                                     |
| [Elementos de vídeo](video-item-attributes.md)                                                       | Muestra los atributos que están disponibles para un elemento de vídeo en la biblioteca.                                                     |



 

Para obtener más información sobre los atributos de metadatos, vea las [directrices de uso de metadatos de Windows Media](/previous-versions/ms867702(v=msdn.10)).
