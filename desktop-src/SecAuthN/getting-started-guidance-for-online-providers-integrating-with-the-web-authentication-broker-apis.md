# <a name="getting-started-guidance-for-online-providers-integrating-with-the-web-authentication-broker-apis"></a>Guía de introducción para proveedores en línea que se integran con las API del Agente de autenticación web

Esta sección guía a los desarrolladores web y proveedores en línea para crear páginas web adaptadas a Windows 8 API del Agente de autenticación web. Proporciona recomendaciones visuales e interactivas para una experiencia de usuario de un extremo a otro de la página web, así como cómo habilitar las funcionalidades de inicio de sesión único (SSO).
