# <a name="programming-guide-xinput-game-controller-apis"></a>Guía de programación (API de controlador de juegos XInput)

Esta guía contiene información sobre cómo usar la API XInput para interactuar con el controlador de Xbox cuando se conecta a un Windows PC.

- [Tareas iniciales con XInput:](getting-started-with-xinput.md) esta sección contiene una descripción de XInput y describe cómo empezar a usar XInput en la aplicación.
- [Versiones de XInput:](xinput-versions.md) esta sección contiene información sobre las versiones de XInput.
- [XInput y DirectInput:](xinput-and-directinput.md) esta sección contiene información sobre las diferencias entre XInput y DirectInput, y cómo puede usar ambos en paralelo.
- [Dispositivos DirectInput y XUSB:](directinput-and-xusb-devices.md) contiene información sobre las asignaciones de dispositivos de la clase Xbox Common Controller (XUSB).
- [Subtipos XINPUT y Controller:](xinput-and-controller-subtypes.md) enumera los subtipos de controlador disponibles en XInput.

Para obtener más información sobre métodos de API específicos, vea la [Referencia de programación](programming-reference.md).
