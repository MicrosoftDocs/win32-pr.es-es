# <a name="conversion-plug-ins-programming-reference"></a>Referencia de programación de complementos de conversión

El kit de desarrollo de software (SDK) de Microsoft Windows Media Player admite una interfaz para los complementos de conversión que convierten un formato de archivo multimedia digital en otro. El SDK también especifica los requisitos para registrar formatos de archivo para la conversión por parte de un complemento determinado. En las secciones siguientes se describen estos detalles.



| Sección                                                                | Descripción                                                                                                                                                                                           |
|------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Registrar complementos de conversión](registering-conversion-plug-ins.md) | Describe cómo registrar formatos de archivos multimedia digitales para la conversión.                                                                                                                                  |
| [Interfaz IWMPConvert](/previous-versions/windows/desktop/api/wmpservices/nn-wmpservices-iwmpconvert)                               | Proporciona métodos para habilitar los complementos de conversión de Windows Media Player para convertir archivos multimedia digitales creados con tecnologías no proporcionadas por Microsoft, en formato de sistema avanzado (ASF). |
