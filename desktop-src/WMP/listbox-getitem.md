# <a name="listboxgetitem"></a>LISTBOX.getItem

El **método getItem** recupera el texto del elemento con el índice especificado.

``` syntax
        elementID.getItem(index)
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*Índice*
</dt> <dd>

**Number** (**long**) que contiene el índice de la línea que se recuperará.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Este método devuelve una **cadena**.

|--------------------|---------------------------------------------------------|
