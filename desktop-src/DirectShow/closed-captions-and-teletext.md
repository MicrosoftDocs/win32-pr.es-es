# <a name="closed-captions-and-teletext"></a>Subtítulos y teletexto

Las señales de televisión analógica pueden contener información adicional en el intervalo de blanking vertical (VBI), incluidos los subtítulos y el teletexto estándar universal (elemento WST). DirectShow proporciona un conjunto de filtros para descodificar y ver el texto de CC y elemento WST.

-   [Ver subtítulos (CC)](viewing-closed-captions.md)
-   [Ver teletexto estándar del mundo](viewing-world-standard-teletext.md)
