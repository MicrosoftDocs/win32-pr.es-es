# <a name="other-login-workstations-attribute"></a>Otro atributo-login-Workstations

Estaciones de trabajo que no son de Windows NT o LAN Manager desde las que un usuario puede iniciar sesión.

> [!Note]  
> Active Directory no usa ni rellena este campo.

 



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Otro: inicios de sesión-estaciones de trabajo                    |
| Nombre para mostrar de LDAP | otherLoginWorkstations                      |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.91                       |
| System-ID-GUID    | bf9679f1-0de6-11d0-a285-00aa003049e2        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
