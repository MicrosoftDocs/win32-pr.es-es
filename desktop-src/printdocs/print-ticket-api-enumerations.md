# <a name="print-ticket-api-enumerations"></a>Enumeraciones de API de vales de impresión

La API print ticket usa las enumeraciones siguientes.

## <a name="in-this-section"></a>En esta sección



| Enumeración                                                   | Descripción                                                                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**EDefaultDevmodeType**](/windows/win32/api/prntvpt/ne-prntvpt-edefaultdevmodetype)<br/> | Permite a los usuarios especificar qué [**DEVMODE**](/windows/win32/api/wingdi/ns-wingdi-devmodea) usar como origen de los valores predeterminados cuando un vale de impresión no especifica toda la configuración posible.<br/> |
| [**EPrintTicketScope**](/windows/desktop/api/prntvpt/ne-prntvpt-eprintticketscope)<br/>     | Especifica el ámbito de un vale de impresión.<br/>                                                                                                                       |
