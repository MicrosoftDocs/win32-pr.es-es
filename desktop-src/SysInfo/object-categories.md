# <a name="object-categories"></a>Categorías de objetos

El sistema proporciona tres categorías de objetos: usuario, interfaz de dispositivo gráfico (GDI) y kernel. El sistema utiliza objetos de usuario para admitir la administración de ventanas, objetos GDI para admitir gráficos y objetos de kernel para admitir la administración de memoria, la ejecución de procesos y las comunicaciones entre procesos (IPC). Para obtener información sobre cómo crear y usar un objeto específico, consulte la información general asociada.
