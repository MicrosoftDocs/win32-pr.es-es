# <a name="update-assessor-interfaces"></a>Actualizar las interfaces de asesor

El proveedor de descriptores de acceso de actualización usa las siguientes interfaces.



| Función                               | Descripción                                                                                                                 |
|----------------------------------------|-----------------------------------------------------------------------------------------------------------------------------|
| [**IWaaSAssessor**](/windows/desktop/api/waasapi/nn-waasapi-iwaasassessor) | Obtiene la evaluación de la actualización del sistema operativo comparando la compilación más reciente de Microsoft con la compilación que se ejecuta en el dispositivo actual. |
