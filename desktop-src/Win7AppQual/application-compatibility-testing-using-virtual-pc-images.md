# <a name="application-compatibility-testing-using-virtual-pc-images"></a>Prueba de compatibilidad de aplicaciones utilizando imágenes de Virtual PC

Para empezar a probar la compatibilidad de aplicaciones web, debe establecer un entorno de pruebas. El proceso para compilar y configurar equipos para esta tarea es lento y a menudo costoso.

Para facilitar las pruebas de compatibilidad, puede usar las imágenes gratuitas y precompiladas de compatibilidad de aplicaciones virtual (VPC). Las imágenes de VPC son imágenes de unidad de disco duro preconfiguradas que se pueden cargar en equipos existentes que ejecutan el [software de pc virtual](https://support.microsoft.com/help/14223/windows-xp-end-of-support). Estas imágenes permiten probar exhaustivamente varias versiones de Microsoft Windows y Windows Internet Explorer con aplicaciones existentes en un breve período de tiempo.

Las imágenes de VPC están disponibles para todas las versiones admitidas actualmente de Windows y contienen muchos recursos y aplicaciones de pruebas de compatibilidad. Para buscar las imágenes de prueba, consulte Internet Explorer [Application Compatibility IMAGE IMAGE (Imagen de LA VPC de compatibilidad de aplicaciones).](https://www.microsoft.com/Downloads/details.aspx?FamilyID=21eabb90-958f-4b64-b5f1-73d0a413c8ef&amp;displaylang=en)
