# <a name="tb_isbuttonchecked-message"></a>Mensaje \_ ISBUTTONCHECKED de TB

Determina si el botón especificado en una barra de herramientas está activado.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>

Identificador de comando del botón.

</dd> <dt>

*lParam* 
</dt> <dd>Debe ser cero.</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Devuelve un valor distinto de cero si el botón está activado o cero en caso contrario.
