# <a name="win32_methodparameterclass-class"></a>\_Clase Win32 MethodParameterClass

La [clase abstracta WMI](/windows/desktop/WmiSdk/retrieving-a-class) **\_ MethodParameterClass de Win32** deriva cualquier clase definida como contenedor de valores de parámetro que se pasarán a un método. No tener propiedades propias, esta clase actúa como nodo de clasificación. Un ejemplo similar es [**CIM \_ PhysicalElement**](cim-physicalelement.md). La derivación de **CIM \_ PhysicalElement** indica que la clase describe una entidad física en lugar de lógica. En el caso de **Win32 \_ MethodParameterClass**, las clases derivadas de ella se crean específicamente para pasar parámetros a un método.

## <a name="syntax"></a>Sintaxis

## <a name="members"></a>Miembros

La **clase \_ MethodParameterClass de Win32** no define ningún miembro.
