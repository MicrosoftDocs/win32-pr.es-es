# <a name="mf_devicestream_max_frame_buffers-attribute"></a>\_Atributo MF DEVICESTREAM \_ Max \_ Frame \_ buffers

Especifica el número máximo de fotogramas que el origen de captura de vídeo almacenará en búfer para esta secuencia.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|-------------------------------------|------------------------------------------------------------------------------------|
| Cliente mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows 8 \[\]<br/>                                         |
| Servidor mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows Server 2012 \[\]<br/>                               |
| Encabezado<br/>                   | <dl> <dt>Mfidl. h</dt> </dl> |
