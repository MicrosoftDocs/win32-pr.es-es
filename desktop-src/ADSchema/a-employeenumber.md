# <a name="employee-number-attribute"></a>Employee-Number atributo)

Número asignado a un empleado distinto del identificador.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Employee-Number                             |
| Nombre para mostrar de LDAP | employeeNumber                              |
| Tamaño              | \-                                          |
| Actualizar privilegio  | Administrador de dominio                        |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.2.610                      |
| System-ID-GUID    | a8df73ef-c5ea-11d1-bbcb-0080c76670c0        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
