# <a name="visible-property-commandswindow-object"></a>Propiedad visible (objeto CommandsWindow)

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Devuelve o establece si la ventana comandos de voz está visible (abrir).

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintáctica** 
</dt> <dd>

*agente * * *. CommandsWindow. visible* *  \[  =  *booleano*\]



| Parte      | Descripción                                                                                                                                                                             |
|-----------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| *boolean* | Una expresión booleana que especifica si la ventana comandos de voz está visible.<br/> **True** La ventana está visible.<br/> **False** La ventana está oculta (cerrada).<br/> |



 

</dd> </dl>

## <a name="remarks"></a>Observaciones

El servidor muestra la ventana en función de la ubicación y el tamaño establecidos por el usuario.

> [!Note]  
> El usuario puede invalidar esta propiedad.
