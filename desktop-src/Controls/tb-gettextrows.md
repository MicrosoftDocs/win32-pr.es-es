# <a name="tb_gettextrows-message"></a>TB \_ GETTEXTROWS message

Recupera el número máximo de filas de texto que se pueden mostrar en un botón de la barra de herramientas.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>Debe ser cero.</dd> <dt>

*lParam* 
</dt> <dd>Debe ser cero.</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Devuelve un valor INT que representa el número máximo de filas de texto que el control mostrará para un botón.
