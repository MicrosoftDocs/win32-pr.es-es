# <a name="directxmath-programming-guide"></a>Guía de programación de DirectXMath

DirectXMath proporciona una solución matemática optimizada para Windows.

## <a name="in-this-section"></a>En esta sección



| Tema                                                             | Descripción                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Introducción](pg-xnamath-getting-started.md)<br/>      | La biblioteca de DirectXMath implementa una interfaz óptima y portable para operaciones de álgebra aritmética y lineal en vectores de punto flotante de precisión sencilla (2D, 3D y 4D) o matrices (3 × 3 y 4 × 4). <br/>                                                    |
| [Novedades](pg-xnamath-whatsnew.md)<br/>                  | La biblioteca de DirectXMath se basa en la [versión 2,04 de la biblioteca SIMD de C++ matemática de XNA](https://walbourn.github.io/). Aquí se describe el modo en que DirectXMath difiere de la matemáticas de XNA y cómo difieren las versiones de DirectXMath. <br/> |
| [Migración de código](pg-xnamath-migration.md)<br/>             | En esta información general se describen los cambios necesarios para migrar el código existente mediante la biblioteca matemática de XNA a la biblioteca de DirectXMath.<br/>                                                                                                                                 |
| [Trabajo con D3DXMath](pg-xnamath-migration-d3dx.md)<br/> | D3DXMath es una biblioteca auxiliar matemática para aplicaciones Direct3D. <br/>                                                                                                                                                                                                |
| [Optimización del código](pg-xnamath-optimizing.md)<br/>         | En este tema se describen las consideraciones y las estrategias de optimización con la biblioteca de DirectXMath.<br/>                                                                                                                                                                |
| [Elementos internos de la biblioteca](pg-xnamath-internals.md)<br/>          | En este tema se describe el diseño interno de la biblioteca de DirectXMath.<br/>                                                                                                                                                                                         |
