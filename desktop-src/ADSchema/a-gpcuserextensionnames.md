# <a name="gpc-user-extension-names-attribute"></a>GPC: atributo User-Extension-names

Lo usa el objeto directiva de grupo para las directivas de usuario.



| Entrada | Value |
|-------------------|---------------------------------------------------------------------------------|
| CN                | GPC-User-Extension-names                                                        |
| Nombre para mostrar de LDAP | gPCUserExtensionNames                                                           |
| Tamaño              | Depende del número de extensiones del lado cliente que tienen una directiva en este GPO. |
| Actualizar privilegio  | Administrador de dominio o de directivas.                                                 |
| Frecuencia de actualización  | Cada vez que se actualiza un GPO a través de GPE.                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1349                                                         |
| System-ID-GUID    | 42a75fc6-783f-11d2-9916-0000f87a57d4                                            |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md)                                     |
