# <a name="manipulating-the-depth-buffer-direct3d-9"></a>Manipular el búfer de profundidad (Direct3D 9)

Los búferes de profundidad están asociados al dispositivo. Las aplicaciones deben trasladar los búferes de profundidad cuando establecen los destinos de representación. Los métodos [**IDirect3DDevice9:: GetDepthStencilSurface**](/windows/desktop/api) y [**IDirect3DDevice9:: SetDepthStencilSurface**](/windows/win32/api/d3d9helper/nf-d3d9helper-idirect3ddevice9-setdepthstencilsurface) se usan para manipular los búferes de profundidad.
