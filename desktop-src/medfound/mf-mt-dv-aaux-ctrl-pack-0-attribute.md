# <a name="mf_mt_dv_aaux_ctrl_pack_0-attribute"></a>\_Atributo MF MT \_ DV \_ AAUX \_ Ctrl \_ Pack \_ 0

Paquete de control de código fuente auxiliar de audio (AAUX) para el primer bloque de audio en un tipo de medio de vídeo digital (DV).

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

Este atributo corresponde al miembro **dwDVAAuxCtl** de la estructura [**DVINFO**](/windows/win32/api/strmif/ns-strmif-dvinfo) de DirectShow.

La constante GUID para este atributo se exporta desde mfuuid. lib.
