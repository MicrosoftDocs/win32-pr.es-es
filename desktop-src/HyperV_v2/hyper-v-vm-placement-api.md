# <a name="hyper-v-vm-placement-api"></a>API de selección de ubicación de máquinas virtuales de Hyper-V

La API de selección de ubicación de máquinas virtuales (VM) de Hyper-V contiene la información de compatibilidad de las máquinas virtuales o del sistema de hospedaje del equipo.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                    | Descripción                                                                                                                                                |
|--------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Compatibilidad de \_ MsvmVector**](msvm-compatibilityvector.md)<br/> | Hace referencia a la información de compatibilidad de una máquina virtual (VM) (cuando se ejecuta en un sistema de equipo de máquina virtual) o un host (cuando se ejecuta en un sistema de equipo host).<br/> |
