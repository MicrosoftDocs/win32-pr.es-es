# <a name="mergemodulecabinet"></a>MergeModule.CABinet

Todos los archivos entregados por el módulo de combinación deben almacenarse dentro de un archivo de archivador que se inserta como una secuencia en el almacenamiento estructurado del módulo de mezcla. En un módulo de combinación estándar, el nombre de este archivador es siempre: MergeModule.CABconjunto de inet. Para obtener más información, vea [Generating MergeModule.CABinet Cabinet Files](generating-mergemodule-cabinet-cabinet-files.md).
