# <a name="id3dxanimationcontrollergetmaxnumanimationsets-method"></a>ID3DXAnimationController:: GetMaxNumAnimationSets (método)

Obtiene el número máximo de conjuntos de animaciones que el controlador de animación puede admitir.

## <a name="syntax"></a>Sintaxis


```C++
UINT GetMaxNumAnimationSets();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

El número máximo de conjuntos de animaciones que el controlador puede administrar.
