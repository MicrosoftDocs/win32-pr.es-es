# <a name="r-upnp-apis"></a>R (API de UPnP)

Contiene términos relacionados con UPnP que comienzan por la letra R.

[A](a-gly.md) B [C](c-gly.md) [D](d-gly.md) [E](e-gly.md) F G [H I](h-gly.md) J K L M N [O](n-gly.md) [P P](p-gly.md) Q R [S](s-gly.md) T [U](u-gly.md) V W X Y Z

<dl> <dt>

<span id="upnp.r_1_gly"></span><span id="UPNP.R_1_GLY"></span>**directorio de recursos**
</dt> <dd>

El directorio de recursos es un directorio local para el dispositivo donde se encuentran archivos como los archivos de icono.

</dd> <dt>

<span id="upnp.r_2_gly"></span><span id="UPNP.R_2_GLY"></span>**dispositivo raíz**
</dt> <dd>

Un dispositivo raíz es el dispositivo de nivel superior de un árbol de dispositivos. Por ejemplo, si está accediendo a un dispositivo de combinación de VCR/TV denominado estación de visualización, hay tres dispositivos: la estación de visualización, la VCR y el televisor. La estación de visualización se considera el dispositivo raíz.

</dd> <dt>

<span id="upnp.r_3_gly"></span><span id="UPNP.R_3_GLY"></span>**dispositivo en ejecución**
</dt> <dd>

Un dispositivo en ejecución es un dispositivo que ya ha cargado su código antes del registro.

Se trata de dispositivos para los que no se conoce una descripción del servicio antes del registro.

Algunos ejemplos de estos dispositivos son los puentes, como un puente SCP de UPnP. Este puente registra los dispositivos en ejecución con el host del dispositivo.

</dd> </dl>
