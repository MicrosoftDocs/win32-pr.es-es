# <a name="introduction-windows-7-developer-guide"></a>Introducción (Windows 7 Developer Guide)

El desarrollo de software nunca ha sido más emocionante. El mundo nunca ha sido más conectado, interactivo y móvil. Las experiencias de usuario y el contenido nunca han tenido mayor fidelidad. Los avances en conectividad, movilidad, interfaces naturales, gráficos y medios permiten nuevos escenarios y oportunidades para los desarrolladores. Al mismo tiempo, las expectativas nunca han sido mayores para el rendimiento, la flexibilidad y la interoperabilidad. Una plataforma de desarrolladores de primera clase debe proporcionar una base flexible y completa para los desarrolladores y sus soluciones, lo que les permite crear aplicaciones que les diferencian de la competencia a la vez que maximizan sus inversiones.

El Windows operativo 7 es la plataforma esencial para los desarrolladores. Escuchando a nuestros clientes, mirando con antelación a los escenarios del mañana y a partir de las funcionalidades de Windows Vista, Microsoft ofrece a los desarrolladores una amplia variedad de opciones y funcionalidades dentro de la plataforma para desarrolladores de Windows, a la vez que permite a los desarrolladores ofrecer soluciones creativos que son móviles, conectadas, de alta fidelidad y proporcionan una experiencia de usuario muy intuitiva. Lo más importante es que los aspectos básicos de la plataforma para desarrolladores, como la seguridad, el rendimiento y la compatibilidad, son las principales prioridades Windows 7.

En esta guía se resumen los avances clave del desarrollador en cada una de las tres áreas siguientes:

[Solid Foundation](solid-foundation.md)

Las experiencias de usuario excelentes comienzan con una base sólida. Windows 7 ofrece una plataforma de desarrollo sólida y herramientas innovadoras que proporcionan más opciones, mayor compatibilidad de aplicaciones, mejor rendimiento y compatibilidad sofisticada con documentos. Esto da como resultado un entorno sencillo y más confiable para las aplicaciones y uno familiar e intuitivo para los usuarios.

Hemos mantenido la interfaz de usuario coherente, predecible y fácil de usar, al tiempo que se ha agregado compatibilidad multi táctil, desplazamiento y otras características de interacción intuitivas que aprovechan las funcionalidades más recientes de pc y dispositivo móvil. Con Windows 7, los desarrolladores tienen la capacidad de crear las experiencias de usuario correctas para sus aplicaciones. (Consulte [las Windows de interacción de la experiencia del usuario).](/windows/apps/desktop/)

[Experiencias de aplicación más enriquecciones](richer-application-experiences.md)

Windows 7 permite a los desarrolladores compilar aplicaciones hoy en día que se ejecutarán en los equipos de mañana. Los dispositivos y software habilitados para el servicio, como teléfonos móviles, reproductores multimedia portátiles y cámaras digitales, exigen conectividad constante y aplicaciones avanzadas. Windows 7 ofrece una plataforma para ambos, lo que facilita a los desarrolladores aprovechar las funcionalidades y características del hardware de próxima generación, al tiempo que garantiza que los usuarios estén siempre activos y siempre conectados. (Vea [Windows Centro para desarrolladores de hardware).](https://www.microsoft.com/whdc/default.mspx)

[Lo mejor de Windows web](the-best-of-windows-and-the-web.md)

Con Windows 7, es fácil conectarse y permanecer conectado. Windows redes ofrece a los desarrolladores opciones para mejorar el almacenamiento en caché y el uso compartido de datos para mejorar el rendimiento de la red. Los diagnósticos de red de Windows 7 dan a los desarrolladores información pertinente para supervisar los problemas de red. Windows Internet Explorer 8 es la plataforma esencial para crear experiencias web rápidas y enriquecciones. Tanto si el objetivo es conectar dispositivos, conectarse a una red o conectarse a *Internet,* Windows 7 proporciona una plataforma mejorada. (Vea [Desarrollo web).](https://msdn.microsoft.com/library/aa155073.aspx)
