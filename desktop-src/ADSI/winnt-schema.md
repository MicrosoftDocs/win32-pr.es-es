# <a name="winnt-schema"></a>Esquema de Winnt

El esquema WinNT define el conjunto de objetos ADSI y su jerarquía de clases para una red Windows NT 4,0. Los [objetos ADSI de WinNT](adsi-objects-of-winnt.md) incluyen usuario, equipo, dominio, grupo, recurso, sesión, servicio y otros. La sección [jerarquía de clases de objetos WinNT](winnt-object-class-hierarchy.md) siguiente describe las relaciones jerárquicas entre estos objetos.

Cada objeto implementa un conjunto de interfaces ADSI. Sin embargo, de una interfaz implementada, no se admiten todas las propiedades y los métodos. [Las propiedades obligatorias y opcionales del esquema WinNT](winnt-schemaampaposs-mandatory-and-optional-properties.md) enumeran las propiedades admitidas por las clases de objeto Winnt.
