| CN                | GP-Options                           |
| Ldap-Display-Name | gPOptions                            |
| Size              | \-                                   |
| Privilegio actualizar  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.892               |
