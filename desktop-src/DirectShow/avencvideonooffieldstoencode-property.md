# <a name="avencvideonooffieldstoencode-property"></a>Propiedad AVEncVideoNoOfFieldsToEncode

Especifica el número de campos que se codifican.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncVideoNoOfFieldsToEncode**

## <a name="remarks"></a>Observaciones

En el caso del vídeo progresiva, establezca esta propiedad en el doble del número de fotogramas que se codifican.
