# <a name="elementshouldbeoffscreen"></a>ElementShouldBeOffScreen

## <a name="text"></a>Texto

El elemento debe estar fuera de la pantalla

## <a name="type"></a>Tipo

Advertencia

## <a name="description"></a>Descripción

El rectángulo delimitador del elemento recorta el rectángulo delimitador de su elemento primario. El elemento debe omitirse y considerarse fuera de los límites del escritorio.
