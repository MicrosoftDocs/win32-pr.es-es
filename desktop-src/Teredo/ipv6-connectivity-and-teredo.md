# <a name="ipv6-connectivity-and-teredo"></a>Conectividad IPv6 y Teredo

La mayoría de las redes son solo IPv4 y la conectividad IPv6 de un extremo a otro no se obtiene fácilmente. Además, la mayoría de los dispositivos NAT incluso limitan la Conectividad IPv4 de un extremo a otro. Con Windows Vista, es posible la conectividad de un extremo a otro de IPv6 entre dos hosts cualesquiera. Los desarrolladores ya no tienen que preocuparse de la conectividad de un extremo a otro, sino que crean nuevos escenarios de aplicación que les permiten aprovechar esta conectividad.

El objetivo de Teredo, como tecnología de transición IPv6, es que la conectividad IPv6 de un extremo a otro sea sencilla y automática. En muchos casos, puede que no sea necesario que los desarrolladores sepan que las tecnologías de transición como Teredo existen en Windows Vista. Windows Vista tiene la capacidad de usar Teredo en segundo plano como un método sin problemas de conectividad IPv6.
