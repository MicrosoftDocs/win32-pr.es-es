# <a name="direct2d-helper-classes"></a>Clases auxiliares de Direct2D

El espacio de nombres D2D1 define las siguientes clases.

## <a name="in-this-section"></a>En esta sección



| Tema                                       | Descripción                                                                                                                              |
|---------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| [**ColorF**](/windows/win32/api/d2d1helper/nl-d2d1helper-colorf)<br/>         | Define un conjunto de constantes que representa los colores conocidos y proporciona métodos útiles para crear instancias de distintos colores.<br/>  |
| [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f)<br/> | La clase [**Matrix3x2F**](/windows/win32/api/d2d1helper/nl-d2d1helper-matrix3x2f) representa una matriz de 3 por 2 y proporciona métodos útiles para crear matrices.<br/> |
| [**Matrix4x3F**](matrix4x3f.md)<br/> | La clase [**Matrix4x3F**](matrix4x3f.md) representa una matriz de 4 por 3 y proporciona métodos útiles para crear matrices.<br/> |
| [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f)<br/> | La clase [**Matrix4x4F**](/windows/desktop/api/d2d1_1helper/nl-d2d1_1helper-matrix4x4f) representa una matriz de 4 por 4 y proporciona métodos útiles para crear matrices.<br/> |
| [**Matrix5x4F**](matrix5x4f.md)<br/> | La clase [**Matrix5x4F**](matrix5x4f.md) representa una matriz de 5 por 4 y proporciona métodos útiles para crear matrices.<br/> |
