# <a name="about-the-hyper-v-wmi-provider"></a>Acerca del proveedor WMI de Hyper-V

El proveedor WMI de Hyper-V permite a los desarrolladores y generadores de scripts crear rápidamente herramientas, utilidades y mejoras personalizadas para la plataforma de virtualización. Las interfaces WMI pueden administrar todos los aspectos de los servicios de Hyper-V.

## <a name="about-microsoft-windows-server-2008-r2-hyper-v"></a>Acerca de Microsoft Windows Server 2008 R2 Hyper-V

Microsoft Windows Server 2008 R2 Hyper-V permite a los administradores del sistema consolidar servidores de hardware independientes en un único servidor que ejecuta Microsoft Windows Server 2008 R2 como sistema operativo host.

Cada una de las máquinas virtuales hospedadas se ejecuta en su propio entorno virtual independiente y aislado. Esto permite al administrador administrar, proporcionar y mantener fácilmente un gran número de máquinas virtuales, a la vez que reduce la necesidad de ejecutar varias soluciones de hardware para usar varios productos y sistemas operativos.

Otra ventaja del entorno de máquina virtual está en pruebas y soporte técnico. Las nuevas configuraciones de servidor y sistema se pueden implementar y probar en paralelo con el sistema existente sin requerir costosos tiempos de inactividad durante la fase de implementación. Cada máquina virtual se puede configurar para que use configuraciones diferentes mientras se ejecuta en una plataforma estándar para generar mejores resultados de pruebas. Gracias a la compatibilidad con los sistemas operativos anteriores, uno puede admitir y probar el hardware, los sistemas y las aplicaciones heredados. La compatibilidad con instantáneas le permite tomar una instantánea del estado de una máquina virtual para que pueda volver a un evento anterior si es necesario.

Hyper-V expone una interfaz enriquecida que permite al usuario supervisar y controlar el entorno de la máquina virtual. La mayor parte de Hyper-V se puede controlar mediante el proveedor WMI, lo que permite una personalización sencilla y eficaz de las máquinas virtuales.
