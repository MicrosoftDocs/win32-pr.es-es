# <a name="waveactiveanytrue-function"></a>WaveActiveAnyTrue función)

Devuelve true si la expresión es verdadera en cualquiera de las calles activas de la onda actual.

## <a name="syntax"></a>Sintaxis

``` syntax
bool WaveActiveAnyTrue(
   bool expr
);
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

*expr* 
</dt> <dd>

Expresión booleana que se va a evaluar.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

True si la expresión es verdadera en cualquier calle.

## <a name="remarks"></a>Observaciones

Esta función se admite desde el modelo de sombreador 6,0 en todas las fases del sombreador.
