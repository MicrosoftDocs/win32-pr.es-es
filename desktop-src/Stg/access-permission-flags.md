# <a name="access-permission-flags"></a>Marcas de permisos de acceso

Las marcas de permiso de acceso especifican el tipo de acceso que tiene una aplicación cliente a un objeto abierto: lectura, escritura o lectura/escritura. El permiso de lectura permite a una aplicación leer el contenido de una secuencia pero no escribir los cambios en la secuencia. El permiso de escritura permite a una aplicación llamar a una función que confirma los cambios en el almacenamiento de un objeto, pero no para leer el contenido de la secuencia. El permiso de lectura y escritura permite que una aplicación Lea las secuencias del objeto o escriba en el almacenamiento del objeto.
