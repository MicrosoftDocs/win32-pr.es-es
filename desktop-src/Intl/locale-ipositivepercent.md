# <a name="locale_ipositivepercent"></a>configuración regional \_ IPOSITIVEPERCENT

**Windows 7 y versiones posteriores:** Patrón de formato de porcentaje positivo para la configuración regional. Solo se puede indicar un patrón.



| Value | Formato                                    |
|-------|-------------------------------------------|
| 0     | Número, espacio, porcentaje; por ejemplo, \# % |
| 1     | Número, porcentaje; por ejemplo, \#%         |
| 2     | Porcentaje, número; por ejemplo,%\#         |
| 3     | Porcentaje, espacio, número; por ejemplo,% \# |
