# <a name="preferred-ou-attribute"></a>Preferido: atributo de unidad organizativa

|-------------------|-----------------------------------------|
| CN                | Preferida: OU                            |
| Nombre para mostrar de LDAP | preferredOU                             |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.97                   |
| System-ID-GUID    | bf9679ff-0de6-11d0-a285-00aa003049e2    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
