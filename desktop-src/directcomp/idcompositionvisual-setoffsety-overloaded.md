# <a name="idcompositionvisualsetoffsety-methods"></a>Métodos IDCompositionVisual::SetOffsetY

Cambia o anima el valor de la propiedad OffsetY de este objeto visual, modificando la posición vertical del objeto visual con respecto a su elemento primario.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                                | Descripción                                           |
|:------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| [**SetOffsetY(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setoffsety(float))                                     | Cambia el valor de la propiedad OffsetY.<br/> |
| [**SetOffsetY(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionvisual-setoffsety(idcompositionanimation)) | Anima el valor de la propiedad OffsetY.<br/> |
