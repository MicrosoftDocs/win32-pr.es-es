# <a name="editboxfontsize"></a>EDITBOX. FontSize

El atributo **FontSize** especifica o recupera el tamaño de fuente para el control de cuadro de edición.

``` syntax
        elementID.fontSize
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **número** de lectura y escritura (**Long**) que especifica el tamaño de fuente en puntos. El valor predeterminado es 10.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------------|
| Versión<br/> | Windows Media Player para Windows XP o posterior<br/> |
