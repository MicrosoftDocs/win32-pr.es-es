# <a name="columncolumnname"></a>COLUMNA. columnName

El atributo **columnName** especifica o recupera un nombre de columna en el control de **lista de reproducción** .

``` syntax
        elementID.columnName
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de lectura/escritura.

## <a name="remarks"></a>Observaciones

El nombre de la columna es el nombre descriptivo que aparece en el encabezado de la columna.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------|
| Versión<br/> | Windows Media Player 9 series o posterior<br/> |
