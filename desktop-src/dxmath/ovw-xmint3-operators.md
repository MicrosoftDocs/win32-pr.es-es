# <a name="xmint3-operators"></a>Operadores XMINT3

> [!Note]  
> Los operadores enumerados aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                             | Descripción                                                                                                      |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**operator =**](/windows/win32/api/directxmath/nf-directxmath-xmint3-operator-assign)<br/> | Asigna los datos del componente vectorial de una instancia de `XMINT3` a la instancia actual de `XMINT3` . <br/> |
