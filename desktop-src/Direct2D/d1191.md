# <a name="d1191-gdi-compatible-bitmap-option-was-used-incorrectly"></a>D1191: se usó incorrectamente la opción de mapa de bits compatible con GDI

\_ \_ Las opciones \_ de mapa \_ de bits de D2D1 compatibles con GDI deben combinarse con el \_ destino de opciones de mapa de bits D2D1 \_ \_






 

## <a name="possible-causes"></a>Causas posibles

La aplicación especificó una combinación incompatible de las \_ marcas de opciones de mapa de bits de D2D1 \_ .

## <a name="possible-fixes"></a>Posibles correcciones

Agregue \_ \_ \_ el destino de opciones de mapa de bits D2D1 a las \_ marcas de opciones de mapa de bits D2D1 \_ .
