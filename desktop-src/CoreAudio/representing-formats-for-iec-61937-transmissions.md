# <a name="representing-formats-for-iec-61937-transmissions"></a><span data-ttu-id="5ceef-103">Representación de formatos para transmisiones IEC 61937</span><span class="sxs-lookup"><span data-stu-id="5ceef-103">Representing Formats for IEC 61937 Transmissions</span></span>

<span data-ttu-id="5ceef-104">Con el aumento de los dispositivos de almacenamiento multimedia que requieren formatos de audio comprimidos, las aplicaciones deben identificar, describir y usar una variedad de contenido de audio codificado nuevo para transmitir contenido desde equipos a dispositivos como el receptor HDMI o DisplayPort.</span><span class="sxs-lookup"><span data-stu-id="5ceef-104">With the increase in media storage devices that require compressed audio formats, applications must identify, describe, and use a variety of new encoded audio content for transmitting content from PCs to devices such as HDMI or DisplayPort receiver.</span></span>

<span data-ttu-id="5ceef-105">Para representar una secuencia de audio codificada que se transmitirá a través de una interfaz compatible con IEC 61937, una aplicación debe proporcionar:</span><span class="sxs-lookup"><span data-stu-id="5ceef-105">To represent an encoded audio stream to be transmitted over an IEC 61937-compatible interface, an application must provide:</span></span>

-   <span data-ttu-id="5ceef-106">Características de una secuencia de audio codificada que se va a transmitir.</span><span class="sxs-lookup"><span data-stu-id="5ceef-106">The characteristics of an encoded audio stream to be transmitted.</span></span>

-   <span data-ttu-id="5ceef-107">Características de una secuencia de audio descodificada en el dispositivo de destino.</span><span class="sxs-lookup"><span data-stu-id="5ceef-107">The characteristics of a decoded audio stream on the target device.</span></span>

<span data-ttu-id="5ceef-108">En Windows Vista y sistemas operativos Windows anteriores, una aplicación puede deducir el nivel de calidad de un formato de audio a partir del número de canales, el tamaño de la muestra y la velocidad de datos de una secuencia de audio que usa el formato.</span><span class="sxs-lookup"><span data-stu-id="5ceef-108">In Windows Vista and earlier Windows operating systems, an application can infer the quality level of an audio format from the number of channels, the sample size, and the data rate of an audio stream that uses the format.</span></span> <span data-ttu-id="5ceef-109">En el caso de un formato PCM, esta información está disponible en los miembros **nChannels**, **nSamplesPerSec** y **nAvgBytesPerSec** de la **estructuraSAMPLEATEX** que especifica el formato.</span><span class="sxs-lookup"><span data-stu-id="5ceef-109">For a PCM format, this information is available from the **nChannels**, **nSamplesPerSec**, and **nAvgBytesPerSec** members of the **WAVEFORMATEX** structure that specifies the format.</span></span> <span data-ttu-id="5ceef-110">En el caso de un formato que no es PCM, se ha comandos a estos tres miembros para almacenar información sobre los datos comprimidos en la secuencia de audio.</span><span class="sxs-lookup"><span data-stu-id="5ceef-110">For a non-PCM format, these three members have been commandeered to store information about the compressed data in the audio stream.</span></span> <span data-ttu-id="5ceef-111">Por lo tanto, la estructura **DESATEX** carece de información sobre el número efectivo de canales, el tamaño de la muestra y la velocidad de datos de la secuencia de audio que no es PCM después de que la secuencia se descomprima y se reproduce.</span><span class="sxs-lookup"><span data-stu-id="5ceef-111">Thus, the **WAVEFORMATEX** structure lacks information about the effective number of channels, sample size, and data rate of the non-PCM audio stream after the stream is decompressed and played.</span></span> <span data-ttu-id="5ceef-112">En función de la información de esta estructura, es posible que un usuario o una aplicación tengan dificultades para deducir el nivel de calidad de la secuencia que no es de PCM.</span><span class="sxs-lookup"><span data-stu-id="5ceef-112">Based on the information in this structure, a user or an application might have difficulty inferring the quality level of the non-PCM stream.</span></span>

<span data-ttu-id="5ceef-113">**EL ELEMENTO DESATEX** se ha ampliado a la **estructura DESATEXTENSIBLE** para proporcionar las características de flujo adicionales.</span><span class="sxs-lookup"><span data-stu-id="5ceef-113">The **WAVEFORMATEX** was extended to the **WAVEFORMATEXTENSIBLE** structure to provide the extra stream characteristics.</span></span> <span data-ttu-id="5ceef-114">Sin embargo, esta estructura tampoco es adecuada para describir la secuencia para las transmisiones IEC 61937 porque estaba diseñada para representar un único conjunto de características y se usaba para datos PCM de varios canales sin comprimir.</span><span class="sxs-lookup"><span data-stu-id="5ceef-114">However, this structure is also not adequate in describing the stream for IEC 61937 transmissions because it was intended to represent a single set of characteristics and used for uncompressed, multi-channel PCM data.</span></span>

<span data-ttu-id="5ceef-115">En Windows 7, el sistema operativo soluciona este problema al proporcionar compatibilidad con una nueva estructura, **SOAPATEXTENSIBLE \_ IEC61937,** que extiende la estructura **DESATEXTENSIBLE** para almacenar dos conjuntos de características de secuencia de audio: el formato de audio codificado antes de la transmisión y las características de la secuencia de audio después de que se haya descodificado.</span><span class="sxs-lookup"><span data-stu-id="5ceef-115">In Windows 7, the operating system addresses this problem by providing support for a new structure, **WAVEFORMATEXTENSIBLE\_IEC61937** which extends **WAVEFORMATEXTENSIBLE** structure to store two sets of audio stream characteristics: the encoded audio format before transmission and characteristics of the audio stream after it has been decoded.</span></span> <span data-ttu-id="5ceef-116">La nueva estructura especifica explícitamente el número efectivo de canales, el tamaño de la muestra y la velocidad de datos de un formato que no es PCM.</span><span class="sxs-lookup"><span data-stu-id="5ceef-116">The new structure explicitly specifies the effective number of channels, sample size, and data rate of a non-PCM format.</span></span> <span data-ttu-id="5ceef-117">Con esta información, una aplicación puede deducir el nivel de calidad de la secuencia que no es PCM después de descomprimida y reproducida.</span><span class="sxs-lookup"><span data-stu-id="5ceef-117">With this information, an application can infer the quality level of the non-PCM stream after it is decompressed and played.</span></span>

-   <span data-ttu-id="5ceef-123">El **miembro wFormatTag** de **WAVEATEX** debe contener WAVE \_ FORMAT EXTENSIBLE ( \_ `FormatExt.Format.wFormatTag = WAVE_FORMAT_EXTENSIBLE` ).</span><span class="sxs-lookup"><span data-stu-id="5ceef-123">The **wFormatTag** member of **WAVEFORMATEX** must contain WAVE\_FORMAT\_EXTENSIBLE (`FormatExt.Format.wFormatTag = WAVE_FORMAT_EXTENSIBLE`).</span></span>
