# <a name="igrfcache-interface"></a>Interfaz IGrfCache

> [!Note]  
> \[En desuso. Esta API se puede quitar de las versiones futuras de Windows.\]

 

La interfaz `IGrfCache` no se admite.

## <a name="members"></a>Miembros

La interfaz **IGrfCache** hereda de la interfaz [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) , pero no tiene miembros adicionales.
