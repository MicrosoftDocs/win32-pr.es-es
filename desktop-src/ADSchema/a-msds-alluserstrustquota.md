# <a name="ms-ds-all-users-trust-quota-attribute"></a>Atributo MS-DS-All-Users-Trust-quota

Se usa para aplicar una cuota de usuarios combinados en el número total de objetos de Trusted-Domain creados mediante el derecho de acceso de control nuevo, crear-entrada-bosque-confianza.



| Entrada | Value |
|-------------------|-------------------------------------------|
| CN                | MS-DS-All-Users-Trust-quota               |
| Nombre para mostrar de LDAP | msDS-AllUsersTrustQuota                   |
| Tamaño              | \-                                        |
| Actualizar privilegio  | Administrador de dominio                      |
| Frecuencia de actualización  | Al crear el bosque y raramente después. |
| Attribute-Id      | 1.2.840.113556.1.4.1789                   |
| System-ID-GUID    | d3aa4a5c-4e03-4810-97aa-2b339e7a434b      |
| Sintaxis            | [**Enumeración**](s-enumeration.md)      |
