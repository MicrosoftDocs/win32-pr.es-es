# <a name="regionexclude-methods"></a>Métodos Region.Exclude

En este tema se enumeran los métodos Exclude de la [**clase Region.**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) Para obtener una lista completa de los métodos de la **clase Region,** vea [Region Methods](-gdiplus-class-region-methods.md).

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                 | Descripción                                                                                                                                                                                         |
|:-----------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Exclude(Rect&)**](/previous-versions//ms534825(v=vs.85))     | El [**método Region::Exclude**](/previous-versions//ms534825(v=vs.85)) actualiza esta región a la parte de sí misma que no forma una intersección con el interior del rectángulo especificado.<br/>  |
| [**Exclude(RectF&)**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstrectf_))   | El [**método Region::Exclude**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstrectf_)) actualiza esta región a la parte de sí misma que no forma una intersección con el interior del rectángulo especificado.<br/> |
| [**Exclude(Region \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstregion))     | El [**método Region::Exclude**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstregion)) actualiza esta región a la parte de sí misma que no forma intersección con otra región.<br/>                         |
| [**Exclude(GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstgraphicspath)) | El [**método Region::Exclude**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstgraphicspath)) actualiza esta región a la parte de sí misma que no forma una intersección con el interior de la ruta de acceso especificada.<br/>            |
