# <a name="cryptography-data-types"></a>Tipos de datos de criptografía

Los siguientes tipos de datos se utilizan en las funciones, interfaces y objetos de criptografía.



| Tipo de datos                                                                      | Descripción                                                                                                                                                                                      |
|--------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**identificador de ALG \_**](alg-id.md)                                                      | Especifica los identificadores de algoritmo.                                                                                                                                                                 |
| [**\_respuesta de \_ OCSP del servidor de HCERT \_**](hcert-server-ocsp-response.md)            | Representa un identificador de una respuesta de OCSP asociada con una cadena de certificados de servidor.                                                                                                              |
| [**HCRYPTHASH**](hcrypthash.md)                                               | Especifica los identificadores de un [*objeto hash*](../secgloss/h-gly.md).                                                                                      |
| [**HCRYPTKEY**](hcryptkey.md)                                                 | Especifica los identificadores de las claves criptográficas.                                                                                                                                                         |
| [**HCRYPTOIDFUNCADDR**](hcryptoidfuncaddr.md)                                 | Representa un identificador de una función que se puede instalar mediante un [*identificador de objeto*](../secgloss/o-gly.md) (OID).                 |
| [**HCRYPTOIDFUNCSET**](hcryptoidfuncset.md)                                   | Representa un identificador para un conjunto de funciones que se pueden instalar mediante un OID.                                                                                                                 |
| [**HCRYPTPROV \_ heredado**](hcryptprov-legacy.md)                                | Se usa para reemplazar el tipo de datos [**HCRYPTPROV**](hcryptprov.md) en el que ya no se usa el tipo de datos **HCRYPTPROV** .                                                                             |
| [**\_identificador de clave HCRYPTPROV o \_ NCRYPT \_ \_**](hcryptprov-or-ncrypt-key-handle.md) | Especifica un identificador para un [*proveedor de servicios criptográficos*](../secgloss/c-gly.md) (CSP) de CryptoAPI o un CSP de CNG. |
| [**HCRYPTPROV**](hcryptprov.md)                                               | Especifica un identificador para un CSP de CryptoAPI.                                                                                                                                                           |
| [**identificador de KEYSVCC \_**](keysvcc-handle.md)                                      | Especifica los identificadores para el servicio de clave.                                                                                                                                                            |
