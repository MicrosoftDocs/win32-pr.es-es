<span data-ttu-id="b5dde-116">La experiencia del usuario para el Asistente para publicación web y el Asistente para la ordenación de impresión en línea es similar, pero el punto de entrada de cada asistente es diferente.</span><span class="sxs-lookup"><span data-stu-id="b5dde-116">The user experience for the Web Publishing Wizard and Online Print Ordering Wizard is similar, but the entry point for each wizard differs.</span></span> <span data-ttu-id="b5dde-117">En el Asistente para publicación Web, una tarea titulada **publicar esta carpeta en Web** está disponible en la lista de **tareas de archivos y carpetas** a la izquierda de la mayoría de las vistas de carpeta.</span><span class="sxs-lookup"><span data-stu-id="b5dde-117">For the Web Publishing Wizard, a task titled **Publish this folder to the Web** is available from the **File and Folder Tasks** list to the left of most folder views.</span></span> <span data-ttu-id="b5dde-118">El texto de la tarea varía en función del contenido seleccionado.</span><span class="sxs-lookup"><span data-stu-id="b5dde-118">The text for the task varies based on selected content.</span></span> <span data-ttu-id="b5dde-119">Por ejemplo, si se selecciona un archivo, la tarea Lee **publicar este archivo en la web** , tal como se muestra en el ejemplo siguiente.</span><span class="sxs-lookup"><span data-stu-id="b5dde-119">For instance, if a file is selected, the task reads **Publish this file to the Web** as shown in the following example.</span></span>
