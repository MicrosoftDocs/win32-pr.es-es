# <a name="microsoft-dss-cryptographic-provider"></a>Proveedor de servicios criptográficos de Microsoft DSS

El proveedor de servicios criptográficos de Microsoft DSS admite el uso de hash, la firma de datos y la comprobación de firmas mediante el algoritmo [*hash seguro*](../secgloss/s-gly.md) (SHA) y los algoritmos [*estándar de firma digital*](../secgloss/d-gly.md) (DSS). Se puede exportar fuera del Estados Unidos y Canadá.
