# <a name="protocol-handlers-interfaces"></a>Interfaces de controladores de protocolo

Esta sección de referencia de Windows Search se compone de interfaces de controlador de protocolo.


## <a name="in-this-section"></a>En esta sección



| Tema                                                                                   | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|-----------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IItemPropertyBag**](iitempropertybag.md)<br/>                                 | Define métodos para obtener información acerca de las propiedades de un elemento de búsqueda. Esta interfaz solo se admite en Windows XP y Windows Server 2003 y ya no debe usarse.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**IOpLockStatus**](/windows/desktop/api/Searchapi/nn-searchapi-ioplockstatus)<br/>                               | Proporciona métodos para comprobar el bloqueo oportunista que usa Microsoft Windows Desktop Search (WDS) en los elementos durante la indexación. Si otro proceso bloquea el archivo de forma no compatible, WDS perderá su bloqueo y permitirá que el otro proceso tenga el archivo. Este mecanismo permite que WDS se ejecute en segundo plano. Por lo tanto, WDS debe comprobar sus bloqueos para asegurarse de que otro proceso no haya tenido prioridad mientras WDS indexa el elemento. <br/> Un objeto [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) de terceros puede implementar esta interfaz si el almacén de datos subyacente proporciona un mecanismo para realizar el seguimiento del acceso simultáneo a los elementos. Si esta interfaz se expone mediante **IUrlAccessor**, WDS comprobará el [**IOpLockStatus**](/windows/desktop/api/Searchapi/nn-searchapi-ioplockstatus) de elementos mientras se indexan los elementos de ese almacén. <br/> |
| [**IProtocolHandlerServices**](/previous-versions/windows/desktop/legacy/bb231447(v=vs.85))<br/>         |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IProtocolHandlerSite**](/windows/desktop/api/Searchapi/nn-searchapi-iprotocolhandlersite)<br/>                 | Proporciona métodos para que el objeto [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) de un controlador de protocolo consulte al demonio de filtro el filtro adecuado para el elemento URL. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**ISearchProtocol**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocol)<br/>                           | Proporciona métodos para invocar, inicializar y administrar objetos [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) . El host del protocolo llama a los métodos de esta interfaz al procesar las direcciones URL del recopilador. <br/> El controlador de protocolo implementa el protocolo para tener acceso a un origen de contenido en su formato nativo. Utilice esta interfaz para implementar un controlador de protocolo personalizado para expandir los orígenes de datos que se pueden indizar. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISearchProtocol2**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocol2)<br/>                         | Proporciona métodos para invocar, inicializar y administrar objetos [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) . El host del protocolo llama a los métodos de esta interfaz al procesar las direcciones URL del recopilador. <br/> El controlador de protocolo implementa el protocolo para tener acceso a un origen de contenido en su formato nativo. Utilice esta interfaz para implementar un controlador de protocolo personalizado para expandir los orígenes de datos que se pueden indizar. <br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**ISearchProtocolThreadContext**](/windows/desktop/api/Searchapi/nn-searchapi-isearchprotocolthreadcontext)<br/> | Esta interfaz opcional permite al controlador de protocolo realizar una acción en el subproceso utilizado para filtrar en el host del protocolo. Cuando se inicia el host de protocolo, primero inicializa todos los controladores de protocolo y, a continuación, crea los subprocesos de filtrado. Los métodos de esta interfaz permiten a los controladores de protocolo administrar sus recursos utilizados por un subproceso de filtrado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [**ISearchProtocolUI**](-search-isearchprotocolui.md)<br/>                       | Proporciona un método para invocar objetos [**ISearchItem**](-search-isearchitem.md) . El host del protocolo llama a los métodos de esta interfaz al procesar las direcciones URL del recopilador. El controlador de protocolo implementa el protocolo para tener acceso a un origen de contenido en su formato nativo y esta interfaz implementa un controlador de protocolo personalizado para expandir los orígenes de datos que se pueden indizar.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor)<br/>                                 | Proporciona métodos para procesar un elemento individual en un origen de contenido cuya dirección URL proporciona el recopilador al host de filtro. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**IUrlAccessor2**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor2)<br/>                               | Extiende la funcionalidad de la interfaz [**IUrlAccessor**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IUrlAccessor3**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor3)<br/>                               | Extiende la funcionalidad de la interfaz [**IUrlAccessor2**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor2) con el método [**IUrlAccessor3:: GetImpersonationSidBlobs**](/windows/desktop/api/Searchapi/nf-searchapi-iurlaccessor3-getimpersonationsidblobs) para identificar los identificadores de seguridad (SID) del usuario para una dirección URL especificada.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**IUrlAccessor4**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor4)<br/>                               | Extiende la funcionalidad de la interfaz [**IUrlAccessor3**](/windows/desktop/api/Searchapi/nn-searchapi-iurlaccessor3) con el método [**IUrlAccessor4:: ShouldIndexItemContent**](/windows/desktop/api/Searchapi/nf-searchapi-iurlaccessor4-shouldindexitemcontent) que identifica si se debe indizar el contenido del elemento. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
