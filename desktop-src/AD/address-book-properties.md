# <a name="user-address-book-attributes"></a>Atributos de la libreta de direcciones de usuario

Los atributos de la libreta de direcciones se usan para proporcionar información e identificación complementaria para un usuario. El usuario y la organización del usuario definen su contenido. Estos atributos pueden verse y administrarse mediante el complemento Active Directory usuarios y equipos o la libreta de direcciones de Windows (WAB).



| Atributo                         | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                   |
|-----------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **c**                             | El país o región en la dirección del usuario. El país o región se representa como el código de país o región de dos caracteres basado en ISO-3166. Para ver los códigos válidos, consulte [valores de CountryCode](values-for-the-countrycode-and-c-properties.md).<br/>                                                                                                                                                                                        |
| **ustedes**                            | El país o la región donde se encuentra el usuario.                                                                                                                                                                                                                                                                                                                                                                                              |
| **apunte**                         | Comentario. Esta cadena puede ser una cadena nula.                                                                                                                                                                                                                                                                                                                                                                                                  |
| **department**                    | El nombre del departamento en el que trabaja el usuario.                                                                                                                                                                                                                                                                                                                                                                                          |
| **description**                   | Descripción que se va a mostrar para el usuario.                                                                                                                                                                                                                                                                                                                                                                                                      |
| **displayName**                   | Nombre que se muestra en la libreta de direcciones de un usuario determinado. Suele ser la combinación del nombre del usuario, el inicial y el apellido.                                                                                                                                                                                                                                                                                        |
| **directReports**                 | La lista de usuarios que notifican directamente al usuario. Los usuarios que aparecen como informes son aquellos que tienen el atributo de **Administrador** establecido en este usuario. Cada elemento de la lista es una referencia vinculada al objeto que representa al usuario; por lo tanto, el servidor de Active Directory actualiza automáticamente este atributo cuando el atributo de **Administrador** de un usuario agrega o quita este usuario como administrador. Los elementos se representan como nombres distintivos. |
| **facsimileTelephoneNumber**      | El número de teléfono del equipo de fax del trabajo del usuario.                                                                                                                                                                                                                                                                                                                                                                                      |
| **givenName**                     | EL nombre del usuario.                                                                                                                                                                                                                                                                                                                                                                                                      |
| **homePhone**                     | Número de teléfono principal del usuario.                                                                                                                                                                                                                                                                                                                                                                                               |
| **iniciales**                      | Las iniciales para las partes del nombre completo del usuario. Puede usarse como inicial central en la libreta de direcciones de Windows.                                                                                                                                                                                                                                                                                                                           |
| **ipPhone**                       | Utilizado por telefonía.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **l**                             | La localidad, como la ciudad o la ciudad, en la dirección del usuario.                                                                                                                                                                                                                                                                                                                                                                                |
| **managedObjects**                | Lista de objetos administrados por el usuario. Los objetos enumerados son aquellos que tienen el atributo **managedBy** establecido en este usuario. Cada elemento de la lista es una referencia vinculada al objeto administrado; por lo tanto, el servidor de Active Directory actualiza automáticamente el atributo **managedObjects** cuando el atributo **managedBy** de un objeto agrega o quita este usuario como administrador. Los elementos se representan como nombres distintivos.  |
| **Manager**                       | Usuario que es el administrador del usuario. El objeto de usuario del administrador contiene un atributo **directReports** que contiene referencias a todos los objetos de usuario que tienen el atributo **Manager** establecido en el objeto de usuario del administrador.                                                                                                                                                                                                                     |
| **Movilidad**                        | EL número de teléfono móvil principal del usuario.                                                                                                                                                                                                                                                                                                                                                                                           |
| **otherFacsimileTelephoneNumber** | La lista de números de teléfono de las máquinas de fax alternativas para el usuario.                                                                                                                                                                                                                                                                                                                                                                         |
| **otherIpPhone**                  | Utilizado por telefonía.                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **otherMobile**                   | Lista de números de teléfono móvil alternativos para el usuario.                                                                                                                                                                                                                                                                                                                                                                                |
| **otherPager**                    | Lista de números de teléfono del buscapersonas alternativos para el usuario.                                                                                                                                                                                                                                                                                                                                                                                   |
| **otherTelephone**                | Lista de números de teléfono de negocio alternativos para el usuario.                                                                                                                                                                                                                                                                                                                                                                                |
| **buscapersonas**                         | Número de teléfono del buscapersonas principal del usuario.                                                                                                                                                                                                                                                                                                                                                                                              |
| **physicalDeliveryOfficeName**    | La ubicación del lugar de trabajo del usuario.                                                                                                                                                                                                                                                                                                                                                                                          |
| **postalAddress**                 | Dirección postal del usuario.                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **postalCode**                    | El código postal de la dirección del usuario. El código postal es específico del país o la región del usuario. En Estados Unidos de América, este atributo contiene el código ZIP.                                                                                                                                                                                                                                                               |
| **postOfficeBox**                 | Número o identificador del cuadro de oficina de correos del usuario.                                                                                                                                                                                                                                                                                                                                                                                       |
| **sn**                            | Los apellidos del usuario.                                                                                                                                                                                                                                                                                                                                                                                                |
| **primer**                            | El estado o provincia de la dirección del usuario.                                                                                                                                                                                                                                                                                                                                                                                                  |
| **streetAddress**                 | La dirección postal del lugar de trabajo del usuario.                                                                                                                                                                                                                                                                                                                                                                                           |
| **telephoneNumber**               | El número de teléfono principal del lugar de trabajo del usuario.                                                                                                                                                                                                                                                                                                                                                                                 |
| **title**                         | El puesto del usuario. Este atributo se usa normalmente para indicar el puesto de trabajo formal, como Programador Senior, en lugar de la clase ocupacional, como programador. Normalmente no se utiliza para los títulos de "sufijo", como Esq. o DDS. Ejemplos: Director de administración, programador II, profesor asociado y responsable de desarrollo.<br/>                                                                                                    |
| **url**                           | Lista de direcciones URL de las páginas web alternativas del usuario.                                                                                                                                                                                                                                                                                                                                                                                           |
| **wWWHomePage**                   | La dirección URL de la página web principal del usuario.                                                                                                                                                                                                                                                                                                                                                                                                       |
