# <a name="about-lines-and-curves"></a>Acerca de las líneas y curvas

Muchos tipos de aplicaciones usan líneas y curvas para dibujar la salida de gráficos en dispositivos de tramas. El diseño asistido por PC (CAD) y las aplicaciones de dibujo usan líneas y curvas para describir objetos, especificar los centros de objetos, las dimensiones de los objetos, etc. Las aplicaciones de hoja de cálculo usan líneas y curvas para dibujar cuadrículas, gráficos y gráficos. Las aplicaciones de procesamiento de texto usan líneas para crear reglas y bordes en una página de texto.
