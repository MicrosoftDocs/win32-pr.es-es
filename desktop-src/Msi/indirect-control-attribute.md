# <a name="indirect-control-attribute"></a>Atributo de control indirecto

El atributo de control indirecto especifica si se hace referencia indirectamente al valor mostrado o modificado por este control. Si se establece este bit, el control muestra o cambia el valor de la propiedad que tiene el identificador que aparece en la columna de propiedades de la [tabla de control](control-table.md). Si no se establece este bit, el control muestra o cambia el valor de la propiedad en la columna de propiedades de la tabla de control.
