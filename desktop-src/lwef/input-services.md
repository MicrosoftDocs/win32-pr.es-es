# <a name="input-services"></a>Servicios de entrada

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Una aplicación cliente proporciona la interfaz de usuario principal para la interacción con un carácter. Puede programar un carácter para responder a cualquier forma de entrada, desde los clics de botón hasta el texto escrito. Además, el agente de Microsoft proporciona eventos para que pueda programar lo que ocurre cuando el usuario hace clic, hace doble clic en él o arrastra el carácter. El servidor pasa las coordenadas del puntero y cualquier estado de tecla modificador para estos eventos.
