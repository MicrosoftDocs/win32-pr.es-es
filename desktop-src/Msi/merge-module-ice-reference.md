# <a name="merge-module-ice-reference"></a>Referencia de módulo de combinación ICE

En la lista siguiente se proporcionan vínculos a cada módulo de combinación ICE.



| ICEM                 | Breve descripción del módulo de combinación ICE                                                                                                                                                                                                                    |
|----------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [ICEM01](icem01.md) | Valida que el mecanismo de ICE funciona.                                                                                                                                                                                                             |
| [ICEM02](icem02.md) | Valida que todas las exclusiones y dependencias del módulo se relacionan con el módulo actual.                                                                                                                                                                      |
| [ICEM03](icem03.md) | Valida que todas las acciones del módulo son acciones base o derivan de una acción base.                                                                                                                                                           |
| [ICEM04](icem04.md) | Comprueba que las tablas vacías necesarias del módulo de combinación están vacías.                                                                                                                                                                                 |
| [ICEM05](icem05.md) | Comprueba si hay asociaciones no válidas con componentes.                                                                                                                                                                                                         |
| [ICEM06](icem06.md) | Comprueba las referencias no válidas a las características del módulo.                                                                                                                                                                                                 |
| [ICEM07](icem07.md) | Valida que el orden de los archivos en las tablas de secuencia y en [MergeModule.CABarchivo inet](mergemodule-cabinet.md) coincide.                                                                                                                               |
| [ICEM08](icem08.md) | Garantiza que un módulo no excluya algo del que depende.                                                                                                                                                                                          |
| [ICEM09](icem09.md) | Comprueba que el módulo de combinación controla de forma segura los directorios predefinidos.                                                                                                                                                                                    |
| [ICEM10](icem10.md) | Comprueba que un módulo de combinación no contiene propiedades no permitidas en la [tabla de propiedades](property-table.md).                                                                                                                                         |
| [ICEM11](icem11.md) | Comprueba que un módulo de combinación configurable muestra la tabla [ModuleConfiguration](moduleconfiguration-table.md) y la [tabla ModuleSubstitution](modulesubstitution-table.md) en la [tabla ModuleIgnoreTable](moduleignoretable-table.md) del módulo. |
| [ICEM12](icem12.md) | Comprueba que en una tabla ModuleSequence, las acciones estándar tienen números de secuencia y acciones personalizadas tienen valores BaseAction y after.                                                                                                                     |
| [ICEM13](icem13.md) | Comprueba que la Directiva del publicador y los ensamblados de configuración no se incluyen en el módulo de combinación.                                                                                                                                                        |
| [ICEM14](icem14.md) | Valida la columna de valor de la [tabla ModuleSubstitution](modulesubstitution-table.md).                                                                                                                                                              |
