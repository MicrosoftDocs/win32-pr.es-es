# <a name="iwindowsdriverupdate2-methods"></a>Métodos IWindowsDriverUpdate2

La interfaz [**IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) define el método siguiente.



| Método                                                   | Descripción                                       |
|----------------------------------------------------------|---------------------------------------------------|
| [**CopyToCache**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-copytocache) | Copia los archivos binarios de actualización externos en una actualización. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
