# <a name="requirements-for-online-music-stores"></a>Requisitos para las tiendas de música en línea

Todos los requisitos que debe cumplir su tienda en línea se detallan en el contrato de lista del servicio multimedia en línea para Windows Media Player. Para asegurarse de que la tienda en línea proporciona una experiencia de usuario excelente, hemos resaltado algunos requisitos clave relacionados con la personalización y la reproducción.

-   La página inicial del almacén expuesto en la interfaz de usuario de Windows Media Player debe ser funcional sin necesidad de que el usuario inicie sesión.
-   Se prohíben los anuncios emergentes que inician una ventana de explorador independiente.
-   Toda la experiencia de exploración, compra y reproducción debe representarse dentro de la interfaz de usuario del Media Player de Windows y no en una ventana de explorador emergente independiente ni en un reproductor multimedia incrustado.
-   La instalación de Systray de una aplicación y los procesos en segundo plano que se ejecutan continuamente están prohibidos.
-   Las acciones discretas, como el inicio de sesión, la facturación, la compra de CDs físicos u otras características relacionadas fuera del servicio de medios digitales principales, pueden vincularse a una instancia de explorador fuera del reproductor para la característica de medios no digitales.
-   Las licencias proporcionadas por el almacén deben ser válidas independientemente de la conexión del almacén a Windows Media Player. Para obtener más información, consulte la sección término y terminación del contrato de lista de servicios multimedia en línea para Windows Media Player.

De vez en cuando, Microsoft puede actualizar los requisitos de idoneidad y incorporación y publicarlos en el [sitio web de Windows Media](https://www.microsoft.com/windows/windowsmedia/default.mspx) ( https://www.microsoft.com/windows/windowsmedia/default.mspx) .
