# <a name="developing-assistive-technology-for-windows"></a>Desarrollo de tecnología de asistencia para Windows

Los productos de tecnología de asistencia de terceros, como lectores de pantalla, lupas y hardware de especialización, son herramientas esenciales para muchos usuarios de Windows (y otros productos de Microsoft).

En esta introducción se describe cómo crear herramientas de tecnología de asistencia compatibles con los productos de Microsoft para personas con visión, aprendizaje, movilidad y discapacidades o limitaciones de lenguaje o comunicación.
