# <a name="modifying-configuration-information-in-the-database"></a>Modificar la información de configuración en la base de datos

La extensión de complemento de datos adjuntos debe permitir que los usuarios cambien la información de configuración a través de su nodo de análisis. Esto permite a los usuarios realizar cambios y, a continuación, volver a aplicar la configuración a lo largo del tiempo. La lógica para hacerlo debe ser idéntica a la modificación de la información en los archivos de configuración. Los cambios realizados tendrán efecto en la configuración guardada en la base de datos de seguridad.
