# <a name="tapi-3-terminal-manager"></a>Administrador de terminales TAPI 3

El administrador de terminal de TAPI 3 se crea mediante Tapi3.dll a petición de un MSP y lo usa el MSP para tareas como la creación y eliminación de terminales dinámicos.

Los detalles de implementación precisos para los terminales dinámicos de MSP quedan fuera del ámbito de esta documentación. Sin embargo, muchos autores pueden optar por usar algunas de las capacidades del terminal de streaming multimedia (MST). Este terminal se basa en DirectShow y los filtros específicos de un protocolo determinado se pueden escribir y usar junto con el MST.
