# <a name="benefits-of-using-active-directory-service-interfaces"></a>Ventajas del uso de Active Directory interfaces de servicio

ADSI ofrece muchas ventajas a los administradores del sistema, como se describe en la tabla siguiente.



| Característica                   | Ventaja                                                                                                                                                                                                                                                                                                                             |
|---------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Abrir                      | Cualquier proveedor de directorios puede implementar un proveedor adsi; los usuarios obtienen libertad de elección en los servicios de directorio sin sacrificar la capacidad de administración.                                                                                                                                                                                        |
| Independiente de DS            | Las aplicaciones administrativas no están estrechamente enlazadas al servicio de directorio de un proveedor determinado. Las aplicaciones administrativas y otras aplicaciones habilitadas para directorios se pueden desarrollar sin necesidad de comprender las API de directorio específicas del proveedor. La misma aplicación puede funcionar en varios directorios. Se reduce el tiempo de desarrollo y los costos de soporte técnico. |
| Compatibilidad con varios idiomas | Los objetos ADSI proporcionan un acceso sencillo a los servicios de directorio a través del modelo de objetos de componente. Las aplicaciones COM se pueden escribir en lenguajes como Microsoft Visual Basic, Microsoft Visual Basic Scripting Edition (VBScript) y C/C++.                                                                                             |
| Modelo de programación simple  | ADSI consta de un pequeño conjunto de interfaces fáciles de aprender.                                                                                                                                                                                                                                                                          |
| Scriptable                | Cualquier lenguaje compatible con Automation (por ejemplo, Visual Basic, VBScript, Perl, Rexx, etc.) se puede usar para desarrollar aplicaciones de servicio de directorio. Los administradores y desarrolladores pueden usar las herramientas que ya conocen. Se mejora la productividad: se reduce el tiempo de desarrollo y los costos de soporte técnico.                               |
| Funcionalmente enriquecido         | Los ISV y los usuarios finales sofisticados pueden desarrollar aplicaciones graves con los mismos modelos ADSI que se usan para aplicaciones administrativas con scripts simples.                                                                                                                                                                         |
| Extensible                | Los proveedores de directorios, isv y usuarios finales pueden ampliar ADSI con nuevos objetos y funciones para agregar valor o satisfacer necesidades únicas.                                                                                                                                                                                                          |
| OLE DB en cuenta              | ADSI proporciona una interfaz OLE DB para que los programadores familiarizados con la programación de bases de OLE DB puedan ser productivos rápidamente.                                                                                                                                                                                                  |
