# <a name="taborderinquestion"></a>TabOrderInQuestion

## <a name="text"></a>Texto

El orden de tabulación puede no seguir las directrices Windows estándar

## <a name="type"></a>Tipo

Informativo

## <a name="description"></a>Descripción

El uso de la navegación por teclado estándar (Tab o Mayús+Tab) expone un orden de tabulación no estándar. Por ejemplo, en función de la configuración del sistema local, el orden de tabulación suele imitar el orden de lectura.

Este problema provoca problemas para las personas que dependen de un lector de pantalla y un teclado para la navegación, ya que el recorrido de elementos puede ser errático e impredecible.
