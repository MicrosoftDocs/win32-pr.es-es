# <a name="object-model-reference-for-scripting"></a>Referencia del modelo de objetos para scripting

La referencia del modelo de objetos para scripting contiene información detallada sobre el modelo de objetos de control ActiveX de Windows Media Player. La información de esta sección se presenta en un estilo diseñado para su uso con lenguajes de scripts como Microsoft JScript.

> [!Note]  
> Todos los métodos, propiedades y eventos son totalmente compatibles con Windows Media Player 10 Mobile o posterior, a menos que se indique explícitamente lo contrario.

 

La referencia del modelo de objetos para scripting contiene documentación para los siguientes objetos y sus métodos, propiedades y eventos asociados.



| Object                                              | Descripción                                                                                                                                                                                    |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [-](cdrom-object.md)                           | Métodos y propiedades para tener acceso a un CD o DVD en su unidad.                                                                                                                                 |
| [CdromCollection](cdromcollection-object.md)       | Métodos y propiedades para tener acceso a una colección de unidades de CD o DVD.                                                                                                                         |
| [ClosedCaption](closedcaption-object.md)           | Propiedades para incluir leyendas con un elemento multimedia.                                                                                                                                           |
| [Controles](controls-object.md)                     | Métodos y propiedades que representan los controles de transporte de Windows Media Player, como reproducir, detener y pausar.                                                                             |
| [DISCOS](dvd-object.md)                               | Propiedad, métodos y eventos para trabajar con DVDs.                                                                                                                                         |
| [Error](error-object.md)                           | Métodos y propiedades que proporcionan acceso a una colección de objetos **ErrorItem** .                                                                                                              |
| [ErrorItem](erroritem-object.md)                   | Propiedades que proporcionan información sobre los errores.                                                                                                                                              |
| [Elementos multimedia](media-object.md)                           | Métodos y propiedades relacionados con elementos multimedia.                                                                                                                                                |
| [MediaCollection](mediacollection-object.md)       | Métodos que proporcionan acceso a una colección de objetos **multimedia** .                                                                                                                              |
| [MetadataPicture](metadatapicture-object.md)       | Propiedades para recuperar información sobre los valores de imagen del atributo de metadatos de **WM/imagen** .                                                                                            |
| [MetadataText](metadatatext-object.md)             | Propiedades para recuperar los metadatos de los atributos de metadatos de texto complejos.                                                                                                                    |
| [Network](network-object.md)                       | Propiedades relacionadas con la conexión de red de Windows Media Player.                                                                                                                         |
| [Reproductor](player-object.md)                         | Métodos, propiedades y eventos a los que se puede programar Windows Media Player para que respondan.                                                                                                     |
| [PlayerApplication](playerapplication-object.md)   | Métodos y propiedades para cambiar entre un control de Windows Media Player remoto y el modo completo del reproductor. Solo se puede utilizar con programas de C++ que incrustan el control en modo remoto. |
| [Lista de reproducción](playlist-object.md)                     | Métodos y propiedades para manipular listas de elementos multimedia.                                                                                                                                  |
| [PlaylistArray](playlistarray-object.md)           | Un método y una propiedad para tener acceso a una colección de objetos de **lista de reproducción** por número de índice.                                                                                                    |
| [PlaylistCollection](playlistcollection-object.md) | Métodos para organizar una colección de objetos de **lista de reproducción** .                                                                                                                                   |
| [Consultar](query-object.md)                           | Métodos para modificar una consulta compuesta.                                                                                                                                                        |
| [Configuración](settings-object.md)                     | Propiedades que permiten la especificación o recuperación de la configuración de Windows Media Player.                                                                                                         |
| [StringCollection](stringcollection-object.md)     | Un método y una propiedad para manipular colecciones de cadenas.                                                                                                                               |
