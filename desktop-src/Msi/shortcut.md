# <a name="shortcut"></a>Acceso directo

Normalmente, el tipo de datos de acceso directo se usa en la columna de destino de la [tabla de acceso directo](shortcut-table.md). Si contiene corchetes ( \[ \] ), el destino del acceso directo se evalúa como una cadena con formato. De lo contrario, el acceso directo se evalúa como un [identificador](identifier.md) y debe ser una clave externa válida en la [tabla de características](feature-table.md).
