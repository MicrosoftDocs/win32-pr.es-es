# <a name="properties-of-icenroll4"></a>Propiedades de ICEnroll4

Las siguientes propiedades se definen mediante la interfaz [**ICEnroll4**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll4) o son propiedades que la interfaz **ICEnroll4** hereda de [**ICEnroll**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll), [**ICEnroll2**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll2)y [**ICEnroll3**](/windows/desktop/api/Xenroll/nn-xenroll-icenroll3).



| Propiedad                                                                               | Descripción                                                                                                                                                                                                                                                                                                                                                         |
|----------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CAStoreFlags**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_castoreflags)                                         | Establece o recupera una marca que controla el almacén de la [*entidad de certificación*](../secgloss/c-gly.md) (CA) cuando se abre el almacén.<br/>                                                                                                                                                  |
| [**CAStoreName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_castorename)                                           | Establece o recupera el nombre del almacén donde se conservan todos los certificados que no son "raíz" y "MY".<br/>                                                                                                                                                                                                                                                         |
| [**CAStoreType**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_castoretype)                                           | Establece o recupera el tipo de almacén que se va a usar para el almacén especificado por la propiedad [**CAStoreName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_castorename) .<br/>                                                                                                                                                                                                                         |
| [**ClientId**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-get_clientid)                                                 | Establece o recupera el atributo de solicitud de ID. de cliente.<br/>                                                                                                                                                                                                                                                                                                       |
| [**ContainerName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_containername)                                       | Establece o recupera el nombre del contenedor de [*claves*](../secgloss/k-gly.md) que se va a usar.<br/>                                                                                                                                                                                                                      |
| [**DeleteRequestCert**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_deleterequestcert)                               | Establece o recupera un valor booleano que determina si se eliminan los certificados ficticios en el almacén de solicitudes.<br/>                                                                                                                                                                                                                                           |
| [**EnableSMIMECapabilities**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-get_enablesmimecapabilities)                   | Controla si PKCS10 contendrá un atributo firmado para las capacidades de [*extensiones seguras multipropósito de correo Internet*](../secgloss/s-gly.md) (S/MIME).<br/>                                                                            |
| [**EnableT61DNEncoding**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll2-get_enablet61dnencoding)                           | Establece o recupera un valor booleano que determina si el nombre distintivo de la solicitud se codifica como una cadena T61 en lugar de como una cadena [*Unicode*](../secgloss/u-gly.md) .<br/>                                                                                                                               |
| [**GenKeyFlags**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_genkeyflags)                                           | Establece o recupera los valores pasados a [**CryptGenKey**](/windows/desktop/api/Wincrypt/nf-wincrypt-cryptgenkey) cuando se genera la solicitud de certificado.<br/>                                                                                                                                                                                                                                     |
| [**HashAlgID**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-get_hashalgid)                                               | Establece o recupera el [*algoritmo hash*](../secgloss/h-gly.md) que se usa al firmar una \# [*solicitud de certificado*](../secgloss/c-gly.md)PKCS 10.<br/>                                                                                      |
| [**HashAlgorithm**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_hashalgorithm)                                       | Establece o recupera solo el [*algoritmo hash*](../secgloss/h-gly.md) de firma utilizado para firmar el archivo PKCS \# 10.<br/>                                                                                                                                                                                      |
| [**IncludeSubjectKeyID**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-get_includesubjectkeyid)                           | Establece o recupera un valor booleano que controla si se incluye una extensión de identificador de clave de sujeto en la solicitud de certificado.<br/>                                                                                                                                                                                                                       |
| [**Especificación**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_keyspec)                                                   | Establece o recupera el tipo de clave generada.<br/>                                                                                                                                                                                                                                                                                                             |
| [**LimitExchangeKeyToEncipherment**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-get_limitexchangekeytoencipherment)     | Establece o recupera un valor booleano que determina si una solicitud AT \_ KEYEXCHANGE contiene los usos de la [*firma digital*](../secgloss/d-gly.md) y la clave sin repudio.<br/>                                                                                                                       |
| [**MyStoreFlags**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_mystoreflags)                                         | Establece o recupera la ubicación del registro utilizada para mi almacén.<br/>                                                                                                                                                                                                                                                                                               |
| [**MyStoreName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_mystorename)                                           | Establece o recupera el nombre del almacén en el que se conservan los certificados con [*claves privadas*](../secgloss/p-gly.md) vinculadas.<br/>                                                                                                                                                                                    |
| [**MyStoreType**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_mystoretype)                                           | Establece o recupera el tipo de almacén especificado por la propiedad [**MyStoreName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_mystorename) .<br/>                                                                                                                                                                                                                                              |
| [**PrivateKeyArchiveCertificate**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-get_privatekeyarchivecertificate)         | Establece o recupera el certificado que se usa para archivar una clave privada con una \# solicitud PKCS 7 o de [*Administración de certificados sobre CMS*](../secgloss/c-gly.md) (CMC).<br/>                                                                                                |
| [**ProviderFlags**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_providerflags)                                       | Establece o recupera el tipo de proveedor.<br/>                                                                                                                                                                                                                                                                                                                     |
| [**ProviderName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_providername)                                         | Establece o recupera el nombre del proveedor de [*servicios criptográficos*](../secgloss/c-gly.md) (CSP) que se va a usar.<br/>                                                                                                                                                             |
| [**ProviderType**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_providertype)                                         | Establece o recupera el tipo de proveedor.<br/>                                                                                                                                                                                                                                                                                                                  |
| [**PVKFileName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_pvkfilename)                                           | Establece o recupera el nombre del archivo que contendrá las claves exportadas.<br/>                                                                                                                                                                                                                                                                                  |
| [**RequestStoreFlags**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_requeststoreflags)                               | Establece o recupera la ubicación del registro utilizada para el almacén de solicitudes.<br/>                                                                                                                                                                                                                                                                                      |
| [**RequestStoreName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_requeststorename)                                 | Establece o recupera el nombre del almacén que contiene el certificado ficticio. Este certificado ficticio, junto con las claves privadas agregadas, permanece en el almacén de solicitudes hasta que una [*entidad de certificación*](../secgloss/c-gly.md) procesa la solicitud y responde con un archivo PKCS \# 7.<br/> |
| [**RequestStoreType**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_requeststoretype)                                 | Establece o recupera el tipo de almacén que se va a usar para el almacén especificado por la propiedad [**RequestStoreName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_requeststorename) .<br/>                                                                                                                                                                                                               |
| [**ReuseHardwareKeyIfUnableToGenNew**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll3-get_reusehardwarekeyifunabletogennew) | Establece o recupera un valor booleano que determina la acción realizada por el objeto de control de inscripción de certificados si se produce un error al generar una nueva clave.<br/>                                                                                                                                                                                    |
| [**RootStoreFlags**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_rootstoreflags)                                     | Establece o recupera la ubicación del registro utilizada para el almacén raíz.<br/>                                                                                                                                                                                                                                                                                         |
| [**RootStoreName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_rootstorename)                                       | Establece o recupera el nombre del almacén raíz donde se conservan todos los [*certificados raíz*](../secgloss/r-gly.md) autofirmados de confianza intrínsecamente.<br/>                                                                                                                                                   |
| [**RootStoreType**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_rootstoretype)                                       | Establece o recupera el tipo de almacén que se va a usar para el almacén especificado por la propiedad [**RootStoreName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_rootstorename) .<br/>                                                                                                                                                                                                                     |
| [**SignerCertificate**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-put_signercertificate)                               | Establece el certificado de firma.<br/>                                                                                                                                                                                                                                                                                                                            |
| [**SPCFileName**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_spcfilename)                                           | Establece o recupera el nombre del archivo en el que se va a escribir el PKCS \# 7 (en formato **BSTR** ) resultante codificado en Base64 como devuelto por la [*entidad de certificación*](../secgloss/c-gly.md).<br/>                                                                                          |
| [**Huella**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll4-get_thumbprint)                                             | Establece o recupera un [*hash*](../secgloss/h-gly.md) de los datos del certificado.<br/>                                                                                                                                                                                                                                              |
| [**UseExistingKeySet**](/windows/win32/api/xenroll/nf-xenroll-icenroll-get_useexistingkeyset)                               | Establece o recupera un valor booleano que determina si se deben usar las claves existentes.<br/>                                                                                                                                                                                                                                                              |
| [**WriteCertToCSP**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll-get_writecerttocsp)                                     | Establece o recupera un valor booleano que determina si un certificado se debe escribir en el [*proveedor de servicios criptográficos*](../secgloss/c-gly.md) (CSP).<br/>                                                                                                     |
| [**WriteCertToUserDS**](/windows/desktop/api/Xenroll/nf-xenroll-icenroll2-get_writecerttouserds)                               | Establece o recupera un valor booleano que determina si el certificado se escribe en el almacén de Active Directory del usuario.<br/>                                                                                                                                                                                                                               |
