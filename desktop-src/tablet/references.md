# <a name="references-tablet-pc"></a>Referencias (Tablet PC)

Para obtener información sobre cómo habilitar el reconocimiento de voz en la aplicación, consulte el SDK de Voz de Microsoft, que se puede encontrar en [Microsoft Speech SDK](https://www.microsoft.com/speech/download/sdk51/). Este sitio también contiene información general sobre las directrices de diseño de aplicaciones Speech API Microsoft.

Para obtener información sobre la accesibilidad de Microsoft, vea [Accesibilidad de Microsoft.](https://www.microsoft.com/enable/)

Para obtener más información sobre Microsoft Active Accessibility directrices para Windows aplicaciones basadas en aplicaciones, vea la [sección Accesibilidad.](../accessibility/accessibility.md)
