# <a name="managed-reference-for-wmi-windows-powershell-command-classes"></a>Referencia administrada para las clases de comandos de Windows PowerShell de WMI

Windows PowerShell implementa la funcionalidad de Instrumental de administración de Windows (WMI) a través de un conjunto de cmdlets. Puede usar estos cmdlets para completar las tareas de un extremo a otro que son necesarias para administrar equipos locales y remotos.

Consulte [Windows PowerShell](https://msdn.microsoft.com/library/dd835506(v=vs.85).aspx) para obtener más información.

## <a name="wmi-powershell-classes"></a>Clases de PowerShell de WMI

**Espacio de nombres**: Microsoft. PowerShell. Commands

**Ensamblado**: Microsoft. PowerShell. Commands. Management

**Dll**: Microsoft.PowerShell.Commands.Management.dll

Windows PowerShell implementa estas clases de comandos WMI. Estas clases se incluyen en este kit de desarrollo de software (SDK) solo para integridad. Los miembros de estas clases no se pueden usar directamente, ni deben usarse para derivar otras clases.



| Clase                   | Descripción                                                                                                                                                                                                                                 |
|-------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| GetWmiObjectCommand     | Obtiene instancias de clases WMI o información sobre las clases disponibles.<br/> Consulte el cmdlet [Get-WMIObject](/previous-versions//dd315295(v=technet.10)) para obtener ejemplos e información detallada sobre los parámetros.<br/> |
| InvokeWmiMethod         | Llama a los métodos de WMI.<br/> Consulte el cmdlet [Invoke-WmiMethod](/previous-versions//dd315300(v=technet.10)) para obtener ejemplos e información detallada sobre los parámetros.<br/>                                                     |
| RegisterWmiEventCommand | Se suscribe a un evento WMI.<br/> Consulte el cmdlet [Register-WmiEvent](/previous-versions//dd315242(v=technet.10)) para obtener ejemplos e información detallada sobre los parámetros.<br/>                                            |
| RemoveWmiObject         | Elimina una instancia de una clase WMI existente.<br/> Consulte el cmdlet [Remove-WMIObject](/previous-versions//dd347605(v=technet.10)) para obtener ejemplos e información detallada sobre los parámetros.<br/>                          |
| SetWmiInstance          | Crea o actualiza una instancia de una clase WMI existente.<br/> Consulte el cmdlet [set-WmiInstance](/previous-versions//dd315391(v=technet.10)) para obtener ejemplos e información detallada sobre los parámetros.<br/>                |
