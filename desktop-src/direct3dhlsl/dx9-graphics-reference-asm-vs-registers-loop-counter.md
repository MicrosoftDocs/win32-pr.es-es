# <a name="loop-counter-register-hlsl-vs-reference"></a>Registro de contadores de bucle (referencia de HLSL VS)

El único registro de este banco es el registro del contador de bucle actual (aL). Se incrementa automáticamente en cada ejecución del [bucle, en lugar](loop---vs.md)de. [ENDLOOP: bloque vs](endloop---vs.md) . Por lo tanto, se puede usar en el bloque para el direccionamiento relativo si es necesario y no es válido para usarlo fuera del bucle.
