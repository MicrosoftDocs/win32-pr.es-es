# <a name="adding-routes"></a>Agregar rutas

Una vez que un cliente ha detectado una ruta, el cliente puede agregarla a la tabla de enrutamiento llamando a la función [**RtmAddNextHop**](/windows/desktop/api/Rtmv2/nf-rtmv2-rtmaddnexthop) .

Un cliente puede quitar rutas de la tabla de enrutamiento llamando a la función [**RtmDeleteRouteToDest**](/windows/desktop/api/Rtmv2/nf-rtmv2-rtmdeleteroutetodest) .

Para ver el código de ejemplo que muestra cómo usar estas funciones, consulte [Agregar y actualizar rutas mediante RtmAddRouteToDest](add-and-update-routes-using-rtmaddroutetodest.md).
