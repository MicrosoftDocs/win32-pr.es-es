# <a name="parental-controls-robustness"></a>Solidez de los controles parentales

Tenga en cuenta que los problemas de ingeniería social relacionados con la aplicación de inicios de sesión basados en identidades, junto con los límites prácticos de los diseños de restricciones, normalmente impiden que las soluciones de seguridad sean robustas para los elementos secundarios determinados para outwitr los controles. El principio de Microsoft es que no podemos proteger a los niños que realizan esfuerzos intencionados sofisticados para eludir las restricciones. Podemos ayudarle a protegerse de acceso intencionado imprevisto o poco sofisticado. Es importante no establecer las expectativas de los padres y tutores demasiado alto en las soluciones de control parental.
