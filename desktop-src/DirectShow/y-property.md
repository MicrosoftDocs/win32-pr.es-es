# <a name="y-property"></a>Propiedad y

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

La `y` propiedad establece o recupera la posición y de la esquina superior izquierda del rectángulo.

``` syntax
[ iYPos = ] DVDRect.y
```

## <a name="return-value"></a>Valor devuelto

Devuelve la posición Y de la esquina superior izquierda del rectángulo como un entero.

## <a name="remarks"></a>Observaciones

Esta propiedad es de lectura/escritura y no tiene ningún valor predeterminado.
