# <a name="stackedby-argument-the-windows-shell"></a>Argumento STACKEDBY (Shell de Windows)

El `stackedby` argumento especifica la columna de propiedad por la que se van a apilar los resultados. Puede apilar por cualquier propiedad válida del sistema de propiedades.

## <a name="example"></a>Ejemplo


```
search:query=vacation&stackedby=System.Author
```



### <a name="argument-information"></a>Información de argumentos



|                          |                                         |
|--------------------------|-----------------------------------------|
| Sistema operativo mínimo | Windows Vista con Service Pack 1 (SP1) |
