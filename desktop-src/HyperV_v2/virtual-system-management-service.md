# <a name="virtual-system-management-service"></a>Servicio de administración del sistema virtual

El perfil de sistema virtual describe los objetos que componen una máquina virtual. Estos objetos incluyen el sistema base, los dispositivos que componen el sistema, la configuración del sistema y sus dispositivos, y el servicio de administración que realiza operaciones en el sistema.
