# <a name="simple-data-binding"></a>enlace de datos simple

La arquitectura de controles ActiveX define un mecanismo de enlace de datos, por el cual un control ActiveX puede especificar que una o varias de sus propiedades son enlazables. En la mayoría de los casos, un control enlazado a datos no debe requerir absolutamente el enlace de datos, de modo que se pueda insertar en un contenedor que no admita el enlace de datos. Obviamente, en tal situación, es posible que se reduzca la funcionalidad del control.
