# <a name="diskcost-dialog"></a>Cuadro de diálogo DiskCost

Normalmente, se muestra un cuadro de diálogo DiskCost cuando se establece la propiedad [**OutOfDiskSpace**](outofdiskspace.md) . Esto indica que no hay suficiente espacio en disco para la instalación seleccionada.

Este cuadro de diálogo también se puede activar en el cuadro de [diálogo de selección](selection-dialog.md) para proporcionar al usuario una visión general del espacio en disco necesario para la instalación.

Un cuadro de diálogo DiskCost es un cuadro de diálogo modal. Normalmente contiene un [pulsador](pushbutton-control.md) que devuelve el usuario al cuadro de diálogo anterior y un [control VolumeCostList](volumecostlist-control.md).
