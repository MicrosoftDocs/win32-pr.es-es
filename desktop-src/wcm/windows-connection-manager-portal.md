# <a name="windows-connection-manager"></a>Administrador de conexiones de Windows

## <a name="purpose"></a>Propósito

El administrador de conexiones de Windows (WCM) permite crear y configurar el software de Connection Manager.

## <a name="developer-audience"></a>Audiencia de desarrolladores

El administrador de conexiones de Windows está diseñado para que lo usen los desarrolladores con el software de desarrollo de C/C++.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

La API del administrador de conexiones de Windows se admite en Windows 8 y Windows Server 2012.
