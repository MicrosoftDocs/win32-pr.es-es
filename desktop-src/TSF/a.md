# <a name="a-text-services-framework"></a>A (marco de trabajo de servicios de texto)

<dl> <dt>

<span id="_tsf_anchor_gly"></span><span id="_TSF_ANCHOR_GLY"></span>**Premium**
</dt> <dd>

Uno de un conjunto de elementos que marcan el inicio y el final de un intervalo de texto. Consulte [delimitadores](ranges.md).

</dd> <dt>

<span id="_tsf_application_character_position_gly"></span><span id="_TSF_APPLICATION_CHARACTER_POSITION_GLY"></span>**posición de carácter de aplicación (ACP)**
</dt> <dd>

Posición de carácter de base cero, dentro de una secuencia de texto continua, que es relativa al inicio de la secuencia. Un ACP se indica con un valor entero largo. Vea [almacenes de texto](text-stores.md).

</dd> </dl>
