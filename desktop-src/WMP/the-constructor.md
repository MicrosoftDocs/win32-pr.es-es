# <a name="the-constructor"></a>Constructor

El constructor que el asistente proporciona almacena un puntero a la clase de complemento principal en una variable miembro. El asistente usa esta referencia para obtener acceso a las variables de miembro declaradas en la clase CSearch. No se necesita ningún cambio de implementación.
