# <a name="methodexceptionoccured"></a>MethodExceptionOccured

## <a name="text"></a>Texto

Excepción \\ " {0} \\ "

## <a name="type"></a>Tipo

Error

## <a name="description"></a>Descripción

Se encontró una excepción inesperada o HRESULT. Algunas excepciones/HRESULTS son válidas, pero en este caso, no se esperan los resultados. Este error debe solucionarse.
