# <a name="mergemodulecabinet"></a>MergeModule.CABinet

Cada archivo entregado por el módulo de combinación debe almacenarse en un archivo. cab que se incrusta como un flujo en el almacenamiento estructurado del módulo de combinación. En un módulo de combinación estándar, el nombre de este archivo. cab siempre es: MergeModule.CABinet. Para obtener más información, consulte [generación de archivos. cab de MergeModule.CABinet](generating-mergemodule-cabinet-cabinet-files.md).
