# <a name="colorcolor-constructors"></a>Constructores Color.Color

En este tema se enumeran los constructores de la [**clase**](/windows/win32/api/gdipluscolor/nl-gdipluscolor-color) Color. Para obtener una lista de clases completa, vea **Clase de color**.

### <a name="overload-list"></a>Lista de sobrecarga



| Constructor                                                               | Descripción                                                                                                                                                                                                         |
|:--------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Color(ARGB)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb))                   | Crea un [**objeto Color::Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inargb)) mediante un **valor ARGB.**<br/>                                                                                                    |
| [**Color(BYTE, BYTE, BYTE)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte))        | Crea un [**objeto Color::Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte)) utilizando los valores especificados para los componentes rojo, verde y azul. Este constructor establece el componente alfa en 255 (opaco).<br/> |
| [**Color(BYTE, BYTE, BYTE, BYTE)**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) | Crea un [**objeto Color::Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color(inbyte_inbyte_inbyte_inbyte)) utilizando los valores especificados para los componentes alfa, rojo, verde y azul.<br/>                                                   |
| [**Color()**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color)                            | Crea un [**objeto Color::Color**](/windows/win32/api/gdipluscolor/nf-gdipluscolor-color-color) y lo inicializa en negro opaco. Éste es el constructor predeterminado.<br/>                                                                |
