# <a name="create-a-windows-runtime-component-for-device-access"></a>Creación de un componente Windows Runtime para el acceso a dispositivos

Puede crear un componente Windows runtime para encapsular el componente que tiene acceso al controlador. Después, puede escribir una aplicación Windows Store para el dispositivo en C# o JavaScript.

## <a name="instructions"></a>Instructions

Consulte el [ejemplo de acceso de controlador personalizado](https://github.com/microsoftarchive/msdn-code-gallery-microsoft/tree/411c271e537727d737a53fa2cbe99eaecac00cc0/Official%20Windows%20Platform%20Sample/Custom%20driver%20access%20sample). Este ejemplo implementa un componente Windows Runtime para proporcionar la funcionalidad de un dispositivo especializado y usa la funcionalidad en una aplicación de dispositivo Windows Store.
