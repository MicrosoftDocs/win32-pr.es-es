# <a name="ms-ds-has-full-replica-ncs-attribute"></a>atributo MS-DS-tiene-Full-Replica-NC

Identifica las particiones que se mantienen como réplicas completas.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-DS-tiene-Full-Replica-NC              |
| Nombre para mostrar de LDAP | msDS-hasFullReplicaNCs                  |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1925                 |
| System-ID-GUID    | 1d3c2d18-42d0-4868-99fe-0eca1e6fa9f3    |
