# <a name="editboxjustification"></a>EDITBOX.justification

El **atributo** de justificación especifica o recupera la alineación del texto dentro del control de cuadro de edición.

``` syntax
        elementID.justification
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una cadena de **lectura** y escritura que contiene uno de los valores siguientes.



| Value  | Descripción                                                        |
|--------|--------------------------------------------------------------------|
| Left   | Predeterminada. Alinea el texto con el borde izquierdo del control de cuadro de edición. |
| Right  | Alinea el texto con el borde derecho del control de cuadro de edición.         |
| Center | Alinea el texto con el centro horizontal del control de cuadro de edición.  |
