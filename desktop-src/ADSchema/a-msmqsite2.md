# <a name="msmq-site-2-attribute"></a>Atributo MSMQ-Site-2

DN del segundo sitio de un par que está conectado.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | MSMQ-Site-2                             |
| Ldap-Display-Name | mSMQSite2                               |
| Size              | \-                                      |
| Actualizar privilegios  | \-                                      |
| Frecuencia de actualización  | Cada vez que se crea una línea de sitio de MSMQ. |
| Attribute-Id      | 1.2.840.113556.1.4.944                  |
| System-Id-Guid    | 9a0dc338-c100-11d1-gni5-0080c76670c0    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
