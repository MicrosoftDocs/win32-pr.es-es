# <a name="winsock-spi-structures"></a>Estructuras SPI de Winsock

En la lista siguiente se proporcionan descripciones concisas de cada estructura SPI de Winsock. Para obtener información adicional sobre cualquier estructura SPI de Winsock, haga clic en el nombre de la estructura.



| Estructura SPI de Winsock                                         | Descripción                                                                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**RUTINA DE \_ NSP**](/windows/desktop/api/Ws2spi/ns-ws2spi-nsp_routine)                           | Contiene información sobre las funciones implementadas por un proveedor de servicios de espacio de nombres versión 1 (NSPv1). |
| [**RUTINA DE \_ NSPV2**](/windows/desktop/api/Ws2spi/ns-ws2spi-nspv2_routine)                       | Contiene información sobre las funciones implementadas por un proveedor de servicios de espacio de nombres versión 2 (NSPv2). |
| [**WSATHREADID**](/windows/desktop/api/Ws2spi/ns-ws2spi-wsathreadid)                          | Permite a un proveedor identificar un subproceso en el que se pueden poner en cola las llamadas a procedimiento asincrónico (APC).           |
| [**INFORMACIÓN DE AUDITORÍA DEL PROVEEDOR DE WSC \_ \_ \_**](/windows/desktop/api/Ws2spi/ns-ws2spi-wsc_provider_audit_info) | Contiene información de auditoría para una entrada de proveedor de servicios por capas (LSP).                                        |
| [**WSPDATA**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspdata)                                  | Contiene información del proveedor de servicios.                                                                        |
| [**TABLA \_ WSPPROC**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspproc_table)                       | Contiene una tabla de punteros a funciones del proveedor de servicios.                                                   |
| [**WSPUPCALLTABLE**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspupcalltable)                      | Contiene una tabla de punteros a funciones de llamada upcall del proveedor de servicios.                                            |
