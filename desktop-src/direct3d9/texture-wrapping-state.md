# <a name="texture-wrapping-state-direct3d-9"></a>Estado de ajuste de textura (Direct3D 9)

Los estados de representación de D3DRS WRAP0 a D3DRS WRAP7 habilitan y deshabilitan el ajuste u- y v para varias texturas en la cascada \_ \_ multitexto del dispositivo. Puede establecer estos estados de representación en una combinación de las marcas D3DWRAPCOORD \_ 0, D3DWRAPCOORD \_ 1, D3DWRAPCOORD 2 y \_ D3DWRAPCOORD 3 para habilitar el ajuste en las direcciones primera, segunda, tercera y cuarta de \_ la textura. Use un valor de cero para deshabilitar el ajuste por completo. El ajuste de textura está deshabilitado en todas las direcciones para todas las fases de textura de forma predeterminada.

Para obtener información general conceptual, vea [Ajuste de texturas (Direct3D 9).](texture-wrapping.md)
