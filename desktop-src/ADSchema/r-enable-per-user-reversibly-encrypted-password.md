# <a name="enable-per-user-reversibly-encrypted-password-extended-right"></a>Enable-Per-User-Vertibly-Encrypted-Password extended right

Derecho de acceso de control extendido que permite a los usuarios habilitar o deshabilitar la configuración de "contraseña cifrada reversible" para objetos de usuario y equipo.



| Entrada | Valor |
|--------------|-----------------------------------------------|
| CN           | Enable-Per-User-Vertibly-Encrypted-Password |
| Display-Name | Habilitar contraseña cifrada invertiblemente por usuario |
| Rights-GUID  | 05c74c5e-4deb-43b4-bd9f-86664c2a7fd5          |
