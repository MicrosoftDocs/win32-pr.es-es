# <a name="texture1dsamplecmp-methods"></a>Métodos Texture1D::SampleCmp

Muestrea [**un objeto Texture1D**](sm5-object-texture1d.md)con un valor de comparación para rechazar muestras.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                         | Descripción                                                                                                                                                                           |
|:-----------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp(S,float,float,int)**](dx-graphics-hlsl-to-samplecmp.md)                          | Muestrea una textura con un valor de comparación para rechazar muestras.<br/>                                                                                                             |
| [**SampleCmp(S,float,float,int,float)**](t1d-samplecmp-s-float-float-int-float-.md)           | Muestrea una textura con un valor de comparación para rechazar muestras, con un valor opcional para fijar los valores de nivel de detalle (LOD) de la muestra.<br/>                                     |
| [**SampleCmp(S,float,float,int,float,uint)**](t1d-samplecmp-s-float-float-int-float-uint-.md) | Muestrea una textura con un valor de comparación para rechazar muestras, con un valor opcional para fijar los valores de nivel de detalle (LOD) de la muestra. Devuelve el estado de la operación.<br/> |
