# <a name="translating-to-c-from-java"></a>Traducir a C++ desde Java

Java administra los punteros y la administración de memoria implícitamente. En C++, la aplicación es responsable de la asignación y desasignación de memoria, así como de realizar cualquier aritmética de puntero necesaria.
