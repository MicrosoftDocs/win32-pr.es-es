# <a name="ms-ds-approx-immed-subordinates-attribute"></a>atributo MS-DS-aprox-immed-Subordinates

El valor devuelto por este atributo se basa en los tamaños de índice. Esto puede estar desactivado por +/10% en contenedores grandes y el error no está enlazado teóricamente, pero el uso de este atributo ayuda a la interfaz de usuario a mostrar el contenido de un contenedor.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-DS-aprox-immed-subordinados      |
| Nombre para mostrar de LDAP | msDS-aprox-immed-subordinados       |
| Tamaño              | \-                                   |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1669              |
| System-ID-GUID    | e185d243-f6ce-4adb-b496-b0c005d7823c |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
