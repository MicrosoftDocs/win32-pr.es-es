# <a name="mf_stf_version_date-attribute"></a>Atributo MF \_ STF \_ VERSION \_ DATE

Este atributo almacena una [filetime que](/windows/desktop/api/minwinbase/ns-minwinbase-filetime) representa la marca de fecha y hora de un generador de transformaciones de sensor.

## <a name="data-type"></a>Tipo de datos

**UINT64**

| Servidor mínimo compatible<br/> | No se admite ninguno<br/>                                                          |
