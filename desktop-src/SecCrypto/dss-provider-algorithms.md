# <a name="dss-provider-algorithms"></a>Algoritmos de proveedor de DSS

En la tabla siguiente se enumeran los algoritmos admitidos por el proveedor de servicios criptográficos de Microsoft DSS.



| Id. de algoritmo    | Descripción                                 | Comentarios                                                                                                        |
|-----------------|---------------------------------------------|-----------------------------------------------------------------------------------------------------------------|
| CALG \_ MD5       | Algoritmo de hash MD5.                      | Solo se proporciona para la hash.                                                                                      |
| CALG \_ SHA       | Algoritmo de hash SHA.                      | Debe usarse para las firmas de DSS.                                                                                |
| CALG \_ SHA1      | Igual que CALG \_ SHA.                          | Sin comentarios.                                                                                                     |
| CALG \_ DSS \_ SIGN | Algoritmo de firma de clave pública o privada de DSS. | Longitud de clave: se puede establecer 512 bits en 1024 bits en incrementos de 64 bits. Longitud de clave predeterminada: 1024 bits.<br/> |
