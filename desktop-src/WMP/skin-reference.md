# <a name="skin-reference"></a>Referencia de máscara

Esta referencia se divide en las secciones siguientes.



| Sección                        | Descripción                                                                     |
|--------------------------------|---------------------------------------------------------------------------------|
| [Carátula de álbum](album-art.md)     | Información sobre cómo incluir carátulas de álbum en máscaras.                                 |
| [Mapas de bits](bitmaps.md)         | Información sobre los archivos de arte utilizados en la creación de máscaras.                          |
| [Botones](buttons.md)         | Información sobre los controles de pulsador disponibles en las máscaras.                       |
| [Descripción](description.md) | Información sobre cómo especificar dimensiones y orientación de la máscara.                   |
| [Marquesina](marquee.md)         | Información sobre el uso de cuadros de texto de desplazamiento en máscaras.                          |
| [Clasificaciones](ratings.md) \*      | Información sobre el valor de clasificación asignado a un elemento multimedia.                   |
| [Estado](status.md)           | Información sobre cómo incluir una presentación de estado en las máscaras.                          |
| [Texto](text.md)               | Información sobre el uso de cuadros de texto para mostrar información en máscaras.             |
| [Trackbars](trackbars.md)     | Información sobre el uso de controles deslizantes de TrackBar para el volumen y la posición de búsqueda. |
| [Vídeo](video.md)             | Información sobre cómo incluir un área de presentación de vídeo en las máscaras.                      |



 

\*Novedades de Windows Media Player 10,1 Mobile.

Cada sección corresponde a una sección del archivo de definición de máscara que se debe proporcionar para usar elementos concretos. Las secciones se enumeran aquí en orden alfabético. Para obtener información sobre el orden en el que deben aparecer en un archivo de definición de máscara, vea [crear un archivo de definición de máscara](creating-a-skin-definition-file.md). Los detalles sobre la composición del archivo de definición de máscara se proporcionan en la sección [acerca de las máscaras](about-skins-mobile.md) .
