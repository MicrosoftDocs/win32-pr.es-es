# <a name="translating-to-jscript"></a>Traducir a JScript

JScript es la implementación de Microsoft de la especificación del lenguaje ECMA 262. Es compatible con Internet Explorer y Netscape Navigator. JScript es el primer lenguaje de scripting que se ajusta al estándar ECMA 262.

JScript es compatible con una amplia gama de exploradores Web, incluidos Internet Explorer y navegador.

Además de las características de lenguaje proporcionadas por JScript, el host de scripting también puede exponer un modelo de objetos al que pueda tener acceso el script. Por ejemplo, Internet Explorer expone un modelo de objetos que permite que los scripts interactúen con el explorador y lo controlen mediante programación. Para obtener más información sobre cómo trabajar con estos modelos de objetos, consulte la documentación del host de scripting.

Para obtener más información sobre JScript, vea la referencia del lenguaje JScript. Para obtener más información, vea la [especificación del lenguaje ECMA 262](https://www.ecma-international.org/publications/standards/Ecma-262.htm).

En los temas siguientes se describen los problemas que debe tener en cuenta al traducir un script a JScript desde JavaScript o VBScript:

-   [Traducir a JScript desde JavaScript](translating-to-jscript-from-javascript.md)
