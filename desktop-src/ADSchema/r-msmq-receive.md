# <a name="msmq-receive-extended-right"></a>MSMQ-recibir derecho extendido

|--------------|--------------------------------------|
| CN           | MSMQ: recepción                         |
| Display-Name | Recibir mensaje                      |
| Rights-GUID  | 06bd3200-df3e-11d1-9c86-006008764d0e |
