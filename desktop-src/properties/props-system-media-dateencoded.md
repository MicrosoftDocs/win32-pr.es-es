# <a name="systemmediadateencoded"></a>System. Media. DateEncoded

Representa la fecha y hora en que se ha codificado el archivo. La fecha y hora está en UTC (en el documento, no en el sistema de archivos).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.Media.DateEncoded
   shellPKey = PKEY_Media_DateEncoded
   formatID = 2E4B640D-5019-46D8-8881-55414CC5CAA0
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = DateTime
```

## <a name="windows-vista"></a>Windows Vista

```
propertyDescription
   name = System.Media.DateEncoded
   shellPKey = PKEY_Media_DateEncoded
   formatID = 2E4B640D-5019-46D8-8881-55414CC5CAA0
   propID = 100
   SearchInfo
      IsColumn = true
   typeInfo
      type = DateTime
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
