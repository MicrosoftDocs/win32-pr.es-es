# <a name="conversion-of-mts-packages-to-com-applications"></a>Conversión de paquetes MTS a aplicaciones COM+

COM+ se puede ver como una combinación de Microsoft Transaction Server (MTS) y COM, junto con la introducción de nuevas características.

En su mayor parte, los paquetes MTS se convierten en aplicaciones COM+ durante el Windows de instalación. Sin hacer nada más allá de la configuración típica, puede aprovechar todas las nuevas características de COM+. Para obtener una lista y una breve explicación de las mejoras más significativas de COM+, vea [Información general de programación de COM+.](com--programming-overview.md)

Si desea obtener más información sobre la conversión automática de paquetes MTS en aplicaciones COM+ durante el proceso de instalación de Windows, así como sobre cómo solucionar los errores que pueden producirse durante este proceso, vea Conversión automática desde [MTS.](automatic-conversion-from-mts.md)

Si desea obtener información sobre cómo convertir manualmente paquetes MTS en aplicaciones COM+, consulte [Conversión manual de MTS.](manual-conversion-from-mts.md)

También debe tener en cuenta que la instalación de COM+ presenta varias funcionalidades nuevas que los paquetes MTS pueden usar como aplicaciones COM+. Estos se explican en Resultados y problemas [de conversión de COM+.](com--conversion-results-and-issues.md)
