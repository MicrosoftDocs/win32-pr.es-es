# <a name="changecurrentangle"></a>ChangeCurrentAngle

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

El `ChangeCurrentAngle` evento se envía cuando el disco habilita o deshabilita el cambio del ángulo.

``` syntax
ChangeCurrentAngle(bEnabled)
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

<span id="bEnabled"></span><span id="benabled"></span><span id="BENABLED"></span>*bEnabled*
</dt> <dd>

Especifica si la operación está habilitada o deshabilitada como un valor booleano.

</dd> </dl>
