# <a name="pathgradientbrushsetcenterpoint-methods"></a>Métodos PathGradientBrush. SetCenterPoint

En este tema se enumeran los métodos SetCenterPoint de la clase [**PathGradientBrush**](/windows/win32/api/gdipluspath/nl-gdipluspath-pathgradientbrush) . Para obtener una lista completa de los métodos de la clase **PathGradientBrush** , consulte [métodos PathGradientBrush](-gdiplus-class-pathgradientbrush-methods.md).

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                           | Descripción                                                                                                                                                                                                                                                                                                                                    |
|:-------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SetCenterPoint (punto&)**](/windows/win32/api/gdipluspath/nf-gdipluspath-pathgradientbrush-setcenterpoint(inconstpoint_))   | El método [**PathGradientBrush:: SetCenterPoint**](/windows/win32/api/gdipluspath/nf-gdipluspath-pathgradientbrush-setcenterpoint(inconstpoint_)) establece el punto central de este pincel de degradado de trazado. De forma predeterminada, el punto central se encuentra en el centroide del trazado del límite del pincel, pero puede establecer el punto central en cualquier ubicación dentro o fuera del trazado.<br/>  |
| [**SetCenterPoint (& de PointF)**](/previous-versions//ms535095(v=vs.85)) | El método [**PathGradientBrush:: SetCenterPoint**](/previous-versions//ms535095(v=vs.85)) establece el punto central de este pincel de degradado de trazado. De forma predeterminada, el punto central se encuentra en el centroide del trazado del límite del pincel, pero puede establecer el punto central en cualquier ubicación dentro o fuera del trazado.<br/> |
