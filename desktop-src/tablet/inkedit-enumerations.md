# <a name="inkedit-enumerations"></a>Enumeraciones de InkEdit

Esta sección contiene enumeraciones que pertenecen al control InkEdit.



| Enumeración                                            | Descripción                                                                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AppearanceConstants**](/windows/desktop/api/inked/ne-inked-appearanceconstants)     | Especifica si el control aparece plano o 3D.<br/>                                                                                            |
| [**BorderStyleConstants**](/windows/desktop/api/inked/ne-inked-borderstyleconstants)   | Especifica si el control tiene un borde.<br/>                                                                                                   |
| [**InkApplicationGesture**](/windows/desktop/api/msinkaut/ne-msinkaut-inkapplicationgesture) | Especifica el interés en un conjunto de gestos específicos de la aplicación.<br/>                                                                             |
| [**InkDisplayMode**](/windows/desktop/api/inked/ne-inked-inkdisplaymode)               | Especifica si una selección aparece como entrada de lápiz o texto.<br/>                                                                                         |
| [**InkEditStatus**](/windows/desktop/api/inked/ne-inked-inkeditstatus)                 | Especifica si el control [InkEdit](inkedit-control-reference.md) está inactivo, recopilando entradas manuscritas o reconociendo entradas de lápiz.<br/>                           |
| [**InkInsertMode**](/windows/desktop/api/inked/ne-inked-inkinsertmode)                 | Especifica cómo se inserta la entrada de lápiz en el control [InkEdit](inkedit-control-reference.md) .<br/>                                                      |
| [**InkMode**](/windows/desktop/api/inked/ne-inked-inkmode)                             | Especifica la configuración del modo de recopilación para la tinta dibujada: Si la colección de tintas está deshabilitada, se recopila la tinta o se recopilan los movimientos y las entradas de lápiz.<br/> |
| [**InkMouseButton**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousebutton)               | Especifica qué botón del mouse se presionó.<br/>                                                                                                     |
| [**InkMousePointer**](/windows/desktop/api/msinkaut/ne-msinkaut-inkmousepointer)             | Especifica el tipo de puntero del mouse que aparece.<br/>                                                                                             |
| [**MouseButton**](/windows/desktop/api/inked/ne-inked-mousebutton)                     | Especifica qué botón del mouse se presionó.<br/>                                                                                                     |
| [**ScrollBarsConstants**](/windows/desktop/api/inked/ne-inked-scrollbarsconstants)     | Especifica cómo las barras de desplazamiento de un control [InkEdit](inkedit-control-reference.md) aparecen en la pantalla.<br/>                                    |
| [**SelAlignmentConstants**](/windows/desktop/api/inked/ne-inked-selalignmentconstants) | Especifica la alineación del párrafo con respecto a los márgenes del control [InkEdit](inkedit-control-reference.md) .<br/>                     |
