# <a name="erasing-ink-with-the-pen"></a>Borrar la entrada de lápiz con el lápiz

Además de habilitar la escritura a mano y el dibujo, puede compilar la aplicación para admitir el uso del lápiz de tableta para borrar la entrada de lápiz. Use el [**objeto InkOverlay**](inkoverlay-class.md) para hacerlo.

En las secciones siguientes se describe el borrado de la entrada de lápiz con la tableta.

-   [Borrado mediante el objeto InkOverlay](erasing-by-using-the-inkoverlay-object.md)
-   [Borrado mediante el lápiz](erasing-by-using-the-pen.md)
