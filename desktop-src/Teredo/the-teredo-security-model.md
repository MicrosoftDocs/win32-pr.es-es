# <a name="the-teredo-security-model"></a>El modelo de seguridad de Teredo

La tecnología [Teredo](about-teredo.md) , junto con Windows Security Center y [firewall de Windows](/previous-versions/windows/desktop/ics/windows-firewall-start-page), proporciona seguridad mejorada para el usuario final. En el caso de Firewall de Windows, el tráfico IPv6 enviado y recibido a través de la interfaz Teredo se procesa igual que el tráfico IPv6 nativo. La interfaz Teredo se reconoce como otro minipuerto en el sistema.

Para obtener más información sobre el uso, vea [implementar el modelo de seguridad de Teredo](implementing-the-teredo-security-model.md).
