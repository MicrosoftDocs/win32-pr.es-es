# <a name="windows-installer-and-logo-requirements"></a>Requisitos de Windows Installer y logotipo

El logotipo certificado para Microsoft Windows identifica los productos que se han comprobado mediante pruebas independientes para cumplir con la especificación de la aplicación para Windows. Esta especificación fue desarrollada por Microsoft en cooperación con clientes y otros desarrolladores para ofrecer un mapa de carreteras para crear aplicaciones confiables y fáciles de administrar. Los proveedores de software que cumplen con la especificación se reúnen en el logotipo certificado para Microsoft Windows y, a continuación, conceden una licencia del logotipo para su uso en su paquete de productos, publicidad, material publicitario y otros materiales de marketing.

Para obtener más información acerca de la certificación de Windows Vista, Windows XP y Windows Server 2008 para las aplicaciones, consulte el [Microsoft Partner Network]( https://partner.microsoft.com).

Para obtener más información acerca de la certificación Windows Server 2003 y Windows 2000 para las aplicaciones, vea [https://msdn.microsoft.com/certification](https://www.microsoft.com/windowsserver2003/partners/isvs/cfw.mspx) .

Si va a crear un paquete de instalación para la aplicación, puede aprovechar las capacidades de la Microsoft Windows Installer para ayudar a cumplir algunos de los requisitos de certificación. En la tabla siguiente se proporcionan vínculos a información en Windows Installer documentación pertinente para algunos de los requisitos de certificación. Tenga en cuenta que no todos los requisitos para la certificación descritos en la especificación de la aplicación dependen de Windows Installer.
