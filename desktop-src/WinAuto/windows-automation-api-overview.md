# <a name="windows-automation-api-overview"></a>Introducción a la API de automatización de Windows

En esta sección se proporciona información general de alto nivel de la API de automatización de Microsoft Windows 3,0, que incluye el Active Accessibility heredado de Microsoft, y la implementación de Windows de la especificación de la automatización de la interfaz de usuario de Microsoft. En la información general se resaltan las similitudes y diferencias entre Microsoft Active Accessibility y la automatización de la interfaz de usuario, se describen los componentes y las características que permiten que las dos tecnologías funcionen juntas y se proporcionan directrices para elegir la tecnología que se va a implementar.

Esta sección contiene los siguientes temas:

-   [Comparación de Microsoft Active Accessibility y UI Automation](microsoft-active-accessibility-and-ui-automation-compared.md)
-   [Arquitectura e interoperabilidad](architecture-and-interoperability.md)
-   [Especificación de UI Automation](ui-automation-specification.md)
-   [interoperabilidad de 32 y 64 bits](32-bit-and-64-bit-interoperability.md)
-   [Procedimientos de accesibilidad recomendados](accessibility-best-practices.md)
