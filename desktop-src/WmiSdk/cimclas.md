# <a name="cim-classes-wmi"></a>Clases CIM (WMI)

Las clases cim (CIM) del Grupo de tareas de administración distribuida (DMTF Modelo de información común) son las clases primarias en las que se han creado las clases WMI. En la tabla siguiente se enumeran las categorías de clases CIM.

> [!IMPORTANT]
> Las clases CIM (Modelo de información común) DMTF (Distributed Management Task Force) son las clases primarias en las que se han creado las clases WMI. WMI admite actualmente solo los esquemas [de la versión CIM 2.x](https://dmtf.org/standards/cim/schemas). Para obtener más información sobre CIM, [vea Modelo de información común](common-information-model.md).
