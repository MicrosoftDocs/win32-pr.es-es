# <a name="hcryptprov_legacy"></a>HCRYPTPROV \_ LEGACY

El tipo de datos **HCRYPTPROV \_ LEGACY** se usa para reemplazar el tipo de datos [**HCRYPTPROV**](hcryptprov.md) donde ya no se usa el tipo de datos **HCRYPTPROV.** Los parámetros de función y las estructuras de datos de **tipo HCRYPTPROV \_ LEGACY** deben establecerse en **NULL.**


```C++
typedef ULONG_PTR HCRYPTPROV_LEGACY;
```
