# <a name="routing-protocols"></a>Protocolos de enrutamiento

## <a name="purpose"></a>Propósito

Los desarrolladores pueden usar las API del Protocolo de enrutamiento para implementar protocolos de enrutamiento.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Las API de enrutamiento están diseñadas para que las usen los programadores de C/C++. Los programadores también deben estar familiarizados con los conceptos de red.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

Los protocolos de enrutamiento son una tecnología basada en servidor. Toda la funcionalidad de los protocolos de enrutamiento se incorpora en Windows Server 2008, Windows Server 2003 y Windows 2000 Server. Las aplicaciones de enrutamiento no se pueden ejecutar en sistemas operativos cliente, como Windows XP o Windows Vista. Para obtener información más específica acerca de los sistemas operativos que admiten una función determinada, consulte las secciones de requisitos en la documentación de.
