# <a name="working-with-the-library"></a>Trabajar con la biblioteca

La biblioteca es una base de datos de información sobre el contenido multimedia que se almacena en el equipo del usuario o que se ha reproducido allí. Parte de la información se muestra en la característica biblioteca de Windows Media Player. Se puede obtener acceso a un conjunto mayor de información mediante programación.

Los elementos multimedia se pueden organizar en listas de reproducción. Los usuarios o el código pueden crear listas de reproducción. El modelo de objetos de Windows Media Player proporciona objetos y métodos que puede usar para manipular elementos multimedia individuales y listas de reproducción.

En las siguientes secciones se explican las técnicas para trabajar con la biblioteca:

-   [Acceso a bibliotecas](library-access.md)
-   [Administrar elementos multimedia](managing-media-items.md)
-   [Atributos de elemento multimedia](media-item-attributes.md)
-   [Administrar listas de reproducción](managing-playlists.md)
