# <a name="d3dcursorcaps"></a>D3DCURSORCAPS

Marcas de funcionalidad del cursor del controlador.



| \#Definir              | Value       | Descripción                                                                                     |
|-----------------------|-------------|-------------------------------------------------------------------------------------------------|
| D3DCURSORCAPS \_ COLOR  | 0x00000001L | El controlador admite cursor de color de hardware en al menos modos de alta resolución (alto >= 400). |
| D3DCURSORCAPS \_ LOWRES | 0x00000002L | El controlador admite el cursor de color de hardware en modos de baja resolución (alto < 400).            |



 

## <a name="constant-information"></a>Información constante



| Requisito                         | Value           |
|--------------------------|------------|
| Encabezado                   | d3d9caps.h |
| Sistema operativo mínimo | Windows 98 |
