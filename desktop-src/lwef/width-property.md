# <a name="width-property-commandswindow-object"></a>Propiedad width (objeto CommandsWindow)

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Devuelve un valor entero que especifica el ancho, en píxeles, de la ventana comandos de voz.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintáctica** 
</dt> <dd>

*agente * * *. CommandsWindow. width**

</dd> </dl>

### <a name="remarks"></a>Observaciones

El servidor muestra la ventana comandos de voz en función de la posición y el tamaño establecidos por el usuario.
