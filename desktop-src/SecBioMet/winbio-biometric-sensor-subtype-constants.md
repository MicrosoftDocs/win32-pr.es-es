# <a name="winbio_biometric_sensor_subtype-constants"></a>WINBIO \_ constantes de \_ subtipo del sensor biométrico \_

Las constantes siguientes se pueden usar como una máscara de máscara para el parámetro *Capabilities* de la estructura de esquema de la [**\_ unidad \_ WINBIO**](winbio-unit-schema.md) . Estos se refieren a las capacidades del sensor incorporado.



| Constante                                                                                                                                                                                                            | Descripción                                        |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------|
| <span id="WINBIO_SENSOR_SUBTYPE_UNKNOWN"></span><span id="winbio_sensor_subtype_unknown"></span><dl> <dt>**subtipo de sensor de WINBIO \_ \_ \_ desconocido**</dt> </dl>     | No se conocen los subtipos de sensor.<br/>      |
| <span id="WINBIO_FP_SENSOR_SUBTYPE_SWIPE"></span><span id="winbio_fp_sensor_subtype_swipe"></span><dl> <dt>**WINBIO \_ de \_ subtipo de sensor FP \_ \_**</dt> </dl> | El sensor admite los deslizamientos de huella digital.<br/> |
| <span id="WINBIO_FP_SENSOR_SUBTYPE_TOUCH"></span><span id="winbio_fp_sensor_subtype_touch"></span><dl> <dt>**\_toque de \_ subtipo de sensor FP \_ \_ de WINBIO**</dt> </dl> | El sensor es compatible con los dedos.<br/>     |
