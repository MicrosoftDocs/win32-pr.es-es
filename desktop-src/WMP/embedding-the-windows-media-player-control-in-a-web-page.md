# <a name="embedding-the-windows-media-player-control-in-a-web-page"></a>Insertar el control Reproductor de Windows Media en una página web

Al insertar el control Reproductor de Windows Media en una página web, puede tomar un enfoque básico o un enfoque más elaborado.

En el enfoque básico, el control se usa simplemente para reproducir música de fondo o para agregar una presentación de vídeo a la página.

En el enfoque más elaborado, la página se hace interactiva mediante la adición de código de script para manipular el control a través del modelo de objetos player. Esto le permite proporcionar una interfaz de usuario de su propio diseño o responder dinámicamente a comandos o eventos de script, como transiciones de medios.

Para obtener más información, [vea Using the Reproductor de Windows Media Control in a Web Page](using-the-windows-media-player-control-in-a-web-page.md).
