# <a name="ms-pki-template-minor-revision-attribute"></a>Atributo ms-PKI-Template-Minor-Revision

Realiza un seguimiento de los atributos de la clase que cambian. Sin embargo, esto no desencadenará la inscripción automática.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Template-Minor-Revision                                                                    |
| Ldap-Display-Name | msPKI-Template-Minor-Revision                                                                     |
| Size              | 4 bytes                                                                                           |
| Actualizar privilegios  | Administrador de dominio                                                                              |
| Frecuencia de actualización  | Cuando se edita, crea o clona el objeto de plantilla de certificado (ms-PKI-Certificate-Template). |
| Attribute-Id      | 1.2.840.113556.1.4.1435                                                                           |
| System-Id-Guid    | 13f5236c-1884-46b1-b5d0-484e38990d58                                                              |
| Syntax            | [**Enumeración**](s-enumeration.md)                                                              |
