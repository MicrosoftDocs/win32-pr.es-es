# <a name="itsubstreamcontrol-mspi"></a>ITSubStreamControl (MSPI)

La interfaz [**ITSubStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstreamcontrol) se expone en el objeto de secuencia si hay un MSP asociado e implementa subsecuencias. Los métodos de esta interfaz permiten a una aplicación crear y quitar subsecuencias.

La interfaz [**ITSubStreamControl**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstreamcontrol) se implementa mediante un MSP. Consulte **ITSubStreamControl** en la sección interfaz MSP para obtener más información sobre esta interfaz.
