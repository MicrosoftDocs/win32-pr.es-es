# <a name="r"></a>R

## <a name="rendering-intent"></a>intención de representación

Se aproxima a la representación del color de un espacio de color a otro. Las intenciones de representación que se usan en los perfiles DE C#, se definen mediante International Color Consortium (ICC).

## <a name="rgb"></a>RGB

El espacio de colores Rojo, Verde y Azul.

## <a name="rgb-triple"></a>RGB Triple

Cantidad que contiene valores de espacio de color RGB.
