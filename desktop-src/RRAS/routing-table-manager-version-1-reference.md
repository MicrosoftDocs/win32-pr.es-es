# <a name="routing-table-manager-version-1-reference"></a>Referencia de la versión 1 del administrador de tablas de enrutamiento

**Windows Server 2003:** Esta API se ha sustituido por la API del [Administrador de tablas de enrutamiento versión 2](about-routing-table-manager-version-2.md) y no estará disponible más allá de Windows Server 2003. Las nuevas aplicaciones deben usar la API del administrador de tablas de enrutamiento versión 2.

Las siguientes funciones, estructuras y constantes proporcionan una interfaz que los protocolos de enrutamiento pueden utilizar para tener acceso a las tablas de enrutamiento que mantiene el administrador de tablas de enrutamiento:

-   [Funciones de la versión 1 del administrador de tablas de enrutamiento](routing-table-manager-version-1-functions.md)
-   [Estructuras de la versión 1 del administrador de tablas de enrutamiento](routing-table-manager-version-1-structures.md)
