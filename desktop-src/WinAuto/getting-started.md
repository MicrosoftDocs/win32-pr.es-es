# <a name="getting-started-microsoft-active-accessibility"></a>Introducción (Microsoft Active Accessibility)

Microsoft Active Accessibility es un conjunto de interfaces COM (modelo de objetos componentes) y elementos de API que proporcionan una manera confiable de exponer y recopilar información sobre los elementos de interfaz de usuario y el contenido web basados en Microsoft Windows. Con esta información, los proveedores de tecnología de asistencia pueden representar la interfaz de usuario en formatos alternativos, como voz o Braille, y las aplicaciones de comando de voz y control pueden manipular la interfaz de forma remota. Microsoft Active Accessibility se basa en la tecnología de Windows y solo se puede usar en conjunción con controles basados en Windows y otras aplicaciones de Windows.
