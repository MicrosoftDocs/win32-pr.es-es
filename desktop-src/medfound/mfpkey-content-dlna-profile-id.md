# <a name="mfpkey_content_dlna_profile_id-property"></a>\_Propiedad de \_ \_ identificador de perfil DLNA de contenido de MFPKEY \_

Contiene el identificador de Perfil de la Alianza de redes digitales (DLNA) para un origen de medios.



Tipo de datos

Tipo PROPVARIANT (VT)

Miembro de PROPVARIANT

**WCHAR \_ t \** _

VT \_ LPWStr

_ *pwszVal**



## <a name="remarks"></a>Observaciones

Un identificador de perfil DLNA es una cadena que identifica el perfil de codificación del contenido.

Los identificadores de perfil se describen en las [instrucciones de interoperabilidad de dispositivos en red DLNA](https://www.dlna.org/about/organization). (Es posible que este recurso no esté disponible en algunos idiomas y países).
