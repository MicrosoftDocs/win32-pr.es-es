# <a name="netconfiggetall"></a>NetConfigGetAll

La función **NetConfigGetAll** está obsoleta. Se incluye por compatibilidad con las versiones de 16 bits de Windows. Otras aplicaciones deben usar el [registro](/windows/desktop/SysInfo/registry).
