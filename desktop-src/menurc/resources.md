# <a name="menus-and-other-resources"></a>Menús y otros recursos

Un *recurso* es datos binarios que se pueden agregar al archivo ejecutable de una aplicación basada en Windows. Un recurso puede ser estándar o definido. Los datos de un *recurso estándar* describen un icono, un cursor, un menú, un cuadro de diálogo, un mapa de bits, un metarchivo mejorado, una fuente, una tabla de aceleradores, una entrada de tabla de mensajes, una entrada de tabla de cadenas o información de versión. Un *recurso definido* por la aplicación, también denominado *recurso personalizado*, contiene los datos necesarios para una aplicación específica.
