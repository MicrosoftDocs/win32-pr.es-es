# <a name="idcompositionscaletransformsetcenterx-methods"></a>IDCompositionScaleTransform:: SetCenterX (métodos)

Cambia o anima el valor de la propiedad CenterX de una transformación de escala 2D. La propiedad CenterX especifica la coordenada x del punto sobre el que se realiza el ajuste de escala.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                                        | Descripción                                            |
|:--------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setcenterx(float))                                     | Cambia el valor de la propiedad CenterX.<br/>  |
| [**SetCenterX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionscaletransform-setcenterx(idcompositionanimation)) | Anima el valor de la propiedad CenterX.<br/> |
