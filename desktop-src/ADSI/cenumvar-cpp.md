| Método                                          | Descripción                                                                           |
|-------------------------------------------------|---------------------------------------------------------------------------------------|
| **CSampleDSEnumVariant::CSampleDSEnumVariant**  | Constructor estándar.                                                                 |

| **CSampleDSEnumVariant::QueryInterface**        | Implementación [**estándar de IUnknown::QueryInterface.**](/windows/win32/api/unknwn/nf-unknwn-iunknown-queryinterface(q)) |
| **CSampleDSEnumVariant::AddRef**                | Implementación [**estándar de IUnknown::AddRef.**](/windows/win32/api/unknwn/nf-unknwn-iunknown-addref)                 |
| **CSampleDSEnumVariant::Release**               | Implementación [**de IUnknown::Release**](/windows/win32/api/unknwn/nf-unknwn-iunknown-release) estándar.               |
| **CSampleDSEnumVariant::Skip**                  | Implementación **IEnumXXXX::Skip** estándar.                                          |
| **CSampleDSEnumVariant::Reset**                 | **IEnumXXXX estándar::Reset** implementation.                                         |
| **CSampleDSEnumVariant::Clone**                 | Implementación **estándar de IEnumXXXX::Clone.**                                         |
