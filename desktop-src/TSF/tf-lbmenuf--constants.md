# <a name="tf_lbmenuf_-constants"></a>Constantes de TF \_ LBMENUF \_ \*

Las constantes ** \_ TF \_ \* LBMENUF* _ se usan en el método [ITfMenu:: AddMenuItem](/windows/desktop/api/Ctfutb/nf-ctfutb-itfmenu-addmenuitem) para especificar las características de un elemento de menú en la barra de idioma.



| Constante o valor                                                                                                                                                                                                                                         | Descripción                                     |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------|
| <span id="TF_LBMENUF_CHECKED"></span><span id="tf_lbmenuf_checked"></span><dl> <dt>_ * TF \_ LBMENUF \_ activado * *</dt> <dt>0x01</dt> </dl>                | El elemento de menú está activado.<br/>            |
| <span id="TF_LBMENUF_SUBMENU"></span><span id="tf_lbmenuf_submenu"></span><dl> <dt>**TF \_ \_Submenú LBMENUF**</dt> <dt>0x02</dt> </dl>                | El elemento de menú es un submenú.<br/>          |
| <span id="TF_LBMENUF_SEPARATOR"></span><span id="tf_lbmenuf_separator"></span><dl> <dt>**TF \_ \_Separador de LBMENUF**</dt> <dt>0x04</dt> </dl>          | El elemento de menú es un separador.<br/>        |
| <span id="TF_LBMENUF_RADIOCHECKED"></span><span id="tf_lbmenuf_radiochecked"></span><dl> <dt>**TF \_ LBMENUF \_ RADIOchecked**</dt> <dt>0x08</dt> </dl> | El elemento de menú es una marca de verificación de radio.<br/> |
| <span id="TF_LBMENUF_GRAYED"></span><span id="tf_lbmenuf_grayed"></span><dl> <dt>**TF \_ LBMENUF en \_ gris**</dt> <dt></dt> </dl>                   | El elemento de menú está deshabilitado.<br/>           |
