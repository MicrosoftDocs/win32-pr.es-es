# <a name="d1167-read-map-option-requires-cpu-readable-bitmap"></a>D1167: la opción de asignación de lectura requiere un mapa de bits legible de la CPU

\_ \_ Las opciones \_ de asignación de D2D1 leídas solo son válidas en un mapa de bits legible de la CPU






 

## <a name="possible-causes"></a>Causas posibles

Direct2D puede generar este mensaje si una aplicación llama a ID2D1Bitmap1:: Map con el \_ atributo de lectura de las opciones de asignación D2D1 \_ \_ , pero el mapa de bits no es legible para la CPU.

## <a name="possible-fixes"></a>Posibles correcciones

Si una aplicación requiere acceso de lectura mediante ID2D1Bitmap1:: Map, cree el mapa de bits con la opción de lectura de mapa de bits D2D1 opciones de la \_ \_ \_ CPU \_ .
