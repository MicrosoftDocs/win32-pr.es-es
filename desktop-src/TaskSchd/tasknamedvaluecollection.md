# <a name="tasknamedvaluecollection-object"></a>Objeto TaskNamedValueCollection

Objeto de scripting que contiene una colección de pares de nombre y valor de objeto [**TaskNamedValuePair**](tasknamedvaluepair.md) .

## <a name="members"></a>Miembros

El objeto **TaskNamedValueCollection** tiene estos tipos de miembros:

-   [Métodos](#methods)
-   [Propiedades](#properties)

### <a name="methods"></a>Métodos

El objeto **TaskNamedValueCollection** tiene estos métodos.



| Método                                            | Descripción                                                        |
|:--------------------------------------------------|:-------------------------------------------------------------------|
| [**Claridad**](tasknamedvaluecollection-clear.md)   | Borra toda la colección de pares nombre-valor.<br/>       |
| [**A**](tasknamedvaluecollection-create.md) | Crea un par de nombre y valor en la colección.<br/>            |
| [**Retirar**](tasknamedvaluecollection-remove.md) | Quita un par nombre-valor seleccionado de la colección.<br/> |



 

### <a name="properties"></a>Propiedades

El objeto **TaskNamedValueCollection** tiene estas propiedades.



| Propiedad                                                   | Descripción                                                        |
|:-----------------------------------------------------------|:-------------------------------------------------------------------|
| [**Contabiliza**](tasknamedvaluecollection-count.md)<br/> | Obtiene el número de pares de nombre y valor de la colección.<br/>  |
| [**Elemento**](tasknamedvaluecollection-item.md)<br/>   | Obtiene el par de nombre y valor especificado de la colección.<br/> |
