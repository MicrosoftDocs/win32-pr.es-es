# <a name="parental-controls"></a>Controles parentales

## <a name="purpose"></a>Propósito

Como parte de los esfuerzos de seguridad de la familia de Microsoft, Windows proporciona funcionalidad de controles parentales para supervisar y limitar la exposición de usuarios de equipos seleccionados a los riesgos en línea y contenido inadecuado.

La tecnología de controles parentales de Windows está diseñada para ayudar a los padres o guardianes diligentes a garantizar el acceso a los materiales adecuados por edad o nivel de madurez para los que están bajo su protección. Proporciona una infraestructura extensible además de funcionalidades integradas.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Se recomienda a los proveedores de software independientes (ISV) de productos que puedan exponer a los usuarios a riesgos de seguridad o que estén trabajando actualmente para proteger a los usuarios de estos riesgos para integrar los controles parentales.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

Esta funcionalidad se introdujo en Windows Vista. La funcionalidad de controles parentales solo está disponible en sistemas operativos de consumidor seleccionados. no está diseñado para su uso con versiones de Windows Server. Esta funcionalidad no se admite en Windows 10.
