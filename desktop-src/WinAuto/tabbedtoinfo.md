# <a name="tabbedtoinfo"></a>TabbedToInfo

## <a name="text"></a>Texto

**Mensaje 1**

Generación de un evento de foco adicional

**Mensaje 2**

Volver al principio

## <a name="type"></a>Tipo

Informativo

## <a name="description"></a>Descripción

**Mensaje 1**

Un solo elemento ha desencadenado varios eventos de foco al usar la navegación con el teclado estándar (Tabulación o Mayús+Tab). Se descartan los eventos falsos.

**Mensaje 2**

El foco se desplaza de nuevo al elemento inicial en el orden de tabulación. La tabulación finalizó en este elemento porque el orden de tabulación aparece circular.
