# <a name="fog-types-direct3d-9"></a>Tipos de niebla (Direct3D 9)

Hay dos maneras de agregar niebla a una escena: con niebla de píxeles o niebla de vértice. En los temas siguientes se muestran las fórmulas que se usan en las ecuaciones de niebla, así como la implementación de la niebla de vértices y píxeles. Una aplicación puede implementar la niebla con un sombreador de vértices y la niebla de píxeles simultáneamente si se desea.

-   [Fórmulas de niebla](fog-formulas.md)
-   [Parámetros de niebla](fog-parameters.md)
-   [Fusión de niebla](fog-blending.md)
-   [Color de niebla](fog-color.md)
-   [Niebla de vértices](vertex-fog.md)
-   [Niebla de píxeles](pixel-fog.md)

La combinación de niebla se controla mediante Estados de representación; no forma parte de la canalización de píxeles programable.
