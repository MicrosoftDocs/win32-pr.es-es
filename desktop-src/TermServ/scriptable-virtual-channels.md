# <a name="scriptable-virtual-channels"></a>Canales virtuales con scripts

Para obtener información sobre el uso de canales virtuales que admiten scripts, consulte [implementación de canales virtuales con scripts mediante conexión web a escritorio remoto](implementing-scriptable-virtual-channels-using-remote-desktop-web-connection.md).

Para obtener información sobre los canales virtuales, consulte [servicios de escritorio remoto de canales virtuales](terminal-services-virtual-channels.md).

**Windows XP:** Windows XP incluye Conexión web a Escritorio remoto, que es una versión actualizada del Cliente avanzado de Terminal Services (TSAC). Conexión web a Escritorio remoto admite [Protocolo de escritorio remoto](remote-desktop-protocol.md) (RDP) 5,1 y es compatible con versiones anteriores de RDP 5,0.
