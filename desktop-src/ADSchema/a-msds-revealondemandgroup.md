# <a name="ms-ds-reveal-ondemand-group-attribute"></a>atributo MS-DS-Reveal-OnDemand-Group

Se usa con RODC para definir qué usuarios, equipos y grupos pueden almacenar en caché sus contraseñas en un RODC.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-DS-Reveal-OnDemand-Group             |
| Nombre para mostrar de LDAP | msDS-RevealOnDemandGroup                |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1928                 |
| System-ID-GUID    | 303d9f4a-1dd6-4b38-8fc5-33afe8c988ad    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
