# <a name="wmparentalratingreason"></a>WM/ParentalRatingReason

El **atributo WM/ParentalRatingReason** contiene los motivos de la clasificación parental asignada al contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMParentalRatingReason

## <a name="data-type"></a>Tipo de datos

**CADENA DE TIPO WMT \_ \_**

## <a name="remarks"></a>Comentarios

Este atributo se usa para la difusión de televisión y otro contenido de vídeo al que se asigna una clasificación parental.
