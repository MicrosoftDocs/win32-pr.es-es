# <a name="about-the-control-point-api"></a>Acerca de la API de punto de control

La tecnología UPnP permite redes dinámicas de dispositivos inteligentes, equipos y otros puntos de control. Las redes dinámicas están disponibles en diversos medios físicos. La implementación de Microsoft de la API de punto de control con tecnología UPnP consta de un conjunto de objetos COM que se usan para escribir aplicaciones y páginas web que encuentran y controlan dispositivos basados en UPnP. Por ejemplo, un VCR compatible con UPnP se puede controlar desde una página web diseñada específicamente para controlarlo. Para ampliar este concepto, los desarrolladores pueden controlar muchos dispositivos desde una ubicación central, remota o del mismo nivel.

Las aplicaciones creadas con la API de punto de control con tecnología UPnP van desde scripts insertados en páginas HTML hasta programas de C++ completos. La API de punto de control con tecnología UPnP admite el sistema de desarrollo de Microsoft Visual Basic, Microsoft Visual Basic Scripting Edition (VBScript) y C++.

En las secciones siguientes se explica, en general, cómo funciona la API de punto de control con tecnología UPnP:

-   [Conceptos básicos del punto de control](control-point-basics.md)
-   [Búsqueda de dispositivos](finding-devices.md)
-   [Descripción de dispositivos](describing-devices.md)
-   [Controlar dispositivos](controlling-devices.md)

Todos los ejemplos de la documentación de la api de punto de control con tecnología UPnP hacen referencia a dispositivos ficticios, tipos de dispositivos y cadenas.
