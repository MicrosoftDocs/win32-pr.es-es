# <a name="windows-installer-development-tools"></a>Windows Herramientas de desarrollo del instalador

Las siguientes herramientas solo están disponibles en los componentes Windows [SDK para Windows instaladores](platform-sdk-components-for-windows-installer-developers.md)de .



| Utilidad                          | Descripción                                                                                                                                                                                                                                      |
|----------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Instmsi.exe](instmsi-exe.md)   | Paquete redistribuible para instalar el instalador de Windows en Windows sistemas operativos anteriores a Windows Me.                                                                                                                               |
| [Msicert.exe](msicert-exe.md)   | Rellena la tabla MsiDigitalSignature y la tabla MsiDigitalCertificate con la información de firma digital que pertenece a archivos de contenedor externos en la tabla Multimedia.                                                                           |
| [Msidb.exe](msidb-exe.md)       | Importa y exporta tablas y secuencias de base de datos, combina bases de datos y aplica transformaciones.                                                                                                                                                       |
| [Msifiler.exe](msifiler-exe.md) | Rellena la tabla File con versiones, idiomas y tamaños de archivo en función de un directorio de origen. También puede actualizar la tabla MsiFileHash con hashes de archivo.                                                                                      |
| [Msiinfo.exe](msiinfo-exe.md)   | Edita o muestra el [flujo de información de resumen.](summary-information-stream.md)                                                                                                                                                                  |
| [Msimerg.exe](msimerg-exe.md)   | Combina una base de datos en otra.                                                                                                                                                                                                                |
| [Msimsp.exe](msimsp-exe.md)     | Herramienta de creación de revisiones. El método recomendado para generar un paquete de revisión es usar una herramienta de creación de revisiones como Msimsp.exe con PATCHWIZ.DLL.                                                                                                 |
| [Msistuff.exe](msistuff-exe.md) | Muestra o configura los recursos en el ejecutable de arranque Setup.exe programa previo.                                                                                                                                                                      |
| [Msitool.mak](msitool-mak.md)   | Archivo Make que se puede usar para realizar herramientas y acciones personalizadas.                                                                                                                                                                                      |
| [Msitran.exe](msitran-exe.md)   | Genera una transformación o aplica un archivo de transformación a una base de datos.                                                                                                                                                                                 |
| [Msival2.exe](msival2-exe.md)   | Ejecuta uno o un conjunto de [evaluadores de coherencia interno: TIC.](internal-consistency-evaluators-ices.md)                                                                                                                                       |
| [Msizap.exe](msizap-exe.md)     | Quita Windows del instalador de un producto o de todos los productos instalados en una máquina.                                                                                                                                                      |
| [Orca.exe](orca-exe.md)         | Editor de bases de datos. Crea y edita .msi archivos y módulos de combinación.                                                                                                                                                                                 |
| [PATCHWIZ.DLL](patchwiz-dll.md) | Genera un paquete de Windows de revisión del [instalador](patch-packages.md) a partir de un archivo de propiedades de creación de revisiones (archivo .csv). El método recomendado para generar un paquete de revisión es usar una herramienta de creación de revisiones como Msimsp.exe con PATCHWIZ.DLL. |
| [Wilogutl.exe](wilogutl-exe.md) | Ayuda al análisis de archivos de registro de una instalación Windows Installer y muestra soluciones sugeridas a los errores.                                                                                                                              |
