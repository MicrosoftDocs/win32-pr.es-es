# <a name="elements-of-the-sdo-api"></a>Elementos de la API de SDO

La API de SDO está diseñada en torno a un modelo de objetos que consta de objetos y colecciones de objetos. En el modelo de objetos SDO, los objetos y colecciones se organizan en una jerarquía, es decir, los objetos y colecciones contienen otros objetos y colecciones.

Los objetos se manipulan recuperando y estableciendo propiedades de objeto.

En las secciones siguientes se describe cómo la API de SDO usa objetos, propiedades de objeto y colecciones:

-   [Objetos y propiedades](/windows/desktop/Nps/sdo-objects-and-properties)
-   [Jerarquía del modelo de objetos](/windows/desktop/Nps/sdo-object-model-hierarchy)
-   [Atributos](/windows/desktop/Nps/sdo-sdo-supported-attributes)
