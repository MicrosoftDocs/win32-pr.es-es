# <a name="tv-audio-filter"></a>Filtro de audio de TV

El filtro de audio de TV proporciona el control de la descodificación de audio de televisión, la selección de estéreo o monoaural y la selección del programa de audio secundario (SAP).



|                                          |                                                                                                                                |
|------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------|
| Interfaces de filtro                        | [**IAMTVAudio**](/windows/desktop/api/Strmif/nn-strmif-iamtvaudio), **IPersistPropertyBag**, **ISpecifyPropertyPages**, [**IKsPropertySet**](ikspropertyset.md) |
| Tipos de medios de anclaje de entrada                    | MEDIATYPE \_ AnalogAudio                                                                                                         |
| Interfaces de PIN de entrada                     | **IKsPropertySet**                                                                                                             |
| Tipos de medios de anclaje de salida                   | MEDIATYPE \_ AnalogAudio                                                                                                         |
| Interfaces de clavija de salida                    | **IKsPropertySet**                                                                                                             |
| Identificador CLSID                             | CLSID \_ TVAudioFilter                                                                                                           |
| CLSID de la página de propiedades                      | \_TVAUDIO KSCATEGORY \_ AM                                                                                                        |
| [Fundament](merit.md)                       | MÉRITO \_ \_ no se \_ usa                                                                                                            |
| [Categoría de filtro](filter-categories.md) | CLSID \_ AudioInputDeviceCategory                                                                                                |
