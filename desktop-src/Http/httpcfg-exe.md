# <a name="httpcfgexe"></a>HttpCfg.exe

Esta herramienta le ayuda a configurar la API del servidor HTTP ajustando las API [de configuración HTTP](configuration-apis.md) para los parámetros siguientes:

-   Lista de escucha de IP
-   Certificados SSL
-   Reservas de espacios de nombres

HttpCfg.exe se suministra con el sistema operativo a partir de Windows Server 2003 como un archivo ejecutable binario, junto con un archivo de ayuda que describe cómo usarlo. Se puede instalar desde el disco de distribución del sistema operativo como una herramienta de soporte técnico.

HttpCfg.exe también está disponible como descarga; consulte [Windows herramientas de soporte técnico de XP SP2](https://www.microsoft.com/downloads/details.aspx?familyid=49AE8576-9BB9-4126-9761-BA8011FABF38).

**Windows Vista y Windows Server 2008:** HttpCfg.exe está obsoleto y se reemplaza por una funcionalidad similar en netsh.exe.
