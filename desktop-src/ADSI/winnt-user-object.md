# <a name="winnt-user-object"></a>Objeto de usuario WinNT

El objeto Usuario winNT representa una cuenta de usuario en un Windows nt 4.0. El objeto presenta características especiales. En una instancia, no admite todos los métodos de propiedad de [**la interfaz IADsUser.**](/windows/desktop/api/Iads/nn-iads-iadsuser) En una segunda instancia, admite algunas propiedades personalizadas a las que solo se puede acceder con el método [**IADs.Get**](/windows/desktop/api/Iads/nf-iads-iads-get) o [**IADs.Put.**](/windows/desktop/api/Iads/nf-iads-iads-put)

Para más información sobre los objetos de usuario de WinNT, consulte:

-   [Propiedades IADsUser no admitidas](unsupported-iadsuser-properties.md)
-   [Propiedades de usuario personalizado de WinNT](winnt-custom-user-properties.md)
-   [Ejemplos de administración de cuentas de usuario de WinNT](winnt-user-account-management-examples.md)
