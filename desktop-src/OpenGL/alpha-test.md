# <a name="alpha-test"></a>Prueba alfa

La prueba alfa (realizada solo en el modo RGBA) descarta un fragmento, dependiendo del resultado de una comparación entre el valor alfa del fragmento y un valor de referencia constante. Especifique la función de comparación y el valor de referencia con [**glAlphaFunc**](glalphafunc.md).
