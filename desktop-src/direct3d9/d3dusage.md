# <a name="d3dusage"></a>D3DUSAGE

Opciones de uso que identifican cómo se van a usar los recursos.

En la tabla siguiente se resumen las opciones de uso disponibles.



| Constante                                                                                                                                                                                                                                          | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="D3DUSAGE_AUTOGENMIPMAP"></span><span id="d3dusage_autogenmipmap"></span><dl> <dt>**D3DUSAGE \_ AUTOGENMIPMAP**</dt> </dl>                                                          | El recurso generará automáticamente mapas MIP. Vea [generación automática de mapas MIP (Direct3D 9)](automatic-generation-of-mipmaps.md). La generación automática de mapas MIP no se admite para las texturas de volumen y las superficies o texturas de la galería de símbolos de profundidad. Este uso no es válido para un recurso de la memoria del sistema (**D3DPOOL \_ SYSTEMMEM**).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="D3DUSAGE_DEPTHSTENCIL"></span><span id="d3dusage_depthstencil"></span><dl> <dt>**D3DUSAGE \_ DEPTHSTENCIL**</dt> </dl>                                                             | El recurso será un búfer de estarcido de profundidad. **D3DUSAGE \_ DEPTHSTENCIL** solo se puede usar con **D3DPOOL \_ default**.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="D3DUSAGE_DMAP"></span><span id="d3dusage_dmap"></span><dl> <dt>**D3DUSAGE \_ DMAP**</dt> </dl>                                                                                     | El recurso será un mapa de desplazamiento.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="D3DUSAGE_DONOTCLIP"></span><span id="d3dusage_donotclip"></span><dl> <dt>**D3DUSAGE \_ DONOTCLIP**</dt> </dl>                                                                      | Se establece para indicar que el contenido del búfer de vértices nunca requerirá un recorte. Al representar con búferes que tienen esta marca establecida, el estado de representación de **\_ recorte de D3DRS** debe establecerse en false.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <span id="D3DUSAGE_DYNAMIC"></span><span id="d3dusage_dynamic"></span><dl> <dt>**D3DUSAGE \_ dinámico**</dt> </dl>                                                                            | Se establece para indicar que el búfer de vértices requiere el uso de memoria dinámica. Esto resulta útil para los controladores porque les permite decidir dónde colocar el búfer. En general, los búferes de vértices estáticos se colocan en la memoria de vídeo y los búferes de vértices dinámicos se colocan en la memoria AGP. Tenga en cuenta que no hay ningún uso estático independiente. Si no especifica **D3DUSAGE \_ Dynamic**, el búfer de vértices se convierte en estático. **D3DUSAGE \_ DYNAMIC** se aplica estrictamente a través de las marcas de bloqueo **D3DLOCK \_ discard** y **D3DLOCK \_ NOOVERWRITE** . Como resultado, **D3DLOCK \_ discard** y **D3DLOCK \_ NOOVERWRITE** solo son válidos en los búferes de vértices creados con **D3DUSAGE \_ Dynamic**. No son marcas válidas en los búferes de vértices estáticos. Para obtener más información, vea [administrar recursos (Direct3D 9)](managing-resources.md). <br/> Para obtener más información sobre el uso de búferes de vértices dinámicos, vea [optimizaciones de rendimiento (Direct3D 9)](performance-optimizations.md).<br/> **D3DUSAGE \_** Los **\_ administrados** de forma dinámica y D3DPOOL son incompatibles y no se deben usar juntos. Vea [**D3DPOOL**](./d3dpool.md).<br/> Las texturas pueden especificar **D3DUSAGE \_ dinámico**. Sin embargo, las texturas administradas no pueden usar **D3DUSAGE \_ Dynamic**. Para obtener más información sobre las texturas dinámicas, vea [usar texturas dinámicas](performance-optimizations.md). <br/> |
| <span id="D3DUSAGE_NONSECURE"></span><span id="d3dusage_nonsecure"></span><dl> <dt>**D3DUSAGE no \_ seguro**</dt> </dl>                                                                      | Permite que una superficie compartida creada por una aplicación segura se abra con una aplicación no segura que tenga el identificador compartido. <br/> **Diferencias entre Direct3D 9 y Direct3D 9Ex:** Esta marca solo está disponible en Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="D3DUSAGE_NPATCHES"></span><span id="d3dusage_npatches"></span><dl> <dt>**D3DUSAGE \_ NPATCHES**</dt> </dl>                                                                         | Se establece para indicar que el búfer de vértices se va a usar para dibujar N-patches.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <span id="D3DUSAGE_POINTS"></span><span id="d3dusage_points"></span><dl> <dt>**\_Puntos D3DUSAGE**</dt> </dl>                                                                               | Se establece para indicar que el búfer de vértice o de índice se usará para los sprites de punto de dibujo. El búfer se cargará en la memoria del sistema si es necesario el procesamiento de vértices de software para emular los sprites de punto.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="D3DUSAGE_RENDERTARGET"></span><span id="d3dusage_rendertarget"></span><dl> <dt>**D3DUSAGE \_ RENDERTARGET**</dt> </dl>                                                             | El recurso será un destino de representación. **D3DUSAGE \_ RENDERTARGET** solo se puede usar con **D3DPOOL \_ default**.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="D3DUSAGE_RTPATCHES"></span><span id="d3dusage_rtpatches"></span><dl> <dt>**D3DUSAGE \_ RTPATCHES**</dt> </dl>                                                                      | Se establece para indicar que el búfer de vértices se va a utilizar para dibujar primitivas de orden superior.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="D3DUSAGE_SOFTWAREPROCESSING"></span><span id="d3dusage_softwareprocessing"></span><dl> <dt>**D3DUSAGE \_ SOFTWAREPROCESSING**</dt> </dl>                                           | Si se usa esta marca, el procesamiento de vértices se realiza en el software. Si no se usa esta marca, el procesamiento de vértices se realiza en el hardware.<br/> La marca **D3DUSAGE \_ SOFTWAREPROCESSING** se puede establecer cuando el procesamiento de vértices de modo mixto o software (**D3DCREATE \_ mixto \_ VERTEXPROCESSING**  /  **D3DCREATE \_ software \_ VERTEXPROCESSING**) está habilitado para ese dispositivo. **D3DUSAGE \_ SOFTWAREPROCESSING** debe establecerse para que los búferes se utilicen con el procesamiento de vértices de software en modo mixto, pero no se debe establecer para el mejor rendimiento posible al usar el procesamiento de índices de hardware en modo mixto (**D3DCREATE de \_ hardware \_ VERTEXPROCESSING**). Sin embargo, el establecimiento de **D3DUSAGE \_ SOFTWAREPROCESSING** es la única opción cuando se usa un solo búfer con el procesamiento de vértices de hardware y software. **D3DUSAGE \_ SOFTWAREPROCESSING** se permite para dispositivos mixtos y de software.<br/> **D3DUSAGE \_ SOFTWAREPROCESSING** se usa con [**CheckDeviceFormat**](/windows/win32/api/d3d9/nf-d3d9-idirect3d9-checkdeviceformat) para averiguar si se puede usar un formato de textura determinado como textura de vértice durante el procesamiento de vértices de software. Si es posible, la textura se debe crear en **el \_ principio de D3DPOOL**.<br/>                                                                                                                                                                 |
| <span id="D3DUSAGE_TEXTAPI"></span><span id="d3dusage_textapi"></span><dl> <dt>**D3DUSAGE \_ TEXTAPI**</dt> </dl>                                                                            | Esta marca de uso debe especificarse para los búferes de vértices y las superficies de origen, que se usan en las llamadas a [**ComposeRects**](/windows/desktop/api/d3d9/nf-d3d9-idirect3ddevice9ex-composerects). No se pueden usar T no usados con esta marca de uso para el filtrado de texturas. Los búferes de vértices, creados con esta marca de uso, no se pueden usar como orígenes de flujo de entrada. <br/> **Diferencias entre Direct3D 9 y Direct3D 9Ex:** Esta marca solo está disponible en Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="D3DUSAGE_WRITEONLY"></span><span id="d3dusage_writeonly"></span><dl> <dt>**D3DUSAGE \_ WRITEONLY**</dt> </dl>                                                                      | Informa al sistema de que la aplicación escribe solo en el búfer de vértices. El uso de esta marca permite al controlador elegir la mejor ubicación de memoria para operaciones de escritura y representación eficientes. Se producirá un error al intentar leer desde un búfer de vértice creado con esta funcionalidad. Los búferes creados con **D3DPOOL \_ predeterminados** que no especifican **D3DUSAGE \_ WRITEONLY** pueden experimentar una penalización de rendimiento grave. **D3DUSAGE \_ WRITEONLY** solo afecta al rendimiento de los búferes **\_ predeterminados de D3DPOOL** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="D3DUSAGE_RESTRICTED_CONTENT"></span><span id="d3dusage_restricted_content"></span><dl> <dt>**\_Contenido restringido de D3DUSAGE \_**</dt> </dl>                                          | Al establecer esta marca, se indica que el recurso puede contener contenido protegido.<br/> **Diferencias entre Direct3D 9 y Direct3D 9Ex:** Esta marca solo está disponible en Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <span id="D3DUSAGE_RESTRICT_SHARED_RESOURCE"></span><span id="d3dusage_restrict_shared_resource"></span><dl> <dt>**D3DUSAGE \_ restringir \_ \_ recurso compartido**</dt> </dl>                       | La configuración de esta marca indica que se debe restringir el acceso al recurso compartido. <br/> **Diferencias entre Direct3D 9 y Direct3D 9Ex:** Esta marca solo está disponible en Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |

El autor de la llamada debe crear un canal autenticado con el controlador. A continuación, el controlador debe permitir el acceso a los procesos que intentan abrir ese recurso compartido.<br/> **Diferencias entre Direct3D 9 y Direct3D 9Ex:** Esta marca solo está disponible en Direct3D 9Ex.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |

### <a name="usage-and-resource-combinations"></a>Combinaciones de recursos y uso

Los usos se especifican cuando se crea un recurso o se especifican con [**CheckDeviceType**](/windows/win32/api/d3d9/nf-d3d9-idirect3d9-checkdevicetype) para probar la capacidad de un recurso existente. En la tabla siguiente se identifican los usos que se pueden aplicar a los tipos de recursos.



| Uso                            | Creación de búfer de vértices | Creación de búfer de índice | Crear textura | Creación de textura de cubo | Creación de textura de volumen | Creación de Surface | Comprobar el formato del dispositivo |
|----------------------------------|----------------------|---------------------|----------------|---------------------|-----------------------|----------------|---------------------|
| **D3DUSAGE \_ AUTOGENMIPMAP**      |                      |                     | x              | x                   |                       |                | x                   |
| **D3DUSAGE \_ DEPTHSTENCIL**       |                      |                     | x              | x                   |                       | x              | x                   |
| **D3DUSAGE \_ DMAP**               |                      |                     | x              |                     |                       |                | x                   |

| **D3DUSAGE no \_ seguro**          | x                    | x                   | x              | x                   | x                     | x              | x                   |

Para obtener más información sobre los tipos de grupo y sus restricciones con determinados usos, vea [**D3DPOOL**](./d3dpool.md).
