# <a name="properties-of-iocspcaconfigurationcollection"></a>Propiedades de IOCSPCAConfigurationCollection

La interfaz [**IOCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) define las siguientes propiedades.



| Propiedad                                                               | Descripción                                                                                                                                                                              |
|------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get__newenum)    | Obtiene un enumerador para el conjunto de configuración.                                                                                                                                            |
| [**Contabiliza**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_count)           | Obtiene el número de configuraciones de [*entidad de certificación*](../secgloss/c-gly.md) (CA) del conjunto de configuración. |
| [**Elemento**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_item)             | Obtiene una configuración de CA identificada por el índice en el conjunto de configuración.                                                                                                                    |
| [**ItemByName**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-get_itembyname) | Obtiene una configuración de CA identificada por el nombre en el conjunto de configuración.                                                                                                                     |
