# <a name="texture2darraygathergreen-methods"></a>Texture2DArray:: GatherGreen (métodos)

Devuelve los componentes verdes de los cuatro valores de textura de un [**Texture2DArray**](sm5-object-texture2darray.md)que se utilizarían en una operación de filtrado bilineal.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                                          | Descripción                                                                                                                                       |
|:----------------------------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherGreen (S, Float, int)**](sm5-object-texture2darray-gathergreen.md)                                        | Devuelve los componentes verdes de los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal.<br/>                                 |
| [**GatherGreen (S, Float, int, uint)**](t2darray-gathergreen-s-float-int-uint-.md)                                  | Devuelve los componentes verdes de los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal, junto con el estado de asignación de mosaicos.<br/> |
| [**GatherGreen (S, Float, INT2, INT2, INT2, INT2)**](t2darray-gathergreen-s-float-int2-int2-int2-int2-.md)            | Devuelve los componentes verdes de los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal.<br/>                                 |
| [**GatherGreen (S, Float, INT2, INT2, INT2, INT2, uint)**](t2darray-gathergreen-s-float-int2-int2-int2-int2-uint-.md)  | Devuelve los componentes verdes de los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal, junto con el estado de asignación de mosaicos.<br/> |
