# <a name="frs-root-path-attribute"></a>Atributo FRS-root-path

Ruta de acceso a la raíz del árbol del sistema de archivos replicados.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | FRS-raíz-ruta de acceso                               |
| Nombre para mostrar de LDAP | fRSRootPath                                 |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.487                      |
| System-ID-GUID    | 1be8f174-a9ff-11d0-afe2-00c04fd930c9        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
