# <a name="dvdrect-object"></a>DVDRect (objeto)

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

> [!Note]  
> Esta API está en desuso. Para obtener información sobre la reproducción y navegación de DVD DirectShow, vea [Aplicaciones de DVD.](dvd-applications.md)

 

Las propiedades del objeto permiten a una aplicación de scripting especificar y recibir coordenadas de rectángulo en `DVDRect` [métodos MSWebDVD](mswebdvd-object.md) como [**GetButtonRect,**](getbuttonrect-method.md) [**GetClipVideoRect,**](getclipvideorect-method.md) [**GetVideoSize**](getvideosize-method.md) y [**SetClipVideoRect.**](setclipvideorect-method.md)



| Propiedades                        | Descripción                                                            |
|-----------------------------------|------------------------------------------------------------------------|
| [**X**](x-property.md)           | Establece o recupera la posición x de la esquina superior izquierda del rectángulo. |
| [**y**](y-property.md)           | Establece o recupera la posición y de la esquina superior izquierda del rectángulo. |
| [**Ancho**](width-property.md)   | Establece o recupera el ancho del rectángulo, en píxeles.               |
| [**Alto**](height-property.md) | Establece o recupera el alto del rectángulo, en píxeles.              |
