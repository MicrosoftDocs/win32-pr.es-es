# <a name="featureinfotitle-property"></a>FeatureInfo. title (propiedad)

La propiedad **title** es una propiedad de solo lectura que devuelve el título de la característica en la columna title de la [tabla de características](feature-table.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = FeatureInfo.Title
```



## <a name="property-value"></a>Valor de propiedad
