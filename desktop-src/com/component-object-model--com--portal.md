# <a name="component-object-model-com"></a>Modelo de objetos componentes (COM)

## <a name="purpose"></a>Propósito

COM es un sistema orientado a objetos, distribuido e independiente de la plataforma para crear componentes de software binarios que pueden interactuar. COM es la tecnología fundamental para las tecnologías OLE (documentos compuestos) y ActiveX de Microsoft (componentes habilitados para Internet).

## <a name="where-applicable"></a>Donde sea aplicable

Los objetos COM se pueden crear con diversos lenguajes de programación. Los lenguajes orientados a objetos, como C++, proporcionan mecanismos de programación que simplifican la implementación de objetos COM. Estos objetos pueden estar dentro de un único proceso, en otros procesos, incluso en equipos remotos.
