# <a name="configuration-constants"></a>Constantes de configuración

En esta sección se proporcionan las especificaciones de referencia [**para las constantes SystemParametersInfo**](/windows/win32/api/winuser/nf-winuser-systemparametersinfoa) relacionadas con los [atributos del sistema](configuration.md) de configuración.

## <a name="in-this-section"></a>En esta sección



| Tema                                                             | Descripción                                                                                                                                                            |
|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Visualización de contactos**](contact-visualization.md)<br/> | Las aplicaciones o marcos de interfaz de usuario usan las siguientes constantes para identificar cómo se procesan los comentarios de la interfaz de usuario cuando se detecta un contacto de entrada.<br/>               |
| [**Visualización de gestos**](gesture-visualization.md)<br/> | Las aplicaciones o marcos de interfaz de usuario usan las siguientes constantes para identificar cómo se procesan los comentarios de la interfaz de usuario cuando se detecta uno de los gestos enumerados.<br/>     |
| [**Visualización de lápiz**](pen-visualization.md)<br/>         | Las aplicaciones o marcos de interfaz de usuario usan las siguientes constantes para identificar cómo se procesan los comentarios de la interfaz de usuario cuando se detecta uno de los gestos de lápiz enumerados.<br/> |
