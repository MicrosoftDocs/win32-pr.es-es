# <a name="dropdownplaylist"></a>LISTA DESPLEGABLEPLAYLIST

Se trata de un elemento **PLAYLIST** predefinido con los siguientes valores predeterminados.

``` syntax
playlistItemsVisible="false"
```

## <a name="remarks"></a>Comentarios

Esto creará una lista de reproducción desplegable **simple** sin elementos mostrados. Todas las propiedades de esta **lista** de reproducción se pueden invalidar si se especifican explícitamente.

|--------------------|----------------------------------------------|
