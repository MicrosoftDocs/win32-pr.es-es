# <a name="previous-versions-of-windows-media-metafiles-deprecated"></a>Versiones anteriores de Windows metarchivos multimedia (en desuso)

En este tema se documenta una característica que ya no se usa en la versión actual de Windows metarchivos multimedia.

En esta sección se documenta una parte de la versión 2 de Windows metarchivos multimedia. La versión 2 no es la versión actual de los metarchivos multimedia de Windows, pero hay algunos casos en los que Servicios de Windows Media proporciona a los clientes metarchivos que están en el formato de la versión 2. Windows Los metarchivos multimedia también se conocen como archivos advanced stream redirector (ASX).

En esta sección se documenta los siguientes elementos.



| Elemento                                                   | Descripción                                                    |
|--------------------------------------------------------|----------------------------------------------------------------|
| [elemento reference](reference-element--deprecated.md) | Contiene una lista de direcciones URL de archivos multimedia.                        |
| [Parámetro MSWMExt](mswmext-parameter--deprecated.md) | Indica a un programa cliente el formato de un archivo al que se hace referencia. |
