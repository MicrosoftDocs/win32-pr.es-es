# <a name="resource-reference-direct3d-11-graphics"></a>Referencia de recursos (gráficos de Direct3D 11)

La API de Direct3D define varios elementos de API para ayudarle a crear y administrar recursos.

| Tema                                                                              | Descripción                                                                                                                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Interfaces de recursos](d3d11-graphics-reference-resource-interfaces.md)<br/> | Hay una serie de interfaces para los dos tipos básicos de recursos: búferes y texturas. También hay interfaces para vistas de recursos. <br/> Una aplicación usa una vista para enlazar un recurso a una fase de canalización. La vista define cómo se puede acceder al recurso durante la representación. En esta sección se describen las interfaces de vista. <br/> |
