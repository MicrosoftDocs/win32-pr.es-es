# <a name="corecpp"></a>Núcleo. Cpp

En el componente de proveedor de ejemplo, el ejemplo de código, que se usa para inicializar las propiedades básicas de un objeto genérico, está en core.cpp. Los métodos admitidos se enumeran en la tabla siguiente.



| Método                                   | Descripción                                                                                              |
|------------------------------------------|----------------------------------------------------------------------------------------------------------|
| **CCoreADsObject::InitializeCoreObject** | Inicialice las propiedades básicas de los [**IAD para**](/windows/desktop/api/Iads/nn-iads-iads) este objeto, una comodidad para la creación de objetos. |
| **CCoreADsObject::CCoreADsObject**       | Creador estándar.                                                                                        |
| **CCoreADsObject::~CCoreADsObject**      | Destructor estándar.                                                                                     |
