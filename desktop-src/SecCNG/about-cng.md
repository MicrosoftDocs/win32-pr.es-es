# <a name="about-cng"></a>Acerca de CNG

Cryptography API: Next Generation (CNG) es el sustituto a largo plazo de [*CryptoAPI*](/windows/desktop/SecGloss/c-gly). CNG está diseñado para ser extensible en muchos niveles y en un comportamiento independiente de la criptografía.

## <a name="features"></a>Características

Para obtener información sobre las características de CNG, vea [características de CNG](cng-features.md).

## <a name="cryptographic-primitives"></a>Primitivos criptográficos

Para obtener más información sobre las primitivas criptográficas en CNG, consulte [primitivas criptográficas](cryptographic-primitives.md).

## <a name="key-storage-and-retrieval"></a>Almacenamiento y recuperación de claves

Para obtener más información sobre el almacenamiento y la recuperación de claves en CNG, consulte [almacenamiento y recuperación de claves](key-storage-and-retrieval.md).

## <a name="key-import-and-export"></a>Importación y exportación de claves

Para obtener más información sobre la importación y exportación de claves, consulte [importación y exportación de claves](key-import-and-export.md).

## <a name="data-protection-api-next-generation"></a>API de protección de datos: próxima generación

CNG proporciona un conjunto de API que se pueden usar para cifrar y descifrar fácilmente el contenido de las entidades de seguridad en varios equipos. Para obtener más información, vea [DPAPI de CNG](cng-dpapi.md).
