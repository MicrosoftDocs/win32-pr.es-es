# <a name="about-the-stringcollection-object"></a>Acerca del objeto StringCollection

El objeto **StringCollection** rige las colecciones de cadenas, que se utilizan para proporcionar funciones de control de cadenas para el objeto **MediaCollection** . El objeto **StringCollection** se puede recuperar mediante el método **GetAttributeStringCollection** del objeto **MediaCollection** . Puede utilizar esto para trabajar con colecciones de cadenas que representan los valores de varios atributos de elemento multimedia.
