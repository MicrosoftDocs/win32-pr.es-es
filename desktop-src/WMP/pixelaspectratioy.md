# <a name="pixelaspectratioy"></a>PixelAspectRatioY

El atributo **PixelAspectRatioY** es la parte vertical de la relación de aspecto de un vídeo.

## <a name="applies-to"></a>Se aplica a

-   [Elementos de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Observaciones

Para un vídeo que tiene una relación de aspecto de 16:9, **PixelAspectRatioY** sería igual a "9".
