# <a name="am_rate_simpleratechange-property"></a>\_Propiedad SimpleRateChange de tasa AM \_

Esta propiedad se utiliza para enviar los cambios de velocidad al descodificador. El tipo de datos de esta propiedad es una estructura **AM \_ SimpleRateChange** , que proporciona la nueva velocidad de reproducción y el tiempo de presentación cuando la nueva tasa surte efecto.



|                   |                                                     |
|-------------------|-----------------------------------------------------|
| GUID del conjunto de propiedades | \_TSRateChange KSPROPSETID \_ AM                       |
| Id. de propiedad       | Tasa de AM \_ \_ SimpleRateChange                          |
| Tipo de datos         | [**\_SIMPLERATECHANGE AM**](/previous-versions/windows/desktop/api/Dvdmedia/ns-dvdmedia-am_simpleratechange) |
