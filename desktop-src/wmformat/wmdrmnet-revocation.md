# <a name="wmdrmnet_revocation"></a>Revocación de WMDRMNET \_

La propiedad de **\_ revocación de WMDRMNET** recupera la lista de revocación para Windows Media DRM 10 para dispositivos de red.

## <a name="global-constant"></a>Constante global

g \_ \_ revocación de wszWMDRMNET

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

La lista de revocación contiene los certificados de los dispositivos que se han revocado.
