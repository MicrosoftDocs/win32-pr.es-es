# <a name="regions-windows-gdi"></a>Regiones (Windows GDI)

Una *región* es un rectángulo, polígono o elipse (o una combinación de dos o más de estas formas) que se pueden rellenar, pintar, invertir, enmarcar y usar para realizar pruebas de posición (pruebas de la ubicación del cursor).

-   [Utilizar regiones](using-regions.md)
