# <a name="multimedia-input"></a>Entrada multimedia

Windows proporciona funciones que permiten a una aplicación recibir datos multimedia. Las funciones de entrada de multimedia se introdujeron en Windows de 16 bits. Muchas de sus características se han sustituido por características de DirectX. Para obtener más información, consulte la documentación de DirectX.

Los temas siguientes se describen en la documentación de entrada multimedia.

-   [E/s de archivos multimedia](multimedia-file-i-o.md)
-   [Joysticks](joysticks.md)
-   [Temporizadores multimedia](multimedia-timers.md)
