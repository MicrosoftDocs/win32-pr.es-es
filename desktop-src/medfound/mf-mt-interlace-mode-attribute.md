# <a name="mf_mt_interlace_mode-attribute"></a>Atributo \_ MF MT \_ INTERLACE \_ MODE

Describe cómo se entrelazan los fotogramas de un tipo de medio de vídeo.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Comentarios

El valor de este atributo es un miembro de la [**enumeración MFVideoInterlaceMode.**](/windows/desktop/api/mfobjects/ne-mfobjects-mfvideointerlacemode)

La constante GUID para este atributo se exporta desde mfuuid.lib.
