# <a name="windows-server"></a>Windows Server

Windows Server es un grupo de sistemas operativos diseñado por Microsoft que admite la administración de nivel empresarial, el almacenamiento de datos, las aplicaciones y las comunicaciones. Las versiones anteriores de Windows Server se centraban en la estabilidad, seguridad, redes y diversas mejoras en el sistema de archivos. Otras mejoras también han incluido mejoras en las tecnologías de implementación, así como una mayor compatibilidad de hardware. Microsoft también ha creado SKU especializadas de Windows Server que se centran en los mercados domésticos y de pequeñas empresas. Windows Server 2012 R2 es la versión más reciente de Windows Server y se centra en la informática en la nube.
