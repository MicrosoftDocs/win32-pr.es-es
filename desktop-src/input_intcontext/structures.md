# <a name="interaction-context-structures"></a>Estructuras de contexto de interacción

Los temas de esta sección proporcionan las especificaciones de referencia para las [estructuras de contexto de](interaction-context-portal.md) interacción.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|---|---|
| [**PARÁMETRO \_ CROSS SLIDE \_**](/windows/win32/api/interactioncontext/ns-interactioncontext-cross_slide_parameter)<br/>                           | Define el umbral de deslizamiento cruzado y su umbral de distancia.<br/>                                                                                                    |
| [**DIAPOSITIVA CRUZADA \_ DE ARGUMENTOS \_ DE \_ INTERACCIÓN**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_cross_slide)<br/>  | Define el estado de la interacción entre diapositivas.<br/>                                                                                                                |
| [**MANIPULACIÓN \_ DE ARGUMENTOS DE \_ INTERACCIÓN**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_manipulation)<br/> | Define el estado de una manipulación.<br/>                                                                                                                             |
| [**TAP \_ DE ARGUMENTOS DE \_ INTERACCIÓN**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_arguments_tap)<br/>                   | Define el estado de la interacción de pulsación.<br/>                                                                                                                        |
| [**CONFIGURACIÓN DEL \_ CONTEXTO DE \_ INTERACCIÓN**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_context_configuration)<br/>   | Define la configuración de un [objeto de contexto de](interaction-context-portal.md) interacción que habilita, deshabilita o modifica el comportamiento de una interacción.<br/> |
| [**SALIDA DEL \_ CONTEXTO DE \_ INTERACCIÓN**](/windows/win32/api/interactioncontext/ns-interactioncontext-interaction_context_output)<br/>                 | Define la salida del objeto [Contexto de](interaction-context-portal.md) interacción.<br/>                                                                          |
| [**TRANSFORMACIÓN DE \_ MANIPULACIÓN**](/windows/win32/api/interactioncontext/ns-interactioncontext-manipulation_transform)<br/>                          | Define los datos de transformación para una manipulación.<br/>                                                                                                              |
| [**VELOCIDAD DE \_ MANIPULACIÓN**](/windows/win32/api/interactioncontext/ns-interactioncontext-manipulation_velocity)<br/>                            | Define los datos de velocidad de una manipulación.<br/>                                                                                                                     |
