# <a name="ms-ts-primary-desktop-attribute"></a>atributo MS-TS-Primary-Desktop

Este atributo representa el vínculo hacia delante al escritorio principal de un usuario.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-TS-principal-escritorio                   |
| Nombre para mostrar de LDAP | msTSPrimaryDesktop                      |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.2073                 |
| System-ID-GUID    | 29259694-09e4-4237-9f72-9306ebe63ab2    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
