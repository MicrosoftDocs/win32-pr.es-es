# <a name="advanced-systems-format-asf-files"></a>Archivos de formato de sistemas avanzados (ASF)

Todos los objetos del SDK Windows Media Format funcionan con archivos que se ajustan a la estructura de archivos del formato de sistemas avanzados (ASF). Estos archivos se conocen como archivos ASF en esta documentación. Para obtener más información sobre la estructura de los archivos ASF, vea [Información general sobre el formato ASF](overview-of-the-asf-format.md).

A veces, *el Windows archivo multimedia* se usa para describir los archivos creados mediante este SDK u otras tecnologías Windows Media. No es preciso llamar a un archivo ASF como un archivo Windows Media a menos que contenga solo contenido codificado mediante códecs Windows Media. Para obtener más información, consulte las instrucciones para el uso de las extensiones de archivo .wma y .wmv en la sección Instrucciones de extensión de [nombre de archivo.](file-name-extension-guidelines.md)
