# <a name="getting-started-tablet-pc"></a>Introducción (Tablet PC)

Tablet PC es un equipo personal totalmente funcional con tecnología de Microsoft Windows y diseñado para aplicaciones con lápiz habilitado, con lápiz habilitado y habilitado para voz. La combinación de software y hardware en Tablet PC permite que estos métodos de interacción con el usuario creen una experiencia informática enriquecida, interactiva y productiva para los usuarios.

La plataforma de Tablet PC permite la entrada y salida de datos de escritura a mano y voz en un Tablet PC, así como el intercambio de estos datos con otros equipos. Los componentes de la plataforma de Tablet PC instalados por el SDK de Windows 7 le permiten crear aplicaciones habilitadas para lápiz y con lápiz para Tablet PC.
