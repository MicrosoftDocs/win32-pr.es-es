# <a name="duplicating-a-pdu"></a>Duplicación de una PDU

La [**función SnmpDuplicatePdu**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmpduplicatepdu) duplica una PDU y asigna la memoria necesaria. Para liberar los recursos asignados por **SnmpDuplicatePdu** para una nueva PDU, una aplicación WinSNMP debe llamar a la [**función SnmpFreePdu.**](/windows/desktop/api/Winsnmp/nf-winsnmp-snmpfreepdu)
