# <a name="dns-zones"></a>Zonas DNS

Una zona DNS es un conjunto de archivos o registros (más concretamente, una base de datos de entradas de registro de recursos) que corresponde a parte del espacio de nombres jerárquico DNS. Las zonas DNS se usan para delinear qué servidores DNS son responsables (autoritativo) de resolver consultas de resolución de nombres para una sección determinada de la jerarquía DNS. Las zonas DNS difieren de la estructura de dominio de la manera siguiente: las zonas pueden estar compuestas por uno o varios dominios DNS. Una zona del gadgets.widgets.microsoft.com de dominio podría ser autoritativa para los dominios de widgets y de widgets. En otras palabras, no es necesario que las zonas DNS tengan una relación uno a uno con los dominios DNS.
