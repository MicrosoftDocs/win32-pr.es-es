# <a name="protected-files"></a>Archivos protegidos

Cuando un archivo está protegido mediante DRM de Windows Media, se cifra la sección de datos del archivo. El proceso de protección de un archivo también se denomina empaquetar el archivo. Las API extendidas del cliente DRM de Windows Media no le permiten empaquetar archivos. Puede empaquetar archivos mediante los objetos del SDK de Windows Media Rights Manager o del SDK de Windows Media Format.

Un usuario no puede tener acceso a los datos de un archivo protegido sin obtener primero una licencia para ese archivo.
