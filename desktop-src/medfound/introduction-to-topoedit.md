# <a name="introduction-to-topoedit"></a>Introducción a TopoEdit

En Media Foundation, los datos multimedia se procesan a través de la capa de canalización y la sesión multimedia controla el flujo de datos y proporciona controles de reproducción. Los datos de cada componente de canalización (origen, transformación y receptor) se representan como un *nodo de topología*. El *objeto Topology* es una colección de nodos Topology y representa la ruta de acceso completa del flujo de datos a través de la canalización de la sesión multimedia.

Para obtener información acerca de las topologías, consulte [acerca de las topologías](about-topologies.md).

TopoEdit es una herramienta visual para compilar y probar topologías. Con TopoEdit, puede realizar las siguientes tareas:

-   Cree una topología agregando varios nodos de topología, como los nodos de origen, transformación y salida.

-   Conecte y resuelva la topología.

-   Pruebe la funcionalidad de la topología a través de la reproducción.
