# <a name="moving-regions"></a>Mover regiones

Una aplicación mueve una región mediante una llamada a la función [**OffsetRgn**](/windows/desktop/api/Wingdi/nf-wingdi-offsetrgn) . Los desplazamientos especificados a lo largo del eje x e y determinan el número de unidades lógicas que se van a desplazar hacia la izquierda o hacia la derecha y hacia arriba o hacia abajo.
