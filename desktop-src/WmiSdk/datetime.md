# <a name="datetime"></a>DATETIME

El tipo de datos DATETIME es una cadena BSTR VT de longitud fija que representa una fecha \_ y hora o un intervalo específicos. Para obtener más información sobre DATETIME como una fecha, vea [Formato de fecha y hora](date-and-time-format.md). Para obtener más información sobre DATETIME como intervalo, vea [Formato de intervalo](interval-format.md).
