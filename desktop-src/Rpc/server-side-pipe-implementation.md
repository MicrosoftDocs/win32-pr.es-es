# <a name="server-side-pipe-implementation"></a>Implementación de Server-Side Pipe

Los programas de servidor para las aplicaciones distribuidas que utilizan canalizaciones no necesitan implementar las funciones de inserción, extracción o asignación. Tienen que contener procedimientos que los clientes pueden invocar de forma remota para iniciar las transferencias de datos. En los temas siguientes, en esta sección se explica cómo se pueden implementar los procedimientos remotos del servidor.

-   [Implementar canalizaciones de entrada en el servidor](implementing-input-pipes-on-the-server.md)
-   [Implementación de canalizaciones de salida en el servidor](implementing-output-pipes-on-the-server.md)
