# <a name="font-embedding-reference"></a>Referencia de incrustación de fuentes

La biblioteca de servicios de incrustación de fuentes proporciona un mecanismo para agrupar las fuentes TrueType y OpenType de Microsoft en un documento o archivo. Normalmente, un documento que contiene fuentes incrustadas necesita esas fuentes para representar el documento en otro equipo. La incrustación de una fuente garantiza que una fuente especificada en un archivo estará presente en el equipo que recibe el archivo. No obstante, algunas fuentes no se pueden migrar a otros equipos debido a problemas de copyright que limitan la distribución.

Los elementos siguientes se usan con los servicios de incrustación de fuentes.

-   [Funciones de incrustación de fuentes](font-embedding-functions.md)
-   [Estructuras de incrustación de fuentes](font-embedding-services-structures.md)
-   [Mensajes de error de incrustación de fuentes](font-embedding-function-error-messages.md)
