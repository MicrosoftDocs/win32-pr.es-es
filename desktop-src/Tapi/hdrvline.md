# <a name="hdrvline"></a>HDRVLINE

El **tipo de datos HDRVLINE** representa el identificador opaco de un proveedor de servicios para una estructura de datos de línea. El proveedor de servicios debe resolver un valor de este tipo en una referencia a la instancia de estructura de datos adecuada.
