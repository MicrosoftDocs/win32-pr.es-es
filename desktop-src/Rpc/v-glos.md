<dl> <dt>

<span id="_rpc_varying_array_glos"></span><span id="_RPC_VARYING_ARRAY_GLOS"></span>**Matriz variable**
</dt> <dd>

Matriz cuyo intervalo de elementos transmitidos está determinado en tiempo de ejecución por otro parámetro, estructura o expresión. Consulte también [*Matriz compatible y*](c-glos.md) [*Abrir matriz*](o-glos.md).

</dd> <dt>

<span id="_rpc_version_number_glos"></span><span id="_RPC_VERSION_NUMBER_GLOS"></span>**número de versión**
</dt> <dd>

Dos números, separados por un separador decimal, que identifican la versión de una interfaz. Para que sea compatible, el número de versión principal (el número a la izquierda del separador decimal) debe ser el mismo para el cliente y el servidor, y el número de versión secundaria del servidor debe ser mayor o igual que el número de versión secundaria del cliente.

</dd> </dl>
