# <a name="obtaining-a-root-oid-from-an-iso-name-registration-authority"></a>Obtener un OID raíz de una entidad de registro de nombres ISO

La manera preferida de obtener un identificador de objeto raíz (OID) es solicitar uno a una autoridad de registro de nombres de la Organización internacional de estándares (ISO). Se trata de una acción única; cuando haya obtenido un OID raíz, el espacio de OID que define es el de usted y puede administrarlo usted mismo.

Normalmente hay una cuota asociada al registro de un nombre de organización y la recepción de un OID raíz. Las tarifas y las directivas de registro varían según el país o región. En la Estados Unidos, iso NRA es el American National Standards Institute (ANSI). Para obtener más información sobre el procedimiento de registro ANSI y la programación de cuotas, vea https://web.ansi.org . La norma ISO mantiene una lista de organizaciones miembro en https://www.iso.ch . Si está fuera del Estados Unidos, debe ponerse en contacto con la organización miembro iso de su país o región para obtener información de registro de nombres.

Independientemente del origen usado para obtener el OID, si tiene previsto ampliar el esquema de Active Directory y desea solicitar el logotipo certified for Windows, debe registrar el OID en Microsoft. Para obtener más información sobre cómo registrar el OID con Microsoft, vea [Obtener un identificador de objeto de Microsoft.](obtaining-an-object-identifier-from-microsoft.md)
