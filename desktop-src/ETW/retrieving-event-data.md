# <a name="retrieving-event-data"></a>Recuperar datos de eventos

Para consumir datos específicos de eventos, el consumidor debe conocer el formato de los datos de evento. Esto no es un problema si está consumiendo eventos de su propio proveedor porque conoce el formato de los datos. Sin embargo, para consumir un evento de cualquier proveedor, el proveedor debe publicar el diseño de los datos de evento. Para obtener información general sobre cómo funcionan los metadatos de eventos y cómo publicar los metadatos de eventos para que otros usuarios puedan usarlo, consulte [información general sobre metadatos de eventos](event-metadata-overview.md).

Para consumir eventos en Windows Vista que se publicaron con los archivos manifest, MOF o TMF, así como los eventos TraceLogging, vea [recuperar datos de eventos mediante TDH](retrieving-event-data-using-tdh.md).

Para consumir eventos anteriores a Windows Vista que se publicaron con MOF, consulte [recuperar datos de eventos mediante MOF](retrieving-event-data-using-mof.md).
