# <a name="frs-file-filter-attribute"></a>Atributo FRS-File-Filter

Una lista de extensiones de nombre de archivo excluidas de la replicación de archivos.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Filtro de archivos de FRS                             |
| Nombre para mostrar de LDAP | fRSFileFilter                               |
| Tamaño              | \-                                          |
| Actualizar privilegio  | Administrador de dominio                        |
| Frecuencia de actualización  | Cuando se configura la replicación.                  |
| Attribute-Id      | 1.2.840.113556.1.4.483                      |
| System-ID-GUID    | 1be8f170-a9ff-11d0-afe2-00c04fd930c9        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
