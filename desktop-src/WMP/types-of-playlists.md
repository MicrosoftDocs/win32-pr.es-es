# <a name="types-of-playlists"></a>Tipos de listas de reproducción

Las listas de reproducción de metarchivos de Windows Media, simplemente denominadas listas de reproducción en este documento, tienen extensiones. Wax,. wvx o. ASX. Las listas de reproducción con estas extensiones se utilizan para tener acceso a los archivos de Windows Media con las extensiones de nombre de archivo. WMA,. WMV y. ASF, respectivamente.
