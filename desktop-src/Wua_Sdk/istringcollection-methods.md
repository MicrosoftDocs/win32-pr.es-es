# <a name="istringcollection-methods"></a>Métodos IStringCollection

La interfaz [**IStringCollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) define los siguientes métodos.



| Método                                         | Descripción                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Agréguela**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-add)           | Agrega un elemento a la colección.                                |
| [**Claridad**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-clear)       | Quita todos los elementos de la colección.                  |
| [**Copiar**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-copy)         | Crea una copia de lectura/escritura profunda de la colección.              |
| [**Introducir**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-insert)     | Inserta un elemento en la colección situada en la posición especificada. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-removeat) | Quita de la colección un elemento situado en el índice especificado.   |
