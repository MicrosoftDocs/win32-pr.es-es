# <a name="task-scheduler-reference"></a>Programador de tareas referencia

En esta sección se describen las API, los objetos de scripting y el esquema XML que proporciona el Programador de tareas.

Los temas de API incluyen una descripción de la API, la sintaxis de declaración de la API, comentarios que describen condiciones especiales para la API y un bloque de requisitos que describe qué plataforma, archivos de encabezado y bibliotecas son necesarios.

Los temas de objeto de scripting incluyen una descripción del objeto, la sintaxis de declaración del objeto, comentarios que describen condiciones especiales para el objeto y un bloque de requisitos que describe qué plataforma y bibliotecas de tipos son necesarias.

Los temas de esquema XML incluyen una descripción del elemento, el elemento primario, el elemento secundario y la información de atributos, y comentarios que describen condiciones especiales.
