# <a name="client-functions"></a>Funciones de cliente

Esta sección contiene información acerca de las funciones de cliente utilizadas con Microsoft Active Accessibility.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                             | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
|-----------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AccessibleChildren**](/windows/desktop/api/Oleacc/nf-oleacc-accessiblechildren)<br/>                       | Recupera el identificador secundario o [**IDispatch**](idispatch-interface.md) de cada elemento secundario dentro de un objeto contenedor accesible.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**AccessibleObjectFromEvent**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfromevent)<br/>         | Recupera la dirección de la interfaz [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) para el objeto que generó el evento que la función de enlace de eventos del cliente está procesando actualmente.<br/>                                                                                                                                                                                                                                                                                                                                                                             |
| [**AccessibleObjectFromPoint**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfrompoint)<br/>         | Recupera la dirección del puntero de interfaz [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) para el objeto que se muestra en un punto especificado de la pantalla.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                       |
| [**AccessibleObjectFromWindow**](/windows/desktop/api/Oleacc/nf-oleacc-accessibleobjectfromwindow)<br/>       | Recupera la dirección de la interfaz especificada para el objeto asociado a la ventana especificada.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| [**GetOleaccVersionInfo**](/windows/desktop/api/Oleacc/nf-oleacc-getoleaccversioninfo)<br/>                   | Recupera el número de versión y el número de compilación del archivo de Microsoft Active Accessibility Oleacc.dll.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| [**GetProcessHandleFromHwnd**](getprocesshandlefromhwnd.md)<br/>           | Recupera un identificador de proceso de un identificador de ventana.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**GetRoleText**](/windows/desktop/api/Oleacc/nf-oleacc-getroletexta)<br/>                                     | Recupera la cadena localizada que describe el rol del objeto para el valor de rol especificado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**GetStateText**](/windows/desktop/api/Oleacc/nf-oleacc-getstatetexta)<br/>                                   | Recupera una cadena traducida que describe el estado de un objeto para una sola marca de bits de estado predefinida. Dado que los valores de estado son una combinación de una o varias marcas de bits, los clientes llaman a esta función más de una vez para recuperar todas las cadenas de estado.<br/>                                                                                                                                                                                                                                                                                                                      |
| [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook)<br/>                             | Establece una función de enlace de eventos para un intervalo de eventos.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| [**UnhookWinEvent**](/windows/desktop/api/Winuser/nf-winuser-unhookwinevent)<br/>                               | Quita una función de enlace de eventos creada por una llamada anterior a [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| [**WindowFromAccessibleObject**](/windows/desktop/api/Oleacc/nf-oleacc-windowfromaccessibleobject)<br/>       | Recupera el identificador de ventana que corresponde a una instancia determinada de una interfaz [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| [*Función de devolución de llamada WinEventProc*](/windows/desktop/api/Winuser/nc-winuser-wineventproc)<br/> | Función de devolución de llamada definida por la aplicación (o enlace) que el sistema llama en respuesta a los eventos generados por un objeto accesible. La función de enlace procesa las notificaciones de eventos según sea necesario. Los clientes instalan la función de enlace y solicitan tipos específicos de notificaciones de eventos mediante una llamada a [**SetWinEventHook**](/windows/desktop/api/Winuser/nf-winuser-setwineventhook).<br/> El tipo [**WINEVENTPROC**](/previous-versions/windows/desktop/legacy/dd373882(v=vs.85)) define un puntero a esta función de devolución de llamada. [*WinEventProc*](/windows/desktop/api/winuser/nc-winuser-wineventproc) es un marcador de posición para el nombre de la función definida por la aplicación.<br/> |
