# <a name="playerplaylistcollection"></a>Player.playlistCollection

La **propiedad playlistCollection** recupera el **objeto PlaylistCollection.**

## <a name="syntax"></a>Syntax

*player* . **playlistCollection**

## <a name="possible-values"></a>Valores posibles

Esta propiedad es un objeto **PlaylistCollection de** solo lectura.

## <a name="remarks"></a>Comentarios

Para recuperar el valor de esta propiedad, se requiere acceso de lectura a la biblioteca. Para obtener más información, vea [Acceso a la biblioteca.](library-access.md)
