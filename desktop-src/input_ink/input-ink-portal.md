# <a name="ink-input"></a>Entrada de entrada de lápiz

Obtenga información sobre las distintas API de entrada de entrada de escritorio Windows aplicaciones.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|---|---|
| [Presentador de entrada manuscrita](ink-presenter.md)<br/> | Las API de presentador de entrada de lápiz permiten a las aplicaciones win32 de Microsoft administrar la entrada, el procesamiento y la representación de la entrada de lápiz (estándar y modificada) a través de un objeto [**InkPresenter**](/uwp/api/Windows.UI.Input.Inking.InkPresenter) insertado en el árbol visual [DirectComposition](../directcomp/directcomposition-portal.md) de la aplicación.<br/> |
| [Representador de entrada manuscrita](ink-renderer.md)<br/>   | Las [](ink-renderer.md) API de representador de entrada de lápiz permiten la representación de trazos de entrada de lápiz en el contexto de dispositivo Direct2D designado de una aplicación universal Windows, en lugar del control [**InkCanvas predeterminado.**](/uwp/api/Windows.UI.Xaml.Controls.InkCanvas)<br/>                                                                        |
