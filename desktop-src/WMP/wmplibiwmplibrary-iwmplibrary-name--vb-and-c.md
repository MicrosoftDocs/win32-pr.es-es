# <a name="iwmplibraryname-property"></a>IWMPLibrary:: Name (propiedad)

La propiedad **Name** obtiene el nombre para mostrar de la biblioteca actual.

## <a name="syntax"></a>Sintaxis


```CSharp
public System.String name {get; set;}
```


```VB

Public ReadOnly Property name As System.String
```





## <a name="property-value"></a>Valor de propiedad

**System. String** que es el nombre de la biblioteca actual.
