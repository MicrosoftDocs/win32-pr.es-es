# <a name="texture2darraygather-methods"></a>Texture2DArray:: Gather (métodos)

Devuelve los cuatro valores de textura de un [**Texture2DArray**](sm5-object-texture2darray.md) que se utilizarían en una operación de filtrado bilineal.

Consulte la documentación de [gather4](./gather4--sm5---asm-.md) para obtener más información que describe la instrucción DXBC subyacente.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                | Descripción                                                                                                               |
|:----------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------|
| [**Gather (S, Float, int)**](sm5-object-texture2darray-gather.md)        | Devuelve los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal.<br/>                                 |
| [**Gather (S, Float, int, uint)**](t2darray-gather-s-float-int-uint-.md)  | Devuelve los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal, junto con el estado de asignación de mosaicos.<br/> |
