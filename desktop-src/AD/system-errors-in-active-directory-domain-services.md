# <a name="system-errors-in-active-directory-domain-services"></a>Errores del sistema en Active Directory Domain Services

Este tema contiene los valores devueltos de error del sistema en funciones de Active Directory Domain Services.



| Value                             | Descripción                                                           |
|-----------------------------------|-----------------------------------------------------------------------|
| **hrFileClose**<br/>        | No se puede cerrar el archivo de DOS.<br/>                              |
| **hrOutOfThreads**<br/>     | No se puede iniciar un subproceso porque no hay ninguno disponible.<br/> |
| **hrTooManyIO**<br/>        | El sistema está ocupado porque hay demasiadas operaciones de e/s.<br/>        |
| **hrBFNotSynchronous**<br/> | La página de búfer se ha expulsado.<br/>                          |
| **hrBFPageNotFound**<br/>   | No se encuentra la página.<br/>                                   |
| **hrBFInUse**<br/>          | No se puede abandonar el búfer.<br/>                              |
