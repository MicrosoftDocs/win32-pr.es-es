# <a name="cbaseinputpinisflushing-method"></a>Método CBaseInputPin.IsFlushing

El `IsFlushing` método consulta si el filtro se va a vaciar actualmente.

## <a name="syntax"></a>Sintaxis


```C++
BOOL IsFlushing();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve la [**variable miembro CBaseInputPin::m \_ bFlushing.**](cbaseinputpin-m-bflushing.md)
