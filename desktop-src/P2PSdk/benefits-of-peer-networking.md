# <a name="benefits-of-peer-networking"></a>Ventajas de las redes del mismo nivel

La infraestructura punto a punto proporciona una solución completa para crear aplicaciones de red punto a punto. Puede usar las API de proveedor de grafos punto a punto, agrupación, Identity Manager y proveedor de espacios de nombres PNRP para crear aplicaciones punto a punto nuevas, emocionantes y sólidas. La infraestructura punto a punto proporciona una infraestructura de  red sólida para que pueda concentrarse en el desarrollo de aplicaciones, ya que la infraestructura se desarrolla en su lugar.

## <a name="design-features"></a>Características de diseño

La infraestructura punto a punto proporciona las siguientes características:

| Característica            | Descripción                                          |
|--------------------|------------------------------------------------------|
| **Seguro**         | Solidez si hay errores o ataques dirigidos |
| **Escalable**       | Escalabilidad desde redes LAN sencillas a toda Internet  |
| **Sin servidor**     | No hay ningún punto central de error                          |
| **Ajuste automático**    | Se adapta a los cambios dentro de la infraestructura          |
| **Reparación propia** | Se repara automáticamente y se corrige a sí mismo            |
| **Uso compartido**        | Habilita el uso compartido desde puntos de conexión perimetrales de red.       |



 

Mediante el uso de la infraestructura punto a punto, puede crear los siguientes tipos de aplicaciones:

-   Redes y comunicaciones en tiempo real
-   Colaboración
-   Distribución de contenido
-   Coordinación del procesamiento distribuido
-   Tecnologías y protocolos de Internet

## <a name="alternative-to-dns"></a>Alternativa a DNS

Dado que las entidades punto a punto participan y se comunican ad hoc, la infraestructura punto a punto es una alternativa eficaz a DNS para la resolución de nombres. La infraestructura punto a punto usa la API del proveedor de espacios de nombres PNRP para proporcionar un mecanismo sin servidor para resolver nombres punto a punto.
