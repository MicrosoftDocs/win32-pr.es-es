# <a name="externalignoreiehistory"></a>External.ignoreIEHistory

> [!Note]  
> En este tema se describe la funcionalidad diseñada para su uso por las tiendas en línea. No se admite el uso de esta funcionalidad fuera del contexto de una tienda en línea.

 

La **propiedad ignoreIEHistory** especifica si Reproductor de Windows Media debe omitir Internet Explorer historial.

``` syntax
window.external.ignoreIEHistory
      
```

## <a name="possible-values"></a>Valores posibles

Esta propiedad es un booleano de solo **escritura.** **TRUE** indica que Reproductor de Windows Media debe omitir el historial.
