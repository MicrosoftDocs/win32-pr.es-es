# <a name="videoonvideoend"></a>VÍDEO. onvideoend

El controlador de eventos **onvideoend** controla un evento que se produce cuando el vídeo detiene la representación y se descarga.

``` syntax

onvideoend 
```

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
