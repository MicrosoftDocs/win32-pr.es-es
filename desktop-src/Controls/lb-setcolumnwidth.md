# <a name="lb_setcolumnwidth-message"></a>\_Mensaje lb SETCOLUMNWIDTH

Establece el ancho, en píxeles, de todas las columnas de un cuadro de lista de varias columnas.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>

Especifica el ancho, en píxeles, de todas las columnas.

</dd> <dt>

*lParam* 
</dt> <dd>

Este parámetro no se utiliza.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Este mensaje no devuelve ningún valor.
