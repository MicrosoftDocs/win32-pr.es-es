# <a name="count-of-properties"></a>Recuento de propiedades

Este tipo de datos **DWORD** proporciona un recuento de los valores de propiedad de la sección. Un conjunto de propiedades puede contener cualquier número de valores de propiedad. Los lectores deben ser capaces de controlar el caso en el que no hay ninguna propiedad.
