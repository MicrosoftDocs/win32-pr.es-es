# <a name="value-headerfieldtype-element"></a>Elemento Value (headerFieldType)

Contiene el valor de un campo de encabezado en un mensaje de correo electrónico.

``` syntax
<xs:element name="Value"
    type="string"
 />
```

El **elemento Value** se define mediante el tipo complejo [**headerFieldType.**](taskschedulerschema-headerfieldtype-complextype.md)

## <a name="parent-element"></a>Elemento primario



| Elemento                                                                         | Derivado de                                                               | Descripción                                                    |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------|----------------------------------------------------------------|
| [**HeaderField**](taskschedulerschema-headerfield-headerfieldstype-element.md) | [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) | Contiene un campo para un encabezado en un mensaje de correo electrónico. <br/> |
