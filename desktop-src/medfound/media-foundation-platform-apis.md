# <a name="media-foundation-platform-apis"></a>API de Media Foundation Platform

El nivel de plataforma de Media Foundation contiene primitivos y objetos auxiliares usados por las otras capas.

Esta sección contiene los siguientes temas:



| Tema                                                                           | Descripción                                                                                                                                                       |
|---------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Inicialización de la plataforma Media Foundation](initializing-media-foundation.md) | Cómo inicializar la plataforma Media Foundation.                                                                                                                  |
| [Media Foundation y COM](media-foundation-and-com.md)                        | Describe la interacción entre COM y Microsoft Media Foundation, y define algunos procedimientos recomendados que se deben usar al desarrollar componentes de complemento de Media Foundation. |
| [Métodos de devolución de llamada asincrónica](asynchronous-callback-methods.md)              | Cómo llamar a métodos asincrónicos y cómo implementar operaciones asincrónicas en Media Foundation.                                                                |
| [Colas de trabajo](work-queues.md)                                                  | Una cola de trabajo es una manera eficaz de realizar operaciones asincrónicas en otro subproceso.                                                                            |
| [Generadores de eventos multimedia](media-event-generators.md)                            | Cómo recibir eventos asincrónicos y cómo generar eventos en Media Foundation.                                                                                   |
| [Interfaces de servicio](service-interfaces.md)                                    | Una interfaz de servicio es una interfaz COM que proporciona un objeto, pero que se expone a la aplicación a través de otro objeto.                                             |
| [Objetos de activación](activation-objects.md)                                    | Un objeto de activación es un objeto que crea otro objeto.                                                                                                    |
| [Reloj de presentación](presentation-clock.md)                                    | El reloj de la presentación genera la hora de reloj que se usa para controlar la reproducción y también para las secuencias de audio y vídeo sincrónicas.                                |
