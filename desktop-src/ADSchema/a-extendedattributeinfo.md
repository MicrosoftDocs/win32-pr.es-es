| CN                | Extended-Attribute-info                     |
| Nombre para mostrar de LDAP | extendedAttributeInfo                       |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.909                      |
