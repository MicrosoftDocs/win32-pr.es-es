# <a name="print-keep-printed-jobs-attribute"></a>Atributo de trabajos de impresión y mantenimiento impresos

**True** si se conservan los trabajos impresos.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Trabajos de impresión y mantenimiento impresos              |
| Nombre para mostrar de LDAP | printKeepPrintedJobs                 |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.275               |
| System-ID-GUID    | ba305f6d-47e3-11d0-a1a6-00c04fd930c9 |
| Sintaxis            | [**Booleano**](s-boolean.md)         |
