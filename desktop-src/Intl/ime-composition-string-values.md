# <a name="ime-composition-string-values"></a>Valores de cadena de composición de IME

Estos valores se usan con la [**\_ \_ composición**](wm-ime-composition.md)de [**ImmGetCompositionString**](/windows/desktop/api/Imm/nf-imm-immgetcompositionstringa) y el IME de WM.



| Value                 | Descripción                                                                                |
|-----------------------|--------------------------------------------------------------------------------------------|
| GC \_ COMPATTR         | Recupere o actualice el atributo de la cadena de composición.                                |
| GC \_ COMPCLAUSE       | Recuperar o actualizar la información de la cláusula de la cadena de composición.                           |
| GC \_ COMPREADATTR     | Recuperar o actualizar los atributos de la cadena de lectura de la composición actual.        |
| GC \_ COMPREADCLAUSE   | Recuperar o actualizar la información de la cláusula de lectura de la cadena de composición. |
| GC \_ COMPREADSTR      | Recupera o actualiza la cadena de lectura de la composición actual.                          |
| GC \_ COMPSTR          | Recuperar o actualizar la cadena de composición actual.                                         |
| GC \_ CURSORPOS        | Recupera o actualiza la posición del cursor en la cadena de composición.                              |
| GC \_ DELTASTART       | Recupera o actualiza la posición inicial de los cambios en la cadena de composición.             |
| GC \_ RESULTCLAUSE     | Recuperar o actualizar la información de la cláusula de la cadena de resultado.                                |
| GC \_ RESULTREADCLAUSE | Recuperar o actualizar la información de las cláusulas de la cadena de lectura.                               |
| GC \_ RESULTREADSTR    | Recuperar o actualizar la cadena de lectura.                                                     |
| RESULTSTR de GC \_        | Recupere o actualice la cadena del resultado de la composición.                                   |
