# <a name="linked-objects"></a><span data-ttu-id="928fc-103">Objetos vinculados</span><span class="sxs-lookup"><span data-stu-id="928fc-103">Linked Objects</span></span>

<span data-ttu-id="928fc-104">Cuando se inserta un vínculo a un objeto en un documento compuesto, los datos de origen o el *origen del vínculo* continúan residiendo dondequiera que se crearon inicialmente, normalmente en otro documento.</span><span class="sxs-lookup"><span data-stu-id="928fc-104">When a link to an object is inserted in a compound document, the source data, or *link source*, continues to reside wherever it was initially created, usually in another document.</span></span> <span data-ttu-id="928fc-105">El documento compuesto solo contiene una referencia, o un *vínculo*, a los datos reales almacenados en el origen del vínculo, junto con información sobre cómo presentar esos datos al usuario.</span><span class="sxs-lookup"><span data-stu-id="928fc-105">The compound document contains only a reference, or *link*, to the actual data stored at the link source, along with information about how to present that data to the user.</span></span>

<span data-ttu-id="928fc-106">La activación de un vínculo ejecuta la aplicación de servidor del origen del vínculo, que el usuario necesita para editar o manipular de otro modo los datos del vínculo.</span><span class="sxs-lookup"><span data-stu-id="928fc-106">Activating a link runs the link source's server application, which the user requires in order to edit or otherwise manipulate the link data.</span></span> <span data-ttu-id="928fc-107">La vinculación mantiene el tamaño de un documento compuesto pequeño.</span><span class="sxs-lookup"><span data-stu-id="928fc-107">Linking keeps the size of a compound document small.</span></span> <span data-ttu-id="928fc-108">También resulta útil cuando un usuario mantiene el origen de datos y debe compartirse entre varios usuarios.</span><span class="sxs-lookup"><span data-stu-id="928fc-108">It is also useful when the data source is maintained by someone else and must be shared among many users.</span></span> <span data-ttu-id="928fc-109">Si la persona que mantiene el origen del vínculo cambia los datos, el cambio se actualiza automáticamente en todos los documentos que contienen un vínculo a esos datos.</span><span class="sxs-lookup"><span data-stu-id="928fc-109">If the person maintaining the link source changes the data, the change is automatically updated in all documents containing a link to that data.</span></span> <span data-ttu-id="928fc-110">Además de crear vínculos simples, los usuarios pueden anidar vínculos y combinar objetos vinculados e incrustados para crear documentos complejos.</span><span class="sxs-lookup"><span data-stu-id="928fc-110">In addition to creating simple links, users can nest links and combine linked and embedded objects to create complex documents.</span></span>
