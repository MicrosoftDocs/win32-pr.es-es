# <a name="apijetsetcurrentindex3-method"></a>Método API. JetSetCurrentIndex3

Establece el índice actual de un cursor.

**Espacio de nombres:**  [Microsoft. ISAM. esent. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Ensamblado:**  Microsoft. ISAM. esent. Interop (en Microsoft.Isam.Esent.Interop.dll)

## <a name="syntax"></a>Sintaxis

``` vb
'Declaration
Public Shared Sub JetSetCurrentIndex3 ( _
    sesid As JET_SESID, _
    tableid As JET_TABLEID, _
    index As String, _
    grbit As SetCurrentIndexGrbit, _
    itagSequence As Integer _
)
'Usage
Dim sesid As JET_SESID
Dim tableid As JET_TABLEID
Dim index As String
Dim grbit As SetCurrentIndexGrbit

    tableid, index, grbit, itagSequence)
```

``` csharp
public static void JetSetCurrentIndex3(

)
```

#### <a name="parameters"></a>Parámetros

  - sesid  
    Tipo: [Microsoft.ISAM.esent.Interop.JET_SESID](./jet-sesid-structure.md)  
    
    La sesión que se va a usar.

<!-- end list -->

  - TABLEID  
    Tipo: [Microsoft.ISAM.esent.Interop.JET_TABLEID](./jet-tableid-structure.md)  
    
    Cursor en el que se va a establecer el índice.

<!-- end list -->

  - índice  
    Tipo: [System. String](/dotnet/api/system.string)  
    
    Nombre del índice que se va a seleccionar. Si es null o está vacío, se seleccionará el índice principal.

<!-- end list -->

  - grbit  
    Tipo: [Microsoft. ISAM. esent. Interop. SetCurrentIndexGrbit](./setcurrentindexgrbit-enumeration.md)  
    
    Establecer opciones de índice.

<!-- end list -->

  - itagSequence  
    Tipo: [System. Int32](/dotnet/api/system.int32)  
    
    Número de secuencia del valor de la columna con varios valores que se usará para colocar el cursor en el nuevo índice. Este parámetro solo se usa junto con [nomove](./setcurrentindexgrbit-enumeration.md). Cuando este parámetro no está presente o se establece en cero, se supone que su valor es 1.
