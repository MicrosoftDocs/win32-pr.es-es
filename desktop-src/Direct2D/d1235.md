# <a name="d1235-target-rendered-to-from-multiple-dcs"></a>D1235: el destino se representa desde varios DC

Se intentó representar en una imagen mientras aún estaba siendo representada por un contexto de dispositivo anterior. Se debe llamar a EndDraw en ese contexto de dispositivo antes de la representación con el contexto de dispositivo actual. Cuando se usa la capa de depuración, también se genera la dirección de este contexto de dispositivo.
