# <a name="locale_sdayname-constants"></a>Constantes de SDAYNAME de configuración regional \_ \*

En este tema se definen las constantes de SDAYNAME de configuración regional \_ \* utilizadas por NLS.



| Value             | Significado                                                                                                                                 |
|-------------------|-----------------------------------------------------------------------------------------------------------------------------------------|
| Configuración regional \_ SDAYNAME1 | Nombre largo nativo del lunes. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.    |
| Configuración regional \_ SDAYNAME2 | Nombre largo nativo del martes. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.   |
| Configuración regional \_ SDAYNAME3 | Nombre largo nativo para el miércoles. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación. |
| Configuración regional \_ SDAYNAME4 | Nombre largo nativo del jueves. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.  |
| Configuración regional \_ SDAYNAME5 | Nombre largo nativo del viernes. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.    |
| Configuración regional \_ SDAYNAME6 | Nombre largo nativo del sábado. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.  |
| Configuración regional \_ SDAYNAME7 | Nombre largo nativo para el domingo. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.    |
