# <a name="provider-eventstype-element"></a>provider (EventsType) (Elemento)

``` syntax

    type="ProviderType"
 />
```

El **elemento** provider se define mediante el [**tipo complejo EventsType.**](eventmanifestschema-eventstype-complextype.md)
