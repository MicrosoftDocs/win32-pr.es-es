# <a name="texturecubesamplebias-methods"></a>Métodos TextureCube::SampleBias

Muestrea una textura después de aplicar el valor de sesgo al nivel de mapa mip.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                     | Descripción                                                                                                                                                                                  |
|:-------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleBias(S,float,float)**](dx-graphics-hlsl-to-samplebias.md)                        | Muestrea una textura después de aplicar el valor de sesgo al nivel de mapa mip.<br/>                                                                                                             |
| [**SampleBias(S,float,float,float)**](tcube-samplebias-s-float-float-float-.md)           | Muestrea una textura después de aplicar el valor de sesgo al nivel de mapa mip, con un valor opcional para fijar los valores de nivel de detalle (LOD) de la muestra.<br/>                                     |
| [**SampleBias(S,float,float,float,uint)**](tcube-samplebias-s-float-float-float-uint-.md) | Muestrea una textura después de aplicar el valor de sesgo al nivel de mapa mip, con un valor opcional para fijar los valores de nivel de detalle (LOD) de la muestra. Devuelve el estado de la operación.<br/> |
