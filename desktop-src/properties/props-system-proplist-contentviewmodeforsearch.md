# <a name="systemproplistcontentviewmodeforsearch"></a>System. proplist. ContentViewModeForSearch

Identifica la lista de propiedades que se van a mostrar en el modo de vista de contenido de un elemento en el contexto de búsqueda.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7

```
propertyDescription
   name = System.PropList.ContentViewModeForSearch
   shellPKey = PKEY_PropList_ContentViewModeForSearch
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 14
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
