# <a name="xmfloat3se-operators"></a>Operadores XMFLOAT3SE

La estructura [**XMFLOAT3SE**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmfloat3se) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                               | Descripción                                                                                                             |
|-----------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**operador UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmfloat3se-operatoruint32_t)<br/> | Devuelve una instancia de `uint32_t` que contiene los componentes de la `XMFLOAT3SE` instancia en un formato empaquetado.<br/> |
| [**operador =**](xmfloat3se-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMFLOAT3SE` .<br/>                                       |
