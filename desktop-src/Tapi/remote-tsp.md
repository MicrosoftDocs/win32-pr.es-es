# <a name="remote-tsp"></a>TSP remoto

El TSP remoto (Remotesp.tsp) proporciona acceso a los recursos de comunicaciones que no son locales en el equipo actual, como UN ARCHIVO SWITCH. Cuando el TSP remoto está configurado para apuntar a un servidor determinado, los recursos aparecen en una aplicación local como si existieran en el equipo del usuario.

Tcmsetup es una utilidad de línea de comandos que habilita la configuración de un TSP remoto. Para obtener más información sobre esta utilidad, consulte Windows Ayuda o el kit Windows Resource Kit.

El TSP remoto se instala con los sistemas operativos Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Me, Windows 98 y Windows 95.
