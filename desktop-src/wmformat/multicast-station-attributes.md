# <a name="multicast-station-attributes"></a>Atributos de la estación de multidifusión

Cuando un archivo se transmite desde una estación de multidifusión, la estación puede imponer algunos atributos en el archivo. Estos atributos, que se enumeran en la tabla siguiente, no se almacenan realmente en el archivo y solo están disponibles cuando el archivo se transmite por secuencias. Contienen información sobre la estación y suelen ser idénticas para todo el contenido de la estación.



| Atributo de estación de multidifusión                 | Identificador global      | Tipo de datos             |
|---------------------------------------------|------------------------|-----------------------|
| [**\_Dirección NSC**](nsc-address.md)         | g \_ wszWMNSCAddress     | **\_cadena de tipo WMT \_** |
| [**Descripción de NSC \_**](nsc-description.md) | g \_ wszWMNSCDescription | **\_cadena de tipo WMT \_** |
| [**\_Correo electrónico NSC**](nsc-email.md)             | g \_ wszWMNSCEmail       | **\_cadena de tipo WMT \_** |
| [**\_Nombre NSC**](nsc-name.md)               | g \_ wszWMNSCName        | **\_cadena de tipo WMT \_** |
| [**\_Teléfono NSC**](nsc-phone.md)             | g \_ wszWMNSCPhone       | **\_cadena de tipo WMT \_** |
