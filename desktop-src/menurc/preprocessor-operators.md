# <a name="preprocessor-operators"></a>Operadores de preprocesador

RC admite el uso de los operadores de preprocesamiento estándar de C en las definiciones de macro. Estos operadores se describen en la tabla siguiente.



| Operator | Descripción                                                 |
|----------|-------------------------------------------------------------|
| \#       | Incluye el argumento entre comillas.                            |
| \#@      | Incluye el argumento entre comillas simples.                     |
| \#\#     | Concatena los tokens que se usan como argumentos para formar otros tokens. |
