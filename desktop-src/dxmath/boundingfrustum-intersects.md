# <a name="boundingfrustumintersects-methods"></a>Métodos BoundingFrustum. Intersects

Comprueba el [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) para la intersección con otro objeto.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                           | Descripción                                                                                                                                |
|:-------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------|
| [**BoundingFrustum:: Intersects (XMVECTOR)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-intersects(fxmvector))                   | Prueba el [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) para la intersección con un plano.<br/>                                              |
| [**BoundingFrustum:: Intersects (const BoundingBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-intersects(constboundingbox_))         | Pruebe [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) para la intersección con [**BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox).<br/>                 |
| [**BoundingFrustum:: Intersects (const BoundingSphere&)**](/previous-versions/windows/desktop/legacy/hh855929(v=vs.85))      | Pruebe [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) para la intersección con un [**BoundingSphere**](/windows/win32/api/directxcollision/ns-directxcollision-boundingsphere).<br/>           |
| [**BoundingFrustum:: Intersects (const BoundingFrustum&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-intersects(constboundingfrustum_))     | Pruebe el [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) para la intersección con otro **BoundingFrustum**.<br/>                          |
| [**BoundingFrustum:: Intersects (XMVECTOR, XMVECTOR, Float&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-intersects(fxmvector_fxmvector_float_))   | Pruebe el [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) para la intersección con un rayo.<br/>                                                |
| [**BoundingFrustum:: Intersects (XMVECTOR, XMVECTOR, XMVECTOR)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-intersects(fxmvector_fxmvector_fxmvector)) | Pruebe el [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) para la intersección con un triángulo.<br/>                                           |
| [**BoundingFrustum:: Intersects (const BoundingOrientedBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingfrustum-intersects(constboundingorientedbox_)) | Pruebe [**BoundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum) para la intersección con un [**BoundingOrientedBox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox).<br/> |
