# <a name="modelurl-element"></a>elemento modelURL

Especifica la dirección URL del modelo de dispositivo.

## <a name="usage"></a>Uso

``` syntax
<modelURL/>
```

## <a name="attributes"></a>Atributos

No hay atributos.

## <a name="child-elements"></a>Elementos secundarios

No hay elementos secundarios.
