# <a name="packcpp"></a>Embalaje. CPP

En el componente proveedor de ejemplo, se encuentra un ejemplo de código de empaquetar y desempaquetar tipos de datos en variantes en Pack. cpp. Las siguientes funciones se usan cuando las propiedades se cargan y se quitan de la memoria caché de propiedades:

-   **PackStringinVariant**
-   **UnpackStringinVariant**
-   **PackLONGinVariant**
-   **UnpackLONGfromVariant**
-   **PackDATEinVariant**
-   **UnpackDATEinVariant**
-   **PackVARIANT \_ BOOLinVariant**
-   **UnpackVARIANT \_ BOOLfromVariant**
-   **PackVARIANTinVariant**
-   **UnpackVARIANTfromVariant**
