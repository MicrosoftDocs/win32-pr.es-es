# <a name="regionchange-method"></a>Método RegionChange

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

El `RegionChange` método muestra un cuadro de diálogo del sistema que permite al usuario cambiar la región asociada a la unidad de DVD.

``` syntax
MSWebDVD.RegionChange()
```

## <a name="return-value"></a>Valor devuelto

No de devuelve ningún valor.

## <a name="remarks"></a>Observaciones

La región de una unidad de DVD solo se puede cambiar cinco veces.
