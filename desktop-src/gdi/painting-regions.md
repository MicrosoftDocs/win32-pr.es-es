# <a name="painting-regions"></a>Áreas de dibujo

Una aplicación rellena el interior de una región mediante el pincel seleccionado actualmente en un contexto de dispositivo mediante la función [**PaintRgn**](/windows/desktop/api/Wingdi/nf-wingdi-paintrgn) . Esta función utiliza los modos de relleno de polígono actuales (alternativos y de bobinado).
