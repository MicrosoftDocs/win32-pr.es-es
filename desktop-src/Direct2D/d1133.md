# <a name="d1133-buffer-precision-unknown-not-allowed"></a>D1133: no se permite la precisión de búfer desconocida

\_ \_ No se permite la precisión del búfer de D2D1 \_ desconocida como parámetro de esta API.






 

## <a name="possible-causes"></a>Causas posibles

Direct2D permite que una aplicación especifique una precisión de búfer deseada para algunas operaciones. Por ejemplo, la precisión usada para la canalización de imágenes se especifica mediante ID2D1DeviceContext:: SetRenderingControls. El \_ especificador de precisión de búfer \_ desconocido permite que una aplicación sea anterior a esta configuración y Direct2D usará un valor predeterminado, pero este especificador no se permite en algunas operaciones. En concreto, es posible que no se permita durante la creación de una colección de delimitadores de degradado.

## <a name="possible-fixes"></a>Posibles correcciones

Asegúrese de que el \_ especificador de precisión de búfer \_ desconocido no se usa con ID2D1DeviceContext:: CreateGradientStopCollection. Usar un valor de precisión conocido; considere la posibilidad de usar la precisión del formato de la superficie de destino para la operación.
