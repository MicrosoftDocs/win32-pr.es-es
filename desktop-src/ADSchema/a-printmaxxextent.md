La región de impresión horizontal máxima.

|-------------------|--------------------------------------|
| CN                | Print-Max-X-extensión                   |
| Nombre para mostrar de LDAP | printMaxXExtent                      |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.277               |
