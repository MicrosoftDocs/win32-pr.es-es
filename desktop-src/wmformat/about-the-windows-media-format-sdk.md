# <a name="about-the-windows-media-format-sdk"></a>Acerca del SDK Windows Media Format

El Kit de desarrollo de software (SDK) de Microsoft® Windows Media® Format permite a los desarrolladores crear aplicaciones que manipulan medios digitales almacenados en archivos que se ajustan a la estructura de archivos de formato de sistemas avanzados (ASF). La estructura de ASF es un formato de contenedor versátil para medios de [*streaming arbitrarios.*](wmformat-glossary.md) Para obtener más información sobre la estructura de los archivos ASF, vea [Información general sobre el formato ASF](overview-of-the-asf-format.md).

Este SDK también proporciona compatibilidad para codificar y descodar contenido de audio y vídeo mediante los códecs Windows media. Cada códec (descompresión o descompresión) está optimizado para obtener la mejor calidad para un tipo específico de contenido. Para obtener más información sobre los códecs admitidos, vea [Características de códec.](codec-features.md)

Además de la compatibilidad general con el formato de archivo y la integración de códecs, el SDK Windows Media Format proporciona objetos para editar información descriptiva sobre medios digitales. La lectura de archivos y la compatibilidad con metadatos también se incluyen para archivos MP3, lo que le permite crear aplicaciones más flexibles.
