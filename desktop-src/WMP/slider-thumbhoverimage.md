# <a name="sliderthumbhoverimage"></a>SLIDEr. thumbHoverImage

El atributo **thumbHoverImage** especifica o recupera la imagen del control de posición que aparece al mantener el mouse sobre él.

``` syntax
        elementID.thumbHoverImage
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de lectura/escritura que contiene el nombre de un archivo de imagen.

## <a name="remarks"></a>Observaciones

**ThumbHoverImage** es opcional. Si no se proporciona, el control deslizante no cambiará de aspecto cuando se mantenga el mouse sobre él.

Los formatos admitidos son BMP, JPG, PNG y GIF (sin incluir los GIF animados).
