# <a name="enumerating-routing-table-entries"></a>Enumerar entradas de la tabla de enrutamiento

Las funciones de enumeración permiten a un cliente recuperar información acerca de un tipo específico de objeto de tabla de enrutamiento (rutas, destinos y próximos saltos). Tanto los protocolos de enrutamiento como los programas de administración pueden utilizar estas funciones para buscar datos específicos.

Para ver el código de ejemplo que muestra cómo usar estas funciones, consulte [enumerar todos los destinos](enumerate-all-destinations.md) y [enumerar todas las rutas](enumerate-all-routes.md).
