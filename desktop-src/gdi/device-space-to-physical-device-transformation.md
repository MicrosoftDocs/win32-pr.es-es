# <a name="device-space-to-physical-device-transformation"></a>Device-Space a Physical-Device transformación

La transformación espacio de dispositivo en dispositivo físico es única en varios aspectos. Por ejemplo, se limita a la traducción y se controla mediante el sistema. El único propósito de esta transformación es asegurarse de que el origen del espacio del dispositivo se asigna al punto adecuado en el dispositivo físico. No hay funciones para establecer esta transformación, ni ninguna función para recuperar los datos relacionados.
