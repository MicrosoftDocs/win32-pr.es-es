# <a name="object-roles"></a>Roles de objeto

En este tema se describen los valores constantes que se usan para describir los roles de varios objetos de interfaz de usuario en una aplicación. Las constantes de rol se definen en el archivo oleacc. h.

Solo debe usar las constantes de rol enumeradas aquí. no agregue roles o roles personalizados que no estén predefinidos.

Antes de usar estos roles de objeto, los desarrolladores de aplicaciones cliente deben usar la herramienta [inspeccionar](inspect-objects.md) para comprobar que los elementos de la interfaz de usuario están usando los roles de objeto.

Para recuperar el rol de un objeto, los clientes llaman al método [**IAccessible:: get \_ accRole**](/windows/desktop/api/Oleacc/nf-oleacc-iaccessible-get_accrole) , que debe devolver uno de los valores descritos en la tabla. Para recuperar una cadena traducida que describa el rol del objeto, los clientes llaman a la función [**GetRoleText**](/windows/desktop/api/Oleacc/nf-oleacc-getroletexta) con el valor de rol. Las cadenas localizadas para los valores de rol se encuentran en el archivo de oleaccrc.dll.
