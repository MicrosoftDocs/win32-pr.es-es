# <a name="taskfoldercollectionitem-property"></a>TaskFolderCollection. Item (propiedad)

En el caso de scripting, obtiene la carpeta especificada de la colección.

## <a name="syntax"></a>Sintaxis


```VB
TaskFolderCollection.Item( _
  ByVal Index _
) As TaskFolder
```



## <a name="property-value"></a>Valor de propiedad

Objeto [**TaskFolder**](taskfolder.md) que representa la carpeta solicitada.
