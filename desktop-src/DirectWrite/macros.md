# <a name="directwrite-macros"></a>Macros de DirectWrite

DirectWrite define las siguientes macros.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|-|-|
| [**DWRITE_MAKE_FONT_AXIS_TAG**](/windows/win32/api/dwrite_3/nf-dwrite_3-dwrite_make_font_axis_tag) | Crea una etiqueta OpenType para un eje de fuentes. |
| [**DWRITE \_ Make \_ \_ etiqueta OPENTYPE**](/windows/win32/api/dwrite/nf-dwrite-dwrite_make_opentype_tag) | Crea una etiqueta OpenType como entero de 32 bits, de modo que el primer carácter de la etiqueta es el byte más bajo (menos significativo en las arquitecturas little endian), que se puede usar para comparar con las etiquetas del archivo de fuente. Esta macro es compatible con la etiqueta de la [**\_ característica de fuente \_ \_ DWRITE**](/windows/win32/api/dwrite/ne-dwrite-dwrite_font_feature_tag). |
