# <a name="interfaces-for-service-providers-and-applications"></a>Interfaces para proveedores de servicios y aplicaciones

Los proveedores de servicios y las aplicaciones que usan el SDK de Administrador de dispositivos de Windows Media pueden utilizar las siguientes interfaces.



| Interfaz                                                | Descripción                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**IWMDMLogger**](/windows/desktop/api/wmdmlog/nn-wmdmlog-iwmdmlogger)                       | Registra entradas en un archivo de registro común.                                                        |
| [**IComponentAuthenticate**](/windows/desktop/api/mswmdm/nn-mswmdm-icomponentauthenticate) | Proporciona comunicación cifrada y segura entre los módulos de Windows Media Administrador de dispositivos. |
