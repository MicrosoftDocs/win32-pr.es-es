# <a name="window-stations-and-desktops"></a>Estaciones de ventana y equipos de escritorio

Windows proporciona tres categorías principales de objetos: interfaz de usuario, interfaz de dispositivo gráfico (GDI) y kernel. Los [objetos de kernel](/windows/desktop/SysInfo/kernel-objects) son protegibles, mientras que los objetos de [usuario](/windows/desktop/SysInfo/user-objects) y los [objetos GDI](/windows/desktop/SysInfo/gdi-objects) no lo son. Por lo tanto, para proporcionar seguridad adicional, los objetos de interfaz de usuario se administran mediante estaciones de ventana y equipos de escritorio, que a su vez son objetos protegibles.

Para obtener más información, vea los temas siguientes:

-   [Acerca de las estaciones de ventana y los escritorios](about-window-stations-and-desktops.md)
-   [Estación de ventana y referencia de escritorio](window-station-and-desktop-reference.md)
