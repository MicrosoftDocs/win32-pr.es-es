# <a name="logical-operations"></a>Operaciones lógicas

Se puede aplicar una operación lógica entre el fragmento y el valor almacenado en la ubicación correspondiente del búfer de fotogramas. el resultado reemplaza el valor actual del búfer de fotogramas. Elija la operación lógica deseada con [**glLogicOp**](gllogicop.md). Las operaciones lógicas solo se realizan en índices de color, nunca en valores RGBA.
