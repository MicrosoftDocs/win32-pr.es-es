# <a name="hyper-v-wmi-classes"></a>Clases WMI de Hyper-V

A continuación se muestran las clases WMI de Hyper-V.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                 | Descripción                                                                                                                                                                                                                                                                                 |
|---------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Clases CIM](cim-classes.md)<br/>                                             | Las clases CIM son clases base de las que derivan todos los objetos WMI.<br/>                                                                                                                                                                                                               |
| [Clases de BIOS](bios-classes.md)<br/>                                           | El BIOS virtual es una imagen de software que se carga en la RAM para configurar algunos de los aspectos básicos de y arrancar un equipo. Hay un elemento BIOS por sistema y ese elemento no se puede reemplazar ni quitar.<br/>                                                 |
| [Clases de entrada](input-classes.md)<br/>                                         | Estas clases representan los dispositivos de entrada de usuario. Una máquina virtual siempre tiene una instancia de cada clase asociada.<br/>                                                                                                                                             |
| [Clases de Integration Services](integration-services-classes.md)<br/>           | Los servicios de integración son servicios de software que se ejecutan en la parte superior del sistema operativo invitado dentro de una máquina virtual secundaria y como parte de la pila de virtualización en el sistema operativo de administración para proporcionar cierto nivel de integración con el sistema operativo de administración.<br/> |
| [Clases de memoria](memory-classes.md)<br/>                                       | Las clases de memoria modelan la memoria de una máquina virtual.<br/>                                                                                                                                                                                                                       |
| [Clases de procesador](processor-classes.md)<br/>                                 | Las clases de procesador modelan un procesador para una máquina virtual.<br/>                                                                                                                                                                                                                   |
| [Clases de registro de perfiles](profile-registration-classes.md)<br/>           | Clases WMI de virtualización relacionadas con el registro de perfiles.<br/>                                                                                                                                                                                                                      |
| [Clases de Escritorio remoto](remotefx-classes.md)<br/>                             | Las clases WMI de virtualización Escritorio remoto se usan para administrar los recursos de Escritorio remoto y RemoteFX y las máquinas virtuales.<br/>                                                                                                                                                     |
| [Clases para la administración de recursos](resource-management-classes.md)<br/>             | Clases WMI de virtualización relacionadas con la administración de recursos.<br/>                                                                                                                                                                                                                       |
| [Clases de dispositivos serie](serial-devices-classes.md)<br/>                       | Los dispositivos serie de una máquina virtual se componen de controladores serie y puertos serie. Cada máquina virtual tiene exactamente un controlador serie y cada controlador serie tiene exactamente dos puertos serie.<br/>                                                                         |
| [Clases de almacenamiento](storage-classes.md)<br/>                                     | Los objetos de almacenamiento se dividen en tres tipos: Controladores, unidades y elementos multimedia.<br/>                                                                                                                                                                                              |
| [Clases de vídeo](video-classes.md)<br/>                                         | Todas las máquinas virtuales reflejan la presencia de un controlador de vídeo S3 emulado y un controlador de vídeo sintético y acelerado.<br/>                                                                                                                                                     |
| [Clases de sistema virtual](virtual-system-classes.md)<br/>                       | Clases WMI de virtualización relacionadas con la máquina virtual.<br/>                                                                                                                                                                                                                       |
| [Clases de administración de sistema virtual](virtual-system-management-classes.md)<br/> | Clases WMI de virtualización relacionadas con la administración de máquinas virtuales.<br/>                                                                                                                                                                                                                |
