# <a name="network-list-manager"></a>Administrador de lista de redes

## <a name="purpose"></a>Propósito

La API del administrador de lista de redes permite a las aplicaciones recuperar una lista de conexiones de red disponibles. Las aplicaciones pueden filtrar las redes, en función de los atributos y las firmas, y elegir las redes más adecuadas para su tarea. La infraestructura del administrador de lista de redes notifica a las aplicaciones los cambios en el entorno de red, lo que permite a las aplicaciones actualizar dinámicamente las conexiones de red.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Network List Manager está diseñado para que lo usen los programadores de COM.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

La API del administrador de lista de redes se admite a partir de Windows Vista.
