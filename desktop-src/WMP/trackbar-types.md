# <a name="trackbar-types"></a>TrackBar (tipos)

No es necesario incluir controles deslizantes ni trackbars, pero pueden ser muy útiles para permitir que el usuario ajuste el volumen y la posición de reproducción. Hay dos tipos de trackbars:

-   Seek
-   Volumen

Para obtener más información sobre los controles deslizantes, consulte la sección [Trackbars](trackbars.md) de la referencia de máscara.
