# <a name="guidcpp"></a>Volumen. CPP

En el componente de proveedor de ejemplo, los GUID necesarios para los objetos de componente de proveedor de ejemplo se encuentran en GUID. cpp. Los GUID se definen para los siguientes elementos:

-   CLSID \_ SampleDSNamespace
-   CLSID \_ SampleDSProvider
-   IDENTIFICADOR de \_ SampleDSOle
-   CLSID \_ SampleDSGenObject
-   CLSID \_ SampleDSSchema
-   CLSID \_ SampleDSClass
-   CLSID \_ SampleDSProperty
