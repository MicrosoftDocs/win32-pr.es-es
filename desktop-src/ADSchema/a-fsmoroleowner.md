# <a name="fsmo-role-owner-attribute"></a>Atributo FSMO-role-Owner

Operación de Single-Master flexible: nombre distintivo del controlador de dominio en el que se puede modificar el esquema.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | FSMO: rol-Propietario                         |
| Nombre para mostrar de LDAP | fSMORoleOwner                           |
| Tamaño              | \-                                      |
| Actualizar privilegio  | Administrador de esquema                    |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.369                  |
| System-ID-GUID    | 66171887-8f3c-11d0-afda-00c04fd930c9    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
