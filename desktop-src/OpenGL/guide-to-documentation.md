# <a name="guide-to-documentation"></a>Guía de la documentación

La documentación establecida para OpenGL en Windows incluye cinco elementos.

-   El *manual de referencia de OpenGL* incluye información general sobre cómo funciona OpenGL y un conjunto de páginas de referencia detalladas. Las páginas de referencia cubren todas las funciones OpenGL distintas 115, así como las funciones 43 de la biblioteca de la utilidad OpenGL (GLU).

-   La *Guía de programación de OpenGL* explica cómo crear programas de gráficos mediante OpenGL. Incluye debates sobre los temas principales siguientes:

    -   Dibujar formas geométricas
    -   Píxeles, mapas de bits, fuentes e imágenes
    -   Visualización y transformaciones de matriz
    -   Asignación de textura
    -   Mostrar listas
    -   Técnicas compuestas avanzadas
    -   Color
    -   Evaluadores y NURBS
    -   Iluminación
    -   Selección y comentarios
    -   Fusión, suavizado de contorno y niebla
    -   Técnicas avanzadas

    Además, la *Guía de programación de OpenGL* contiene apéndices que describen la biblioteca de utilidades de OpenGL y la biblioteca auxiliar de la guía de programación de OpenGL.

-   El tercer elemento de documentación es esta información general. Describe la implementación de Windows de OpenGL y proporciona información general sobre sus componentes. Describe los conceptos de representación de contextos, formatos de píxeles y búferes. las funciones WGL que conectan OpenGL a los sistemas de ventanas de Windows; y las funciones de Windows que admiten formatos de píxel por ventana y doble búfer de ventanas para ventanas de gráficos OpenGL. Las funciones WGL y las funciones son específicas de la implementación de Windows de OpenGL.

-   El cuarto elemento de documentación es el conjunto de páginas de referencia para las funciones de WGL, las funciones de Windows que se acaban de mencionar y la estructura de datos [**PIXELFORMATDESCRIPTOR**](/windows/win32/api/wingdi/ns-wingdi-pixelformatdescriptor) .

-   El quinto elemento de documentación es la guía de migración. Describe cómo trasladar el código OpenGL existente desde otros entornos a Windows.

Los dos primeros elementos son los libros de OpenGL oficiales; no se incluyen en el Microsoft Windows SDK.
