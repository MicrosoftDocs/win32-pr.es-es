# <a name="rectfrectf-constructors"></a>Constructores RectF. RectF

En este tema se enumeran los constructores de la clase [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) . Para obtener una lista completa de clases, consulte **clase RectF**.

### <a name="overload-list"></a>Lista de sobrecarga



| Constructor                                                                        | Descripción                                                                                                                                                                                                                                                                                                                     |
|:-----------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**RectF (PointF&, SizeF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inconstpointf__inconstsizef_))         | Crea un objeto [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) mediante un objeto [**PointF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-pointf) para inicializar los miembros de datos **X** e y y usa un objeto [**SizeF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-sizef) para inicializar los miembros **de datos de** **ancho** y **alto** de este rectángulo.<br/> |
| [**RectF (REAL, REAL, REAL, REAL)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inreal_inreal_inreal_inreal)) | Crea un objeto [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) con cuatro enteros para inicializar los miembros de datos **X**, **y**, **ancho** y **alto** .<br/>                                                                                                                                                          |
| [**RectF()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-rectf(inconstpointf__inconstsizef_))                                     | Crea un objeto [**RectF**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-rectf) e inicializa los miembros de datos **X**, **y**, **ancho** y **alto** en cero. Éste es el constructor predeterminado.<br/>                                                                                                                                      |
