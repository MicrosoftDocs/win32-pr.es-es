# <a name="codecapi_avencadaptivemode-property"></a>\_Propiedad AVEncAdaptiveMode de CODECAPI

Especifica el modo de codificación adaptable para un codificador de vídeo.

## <a name="data-type"></a>Tipo de datos

**ULong** (VT \_ UI4)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncAdaptiveMode**

## <a name="property-value"></a>Valor de propiedad

El valor de esta propiedad es una operación OR bit a bit de los **valores de la** enumeración [**eAVEncAdaptiveMode**](/windows/win32/api/codecapi/ne-codecapi-eavencadaptivemode) .
