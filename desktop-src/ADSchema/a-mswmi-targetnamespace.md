# <a name="ms-wmi-targetnamespace-attribute"></a>atributo MS-WMI-TargetNameSpace

Espacio de nombres en el que se va a crear el objeto.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-WMI-TargetNameSpace                      |
| Nombre para mostrar de LDAP | msWMI-TargetNameSpace                       |
| Tamaño              | Menos de veinte caracteres.                |
| Actualizar privilegio  | Administrador de directiva de grupo                  |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1646                     |
| System-ID-GUID    | 1c4ab61f-3420-44e5-849d-8b5dbf60feb7        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
