# <a name="d2d1_matrix_5x4_f"></a>MATRIZ D2D1 \_ \_ 5X4 \_ F

Representa una matriz de 5 por 4.


```C++
typedef D2D_MATRIX_5X4_F D2D1_MATRIX_5X4_F;
```



## <a name="remarks"></a>Comentarios

**D2D1 \_ MATRIX \_ 5X4** es un nuevo nombre para la estructura [**\_ \_ MATRIZ 5X4 \_ F de D2D.**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f) Para obtener una lista de los campos proporcionados por la matriz, vea [**D2D \_ MATRIX \_ 5X4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f).
