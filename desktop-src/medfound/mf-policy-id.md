# <a name="mf_policy_id-attribute"></a>\_Atributo de \_ ID. de directiva MF

Identificador que se puede establecer en un [IMFOutputPolicy](/windows/win32/api/mfidl/nn-mfidl-imfoutputpolicy).

## <a name="data-type"></a>Tipo de datos

**UINT32** (se trata como **bool**)

## <a name="remarks"></a>Observaciones

El valor se puede recibir desde el evento [MEPolicySet](./mepolicyset.md) media. Se almacena como una carga de tipo **VT_UI4** en el evento **MEPolicySet** .
