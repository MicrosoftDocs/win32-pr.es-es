# <a name="systemcategory"></a>System. Category

En desuso. La categoría que se puede asignar a un elemento, como un documento o un archivo. Esta propiedad se hereda de las propiedades de documento OLE y está en desuso para Windows Vista. En su lugar, se deben usar palabras clave. El código anterior trata esta propiedad como VT \_ LPSTR.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Category
   shellPKey = PKEY_Category
   formatID = D5CDD502-2E9C-101B-9397-08002B2CF9AE
   propID = 2
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = Multivalue String
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
