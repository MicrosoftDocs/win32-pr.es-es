# <a name="spdif-audio-output"></a>Salida de audio S/PDIF

Algunos dispositivos de audio usan el formato de interconexión digital de Sony/Philips (S/PDIF) para transportar datos de audio. Ahora puede descodificar audio a S/PDIF desde audio codificado con el códec de Windows Media Professional.
