# <a name="win32-text-api-comparison"></a>Comparación de la API de texto de Win32

Para los desarrolladores que están migrando su código de aplicación de Win32, en la tabla siguiente se enumeran las API de texto de Win32 y el equivalente aproximado en DirectWrite.

|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

| [**AddFontResource**](/windows/win32/api/wingdi/nf-wingdi-addfontresourcea)                 | Agrega un recurso de fuente a la tabla de fuentes del sistema.                                                                        | No hay ningún equivalente. [**AddFontResource**](/windows/win32/api/wingdi/nf-wingdi-addfontresourcea) se agrega a la colección de fuentes del sistema como parte del paso de instalación de fuentes de GDI. DirectWrite rellena la colección de fuentes del sistema por adelantado y supervisa automáticamente las actualizaciones, por lo que no se necesita ninguna instalación.                                                                              |
| [**AddFontResourceEx**](/windows/win32/api/wingdi/nf-wingdi-addfontresourceexa)             | Agrega una fuente privada o no Enumerable a la tabla de fuentes del sistema.                                                       | Lo mismo que antes.                                                                                                                                                                                                                                                                                                                                                       |
