# <a name="windows-virtual-pc"></a>Windows Virtual PC

\[Windows Virtual PC ya no está disponible para su uso a partir de Windows 8. En su lugar, use el [proveedor de WMI de Hyper-V (V2)](../hyperv_v2/windows-virtualization-portal.md).\]

## <a name="purpose"></a>Propósito

En esta documentación se proporciona información acerca de la interfaz COM de Windows Virtual PC.

## <a name="where-applicable"></a>Donde sea aplicable

Windows Virtual PC es la última tecnología de virtualización de Microsoft; permite ejecutar muchas aplicaciones de productividad en un entorno de Windows virtual, con un solo clic, directamente desde Windows 7. Puede crear máquinas virtuales independientes encima de su escritorio de Windows 7. Cada máquina virtual emula un sistema de hardware completo, desde el procesador a la tarjeta de red, en un entorno de software independiente y aislado, lo que permite el funcionamiento simultáneo de sistemas incompatibles de otro modo.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Las interfaces COM de Windows Virtual PC son para desarrolladores que crean aplicaciones cliente que automatizan la implementación y el funcionamiento de las máquinas virtuales.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

Windows Virtual PC requiere una de las siguientes ediciones de Windows 7: Home Basic, Home Premium, Professional, Ultimate o Enterprise Edition.

Para obtener los archivos de encabezado y de biblioteca relacionados, instale el Windows SDK para Windows 7 desde el [centro de descarga de Microsoft](https://www.microsoft.com/download/details.aspx?id=8279).
