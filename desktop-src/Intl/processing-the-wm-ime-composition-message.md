# <a name="processing-the-wm_ime_composition-message"></a><span data-ttu-id="e1bd0-103">Procesar el mensaje de composición del \_ IME de WM \_</span><span class="sxs-lookup"><span data-stu-id="e1bd0-103">Processing the WM\_IME\_COMPOSITION Message</span></span>

<span data-ttu-id="e1bd0-104">Una aplicación compatible con IME que procesa el mensaje de [**\_ \_ composición del IME de WM**](wm-ime-composition.md) comprueba los bits en el parámetro *lParam* y llama a la función [**ImmGetCompositionString**](/windows/desktop/api/Imm/nf-imm-immgetcompositionstringa) para recuperar la cadena o los datos indicados.</span><span class="sxs-lookup"><span data-stu-id="e1bd0-104">An IME-aware application that processes the [**WM\_IME\_COMPOSITION**](wm-ime-composition.md) message tests the bits in *lParam* parameter and calls the [**ImmGetCompositionString**](/windows/desktop/api/Imm/nf-imm-immgetcompositionstringa) function to retrieve the indicated string or data.</span></span> <span data-ttu-id="e1bd0-105">En el siguiente ejemplo se comprueba la cadena de resultado, se asigna memoria suficiente para la cadena y se recupera la cadena de resultado del IME.</span><span class="sxs-lookup"><span data-stu-id="e1bd0-105">The following example checks for the result string, allocates sufficient memory for the string, and retrieves the result string from the IME.</span></span>


```C++
HIMC hIMC;
DWORD dwSize;
HGLOBAL hstr;
LPSTR lpstr;

case WM_IME_COMPOSITION:
    if (lParam & GCS_RESULTSTR) 
    {
        hIMC = ImmGetContext(hWnd);

        if (!hIMC)
            MyError(ERROR_NULLCONTEXT);

        // Get the size of the result string. 
        dwSize = ImmGetCompositionString(hIMC, GCS_RESULTSTR, NULL, 0);

        // increase buffer size for terminating null character,  
        //   maybe it is in UNICODE 
        dwSize += sizeof(WCHAR);

        hstr = GlobalAlloc(GHND,dwSize);
        if (hstr == NULL)
             MyError(ERROR_GLOBALALLOC);

        lpstr = (LPSTR)GlobalLock(hstr);
        if (lpstr == NULL)
             MyError(ERROR_GLOBALLOCK);

        // Get the result strings that is generated by IME into lpstr. 
        ImmGetCompositionString(hIMC, GCS_RESULTSTR, lpstr, dwSize);
        ImmReleaseContext(hWnd, hIMC);

        // add this string into text buffer of application 

        GlobalUnlock(hstr);
        GlobalFree(hstr);
    }
```
