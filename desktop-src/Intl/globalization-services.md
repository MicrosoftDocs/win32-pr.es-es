# <a name="globalization-services"></a>Servicios de globalización

Los servicios de globalización exponen compatibilidad con la globalización básica para idiomas y regiones que permiten la creación de aplicaciones globalizadas que se aplican a una amplia gama de clientes internacionales. Esta tecnología proporciona la infraestructura de globalización para admitir una experiencia de usuario de Windows coherente y lingüísticamente adecuada. Los servicios de globalización incluyen las siguientes tecnologías:

-   [Servicios lingüísticos extendidos](extended-linguistic-services.md). Proporciona una variedad de servicios basados en texto para desarrolladores de Windows (incluida la detección de lenguajes, la transliteración y otras funciones lingüísticas principales) y la plataforma e interfaces para que las aplicaciones consuman los servicios con facilidad, así como las interfaces que permiten a los desarrolladores agregar servicios propios.
-   [Componentes internacionales para Unicode (ICU)](international-components-for-unicode--icu-.md). Proporciona un conjunto de API de globalización de código abierto de uso generalizado. ICU emplea el amplio repositorio de datos de configuración regional común (CLDR) de Unicode, como la biblioteca de datos, que proporciona compatibilidad de globalización para las aplicaciones de software. Las ICU son muy portátiles y proporcionan a las aplicaciones los mismos resultados en todas las plataformas.
-   [Compatibilidad con el idioma nacional](national-language-support.md). Proporciona datos y API internacionales para admitir el formato de fecha, hora y calendario, distribuciones del teclado, comportamiento de ordenación, aspectos de la habilitación del texto y otra información básica de la configuración regional.
-   [API](spell-checker-api.md)de revisión ortográfica. Proporciona una API para que los desarrolladores consuman la funcionalidad de revisión ortográfica para comprobar el texto, obtener sugerencias y mantener la configuración y los diccionarios de usuario. Los desarrolladores también pueden crear comprobadores ortográficos que se integran en el marco de revisión ortográfica como proveedores.
