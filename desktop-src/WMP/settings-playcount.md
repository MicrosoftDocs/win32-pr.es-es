# <a name="settingsplaycount"></a>Settings. playCount

La propiedad **playCount** especifica o recupera el número de veces que se reproducirá un elemento multimedia.

## <a name="syntax"></a>Sintaxis

Player. Settings. playCount

## <a name="possible-values"></a>Valores posibles

Esta propiedad es un **número** de lectura y escritura (**Long**) con un valor mínimo de uno y un valor predeterminado de uno.

**Windows Media Player 10 Mobile**: no se admite esta propiedad.
