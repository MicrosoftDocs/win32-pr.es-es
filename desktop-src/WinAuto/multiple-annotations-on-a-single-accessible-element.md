# <a name="multiple-annotations-on-a-single-accessible-element"></a>Varias anotaciones en un único elemento accesible

Actualmente, un elemento accesible solo admite una anotación por propiedad. Las anotaciones sucesivas de una propiedad reemplazan e invalidan las anotaciones anteriores de esa propiedad.
