# <a name="udm_getbuddy-message"></a>\_Mensaje GETBUDDY UDM

Recupera el identificador de la ventana relacionada actual.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>Debe ser cero.</dd> <dt>

*lParam* 
</dt> <dd>Debe ser cero.</dd> </dl>

## <a name="return-value"></a>Valor devuelto

El valor devuelto es el identificador de la ventana relacionada actual.
