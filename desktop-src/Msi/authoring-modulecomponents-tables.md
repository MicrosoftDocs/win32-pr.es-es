# <a name="authoring-modulecomponents-tables"></a>Creación de tablas ModuleComponents

Enumera todos los componentes entregados por el módulo de combinación en la [tabla ModuleComponents](modulecomponents-table.md). Tenga en cuenta que los componentes del módulo de combinación también se especifican en la tabla de componentes. Para obtener más información, vea [crear tablas de componentes de módulo de combinación](authoring-merge-module-component-tables.md).
