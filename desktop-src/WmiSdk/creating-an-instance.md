# <a name="creating-an-instance"></a>Creación de una instancia

Una instancia de es un caso de una clase que describe un objeto determinado en la empresa. Por ejemplo, si tiene dos unidades de CD-ROM en una plataforma de escritorio, podría usar dos instancias de la clase [**\_ CDROMDrive de Win32**](/windows/desktop/CIMWin32Prov/win32-cdromdrive) para describir ambas. Dado que una instancia representa una propiedad real de la empresa, es más probable que cree más instancias que clases. Cada clase a partir de la que se crea una instancia tiene una o varias propiedades clave. No se puede crear una instancia con más de 256 claves.

Para obtener más información sobre cómo crear instancias, vea los temas siguientes:

-   [Creación y declaración de una instancia mediante C++](creating-and-declaring-an-instance-using-c-.md)
-   [Creación de una instancia mediante MOF](creating-an-instance-using-mof.md)
-   [Crear un objeto mediante VBScript](creating-an-object-using-vbscript.md)
