# <a name="tablet-pc"></a>Tablet PC

Tablet PC es un equipo personal totalmente funcional orientado a aplicaciones habilitadas para lápiz, escritura a mano y habilitadas para voz. La combinación de software y hardware en una tableta PC habilita estos métodos de interacción del usuario y permite una experiencia informática enriquecente, interactiva y productiva para los usuarios.

La tecnología tablet PC de Windows permite la entrada y salida de datos de escritura a mano y voz en un tablet PC, así como el intercambio de estos datos con otros equipos.

Windows Touch, introducido en Windows 7, permite gestos multitáctil y una experiencia táctil mejorada para los usuarios.
