# <a name="namedqueries-metadatatype-element"></a>elemento namedQueries (MetadataType)

No se usa. Define una lista de consultas con nombre que consultan un valor en la cadena del mensaje de evento y realizan una acción especificada si se encuentran.

``` syntax
<xs:element name="namedQueries"
    type="NamedQueryType"
 />
```

El **elemento namedQueries** se define mediante el [**tipo complejo MetadataType.**](eventmanifestschema-metadatatype-complextype.md)
