# <a name="wpd_command_mtp_ext_get_supported_vendor_opcodes-command"></a>Comando de WPD comando \_ \_ ext de proveedor de \_ \_ \_ \_ códigos de descompatibilidad \_

El comando de WPD de comando de la extensión de código de desbloqueo de comandos de la **\_ extensión de \_ \_ \_ \_ \_ código \_** de la extensión de comando No hay ninguna fase de datos subsiguiente asociada a este comando.

## <a name="command-category"></a>Categoría de comando

**Categoría de WPD de \_ \_ \_ operaciones de proveedor ext MTP \_ \_**

## <a name="parameters"></a>Parámetros

Este comando no contiene parámetros.

## <a name="return-value"></a>Valor devuelto

El controlador devuelve los resultados siguientes.



| Resultado                                                | VarType | Descripción                                                                                              |
|-------------------------------------------------------|---------|----------------------------------------------------------------------------------------------------------|
| **propiedad de WPD \_ \_ \_ códigos de \_ operación ext VENDOR \_ \_ MTP** | VT \_ UI4 | Obligatorio. Un **IPortableDevicePropVariantCollection** que contiene todos los códigos de operación ampliados por el proveedor. |



 

## <a name="calling-methods"></a>Llamar a métodos

Solo se puede llamar directamente mediante [**IPortableDevice:: SendCommand**](/windows/desktop/api/PortableDeviceApi/nf-portabledeviceapi-iportabledevice-sendcommand).
