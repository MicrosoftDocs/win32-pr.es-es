# <a name="ms-ds-top-quota-usage-attribute"></a>atributo de uso de cuota de MS-DS-Top

La lista de usuarios de cuota superior actualmente en la base de datos de directorio, ordenada por el uso de cuota decreciente.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-DS-Top-quota-uso                       |
| Nombre para mostrar de LDAP | msDS-TopQuotaUsage                          |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1850                     |
| System-ID-GUID    | 7b7cce4f-f1f5-4bb6-b7eb-23504af19e75        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
