# <a name="temporary-creation-flags"></a>Marcas de creación temporales

La marca de creación temporal indica que el archivo subyacente se destruye automáticamente cuando se libera el objeto de almacenamiento raíz. Esta funcionalidad es muy útil para crear archivos temporales.
