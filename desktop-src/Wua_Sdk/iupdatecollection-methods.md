# <a name="iupdatecollection-methods"></a>Métodos IUpdateCollection

La interfaz [**IUpdateCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatecollection) define los siguientes métodos.



| Método                                         | Descripción                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Agréguela**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-add)           | Agrega un elemento a la colección.                                |
| [**Claridad**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-clear)       | Quita todos los elementos de la colección.                  |
| [**Copiar**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-copy)         | Crea una copia de lectura/escritura superficial de la colección.           |
| [**Introducir**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-insert)     | Inserta un elemento en la colección situada en la posición especificada. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatecollection-removeat) | Quita de la colección un elemento situado en el índice especificado.   |
