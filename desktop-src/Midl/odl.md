# <a name="odl-attribute"></a>odl (atributo)

MKTYPLIB requería el **\[ atributo odl \]** en las interfaces ODL. El compilador MIDL no requiere el **\[ atributo odl; \]** solo se reconoce por compatibilidad con archivos **ODL anteriores.**

> [!Note]  
> La Mktyplib.exe está obsoleta. En su lugar, use el compilador MIDL.
