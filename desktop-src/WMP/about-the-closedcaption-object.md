# <a name="about-the-closedcaption-object"></a>Acerca del objeto ClosedCaption

El **objeto ClosedCaption** rige la interfaz de subtítulos para Reproductor de Windows Media. El **objeto ClosedCaption** se obtiene a través de **la propiedad closedCaption** del objeto **Player.** Por ejemplo, para obtener el valor **de la propiedad SAMIFileName,** escriba lo siguiente:


```C++
myfile = player.closedcaption.SAMIFileName;

```
