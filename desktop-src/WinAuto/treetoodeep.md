# <a name="treetoodeep"></a>TreeTooDeep

## <a name="text"></a>Texto

El árbol tiene {0} niveles de profundidad.

## <a name="type"></a>Tipo

Advertencia

## <a name="description"></a>Descripción

El árbol de elementos puede ser demasiado profundo.

Este problema causa problemas para las personas que confían en un lector de pantalla y un teclado para la navegación porque parece que los elementos de recorrido van a tardar mucho tiempo y pueden aparecer cíclicos.

## <a name="possible-causes"></a>Causas posibles

-   Varios elementos o sus elementos primarios son controles personalizados que no implementan el tabulador correctamente. Por ejemplo, la propiedad estado de MSAA no se actualiza correctamente.
-   La interfaz de usuario de la aplicación es demasiado compleja.
