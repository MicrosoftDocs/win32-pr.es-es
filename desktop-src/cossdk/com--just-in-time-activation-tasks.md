En los temas siguientes se proporcionan instrucciones paso a paso para usar la activación Just-In-Time (JIT) en COM+.



| Tema                                                                                                       | Descripción                                                                                                                               |
|-------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [Habilitación de la activación JIT para un componente](enabling-jit-activation-for-a-component.md)<br/>           | Proporciona un procedimiento paso a paso para activar el servicio de activación JIT de COM+ para los componentes de COM+.<br/>                     |
| [Establecer el bit listo](setting-the-done-bit.md)<br/>                                                 | Proporciona información sobre cómo establecer el bit listo para los componentes activados por JIT.<br/>                                             |
