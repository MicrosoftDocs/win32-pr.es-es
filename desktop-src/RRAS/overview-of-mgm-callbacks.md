# <a name="overview-of-mgm-callbacks"></a>Información general sobre las devoluciones de llamada MGM

Hay dos tipos de devoluciones de llamada en la API MGM:

-   [Devoluciones de llamada de protocolo de enrutamiento](routing-protocol-callbacks.md)
-   [Habilitar y deshabilitar devoluciones de llamada de IGMP](igmp-enable-and-disable-callbacks.md)

Estas devoluciones de llamada, combinadas con las API MGM, permiten la comunicación continua entre los protocolos de enrutamiento, IGMP y el administrador del grupo de multidifusión.
