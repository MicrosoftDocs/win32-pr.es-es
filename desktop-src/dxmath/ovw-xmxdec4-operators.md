# <a name="xmxdec4-operators"></a>Operadores XMXDEC4

La estructura [**XMXDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdec4) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                            | Descripción                                                                                                           |
|--------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operador UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmxdec4-operatoruint32_t)<br/> | Devuelve una instancia de `uint32_t` que contiene los componentes de la `XMXDEC4` instancia en un formato empaquetado. <br/> |
| [**operador =**](xmxdec4-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMXDEC4` . <br/>                                       |
