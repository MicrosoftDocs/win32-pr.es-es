# <a name="objectcpp"></a>Objeto. Cpp

En el componente de proveedor de ejemplo, un ejemplo de código de listas de objetos, incluido el uso de un filtro, está en Object.cpp. Los métodos admitidos se enumeran en la tabla siguiente.



| Object                               | Descripción                                 |
|--------------------------------------|---------------------------------------------|
| **ObjectTypeList::ObjectTypeList**   | Creator: cree una lista mediante el filtro .     |
| **ObjectTypeList::~ObjectTypeList**  | Destructor estándar.                        |
| **ObjectTypeList::GetCurrentObject** | Recupere el objeto actual de la lista.  |
| **ObjectTypeList::Next**             | Suba el índice.                             |
| **ObjectTypeList::Reset**            | Restablezca el índice.                            |
| **IsValidFilter**                    | Compruebe la integridad del filtro.          |
| **BuildDefaultObjectArray**          | Cree una matriz de objetos mediante el filtro . |
| **BuildObjectArray**                 | Cree una matriz de objetos.                      |
