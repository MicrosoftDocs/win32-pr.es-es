# <a name="nismapentry-attribute"></a>atributo nisMapEntry

Contiene una entrada de mapa para un mapa no estándar.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | nisMapEntry                          |
| Nombre para mostrar de LDAP | nisMapEntry                          |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.3.6.1.1.1.1.27                     |
| System-ID-GUID    | 4a95216e-fcc0-402e-b57f-5971626148a9 |
| Sintaxis            | [**String(IA5)**](s-string-ia5.md)  |
