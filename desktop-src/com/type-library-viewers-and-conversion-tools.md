# <a name="type-library-viewers-and-conversion-tools"></a>Visores y herramientas de conversión de bibliotecas de tipos

Las bibliotecas de tipos contienen la especificación de uno o más elementos COM, incluidas clases, interfaces, enumeraciones, etc. Estos archivos se almacenan en un formato binario estándar. Una biblioteca de tipos puede ser un archivo independiente con la extensión de nombre de archivo. tlb o puede almacenarse como un recurso en un archivo ejecutable, que puede tener una extensión de nombre de archivo. ocx,. dll o. exe. Los visores y las herramientas de conversión de la biblioteca de tipos que se describen a continuación leen este formato para obtener información sobre los elementos COM de la biblioteca.

Para poder programar un objeto en un lenguaje de programación determinado, debe ser capaz de ver su biblioteca de tipos en ese idioma. Esto le proporciona la sintaxis adecuada para las clases, interfaces, métodos, propiedades y eventos del objeto COM.

Los productos de desarrollo de Microsoft proporcionan las siguientes herramientas que puede usar para generar, extraer y ver la información de la biblioteca de tipos.

-   [Visor de objetos OLE y COM](ole-com-object-viewer.md)
-   [Compilador MIDL](midl-compiler.md)
-   [MkTypLib](mktyplib-command-line-tool.md)

## <a name="visual-basic"></a>Visual Basic

-   [Visual Basic Examinador de objetos](visual-basic-object-browser.md)

## <a name="java"></a>Java

Para obtener información general sobre cómo estas herramientas interactúan con las bibliotecas de tipos, vea [cómo herramientas de desarrollo usar bibliotecas de tipos](how-developer-tools-use-type-libraries.md).
