# <a name="em_reconversion-message"></a>Mensaje DE \_ EM RECONVERSION

Invoca el cuadro de diálogo de reconversión del Editor de métodos de entrada (IME).

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>

No se usa; debe ser cero.

</dd> <dt>

*lParam* 
</dt> <dd>

No se usa; debe ser cero.

</dd> </dl>
