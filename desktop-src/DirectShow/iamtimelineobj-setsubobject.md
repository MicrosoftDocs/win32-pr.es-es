> [!Note]  
> \[En desuso. Esta API se puede quitar de las versiones futuras de Windows.\]

 

No se admite.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT SetSubObject(
   IUnknown *newVal
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*newVal* 
</dt> <dd>

Reservado.

</dd> </dl>
