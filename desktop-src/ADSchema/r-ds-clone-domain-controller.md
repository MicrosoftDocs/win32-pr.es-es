# <a name="ds-clone-domain-controller-extended-right"></a>DS-Clone-Domain-Controller Extended Right

Permita que un controlador de dominio cree un clon de sí mismo.



| Entrada | Value |
|--------------|----------------------------------------|
| CN           | DS-Clone-Domain-Controller             |
| Display-Name | Permitir a un controlador de dominio crear un clon de sí mismo |
| Rights-GUID  | 3e0f7e18-2c7a-4c10-ba82-4d926db99a3e   |
