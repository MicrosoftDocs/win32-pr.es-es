# <a name="wmp-sdk-samples"></a>Ejemplos del SDK de WMP

El SDK de Windows Media Player incluye una variedad de ejemplos que muestran muchas de las técnicas de programación descritas en este documento.

-   Los ejemplos de scripting muestran cómo incrustar el control ActiveX del reproductor en una página web. En un ejemplo completo de página web se muestra cómo detectar la versión actual del control ActiveX del reproductor.
-   Un ejemplo de máscaras muestra cómo cambiar la apariencia visual y el comportamiento de Windows Media Player.
-   Un asistente para Microsoft Visual C++ puede crear un ejemplo de trabajo único de varios tipos de complementos: un complemento de visualización, un complemento de la interfaz de usuario, un complemento de representación y un complemento de procesamiento de señal digital.
-   Los ejemplos de tiendas en línea incluyen un asistente para Microsoft Visual C++ que puede crear un único ejemplo de trabajo de un objeto COM de tienda en línea.
-   Los ejemplos de Microsoft Visual C++ muestran la incrustación del control ActiveX de Windows Media Player, la comunicación remota del control ActiveX del reproductor y el uso de la biblioteca.
-   Un ejemplo de C# muestra cómo utilizar el control ActiveX de Windows Media Player con el .NET Framework.
-   En un ejemplo de .NET Visual Basic se muestra el uso del control ActiveX de Windows Media Player con la .NET Framework.

Para usar los ejemplos, debe instalar el Windows SDK, que incluye el SDK de Windows Media Player. Para descargar el Windows SDK, vaya a la página [Windows SDK](https://msdn.microsoft.com/windows/bb980924.aspx) del centro para desarrolladores de Windows.
