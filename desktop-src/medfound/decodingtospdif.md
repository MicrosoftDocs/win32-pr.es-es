# <a name="decoding-to-spdif-microsoft-media-foundation"></a>Codificación en S/PDIF (Microsoft Media Foundation)

El identificador de clase para el objeto descodificador que transforma el contenido de Professional media audio 9.1 de Windows para su entrega a través del formato de interconexión digital (S/PDIF) es CLSID \_ CWMAudioSpdTxDMO. Este objeto se usa como lo haría con cualquier otro descodificador. Admite las mismas propiedades que el descodificador Windows Media Audio 9 . Los tipos de salida enumerados por este descodificador son todos formatos S/PDIF.

Para admitir completamente la salida de S/PDIF, el equipo cliente debe tener instalado hardware especial.
