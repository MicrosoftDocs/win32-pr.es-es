No se admite este método.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT MoveEverythingBy2(
   REFTIME Start,
   REFTIME MoveBy
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*Iniciar* 
</dt> <dd>

Reservado.

</dd> <dt>

</dt> <dd>
