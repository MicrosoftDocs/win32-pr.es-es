# <a name="com-object-constructor-strings"></a>Cadenas del constructor de objetos COM+

Las cadenas del constructor de objetos COM+ admiten la programación genérica. Permiten escribir un único componente con un grado de generalidad para que se pueda personalizar para una tarea determinada mediante cadenas de constructor especificadas de forma administrativa.
