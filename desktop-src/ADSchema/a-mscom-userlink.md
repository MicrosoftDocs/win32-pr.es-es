# <a name="ms-com-userlink-attribute"></a>atributo MS-COM-UserLink

Vínculo que se usa para asociar un PartitionSet COM+ a un objeto de usuario.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-COM-UserLink                         |
| Nombre para mostrar de LDAP | msCOM-UserLink                          |
| Tamaño              | \-                                      |
| Actualizar privilegio  | El sistema establece este valor.        |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1425                 |
| System-ID-GUID    | 9e6f3a4d-242c-4f37-b068-36b57f9fc852    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
