# <a name="ms-ds-nc-repl-outbound-neighbors-attribute"></a>atributo MS-DS-NC-REPL-Outbound-Neighbors

Asociados de replicación para esta partición. Este servidor envía datos de replicación a estos otros servidores, que actúan como destinos. Este servidor notificará a estos otros servidores cuando haya nuevos datos disponibles.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-DS-NC-REPL-Outbound-Neighbors            |
| Nombre para mostrar de LDAP | msDS-NCReplOutboundNeighbors                |
| Tamaño              | \-                                          |
| Actualizar privilegio  | El sistema establece este valor.            |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1706                     |
| System-ID-GUID    | 855f2ef5-a1c5-4cc4-ba6d-32522848b61f        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
