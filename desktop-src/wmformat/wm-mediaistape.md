# <a name="wmmediaistape"></a>WM/MediaIsTape

El **atributo WM/MediaIsTape** indica si el contenido se difundió desde una cinta registrada previamente.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaIsTape

## <a name="data-type"></a>Tipo de datos

**WMT \_ TYPE \_ BOOL**

## <a name="remarks"></a>Comentarios

Este atributo se usa para la televisión de difusión.
