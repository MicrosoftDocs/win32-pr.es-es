# <a name="using-system-profiles"></a>Usar perfiles del sistema

Los perfiles del sistema que se incluyen con el SDK de Windows Media Format se diseñaron para cubrir las necesidades básicas de contenido multimedia digital que se compone de audio y vídeo. Debido a la gran variedad de características disponibles y al aumento relacionado en posibles escenarios de medios digitales, los perfiles del sistema se han desacentuado. El conjunto más reciente de perfiles del sistema hace uso de los códecs de Windows Media de la versión 8 que se lanzaron con Windows XP. A pesar de no admitir la versión más reciente de los códecs, los perfiles del sistema suponen un excelente punto de partida para crear perfiles que se adapten a sus necesidades.

En los temas siguientes se describe cómo usar los perfiles del sistema como base para los archivos ASF.
