## <a name="component-and-template-types"></a>Tipos de componentes y plantillas

Las declaraciones de función intrínseca de HLSL usan tipos de componentes y tipos de plantilla para los argumentos de parámetros de entrada y valores devueltos. En la tabla siguiente se enumeran los tipos disponibles.



| Estos tipos de plantilla | Descripción                                           | Compatibilidad con estos tipos de datos                                        |
|----------------------|-------------------------------------------------------|-----------------------------------------------------------------|
| matriz               | hasta 16 componentes en función de la declaración      | [Tipos de HLSL básicos](dx-graphics-hlsl-data-types.md)             |
| object               | objeto de muestra                                        | *muestra*, *sampler1D*, *sampler2D*, *sampler3D*, *samplerCUBE* |
| escalar               | 1 componente                                           | [Tipos de HLSL básicos](dx-graphics-hlsl-data-types.md)             |
| vector               | 1 componente como mínimo, 4 componentes como máximo (inclusivo) | [Tipos de HLSL básicos](dx-graphics-hlsl-data-types.md)             |
