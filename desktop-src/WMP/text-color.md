# <a name="text-color"></a>Color del texto

Se debe especificar el valor de color del tipo. El valor debe especificarse como tres enteros positivos separados por comas. Los valores representan la cantidad de rojo, verde y azul para un color de mapa de bits y oscilan entre 0 y 255.
