### <a name="step-1-provide-the-root-iaccessible-interface-on-behalf-of-the-windowless-control"></a><span data-ttu-id="574ed-118">Paso 1: proporcionar la interfaz IAccessible raíz en nombre del control sin ventana.</span><span class="sxs-lookup"><span data-stu-id="574ed-118">Step 1: Provide the root IAccessible interface on behalf of the windowless control.</span></span>
