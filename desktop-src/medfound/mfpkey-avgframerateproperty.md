# <a name="mfpkey_avgframerate-property"></a>\_Propiedad AVGFRAMERATE de MFPKEY

Especifica la velocidad media de fotogramas del contenido de vídeo, en fotogramas por segundo.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

g \_ wszWMVCAvgFrameRate

## <a name="data-type"></a>Tipo de datos
