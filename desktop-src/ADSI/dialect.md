# <a name="dialect"></a>Dialect

Dos de las interfaces ADSI para buscar en un directorio, ADO y OLE DB, utilizan una cadena de búsqueda para especificar los parámetros de la búsqueda. Los parámetros de búsqueda incluyen la base de la búsqueda, el filtro de búsqueda, los atributos que se van a recuperar y la profundidad de la búsqueda.

Puede especificar la cadena de búsqueda en uno de dos dialectos:

-   [Dialecto LDAP](ldap-dialect.md)
-   [Dialecto SQL](sql-dialect.md)
