# <a name="additional-files-required-to-generate-a-type-library"></a>Archivos adicionales necesarios para generar una biblioteca de tipos

Para compilar un archivo IDL que contiene una instrucción de biblioteca, los archivos DLL de automatización OLE y OLE deben estar en el sistema. Estos archivos se instalan automáticamente.
