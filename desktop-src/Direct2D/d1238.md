# <a name="d1238-source-transform-single-channel-srgb"></a>D1238: SRGB de canal único de transformación de origen

ID2D1SourceTransform no permite que D2D1 BUFFER PRECISION SRGB se \_ combine con \_ \_ D2D1 \_ CHANNEL DEPTH \_ \_ 1.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación especificó D2D1 BUFFER PRECISION SRGB y \_ \_ \_ D2D1 CHANNEL DEPTH 1 en \_ un \_ \_ ID2D1SourceTransform.

## <a name="possible-fixes"></a>Posibles correcciones

En su lugar, se deben usar \_ D2D1 CHANNEL DEPTH DEFAULT o \_ \_ D2D1 \_ CHANNEL DEPTH \_ \_ 4.
