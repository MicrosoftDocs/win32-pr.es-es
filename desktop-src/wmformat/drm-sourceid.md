# <a name="drm_sourceid"></a>Endrm \_ SourceID

El **atributo \_ SourceID de DRM** contiene un identificador que asocia contenido protegido con un origen.

## <a name="global-constant"></a>Constante global

g \_ wszWMDRM \_ SourceID

## <a name="data-type"></a>Tipo de datos

**tipo de WMT \_ \_ DWORD**

## <a name="remarks"></a>Observaciones

Los identificadores de origen se utilizan para diferenciar el contenido que está protegido de forma no estándar del contenido protegido mediante el uso de DRM de Windows Media estándar.
