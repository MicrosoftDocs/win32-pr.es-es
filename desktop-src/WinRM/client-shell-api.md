# <a name="winrm-client-shell-api"></a>API de Shell de cliente de WinRM

La interfaz de programación de aplicaciones (API) del Shell de cliente de Windows Remote Management (WinRM) proporciona funcionalidad para crear y administrar shells y operaciones de shell, comandos y flujos de datos en equipos remotos.

La sección referencia de la API de Shell de cliente de WinRM contiene información detallada sobre los siguientes elementos de API:

-   [Funciones adicionales de la API de Shell de cliente](client-shell-api-additional-functions.md)
-   [Enumeraciones de API de Shell de cliente](client-shell-api-enumerations.md)
-   [Estructuras y definiciones de la API de Shell de cliente](client-shell-api-structures.md)
