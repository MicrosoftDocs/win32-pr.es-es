# <a name="d1255-tile-opacity-metadata-conflict"></a>D1255: Conflicto de metadatos de opacidad de mosaico

El rectángulo opaco calculado para un gráfico de efectos superpuesto a una región marcada como transparente. Esto puede ocurrir debido al uso incorrecto de un efecto Atlas o de una transformación de ajuste de límites.
