# <a name="displayable-address"></a>Dirección que se pueda mostrar

La dirección que se pueda mostrar es una cadena que presenta la dirección en un formato adecuado para su presentación en una interfaz de usuario.

**TAPI 2. x:** Vea [**lineGetCallInfo**](/windows/win32/api/tapi/nf-tapi-linegetcallinfo), **dwDisplayableAddressSize** y **dwDisplayableAddressOffset** miembros de [**LINECALLINFO**](/windows/win32/api/tapi/ns-tapi-linecallinfo).

**TAPI 3. x:** Vea [**ITCallInfo:: get \_ CallInfoString**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-get_callinfostring) o [**ITCallInfo::p UT \_ CallInfoString**](/windows/desktop/api/tapi3if/nf-tapi3if-itcallinfo-put_callinfostring), llamado con el miembro **CIS \_ DISPLAYABLEADDRESS** de [**CALLINFO \_ cadena**](/windows/desktop/api/Tapi3if/ne-tapi3if-callinfo_string).
