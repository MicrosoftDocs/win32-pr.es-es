# <a name="resourcelocatorclearoptions-method"></a>ResourceLocator. ClearOptions, método

Quita todas [*las opciones*](windows-remote-management-glossary.md) del objeto [**ResourceLocator**](resourcelocator.md) . Puede proporcionar un objeto [**ResourceLocator**](resourcelocator.md) en lugar de especificar un URI de recurso en operaciones de objetos de [**sesión**](session.md) como [**Session. Get**](session-get.md), [**Session. put**](session-put.md)o [**Session. Enumerate**](session-enumerate.md).

## <a name="syntax"></a>Sintaxis


```VB
ResourceLocator.ClearOptions()
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="remarks"></a>Observaciones

**IWSManResourceLocator:: ClearOptions** es el método de C++ correspondiente.
