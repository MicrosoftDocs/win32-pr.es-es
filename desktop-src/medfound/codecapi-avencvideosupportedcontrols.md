# <a name="codecapi_avencvideosupportedcontrols-property"></a>\_Propiedad AVEncVideoSupportedControls de CODECAPI

Establece los controles admitidos para un codificador de vídeo.

## <a name="data-type"></a>Tipo de datos

**UINT64 (VT \_ UI8)**

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncVideoSupportedControls**

## <a name="remarks"></a>Observaciones

Esta propiedad solo se usa con [codificadores de cámara H. 264 UVC 1,5](camera-encoder-h264-uvc-1-5.md).
