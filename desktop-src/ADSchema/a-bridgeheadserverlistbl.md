# <a name="bridgehead-server-list-bl-attribute"></a>Cabeza de puente: atributo Server-List-BL

La lista de servidores que son cabezas de puente para la replicación.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | Cabeza de puente-servidor-lista-BL               |
| Nombre para mostrar de LDAP | bridgeheadServerListBL                  |
| Tamaño              | \-                                      |
| Actualizar privilegio  | El sistema establece este valor.        |
| Frecuencia de actualización  | Cada vez que se configura un sitio.               |
| Attribute-Id      | 1.2.840.113556.1.4.820                  |
| System-ID-GUID    | d50c2cdb-8951-11d1-aebc-0000f80367c1    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
