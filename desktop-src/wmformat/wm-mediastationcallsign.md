# <a name="wmmediastationcallsign"></a>WM/MediaStationCallSign

El atributo **WM/MediaStationCallSign** contiene el signo de llamada de la estación que difunde el contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaStationCallSign

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Este atributo se usa para el televisor y la radio de difusión.
