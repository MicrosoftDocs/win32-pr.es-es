# <a name="texture3doperator--function"></a>Función Texture3D::Operator

Devuelve una variable de recurso de solo lectura.

## <a name="syntax"></a>Sintaxis

``` syntax
R Operator[](
  in uint3 pos
);
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

*pos* \[ En\]
</dt> <dd>

Tipo: **uint3**

Posición del índice. Contiene las coordenadas (x, y, z).

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Tipo: **R**

Una variable de recurso de solo lectura.

## <a name="remarks"></a>Comentarios

Este método siempre tiene acceso al primer nivel de mip. Para especificar otros niveles de mip, use en su [**lugar \[ \] \[ \] mip.operator.**](sm5-object-texture3d-mipsoperatorindex.md)

Esta función es compatible con los siguientes tipos de sombreadores:



| Vértice | Casco | Domain | Geometría | Píxel | Compute |
|--------|------|--------|----------|-------|---------|
| x      | x    | x      | x        | x     | x       |
