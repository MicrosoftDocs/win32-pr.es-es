# <a name="attribute-security-guid-attribute"></a>Atributo-Security-GUID (atributo)

GUID que se va a utilizar para aplicar las credenciales de seguridad a un conjunto de objetos.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | Attribute-Security-GUID                               |
| Nombre para mostrar de LDAP | attributeSecurityGUID                                 |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | Administrador de esquema                                  |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.149                                |
| System-ID-GUID    | bf967924-0de6-11d0-a285-00aa003049e2                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
