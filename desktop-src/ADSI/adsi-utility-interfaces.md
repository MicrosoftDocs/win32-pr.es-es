# <a name="adsi-utility-interfaces"></a>Interfaces de la utilidad ADSI

ADSI proporciona varias interfaces de utilidad para objetos ADSI que pueden ayudar en el acceso y el mantenimiento de la Active Directory. Estas interfaces son:

-   [**IADsDeleteOps**](iadsdeleteops-interface.md)
-   [**IADsObjectOptions**](iadsobjectoptions-interface.md)
-   [**IADsPathname**](iadspathname-interface.md)
-   [**IADsNameTranslate**](iadsnametranslate-interface.md)
