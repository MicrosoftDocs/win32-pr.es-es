# <a name="what-is-text-services-framework"></a>¿Qué es Text Services Framework?

Text Services Framework (TSF) es un sistema extensible, independiente del idioma y independiente del dispositivo para las tecnologías de entrada de texto y lenguaje.

Las aplicaciones habilitadas para TSF pueden recibir entradas de texto de cualquier servicio de texto que admita TSF sin tener que tener en cuenta ningún detalle del origen de texto.

Los servicios de texto suministran texto a las aplicaciones de una manera que no requiere ningún conocimiento detallado de las aplicaciones que producen o reciben el texto. Por ejemplo, un servicio de texto puede proporcionar entrada de texto desde dispositivos de entrada de voz o escritura a mano.
