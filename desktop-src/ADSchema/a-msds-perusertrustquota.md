# <a name="ms-ds-per-user-trust-quota-attribute"></a>Atributo MS-DS-Per-User-Trust-Quota

Se usa para aplicar una cuota por usuario para crear objetos Trusted-Domain autorizados por el nuevo derecho de acceso de control, Create-Inbound-Forest-Trust. Este atributo limita el número de objetos Trusted-Domain que puede crear un único usuario que no es administrador.



| Entrada | Valor |
|-------------------|-------------------------------------------|
| CN                | MS-DS-per-user-trust-quota                |
| Ldap-Display-Name | msDS-PerUserTrustQuota                    |
| Size              | \-                                        |
| Privilegio actualizar  | Administrador de dominio                      |
| Frecuencia de actualización  | En la creación del bosque y rara vez después de eso. |
| Attribute-Id      | 1.2.840.113556.1.4.1788                   |
| System-Id-Guid    | d161adf0-ca24-4993-a3aa-8b2c981302e8      |
| Syntax            | [**Enumeración**](s-enumeration.md)      |
