# <a name="lsa-user-logon-authentication"></a>Autenticación de inicio de sesión de usuario LSA

El LSA controla el inicio de sesión y la autenticación del usuario en el equipo local, y si el paquete de autenticación que procesa la solicitud de inicio de sesión admite la autenticación de paso a través, el LSA también puede iniciar sesión a los usuarios en otros equipos de la red. El LSA proporciona acceso a los paquetes de autenticación para los proveedores de soporte técnico de seguridad (SSP). Se realiza uno de los dos tipos de autenticación siguientes:

-   [Autenticación interactiva](interactive-authentication.md)
-   [Autenticación no interactiva](noninteractive-authentication.md)
