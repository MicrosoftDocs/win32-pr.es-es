# <a name="multimedia-audio"></a>Audio multimedia

Windows proporciona funciones que permiten a una aplicación agregar servicios de audio. Las funciones de audio multimedia se introdujeron en Windows de 16 bits. Muchas de sus características se han sustituido por características de DirectX. Para obtener más información, consulte la documentación de DirectX.

Los temas siguientes se describen en la documentación de audio multimedia.

-   [Administrador de compresión de audio](audio-compression-manager.md)
-   [Mezcladores de audio](audio-mixers.md)
-   [Interfaz digital de instrumentos digitales (MIDI)](musical-instrument-digital-interface--midi.md)
-   [Audio de onda](waveform-audio.md)

También se puede reproducir audio mediante la interfaz de control de medios. Para obtener más información, consulte [MCI](mci.md).
