# <a name="dns-zones"></a>Zonas DNS

Una zona DNS es un conjunto de archivos o registros (más concretamente, una base de datos de entradas de registros de recursos) que corresponde a parte del espacio de nombres jerárquico de DNS. Las zonas DNS se utilizan para delimitar qué servidores DNS son responsables (autoritativos) para resolver consultas de resolución de nombres para una sección determinada de la jerarquía de DNS. Las zonas DNS se diferencian de la estructura de dominios de la siguiente manera: las zonas pueden estar compuestas de uno o más dominios DNS. Una zona en el árbol de dominio de gadgets.widgets.microsoft.com puede ser autoritativa para los dominios de gadgets y widgets. En otras palabras, no hay ningún requisito para que las zonas DNS tengan una relación de uno a uno con los dominios DNS.
