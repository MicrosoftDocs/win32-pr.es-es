# <a name="printer-object-user-interface-mapping"></a>Asignación de interfaz de usuario del objeto de impresora

En este tema se describen las hojas de propiedades de objetos de impresora del complemento usuarios y equipos de Active Directory.

## <a name="general-property-sheet"></a>Hoja de propiedades general

En la tabla siguiente se muestran las etiquetas de la interfaz de usuario de la hoja de propiedades **General** .



| Etiqueta de la interfaz de usuario           | Atributo en Active Directory Domain Services |
|--------------------|-----------------------------------------------|
| Location           | ubicación                                      |
| Modelo              | driverName                                    |
| Descripción        | description                                   |
| Color              | printColor                                    |
| Staple             | printStaplingSupported                        |
| Dos caras       | imprimir DuplexSupported                         |
| Menú de impresión      | printRate                                     |
| Resolución máxima | printMaxResolutionSupported                   |
