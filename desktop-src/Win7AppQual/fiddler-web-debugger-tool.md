# <a name="fiddler-web-debugger-tool"></a>Herramienta de depurador web Fiddler

La herramienta Fiddler le ayuda a depurar aplicaciones web mediante la captura del tráfico de red entre los equipos de Internet y de prueba. La herramienta le permite inspeccionar los datos entrantes y salientes para supervisar y modificar solicitudes y respuestas antes de que el explorador los reciba. Fiddler también incluye un potente subsistema de scripting basado en eventos, que se puede ampliar mediante el uso de cualquier lenguaje .NET Framework.

Fiddler y las opciones de reproducción de HTTP pueden ayudarle a solucionar problemas del cliente con aplicaciones web mediante la creación de una copia sin conexión del sitio de prueba. Con estas herramientas, puede crear imágenes sin conexión de la experiencia de exploración y, a continuación, empaquetar y analizar los resultados para obtener información de depuración más detallada.

Para descargar el complemento Fiddler, vaya a la página de [Complementos de Internet Explorer](https://www.microsoft.com/iegallery) . Para obtener más información sobre cómo solucionar problemas con Fiddler y las herramientas relacionadas, vea la entrada [de blog sobre cómo usar Fiddler y la reproducción http para tener una copia sin conexión de su sitio](https://blogs.msdn.com/b/ricardo/archive/2010/06/22/how-to-user-fiddler-and-http-replay-to-have-an-offline-copy-of-your-site.aspx) y la entrada de blog [solución de problemas de autenticación con Fiddler](https://blogs.msdn.com/b/ieinternals/archive/2009/11/22/clipboard-paste-bug-in-windows-7-password-prompt.aspx) .
