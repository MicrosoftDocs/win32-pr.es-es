# <a name="systemdevicesaepcontainersupportsrendering"></a>System.Devices.AepContainer.SupportsRendering

Indica si el contenedor de puntos de conexión de asociación de dispositivos admite la representación

## <a name="windows-10-version-1703-windows-10-version-1607"></a>Windows 10, versión 1703, Windows 10, versión 1607

```
propertyDescription
   name = System.Devices.AepContainer.SupportsRendering
   shellPKey = PKEY_Devices_AepContainer_SupportsRendering
   formatID = 6AF55D45-38DB-4495-ACB0-D4728A3B8314
   propID = 10
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
