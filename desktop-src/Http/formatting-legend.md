# <a name="formatting-legend"></a>Leyenda de formato

En la sección se utilizan las siguientes convenciones de formato.

| Formato                                                                    | Significado                                                        |
|---------------------------------------------------------------------------|----------------------------------------------------------------|
| Cursiva                                                                    | Información que el usuario debe proporcionar                          |
| Bold                                                                      | Elementos que el usuario debe escribir exactamente como se indica              |
| Puntos suspensivos (...)                                                            | Parámetro que se puede repetir varias veces en una línea de comandos |
| Entre corchetes ( \[ \] )                                                   | Elementos opcionales                                                 |
| Entre llaves ( {} ); opciones separadas por canalización ( \| ). Ejemplo: { \| impares} | Conjunto de opciones de las que el usuario debe elegir sólo una.        |
| Fuente Courier                                                              | Código o resultado del programa                                         |
