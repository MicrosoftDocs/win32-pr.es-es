# <a name="ui_pkey_categoryid"></a>UI \_ PKEY \_ CategoryId

Identifica la propiedad \_ PKEY CategoryId de la \_ interfaz de usuario.

```
propertyDescription
   name = UI_PKEY_CategoryId
   shellPKey = UI_PKEY_CategoryId
   formatID = 00000103-7363-696e-8441798acf5aebb7
   propID = 103
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Comentarios

La interfaz de usuario PKEY CategoryId la usa una aplicación para consultar los id. de las categorías que se usan para agrupar elementos relacionados \_ en un control de \_ galería.

El valor de propiedad es el índice de base cero de un elemento de una colección de categorías.

En la captura de pantalla siguiente se muestra  el uso de categorías en  un [**menú SplitButton**](windowsribbon-element-splitbutton.md) donde Las formas de selección tienen un identificador de 0 y Las opciones de selección tienen un identificador de 1.
