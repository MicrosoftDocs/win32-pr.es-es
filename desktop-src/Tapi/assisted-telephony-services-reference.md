# <a name="assisted-telephony-services-reference"></a>Referencia de servicios de telefonía asistidos

Las funciones de telefonía asistida se muestran en la tabla siguiente. Las funciones de telefonía asistida permiten que cualquier aplicación establezca llamadas de voz y llamadas multimedia. Para obtener información general, consulte la introducción a la [telefonía asistida](./assisted-telephony-overview.md) .

## <a name="assisted-telephony-services-functions"></a>Funciones de servicios de telefonía asistidos



| Función                                             | Descripción                                                 |
|------------------------------------------------------|-------------------------------------------------------------|
| [**tapiRequestMakeCall**](/windows/desktop/api/Tapi/nf-tapi-tapirequestmakecall)   | Envía una solicitud para realizar una llamada de voz.                    |
| [**tapiRequestMediaCall**](tapirequestmediacall.md) | Obsoleto. No debe usarse.                                       |
| [**tapiRequestDrop**](/windows/desktop/api/Tapi/nf-tapi-tapirequestdrop)           | Obsoleto. No debe usarse.                                       |
| [**tapiGetLocationInfo**](/windows/desktop/api/Tapi/nf-tapi-tapigetlocationinfo)   | Devuelve códigos que identifican el país o región, y la ciudad o el área. |
