# <a name="wmpartofset"></a>WM/PartOfSet

El atributo **WM/PartOfSet** contiene el número de pieza y el número total de elementos del conjunto al que pertenece el archivo. Los dos valores se separan con una barra diagonal.

## <a name="global-constant"></a>Constante global

g \_ wszWMPartOfSet

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="example"></a>Ejemplo



| Tipo de archivo | Valor de ejemplo |
|-----------|---------------|
| All       | "2/3"         |
