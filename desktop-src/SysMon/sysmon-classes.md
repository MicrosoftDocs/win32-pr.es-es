# <a name="sysmon-classes"></a>Clases SYSMON

Puede utilizar las siguientes clases para configurar el control del monitor de sistema (SYSMON).



| Nombre de la clase                             | Descripción                                                                                                  |
|----------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [**Elemento de**](counteritem.md)     | Proporciona información sobre las propiedades de los elementos de contador.                                                                  |
| [**Counters**](counters.md)           | Proporciona los métodos para administrar una colección de objetos de [**elementos de elemento**](counteritem.md) .                |
| [**LogFileItem**](logfileitem.md)     | Proporciona una propiedad para recuperar la ruta de acceso de un único archivo de registro al que tiene acceso el control de monitor del sistema.     |
| [**LogFiles**](logfiles.md)           | Proporciona los métodos y las propiedades para administrar una colección de objetos [**LogFileItem**](logfileitem.md) . |
| [**SystemMonitor**](systemmonitor.md) | Proporciona los métodos y las propiedades que se usan para configurar el control de monitor del sistema.                            |
