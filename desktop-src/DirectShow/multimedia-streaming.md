# <a name="multimedia-streaming"></a>Transmisión por secuencias multimedia

> [!Note]  
> Estas API están en desuso. Las aplicaciones deben usar el filtro de [**enganche de ejemplo**](sample-grabber-filter.md) o implementar un filtro personalizado para obtener datos de un gráfico de filtros de DirectShow.

 

Streaming multimedia es un conjunto de API que permiten a una aplicación obtener datos de audio y vídeo de un gráfico de filtros de DirectShow sin necesidad de escribir un filtro DirectShow personalizado.
