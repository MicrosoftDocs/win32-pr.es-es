# <a name="cutscene-sample"></a>Ejemplo de CutScene

## <a name="description"></a>Descripción

El ejemplo CutScene reproduce un archivo de vídeo en modo de pantalla completa hasta que el usuario escribe ESC, ENTER o la barra espaciadora.

Ejecute este ejemplo desde la línea de comandos, con el nombre de un archivo de vídeo como un argumento de línea de comandos.

## <a name="downloading-the-sample"></a>Descargar el ejemplo

Para descargar los ejemplos del SDK de DirectShow, instale la versión más reciente de la [Windows SDK](https://msdn.microsoft.com/windowsvista/bb980924.aspx).

Este ejemplo se instala en la siguiente ruta de acceso: ejemplos *\[ raíz \] del SDK* \\ \\ multimedia \\ DirectShow \\ reproductores \\ CutScene.
