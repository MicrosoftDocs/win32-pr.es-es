# <a name="windows-design-basics"></a>Windows (conceptos básicos de diseño)

> [!NOTE]
> Esta guía de diseño se creó para Windows 7 y no se ha actualizado para las versiones más recientes de Windows. Gran parte de las instrucciones se siguen aplicando en principio, pero la presentación y los ejemplos no reflejan nuestra [guía de diseño actual.](/windows/uwp/design/)

Windows son los principales "lienzos" o superficies de interfaz de usuario de la aplicación de escritorio, incluidas las ventanas principales y los elementos emergentes, diálogos y asistentes. Siga estas directrices a la hora de decidir qué superficie usar y cómo usarlas mejor.

## <a name="in-this-section"></a>En esta sección




| Tema | Descripción | 
|-------|-------------|
| <a href="win-window-mgt.md">Administración de ventanas</a><br /> | En este artículo se trata la colocación predeterminada de las ventanas cuando se muestran inicialmente en la pantalla, su orden de apilamiento con respecto a otras ventanas (orden<a href="glossary.md">Z),</a>su tamaño inicial y cómo su pantalla afecta al foco de entrada.<br /> | 
| <a href="win-window-frames.md">Marcos de ventana</a><br /> | La mayoría de los programas deben usar marcos de ventana estándar. Las aplicaciones inmersivas pueden tener un modo de pantalla completa que oculta el marco de la ventana. Considere la posibilidad de usar el cristal estratégicamente para una apariencia más sencilla, más ligera y más cohesiva. <br /> | 
| <a href="win-dialog-box.md">Cuadros de diálogo</a><br /> | Un cuadro de diálogo es una ventana secundaria que permite a los usuarios realizar un comando, hacer una pregunta a los usuarios o proporcionar a los usuarios información o comentarios de progreso.<br /> | 
| <a href="win-common-dlg.md">Cuadros de diálogo comunes</a><br /> | Los cuadros Windows comunes de Microsoft constan de los cuadros de diálogo Abrir archivo, Guardar archivo, Abrir carpeta, Buscar y reemplazar, Imprimir, Configuración de página, Fuente y Color.<br /> | 
| <a href="win-wizards.md">Asistentes</a><br /> | A pesar de ese nombre fantástico y llorón, los asistentes no son realmente una forma especial de interfaz de usuario y solo tienen un intervalo determinado de utilidad. <br /> | 
| <a href="win-property-win.md">Propiedades Windows</a><br /> | La ventana de propiedades es el nombre colectivo de los siguientes tipos de interfaces de usuario (URI):<br /><ul><li>Hoja de propiedades: se usa <strong>para ver y cambiar las propiedades de un objeto</strong>o colección de objetos en un cuadro de diálogo .</li><li>Inspector de propiedad: se usa <strong>para ver y cambiar las propiedades de un objeto</strong>o colección de objetos en un panel .</li><li>Cuadro de diálogo Opciones: se usa <strong>para ver y cambiar las opciones de una aplicación.</strong></li></ul> |
