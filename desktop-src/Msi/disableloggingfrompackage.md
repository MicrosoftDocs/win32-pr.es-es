# <a name="disableloggingfrompackage"></a>DisableLoggingFromPackage

El valor de esta [](system-policy.md) directiva de sistema por equipo se establece en "1" para deshabilitar el registro especificado para el paquete por la propiedad [**MsiLogging**](msilogging.md) para todos los usuarios del equipo.

La [**propiedad MsiLogging**](msilogging.md) y la directiva DisableLoggingFromPackage requieren Windows Installer 4.0.

## <a name="registry-key"></a>Clave del Registro

**HKEY \_ Local \_ MACHINE** \\ **Software** \\ **Policies** \\ **Microsoft** \\ **Windows** \\ **Installer**

## <a name="data-type"></a>Tipo de datos

**REG \_ DWORD**
