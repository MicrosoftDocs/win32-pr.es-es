# <a name="locale_imonlzero"></a>CONFIGURACIÓN REGIONAL \_ IMONLZERO

Especificador para ceros a la izquierda en campos de mes solo para una fecha corta. Es preferible que la aplicación use la [constante LOCALE \_ SSHORTDATE](locale-sshortdate.md) en lugar de LOCALE \_ IMONLZERO.



| Value | Significado                     |
|-------|-----------------------------|
| 0     | Sin ceros a la izquierda durante meses |
| 1     | Ceros a la izquierda durante meses    |
