# <a name="what-is-text-services-framework"></a>¿Qué es el marco de trabajo de servicios de texto?

El marco de trabajo de servicios de texto (TSF) es un sistema independiente, independiente del lenguaje y extensible para la entrada de texto y tecnologías de lenguaje.

Las aplicaciones habilitadas para TSF pueden recibir entradas de texto de cualquier servicio de texto que admita TSF sin tener que conocer los detalles del origen de texto.

Los servicios de texto proporcionan texto a las aplicaciones de forma que no requieren ningún conocimiento detallado de las aplicaciones que producen o reciben el texto. Por ejemplo, un servicio de texto puede proporcionar datos de texto de los dispositivos de entrada de voz o de escritura a mano.
