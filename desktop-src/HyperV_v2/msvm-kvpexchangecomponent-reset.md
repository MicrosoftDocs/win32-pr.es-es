# <a name="reset-method-of-the-msvm_kvpexchangecomponent-class"></a>Método Reset de la clase Msvm \_ KvpExchangeComponent

Solicita un restablecimiento.

## <a name="syntax"></a>Sintaxis


```mof
uint32 Reset();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

Este método devuelve uno de los siguientes valores:

<dl> <dt>
