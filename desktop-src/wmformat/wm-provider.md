# <a name="wmprovider"></a>WM/proveedor

El atributo **WM/Provider** contiene el nombre del proveedor de contenido de metadatos. Los metadatos se suelen obtener a través de un servicio.

## <a name="global-constant"></a>Constante global

g \_ wszWMProvider

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**
