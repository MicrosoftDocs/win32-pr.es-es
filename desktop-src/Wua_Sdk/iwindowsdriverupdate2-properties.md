# <a name="iwindowsdriverupdate2-properties"></a>Propiedades de IWindowsDriverUpdate2

La [**interfaz IWindowsDriverUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate) define las siguientes propiedades.



| Propiedad                                                       | Descripción                                                                                                             |
|----------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**CveIDs**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_cveids)                 | Contiene una colección de los identificadores de vulnerabilidades y exposición comunes (CVE) asociados a una actualización. |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_ispresent)           | Obtiene un valor booleano que indica si una actualización está instalada en el equipo.                                     |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdate2-get_rebootrequired) | Obtiene un valor booleano que indica si el equipo debe reiniciarse después de instalar o desinstalar una actualización.    |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz.

-   [**IWindowsDriverUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdate)
