# <a name="table-names"></a>Nombres de tabla

Los nombres de tabla están limitados a 31 caracteres. Si un nombre de tabla contiene un punto (una práctica que se permite pero no se recomienda), el nombre debe incluirse entre marcas de acento grave \` (valor ASCII 96), por ejemplo, \` filename.ext \` . Esta sintaxis de escape evita conflictos entre el nombre de tabla y las palabras reservadas, como SQL palabras clave. Para obtener más información, [vea SQL sintaxis](sql-syntax.md).
