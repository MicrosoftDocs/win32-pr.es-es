# <a name="multilingual-user-interface"></a>Interfaz de usuario multilingüe

La interfaz de usuario multilingüe (MUI) permite la localización de interfaces de usuario para aplicaciones globalizadas. MUI también admite la creación de recursos para cualquier número de idiomas de la interfaz de usuario.

En esta documentación se describe MUI y se indica cómo agregar la funcionalidad de MUI a la aplicación totalmente globalizada. Sin embargo, no intenta describir todos los detalles del desarrollo de aplicaciones globalizadas en Windows. Para obtener más información sobre este tipo, visite el centro para desarrolladores de Microsoft [Go global](https://msdn.microsoft.com/goglobal).

La documentación de MUI incluye las siguientes secciones:

-   [Acerca de la interfaz de usuario multilingüe](about-multilingual-user-interface.md)
-   [Uso de la interfaz de usuario multilingüe](using-multilingual-user-interface.md)
-   [Referencia de la interfaz de usuario multilingüe](multilingual-user-interface-reference.md)
