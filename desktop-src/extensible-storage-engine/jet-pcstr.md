# <a name="jet_pcstr"></a>JET_PCSTR


_**Se aplica a:** Windows | Windows Server_

## <a name="jet_pcstr"></a>JET_PCSTR

El tipo de datos **JET_PCSTR** contiene una cadena **ASCII** constante terminada en null (char \* ).

**Windows Vista: JET_PCSTR** se incorpora en Windows Vista.

```cpp
    typedef __nullterminated const char *  JET_PCSTR;
```

### <a name="data-types"></a>Tipo de datos

JET_PCSTR

Cadena ASCII constante (Char) terminada en NULL \* .
