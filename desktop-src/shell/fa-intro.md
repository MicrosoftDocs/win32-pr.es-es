## <a name="additional-resources"></a>Recursos adicionales

-   Establecer el acceso al programa y los valores predeterminados del equipo (SPAD) es un Windows Panel de control que permite a los usuarios con privilegios administrativos establecer un valor predeterminado de la máquina y ocultar o mostrar una aplicación. Las aplicaciones Multimedia, Mail, Browser, Messenger y Java son ejemplos de aplicaciones registradas en SPAD. Establecer los programas predeterminados (SYDP) es un Windows Panel de control, que funciona con privilegios limitados y permite a los usuarios establecer un valor predeterminado de usuario. Cualquier aplicación puede registrarse en SYDP. Para obtener información sobre el registro de aplicaciones SPAD y SYDP, vea [Guidelines for File Associations and Default Programs](appguide-fa-defpro.md)y Set Program Access and Computer [Defaults (SPAD).](cpl-setprogramaccess.md)
-   Para obtener información conceptual relacionada, vea [Información general de verbos y asociaciones de archivo.](fa-verbs.md)
-   Para crear un almacén de datos de Shell, vea [Implementing the Basic Folder Object Interfaces](nse-implement.md).

Para obtener documentación de referencia relacionada, consulte los temas siguientes:

-   Para ejecutar un verbo en un elemento de Shell, vea el [**método InvokeVerb.**](folderitem-invokeverb.md)
-   Para recuperar una colección de verbos que se pueden ejecutar en un elemento de Shell, vea el [**método Verbos.**](folderitem-verbs.md)
-   Para realizar una operación en un archivo especificado, vea las [**funciones ShellExecute**](/windows/desktop/api/Shellapi/nf-shellapi-shellexecutea) o [**ShellExecuteEx.**](/windows/desktop/api/Shellapi/nf-shellapi-shellexecuteexa)
-   Para obtener una lista de los tipos percibidos predeterminados, vea la [**enumeración PERCEIVED.**](/windows/win32/api/shtypes/ne-shtypes-perceived)
-   Para recuperar el tipo percibido de un archivo en función de su extensión, vea la [**función AssocGetPerceivedType.**](/windows/desktop/api/Shlwapi/nf-shlwapi-assocgetperceivedtype)
