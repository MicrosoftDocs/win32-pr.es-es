# <a name="about-the-task-scheduler"></a>Acerca de la Programador de tareas

El Programador de tareas servicio le permite realizar tareas automatizadas en un equipo elegido. Con este servicio, puede programar cualquier programa para que se ejecute en un momento adecuado para usted o cuando se produzca un evento específico. El Programador de tareas supervisa el tiempo o los criterios de evento que elija y, a continuación, ejecuta la tarea cuando se cumplen esos criterios.

## <a name="where-task-scheduler-is-installed"></a>Dónde Programador de tareas está instalado

El Programador de tareas se instala automáticamente con varios sistemas operativos de Microsoft.

Programador de tareas 1.0 se instala con los sistemas operativos Windows Server 2003, Windows XP y Windows 2000.

Programador de tareas 2.0 se instala con Windows Vista y Windows Server 2008.

La API Programador de tareas 2.0 debe usarse en el desarrollo de aplicaciones que usan el servicio Programador de tareas en Windows Vista. Para obtener más información, [vea Programador de tareas referencia](task-scheduler-reference.md)de .

Programador de tareas se inicia cada vez que se inicia el sistema operativo. Se puede ejecutar a través de la interfaz gráfica Programador de tareas usuario (GUI) o a través de Programador de tareas API descrita en este SDK.
