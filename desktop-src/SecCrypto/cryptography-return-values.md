# <a name="cryptography-return-values"></a>Valores devueltos de criptografía

Las funciones de criptografía usan los siguientes valores devueltos y se definen en Wincrypt. h. Los valores devueltos de criptografía se clasifican en función del uso como se indica a continuación.



| Valor devuelto                                                                       | Descripción                                                                                                |
|------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [Valores devueltos de codificación/descodificación ASN. 1](asn-1-encoding-decoding-return-values.md) | Constantes predefinidas que devuelven las funciones que realizan la codificación o descodificación de ASN. 1.                    |
| [Valores devueltos de certificado y confianza](certificate-and-trust-return-values.md)     | Constantes predefinidas que se devuelven cuando se determina la validez y el estado de confianza de un certificado. |



 

Para obtener una lista de los valores devueltos de CAPICOM, consulte el [**\_ \_ código de error de CAPICOM**](capicom-error-code.md).
