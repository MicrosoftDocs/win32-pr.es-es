# <a name="use-appropriate-timing-and-emphasis"></a>Usar el tiempo y el énfasis adecuados

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Al igual que todos los comentarios, la eficacia de la salida de voz depende del tiempo y el énfasis. Se puede comunicar una gran cantidad de información en el ritmo, el volumen y el tono que se usa cuando se habla algo. Si usa un motor de texto a voz como voz del carácter, la mayoría de los motores le permiten establecer la velocidad, las pausas, el tono y el énfasis de palabras o frases. Puede usar estos atributos para indicar el interés y la comprensión del carácter, así como para dirigir la atención del usuario o para indicar la intención del carácter. Para obtener más información sobre cómo establecer atributos de voz, vea [Etiquetas de salida de](microsoft-agent-speech-output-tags.md)voz de Microsoft Agent . Si usa archivos de sonido como salida del carácter, tenga en cuenta estos factores también en el audio grabado.
