# <a name="bluetooth-and-getpeername"></a>Bluetooth y getpeername

La función [**getpeername**](/windows/desktop/api/winsock/nf-winsock-getpeername) se usa para recuperar la dirección Bluetooth del dispositivo Bluetooth del mismo nivel. La función **getpeername** solo es válida para los sockets Bluetooth conectados.

Para obtener más información, consulte [Windows Sockets](/windows/desktop/WinSock/windows-sockets-start-page-2).
