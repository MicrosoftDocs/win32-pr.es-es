# <a name="introduction-to-microsoft-agent"></a>Introducción a Microsoft Agent

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

El agente de Microsoft es un conjunto de servicios de software programable que admite la presentación de caracteres animados interactivos. Los desarrolladores pueden usar caracteres como asistentes interactivos para introducir, guiar, entretener o mejorar de algún otro modo sus páginas web o aplicaciones, además del uso convencional de ventanas, menús y controles.

El agente de Microsoft permite a los desarrolladores de software y a los autores Web incorporar un nuevo formulario de interacción con el usuario, conocido como interfaces de conversación, que aprovecha los aspectos naturales de la comunicación social humana. Además de la entrada del mouse y del teclado, Microsoft Agent incluye compatibilidad opcional con el reconocimiento de voz para que las aplicaciones puedan responder a los comandos de voz. Los caracteres pueden responder mediante voz sintetizada, audio grabado o texto en un globo de palabra de dibujo.

El enfoque de la interfaz de conversación que facilitan los servicios del agente de Microsoft no reemplaza el diseño convencional de la interfaz gráfica de usuario (GUI). En su lugar, la interacción de caracteres puede combinarse fácilmente con los componentes de interfaz convencionales, como ventanas, menús y controles, para ampliar y mejorar la interfaz de la aplicación.

Las interfaces de programación de Microsoft Agent facilitan la animación de un carácter para responder a los datos proporcionados por el usuario. Los caracteres animados aparecen en su propia ventana, lo que proporciona la máxima flexibilidad para el lugar en el que se pueden mostrar en la pantalla. El agente de Microsoft incluye un control ActiveX que hace que sus servicios estén accesibles para los lenguajes de programación que admiten ActiveX, incluidos los lenguajes de scripting web como Visual Basic Scripting Edition (VBScript). Esto significa que la interacción de caracteres se puede programar incluso desde páginas HTML mediante la <OBJECT> etiqueta.
