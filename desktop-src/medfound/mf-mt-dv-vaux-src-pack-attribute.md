# <a name="mf_mt_dv_vaux_src_pack-attribute"></a>MF \_ MT \_ DV \_ Vaux \_ src \_ Pack atributo

Paquete de origen de vídeo auxiliar (VAUX) en un tipo de medio de vídeo digital (DV).

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

Este atributo corresponde al miembro **dwDVVAuxSrc** de la estructura [**DVINFO**](/windows/win32/api/strmif/ns-strmif-dvinfo) de DirectShow.

La constante GUID para este atributo se exporta desde mfuuid. lib.
