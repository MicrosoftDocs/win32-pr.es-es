# <a name="localization-display-id-attribute"></a>Localization: display-ID (atributo)

Se usa para indexar en el archivo Extrts.mc para obtener el displayName adaptado para los objetos con fines de interfaz de usuario.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Localización: ID. de presentación              |
| Nombre para mostrar de LDAP | localizationDisplayId                |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | Administrador de esquema                 |
| Frecuencia de actualización  | Un identificador nunca debe cambiarse.       |
| Attribute-Id      | 1.2.840.113556.1.4.1353              |
| System-ID-GUID    | a746f0d1-78d0-11d2-9916-0000f87a57d4 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
