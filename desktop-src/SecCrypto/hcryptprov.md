# <a name="hcryptprov"></a>HCRYPTPROV

El **tipo de datos HCRYPTPROV** se usa para representar identificadores a los CSP. Estos identificadores se usan para indicar qué módulo CSP realiza operaciones específicas.


```C++
typedef ULONG_PTR HCRYPTPROV;
```
