# <a name="about-the-winsnmp-api"></a>Acerca de la API WinSNMP

\[SNMP está disponible para su uso en los sistemas operativos especificados en la sección de requisitos. En versiones posteriores podría modificarse o no estar disponible. En su lugar, use [administración remota de Windows](/windows/desktop/WinRM/portal), que es la implementación de Microsoft de WS-MAN.\]

La API WinSNMP es una interfaz para el desarrollo de aplicaciones de administración de redes habilitadas para SNMP.

La API WinSNMP proporciona las siguientes características:

-   Tecnología de habilitación de SNMP para aplicaciones de administración de redes
-   Compatibilidad con SNMP versión 1 (SNMPv1) y SNMP versión 2C (SNMPv2C)

Además de las operaciones del administrador SNMP, la versión 2,0 de la API WinSNMP también admite operaciones del agente SNMP.

La API WinSNMP admite aplicaciones de 32 bits y se ejecuta en entornos de un solo subproceso. La compatibilidad con la API WinSNMP está disponible para las aplicaciones que se ejecutan en el entorno operativo Windows 2000 o posterior.
