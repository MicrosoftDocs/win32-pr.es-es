# <a name="retrieving-file-type-information"></a>Recuperando información de tipo de archivo

La función [**GetFileType**](/windows/desktop/api/FileAPI/nf-fileapi-getfiletype) recupera el tipo de un archivo: disco, carácter (como una consola), canalización o desconocido. La función [**GetBinaryType**](/windows/desktop/api/WinBase/nf-winbase-getbinarytypea) determina si un archivo es ejecutable y, en ese caso, el tipo de archivo ejecutable que es. Consulte **GetBinaryType** para obtener una lista de los tipos de archivos ejecutables admitidos.
