# <a name="albumtitlesortorder-attribute"></a>Atributo AlbumTitleSortOrder

El atributo **AlbumTitleSortOrder** es el valor que se va a usar para ordenar la columna del **título del álbum** en la biblioteca.

## <a name="applies-to"></a>Se aplica a

-   [Elementos de audio](audio-item-attributes.md)

## <a name="remarks"></a>Observaciones

Este atributo solo se almacena en la biblioteca.

Para determinar si puede cambiar el valor de este atributo, use el método [media. isReadOnlyItem](media-isreadonlyitem.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|-----------------------------------------------|
| Versión<br/> | Solo Windows Media Player 9 series<br/> |
