# <a name="itemwidth-property"></a>Item. width (propiedad)

Ancho, en píxeles, del elemento. Solo lectura.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Item.Width
```



## <a name="property-value"></a>Valor de propiedad

Variable que recibe el ancho.

## <a name="remarks"></a>Observaciones

Si el elemento es una cámara digital, la propiedad **ancho** representa el ancho de las imágenes que genera la cámara. Si el elemento es un escáner, esta propiedad representa el ancho de la cama de digitalización. Si el elemento es una imagen, esta propiedad representa el ancho real de la imagen.
