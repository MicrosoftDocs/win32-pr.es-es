# <a name="directwrite-functions"></a>Funciones de DirectWrite

DirectWrite proporciona la siguiente función. Todas las demás operaciones se realizan mediante los objetos de interfaz de DirectWrite creados por el objeto [**IDWriteFactory**](/windows/win32/api/dwrite/nn-dwrite-idwritefactory) que se crea con la función siguiente.

## <a name="in-this-section"></a>En esta sección



| Tema                                                         | Descripción                                                                                                             |
|---------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**DWriteCreateFactory**](/windows/win32/api/dwrite/nf-dwrite-dwritecreatefactory)<br/> | Crea un objeto de generador de DirectWrite que se usa para la creación posterior de objetos de DirectWrite individuales.<br/> |
