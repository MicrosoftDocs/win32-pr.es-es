# <a name="ctransinplacefiltertypesmatch-method"></a>CTransInPlaceFilter. TypesMatch, método

El `TypesMatch` método determina si el tipo de medio de entrada coincide con el tipo de medio de salida.

## <a name="syntax"></a>Sintaxis


```C++
BOOL TypesMatch();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve **true** si los tipos de medios de entrada y salida son iguales, o **false** en caso contrario.
