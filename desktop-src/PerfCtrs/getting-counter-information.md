# <a name="getting-counter-information"></a>Obtención de información de contador

Para recuperar información sobre un contador, como el tipo de contador, el texto de ayuda y la ruta de acceso, llame a la función [**PdhGetCounterInfo**](/windows/desktop/api/Pdh/nf-pdh-pdhgetcounterinfoa) . La función devuelve la información del contador como una estructura de [**\_ \_ información del contador PDH**](/windows/desktop/api/Pdh/ns-pdh-pdh_counter_info_a) .
