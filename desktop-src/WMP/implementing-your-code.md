# <a name="implementing-your-code"></a>Implementar el código

Una vez que haya completado el proceso de compilación, probablemente querrá mejorar el código generado por el asistente. En caso contrario, la visualización tendrá un aspecto muy similar a las barras. Lo que necesita hacer es cambiar la implementación de la función de **representación** y, posiblemente, la función **RenderWindowed** .

En las secciones siguientes se proporcionan detalles sobre cómo modificar el código de ejemplo generado por el Asistente para complementos:

-   [Implementación de render](implementing-render.md)
-   [Implementar otras funciones](implementing-other-functions.md)
-   [Valores preestablecidos](presets.md)
