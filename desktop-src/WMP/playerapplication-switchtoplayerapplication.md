# <a name="playerapplicationswitchtoplayerapplication-method"></a>PlayerApplication. switchToPlayerApplication, método

El método **switchToPlayerApplication** cambia un control de Windows Media Player remoto al modo completo del reproductor.

## <a name="syntax"></a>Sintaxis


```JScript
PlayerApplication.switchToPlayerApplication()
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.

## <a name="remarks"></a>Observaciones

Este método solo se usa cuando la comunicación remota de Windows Media Player el control.

**Windows Media Player 10 Mobile:** Este método no se admite.
