# <a name="the-listening-tip-window"></a>Ventana Sugerencia de escucha

\[Microsoft Agent está en desuso a partir de Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Si la voz está habilitada, aparece una ventana de información sobre herramientas especial cuando el usuario presiona la tecla de inserción para hablar para comenzar la entrada de voz. La sugerencia de escucha muestra información contextual relacionada con el estado de entrada actual de Microsoft Agent. Si no se ha instalado o deshabilitado el reconocimiento de voz compatible, no aparece la sugerencia de escucha.
