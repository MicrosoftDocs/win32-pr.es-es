# <a name="m-wmi"></a>M (WMI)

[A](gloss-a.md) B [C](gloss-c.md) [D](gloss-d.md) [E](gloss-e.md) [F](gloss-f.md) G [H](gloss-h.md) [I](gloss-i.md) J [K](gloss-k.md) [L M](gloss-l.md) N [O](gloss-n.md) [](gloss-o.md) [P P](gloss-p.md) [Q](gloss-q.md) [R](gloss-r.md) [S](gloss-s.md) [T](gloss-t.md) U V [W](gloss-w.md) X Y Z

<dl> <dt>

<span id="wmi.gloss_managed_object"></span><span id="WMI.GLOSS_MANAGED_OBJECT"></span>**objeto administrado**
</dt> <dd>

Componente del sistema de hardware o software que se representa como una instancia de una clase WMI.

</dd> <dt>

<span id="wmi.gloss_managed_object_format"></span><span id="WMI.GLOSS_MANAGED_OBJECT_FORMAT"></span>**Managed Object Format (MOF)**
</dt> <dd>

Lenguaje basado en el lenguaje de definición de interfaz (IDL) que describe la información de administración.

</dd> <dt>

<span id="wmi.gloss_management_information_base"></span><span id="WMI.GLOSS_MANAGEMENT_INFORMATION_BASE"></span>**base de información de administración (MIB)**
</dt> <dd>

Base de datos que almacena una colección de representaciones de objetos de red.

</dd> <dt>

<span id="wmi.gloss_metamodel"></span><span id="WMI.GLOSS_METAMODEL"></span>**Metamodelo**
</dt> <dd>

Componente CIM que describe las entidades y relaciones que representan *objetos administrados.* Por ejemplo, las clases, instancias y asociaciones se incluyen en el metamodelo.

</dd> <dt>

<span id="wmi.gloss_metaschema"></span><span id="WMI.GLOSS_METASCHEMA"></span>**metaschema**
</dt> <dd>

Esquema [*del*](gloss-s.md) *metamodelo*.

</dd> <dt>

<span id="wmi.gloss_method_provider"></span><span id="WMI.GLOSS_METHOD_PROVIDER"></span>**proveedor de métodos**
</dt> <dd>

Proveedor que permite el acceso WMI a los métodos de una clase. Vea también [Escribir un proveedor de métodos.](writing-a-method-provider.md)

</dd> <dt>

<span id="wmi.gloss_mib"></span><span id="WMI.GLOSS_MIB"></span>**Mib**
</dt> <dd>

Consulte *la base de información de administración (MIB).*

</dd> <dt>

<span id="wmi.gloss_mof"></span><span id="WMI.GLOSS_MOF"></span>**Mof**
</dt> <dd>

Vea *Managed Object Format (MOF).*

</dd> <dt>

<span id="wmi.gloss_mof_file"></span><span id="WMI.GLOSS_MOF_FILE"></span>**Archivo MOF**
</dt> <dd>

Archivo de texto que contiene definiciones de clases e instancias especificadas mediante *Managed Object Format (MOF).*

</dd> </dl>
