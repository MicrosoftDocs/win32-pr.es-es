# <a name="installing-your-character"></a>Instalación del carácter

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Para que el carácter se muestre en la Galería del asistente, instáleno en la carpeta Microsoft Office Actors dentro de la carpeta \\ Datos de aplicación del \\ usuario. La carpeta Datos de la aplicación se puede encontrar en Windows directorio o en el directorio Perfiles de usuario dentro del Windows usuario. Por ejemplo, en Windows 2000, puede instalar el carácter en "C: \\ Documents and Configuración \\ YourLoginName \\ Application Data Microsoft Office \\ \\ \\ Actors".
