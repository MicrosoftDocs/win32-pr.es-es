# <a name="com-queued-components"></a>Componentes en cola de COM+

La característica de componentes en cola COM+ permite a un cliente solicitar un servicio de una aplicación de servidor COM+, incluso cuando la aplicación de servidor no está disponible temporalmente. La solicitud de cliente se almacena en una cola y se procesa cuando el servidor está disponible.

> [!Note]  
> Para usar el servicio de componentes en cola de COM+, debe tener [Message Queue Server](/previous-versions/windows/desktop/legacy/ms711472(v=vs.85)) instalado en el equipo.
