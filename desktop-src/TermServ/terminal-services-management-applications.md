# <a name="remote-desktop-services-management-applications"></a>Aplicaciones de administración de Servicios de Escritorio remoto

Servicios de Escritorio remoto admite las siguientes aplicaciones de administración.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[Extensión ADSI para Servicios de Escritorio remoto configuración de usuario](adsi-extension-for-terminal-services-user-configuration.md)
</dt> <dd>

Una extensión de la interfaz de servicio de Active Directory (ADSI) es un objeto COM especial que permite la extensión de un objeto COM.

</dd> <dt>

[Servicios de Escritorio remoto permisos](terminal-services-permissions.md)
</dt> <dd>

Puede usar los permisos proporcionados para Servicios de Escritorio remoto para controlar el modo en que los usuarios y los grupos acceden al servidor.

</dd> </dl>

El [proveedor de servicios de escritorio remoto WMI](terminal-services-wmi-provider.md) proporciona acceso de WMI a la información que los administradores pueden configurar mediante el complemento configuración/conexiones de Microsoft Management Console (MMC) de servicios de escritorio remoto.

La [extensión ADSI para servicios de escritorio remoto configuración de usuario](adsi-extension-for-terminal-services-user-configuration.md) extiende un objeto ADSI que permite a los administradores de dominio escribir scripts que automaticen el mantenimiento de las propiedades de usuario específicas de servicios de escritorio remoto.

Para obtener una lista de los permisos que puede establecer y las tareas que los permisos permiten, [servicios de escritorio remoto permisos](terminal-services-permissions.md).

Para obtener más información, vea la [Referencia del proveedor de WMI servicios de escritorio remoto](terminal-services-wmi-provider-reference.md) y [el uso de la extensión ADSI para servicios de escritorio remoto configuración de usuario](using-the-adsi-extension-for-terminal-services-user-configuration.md).
