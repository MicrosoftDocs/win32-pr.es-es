# <a name="cimagepaletteremovepalette-method"></a>CImagePalette. RemovePalette, método

El `RemovePalette` método elimina la paleta lógica existente y llama a [**CBaseWindow:: UnsetPalette**](cbasewindow-unsetpalette.md) en el objeto **CBaseWindow** .

## <a name="syntax"></a>Sintaxis


```C++
HRESULT RemovePalette();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve S \_ correcto.
