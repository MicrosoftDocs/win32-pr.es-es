# <a name="ui_pkey_increment"></a>Incremento de UI \_ PKEY \_

Identifica la propiedad de incremento de PKEY de la interfaz de usuario \_ \_ .

```
propertyDescription
   name = UI_PKEY_Increment
   shellPKey = UI_PKEY_Increment
   formatID = 00000205-7363-696e-8441798acf5aebb7
   propID = 205
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Observaciones

\_ \_ Una aplicación usa el incremento de PKEY de la interfaz de usuario para consultar Cuánto aumenta o disminuye el valor del campo de edición del control de [número](windowsribbon-controls-spinner.md) al hacer clic en el botón de incremento o decremento.
