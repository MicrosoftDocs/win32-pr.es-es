# <a name="using-service-security-attachments"></a>Uso de datos adjuntos de seguridad del servicio

En esta sección se describe cómo crear los componentes de datos adjuntos necesarios para admitir una configuración específica del servicio en los complementos de configuración de seguridad. Deberá implementar y registrar dos componentes: un complemento de extensión para mostrar una interfaz de usuario y un motor de datos adjuntos para procesar las solicitudes de configuración y análisis.

Para obtener más información, consulte los siguientes temas:

-   [Crear una extensión de complemento de datos adjuntos](creating-an-attachment-snap-in-extension.md)
-   [Registro de una extensión de complemento de datos adjuntos](registering-an-attachment-snap-in-extension.md)
-   [Crear un motor de datos adjuntos](creating-an-attachment-engine.md)
-   [Registro de un motor de datos adjuntos](registering-an-attachment-engine.md)
