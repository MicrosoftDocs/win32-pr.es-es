# <a name="multiple-threads"></a>Varios subprocesos

Un *subproceso* es la entidad dentro de un proceso que se puede programar para su ejecución. Todos los subprocesos de un proceso comparten su espacio de direcciones virtuales y los recursos del sistema. Cada proceso se inicia con un solo subproceso, pero puede crear subprocesos adicionales a partir de cualquiera de sus subprocesos.
