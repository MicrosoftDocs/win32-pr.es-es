# <a name="ts_char_-constants"></a>\_Constantes de caracteres de TS \_ \*

Las constantes de caracteres de TS \_ \_ \* describen el tipo de carácter.



| Constante o valor                                                                                                                                                                                                                                   | Descripción                                                                                         |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| <span id="TS_CHAR_EMBEDDED"></span><span id="ts_char_embedded"></span><dl> <dt>**TS \_ CARÁCTER \_ incrustado**</dt> <dt>(0xfffc)</dt> </dl>          | El carácter es un carácter de reemplazo de objeto Unicode 2,1.<br/>                             |
| <span id="TS_CHAR_REGION"></span><span id="ts_char_region"></span><dl> <dt>**TS \_ \_Región char**</dt> <dt>(0)</dt> </dl>                     | El carácter es un límite de región.<br/>                                                      |
| <span id="TS_CHAR_REPLACEMENT"></span><span id="ts_char_replacement"></span><dl> <dt>**TS \_ \_Reemplazo de char**</dt> <dt>(0xfffd)</dt> </dl> | El carácter es un carácter de marcador de posición de texto oculto o un carácter de reemplazo Unicode.<br/> |
