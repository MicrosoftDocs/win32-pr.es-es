# <a name="viewing-drm-attributes-in-the-metadata-editor"></a>Visualización de atributos drm en el Editor de metadatos

El Editor de metadatos expone la interfaz IWMDRMEditor, que permite a las aplicaciones de edición examinar determinados atributos de un encabezado DRM en un archivo ASF protegido, y los atributos de la licencia de contenido, si hay alguno, incluso si la aplicación no tiene la biblioteca estática específica de DRM (wmstubdrm.lib) necesaria para las aplicaciones de escritor y reproductor DRM totalmente habilitadas.

> [!Note]  
> DRM no es compatible con la versión basada en x64 de este SDK.
