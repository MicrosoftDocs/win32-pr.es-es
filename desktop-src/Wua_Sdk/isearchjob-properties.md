# <a name="isearchjob-properties"></a>Propiedades de ISearchJob

La [**interfaz ISearchJob**](/windows/desktop/api/Wuapi/nn-wuapi-isearchjob) define las siguientes propiedades.



| Propiedad                                      | Descripción                                                                                                                                                 |
|-----------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**AsyncState**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-get_asyncstate)   | Obtiene el objeto de estado específico del autor de la llamada que se pasa al [**método IUpdateSearch.BeginSearch.**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch)                         |
| [**IsCompleted**](/windows/desktop/api/Wuapi/nf-wuapi-isearchjob-get_iscompleted) | Obtiene un valor booleano que indica si la llamada al método [**IUpdateSearch.BeginSearch**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) está completamente procesada. |
