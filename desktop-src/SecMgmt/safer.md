# <a name="safer"></a>Más seguro

Las funciones de la API más segura proporcionan a cualquier aplicación que inicie programas de orígenes externos la capacidad de consultar la Directiva de seguridad para su aprobación antes de que se inicie un ejecutable. Se puede llamar a estas funciones antes de cargar y ejecutar un archivo ejecutable o contenido activo. El tipo de aplicaciones en las que la API más segura es útil son las aplicaciones que controlan los datos adjuntos (como los clientes de correo electrónico y la mensajería instantánea que puede transferir archivos) y los intérpretes de scripts. La compatibilidad con las funciones de la API más segura comienza con Windows XP. Para obtener más información, consulte [funciones más seguras](management-functions.md).
