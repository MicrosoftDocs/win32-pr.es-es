# <a name="presentation-address-attribute"></a>Presentation-Address atributo)

Especifica una dirección de presentación asociada a un objeto que representa una entidad de aplicación OSI.

|-------------------|-----------------------------------------------------------------------|
| CN                | Presentation-Address                                                  |
| Nombre para mostrar de LDAP | presentationAddress                                                   |
| Tamaño              | \-                                                                    |
| Actualizar privilegio  | \-                                                                    |
| Frecuencia de actualización  | \-                                                                    |
| Attribute-Id      | 2.5.4.29                                                              |
| System-ID-GUID    | a8df744b-c5ea-11d1-bbcb-0080c76670c0                                  |
| Sintaxis            | [**Object(Presentation-Address)**](s-object-presentation-address.md) |
