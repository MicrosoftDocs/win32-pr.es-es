# <a name="exposing-text-programmatically"></a>Exponer texto mediante programación

Exponer texto mediante programación proporciona una gran ventaja a las ayudas de accesibilidad, ya que proporciona una mayor confiabilidad y, en algunos casos, una interpretación más enriquecida de la información.

Hay varios métodos básicos para exponer objetos para exponer texto mediante programación. Sin embargo, ningún método es efectivo con todas las ayudas de accesibilidad, por lo que también es necesario dibujar texto en la pantalla para que se pueda reconocer. Para obtener información sobre cómo exponer texto de grandes cantidades, vea [usar Active Accessibility para exponer texto](using-active-accessibility-to-expose-text.md).
