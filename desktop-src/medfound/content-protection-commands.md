# <a name="content-protection-commands"></a>Content Protection comandos

Enumera los comandos del método [**IDirect3DAuthenticatedChannel9::Configure.**](/windows/desktop/api/d3d9/nf-d3d9-idirect3dauthenticatedchannel9-configure)



| Get-Help                                                                                                           | Descripción                                                                                                           |
|-------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**D3DAUTHENTICATEDCONFIGURE \_ INITIALIZE**](d3dauthenticatedconfigure-initialize.md)                             | Inicializa el canal autenticado.                                                                                |
| [**D3DAUTHENTICATEDCONFIGURE \_ CRYPTOSESSION**](d3dauthenticatedconfigure-cryptosession.md)                       | Asocia una sesión criptográfica a un dispositivo descodificador directX Video Acceleration 2 (DXVA-2) y a un dispositivo Direct3D. |
| [**D3DAUTHENTICATEDCONFIGURE \_ ENCRYPTIONWHENACCESSIBLE**](d3dauthenticatedconfigure-encryptionwhenaccessible.md) | Establece el nivel de cifrado que se realiza antes de que el contenido protegido sea accesible para la CPU o el bus.         |
| [**D3DAUTHENTICATEDCONFIGURE \_ PROTECTION**](d3dauthenticatedconfigure-protection.md)                             | Habilita o deshabilita la protección del dispositivo.                                                                        |
| [**D3DAUTHENTICATEDCONFIGURE \_ SHAREDRESOURCE**](d3dauthenticatedconfigure-sharedresource.md)                     | Habilita un proceso para abrir un recurso compartido o deshabilita que un proceso abra recursos compartidos.                     |
