<span data-ttu-id="624aa-150">Normalmente, una aplicación crea un cuadro de diálogo mediante la función [**DialogBox**](/windows/desktop/api/Winuser/nf-winuser-dialogboxa) o [**CreateDialog**](/windows/desktop/api/Winuser/nf-winuser-createdialoga) .</span><span class="sxs-lookup"><span data-stu-id="624aa-150">An application typically creates a dialog box by using either the [**DialogBox**](/windows/desktop/api/Winuser/nf-winuser-dialogboxa) or [**CreateDialog**](/windows/desktop/api/Winuser/nf-winuser-createdialoga) function.</span></span> <span data-ttu-id="624aa-151">**DialogBox** crea un cuadro de diálogo modal; **CreateDialog** crea un cuadro de diálogo no modal.</span><span class="sxs-lookup"><span data-stu-id="624aa-151">**DialogBox** creates a modal dialog box; **CreateDialog** creates a modeless dialog box.</span></span> <span data-ttu-id="624aa-152">Estas dos funciones cargan una plantilla de cuadro de diálogo desde el archivo ejecutable de la aplicación y crean una ventana emergente que coincide con las especificaciones de la plantilla.</span><span class="sxs-lookup"><span data-stu-id="624aa-152">These two functions load a dialog box template from the application's executable file and create a pop-up window that matches the template's specifications.</span></span> <span data-ttu-id="624aa-153">Hay otras funciones que crean un cuadro de diálogo mediante el uso de plantillas en memoria; pasan información adicional al procedimiento del cuadro de diálogo a medida que se crea el cuadro de diálogo.</span><span class="sxs-lookup"><span data-stu-id="624aa-153">There are other functions that create a dialog box by using templates in memory; they pass additional information to the dialog box procedure as the dialog box is created.</span></span>

## <a name="dialog-box-owner-window"></a><span data-ttu-id="624aa-159">Ventana propietaria del cuadro de diálogo</span><span class="sxs-lookup"><span data-stu-id="624aa-159">Dialog Box Owner Window</span></span>

<span data-ttu-id="624aa-169">Cuando un cuadro de diálogo no modal no tiene ningún propietario, el sistema no oculta ni destruye el cuadro de diálogo cuando se ocultan o destruyen otras ventanas de la aplicación.</span><span class="sxs-lookup"><span data-stu-id="624aa-169">When a modeless dialog box has no owner, the system neither hides nor destroys the dialog box when other windows in the application are hidden or destroyed.</span></span> <span data-ttu-id="624aa-170">Aunque esto no anula el propósito del cuadro de diálogo no modal, requiere que la aplicación lleve a cabo un procesamiento especial para asegurarse de que el cuadro de diálogo se oculta y se destruye en los momentos adecuados.</span><span class="sxs-lookup"><span data-stu-id="624aa-170">Although this does not defeat the purpose of the modeless dialog box, it requires that the application carry out special processing to ensure the dialog box is hidden and destroyed at appropriate times.</span></span>
