# <a name="deleting-a-reservation"></a>Eliminación de una reserva

El proceso para quitar una reserva comienza con una comprobación de acceso que es idéntica a la descrita anteriormente. El ID. de usuario del proceso de llamada debe tener derechos de acceso al *nodo primario* de la entrada de reserva que se va a eliminar.
