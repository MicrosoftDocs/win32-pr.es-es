# <a name="fourcc-attribute"></a>Atributo FourCC

El **atributo FourCC** es un valor hexadecimal que identifica el códec necesario para representar un archivo de vídeo.

## <a name="applies-to"></a>Se aplica a

-   [Elementos de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Comentarios

Para determinar si puede cambiar el valor de este atributo, use el [método Media.isReadOnlyItem.](media-isreadonlyitem.md)

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------|
| Versión<br/> | Reproductor de Windows Media 10 o posterior<br/> |
