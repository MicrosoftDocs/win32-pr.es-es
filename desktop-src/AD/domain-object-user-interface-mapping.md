# <a name="domain-object-user-interface-mapping"></a>Asignación de interfaz de usuario de objetos de dominio

En este tema se describen las hojas de propiedades de objetos de dominio en el complemento usuarios y equipos de Active Directory.

## <a name="general-property-sheet"></a>Hoja de propiedades general

En la tabla siguiente se muestran las etiquetas de la interfaz de usuario en la hoja de propiedades **General** .



| Etiqueta de la interfaz de usuario                       | Atributo de Active Directory |
|--------------------------------|----------------------------|
| Nombre de dominio (anterior a Windows 2000) | DC\*                       |
| Descripción                    | description                |



 

\*El atributo DC proporciona acceso al primero de los componentes de dominio que no es necesariamente el mismo que el DN de NetBIOS.
