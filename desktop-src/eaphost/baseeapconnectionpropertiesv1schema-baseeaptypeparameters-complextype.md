# <a name="baseeaptypeparameters-complex-type---connection-properties"></a>Tipo complejo de BaseEapTypeParameters: propiedades de conexión

El tipo complejo **BaseEapTypeParameters** es un elemento base del elemento específico del método que aparece en el elemento derivado del [esquema BaseEapConnectionPropertiesV1](baseeapconnectionpropertiesv1schema-schema.md).

``` syntax
<xs:complexType name="BaseEapTypeParameters"
    abstract="true"
 />
```

## <a name="remarks"></a>Observaciones

**BaseEapTypeParameters** es un elemento abstracto. Un elemento derivado de **BaseEapTypeParameters** debe usarse en documentos de instancia.
