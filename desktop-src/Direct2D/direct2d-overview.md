# <a name="about-direct2d"></a><span data-ttu-id="22c94-112">Acerca de Direct2D</span><span class="sxs-lookup"><span data-stu-id="22c94-112">About Direct2D</span></span>

<span data-ttu-id="22c94-113">En este tema se presenta Direct2D, una API que proporciona a los desarrolladores de Win32 la capacidad de realizar tareas de representación de gráficos en 2D con un rendimiento y una calidad visual superiores.</span><span class="sxs-lookup"><span data-stu-id="22c94-113">This topic introduces Direct2D, an API that provides Win32 developers with the ability to perform 2-D graphics rendering tasks with superior performance and visual quality.</span></span>

## <a name="what-is-direct2d"></a><span data-ttu-id="22c94-114">¿Qué es Direct2D?</span><span class="sxs-lookup"><span data-stu-id="22c94-114">What is Direct2D?</span></span>

<span data-ttu-id="22c94-115">Direct2D es una API de gráficos 2D de modo inmediato y con aceleración de hardware que proporciona alto rendimiento y representación de alta calidad para geometría 2D, mapas de bits y texto.</span><span class="sxs-lookup"><span data-stu-id="22c94-115">Direct2D is a hardware-accelerated, immediate-mode 2-D graphics API that provides high performance and high-quality rendering for 2-D geometry, bitmaps, and text.</span></span> <span data-ttu-id="22c94-116">La API de Direct2D está diseñada para interoperar con el código existente que usa GDI, GDI+ o Direct3D.</span><span class="sxs-lookup"><span data-stu-id="22c94-116">The Direct2D API is designed to interoperate with existing code that uses GDI, GDI+, or Direct3D.</span></span>

<span data-ttu-id="22c94-117">Direct2D está diseñado principalmente para su uso por las siguientes clases de desarrolladores:</span><span class="sxs-lookup"><span data-stu-id="22c94-117">Direct2D is designed primarily for use by the following classes of developers:</span></span>

-   <span data-ttu-id="22c94-118">Desarrolladores de aplicaciones nativas grandes y de escala empresarial.</span><span class="sxs-lookup"><span data-stu-id="22c94-118">Developers of large, enterprise-scale, native applications.</span></span>
-   <span data-ttu-id="22c94-119">Desarrolladores que crean bibliotecas y kits de biblioteca de controles para su uso por parte de desarrolladores de nivel inferior.</span><span class="sxs-lookup"><span data-stu-id="22c94-119">Developers who create control toolkits and libraries for consumption by downstream developers.</span></span>

-   <span data-ttu-id="22c94-121">Los desarrolladores que usan gráficos de Direct3D y necesitan una representación de texto, de alto rendimiento y sencilla para menús, elementos de interfaz de usuario (IU) y pantallas de visualización (HUDs).</span><span class="sxs-lookup"><span data-stu-id="22c94-121">Developers who use Direct3D graphics and need simple, high-performance 2-D and text rendering for menus, user interface (UI) elements, and Heads-up Displays (HUDs).</span></span>

## <a name="why-direct2d"></a><span data-ttu-id="22c94-122">¿Por qué está Direct2D?</span><span class="sxs-lookup"><span data-stu-id="22c94-122">Why Direct2D?</span></span>

<span data-ttu-id="22c94-123">Entre las motivaciones principales para crear una nueva API de gráficos 2D en Microsoft Windows se incluyen las siguientes:</span><span class="sxs-lookup"><span data-stu-id="22c94-123">The primary motivations for creating a new 2-D graphics API in Microsoft Windows include the following:</span></span>

-   <span data-ttu-id="22c94-124">Para mantener el ritmo del aumento de la riqueza visual a la que están acostumbrados los usuarios de Windows.</span><span class="sxs-lookup"><span data-stu-id="22c94-124">To keep pace with the increasing level of visual richness that Windows users are accustomed to.</span></span>
-   <span data-ttu-id="22c94-125">Para permitir que los desarrolladores escriban código de representación 2D que se escala directamente con el hardware de procesamiento de gráficos del equipo en el que se está ejecutando.</span><span class="sxs-lookup"><span data-stu-id="22c94-125">To enable developers to write 2-D rendering code that scales directly with the graphics processing hardware of the PC it is running on.</span></span>
-   <span data-ttu-id="22c94-126">Permite a los desarrolladores escribir código para representar gráficos 2D que se pueden ejecutar en el contexto de un servicio.</span><span class="sxs-lookup"><span data-stu-id="22c94-126">To enable developers to write code for rendering 2-D graphics that can run in the context of a service.</span></span>

<span data-ttu-id="22c94-127">En los últimos años, los usuarios finales han empezado a esperar una mayor fidelidad visual en las experiencias digitales.</span><span class="sxs-lookup"><span data-stu-id="22c94-127">In recent years, end users have begun to expect greater visual fidelity in digital experiences.</span></span> <span data-ttu-id="22c94-128">Esta tendencia se refleja en la electrónica del consumidor.</span><span class="sxs-lookup"><span data-stu-id="22c94-128">This trend is reflected in consumer electronics.</span></span> <span data-ttu-id="22c94-129">Los dispositivos GPS, los dispositivos de reproducción multimedia, los teléfonos móviles y las cámaras digitales ofrecen experiencias continuamente más enriquecidas año tras año.</span><span class="sxs-lookup"><span data-stu-id="22c94-129">GPS devices, media playback devices, mobile phones, and digital cameras deliver consistently richer experiences year after year.</span></span> <span data-ttu-id="22c94-130">Esta tendencia también puede verse en la diversidad de contenido gráfico en películas, televisores, juegos de vídeo y en la Web.</span><span class="sxs-lookup"><span data-stu-id="22c94-130">This trend can also be seen in the diversity of graphical content in film, television, video games, and on the Web.</span></span> <span data-ttu-id="22c94-131">Para seguir el ritmo de estos cambios, se pide a los desarrolladores de forma coherente que realicen sus aplicaciones Windows existentes en el siguiente nivel de riqueza visual.</span><span class="sxs-lookup"><span data-stu-id="22c94-131">To keep pace with these changes, developers are consistently asked to take their existing Windows applications to the next level of visual richness.</span></span>

<span data-ttu-id="22c94-132">Los procesadores de gráficos de los equipos modernos de Windows también han evolucionado de forma constante, por avances en gráficos de juegos de vídeo y en partes de la experiencia de Windows, como Windows Media Center y Aero.</span><span class="sxs-lookup"><span data-stu-id="22c94-132">Graphics processors in modern Windows PCs have also been evolving steadily, driven by advances in video game graphics and parts of the Windows experience like Windows Media Center and Aero.</span></span> <span data-ttu-id="22c94-133">Algunas aplicaciones de Windows pueden aprovechar las GPU modernas mediante Microsoft Direct3D y Windows Presentation Foundation (WPF).</span><span class="sxs-lookup"><span data-stu-id="22c94-133">Some Windows applications can take advantage of modern GPUs by using Microsoft Direct3D and Windows Presentation Foundation (WPF).</span></span> <span data-ttu-id="22c94-134">Mientras que Direct3D sirve aplicaciones de gráficos 3D de gama alta y WPF aborda las necesidades de los desarrolladores de .NET, existen huecos para los desarrolladores que tienen grandes códigos base de representación de código basados en GDI e GDI+, o que desean incorporar gráficos 2D de alta calidad en sus aplicaciones basadas en Direct3D.</span><span class="sxs-lookup"><span data-stu-id="22c94-134">While Direct3D serves high-end 3-D graphics applications and WPF addresses the needs of .NET developers, there are gaps for developers who have large existing codebases of rendering code based on GDI and GDI+ or who want to incorporate high-quality 2-D graphics in their Direct3D-based applications.</span></span>

<span data-ttu-id="22c94-135">Por último, la necesidad de una API de gráficos que se puede usar en un servicio se ha convertido en un requisito emergente para los desarrolladores que trabajan en escenarios empresariales y de desarrollo web.</span><span class="sxs-lookup"><span data-stu-id="22c94-135">Finally, the need for a graphics API that can be used in a service has become an emerging requirement for developers working in enterprise and Web-development scenarios.</span></span> <span data-ttu-id="22c94-136">Las API de representación existentes se centran en la representación del lado cliente en una sesión de un solo usuario.</span><span class="sxs-lookup"><span data-stu-id="22c94-136">Existing rendering APIs focus on client-side rendering in a single user session.</span></span> <span data-ttu-id="22c94-137">Como tal, pueden ser breves en áreas de robustez y escalabilidad cuando se usan en un contexto de servicio.</span><span class="sxs-lookup"><span data-stu-id="22c94-137">As such, they can fall short in areas of robustness and scalability when used in a service context.</span></span> <span data-ttu-id="22c94-138">Se necesita una nueva API para solucionar este requisito.</span><span class="sxs-lookup"><span data-stu-id="22c94-138">A new API is required to address this.</span></span>

## <a name="high-performance-with-maximum-availability"></a><span data-ttu-id="22c94-139">Alto rendimiento con disponibilidad máxima</span><span class="sxs-lookup"><span data-stu-id="22c94-139">High Performance with Maximum Availability</span></span>

## <a name="interoperability"></a><span data-ttu-id="22c94-163">Interoperabilidad</span><span class="sxs-lookup"><span data-stu-id="22c94-163">Interoperability</span></span>

<span data-ttu-id="22c94-164">La integración de la representación basada en Direct2D es más fácil para los desarrolladores a través de la interoperabilidad de nivel de superficie con GDI y Direct3D.</span><span class="sxs-lookup"><span data-stu-id="22c94-164">Integrating Direct2D-based rendering is made easier for developers through surface-level interoperability with GDI and Direct3D.</span></span> <span data-ttu-id="22c94-165">Las aplicaciones que representan contenido principalmente con GDI, GDI+ o Direct3D pueden empezar a usar Direct2D para representar áreas específicas de su aplicación y, con el tiempo, se mueven a un modelo donde la representación se realiza principalmente a través de Direct2D, usando GDI principalmente para complementos o extensibilidad heredada.</span><span class="sxs-lookup"><span data-stu-id="22c94-165">Applications that render content primarily with GDI, GDI+, or Direct3D, can begin by using Direct2D to render specific areas of their application, and over time move to a model where rendering is performed primarily via Direct2D, using GDI primarily for plug-ins or legacy extensibility.</span></span>
