# <a name="private-communications-technology"></a>Tecnología de comunicaciones privadas

La tecnología de comunicaciones privadas (PCT) se ha sustituido por [*Capa de sockets seguros*](../secgloss/s-gly.md) 3,0 y el protocolo de seguridad de la [capa de transporte](transport-layer-security-protocol.md).

Schannel solo admite PCT 1,0 por compatibilidad con versiones anteriores. no debe usarse para el desarrollo nuevo. PCT es compatible con la mayoría de los conjuntos de cifrado compatibles con SSL 2,0.
