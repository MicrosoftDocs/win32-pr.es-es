# <a name="icem01"></a>ICEM01

ICEM01 valida que el mecanismo de ICE funciona. Esta ICE usa la propiedad [**Time**](time.md) para obtener la hora y devuelve la hora del sistema o ninguna.

El módulo de combinación ICEs se almacena en un archivo. Cub de módulo de combinación denominado Mergemod. Cub y no en el archivo. Cub que contiene el ICEs usado para la validación del paquete.

## <a name="result"></a>Resultado

ICEM01 envía un mensaje que le indica la hora a la que el instalador llamó a ICEM01. Esta ICE nunca debe devolver un error.
