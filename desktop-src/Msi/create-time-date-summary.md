# <a name="create-timedate-summary-property"></a>Crear propiedad de Resumen de fecha y hora

La propiedad de **Resumen crear fecha y hora** transmite la hora y la fecha en que un autor creó el paquete de instalación, la transformación o el paquete de revisión.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Versión<br/> | Windows Installer 5,0 en Windows Server 2012, Windows 8, Windows Server 2008 R2 o Windows 7. Windows Installer 4,0 o Windows Installer 4,5 en Windows Server 2008 o Windows Vista. Windows Installer en Windows Server 2003 o Windows XP<br/> |
