# <a name="errortype-property"></a>Error. Type (propiedad)

La propiedad **tipo** de solo lectura devuelve uno de los valores de msmErrorType, que indica el tipo de error representado por este objeto.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Error.Type
```



## <a name="property-value"></a>Valor de propiedad

## <a name="c"></a>C++
