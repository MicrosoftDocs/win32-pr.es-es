# <a name="creating-an-instance-of-the-control"></a>Crear una instancia del control

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

El acceso al control de Microsoft Agent requiere que primero cree una instancia del control. La manera más fácil de hacerlo es colocar una instancia del control en un formulario. Es posible que tenga que agregar el control al cuadro de herramientas antes de agregarlo al formulario. Dependiendo del lenguaje de programación, también puede crear el control en tiempo de ejecución.
