# <a name="pointpoint-constructors"></a>Constructores Point. Point

En este tema se enumeran los constructores de la clase [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) . Para obtener una lista completa de clases, vea **Point (clase**).

### <a name="overload-list"></a>Lista de sobrecarga



| Constructor                                                | Descripción                                                                                                                                                                      |
|:-----------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Punto (tamaño&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inconstsize_))   | Crea un objeto de [**punto**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) mediante un objeto de [**tamaño**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-size) para inicializar los miembros de datos **X** **e y.**<br/> |
| [**Punto (punto&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inconstpoint_)) | Crea un nuevo objeto [**Point**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) y copia los miembros de datos de otro objeto **Point** .<br/>                                           |
| [**Point (INT, INT)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point(inint_inint))  | Crea un objeto de [**punto**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) mediante dos enteros para inicializar los miembros **de datos** **X** e y.<br/>                                       |
| [**Punto ()**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-point-point)             | Crea un objeto de [**punto**](/windows/desktop/api/gdiplustypes/nl-gdiplustypes-point) e inicializa los miembros de datos **X** **e y a** cero. Éste es el constructor predeterminado.<br/>               |
