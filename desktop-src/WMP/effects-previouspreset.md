# <a name="effectspreviouspreset"></a>EFFECTs. previousPreset

El método **previousPreset** muestra el valor preestablecido anterior de la visualización actual.

``` syntax
        elementID.previousPreset()
```

## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.

## <a name="remarks"></a>Observaciones

Si el valor predeterminado actual es el primero de la lista de la visualización actual, el último valor preestablecido se convierte en actual.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|------------------------------------------------------|
| Versión<br/> | Windows Media Player versión 7,0 o posterior<br/> |
