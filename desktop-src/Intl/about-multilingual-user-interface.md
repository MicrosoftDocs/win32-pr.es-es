# <a name="about-multilingual-user-interface"></a>Acerca de Interfaz de usuario multilingüe

Interfaz de usuario multilingüe (MUI) es una tecnología que proporciona a los usuarios una interfaz de usuario localizada para aplicaciones globalizadas y administración de recursos de lenguaje de interfaz de usuario en el Windows operativo. Se proporciona compatibilidad para agregar la funcionalidad de MUI a las aplicaciones globalizadas para que se ejecuten en Windows Vista y versiones posteriores, así como en muchos sistemas operativos Windows Vista. Los modelos de localización y administración de recursos de MUI mejoran el desarrollo, las pruebas y la compatibilidad con software listo para el mundo.

Entre las características de la tecnología DEA, se incluyen las siguientes:

-   Administración simplificada de idiomas para la localización de software. MUI permite cambiar la configuración de idioma del sistema operativo a un idioma compatible según las preferencias del usuario.
-   Tecnología de recursos innovadora basada en la división del código binario de la aplicación de los archivos de recursos de lenguaje. Puede agregar recursos para un lenguaje adicional sin necesidad de volver a compilar o volver a vincular la aplicación. Los recursos localizados adicionales se convierten en complementos opcionales.
-   Interfaz completa de programación de aplicaciones (API). Windows Vista y versiones posteriores exponen la API de MUI para que la use en la adición de la funcionalidad de MUI a las aplicaciones globalizadas. Las funciones de administración de idiomas permiten que las aplicaciones admitan una amplia variedad de lenguajes de interfaz de usuario. La API también incluye funciones de carga de recursos que permiten al cargador de recursos cargar recursos en Windows Vista y versiones posteriores, así como sistemas operativos Windows Vista.
-   Un conjunto de herramientas de administración de paquetes de idioma que proporcionan flexibilidad en la administración de imágenes de implementación internacional. Estas herramientas no están dirigidas a los desarrolladores y, por tanto, no se tratan en el SDK. Puede encontrar más información en el Windows Automated Installation Kit disponible en Technet.

La ventaja más visible de MUI es que varios usuarios pueden compartir la misma estación de trabajo y ver la interfaz de usuario en distintos idiomas. Las empresas y oem se beneficiarán de la funcionalidad que tienen para implantar, admitir y mantener imágenes multilingües con una sola instalación. Sin embargo, quizás la principal ventaja de MUI se debe a las eficiencias obtenidas al desarrollar, crear y dar servicio a la aplicación. Puede enviar un archivo binario de funcionalidad básica aplicable a todas las plataformas, independientemente del lenguaje de interfaz de usuario, lo que reduce significativamente los esfuerzos de desarrollo y pruebas. Si tiene que emitir una actualización o un Service Pack, se aplicará a todos los idiomas admitidos sin ningún esfuerzo de ingeniería adicional. La compatibilidad posterior con idiomas adicionales se convierte en un proyecto de localización en lugar de un proyecto de desarrollo de software completo.
