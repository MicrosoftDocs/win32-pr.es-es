# <a name="nonencapsulated-unions"></a>Uniones no superadas

Una unión no superada es una unión que no está contenida en una estructura con su discriminante. La unión nonencapsulated **\[ \_ \]** se indica mediante la presencia del tipo de modificador de atributo de tipo y el modificador de atributo de **\[ campo \_ es \]**.

Para obtener una descripción del formato de una unión no superada, vea [**union**](union.md).

La forma de las uniones debe ser la misma entre plataformas para garantizar la interconectividad.
