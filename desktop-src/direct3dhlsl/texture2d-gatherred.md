# <a name="texture2dgatherred-methods"></a>Texture2D:: GatherRed (métodos)

Devuelve los componentes rojo de los cuatro valores de textura de un [**Texture2D**](sm5-object-texture2d.md)que se utilizarían en una operación de filtrado bilineal.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                                  | Descripción                                                                                                                                     |
|:--------------------------------------------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherRed (S, Float, int)**](sm5-object-texture2d-gatherred.md)                                         | Devuelve los componentes rojo de los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal.<br/>                                 |
| [**GatherRed (S, Float, int, uint)**](t2d-gatherred-s-float-int-uint-.md)                                   | Devuelve los componentes rojo de los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal, junto con el estado de asignación de mosaicos.<br/> |
| [**GatherRed (S, Float, INT2, INT2, INT2, INT2)**](t2d-gatherred-s-float-int2-int2-int2-int2-.md)             | Devuelve los componentes rojo de los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal.<br/>                                 |
| [**GatherRed (S, Float, INT2, INT2, INT2, INT2, uint)**](t2d-gathergreen-s-float-int2-int2-int2-int2-uint-.md) | Devuelve los componentes rojo de los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal, junto con el estado de asignación de mosaicos.<br/> |
