# <a name="xmshort4-operators"></a>Operadores XMSHORT4

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                               | Descripción                                                                                                          |
|-------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**operador =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmshort4-operator-assign)<br/> | Asigna los datos del componente de vector de una instancia de `XMSHORT4` a la instancia actual de `XMSHORT4` . <br/> |
