# <a name="locale_idaylzero"></a>CONFIGURACIÓN REGIONAL \_ IDAYLZERO

Especificador para ceros iniciales en campos de día solo para una fecha corta. El número máximo de caracteres permitido para esta cadena es dos, incluido un carácter nulo de terminación. Es preferible que la aplicación use la [constante \_ LOCALE SSHORTDATE](locale-sshortdate.md) en lugar de LOCALE \_ IDAYLZERO.



| Value | Significado                   |
|-------|---------------------------|
| 0     | Sin ceros a la izquierda durante días |
| 1     | Ceros iniciales para días    |
