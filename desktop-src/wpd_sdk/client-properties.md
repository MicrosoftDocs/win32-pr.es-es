# <a name="client-properties"></a>Propiedades del cliente

Los dispositivos portátiles de Windows admiten las siguientes propiedades de cliente.



| Propiedad                                          | VarType                   | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
|---------------------------------------------------|---------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_ \_ acceso deseado al cliente de \_ WPD**                  | **VT \_ UI4**               | Valor que especifica que el cliente requiere permisos de acceso únicos, como GENÉRICOs de \_ solo lectura. Si no se establece esta propiedad, WPD supone una \_ escritura genérica de lectura genérica \| \_ , que es el valor predeterminado recomendado.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **\_ \_ versión principal del cliente de WPD \_**                   | **VT \_ UI4**               | Número que especifica la versión principal del cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **\_ \_ versión secundaria del cliente de WPD \_**                   | **VT \_ UI4**               | Número que especifica la versión secundaria del cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| **\_nombre del cliente de WPD \_**                             | **VT \_ LPWStr**            | Cadena de nombre para la aplicación cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| **\_revisión del cliente de WPD \_**                         | **VT \_ UI4**               | Número que especifica la revisión (o el número de compilación) del cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **\_ \_ \_ calidad \_ de servicio de la seguridad del cliente de WPD \_**   | **VT \_ UI4**               | Valor que especifica la calidad de servicio de seguridad para la conexión con el controlador. Esto se relaciona con la calidad de las marcas de servicio de la seguridad de **CreateFile**.<br/> Por ejemplo, estas marcas permiten o deniegan a un controlador la suplantación del cliente. \_Se asume la identificación de seguridad si no se proporciona este valor. Normalmente, las aplicaciones deben establecer este valor en \_ SUplantación de seguridad, ya que algunos controladores requerirán la suplantación para poder funcionar. Sin embargo, es posible que las aplicaciones que se ejecutan con estrictos requisitos de seguridad con un subconjunto conocido de dispositivos deseen restringir la capacidad del controlador para identificar o suplantar al autor de la llamada. Tenga en cuenta que reducir el nivel de suplantación a cualquier valor inferior a la \_ SUplantación de seguridad puede hacer que algunos controladores tengan una funcionalidad reducida.<br/>                                         |
| **\_modo de \_ uso compartido de cliente de WPD \_**                      | **VT \_ UI4**               | Valor que especifica el modo de uso compartido que el cliente solicita al controlador. Los valores posibles son los mismos que para **CreateFile**, por ejemplo, \_ lectura de recurso compartido de archivos \_ , escritura de \_ recurso compartido de archivos \_ , etc.<br/> \_ \_ \| \_ \_ Se supone que se ha leído el recurso compartido de archivos lectura recurso compartido de archivos si el cliente no especifica este parámetro. Este es el valor predeterminado recomendado. Sin embargo, cuando se requiere un modo de recurso compartido diferente, por ejemplo, \_ lectura de recurso compartido de archivos \_ para solicitar solo el uso compartido de lectura, este parámetro es obligatorio.<br/>                                                                                                                                                                                                                                                                                                                                                          |
| **\_clave privada de la \_ \_ aplicación WMDRM \_ \_ del cliente de WPD** | **VT \_ Vector \_ \| VT \_ UI1** | Valor que especifica la clave privada de la aplicación Windows Media DRM del cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| **\_certificado de \_ \_ aplicación WMDRM del cliente \_ de WPD**  | **VT \_ Vector \_ \| VT \_ UI1** | Valor que especifica el certificado de Windows Media DRM del cliente.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| **\_cookie de \_ evento de cliente de WPD \_**                    | **VT \_ LPWStr**            | Una cookie suministrada por el cliente devuelta por el controlador en eventos publicados como resultado directo de las operaciones emitidas por este cliente. Se trata de una propiedad opcional que el controlador puede proporcionar como un parámetro de evento. Los clientes suelen usar este valor para optimizar el control de eventos. Por ejemplo, si el cliente elimina un objeto, su controlador de eventos puede optar por omitir el evento resultante sabiendo que ya tomó la acción adecuada (por ejemplo, actualizó la vista en el subproceso que emitió la eliminación).                                                                                                                                                                                                                                                                                                                                                                |
| **\_ \_ tamaño mínimo del \_ búfer de resultados de \_ cliente de WPD \_**   | **VT \_ UI4**               | Especifica el tamaño mínimo de búfer (en bytes) para enviar comandos al controlador. Los clientes pueden utilizar esta opción para recomendar el tamaño mínimo del búfer interno que se usa para enviar comandos al controlador. Esto puede dar lugar a un menor consumo de memoria para los clientes que solo envían comandos con tamaños de datos pequeños, como obtener o establecer propiedades. El controlador puede invalidar este tamaño de búfer estableciendo **PortableDeviceResultsBufferMin** en su sección de parámetros de dispositivo. Este tamaño de búfer debe ser 1024 o superior. Si ni el cliente ni el controlador establecen el tamaño del búfer, el valor predeterminado es 1 MB.                                                                                                                                                                                                                                                  |
| **\_ \_ \_ cierre manual del cliente \_ de WPD \_ desconectado**    | **VT \_ bool**              | Una opción avanzada para los clientes que llaman manualmente a **IPortableDevice:: Close** o **IPortableDeviceService:: Close** para cada objeto desconectado, en lugar de depender de la API para hacerlo. Se asume FALSE si el cliente no especifica este parámetro. Este es el valor predeterminado recomendado si el cliente no está escuchando **mensajes \_ DEVICECHANGE de WM** . Los clientes que especifican esta opción normalmente escuchan los mensajes de **WM \_ DEVICECHANGE** para el dispositivo y, a continuación, cierran manualmente las instancias de **IPortableDevice** y **IPortableDeviceService** en la desconexión del dispositivo. Los clientes que eligen esta opción deben administrar cuidadosamente la duración de cada instancia de **IPortableDevice** y **IPortableDeviceService** mediante la liberación de estos objetos cuando el dispositivo está desconectado. Si no lo hace, se producirán pérdidas en los procesos de WUDFHost.exe para el controlador. |
