# <a name="loading-and-previewing-a-project"></a>Carga y vista previa de un Project

\[Esta API no se admite y puede modificarse o no estar disponible en el futuro.\]

En este tema se describe cómo cargar y obtener una vista previa de un archivo de proyecto [con DirectShow Editing Services](directshow-editing-services.md) (DES). Presenta una aplicación de ejemplo que carga un archivo de proyecto XML en una escala de tiempo vacía y representa la escala de tiempo para la versión preliminar. Puede modificar la aplicación de ejemplo para que funcione con cualquier archivo de proyecto XML de ejemplo que se incluye con este SDK.

Este artículo contiene los temas siguientes.

-   [Carga de un Project archivo](loading-a-project-file.md)
-   [Vista previa de la Project](previewing-the-project.md)
-   [Vista previa de un Project: código de ejemplo](previewing-a-project--example-code.md)
