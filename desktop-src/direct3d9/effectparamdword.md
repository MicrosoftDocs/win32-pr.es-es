# <a name="effectparamdword"></a>EffectParamDWord

Define un valor DWORD de efecto. Esta plantilla ha reemplazado la plantilla [**EffectDWord**](effectdword.md) .

``` syntax
template EffectParamDWord
{
    < E13963BC-AE51-4c5d-B00F-CFA3A9D97CE5 >
    STRING ParamName;
    DWORD Value;
} 
```

Donde:

-   ParamName: nombre del parámetro.
-   Valor del parámetro de valor.
