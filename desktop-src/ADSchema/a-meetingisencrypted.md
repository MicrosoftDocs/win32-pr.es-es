# <a name="meetingisencrypted-attribute"></a>Atributo meetingIsEncrypted

Esto es **TRUE** si se va a cifrar la reunión.



| Entrada | Value |
|-------------------|----------------------------------------------------------------------------------|
| CN                | meetingIsEncrypted                                                               |
| Ldap-Display-Name | meetingIsEncrypted                                                               |
| Size              | \-                                                                               |
| Actualizar privilegios  | Cualquiera puede actualizar este objeto en función de la seguridad del objeto que se va a crear. |
| Frecuencia de actualización  | \-                                                                               |
| Attribute-Id      | 1.2.840.113556.1.4.585                                                           |
| System-Id-Guid    | 11b6cc8e-48c4-11d1-a9c3-0000f80367c1                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
