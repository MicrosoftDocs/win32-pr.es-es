# <a name="whats-new-for-printing-in-windows-7"></a>Novedades de la impresión en Windows 7

Describe las novedades de los documentos y la impresión en Windows 7.

## <a name="documents-and-printing"></a>Documentos e impresión 

Los documentos y la impresión se han mejorado en Windows 7 para incluir las siguientes características y mejoras nuevas.



|                                                                                                                |                                                                                                                                                                                                                                                                                                                          |
|----------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [API de documento XPS](/previous-versions/windows/desktop/dd316976(v=vs.85))<br/>                                                        | La [API de documento XPS](/previous-versions/windows/desktop/dd316976(v=vs.85)) le permite crear documentos XPS desde su programa Win32 y guardarlos como un archivo de documento o enviarlos directamente a una impresora.<br/> También puede usar la [API de documento XPS](/previous-versions/windows/desktop/dd316976(v=vs.85)) para modificar documentos XPS existentes en un programa Win32.<br/> |
| [API de firma digital XPS](/previous-versions/windows/desktop/ff819108(v=vs.85))<br/>                                      | La [API de firma digital XPS](/previous-versions/windows/desktop/ff819108(v=vs.85)) le permite agregar firmas digitales a documentos XPS desde su programa Win32.<br/> También puede usar la API de firma digital XPS para comprobar si un documento XPS es auténtico y no alterado.<br/>                                          |
| [API de impresión XPS](xps-printing.md)<br/>                                                                   | La [API de impresión XPS](xpsprint-api.md) le permite imprimir documentos XPS desde su programa Win32.<br/>                                                                                                                                                                                                                   |
| Mejoras en el [convertidor de documentos XPS de Microsoft (MXDC)](microsoft-xps-document-converter--mxdc-.md)<br/> | El [MXDC](microsoft-xps-document-converter--mxdc-.md) se ha mejorado para que pueda usar la funcionalidad GDI existente para crear contenido XPS e imprimir en la ruta de impresión XPS desde los programas Win32 con mayor fidelidad gráfica que la que era posible en Windows Vista.<br/>                                   |
