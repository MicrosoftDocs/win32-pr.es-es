# <a name="winnt-schema"></a>Esquema de WinNT

El esquema WinNT define el conjunto de objetos ADSI y su jerarquía de clases para una Windows NT 4.0. Los [objetos ADSI de WinNT](adsi-objects-of-winnt.md) incluyen Usuario, Equipo, Dominio, Grupo, Recurso, Sesión, Servicio, etc. La [sección Jerarquía de clases de objetos WinNT](winnt-object-class-hierarchy.md) que sigue describe las relaciones jerárquicas entre estos objetos.

Cada objeto implementa un conjunto de interfaces ADSI. Sin embargo, de una interfaz implementada, no se admiten todas las propiedades y métodos. [Propiedades obligatorias y](winnt-schemaampaposs-mandatory-and-optional-properties.md) opcionales del esquema WinNT enumera las propiedades admitidas por las clases de objetos WinNT.
