# <a name="candidate-lists"></a>Listas de candidatos

Una lista de candidatos se representa mediante una estructura [**CANDIDATELIST**](/windows/win32/api/imm/ns-imm-candidatelist) que contiene una matriz de cadenas que especifica los caracteres o cadenas de caracteres entre los que el usuario puede elegir. La aplicación compatible con el IME puede recuperar información de la lista de candidatos mediante las funciones [**ImmGetCandidateListCount**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelistcounta) y [**ImmGetCandidateList**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelista) .
