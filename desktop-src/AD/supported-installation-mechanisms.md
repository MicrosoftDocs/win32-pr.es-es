# <a name="supported-installation-mechanisms"></a>Mecanismos de instalación admitidos

Active Directory Domain Services admiten cuatro mecanismos para extender el esquema de Active Directory. Puede extender el esquema de Active Directory mediante

-   [Scripts LDIF](ldif-scripts.md) (preferido)
-   [Scripts de valores separados por comas (CSV)](comma-separated-value-csv-scripts.md)
-   [Extensión de programación](programmatic-extension.md)
-   y [extendiendo la interfaz de usuario para los objetos de directorio](extending-the-user-interface-for-directory-objects.md).
