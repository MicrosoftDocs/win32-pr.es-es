# <a name="microsoft-digest-ssp"></a>Microsoft Digest SSP

Microsoft Digest es un [*proveedor de compatibilidad para seguridad*](../secgloss/s-gly.md) (SSP) que implementa el protocolo de acceso implícita, un protocolo de autenticación ligera para entidades implicadas en las comunicaciones basadas en el protocolo de transferencia de hipertexto (http) o el nivel de seguridad de autenticación simple (SASL).

Microsoft Digest proporciona un mecanismo de respuesta de desafío simple para autenticar a los clientes. Este SSP está pensado para que lo usen las aplicaciones cliente/servidor mediante el uso de comunicaciones basadas en HTTP o SASL.

> [!IMPORTANT]
> No se admite la autenticación implícita entre bosques.
