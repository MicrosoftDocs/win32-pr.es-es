# <a name="rectfcontains-methods"></a>RectF. Contains (métodos)

En este tema se enumeran los métodos Contains de la clase [**RectF**](/windows/win32/api/gdiplustypes/nl-gdiplustypes-rectf) . Para obtener una lista completa de los métodos de la clase **RectF** , consulte [métodos RectF](-gdiplus-class-rectf-methods.md).

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                            | Descripción                                                                                                                                           |
|:------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Contains (RectF&)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstrectf_))   | El método [**RectF:: Contains**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstrectf_)) determina si otro rectángulo está dentro de este rectángulo.<br/>   |
| [**Contains (& de PointF)**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstpointf_))    | El método [**RectF:: Contains**](/windows/win32/api/gdiplustypes/nf-gdiplustypes-rectf-contains(inconstpointf_)) determina si un punto está dentro de este rectángulo.<br/>               |
| [**Contains (REAL, REAL)**](/previous-versions//ms534956(v=vs.85)) | El método [**RectF:: Contains**](/previous-versions//ms534956(v=vs.85)) determina si el punto (*x*, *y*) está dentro de este rectángulo.<br/> |
