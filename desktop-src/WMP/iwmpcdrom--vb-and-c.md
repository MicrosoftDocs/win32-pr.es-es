# <a name="iwmpcdrom-vb-and-c-interface"></a>Interfaz IWMPCdrom (VB y C#)

Proporciona una manera de tener acceso a un CD o DVD en su unidad.

La interfaz **IWMPCdrom** expone las siguientes propiedades.

## <a name="members"></a>Miembros

La interfaz **IWMPCdrom (VB y C#)** tiene estos tipos de miembros:

-   [Métodos](#methods)
-   [Propiedades](#properties)

### <a name="methods"></a>Métodos

La interfaz **IWMPCdrom (VB y C#)** tiene estos métodos.



| Método                                                     | Descripción                                     |
|:-----------------------------------------------------------|:------------------------------------------------|
| [**expulsar**](wmplibiwmpcdrom-iwmpcdrom-eject--vb-and-c.md) | Expulsa el CD o DVD de la unidad.<br/> |



 

### <a name="properties"></a>Propiedades

La interfaz **IWMPCdrom (VB y C#)** tiene estas propiedades.



| Propiedad                                                                                | Tipo de acceso          | Descripción                                                                                                                                  |
|:----------------------------------------------------------------------------------------|:---------------------|:---------------------------------------------------------------------------------------------------------------------------------------------|
| [**driveSpecifier**](wmplibiwmpcdrom-iwmpcdrom-drivespecifier--vb-and-c.md)<br/> | Solo lectura<br/> | Obtiene la letra de unidad de CD o DVD.<br/>                                                                                                  |
| [**Lista de reproducción**](wmplibiwmpcdrom-iwmpcdrom-playlist--vb-and-c.md)<br/>             | Solo lectura<br/> | Obtiene una interfaz [**IWMPPlaylist**](/previous-versions/windows/desktop/api/wmp/nn-wmp-iwmpplaylist) que representa las pistas de un CD o las entradas de título de nivel de raíz de un DVD.<br/> |
