# <a name="iupdatehistoryentrycollection-properties"></a>Propiedades de IUpdateHistoryEntryCollection

La interfaz [**IUpdateHistoryEntryCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatehistoryentrycollection) define las siguientes propiedades.



| Propiedad                                                    | Descripción                                                                                                          |
|-------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatehistoryentrycollection-get__newenum) | Obtiene una interfaz [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) que se utiliza para enumerar la colección. |
| [**Contabiliza**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatehistoryentrycollection-get_count)        | Obtiene el número de elementos de la colección.                                                                       |
| [**Elemento**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatehistoryentrycollection-get_item)          | Obtiene una interfaz [**IUpdateHistoryEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatehistoryentry) en la colección.                              |
