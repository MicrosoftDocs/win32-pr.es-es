# <a name="suggested-advtuisequence"></a>AdvtUISequence sugerido

No cree la tabla AdvtUISequence. El instalador no usa esta tabla. La tabla AdvtUISequence no debe existir en la base de datos de instalación o se debe dejar vacía. El nombre de la aplicación que se anunciará debe pasarse al instalador durante la inicialización.
