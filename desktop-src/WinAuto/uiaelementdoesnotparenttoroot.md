# <a name="uiaelementdoesnotparenttoroot"></a>UiaElementDoesNotParentToRoot

## <a name="text"></a>Texto

La cadena primaria del elemento no va al elemento raíz

## <a name="type"></a>Tipo

Advertencia

## <a name="description"></a>Descripción

Se rompe la navegación del elemento especificado a la raíz del árbol de la interfaz de usuario.

Este problema puede producir resultados imprevisibles para una aplicación que intenta navegar por el árbol de la interfaz de usuario.

## <a name="possible-causes"></a>Causas posibles

Implementación incorrecta de la jerarquía de elementos primarios y secundarios del árbol de la interfaz de usuario.
