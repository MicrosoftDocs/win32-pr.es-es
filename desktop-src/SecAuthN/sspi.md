# <a name="sspi"></a>SSPI

En esta sección se ofrece información acerca de la [*interfaz del proveedor de compatibilidad para seguridad*](../secgloss/s-gly.md) (SSPI). SSPI permite a una aplicación utilizar diversos modelos de seguridad disponibles en un equipo o una red sin cambiar la interfaz para el sistema de seguridad. Entre los temas se incluyen los siguientes:

-   [Modelo SSPI](sspi-model.md)
-   [Opciones de SSPI para aplicaciones distribuidas](sspi-options-for-distributed-applications.md)
-   [Semántica de contexto SSPI](sspi-context-semantics.md)
-   [Paquetes de SSP proporcionados por Microsoft](ssp-packages-provided-by-microsoft.md)
-   [Paquetes de seguridad personalizados](custom-security-packages.md)
-   [Información de error extendida](extended-error-information.md)
