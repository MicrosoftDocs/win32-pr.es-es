# <a name="file-terminals"></a>Terminales de archivo

En TAPI 3,1, el objeto terminal se expande para incluir un terminal de archivo. Los terminales de archivo proporcionan un medio sencillo para realizar operaciones comunes, por ejemplo, la grabación de una secuencia de audio o secuencias en un archivo o una reproducción posterior del audio grabado en una secuencia.

Se usan cinco objetos básicos para manipular los terminales de archivo:

-   Terminal de reproducción de archivos
-   Terminal de grabación de archivos
-   Pista de reproducción de archivos
-   Pista de grabación de archivos
-   Evento de archivo

Los objetos de terminal de archivo se pueden considerar como una abstracción para el almacenamiento, que suele ser un archivo. Las pistas corresponden a los flujos de datos de audio individuales de ese almacenamiento, como cuando se graba la voz.

Para obtener más información acerca de los terminales de archivos, consulte:

-   [Terminal de reproducción de archivos y terminal de grabación de archivos](file-playback-terminal-and-file-recording-terminal.md)
-   [Seguimiento de terminales](track-terminals.md)

Para obtener más información y ejemplos de código y listas de tareas relacionadas con los terminales de archivos, consulte [uso de terminales de archivos](using-file-terminals.md).
