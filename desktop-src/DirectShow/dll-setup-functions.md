# <a name="dll-setup-functions"></a>Funciones de instalación de DLL

Estas funciones registran un filtro.



| Función                                                         | Descripción                                        |
|------------------------------------------------------------------|----------------------------------------------------|
| [**AMovieDllRegisterServer**](amoviedllregisterserver.md)       | Obsoleto.                                          |
| [**AMovieDllRegisterServer2**](amoviedllregisterserver2.md)     | Registra y anula el registro de filtros.                 |
| [**AMovieDllUnregisterServer**](amoviedllunregisterserver.md)   | Obsoleto.                                          |
| [**AMovieSetupRegisterFilter**](amoviesetupregisterfilter.md)   | Obsoleto.                                          |
| [**AMovieSetupRegisterFilter2**](amoviesetupregisterfilter2.md) | Registra el mérito, los pin y los tipos de medios de un filtro. |
