# <a name="swbempropertyqualifiers_-property"></a>SWbemProperty. Qualifiers \_ (propiedad)

La propiedad **Qualifiers \_** del objeto [**SWbemProperty**](swbemproperty.md) devuelve un objeto [**SWbemQualifierSet**](swbemqualifierset.md) que es una colección de calificadores para esta propiedad WMI. Esta propiedad es de solo lectura.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemProperty.Qualifiers_ As Object
```
