# <a name="wmcontentgroupdescription"></a>WM/ContentGroupDescription

El **atributo WM/ContentGroupDescription** contiene una descripción del grupo de contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMContentGroupDescription

## <a name="data-type"></a>Tipo de datos

**CADENA DE TIPO WMT \_ \_**

## <a name="remarks"></a>Comentarios

Los grupos de contenido son colecciones de medios a los que pueden pertenecer archivos individuales. Por ejemplo, un archivo de audio puede ser una canción que es una pista de un disco compacto que forma parte de un conjunto con formato boxed. De forma similar, un clip de vídeo puede formar parte de una serie.
