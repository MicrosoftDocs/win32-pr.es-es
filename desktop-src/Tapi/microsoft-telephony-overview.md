# <a name="microsoft-telephony-overview"></a>Introducción a la telefonía de Microsoft

La telefonía integra equipos con redes y dispositivos de comunicaciones. En telefonía clásica, el dispositivo era un teléfono y la red era la red telefónica conmutada pública (RTC). La telefonía moderna amplía continuamente la gama de dispositivos y redes, y actualmente abarca dispositivos como cámaras y redes, como Internet.

Las posibles aplicaciones de telefonía incluyen:

-   Conferencia IP Multimedia multidifusión
-   Llamadas de voz a través de Internet (VoIP)
-   Aplicaciones cliente y servidor del centro de distribución de llamadas automáticas (ACD)
-   Llamadas de voz básicas en la RTC
-   Controles de tipo PBX como el estacionamiento de llamadas y el reenvío selectivo en una red telefónica corporativa sin necesidad de invertir en hardware especializado.
-   Sistemas de respuesta interactiva de voz (IVR)
-   Colaboración en tiempo real
