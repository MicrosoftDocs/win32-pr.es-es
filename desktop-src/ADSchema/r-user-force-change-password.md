# <a name="user-force-change-password-extended-right"></a>Derecho extendido para forzar el cambio de contraseña del usuario

Permite restablecer una contraseña en una cuenta de usuario.



| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | User-Force-Change-Password           |
| Display-Name | Restablecimiento de contraseña                       |
| Rights-GUID  | 00299570-246d-11d0-a768-00aa006e0529 |
