# <a name="using-standard-dialog-boxes-to-handle-objects-in-active-directory-domain-services"></a>Usar cuadros de diálogo estándar para controlar objetos en Active Directory Domain Services

Windows proporciona una biblioteca de cuadros de diálogo comunes para operaciones comunes, como abrir archivos, examinar archivos, etc. A partir de Windows 2000, el sistema proporciona cuadros de diálogo que se pueden usar para las operaciones de interfaz de usuario comunes en Active Directory Domain Services.

Los siguientes cuadros de diálogo están disponibles para su uso con Active Directory Domain Services:

-   [Selector de objetos de directorio](directory-object-picker.md)
-   [Explorador de dominio](domain-browser.md)
-   [Explorador de contenedor](container-browser.md)
-   [Hojas de propiedades de Active Directory usuarios y equipos](active-directory-users-and-computers-property-sheets.md)

Además, los asistentes para crear Active Directory Domain Services también se pueden invocar desde una aplicación de. Para obtener más información, vea [invocación de los asistentes para creación desde la aplicación](invoking-creation-wizards-from-your-application.md).
