# <a name="media-sources"></a>Orígenes multimedia

*Los orígenes multimedia* son objetos que generan datos multimedia en la canalización Media Foundation datos. En esta sección se describen con detalle las API de origen de medios. Lea esta sección si está implementando un origen multimedia personalizado o usando un origen multimedia fuera de la canalización Media Foundation datos.

Si la aplicación usa la capa de control, solo debe usar un subconjunto limitado de las API de origen multimedia. Para obtener información, vea el tema [Using Media Sources with the Media Session](using-media-sources-with-the-media-session.md).
