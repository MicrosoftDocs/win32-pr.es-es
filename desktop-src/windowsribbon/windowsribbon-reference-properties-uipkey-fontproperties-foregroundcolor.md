# <a name="ui_pkey_fontproperties_foregroundcolor"></a>UI \_ PKEY \_ FontProperties \_ foregroundcolor

Identifica la propiedad PKEY de la interfaz de usuario \_ \_ FontProperties \_ foregroundcolor.

```
propertyDescription
   name = UI_PKEY_FontProperties_ForegroundColor
   shellPKey = UI_PKEY_FontProperties_ForegroundColor
   formatID = 00000308-7363-696e-8441798acf5aebb7
   propID = 308
   typeInfo
      type = COLORREF
```

## <a name="remarks"></a>Observaciones

La \_ interfaz \_ de usuario PKEY FontProperties \_ foregroundcolor se usa en una aplicación, junto con la [interfaz de usuario \_ PKEY \_ FontProperties \_ ForegroundColorType](windowsribbon-reference-properties-uipkey-fontproperties-foregroundcolortype.md), para consultar la configuración de la galería de **colores de texto** .

El valor predeterminado es `0x00000000`.
