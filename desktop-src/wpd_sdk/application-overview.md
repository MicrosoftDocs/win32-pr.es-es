# <a name="application-overview"></a>Información general de la aplicación

Dispositivos portátiles de Windows (WPD) proporciona una infraestructura que permite el acceso de equipos a dispositivos multifunción que almacenan varios medios y contenido no multimedia. Esta infraestructura permite dispositivos convergentes y se centra inicialmente en dispositivos de almacenamiento inteligente, como reproductores multimedia portátiles, cámaras digitales fijas y teléfonos móviles.

WPD consta de una interfaz de controlador de dispositivo (DDI) y un conjunto de interfaces de programación de aplicaciones (API). En esta información general se describe cómo puede aprovechar las ventajas de estas API en su aplicación Windows.

La infraestructura de WPD ofrece mecanismos de control y representación de dispositivos muy extensibles. Esto permite a los proveedores de soluciones verticales usar la API de WPD para crear experiencias de usuario mejoradas que se encuentran fuera del conjunto estandarizado de WPD. Entre los ejemplos se incluyen software proporcionado por el proveedor que realiza actualizaciones de firmware o diagnósticos de dispositivos (para soporte remoto).
