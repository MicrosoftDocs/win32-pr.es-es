# <a name="candidate-lists"></a>Listas de candidatos

Una lista de candidatos se representa mediante una estructura [**CANDIDATELIST**](/windows/win32/api/imm/ns-imm-candidatelist) que contiene una matriz de cadenas que especifican los caracteres o cadenas de caracteres entre los que puede elegir el usuario. La aplicación con IME puede recuperar información de la lista de candidatos mediante las funciones [**ImmGetCandidateListCount**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelistcounta) [**e ImmGetCandidateList.**](/windows/desktop/api/Imm/nf-imm-immgetcandidatelista)
