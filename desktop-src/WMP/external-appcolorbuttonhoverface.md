# <a name="externalappcolorbuttonhoverface"></a>External.appColorButtonHoverFace

> [!Note]  
> En este tema se describe la funcionalidad diseñada para su uso por las tiendas en línea. No se admite el uso de esta funcionalidad fuera del contexto de una tienda en línea.

 

La **propiedad appColorButtonHoverFace** recupera el color de desplazamiento del botón actual para la interfaz Reproductor de Windows Media usuario. Este color se muestra cuando el usuario mueve el puntero del mouse sobre un botón.

``` syntax
window.external.appColorButtonHoverFace
      
```

## <a name="possible-values"></a>Valores posibles

Esta propiedad es una cadena de solo **lectura.**

## <a name="remarks"></a>Comentarios

Los usuarios pueden cambiar el color de la interfaz Reproductor de Windows Media usuario. Puede usar esta propiedad para personalizar la apariencia de la página web hospedada para que coincida con el reproductor.
