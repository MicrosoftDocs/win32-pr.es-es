# <a name="dynamic-virtual-channels-reference"></a>Referencia de canales virtuales dinámicos

Las siguientes interfaces de cliente de canal virtual dinámico (DVC) son compatibles con Servicios de Escritorio remoto (anteriormente conocido como Terminal Services).

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[API de cliente de DVC](dvc-client-apis.md)
</dt> <dd>

Las API de cliente de canal virtual dinámico (DVC) se implementan específicamente para el cliente de Conexión a Escritorio remoto (RDC) de la conexión.

</dd> <dt>

[API de servidor DVC](dvc-server-apis.md)
</dt> <dd>

Las API de servidor de canal virtual dinámico (DVC) se implementan mediante el servidor de Conexión a Escritorio remoto (RDC) de la conexión.

</dd> </dl>
