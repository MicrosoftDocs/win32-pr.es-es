# <a name="eventmanifest-schema-elements"></a>Elementos de esquema EventManifest

A continuación se muestran los elementos que define el esquema EventManifest. Esta sección contiene los nombres de los elementos que se usan en el manifiesto; sin embargo, para obtener los detalles de cada elemento, vea el tipo complejo que contiene el elemento.



| Elemento                                                                                                                  | Descripción                                                                                                                                                                                                                                                                          |
|--------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**autocopia de seguridad (ChannelLoggingType)**](eventmanifestschema-autobackup-channelloggingtype-element.md)                     | Determina si se debe crear un nuevo archivo de registro cuando el archivo de registro actual alcanza su tamaño máximo.<br/>                                                                                                                                                                           |
| [**binario (TemplateItemType)**](eventmanifestschema-binary-templateitemtype-element.md)                                 | Contiene los datos binarios proporcionados por la API del [registro de eventos](/windows/desktop/EventLog/event-logging) .<br/>                                                                                                                                                                                     |
| [**Mapa de bits (MapType)**](eventmanifestschema-bitmap-maptype-element.md)                                                   | Define una lista de pares de nombre/valor que asignan valores de bit y valores de cadena.<br/>                                                                                                                                                                                                 |
| [**bufferSize (ChannelPublishingType)**](eventmanifestschema-buffersize-channelpublishingtype-element.md)               | Contiene la cantidad de memoria, en kilobytes, asignada para los búferes de la sesión.<br/>                                                                                                                                                                                           |
| [**canal (ChannelListType)**](eventmanifestschema-channel-channellisttype-element.md)                                 | Define un canal en el que se pueden registrar los eventos.<br/>                                                                                                                                                                                                                          |
| [**canales (MetadataType)**](eventmanifestschema-channels-metadatatype-element.md)                                     | Define una lista de canales en los que los proveedores pueden registrar eventos.<br/>                                                                                                                                                                                                             |
| [**canales (ProviderType)**](eventmanifestschema-channels-providertype-element.md)                                     | Define una lista de canales en los que los proveedores pueden registrar eventos.<br/>                                                                                                                                                                                                             |
| [**clockType (ChannelPublishingType)**](eventmanifestschema-clocktype-channelpublishingtype-element.md)                 | Resolución del reloj que se va a usar al registrar la marca de tiempo para cada evento.<br/>                                                                                                                                                                                                   |
| [**controlGuid (ChannelPublishingType)**](eventmanifestschema-controlguid-channelpublishingtype-element.md)             | Identifica el GUID de sesión para una sesión de ETW que contiene eventos WPP.<br/>                                                                                                                                                                                                  |
| [**datos (StructDefinitionType)**](eventmanifestschema-data-structdefinitiontype-element.md)                             | Define un elemento de datos que desea incluir en la estructura.<br/>                                                                                                                                                                                                            |
| [**datos (TemplateItemType)**](eventmanifestschema-data-templateitemtype-element.md)                                     | Define un elemento de datos que desea incluir con el evento.<br/>                                                                                                                                                                                                              |
| [**Descripción (InputType)**](eventmanifestschema-description-inputtype-element.md)                                     | Descripción del tipo de entrada.<br/>                                                                                                                                                                                                                                          |
| [**evento (DefinitionType)**](eventmanifestschema-event-definitiontype-element.md)                                       | Define un evento que el proveedor puede registrar.<br/>                                                                                                                                                                                                                              |
| [**Event (OpCode)**](eventmanifestschema-event-opcode-element.md)                                                       | No disponible. **Windows Server 2008 y Windows Vista:** Define un evento para un código de operación específico de la tarea. No está disponible a partir de Windows 7.<br/> <br/>                                                                                                                  |
| [**evento (TaskEventDefinitionType)**](eventmanifestschema-event-taskeventdefinitiontype-element.md)                     | No disponible. **Windows Server 2008 y Windows Vista:** Define un evento para un evento específico de tarea. No está disponible a partir de Windows 7.<br/> <br/>                                                                                                                   |
| [**eventos (InstrumentationType)**](eventmanifestschema-events-instrumentationtype-element.md)                           | Contiene una lista de proveedores definidos en el manifiesto.<br/>                                                                                                                                                                                                            |
| [**eventos (ProviderType)**](eventmanifestschema-events-providertype-element.md)                                         | Contiene una lista de los eventos que define el proveedor.<br/>                                                                                                                                                                                                                  |
| [**importChannel (ChannelListType)**](eventmanifestschema-importchannel-channellisttype-element.md)                     | Identifica un canal definido por otro proveedor.<br/>                                                                                                                                                                                                           |
| [**instrumentationManifest**](eventmanifestschema-instrumentationmanifest-element.md)                                   | Nodo raíz del manifiesto.<br/>                                                                                                                                                                                                                                            |
| [**instrumentación (instrumentationManifest)**](eventmanifestschema-instrumentation-instrumentationmanifest-element.md) | Define los tipos de instrumentación que contiene el manifiesto (por ejemplo, la instrumentación de eventos o de contadores).<br/>                                                                                                                                                          |
| [**Intype (InputTypeListType)**](eventmanifestschema-intype-inputtypelisttype-element.md)                               | Define un tipo de datos de entrada.<br/>                                                                                                                                                                                                                                               |
| [**Intypes (TypeListType)**](eventmanifestschema-intypes-typelisttype-element.md)                                       | Contiene una lista de definiciones de tipo de entrada.<br/>                                                                                                                                                                                                                                |
| [**palabra clave (KeywordListType)**](eventmanifestschema-keyword-keywordlisttype-element.md)                                 | Define una palabra clave que identifica una categoría de eventos.<br/>                                                                                                                                                                                                                   |
| [**Palabras clave (ChannelPublishingType)**](eventmanifestschema-keywords-channelpublishingtype-element.md)                   | Máscara de máscara que determina la categoría de eventos que se escriben en el canal.<br/>                                                                                                                                                                                         |
| [**Palabras clave (MetadataType)**](eventmanifestschema-keywords-metadatatype-element.md)                                     | Define una lista de palabras clave que clasifican eventos.<br/>                                                                                                                                                                                                                        |
| [**Palabras clave (ProviderType)**](eventmanifestschema-keywords-providertype-element.md)                                     | Define una lista de palabras clave que clasifican eventos.<br/>                                                                                                                                                                                                                        |
| [**latencia (ChannelPublishingType)**](eventmanifestschema-latency-channelpublishingtype-element.md)                     | Tiempo de espera antes de vaciar los búferes, en milisegundos.<br/>                                                                                                                                                                                                            |
| [**nivel (ChannelPublishingType)**](eventmanifestschema-level-channelpublishingtype-element.md)                         | Nivel de gravedad de los eventos que se van a escribir en el canal.<br/>                                                                                                                                                                                                                 |
| [**nivel (LevelListType)**](eventmanifestschema-level-levellisttype-element.md)                                         | Define un valor de gravedad que determina el nivel de detalle de los eventos durante el registro.<br/>                                                                                                                                                                                          |
| [**niveles (MetadataType)**](eventmanifestschema-levels-metadatatype-element.md)                                         | Define una lista de niveles que especifican la gravedad de un evento.<br/>                                                                                                                                                                                                           |
| [**niveles (ProviderType)**](eventmanifestschema-levels-providertype-element.md)                                         | Define una lista de niveles que especifican la gravedad de un evento.<br/>                                                                                                                                                                                                           |
| [**localización (instrumentationManifest)**](eventmanifestschema-localization-instrumentationmanifest-element.md)       | Define un grupo de recursos localizados al que se hace referencia en el manifiesto.<br/>                                                                                                                                                                                               |
| [**registro (ChannelType)**](eventmanifestschema-logging-channeltype-element.md)                                         | Define las propiedades del archivo de registro que respalda el canal, como su capacidad y si son secuenciales o circulares.<br/>                                                                                                                                             |
| [**asignación (BitMapType)**](eventmanifestschema-map-bitmaptype-element.md)                                                   | Define la asignación entre un valor de bit y un valor de cadena.<br/>                                                                                                                                                                                                               |
| [**asignación (PatternMapType)**](eventmanifestschema-map-patternmaptype-element.md)                                           | Define las expresiones regulares que se usan para buscar una cadena coincidente en la cadena de mensaje y modificarla.<br/>                                                                                                                                                                        |
| [**asignación (ValueMapType)**](eventmanifestschema-map-valuemaptype-element.md)                                               | Define la asignación entre un valor entero y un valor de cadena.<br/>                                                                                                                                                                                                          |
| [**Maps (ProviderType)**](eventmanifestschema-maps-providertype-element.md)                                             | Define una lista de pares de nombre/valor a los que se puede hacer referencia en la sección de plantilla del manifiesto.<br/>                                                                                                                                                                        |
| [**maxBuffers (ChannelPublishingType)**](eventmanifestschema-maxbuffers-channelpublishingtype-element.md)               | Número máximo de búferes que se van a asignar a la sesión.<br/>                                                                                                                                                                                                                |
| [**maxSize (ChannelLoggingType)**](eventmanifestschema-maxsize-channelloggingtype-element.md)                           | Tamaño máximo, en bytes, del archivo de registro.<br/>                                                                                                                                                                                                                              |
| [**mensaje (tabla de mensajes)**](eventmanifestschema-message-messagetable-element.md)                                      | Contiene una referencia a una cadena en la sección de localización del manifiesto.<br/>                                                                                                                                                                                             |
| [**Elemento messageTable (EventsType)**](eventmanifestschema-messagetable-instrumentationtype-element.md)                | Contiene una referencia a una cadena en la sección de localización del manifiesto.<br/>                                                                                                                                                                                             |
| [**metadatos (instrumentationManifest)**](eventmanifestschema-metadata-instrumentationmanifest-element.md)               | Contiene valores globales a los que se puede hacer referencia en otros manifiestos.<br/>                                                                                                                                                                                                         |
| [**minBuffers (ChannelPublishingType)**](eventmanifestschema-minbuffers-channelpublishingtype-element.md)               | Número mínimo de búferes que se van a asignar a la sesión.<br/>                                                                                                                                                                                                                |
| [**namedQueries (MetadataType)**](eventmanifestschema-namedqueries-metadatatype-element.md)                             | No se utiliza. Define una lista de consultas con nombre que consultan la cadena de mensaje de evento de un valor y realizan una acción especificada si se encuentran.<br/>                                                                                                                                        |
| [**namedQueries (ProviderType)**](eventmanifestschema-namedqueries-providertype-element.md)                             | No se utiliza. Define una lista de consultas con nombre que consultan la cadena de mensaje de evento de un valor y realizan una acción especificada si se encuentran.<br/>                                                                                                                                        |
| [**OpCode (OpcodeListType)**](eventmanifestschema-opcode-opcodelisttype-element.md)                                     | Contiene un valor numérico que identifica la actividad o un punto dentro de una actividad que la aplicación estaba realizando cuando generó el evento (por ejemplo, inicialización o cierre).<br/>                                                                                |
| [**OpCode (TaskEventDefinitionType)**](eventmanifestschema-opcode-taskeventdefinitiontype-element.md)                   | No disponible. **Windows Server 2008 y Windows Vista:** Contiene un código de operación específico de la tarea para un evento. Se supone que todas las definiciones de código de operación son específicas de la tarea para la tarea que contiene las definiciones de evento. No está disponible a partir de Windows 7.<br/> <br/> |
| [**OpCodes (MetadataType)**](eventmanifestschema-opcodes-metadatatype-element.md)                                       | Define una lista de códigos de tareas que se pueden usar para agrupar eventos dentro de una tarea.<br/>                                                                                                                                                                                                 |
| [**códigos de la (ProviderType)**](eventmanifestschema-opcodes-providertype-element.md)                                       | Define una lista de códigos de tareas que se pueden usar para agrupar eventos dentro de una tarea.<br/>                                                                                                                                                                                                 |
| [**códigos de la (TaskType)**](eventmanifestschema-opcodes-tasktype-element.md)                                               | Define una lista de códigos de tiempo específicos de la tarea.<br/>                                                                                                                                                                                                                                  |
| [**outtype (InputType)**](eventmanifestschema-outtype-inputtype-element.md)                                             | Una lista de tipos de datos de salida que determinan cómo se representan los datos.<br/>                                                                                                                                                                                                      |
| [**patternMap (PatternMapListType)**](eventmanifestschema-patternmap-patternmaplisttype-element.md)                     | Define una asignación entre dos expresiones regulares. Una expresión se utiliza para buscar una cadena coincidente en la cadena de mensaje y la otra se usa para modificar la cadena antes de que el servicio la vuelva a colocar en la cadena de mensaje.<br/>                                             |
| [**patternMaps (NamedQueryType)**](eventmanifestschema-patternmaps-namedquerytype-element.md)                           | Define una lista de pares de expresiones regulares que se utilizan para modificar la cadena de mensaje.<br/>                                                                                                                                                                                     |
| [**proveedor (EventsType)**](eventmanifestschema-provider-eventstype-element.md)                                         | Define un proveedor y los metadatos que usa para definir sus eventos.<br/>                                                                                                                                                                                                    |
| [**publicación (ChannelType)**](eventmanifestschema-publishing-channeltype-element.md)                                   | Define las propiedades de registro para la sesión que utiliza el canal.<br/>                                                                                                                                                                                                     |
| [**recursos (LocalizationType)**](eventmanifestschema-resources-localizationtype-element.md)                           | Define un grupo de tablas de cadenas que contienen las cadenas localizadas a las que se hace referencia en el manifiesto.<br/>                                                                                                                                                                  |
| [**retención (ChannelLoggingType)**](eventmanifestschema-retention-channelloggingtype-element.md)                       | Determina si el archivo de registro es un archivo de registro secuencial o circular. <br/>                                                                                                                                                                                                    |
| [**SID (ChannelPublishingType)**](eventmanifestschema-sidtype-channelpublishingtype-element.md)                     | Determina si se debe incluir un identificador de seguridad (SID) de la entidad de seguridad con cada evento escrito en el canal.<br/>                                                                                                                                                        |
| [**cadena (StringTableType)**](eventmanifestschema-string-stringtabletype-element.md)                                   | Define una cadena traducida.<br/>                                                                                                                                                                                                                                               |
| [**stringTable (LocalizationType)**](eventmanifestschema-stringtable-localizationtype-element.md)                       | Define una lista de cadenas traducidas a las que se puede hacer referencia en el manifiesto.<br/>                                                                                                                                                                                              |
| [**stringTable (recursos)**](eventmanifestschema-stringtable-localizationtype-element.md)                              | Define una lista de cadenas traducidas a las que se puede hacer referencia en el manifiesto.<br/>                                                                                                                                                                                              |
| [**struct (TemplateItemType)**](eventmanifestschema-struct-templateitemtype-element.md)                                 | Define una estructura que incluye uno o más elementos de datos que desea incluir con el evento.<br/>                                                                                                                                                                         |
| [**tarea (DefinitionType)**](eventmanifestschema-task-definitiontype-element.md)                                         | No disponible. **Windows Server 2008 y Windows Vista:** Define un evento específico de la tarea que el proveedor puede registrar. No está disponible a partir de Windows 7.<br/> <br/>                                                                                                     |
| [**tarea (TaskListType)**](eventmanifestschema-task-tasklisttype-element.md)                                             | Define un componente o subcomponente de una aplicación.<br/>                                                                                                                                                                                                                    |
| [**tareas (ProviderType)**](eventmanifestschema-tasks-providertype-element.md)                                           | Define una lista de tareas que un proveedor puede usar para agrupar los eventos.<br/>                                                                                                                                                                                                          |
| [**plantilla (TemplateListType)**](eventmanifestschema-template-templatelisttype-element.md)                             | Plantilla que define los datos que se van a incluir con un evento.<br/>                                                                                                                                                                                                                |
| [**plantillas (ProviderType)**](eventmanifestschema-templates-providertype-element.md)                                   | Define una lista de plantillas que especifican los datos que se van a incluir con los eventos.<br/>                                                                                                                                                                                             |
| [**tipos (MetadataType)**](eventmanifestschema-types-metadatatype-element.md)                                           | Define una lista de tipos XML.<br/>                                                                                                                                                                                                                                              |
| [**UserData (TemplateItemType)**](eventmanifestschema-userdata-templateitemtype-element.md)                             | Fragmento XML que se utiliza para representar los datos de evento.<br/>                                                                                                                                                                                                                    |
| [**valueMap (MapType)**](eventmanifestschema-valuemap-maptype-element.md)                                               | Define una lista de pares de nombre y valor que asignan valores enteros y valores de cadena.<br/>                                                                                                                                                                                             |
| [**xmlType (XmlTypeListType)**](eventmanifestschema-xmltype-xmltypelisttype-element.md)                                 | Define un tipo XML.<br/>                                                                                                                                                                                                                                                      |
| [**xmlTypes (TypeListType)**](eventmanifestschema-xmltypes-typelisttype-element.md)                                     | Define una lista de tipos de salida que el servicio usa para determinar cómo se representa un tipo de datos de entrada.<br/>                                                                                                                                                                       |
