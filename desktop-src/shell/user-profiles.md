# <a name="user-profiles"></a>Perfiles de usuario

La información de configuración que se puede conservar para cada usuario se guarda en los [perfiles de usuario](about-user-profiles.md). Esta información incluye la organización del escritorio, los grupos de programas personales, los elementos de programa de los grupos de programas personales, los colores de pantalla, los protectores de pantalla, las conexiones de red, las conexiones de impresora, la configuración del mouse, el tamaño y la posición de la ventana, etc. Cuando un usuario inicia sesión, el sistema carga el perfil del usuario y configura el entorno de acuerdo con la información del perfil.

Esta sección incluye los temas siguientes.

-   [Acerca de los perfiles de usuario](about-user-profiles.md)
-   [Referencia de perfiles de usuario](user-profiles-reference.md)
