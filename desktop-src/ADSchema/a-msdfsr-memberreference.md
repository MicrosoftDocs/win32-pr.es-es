# <a name="ms-dfsr-memberreference-attribute"></a>atributo MS-DFSR-MemberReference

Contiene el vínculo hacia delante al objeto [**MS-DFSR-Member**](c-msdfsr-member.md) .



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-DFSR-MemberReference                 |
| Nombre para mostrar de LDAP | msDFSR-MemberReference                  |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.100             |
| System-ID-GUID    | 261337aa-f1c3-44b2-bbea-c88d49e6f0c7    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
