# <a name="vss-backups-and-restores-of-the-cluster-database"></a>Copias de seguridad de VSS y restauraciones de la base de datos de clúster

Los clústeres de conmutación por error usan el Servicio de instantáneas de volumen (VSS) para realizar copias de seguridad y restaurar la configuración del clúster de conmutación por error. Para obtener más información, consulte [copia de seguridad y restauración de la configuración del clúster de conmutación por error mediante VSS](/previous-versions/windows/desktop/mscs/backing-up-and-restoring-the-failover-cluster-configuration-using-vss).
