# <a name="french-overseas-territories"></a>Territorios de ultramar en francés

Para obtener una explicación de las tablas siguientes, vea [tablas y asignaciones](tables-and-assignments.md).

Los índices de frecuencia de la tabla de difusión del territorio francés de ultramar son adecuados para su uso como números de canal; los índices de frecuencia de cable no son adecuados para su uso como números de canal.

### <a name="french-overseas-territories-broadcast-assignments"></a>Las asignaciones de difusión de territorios de ultramar en francés


```C++
         4L,            // Lowest channel
         9L,            // Highest channel

       175250000L,      // 4
       183250000L,      // 5
       191250000L,      // 6
       199250000L,      // 7
       207250000L,      // 8
       215250000L,      // 9
```



### <a name="french-overseas-territories-cable-assignments"></a>Asignaciones de cable de territorios de ultramar en francés


```C++
         4L,            // Lowest channel
         9L,            // Highest channel

       175250000L,      // 4
       183250000L,      // 5
       191250000L,      // 6
       199250000L,      // 7
       207250000L,      // 8
       215250000L,      // 9
```
