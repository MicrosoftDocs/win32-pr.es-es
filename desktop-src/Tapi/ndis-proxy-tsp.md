# <a name="ndis-proxy-tsp"></a>NDIS Proxy TSP

El proxy NDIS es un componente en modo kernel que expone miniportas de NDIS como líneas TAPI. El TSP del proxy NDIS (Ndptsp.tsp) permite a TAPI comunicarse con el proxy NDIS.

El TSP del proxy NDIS se instala con los sistemas operativos Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Edition, Windows 98 y Windows 95.
