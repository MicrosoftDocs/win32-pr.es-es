# <a name="ms-pki-key-recovery-agent-class"></a>ms-PKI-Key-Recovery-Agent (Clase)

Se crea una instancia de objeto del agente de recuperación de claves (SSM) para cada servidor de certificados instalado (con un nombre común único) durante la instalación del servidor de certificados. Si dos CA tienen el mismo nombre común durante la instalación de la ca, compartirán una única instancia de objeto DEV.



| Entrada | Value |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | ms-PKI-Key-Recovery-Agent                                                                                                                                                    |
| Ldap-Display-Name | msPKI-Key-Recovery-Agent                                                                                                                                                     |
| Actualizar privilegios  | Un administrador que instale una ENTIDAD de certificación deberá poder crear una instancia DE LAN en el contenedor DE LAN. Los servidores de certificados instalados deben poder actualizar el atributo userCertificate. |
| Frecuencia de actualización  | Se agregarán algunos certificados como máximo cada pocos meses.                                                                                                                   |
| Schema-Id-Guid    | 26ccf238-a08e-4b86-9a82-a8c9ac7ee5cb                                                                                                                                         |
