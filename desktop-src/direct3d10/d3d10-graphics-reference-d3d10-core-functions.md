# <a name="direct3d-10-core-functions"></a>Funciones principales de Direct3D 10

Esta sección contiene información sobre las siguientes funciones principales:



| Functions                                                                | Descripción                                     |
|--------------------------------------------------------------------------|-------------------------------------------------|
| [**D3D10CreateBlob**](/windows/desktop/api/D3D10Misc/nf-d3d10misc-d3d10createblob)                               | Cree un búfer.                                |
| [**D3D10CreateDevice**](/windows/desktop/api/D3D10Misc/nf-d3d10misc-d3d10createdevice)                           | Cree un dispositivo Direct3D 10.                    |
| [**D3D10CreateDevice1**](/windows/desktop/api/D3D10_1/nf-d3d10_1-d3d10createdevice1)                         | Cree un dispositivo Direct3D 10,1.                  |
| [**D3D10CreateDeviceAndSwapChain**](/windows/desktop/api/D3D10Misc/nf-d3d10misc-d3d10createdeviceandswapchain)   | Cree un dispositivo Direct3D 10 y una cadena de intercambio.   |
| [**D3D10CreateDeviceAndSwapChain1**](/windows/desktop/api/D3D10_1/nf-d3d10_1-d3d10createdeviceandswapchain1) | Cree un dispositivo Direct3D 10,1 y una cadena de intercambio. |
