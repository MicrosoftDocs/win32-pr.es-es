# <a name="dn-reference-update-attribute"></a>Atributo DN-Reference-Update

Si se cambia el nombre de un objeto, este atributo se utiliza para realizar el seguimiento de todos los nombres anteriores y actuales asignados a un objeto para que los objetos vinculados puedan encontrarlos.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | DN-Reference-actualizar                     |
| Nombre para mostrar de LDAP | dNReferenceUpdate                       |
| Tamaño              | \-                                      |
| Actualizar privilegio  | Lo establece el sistema.              |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1242                 |
| System-ID-GUID    | 2df90d86-009f-11d2-aa4c-00c04fd7d83a    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
