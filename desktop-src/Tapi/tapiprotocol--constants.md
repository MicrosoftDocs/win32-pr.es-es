# <a name="tapiprotocol_-constants"></a>Constantes de TAPIPROTOCOL \_

Los siguientes protocolos están definidos por TAPI 3 y los CLSID están en Tapi3. lib. Los proveedores de servicios pueden definir protocolos adicionales.

<dl> <dt>

<span id="CLSID_TAPIPROTOCOL_PSTN"></span><span id="clsid_tapiprotocol_pstn"></span>**\_TAPIPROTOCOL de CLSID**
</dt> <dd> <dl> <dt>



Protocolo de red de telefonía conmutada pública.


</dt> </dl> </dd> <dt>

<span id="CLSID_TAPIPROTOCOL_H323"></span><span id="clsid_tapiprotocol_h323"></span>**CLSID TAPIPROTOCOL \_ H323**
</dt> <dd> <dl> <dt>



Protocolo H323.


</dt> </dl> </dd> <dt>

<span id="CLSID_TAPIPROTOCOL_Multicast"></span><span id="clsid_tapiprotocol_multicast"></span><span id="CLSID_TAPIPROTOCOL_MULTICAST"></span>**Multidifusión TAPIPROTOCOL CLSID \_**
</dt> <dd> <dl> <dt>



Protocolo de multidifusión.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Observaciones

Las versiones **BSTR** de las constantes de protocolo se declaran para el uso de aplicaciones Visual Basic.

-   BSTR TAPIPROTOCOL \_ cadena \_ PSTN
-   BSTR TAPIPROTOCOL \_ cadena \_ H323
-   Cadena BSTR \_ TAPIPROTOCOL \_ multidifusión
