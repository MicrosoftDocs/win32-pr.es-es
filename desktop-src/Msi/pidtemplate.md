# <a name="pidtemplate-property"></a>Propiedad PIDTemplate

La propiedad **PIDTemplate** contiene la cadena que se usa como plantilla para la propiedad [**PIDKEY**](pidkey.md) . Para obtener la sintaxis utilizada en la plantilla, vea el tipo de control [MaskedEdit](maskededit-control.md) . La [Acción ValidateProductID](validateproductid-action.md) usa este valor para validar el identificador de producto.

Windows Installer en Windows Server 2003 o Windows XP. Consulte los [requisitos de Run-Time de Windows Installer](windows-installer-portal.md) para obtener información sobre la Service Pack mínima de Windows que requiere una versión Windows Installer.<br/> |
