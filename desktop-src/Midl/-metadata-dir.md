# <a name="metadata_dir-switch-midlrt"></a>modificador/metadata_dir (MIDLRT)

El conmutador **/Metadata \_ dir** especifica uno o más directorios que contienen archivos de metadatos de plataforma.

``` syntax
midlrt /metadata_dir metadata_directory
```

## <a name="switch-options"></a>Opciones de conmutador

<dl> <dt>

*directorio de metadatos \_* 
</dt> <dd>

Especifica uno o más directorios que contienen archivos de metadatos de plataforma.

</dd> </dl>

## <a name="remarks"></a>Observaciones

Use este modificador para especificar la ubicación del archivo de metadatos principal para Windows, que se denomina Windows. winmd.

## <a name="examples"></a>Ejemplos

**midlrt/Metadata \_ dir dir1 dir2**
