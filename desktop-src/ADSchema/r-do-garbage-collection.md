# <a name="do-garbage-collection-extended-right"></a>Do: derecho extendido de recolección de elementos no utilizados

Derecho extendido para obligar al servicio de directorio a realizar la recolección de elementos no utilizados. Control derecho para obligar al servicio de directorio a realizar la recolección de elementos no utilizados.



| Entrada | Value |
|--------------|--------------------------------------|
| CN           | Do: recolección de elementos no utilizados                |
| Display-Name | Recolección de elementos no utilizados                |
| Rights-GUID  | fec364e0-0a98-11d1-adbb-00c04fd8d5cd |
