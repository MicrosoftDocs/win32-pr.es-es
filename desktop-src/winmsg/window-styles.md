# <a name="window-styles"></a>Estilos de ventana

A continuación se muestran los estilos de ventana. Una vez creada la ventana, estos estilos no se pueden modificar, salvo que se indique lo contrario.

|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="WS_BORDER"></span><span id="ws_border"></span><dl> <dt>**WS \_ BORDE**</dt> <dt>0x00800000L</dt> </dl>                                                                                                                          | La ventana tiene un borde de línea fina.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="WS_CAPTION"></span><span id="ws_caption"></span><dl> <dt>**WS \_ TÍTULO**</dt> <dt>0x00C00000L</dt> </dl>                                                                                                                       | La ventana tiene una barra de título (incluye el estilo de **\_ borde de WS** ).<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="WS_CHILD"></span><span id="ws_child"></span><dl> <dt>**WS \_**</dt> <dt>0x40000000L</dt> secundario </dl>                                                                                                                             | La ventana es una ventana secundaria. Una ventana con este estilo no puede tener una barra de menús. Este estilo no se puede usar con el estilo de **\_ ventana emergente de WS** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="WS_CHILDWINDOW"></span><span id="ws_childwindow"></span><dl> <dt>**WS \_ CHILDWINDOW**</dt> <dt>0x40000000L</dt> </dl>                                                                                                           | Igual que el **estilo \_ secundario WS** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |

Este estilo se usa al crear la ventana primaria.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| <span id="WS_CLIPSIBLINGS"></span><span id="ws_clipsiblings"></span><dl> <dt>**WS \_ CLIPSIBLINGS**</dt> <dt>0x04000000L</dt> </dl>                                                                                                        | Recorta las ventanas secundarias en relación con las otras; es decir, cuando una ventana secundaria determinada recibe un mensaje de [**\_ dibujo de WM**](../gdi/wm-paint.md) , el estilo **WS \_ CLIPSIBLINGS** recorta todas las demás ventanas secundarias superpuestas fuera de la región de la ventana secundaria que se va a actualizar. Si no se especifica **WS \_ CLIPSIBLINGS** y las ventanas secundarias se superponen, es posible que, al dibujar en el área de cliente de una ventana secundaria, dibuje en el área de cliente de una ventana secundaria vecina.<br/>                                                                                                                                                                                |
| <span id="WS_DISABLED"></span><span id="ws_disabled"></span><dl> <dt>**WS \_ Deshabilitado**</dt> <dt>0x08000000L</dt> </dl>                                                                                                                    | La ventana está deshabilitada inicialmente. Una ventana deshabilitada no puede recibir entradas del usuario. Para cambiar esto una vez creada una ventana, use la función [**EnableWindow**](/windows/win32/api/winuser/nf-winuser-enablewindow) .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="WS_DLGFRAME"></span><span id="ws_dlgframe"></span><dl> <dt>**WS \_ DLGFRAME**</dt> <dt>0x00400000L</dt> </dl>                                                                                                                    | La ventana tiene un borde de un estilo que se utiliza normalmente con cuadros de diálogo. | <span id="WS_GROUP"></span><span id="ws_group"></span><dl> <dt>**WS \_ Grupo**</dt> <dt>0x00020000L</dt> </dl>                                                                                                                             | La ventana es el primer control de un grupo de controles. Normalmente, el primer control de cada grupo tiene el estilo **WS \_ TABSTOP** , de modo que el usuario pueda moverse de un grupo a un grupo. Posteriormente, el usuario puede cambiar el foco de teclado de un control del grupo al siguiente control del grupo mediante las teclas de dirección.<br/> Puede activar y desactivar este estilo para cambiar la navegación del cuadro de diálogo. Para cambiar este estilo una vez creada una ventana, use la función [**SetWindowLong**](/windows/win32/api/winuser/nf-winuser-setwindowlonga) .<br/> |
| <span id="WS_HSCROLL"></span><span id="ws_hscroll"></span><dl> <dt>**WS \_ HSCROLL**</dt> <dt>0x00100000L</dt> </dl>                                                                                                                       | La ventana tiene una barra de desplazamiento horizontal.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="WS_ICONIC"></span><span id="ws_iconic"></span><dl> <dt>**WS \_**</dt> <dt>0X20000000L</dt> de iconos </dl>                                                                                                                          | La ventana está minimizada inicialmente. | <span id="WS_MAXIMIZE"></span><span id="ws_maximize"></span><dl> <dt>**WS \_ MAXIMIZAr**</dt> <dt>0x01000000L</dt> </dl>                                                                                                                    | La ventana está maximizada inicialmente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="WS_MAXIMIZEBOX"></span><span id="ws_maximizebox"></span><dl> <dt>**WS \_ MAXIMIZEBOX**</dt> <dt>0x00010000L</dt> </dl>                                                                                                           | La ventana tiene un botón maximizar. No se puede combinar con el estilo **WS \_ ex \_ CONTEXTHELP** . También se debe especificar el estilo **WS \_ SYSMENU** . <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="WS_MINIMIZE"></span><span id="ws_minimize"></span><dl> <dt>**WS \_ MINIMIZAR**</dt> <dt>0x20000000L</dt> </dl>                                                                                                                    | La ventana está minimizada inicialmente. Igual que el estilo de **\_ iconos de WS** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| <span id="WS_MINIMIZEBOX"></span><span id="ws_minimizebox"></span><dl> <dt>**WS \_ MINIMIZEBOX**</dt> <dt>0x00020000L</dt> </dl>                                                                                                           | La ventana tiene un botón minimizar. No se puede combinar con el estilo **WS \_ ex \_ CONTEXTHELP** . También se debe especificar el estilo **WS \_ SYSMENU** . <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <span id="WS_OVERLAPPED"></span><span id="ws_overlapped"></span><dl> <dt>**WS \_ 0x00000000L superpuesto**</dt> <dt></dt> </dl>                                                                                                              | La ventana es una ventana superpuesta. Una ventana superpuesta tiene una barra de título y un borde. Igual que el estilo en **\_ mosaico de WS** .<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| <span id="WS_OVERLAPPEDWINDOW"></span><span id="ws_overlappedwindow"></span><dl> <dt>**WS \_ OVERLAPPEDWINDOW**</dt> <dt>(WS \_ superpuesto \| WS \_ Caption \| WS \_ SYSMENU \| WS \_ THICKFRAME \| WS \_ MINIMIZEBOX \| WS \_ MAXIMIZEBOX)</dt> </dl> | La ventana es una ventana superpuesta.
