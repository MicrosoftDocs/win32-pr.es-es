# <a name="validation-automation"></a>Automatización de validación

Puede usar el Evalcom2.dll para implementar operaciones de validación para los paquetes de instalación y los módulos de combinación.

Las secciones siguientes de esta documentación contienen información sobre el uso de la Windows installer desde las herramientas de creación:

-   [Uso de Evalcom2](using-evalcom2.md)
-   [EvalCom2 Interfaces](evalcom2-interfaces.md)

Se recomienda instalar Evalcom2.dll mediante el instalador de Windows. El paquete Orca.msi proporcionado con el Kit de desarrollo de software (SDK) de Microsoft Windows para Windows XP con Service Pack 2 (SP2) o una versión posterior instala Evalcom2.dll. El identificador del componente que contiene la interfaz COM es {53315473-B8D6-470A-9951-D9AE5C11FC91}.

Para obtener información sobre cómo obtener el SDK Windows, consulte Componentes Windows [SDK para desarrolladores Windows Installer](platform-sdk-components-for-windows-installer-developers.md).
