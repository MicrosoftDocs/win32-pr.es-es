# <a name="about-the-winsnmp-20-addendum"></a>Acerca del anexo de WinSNMP 2.0

El anexo de la versión 2.0 de WinSNMP es una actualización de la especificación de la API de WinSNMP Manager, versión 1.1a. (En este momento, ninguna especificación define la versión 2.0 de la API winSNMP). El anexo se desarrolló conjuntamente con la cooperación, la entrada y el soporte técnico de varias empresas, asociaciones e individuos.

La especificación WinSNMP 1.1a define la interfaz de programación de aplicaciones de WinSNMP Manager para desarrollar aplicaciones de administración de red basadas en SNMP. En el anexo de la versión 2.0 de WinSNMP, se cambia el nombre de la interfaz como api WinSNMP. Esto se debe a que WinSNMP 2.0 admite operaciones SNMP de agente y administrador. Debe estar familiarizado con ambos documentos si está programando en el entorno de WinSNMP.

La implementación de Microsoft WinSNMP es compatible con WinSNMP 2.0. Admite todas las funciones y operaciones definidas en la especificación WinSNMP 1.1a y el anexo WinSNMP 2.0.
