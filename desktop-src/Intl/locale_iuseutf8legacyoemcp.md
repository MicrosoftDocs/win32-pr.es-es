# <a name="locale_iuseutf8legacyoemcp"></a>CONFIGURACIÓN REGIONAL \_ IUSEUTF8LEGACYOEMCP

**A partir de la siguiente versión de Windows:** Se recomienda que las aplicaciones usen Unicode en su lugar. En las máquinas que el usuario ha configurado para usar una página de códigos UTF-8, LOCALE_IUSEUTF8LEGACYOEMCP es la página de códigos OEM heredada que se habría usado en una máquina de Windows heredada que no sea UTF-8.
