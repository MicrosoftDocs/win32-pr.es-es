# <a name="ms-ds-preferred-gc-site-attribute"></a>Atributo ms-DS-Preferred-GC-Site

El administrador de cuentas de seguridad usa el **atributo ms-DS-Preferred-GC-Site** para la expansión de grupos durante la evaluación del token.



| Entrada | Valor |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Preferred-GC-Site                 |
| Ldap-Display-Name | msDS-Preferred-GC-Site                  |
| Size              | \-                                      |
| Actualizar privilegios  | Administrador de dominio                    |
| Frecuencia de actualización  | A discreción del administrador.      |
| Attribute-Id      | 1.2.840.113556.1.4.1444                 |
| System-Id-Guid    | d921b50a-0ab2-42cd-87f6-09cf83a91854    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
