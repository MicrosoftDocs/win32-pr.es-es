# <a name="d1240-source-tranform-still-mapped"></a>D1240: transformación de origen todavía asignado

El mapa de bits pasado a ID2D1SourceTransform::D RAW no se ha desasignado nunca.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación llamó a ID2D1Bitmap1:: Map pero no llamó a ID2D1Bitmap1:: desasignación en un mapa de bits pasado a ID2D1SourceTransform::D RAW.

## <a name="possible-fixes"></a>Posibles correcciones

Asegúrese de que se llama a unwith cuando se llama a Map.
