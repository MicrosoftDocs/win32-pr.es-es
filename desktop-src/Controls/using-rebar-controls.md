# <a name="using-rebar-controls"></a>Usar controles rebar

Esta sección contiene temas que muestran cómo crear y usar los controles rebar.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Cómo crear controles rebar](create-rebar-controls.md)<br/> | Una aplicación crea un control rebar llamando a la función [**CreateWindowEx**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) y especificando [**REBARCLASSNAME**](common-control-window-classes.md) como clase de ventana. La aplicación debe registrar primero la clase de ventana mediante una llamada a la función [**InitCommonControlsEx**](/windows/desktop/api/Commctrl/nf-commctrl-initcommoncontrolsex) , especificando el \_ \_ bit de las clases de ICC Cool en la estructura [**InitCommonControlsEx**](/windows/win32/api/commctrl/ns-commctrl-initcommoncontrolsex) correspondiente. <br/> |
