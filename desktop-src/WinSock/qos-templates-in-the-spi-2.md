# <a name="qos-templates-in-the-spi"></a>Plantillas de QoS en el SPI

Los detalles de implementación de calidad de servicio (QoS) se proporcionan en una sección de QoS independiente y dedicada del Kit de desarrollo de software (SDK) de Microsoft Windows. Consulte la sección [Calidad](/previous-versions/windows/desktop/qos/qos-start-page) de servicio del SDK de Windows para obtener información sobre las directrices de uso de QoS, incluidas las plantillas de QoS disponibles y sus valores [**FLOWSPEC asociados.**](/windows/win32/api/qos/ns-qos-flowspec)
