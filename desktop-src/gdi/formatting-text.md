# <a name="formatting-text-windows-gdi"></a>Aplicar formato a texto (Windows GDI)

Las funciones de formato se pueden dividir en tres categorías: las que recuperan o establecen los [atributos de formato de texto](text-formatting-attributes.md) para un contexto de dispositivo, las que recuperan el ancho de [caracteres](character-widths.md)y las que recuperan el [ancho y el alto](string-widths-and-heights.md)de la cadena.
