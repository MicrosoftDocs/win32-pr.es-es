# <a name="using-transforms"></a>Usar transformaciones

Las transformaciones aumentan la flexibilidad de la instalación de la aplicación al proporcionar una manera de aplicar los cambios a una base de datos sin modificar la base de datos original. Para obtener más información, vea [transformaciones de base de datos](database-transforms.md).

En las siguientes secciones se describe el uso de transformaciones:

[Aplicar transformaciones](applying-transforms.md)

 

[Generar una transformación](generating-a-transform.md)

 

[Usar transformaciones para agregar recursos](using-transforms-to-add-resources.md)
