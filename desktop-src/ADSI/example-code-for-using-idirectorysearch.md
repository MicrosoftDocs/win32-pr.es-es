# <a name="example-code-for-using-idirectorysearch"></a>Código de ejemplo para usar IDirectorySearch

El Kit de desarrollo de software (SDK) de plataforma incluye un tema de ejemplo de código, DsSrch, que proporciona una sencilla aplicación de símbolo del sistema que usa la interfaz [**IDirectorySearch**](/windows/desktop/api/Iads/nn-iads-idirectorysearch) para consultar un servicio de directorio. El ejemplo de código completo se incluye en el SDK de plataforma en el directorio \\ \\ \\ \\ DsSrch Samples NetDs ADSI Samples General \\ .
