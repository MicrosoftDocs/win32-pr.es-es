# <a name="the-propertysheet-object"></a>El objeto PropertySheet

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

El [**objeto PropertySheet**](https://www.bing.com/search?q=**PropertySheet**) proporciona varias propiedades que puede usar si desea manipular el carácter en relación con la hoja de propiedades de Microsoft Agent (también conocida como la ventana Opciones avanzadas de caracteres).

-   [**Alto**](height-property-pso.md)
-   [**Izquierda**](left-property-pso.md)
-   [**Página**](page-property.md)
-   [**Arriba**](top-property-pso.md)
-   [**Visible**](visible-property.md)
-   [**Ancho**](width-property-pso.md)

Si consulta las [**propiedades Height**](height-property-pso.md), [**Left**](left-property-pso.md), [**Top**](top-property-pso.md)y [**Width**](width-property-pso.md) antes de que se haya mostrado la hoja de propiedades, sus valores devuelven como cero (0). Una vez que se muestran, estas propiedades devuelven la última posición y tamaño de la ventana (en relación con la resolución de pantalla actual).
