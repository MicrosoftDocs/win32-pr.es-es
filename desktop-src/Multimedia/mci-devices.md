# <a name="mci-devices"></a>Dispositivos MCI

Cada dispositivo multimedia MCI admite un conjunto básico de comandos MCI de una forma que tenga sentido para ese dispositivo. Por ejemplo, el comando [**Play**](play.md) ([**MCI \_ Play**](mci-play.md)) hace que el dispositivo abierto reproduzca un archivo o una pista, independientemente del tipo de datos con el que funcione el dispositivo. En los temas siguientes se describen los dispositivos MCI y cómo responden a los comandos MCI estándar.

-   [Control de dispositivo](device-control.md)
-   [Reproducir y colocar](playback-and-positioning.md)
-   [Tipos de dispositivos](device-types.md)
-   [Nombres de dispositivo](device-names.md)
-   [Compatibilidad con controladores para comandos MCI](driver-support-for-mci-commands.md)
-   [Comportamiento predeterminado de los controladores](default-behavior-of-drivers.md)
