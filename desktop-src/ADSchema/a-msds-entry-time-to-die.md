# <a name="ms-ds-entry-time-to-die-attribute"></a>atributo MS-DS-entry-Time-to-Die

Contiene el tiempo de expiración absoluto de un objeto dinámico en el directorio.



| Entrada | Value |
|-------------------|---------------------------------------------------------------|
| CN                | MS-DS-entry-Time-to-Die                                       |
| Nombre para mostrar de LDAP | msDS-entry-Time-to-Die                                        |
| Tamaño              | \-                                                            |
| Actualizar privilegio  | El sistema establece este valor.                              |
| Frecuencia de actualización  | \-                                                            |
| Attribute-Id      | 1.2.840.113556.1.4.1622                                       |
| System-ID-GUID    | e1e9bad7-c6dd-4101-A843-794cec85b038                          |
| Sintaxis            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
