# <a name="specifying-protocol-information"></a>Especificar información de protocolo

El Monitor de red puede agregar información a la tabla de fotogramas especificando qué protocolo de bajo nivel (IP o IPX) se encontró y el desplazamiento hasta el principio del protocolo. Para agregar esta información a la tabla de fotogramas, establezca el filtro de captura ANY.
