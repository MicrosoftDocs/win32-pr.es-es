# <a name="router-manager"></a>Administrador de enrutadores

El administrador de enrutadores es el componente de un Windows NT/Windows 2000 que inicia y administra los distintos componentes de enrutamiento. Los componentes incluyen los protocolos de enrutamiento, el administrador de tablas de enrutamiento y el administrador de grupos de multidifusión.
