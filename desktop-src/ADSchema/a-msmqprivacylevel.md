| CN                | Nivel de privacidad de MSMQ                   |
| Ldap-Display-Name | mSMQPrivacyLevel                     |
| Size              | \-                                   |
| Actualizar privilegios  | \-                                   |

| System-Id-Guid    | 9a0dc327-c100-11d1-gni5-0080c76670c0 |
