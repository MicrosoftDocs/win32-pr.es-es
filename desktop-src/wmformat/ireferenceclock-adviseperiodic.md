# <a name="ireferenceclockadviseperiodic-method"></a>IReferenceClock:: AdvisePeriodic (método)

Este método no se implementa.

En otras implementaciones de la interfaz **IReferenceClock** , como en el componente DirectShow® de Microsoft® DirectX®, el método **AdvisePeriodic** se usa para crear una solicitud de notificación periódica que señaliza el objeto de evento especificado cada vez que transcurre el intervalo de tiempo especificado. Este SDK no implementa la funcionalidad de este método, y las llamadas realizadas a él darán como resultado un valor devuelto de E \_ NOTIMPL.

## <a name="syntax"></a>Sintaxis


```C++
 AdvisePeriodic();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.
