# <a name="mf_toponode_locked-attribute"></a>MF \_ TOPONODE \_ atributo bloqueado

Especifica si los tipos de medios se pueden cambiar en este nodo de topología.

## <a name="data-type"></a>Tipo de datos

**UINT32**

Trata como un valor booleano.

## <a name="remarks"></a>Observaciones

Este atributo se aplica a todos los tipos de nodo.

Si el valor de este atributo es distinto de cero, el cargador de topología no cambiará los tipos de medios. Este atributo se establece en **true** cuando el nodo está en uso.

La constante GUID para este atributo se exporta desde mfuuid. lib.
