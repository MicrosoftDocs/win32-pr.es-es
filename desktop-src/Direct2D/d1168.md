# <a name="d1168-write-map-option-requires-cpu-writable-bitmap"></a>D1168: La opción Escribir mapa requiere mapa de bits grabable de CPU

D2D1 \_ MAP OPTIONS WRITE solo es válido en un mapa de bits que admite escritura de \_ \_ CPU






 

## <a name="possible-causes"></a>Causas posibles

Direct2D puede generar este mensaje si una aplicación llama a ID2D1Bitmap1::Map con el atributo D2D1 MAP OPTIONS WRITE, pero el mapa de bits no se puede escribir \_ \_ en LA \_ CPU.

## <a name="possible-fixes"></a>Posibles correcciones

Si una aplicación requiere acceso de escritura mediante ID2D1Bitmap1::Map, asegúrese de que el recurso de Direct3D subyacente tiene el atributo D3D11 \_ CPU \_ ACCESS \_ WRITE.
