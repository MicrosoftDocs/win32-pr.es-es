# <a name="brushes-windows-gdi"></a>Pinceles (Windows GDI)

Un *pincel* es una herramienta de gráficos que las aplicaciones usan para pintar el interior de polígonos, elipses y trazados. Las aplicaciones de dibujo utilizan pinceles para pintar formas; las aplicaciones de procesamiento de texto usan pinceles para pintar reglas. las aplicaciones de diseño asistido por PC (CAD) utilizan pinceles para pintar el interior de las vistas transversales. las aplicaciones de hoja de cálculo y usan pinceles para pintar las secciones de los gráficos circulares y las barras de los gráficos de barras.
