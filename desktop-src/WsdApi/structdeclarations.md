# <a name="structdeclarations-element"></a>Elemento structDeclarations

Genera declaraciones de estructura de C para tipos conocidos.

## <a name="usage"></a>Uso

``` syntax
<structDeclarations/>
```

## <a name="attributes"></a>Atributos

No hay atributos.

## <a name="child-elements"></a>Elementos secundarios

No hay elementos secundarios.

## <a name="parent-elements"></a>Elementos primarios



| Elemento                         | Descripción                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**Archivo**](file.md)<br/> | Genera un archivo del generador de código.<br/> <br/> |



## <a name="remarks"></a>Comentarios

Gran parte del código generado y el código de aplicación hacen referencia a las estructuras para los tipos conocidos. Este elemento se usa para generar archivos de incluir.
