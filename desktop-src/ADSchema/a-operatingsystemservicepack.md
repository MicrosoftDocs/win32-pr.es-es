# <a name="operating-system-service-pack-attribute"></a>Operating-System-Service-Pack (atributo)

La cadena del identificador del Service Pack del sistema operativo (por ejemplo, SP3).



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Sistema operativo-Service-Pack               |
| Nombre para mostrar de LDAP | operatingSystemServicePack                  |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.365                      |
| System-ID-GUID    | 3e978927-8c01-11d0-afda-00c04fd930c9        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
