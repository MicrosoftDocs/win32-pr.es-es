# <a name="recordfieldcount-property"></a>Propiedad record. FieldCount

La propiedad **FieldCount** del objeto [**Record**](record-object.md) es una propiedad de solo lectura que devuelve el número de campos del registro. El acceso de lectura a los campos más allá de este recuento devuelve valores NULL. Error de acceso de escritura.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Record.FieldCount
```
