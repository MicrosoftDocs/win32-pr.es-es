# <a name="printticket-specific-property-descriptions"></a>Descripciones de propiedades específicas de PrintTicket

Este tema no es actual. Para obtener la información más reciente, vea [La especificación de esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

En esta sección se describen las palabras clave públicas que pueden aparecer en printticket.

No es necesario que se admiten todas las palabras clave. Los implementadores de PrintTicket PUEDEN admitir la funcionalidad solo si es aplicable a su propia implementación específica.
