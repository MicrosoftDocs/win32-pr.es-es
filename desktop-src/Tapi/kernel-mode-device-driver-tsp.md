# <a name="kernel-mode-device-driver-tsp"></a>Kernel-Mode de proveedor de dispositivos TSP

El TSP de modo kernel (Kmddsp. TSP) permite que una aplicación administre miniports WAN heredado (por ejemplo, controladores NDIS 4,0) como dispositivos TAPI.

El TSP en modo kernel se instala con los sistemas operativos Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Millennium Edition, Windows 98 y Windows 95.
