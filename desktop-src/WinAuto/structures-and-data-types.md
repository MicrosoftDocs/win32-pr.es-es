# <a name="structures-and-data-types"></a>Estructuras y tipos de datos

Esta sección contiene información sobre las estructuras y los tipos de datos utilizados con Microsoft Active Accessibility.

## <a name="in-this-section"></a>En esta sección



| Tema                                             | Descripción                                                                                                                                                              |
|---------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSAAMENUINFO**](/windows/win32/api/oleacc/ns-oleacc-msaamenuinfo)<br/>   | Los desarrolladores de servidores usan para exponer los nombres de los elementos de menú dibujados por el propietario.<br/>                                                                                      |
| [**HWINEVENTHOOK**](hwineventhook.md)<br/> | Se usa para declarar una función de enlace de eventos de ventana.<br/>                                                                                                                 |
| [**WINEVENTPROC**](/previous-versions/windows/desktop/legacy/dd373882(v=vs.85))<br/>   | Se usa para declarar un puntero a una función de devolución de llamada definida por la aplicación a la que el sistema llama en respuesta a eventos generados por un objeto accesible.<br/> |
