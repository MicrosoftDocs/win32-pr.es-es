# <a name="vol-table-idx-guid-attribute"></a>Atributo Vol-Table-IDX-GUID

Identificador de índice de una entrada de tabla de seguimiento de vínculos.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | Vol-Table-IDX-GUID                                    |
| Nombre para mostrar de LDAP | volTableIdxGUID                                       |
| Tamaño              | 16 bytes                                              |
| Actualizar privilegio  | El sistema establece este valor.                      |
| Frecuencia de actualización  | Siempre que se crea un nuevo vínculo a un archivo.             |
| Attribute-Id      | 1.2.840.113556.1.4.334                                |
| System-ID-GUID    | 1f0075fb-7e40-11d0-afd6-00c04fd930c9                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
