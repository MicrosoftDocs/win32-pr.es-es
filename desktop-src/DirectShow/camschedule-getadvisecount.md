# <a name="camschedulegetadvisecount-method"></a>CAMSchedule. GetAdviseCount, método

El `GetAdviseCount` método recupera el número de solicitudes de notificación pendientes.

## <a name="syntax"></a>Sintaxis


```C++
DWORD GetAdviseCount();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve el número de solicitudes de notificación pendientes.
