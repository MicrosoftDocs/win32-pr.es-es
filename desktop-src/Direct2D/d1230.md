# <a name="d1230-unknown-buffer-precision-support"></a>D1230: compatibilidad de precisión de búfer desconocida

Se realizó una llamada a IsBufferPrecisionSupported con la \_ precisión del búfer de D2D1 \_ \_ desconocida. Esta API devuelve false, aunque la \_ precisión de búfer desconocido de D2D1 \_ \_ puede ser válida como entrada para algunas API.






 

## <a name="possible-causes"></a>Causas posibles

Se realizó una llamada a IsBufferPrecisionSupported con la \_ precisión del búfer de D2D1 \_ \_ desconocida.

## <a name="possible-fixes"></a>Posibles correcciones

Determine si \_ \_ la precisión del búfer de D2D1 \_ desconocida es una precisión de búfer válida para una API determinada; para ello, lea la documentación de esa API en lugar de intentar determinar esto mediante programación.
