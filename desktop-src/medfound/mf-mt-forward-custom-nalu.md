# <a name="mf_mt_forward_custom_nalu-attribute"></a>\_ \_ \_ Atributo Nalu personalizado MF MT Forward \_

Especifica que el descodificador debe reenviar los tipos de unidad de la capa de abstracción de red (NAL) en los ejemplos de salida.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

Si el descodificador analiza un NALU, no se reenviará.
