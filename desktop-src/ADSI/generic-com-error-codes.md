# <a name="generic-com-error-codes"></a>Códigos de error COM genéricos

Los códigos de error COM genéricos expresan el estado de la operación tal y como lo producen todos los módulos COM. La tabla siguiente contiene algunos ejemplos de estos códigos de error en cualquier plataforma Win32.



| Código de error     | Valor hexadecimal  | Descripción              |
|----------------|------------|--------------------------|
| E \_ anular       | 0x80004004 | La operación se anuló.       |
| E \_ FAIL        | 0x80004005 | Error no especificado.       |
| E \_ NOinterface | 0x80004002 | No se admite la interfaz. |
| E \_ NOTIMPL     | 0x80004001 | Sin implementar.         |
| \_puntero E     | 0x80004003 | Puntero no válido.         |
| E \_ inesperado  | 0x8000FFFF | Error catastrófico.    |
