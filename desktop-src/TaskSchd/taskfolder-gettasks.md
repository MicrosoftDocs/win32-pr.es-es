# <a name="taskfoldergettasks-property"></a>Propiedad TaskFolder. GetTasks

En el caso de scripting, obtiene todas las tareas de la carpeta.

## <a name="syntax"></a>Sintaxis


```VB
TaskFolder.GetTasks( _
  ByVal flags, _
  ByRef ppTasks _
)
```
