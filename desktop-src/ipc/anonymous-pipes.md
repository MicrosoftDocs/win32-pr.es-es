# <a name="anonymous-pipes"></a>Canalizaciones anónimas

Una *canalización anónima* es una canalización unidireccional sin nombre que normalmente transfiere datos entre un proceso primario y un proceso secundario. Las canalizaciones anónimas siempre son locales. no se pueden usar para la comunicación a través de una red.

Para obtener más información, vea los temas siguientes:

-   [Operaciones de canalización anónimas](anonymous-pipe-operations.md)
-   [Herencia de controladores de canalización](pipe-handle-inheritance.md)
-   [Seguridad de canalización anónima y derechos de acceso](anonymous-pipe-security-and-access-rights.md)
