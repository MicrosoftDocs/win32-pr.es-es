# <a name="tapi-22"></a>TAPI 2,2

TAPI 2,2 proporciona funciones especializadas de administración del centro de llamadas. En TAPI 2,1 y versiones anteriores, el software del centro de llamadas requirió una asignación cuidadosa de líneas y direcciones a la función centro de llamadas. Las funciones de TAPI 2,1 se coordinan con las capacidades del centro de llamadas de TAPI 3. x y facilitan la creación de sistemas ACD.

**TAPI 2. x:** Para obtener más información, vea [control de centro de llamadas](call-center-control.md).

**TAPI 3. x:** Para obtener más información, vea [controles del centro de llamadas](./about-call-center-controls.md).
