# <a name="certificate-services-features"></a>Características de servicios de Certificate Server

Los servicios de Certificate Server permiten a una organización administrar la emisión, renovación y revocación de certificados. Tiene una serie de características que hacen que sea útil para las organizaciones que no decidan confiar en entidades de certificación externas y que necesiten una herramienta flexible que se pueda adaptar a las necesidades de la organización.
