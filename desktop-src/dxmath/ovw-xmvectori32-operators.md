# <a name="xmvectori32-operators"></a>Operadores XMVECTORI32

La estructura [**XMVECTORI32**](xmvectori32-data-type.md) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                               | Descripción                                                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**operador \_ \_ m128i**](/previous-versions/windows/desktop/legacy/ee421164(v=vs.85))<br/> | Devuelve el objeto que implementa la instancia actual de [**XMVECTORI32**](xmvectori32-data-type.md) convertido en una instancia de [ \_ \_ m128i](/cpp/cpp/m128i) de \[ sistemas de Windows que solo admite [SSE](/previous-versions/visualstudio/visual-studio-2010/t467de55(v=vs.100)) .\]<br/> |
| [**operador XMVECTOR**](/previous-versions/windows/desktop/legacy/ee421165(v=vs.85))<br/> | Devuelve la instancia de `XMVECTOR` que implementa la instancia actual de `XMVECTORI32` .<br/>                                                                                                                                                                                                         |
