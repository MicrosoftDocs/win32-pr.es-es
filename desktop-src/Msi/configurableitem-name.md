# <a name="configurableitemname-property"></a>Propiedad ConfigurableItem.Name

La propiedad **Name** del objeto [**ConfigurableItem**](configurableitem-object.md) devuelve el valor de la columna Name de la tabla ModuleConfiguration.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = ConfigurableItem.Name
```



## <a name="property-value"></a>Valor de propiedad

## <a name="c"></a>C++

Vea función [**Get \_ Name**](/windows/desktop/api/Mergemod/nf-mergemod-imsmconfigurableitem-get_name) .
