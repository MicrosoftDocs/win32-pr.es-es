-   <span data-ttu-id="537f3-127">La longitud del valor hash viene determinada por el tipo de algoritmo utilizado y su longitud no varía en función del tamaño del mensaje.</span><span class="sxs-lookup"><span data-stu-id="537f3-127">The length of the hash value is determined by the type of algorithm used, and its length does not vary with the size of the message.</span></span> <span data-ttu-id="537f3-128">Las longitudes de los valores hash más comunes son 128 o 160 bits.</span><span class="sxs-lookup"><span data-stu-id="537f3-128">The most common hash value lengths are either 128 or 160 bits.</span></span>
