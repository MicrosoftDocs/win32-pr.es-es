# <a name="multimedia-timers"></a>Temporizadores multimedia

Los servicios de temporizador de multimedia permiten a una aplicación programar eventos de temporizador periódicos; es decir, la aplicación puede solicitar y recibir mensajes de temporizador en intervalos especificados por la aplicación.
