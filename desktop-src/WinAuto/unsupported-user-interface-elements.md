# <a name="unsupported-user-interface-elements"></a>Elementos de la interfaz de usuario no admitidos

Microsoft Active Accessibility no es compatible con los controles comunes que se enumeran en la tabla siguiente.



| Nombre del control                  | Nombre de clase de ventana   |
|-------------------------------|---------------------|
| Controles de selector de fecha y hora | \_clase DATETIMEPICK |
| Controles de paginación                | \_PAGESCROLLER CT    |
| Rebar (controles)                | REBARCLASSNAME      |



 

Microsoft Active Accessibility no admite los controles predefinidos y los elementos de ventana que se muestran en la tabla siguiente.



| Nombre de clase de ventana | Descripción                                                 |
|-------------------|-------------------------------------------------------------|
| DDEMLEvent        | Windows NT: clase para eventos DDEML                      |
| Message           | Windows 2000 y versiones posteriores: la clase de una ventana de solo mensaje |
| \#32772           | Windows NT: la clase de los títulos de los iconos                       |
