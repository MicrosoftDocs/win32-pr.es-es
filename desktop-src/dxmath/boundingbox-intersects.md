# <a name="boundingboxintersects-methods"></a>Métodos BoundingBox.Intersects

Comprueba la intersección de BoundingBox con otro objeto .

| Método                                                                                   | Descripción                                                                                                                        |
|:-----------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------|
| [**BoundingBox::Intersects (XMVECTOR)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-intersects(fxmvector))                   | Pruebe el control BoundingBox para la intersección con un plano.<br/>                                                                     |
| [**BoundingBox::Intersects (const BoundingBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-intersects(constboundingbox_))         | Comprueba la intersección de BoundingBox con otro BoundingBox.<br/>                                                        |
| [**BoundingBox::Intersects (const BoundingSphere&)**](/previous-versions/windows/desktop/legacy/hh437825(v=vs.85))      | Comprueba la intersección de BoundingBox con un BoundingSphere.<br/>                                                           |
| [**BoundingBox::Intersects (const BoundingFrustum&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-intersects(constboundingfrustum_))     | Pruebe la [**intersección de BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) con [**un boundingFrustum**](/windows/win32/api/directxcollision/ns-directxcollision-boundingfrustum).<br/>         |
| [**BoundingBox::Intersects (XMVECTOR,XMVECTOR,float&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-intersects(fxmvector_fxmvector_float_))   | Pruebe boundingBox para la intersección con un rayo.<br/>                                                                       |
| [**BoundingBox::Intersects (XMVECTOR,XMVECTOR,XMVECTOR)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-intersects(fxmvector_fxmvector_fxmvector)) | Pruebe boundingBox para la intersección con un triángulo.<br/>                                                                  |
| [**BoundingBox::Intersects (const BoundingOrientedBox&)**](/windows/win32/api/directxcollision/nf-directxcollision-boundingbox-intersects(constboundingorientedbox_)) | Pruebe la [**intersección de BoundingBox**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) con [**un boundingOrientedBox**](/windows/win32/api/directxcollision/ns-directxcollision-boundingorientedbox).<br/> |
