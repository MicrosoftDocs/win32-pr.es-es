# <a name="xmint3-operators"></a>Operadores XMINT3

La estructura [**XMINT3**](/windows/win32/api/directxmath/ns-directxmath-xmint3) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                             | Descripción                                                                                                      |
|-----------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| [**operador =**](/windows/win32/api/directxmath/nf-directxmath-xmint3-operator-assign)<br/> | Asigna los datos del componente de vector de una instancia de `XMINT3` a la instancia actual de `XMINT3` . <br/> |
