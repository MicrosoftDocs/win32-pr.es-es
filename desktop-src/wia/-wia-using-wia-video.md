# <a name="using-wia-video"></a>Uso de WIA Video

WIA proporciona la interfaz [**IWiaVideo**](/windows/desktop/api/Wiavideo/nn-wiavideo-iwiavideo) para permitir que las aplicaciones muestren vídeo de streaming en una ventana, capturen imágenes fijas de la secuencia de vídeo y guarden esas imágenes como archivos JPEG.

En las secciones siguientes se explica cómo usar Windows de vídeo de adquisición de imágenes (WIA):

-   [Vista previa del vídeo](-wia-previewing-video.md)
-   [Captura de una imagen fija de una secuencia de vídeo](-wia-capturing-a-still-image-from-a-video-stream.md)

> [!Note]  
> WIA no admite dispositivos de vídeo en Windows Server 2003, Windows Vista ni versiones posteriores. Para esas versiones de la Windows, use [DirectShow](/previous-versions//ms783323(v=vs.85)) para adquirir imágenes a partir de vídeo.
