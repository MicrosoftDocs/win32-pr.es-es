# <a name="popup-element"></a>Elemento POPUP

El elemento **popup** proporciona una manera para que los usuarios seleccionen elementos de una lista. Estos elementos se pueden especificar mediante elementos **Item** como elementos secundarios del elemento **popup** . El elemento **popup** es idéntico al elemento **ListBox** , excepto que el valor predeterminado del atributo **popup** es true. Cuando **popUp** se establece en true, el control solo se muestra cuando se llama al método **Show** y su ancho se establece en el ancho del contenido (se omite el atributo de **ancho** ambiente). El control se oculta automáticamente cuando el usuario selecciona un elemento de la lista o cuando se llama al método de **descartado** .

> [!Note]  
> Este elemento requiere Windows Media Player para Windows XP o posterior.
