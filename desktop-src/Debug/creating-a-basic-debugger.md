# <a name="creating-a-basic-debugger"></a>Crear un depurador básico

La [depuración de un proceso en ejecución](debugging-a-running-process.md) describe cómo adjuntar el depurador a un proceso en ejecución. Al [escribir el bucle principal del depurador](writing-the-debugger-s-main-loop.md) se describen las funciones que se usan en el bucle principal de ejecución del depurador.

Aunque la mayoría de las funciones de depuración se usan para crear un depurador, varias funciones están diseñadas para su uso en el proceso que se está depurando. Para obtener más información, vea [comunicar con el depurador](communicating-with-the-debugger.md).
