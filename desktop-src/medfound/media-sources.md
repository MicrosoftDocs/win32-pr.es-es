# <a name="media-sources"></a>Orígenes multimedia

Los *orígenes multimedia* son objetos que generan datos multimedia en la canalización Media Foundation. En esta sección se describen los detalles de las API de origen de medios. Lea esta sección si va a implementar un origen multimedia personalizado o a través de un origen multimedia fuera de la canalización Media Foundation.

Si la aplicación usa la capa de control, solo necesita usar un subconjunto limitado de las API de origen multimedia. Para obtener más información, consulte el tema [uso de orígenes multimedia con la sesión multimedia](using-media-sources-with-the-media-session.md).
