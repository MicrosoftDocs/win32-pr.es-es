# <a name="listboxappenditem"></a>LISTBOX. appendItem

El método **appendItem** inserta texto nuevo al final del control de cuadro de lista.

``` syntax
        elementID.appendItem(newVal)
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

<span id="newVal"></span><span id="newval"></span><span id="NEWVAL"></span>*newVal*
</dt> <dd>

**Cadena** que contiene el texto que se va a insertar.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------------|
| Versión<br/> | Windows Media Player para Windows XP o posterior<br/> |
