# <a name="systemdevicesaepbluetoothcodserviceslimiteddiscovery"></a>System. Devices. AEP. Bluetooth. COD. Services. LimitedDiscovery

Detección limitada de la clase Bluetooth de servicio de dispositivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511

```
propertyDescription
   name = System.Devices.Aep.Bluetooth.Cod.Services.LimitedDiscovery
   shellPKey = PKEY_Devices_Aep_Bluetooth_Cod_Services_LimitedDiscovery
   formatID = 5FBD34CD-561A-412E-BA98-478A6B0FEF1D
   propID = 4
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
