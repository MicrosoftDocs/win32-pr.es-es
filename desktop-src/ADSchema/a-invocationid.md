# <a name="invocation-id-attribute"></a>Invocation-Id atributo

Se usa para identificar de forma única Microsoft Exchange Server directorio de la organización.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | Invocation-Id                                         |
| Ldap-Display-Name | invocationId                                          |
| Size              | \-                                                    |
| Privilegio actualizar  | \-                                                    |
| Frecuencia de actualización  | Cuando el Exchange Server está instalado.                |
| Attribute-Id      | 1.2.840.113556.1.2.115                                |
| System-Id-Guid    | bf96798e-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
