# <a name="fatal-application-exit"></a>Salida de aplicación fatal

La función **FatalAppExit** muestra un cuadro de mensaje y finaliza la aplicación cuando el usuario cierra el cuadro de mensaje. Esta función solo se debe usar como último recurso, ya que no puede liberar la memoria o los archivos que pertenecen a la aplicación.
