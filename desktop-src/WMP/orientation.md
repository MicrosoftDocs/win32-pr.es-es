# <a name="orientation"></a>Orientación

Algunos Windows dispositivos basados en Mobile se pueden usar en distintas orientaciones físicas. Las orientaciones disponibles incluyen:

-   **Vertical.** El dispositivo se mantiene verticalmente para que sea más alto que ancho.
-   **Horizontal**. El dispositivo se mantiene horizontalmente para que sea más ancho que alto.
-   **Cuadrado.** Algunos dispositivos tienen una pantalla en forma cuadrada, lo que significa que hay un modo de orientación único.

Cada máscara que cree debe especificar una sola orientación. Debe proporcionar una máscara independiente para cada orientación. El elegidor de máscara solo mostrará las máscaras que se crearon para la orientación actual y las dimensiones de la pantalla del dispositivo.
