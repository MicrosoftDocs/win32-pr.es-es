# <a name="systemmonitoryaxislabel-property"></a>Propiedad SystemMonitor.YAxisLabel

Recupera o establece la etiqueta del eje vertical (Y) del gráfico.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
Property YAxisLabel As String
```



## <a name="property-value"></a>Valor de propiedad

Etiqueta del eje vertical (Y) del gráfico. La longitud máxima del título es de 128 caracteres. Si el título supera los 128 caracteres, el título se trunca.

**Windows XP y Windows 2000:** No hay ningún límite en la longitud del título.
