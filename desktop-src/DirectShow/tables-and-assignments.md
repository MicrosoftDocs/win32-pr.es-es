### <a name="frequency-tables"></a>Tablas de frecuencia

En todo el mundo, hay un conjunto finito y conocido de estándares de frecuencia de difusión. Estos estándares se han identificado por su país o región de origen, pero en muchos casos, varios países o regiones comparten el mismo estándar. El filtro Tv Tuner define una tabla para cada estándar.

Sin embargo, no hay ningún conjunto finito conocido de estándares de frecuencia de cable. Los proveedores de cable de algunos países o regiones pueden difundir la programación según las frecuencias que elijan. Las frecuencias de varios estándares suelen ser mixtas. En la mayoría de los casos, esta práctica se correlaciona directamente con el país o región de origen para cada uno de los estándares de frecuencia. El filtro tv tuner define tablas para los pocos estándares de cable (y países o regiones) que tienen asignaciones de frecuencia de cable confiables. Para todos los demás, usa la [tabla de frecuencia unicable](unicable-frequency-table.md).

Las tablas de frecuencia de las secciones siguientes se extraen textualmente de las definiciones de archivo de recursos que se usaron para compilar las tablas en el filtro de TV Tuner. Cada tabla comienza con su identificador de recurso. Puede usar estos identificadores de recursos para hacer referencia cruzada desde la tabla [Asignaciones](country-region-assignments.md) de país o región para determinar qué tablas están en uso para cada país o región o configuración regional.

Las tablas de frecuencia incluyen los números teóricos de canal superior e inferior (o índices de frecuencia) para cada código de país o región. Este intervalo se puede ampliar mediante el Registro; Para obtener más información, vea [Invalidaciones de frecuencia.](frequency-overrides.md)
