# <a name="window-classes-commctrlh"></a>Clases de ventana (CommCtrl.h)

En esta sección se enumeran los nombres de clase de ventana proporcionados por la biblioteca de controles común.



| Constante                                                                                                                                                                     | Descripción                                                                                                                                                                                                                                                    |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="ANIMATE_CLASS"></span><span id="animate_class"></span><dl> <dt>**ANIMATE \_ (CLASE)**</dt> </dl>                | Crea controles de animación. Estos controles muestran de forma silenciosa un clip de vídeo de audio intercalado (AVI).<br/>                                                                                                                                                  |
| <span id="DATETIMEPICK_CLASS"></span><span id="datetimepick_class"></span><dl> <dt>**DATETIMEPICK \_ (CLASE)**</dt> </dl> | Crea controles selectores de fecha y hora. Estos controles proporcionan una interfaz sencilla para intercambiar información de fecha y hora con un usuario. <br/>                                                                                                                |
| <span id="HOTKEY_CLASS"></span><span id="hotkey_class"></span><dl> <dt>**HOTKEY \_ (CLASE)**</dt> </dl>                   | Crea controles de tecla de acceso. Estos controles hacen que sea fácil para el usuario definir las teclas de acceso rápido. <br/>                                                                                                                                                             |
| <span id="LINK_CLASS"></span><span id="link_class"></span><dl> <dt>**LINK \_ (CLASE)**</dt> </dl>                         | Crea controles syslink. Estos controles se usan con vínculos. <br/>                                                                                                                                                                                      |
| <span id="MONTHCAL_CLASS"></span><span id="monthcal_class"></span><dl> <dt>**MONTHCAL \_ (CLASE)**</dt> </dl>             | Crea controles de calendario mensual. Estos controles proporcionan una manera sencilla para que un usuario seleccione una fecha de una interfaz conocida. <br/>                                                                                                                        |
| <span id="NATIVEFNTCTL_CLASS"></span><span id="nativefntctl_class"></span><dl> <dt>**NATIVEFNTCTL \_ (CLASE)**</dt> </dl> | Crea controles de fuente nativos. Estos controles se usan con fuentes nativas. <br/>                                                                                                                                                                           |
| <span id="PROGRESS_CLASS"></span><span id="progress_class"></span><dl> <dt>**PROGRESS \_ (CLASE)**</dt> </dl>             | Crea barras de progreso. Estos controles indican el progreso de una operación larga.<br/>                                                                                                                                                                 |
| <span id="REBARCLASSNAME"></span><span id="rebarclassname"></span><dl> <dt>**REBARCLASSNAME**</dt> </dl>              | Crea controles rebar. Estos controles actúan como un contenedor para las ventanas secundarias. <br/>                                                                                                                                                                       |
| <span id="STANDARD_CLASSES"></span><span id="standard_classes"></span><dl> <dt>**CLASES \_ ESTÁNDAR**</dt> </dl>       | Crea controles User32 intrínsecos, como botón, edición y barra de desplazamiento.<br/>                                                                                                                                                                            |
| <span id="STATUSCLASSNAME"></span><span id="statusclassname"></span><dl> <dt>**STATUSCLASSNAME**</dt> </dl>           | Crea ventanas de estado. Estos controles muestran información de estado en una ventana horizontal.<br/>                                                                                                                                                           |
| <span id="TOOLBARCLASSNAME"></span><span id="toolbarclassname"></span><dl> <dt>**TOOLBARCLASSNAME**</dt> </dl>        | Crea barras de herramientas. Estos controles contienen botones que llevan a cabo comandos de menú. <br/>                                                                                                                                                                     |
| <span id="TOOLTIPS_CLASS"></span><span id="tooltips_class"></span><dl> <dt>**TOOLTIPS \_ (CLASE)**</dt> </dl>             | Crea controles de información sobre herramientas. Estos controles muestran una pequeña ventana emergente que contiene una línea de texto que describe el propósito de una herramienta en una aplicación.<br/>                                                                                            |
| <span id="TRACKBAR_CLASS"></span><span id="trackbar_class"></span><dl> <dt>**TRACKBAR \_ (CLASE)**</dt> </dl>             | Crea barras de seguimiento. Estos controles permiten al usuario seleccionar entre un intervalo de valores moviendo un control deslizante. <br/>                                                                                                                                                   |
| <span id="UPDOWN_CLASS"></span><span id="updown_class"></span><dl> <dt>**UPDOWN \_ (CLASE)**</dt> </dl>                   | Crea controles de arriba a abajo. Estos controles combinan un par de flechas con un control de edición. Al hacer clic en las flechas se incrementa o disminuye el valor del control de edición. <br/>                                                                                |
| <span id="WC_BUTTON"></span><span id="wc_button"></span><dl> <dt>**BOTÓN \_ WC**</dt> </dl>                            | Crea controles de botón. Estos controles permiten a un usuario iniciar una acción.<br/>                                                                                                                                                                        |
| <span id="WC_COMBOBOX"></span><span id="wc_combobox"></span><dl> <dt>**WC \_ COMBOBOX**</dt> </dl>                      | Crea controles de cuadro combinado. Estos controles contienen una lista desplegable desde la que se puede seleccionar un solo elemento. <br/>                                                                                                                                       |
| <span id="WC_COMBOBOXEX"></span><span id="wc_comboboxex"></span><dl> <dt>**WC \_ COMBOBOXEX**</dt> </dl>                | Crea controles ComboBoxEx. Estos controles proporcionan una extensión del control de cuadro combinado que proporciona compatibilidad nativa con imágenes de elementos. <br/>                                                                                                            |
| <span id="WC_EDIT"></span><span id="wc_edit"></span><dl> <dt>**WC \_ EDIT**</dt> </dl>                                  | Crea controles de edición. Estos controles contienen texto editable. <br/>                                                                                                                                                                                       |
| <span id="WC_HEADER"></span><span id="wc_header"></span><dl> <dt>**ENCABEZADO \_ WC**</dt> </dl>                            | Crea controles de encabezado. Estos controles muestran encabezados en la parte superior de las columnas de información y permiten al usuario ordenar la información haciendo clic en los encabezados. <br/>                                                                                      |
| <span id="WC_LISTBOX"></span><span id="wc_listbox"></span><dl> <dt>**WC \_ LISTBOX**</dt> </dl>                         | Crea controles de cuadro de lista. Estos controles contienen listas de elementos. <br/>                                                                                                                                                                                   |
| <span id="WC_IPADDRESS"></span><span id="wc_ipaddress"></span><dl> <dt>**WC \_ IPADDRESS**</dt> </dl>                   | Crea controles de dirección IP. Estos controles son similares a un control de edición, pero permiten escribir una dirección numérica en formato de protocolo de Internet (IP). <br/>                                                                                       |
| <span id="WC_LINK"></span><span id="wc_link"></span><dl> <dt>**WC \_ LINK**</dt> </dl>                                  | Crea controles SysLink. Estos controles contienen vínculos de hipertexto.<br/>                                                                                                                                                                                   |
| <span id="WC_LISTVIEW"></span><span id="wc_listview"></span><dl> <dt>**WC \_ LISTVIEW**</dt> </dl>                      | Crea controles de vista de lista. Estos controles muestran una colección de elementos, cada uno de los que consta de un icono y una etiqueta, y proporcionan varias maneras de organizar los elementos.<br/>                                                                                    |
| <span id="WC_NATIVEFONTCTL"></span><span id="wc_nativefontctl"></span><dl> <dt>**WC \_ NATIVEFONTCTL**</dt> </dl>       | Crea controles de fuente nativos. Se trata de controles invisibles que funcionan en segundo plano para permitir que los controles predefinidos de un cuadro de diálogo muestren el idioma actual del sistema.<br/>                                                                         |
| <span id="WC_PAGESCROLLER"></span><span id="wc_pagescroller"></span><dl> <dt>**WC \_ PAGESCROLLER**</dt> </dl>          | Crea controles de paginación. Estos controles se usan para contener y desplazarse por otra ventana. <br/>                                                                                                                                                              |
| <span id="WC_SCROLLBAR"></span><span id="wc_scrollbar"></span><dl> <dt>**BARRA \_ DE DESPLAZAMIENTO DE WC**</dt> </dl>                   | Crea controles de barra de desplazamiento. Estos controles permiten al usuario desplazarse por el contenido de una ventana.<br/>                                                                                                                                                     |
| <span id="WC_STATIC"></span><span id="wc_static"></span><dl> <dt>**WC \_ STATIC**</dt> </dl>                            | Crea controles estáticos. Estos controles contienen texto no editable. <br/>                                                                                                                                                                                  |
| <span id="WC_TABCONTROL"></span><span id="wc_tabcontrol"></span><dl> <dt>**WC \_ TABCONTROL**</dt> </dl>                | Crea controles de ficha. Estos controles definen varias páginas para la misma área de una ventana o cuadro de diálogo. Cada página consta de un conjunto de información o un grupo de controles que una aplicación muestra cuando el usuario selecciona la pestaña correspondiente.<br/> |
| <span id="WC_TREEVIEW"></span><span id="wc_treeview"></span><dl> <dt>**WC \_ TREEVIEW**</dt> </dl>                      | Crea controles de vista de árbol. Estos controles muestran una lista jerárquica de elementos. Cada elemento consta de una etiqueta y un mapa de bits opcional.<br/>                                                                                                              |
