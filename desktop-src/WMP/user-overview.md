# <a name="user-overview"></a>Información general del usuario

Para el usuario, las visualizaciones son imágenes que bailen en el tiempo a la música. Proporcionan entretenimiento adicional para mejorar las experiencias musicales. Cuanto más entretenido sea la visualización, mejor.

Aparece una visualización típica como se muestra en la siguiente ilustración.

![visualización típica](images/vizanim.gif)

## <a name="standard-visualizations"></a>Visualizaciones estándar

Windows Media Player incluye varias visualizaciones, como las siguientes:

-   Ambiente
-   Plenoptic
-   Intenso

## <a name="standard-presets"></a>Valores preestablecidos estándar

Además, cada visualización incluye valores predeterminados específicos que modifican el modo en que se muestra la visualización. Por ejemplo, la visualización de pico tiene dos valores preestablecidos:

-   Intenso
-   Amoeba

Ambos efectos forman parte de la misma visualización, pero tienen valores predeterminados diferentes. Un valor preestablecido es un conjunto de valores para los atributos de una visualización determinada.

## <a name="skin-attributes"></a>Atributos de máscara

Si crea máscaras para Windows Media Player, puede mostrar las visualizaciones estándar en las máscaras mediante el elemento **Effects** . Para obtener más información sobre el uso de visualizaciones en las máscaras, vea [máscaras de Windows Media Player](windows-media-player-skins.md).
