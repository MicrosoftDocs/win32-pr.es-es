# <a name="d1173-texture-cube-interop-unsupported"></a>D1173: interoperabilidad de cubos de textura no admitida

Las texturas marcadas como D3D11RESOURCE \_ Misc \_ TEXTURECUBE no se pueden usar en Direct2D.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación intentó interoperar una textura de Direct3D con la \_ opción D3D11RESOURCE Misc \_ TEXTURECUBE especificada.

## <a name="possible-fixes"></a>Posibles correcciones

Abstenerse de las texturas interoperables que tienen el especificador TEXTURECUBE. En su lugar, reorganice el diseño de representación de la aplicación o considere la posibilidad de interoperar con un intermediario.
