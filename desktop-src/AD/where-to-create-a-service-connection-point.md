# <a name="where-to-create-a-service-connection-point"></a><span data-ttu-id="dc875-108">Dónde crear un punto de conexión de servicio</span><span class="sxs-lookup"><span data-stu-id="dc875-108">Where to Create a Service Connection Point</span></span>
