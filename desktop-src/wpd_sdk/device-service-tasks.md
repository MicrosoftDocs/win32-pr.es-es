# <a name="device-service-tasks"></a>Tareas de Device Service

La API de WPD admite una serie de tareas de servicios de dispositivo. Esto incluye las tareas siguientes:

-   Enumeración de servicios de dispositivo.
-   Abrir un servicio de dispositivo.
-   Enumeración de contenido en un servicio de dispositivo.
-   Recuperación de eventos de servicio admitidos.
-   Recuperación de formatos de servicio admitidos.
-   Recuperación de los métodos de servicio admitidos.
-   Recuperación de los servicios abstractos admitidos.
-   Recuperación de propiedades de objeto de contenido.
-   Escribir propiedades de objeto de contenido.
-   Invocar métodos de servicio de forma sincrónica.
-   Invocación asincrónica de métodos de servicio.

Estas operaciones se describen en las secciones siguientes y se muestran en la aplicación WpdServicesApiSample. Además, la API de WPD también admite la transferencia de contenido, el movimiento de contenido y la eliminación de contenido de un servicio de dispositivo.
