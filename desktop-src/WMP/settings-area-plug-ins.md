# <a name="settings-area-plug-ins"></a>Configuración Complementos de área

Para Microsoft Reproductor de Windows Media 11 y versiones anteriores, los complementos de interfaz de  usuario del área de configuración se muestran en el área Herramientas del panel Reproducción ahora debajo del área de presentación principal y encima de los controles de transporte. Para Microsoft Reproductor de Windows Media 12 y versiones posteriores, los complementos de interfaz de usuario del área de configuración aparecen como ventanas independientes. Solo se puede habilitar un complemento de área de configuración a la vez.

Configuración complementos de área de trabajo son útiles para mostrar paneles de control personalizados similares a los controles de configuración de vídeo o el igualador de gráficos.
