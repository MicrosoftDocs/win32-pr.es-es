# <a name="acquisitiontimeyearmonthday-attribute"></a>Atributo AcquisitionTimeYearMonthDay

El **atributo AcquisitionTimeYearMonthDay** es la parte de año, mes y día de la fecha y hora en que se agregó el elemento a la biblioteca.

## <a name="applies-to"></a>Se aplica a

-   [Elementos de audio](audio-item-attributes.md)
-   [Otros elementos](other-item-attributes.md)
-   [Elementos de fotos](photo-item-attributes.md)
-   [Listas](playlist-attributes-ref.md)
-   [Elementos de radio](radio-item-attributes.md)
-   [Elementos de vídeo](video-item-attributes.md)

## <a name="remarks"></a>Comentarios

Este atributo solo se almacena en la biblioteca.

Para determinar si puede cambiar el valor de este atributo, use el [método Media.isReadOnlyItem.](media-isreadonlyitem.md)
