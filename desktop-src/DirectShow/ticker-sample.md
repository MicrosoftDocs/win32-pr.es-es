# <a name="ticker-sample"></a>Ejemplo de teletipo

## <a name="description"></a>Descripción

En este ejemplo se usa el representador de combinación de vídeo para mezclar vídeo y texto. Usa la interfaz [**IVMRMixerBitmap9**](/previous-versions/windows/desktop/api/Vmr9/nn-vmr9-ivmrmixerbitmap9) para mezclar texto en la parte inferior de la ventana de vídeo.

La aplicación crea un mapa de bits con una cadena de texto predeterminada y lo desplaza por la parte inferior de la pantalla. Para cambiar el texto, elija **establecer cadena de texto...** en el menú del **tablero de cotizaciones** . Para establecer la fuente, elija **establecer fuente...** en el menú del **tablero de cotizaciones** . Si selecciona **usar imagen estática** en el menú del **tablero de cotizaciones** , la aplicación cambia de una cadena de texto dinámico a una imagen de mapa de bits estática.
