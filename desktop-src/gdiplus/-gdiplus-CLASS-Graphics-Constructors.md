# <a name="graphicsgraphics-constructors"></a>Graphics. Graphics (constructores)

En este tema se enumeran los constructores de la clase [**Graphics**](/windows/win32/api/gdiplusgraphics/nl-gdiplusgraphics-graphics) . Para obtener una lista de clases completa, vea **Graphics (clase**).

### <a name="overload-list"></a>Lista de sobrecarga



| Constructor                                                                   | Descripción                                                                                                                                                                        |
|:------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Gráficos (HDC)**](/previous-versions//ms536160(v=vs.85))                | Crea un objeto [**Graphics:: Graphics**](/previous-versions//ms536160(v=vs.85)) que está asociado a un contexto de dispositivo especificado.<br/>                                |
| [**Gráficos (imagen \* )**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inimage))          | Crea un objeto [**Graphics:: Graphics**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inimage)) que está asociado a un objeto [**Image**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-image) .<br/>   |
| [**Gráficos (HWND, BOOL)**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhwnd_inbool))     | Crea un objeto [**Graphics:: Graphics**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhwnd_inbool)) que está asociado a una ventana especificada.<br/>                                   |
| [**Gráficos (HDC, HANDLE)**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhdc_inhandle)) | Crea un objeto [**Graphics:: Graphics**](/windows/win32/api/gdiplusgraphics/nf-gdiplusgraphics-graphics-graphics(inhdc_inhandle)) que está asociado a un contexto de dispositivo especificado y a un dispositivo especificado.<br/> |
