# <a name="syntax-conventions"></a>Convenciones de sintaxis

En la tabla siguiente se enumeran los términos que se usan en las instrucciones de sintaxis de la referencia del modelo de scripting de adquisición de imágenes de Windows (WIA). 

| Término                      | Descripción                                                    |
|---------------------------|----------------------------------------------------------------|
| *Nombre de archivo*                | La ruta de acceso y el nombre de un archivo.                                   |
| *objDeviceInfo*           | Objeto [**DeviceInfo**](-wia-deviceinfo.md) .                |
| *objDeviceInfoCollection* | Colección de objetos [**DeviceInfo**](-wia-deviceinfo.md) . |
| *objItem*                 | Objeto de [**elemento**](-wia-item.md) .                           |
| *objItemCollection*       | Colección de objetos [**Item**](-wia-item.md) .             |
| *objWia*                  | Objeto [**WIA**](-wia-wia.md) .                              |
| *PropID*                  | Una constante de propiedad de WIA.                                       |
| *var*                     | Valor de **variante** .                                           |
