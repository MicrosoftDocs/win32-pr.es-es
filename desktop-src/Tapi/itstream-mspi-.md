# <a name="itstream-mspi"></a>ITStream (MSPI)

La interfaz [**ITStream**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) está disponible si hay un MSP asociado a la llamada. Los métodos de esta interfaz permiten a una aplicación recuperar información en un flujo; para iniciar, pausar o detener la secuencia; y para seleccionar o anular la selección de terminales en una secuencia.

La interfaz [**ITStream**](/windows/win32/api/tapi3if/nn-tapi3if-itstream) se implementa mediante un MSP. Consulte **ITStream** en la sección interfaz MSP para obtener más información sobre esta interfaz.
