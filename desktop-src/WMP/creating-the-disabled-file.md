# <a name="creating-the-disabled-file"></a>Crear el archivo deshabilitado

El archivo deshabilitado es similar al archivo insertado, salvo que se usa otro color (rojo) para mostrar cuando algo está deshabilitado.

La siguiente imagen es el archivo deshabilitado.

![archivo deshabilitado](images/ceswmdis.png)

Tenga en cuenta que los archivos como este se podrían recortar en los lados derecho e inferior. El tamaño real del mapa de bits no es importante, solo que los elementos se alinean con el mapa de bits de la región para la prueba de posicionamiento, medido desde las esquinas superiores izquierdas (incluidos los desplazamientos).
