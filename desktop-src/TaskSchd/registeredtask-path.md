# <a name="registeredtaskpath-property"></a>RegisteredTask. Path (propiedad)

En el caso de scripting, obtiene la ruta de acceso donde se almacena la tarea registrada.

## <a name="syntax"></a>Sintaxis


```VB
RegisteredTask.Path As String
```



## <a name="property-value"></a>Valor de propiedad

La ruta de acceso donde se almacena la tarea registrada.
