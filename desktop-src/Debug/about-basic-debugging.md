# <a name="about-basic-debugging"></a>Acerca de la depuración básica

Las [funciones de depuración](debugging-functions.md) se pueden usar para crear un depurador básico orientado a eventos. *Orientado a eventos* significa que se notifica al depurador cada vez que se producen determinados eventos en el proceso que se está depurando. La notificación permite al depurador tomar las medidas adecuadas en respuesta a los eventos.

Para obtener información general sobre los términos de depuración, vea [terminología de depuración](debugging-terminology.md).

La [compatibilidad de depuración de las funciones de proceso, subproceso y excepción](debug-support-from-process-thread-and-exception-functions.md) describe las características específicas de depuración de ciertas funciones de proceso, subproceso y control de excepciones.

[Crear un depurador básico](creating-a-basic-debugger.md) describe el uso de las funciones de depuración básicas para crear un depurador sencillo. Estas funciones permiten a una aplicación esperar a la depuración de eventos, producir excepciones de punto de interrupción, transferir el control de ejecución al depurador, etc.

[Depurar eventos](debugging-events.md) describe el mecanismo de eventos de depuración. Los eventos de depuración hacen que el sistema notifique al depurador.
