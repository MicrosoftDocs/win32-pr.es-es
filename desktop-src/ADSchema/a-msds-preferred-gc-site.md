# <a name="ms-ds-preferred-gc-site-attribute"></a>atributo MS-DS-Preferred-GC-site

El atributo **MS-DS-Preferred-GC-site** lo utiliza el administrador de cuentas de seguridad para la expansión de grupos durante la evaluación de tokens.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-DS-preferido-GC-sitio                 |
| Nombre para mostrar de LDAP | msDS-preferido-GC-sitio                  |
| Tamaño              | \-                                      |
| Actualizar privilegio  | Administrador de dominio                    |
| Frecuencia de actualización  | A discreción del administrador.      |
| Attribute-Id      | 1.2.840.113556.1.4.1444                 |
| System-ID-GUID    | d921b50a-0ab2-42cd-87f6-09cf83a91854    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
