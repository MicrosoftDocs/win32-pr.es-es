# <a name="avencvideoinputcolorprimaries-property"></a>Propiedad AVEncVideoInputColorPrimaries

Especifica los valores de color principal para el vídeo de entrada.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncVideoInputColorPrimaries**

## <a name="property-value"></a>Valor de propiedad

El valor de esta propiedad es un miembro de la [**enumeración eAVEncVideoColorPrimaries.**](/windows/desktop/api/codecapi/ne-codecapi-eavencvideocolorprimaries)
