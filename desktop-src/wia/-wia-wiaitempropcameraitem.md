# <a name="camera-wia-item-property-constants"></a>Constantes de propiedad del elemento de WIA de cámara

Las siguientes constantes de **Word** forman el conjunto de propiedades de elemento de cámara de adquisición de imágenes de Windows (WIA) válidas.

> [!Note]  
> WIA no admite cámaras en Windows Vista o posterior. En el caso de las versiones de Windows, use la API de dispositivo portátil de Windows (WPD) que se describe en el kit de desarrollo de controladores de Windows (DDK) para adquirir imágenes de cámaras.

 

El prefijo "WIA \_ IPC \_ " indica una propiedad de elemento para las cámaras y es la Convención de nomenclatura utilizada en C/C++. Con fines de scripting, estas constantes usan el prefijo "CameraPicture" y forman parte del tipo enumerado [WiaItemPropertyId](-wia-wiaitempropertyid.md) . El nombre de miembro correspondiente de esa enumeración de script aparece entre paréntesis al lado del nombre de la constante de C/C++ en la lista siguiente.
