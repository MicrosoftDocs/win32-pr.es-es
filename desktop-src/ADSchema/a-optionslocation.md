# <a name="options-location-attribute"></a>Options-Location atributo)

En el caso de DHCP, la ubicación de las opciones contiene el DN para los sitios alternativos que contienen la información de las opciones.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Options-Location                     |
| Nombre para mostrar de LDAP | optionsLocation                      |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.713               |
| System-ID-GUID    | 963d274e-48be-11d1-a9c3-0000f80367c1 |
| Sintaxis            | [**String(IA5)**](s-string-ia5.md)  |
