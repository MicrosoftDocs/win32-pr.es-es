# <a name="uninstalling-custom-actions"></a>Desinstalación de acciones personalizadas

Los desarrolladores de paquetes de instalación que hacen uso de acciones personalizadas deben determinar en cada caso cómo condicionar la acción personalizada para que funcione correctamente cuando se desinstale el paquete. Depende del autor del paquete decidir si desea que la acción personalizada se ejecute durante la desinstalación, no se ejecute en la desinstalación o para ejecutar una acción personalizada alternativa en la desinstalación.
