</dt> <dd>

Un estándar de interconexión de sistemas abiertos (OSI).

</dd> <dt>

<span id="_security_x.208_gly"></span><span id="_SECURITY_X.208_GLY"></span>**X. 208**
</dt> <dd>

El estándar ITU que define ASN. 1.

</dd> <dt>

<span id="_security_x.209_gly"></span><span id="_SECURITY_X.209_GLY"></span>**X. 209**
</dt> <dd>

Vea también [*reglas de codificación básicas*](b-gly.md).

</dd> <dt>

<span id="_security_x.400_gly"></span><span id="_SECURITY_X.400_GLY"></span>**X. 400**
</dt> <dd>

El estándar ITU que define los tokens de mensaje.

</dd> <dt>

<span id="_security_x.500_gly"></span><span id="_SECURITY_X.500_GLY"></span>**X. 500**
</dt> <dd>

El estándar ITU que define la estructura de un diccionario global.

</dd> <dt>

<span id="_security_x.509_gly"></span><span id="_SECURITY_X.509_GLY"></span>**X. 509**
</dt> <dd>

Un estándar reconocido internacionalmente para los certificados que define sus partes necesarias.

</dd> <dt>

<span id="_security_x509_asn_encoding_gly"></span><span id="_SECURITY_X509_ASN_ENCODING_GLY"></span>**\_Codificación ASN de X509 \_**
</dt> <dd>

Tipo de codificación que especifica la codificación del certificado. Los tipos de codificación de certificados se almacenan en la palabra de orden inferior de un **DWORD** (el valor es: 0x00000001). Este tipo de codificación es funcionalmente igual que la codificación de ASN de CIFRAdo \_ \_ .

</dd> <dt>

<span id="_security_xor_gly"></span><span id="_SECURITY_XOR_GLY"></span>**XO**
</dt> <dd>

**Or** exclusivo. Una operación booleana que produce true solo si uno de sus operandos es true y el otro es false. Si ambos operandos son iguales (true o false), la operación da como resultado false.

</dd> </dl>
