# <a name="using-ascii-control-codes-0x000d-and-0x000a"></a>Usar códigos de control ASCII 0x000D y 0x000A

El estándar Unicode no prescribe significados específicos para los códigos de control 0x000D (retorno de carro) y 0x000A (avance de la alimentación). No es necesario usar estos códigos en combinación. Si se usa de forma individual, el código solo puede representarse o ambos códigos juntos. La aplicación siempre debe determinar lo que representan estos códigos.
