# <a name="channel-com"></a>Canal (COM)

El canal tiene la responsabilidad de transmitir todos los mensajes entre el cliente y el objeto a través del límite del proceso. El canal se ha diseñado para funcionar de forma transparente con diferentes tipos de canal, es compatible con RPC estándar de OSF DCE y admite aplicaciones únicas y multiproceso.
