# <a name="video-for-windows"></a>Vídeo para Windows

El vídeo para Windows (VFW) proporciona funciones que permiten que una aplicación procese datos de vídeo. VFW se presentó en Windows de 16 bits. Muchas de sus características se han sustituido por características de DirectX. Para obtener más información, consulte la documentación de DirectX.

Los temas siguientes se describen en la documentación de vídeo para Windows.

-   [Funciones y macros de AVIFile](avifile-functions-and-macros.md)
-   [Administrador de compresión de vídeo](video-compression-manager.md)
-   [Captura de vídeo](video-capture.md)
-   [Controladores personalizados de archivos y secuencias](custom-file-and-stream-handlers.md)
-   [DrawDib](drawdib.md)
