# <a name="iupdatesearcher2-properties"></a>Propiedades de IUpdateSearcher2

La [**interfaz IUpdateSearcher2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher2) define la siguiente propiedad.



| Propiedad                                                                  | Descripción                                                                           |
|---------------------------------------------------------------------------|---------------------------------------------------------------------------------------|
| [**IgnoreDownloadPriority**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher2-get_ignoredownloadpriority) | Obtiene y establece un valor booleano que indica si se debe omitir la prioridad de descarga. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz.

-   [**IUpdateSearcher**](/windows/desktop/api/Wuapi/nn-wuapi-iupdatesearcher)
