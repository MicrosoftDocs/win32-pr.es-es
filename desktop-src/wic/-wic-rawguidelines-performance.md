# <a name="performance-windows-imaging-component"></a><span data-ttu-id="fa063-103">Rendimiento (componente de Windows Imaging)</span><span class="sxs-lookup"><span data-stu-id="fa063-103">Performance (Windows Imaging Component)</span></span>

<span data-ttu-id="fa063-104">El objetivo de Microsoft es que el rendimiento del procesamiento sin procesar a través de la interfaz de Windows Imaging Component (WIC) sea equivalente a las herramientas y aplicaciones de procesamiento sin formato líderes del sector.</span><span class="sxs-lookup"><span data-stu-id="fa063-104">The Microsoft goal is that the performance of RAW processing through the Windows Imaging Component (WIC) interface be equivalent to industry-leading RAW processing tools and applications.</span></span> <span data-ttu-id="fa063-105">Para proporcionar una guía de rendimiento, se realizó un conjunto de pruebas exhaustivas que comparan el rendimiento de las herramientas de flujo de trabajo sin procesar conocidas.</span><span class="sxs-lookup"><span data-stu-id="fa063-105">To provide performance guidance, a set of comprehensive tests comparing the performance of popular RAW workflow tools was performed.</span></span> <span data-ttu-id="fa063-106">Las pruebas incluyen las versiones de sistema operativo de Windows y Apple Macintosh de los productos de procesamiento sin procesar, archivos sin procesar de ejemplo de cámaras de Reflex de una sola lente (SLR) actuales y una serie de funciones que representan los pasos comunes del flujo de trabajo.</span><span class="sxs-lookup"><span data-stu-id="fa063-106">The tests included both Windows and Apple Macintosh operating system versions of RAW processing products, sample RAW files from current single-lens reflex (SLR) cameras, and a series of functions that represent common workflow steps.</span></span>

<span data-ttu-id="fa063-107">En función de estos resultados, Microsoft sugiere las siguientes recomendaciones a los asociados para el rendimiento del procesamiento sin procesar:</span><span class="sxs-lookup"><span data-stu-id="fa063-107">Based on these results, Microsoft suggests the following recommendations to partners for RAW processing performance:</span></span>

-   <span data-ttu-id="fa063-108">El rendimiento de línea base debe ser igual o superior a los tiempos de procesamiento de las aplicaciones y herramientas de procesamiento sin formato proporcionadas por el fabricante actual.</span><span class="sxs-lookup"><span data-stu-id="fa063-108">Baseline performance should be equal to or better than the processing times of current manufacturer-supplied RAW processing applications and tools.</span></span>
-   <span data-ttu-id="fa063-109">Los objetivos de rendimiento óptimos deben coincidir o superar la aplicación de mayor rendimiento sin sacrificar la calidad (en estas pruebas, la cámara RAW de Photoshop generalmente se realiza más rápido).</span><span class="sxs-lookup"><span data-stu-id="fa063-109">Optimal performance targets should match or exceed the highest performing application without any sacrifice in quality (in these tests, Photoshop Camera RAW generally performed fastest).</span></span>
