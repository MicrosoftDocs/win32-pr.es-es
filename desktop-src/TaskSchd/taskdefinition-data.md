# <a name="taskdefinitiondata-property"></a>Propiedad TaskDefinition.Data

Para el scripting, obtiene o establece los datos asociados a la tarea. El servicio de Programador de tareas omite estos datos, pero los usan terceros que desean ampliar el formato de la tarea.

## <a name="syntax"></a>Syntax


```VB
TaskDefinition.Data As String
```



## <a name="property-value"></a>Valor de propiedad

Datos asociados a la tarea.
