# <a name="systemdeviceswifidirectmiracastversion"></a>System.Devices.WiFiDirect.MiracastVersion

Indica la versión del protocolo Miracast si Wi-Fi dispositivo directo es Miracast compatible.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507

```
propertyDescription
   name = System.Devices.WiFiDirect.MiracastVersion
   shellPKey = PKEY_Devices_WiFiDirect_MiracastVersion
   formatID = 1506935D-E3E7-450F-8637-82233EBE5F6E
   propID = 8
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = UInt32
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
