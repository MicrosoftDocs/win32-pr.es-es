# <a name="new-features-and-enhancements"></a>Nuevas características y mejoras

En esta sección del documento se describen las mejoras y las nuevas características que pueden mejorar la experiencia de los usuarios y los desarrolladores. Sin embargo, algunos de ellos tienen impactos que los desarrolladores deben tener en cuenta con respecto a sus productos nuevos y existentes. A continuación se muestra la lista de nuevas características y mejoras, agrupadas por área de características:

## <a name="security"></a>Seguridad

-   Antimalware de inicio temprano
-   Requisitos de firma de características de arranque seguras para controladores en modo kernel
-   Arranque medido

## <a name="performance"></a>Rendimiento

-   Aplicaciones de inicio
-   Sistemas de archivos resistentes
-   Compatibilidad con la API del servidor de archivos
-   Mantenimiento automático

## <a name="usability"></a>Facilidad de uso

-   Editores de métodos de entrada de terceros

## <a name="storage-and-file-system"></a>Storage y sistema de archivos

-   Seguimiento de los intervalos modificados de un archivo en Windows 8.1
-   La nueva API permite que las aplicaciones envíen sugerencias "TRIM y Unmap" a los medios de almacenamiento.
-   E/S de múltiples rutas ahora admite bloques de solicitud de almacenamiento extendidos
-   Sistemas de archivos resistentes
-   Compatibilidad con la API del servidor de archivos
-   Nueva característica Historial de archivos de datos
-   El sistema operativo ahora controla la potencia de las unidades de disco óptico.

## <a name="usb"></a>USB

-   Compatibilidad con USB 3.0
