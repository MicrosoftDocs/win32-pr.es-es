# <a name="jet_ossnapid"></a>JET_OSSNAPID


_**Se aplica a:** Windows | Windows Server_

## <a name="jet_ossnapid"></a>JET_OSSNAPID

El tipo de datos **JET_OSSNAPID** contiene un identificador para una instantánea de la base de datos.

```cpp
    typedef JET_API_PTR JET_OSSNAPID;
```

### <a name="data-types"></a>Tipo de datos

JET_OSSNAPID

Identificador de una instantánea de la base de datos. Este identificador se usa en los elementos de la API de JET que participan en la copia de seguridad de instantánea.

**Null** se puede usar para indicar un identificador no válido.
