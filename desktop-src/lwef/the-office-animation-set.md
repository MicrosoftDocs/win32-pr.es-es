# <a name="the-office-animation-set"></a>Conjunto Office animación

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

En la tabla siguiente se enumeran las animaciones definidas para Microsoft Office 2000 caracteres. Si piensa usar el carácter en Microsoft Office, debe admitir todas las animaciones de esta tabla. Además, puede agregar cualquier otra animación que se vivo, pero tenga en cuenta que Microsoft Office no las llamará. Las animaciones con asteriscos ( \* ) deben tener un bucle del 100 %. Otras animaciones deben ser breves.



| Animación               | Estado del agente    | Ejemplo de cuando se usa                                                                                 | Ejemplos de animación específicos                                                                                                               |
|-------------------------|----------------|------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| **Alerta**               | Ninguno           | Cuando el carácter quiere alertar al usuario                                                           | El carácter busca al usuario.                                                                                                              |
| **CheckingSomething\*** | Ninguno           | Corrector ortográfico, revisión gramatical                                                                            | El carácter busca algo en un libro de referencia                                                                                          |
| **Felicitar**        | Ninguno           | Completar un asistente                                                                                    | Big uno, aspecto de relieve, agotado, pero satisfecho                                                                                                 |
| **EmptyTrash**          | Ninguno           | La papelera se vacía en Outlook                                                                          | La papelera de luces de caracteres se puede encender                                                                                                        |
| **Explicación**             | Ninguno           | Cuando el carácter quiere explicar algo al usuario                                            | Examina brevemente, pero con cuidado, al usuario y, a continuación, mira hacia otro lado.                                                                                     |
| **GestureDown**         | GesturingDown  | El carácter señala algo en la pantalla                                                         | El carácter examina al usuario y, a continuación, apunta y examina la pantalla.                                                                           |
| **GestureLeft**         | GesturingLeft  | El carácter señala algo en la pantalla, como un tema de ayuda o un fragmento de interfaz de usuario                  | El carácter examina al usuario y, a continuación, apunta y examina la pantalla.                                                                           |
