# <a name="ms-dfsr-tombstoneexpiryinmin-attribute"></a>atributo MS-DFSR-TombstoneExpiryInMin

Contiene la duración del registro de marcadores de exclusión, en minutos.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-TombstoneExpiryInMin         |
| Nombre para mostrar de LDAP | msDFSR-TombstoneExpiryInMin          |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.11           |
| System-ID-GUID    | 23e35d4c-e324-4861-a22f-e199140dae00 |
