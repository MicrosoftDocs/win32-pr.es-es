# <a name="touch-hit-testing-structures"></a>Estructuras de pruebas de posicionamiento táctiles

En los temas de esta sección se proporcionan las especificaciones de referencia para las estructuras de [prueba de posicionamiento táctil](touch-hit-testing-portal.md) .

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
| --- | --- |
| [**TOUCH_HIT_TESTING_INPUT**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_input)<br/> | Contiene información sobre el área de contacto táctil indicada por el digitalizador táctil.<br/> |
| [**TOUCH_HIT_TESTING_PROXIMITY_EVALUATION**](/windows/win32/api/winuser/ns-winuser-touch_hit_testing_proximity_evaluation)<br/> | Contiene la puntuación de la prueba de posicionamiento que indica si el objeto es el objetivo probable del área de contacto táctil, con respecto a otros objetos que forman una intersección con el área de contacto táctil.<br/> |
