# <a name="language-names"></a>Nombres de idioma

Un nombre de idioma utiliza esencialmente el mismo formato que un [nombre de configuración regional](locale-names.md). Por ejemplo, el nombre de idioma de la configuración regional en inglés (Belice) es "en-BZ". Cada nombre de idioma corresponde a un [identificador de idioma](language-identifiers.md).

Las principales diferencias entre el formato de un nombre de idioma y el formato de un nombre de configuración regional son:

-   Un nombre de lenguaje nunca incluye un criterio de ordenación. Por lo tanto, el nombre de idioma correspondiente al nombre de configuración regional "Hu-HU \_ technl" (idioma húngaro, Hungría, con ordenación técnica) es simplemente "Hu-Hu".
-   Un nombre de idioma puede ser "neutro" con respecto a la configuración regional. Por ejemplo, "en" (Inglés) es un nombre de idioma neutro válido, como se diferencia de "en-BZ" (Inglés, Belice).
