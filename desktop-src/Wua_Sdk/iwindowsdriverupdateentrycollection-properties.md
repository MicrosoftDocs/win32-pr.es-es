# <a name="iwindowsdriverupdateentrycollection-properties"></a>Propiedades de IWindowsDriverUpdateEntryCollection

La interfaz [**IWindowsDriverUpdateEntryCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentrycollection) define las siguientes propiedades.



| Propiedad                                                          | Descripción                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentrycollection-get__newenum) | Obtiene una interfaz [**IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) que se utiliza para enumerar la colección. |
| [**Contabiliza**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentrycollection-get_count)        | Obtiene el número de elementos de la colección.                                                                       |
| [**Elemento**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentrycollection-get_item)          | Obtiene un intrface [**IWindowsDriverUpdateEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentry) en la colección.                   |
