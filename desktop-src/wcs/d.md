# <a name="d"></a>D

## <a name="dac"></a>DAC

Convertidor de digital a análogo.

## <a name="device-link-profile"></a>perfil de vínculo de dispositivo

Un archivo que contiene una concatenación de conversiones de color que se usan con frecuencia.

## <a name="device-profile"></a>perfil de dispositivo

Un archivo que contiene información sobre cómo caracterizar las funcionalidades de color de un dispositivo. Los perfiles de dispositivo DEER convierten los colores de la gama de un dispositivo en el PCS. También contiene la información necesaria para la conversión de colores del PCS en la gama del dispositivo. Los perfiles de dispositivo XML de WCS proporcionan los datos de medida y las referencias algorítmicas que se convierten entre el espacio del dispositivo nativo y CIEXYZ.

## <a name="dmp"></a>DMP

El formato de perfil de modelo de dispositivo de color es un formato de archivo basado en XML que contiene las medidas necesarias para describir la relación entre los valores de espacio de color del dispositivo nativo y los valores CIEXYZ.
