# <a name="sample-text-section"></a>Sección de texto de ejemplo

Las líneas siguientes muestran una sección text típica de un archivo de definición de máscara:


```C++
//  <Type>       <Location>     <Align>  <Font>          <Color>

//  ------       ----------     -------  ------          -------

    Time         180,46,50,30   Right    Tahoma,16,N     255,255,255

    Playlist     3,46,175,20    Left     Tahoma,10,N     255,255,255

    Track#       10,68,40,20    Left     Tahoma,14,N     255,255,255

```



En el ejemplo anterior se definen tres cuadros de presentación de texto que muestran la hora actual dentro del elemento multimedia, la lista de reproducción actual y el número de pista actual.
