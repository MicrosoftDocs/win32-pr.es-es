<dl> <dt>

<span id="_security_x.200_gly"></span><span id="_SECURITY_X.200_GLY"></span>**X.200**
</dt> <dd>

Un estándar de interconexión de sistemas abiertos (OSI).

</dd> <dt>

<span id="_security_x.208_gly"></span><span id="_SECURITY_X.208_GLY"></span>**X.208**
</dt> <dd>

El estándar itu que define ASN.1.

</dd> <dt>

<span id="_security_x.209_gly"></span><span id="_SECURITY_X.209_GLY"></span>**X.209**
</dt> <dd>

El estándar itu que define las codificaciones BER para objetos ASN.1.

Vea también Reglas [*básicas de codificación*](b-gly.md).

</dd> <dt>

<span id="_security_x.400_gly"></span><span id="_SECURITY_X.400_GLY"></span>**X.400**
</dt> <dd>

El estándar itu que define los tokens de mensaje.

</dd> <dt>

<span id="_security_x.500_gly"></span><span id="_SECURITY_X.500_GLY"></span>**X.500**
</dt> <dd>

El estándar itu que define la estructura de un diccionario global.

</dd> <dt>

<span id="_security_x.509_gly"></span><span id="_SECURITY_X.509_GLY"></span>**X.509**
</dt> <dd>

Un estándar reconocido internacionalmente para los certificados que define sus partes necesarias.

</dd> <dt>

<span id="_security_x509_asn_encoding_gly"></span><span id="_SECURITY_X509_ASN_ENCODING_GLY"></span>**CODIFICACIÓN \_ DE ASN X509 \_**
</dt> <dd>

Tipo de codificación que especifica la codificación del certificado. Los tipos de codificación de certificado se almacenan en la palabra de orden bajo de **un DWORD** (el valor es: 0x00000001). Este tipo de codificación es funcionalmente el mismo que CRYPT \_ ASN \_ ENCODING.

</dd> <dt>

<span id="_security_xor_gly"></span><span id="_SECURITY_XOR_GLY"></span>**Xor**
</dt> <dd>

OR **exclusivo.** Una operación booleana que produce true solo si uno de sus operandos es true y el otro es false. Si ambos operandos son iguales (true o false), la operación produce false.

</dd> </dl>
