# <a name="sensor_category_orientation"></a>\_orientación de categoría de sensor \_

La \_ categoría orientación de categoría de sensor \_ contiene sensores que proporcionan información sobre la orientación física. Los compuestos proporcionan orientación de navegación, como las basadas en el norte magnético. Inclinómetros miden la pendiente o la elevación. Los sensores de distancia miden la proximidad de algún objeto al sensor.
