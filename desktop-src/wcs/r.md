# <a name="r"></a>R

## <a name="rendering-intent"></a>intención de representación

Enfoques para representar el color de un espacio de colores a otro. Los efectos de representación que se usan en los perfiles de ICC se definen mediante el International color Consortium (ICC).

## <a name="rgb"></a>RGB

Espacio de colores rojo, verde y azul.

## <a name="rgb-triple"></a>RGB triple

Una cantidad que contiene valores de espacio de colores RGB.
