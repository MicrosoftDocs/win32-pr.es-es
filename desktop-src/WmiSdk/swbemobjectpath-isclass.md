# <a name="swbemobjectpathisclass-property"></a>Propiedad SWbemObjectPath.IsClass

La **propiedad IsClass** del objeto [**SWbemObjectPath**](swbemobjectpath.md) es un valor booleano que indica si esta ruta de acceso representa una clase. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [Document Conventions for the Scripting API](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemObjectPath.IsClass As Boolean
```
