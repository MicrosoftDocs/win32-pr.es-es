# <a name="retrieving-event-data-using-tdh"></a>Recuperación de datos de eventos mediante TDH

Para consumir datos específicos de eventos, el consumidor debe conocer el formato de los datos de evento. Si el proveedor usaba un manifiesto, MOF o archivos TMF para publicar el formato de los datos de evento, puede utilizar las funciones de la aplicación auxiliar de datos de seguimiento (TDH) para analizar los datos de evento. Las funciones de TDH están disponibles a partir de Windows Vista.

En los temas siguientes se muestra cómo usar TDH para recuperar datos de eventos y metadatos.

-   [Usar TdhFormatProperty para consumir datos de eventos](using-tdhformatproperty-to-consume-event-data.md)
-   [Usar TdhGetProperty para consumir datos de eventos](using-tdhgetproperty-to-consume-event-data.md)
-   [Recuperar metadatos de eventos](retrieving-event-metadata.md)
-   [Enumeración de proveedores](enumerating-providers.md)
