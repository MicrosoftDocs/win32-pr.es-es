# <a name="elementshouldbeoffscreen"></a>ElementShouldBeOffScreen

## <a name="text"></a>Texto

El elemento debe estar oculto

## <a name="type"></a>Tipo

Advertencia

## <a name="description"></a>Descripción

El rectángulo delimitador del elemento primario recorta el rectángulo delimitador del elemento. El elemento debe omitirse y considerarse fuera de los límites del escritorio.
