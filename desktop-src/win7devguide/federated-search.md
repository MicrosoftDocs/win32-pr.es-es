# <a name="federated-search"></a>Búsqueda federada

Windows 7 admite la búsqueda de documentos más allá del propio equipo del usuario. Los desarrolladores y profesionales de TI pueden permitir que se busquen en Windows 7 los motores de búsqueda, los repositorios de documentos, las aplicaciones web y los almacenes de datos de propiedad sin necesidad de escribir e implementar el código de cliente. Esto permite a los usuarios finales buscar su intranet corporativa o la web con la misma facilidad con la que pueden buscar sus archivos locales, todo ello en la misma interfaz de Windows conocida.

Los desarrolladores y profesionales de TI pueden agregar compatibilidad con Windows 7 compatible con *OpenSearch* a cualquier aplicación Web de búsqueda existente agregando la salida *RSS* o *Atom* , tal como se define en el estándar público de [OpenSearch](http://www.opensearch.org/) . Después, los desarrolladores y los usuarios avanzados pueden crear los archivos de descripción de *OpenSearch* para definir los parámetros de conexión que permiten al cliente de escritorio crear un conector de búsqueda para el servicio.

El servidor de búsqueda de SharePoint también puede consultar estos servicios de *OpenSearch* compatibles. Esto también permite el acceso al servicio de búsqueda desde la interfaz web del centro de búsqueda de *SharePoint* .
