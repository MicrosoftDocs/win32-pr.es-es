# <a name="mft_enum_adapter_luid-attribute"></a>Atributo \_ LUID del ADAPTADOR DE \_ \_ ENUM de MFT

Especifica el identificador único de un adaptador de vídeo. Use este atributo al llamar a [**MFTEnum2 para**](/windows/desktop/api/mfapi/nf-mfapi-mftenum2) enumerar los MFT asociados a un adaptador específico.

## <a name="data-type"></a>Tipo de datos

**[**LUID**](/windows/win32/api/devicetopology/ns-devicetopology-luid)**
