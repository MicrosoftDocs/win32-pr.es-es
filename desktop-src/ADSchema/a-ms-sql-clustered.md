# <a name="ms-sql-clustered-attribute"></a>Atributo de clúster de MS-SQL

True si el servidor está en clúster.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | En clúster de MS-SQL                     |
| Nombre para mostrar de LDAP | en clúster de mS-SQL                     |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | Al iniciar el sistema.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1373              |
| System-ID-GUID    | 7778bd90-ccee-11d2-9993-0000f87a57d4 |
| Sintaxis            | [**Booleano**](s-boolean.md)         |
