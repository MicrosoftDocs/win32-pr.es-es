# <a name="parental-controls-minimum-compliance-api"></a>API de cumplimiento mínimo de controles parentales

La API de cumplimiento mínima implementa métodos de interfaz simples para que las aplicaciones obtengan información de estado básica de una cuenta de usuario controlada con control parental, su configuración de restricciones web y la configuración de restricciones de juegos.

-   [**IWindowsParentalControls**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwindowsparentalcontrols)
-   [**IWPCGamesSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcgamessettings)
-   [**IWPCSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcsettings)
-   [**IWPCWebSettings**](/windows/desktop/api/Wpcapi/nn-wpcapi-iwpcwebsettings)
