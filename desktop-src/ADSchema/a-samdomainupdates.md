| CN                | SAM-Domain-Actualizaciones                                    |
| Nombre para mostrar de LDAP | samDomainUpdates                                      |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
