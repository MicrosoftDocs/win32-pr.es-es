# <a name="axwindowsmediaplayerplaylistcollection-property"></a>Propiedad AxWindowsMediaPlayer. playlistCollection

La propiedad playlistCollection obtiene una interfaz **IWMPPlaylistCollection** .

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```CSharp
public IWMPPlaylistCollection playlistCollection {get;}
```


```VB

Public ReadOnly Property playlistCollection As IWMPPlaylistCollection
```





## <a name="property-value"></a>Valor de propiedad

WMPLib. Interfaz **IWMPPlaylistCollection** .

## <a name="remarks"></a>Observaciones

Para recuperar el valor de esta propiedad, se requiere acceso de lectura a la biblioteca. Para obtener más información, vea [acceso a la biblioteca](library-access.md).
