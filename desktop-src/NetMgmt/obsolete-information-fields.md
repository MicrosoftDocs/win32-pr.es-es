# <a name="obsolete-information-fields"></a>Campos de información obsoletos

Muchos de los campos de información de las estructuras de información principales estarán obsoletos. Estos campos permanecerán en la estructura de información por compatibilidad con versiones de 16 bits de Windows y devolverán un valor predeterminado inteligente en otros sistemas Windows datos.
