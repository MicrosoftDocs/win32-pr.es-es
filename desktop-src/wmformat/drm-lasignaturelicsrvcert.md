# <a name="drm_lasignaturelicsrvcert"></a>DRM \_ LASignatureLicSrvCert

El **atributo \_ DRM LASignatureLicSrvCert** contiene el certificado que comprueba el certificado contenido en [**DRM \_ LASignatureCert**](drm-lasignaturecert.md).

## <a name="global-constant"></a>Constante global

g \_ wszWMDRM \_ LASignatureLicSrvCert

## <a name="data-type"></a>Tipo de datos

**CADENA DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentarios

Esta propiedad se puede establecer con el [**método IWMDRMWriter::SetDRMAttribute.**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmdrmwriter-setdrmattribute) No es accesible para el objeto de lector.
