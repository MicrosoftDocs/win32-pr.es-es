# <a name="deviceinfouiclsid-property"></a>Propiedad DeviceInfo. UIClsid

Recupera el identificador de clase de la interfaz de usuario proporcionada por el fabricante para este dispositivo de hardware de adquisición de imágenes de Windows (WIA). El valor es una representación de cadena de un GUID.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = DeviceInfo.UIClsid
```



## <a name="property-value"></a>Valor de propiedad

Cadena que va a recibir el identificador de clase.
