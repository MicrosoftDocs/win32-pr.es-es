# <a name="dns-tombstoned-attribute"></a>DNS-Tombstoned atributo

True si este objeto se ha convertido en un objeto de desecho. Este atributo existe para facilitar y acelerar la búsqueda de registros con valores de desecho. Los objetos con lápida son objetos que se han eliminado, pero que aún no se han quitado del directorio.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | DNS-Tombstoned                       |
| Ldap-Display-Name | dNSTombstoned                        |
| Size              | 4 bytes                              |
| Privilegio actualizar  | El sistema establece este valor.     |
| Frecuencia de actualización  | Siempre que se elimina un objeto.       |
| Attribute-Id      | 1.2.840.113556.1.4.1414              |
| System-Id-Guid    | d5eb2eb7-be4e-463b-a214-634a44d7392e |
| Syntax            | [**Boolean**](s-boolean.md)         |
