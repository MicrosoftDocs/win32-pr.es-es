# <a name="controltyperequiredpatternnotsupported"></a>ControlTypeRequiredPatternNotSupported

## <a name="text"></a>Texto

El elemento con un ControlType de {0} debe admitir {1} pero este elemento no

## <a name="type"></a>Tipo

Error

## <a name="description"></a>Descripción

La compatibilidad de automatización de la interfaz de usuario del elemento no cumple con la especificación de automatización de la interfaz de usuario porque el elemento debe admitir la lista proporcionada de patrones de control, pero no lo hace. Como resultado, es posible que este elemento no esté expuesto correctamente a las tecnologías de asistencia, lo que podría tener un impacto serio en la experiencia del usuario.

## <a name="possible-causes"></a>Causas posibles

-   Implementación de automatización de la interfaz de usuario incompleta.
-   La propiedad ControlType no se ha establecido correctamente.
