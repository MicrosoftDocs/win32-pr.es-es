# <a name="shader-functions-direct3d-10-graphics"></a>Funciones de sombreador (gráficos de Direct3D 10)

Esta sección contiene información sobre las siguientes funciones de sombreador:



| Functions                                                                          | Descripción                                                                 |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------|
| [**D3D10CompileShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10compileshader)                                   | Compile un sombreador.                                                           |
| [**D3D10DisassembleShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10disassembleshader)                           | En desuso. Vea [**D3DX10DisassembleShader.**](d3dx10disassembleshader.md) |
| [**D3D10GetGeometryShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getgeometryshaderprofile)             | Obtenga el perfil de sombreador de geometría más adecuado para un dispositivo determinado.              |
| [**D3D10GetInputAndOutputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getinputandoutputsignatureblob) | Obtiene un búfer que contiene firmas de sombreador.                               |
| [**D3D10GetInputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getinputsignatureblob)                   | Obtiene un búfer que contiene las firmas de entrada del sombreador.                         |
| [**D3D10GetOutputSignatureBlob**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getoutputsignatureblob)                 | Obtiene un búfer que contiene las firmas de salida del sombreador.                        |
| [**D3D10GetPixelShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getpixelshaderprofile)                   | Obtenga el perfil de sombreador de píxeles más adecuado para un dispositivo determinado.                 |
| [**D3D10GetShaderDebugInfo**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getshaderdebuginfo)                         | Obtener información de depuración de sombreador de un sombreador compilado.                               |
| [**D3D10GetVertexShaderProfile**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10getvertexshaderprofile)                 | Obtenga el perfil del sombreador de vértices más adecuado para un dispositivo determinado.                |
| [**D3D10PreprocessShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10preprocessshader)                             | Genere una cadena de texto que contenga la secuencia de tokens del sombreador.               |
| [**D3D10ReflectShader**](/windows/desktop/api/D3D10Shader/nf-d3d10shader-d3d10reflectshader)                                   | En desuso. Vea [**D3DX10ReflectShader.**](d3dx10reflectshader.md)         |
