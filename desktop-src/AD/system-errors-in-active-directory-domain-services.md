# <a name="system-errors-in-active-directory-domain-services"></a>Errores del sistema en Active Directory Domain Services

Este tema contiene valores devueltos de errores del sistema en las funciones Active Directory Domain Services.



| Value                             | Descripción                                                           |
|-----------------------------------|-----------------------------------------------------------------------|
| **hrFileClose**<br/>        | No se puede cerrar el archivo DOS.<br/>                              |
| **hrOutOfThreads**<br/>     | No se puede iniciar un subproceso porque no hay ninguno disponible.<br/> |
| **hrTooManyIO**<br/>        | El sistema está ocupado porque hay demasiadas E/S.<br/>        |
| **hrBFNotSynchronous**<br/> | Se ha expulsado la página del búfer.<br/>                          |
| **hrBFPageNotFound**<br/>   | No se puede encontrar la página.<br/>                                   |
| **hrBFInUse**<br/>          | No se puede abandonar el búfer.<br/>                              |
