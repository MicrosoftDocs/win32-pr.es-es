# <a name="header-and-lib-files"></a>Archivos de encabezado y biblioteca

Los siguientes archivos de encabezado y biblioteca admiten el desarrollo de Microsoft Active Accessibility aplicaciones.



| Archivo         | Descripción                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Winuser.h    | Archivo de encabezado de C y C++ para usar Windows controles                                                                                                                             |
| Oleacc.h     | Archivo de encabezado de C y C++ que contiene las definiciones necesarias para Microsoft Active Accessibility clientes y servidores                                                                    |
| Winver.h     | Archivo de encabezado de C y C++ para obtener información de versión                                                                                                                                |
| Oleacc.lib   | Biblioteca de importación para Oleacc.dll                                                                                                                                                |
| Oleacc.dll   | Biblioteca de vínculos dinámicos que proporciona el tiempo Microsoft Active Accessibility ejecución, administra las solicitudes de Microsoft Active Accessibility clientes y los servidores proxy comunes de la interfaz de usuario |
| Oleaccrc.dll | Biblioteca multilingüe de vínculos dinámicos de recursos que Oleacc.dll                                                                                                                |
