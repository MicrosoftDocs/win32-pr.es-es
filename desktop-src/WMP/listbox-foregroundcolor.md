# <a name="listboxforegroundcolor"></a>LISTBOX. foregroundColor

El atributo **foregroundColor** especifica o recupera el color del texto en el control de cuadro de lista.

``` syntax
        elementID.foregroundColor
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de lectura/escritura que contiene cualquier valor de color de Microsoft Internet Explorer. Tiene un valor predeterminado igual al color del texto de Windows.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------------|
| Versión<br/> | Windows Media Player para Windows XP o posterior<br/> |
