# <a name="msvm_ethernetswitchportfeaturesettingdata-class"></a>Clase Msvm \_ EthernetSwitchPortFeatureSettingData

Clase base abstracta para las clases que representan la configuración de una característica de puerto de conmutador Ethernet.

La sintaxis siguiente se Managed Object Format código de Managed Object Format (MOF) e incluye todas las propiedades heredadas.

## <a name="syntax"></a>Sintaxis

``` syntax
[Abstract, Provider("VmmsWmiInstanceAndMethodProvider"), AMENDMENT]
class Msvm_EthernetSwitchPortFeatureSettingData : Msvm_FeatureSettingData
{
  string InstanceID;
  string Caption;
  string Description;
  string ElementName;
};
```
