# <a name="bits-c-application-examples"></a>Ejemplos de aplicaciones de BITS C++

En la tabla siguiente se enumeran los ejemplos de C++ de esta sección.

Estas clases comunes incluyen una clase de devolución de llamada, una clase de error genérica y una clase auxiliar de adquisición de recursos para la [función COM CoInitializeEx.](/windows/win32/api/combaseapi/nf-combaseapi-coinitializeex) |

| [Ejemplo: Uso de la codificación de autenticación SSPI con BITS.](example-using-sspi-authentication-encoding-with-bits.md)                                                 | En este ejemplo se muestra el uso de la autenticación de la interfaz de proveedor de compatibilidad de seguridad (SSPI) y BITS para obtener las credenciales de un usuario, codificar las credenciales y establecer las credenciales codificadas en un trabajo de transferencia de BITS.                                                                |
