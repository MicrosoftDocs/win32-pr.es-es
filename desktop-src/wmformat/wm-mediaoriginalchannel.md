# <a name="wmmediaoriginalchannel"></a>WM/MediaOriginalChannel

El atributo **WM/MediaOriginalChannel** contiene el canal en el que se difundió el contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaOriginalChannel

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Este atributo se utiliza para la televisión de difusión.
