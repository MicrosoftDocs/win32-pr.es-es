# <a name="properties-of-imscepsetup"></a>Propiedades de IMSCEPSetup

La interfaz [**IMSCEPSetup**](/windows/desktop/api/Casetup/nn-casetup-imscepsetup) define las siguientes propiedades.



| Propiedad                                                 | Descripción                                                                                                                                              |
|----------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSCEPErrorId**](/windows/desktop/api/Casetup/nf-casetup-imscepsetup-get_msceperrorid)         | Obtiene el identificador de la información de error adicional relacionada con una especificación de NDES con error. Cualquier llamada al método en el objeto primario restablece esta propiedad.          |
| [**MSCEPErrorString**](/windows/desktop/api/Casetup/nf-casetup-imscepsetup-get_msceperrorstring) | Obtiene los datos de cadena para la información de error adicional relacionada con una especificación de NDES con error. Cualquier llamada al método en el objeto primario restablece esta propiedad. |
