# <a name="user-configurable-elements"></a>Elementos configurables por el usuario

Este tema no está actualizado. Para obtener la información más reciente, consulte la [especificación del esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

En esta sección se describen las palabras clave públicas que pueden aparecer en un documento de PrintCapabilities o en PrintTicket.

No es necesario que se admitan todas las palabras clave y elementos de las palabras clave. Los implementadores de PrintTicket y PrintCapabilities solo deben admitir la funcionalidad si es aplicable al dispositivo o controlador de destino.

## <a name="element-type-implementations-for-user-configurable-elements"></a>Implementaciones de tipo de elemento para elementos configurables por el usuario

Es importante prestar atención a los elementos primarios y secundarios permitidos en el contenido estructural para cada palabra clave. Para obtener más información, consulte la sección [Resumen de tipos de elemento](./summary-of-element-types.md) .
