# <a name="topoedit-toolbar"></a>Barra de herramientas TopoEdit

De forma predeterminada, la barra de herramientas se muestra en la parte superior de la ventana de la aplicación y debajo de la barra de menús. La barra de herramientas proporciona acceso rápido al mouse para los elementos de [menú controles](controls-menu.md) . Los botones de la barra de herramientas solo se habilitan después de que una topología se resuelva correctamente.

En la captura de pantalla siguiente se muestra la barra de herramientas TopoEdit.

![captura de pantalla que muestra la barra de herramientas topoedit](images/410257c5-ae5c-454c-a9a8-ba05fc5235b5.jpg)
