# <a name="debugging-shaders-in-visual-studio"></a>Depurar sombreadores en Visual Studio

La herramienta más reciente para la depuración de sombreadores ahora se incluye como una característica de Microsoft Visual Studio, denominada depurador de gráficos de Visual Studio. Esta nueva herramienta sustituye a la herramienta [PIX](/windows/desktop/directx-sdk--august-2009-) para Windows. El depurador de gráficos de Visual Studio ha mejorado considerablemente el uso, la compatibilidad con Windows 8 y Direct3D 11,1, y la integración con las características de Visual Studio tradicionales, como las pilas de llamadas y las ventanas de depuración para la depuración de [HLSL](dx-graphics-hlsl.md) . Para obtener más información sobre esta nueva característica, consulte [depuración de gráficos de DirectX](/visualstudio/debugger/visual-studio-graphics-diagnostics?view=vs-2015).
