# <a name="wmtracknumber"></a>WM/TrackNumber

El atributo **WM/TrackNumber** contiene el número de pista del contenido. Este atributo se basa en 1.

## <a name="global-constant"></a>Constante global

g \_ wszWMTrackNumber

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Muchas aplicaciones existentes escriben el valor de **WM/TrackNumber** como **DWORD**. Si crea una aplicación que reproduzca archivos de orígenes desconocidos, debe incluir código para controlar los valores de cadena y **DWORD** .
