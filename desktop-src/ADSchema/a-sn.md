# <a name="surname-attribute"></a>Atributo Surname

Este atributo contiene la familia o los apellidos de un usuario.



| Entrada | Value |
|-------------------|----------------------------------------------------------------------------------|
| CN                | Surname                                                                          |
| Ldap-Display-Name | sn                                                                               |
| Size              | \-                                                                               |
| Privilegio actualizar  | Cualquier persona puede actualizar este objeto en función de la seguridad del objeto que se va a crear. |
| Frecuencia de actualización  | \-                                                                               |
| Attribute-Id      | 2.5.4.4                                                                          |
| System-Id-Guid    | bf967a41-0de6-11d0-a285-00aa003049e2                                             |
| Syntax            | [**String(Unicode)**](s-string-unicode.md)                                      |
