# <a name="mobile-broadband"></a>Banda ancha móvil

## <a name="purpose"></a>Propósito

Mobile Broadband API se usa para implementar la conectividad a redes de telefonía móvil. Las aplicaciones no deben comunicarse directamente con estos dispositivos de banda ancha móvil. En su lugar, deben usar Mobile Broadband API.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Mobile Broadband API está diseñada para desarrolladores de C++.

También hay una interfaz para desarrolladores de .NET que usan C \# . Para obtener más información, lea [estas whitepaper .](https://www.microsoft.com/whdc/connect/wireless/MB_ManagedCode.mspx)

La API de banda ancha móvil se puede usar en aplicaciones de terceros que necesitan controlar y administrar interfaces de banda ancha móvil.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

La compatibilidad con banda ancha móvil comienza con Windows 7.
