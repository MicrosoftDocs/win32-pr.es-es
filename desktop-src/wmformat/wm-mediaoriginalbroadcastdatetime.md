# <a name="wmmediaoriginalbroadcastdatetime"></a>WM/MediaOriginalBroadcastDateTime

El **atributo WM/MediaOriginalBroadcastDateTime** contiene la fecha y hora en que se difundió el contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaOriginalBroadcastDateTime

## <a name="data-type"></a>Tipo de datos

**CADENA DE TIPO WMT \_ \_**

## <a name="remarks"></a>Comentarios

Este atributo se usa para la difusión de televisión o radio.
