# <a name="tf_-constants"></a>TF ( \_ \* constantes)

Las llamadas a funciones de Atom de TSF usan estas constantes para administrar los procesos de entrada de subprocesos.



| Constante o valor                                                                                                                                                                                                                                                                       | Descripción                                                                                          |
|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------|
| <span id="TF_PROCESS_ATOM"></span><span id="tf_process_atom"></span><dl> <dt>**TF \_ PROCESAR texto \_ Atom**</dt> <dt>( \_ CTF \_ Process \_ Atom \_ )</dt> </dl>                               | Se utiliza para notificar a las funciones Atom que ha iniciado un proceso de entrada de subproceso.<br/>                  |
| <span id="TF_ENABLE_PROCESS_ATOM"></span><span id="tf_enable_process_atom"></span><dl> <dt>**TF \_ HABILITAción del procesamiento de texto \_ \_ Atom**</dt> <dt>( \_ CTF \_ enable \_ Process \_ Atom \_ )</dt> </dl> | Se usa para notificar a TSF, a través de las funciones de Atom, que se ha habilitado un proceso de entrada de subprocesos.<br/> |
