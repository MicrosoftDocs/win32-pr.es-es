# <a name="pixels"></a>píxeles

Los fragmentos se convierten en píxeles en el fotogramas. El fotogramas se organiza en un conjunto de búferes lógicos, el color, la profundidad, el estarcido y los búferes de acumulación. El búfer de color se compone de una parte delantera izquierda, frontal derecha, posterior izquierda, atrás a la derecha y un número de búferes auxiliares. Puede emitir funciones para controlar estos búferes y leer o copiar directamente píxeles de ellos. (Tenga en cuenta que el contexto de OpenGL concreto que está usando no puede proporcionar todos estos búferes).

-   [Operaciones de fotogramas](framebuffer-operations.md)
-   [Leer o copiar píxeles](reading-or-copying-pixels.md)
-   [Referencia de píxeles](pixels-reference.md)
