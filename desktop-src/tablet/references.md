# <a name="references-tablet-pc"></a>Referencias (Tablet PC)

Para obtener información acerca de cómo habilitar el reconocimiento de voz en su aplicación, consulte el SDK de Microsoft Speech, que se encuentra en el [SDK de Microsoft Speech](https://www.microsoft.com/speech/download/sdk51/). Este sitio también contiene información general sobre las directrices de diseño de aplicaciones y Speech API de Microsoft.

Para obtener información sobre la accesibilidad de Microsoft, vea [accesibilidad de Microsoft](https://www.microsoft.com/enable/).

Para obtener más información sobre las directrices de Microsoft Active Accessibility para las aplicaciones basadas en Windows, consulte la sección [accesibilidad](../accessibility/accessibility.md) .
