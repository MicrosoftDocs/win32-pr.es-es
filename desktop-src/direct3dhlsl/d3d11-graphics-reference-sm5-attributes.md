# <a name="shader-model-5-attributes"></a>Atributos del modelo de sombreador 5

El modelo de sombreador 5 implementa los atributos del modelo de sombreador 4 y versiones anteriores (vea el SDK de DirectX), así como los nuevos atributos siguientes:

-   [dominio](sm5-attributes-domain.md)
-   [earlydepthstencil](sm5-attributes-earlydepthstencil.md)
-   [repetición](sm5-attributes-instance.md)
-   [maxtessfactor](sm5-attributes-maxtessfactor.md)
-   [numthreads](sm5-attributes-numthreads.md)
-   [outputcontrolpoints](sm5-attributes-outputcontrolpoints.md)
-   [outputtopology](sm5-attributes-outputtopology.md)
-   [particiones](sm5-attributes-partitioning.md)
-   [patchconstantfunc](sm5-attributes-patchconstantfunc.md)

Todos estos atributos son obligatorios (excepto el atributo de **instancia** ). Si no se usan, se producirán errores de compilación.
