# <a name="iautomaticupdates-methods"></a>Métodos IAutomaticUpdates

La [**interfaz IAutomaticUpdates**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdates) define los métodos siguientes.



| Método                                                             | Descripción                                                         |
|--------------------------------------------------------------------|---------------------------------------------------------------------|
| [**DetectNow**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-detectnow)                   | Comienza la actualización automática si aún no se ha iniciado.            |
| [**EnableService**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-enableservice)           | Habilita todos los componentes que Actualizaciones automáticas requiere.         |
| [**Pausar**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-pause)                           | Pausa la actualización automática.                                          |
| [**Reanudar**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-resume)                         | Reinicia la actualización automática si está en pausa.                        |
| [**ShowSettingsDialog**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdates-showsettingsdialog) | Muestra un cuadro de diálogo que contiene la configuración de Actualizaciones automáticas. |
