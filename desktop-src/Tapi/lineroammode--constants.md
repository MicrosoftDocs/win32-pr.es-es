# <a name="lineroammode_-constants"></a>Constantes de LINEROAMMODE \_

Las constantes de marcador de bits **LINEROAMMODE \_** describen el estado de itinerancia de un dispositivo de línea.

<dl> <dt>

<span id="LINEROAMMODE_HOME"></span><span id="lineroammode_home"></span>**\_Página principal de LINEROAMMODE**
</dt> <dd> <dl> <dt>



La línea está conectada al nodo de red doméstica.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_ROAMA"></span><span id="lineroammode_roama"></span>**LINEROAMMODE \_ roaming**
</dt> <dd> <dl> <dt>



La línea está conectada a la portadora itinerancia y las llamadas se cobran en consecuencia.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_ROAMB"></span><span id="lineroammode_roamb"></span>**LINEROAMMODE \_ ROAMB**
</dt> <dd> <dl> <dt>



La línea está conectada a la portadora de itinerancia y las llamadas se cobran en consecuencia.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_UNAVAIL"></span><span id="lineroammode_unavail"></span>**LINEROAMMODE no \_ disponible**
</dt> <dd> <dl> <dt>



El modo de itinerancia no está disponible y no se conocerá.


</dt> </dl> </dd> <dt>

<span id="LINEROAMMODE_UNKNOWN"></span><span id="lineroammode_unknown"></span>**LINEROAMMODE \_ desconocido**
</dt> <dd> <dl> <dt>



El modo de itinerancia es desconocido actualmente, pero se puede conocer más adelante.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Observaciones

Sin extensibilidad. Todos los 32 bits están reservados.
