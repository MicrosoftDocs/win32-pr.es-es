# <a name="windows-defender"></a>Windows Defender

Windows Defender ayuda a proteger los equipos contra las ventanas emergentes, el rendimiento lento y las amenazas de seguridad causadas por spyware y otro software no deseado. Las aplicaciones pueden consultar información como el estado del equipo, información de malware específica e historial de malware para el cliente, y pueden invocar exámenes a petición.
