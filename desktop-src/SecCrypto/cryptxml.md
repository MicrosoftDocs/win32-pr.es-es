# <a name="cryptxml"></a>CryptXML

CryptXML proporciona un conjunto de bajo nivel de API nativas que implementan la creación y comprobación de firmas digitales XML en función de la recomendación de W3C sobre la sintaxis y el procesamiento de la firma XML (segunda edición). Los desarrolladores deben estar familiarizados con este estándar para usar de forma eficaz las API de firma digital CryptXML.

CryptXML contiene los siguientes temas:

-   [Información general sobre la firma digital XML](xml-digital-signature-overview.md)
-   [Funcionalidad de la API de firma digital XML](xml-digital-signature-api-functionality.md)
-   [Algoritmos criptográficos de firma digital XML](xml-digital-signature-cryptographic-algorithms.md)
-   [Extensiones criptográficas de firmas digitales XML](xml-digital-signature-cryptographic-extensions.md)
-   [Consideraciones de seguridad de las firmas digitales XML](xml-digital-signature-security-considerations.md)
