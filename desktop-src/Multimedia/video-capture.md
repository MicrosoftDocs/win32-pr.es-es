# <a name="video-capture-windows-multimedia"></a>Captura de vídeo (Windows Multimedia)

Puede incorporar fácilmente funcionalidades de captura de vídeo en la aplicación mediante la clase de ventana AVICap. AVICap proporciona a las aplicaciones una interfaz sencilla basada en mensajes para acceder al hardware de adquisición de audio y vídeo y forma de onda y para controlar el proceso de captura de vídeo en streaming en disco.

-   [Uso de captura de vídeo](using-video-capture.md)
-   [Referencia de captura de vídeo](video-capture-reference.md)
