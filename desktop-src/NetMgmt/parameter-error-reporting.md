# <a name="parameter-error-reporting"></a>Informes de errores de parámetros

Las funciones de administración de red agregar y SetInfo devuelven un índice para un parámetro que tiene un error. El autor de la llamada puede pasar un puntero **nulo** para el parámetro *\_ Err PARM* que indica que la función no debe establecer el campo.
