# <a name="understanding-the-application-compatibility-challenge"></a>Descripción del desafío de la compatibilidad de aplicaciones

Windows Internet Explorer 8 es el explorador predeterminado en el Windows operativo 7. Internet Explorer 8 mejora muchas características, incluida una arquitectura de plataforma mejorada diseñada para que la web sea más segura y rápida para los usuarios y sea más fácil de configurar y administrar para los profesionales de TI, y para crear contenido innovador para desarrolladores web.

Algunas de estas características usan nuevos motores de scripting y representación, que son los elementos principales que Windows Internet Explorer para mostrar contenido HTML y CSS y ejecutar scripts. Los cambios arquitectónicos y las mejoras de la plataforma presentan problemas de compatibilidad de aplicaciones que puede experimentar al migrar aplicaciones a Internet Explorer 8 desde un explorador anterior.

Internet Explorer 8 incluye cierta compatibilidad con versiones anteriores, pero hay algunos cambios (incluida la compatibilidad con la compatibilidad con estándares web y las características de seguridad) que pueden requerir que actualice las aplicaciones web creadas para Microsoft Internet Explorer 6 o versiones anteriores.

Para obtener más información sobre cómo probar y solucionar problemas de compatibilidad para Internet Explorer 8, vea [Addressing Application Compatibility When Migrating to Internet Explorer 8 – Information for Information Technology Professionals](/previous-versions/windows/it-pro/windows-7/ff955640(v=ws.10)).
