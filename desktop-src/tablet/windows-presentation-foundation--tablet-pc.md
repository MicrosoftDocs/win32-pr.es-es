# <a name="windows-presentation-foundation-tablet-pc"></a>Windows Presentation Foundation (Tablet PC)

Las características de Tablet PC del Windows Presentation Foundation (WPF) no requieren ninguna configuración especial.

Siga las instrucciones para configurar WPF para usar el InkCanvas y otros componentes de WPF.
