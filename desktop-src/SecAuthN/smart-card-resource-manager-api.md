# <a name="smart-card-resource-manager-api"></a>Smart Card Resource Manager API

La [*API de Resource Manager*](../secgloss/r-gly.md) proporciona la siguiente funcionalidad.



| Tema                                                                                               | Descripción                                                                                                                                                                                                                                                                 |
|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Funciones de consulta de base de datos de tarjeta inteligente](smart-card-database-query-functions.md)<br/>           | Consulte la base de datos de tarjetas inteligentes.<br/>                                                                                                                                                                                                                                   |
| [Funciones de administración de bases de datos de tarjetas inteligentes](smart-card-database-management-functions.md)<br/> | Administrar la base de datos de tarjetas inteligentes.<br/>                                                                                                                                                                                                                                  |
| [Resource Manager funciones de contexto](resource-manager-context-functions.md)<br/>             | Administre el contexto de las operaciones de base de datos del administrador de recursos.<br/>                                                                                                                                                                                                |
| [Resource Manager support (Función)](resource-manager-support-function.md)<br/>               | Liberar memoria asignada.<br/>                                                                                                                                                                                                                                        |
| [Funciones de seguimiento de tarjetas inteligentes](smart-card-tracking-functions.md)<br/>                       | Realice un seguimiento de las tarjetas inteligentes dentro de los lectores.<br/>                                                                                                                                                                                                                                |
| [Funciones de acceso de lector y tarjeta inteligente](smart-card-and-reader-access-functions.md)<br/>     | Conectar y comunicarse con una tarjeta inteligente, incluida la transferencia de datos mediante [*T=0,*](../secgloss/t-gly.md) [*T=1*](../secgloss/t-gly.md)y protocolos sin procesar.<br/> |
| [Funciones de acceso directo a tarjetas](direct-card-access-functions.md)<br/>                         | Comunicarse con tarjetas que pueden no cumplir las especificaciones iso 7816.<br/>                                                                                                                                                                                      |
