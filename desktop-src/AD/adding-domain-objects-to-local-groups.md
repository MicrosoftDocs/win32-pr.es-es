<span data-ttu-id="cd727-106">Al administrar grupos locales de dominio en un dominio de Windows 2000 mediante ADSI, se usa normalmente el proveedor LDAP.</span><span class="sxs-lookup"><span data-stu-id="cd727-106">When managing domain local groups on a Windows 2000 domain using ADSI, the LDAP provider is normally used.</span></span> <span data-ttu-id="cd727-107">Sin embargo, al administrar grupos locales en servidores miembro y un equipo que ejecuta Windows 2000 Professional, se debe usar el proveedor de Winnt.</span><span class="sxs-lookup"><span data-stu-id="cd727-107">When managing local groups on member servers and a computer running Windows 2000 Professional, however, the WinNT provider must be used.</span></span>

<span data-ttu-id="cd727-108">Solo se pueden crear grupos locales en servidores miembro y Windows 2000 Professional.</span><span class="sxs-lookup"><span data-stu-id="cd727-108">Only local groups can be created on member servers and Windows 2000 Professional.</span></span> <span data-ttu-id="cd727-109">Sin embargo, los grupos locales pueden contener:</span><span class="sxs-lookup"><span data-stu-id="cd727-109">However, the local groups can contain:</span></span>

<span data-ttu-id="cd727-113">**Para agregar un objeto de dominio a un grupo local**</span><span class="sxs-lookup"><span data-stu-id="cd727-113">**To add a domain object to a local group**</span></span>

1.  <span data-ttu-id="cd727-114">Enlace a la interfaz [**IADsContainer**](/windows/desktop/api/iads/nn-iads-iadscontainer) del equipo que contiene el grupo local al que se va a agregar un miembro.</span><span class="sxs-lookup"><span data-stu-id="cd727-114">Bind to the [**IADsContainer**](/windows/desktop/api/iads/nn-iads-iadscontainer) interface of the computer that contains the local group to add a member to.</span></span> <span data-ttu-id="cd727-115">El enlace se debe realizar mediante una cuenta que tenga derechos suficientes para acceder a ese equipo.</span><span class="sxs-lookup"><span data-stu-id="cd727-115">The binding must be performed using an account that has sufficient rights to access that computer.</span></span> <span data-ttu-id="cd727-116">La cadena de enlace debe tener el formato "WinNT:// <computer name> , equipo" donde " &lt; 3.  <span data-ttu-id="cd727-121">Enlace a la interfaz [**IADsGroup**](/windows/desktop/api/iads/nn-iads-iadsgroup) del grupo local al que se va a agregar un miembro.</span><span class="sxs-lookup"><span data-stu-id="cd727-121">Bind to the [**IADsGroup**](/windows/desktop/api/iads/nn-iads-iadsgroup) interface of the local group to which a member will be added.</span></span>

5.  <span data-ttu-id="cd727-123">Agregue el usuario o grupo al grupo local con el método [**IADsGroup.
