# <a name="messages-touch-input"></a>Mensajes (entrada táctil)

Esta sección contiene mensajes para la entrada multitoque.

Los mensajes siguientes se usan para la entrada multitoque.



| Message                           | Descripción                                                                                                         |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------|
| [**táctil de WM \_**](wm-touchdown.md) | Indica que uno o más puntos táctiles, como un dedo o un lápiz, han tocado una superficie del digitalizador sensible al tacto. |
