# <a name="swbempropertysetcount-property"></a>SWbemPropertySet. Count (propiedad)

Use la propiedad **Count** Automation del objeto [**SWbemPropertySet**](swbempropertyset.md) para determinar el número de elementos que hay en la colección **SWbemPropertySet** . Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemPropertySet.Count As Integer
```
