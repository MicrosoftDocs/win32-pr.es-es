# <a name="settings-area-plug-ins"></a>Complementos de área de configuración

En Microsoft Windows Media Player 11 y versiones anteriores, los complementos de la interfaz de usuario del área de configuración se muestran en el área herramientas del panel **reproducción en curso** debajo del área de presentación principal y encima de los controles de transporte. En Microsoft Windows Media Player 12 y versiones posteriores, los complementos de la interfaz de usuario del área de configuración aparecen como ventanas independientes. Solo se puede habilitar un complemento de área de configuración a la vez.

Los complementos de área de configuración son útiles para mostrar paneles de control personalizados similares a los controles de configuración de vídeo o ecualizador gráfico.
