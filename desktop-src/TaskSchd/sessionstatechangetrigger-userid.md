# <a name="sessionstatechangetriggeruserid-property"></a>SessionStateChangeTrigger. UserId (propiedad)

En el caso de scripting, obtiene o establece el usuario para la sesión de Terminal Server. Cuando se detecta un cambio de estado de sesión para este usuario, se inicia una tarea.

## <a name="syntax"></a>Sintaxis


```VB
SessionStateChangeTrigger.UserId As String
```



## <a name="property-value"></a>Valor de propiedad

Usuario para la sesión de Terminal Server.
