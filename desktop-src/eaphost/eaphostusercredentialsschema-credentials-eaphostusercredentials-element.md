# <a name="credentials-eaphostusercredentials-element"></a>Credentials (elemento) (EapHostUserCredentials)

El elemento **Credentials (EapHostUserCredentials)** se usa cuando la configuración del método está en formato de texto XML en lugar de un BLOB binario.

``` syntax
<xs:element name="Credentials"
    type="BaseEapMethodUserCredentials"
 />
```

El elemento **Credentials** se define mediante el elemento [**EapHostUserCredentials**](eaphostusercredentialsschema-eaphostusercredentials-element.md) .

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|-------------------------------------|------------------------------------------------------|
| Cliente mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows Vista \[\]<br/>       |
| Servidor mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows Server 2008 \[\]<br/> |
