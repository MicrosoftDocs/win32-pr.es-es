# <a name="systemmonitoronsamplecollected-event"></a>Evento SystemMonitor. OnSampleCollected

Le informa cuando se han recopilado valores de ejemplo para los contadores.

## <a name="syntax"></a>Sintaxis


```VB
SystemMonitor.OnSampleCollected()
```



## <a name="parameters"></a>Parámetros

Este evento no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este evento no devuelve un valor.
