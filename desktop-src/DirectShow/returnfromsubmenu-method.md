# <a name="returnfromsubmenu-method"></a>Método ReturnFromSubmenu

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

El `ReturnFromSubmenu` método devuelve la presentación de un submenú a su menú primario o al título actual si el menú es un menú de nivel superior.

``` syntax
MSWebDVD.ReturnFromSubmenu()
```

## <a name="return-value"></a>Valor devuelto

No de devuelve ningún valor.
