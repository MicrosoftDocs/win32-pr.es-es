# <a name="router-management-mib-reference"></a>Referencia de MIB de administración de enrutadores

En esta documentación se describen los elementos de referencia que se usan en la API de Base de información de administración (MIB) específica de la administración de enrutadores. Use estos elementos de referencia para consultar y establecer los valores de las variables de MIB. Los administradores de enrutadores y los protocolos de enrutamiento que los servicios de administrador de enrutadores exportan las variables de MIB:

-   [Funciones miB de administración de enrutadores](mib-functions.md)
-   [Constantes de transporte y protocolo](transport-and-protocol-constants.md)

Las estructuras que definen variables de MIB se documentan en el tema de redes de [API base](/previous-versions/windows/desktop/mib/management-information-base-reference) de administración independiente de la tecnología.
