# <a name="the-agent-object-model"></a>Modelo de objetos del Agente

\[Microsoft Agent está en desuso a partir de Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

El modelo de objetos de Microsoft Agent consta de los siguientes objetos:

-   Solicitud
-   Agente (control)
-   Caracteres (colección)
-   Carácter
-   Comandos (colección)
-   Get-Help
-   Globo
-   AnimationNames (colección)
-   SpeechInput
-   AudioOutput
-   CommandsWindow
-   PropertySheet

Estos objetos se organizan en la siguiente jerarquía. (La línea de puntos que sigue a un objeto indica que pueden existir varios objetos).
