# <a name="direct3d-11-graphics"></a>Gráficos de Direct3D 11

Puede usar gráficos de Microsoft Direct3D 11 para crear gráficos 3D para juegos y aplicaciones científicas y de escritorio.

Esta sección contiene información sobre la programación con gráficos de Direct3D 11.

Para obtener más información, [vea Características de Direct3D 11.](direct3d-11-features.md)



|                                   | Admitido                                                                                                  |
|-----------------------------------|---------------------------------------------------------------------------------------------------|
| **Entornos de tiempo de ejecución compatibles**    | Windows/C++            |
| **Idiomas de programación recomendados** | C/C++                                                                                             |
| **Cliente mínimo compatible**          | Windows 7             |
| **Servidor mínimo compatible**          | Windows Server 2008 R2 |

 

| Tema                                                                          | Descripción                                                                                                                                                                                           |

| [Novedades de Direct3D 11](dx-graphics-overviews-introduction.md)<br/> | En esta sección se describen las características agregadas en Direct3D 11, Direct3D 11.1 y Direct3D 11.2.<br/>                                                                                                    |
| [Programming Guide for Direct3D 11 (Guía de programación para Direct3D 11)](dx-graphics-overviews.md)<br/>      | La guía de programación contiene información sobre cómo usar la canalización programable de Direct3D 11 para crear gráficos 3D en tiempo real para juegos, así como aplicaciones científicas y de escritorio.<br/> |
| [Referencia de Direct3D 11](atoc-d3d11-graphics-reference.md)<br/>          | Esta sección contiene las páginas de referencia para la programación de gráficos basada en Direct3D 11.<br/>                                                                                                      |



 

Además de que Windows 7 y versiones posteriores admiten Direct3D 11 y Windows Server 2008 R2 y versiones posteriores, Direct3D 11 está disponible de nivel inferior para Windows Vista con Service Pack 2 (SP2) y Windows Server 2008 mediante la descarga de [KB 971644](https://support.microsoft.com/kb/971644) y [KB 971512.](https://support.microsoft.com/kb/971512/)

Para obtener información sobre las nuevas características de Direct3D 11.1 que se incluyen con Windows 8, Windows Server 2012 y están parcialmente disponibles en Windows 7 y Windows Server 2008 R2 a través de la actualización de plataforma para [Windows 7,](https://support.microsoft.com/kb/2670838)vea Características de [Direct3D 11.1](direct3d-11-1-features.md) y la actualización de plataforma [para Windows 7.](/windows/desktop/direct3darticles/platform-update-for-windows-7)
