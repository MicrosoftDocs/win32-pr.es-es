# <a name="oidname-property"></a>OID. Name (propiedad)

\[La propiedad **nombre** está disponible para su uso en los sistemas operativos especificados en la sección requisitos. En su lugar, use la [**clase OID**](/dotnet/api/system.security.cryptography.oid?view=netcore-3.1) en el espacio de nombres [**System. Security. Cryptography**](/dotnet/api/system.security.cryptography?view=dotnet-plat-ext-3.1&preserve-view=true) .\]

La propiedad **Name** establece o recupera el nombre definido por CAPICOM para el identificador. Esta es la propiedad predeterminada.

## <a name="syntax"></a>Sintaxis


```VB
OID.Name As CAPICOM_OID
```
