# <a name="using-the-dimmer-device-sample-code"></a>Uso del código de ejemplo del dispositivo de dimmer

Platform Software Development Kit (SDK) incluye código de ejemplo de dispositivo de dimmer. Este código de ejemplo muestra un dispositivo sencillo basado en UPnP implementado mediante device host API con tecnología UPnP.

El directorio DCO \_ DimmerService contiene la implementación del dispositivo .dll. El directorio RegisterDevice contiene la implementación para registrar el dispositivo .dll con el servicio de registrador UPnP en el equipo. El directorio StepByStep contiene proyectos VC6.0 que corresponden a la "Guía paso a paso" incluida en el Readme.txt paso. La "Guía paso a paso" es para desarrolladores que no tienen experiencia atl/COM anterior.
