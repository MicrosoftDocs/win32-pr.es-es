# <a name="native-wifi"></a>Wi-Fi nativo

## <a name="purpose"></a>Propósito

El componente de configuración automática Wi-Fi nativo configura, conecta y desconecta de las redes inalámbricas. Native Wifi puede almacenar perfiles en las redes con las que interactúa en forma de documentos XML.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Native Wifi API está diseñada para desarrolladores de C/C++. Los programadores deben estar familiarizados con la terminología y los conceptos de redes inalámbricas.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

El componente Wi-Fi nativo requiere clientes que ejecuten Windows Vista, Windows XP con Service Pack 3 (SP3) o wireless LAN API para Windows XP con Service Pack 2 (SP2).
