# <a name="configurableitemname-property"></a>ConfigurableItem.Name propiedad

La **propiedad Name** del objeto [**ConfigurableItem**](configurableitem-object.md) devuelve el valor de la columna Name de la tabla ModuleConfiguration.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = ConfigurableItem.Name
```



## <a name="property-value"></a>Valor de propiedad

## <a name="c"></a>C++
