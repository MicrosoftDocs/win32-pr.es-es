# <a name="iupdate4-properties"></a>Propiedades de IUpdate4

La interfaz [**IUpdate4**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate4) define la propiedad siguiente.



| Propiedad                            | Descripción                                                            |
|-------------------------------------|------------------------------------------------------------------------|
| [**Perusuario**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate4-get_peruser) | Obtiene un valor booleano que indica si se trata de una actualización por usuario. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea las siguientes interfaces:

-   [**IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3)
-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
