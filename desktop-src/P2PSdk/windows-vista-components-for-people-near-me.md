# <a name="windows-vista-components-for-people-near-me"></a>Componentes de Windows Vista para People Near me

[People Near me](about-people-near-me.md) (PNM) usa los siguientes componentes de Windows Vista:

### <a name="web-service-discovery"></a>Descubrimiento de servicios Web

El componente de detección de servicios web permite la publicación y detección de servicios en la subred local. En PNM, un usuario es un servicio que se anuncia en la subred local. Para obtener más información sobre la detección de servicios Web, vea [detección de servicios Web XML](/previous-versions/dotnet/netframework-4.0/fxx6cfx2(v=vs.100)).

### <a name="schannel"></a>Schannel

El componente Schannel permite la comunicación cifrada entre dos extremos.

### <a name="windows-sockets"></a>Windows Sockets

Las aplicaciones usan el componente Windows Sockets para las comunicaciones a través de TCP/IP.

### <a name="next-generation-tcpip-stack"></a>Pila TCP/IP de próxima generación

Nueva pila TCP/IP en Windows Vista que tiene integrado IPv6 y habilitado de forma predeterminada. Para obtener más información, consulte [pila TCP/IP de próxima generación en Windows Vista](https://www.microsoft.com/technet/itsolutions/network/tcpip/default.mspx).
