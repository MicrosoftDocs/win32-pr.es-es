# <a name="building-a-dsp-plug-in"></a>Crear un complemento DSP

Puede crear un complemento DSP mediante Visual Studio y el Asistente para complementos de Windows Media Player. Para obtener información sobre cómo configurar el asistente y crear un complemento, vea [crear un complemento](building-a-plug-in.md).

Al ejecutar el asistente, se le ofrecerá la posibilidad de elegir entre varios tipos de complementos. Elija **complemento DSP**.

En los temas siguientes se describen las características de los complementos de DSP de audio y vídeo de ejemplo que se pueden crear con el asistente.

-   [Acerca del complemento DSP de audio de ejemplo](about-the-sample-audio-dsp-plug-in.md)
-   [Acerca del complemento DSP de vídeo de ejemplo](about-the-sample-video-dsp-plug-in.md)
