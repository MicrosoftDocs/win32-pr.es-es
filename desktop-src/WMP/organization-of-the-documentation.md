# <a name="organization-of-the-documentation"></a>Organización de la documentación

El SDK de Windows Media Player documenta una amplia variedad de tecnologías relacionadas con Windows Media Player. Para que sea más fácil encontrar la información que necesita, la documentación se divide en secciones que le ayudan a encontrar rápidamente temas sobre una característica o tecnología determinada.

Cuando corresponda, las secciones principales incluyen tres subsecciones: acerca de, guía de programación y referencia. En la sección acerca de se describe la característica o tecnología específica para ayudarle a entender cómo puede usarla. La sección Guía de programación agrega tutoriales que le ofrecen más información sobre la tecnología con instrucciones paso a paso. En la sección de referencia se detallan las interfaces, objetos, métodos, propiedades, eventos, elementos y atributos que están disponibles para su uso.
