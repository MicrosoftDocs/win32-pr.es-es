# <a name="converting-a-binary-sid-to-string-format-in-c"></a>Convertir un SID binario en formato de cadena en C++

Las funciones [**ConvertSidToStringSid**](/windows/desktop/api/Sddl/nf-sddl-convertsidtostringsida) y [**ConvertStringSidToSid**](/windows/desktop/api/Sddl/nf-sddl-convertstringsidtosida) convierten un [*identificador de seguridad*](/windows/desktop/SecGloss/s-gly) (SID) a y desde el formato de cadena. Para obtener una descripción del formato de cadena SID, vea [SID Components](sid-components.md).
