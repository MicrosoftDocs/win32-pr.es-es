# <a name="remote-desktop-services-user-configuration"></a>Servicios de Escritorio remoto configuración de usuario

Las [**funciones WTSQueryUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtsqueryuserconfiga) y [**WTSSetUserConfig**](/windows/desktop/api/Wtsapi32/nf-wtsapi32-wtssetuserconfiga) permiten establecer y recuperar información de configuración para una cuenta de usuario especificada. El [**WTS \_ de \_ enumeración CONFIG CLASS**](/windows/desktop/api/Wtsapi32/ne-wtsapi32-wts_config_class) define los parámetros de configuración de usuario.
