# <a name="implementing-your-code"></a>Implementación del código

Una vez que haya completado el proceso de compilación, probablemente querrá mejorar el código generado por el asistente. De lo contrario, la visualización se parecerá mucho a barras. Lo que debe hacer es cambiar la implementación de la **función Render** y, posiblemente, la **función RenderWindowed.**

En las secciones siguientes se proporcionan detalles sobre cómo modificar el código de ejemplo generado por el asistente para complementos:

-   [Implementación de Render](implementing-render.md)
-   [Implementar otras funciones](implementing-other-functions.md)
-   [Valores preestablecidos](presets.md)
