# <a name="the-policy-object-type"></a>El tipo de objeto de Directiva

El objeto de [**Directiva**](policy-object.md) es un objeto contenedor. Los siguientes tipos de objetos pueden residir en este contenedor:

-   Objetos de [**cuenta**](account-object.md)
-   Objeto [**TrustedDomain**](trusteddomain-object.md)
-   Objeto Secret

Solo hay una instancia del objeto de [**Directiva**](policy-object.md) . No se puede crear ni destruir.
