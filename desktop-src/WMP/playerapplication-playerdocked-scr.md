# <a name="playerapplicationplayerdocked"></a>PLAYERAPPLICATION.playerDocked

El atributo **playerDocked** recupera un valor que indica si el Media Player de Windows está en un estado acoplado.

``` syntax
        elementID.playerDocked
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **valor booleano** de solo lectura.



| Value | Descripción                       |
|-------|-----------------------------------|
| True  | Windows Media Player está acoplado.   |
| False | Windows Media Player está desacoplado. |



 

## <a name="remarks"></a>Observaciones

Este atributo solo se usa cuando la comunicación remota del control de Media Player de Windows.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------|
| Versión<br/> | Windows Media Player 9 series o posterior<br/> |
