# <a name="classes-and-interfaces"></a>Clases e interfaces

Esta sección contiene referencias de clase e interfaz para la inercia y las manipulaciones.

Las siguientes clases e interfaces se usan para la inercia y las manipulaciones.



| Clase o interfaz                                          | Descripción                                                                  |
|----------------------------------------------------------|------------------------------------------------------------------------------|
| [**\_IManipulationEvents**](/windows/win32/api/manipulations/nn-manipulations-_imanipulationevents)    | Controla los eventos de manipulación.                                                 |
| [**IInertiaProcessor**](/windows/desktop/api/manipulations/nn-manipulations-iinertiaprocessor)           | Controla los cálculos relacionados con el movimiento de objetos Windows Touch.              |
| [**IManipulationProcessor**](/windows/desktop/api/manipulations/nn-manipulations-imanipulationprocessor) | Proporciona funcionalidad para supervisar y responder a Windows entrada táctil. |
