# <a name="xmunibble4-operators"></a>Operadores XMUNIBBLE4

La [**estructura XMUNIBBLE4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmunibble4) expone los operadores siguientes.

> [!Note]  
> Los operadores enumerados aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                               | Descripción                                                                                                              |
|-----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**operador uint16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmunibble4-operatoruint16_t)<br/> | Devuelve una instancia de `uint16_t` que contiene los componentes de la instancia en un formato `XMUNIBBLE4` empaquetado. <br/> |
| [**operator =**](xmunibble4-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMUNIBBLE4` . <br/>                                       |
