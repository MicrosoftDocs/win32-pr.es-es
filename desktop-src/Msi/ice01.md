# <a name="ice01"></a>ICE01

ICE01 valida que el mecanismo de ICE funciona. Esta ICE usa la propiedad [**Time**](time.md) para obtener la hora y devuelve la hora del sistema o ninguna.

## <a name="result"></a>Resultado

ICE01 envía un mensaje que le indica la hora a la que el instalador llamó a ICE01. Esta ICE nunca debe devolver un error.
