# <a name="resource-management-classes"></a>Clases para la administración de recursos

Las siguientes son clases WMI de virtualización relacionadas con la administración de recursos.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                                        | Descripción                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ AbstractResourcePoolSettingData**](msvm-abstractresourcepoolsettingdata.md)<br/>             | Representa la configuración de una [**instancia de \_ ResourcePool de Msvm**](msvm-resourcepool.md) que no está relacionada con la asignación.<br/>                                |
| [**Asignación de \_ MsvmCapabilities**](msvm-allocationcapabilities.md)<br/>                               | Define los medios por los que un cliente puede detectar el intervalo válido de configuración predeterminada para un recurso virtual.<br/>                                          |
| [**Elemento \_ MsvmAllocatedFromPool**](msvm-elementallocatedfrompool.md)<br/>                           | Asocia una instancia de un recurso asignado al grupo de recursos desde el que se asignó.<br/>                                                   |
| [**Elemento \_ MsvmCapabilities**](msvm-elementcapabilities.md)<br/>                                     | Representa la asociación entre los elementos administrados y sus funcionalidades.<br/>                                                                           |
| [**Msvm \_ FeatureSettingData**](msvm-featuresettingdata.md)<br/>                                       | Clase abstracta que representa la configuración de una característica específica de un sistema o componente.<br/>                                                           |
| [**Característica de \_ MsvmSettingsDefineCapabilities**](msvm-featuresettingsdefinecapabilities.md)<br/>         | Proporciona un vínculo entre la instancia de funcionalidades de la característica de conmutador Ethernet y la configuración mínima, máxima, incremental y predeterminada de un recurso.<br/> |
| [**Msvm \_ ResourceAllocationFromPool**](msvm-resourceallocationfrompool.md)<br/>                       | Asocia una instancia de una asignación de recursos al grupo de recursos desde el que se asigna.<br/>                                                    |
| [**Msvm \_ ResourceAllocationSettingData**](msvm-resourceallocationsettingdata.md)<br/>                 | Representa los estados de asignación actuales y registrados de un recurso virtual.<br/>                                                                          |
| [**ResourcePool de Msvm \_**](msvm-resourcepool.md)<br/>                                                   | Describe un tipo de recurso virtual disponible para su uso en máquinas virtuales.<br/>                                                                           |
| [**Msvm \_ ResourcePoolConfigurationCapabilities**](msvm-resourcepoolconfigurationcapabilities.md)<br/> | Describe las funcionalidades de la clase [**\_ ResourcePoolConfigurationService de Msvm**](msvm-resourcepoolconfigurationservice.md) asociada.<br/>           |
| [**Msvm \_ ResourcePoolConfigurationService**](msvm-resourcepoolconfigurationservice.md)<br/>           | Proporciona la administración activa de grupos de recursos.<br/>                                                                                                     |
| [**Msvm \_ ResourcePoolSettingData**](msvm-resourcepoolsettingdata.md)<br/>                             | Representa la configuración de una [**instancia de \_ ResourcePool de Msvm**](msvm-resourcepool.md) que no está relacionada con la asignación.<br/>                                |
| [**Configuración de \_ MsvmDefineCapabilities**](msvm-settingsdefinecapabilities.md)<br/>                       | Proporciona un vínculo entre la instancia de funcionalidades y la configuración mínima, máxima, incremental y predeterminada de un recurso.<br/>                         |
