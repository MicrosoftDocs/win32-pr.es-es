# <a name="cooker_min"></a>COOKER \_ mín.

La \_ fórmula Cooker min Counter Type identifica el valor más pequeño de un conjunto de observaciones de una propiedad en una instancia de [**\_ PerfRawData de Win32**](/windows/desktop/CIMWin32Prov/win32-perfrawdata) . Este valor mínimo calculado se coloca en una propiedad de una instancia de [**\_ PerfFormattedData de Win32**](/windows/desktop/CIMWin32Prov/win32-perfformatteddata) . Este tipo de contador solo se define en WMI y no está disponible para las tecnologías de supervisión de rendimiento, como los [contadores de rendimiento versión 6,0](/windows/desktop/PerfCtrs/performance-counters-portal).

Para obtener más información sobre la fórmula de tipo de contador, vea [tipos de contadores](/previous-versions/windows/it-pro/windows-server-2003/cc785636(v=ws.10)).

Para obtener más información acerca de los proveedores y scripting de WMI de alto rendimiento, consulte [tipos de contadores de rendimiento de WMI](wmi-performance-counter-types.md).
