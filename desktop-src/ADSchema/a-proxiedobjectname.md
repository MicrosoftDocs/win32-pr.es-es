# <a name="proxied-object-name-attribute"></a>Atributo de nombre de objeto de proxy

Este atributo lo usa internamente Active Directory para ayudar a realizar el seguimiento de los movimientos entre dominios.



| Entrada | Value |
|-------------------|-------------------------------------------------|
| CN                | Nombre-objeto-proxy                             |
| Nombre para mostrar de LDAP | proxiedObjectName                               |
| Tamaño              | \-                                              |
| Actualizar privilegio  | Lo utiliza el sistema.                     |
| Frecuencia de actualización  | \-                                              |
| Attribute-Id      | 1.2.840.113556.1.4.1249                         |
| System-ID-GUID    | e1aea402-cd5b-11d0-afff-0000f80367c1            |
| Sintaxis            | [**Object(DN-Binary)**](s-object-dn-binary.md) |
