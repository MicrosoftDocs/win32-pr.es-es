# <a name="methodexceptionoccured"></a>MethodExceptionOccured

## <a name="text"></a>Texto

Se produjo una excepción \\ " {0} \\ "

## <a name="type"></a>Tipo

Error

## <a name="description"></a>Descripción

Excepción inesperada o HRESULT. Algunas excepciones o HRESULT son válidos, pero en este caso no se esperan los resultados. Este error debe corregirse.
