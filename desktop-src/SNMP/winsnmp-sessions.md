# <a name="winsnmp-sessions"></a>Sesiones de WinSNMP

Una *sesión* es la unidad básica de administración de recursos y comunicaciones entre una aplicación WinSNMP que realiza la llamada y la implementación de Microsoft WinSNMP. Se recomienda que una aplicación use sesiones para organizar sus operaciones y reducir la demanda de los recursos de la implementación.

Para obtener más información, vea [Abrir y cerrar una sesión winSNMP.](opening-and-closing-a-winsnmp-session.md)
