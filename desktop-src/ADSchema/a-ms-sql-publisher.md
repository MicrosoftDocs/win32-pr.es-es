# <a name="ms-sql-publisher-attribute"></a>Atributo MS-SQL-Publisher

El nombre de la base de datos del publicador para la replicación.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Publicador de MS-SQL                            |
| Nombre para mostrar de LDAP | Publicador de mS-SQL                            |
| Tamaño              | \-                                          |
| Actualizar privilegio  | Administrador de dominio                        |
| Frecuencia de actualización  | Cuando se configura la replicación.                  |
| Attribute-Id      | 1.2.840.113556.1.4.1402                     |
| System-ID-GUID    | c1676858-d34b-11d2-999a-0000f87a57d4        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
