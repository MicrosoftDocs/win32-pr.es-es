# <a name="id2d1devicecontext2createimagesourcefromwic-methods"></a>ID2D1DeviceContext2:: CreateImageSourceFromWic (métodos)

Crea un objeto de origen de imagen a partir de un origen de mapa de bits de WIC y rellena toda la memoria de píxeles dentro del origen de la imagen. La imagen se carga y se almacena al usar una cantidad mínima de memoria.
