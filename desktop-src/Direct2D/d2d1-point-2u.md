# <a name="d2d1_point_2u"></a>Punto de D2D1 \_ \_ 2U

Representa un par de coordenadas x e y en un espacio bidimensional.


```C++
typedef D2D_POINT_2U D2D1_POINT_2U;
```



## <a name="remarks"></a>Observaciones

Los puntos en Direct2D se representan mediante las estructuras [**D2D1 \_ punto \_ 2F**](d2d1-point-2f.md) o **D2D1 \_ punto \_ 2U** . Ambos contienen un par de coordenada x e y en un espacio bidimensional. La **estructura D2D1 \_ Point \_ 2F** almacena las coordenadas como valores **float** y la estructura **D2D1 \_ Point \_ 2U** las almacena como valores **UINT32** .

**D2D1 \_ El punto \_ 2U** es un nombre nuevo para el tipo ya definido [**D2D \_ punto \_ 2U**](/windows/desktop/api/dcommon/ns-dcommon-d2d_point_2u).
