# <a name="sessioninstaller-property"></a>Propiedad Session. Installer

La propiedad **instalador** de solo lectura del objeto de [**sesión**](session-object.md) devuelve el objeto **instalador** activo.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Session.Installer
```



## <a name="property-value"></a>Valor de propiedad
