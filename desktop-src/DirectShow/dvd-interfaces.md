# <a name="dvd-interfaces"></a>DVD Interfaces

Estas interfaces admiten la reproducción y navegación de DVD.



| Interfaz                                    | Descripción                                                                                            |
|----------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**IAMDecoderCaps**](/windows/desktop/api/Strmif/nn-strmif-iamdecodercaps)     | Implementado por descodificadores para indicar varias funcionalidades.                                              |
| [**IDvdCmd**](/windows/desktop/api/Strmif/nn-strmif-idvdcmd)                   | Bloquee el [filtro del navegador de DVD](dvd-navigator-filter.md) hasta que se inicie o complete un comando.        |
| [**IDvdControl2**](/windows/desktop/api/Strmif/nn-strmif-idvdcontrol2)         | Navegue y reprodgue DVD-Video títulos, incluidos los títulos de title.                                          |
| [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) | Cree un gráfico de filtros para DVD-Video reproducción desde el software y el hardware disponibles en el sistema.    |
| [**IDvdInfo2**](/windows/desktop/api/Strmif/nn-strmif-idvdinfo2)               | Consulta de varios atributos de un disco de DVD o del estado del navegador de DVD.                               |
| [**IDvdState**](/windows/desktop/api/Strmif/nn-strmif-idvdstate)               | Guarde el estado de la sesión del usuario en el disco, incluida la ubicación de reproducción, el nivel parental y la región. |
