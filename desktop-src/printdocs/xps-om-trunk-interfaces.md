# <a name="xps-om-trunk-interfaces"></a>Interfaces troncales de XPS OM

Las interfaces de nivel de tronco de la API de documentos XPS describen la estructura de un documento XPS a medida que el modelo de objetos XPS lo representa.

<dl>

[Interfaces de paquetes de XPS OM](xps-object-model-package-interfaces.md) Las interfaces de paquete describen las características y las operaciones que actúan en el nivel superior de un OM de XPS.  
[Interfaces de documento XPS OM](xps-object-model-document-interfaces.md) Las interfaces de documento describen la organización y la secuencia de FixedDocuments del paquete y los documentos de FixedPages.  
[Interfaces de elementos de XPS OM](working-with-xps-object-model-part-interfaces.md) Las interfaces de elementos describen los recursos utilizados por el documento y sus componentes.  
</dl>
