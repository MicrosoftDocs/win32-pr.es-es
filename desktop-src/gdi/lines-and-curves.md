# <a name="lines-and-curves"></a>Líneas y curvas

Las líneas y curvas se usan para dibujar la salida de gráficos en dispositivos de trama. Como se describe en  esta información general, una línea es un conjunto de píxeles resaltados en una pantalla de trama (o un conjunto de puntos en una página impresa) identificados por dos puntos: un punto inicial y un punto final. Una *curva normal* es un conjunto de píxeles resaltados en una pantalla de trama (o puntos en una página impresa) que define el perímetro (o parte del perímetro) de una sección cónica. Una *curva irregular* es un conjunto de píxeles que define una curva que no se ajusta al perímetro de una sección cónica.
