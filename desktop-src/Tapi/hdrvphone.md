# <a name="hdrvphone"></a>HDRVPHONE

El tipo de datos **HDRVPHONE** representa el identificador opaco de un proveedor de servicios para una estructura de datos de teléfono. El proveedor de servicios debe resolver un valor de este tipo en una referencia a la instancia de la estructura de datos adecuada.
