# <a name="dvd-graph-builder"></a>DVD Graph Builder

DVD Graph Builder es un objeto auxiliar para crear gráficos de reproducción de DVD. Las aplicaciones pueden usar este componente para crear gráficos de filtro para la reproducción y navegación de DVD. Cree este objeto mediante una llamada **a CoCreateInstance**.



| Etiqueta | Valor |
|------------------|----------------------------------------------|
| Identificador de clase | CLSID \_ DvdGraphBuilder                       |
| Interfaces       | [**IDvdGraphBuilder**](/windows/desktop/api/Strmif/nn-strmif-idvdgraphbuilder) |
