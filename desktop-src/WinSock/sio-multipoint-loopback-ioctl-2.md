# <a name="sio_multipoint_loopback-ioctl"></a>SIO \_ MULTIPOINT \_ LOOPBACK Ioctl

Cuando se usan sockets hoja d en un plano de datos no raíz, generalmente es conveniente poder controlar si el tráfico enviado también se recibe de nuevo en el \_ mismo socket. El código de comando \_ SIO MULTIPOINT LOOPBACK para \_ [**WSPIoctl**](/previous-versions/windows/hardware/network/ff566296(v=vs.85)) se usa para habilitar o deshabilitar el bucleback del tráfico multipunto.
