# <a name="winnt-user-object"></a>Objeto de usuario de Winnt

El objeto de usuario WinNT representa una cuenta de usuario en un dominio de Windows NT 4,0. El objeto exhibe características especiales. En una instancia de, no es compatible con todos los métodos de propiedad de la interfaz [**IADsUser**](/windows/desktop/api/Iads/nn-iads-iadsuser) . En una segunda instancia, admite algunas propiedades personalizadas a las que solo se puede tener acceso con el método [**IADs. Get**](/windows/desktop/api/Iads/nf-iads-iads-get) o [**IADs. put**](/windows/desktop/api/Iads/nf-iads-iads-put) .

Para obtener más información acerca de los objetos de usuario de WinNT, vea:

-   [Propiedades de IADsUser no admitidas](unsupported-iadsuser-properties.md)
-   [Propiedades de usuario personalizadas de Winnt](winnt-custom-user-properties.md)
-   [Ejemplos de administración de cuentas de usuario de Winnt](winnt-user-account-management-examples.md)
