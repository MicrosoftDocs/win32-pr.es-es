# <a name="isearchresult-properties"></a>Propiedades de ISearchResult

La interfaz [**ISearchResult**](/windows/desktop/api/Wuapi/nn-wuapi-isearchresult) define las siguientes propiedades.



| Propiedad                                               | Descripción                                                                                                   |
|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**ResultCode**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_resultcode)         | Obtiene una enumeración [**OperationResultCode**](/windows/win32/api/wuapi/ne-wuapi-operationresultcode) que especifica el resultado de una búsqueda. |
| [**RootCategories**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_rootcategories) | Obtiene una colección de interfaces de las categorías raíz que están actualmente disponibles en el equipo.             |
| [**Actualizaciones**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_updates)               | Obtiene una colección de interfaces de las actualizaciones que son el resultado de una búsqueda.                                        |
| [**Advertencias**](/windows/desktop/api/Wuapi/nf-wuapi-isearchresult-get_warnings)             | Obtiene una colección de las advertencias que son el resultado de una búsqueda.                                                  |
