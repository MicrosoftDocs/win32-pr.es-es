# <a name="use-the-active-voice"></a>Uso de la voz activa

\[Microsoft Agent está en desuso a partir de Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

Al usar la salida de voz para proporcionar información de directiva o para generar una respuesta del usuario, use la voz activa y especifique claramente la acción esperada del usuario. En el ejemplo siguiente se muestran las diferencias:



| Directiva                            |  Evaluación                   |
|--------------------------------------|-------------------------------|
| Voy a repetir el número.           | Ninguna acción del usuario                |
| El número se repetirá.         | Voz pasiva, sin acción del usuario |
| Escuche mientras se repite el número. | Voz pasiva                 |
| Escuche la repetición.            | Mejor opción                   |



 

Además, cree la salida para mostrar la información clave al final de la frase, como se muestra en los ejemplos siguientes:



| En vez de esto...                                                 | Use                                           |
|--------------------------------------------------|--------------------------------------------|
| "¿Es tres el dígito siguiente?"           | "¿Es el siguiente dígito tres?"            |
| "Haga clic en Aceptar para comenzar".                 | "Para empezar, haga clic en Aceptar".                 |
| "Diga "Listo" para completar el pedido". | "Para completar el pedido, diga "Listo"". |
