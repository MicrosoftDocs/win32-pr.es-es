# <a name="the-sensor-collection-object"></a>El objeto De colección de sensores

El objeto de colección de sensores contiene un grupo de objetos de sensor.

A veces querrá trabajar con sensores como grupo. Por ejemplo, puede usar el administrador de sensores para detectar todos los sensores de movimiento disponibles. El objeto de colección de sensores proporciona métodos para trabajar con un grupo de objetos de sensor. Puede usar la interfaz [**ISensorCollection para**](/windows/desktop/api/sensorsapi/nn-sensorsapi-isensorcollection) llamar a métodos que cuentan, agregan, quitan y recuperan sensores de la colección.
