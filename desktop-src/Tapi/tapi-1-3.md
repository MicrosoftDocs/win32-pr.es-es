# <a name="tapi-13"></a>TAPI 1,3

TAPI 1,3 (que a veces se conoce como TAPI 1,03) era la primera versión de lanzamiento de la especificación de TAPI. Los archivos binarios del sistema TAPI 1,3 (Tapi.dll y otros) se encuentran en el SDK de TAPI 1,0, que solo se puede encontrar en la suscripción de Microsoft Developer Network (MSDN) Level II. Los archivos binarios del sistema (que se encuentran en el directorio Redist) son redistribuibles, pero están diseñados para ejecutarse solo en Microsoft Windows 3,1 y Windows para grupos de trabajo 3,11. No se admiten y no funcionarán en ninguna versión de Windows de 32 bits.

Los encabezados y las bibliotecas que se usan para desarrollar una aplicación TAPI 1,3 solo están disponibles en el SDK de TAPI 1,0 y solo admiten el desarrollo de 16 bits. De hecho, las bibliotecas de TAPI 1,3 son las únicas bibliotecas TAPI de 16 bits disponibles y deben usarse para todas las aplicaciones de 16 bits, independientemente de la plataforma de destino o la versión de destino.

Los archivos binarios del sistema TAPI 1,3 solo admiten Win16, TAPI 1,3 profesionales y aplicaciones.
