# <a name="retrieving-amended-classes"></a>Recuperar clases modificadas

Puede recuperar y modificar una clase modificada especificando una configuración regional y estableciendo una marca que indique que desea información localizada. El procedimiento usado para recuperar definiciones de clase localizadas depende de cómo se conecte a WMI. Para obtener más información, consulte [recuperación de clases modificadas mediante la API de scripting para WMI](retrieving-amended-classes-using-the-scripting-api-for-wmi.md) y [recuperación de clases modificadas mediante la API com para WMI](retrieving-amended-classes-using-the-com-api-for-wmi.md).
