# <a name="texture2darraygatherblue-methods"></a>Texture2DArray:: GatherBlue (métodos)

Devuelve los componentes azules de los cuatro valores de textura de un [**Texture2DArray**](sm5-object-texture2darray.md)que se utilizarían en una operación de filtrado bilineal.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                                        | Descripción                                                                                                                                      |
|:--------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GatherBlue (S, Float, int)**](sm5-object-texture2darray-gatherblue.md)                                        | Devuelve los componentes azules de los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal.<br/>                                 |
| [**GatherBlue (S, Float, int, uint)**](t2darray-gatherblue-s-float-int-uint-.md)                                  | Devuelve los componentes azules de los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal, junto con el estado de asignación de mosaicos.<br/> |
| [**GatherBlue (S, Float, INT2, INT2, INT2, INT2)**](t2array-gatherblue-s-float-int2-int2-int2-int2-.md)             | Devuelve los componentes azules de los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal.<br/>                                 |
| [**GatherBlue (S, Float, INT2, INT2, INT2, INT2, uint)**](t2darray-gatherblue-s-float-int2-int2-int2-int2-uint-.md)  | Devuelve los componentes azules de los cuatro valores de textura que se utilizarían en una operación de filtrado bilineal, junto con el estado de asignación de mosaicos.<br/> |
