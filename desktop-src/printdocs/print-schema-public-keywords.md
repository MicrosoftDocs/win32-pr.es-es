# <a name="print-schema-public-keywords"></a>Palabras clave públicas de esquema de impresión

Este tema no está actualizado. Para obtener la información más reciente, consulte la [especificación del esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

En la sección siguiente se especifican las palabras clave públicas que se definen para el esquema de impresión.

## <a name="print-schema-public-keywords-usage-for-print-capabilities"></a>Uso de palabras clave públicas de esquema de impresión para funcionalidades de impresión

Las palabras clave especificadas en las palabras clave públicas de PrintCapabilities pueden aparecer en un documento de capacidades de impresión. Para obtener más información sobre cómo interactúan estas palabras clave con la tecnología de PrintCapabilities, consulte [información general de la sección PrintCapabilities](overview-of-the-printcapabilities.md) .

Las palabras clave públicas específicas de PrintTicket no deben aparecer en un documento de PrintCapabilities.

## <a name="print-schema-public-keywords-usage-for-printticket"></a>Imprimir el uso de palabras clave públicas de esquema para PrintTicket

Las palabras clave especificadas en las palabras clave públicas de PrintTicket pueden aparecer en un PrintTicket. Las palabras clave de PrintTicket se implementan como un tipo de elemento de propiedad. Para obtener más información sobre cómo estas palabras clave interactúan con la tecnología de PrintTicket, consulte la sección [información de configuración no relacionada con PrintCapabilities](configuration-information-not-related-to-printcapabilities.md) .

Las palabras clave especificadas en las palabras clave públicas de PrintCapabilities pueden aparecer en un PrintTicket dependiendo de la implementación del PrintTicket en una aplicación o un controlador. Esto proporciona selecciones para los atributos de dispositivo definidos en el documento de PrintCapabilities. Para obtener más información sobre la interacción entre las palabras clave de PrintCapabilities y el PrintTicket, consulte [el propósito de la sección PrintTicket](purpose-of-the-printticket.md) .
