# <a name="modify-time-stamp-attribute"></a>Atributo Modify-Time-Stamp

Atributo calculado que representa la fecha en que se cambió por última vez este objeto. Este valor no se replica.



| Entrada | Value |
|-------------------|---------------------------------------------------------------|
| CN                | Modificar: marca de tiempo                                             |
| Nombre para mostrar de LDAP | modifyTimeStamp                                               |
| Tamaño              | 8 bytes                                                       |
| Actualizar privilegio  | El sistema establece este valor.                              |
| Frecuencia de actualización  | \-                                                            |
| Attribute-Id      | 2.5.18.2                                                      |
| System-ID-GUID    | 9a7ad94a-ca53-11d1-bbd0-0080c76670c0                          |
| Sintaxis            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
