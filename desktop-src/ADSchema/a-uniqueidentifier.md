# <a name="uniqueidentifier-attribute"></a>uniqueIdentifier (atributo)

El tipo de atributo uniqueIdentifier especifica un "identificador único" para un objeto representado en el directorio.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | uniqueIdentifier                            |
| Nombre para mostrar de LDAP | uniqueIdentifier                            |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 0.9.2342.19200300.100.1.44                  |
| System-ID-GUID    | ba0184c7-38c5-4bed-a526-75421470580c        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
