# <a name="dsp-plug-in-enumeration-types"></a>Tipos de enumeración de complementos DSP

El SDK de Media Player de Windows implementa el siguiente tipo de enumeración para crear complementos de DSP.



| Tipo de enumeración                                        | Descripción                                                                                    |
|---------------------------------------------------------|------------------------------------------------------------------------------------------------|
| [\_Cap WMPPlugin](/previous-versions/windows/desktop/api/wmpservices/ne-wmpservices-wmpplugin_caps)                   | Se usa con **IWMPPlugin:: GetCaps** para indicar si el complemento puede convertir entre formatos. |
| [WMPServices \_ StreamState](/previous-versions/windows/desktop/api/wmpservices/ne-wmpservices-wmpservices_streamstate) | Indica si la secuencia está actualmente detenida, en pausa o en reproducción.                     |
