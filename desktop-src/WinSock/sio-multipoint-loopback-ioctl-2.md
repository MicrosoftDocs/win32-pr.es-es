# <a name="sio_multipoint_loopback-ioctl"></a>\_Ioctl de \_ bucle invertido de SiO

Cuando se \_ usan Sockets hoja d en un plano de datos no raíz, por lo general es conveniente poder controlar si el tráfico enviado también se recibe en el mismo Socket. El \_ \_ código de comando SiO Multipoint loopback para [**WSPIoctl**](/previous-versions/windows/hardware/network/ff566296(v=vs.85)) se usa para habilitar o deshabilitar el bucle invertido de tráfico multipoint.
