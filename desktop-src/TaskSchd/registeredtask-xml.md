# <a name="registeredtaskxml-property"></a>Propiedad RegisteredTask.XML

En el caso de scripting, obtiene la información de registro con formato XML para la tarea registrada.

## <a name="syntax"></a>Sintaxis


```VB
RegisteredTask.XML As String
```



## <a name="property-value"></a>Valor de propiedad

Información de registro con formato XML para la tarea registrada.
