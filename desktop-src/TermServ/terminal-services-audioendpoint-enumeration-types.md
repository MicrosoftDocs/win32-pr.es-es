# <a name="remote-desktop-services-audioendpoint-enumeration-types"></a>Servicios de Escritorio remoto enumeración AudioEndpoint

Los siguientes tipos de enumeración se usan con audioEndpoint API.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**MARCAS DE \_ POSICIÓN DE AE \_**](/windows/desktop/api/Audioengineendpoint/ne-audioengineendpoint-ae_position_flags)
</dt> <dd>

Define constantes para la [**estructura \_ AE CURRENT \_ POSITION.**](/windows/desktop/api/Audioengineendpoint/ns-audioengineendpoint-ae_current_position) Estas constantes describen el grado de validez de la posición actual.

</dd> <dt>

[**MARCAS \_ DE BÚFER DE APO \_**](/windows/desktop/api/Audioapotypes/ne-audioapotypes-apo_buffer_flags)
</dt> <dd>

Define las marcas de validación del búfer para la [**estructura \_ APO CONNECTION \_ PROPERTY**](/windows/desktop/api/Audioapotypes/ns-audioapotypes-apo_connection_property) asociada a cada conexión APO.

</dd> </dl>

## <a name="remarks"></a>Comentarios

La SERVICIOS DE ESCRITORIO REMOTO AudioEndpoint es para su uso en Escritorio remoto escenarios; no es para las aplicaciones cliente.
