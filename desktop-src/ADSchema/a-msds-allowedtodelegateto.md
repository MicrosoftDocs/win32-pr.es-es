# <a name="ms-ds-allowed-to-delegate-to-attribute"></a>atributo MS-DS-allowed-to-Delegate

Se trata de un atributo de objetos de cuenta de servicio (equipo o cuenta de usuario). Contiene una lista de nombres de entidad de seguridad de servicio (SPN). Este atributo se utiliza para configurar un servicio de modo que pueda obtener vales de servicio que se pueden usar para la delegación restringida.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-DS-permitido a delegado a                |
| Nombre para mostrar de LDAP | msDS-AllowedToDelegateTo                    |
| Tamaño              | de 0 a 64 k                                    |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | Con poca frecuencia                                |
| Attribute-Id      | 1.2.840.113556.1.4.1787                     |
| System-ID-GUID    | 800d94d7-b7a1-42a1-b14d-7cae1423d07f        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
