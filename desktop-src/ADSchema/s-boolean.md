# <a name="boolean-syntax"></a>Sintaxis booleana

Representa un valor booleano.



| Entrada | Valor |
|--------------|--------------------------------------------------------------------------|
| Nombre         | Boolean                                                                  |
| Identificador de sintaxis    | 2.5.5.8                                                                  |
| Id. de OM        | 1                                                                        |
| Tipo DE ARCHIVO    | BOOLEAN                                                                  |
| ADS Type     | ADSTYPE \_ BOOLEAN                                                         |
| Tipo de variante | VT \_ BOOL                                                                 |
| Tipo de SDS     | [System.Boolean](/dotnet/api/system.boolean) |



## <a name="remarks"></a>Comentarios

Para las consultas que incluyen atributos de sintaxis booleana en un filtro, especifique "TRUE" o "FALSE" (por ejemplo, "myboolattr=TRUE").
