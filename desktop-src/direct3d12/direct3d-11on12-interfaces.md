# <a name="11on12-interfaces"></a>Interfaces 11on12
Las siguientes interfaces se declaran en d3d11on12. h.

## <a name="in-this-section"></a>En esta sección
| Tema | Descripción |
|-|-|
| [**ID3D11On12Device**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device) | Controla la creación, el ajuste y la liberación de los recursos de D3D11 para Direct3D 11on12. |
| [**ID3D11On12Device1**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device1) | Permite una mejor interoperabilidad con un componente que se puede entregar a un dispositivo Direct3D 11, pero que quiere aprovechar Direct3D 12 en su lugar. |
| [**ID3D11On12Device2**](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device2) | Permite tomar recursos creados a través de las API de Direct3D 11 y usarlos en Direct3D 12. Esta interfaz extiende [ID3D11On12Device1](/windows/win32/api/d3d11on12/nn-d3d11on12-id3d11on12device1). |
