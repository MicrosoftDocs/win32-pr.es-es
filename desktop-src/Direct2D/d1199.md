# <a name="d1199-changed-command-list-resource"></a>D1199: recurso de lista de comandos cambiado

Se acaba de cambiar una imagen dibujada en una lista de comandos. Esto puede producir consecuencias imprevistas. La lista de comandos contendrá ahora la imagen modificada en lugar del contenido de la imagen en el momento en que se dibujó.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación agregó una imagen a una lista de comandos y después cambió la imagen.

## <a name="possible-fixes"></a>Posibles correcciones

Compruebe que el comportamiento resultante es el esperado. El comportamiento en este caso, donde una lista de comandos es el destino, generará resultados diferentes a los que se configuraron como destino un mapa de bits. Si el comportamiento no es el esperado, es posible que la aplicación desee clonar la imagen antes de actualizarla.
