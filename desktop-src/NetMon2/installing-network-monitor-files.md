# <a name="installing-network-monitor-files"></a>Instalación de Monitor de red Files

Para instalar y ejecutar la parte Monitor de red del Kit de desarrollo de software (SDK) de plataforma, siga este procedimiento.

**Para instalar y ejecutar archivos Monitor de red PLATFORM SDK**

1.  Para instalar los archivos, durante la instalación del SDK de plataforma, seleccione los Monitor de red y el código de ejemplo.

    Una vez completada la instalación, el programa de instalación del SDK de plataforma crea un archivo por lotes, Setenv.bat.

2.  Para inicializar las variables de entorno necesarias, ejecute Setenv.bat.
3.  Para compilar los ejemplos, ejecute **nmake**, que se incluye en Microsoft Visual Studio.
