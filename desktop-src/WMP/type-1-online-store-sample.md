# <a name="type-1-online-store-sample"></a>Ejemplo de tienda en línea de tipo 1

El SDK Reproductor de Windows Media incluye una tienda en línea de ejemplo de tipo 1 que es compatible con Reproductor de Windows Media 11.

Para ver el ejemplo de almacén en línea de tipo 1, configure el registro como se describe en Claves y entradas del Registro para un almacén [en línea de tipo 1.](registry-keys-and-entries-for-a-type-1-online-store.md) La clave del almacén de tipo de ejemplo 1 es 2573.
