# <a name="saving-and-loading-topologies"></a>Guardar y cargar topologías

Con TopoEdit, puede guardar una topología como archivo XML y cargarla más adelante. Cuando se carga una topología completa, se debe volver a resolver. Para obtener información sobre este procedimiento, vea [resolver una topología con TopoEdit](resolving-a-topology-with-topoedit.md).

## <a name="to-save-a-topology"></a>Para guardar una topología

1.  En el menú **Archivo**, haga clic en **Guardar**.

    Se abrirá el cuadro de diálogo **Guardar archivo** .

2.  Navegue hasta la carpeta en la que se debe guardar el archivo de topología.

3.  En el campo **nombre de archivo:** , escriba el nombre del archivo.

4.  Haga clic en **Save**(Guardar).

El archivo de topología se guardará como un archivo XML.

## <a name="to-load-a-saved-topology"></a>Para cargar una topología guardada

1.  En el menú **Archivo**, haga clic en **Abrir**.

    Se abrirá el cuadro de diálogo **Abrir archivo** .

2.  Navegue hasta la carpeta en la que se encuentra el archivo XML de la topología.

3.  Especifique el nombre de archivo en el campo **nombre de archivo:** o seleccione el archivo en la lista.

4.  Haga clic en **Abrir**.

El archivo de topología se carga en el **Panel de topología**.
