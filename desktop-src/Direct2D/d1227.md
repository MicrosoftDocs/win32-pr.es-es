# <a name="d1227-print-rasterization-event"></a>D1227: evento de rasterización de impresión

Este mensaje se emite desde la impresión de Direct2D. Indica que una de las siguientes operaciones provocó una necesidad de rasterización:

-   Uso de una \_ capa D2D1 \_ OPTIONS1 \_ inicializar \_ desde el \_ nivel de fondo.
-   Uso de la copia de origen del modo de mezcla primitivo o cualquier otro modo de mezcla que no sea el origen sobre.
-   Uso de determinados modos compuestos, como la copia de origen ENLAZAda \_ \_ , el destino de \_ salida, el origen \_ encima, el signo más o la inversión.
-   Uso de [**DrawGdiMetafile**](id2d1devicecontext-drawgdimetafile-overload.md).
-   Uso de una transformación 4x4 (o perspectiva) en una llamada a [**DrawBitmap**](id2d1devicecontext-drawbitmap-overload.md) .

Para obtener un rendimiento óptimo, estas operaciones se deben evitar si es posible.
