# <a name="musical-instrument-digital-interface-midi"></a>Interfaz digital de instrumentos digitales (MIDI)

La interfaz digital de instrumentos musicales (MIDI) es un protocolo y un conjunto de comandos para almacenar y transmitir información sobre música. Los dispositivos de salida MIDI interpretan esta información y la usan para sintetizar música.

-   [Acerca de MIDI](about-midi.md)
-   [Uso de MIDI](using-midi.md)
-   [Referencia MIDI](midi-reference.md)
