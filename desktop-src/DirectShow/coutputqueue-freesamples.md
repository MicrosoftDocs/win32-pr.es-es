# <a name="coutputqueuefreesamples-method"></a>COutputQueue. FreeSamples, método

El `FreeSamples` método libera todas las muestras pendientes.

## <a name="syntax"></a>Sintaxis


```C++
void FreeSamples();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.

## <a name="remarks"></a>Observaciones

Este método quita cualquier ejemplo pendiente de la cola y de la matriz de ejemplo.
