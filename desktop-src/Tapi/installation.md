# <a name="installation-telephony-api"></a>Instalación (API de telefonía)

Los orígenes principales de los procedimientos de instalación son el Kit de recursos para el sistema operativo de destino, la documentación de la aplicación y las instrucciones del proveedor de servicios. Para los proveedores de servicios de Microsoft, el kit de recursos adecuado contiene instrucciones.

El proceso de instalación debe incluir la enumeración de "Servicio de telefonía" como dependencia.

Si una aplicación o proveedor de servicios usa el registro para el almacenamiento de datos privados persistentes, la información no debe colocarse en la sección de TAPI del registro. No se garantiza que el formato de esta sección se mantenga en versiones futuras.
