# <a name="terminal"></a>Terminal

Un terminal es un origen o un representador para una dirección y un tipo de medio concretos. El concepto de terminal se utiliza principalmente en TAPI 3 y solo es levemente importante para las aplicaciones TAPI 2. Para obtener más información, consulte la introducción a los [objetos de terminal](terminal-object.md) de TAPI 3.
