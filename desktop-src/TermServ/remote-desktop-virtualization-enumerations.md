# <a name="remote-desktop-virtualization-enumerations"></a>Enumeraciones de virtualización de Escritorio remoto

La API de virtualización de Escritorio remoto admite los siguientes tipos de enumeración.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**\_tipo de cambio de destino \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_change_type)
</dt> <dd>

Especifica el tipo de cambio que se produjo en un destino.

</dd> <dt>

[**Estado de destino \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_state)
</dt> <dd>

Indica el estado de un destino.

</dd> <dt>

[**tipo de destino \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-target_type)
</dt> <dd>

Indica si un destino pertenece a un grupo o conjunto de servidores.

</dd> <dt>

[**Estado de TSSESSION \_**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-tssession_state)
</dt> <dd>

Indica el estado de una sesión.

</dd> <dt>

[**TSSD \_ AddrV46Type**](/windows/desktop/api/SessDirPublicTypes/ne-sessdirpublictypes-tssd_addrv46type)
</dt> <dd>

Indica el tipo de dirección IP.

</dd> <dt>

[**\_SB \_ de TS ordenar \_ por**](/windows/desktop/api/sbtsv/ne-sbtsv-ts_sb_sort_by)
</dt> <dd>

Especifica el criterio de ordenación. Se utiliza como parámetro en el método [**EnumerateTargets**](/windows/desktop/api/sbtsv/nf-sbtsv-itssbresourcepluginstore-enumeratetargets) .

</dd> </dl>
