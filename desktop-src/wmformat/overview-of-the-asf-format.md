# <a name="overview-of-the-asf-format"></a><span data-ttu-id="94321-106">Información general del formato ASF</span><span class="sxs-lookup"><span data-stu-id="94321-106">Overview of the ASF Format</span></span>

<span data-ttu-id="94321-107">El formato de sistemas avanzados (ASF) es un formato de archivo extensible diseñado principalmente para almacenar y reproducir secuencias multimedia digitales sincronizadas y transmitirlas a través de redes.</span><span class="sxs-lookup"><span data-stu-id="94321-107">The Advanced Systems Format (ASF) is an extensible file format designed primarily for storing and playing synchronized digital media streams and transmitting them over networks.</span></span> <span data-ttu-id="94321-108">ASF es el formato de contenedor para Windows Media Audio y el contenido basado en Windows Media Video.</span><span class="sxs-lookup"><span data-stu-id="94321-108">ASF is the container format for Windows Media Audio and Windows Media Video-based content.</span></span> <span data-ttu-id="94321-109">La extensión WMA o WMV se usa para especificar un archivo ASF que contiene contenido codificado con los códecs de Windows Media Audio y/o Windows Media Video.</span><span class="sxs-lookup"><span data-stu-id="94321-109">The extension wma or wmv is used to specify an ASF file that contains content encoded with the Windows Media Audio and/or Windows Media Video codecs.</span></span> <span data-ttu-id="94321-110">El SDK de Windows Media Format se puede usar para crear y leer archivos de Windows Media, así como archivos ASF que contienen otros tipos de datos comprimidos o sin comprimir.</span><span class="sxs-lookup"><span data-stu-id="94321-110">The Windows Media Format SDK can be used to create and read Windows Media files, as well as ASF files that contain other types of compressed or uncompressed data.</span></span>

<span data-ttu-id="94321-111">En esta sección se proporciona una descripción general del formato ASF como información general.</span><span class="sxs-lookup"><span data-stu-id="94321-111">This section provides a general description of the ASF format as background information.</span></span> <span data-ttu-id="94321-112">Dado que los objetos de lectura y escritura controlan todas las tareas de formato y análisis de archivos de bajo nivel, no es necesario tener un conocimiento detallado de ASF antes de usar este SDK para crear archivos ASF.</span><span class="sxs-lookup"><span data-stu-id="94321-112">Because the reader and writer objects handle all low-level file parsing and formatting tasks, it is not necessary to have a detailed understanding of ASF before using this SDK to create ASF files.</span></span> <span data-ttu-id="94321-113">La especificación ASF completa se puede encontrar en el [sitio web de Microsoft](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).</span><span class="sxs-lookup"><span data-stu-id="94321-113">The complete ASF specification can be found on the [Microsoft Web site](https://download.microsoft.com/download/7/9/0/790fecaa-f64a-4a5e-a430-0bccdab3f1b4/ASF_Specification.doc).</span></span>

<span data-ttu-id="94321-114">Los principales objetivos del formato ASF son:</span><span class="sxs-lookup"><span data-stu-id="94321-114">The primary goals of the ASF format are:</span></span>

-   <span data-ttu-id="94321-115">Para admitir la reproducción eficaz desde servidores multimedia, servidores HTTP y dispositivos de almacenamiento local.</span><span class="sxs-lookup"><span data-stu-id="94321-115">To support efficient playback from media servers, HTTP servers, and local storage devices.</span></span>
-   <span data-ttu-id="94321-116">Para admitir tipos de medios escalables como audio y vídeo.</span><span class="sxs-lookup"><span data-stu-id="94321-116">To support scalable media types such as audio and video.</span></span>
-   <span data-ttu-id="94321-117">Para permitir que se presente una única composición multimedia a través de una amplia gama de anchos de banda.</span><span class="sxs-lookup"><span data-stu-id="94321-117">To permit a single multimedia composition to be presented over a wide range of bandwidths.</span></span>
-   <span data-ttu-id="94321-118">Permite el control de la creación de relaciones de flujo multimedia, especialmente en escenarios de ancho de banda restringido.</span><span class="sxs-lookup"><span data-stu-id="94321-118">To allow authoring control over media stream relationships, especially in constrained-bandwidth scenarios.</span></span>
-   <span data-ttu-id="94321-119">Ser independiente de cualquier sistema de composición multimedia, sistema operativo de equipo o protocolo de comunicaciones de datos determinado.</span><span class="sxs-lookup"><span data-stu-id="94321-119">To be independent of any particular multimedia composition system, computer operating system, or data communications protocol.</span></span>

<span data-ttu-id="94321-120">Un archivo ASF puede contener varios flujos independientes o dependientes, como varias secuencias de audio para audio multicanal o secuencias de vídeo de varias velocidades de bits adecuadas para su transmisión a través de diferentes anchos de banda.</span><span class="sxs-lookup"><span data-stu-id="94321-120">An ASF file can contain multiple independent or dependent streams, including multiple audio streams for multichannel audio, or multiple bit rate video streams suitable for transmission over different bandwidths.</span></span> <span data-ttu-id="94321-121">Los flujos pueden estar en cualquier formato comprimido o sin comprimir. sin embargo, la mejor compresión se logra con los códecs de Microsoft Windows Media Audio y de la serie de vídeo 9.</span><span class="sxs-lookup"><span data-stu-id="94321-121">The streams can be in any compressed or uncompressed format; <span data-ttu-id="94321-125">Un archivo ASF se organiza en secciones denominadas "objetos".</span><span class="sxs-lookup"><span data-stu-id="94321-125">An ASF file is organized into sections called "objects."</span></span> <span data-ttu-id="94321-126">Hay tres objetos de nivel superior, un objeto de encabezado y un objeto de datos (ambos obligatorios), además de un objeto de índice data-stu-id="94321-128">Metadata is also stored here.</span></span> <span data-ttu-id="94321-129">El objeto de encabezado es el único objeto de nivel superior que puede contener otros <span data-ttu-id="94321-133">Cada objeto de nivel superior o de nivel inferior comienza con un identificador único global (GUID) y un valor de tamaño.</span><span class="sxs-lookup"><span data-stu-id="94321-133">Each top-level or lower-level object begins with a globally unique identifier (GUID) and a size value.</span></span> <span data-ttu-id="94321-134">Estos números permiten que el lector de archivos analice la información en los lugares adecuados en objetos identificables.</span><span class="sxs-lookup"><span data-stu-id="94321-134">These numbers allow the file reader to parse the information at appropriate places into identifiable objects.</span></span> <span data-ttu-id="94321-135">Debido a estos GUID, se pueden enviar objetos de nivel inferior en cualquier orden y seguir siendo reconocidos.</span><span class="sxs-lookup"><span data-stu-id="94321-135">Because of these GUIDs, lower-level objects can be sent in any order and still be recognized.</span></span> <span data-ttu-id="94321-136">El formato ASF está diseñado para evitar la recepción de datos inexactos.</span><span class="sxs-lookup"><span data-stu-id="94321-136">The ASF format is designed to overcome inaccurate data reception.</span></span> <span data-ttu-id="94321-137">Todavía se puede leer un archivo ASF descargado parcialmente, siempre que contenga el objeto de encabezado y al menos un objeto de datos.</span><span class="sxs-lookup"><span data-stu-id="94321-137">A partially downloaded ASF file can still be read, as long as it contains the Header object and at least one Data object.</span></span>
