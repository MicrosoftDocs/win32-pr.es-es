# <a name="lineremovefromconf_-constants"></a>LINEREMOVEFROMCONF \_ (Constantes)

Las **constantes escalares \_ LINEREMOVEFROMCONF** describen cómo las partes que participan en una llamada de conferencia se pueden quitar de una llamada de conferencia.

<dl> <dt>

<span id="LINEREMOVEFROMCONF_ANY"></span><span id="lineremovefromconf_any"></span>**LINEREMOVEFROMCONF \_ ANY**
</dt> <dd> <dl> <dt>



Cualquier parte participante se puede quitar de la llamada de conferencia.


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_LAST"></span><span id="lineremovefromconf_last"></span>**LINEREMOVEFROMCONF \_ LAST**
</dt> <dd> <dl> <dt>



Solo la parte agregada más recientemente se puede quitar de la llamada de conferencia


</dt> </dl> </dd> <dt>

<span id="LINEREMOVEFROMCONF_NONE"></span><span id="lineremovefromconf_none"></span>**LINEREMOVEFROMCONF \_ NONE**
</dt> <dd> <dl> <dt>



Las partes no se pueden quitar de la llamada de conferencia.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Comentarios

Sin extensibilidad. Los 32 bits están reservados.
