# <a name="uniscribe"></a>Uniscribe

Uniscribe es un conjunto de API que permiten un alto grado de control para la tipografía precisa y para procesar scripts complejos. Tanto los scripts complejos como los sencillos que tienen efectos tipográficos finos requieren un procesamiento especial para mostrarlos y editarlos, ya que los caracteres ("glifos") no están dispuestos de forma sencilla. En el caso de los scripts complejos, las reglas que rigen la forma y el posicionamiento de los glifos se especifican y catalogan en [el estándar Unicode](https://www.unicode.org/).

En este tema se describen diversos aspectos de la tipografía precisa y el procesamiento de scripts complejos, y se presenta de la siguiente manera:

-   [Acerca de Uniscribe](about-uniscribe.md)
-   [Usar Uniscribe](using-uniscribe.md)
-   [Referencia de Uniscribe](uniscribe-reference.md)
