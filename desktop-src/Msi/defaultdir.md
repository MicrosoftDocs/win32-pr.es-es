# <a name="defaultdir"></a>DefaultDir

El tipo de datos DefaultDir es una cadena de texto que contiene un [nombre de archivo](filename.md) válido o un [identificador](identifier.md)válido. Solo se usa en la [tabla de directorios](directory-table.md). Debe ser un identificador si el directorio es un directorio raíz. Si el directorio no es una raíz, este valor debe ser un nombre de archivo o un par de nombre de archivo: nombre de archivo. Tenga en cuenta que se permite "." como nombre de archivo y tiene un significado especial en la tabla de directorios.
