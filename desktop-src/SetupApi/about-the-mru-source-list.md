# <a name="about-the-mru-source-list"></a>Acerca de la lista de orígenes de MRU

La lista de orígenes usada más recientemente (MRU) sigue estando en el sistema del usuario. Las funciones de instalación controlan internamente la creación de nuevas listas de origen y su ubicación en el equipo del usuario. Las funciones de instalación usan esta lista para almacenar información sobre las rutas de acceso de origen usadas en instalaciones anteriores.

Puede usar esta información al solicitar al usuario una ruta de acceso de origen. Por ejemplo, podría crear una lista desplegable de las conexiones de red usadas como rutas de acceso de origen en instalaciones anteriores.

En función de los permisos, puede crear una lista de usuarios, una que sea específica de un usuario determinado o una lista del sistema, que sea la misma para todos los usuarios. Además de las listas de orígenes de usuario y del sistema, puede crear una lista de origen temporal que se descarta cuando se cierra la aplicación de instalación.
