# <a name="rc-diagnostic-messages"></a>Mensajes de diagnóstico rc

En esta sección se describen los mensajes de diagnóstico generados por RC. Muchos de estos mensajes aparecen cuando RC no puede compilar los recursos correctamente. Las descripciones aclaran la causa de cada mensaje.

Una V mayúscula entre paréntesis (V) al principio de una descripción del mensaje indica que el mensaje solo se muestra si RC se ejecuta con la opción **/v** (detallado). Estos mensajes suelen ser informativos y no indican necesariamente errores.

Los mensajes se enumeran en orden alfabético en los temas siguientes:

[A](a.md) [B](b.md) [C](c.md) [E](e.md) [F](f.md) [I](i.md) [N](n.md) [O](o.md) [R](r.md) [T](t.md) [U](u.md) [V](v.md) [W](w.md)
