# <a name="desktopfolder-property"></a>Propiedad DesktopFolder

El instalador establece la propiedad **DesktopFolder** en la ruta de acceso completa de la carpeta de escritorio del usuario actual. Si existe un perfil "todos los usuarios" y se establece la propiedad [**AllUsers**](allusers.md) , esta propiedad se establece en la carpeta del perfil "todos los usuarios".
