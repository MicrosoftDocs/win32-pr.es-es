# <a name="ms-ds-external-key-attribute"></a>atributo MS-DS-external-Key

Una cadena que identifica un objeto en un almacén externo, como un registro en una base de datos.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-DS-external-Key                          |
| Nombre para mostrar de LDAP | msDS-ExternalKey                            |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1833                     |
| System-ID-GUID    | b92fd528-38ac-40d4-818d-0433380837c1        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
