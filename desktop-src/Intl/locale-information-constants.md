# <a name="locale-information-constants"></a>Constantes de información de configuración regional

En esta sección se definen las constantes que usa NLS para especificar determinados fragmentos de información de configuración regional. Todos los nombres empiezan por "LOCALe \_ ". Los nombres de las constantes que representan valores enteros comienzan con "LOCALe \_ I". Todos los valores de estas constantes son cadenas Unicode terminadas en NULL. Las longitudes de cadena máximas incluyen el carácter nulo de terminación. Si no se indica ninguna longitud máxima para una constante, sus valores pueden variar en longitud.

## <a name="combining-constants"></a>Combinar constantes

Las constantes siguientes se pueden combinar con otras constantes mediante el operador binario OR. El resto de las constantes se excluyen mutuamente y, por tanto, no se pueden combinar entre sí.

-   configuración regional \_ NOUSEROVERRIDE
-   número de devolución de la configuración regional \_ \_
-   configuración regional \_ use \_ CP \_ ACP

| configuración regional \_ SLONGDATE   | configuración regional \_ ILDATE                                                                                                     |
