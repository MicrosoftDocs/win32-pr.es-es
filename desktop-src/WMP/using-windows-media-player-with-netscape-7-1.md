# <a name="using-windows-media-player-with-netscape-71"></a>Uso del Reproductor de Windows Media con Netscape 7.1

El explorador Web Netscape 7,1 proporciona compatibilidad nativa con los controles ActiveX de Windows Media Player mediante el componente ActiveX Gecko. Esto significa que puede insertar el control Media Player de Windows en las páginas web para Netscape 7,1 con la misma sintaxis que usaría para Microsoft Internet Explorer. Netscape 7,1 también admite el scripting del control y el control de eventos de un modo similar.

Si desea usar el control Media Player de Windows en Netscape, se recomienda usar Netscape 7,1.

Para obtener más información acerca del uso del control de Media Player de Windows con Netscape 7,1, vea el [sitio web de Netscape](https://developer.mozilla.org/docs/Archive/Windows_Media_in_Netscape).

Para obtener más información acerca del componente ActiveX de Gecko, vea el [sitio web de StreamingMedia.com](https://www.streamingmedia.com/Articles/Editorial/Featured-Articles/The-Windows-Media-ActiveX-Control-8211%3b-Not-Just-for-Internet-Explorer-Anymore-64608.aspx).

La descarga completa de este SDK incluye un ejemplo que muestra la detección del explorador Web. Para obtener más información sobre los ejemplos incluidos con el SDK, vea [ejemplos](samples.md).
