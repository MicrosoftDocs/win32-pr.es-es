# <a name="file-and-directory-linking"></a>Vinculación de archivos y directorios

El sistema de archivos NTFS proporciona la capacidad de crear una representación del sistema de un archivo o un directorio en una ubicación de la estructura de directorios diferente del objeto de archivo o directorio al que se está vinculando. Este proceso se denomina vinculación. En el sistema de archivos NTFS se admiten dos tipos de vínculos: [vínculos físicos y uniones](hard-links-and-junctions.md).

El sistema de archivos NTFS también proporciona el [servicio de seguimiento de vínculos distribuido](distributed-link-tracking-and-object-identifiers.md), que realiza un seguimiento automático de los vínculos a medida que se mueven.
