# <a name="values-for-the-countrycode-and-c-properties"></a>Valores de las propiedades countryCode y c

Los valores de las **propiedades countryCode** **y c** se obtienen del estándar ISO 3166. Este estándar está disponible en el sitio [Organización internacional de normalización](https://www.iso.org/) web. El valor de la propiedad **countryCode** es el código de país numérico de ISO 3166 y el valor de propiedad **c** es la designación de país o región de dos letras de ISO 3166.
