# <a name="directwrite-glossary"></a>Glosario de DirectWrite

<dl> <dt>

<span id="directwrite.directwrite_glossary_bidi__unicode_feature_"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_BIDI__UNICODE_FEATURE_"></span>**BiDi (característica Unicode)**
</dt> <dd>

Se pronuncia "por tinta". Texto que contiene una combinación de palabras, de idiomas diferentes, que se deben leer en distintas direcciones. A menudo requiere un código especial para procesar. "El árabe y el hebreo son lenguajes bidireccionales; el texto va de derecha a izquierda, pero los números van de izquierda a derecha ".

</dd> <dt>

<span id="directwrite.directwrite_glossary_cleartype"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_CLEARTYPE"></span>**ClearType**
</dt> <dd>

Tecnología de presentación de fuentes que mejora enormemente la resolución de presentación de las fuentes, de forma que las letras aparecen nítidas, no irregulares, en la pantalla. ClearType mejora en gran medida la legibilidad del texto en monitores LCD en color con una interfaz digital, como los de los equipos portátiles y las pantallas de escritorio planas de alta calidad.

</dd> <dt>

<span id="directwrite.directwrite_glossary_direct2d"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECT2D"></span>**Direct2D**
</dt> <dd>

Una API de gráficos de modo inmediato y con aceleración de hardware que proporciona alto rendimiento y una representación de alta calidad para geometría 2D, mapas de bits y texto. La API de Direct2D está diseñada para interoperar bien con el código existente que usa GDI, GDI+ o Direct3D.

</dd> <dt>

<span id="directwrite.directwrite_glossary_directwrite"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECTWRITE"></span>**DirectWrite**
</dt> <dd>

Una API de DirectX que proporciona una mejor representación e interoperabilidad de texto de alta calidad con GDI y Direct2D.

</dd> <dt>

<span id="directwrite.directwrite_glossary_directx"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_DIRECTX"></span>**DirectX**
</dt> <dd>

Una extensión del sistema operativo Microsoft Windows. La tecnología DirectX ayuda a los juegos y otros programas a usar las capacidades avanzadas de multimedia del hardware.

</dd> <dt>

<span id="directwrite.directwrite_glossary_gdi"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GDI"></span>**GDI**
</dt> <dd>

Programa ejecutable que procesa las llamadas a funciones gráficas desde una aplicación basada en Windows y pasa esas llamadas al controlador de dispositivo adecuado, que realiza las funciones específicas del hardware que generan la salida. Al actuar como un búfer entre las aplicaciones y los dispositivos de salida, GDI presenta una vista independiente del dispositivo del mundo de la aplicación a la vez que interactúa en un formato dependiente del dispositivo con el dispositivo.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH"></span>**Pictogram**
</dt> <dd>

Representación física de un carácter en una fuente determinada. Los caracteres podrían tener muchos glifos, y cada fuente de un sistema podría definir un glifo diferente para ese carácter.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_composition"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_COMPOSITION"></span>**composición de glifo**
</dt> <dd>

Combinación de dos o más glifos en un único glifo.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_decomposition"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_DECOMPOSITION"></span>**descomposición de glifos**
</dt> <dd>

División de un solo glifo en varios glifos.

</dd> <dt>

<span id="directwrite.directwrite_glossary_glyph_run"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_GLYPH_RUN"></span>**Ejecución de glifos**
</dt> <dd>

Conjunto de glifos en un orden específico, con las mismas características de formato, como el tipo de fuente, el tamaño, el grosor y el estilo.

</dd> <dt>

<span id="directwrite.directwrite_glossary_hardware-accelerated_text"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_HARDWARE-ACCELERATED_TEXT"></span>**texto con aceleración de hardware**
</dt> <dd>

Texto representado por una tecnología que usa la aceleración de hardware para mejorar el rendimiento de la representación.

</dd> <dt>

<span id="directwrite.directwrite_glossary_interoperability"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_INTEROPERABILITY"></span>**interoperabilidad**
</dt> <dd>

Capacidad para que dos o más API funcionen con y transmitan información entre sí.

</dd> <dt>

<span id="directwrite.directwrite_glossary_kerning"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_KERNING"></span>**el interletraje**
</dt> <dd>

Espaciado aplicado entre pares de letras específicos de una palabra determinada.

</dd> <dt>

<span id="directwrite.directwrite_glossary_letterspacing"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_LETTERSPACING"></span>**Espacia**
</dt> <dd>

Ajuste del espaciado entre dos caracteres para crear la apariencia de espaciado uniforme, ajustar el texto a un espacio determinado y ajustar los saltos de línea.

</dd> <dt>

<span id="directwrite.directwrite_glossary_ligature"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_LIGATURE"></span>**ligadura**
</dt> <dd>

Dos o más caracteres combinados para representar un único carácter tipográfico. El script latín moderno usa algunos. Otros scripts usan muchas ligaduras que dependen de la fuente y el estilo. Algunos idiomas poseen ligaduras obligatorias, por ejemplo, Árabe.

</dd> <dt>

<span id="directwrite.directwrite_glossary_smart_pointers"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_SMART_POINTERS"></span>**punteros inteligentes**
</dt> <dd>

Una clase que contiene punteros de interfaz COM que liberarán automáticamente el objeto de interfaz.

</dd> <dt>

<span id="directwrite.directwrite_glossary_swashes"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_SWASHES"></span>**Decora**
</dt> <dd>

Adición ornamental a un carácter que hace que un glifo sea más estilístico.

</dd> <dt>

<span id="directwrite.directwrite_glossary_unicode"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_UNICODE"></span>**Unicode**
</dt> <dd>

Estándar de codificación de caracteres desarrollado por Unicode Consortium que representa casi todos los idiomas escritos del mundo. El repertorio de caracteres Unicode tiene varias formas de representación, incluidas UTF-8, UTF-16 y UTF-32. La mayoría de las interfaces de Windows utilizan el formato UTF-16. (GTMT tiene 20 definiciones).

</dd> <dt>

<span id="directwrite.directwrite_glossary_windows_presentation_foundation__wpf_"></span><span id="DIRECTWRITE.DIRECTWRITE_GLOSSARY_WINDOWS_PRESENTATION_FOUNDATION__WPF_"></span>**Windows Presentation Foundation (WPF)**
</dt> <dd>

Marco de trabajo de GUI usado por .NET micro Framework 3,0, Windows Vista y Microsoft Silverlight. Las clases de GUI de .NET micro Framework se basan de forma flexible en WPF.

</dd> </dl>
