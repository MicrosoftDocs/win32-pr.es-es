# <a name="mf_mt_max_frame_average_luminance_level-attribute"></a>\_Atributo de \_ \_ nivel de \_ luminancia promedio de trama MF \_ MT \_

Especifica el promedio máximo de nivel de luminancia por fotograma del contenido en Nits. Este atributo tiene la misma semántica que **MaxFALL** , tal y como se define en el estándar CEA-861,3.

## <a name="data-type"></a>Tipo de datos

**UINT32**
