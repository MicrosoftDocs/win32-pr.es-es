# <a name="windows7paramlvchunksizemost-field"></a>Campo Windows7Param.LVChunkSizeMost

Este parámetro se usa para recuperar el tamaño del fragmento de datos de valor largo (blob). Establecer y recuperar datos en múltiplo de este tamaño aumenta la eficacia.

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop.Windows7](./microsoft.isam.esent.interop.windows7-namespace.md)
