# <a name="lsa-user-logon-authentication"></a>Autenticación de inicio de sesión de usuario LSA

LSA controla el inicio de sesión y la autenticación de los usuarios en el equipo local y, si el paquete de autenticación que procesa la solicitud de inicio de sesión admite la autenticación de paso a través, la LSA también puede iniciar la sesión de los usuarios en otros equipos de la red. LSA proporciona acceso a los paquetes de autenticación para los proveedores de compatibilidad para seguridad (SSP). Se lleva a cabo uno de los dos tipos de autenticación siguientes:

-   [Autenticación interactiva](interactive-authentication.md)
-   [Autenticación no interactiva](noninteractive-authentication.md)
