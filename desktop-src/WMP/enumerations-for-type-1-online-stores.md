# <a name="enumerations-for-type-1-online-stores"></a>Enumeraciones de las tiendas en línea de tipo 1

> [!Note]  
> En esta sección se describe la funcionalidad diseñada para su uso en tiendas en línea. No se admite el uso de esta funcionalidad fuera del contexto de una tienda en línea.

 

Los siguientes tipos de enumeración están disponibles para las tiendas en línea de tipo 1 mediante código de C++.



| Enumeración                                            | Descripción                                                                                              |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------|
| [WMPAccountType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpaccounttype)                   | Enumeración que define los tipos de cuenta para una tienda en línea.                                              |
| [WMPCallbackNotification](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpcallbacknotification) | Enumeración que define los Estados que va a usar la función de devolución de llamada **IWMPContentPartnerCallback:: Notify** . |
| [WMPPartnerNotification](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmppartnernotification)   | Enumeración que define los Estados operativos de una tienda en línea.                                          |
| [WMPStreamingType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmpstreamingtype)               | Enumeración que especifica el tipo de contenido multimedia de transmisión por secuencias.                                                  |
| [WMPTaskType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptasktype)                         | Enumeración que representa los paneles de tareas de Windows Media Player.                                             |
| [WMPTemplateSize](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptemplatesize)                 | Enumeración que representa los tamaños de plantilla HTML.                                                         |
| [WMPTransactionType](/previous-versions/windows/desktop/api/contentpartner/ne-contentpartner-wmptransactiontype)           | Enumeración que representa un tipo de transacción.                                                          |
