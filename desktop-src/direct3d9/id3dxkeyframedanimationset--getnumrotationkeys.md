# <a name="id3dxkeyframedanimationsetgetnumrotationkeys-method"></a>Método ID3DXKeyframedAnimationSet::GetNumRotationKeys

Obtiene el número de claves de rotación en la animación de fotogramas clave especificada.

## <a name="syntax"></a>Sintaxis


```C++
UINT GetNumRotationKeys(
  [in] UINT Animation
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*Animación* \[ En\]
</dt> <dd>

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

Índice de animación.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

Número de claves de rotación en la animación de fotograma clave especificada.
