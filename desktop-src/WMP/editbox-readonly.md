# <a name="editboxreadonly"></a>EDITBOX. readOnly

El atributo **ReadOnly** especifica o recupera un valor que indica si el texto del control de cuadro de edición es de solo lectura o si se puede editar.

``` syntax
        elementID.readOnly
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **booleano** de lectura/escritura con un valor predeterminado de false.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------------|
| Versión<br/> | Windows Media Player para Windows XP o posterior<br/> |
