# <a name="operating-system-availability-of-wmi-components"></a>Disponibilidad del sistema operativo de los componentes WMI

WMI se instala de forma predeterminada en todas las plataformas de escritorio y servidor de Windows. Sin embargo, algunos proveedores de WMI se pueden instalar o no, dependiendo de la versión del sistema operativo y la configuración. Por ejemplo, el proveedor SNMP no está habilitado de forma predeterminada, y el proveedor de Windows Installer (proveedor MSI) no se instala de forma predeterminada en los sistemas operativos de 64 bits. Para obtener más información sobre la disponibilidad de un proveedor específico, consulte [proveedores de WMI](/windows/desktop/SrvNodes/wmi-mi-omi-providers).
