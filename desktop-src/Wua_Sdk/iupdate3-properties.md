# <a name="iupdate3-properties"></a>Propiedades de IUpdate3

La [**interfaz IUpdate3**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate3) define la siguiente propiedad.



| Propiedad                                  | Descripción                                                                                                             |
|-------------------------------------------|-------------------------------------------------------------------------------------------------------------------------|
| [**BrowseOnly**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate3-get_browseonly) | Obtiene un valor booleano que indica si una actualización solo se puede detectar explorando las actualizaciones disponibles. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea las siguientes interfaces:

-   [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2)
-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
