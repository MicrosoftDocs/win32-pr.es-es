# <a name="xmshortn4-extensions"></a>Extensiones XMSHORTN4

[**XMSHORTN4**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmshortn4) describe el vector 4D para almacenar valores con signo normalizados como enteros de 16 bits con signo (tipo `int16_t` ).

> [!Note]  
> Las características enumeradas aquí solo están disponibles al desarrollar con C++.
