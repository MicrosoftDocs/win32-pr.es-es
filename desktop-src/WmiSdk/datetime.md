# <a name="datetime"></a>DATETIME

El tipo de datos DATETIME es una cadena BSTR monodimensional de longitud fija \_ que representa una fecha y hora específicas o un intervalo. Para obtener más información sobre DATETIME como una fecha, vea [formato de fecha y hora](date-and-time-format.md). Para obtener más información sobre DATETIME como un intervalo, consulte [Interval Format](interval-format.md).
