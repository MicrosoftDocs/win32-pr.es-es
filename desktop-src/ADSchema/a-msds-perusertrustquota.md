# <a name="ms-ds-per-user-trust-quota-attribute"></a>Atributo MS-DS-per-user-Trust-quota

Se utiliza para exigir una cuota por usuario para crear Trusted-Domain objetos autorizados por el permiso de acceso de control nuevo, crear-entrada-bosque-confianza. Este atributo limita el número de objetos de Trusted-Domain que un solo usuario no administrador puede crear.



| Entrada | Value |
|-------------------|-------------------------------------------|
| CN                | MS-DS-per-user-Trust-quota                |
| Nombre para mostrar de LDAP | msDS-PerUserTrustQuota                    |
| Tamaño              | \-                                        |
| Actualizar privilegio  | Administrador de dominio                      |
| Frecuencia de actualización  | Al crear el bosque y raramente después. |
| Attribute-Id      | 1.2.840.113556.1.4.1788                   |
| System-ID-GUID    | d161adf0-ca24-4993-a3aa-8b2c981302e8      |
| Sintaxis            | [**Enumeración**](s-enumeration.md)      |
