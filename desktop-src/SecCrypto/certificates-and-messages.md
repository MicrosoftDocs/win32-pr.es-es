# <a name="certificates-and-messages"></a>Certificados y mensajes

Las funciones criptográficas base utilizan [*certificados*](../secgloss/c-gly.md), y las funciones [*de mensaje de bajo nivel*](../secgloss/l-gly.md) y [*las funciones de mensaje simplificadas*](../secgloss/s-gly.md) también las utilizan. Para enviar o recibir mensajes que incorporan certificados, utilice funciones de mensaje de bajo nivel.

Para obtener ejemplos del uso de certificados con las funciones de mensaje de bajo nivel, vea [ejemplo de programa c: codificar y descodificar un mensaje con signo](example-c-program-encoding-and-decoding-a-countersigned-message.md), [ejemplo de c: codificar y descodificar un mensaje con hash](example-c-program-encoding-and-decoding-a-hashed-message.md).

Para obtener ejemplos del uso de certificados con las funciones de mensaje simplificado, vea [programa C de ejemplo: uso de CryptEncryptMessage y CryptDecryptMessage](example-c-program-using-cryptencryptmessage-and-cryptdecryptmessage.md).

Para obtener información sobre las funciones de mensajes de bajo nivel, consulte [mensajes de nivel bajo](low-level-messages.md).

Para obtener información acerca de las funciones de mensaje simplificado, consulte [mensajes simplificados](simplified-messages.md).
