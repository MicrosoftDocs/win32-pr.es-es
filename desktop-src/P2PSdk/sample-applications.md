# <a name="sample-applications-peer-infrastructure"></a>Aplicaciones de ejemplo (infraestructura del mismo nivel)

Las aplicaciones de ejemplo siguientes se incluyen en el SDK del mismo nivel de Windows XP. Los ejemplos pueden ayudarle a desarrollar sus propias aplicaciones del mismo nivel mediante la infraestructura del mismo nivel.

-   [Chat de grafos](#graph-chat)
-   [Conversación de grupo](#group-chat)
-   [Explorador de grupo](#group-browser)

## <a name="graph-chat"></a>Chat de grafos

La aplicación de ejemplo de chat de grafos es una sencilla aplicación de chat que muestra cómo usar las API de gráficos del mismo nivel y el proveedor de espacios de nombres del Protocolo de resolución de nombres de mismo nivel (PNRP) con la API Winsock 2. La aplicación muestra las siguientes tareas:

-   Crear un gráfico
-   Conexión a un gráfico existente
-   Desconectar de un gráfico existente
-   Enumerar entidades del mismo nivel
-   Agregar registros al gráfico
-   Usar conexiones directas con un gráfico
-   Uso de la infraestructura de eventos y notificaciones con gráficos
-   Registro de nombres con PNRP
-   Resolución de nombres con PNRP
-   Anulación del registro de nombres con PNRP

## <a name="group-chat"></a>Conversación de grupo

La aplicación de ejemplo chat de grupo es una sencilla aplicación de chat que muestra cómo usar las API de agrupación del mismo nivel y de Identity Manager. La aplicación muestra las siguientes tareas:

-   Crear una identidad
-   Crear y obtener información de identidad
-   Enumerar identidades
-   Enumerar grupos asociados a una identidad
-   Creating a group
-   Crear invitaciones para un grupo
-   Conexión a un grupo existente
-   Desconectar de un grupo existente
-   Extraer información de las propiedades del grupo
-   Usar conexiones directas con un grupo
-   Usar las funciones de enumeración dentro de un grupo
-   Enumerar miembros de grupo
-   Agregar registros a un grupo
-   Uso de la infraestructura de eventos y notificaciones con grupos

## <a name="group-browser"></a>Explorador de grupo

La aplicación de ejemplo de explorador de grupo es una sencilla herramienta de administración de grupos del mismo nivel que muestra cómo usar las API de agrupación del mismo nivel y de Identity Manager. La aplicación muestra las siguientes tareas:

-   Enumerar nubes PNRP
-   Enumerar identidades
-   Enumerar grupos asociados a una identidad
-   Crear y eliminar identidades
-   Crear un grupo y asociarlo a una identidad
-   Crear una invitación y guardarla
-   Abrir una invitación y usarla para unirse a un grupo
-   Eliminación de una identidad y pertenencia a grupos
-   Conexión a un grupo existente
-   Desconectar de un grupo existente
-   Extraer información de las propiedades del grupo
-   Usar las funciones de enumeración dentro de un grupo
-   Enumerar miembros de grupo
-   Uso de la infraestructura de eventos y notificaciones con grupos
