# <a name="iupdateservicecollection-properties"></a>Propiedades de IUpdateServiceCollection

La [**interfaz IUpdateServiceCollection**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservicecollection) define las siguientes propiedades.



| Propiedad                                               | Descripción                                                                                                          |
|--------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicecollection-get__newenum) | Obtiene una [**interfaz IEnumVARIANT**](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) que se usa para enumerar la colección. |
| [**Contar**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicecollection-get_count)        | Obtiene el número de elementos de la colección.                                                                       |
| [**Elemento**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateservicecollection-get_item)          | Obtiene y establece una [**interfaz IUpdateService**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateservice) en la colección.                               |
