# <a name="primary-service-provider"></a>Proveedor de servicios principal

Cada [*tarjeta inteligente*](../secgloss/s-gly.md) conocida por el sistema puede tener una interfaz COM registrada enumerada como su proveedor de servicios principal. Cuando se accede a una *tarjeta inteligente* a través de COM, este proveedor [*de servicios*](../secgloss/p-gly.md) principal proporciona las interfaces de control a la tarjeta. Esto permite que los servicios de tarjetas se exponan a una amplia gama de entornos de programación, incluidos Java y el sistema de desarrollo Visual Basic Microsoft.
