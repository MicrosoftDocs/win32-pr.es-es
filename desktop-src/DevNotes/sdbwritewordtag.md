# <a name="sdbwritewordtag-function"></a>SdbWriteWORDTag función)

Escribe un valor de **Word** en la base de datos especificada.

## <a name="syntax"></a>Sintaxis


```C++
BOOL WINAPI SdbWriteWORDTag(
  _In_ PDB  pdb,
  _In_ TAG  tTag,
  _In_ WORD wData
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

archivo *PDB* \[ de\]
</dt> <dd>

Identificador de la base de datos de correcciones de compatibilidad.

</dd> <dt>

*tTag* \[ de\]
</dt> <dd>

ETIQUETA de la entrada. Esta etiqueta debe ser de tipo **etiqueta \_ \_ palabra**.

</dd> <dt>

*wData* \[ de\]
</dt> <dd>

Valor.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

La función devuelve **true** si se ejecuta correctamente o **false** en caso de error.
