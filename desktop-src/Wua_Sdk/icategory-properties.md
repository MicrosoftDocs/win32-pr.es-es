# <a name="icategory-properties"></a>Propiedades de ICategory

La interfaz [**ICategory**](/windows/desktop/api/Wuapi/nn-wuapi-icategory) define las siguientes propiedades.



| Propiedad                                     | Descripción                                                                                       |
|----------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**CategoryID**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_categoryid)   | Obtiene el identificador de la categoría.                                                              |
| [**Children**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_children)       | Obtiene una colección de interfaces que contiene las categorías secundarias de esta categoría.                 |
| [**Descripción**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_description) | Obtiene la descripción de la categoría.                                                             |
| [**Impresión**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_image)             | Obtiene una interfaz que contiene información sobre la imagen que está asociada a la categoría. |
| [**Name**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_name)               | Obtiene el nombre traducido de la categoría.                                                          |
| [**Pedido**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_order)             | Obtiene el orden de presentación recomendado de esta categoría entre sus categorías relacionadas.                 |
| [**Parent**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_parent)           | Obtiene una interfaz que describe la categoría primaria de esta categoría.                            |
| [**Tipo**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_type)               | Obtiene el tipo de la categoría.                                                                    |
| [**Actualizaciones**](/windows/desktop/api/Wuapi/nf-wuapi-icategory-get_updates)         | Obtiene una interfaz que contiene una colección de actualizaciones que pertenecen inmediatamente a la categoría.  |
