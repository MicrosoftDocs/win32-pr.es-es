# <a name="error-codes-windows-media-format-11-sdk"></a>Códigos de error (Windows SDK de formato multimedia 11)

Microsoft ha definido los siguientes códigos de error y de éxito para su uso en Windows SDK de formato multimedia. Estos códigos son un subconjunto de una colección más grande que se puede ver en los archivos de encabezado asferr.h (códigos ASF) y nserror.h (otros códigos) que se incluyen en el SDK de formato multimedia de Windows. Se pueden ver códigos de error comunes adicionales en Winerror.h.
