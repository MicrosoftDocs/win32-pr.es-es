# <a name="oafilterstate"></a>OAFilterState

El `OAFilterState` tipo de datos define el estado del gráfico de filtro. El valor es miembro de la [**enumeración FILTER \_ STATE.**](/windows/win32/api/strmif/ne-strmif-filter_state) Este tipo de datos se define para los clientes de OLE Automation, como Microsoft Visual Basic 6.0.


```C++
typedef long OAFilterState;
```
