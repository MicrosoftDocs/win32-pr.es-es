# <a name="device-service-tasks"></a>Tareas del servicio de dispositivo

La API de WPD es compatible con una serie de tareas de servicios de dispositivos. Esto incluye las tareas siguientes:

-   Enumerar servicios de dispositivo.
-   Abrir un servicio de dispositivo.
-   Enumerar el contenido de un servicio de dispositivo.
-   Recuperando eventos de servicio admitidos.
-   Recuperando formatos de servicio admitidos.
-   Recuperando métodos de servicio admitidos.
-   Recuperando servicios abstractos admitidos.
-   Recuperando propiedades de objetos de contenido.
-   Escribir propiedades de contenido-objeto.
-   Invocar métodos de servicio sincrónicamente.
-   Invocar métodos de servicio de forma asincrónica.

Estas operaciones se describen en las secciones siguientes y se muestran en la aplicación WpdServicesApiSample. Además, la API de WPD también admite la transferencia de contenido, el traslado de contenido y la eliminación de contenido de un servicio de dispositivo.
