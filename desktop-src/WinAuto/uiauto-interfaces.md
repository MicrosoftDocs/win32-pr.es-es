# <a name="interfaces-for-providers"></a>Interfaces para proveedores

En esta sección se describen las interfaces básicas implementadas por los proveedores de automatización de la interfaz de usuario para las aplicaciones Win32 de Microsoft.

## <a name="in-this-section"></a>En esta sección



| Interfaz                                                                                                    | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
|--------------------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAccessibleEx**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iaccessibleex)<br/>                                                     | Expone métodos a los que Microsoft UI Automation llama para recuperar información adicional acerca de un control que admite Microsoft Active Accessibility.<br/>                                                                                                                                                                                                                                                                                                                |
| [**IAccessibleHostingElementProviders**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iaccessiblehostingelementproviders)<br/>                  | Un objeto de Active Accessibility de Microsoft implementa esta interfaz cuando el objeto es la raíz de un árbol de accesibilidad que incluye controles ActiveX de Microsoft sin ventanas que implementan la automatización de la interfaz de usuario. Dado que Microsoft Active Accessibility y la automatización de la interfaz de usuario usan interfaces diferentes, esta interfaz permite a un cliente detectar la lista de controles ActiveX sin ventanas hospedados que admiten la automatización de la interfaz de usuario en caso de que el cliente necesite tratarlos de manera diferente.<br/> |
| [**IProxyProviderWinEventHandler**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iproxyproviderwineventhandler)<br/>                     | Expone un método que los proveedores de proxy implementan para controlar WinEvents.<br/>                                                                                                                                                                                                                                                                                                                                                                                          |
| [**IProxyProviderWinEventSink**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-iproxyproviderwineventsink)<br/>                           | Expone métodos usados por los proveedores de proxy para generar eventos.<br/>                                                                                                                                                                                                                                                                                                                                                                                                              |
| [**IRawElementProviderAdviseEvents**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovideradviseevents)<br/>                 | Expone métodos a los que se llama para notificar al elemento raíz de un fragmento cuando una aplicación cliente de automatización de la interfaz de usuario inicia o finaliza la escucha de eventos en ese fragmento.<br/>                                                                                                                                                                                                                                                                                                |
| [**IRawElementProviderFragment**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragment)<br/>                         | Expone métodos y propiedades en elementos de interfaz de usuario que forman parte de una estructura de más de un nivel de profundidad, como un cuadro de lista o un elemento de lista. Implementado por el proveedor de automatización de la interfaz de usuario.<br/>                                                                                                                                                                                                                                                                                          |
| [**IRawElementProviderFragmentRoot**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragmentroot)<br/>                 | Expone métodos y propiedades en el elemento raíz de un fragmento.<br/>                                                                                                                                                                                                                                                                                                                                                                                                     |
| [**IRawElementProviderHostingAccessibles**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-irawelementproviderhostingaccessibles)<br/> | Esta interfaz se implementa mediante un proveedor de automatización de la interfaz de usuario cuando el proveedor es la raíz de un árbol de accesibilidad que incluye controles sin ventanas que admiten Microsoft Active Accessibility. Como la automatización de la interfaz de usuario y Microsoft Active Accessibility usar interfaces diferentes, esta interfaz permite a un cliente detectar la lista de controles de Microsoft Active Accessibility hospedados en caso de que sea necesario tratarlos de manera diferente.<br/>                                 |
| [**IRawElementProviderHwndOverride**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderhwndoverride)<br/>                 | Expone un método que permite cambiar la posición de los elementos basados en ventanas en el árbol de automatización de la interfaz de usuario del fragmento.<br/>                                                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple)<br/>                             | Define métodos y propiedades que exponen elementos de interfaz de usuario simples.<br/>                                                                                                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple2)<br/>                           | Extiende la interfaz [**IRawElementProviderSimple**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple) para habilitar la invocación mediante programación de los menús contextuales.<br/>                                                                                                                                                                                                                                                                                                                        |
| [**IRawElementProviderSimple3**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple3)<br/>                           | Extiende la interfaz [**IRawElementProviderSimple2**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementprovidersimple2) para habilitar la recuperación de metadatos sobre la manera en que la tecnología accesible debe indicar el tipo de contenido preferido.<br/>                                                                                                                                                                                                                                                                    |
| [**IRawElementProviderWindowlessSite**](/windows/desktop/api/uiautomationcore/nn-uiautomationcore-irawelementproviderwindowlesssite)<br/>         | Un sitio de control ActiveX implementa esta interfaz para permitir que un control ActiveX habilitado para la automatización de la interfaz de usuario exprese su accesibilidad. Esta interfaz permite que el contenedor de control proporcione un puntero [**IRawElementProviderFragment**](/windows/desktop/api/UIAutomationCore/nn-uiautomationcore-irawelementproviderfragment) para el elemento primario o del mismo nivel del control ActiveX sin ventanas, y para proporcionar un identificador en tiempo de ejecución que sea único para el sitio de control.<br/>                                                           |
