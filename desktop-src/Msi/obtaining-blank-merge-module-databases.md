# <a name="obtaining-blank-merge-module-databases"></a>Obtener bases de datos de módulos de combinación en blanco

Obtenga una base de datos de módulo de combinación en blanco. Puede usar el archivo Schema.msm proporcionado con el SDK Windows Installer como base de datos inicial para el módulo de combinación. Para obtener más información, [consulte Windows SDK Components for Windows Installer Developers](platform-sdk-components-for-windows-installer-developers.md).

Los desarrolladores deben crear módulos de mezcla mediante el esquema de base de datos más sencillo que instala sus componentes. El uso de un esquema simple garantiza la mayor compatibilidad para el módulo de combinación. La combinación de un módulo de combinación en un paquete de instalación con un esquema de base de datos diferente suele tener como resultado conflictos de combinación.

Para obtener una lista completa de todas las tablas obligatorias y opcionales de los módulos de combinación, vea Combinar tablas de base [de datos de módulos](merge-module-database-tables.md).
