# <a name="iimageinformation-properties"></a>Propiedades de IImageInformation

La interfaz [**IImageInformation**](/windows/desktop/api/Wuapi/nn-wuapi-iimageinformation) define las siguientes propiedades.



| Propiedad                                     | Descripción                              |
|----------------------------------------------|------------------------------------------|
| [**AltText**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_alttext) | Obtiene el texto alternativo de la imagen.   |
| [**Height**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_height)   | Obtiene el alto de la imagen, en píxeles. |
| [**Origen**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_source)   | Obtiene la ubicación de origen de la imagen.   |
| [**Width**](/windows/desktop/api/Wuapi/nf-wuapi-iimageinformation-get_width)     | Obtiene el ancho de la imagen, en píxeles.  |
