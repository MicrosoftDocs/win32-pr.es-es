# <a name="registration-and-licensing"></a>Registro y licencias

Normalmente, un control se proporciona como un servidor en proceso (.DLL), aunque también puede ser un servidor local o remoto (.EXE).

Normalmente, un control admite el registro automático y crea un conjunto de entradas del Registro cuando se crea una instancia de él. Un control también se puede licenciar para ayudar a evitar el uso no autorizado. Consulte [El modelo de objetos componentes](the-component-object-model.md) para obtener más información sobre el registro y las licencias.

Para obtener más información, [vea ActiveX información del Registro de controles .](activex-controls-registry-information.md)
