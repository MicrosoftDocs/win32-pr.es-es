# <a name="monitor-event-reference-pages"></a>Páginas de referencia de eventos de supervisión

Las páginas de referencia de eventos (O ERP e) se pueden usar para conectar el monitor a la interfaz de usuario de Monitor de red. Se pueden usar para avisar a los usuarios finales acerca de los problemas y condiciones detectados, o bien se pueden usar para proporcionar correcciones o datos explicativos sugeridos.
