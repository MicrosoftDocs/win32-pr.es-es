# <a name="transport-identifiers"></a>Identificadores de transporte

Los siguientes identificadores de transporte (o familia de protocolos) también se enumeran en Mprapi.h.



| Value      | Transporte (familia de protocolos)                               |
|------------|-----------------------------------------------------------|
| PID \_ ATALK | Appletalk                                                 |
| IP \_ de PID    | Protocolo de Internet versión 4                               |
| PID \_ IPX   | Internet Packet Exchange                                  |
| PID \_ NBF   | Protocolo de marcos NetBIOS                                   |
| PID \_ IPV6  | Windows Server 2008 o posterior: Protocolo de Internet versión 6 |
