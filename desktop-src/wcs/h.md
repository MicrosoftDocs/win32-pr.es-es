# <a name="h"></a>H

## <a name="hard-proofing"></a>pruebas de seguridad

Obtener una vista previa o probar una imagen mediante la impresión de una copia de ella.

## <a name="hls"></a>HLS

Un espacio de colores usado a menudo por los intérpretes. Sus componentes son Hue, Lightness y Saturation (croma).

## <a name="hsv"></a>Hsv

Un espacio de colores usado a menudo por los intérpretes. Sus componentes son Hue, Saturación (contraste) y Valor (lumíceo).

## <a name="hue"></a>Hue
