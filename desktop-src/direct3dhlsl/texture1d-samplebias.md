|:-------------------------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

| [**SampleBias (S, Float, Float, int, float)**](t1d-samplebias-s-float-float-int-float-.md)           | Muestrea una textura, después de aplicar el valor de diferencia al nivel de mipmap, con un valor opcional para Clamp valores de nivel de detalle (LOD) de ejemplo en.<br/>                                     |
