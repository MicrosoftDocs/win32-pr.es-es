# <a name="midl-enumerated-types"></a>Tipos enumerados de MIDL

El [**compilador MIDL**](enum.md) no traduce la declaración de enumeración en instrucciones **\# define,** como lo están algunos compiladores de DCE. Se reproduce como una declaración de enumeración **en lenguaje** C en el archivo de encabezado generado.
