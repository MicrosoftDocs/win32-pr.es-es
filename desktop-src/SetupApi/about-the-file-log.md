# <a name="about-the-file-log"></a>Acerca del registro de archivos

Un registro de archivos registra información sobre los archivos copiados en un sistema durante una instalación. Se puede usar para la reparación de emergencia o como una herramienta de diagnóstico.

El archivo que se usa para registrar instalaciones de archivos puede ser el registro del sistema o cualquier otro registro de archivos que especifique.
