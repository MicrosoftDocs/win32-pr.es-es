# <a name="control-point-programming-considerations"></a>Consideraciones de programación de puntos de control

Los desarrolladores que crean aplicaciones basadas en UPnP (o puntos de control) deben usar estas aplicaciones desde un cliente \_ APARTMENTTHREADED COINIT. Hay problemas conocidos al usar la API de punto de control desde un cliente \_ COINIT MULTITHREADED que se ejecuta bajo esfuerzo.
