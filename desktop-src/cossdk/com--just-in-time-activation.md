# <a name="com-just-in-time-activation"></a>Activación Just-in-Time de COM+

La activación Just-in-Time (JIT) es un servicio de COM+ automático que puede ayudarle a usar los recursos del servidor de forma más eficaz, especialmente al escalar verticalmente la aplicación para realizar transacciones de gran volumen.

En los temas siguientes se proporciona información básica y de tareas sobre la activación Just-in-Time de COM+.



| Tema                                                                              | Descripción                                                                                                        |
|------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [Conceptos de activación Just-in-Time de COM+](com--just-in-time-activation-concepts.md) | Proporciona información general sobre las características y ventajas de habilitar el servicio de activación JIT para los componentes COM+. |
| [Tareas de activación Just-in-Time de COM+](com--just-in-time-activation-tasks.md)       | Proporciona instrucciones para configurar la activación JIT de las aplicaciones COM+.                                       |
