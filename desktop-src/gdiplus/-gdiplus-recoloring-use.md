# <a name="recoloring"></a>Cambio de color

Cambiar el color es el proceso de ajustar los colores de la imagen. Algunos ejemplos de cambio de color cambian el color a otro, el ajuste de la intensidad de un color en relación con otro color, el ajuste del brillo o el contraste de todos los colores y la conversión de colores en tonalidades de gris.

En los temas siguientes se explica cómo cambiar el color con más detalle:

-   [Uso de una matriz de colores para transformar un color único](-gdiplus-using-a-color-matrix-to-transform-a-single-color-use.md)
-   [Traducir colores](-gdiplus-translating-colors-use.md)
-   [Escalado de colores](-gdiplus-scaling-colors-use.md)
-   [Girar colores](-gdiplus-rotating-colors-use.md)
-   [Distorsionar colores](-gdiplus-shearing-colors-use.md)
-   [Uso de una tabla de reasignación de colores](-gdiplus-using-a-color-remap-table-use.md)
