# <a name="simple-data-binding"></a>enlace de datos simple

La ActiveX controles define un mecanismo de enlace de datos, por el que un control ActiveX puede especificar que una o varias de sus propiedades son enlazables. En la mayoría de los casos, un control enlazado a datos no debe requerir absolutamente el enlace de datos, por lo que podría insertarse en un contenedor que no admita el enlace de datos. Obviamente, en esta situación, se puede reducir la funcionalidad del control.
