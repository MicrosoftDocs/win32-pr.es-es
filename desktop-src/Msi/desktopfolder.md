# <a name="desktopfolder-property"></a>Propiedad DesktopFolder

El instalador establece la **propiedad DesktopFolder** en la ruta de acceso completa de la carpeta Desktop del usuario actual. Si existe un perfil "Todos los usuarios" y se establece la propiedad [**ALLUSERS,**](allusers.md) esta propiedad se establece en la carpeta del perfil "Todos los usuarios".
