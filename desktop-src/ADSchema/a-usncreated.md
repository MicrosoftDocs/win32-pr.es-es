# <a name="usn-created-attribute"></a>USN-Created atributo

Número de secuencia de actualización (USN) asignado al crear el objeto. Vea también, [**USN-Changed ( Cambiado por USN).**](a-usnchanged.md)



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | USN-Created                          |
| Ldap-Display-Name | uSNCreated                           |
| Size              | 8 bytes                              |
| Privilegio actualizar  | El sistema establece este valor.     |
| Frecuencia de actualización  | Cuando se crea el objeto .          |
| Attribute-Id      | 1.2.840.113556.1.2.19                |
| System-Id-Guid    | bf967a70-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervalo**](s-interval.md)       |
