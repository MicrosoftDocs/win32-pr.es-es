# <a name="creating-a-private-sensor-pool"></a>Creación de un grupo de sensores privados

Un grupo de sensores privado es una colección de unidades biométricas reservadas para uso exclusivo de una aplicación cliente. Los grupos privados admiten métodos de autenticación propietarios y permiten que una aplicación cliente acceda a una unidad biométrica mediante comandos de control especificados por el proveedor.

Los temas siguientes contienen ejemplos de código que muestran cómo crear un grupo de sensores privado.
