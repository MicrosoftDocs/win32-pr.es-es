# <a name="ms-ds-members-for-az-role-bl-attribute"></a>atributo MS-DS-Members-for-AZ-role-BL

Vínculo hacia atrás del grupo o usuario de la aplicación miembro para Az-Role objetos que se vinculan a él.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-DS-Members-for-AZ-role-BL            |
| Nombre para mostrar de LDAP | msDS-MembersForAzRoleBL                 |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | El sistema establece este valor.        |
| Attribute-Id      | 1.2.840.113556.1.4.1807                 |
| System-ID-GUID    | ececcd20-a7e0-4688-9ccf-02ece5e287f5    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
