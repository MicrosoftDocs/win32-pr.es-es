# <a name="ndis-proxy-tsp"></a>TSP de proxy NDIS

El proxy NDIS es un componente de modo kernel que expone NDIS miniports como líneas TAPI. El TSP del proxy NDIS (Ndptsp. TSP) permite a TAPI comunicarse con el proxy NDIS.

El TSP del proxy NDIS se instala con los sistemas operativos Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Millennium Edition, Windows 98 y Windows 95.
