# <a name="ccactive-property"></a>Propiedad CCActive

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

La `CCActive` propiedad establece o recupera el estado actual de los subtítulos (CC).

``` syntax
[ bCCActive = ] MSWebDVD.CCActive
```

## <a name="return-value"></a>Valor devuelto

Devuelve un valor booleano que indica si está activado el subtítulos (CC).

## <a name="remarks"></a>Observaciones

Esta propiedad es de lectura/escritura y su valor predeterminado es false.
