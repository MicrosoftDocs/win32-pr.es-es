# <a name="default-security-descriptor-attribute"></a>Atributo default-Security-Descriptor

El descriptor de seguridad que se va a asignar al objeto cuando se crea por primera vez.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Default-Security-Descriptor                 |
| Nombre para mostrar de LDAP | defaultSecurityDescriptor                   |
| Tamaño              | \-                                          |
| Actualizar privilegio  | Administrador de esquema                        |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.224                      |
| System-ID-GUID    | 807a6d30-1669-11d0-a064-00aa006c33ed        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
