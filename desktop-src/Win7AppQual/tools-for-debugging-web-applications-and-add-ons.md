# <a name="tools-for-debugging-web-applications-and-add-ons"></a>Herramientas para depurar aplicaciones web y Add-Ons

En lugar de corregir problemas de compatibilidad en aplicaciones [web](remediating-web-applications-and-add-ons.md)y complementos, puede hacer que la aplicación web o el sitio web funcionen de forma nativa en el modo estándar Windows Internet Explorer 8 escribiendo el código en los estándares web más recientes.

Para evaluar completamente los problemas de la aplicación o el sitio web, debe investigar la causa principal del problema. Por ejemplo, el equipo Internet Explorer determina las causas principales mediante el uso del Herramientas de desarrollo y la reducción lógica de posibles problemas. En las secciones siguientes se describen varias herramientas para ayudar a simplificar el proceso de depuración.



| Sección                                                                                                                    | Descripción                                                                                                                                                                         |
|----------------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Microsoft Expression Web SuperPreview](microsoft-expression-web-superpreview.md)                                         | Una herramienta de depuración visual independiente que hace que sea más rápido y fácil migrar los sitios de Microsoft Internet Explorer 6 a Windows Internet Explorer 7 o Internet Explorer 8. |
| [Herramienta de prueba de compatibilidad de Internet Explorer (IECTT)](inernet-explorer-compatibility-test-tool--iectt-.md)                  | Componente de la compatibilidad [de aplicaciones de Microsoft Toolkit](/windows-hardware/get-started/adk-install) que puede ayudarle a diagnosticar problemas en las aplicaciones web.       |
| [Herramientas para desarrolladores de Internet Explorer 8](internet-explorer-8-developer-tools.md)                                             | Las herramientas de desarrollo que se incluyen en Internet Explorer 8 para la depuración.                                                                                                         |
| [Internet Explorer Asistente para compatibilidad de 8 (Aomorno)](inernet-explorer-8-compatibility-wizard--aggiorno-.md)              | Una herramienta de terceros que actualiza automáticamente los sitios web para que se represente correctamente en Internet Explorer 8 sin que se puedan infringir las compatibilidades en exploradores anteriores.                                |
| [Herramienta Depurador web de Fiddler](fiddler-web-debugger-tool.md)                                                                 | Herramienta de terceros para capturar el tráfico de red entre Internet y los equipos de prueba.                                                                                           |
| [Prueba de compatibilidad de aplicaciones utilizando imágenes de Virtual PC](application-compatibility-testing-using-virtual-pc-images.md) | Imágenes de PC virtual que simplifican las pruebas de compatibilidad.                                                                                                                              |
