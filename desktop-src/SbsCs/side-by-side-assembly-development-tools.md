# <a name="side-by-side-assembly-development-tools"></a>Herramientas de desarrollo de ensamblados en paralelo

Las siguientes herramientas solo están disponibles en el kit de desarrollo de software (SDK) de Microsoft Windows.



| Utilidad                                | Descripción                                                   |
|----------------------------------------|---------------------------------------------------------------|
| [Manifestchk.vbs](manifestchk-vbs.md) | Una herramienta de validación de manifiestos.                                   |
| [Mt.exe](mt-exe.md)                   | Herramienta de generación de manifiestos que crea archivos y catálogos firmados. |
| [Pktextract.exe](pktextract-exe.md)   | Extrae los tokens de clave pública de los archivos de certificado.            |
