# <a name="webwizardhostcancel-method"></a>WebWizardHost. Cancel (método)

Simula un clic del botón **Cancelar** .

## <a name="syntax"></a>Sintaxis


```JScript
iRetVal = WebWizardHost.Cancel()
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="remarks"></a>Observaciones

El cliente es responsable de responder a este método con el comportamiento esperado cerrando el asistente.
