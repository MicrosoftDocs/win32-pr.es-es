# <a name="windows-accessibility"></a>Accesibilidad de Windows

En esta sección se incluye documentación para desarrolladores de Windows que diseñan aplicaciones accesibles, desarrolladores de tecnología de asistencia que crean herramientas como lectores de pantalla y lupa, e ingenieros de pruebas de software que crean scripts automatizados para probar Windows aplicaciones.

Use los recursos que se proporcionan aquí para crear aplicaciones Windows accesibles que puedan usar tantas personas como sea posible, incluidas aquellas con discapacidades, preferencias personales y estilos de trabajo específicos.
