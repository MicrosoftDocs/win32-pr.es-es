# <a name="appearstonotsupporttabbing"></a>AppearsToNotSupportTabbing

## <a name="text"></a>Texto

Parece que la aplicación no admite la tabulación.

## <a name="type"></a>Tipo

Error

## <a name="description"></a>Descripción

La aplicación no admite la navegación con teclado estándar (Tab o Mayús+Tab).

Este problema provoca problemas de facilidad de uso para las personas que dependen de un lector de pantalla o del teclado para la navegación porque ninguno de los elementos de la aplicación es reconocible.

## <a name="possible-causes"></a>Causas posibles

-   El marco de interfaz de usuario de la aplicación no admite MSAA.
-   MSAA no se implementa en ningún control de la aplicación.
