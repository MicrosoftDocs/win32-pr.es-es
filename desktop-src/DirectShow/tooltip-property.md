# <a name="tooltip-property"></a>ToolTip (propiedad)

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

La `ToolTip` propiedad establece el texto de la información sobre herramientas que aparece cuando el puntero del mouse se encuentra sobre el rectángulo de vídeo **MSWebDVD** .

``` syntax
[ sToolTip = ] MSWebDVD.ToolTip
```

## <a name="return-value"></a>Valor devuelto

Devuelve el texto de información sobre herramientas como una cadena.

## <a name="remarks"></a>Observaciones

Esta propiedad es de lectura/escritura y no tiene ningún valor predeterminado. Si la cadena está vacía, no aparece información sobre herramientas.
