# <a name="__indicationrelated-class"></a>\_\_IndicationRelated (clase)

La **\_ \_ clase abstracta del** sistema IndicationRelated actúa como clase primaria para todas las clases relacionadas con eventos. La clase no contiene ninguna propiedad.

La siguiente sintaxis se simplifica desde el código de Managed Object Format (MOF) e incluye todas las propiedades heredadas. Las propiedades se enumeran en orden alfabético, no en orden MOF.

## <a name="syntax"></a>Sintaxis

``` syntax
[abstract]
class __IndicationRelated : __SystemClass
{
};
```

## <a name="members"></a>Miembros

La **\_ \_ clase IndicationRelated** no define ningún miembro.

## <a name="remarks"></a>Comentarios

La **\_ \_ clase IndicationRelated** se deriva de [**\_ \_ SystemClass**](--systemclass.md), que no tiene propiedades.
