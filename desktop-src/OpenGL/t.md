# <a name="t-opengl"></a>T (OpenGL)

[A](a.md) [B](b.md) [C](c.md) [D](d.md) [E](e.md) [F](f.md) [G](g.md) [H](h.md) [I](i.md) [J K](jk.md) [L](l.md) [M](m.md) [N](n.md) [o](o.md) [p](p.md) [Q](q.md) [R](r.md) [s](s.md) T [U V](u-v.md) [W](w.md) [X Y Z](x-y-z.md)

<dl> <dt>

<span id="opengl_tessellation"></span><span id="OPENGL_TESSELLATION"></span>**tesela**
</dt> <dd>

Reducción de una parte de una superficie analítica en una malla de polígonos o de una parte de una curva analítica en una secuencia de líneas.

</dd> <dt>

<span id="opengl_texel"></span><span id="OPENGL_TEXEL"></span>**textura**
</dt> <dd>

Un elemento Texture. Un textura se obtiene de la memoria de textura y representa el color de la textura que se va a aplicar al fragmento correspondiente. Vea también [Fragment](f.md).

</dd> <dt>

<span id="opengl_texture"></span><span id="OPENGL_TEXTURE"></span>**degrada**
</dt> <dd>

Imagen de una o dos dimensiones utilizada para modificar el color de los fragmentos generados por la rasterización. Vea también [rasterizar](r.md).

</dd> <dt>

<span id="opengl_texture_mapping"></span><span id="OPENGL_TEXTURE_MAPPING"></span>**asignación de textura**
</dt> <dd>

Proceso de aplicar una imagen (la textura) a una primitiva. La asignación de texturas se usa a menudo para agregar realismo a una escena. Por ejemplo, puede aplicar una imagen de una fachada de edificio a un polígono que representa una pared. Vea también textura.

</dd> <dt>

<span id="opengl_texture_matrix"></span><span id="OPENGL_TEXTURE_MATRIX"></span>**matriz de textura**
</dt> <dd>

Matriz de 4x4 que transforma las coordenadas de textura de las coordenadas en las que se especifican en las coordenadas que se utilizan para la interpolación y la búsqueda de textura.

</dd> <dt>

<span id="opengl_transformation"></span><span id="OPENGL_TRANSFORMATION"></span>**transformación**
</dt> <dd>

Un salto de espacio. En OpenGL, las transformaciones se limitan a las transformaciones proyectadas que incluyen todo lo que se puede representar mediante una matriz 4x4. Dichas transformaciones incluyen rotaciones, traducciones, escalas (no uniformes) a lo largo de los ejes de coordenadas, transformaciones de perspectiva y combinaciones de estas.

</dd> <dt>

<span id="opengl_triangle"></span><span id="OPENGL_TRIANGLE"></span>**ubicado**
</dt> <dd>

Polígono con tres bordes. Los triángulos siempre son convexo.

</dd> </dl>
