# <a name="terminal-services-is-now-remote-desktop-services"></a>Terminal Services ya está Servicios de Escritorio remoto

Se ha cambiado el nombre de Terminal Services Servicios de Escritorio remoto.

En Windows Server 2008 R2, se ha cambiado el nombre de todos los Servicios de Escritorio remoto de rol. En la tabla siguiente se enumeran tanto el nombre anterior como el nuevo nombre de cada Servicios de Escritorio remoto servicio de rol.



| Nombre anterior                                                   | Nombre en Windows Server 2008 R2                                         |
|-----------------------------------------------------------------|------------------------------------------------------------------------|
| Terminal Services <br/>                                   | Servicios de Escritorio remoto<br/>                                     |
| Terminal Server<br/>                                      | Host de sesión de Escritorio remoto (Host de sesión de RD)<br/>               |
| Licencias de Terminal Services (licencias de TS)<br/>          | Administrador de licencias de Escritorio remoto (Administrador de licencias de RD)<br/>                     |
| Puerta de enlace de Terminal Services (Puerta de enlace de TS)<br/>               | Puerta de enlace de Escritorio remoto (Puerta de enlace de RD)<br/>                         |
| Agente de sesiones de Terminal Services (Agente de sesiones de TS)<br/> | Conexión a Escritorio remoto Broker (Agente de conexión a Escritorio remoto)<br/>     |
| Terminal Services Web Access (TS Web Access)<br/>         | Escritorio remoto Web Access (Acceso web de Escritorio remoto)<br/>                  |
| Virtualización de Terminal Services<br/>                     | Host de virtualización de Escritorio remoto (Host de virtualización de RD)<br/> |



 

Además, tenga en cuenta que un servidor que tiene instalado el servicio de rol Host de sesión de Escritorio remoto ahora se denomina servidor host de sesión de Escritorio remoto, en lugar de un servidor terminal.
