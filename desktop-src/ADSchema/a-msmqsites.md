# <a name="msmq-sites-attribute"></a>MSMQ-Sites atributo)

La lista de sitios a los que pertenece el equipo (una matriz de los sitios de objectGUIDs, normalmente no más de un GUID).



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | MSMQ-Sites                                            |
| Nombre para mostrar de LDAP | mSMQSites                                             |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.927                                |
| System-ID-GUID    | 9a0dc32a-c100-11d1-bbc5-0080c76670c0                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
