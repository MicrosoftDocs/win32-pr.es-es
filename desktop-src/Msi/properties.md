# <a name="properties-windows-installer"></a>Propiedades (Windows instalador)

Las propiedades son variables globales que Windows Installer usa durante una instalación. En las secciones siguientes se describen las propiedades usadas por el instalador:

-   [Acerca de las propiedades](about-properties.md)
-   [Utilizar propiedades](using-properties.md)
-   [Referencia de propiedad](property-reference.md)

El término "propiedad" también hace referencia a un atributo de un objeto de automatización. Vea [Interfaz de Automation](automation-interface.md).
