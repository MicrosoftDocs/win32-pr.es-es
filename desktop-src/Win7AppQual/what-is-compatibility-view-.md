# <a name="what-is-compatibility-view"></a>¿Qué es la vista de compatibilidad?

La *vista de compatibilidad* es una característica de Windows Internet Explorer 8 que permite al explorador representar una página web de forma casi idéntica a la manera en que Windows Internet Explorer 7 la representaría.

En Internet Explorer 8, la vista de compatibilidad cambia el modo en que el explorador interpreta el código escrito en CSS, HTML y el Document Object Model (DOM) para intentar coincidir con Internet Explorer 7. Un sitio que un usuario ve en la vista de compatibilidad de Internet Explorer 8 es casi idéntico a un sitio que el usuario ve en Internet Explorer 7. Sin embargo, la vista de compatibilidad no cambia el modo en que el explorador interpreta todo el código. Por ejemplo, los cambios en Internet Explorer 8 para el modo en que el explorador controla ActiveX, el analizador, AJAX, JavaScript, las redes y la seguridad pueden seguir provocando problemas de compatibilidad. La vista de compatibilidad no cambia estos comportamientos.

En un entorno empresarial, algunas áreas tienen un riesgo menor de problemas de compatibilidad. Por ejemplo, los sitios web de la zona de intranet utilizan la Vista de compatibilidad de forma predeterminada. Las aplicaciones Web cliente que se representan mediante el control de explorador Web o el WebOC (motor de representación de Internet Explorer) también tienen un bajo riesgo de problemas de compatibilidad porque Internet Explorer 8 tiene como valor predeterminado un modo de compatibilidad para WebOC. Sin embargo, es posible que la configuración predeterminada de la vista de compatibilidad no garantice una compatibilidad completa. Para determinar si un sitio web o una aplicación web es compatible con Internet Explorer 8, debe probar el sitio web o la aplicación Web.

Para obtener más información acerca de las diferencias entre la vista de compatibilidad de Internet Explorer 8 e Internet Explorer 7, consulte el [blog compatibilidad de sitios e Internet Explorer 8](/archive/blogs/ie/site-compatibility-and-ie8). Para obtener una lista de lo que debe comprobar al actualizar a Internet Explorer 8, consulte el [Kit de herramientas de preparación de Internet Explorer 8](https://www.microsoft.com/windows/internet-explorer/readiness/developers.aspx).

Para obtener más información acerca de la vista de compatibilidad, consulte el [blog del equipo de Internet Explorer](/archive/blogs/ie/).
