# <a name="iwindowsdriverupdateentry-properties"></a>Propiedades de IWindowsDriverUpdateEntry

La interfaz [**IWindowsDriverUpdateEntry**](/windows/desktop/api/Wuapi/nn-wuapi-iwindowsdriverupdateentry) define las siguientes propiedades.



| Propiedad                                                                     | Descripción                                                                                                          |
|------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**DeviceProblemNumber**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_deviceproblemnumber) | Obtiene el número de problema del dispositivo que coincide con la actualización del controlador de Windows.                                    |
| [**DeviceStatus**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_devicestatus)               | Obtiene el estado del dispositivo que coincide con la actualización del controlador de Windows.                                            |
| [**DriverClass**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverclass)                 | Obtiene la clase de la actualización del controlador de Windows.                                                                         |
| [**DriverHardwareID**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverhardwareid)       | Obtiene el hardware o el identificador compatible que debe coincidir con la actualización del controlador de Windows para poder instalarse. |
| [**DriverManufacturer**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermanufacturer)   | Obtiene el nombre invariable de idioma del fabricante de la actualización del controlador de Windows.                                   |
| [**DriverModel**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_drivermodel)                 | Obtiene el nombre de modelo invariable de idioma del dispositivo para el que está prevista la actualización del controlador de Windows.                |
| [**DriverProvider**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driverprovider)           | Obtiene el nombre de idioma invariable del proveedor de la actualización del controlador de Windows.                                       |
| [**DriverVerDate**](/windows/desktop/api/Wuapi/nf-wuapi-iwindowsdriverupdateentry-get_driververdate)             | Obtiene la fecha de la versión del controlador de la actualización del controlador de Windows.                                                           |
