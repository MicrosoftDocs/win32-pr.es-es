# <a name="taskfoldername-property"></a>Propiedad TaskFolder.Name

En el caso de scripting, obtiene el nombre que se usa para identificar la carpeta que contiene una tarea.

## <a name="syntax"></a>Sintaxis


```VB
TaskFolder.Name As String
```



## <a name="property-value"></a>Valor de propiedad

Nombre que se usa para identificar la carpeta.
