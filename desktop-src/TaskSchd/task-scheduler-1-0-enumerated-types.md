# <a name="task-scheduler-10-enumerated-types"></a>Tipos enumerados de Programador de tareas 1,0

\[\[Es posible que esta API se modifique o no esté disponible en versiones posteriores del sistema operativo o del producto. En su lugar, use las [Interfaces Programador de tareas 2,0](task-scheduler-2-0-interfaces.md) o los [tipos enumerados de programador de tareas 2,0](task-scheduler-2-0-enumerated-types.md) . \]\]

Describe los tipos enumerados que definen las constantes utilizadas por las API de Programador de tareas 1,0.


Los siguientes tipos enumerados se introducen en Programador de tareas 1,0.



| Nombre                                             | Descripción                                                                        |
|--------------------------------------------------|------------------------------------------------------------------------------------|
| [**\_tipo de desencadenador de tarea \_**](/windows/desktop/api/Mstask/ne-mstask-task_trigger_type) | Define los tipos de [*desencadenadores*](t.md) asociados a una tarea. |
| [**TASKPAGE**](/windows/desktop/api/Mstask/ne-mstask-taskpage)                     | Define el tipo de página de tareas que se va a recuperar.                                     |
