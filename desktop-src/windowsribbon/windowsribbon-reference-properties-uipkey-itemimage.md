# <a name="ui_pkey_itemimage"></a>UI \_ PKEY \_ ItemImage

Identifica la propiedad ItemImage de PKEY de la interfaz de usuario \_ \_ .

```
propertyDescription
   name = UI_PKEY_ItemImage
   shellPKey = UI_PKEY_ItemImage
   formatID = 00000106-7363-696e-8441798acf5aebb7
   propID = 106
   typeInfo
      type = IUIImage
```

## <a name="remarks"></a>Observaciones

\_ \_ Una aplicación usa la interfaz de usuario PKEY ItemImage para consultar la imagen asociada a un elemento o un comando en un control de la galería.

El valor de propiedad es un objeto [**IUIImage**](/windows/desktop/api/uiribbon/nn-uiribbon-iuiimage) .
