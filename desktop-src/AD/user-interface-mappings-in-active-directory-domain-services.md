# <a name="user-interface-mappings-in-active-directory-domain-services"></a>Asignaciones de la interfaz de usuario en Active Directory Domain Services

La interfaz de usuario de Active Directory Domain Services incluye hojas de propiedades y otras interfaces de usuario para establecer y recuperar los atributos de las clases usadas habitualmente. Los temas de esta sección contienen tablas que asignan etiquetas que aparecen en las interfaces de usuario con los atributos correspondientes en el esquema de Active Directory.

-   [Asignaciones para el complemento usuarios y equipos de Active Directory](mappings-for-the-active-directory-users-and-computers-snap-in.md)
