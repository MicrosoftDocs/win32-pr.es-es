# <a name="directshow-editing-services"></a>Servicios de edición de DirectShow

\[Esta API no se admite y puede modificarse o no estar disponible en el futuro.\]

Los servicios de edición de DirectShow (DES) son una interfaz de programación de aplicaciones (API) que simplifica considerablemente las tareas implicadas en la edición de vídeo. DES se basa en la arquitectura de DirectShow principal. Abstrae gran parte de la complejidad de DirectShow y proporciona un conjunto de interfaces diseñadas específicamente para manipular proyectos de edición de vídeo. Como desarrollador de aplicaciones, obtendrá las ventajas de DirectShow dentro de un marco de trabajo mucho más adecuado para crear aplicaciones de edición de vídeo.
