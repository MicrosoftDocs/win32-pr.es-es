# <a name="builtin-creation-time-attribute"></a>Atributo Builtin-Creation-time

El atributo **Builtin-Creation-time** se usa para admitir la replicación en los dominios de Windows NT 4,0.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Integrado en tiempo de creación                |
| Nombre para mostrar de LDAP | builtinCreationTime                  |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.13                |
| System-ID-GUID    | bf96792f-0de6-11d0-a285-00aa003049e2 |
| Sintaxis            | [**Interval**](s-interval.md)       |
