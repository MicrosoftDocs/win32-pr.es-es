# <a name="windows-mixed-reality-com-interfaces"></a>Windows Mixed Reality Interfaces COM

Los temas contenidos en esta sección proporcionan las especificaciones de referencia para Windows Mixed Reality interfaces COM.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|-|-|
| [**ISpatialInteractionManagerInterop**](/windows/desktop/api/spatialinteractionmanagerinterop/nn-spatialinteractionmanagerinterop-ispatialinteractionmanagerinterop) | Habilita la interoperabilidad con un objeto [SpatialInteractionManager](/uwp/api/Windows.UI.Input.Spatial.SpatialInteractionManager) de la Plataforma universal de Windows (UWP) y proporciona acceso a los miembros **de SpatialInteractionManager** para acceder a la entrada del usuario desde las manos, los controladores de movimiento y los comandos de voz del sistema. |
| [**IHolographicSpaceInterop**](/windows/desktop/api/holographicspaceinterop/nn-holographicspaceinterop-iholographicspaceinterop) | Habilita la interoperabilidad con un objeto [HolographicSpace](/uwp/api/windows.graphics.holographic.holographicspace) de la Plataforma universal de Windows (UWP) y proporciona acceso a los miembros **de HolographicSpace** para representar una escena holográfica. |
