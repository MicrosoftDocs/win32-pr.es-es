# <a name="about-mci"></a>Acerca de MCI

MCI proporciona a las aplicaciones funcionalidades independientes del dispositivo para controlar periféricos visuales y de audio. La aplicación puede usar MCI para controlar cualquier dispositivo multimedia compatible, incluidos dispositivos de audio de forma de onda, secuenciadores MIDI, dispositivos de audio CD y dispositivos de vídeo digital (reproducción de vídeo).
