# <a name="the-performance-monitoring-provider"></a>Proveedor de supervisión de rendimiento

Los datos de las bibliotecas de rendimiento están disponibles como clases de contador de rendimiento [wmi](/windows/desktop/CIMWin32Prov/performance-counter-classes) a través del proveedor [WMIPerfClass](wmiperfclass-provider.md) y [el proveedor WMIPerfInst](wmiperfinst-provider.md).
