# <a name="application-compatibility-testing-using-virtual-pc-images"></a>Prueba de compatibilidad de aplicaciones utilizando imágenes de Virtual PC

Para empezar a probar la compatibilidad de aplicaciones Web, debe establecer un entorno de pruebas. El proceso de creación y configuración de equipos para esta tarea lleva mucho tiempo y a menudo es costoso.

Para facilitar las pruebas de compatibilidad, puede usar las imágenes de Virtual PC (VPC) de compatibilidad de aplicaciones precompiladas. Las imágenes de VPC son imágenes de unidad de disco duro preconfiguradas que se pueden cargar en equipos existentes que ejecutan el [software de Virtual PC](https://support.microsoft.com/help/14223/windows-xp-end-of-support). Estas imágenes permiten probar exhaustivamente varias versiones de Microsoft Windows y Windows Internet Explorer con las aplicaciones existentes en un breve período de tiempo.

Las imágenes de VPC están disponibles para todas las versiones de Windows admitidas actualmente y contienen muchos recursos y aplicaciones de pruebas de compatibilidad. Para buscar las imágenes de prueba, consulte [imagen de VPC de compatibilidad de aplicaciones de Internet Explorer](https://www.microsoft.com/Downloads/details.aspx?FamilyID=21eabb90-958f-4b64-b5f1-73d0a413c8ef&amp;displaylang=en).
