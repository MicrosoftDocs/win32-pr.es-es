# <a name="extensions"></a>Extensiones

Las extensiones del conjunto de propiedades, tal y como se definen en el formato del conjunto de propiedades COM original, se han quitado y no se admiten, excepto las de [los conjuntos de propiedades DocumentSummaryInformation y UserDefined](the-documentsummaryinformation-and-userdefined-property-sets.md).
