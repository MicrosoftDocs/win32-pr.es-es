# <a name="getdistinguishedname-method-of-the-microsoftdns_roothints-class"></a>Método GetDistinguishedName de la \_ clase MicrosoftDNS RootHints

El método **GetDistinguishedName** recupera el nombre distintivo de la zona.

## <a name="syntax"></a>Sintaxis


```mof
string GetDistinguishedName();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve el nombre de la zona.
