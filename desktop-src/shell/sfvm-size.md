# <a name="sfvm_size-message"></a>Mensaje de tamaño de SFVM \_

Notifica al objeto de devolución de llamada que se ha cambiado el tamaño de la vista de carpetas. Usado por [**IShellFolderViewCB:: MessageSFVCB**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb).


```C++
SFVM_SIZE
```



## <a name="parameters"></a>Parámetros

Este mensaje no tiene parámetros.
