# <a name="winsnmp-sessions"></a>Sesiones WinSNMP

Una *sesión* es la unidad básica de administración de recursos y comunicaciones entre una aplicación que llama a WinSNMP y la implementación de Microsoft WinSNMP. Se recomienda que una aplicación use sesiones para organizar sus operaciones y reduzca la demanda de los recursos de la implementación.

Para obtener más información, vea [abrir y cerrar una sesión WinSNMP](opening-and-closing-a-winsnmp-session.md).
