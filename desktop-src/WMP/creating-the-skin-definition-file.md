# <a name="creating-the-skin-definition-file"></a>Creación del archivo de definición de máscara

El archivo de definición de máscara es un archivo de texto simple. Puede crear archivos de definición de máscara en cualquier editor de texto que guarde archivos de texto sin formato ASCII, como Microsoft Bloc de notas, Microsoft Visual InterDev® o Microsoft Visual Studio. Puesto que va a escribir XML, no HTML, es posible que no quiera usar un editor para HTML, ya que puede agregar códigos adicionales que no desee o puede que le informe sobre errores que no tiene. Probablemente querrá evitar procesadores de palabras como Microsoft Word; Aunque Word puede guardar archivos de texto sin formato, es posible que no quiera recordar guardarlos como texto cada vez que guarde.

Las secciones siguientes le guiarán en la creación de un archivo de definición de máscara:

-   [Estructura XML del archivo de definición de máscara](skin-definition-file-xml-structure.md)
-   [Comience con THEME y VIEW](start-with-theme-and-view.md)
-   [Agregar BUTTONGROUP](adding-buttongroup.md)
-   [Adición de Play BUTTONELEMENT](adding-the-play-buttonelement.md)
-   [Adición de Close BUTTONELEMENT](adding-the-close-buttonelement.md)
-   [Código completo para la máscara simple](complete-code-for-simple-skin.md)
