# <a name="locale_icurrency"></a>configuración regional \_ ICURRENCY

Posición del símbolo monetario en el modo de moneda positiva.



| Value | Significado                                            |
|-------|----------------------------------------------------|
| 0     | Prefijo, sin separación, por ejemplo, $1,1           |
| 1     | Sufijo, sin separación, por ejemplo, $1,1           |
| 2     | Prefijo, separación de 1 carácter, por ejemplo, $1,1 |
| 3     | Sufijo, separación de 1 carácter, por ejemplo, $1,1 |
