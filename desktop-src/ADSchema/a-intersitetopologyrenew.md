# <a name="inter-site-topology-renew-attribute"></a>Inter-site-Topology: renovar atributo

Esta clase indica la frecuencia con la que el generador de topología entre sitios actualiza el mensaje Keep-Alive que se envía a los controladores de dominio contenidos en el mismo sitio.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Entre sitios-topología: renovación            |
| Nombre para mostrar de LDAP | interSiteTopologyRenew               |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1247              |
| System-ID-GUID    | b7c69e5f-2cc7-11d2-854e-00a0c983f608 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
