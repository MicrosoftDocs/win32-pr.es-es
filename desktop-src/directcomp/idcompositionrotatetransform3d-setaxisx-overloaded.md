# <a name="idcompositionrotatetransform3dsetaxisx-methods"></a>IDCompositionRotateTransform3D:: SetAxisX (métodos)

Cambia o anima el valor de la propiedad AxisX de una transformación de rotación 3D. La propiedad AxisX especifica la coordenada x del vector de eje de rotación. El valor predeterminado es cero.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                                       | Descripción                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisX (float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(float))                                     | Cambia el valor de la propiedad AxisX.<br/>  |
| [**SetAxisX (IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisx(idcompositionanimation)) | Anima el valor de la propiedad AxisX.<br/> |
