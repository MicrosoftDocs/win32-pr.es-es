Representa la información de resumen sobre un evento.

## <a name="syntax"></a>Sintaxis


```C++
} SummaryItem;
```

## <a name="members"></a>Miembros

**Nombre**  
Cadena COM que contiene el nombre del evento resumido.

**Valor**  
Cadena COM que contiene un valor relacionado con el evento.

**Id**  
Id. del evento.

**ParentID**  
IDENTIFICADOR del evento primario, si existe.
