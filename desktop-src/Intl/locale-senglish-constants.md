# <a name="locale_senglish-constants"></a>Constantes \_ SENGLISH de LOCALE \*

En este tema se definen las constantes SENGLISH de LOCALE \_ \* utilizadas por NLS.



| Value                        | Significado                                                                                                                                                                                                                                                  |
|------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SENGLISHCOUNTRYNAME  | **Windows 7 y versiones posteriores:** Nombre en inglés del país o región, por ejemplo, Alemania para Deutschland. Este nombre siempre está restringido a los caracteres que se pueden asignar al subconjunto ASCII de 127 caracteres. <br/>                                      |
| LOCALE \_ SENGLISHDISPLAYNAME  | **Windows 7 y versiones posteriores:** Nombre para mostrar de la configuración regional en inglés. Normalmente, el nombre para mostrar consta del idioma y el país o región, por ejemplo, alemán (Alemania) para Deutsch (Deutschland). <br/>                                           |
| LOCALE \_ SENGLISHLANGUAGENAME | **Windows 7 y versiones posteriores:** Nombre en inglés del idioma en inglés, por ejemplo, alemán para Deutsch, del Estándar ISO internacional 639. Este nombre siempre está restringido a los caracteres que se pueden asignar al subconjunto ASCII de 127 caracteres. <br/> |
