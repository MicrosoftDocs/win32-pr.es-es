# <a name="d1140-stroke-transform-type-uses-default-dpi"></a>D1140: el tipo de transformación de trazo usa PPP predeterminado

Al ensanchar un ID2D1Geometry con un trazo fijo o fino, Direct2D asumirá un DPI de 96. Al representar la geometría ampliada, el ancho del trazo puede cambiar.






 

## <a name="possible-causes"></a>Causas posibles

Cuando Direct2D dibuja una geometría en un destino, amplía la geometría. Sin embargo, una aplicación también puede construir objetos ID2D1Geometry sin un destino y ampliarlos mediante métodos de ID2D1Geometry como Wide, StrokeContainsPoint y GetWidenedBounds. Durante estas operaciones, una aplicación puede usar D2D1 \_ Stroke \_ transformación \_ Type \_ fino o fijo. Normalmente, estos atributos consideran un PPP pero, en ausencia de un destino, no se especifica ningún DPI. Este mensaje informativo advierte que Direct2D considerará un valor de PPP predeterminado de 96,0.

## <a name="possible-fixes"></a>Posibles correcciones

Este mensaje es meramente informativo y no indica necesariamente un problema con la aplicación.
