# <a name="bitsadmin-tool"></a>Herramienta BITSAdmin

BITSAdmin es una herramienta de línea de comandos que puede usar para crear trabajos de descarga o carga y supervisar su progreso. Para obtener documentación completa y completa de la herramienta y todos sus comandos, consulte los ejemplos [bitsadmin](/windows-server/administration/windows-commands/bitsadmin) y [bitsadmin](/windows-server/administration/windows-commands/bitsadmin-examples) en el centro de administración Windows TI Pro Center.

Por ejemplo, para enumerar todos los trabajos de BITS, puede emitir el comando `bitsadmin /list /verbose` . Esta es una captura de pantalla de la salida de ejemplo para ese comando.
