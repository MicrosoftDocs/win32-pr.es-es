# <a name="mru-source-list"></a>Lista de orígenes MRU

La API de instalación proporciona funciones que almacenan los directorios de origen usados más recientemente (MRU). Esta información se almacena en el sistema del usuario y se puede tener acceso a ella en instalaciones posteriores.

En los temas siguientes se describe la lista de orígenes MRU y cómo usar las funciones de configuración para abrir, modificar, examinar y cerrar:

-   [Acerca de la lista de orígenes MRU](about-the-mru-source-list.md)
-   [Usar la lista de origen de MRU](using-the-mru-source-list.md)
-   [Referencia de lista de orígenes MRU](mru-source-list-reference.md)
