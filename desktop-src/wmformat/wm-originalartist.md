# <a name="wmoriginalartist"></a>WM/OriginalArtist

El atributo **WM/OriginalArtist** contiene el nombre del intérprete que generó originalmente el contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMOriginalArtist

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

El artista original se utiliza normalmente para retomar las canciones. Para estos archivos. Use el nombre del [**autor**](author.md) de la banda que grabó originalmente la canción.
