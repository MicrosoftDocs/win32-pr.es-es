# <a name="sio_multipoint_loopback-command-code-for-wsaioctl"></a>Código de \_ comando SIO MULTIPOINT \_ LOOPBACK para WSAIoctl

Cuando se usan sockets de hoja d en un plano de datos no raíz, es conveniente que el tráfico que se envía se reciba de nuevo \_ en el mismo socket. El código de comando \_ SIO MULTIPOINT LOOPBACK para \_ [**WSAIoctl**](/windows/desktop/api/Winsock2/nf-winsock2-wsaioctl) se usa para habilitar o deshabilitar el bucleback del tráfico multipunto.
