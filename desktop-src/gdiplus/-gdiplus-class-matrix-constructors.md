# <a name="matrixmatrix-constructors"></a>Constructores Matrix.Matrix

En este tema se enumeran los constructores de la [**clase Matrix.**](/windows/desktop/api/gdiplusmatrix/nl-gdiplusmatrix-matrix) Para obtener una lista de clases completa, vea **Matrix Class**.

### <a name="overload-list"></a>Lista de sobrecarga



| Constructor                                                                                          | Descripción                                                                                                                                                                          |
|:-----------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Matrix(Rect&,Point \* )**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(inconstrect__inconstpoint))                | Crea un [**objeto Matrix::Matrix**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(inconstrect__inconstpoint)) basado en un rectángulo y un punto.<br/>                                         |
| [**Matrix(RectF&,PointF \* )**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(inconstrectf__inconstpointf))            | Crea un [**objeto Matrix::Matrix**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(inconstrectf__inconstpointf)) basado en un rectángulo y un punto.<br/>                                       |
| [**Matrix(REAL,REAL,REAL,REAL,REAL,REAL)**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(inreal_inreal_inreal_inreal_inreal_inreal)) | Crea e inicializa un objeto [**Matrix::Matrix**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(inreal_inreal_inreal_inreal_inreal_inreal)) basado en seis números que definen una transformación afín.<br/> |
| [**Matrix()**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(constmatrix_))                                                    | Crea e inicializa un objeto [**Matrix::Matrix**](/windows/win32/api/gdiplusmatrix/nf-gdiplusmatrix-matrix-matrix(constmatrix_)) que representa la matriz de identidad.<br/>                                             |
