# <a name="credentials-management"></a>Administración de credenciales

Los desarrolladores que escriben para Windows pueden usar el API de Administración credenciales, incluidas las funciones de Interfaz de usuario de administración de credenciales (IU) para obtener y administrar información de credenciales, como nombres de usuario y contraseñas. Estas funciones solicitan Windows información de la cuenta que se va a usar en lugar de las credenciales establecidas al iniciar sesión. Estas solicitudes suelen producirse cuando las credenciales de inicio de sesión no tienen permisos requeridos por la aplicación.

Las funciones de la interfaz de usuario de administración de credenciales proporcionan interfaces con la apariencia de Windows interfaz de usuario. Estas funciones incluyen opciones personalizables que agregan información del usuario al almacén de credenciales del usuario.

En los temas siguientes se proporciona más información sobre las credenciales API de Administración:

-   [Tipos de credenciales](kinds-of-credentials.md)
-   [Formatos de nombre de usuario](user-name-formats.md)

Para obtener información sobre las funciones de Credential API de Administración, vea [Referencia de autenticación.](authentication-reference.md)
