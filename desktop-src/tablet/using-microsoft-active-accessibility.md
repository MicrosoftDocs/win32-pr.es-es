# <a name="using-microsoft-active-accessibility"></a>Uso de Microsoft Active Accessibility

El motor de reconocimiento de voz de Tablet PC usa Active Accessibility para recuperar información directamente del objeto y recibir una notificación cuando cambien los objetos. Esta interfaz es compatible con los controles estándar de Microsoft Windows y los controles personalizados y las clases de ventana personalizadas también pueden utilizarla.
