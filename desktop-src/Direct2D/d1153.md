# <a name="d1153-closed-command-lists-cannot-be-set-as-targets"></a>D1153: las listas de comandos cerradas no se pueden establecer como destinos

Se intentó establecer una lista de comandos cerrada en un destino.






 

## <a name="possible-causes"></a>Causas posibles

Una vez que se ha cerrado una lista de comandos, una aplicación no debe intentar establecerla como destino. Direct2D considera que esta llamada a ID2D1DeviceContext:: SetTarget es un error porque el contenido de una lista de comandos no puede cambiar una vez que se ha cerrado. La configuración de una lista de comandos como destino indica una intención de representarla.

## <a name="possible-fixes"></a>Posibles correcciones

Para solucionar este mensaje, no llame a ID2D1DeviceContext:: SetTarget con una lista de comandos que se ha cerrado. Si es necesario usar una lista de comandos con contenido nuevo y diferente, cree una nueva lista de comandos.
