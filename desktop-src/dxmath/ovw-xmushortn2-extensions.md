# <a name="xmushortn2-extensions"></a>Extensiones de XMUSHORTN2

[**XMUSHORTN2**](/windows/desktop/api/DirectXPackedVector/ns-directxpackedvector-xmushortn2) describe un vector 2D para almacenar valores normalizados sin signo como enteros de 16 bits sin signo, (tipo `int16_t` ).

> [!Note]  
> Las características que se enumeran aquí solo están disponibles cuando se desarrollan con C++.
