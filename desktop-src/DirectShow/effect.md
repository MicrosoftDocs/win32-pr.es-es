# <a name="effect"></a>Efecto

> [!Note]  
> \[En desuso. Esta API se puede quitar de las versiones futuras de Windows.\]

 

El objeto de efecto produce un efecto de vídeo o audio. Para crear este objeto, llame al método [**IAMTimeline:: CreateEmptyNode**](iamtimeline-createemptynode.md) .

El objeto Effect expone las siguientes interfaces:

-   [**IAMTimelineEffect**](iamtimelineeffect.md)
-   [**IAMTimelineObj**](iamtimelineobj.md)
-   [**IAMTimelineSplittable**](iamtimelinesplittable.md)
-   **IPropertyPage**
