# <a name="coutputqueuem_bterminate-member"></a>Miembro COutputQueue::m \_ bTerminate

Marca que especifica si el subproceso debe finalizar. Si **es TRUE,** el subproceso finaliza.

## <a name="syntax"></a>Sintaxis


```C++
BOOL m_bTerminate;
```
