# <a name="using-the-winhttp-cc-api"></a>Uso de la API de C/C++ de WinHTTP

Esta sección contiene información específica sobre el uso de la interfaz WinHTTP C/C++.

-   Las [sesiones de WinHTTP](winhttp-sessions-overview.md) describen el flujo básico de una aplicación de WinHTTP, incluido el orden en el que se llama a las funciones.
-   Los [identificadores de HINTERNET en WinHTTP](hinternet-handles-in-winhttp.md) describen los distintos tipos de identificadores, las convenciones de nomenclatura de estos identificadores y su estructura jerárquica.
-   [Localizadores de recursos uniformes (URL) en WinHTTP](uniform-resource-locators--urls--in-winhttp.md) describe cómo WinHTTP usa las direcciones URL y proporciona un ejemplo de código para mostrar el uso de las funciones proporcionadas.
-   La [autenticación en WinHTTP](authentication-in-winhttp.md) describe la compatibilidad de WinHTTP con la autenticación de servidor y proxy de las sesiones http.
-   La [autenticación de Passport en WinHTTP](passport-authentication-in-winhttp.md) describe la compatibilidad de WinHTTP con la autenticación de Passport 1,4.
-   [SSL en WinHTTP](ssl-in-winhttp.md) describe la compatibilidad de WinHTTP para el protocolo capa de sockets seguros (SSL).
-   El [uso de WinHTTP como ensamblado simultáneo](using-winhttp-as-a-side-by-side-assembly.md) muestra un manifiesto de aplicación de ejemplo que se puede usar para vincular a winhttp.
-   El [control de cookies en WinHTTP](cookie-handling-in-winhttp.md) describe varias opciones que las aplicaciones cliente de WinHTTP pueden utilizar para controlar las cookies.
-   [Control de errores en WinHTTP](error-handling-in-winhttp.md) describe cómo establecer las opciones de WinHTTP mediante programación desde la aplicación de C/C++.
-   La [compatibilidad con el AutoProxy WinHTTP](winhttp-autoproxy-support.md) describe las distintas formas en que las funciones de la API WinHTTP notifican errores.
