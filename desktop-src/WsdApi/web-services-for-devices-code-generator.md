# <a name="web-services-on-devices-code-generator"></a>Generador de código de Servicios web en dispositivos

El generador de código Servicios web en dispositivos, WsdCodeGen, es una utilidad de línea de comandos que genera código de programa a partir de una descripción del servicio. La descripción del servicio se almacena en archivos WSDL o XSD. WsdCodeGen crea archivos C++ e IDL (lenguaje de definición de interfaz). Los desarrolladores pueden usar WsdCodeGen para crear aplicaciones WSDAPI sin preocuparse de cómo se serializan y representan los datos en la conexión.

WsdCodeGen se puede usar para generar código para un servicio, un cliente o ambos. La utilidad genera código auxiliar para servicios, código proxy para clientes e interfaz y código de tipo para ambos.

Esta herramienta está disponible en Microsoft Windows Software Development Kit (SDK) y en Windows Driver Kit (WDK). Las herramientas del SDK se encuentran en el directorio siguiente: <Windows SDK Install Folder> \\ Bin.

El SDK Windows incluye código de ejemplo generado por WsdCodeGen. Para obtener más información, vea [Ejemplos de WSDAPI.](wsdapi-samples.md)
