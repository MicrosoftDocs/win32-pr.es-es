# <a name="tabbedtoinfo"></a>TabbedToInfo

## <a name="text"></a>Texto

**Mensaje 1**

Desencadenar evento de enfoque adicional

**Mensaje 2**

Volver al principio

## <a name="type"></a>Tipo

Informativo

## <a name="description"></a>Descripción

**Mensaje 1**

Un solo elemento activó varios eventos de foco cuando se usa la navegación de teclado estándar (TAB o Mayús + Tab). Los eventos falsos se descartan.

**Mensaje 2**

Foco desplazado al elemento inicial en el orden de tabulación. La tabulación terminó en este elemento porque el orden de tabulación aparece circular.
