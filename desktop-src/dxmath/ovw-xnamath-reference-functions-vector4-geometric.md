# <a name="directxmath-library-4d-vector-geometric-functions"></a>Biblioteca de DirectXMath (funciones geométricas vectoriales)

Enumera las funciones geométricas vectoriales 4D.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                 | Descripción                                                                   |
|---------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**XMVector4AngleBetweenNormals**](/windows/win32/api/directxmath/nf-directxmath-xmvector4anglebetweennormals)<br/>       | Calcula el ángulo en radianes entre dos vectores 4D normalizados.<br/>        |
| [**XMVector4AngleBetweenNormalsEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4anglebetweennormalsest)<br/> | Calcula el ángulo en radianes entre dos vectores 4D normalizados.<br/>      |
| [**XMVector4AngleBetweenVectors**](/windows/win32/api/directxmath/nf-directxmath-xmvector4anglebetweenvectors)<br/>       | Calcula el ángulo en radianes entre dos vectores 4D.<br/>                   |
| [**XMVector4ClampLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector4clamplength)<br/>                       | Fija la longitud de un vector 4D a un intervalo determinado.<br/>                 |
| [**XMVector4ClampLengthV**](/windows/win32/api/directxmath/nf-directxmath-xmvector4clamplengthv)<br/>                     | Fija la longitud de un vector 4D a un intervalo determinado.<br/>                 |
| [**XMVector4Cross**](/windows/win32/api/directxmath/nf-directxmath-xmvector4cross)<br/>                                   | Calcula el producto 4D cruzado.<br/>                                     |
| [**XMVector4Dot**](/windows/win32/api/directxmath/nf-directxmath-xmvector4dot)<br/>                                       | Calcula el producto escalar entre los vectores 4D.<br/>                       |
| [**XMVector4InBounds**](/windows/win32/api/directxmath/nf-directxmath-xmvector4inbounds)<br/>                             | Comprueba si los componentes de un vector 4D están dentro de los límites establecidos.<br/> |
| [**XMVector4Length**](/windows/win32/api/directxmath/nf-directxmath-xmvector4length)<br/>                                 | Calcula la longitud de un vector 4D.<br/>                                |
| [**XMVector4LengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lengthest)<br/>                           | Calcula la longitud de un vector 4D.<br/>                               |
| [**XMVector4LengthSq**](/windows/win32/api/directxmath/nf-directxmath-xmvector4lengthsq)<br/>                             | Calcula el cuadrado de la longitud de un vector 4D.<br/>                  |
| [**XMVector4Normalize**](/windows/win32/api/directxmath/nf-directxmath-xmvector4normalize)<br/>                           | Devuelve la versión normalizada de un vector 4D.<br/>                     |
| [**XMVector4NormalizeEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4normalizeest)<br/>                     | Calcula la versión normalizada de un vector 4D.<br/>                   |
| [**XMVector4Orthogonal**](/windows/win32/api/directxmath/nf-directxmath-xmvector4orthogonal)<br/>                         | Calcula un vector perpendicular a un vector 4D.<br/>                    |
| [**XMVector4ReciprocalLength**](/windows/win32/api/directxmath/nf-directxmath-xmvector4reciprocallength)<br/>             | Calcula el recíproco de la longitud de un vector 4D.<br/>              |
| [**XMVector4ReciprocalLengthEst**](/windows/win32/api/directxmath/nf-directxmath-xmvector4reciprocallengthest)<br/>       | Calcula el recíproco de la longitud de un vector 4D.<br/>             |
| [**XMVector4Reflect**](/windows/win32/api/directxmath/nf-directxmath-xmvector4reflect)<br/>                               | Refleja un vector 4D de incidente en un vector 4D normal.<br/>          |
| [**XMVector4Refract**](/windows/win32/api/directxmath/nf-directxmath-xmvector4refract)<br/>                               | Refracts un vector 4D de incidente en un vector 4D normal.<br/>          |
| [**XMVector4RefractV**](/windows/win32/api/directxmath/nf-directxmath-xmvector4refractv)<br/>                             | Refracts un vector 4D de incidente en un vector 4D normal.<br/>          |
