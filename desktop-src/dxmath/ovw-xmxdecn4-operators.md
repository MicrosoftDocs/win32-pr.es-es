# <a name="xmxdecn4-operators"></a>Operadores XMXDECN4

La estructura [**XMXDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                             | Descripción                                                                                                            |
|---------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**operador UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmxdecn4-operatoruint32_t)<br/> | Devuelve una instancia de `uint32_t` que contiene los componentes de la `XMXDECN4` instancia en un formato empaquetado. <br/> |
| [**operador =**](xmxdecn4-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMXDECN4` . <br/>                                       |
