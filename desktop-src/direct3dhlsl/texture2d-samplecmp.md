# <a name="texture2dsamplecmp-methods"></a>Texture2D:: SampleCmp (métodos)

| Método                                                                                     | Descripción                                                                                                                                                                           |
|:-------------------------------------------------------------------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SampleCmp (S, Float, Float, int)**](dx-graphics-hlsl-to-samplecmp.md)                      | Muestrea una textura con un valor de comparación para rechazar ejemplos.<br/>                                                                                                             |
| [**SampleCmp (S, Float, Float, int, float)**](samplecmp-s-float-float-int-float-.md)           | Muestrea una textura, utilizando un valor de comparación para rechazar muestras, con un valor opcional para Clamp valores de nivel de detalle (LOD) de ejemplo en.<br/>                                     |
| [**SampleCmp (S, Float, Float, int, Float, uint)**](samplecmp-s-float-float-int-float-uint-.md) | Muestrea una textura, utilizando un valor de comparación para rechazar muestras, con un valor opcional para Clamp valores de nivel de detalle (LOD) de ejemplo en.
