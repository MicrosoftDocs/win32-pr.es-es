# <a name="labeling-owner-drawn-controls"></a><span data-ttu-id="bbbf5-103">Etiquetar controles de Owner-Drawn</span><span class="sxs-lookup"><span data-stu-id="bbbf5-103">Labeling Owner-Drawn Controls</span></span>

<span data-ttu-id="bbbf5-104">Los desarrolladores de aplicaciones deben definir una etiqueta, también conocida como texto de leyenda, para todos los controles dibujados por el propietario, incluso si la etiqueta no está visible en la pantalla.</span><span class="sxs-lookup"><span data-stu-id="bbbf5-104">Application developers should define a label, also known as caption text, for all owner-drawn controls, even if the label is not visible on the screen.</span></span> <span data-ttu-id="bbbf5-105">La etiqueta debe ir inmediatamente antes del control que se va a etiquetar, y debe estar en el siguiente elemento en el orden de tabulación.</span><span class="sxs-lookup"><span data-stu-id="bbbf5-105">The label must come immediately before the control that it is labeling, and must be in the next item in the tab order.</span></span>

<span data-ttu-id="bbbf5-106">Por ejemplo, si define una etiqueta al crear un botón dibujado por el propietario en el que se muestra un gráfico en lugar del texto de título normal, las aplicaciones cliente obtienen la etiqueta y la usan para identificar el control.</span><span class="sxs-lookup"><span data-stu-id="bbbf5-106">For example, if you define a label when creating an owner-drawn button in which a graphic is displayed instead of the normal caption text, client applications get the label and use it to identify the control.</span></span>
