# <a name="component-categories"></a>Categorías del componente

Las categorías de componentes de OLE permiten identificar las capacidades y los requisitos de un componente de software mediante entradas en el Registro. En un escenario en el que es posible que un contenedor no quiera o no pueda admitir un área de funcionalidad, como el enlace de datos, por ejemplo, el contenedor no querrá hospedar controles que requieran enlace de datos para realizar su función de trabajo. Las categorías de componentes permiten identificar áreas de funcionalidad, como el enlace de datos, para que el contenedor de controles pueda evitar los controles que lo den como un requisito. Las categorías de componentes se especifican por separado como parte de OLE y no son específicas de la arquitectura de control ActiveX, la especificación de las categorías de componentes incluye API para la manipulación de las claves del Registro de categorías de componentes.
