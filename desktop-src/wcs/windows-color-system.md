# <a name="windows-color-system"></a>Sistema de colores de Windows

## <a name="purpose"></a>Propósito

Los esquemas de administración de colores se implementan en Microsoft Windows operativos para mejorar la representación del contenido de color en todas las formas de comunicación digital.

El esquema de administración de colores que se usa a partir de Windows 2000, Windows XP y Windows Server 2003 se denomina Administración de color de imagen (ICM) 2.0. El esquema de administración de colores que se usa a partir de Windows Vista se denomina Windows Color System (WCS) 1.0. El esquema de administración de colores de WCS 1.0 es un superconjunto de api ICM API y funcionalidad 2.0.

## <a name="where-applicable"></a>Donde sea aplicable

ICM se puede usar en todas las aplicaciones de Windows 2000 y sistemas operativos posteriores. WCS se puede usar en todas las aplicaciones de Windows Vista y sistemas operativos posteriores.

## <a name="developer-audience"></a>Audiencia de desarrolladores

La API de WCS está diseñada para su uso por programadores de C/C++. Es necesario estar familiarizado con Windows interfaz gráfica de usuario, la arquitectura controlada por mensajes y un conocimiento práctico de los conceptos de administración de colores.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

Las aplicaciones que usan ICM API requieren Windows 2000 o posterior. Las aplicaciones que usan la API de WCS requieren Windows Vista o posterior. Para obtener información específica en tiempo de ejecución sobre una función, consulte la sección Requisitos de la página de referencia de esa función.
