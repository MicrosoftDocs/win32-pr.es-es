# <a name="namespacedefinitions-element"></a>elemento namespaceDefinitions

Genera definiciones de C para tablas de espacio de nombres.

## <a name="usage"></a>Uso

``` syntax
<namespaceDefinitions/>
```

## <a name="attributes"></a>Atributos

No hay atributos.

## <a name="child-elements"></a>Elementos secundarios

No hay elementos secundarios.

## <a name="parent-elements"></a>Elementos primarios



| Elemento                         | Descripción                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**Archivo**](file.md)<br/> | Genera un archivo del generador de código.<br/> <br/> |



## <a name="remarks"></a>Comentarios

Este elemento se usa generalmente en los archivos de código fuente de C para proporcionar las tablas de espacio de nombres declaradas por [**namespaceDeclarations**](namespacedeclarations.md).

## <a name="element-information"></a>Información de elemento



| Etiqueta | Value |
|-------------------------------------|---------------|
| Sistema mínimo compatible<br/> | Windows Vista |
| Puede estar vacío                        | Sí           |
