# <a name="query-schema-elements"></a>Elementos de esquema de consulta

A continuación se muestran los elementos que define el esquema de consulta. Esta sección contiene los nombres de los elementos que usaría en la consulta; sin embargo, para obtener los detalles de cada elemento, vea el tipo complejo que contiene el elemento.



| Elemento                                                                  | Descripción                                                                                                                     |
|--------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**QueryList**](queryschema-querylist-element.md)                       | El nodo raíz de la consulta.<br/>                                                                                          |
| [**Consulta (QueryListType)**](queryschema-query-querylisttype-element.md) | Define un conjunto de selectores y suprimidores que se usan para incluir eventos en o excluir eventos del conjunto de resultados.<br/> |
| [**Select (QueryType)**](queryschema-select-querytype-element.md)       | Consulta XPath que identifica los eventos que se van a incluir en el conjunto de resultados de la consulta.<br/>                                        |
| [**Suprimir (QueryType)**](queryschema-suppress-querytype-element.md)   | Consulta XPath que identifica los eventos que se van a excluir del conjunto de resultados de la consulta.<br/>                                      |
