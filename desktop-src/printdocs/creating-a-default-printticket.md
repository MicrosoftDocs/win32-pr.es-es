# <a name="creating-a-default-printticket"></a>Crear un PrintTicket predeterminado

Este tema no está actualizado. Para obtener la información más reciente, consulte la [especificación del esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

La interfaz del proveedor PrintTicket contiene un método que devuelve un valor de PrintTicket predeterminado. Además, la interfaz del proveedor de PrintCapabilities tiene un método que devuelve un documento de PrintCapabilities predeterminado; es decir, una instantánea de la PrintCapabilities creada mediante el uso de un PrintTicket predeterminado. Este tipo de PrintTicket predeterminado debe generarse automáticamente mediante el siguiente proceso.
