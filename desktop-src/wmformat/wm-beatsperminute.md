# <a name="wmbeatsperminute"></a>WM/BeatsPerMinute

El atributo **WM/BeatsPerMinute** contiene las pulsaciones por minuto del contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMBeatsPerMinute

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

El valor de un atributo **BeatsPerMinute** es un entero almacenado como una cadena.

### <a name="example"></a>Ejemplo



| Tipo de archivo | Valor de ejemplo |
|-----------|---------------|
| Audio     | "120"         |
