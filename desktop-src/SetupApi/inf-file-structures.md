# <a name="inf-file-structures"></a>Estructuras de archivos INF

Las siguientes estructuras se utilizan con archivos INF.



| Estructura                                              | Descripción                        |
|--------------------------------------------------------|------------------------------------|
| [**INFCONTEXT**](/windows/desktop/api/Setupapi/ns-setupapi-infcontext)                   | Contexto posicional en un archivo INF. |
| [**\_información del INF de SP \_**](/windows/desktop/api/Setupapi/ns-setupapi-sp_inf_information) | Información sobre el archivo INF.    |



 

> [!Note]  
> Las funciones de instalación usan internamente la estructura INFCONTEXT y las aplicaciones de instalación no deben modificarlas ni hacer referencia a ellas. Se incluye aquí únicamente con fines informativos.
