# <a name="learn-about-the-eap-and-eaphost-relationship"></a>Más información sobre la relación de EAP y EAPHost

En este tema se describe la relación entre el protocolo de autenticación extensible (EAP) y el host del Protocolo de autenticación extensible. Para el nuevo desarrollo de métodos EAP, vea [host del Protocolo de autenticación extensible](../eaphost/portal.md).

## <a name="eaphost-framework"></a>Marco de EAPHost

EAPHost se considera un marco para EAP por dos motivos.

-   EAPHost es un componente de la infraestructura de Windows que hospeda los complementos EAP
-   EAPHost implementa el protocolo EAP según [RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063). Una autenticación EAP consta del protocolo EAP según [RFC 3748](https://go.microsoft.com/fwlink/p/?linkid=84063)y los aspectos específicos del método EAP del protocolo.
