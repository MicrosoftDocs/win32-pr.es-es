# <a name="ms-dfsr-options-attribute"></a>atributo MS-DFSR-Options

Contiene las opciones del objeto de replicación Sistema de archivos distribuido (DFS).



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Opciones de MS-DFSR                      |
| Nombre para mostrar de LDAP | msDFSR-opciones                       |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.17           |
| System-ID-GUID    | d6d67084-C720-417d-8647-b696237a114c |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
