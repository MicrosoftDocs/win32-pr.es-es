# <a name="hotkey-property"></a>Propiedad HotKey

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Devuelve una cadena que especifica la configuración actual del usuario para la clave de escucha.

</dd> <dt>

<span id="Syntax_"></span><span id="syntax_"></span><span id="SYNTAX_"></span>**Sintáctica** 
</dt> <dd>

*agente * * *. SpeechInput. HotKey**

</dd> </dl>

## <a name="remarks"></a>Observaciones

La propiedad **Hotkey** refleja el valor actual de la clave de escucha en la página entrada de voz de la hoja de propiedades del agente (opciones avanzadas de caracteres). La propiedad es de solo lectura; solo el usuario puede cambiar la configuración.
