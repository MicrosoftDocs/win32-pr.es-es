# <a name="imstscnonscriptablebinarypassword-property"></a>Propiedad IMsTscNonScriptable::BinaryPassword

Esta propiedad ya no está disponible para su uso.

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Syntax


```C++
HRESULT put_BinaryPassword(
  [in]  BSTR newPassword
);

HRESULT get_BinaryPassword(
  [out] BSTR *pBinaryPassword
);
```



## <a name="property-value"></a>Valor de propiedad

La nueva parte de contraseña, en formato binario codificado.

## <a name="error-codes"></a>Códigos de error

Devuelve **E \_ NOTIMPL**.
