# <a name="ms-wmi-id-attribute"></a>atributo MS-WMI-ID

IDENTIFICADOR único de una instancia de objeto. Indizado en el catálogo global.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | IDENTIFICADOR de MS-WMI                                   |
| Nombre para mostrar de LDAP | IDENTIFICADOR de msWMI                                    |
| Tamaño              | \-                                          |
| Actualizar privilegio  | El sistema establece este valor.            |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.1627                     |
| System-ID-GUID    | 9339a803-94b8-47f7-9123-a853b9ff7e45        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
