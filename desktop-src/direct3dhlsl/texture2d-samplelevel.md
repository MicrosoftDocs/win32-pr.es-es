# <a name="texture2dsamplelevel-methods"></a>Métodos Texture2D::SampleLevel

Muestrea [**un objeto Texture2D**](sm5-object-texture2d.md) en el nivel de mapa mip especificado.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                             | Descripción                                                                                        |
|:-----------------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**SampleLevel(S,float,float,int)**](dx-graphics-hlsl-to-samplelevel.md)          | Muestrea una textura en el nivel de mapa mip especificado.<br/>                                        |
| [**SampleLevel(S,float,float,int,uint)**](samplelevel-s-float-float-int-uint-.md) | Muestrea una textura en el nivel de mapa mip especificado y devuelve el estado de la operación.<br/> |
