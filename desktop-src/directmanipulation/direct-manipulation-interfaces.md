# <a name="direct-manipulation-interfaces"></a>Interfaces de manipulación directa

Los temas contenidos en esta sección proporcionan las especificaciones de referencia para las interfaces de [manipulación directa](direct-manipulation-portal.md) .

> [!Note]  
> Al implementar un objeto de [manipulación directa](direct-manipulation-portal.md) , asegúrese de que la implementación de [**IUnknown**](/windows/win32/api/unknwn/nn-unknwn-iunknown) admite el multithreading mediante el recuento de referencias seguro para subprocesos. Para obtener más información, vea [**InterlockedIncrement**](/windows/win32/api/winnt/nf-winnt-interlockedincrement) y [**InterlockedDecrement**](/windows/win32/api/winnt/nf-winnt-interlockeddecrement).

## <a name="in-this-section"></a>En esta sección

| Tema                                                                                                       | Descripción                                                                                                                                                                                                                                                                  |
|-------------------------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IDirectManipulationAutoScrollBehavior**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationautoscrollbehavior)<br/>           | Representa el comportamiento de animación de desplazamiento automático del contenido a medida que se aproxima al límite de un eje o ejes determinados.<br/>                                                                                                                                                   |
| [**IDirectManipulationCompositor**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationcompositor)<br/>                           | Representa un objeto de compositor que asocia contenido manipulado a una superficie de dibujo, como [**Canvas**](/uwp/api/Windows.UI.Xaml.Controls.Canvas).<br/> |
| [**IDirectManipulationCompositor2**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationcompositor2)<br/>                         | Representa un objeto de compositor que asocia el contenido manipulado a las superficies de dibujo a través de varios procesos.<br/>                                                                                                                                               |
| [**IDirectManipulationContent**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationcontent)<br/>                                 | Encapsula el contenido dentro de una ventanilla, donde el contenido representa una superficie visual recortada dentro de la ventanilla.<br/>                                                                                                                                                    |
| [**IDirectManipulationDeferContactService**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationdefercontactservice)<br/>         | Representa un servicio para administrar asociaciones entre un contacto y una ventanilla.<br/>                                                                                                                                                                                  |
| [**IDirectManipulationDragDropBehavior**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationdragdropbehavior)<br/>               | Representa los comportamientos de las interacciones de arrastrar y colocar, que se desencadenan mediante movimientos entre diapositivas u operaciones de presionar y mantener. <br/>                                                                                                                                              |
| [**IDirectManipulationDragDropEventHandler**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationdragdropeventhandler)<br/>       | Define métodos para controlar los eventos de comportamiento de arrastrar y colocar.<br/>                                                                                                                                                                                                              |
| [**IDirectManipulationFrameInfoProvider**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationframeinfoprovider)<br/>             | Representa un objeto de tiempo de conservación que mide la latencia de la infraestructura de composición utilizada por la aplicación y proporciona estos datos para la [manipulación directa](direct-manipulation-portal.md). <br/>                                                            |
| [**IDirectManipulationInteractionEventHandler**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationinteractioneventhandler)<br/> | Define métodos para controlar las interacciones cuando se detectan.<br/>                                                                                                                                                                                                    |
| [**IDirectManipulationManager**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager)<br/>                                 | Proporciona acceso a todas las características de [manipulación directa](direct-manipulation-portal.md) y API disponibles para la aplicación cliente.<br/>                                                                                                                           |
| [**IDirectManipulationManager2**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager2)<br/>                               | Extiende la interfaz [**IDirectManipulationManager**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager) que proporciona acceso a todas las características de [manipulación directa](direct-manipulation-portal.md) y API disponibles para la aplicación cliente. <br/>                              |
| [**IDirectManipulationManager3**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager3)<br/>                               | Extiende la interfaz [**IDirectManipulationManager2**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationmanager2) que proporciona acceso a todas las características de [manipulación directa](direct-manipulation-portal.md) y API disponibles para la aplicación cliente. <br/>                            |
| [**IDirectManipulationPrimaryContent**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationprimarycontent)<br/>                   | Encapsula el contenido principal dentro de una ventanilla.<br/>                                                                                                                                                                                                               |
| [**IDirectManipulationUpdateHandler**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationupdatehandler)<br/>                     | Define métodos para controlar los eventos de actualización de manipulación.<br/>                                                                                                                                                                                                          |
| [**IDirectManipulationUpdateManager**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationupdatemanager)<br/>                     | Administra cómo se envían las actualizaciones del compositor a la [manipulación directa](direct-manipulation-portal.md).<br/>                                                                                                                                                                 |
| [**IDirectManipulationViewport**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationviewport)<br/>                               | Define una región dentro de una ventana (denominada ventanilla) que puede recibir y procesar la entrada de las interacciones del usuario. <br/>                                                                                                                                   |
| [**IDirectManipulationViewport2**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationviewport2)<br/>                             | Proporciona la administración de los comportamientos en una ventanilla. Un comportamiento afecta a la funcionalidad de una parte determinada del flujo de trabajo de [manipulación directa](direct-manipulation-portal.md) . <br/>                                                                                 |
| [**IDirectManipulationViewportEventHandler**](/windows/win32/api/DirectManipulation/nn-directmanipulation-idirectmanipulationviewporteventhandler)<br/>       | Define métodos para controlar el estado y los eventos de actualización de la ventanilla.<br/>                                                                                                                                                                                           |
