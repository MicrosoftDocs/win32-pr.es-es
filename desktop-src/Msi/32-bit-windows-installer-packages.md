# <a name="32-bit-windows-installer-packages"></a>Paquetes de Windows Installer de bits de 32

Un paquete de 32 bits solo consta de [componentes de Windows Installer](windows-installer-components.md) de 32 bits y debe tener el valor "Intel" especificado en el campo plataforma de la propiedad de Resumen de la [**plantilla**](template-summary.md) .

Para obtener más información, consulte [Windows Installer en sistemas operativos de 64](windows-installer-on-64-bit-operating-systems.md) bits y [paquetes de Windows Installer de 64 bits](64-bit-windows-installer-packages.md).
