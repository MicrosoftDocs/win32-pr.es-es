# <a name="typeuri-element"></a>elemento typeUri

Especifica un tipo que se va a incluir desde un archivo XSD.

## <a name="usage"></a>Uso

``` syntax
<typeUri
  type = "character_string"
  uri = "character_string"/>
```

## <a name="attributes"></a>Atributos



| Atributo           | Tipo                         | Obligatorio       | Descripción                                                            |
|---------------------|------------------------------|----------------|------------------------------------------------------------------------|
| **type**<br/> | cadena \_ de caracteres<br/> | Sí<br/> | Nombre del tipo.<br/> <br/>                           |
| **uri**<br/>  | cadena \_ de caracteres<br/> | Sí<br/> | Espacio de nombres del tipo. Debe ser un URI válido.<br/> <br/> |



## <a name="child-elements"></a>Elementos secundarios

No hay elementos secundarios.

## <a name="parent-elements"></a>Elementos primarios



| Elemento                       | Descripción                                                                       |
|-------------------------------|-----------------------------------------------------------------------------------|
| [**Xsd**](xsd.md)<br/> | Especifica un archivo XSD para procesar la información del contrato.<br/> <br/> |
