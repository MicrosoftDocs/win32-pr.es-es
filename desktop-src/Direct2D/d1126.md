# <a name="d1126-alpha-mode-unknown-not-allowed"></a>D1126: no se permite el modo alfa desconocido

\_ \_ \_ No se permite el modo alfa D2D1 desconocido como parámetro de esta API.



|             |         |
|-------------|---------|
| Nivel de error | Advertencia |



 

## <a name="possible-causes"></a>Causas posibles

El \_ especificador de modo alfa \_ desconocido se permite en algunos contextos, como una descripción de formato para ID2D1RenderTarget:: CreateCompatibleRenderTarget. Sin embargo, de forma similar al formato de DXGI \_ \_ desconocido, Direct2D normalmente requiere que los formatos se especifiquen por completo.

## <a name="possible-fixes"></a>Posibles correcciones

Al decidir un especificador de formato, tenga en cuenta el tratamiento de la superficie de alfa para usar un \_ modo alfa que no sea el \_ modo alfa \_ desconocido. Use \_ \_ el modo alfa D2D1 \_ si la superficie no usa alfa. Vea la sección Comentarios en el [**\_ \_ modo alfa de D2D1**](/windows/desktop/api/dcommon/ne-dcommon-d2d1_alpha_mode).
