# <a name="updatestarted-property"></a>Propiedad UpdateStarted

El instalador establece la propiedad **UpdateStarted** cuando se han iniciado cambios en el sistema para esta instalación, incluida la reanudación de una instalación suspendida.

Instrucciones de condición de IU use este valor para seleccionar un cuadro de diálogo. Si se establece esta propiedad, se le preguntará al usuario después de un error y cancelará la restauración o la continuación.
