# <a name="windows-connect-now"></a>Windows Connect Now

## <a name="purpose"></a>Propósito

Windows Connect Now (WCN) permite que los dispositivos móviles e incrustados, los puntos de acceso 802,11 (APs) y los equipos se conecten y intercambien la configuración entre sí de forma segura. WCN está diseñado para el usuario doméstico o de pequeña empresa, lo que proporciona un compromiso razonable entre la facilidad de uso y la seguridad sólida.

## <a name="developer-audience"></a>Audiencia de desarrolladores

La API de Windows Connect Now está diseñada para su uso con C/C++.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

Windows Connect ahora se admite a partir de Windows 7.
