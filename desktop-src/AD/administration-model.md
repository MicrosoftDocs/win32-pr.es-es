# <a name="administration-model"></a>Modelo de administración

Los usuarios autorizados realizan la administración en Active Directory Domain Services. Un usuario está autorizado por una autoridad superior para realizar un conjunto especificado de acciones en un conjunto especificado de objetos y clases de objeto en algunos subárboles identificados del directorio. Esto se denomina *administración delegada*. La administración delegada permite un control muy preciso sobre quién puede hacer y habilitar la delegación de autoridad sin conceder privilegios elevados.
