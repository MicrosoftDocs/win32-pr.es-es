# <a name="windows-security-center"></a>Centro de seguridad de Windows

Seguridad de Windows Center (WSC) es una herramienta de informes completa que ayuda a los usuarios a establecer y mantener una capa de seguridad de protección alrededor de sus sistemas informáticos. Una vez establecida una capa de seguridad, Seguridad de Windows Center no es visible, ya que supervisa el estado de mantenimiento del equipo. Sin embargo, si existen vulnerabilidades, WSC proporciona alertas e instrucciones preceptivas para ayudar al usuario a lograr un estado seguro que se detecta al usuario final a través del Centro de acciones.

Para que las soluciones de seguridad de terceros (antivirus, antimalware o antispyware) sean compatibles con Windows y informen correctamente del estado al Centro de acciones, deben registrarse en Security Center e informar de los cambios de estado posteriores mediante API privadas para comunicarse con WSC. WSC, a su vez, comunica estas actualizaciones al Centro de acciones, donde finalmente se muestran al usuario final.
