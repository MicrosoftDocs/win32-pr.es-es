# <a name="playlistcollectiongetall-method"></a>Método PlaylistCollection.getAll

El **método getAll** recupera un objeto **PlaylistArray** que contiene todas las listas de reproducción de la biblioteca.

## <a name="syntax"></a>Sintaxis


```JScript
retVal = PlaylistCollection.getAll()
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método devuelve un objeto **PlaylistArray.**

## <a name="remarks"></a>Comentarios

Para usar este método, se requiere acceso de lectura a la biblioteca. Para obtener más información, vea [Acceso a la biblioteca](library-access.md).
