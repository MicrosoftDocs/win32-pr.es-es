# <a name="h323-msp"></a>H323 MSP

\[Msp H323 no está disponible para su uso en Windows Vista, Windows Server 2008 y versiones posteriores del sistema operativo. La API de cliente RTC proporciona una funcionalidad similar.\]

El MSP H323 (H323msp.dll), cuando se empareja con su TSP (H323.tsp), permite que una aplicación realice y reciba llamadas de vídeo y audio mediante el protocolo H.323 para telefonía IP.

El MSP de H323 se ha mejorado en Windows 2000 SP1 para habilitar la comunicación entre los clientes H323 y SDP. Ahora expone las interfaces [**IH323LineEx**](ih323lineex.md) [**e IKeyFrameControl.**](ikeyframecontrol.md)

El MSP de H323 se instala con Windows 2000.
