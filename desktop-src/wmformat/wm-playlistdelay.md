# <a name="wmplaylistdelay"></a>WM/PlaylistDelay

El **atributo WM/PlaylistDelay** contiene el número de milisegundos de retraso que debe preceder a la reproducción del archivo en una lista de reproducción.

## <a name="global-constant"></a>Constante global

g \_ wszWMPlaylistDelay

## <a name="data-type"></a>Tipo de datos

**CADENA DE TIPO WMT \_ \_**

## <a name="remarks"></a>Comentarios

No todos los reproductores multimedia respetan el retraso de la lista de reproducción.
