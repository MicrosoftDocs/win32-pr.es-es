# <a name="vml-callout-element"></a>Elemento de llamada de VML

En este tema se describe VML, una característica que está en desuso a partir Windows Internet Explorer 9. Las páginas web y las aplicaciones que se basan en VML deben migrarse a SVG u otros estándares ampliamente admitidos.

> [!Note]  
> A partir de diciembre de 2011, este tema se archivó. Como resultado, ya no se mantiene activamente. Para obtener más información, vea [Contenido archivado.](/previous-versions/windows/internet-explorer/ie-developer/) Para obtener información, recomendaciones e instrucciones sobre la versión actual de Windows Internet Explorer, [vea Internet Explorer Developer Center](https://msdn.microsoft.com/ie/).

 

Define una llamada para una forma.

Los atributos siguientes modifican una llamada.



| Atributo                                                        | Descripción                                                                             |
|------------------------------------------------------------------|-----------------------------------------------------------------------------------------|
| [AccentBar](msdn-online-vml-accentbar-attribute.md)             | Determina si se usará una barra de acentos con la llamada.                         |
| [Ángulo](angle-attribute--callout--vml.md)                       | Define el ángulo que la llamada realiza con respecto al cuadro de límite de la forma. |
| [Distancia](msdn-online-vml-distance-attribute.md)               | Define la distancia de colocación de una llamada.                                                 |
| [Soltar](msdn-online-vml-drop-attribute.md)                       | Determina dónde se colocará la eliminación de una llamada.                                  |
| [DropAuto](msdn-online-vml-dropauto-attribute.md)               | Determina si la llamada tendrá una eliminación automática.                             |
| [Ext](ext-attribute--callout--vml.md)                           | Define cómo se procesa una llamada.                                                     |
| [Brecha](msdn-online-vml-gap-attribute.md)                         | Define la distancia de la llamada desde su rectángulo delimitador.                        |
| [Duración](msdn-online-vml-length-attribute.md)                   | Define la longitud de la llamada.                                                      |
| [LengthSpecified](msdn-online-vml-lengthspecified-attribute.md) | Determina si el **atributo Length** se usará para la llamada.               |
| [MinusX](msdn-online-vml-minusx-attribute.md)                   | Determina si la llamada se invertirá a lo largo del eje X.                              |
| [MinusY](msdn-online-vml-minusy-attribute.md)                   | Determina si la llamada se invertirá a lo largo del eje Y.                              |
| [Activado](on-attribute--callout--vml.md)                             | Determina si una forma es una llamada.                                                |
| [TextBorder](msdn-online-vml-textborder-attribute.md)           | Determina si una llamada tendrá un borde de texto.                                   |
| [Type](type-attribute--callout--vml.md)                         | Define el tipo de llamada.                                                            |



 

**Comentarios:**

Este elemento debe definirse dentro de un [elemento Shape.](shape-element--vml.md)

El **elemento Callout** es una extensión Microsoft Office a VML.
