# <a name="codecapi_avdecvideomaxcodedheight-property"></a>\_Propiedad AVDecVideoMaxCodedHeight de CODECAPI

Establece el alto máximo de la imagen que un descodificador de vídeo aceptará como tipo de entrada.

## <a name="data-type"></a>Tipo de datos

**Long** (**VT \_ I4**)

## <a name="property-guid"></a>GUID de propiedad

CODECAPI \_ AVDecVideoMaxCodedHeight
