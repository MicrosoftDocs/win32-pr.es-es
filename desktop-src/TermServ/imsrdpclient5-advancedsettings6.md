# <a name="imsrdpclient5advancedsettings6-property"></a>IMsRdpClient5:: AdvancedSettings6 (propiedad)

Recupera la interfaz [**IMsRdpClientAdvancedSettings5**](imsrdpclientadvancedsettings5.md) .

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT get_AdvancedSettings6(
  [out] IMsRdpClientAdvancedSettings5 **ppAdvSettings
);
```



## <a name="property-value"></a>Valor de propiedad
