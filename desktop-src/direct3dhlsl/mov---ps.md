# <a name="mov---ps"></a>MOV-PS

Mueve los datos entre los registros.

## <a name="syntax"></a>Sintaxis



| MOV. DST, src |
|--------------|



 

, donde

-   DST es el registro de destino.
-   src es un registro de origen.
