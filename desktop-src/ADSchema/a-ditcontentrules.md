# <a name="dit-content-rules-attribute"></a>Atributo DIT-Content-rules

Esto especifica el contenido permitido de las entradas de una clase de objeto estructural determinada mediante la identificación de un conjunto opcional de clases de objeto auxiliares y atributos obligatorios, opcionales y excluidos. Los atributos colectivos se incluyen en DIT-Content-rules, tal y como se define en RFC 2251.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | DIT-Content-rules                           |
| Nombre para mostrar de LDAP | dITContentRules                             |
| Tamaño              | \-                                          |
| Actualizar privilegio  | El sistema establece este valor.            |
| Frecuencia de actualización  | Siempre que se crea una nueva clase.            |
| Attribute-Id      | 2.5.21.2                                    |
| System-ID-GUID    | 9a7ad946-ca53-11d1-bbd0-0080c76670c0        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
