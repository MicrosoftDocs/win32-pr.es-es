# <a name="bitsadmin-tool"></a>BITSAdmin (herramienta)

BITSAdmin es una herramienta de línea de comandos que puede usar para crear trabajos de descarga o carga y supervisar su progreso. Para obtener documentación completa y completa de la herramienta y de todos sus comandos, consulte los [ejemplos](/windows-server/administration/windows-commands/bitsadmin-examples) de [bitsadmin](/windows-server/administration/windows-commands/bitsadmin) y bitsadmin en el centro de profesionales de TI de Windows.

Por ejemplo, para enumerar todos los trabajos de BITS, puede emitir el comando `bitsadmin /list /verbose` . Esta es una captura de pantalla de la salida de ejemplo para ese comando.
