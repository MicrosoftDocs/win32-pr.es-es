# <a name="methods-of-iocspcaconfigurationcollection"></a>Métodos de IOCSPCAConfigurationCollection

La interfaz [**IOCSPCAConfigurationCollection**](/windows/desktop/api/Certadm/nn-certadm-iocspcaconfigurationcollection) define los métodos siguientes. Los métodos de acceso de propiedad no se muestran aquí. Para ver las propiedades de **IOCSPCAConfigurationCollection,** vea [Propiedades de IOCSPCAConfigurationCollection.](properties-of-iocspcaconfigurationcollection.md)



| Método                                                                                       | Descripción                                                                                                                                                                                    |
|----------------------------------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CreateCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-createcaconfiguration) | Crea una nueva [*configuración de entidad*](../secgloss/c-gly.md) de certificación (CA) y la agrega al conjunto de configuración. |
| [**DeleteCAConfiguration**](/windows/desktop/api/Certadm/nf-certadm-iocspcaconfigurationcollection-deletecaconfiguration) | Quita una configuración de CA con nombre del conjunto de configuración.                                                                                                                                   |
