# <a name="locale_neutral"></a>CONFIGURACIÓN REGIONAL \_ NEUTRA

Configuración regional neutra. Por lo general, esta constante no se usa al llamar a las API de NLS. En su lugar, use [LOCALE \_ SYSTEM DEFAULT \_ o](locale-system-default.md) [LOCALE USER \_ \_ DEFAULT](locale-user-default.md). El valor de LOCALE \_ NEUTRAL es 0x0000.
