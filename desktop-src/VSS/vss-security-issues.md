# <a name="vss-security-issues"></a>Problemas de seguridad de VSS

La infraestructura de VSS se basa internamente en COM y, por lo tanto, permitir que los escritores y los solicitantes se comuniquen requiere seguridad basada en COM. La seguridad COM proporciona mecanismos e instrucciones que debe tener en cuenta un desarrollador de VSS:

-   [Consideraciones de seguridad para escritores](security-considerations-for-writers.md)
-   [Consideraciones de seguridad para los solicitantes](security-considerations-for-requestors.md)
