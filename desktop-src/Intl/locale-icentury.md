# <a name="locale_icentury"></a>configuración regional \_ ICENTURY

Un siglo de 2 o 4 dígitos solo para la fecha corta. El siglo puede tener uno de los valores siguientes. Se prefiere que la aplicación use la [configuración regional \_ SSHORTDATE](locale-sshortdate.md) en lugar de la configuración regional \_ ICENTURY.



| Value | Significado                     |
|-------|-----------------------------|
| 0     | Siglo de 2 dígitos abreviado |
| 1     | Siglo de 4 dígitos completo        |
