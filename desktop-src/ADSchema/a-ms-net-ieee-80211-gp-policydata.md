# <a name="ms-net-ieee-80211-gp-policydata-attribute"></a>atributo MS-net-IEEE-80211-GP-PolicyData

|-------------------|---------------------------------------------|

| Nombre para mostrar de LDAP | MS-net-IEEE-80211-GP-PolicyData             |

| Actualizar privilegio  | \-                                          |
