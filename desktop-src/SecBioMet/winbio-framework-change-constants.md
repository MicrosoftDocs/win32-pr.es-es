# <a name="winbio_framework_change-constants"></a>Constantes de cambio de WINBIO \_ Framework \_

Se pueden usar las siguientes constantes en la función [**WinBioAsyncMonitorFrameworkChanges**](/windows/desktop/api/Winbio/nf-winbio-winbioasyncmonitorframeworkchanges) para identificar el tipo de cambio que se produjo en el marco.

<dl> <dt>

<span id="WINBIO_FRAMEWORK_CHANGE_UNIT"></span><span id="winbio_framework_change_unit"></span>**unidad de cambio de WINBIO \_ Framework \_ \_**
</dt> <dd> <dl> <dt>

0x00000001
</dt> <dt>



Se adjuntó una unidad biométrica al equipo o se desasoció del mismo.


</dt> </dl> </dd> </dl>
