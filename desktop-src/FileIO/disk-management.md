# <a name="disk-management"></a>Administración de discos

Un *disco duro* es un disco rígida dentro de un equipo que almacena y proporciona acceso relativamente rápido a grandes cantidades de datos. Es el tipo de almacenamiento que se usa con más frecuencia con Windows. El sistema también admite medios extraíbles.
