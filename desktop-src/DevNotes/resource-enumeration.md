# <a name="resource-enumeration"></a>Enumeración de recursos

Los desarrolladores pueden usar la enumeración de recursos como parte de la instrumentación que proporciona tipos específicos de información sobre sus aplicaciones.

Microsoft crea instrumentación en el sistema operativo Windows, pero estas características están vinculadas tan estrechamente al sistema operativo que pueden ser de uso limitado para los desarrolladores de aplicaciones. Microsoft está haciendo disponible una interfaz de programación de aplicaciones (API) de enumeración de recursos que se puede usar con los siguientes recursos de instrumentación: asignaciones de montón y control de la información de seguimiento.

En los temas siguientes se describe cómo usar esta API en las aplicaciones:

-   [Acerca de la enumeración de recursos](about-resource-enumeration.md)
-   [Referencia de enumeración de recursos](resource-enumeration-reference.md)
