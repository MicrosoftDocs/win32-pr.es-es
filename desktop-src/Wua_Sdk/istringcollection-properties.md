# <a name="istringcollection-properties"></a>Propiedades de IStringCollection

La [**interfaz IStringCollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) define las siguientes propiedades.



| Propiedad                                        | Descripción                                                                                                                     |
|-------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------|
| [**\_NewEnum**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get__newenum) | Obtiene una [interfaz IEnumVARIANT](/windows/win32/api/oaidl/nn-oaidl-ienumvariant) que se puede usar para enumerar la colección. |
| [**Contar**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_count)        | Obtiene el número de elementos de la colección.                                                                                  |
| [**Elemento**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_item)          | Obtiene o establece una cadena en la colección.                                                                                        |
| [**Readonly**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-get_readonly)  | Obtiene un valor booleano que indica si la colección es de solo lectura.                                                        |
