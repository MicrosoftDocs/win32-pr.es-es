# <a name="ms-ds-cached-membership-time-stamp-attribute"></a>MS-DS-Cached-pertenencia-atributo de marca de tiempo

El administrador de cuentas de seguridad usa el atributo **MS-DS-Cached-Membership-Stamp** para la expansión del grupo durante la evaluación de tokens.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-DS-Cached-Membership-marca de tiempo   |
| Nombre para mostrar de LDAP | msDS-Cached-Membership-marca de tiempo    |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1442              |
| System-ID-GUID    | 3566bf1f-beee-4dcb-8abe-ef89fcfec6c1 |
| Sintaxis            | [**Interval**](s-interval.md)       |
