# <a name="winsock-ipxspx-structures"></a>Estructuras de Winsock IPX/SPX

En la lista siguiente se proporcionan descripciones concisas de cada estructura de Winsock IPX/SPX. Para obtener información adicional sobre cualquier estructura, haga clic en el nombre de la estructura.



| Estructura IPX/SPX                                            | Descripción                                                          |
|--------------------------------------------------------------|----------------------------------------------------------------------|
| [**\_datos de dirección IPX \_**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_address_data)             | Proporciona información acerca de un adaptador específico al que está enlazado IPX. |
| [**\_datos de NETNUM IPX \_**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_netnum_data)               | Proporciona información sobre un número de red IPX especificado.           |
| [**\_datos de SPXCONNSTATUS IPX \_**](/windows/desktop/api/Wsnwlink/ns-wsnwlink-ipx_spxconnstatus_data) | Proporciona información sobre un socket SPX conectado.                   |
