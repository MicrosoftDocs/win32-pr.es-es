# <a name="mfpkey_totalframes-property"></a>\_Propiedad TOTALFRAMES de MFPKEY

Especifica el número total de fotogramas de vídeo pasados al codificador durante el proceso de endocing.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

g \_ wszWMVCTotalFrames

## <a name="data-type"></a>Tipo de datos

VT \_ I4

## <a name="remarks"></a>Observaciones

Puede obtener este valor después de haber terminado de pasar ejemplos.
