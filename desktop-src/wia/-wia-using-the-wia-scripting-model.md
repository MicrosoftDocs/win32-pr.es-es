# <a name="using-the-wia-scripting-model"></a>Uso del modelo de scripting de WIA

A través del modelo de scripting de [WIA,](-wia-wia-scripting-model.md)la funcionalidad de adquisición de imágenes de Windows (WIA) está disponible para lenguajes de scripting como el software de desarrollo de Microsoft JScript y Microsoft Visual Basic Scripting Edition (VBScript) y lenguajes de alto nivel, como el sistema de desarrollo de Microsoft Visual Basic.

> [!Note]  
> Este sistema de scripting se ha reemplazado por Windows de automatización de adquisición de imágenes (WIA). Consulte [Windows capa de automatización de adquisición de imágenes.](/previous-versions/windows/desktop/wiaaut/-wiaaut-startpage)

 

El modelo de scripting de WIA expone objetos que heredan la [interfaz IDispatch.](/previous-versions/windows/desktop/api/oaidl/nn-oaidl-idispatch)

El modelo de scripting ofrece una versión simplificada de WIAAPI.
