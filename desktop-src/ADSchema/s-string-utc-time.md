# <a name="stringutc-time-syntax"></a>Sintaxis de cadena (hora UTC)

Formato de cadena de hora definido por los estándares ASN. 1. Para obtener más información, vea ISO 8601 y X680. Use esta sintaxis para almacenar valores de hora en UTC-Time formato.



| Entrada | Value |
|--------------|----------------------------------------------------------------------------|
| Nombre         | String(UTC-Time)                                                           |
| IDENTIFICADOR de sintaxis    | 2.5.5.11                                                                   |
| IDENTIFICADOR DE OM        | 23                                                                         |
| Tipo MAPI    | SYSTIME                                                                    |
| Tipo ADS     | \_hora UTC de ADSTYPE \_                                                         |
| Tipo Variant | fecha de VT \_                                                                   |
| Tipo de SDS     | [System.DateTime](/dotnet/api/system.datetime) |
