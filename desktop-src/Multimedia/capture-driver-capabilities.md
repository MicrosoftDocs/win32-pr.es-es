# <a name="capture-driver-capabilities"></a>Capacidades del controlador de captura

Puede recuperar las capacidades de hardware del controlador de captura conectado actualmente mediante el mensaje [**de \_ \_ \_ obtención \_ de Cap del controlador Cap de WM**](wm-cap-driver-get-caps.md) (o la macro [**capDriverGetCaps**](/windows/desktop/api/Vfw/nf-vfw-capdrivergetcaps) ). Este mensaje devuelve las capacidades del controlador de captura y el hardware subyacente en la estructura [**CAPDRIVERCAPS**](/windows/win32/api/vfw/ns-vfw-capdrivercaps) .
