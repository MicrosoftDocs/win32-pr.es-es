|-------------------|---------------------------------------------|
| CN                | FRS-Version                                 |
| Nombre para mostrar de LDAP | fRSVersion                                  |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.882                      |
| System-ID-GUID    | 2a132585-9373-11d1-aebc-0000f80367c1        |
