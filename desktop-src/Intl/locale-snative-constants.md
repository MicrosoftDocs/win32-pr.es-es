# <a name="locale_snative-constants"></a>Constantes \_ SNATIVE de LOCALE \*

En este tema se definen las constantes SNATIVE de LOCALE \_ \* usadas por NLS para representar nombres de lenguaje nativos.



| Value                       | Significado                                                                                                                                                                                                                                                            |
|-----------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| LOCALE \_ SNATIVECOUNTRYNAME  | **Windows 7 y versiones posteriores:** Nombre nativo del país o región, por ejemplo, España para España. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.                                                                 |
| LOCALE \_ SNATIVECTRYNAME     | En desuso Windows 7 y versiones posteriores. Nombre nativo del país o región. Vea LOCALE \_ SNATIVECOUNTRYNAME.                                                                                                                                                             |
| LOCALE \_ SNATIVECURRNAME     | **Windows Me/98, Windows 2000:** Nombre nativo de la moneda asociada a la configuración regional, en el idioma nativo de la configuración regional. No hay ningún límite en el número de caracteres permitidos para esta cadena.                                                          |
| LOCALE \_ SNATIVEDIGITS       | Equivalentes nativos de ASCII 0 a 9. El número máximo de caracteres permitido para esta cadena es de once, incluido un carácter nulo de terminación. Por ejemplo, el árabe usa "٠١٢٣٤٥ ٦٧٨٩". Vea también [LOCALE \_ IDIGITSUBSTITUTION](locale-idigitsubstitution.md). |
| LOCALE \_ SNATIVEDISPLAYNAME  | **Windows 7 y versiones posteriores:** Nombre para mostrar de la configuración regional en su idioma nativo, por ejemplo, Deutsch (Deutschland) para la configuración regional alemana (Alemania). <br/>                                                                                                        |
| LOCALE \_ SNATIVELANGNAME     | En desuso Windows 7 y versiones posteriores. Nombre nativo del idioma. Vea LOCALE \_ SNATIVELANGUAGENAME.                                                                                                                                                                  |
| LOCALE \_ SNATIVELANGUAGENAME | **Windows 7 y versiones posteriores:** Nombre nativo del idioma, por ejemplo, Հայերեն para armenio (Armenio). El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación.                                                         |
