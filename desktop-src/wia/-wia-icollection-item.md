# <a name="collectionitem-property"></a>Colección. Item (propiedad)

Devuelve el elemento especificado en la colección.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Collection.Item
```
