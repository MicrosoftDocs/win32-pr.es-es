# <a name="framework-security"></a>Seguridad de .NET Framework

El servicio biométrico de Windows permite que las aplicaciones cliente sin privilegios realicen operaciones biométricas sin proporcionarles acceso directo a muestras biométricas o hardware. El servicio crea un límite de seguridad entre los clientes y los componentes biométricos, tal y como se muestra en la siguiente ilustración.
