# <a name="ui_pkey_decimalplaces"></a>\_DecimalPlaces de ui PKEY \_

Identifica la propiedad \_ PKEY \_ DecimalPlaces de la interfaz de usuario.

```
propertyDescription
   name = UI_PKEY_DecimalPlaces
   shellPKey = UI_PKEY_DecimalPlaces
   formatID = 00000206-7363-696e-8441798acf5aebb7
   propID = 206
   typeInfo
      type = VT_UI4
```

## <a name="remarks"></a>Comentarios

Una aplicación usa decimales PKEY de interfaz de usuario para consultar el número de dígitos que se muestran después del separador decimal en el campo de edición \_ \_ del control [Spinner.](windowsribbon-controls-spinner.md)
