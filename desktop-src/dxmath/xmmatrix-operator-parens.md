# <a name="xmmatrix-operator--operators"></a>Operadores del operador XMMATRIX ()

Tiene acceso a elementos de matriz específicos a los que hace referencia fila y columna desde la instancia actual de `XMMATRIX` .

Tiene acceso a elementos de matriz específicos a los que hace referencia fila y columna desde la instancia actual de [ **XMMATRIX.**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix)

### <a name="overload-list"></a>Lista de sobrecarga




| Operador | Descripción | 
|----------|-------------|
| <a href="/windows/desktop/api/directxmath/nf-directxmath-xmmatrix-operator-function-call(size_t_size_t)"><strong>XMMATRIX::operator () (size_t,size_t)</strong></a> | Devuelve un <code>reference</code> objeto a un elemento de matriz de una instancia de tal como lo especifican los <code>XMMATRIX</code> argumentos de fila y columna.<br /> Este operador devuelve a <code>reference</code> un elemento de matriz de una instancia de <a href="/windows/desktop/api/directxmath/ns-directxmath-xmmatrix"><strong>XMMATRIX</strong></a> tal y como se especifica en los argumentos de fila y columna.<br /><blockquote>[!Note]<br />Este operador solo está disponible cuando se desarrolla con C++.</blockquote><br /> |
