# <a name="dvd_register"></a>REGISTRO DE \_ DVD

El **tipo de datos DVD \_ REGISTER** contiene un valor para un registro de parámetros general de DVD (GPRM) o un registro de parámetros del sistema (SPRM).


```C++
typedef WORD DVD_REGISTER;
```
