# <a name="developer-license-apis"></a>API de licencia para desarrolladores

Use estas API para comprobar, adquirir y quitar una licencia de desarrollador.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                 | Descripción                                             |
|-----------------------------------------------------------------------|---------------------------------------------------------|
| [**CheckDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-checkdeveloperlicense)<br/>     | Comprueba si existe una licencia de desarrollador.<br/> |
| [**AcquireDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-acquiredeveloperlicense)<br/> | Adquiere una licencia para desarrolladores.<br/>                |
| [**RemoveDeveloperLicense**](/windows/desktop/api/wsdevlicensing/nf-wsdevlicensing-removedeveloperlicense)<br/>   | Quita una licencia de desarrollador.<br/>                 |
