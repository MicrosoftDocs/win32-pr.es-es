# <a name="getting-started-developing-user-interfaces-for-windows-applications"></a>Tareas iniciales desarrollar interfaces de usuario para aplicaciones windows

## <a name="purpose"></a>Propósito

En las secciones siguientes se ofrecen instrucciones generales a los desarrolladores que diseñan, implementan y prueban la interfaz de usuario de una aplicación windows.

Además de los principios básicos de diseño de la interfaz de usuario, se proporcionan numerosas recomendaciones y sugerencias que ayudarán a los desarrolladores a proporcionar una experiencia de usuario lo más sencilla, eficaz y divertida posible.

> [!Note]  
> Estas directrices no están diseñadas para ser completas y están sujetas al ámbito y la funcionalidad específicos de una aplicación. Para obtener instrucciones más completas, consulte Las directrices de interacción de la [experiencia del usuario de Windows](../uxguide/guidelines.md).
