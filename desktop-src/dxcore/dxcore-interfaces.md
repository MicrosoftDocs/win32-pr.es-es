# <a name="dxcore-interfaces"></a>Interfaces de DXCore

Las siguientes interfaces, y sus métodos, se declaran en dxcore_interface. h.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|-|-|
| [**IDXCoreAdapter**](./dxcore_interface/nn-dxcore_interface-idxcoreadapter.md) | La interfaz **IDXCoreAdapter**   implementa métodos para recuperar detalles sobre un elemento de adaptador. |
| [**IDXCoreAdapterFactory**](./dxcore_interface/nn-dxcore_interface-idxcoreadapterfactory.md) | La interfaz **IDXCoreAdapterFactory**   implementa métodos para generar objetos de enumeración del adaptador de DXCore y para recuperar los detalles. |
| [**IDXCoreAdapterList**](./dxcore_interface/nn-dxcore_interface-idxcoreadapterlist.md) | La interfaz **IDXCoreAdapterList**   implementa los métodos para recuperar los elementos de adaptador de una lista generada, así como los detalles de la lista. |
