# <a name="iupdateinstaller2-properties"></a>Propiedades de IUpdateInstaller2

La interfaz [**IUpdateInstaller2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller2) define la propiedad siguiente.



| Propiedad                                           | Descripción                                                                                                                       |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [**ForceQuiet**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller2-get_forcequiet) | Obtiene y establece un valor booleano que indica si se fuerza el Windows Installer para instalar las actualizaciones sin la interacción del usuario. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz.

-   [**IUpdateInstaller**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller)
