# <a name="xmvector-operators"></a>Operadores XMVECTOR

La estructura [**XMVECTOR**](xmvector-data-type.md) expone los operadores siguientes.

> [!Note]  
> Los operadores enumerados aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                    | Descripción                                                                                                                                 |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| [**operador +=**](/previous-versions/windows/desktop/legacy/ee421394(v=vs.85))<br/>  | Agrega un valor de punto flotante a una `XMVECTOR` instancia de y devuelve una referencia a la instancia actualizada. <br/>                         |
| [**operator -=**](/previous-versions/windows/desktop/legacy/ee421383(v=vs.85))<br/>    | Resta un valor de punto flotante de la instancia actual de `XMVECTOR` y devuelve el resultado en la instancia actual actualizada. <br/> |
| [**Operador \***](xmvector-operator-mul.md)<br/>    | Operadores de multiplicación<br/>                                                                                                         |
| [**Operador \*=**](xmvector-operator-muleq.md)<br/> | Operadores de asignación de multiplicación<br/>                                                                                              |
| [**operador /**](xmvector-operator-div.md)<br/>     | Operador de división.<br/>                                                                                                               |
| [**operador /=**](xmvector-operator-diveq.md)<br/>  | Operador de asignación de división.<br/>                                                                                                    |
| [**operador :**](xmvector-operator--.md)<br/>       | Operadores de resta y negación<br/>                                                                                               |
| [**operador +**](xmvector-operator-pls.md)<br/>     | Operadores de suma.<br/>                                                                                                              |
