# <a name="how-do-i-decompress-streams-from-an-avi-file-or-other-file-about-which-i-have-no-format-information"></a>Cómo descomprimir secuencias de un archivo AVI u otro archivo sobre el que no tengo información de formato?

Sin información de formato, incluidos los datos de formato extendido, no se pueden descomprimir los datos codificados con un códec Windows media. Los códecs se diseñaron para crear flujos de datos para su uso con el contenedor de archivos de formato de sistemas avanzados (ASF). La sección de encabezado de un archivo ASF almacena la información de formato de todas las secuencias del archivo. Cuando se usa contenido codificado mediante uno de los códecs Windows Media Audio y Video fuera de un archivo ASF, debe asegurarse de que la información de formato esté disponible. La manera de hacerlo varía de un contenedor a otro.
