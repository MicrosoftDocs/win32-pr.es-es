# <a name="locale_ildate"></a>configuración regional \_ ILDATE

Formato de fecha larga: especificador de ordenación. Es preferible que la aplicación use la constante [ \_ SLONGDATE de configuración regional](locale-slongdate.md) en lugar de la configuración regional \_ ILDATE.



| Value | Significado        |
|-------|----------------|
| 0     | Mes-día-año |
| 1     | Día-mes-año |
| 2     | Año-mes-día |
