# <a name="linecallcomplcond_-constants"></a>Constantes de LINECALLCOMPLCOND \_

Las constantes de marcador de bits **LINECALLCOMPLCOND \_** describen las condiciones en las que se puede completar una llamada.

<dl> <dt>

<span id="LINECALLCOMPLCOND_BUSY"></span><span id="linecallcomplcond_busy"></span>**LINECALLCOMPLCOND \_ ocupado**
</dt> <dd> <dl> <dt>



Finalización de la llamada en condiciones de ocupación.


</dt> </dl> </dd> <dt>

<span id="LINECALLCOMPLCOND_NOANSWER"></span><span id="linecallcomplcond_noanswer"></span>**LINECALLCOMPLCOND \_ NOanswer**
</dt> <dd> <dl> <dt>



Finalización de la llamada en las condiciones de timbre no responde.


</dt> </dl> </dd> </dl>

## <a name="remarks"></a>Observaciones

Sin extensibilidad. Todos los 32 bits están reservados.
