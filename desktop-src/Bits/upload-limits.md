# <a name="upload-limits"></a>Límites de carga

Para limitar el tamaño de las cargas, establezca la propiedad de extensión de IIS [**BITSMaximumUploadSize**](bits-iis-extension-properties.md) . En IIS 7, es posible que también tenga que modificar el atributo **maxAllowedContentLength** . De forma predeterminada, el límite de carga de IIS 7 es 30 millones bytes. Para obtener detalles e información sobre cómo cambiar el valor predeterminado de IIS, vea [KB942074](https://support.microsoft.com//kb/942074).
