# <a name="quality-of-service-in-the-windows-sockets-2-spi"></a>Calidad de servicio en Windows Sockets 2 SPI

La calidad de servicio (QoS) se implementa en Windows a través de varios componentes de QoS admitidos en Windows Server 2003, Windows XP y Windows 2000. Una explicación detallada de QoS y sus componentes se encuentra en una sección dedicada a la calidad de servicio, que se encuentra en el Kit de desarrollo de software (SDK) de Microsoft Windows. Cada componente de QoS y el rol que desempeña en la implementación de QoS se explican en esta sección de QoS, con instrucciones adicionales para la implementación de servicios y aplicaciones habilitados para QoS. Consulte la sección [Calidad de servicio (QoS)](/previous-versions/windows/desktop/qos/qos-start-page) de Windows SDK para obtener información más detallada.

En esta sección se describen las funcionalidades de calidad de servicio disponibles para los desarrolladores de Winsock. En la lista siguiente se describen los temas de esta sección:

-   [Modelo de uso](usage-model-2.md)
-   [Actualizaciones de QoS](qos-updates-2.md)
-   [Valores predeterminados en el SPI](default-values-in-the-spi-2.md)
-   [Plantillas de QoS en el SPI](qos-templates-in-the-spi-2.md)
