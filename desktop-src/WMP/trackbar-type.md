# <a name="trackbar-type"></a>Tipo de barra de texto

Debe definir el tipo de barra de texto que desea utilizar. En la tabla siguiente se muestran los tipos que son válidos.



| Tipo   | Descripción                                                                                                                                                                                                                                                                                                     |
|--------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Seek   | Controla la posición de reproducción. Si el valor es cero, el elemento multimedia está al principio. El valor aumenta hasta que se alcanza el final del elemento multimedia. El usuario puede deslizar la imagen Thumb para cambiar el valor y, mientras se reproduce el elemento multimedia, la imagen Thumb se desplaza para que coincida con la posición de reproducción actual. |
| Volumen | Controla el volumen de audio. El valor varía entre 0 y 100. El usuario puede deslizar la imagen Thumb para cambiar el volumen. Si cambia el volumen, la imagen de Thumb se moverá para que coincida con el nuevo volumen.                                                                                                                   |
