# <a name="equalizersettingswowlevel"></a>EQUALIZERSETTINGS.wowLevel

El atributo **wowLevel** especifica o recupera el nivel de efecto SRS WOW.

``` syntax
        elementID.wowLevel
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **número** de lectura/escritura (**largo**) que va de 0 a 100 con un valor predeterminado de 50.

## <a name="remarks"></a>Observaciones

El efecto SRS WOW es un efecto de mejora de audio. Este atributo se omite si **enhancedAudio** está establecido en false.
