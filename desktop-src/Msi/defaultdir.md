# <a name="defaultdir"></a>DefaultDir

El tipo de datos DefaultDir es una cadena de texto que contiene un nombre de archivo [válido](filename.md) o un identificador [válido.](identifier.md) Solo se usa en la tabla [Directory](directory-table.md). Debe ser un identificador si el directorio es un directorio raíz. Si el directorio no es raíz, este valor debe ser un nombre de archivo o un par nombreArchivo:nombreArchivo. Tenga en cuenta que "." se permite como nombre de archivo y tiene un significado especial en la tabla Directory.
