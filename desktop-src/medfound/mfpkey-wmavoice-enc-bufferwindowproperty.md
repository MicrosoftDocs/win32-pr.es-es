# <a name="mfpkey_wmavoice_enc_bufferwindow-property"></a>Propiedad BufferWindow de MFPKEY \_ WMAVOICE \_ ENC \_

Especifica la ventana de búfer, en milisegundos, que se usará para el códec de voz.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

g \_ wszWMACVoiceBuffer

## <a name="data-type"></a>Tipo de datos

VT \_ I4

## <a name="default-value"></a>Valor predeterminado

1500 para 4, 5, 8, 12 y 16 Kbps

2177 para 10 y 20 Kbps
