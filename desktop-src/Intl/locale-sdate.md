# <a name="locale_sdate"></a>configuración regional \_ SDATE

Carácter (s) para el separador de fecha. El número máximo de caracteres permitido para esta cadena es cuatro, incluido un carácter nulo de terminación.

**Windows Vista y versiones posteriores:** Esta constante está en desuso. Use la [configuración regional \_ SSHORTDATE](locale-sshortdate.md) en su lugar. Una configuración regional personalizada no puede tener un único carácter separador uniforme. Por ejemplo, un formato como "12/31, 2006" es válido.
