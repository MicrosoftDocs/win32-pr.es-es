# <a name="locale_sortname"></a>LOCALE \_ SORTNAME

**Windows 7 y versiones posteriores:** Nombre de configuración regional que se va a usar para la ordenación, si es diferente de la configuración regional especificada por [LOCALE \_ SNAME](locale-sname.md). Esta constante la usan las configuraciones regionales personalizadas. Vea también [LOCALE \_ SSORTNAME](locale-ssort-constants.md).
