# <a name="xmxdecn4-extensions"></a>Extensiones de XMXDECN4

[**XMXDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmxdecn4) definen un vector 4D para almacenar valores normalizados y firmados como componentes x-, y-y z-y sin signo de 10 bits, y valores normalizados de 2 bits como componente w-sin signo.

> [!Note]  
> Las características que se enumeran aquí solo están disponibles cuando se desarrollan con C++.
