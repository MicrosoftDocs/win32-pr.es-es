<span data-ttu-id="b964c-206">Las funciones matemáticas D3DX se implementan de forma idéntica en D3DX10 (D3DX10math.h) y D3DX9 (D3DX9math.h).</span><span class="sxs-lookup"><span data-stu-id="b964c-206">The D3DX math functions are implemented identically in D3DX10 (D3DX10math.h) and D3DX9 (D3DX9math.h).</span></span> <span data-ttu-id="b964c-207">Esto puede provocar errores de vínculo si un proyecto implementa código DirectX 9 y DirectX 10 e intenta vincular una función desde un encabezado con la biblioteca opuesta.</span><span class="sxs-lookup"><span data-stu-id="b964c-207">This can cause link errors if a project implements both DirectX 9 and DirectX 10 code, and attempts to link a function from one header with the opposite library.</span></span>
