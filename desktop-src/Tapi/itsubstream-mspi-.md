# <a name="itsubstream-mspi"></a>ITSubStream (MSPI)

La interfaz [**ITSubStream**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) está disponible si un MSP está asociado a la llamada e implementa subsecuencias. Los métodos de esta interfaz permiten a una aplicación recuperar información en un subflujo; para iniciar, pausar o detener el subflujo; y para seleccionar o anular la selección de terminales en una subsecuencia.

La interfaz [**ITSubStream**](/windows/win32/api/tapi3if/nn-tapi3if-itsubstream) se implementa mediante un MSP. Consulte **ITSubStream** en la sección interfaz MSP para obtener más información sobre esta interfaz.
