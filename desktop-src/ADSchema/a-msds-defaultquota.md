# <a name="ms-ds-default-quota-attribute"></a>atributo MS-DS-default-quota

La cuota predeterminada que se aplicará a una entidad de seguridad que crea un objeto en el NC si no existe ninguna especificación de cuota que cubra la entidad de seguridad.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-DS-default-quota                  |
| Nombre para mostrar de LDAP | msDS-DefaultQuota                    |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1846              |
| System-ID-GUID    | 6818f726-674b-441b-8a3a-f40596374cea |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
