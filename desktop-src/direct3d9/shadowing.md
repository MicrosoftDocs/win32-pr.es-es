# <a name="shadowing-direct3d-9"></a>Sombreado (Direct3D 9)

Los búferes de estarcido son una técnica de búfer de profundidad que se puede actualizar a medida que la geometría se representa y se usa de nuevo como máscara para dibujar más geometría. Los efectos comunes son los reflejos, las sombras (una técnica avanzada), la resolución, etc. El [ejemplo ShadowVolume](https://msdn.microsoft.com/library/Ee418792(v=VS.85).aspx) usa búferes de estarcido para implementar sombras en tiempo real. La información adicional se incluye en [la galería de símbolos de dos caras (Direct3D 9)](two-sided-stencil.md).
