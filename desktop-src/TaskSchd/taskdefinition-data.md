# <a name="taskdefinitiondata-property"></a>Propiedad TaskDefinition. Data

En el caso de scripting, obtiene o establece los datos asociados a la tarea. El servicio Programador de tareas omite estos datos, pero lo usan terceros que desean ampliar el formato de la tarea.

## <a name="syntax"></a>Sintaxis


```VB
TaskDefinition.Data As String
```



## <a name="property-value"></a>Valor de propiedad

Los datos que están asociados a la tarea.
