# <a name="winbio_ansi_385_face-constants"></a>WINBIO \_ \_ constantes de \_ tipo ANSI 385

Las siguientes constantes son valores **de \_ \_ subtipo biométrico WINBIO** que se pueden usar para especificar los dos tipos de imágenes de la esfera frontal según se define en ANSI INCITS 385-2004: "formato de reconocimiento facial para el intercambio de datos": resolución completa y baja resolución. En la práctica, el marco biométrico solo usará imágenes de resolución completa para el reconocimiento facial.



| Constante o valor                                                                                                                                                                                                                                                                          | Descripción                                                |
|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------|
| <span id="WINBIO_ANSI_385_FACE_TYPE_UNKNOWN"></span><span id="winbio_ansi_385_face_type_unknown"></span><dl> <dt>**WINBIO \_ \_Tipo de \_ letra ANSI 385 \_ \_ desconocido**</dt> <dt>0</dt> </dl>    | Se desconoce el tipo de imagen de la esfera frontal.<br/>         |
| <span id="WINBIO_ANSI_385_FACE_FRONTAL_FULL"></span><span id="winbio_ansi_385_face_frontal_full"></span><dl> <dt>**WINBIO \_ ANSI \_ 385 con una \_ superficie \_ \_ completa**</dt> <dt>1</dt> </dl>    | El tipo de imagen de la esfera frontal es resolución completa.<br/> |
| <span id="WINBIO_ANSI_385_FACE_FRONTAL_TOKEN"></span><span id="winbio_ansi_385_face_frontal_token"></span><dl> <dt>**WINBIO \_ \_ \_ \_ \_ Token**</dt> <dt>2</dt> de la superficie ANSI 385 </dl> | El tipo de imagen de la esfera frontal es baja resolución.<br/>  |
