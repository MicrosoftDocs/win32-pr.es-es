# <a name="authentication-options-attribute"></a>Authentication-Options atributo)

Opciones de autenticación usadas en ADSI para enlazar a los objetos de servicios de directorio.



| Entrada | Value |
|-------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CN                | Authentication-Options                                                                                                                                                                                                                                                       |
| Nombre para mostrar de LDAP | authenticationOptions                                                                                                                                                                                                                                                        |
| Tamaño              | 4 bytes. Valores definidos en IADS. h: \_ autenticación segura de ADS \_ 0X1, ADS \_ usar \_ cifrado 0X2, ADS \_ usar \_ SSL 0X2, ADS \_ ReadOnly \_ Server 0x4, ADS \_ prompt \_ CREDENTIALS 0x8, ADS \_ sin \_ autenticación 0x10, ADS \_ Fast \_ BIND 0x20, ADS \_ usar \_ firma 0x40, ADS \_ usar el \_ sellado 0x80 |
| Actualizar privilegio  | \-                                                                                                                                                                                                                                                                           |
| Frecuencia de actualización  | \-                                                                                                                                                                                                                                                                           |
| Attribute-Id      | 1.2.840.113556.1.4.11                                                                                                                                                                                                                                                        |
| System-ID-GUID    | bf967928-0de6-11d0-a285-00aa003049e2                                                                                                                                                                                                                                         |
| Sintaxis            | [**Enumeración**](s-enumeration.md)                                                                                                                                                                                                                                         |
