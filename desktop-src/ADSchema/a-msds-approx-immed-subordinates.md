# <a name="ms-ds-approx-immed-subordinates-attribute"></a>Atributo ms-DS-Approx-Immed-Subordinates

El valor devuelto por este atributo se basa en los tamaños de índice. Esto puede estar desactivado en +/ 10 % en contenedores grandes y el error es teóricamente sin enlazar, pero el uso de este atributo ayuda a la interfaz de usuario a mostrar el contenido de un contenedor.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | ms-DS-Approx-Immed-Subordinates      |
| Ldap-Display-Name | msDS-Approx-Immed-Subordinates       |
| Size              | \-                                   |
| Privilegio actualizar  | El sistema establece este valor.     |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1669              |
| System-Id-Guid    | e185d243-f6ce-4adb-b496-b0c005d7823c |
| Syntax            | [**Enumeración**](s-enumeration.md) |
