# <a name="media-foundation-primitives"></a>Media Foundation primitivas

Media Foundation define varios tipos de objetos básicos que se usan en las API de Media Foundation.



| Tema                                       | Descripción                                                                         |
|---------------------------------------------|-------------------------------------------------------------------------------------|
| [Atributos](attributes-and-properties.md) | Los atributos y las propiedades son pares clave-valor almacenados en un objeto.                  |
| [Tipos de medios](media-types.md)              | Un tipo de medio describe el formato de un flujo de medios digitales.                        |
| [Búferes multimedia](media-buffers.md)          | Un búfer de medios administra un bloque de memoria, por lo que se puede compartir entre los objetos. |
| [Ejemplos de medios](media-samples.md)          | Un ejemplo multimedia es un objeto que contiene una lista de búferes multimedia.                  |
