# <a name="separate-window-plug-ins"></a>Complementos de ventana independientes

Los complementos de interfaz de usuario de ventana independientes se muestran en sus propias ventanas. Estas ventanas pueden tener un tamaño fijo o se pueden cambiar de tamaño. Se pueden abrir varias ventanas de complementos de interfaz de usuario diferentes al mismo tiempo.

Los complementos de interfaz de usuario de ventana independientes son útiles cuando se desea que la información o los controles que se proporcione sigan estando visibles cuando el usuario salga del panel **reproducción en curso** o cargue otro complemento de la interfaz de usuario. Los complementos de la interfaz de usuario de una ventana independiente que se ejecuten cuando se cierre Windows Media Player se volverán a cargar y se mostrarán en el mismo tamaño y posición cuando se vuelva a iniciar el reproductor.
