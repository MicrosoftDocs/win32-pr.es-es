# <a name="schema-id-guid-attribute"></a>Schema-ID-GUID (atributo)

Identificador único de un atributo.



| Entrada | Value |
|-------------------|---------------------------------------------------------------------|
| CN                | IDENTIFICADOR de esquema-GUID                                                      |
| Nombre para mostrar de LDAP | schemaIDGUID                                                        |
| Tamaño              | 16 bytes                                                            |
| Actualizar privilegio  | El sistema establece este valor.                                    |
| Frecuencia de actualización  | Este valor se establece cuando se crea el objeto y no se puede cambiar. |
| Attribute-Id      | 1.2.840.113556.1.4.148                                              |
| System-ID-GUID    | bf967923-0de6-11d0-a285-00aa003049e2                                |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md)               |
