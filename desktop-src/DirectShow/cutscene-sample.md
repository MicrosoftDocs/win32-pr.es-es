# <a name="cutscene-sample"></a>Ejemplo de CutScene

## <a name="description"></a>Descripción

El ejemplo CutScene reproduce un archivo de vídeo en modo de pantalla completa hasta que el usuario escribe ESC, ENTRAR o la barra espaciador.

Ejecute este ejemplo desde la línea de comandos, con el nombre de un archivo de vídeo como argumento de línea de comandos.

## <a name="downloading-the-sample"></a>Descargar el ejemplo

Para descargar los ejemplos DirectShow SDK, instale la versión más reciente del [SDK de Windows.](https://msdn.microsoft.com/windowsvista/bb980924.aspx)

Este ejemplo se instala en la siguiente ruta de acceso: *\[ Sdk Root \]* Samples Multimedia DirectShow Players \\ \\ \\ \\ \\ CutScene.
