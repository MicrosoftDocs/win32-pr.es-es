# <a name="retrieving-event-data-using-tdh"></a>Recuperación de datos de eventos mediante TDH

Para consumir datos específicos del evento, el consumidor debe conocer el formato de los datos del evento. Si el proveedor ha usado un manifiesto, MOF o archivos TMF para publicar el formato de los datos del evento, puede usar las funciones del asistente de datos de seguimiento (TDH) para analizar los datos del evento. Las funciones de TDH están disponibles a partir de Windows Vista.

En los temas siguientes se muestra cómo usar TDH para recuperar metadatos y datos de eventos.

-   [Uso de TdhFormatProperty para consumir datos de eventos](using-tdhformatproperty-to-consume-event-data.md)
-   [Uso de TdhGetProperty para consumir datos de eventos](using-tdhgetproperty-to-consume-event-data.md)
-   [Recuperar metadatos de eventos](retrieving-event-metadata.md)
-   [Enumeración de proveedores](enumerating-providers.md)
