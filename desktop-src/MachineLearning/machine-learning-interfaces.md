# <a name="windows-machine-learning-interfaces"></a>Interfaces de Machine Learning de Windows

En los temas siguientes se describen las interfaces COM del SDK de Windows Machine Learning.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                             | Descripción                                                                                                                                                                                |
|-----------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IWinMLEvaluationContext**](/windows/win32/api/winml/nn-winml-iwinmlevaluationcontext)<br/> | Representa el contexto para enlazar entradas y salidas a un modelo WinML.<br/>                                                                                                             |
| [**IWinMLModel**](/windows/win32/api/winml/nn-winml-iwinmlmodel)<br/>                         | Representa un modelo de Machine Learning de Windows con los metadatos correspondientes. incluye descripciones de modelos (nombre, autor, control de versiones, etc.), así como entradas y salidas esperadas. <br/> |
| [**IWinMLRuntime**](/windows/win32/api/winml/nn-winml-iwinmlruntime)<br/>                     | Representa el tiempo de ejecución para cargar y evaluar un modelo WinML.<br/>                                                                                                                      |
| [**IWinMLRuntimeFactory**](/windows/win32/api/winml/nn-winml-iwinmlruntimefactory)<br/>       | Representa el generador que crea el tiempo de ejecución de WinML para la carga y evaluación del modelo.<br/>                                                                                         |
