# <a name="license-revocation-windows-media-format-sdk"></a>Revocación de licencia (SDK de Windows Media Format)

El SDK de Windows Media Format proporciona métodos por los que una aplicación puede interactuar con el sitio web de un emisor de licencias para revocar una licencia de contenido.

Un escenario típico de revocación de licencias es cuando un usuario proporciona un equipo antiguo a otra persona. En este caso, el usuario original puede tener muchas licencias para el contenido protegido almacenado en el equipo, aunque el usuario ya no use el equipo. La revocación de licencias permite al usuario original quitar las licencias del equipo.

El proceso de revocación de licencias implica que los mensajes se pasen entre la aplicación cliente y una aplicación basada en Web creada por el emisor de la licencia.

**Nota:** DRM no es compatible con la versión basada en x64 de este SDK.
