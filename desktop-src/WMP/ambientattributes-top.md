# <a name="ambientattributestop"></a>AmbientAttributes.top

El atributo **Top** especifica o recupera la coordenada superior del control.

``` syntax
        elementID.top
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **número** de lectura/escritura (**Long**) que representa la distancia en píxeles desde el control al borde superior de la **vista** o la **subvista** primaria. Tiene un valor predeterminado de cero. Se permiten números negativos, en cuyo caso el borde superior de la **vista** o la **subvista** recorta el control.
