# <a name="skin-files"></a>Archivos de máscara

Cuando ve una máscara, solo ve la superficie. Detrás de la superficie hay varios archivos que componen la piel completa. Técnicamente hablando, una máscara es un grupo de archivos, donde cada archivo contiene un tipo de información específico. Estos son los tipos de archivos que puede usar para crear una máscara completa.



| Tipo de archivo                                        | Descripción                                                                                                                                                               |
|--------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Archivo de definición de máscara](skin-definition-file.md) | El archivo maestro que define cómo se utilizarán los demás archivos. Se trata de un archivo de texto y tiene una extensión de nombre de archivo. WMS.                                                    |
| [Archivos de imagen](art-files.md)                       | Archivos de imagen que contienen los elementos gráficos de la máscara. Entre ellos se incluyen archivos de mapa de bits, GIF, JPEG y PNG.                                                                 |
| [Archivos JScript](jscript-files.md)               | Si desea crear respuestas más complicadas a los eventos, puede crear archivos de scripts mediante Microsoft JScript. Estos son archivos de texto y tienen una extensión de nombre de archivo. js. |
