# <a name="windows-system-assessment-tool-provider"></a>Proveedor de herramientas de evaluación del sistema de Windows

\[Es posible que WinSAT se modifique o no esté disponible para las versiones después de Windows 8.1.\]

## <a name="purpose"></a>Propósito

La herramienta de evaluación del sistema de Windows (WinSAT) expone una serie de clases que evalúa las características de rendimiento y las capacidades de un equipo. Los desarrolladores pueden usar esta API para desarrollar software que pueda tener acceso a la información de rendimiento y capacidad de un equipo con el fin de determinar la configuración de aplicación óptima en función de las capacidades de rendimiento de ese equipo.

Los usuarios pueden usar los resultados de una evaluación formal para determinar qué escenarios y aplicaciones funcionarán correctamente en un equipo. Por ejemplo, si un paquete de software contiene una clasificación en su paquete, un usuario puede usar la clasificación para determinar si el software se ejecutará correctamente en el equipo.

## <a name="developer-audience"></a>Audiencia de desarrolladores

WinSAT está diseñado para desarrolladores de C, C++ y Visual Basic y para los que escriben scripts.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

WinSAT está disponible en los equipos cliente a partir del sistema operativo Windows Vista.
