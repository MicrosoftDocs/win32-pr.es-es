# <a name="the-characters-object"></a>El objeto Characters

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

La aplicación cliente puede admitir uno o varios caracteres. Además, puede compartir un carácter entre varias aplicaciones. Microsoft Agent define el [**objeto Characters**](/windows/desktop/lwef/the-characters-object) como una colección de caracteres. Para tener acceso a un carácter, cargue los datos del carácter en la colección [**Characters**](/windows/desktop/lwef/the-characters-object) y especifique que el elemento de la colección usa los métodos y propiedades admitidos para ese carácter.

-   [Métodos de objeto Characters](characters-object-methods.md)
-   [Métodos de objeto Characters](character-object-methods.md)
-   [Propiedades del objeto Characters](character-object-properties.md)
