# <a name="glossary-imapi"></a>Glosario (IMAPi)

<dl> <dt>

<span id="imapi.imapi2_glossary_ANSI"></span><span id="imapi.imapi2_glossary_ansi"></span><span id="IMAPI.IMAPI2_GLOSSARY_ANSI"></span>**ANSI**
</dt> <dd>

American National Standards Institute.

</dd> <dt>

<span id="imapi.imapi2_glossary_BurnEngine"></span><span id="imapi.imapi2_glossary_burnengine"></span><span id="IMAPI.IMAPI2_GLOSSARY_BURNENGINE"></span>**Motor de grabación**
</dt> <dd>

Componente que lee datos de un flujo de datos proporcionado (por ejemplo, IStream o e/s. Stream) y escribe los datos en un dispositivo de CD/DVD. El componente debe tener en cuenta el búfer disponible en la unidad, pero no contiene ningún conocimiento del formato de los datos, preparar la unidad para escribir o informar a la unidad de que se ha completado el proceso de escritura.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDDVDDevice"></span><span id="imapi.imapi2_glossary_cddvddevice"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDDVDDEVICE"></span>**Dispositivo de CD/DVD**
</dt> <dd>

Dispositivo que se notifica a sí mismo como un dispositivo "tipo 5". Esto incluye el CD-ROM, DVD-ROM y la mayoría de las unidades de CD y DVD Writer. Además, algunas unidades con medios magnéticos (como IOMEGA REV) se informan como dispositivos "tipo 5" y se muestran como dispositivos de CD-ROM en Windows.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDG"></span><span id="imapi.imapi2_glossary_cdg"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDG"></span>**CD-G** 
</dt> <dd>

Un acrónimo de gráficos de CD, tecnología exclusiva de CDs de audio que permite agregar gráficos de velocidad de bits baja a una pista de audio. Esta tecnología se usa con más frecuencia en CDs usados para karaoke.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_R"></span><span id="imapi.imapi2_glossary_cd_r"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_R"></span>**CD-R**
</dt> <dd>

Un disco compacto de solo escritura.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RandRW"></span><span id="imapi.imapi2_glossary_cd_randrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RANDRW"></span>**CD-R/RW**
</dt> <dd>

Un disco compacto de CD-R o CD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_ROM"></span><span id="imapi.imapi2_glossary_cd_rom"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_ROM"></span>**CD-ROM**
</dt> <dd>

Un disco compacto de solo lectura.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RW"></span><span id="imapi.imapi2_glossary_cd_rw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RW"></span>**CD-RW**
</dt> <dd>

Un disco compacto regrabable.

</dd> <dt>

<span id="imapi.imapi2_glossaryCDText"></span><span id="imapi.imapi2_glossarycdtext"></span><span id="IMAPI.IMAPI2_GLOSSARYCDTEXT"></span>**Texto de CD** 
</dt> <dd>

Tecnología exclusiva de CDs de audio que permite almacenar información de velocidad de bits baja, como el nombre de intérprete y el nombre de la canción en el medio físico. A continuación, las aplicaciones o dispositivos que reproducen el CD resultante recuperan esta información, si existe, para mostrar al usuario.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disc"></span><span id="imapi.imapi2_glossary_disc"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISC"></span>**Disco**
</dt> <dd>

Un término genérico para medios ópticos redondos, como un CD-ROM o un DVD.

</dd> <dt>

<span id="imapi.imapi2_glossary_DiscAtOnce"></span><span id="imapi.imapi2_glossary_discatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISCATONCE"></span>**Disco a la vez** 
</dt> <dd>

Método de escritura en el CD en el que todo el contenido de la imagen de grabación se escribe en el disco en un flujo continuo. Además, los datos no accesibles, como los de cliente potencial y LeadOut, se incluyen en la imagen de grabación.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disk"></span><span id="imapi.imapi2_glossary_disk"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISK"></span>**Discos**
</dt> <dd>

Un término genérico para medios magnéticos redondos, como un disquete o una unidad de disco duro.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dispatch"></span><span id="imapi.imapi2_glossary_dispatch"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISPATCH"></span>**Obra** 
</dt> <dd>

Una designación de interfaz COM donde una interfaz determinada admite el método **IDispatch** para llamar a una función. Consulte MSDN para obtener más información sobre este tema sobre COM avanzado.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dual"></span><span id="imapi.imapi2_glossary_dual"></span><span id="IMAPI.IMAPI2_GLOSSARY_DUAL"></span>**BI** 
</dt> <dd>

Una designación de interfaz COM donde una interfaz determinada admite los métodos VTable y **IDispatch** para llamar a una función. Consulte MSDN para obtener más información sobre estos temas de COM avanzados.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVD"></span><span id="imapi.imapi2_glossary_dvd"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVD"></span>**DISCOS**
</dt> <dd>

Un disco versátil digital. originalmente, esta abreviatura era "disco de vídeo digital".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusR"></span><span id="imapi.imapi2_glossary_dvdplusr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSR"></span>**DVD + R**
</dt> <dd>

Un DVD de una sola escritura. El acrónimo se pronuncia "DVD Plus R".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRDL"></span><span id="imapi.imapi2_glossary_dvdplusrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRDL"></span>**DVD + R DL**
</dt> <dd>

Un disco DVD + R de doble capa. Este tipo de disco también se conoce como "DVD + R de doble capa".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRRW"></span><span id="imapi.imapi2_glossary_dvdplusrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRRW"></span>**DVD + R/RW**
</dt> <dd>

Un disco DVD + R y un disco DVD + RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRW"></span><span id="imapi.imapi2_glossary_dvdplusrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRW"></span>**DVD + RW**
</dt> <dd>

Un DVD regrabable. El acrónimo se pronuncia "DVD Plus RW".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashR"></span><span id="imapi.imapi2_glossary_dvdplusdashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHR"></span>**DVD ± R**
</dt> <dd>

Un disco DVD + R y un disco DVD-R.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRRW"></span>**DVD ± R/RW**
</dt> <dd>

Un disco DVD + R, un disco DVD-R, un disco DVD + RW y un disco DVD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRW"></span>**DVD ± RW**
</dt> <dd>

Un disco DVD + RW y un medio DVD + R

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashR"></span><span id="imapi.imapi2_glossary_dvddashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHR"></span>**DVD-R**
</dt> <dd>

Un DVD de una sola escritura. El acrónimo se pronuncia "DVD Dash R".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRDL"></span><span id="imapi.imapi2_glossary_dvddashrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRDL"></span>**DVD-R DL**
</dt> <dd>

Un medio DVD-R de doble capa.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDRAM"></span><span id="imapi.imapi2_glossary_dvdram"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDRAM"></span>**DVD-RAM**
</dt> <dd>

Un DVD de escritura aleatoria. Este tipo de medio admite la administración de defectos de hardware que identifica y evita el uso de pistas dañadas. Las unidades que admiten este formato son más frecuentes en Japón que en EE. UU.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDROM"></span><span id="imapi.imapi2_glossary_dvdrom"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDROM"></span>**DVD-ROM**
</dt> <dd>

Un DVD de solo lectura.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRRW"></span><span id="imapi.imapi2_glossary_dvddashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRRW"></span>**DVD-R/RW**
</dt> <dd>

Un disco DVD-R y un disco DVD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRW"></span><span id="imapi.imapi2_glossary_dvddashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRW"></span>**DVD-RW**
</dt> <dd>

Un DVD regrabable. El acrónimo se pronuncia "DVD Dash RW".

</dd> <dt>

<span id="imapi.imapi2_glossary_ECMA"></span><span id="imapi.imapi2_glossary_ecma"></span><span id="IMAPI.IMAPI2_GLOSSARY_ECMA"></span>**ECMA** 
</dt> <dd>

La Asociación Europea de fabricantes de equipos es una asociación del sector fundada en 1961 que está dedicada a la estandarización de sistemas de información y comunicación. ECMA edita estándares e informes técnicos. Todas las publicaciones de ECMA están disponibles de forma gratuita.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPI"></span><span id="imapi.imapi2_glossary_imapi"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPI"></span>**IMAPI** 
</dt> <dd>

La API de creación de imágenes maestras que se usa para crear y escribir imágenes maestras (en lugar de un FS en vivo) en dispositivos de CD/DVD. Este término se aplica genéricamente a todas las versiones de la API de masterización de imágenes.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv1"></span><span id="imapi.imapi2_glossary_imapiv1"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV1"></span>**IMAPIv1** 
</dt> <dd>

La versión de IMAPi que se expuso primero en Windows XP. IMAPi 1,0 es compatible con los medios CD-R y CD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv2"></span><span id="imapi.imapi2_glossary_imapiv2"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV2"></span>**IMAPIv2** 
</dt> <dd>

La segunda versión de IMAPi, publicada con Windows Vista.

</dd> <dt>

<span id="imapi.imapi2_glossary_INCITS"></span><span id="imapi.imapi2_glossary_incits"></span><span id="IMAPI.IMAPI2_GLOSSARY_INCITS"></span>**INCITS**
</dt> <dd>

Comité Internacional de estándares de tecnología de la información, una organización acreditada y aprobada por ANSI.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISO9660"></span><span id="imapi.imapi2_glossary_iso9660"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISO9660"></span>**ISO9660**
</dt> <dd>

Un sistema de archivos diseñado para la primera generación de medios de CD. Este sistema de archivos tiene limitaciones de nombre de archivo y ordenación significativas.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISOImage"></span><span id="imapi.imapi2_glossary_isoimage"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISOIMAGE"></span>**Imagen ISO**
</dt> <dd>

Un nombre común para una imagen sector por sector de la totalidad del área de datos de un CD de datos o un disco DVD. El nombre aumentó desde el uso común de la unidad de disco de ISO9660 como sistema de archivos para el CD.

</dd> <dt>

<span id="imapi.imapi2_glossary_Joliet"></span><span id="imapi.imapi2_glossary_joliet"></span><span id="IMAPI.IMAPI2_GLOSSARY_JOLIET"></span>**Joliet**
</dt> <dd>

Un conjunto de extensiones normalizadas para la ISO9660 que admite nombres de archivo Unicode de hasta 64 caracteres de longitud. Consulte la especificación ECMA número 119 para más información.

</dd> <dt>

<span id="imapi.imapi2_glossary_MMC"></span><span id="imapi.imapi2_glossary_mmc"></span><span id="IMAPI.IMAPI2_GLOSSARY_MMC"></span>**COMPLEMENTO**
</dt> <dd>

Conjunto de comandos multimedia definido por el grupo de trabajo T10 de INCITS. MMC define comandos de paquetes para dispositivos de CD y DVD. Los borradores de trabajo actuales de MMC pueden encontrarse en el sitio web de T10.

</dd> <dt>

<span id="imapi.imapi2_glossary_MtFuji"></span><span id="imapi.imapi2_glossary_mtfuji"></span><span id="IMAPI.IMAPI2_GLOSSARY_MTFUJI"></span>**MT. Fuji**
</dt> <dd>

Conjunto de comandos para controlar dispositivos multimedia, similar al conjunto de comandos de MMC. Algunos elementos (como las características o los tipos de medios) se describen en la MT. Documentos de Fuji, mientras que otros se describen primero en los documentos de MMC. Los borradores de trabajo actuales del conjunto de comandos pueden encontrarse en [FTP://ftp.AVC-Pioneer.com](ftp://ftp.avc-pioneer.com/).

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSI"></span><span id="imapi.imapi2_glossary_scsi"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSI"></span>**SCSI**
</dt> <dd>

La interfaz del sistema pequeño del equipo.

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSICommand"></span><span id="imapi.imapi2_glossary_scsicommand"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSICOMMAND"></span>**Comando SCSI**
</dt> <dd>

Cualquier número de comandos de paquete, como los definidos en el documento de MMC. Estos comandos se pueden enviar mediante cualquier mecanismo de transporte adecuado, como SCSI, ATAPI, USB o IEEE1394.

</dd> <dt>

<span id="imapi.imapi2_glossary_SessonAtOnce"></span><span id="imapi.imapi2_glossary_sessonatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_SESSONATONCE"></span>**Sesión a la vez**
</dt> <dd>

Método para crear un CD Plus Disc. Una sesión de escritura registra las pistas de audio en el disco y una segunda sesión escribe una única pista de CD-XA en el disco y cierra el disco.

</dd> <dt>

<span id="imapi.imapi2_glossary_TrackAtOnce"></span><span id="imapi.imapi2_glossary_trackatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_TRACKATONCE"></span>**Seguimiento a la vez**
</dt> <dd>

Método de escritura en el CD en el que los datos de una sola pista se escriben al mismo tiempo. Cuando se escriben todas las pistas, la sesión se cierra como paso final que implica la escritura de las áreas iniciales y de salida del disco.

</dd> <dt>

<span id="imapi.imapi2_glossary_UDF"></span><span id="imapi.imapi2_glossary_udf"></span><span id="IMAPI.IMAPI2_GLOSSARY_UDF"></span>**Formato de disco universal (UDF)**
</dt> <dd>

Este sistema de archivos está definido por OSTA (la Asociación de tecnología de almacenamiento óptico) que admite discos de sesión única y de varias sesiones.

</dd> <dt>

<span id="imapi.imapi2_glossary_VTable"></span><span id="imapi.imapi2_glossary_vtable"></span><span id="IMAPI.IMAPI2_GLOSSARY_VTABLE"></span>**VTBL** 
</dt> <dd>

Una designación de interfaz COM donde una interfaz determinada admite el método vtable de llamar a una función. Consulte MSDN para obtener más información sobre este tema sobre COM avanzado.

</dd> </dl>
