# <a name="locale_sintlsymbol"></a>configuración regional \_ SINTLSYMBOL

Tres caracteres del símbolo monetario internacional especificado en ISO 4217, seguido del carácter que separa esta cadena de la cantidad. El número máximo de caracteres permitido para esta cadena es nueve, incluido un carácter nulo de terminación.
