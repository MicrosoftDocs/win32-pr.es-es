# <a name="ienumparticipantreset-method"></a>IEnumParticipant::Reset (Método)

\[**El** restablecimiento no está disponible para su uso en Windows Vista, Windows Server 2008 y versiones posteriores del sistema operativo. La API de cliente RTC proporciona una funcionalidad similar.\]

El **método Reset** se restablece al principio de la secuencia de enumeración. Este método está oculto de Visual Basic y lenguajes de scripting.
