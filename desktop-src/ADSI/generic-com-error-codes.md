# <a name="generic-com-error-codes"></a>Códigos de error COM genéricos

Los códigos de error COM genéricos expresan el estado de la operación generado por todos los módulos COM. La tabla siguiente contiene algunos ejemplos de estos códigos de error en cualquier plataforma Win32.



| Código de error     | Valor hexadecimal  | Descripción              |
|----------------|------------|--------------------------|
| E \_ ABORT       | 0x80004004 | La operación se anuló.       |
| E \_ FAIL        | 0x80004005 | Error no especificado.       |
| E \_ NOINTERFACE | 0x80004002 | Interfaz no admitida. |
| E \_ NOTIMPL     | 0x80004001 | Sin implementar.         |
| PUNTERO \_ E     | 0x80004003 | Puntero no válido.         |
| E \_ UNEXPECTED  | 0x8000FFFF | Error catastrófico.    |
