# <a name="upcalls-exposed-by-windows-sockets-2-dll"></a>Llamadas arriba expuestas por Windows Sockets 2 DLL

En esta sección se enumeran las llamadas que los proveedores de servicios pueden realizar en el Windows Sockets. Los proveedores de servicios reciben una tabla de envío de llamada upcall como parámetro para la función [**WSPStartup**](/windows/desktop/api/Ws2spi/nf-ws2spi-wspstartup) y usan las entradas de esta tabla para realizar las llamadas upcall. Por lo tanto, un cliente no necesita exportar sus funciones WPU.

No es obligatorio que los proveedores usen todas estas llamadas. En la tabla siguiente se indica qué llamadas upcall deben usarse y cuáles son opcionales.

| Función                                                               | Descripción                                                                                                              | Estado                         | Significado                                                                                                                                                                          |
|------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|--------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**WPUCloseEvent**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpucloseevent)                               | Cierra un identificador de objeto de evento abierto.                                                                                      | Opcional.                      | El proveedor puede usar una llamada Windows en su lugar.                                                                                                                     |
| [**WPUCloseSocketHandle**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpuclosesockethandle)                 | Cierra un identificador de socket asignado por el archivo DLL Windows Sockets.                                                             | Obligatorio.                      | El32.dll Ws2 debe consultar o modificar la información de estado \_ interno asociada al identificador de socket.                                                                       |
| [**WPUCloseThread**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpuclosethread)                             | Cierra un identificador de subproceso para un subproceso de servicio interno.                                                                       |                                |                                                                                                                                                                                  |
| [**WPUCompleteOverlappedRequest**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpucompleteoverlappedrequest) | Entrega una notificación de finalización de E/S superpuesta en la que el mecanismo de finalización es algo distinto del modo de usuario APC.    |                                |                                                                                                                                                                                  |
| [**WPUCreateEvent**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpucreateevent)                             | Crea un nuevo objeto de evento.                                                                                              | Opcional.                      | El proveedor puede usar una llamada Windows en su lugar.                                                                                                                        |
| [**WPUCreateSocketHandle**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpucreatesockethandle)               | Crea un nuevo identificador de socket para proveedores que no sonIFS.                                                                        | Necesario para proveedores que no sonIFS. | El32.dll Ws2 debe consultar o modificar la información de estado \_ interno asociada al identificador de socket.                                                                       |
| [**WPUFDIsSet**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpufdisset)                                     | Comprueba la pertenencia del identificador de socket especificado.                                                                    | Opcional.                      | Se trata simplemente de una función de conveniencia que sabe cómo profundizar en las [**estructuras de fd \_ set.**](/windows/desktop/api/winsock/nf-winsock-fd_set) De todos modos, es posible que un proveedor tenga que profundizar explícitamente en estas estructuras. |
| [**WPUGetProviderPath**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpugetproviderpath)                     | Recupera la ruta de acceso del archivo DLL para el proveedor especificado.                                                                       | Obligatorio.                      | Solo el Ws232.dll saber dónde se ha instalado una capa de protocolo adyacente (potencialmente de otro \_ proveedor).                                                           |
| [**WPUModifyIFSHandle**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpumodifyifshandle)                     | Recibe un identificador IFS modificado (posiblemente) del archivo DLL Windows Sockets.                                                  | Necesario para los proveedores de IFS.    | El32.dll Ws2 debe consultar o modificar la información de estado \_ interno asociada al identificador de socket.                                                                       |
| [**WPUPostMessage**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpupostmessage)                             | Realiza la función [**PostMessage**](/windows/win32/api/winuser/nf-winuser-postmessagea) estándar de una manera que mantiene la compatibilidad con versiones anteriores. | Obligatorio.                      | Windows 2000 y Windows NT. Windows 95 permite enviar un mensaje desde el modo kernel.                                                                                               |
| [**WPUQueryBlockingCallback**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpuqueryblockingcallback)         | Devuelve un puntero a la función de enlace de bloqueo de un subproceso.                                                                  | Obligatorio.                      | No hay ninguna funcionalidad Windows correspondiente. Solo el32.dll \_ Ws2 tiene la información necesaria para ello.                                                                    |
| [**WPUQuerySocketHandleContext**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpuquerysockethandlecontext)   | Obtiene el valor de contexto de un socket (solo proveedores noIFS).                                                                   | Necesario para proveedores que no sonIFS. | El32.dll Ws2 debe consultar o modificar la información de estado \_ interno asociada al identificador de socket.                                                                       |
| [**WPUQueueApc**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpuqueueapc)                                   | Pone en cola un APC en modo de usuario en el subproceso especificado.                                                                          | Opcional.                      | También [**se puede usar QueueUserApc.**](/windows/win32/api/processthreadsapi/nf-processthreadsapi-queueuserapc)                                                                                                                      |
| [**WPUResetEvent**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpuresetevent)                               | Restablece un objeto de evento.                                                                                                  | Opcional.                      | El proveedor puede usar una llamada Windows en su lugar.                                                                                                                        |
| [**WPUSetEvent**](/windows/desktop/api/Ws2spi/nf-ws2spi-wpusetevent)                                   | Establece un objeto de evento.                                                                                                    | Opcional.                      | El proveedor puede usar una llamada Windows en su lugar.                                                                                                                        |
