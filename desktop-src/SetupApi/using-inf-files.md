# <a name="using-inf-files"></a>Uso de archivos INF

Después de crear un archivo INF para el proceso de instalación, las funciones de instalación pueden acceder a esa información para usarla al copiar, eliminar o cambiar el nombre de los archivos y actualizar los INI y las entradas del Registro.

En los temas siguientes se describe cómo usar las funciones de instalación para recuperar información de archivos INF y cómo realizar operaciones de instalación haciendo referencia a un archivo INF:

-   [Creación de un archivo INF](authoring-an-inf-file.md)
-   [Apertura y cierre de un archivo INF](opening-and-closing-an-inf-file.md)
-   [Recuperar información de un archivo INF](retrieving-information-from-an-inf-file.md)
-   [Instalación desde un archivo INF](installing-from-an-inf-file.md)
-   [Especificar un descriptor de seguridad a partir de un archivo INF](specifying-a-security-descriptor-from-an-inf-file.md)
