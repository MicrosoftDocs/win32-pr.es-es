# <a name="firewall-dynamic-keywords-enumerations"></a>Enumeraciones de palabras clave dinámicas de firewall

Las palabras clave dinámicas del firewall contienen las enumeraciones siguientes.

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|-|-|
| [**FW_DYNAMIC_KEYWORD_ADDRESS_ENUM_FLAGS**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_address_enum_flags) | Define constantes que especifican los tipos de objetos que se incluirán en una operación de enumeración. |
| [**FW_DYNAMIC_KEYWORD_ADDRESS_FLAGS**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_address_flags) | Define constantes que especifican cómo se van a resolver las direcciones IP. |
| [**FW_DYNAMIC_KEYWORD_ORIGIN_TYPE**](/windows/win32/api/netfw/ne-netfw-fw_dynamic_keyword_origin_type) | Define constantes que especifican el origen del objeto de dirección de palabra clave dinámica en un [**FW_DYNAMIC_KEYWORD_ADDRESS_DATA0**](/windows/win32/api/netfw/ns-netfw-fw_dynamic_keyword_address_data0). |
