# <a name="playlist-burning"></a>Grabación de listas de reproducción

Las licencias de Windows Media DRM 10 pueden admitir la copia de listas de reproducción de archivos protegidos en un CD de audio de libro rojo. Los emisores de licencias pueden limitar el número de veces que el contenido se puede copiar en un CD en una lista de reproducción en un número total de veces y por el número de veces que se puede copiar una lista de reproducción específica. Para obtener más información acerca de los derechos de grabación de listas de reproducción en licencias, consulte la documentación del SDK de Windows Media Rights Manager 10.

El SDK de Windows Media Format proporciona la capacidad de comprobar rápidamente todos los archivos de una lista de reproducción para asegurarse de que están autorizados para la grabación de listas de reproducción.

**Nota:** DRM no es compatible con la versión basada en x64 de este SDK.
