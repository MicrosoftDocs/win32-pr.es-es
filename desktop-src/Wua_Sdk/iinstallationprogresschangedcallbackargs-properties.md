# <a name="iinstallationprogresschangedcallbackargs-properties"></a>Propiedades de IInstallationProgressChangedCallbackArgs

La interfaz [**IInstallationProgressChangedCallbackArgs**](/windows/desktop/api/Wuapi/nn-wuapi-iinstallationprogresschangedcallbackargs) define la propiedad siguiente.



| Propiedad                                                              | Descripción                                                                                                                        |
|-----------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------|
| [**Progreso**](/windows/desktop/api/Wuapi/nf-wuapi-iinstallationprogresschangedcallbackargs-get_progress) | Obtiene una interfaz que contiene el progreso de la instalación o desinstalación asincrónica en el momento en que se realizó la devolución de llamada. |
