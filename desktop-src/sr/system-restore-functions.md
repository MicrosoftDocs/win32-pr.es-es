# <a name="system-restore-functions"></a>Funciones de restauración del sistema

Las siguientes funciones se usan con Restaurar sistema.



| Función                                             | Descripción                                                                                                   |
|------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**SRRemoveRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srremoverestorepoint) | Elimina el punto de restauración especificado.                                                                          |
| [**SRSetRestorePoint**](/windows/desktop/api/SRRestorePtAPI/nf-srrestoreptapi-srsetrestorepointa)       | Especifica el principio y el final de un conjunto de cambios para que restaurar sistema pueda crear un punto de restauración. |
