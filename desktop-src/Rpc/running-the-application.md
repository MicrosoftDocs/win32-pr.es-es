# <a name="running-the-application"></a>Ejecutar la aplicación

Para ejecutar la aplicación en un único equipo, abra dos ventanas de consola. En la primera ventana, escriba

**Hola**

y en la segunda ventana, escriba

**helloc**

Para experimentar con diferentes secuencias de protocolo, puntos de conexión y otras opciones, compile la aplicación Hello de ejemplo a partir de los archivos de origen en el \\ directorio RPC Hello en el kit de desarrollo de software (SDK) de la plataforma.

> [!Note]  
> La restricción de la **interfaz RPC** puede provocar que se produzca un error de **acceso denegado** en Windows XP con Service Pack 2 (SP2) y versiones posteriores de Windows. Para obtener más información, consulte restricción de la [interfaz RPC](/previous-versions/windows/it-pro/windows-xp/bb457156(v=technet.10)#EGAA).
