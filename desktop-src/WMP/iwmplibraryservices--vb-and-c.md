# <a name="iwmplibraryservices-vb-and-c-interface"></a>Interfaz IWMPLibraryServices (VB y C#)

Proporciona métodos para enumerar las bibliotecas.

## <a name="members"></a>Miembros

La interfaz **IWMPLibraryServices (VB y C#)** tiene estos tipos de miembros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

La interfaz **IWMPLibraryServices (VB y C#)** tiene estos métodos.



| Método                                                                                               | Descripción                                                                                                        |
|:-----------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------------------|
| [**getCountByType**](wmplibiwmplibraryservices-iwmplibraryservices-getcountbytype--vb-and-c.md)     | Devuelve el número de bibliotecas disponibles de un tipo especificado.<br/>                                           |
| [**getLibraryByType**](wmplibiwmplibraryservices-iwmplibraryservices-getlibrarybytype--vb-and-c.md) | Devuelve una interfaz **IWMPLibrary** que representa la biblioteca que tiene el tipo y el índice especificados.<br/> |



 

Obtenga una interfaz **IWMPLibraryServices** mediante la conversión del objeto devuelto desde el método **AxWindowsMediaPlayer. GetOcx** .
