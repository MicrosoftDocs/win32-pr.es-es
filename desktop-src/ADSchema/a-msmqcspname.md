# <a name="msmq-csp-name-attribute"></a>Atributo MSMQ-CSP-Name

El tipo de proveedor de servicios criptográficos que utiliza MSMQ.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Nombre de MSMQ-CSP:                               |
| Nombre para mostrar de LDAP | mSMQCSPName                                 |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.940                      |
| System-ID-GUID    | 9a0dc334-c100-11d1-bbc5-0080c76670c0        |
| Sintaxis            | [**String(Teletex)**](s-string-teletex.md) |
