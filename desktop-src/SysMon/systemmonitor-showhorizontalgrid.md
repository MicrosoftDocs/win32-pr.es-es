# <a name="systemmonitorshowhorizontalgrid-property"></a>Propiedad SystemMonitor.ShowHorizontalGrid

Recupera o establece un valor que determina si las líneas de cuadrícula horizontales se muestran en el gráfico.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
Property ShowHorizontalGrid As Boolean
```



## <a name="property-value"></a>Valor de propiedad

True indica que las líneas de cuadrícula horizontales se muestran en el gráfico; de lo contrario, false. El valor predeterminado de esta propiedad es False.
