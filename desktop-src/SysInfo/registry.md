# <a name="registry"></a>Registro

El *registro* es una base de datos definida por el sistema en la que las aplicaciones y los componentes del sistema almacenan y recuperan los datos de configuración. Los datos almacenados en el registro varían según la versión de Microsoft Windows. Las aplicaciones usan la API del registro para recuperar, modificar o eliminar datos del registro.

No debe editar los datos del registro que no pertenezcan a la aplicación a menos que sea absolutamente necesario. Si hay un error en el registro, es posible que el sistema no funcione correctamente. Si esto ocurre, puede restaurar el registro al estado en que se encontraba la última vez que inició el equipo correctamente. Para obtener más información, consulte la ayuda del sistema operativo.

Para obtener más información sobre la base de datos del registro y el acceso mediante programación a los datos del registro, vea los temas siguientes:

-   [Acerca del registro](about-the-registry.md)
-   [Usar el registro](using-the-registry.md)
-   [Referencia del registro](registry-reference.md)
