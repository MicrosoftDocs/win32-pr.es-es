# <a name="object-property-sheet"></a>Hoja de propiedades de objeto

En este tema se describe la hoja de propiedades de objeto, que es común a todas las clases de objeto del complemento usuarios y equipos de Active Directory.

## <a name="object-property-sheet"></a>Hoja de propiedades de objeto

En la tabla siguiente se muestran las etiquetas de la interfaz de usuario de la hoja de propiedades del **objeto** .



| Etiqueta de la interfaz de usuario                              | Atributo en Active Directory Domain Services | Comentarios                                                   |
|---------------------------------------|-----------------------------------------------|------------------------------------------------------------|
| Nombre de dominio completo del objeto |                                               | Este es el nombre distintivo del objeto en formato canónico. |
| Object (clase)                          | objectClass                                   |                                                            |
| Creado                               | whenCreated                                   |                                                            |
| Modificado                              | whenChanged                                   |                                                            |
| Números de secuencia de actualización: actual      | uSNChanged                                    |                                                            |
| Números de secuencia de actualización: original     | uSNCreated                                    |                                                            |
