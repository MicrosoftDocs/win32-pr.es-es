# <a name="wia-event-identifiers"></a>Identificadores de eventos WIA

Los identificadores de eventos WIA son identificadores únicos globales (GUID) definidos por el Windows adquisición de imágenes (WIA).

Una aplicación WIA puede suscribirse a estos eventos no persistentes mediante [**IwiaDevMgr2::RegisterEventCallbackInterface**](-wia-iwiadevmgr2-registereventcallbackinterface.md) para recibir notificaciones de cambio de estado del dispositivo WIA al que está conectada la sesión de aplicación WIA actual. El mini driver de WIA debe asegurarse de que EL ESTADO DE CONTROL DE DOCUMENTOS de WIA DPS se actualiza correctamente antes de que el controlador \_ señale uno de estos \_ \_ \_ eventos.

Los siguientes son GUID de eventos WIA válidos: 

| GUID                             | Significado                                                                                                                                                                                                                                                                                                                               |
|----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| CANCELACIÓN DE \_ E/S DE \_ EVENTOS DE WIA \_           | La aplicación intenta cancelar una transferencia de datos.                                                                                                                                                                                                                                                                                   |
| COBERTURA DE EVENTOS DE WIA \_ \_ \_ CERRADA        | Todas las cubiertas del analizador están cerradas. Si se han abierto varias cubiertas de escáner, el minidriver WIA debe indicar este evento solo cuando se cerró la última cobertura. Esta característica está disponible a partir de Windows 8.                                                                                                                      |
| WIA \_ EVENT \_ COVER \_ OPEN          | El operador abrió una cubierta del analizador. El dispositivo no puede examinar en este momento. El minidriver WIA debe señalar este evento solo una vez si se abren varias cubiertas de escáner en sucesión. Esta característica está disponible a partir de Windows 8.                                                                                        |
| DISPOSITIVO DE EVENTOS WIA \_ \_ \_ CONECTADO    | Un dispositivo de hardware WIA se conectó al equipo del usuario o se inició.                                                                                                                                                                                                                                                                |
| DISPOSITIVO DE \_ EVENTOS WIA \_ \_ DESCONECTADO | Un dispositivo de hardware WIA se desconectó del equipo del usuario o se detuvo.                                                                                                                                                                                                                                                           |
| DISPOSITIVO DE \_ EVENTOS WIA \_ NO \_ \_ LISTO   | El dispositivo entró en estado no listo. La aplicación debe leer ESTADO DE CONTROL DE DOCUMENTOS de WIA \_ DPS \_ para obtener \_ más \_ información. Esta característica está disponible a partir de Windows 8.                                                                                                                                                         |
| DISPOSITIVO DE \_ EVENTOS WIA \_ \_ LISTO        | El dispositivo entró en estado listo. Se ha borrado el estado anterior no listo señalado con WIA \_ EVENT \_ DEVICE NOT \_ \_ READY. Esta característica está disponible a partir de Windows 8.                                                                                                                                                             |
| WIA \_ EVENT \_ FEEDER \_ EMPTIED      | El operador quitó todos los documentos de la bandeja de entrada del feeder y el feeder está vacío. El dispositivo no puede examinar desde el feeder en este momento. Importante: el minidriver no debe indicar este evento cuando el feeder se queda vacío como resultado de un trabajo de examen ejecutado por una aplicación. Esta característica está disponible a partir de Windows 8. |
| SE HA CARGADO \_ EL \_ FEEDER DE EVENTOS \_ WIA       | El operador colocó documentos en la bandeja de entrada del feeder. El feeder se carga con papel. La aplicación puede iniciar un trabajo de examen desde el feeder en este momento. Esta característica está disponible a partir de Windows 8.                                                                                                                     |
| CIERRE DE LA TAP PLANA \_ \_ DEL EVENTO \_ WIA \_ | El operador cerró la tapa plana. La aplicación puede iniciar un trabajo de examen desde la pantalla plana en este momento. Esta característica está disponible a partir de Windows 8.                                                                                                                                                                 |
| CIERRE PLANO DEL EVENTO WIA \_ \_ \_ \_ ABIERTO   | El operador abrió la tapa plana. El dispositivo no puede examinar desde la pantalla plana en este momento. Esta característica está disponible a partir de Windows 8.                                                                                                                                                                                  |
| ELEMENTO DE \_ EVENTO WIA \_ \_ CREADO        | Se agregó un elemento al árbol de [**objetos IWiaItem**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) o [**IWiaItem2**](-wia-iwiaitem2.md) para un dispositivo WIA.                                                                                                                                                                                                   |
| ELEMENTO DE \_ EVENTO WIA \_ \_ ELIMINADO        | Se eliminó un elemento del árbol de [**objetos IWiaItem**](/windows/desktop/api/wia_xp/nn-wia_xp-iwiaitem) o [**IWiaItem2**](-wia-iwiaitem2.md) para un dispositivo WIA.                                                                                                                                                                                               |
| WIA \_ EVENT \_ POWER \_ RESUME        | La energía se ha restaurado en el dispositivo o equipo.                                                                                                                                                                                                                                                                                    |
| WIA \_ EVENT \_ POWER \_ SUSPEND       | La alimentación se ha suspendido en el dispositivo o equipo.                                                                                                                                                                                                                                                                                   |
| IMAGEN DE CORREO \_ ELECTRÓNICO DE EXAMEN DE EVENTOS DE \_ \_ \_ WIA   | Se inició un examen con la intención de enviar por correo electrónico la imagen resultante.                                                                                                                                                                                                                                                                   |
| IMAGEN DE \_ FAX DE EXAMEN DE EVENTOS \_ \_ DE \_ WIA     | Se inició un examen con la intención de enviar por fax la imagen resultante.                                                                                                                                                                                                                                                                      |
| IMAGEN DE WIA \_ EVENT \_ SCAN \_ IMAGE \_    | Se ha examinado una imagen desde un adaptador de película.                                                                                                                                                                                                                                                                                             |
| IMAGEN DE \_ EXAMEN DE EVENTOS DE \_ \_ WIA          | Se ha examinado una imagen.                                                                                                                                                                                                                                                                                                                 |
| IMAGEN DE \_ EXAMEN DE EVENTOS DE WIA2 \_ \_         | Se ha examinado una imagen. Este evento permite a los analizadores con más de un botón de examen dirigir los botones a diferentes aplicaciones.                                                                                                                                                                                                      |
| WIA \_ EVENT \_ SCAN \_ IMAGE3         | Se ha examinado una imagen. Este evento permite a los analizadores con más de un botón de examen dirigir los botones a diferentes aplicaciones.                                                                                                                                                                                                      |
| IMAGEN4 \_ DEL EXAMEN DE EVENTOS \_ \_ DE WIA         | Se ha examinado una imagen. Este evento permite a los analizadores con más de un botón de examen dirigir los botones a diferentes aplicaciones.                                                                                                                                                                                                      |
| IMAGEN DE \_ OCR DE EXAMEN DE EVENTOS DE \_ \_ \_ WIA     | Se inició un examen con la intención de convertir la imagen resultante con reconocimiento óptico de caracteres (OCR).                                                                                                                                                                                                                         |
| IMAGEN DE \_ IMPRESIÓN DE EXAMEN DE EVENTOS DE \_ \_ \_ WIA   | Se inició un examen con la intención de imprimir la imagen resultante.                                                                                                                                                                                                                                                                    |
| PROXY PROXY \_ DE WIA EVENT \_ PROXY \_           | Sólo para uso interno. No debe usarse.                                                                                                                                                                                                                                                                                           |
| ALMACENAMIENTO DE EVENTOS DE WIA \_ \_ \_ CREADO     | Se agregó una unidad de almacenamiento, como una tarjeta flash, al dispositivo. La aplicación debe volver a crear el árbol de elementos que representa este dispositivo.                                                                                                                                                                                            |
| ALMACENAMIENTO DE EVENTOS DE WIA \_ \_ \_ ELIMINADO     | Una unidad de almacenamiento, como una tarjeta flash, se quitó del dispositivo. La aplicación debe volver a crear el árbol de elementos que representa este dispositivo.                                                                                                                                                                                        |
| ÁRBOL DE \_ EVENTOS WIA \_ \_ ACTUALIZADO        | Se actualizó el árbol de elementos de WIA.                                                                                                                                                                                                                                                                                                        |
| VOLUMEN DE \_ EVENTOS WIA \_ \_ INSERTADO     | Se insertó un dispositivo de almacenamiento, como una tarjeta flash, en el dispositivo.                                                                                                                                                                                                                                                                 |
