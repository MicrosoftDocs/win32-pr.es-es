# <a name="web-standards-and-application-compatibility"></a>Estándares web y compatibilidad de aplicaciones

Los estándares web proporcionan una línea base común de tecnologías que permiten a los usuarios seleccionar un explorador Web sin preocuparse por el uso de los sitios web que desean y que permiten a los diseñadores web y desarrolladores crear un único sitio web que todos pueden usar.

Los estándares del explorador web están evolucionando constantemente y los exploradores Web se actualizan continuamente para adoptar estos estándares. Por lo tanto, los profesionales de ti y los desarrolladores deben asegurarse de que las aplicaciones web son compatibles con los estándares y exploradores Web actuales y emergentes. Este mantenimiento garantiza que las aplicaciones web y los sitios de son compatibles con las nuevas versiones de los exploradores y ayuda a evitar muchos problemas de compatibilidad que podrían producirse.
