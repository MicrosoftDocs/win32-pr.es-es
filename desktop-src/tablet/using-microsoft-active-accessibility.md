# <a name="using-microsoft-active-accessibility"></a>Uso de Microsoft Active Accessibility

El motor de reconocimiento de voz en tablet PC usa Active Accessibility recuperar información directamente del objeto y recibir notificaciones cuando cambian los objetos. Esta interfaz es compatible con los controles estándar de Microsoft Windows y también se puede usar en controles personalizados y clases de ventana personalizadas.
