# <a name="texture2dgather-methods"></a>Métodos Texture2D::Gather

Consulte la documentación de [gather4](./gather4--sm5---asm-.md) para obtener más información sobre la instrucción DXBC subyacente.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                           | Descripción                                                                                         |
|:-----------------------------------------------------------------|:----------------------------------------------------------------------------------------------------|
| [**Gather(S,float,int)**](sm5-object-texture2d-gather.md)       | Muestrea una textura y devuelve los cuatro componentes.<br/>                                       |
| [**Gather(S,float,int,uint)**](t2d-gather-s-float-int-uint-.md) | Muestrea una textura y devuelve los cuatro componentes junto con el estado de la operación.<br/> |
