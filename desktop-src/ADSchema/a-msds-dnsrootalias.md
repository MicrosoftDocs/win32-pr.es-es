# <a name="ms-ds-dnsrootalias-attribute"></a>atributo MS-DS-DnsRootAlias

Se utiliza para almacenar el alias del dominio.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-DS-DnsRootAlias                          |
| Nombre para mostrar de LDAP | msDS-DnsRootAlias                           |
| Tamaño              | Tamaño máximo 255 bytes.                     |
| Actualizar privilegio  | El sistema establece este valor.            |
| Frecuencia de actualización  | Solo durante la reestructuración de dominios.             |
| Attribute-Id      | 1.2.840.113556.1.4.1719                     |
| System-ID-GUID    | 2143acca-eead-4d29-b591-85fa49ce9173        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
