# <a name="ms-authz-last-effective-security-policy-attribute"></a>atributo MS-authz-Last-effectiv-Security-Policy

En el caso de una regla de acceso central, este atributo define el permiso que se aplicó por última vez a los objetos a los que se aplica la regla de acceso central.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-authz-Last-effectived-Policy     |
| Nombre para mostrar de LDAP | msAuthz-LastEffectiveSecurityPolicy         |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2152                     |
| System-ID-GUID    | 8e1685c6-3e2f-48a2-a58d-5af0ea789fa0        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
