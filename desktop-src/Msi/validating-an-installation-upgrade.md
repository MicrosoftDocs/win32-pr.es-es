# <a name="validating-an-installation-upgrade"></a>Validación de una actualización de instalación

Siempre que realice cambios en el paquete, los autores de paquetes de actualización siempre deben ejecutar la validación en sus paquetes antes de intentar instalar el paquete por primera vez y volver a ejecutar la validación. Ejecute la validación en un paquete de actualización de la misma manera que para un paquete de instalación. Para más información, consulte [Validación de una base de datos de instalación.](validating-an-installation-database.md)

Esto completa el paquete de actualización de ejemplo. Para instalar primero la actualización, instale MNP2000.msi, a continuación, instale MNP2001.msi. Al desinstalar MNP2001, los productos originales y de actualización se quitan del equipo.
