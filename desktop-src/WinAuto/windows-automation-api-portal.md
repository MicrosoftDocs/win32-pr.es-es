# <a name="windows-accessibility-api-overview"></a>Windows API de accesibilidad información general

En esta sección se proporcionan información general de alto nivel y una referencia de API detallada para Microsoft Windows Automation API 3.0, que incluye la implementación de Windows de la especificación de Microsoft [Automatización de la interfaz de usuario](./uiauto-specandcommunitypromise.md)y Microsoft Active Accessibility (característica heredada introducida en Windows 95 como complemento de plataforma).

Se describen las similitudes y diferencias entre Microsoft Active Accessibility y Automatización de la interfaz de usuario, los componentes y características que permiten que las dos tecnologías funcionen conjuntamente, y se proporcionan directrices para elegir qué tecnología implementar para su escenario específico.
