# <a name="access-rights-for-active-directory-domain-services-objects"></a>Derechos de acceso para objetos de Active Directory Domain Services

Todos los objetos de directorio usan el mismo conjunto de derechos de acceso predefinidos que corresponden a las operaciones de directorio comunes. Estos derechos se describen en el tipo de enumeración [**ADS \_ Rights \_ enum**](/windows/win32/api/iads/ne-iads-ads_rights_enum) .

Además, puede definir derechos de acceso privado (derechos de acceso de control) para las operaciones que van más allá de las que abarcan los derechos predefinidos. Vea [derechos de control de acceso](control-access-rights.md).
