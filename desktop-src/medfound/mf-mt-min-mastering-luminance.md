# <a name="mf_mt_min_mastering_luminance-attribute"></a>Atributo MF \_ MT \_ MIN \_ MASTERING \_ LUMINANCE

Especifica la luminosidad máxima de la pantalla en la que se autorizó el contenido, en Nits. Este atributo tiene la misma semántica que la luminosidad de maestro de visualización mínima tal como se define en el estándar CEA-861.3. **\_ \_ \_**

## <a name="data-type"></a>Tipo de datos

**UINT32**
