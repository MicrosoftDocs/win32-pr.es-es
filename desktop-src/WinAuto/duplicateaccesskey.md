# <a name="duplicateaccesskey"></a>DuplicateAccessKey

## <a name="text"></a>Texto

Hay un elemento de automatización relacionado con la misma clave de acceso o tecla de aceleración

## <a name="type"></a>Tipo

Error

## <a name="description"></a>Descripción

La tecla de acceso/acelerador asignada al elemento también se asigna a otro elemento en el destino de la comprobación.

Este problema causa problemas para las personas que confían en el teclado para la navegación porque es posible que se invoquen varios elementos o que el foco se Centre con el mismo método abreviado o tecla de aceleración.
