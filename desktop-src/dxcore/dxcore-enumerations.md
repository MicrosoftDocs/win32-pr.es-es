# <a name="dxcore-enumerations"></a>Enumeraciones de DXCore

Las enumeraciones siguientes se declaran en dxcore_interface.h.

## <a name="in-this-section"></a>En esta sección

| Tema y descripción |
|-|
| [**DXCoreAdapterPreference**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterpreference.md). Define constantes que especifican las preferencias del adaptador de DXCore que se usarán como criterios de ordenación de lista. |
| [**DXCoreAdapterProperty**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterproperty.md). Define constantes que especifican las propiedades del adaptador de DXCore. |
| [**DXCoreAdapterState**](./dxcore_interface/ne-dxcore_interface-dxcoreadapterstate.md). Define constantes que especifican tipos de estados de adaptador DXCore. |
| [**DXCoreNotificationType**](./dxcore_interface/ne-dxcore_interface-dxcorenotificationtype.md). Define constantes que especifican tipos de notificaciones generados por objetos [IDXCoreAdapter](./dxcore_interface/nn-dxcore_interface-idxcoreadapter.md) o [IDXCoreAdapterList.](./dxcore_interface/nn-dxcore_interface-idxcoreadapterlist.md) |
| [**DXCoreSegmentGroup**](./dxcore_interface/ne-dxcore_interface-dxcoresegmentgroup.md). Define constantes que especifican la agrupación de segmentos de memoria de un adaptador. |
