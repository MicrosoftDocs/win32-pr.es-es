# <a name="constructing-a-timeline"></a>Construir una escala de tiempo

\[Esta API no se admite y puede modificarse o no estar disponible en el futuro.\]

En este artículo se describe cómo construir una escala de tiempo en los [servicios de edición de DirectShow](directshow-editing-services.md) (des). Presenta una aplicación de consola de ejemplo que crea una escala de tiempo y la representa. La escala de tiempo es mínima y consta de un único grupo de vídeos con un clip de origen, pero muestra la mayoría de los conceptos necesarios para crear escalas de tiempo más complejas.
