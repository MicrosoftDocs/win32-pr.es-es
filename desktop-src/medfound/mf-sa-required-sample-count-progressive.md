# <a name="mf_sa_required_sample_count_progressive-attribute"></a>Atributo MF \_ SA REQUIRED SAMPLE COUNT \_ \_ \_ \_ PROGRESSIVE

Indica el número de muestras que una Microsoft Media Foundation transformación (MFT) requiere que se asigne para el contenido progresiva.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

Este valor se usa si el siguiente nodo de bajada tiene [**un valor DENVIDEOVideoSampleAllocator.**](/windows/desktop/api/mfidl/nn-mfidl-imfvideosampleallocator)
