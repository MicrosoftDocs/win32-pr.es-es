# <a name="globalization-services"></a>Globalization Services

Globalization Services expone compatibilidad de globalización básica para idiomas y regiones para permitir la creación de aplicaciones globalizadas que se aplican a una amplia gama de clientes internacionales. Esta tecnología proporciona la infraestructura de globalización para admitir una experiencia Windows usuario coherente y lingüística y culturalmente adecuada. Globalization Services incluye las siguientes tecnologías:

-   [Servicios lingüísticos extendidos](extended-linguistic-services.md). Proporciona una variedad de servicios basados en texto para desarrolladores de Windows (incluida la detección de idiomas, la transliteración y otras funcionalidades lingüísticas principales) y la plataforma y las interfaces para que las aplicaciones consuman servicios fácilmente, así como interfaces para permitir a los desarrolladores agregar servicios propios.
-   [Componentes internacionales para Unicode (ICU).](international-components-for-unicode--icu-.md) Proporciona un conjunto maduro y ampliamente utilizado de API de globalización de código abierto. ICU usa el amplio repositorio de datos de configuración regional común (CLDR) de Unicode como biblioteca de datos, lo que proporciona compatibilidad de globalización con aplicaciones de software. ICU es ampliamente portátil y proporciona a las aplicaciones los mismos resultados en todas las plataformas.
-   [Compatibilidad con idiomas nacionales](national-language-support.md). Proporciona datos y API internacionales para admitir el formato de fecha, hora y calendario, diseños de teclado, comportamiento de ordenación, aspectos de la habilitación de texto y otra información de configuración regional principal.
-   [Spell Checking API](spell-checker-api.md). Proporciona una API para que los desarrolladores consuman la funcionalidad de revisión ortográfica para comprobar el texto, obtener sugerencias y mantener la configuración y los diccionarios de usuario. Los desarrolladores también pueden crear correctores ortográficas integrados en el marco de revisión ortográfica como proveedores.
