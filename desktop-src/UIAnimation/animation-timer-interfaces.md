# <a name="animation-timer-interfaces"></a>Interfaces del temporizador de animación

Esta sección contiene las especificaciones de referencia de las interfaces del administrador de animaciones de Windows que admiten temporizadores de animación.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                         | Descripción                                                                                   |
|-----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------|
| [**IUIAnimationTimer**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimer)<br/>                                     | Define un temporizador de animación, que proporciona servicios para administrar el control de tiempo de animación.<br/> |
| [**IUIAnimationTimerClientEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerclienteventhandler)<br/> | Define un método para controlar eventos relacionados con los cambios en el estado de cliente del temporizador.<br/>    |
| [**IUIAnimationTimerEventHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimereventhandler)<br/>             | Define métodos para controlar los eventos de control de tiempo.<br/>                                        |
| [**IUIAnimationTimerUpdateHandler**](/windows/desktop/api/UIAnimation/nn-uianimation-iuianimationtimerupdatehandler)<br/>           | Define métodos para controlar los eventos de actualización del tiempo.<br/>                                 |
