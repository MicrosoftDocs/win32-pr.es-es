# <a name="storage-creation-flags"></a>Storage Marcas de creación

Storage marcas de creación especifican qué debe hacer COM si un objeto de almacenamiento, secuencia o lockbytes existente tiene el mismo nombre que un nuevo objeto de almacenamiento o flujo que está creando. El valor predeterminado es devolver un mensaje de error y no crear el nuevo objeto. Solo puede usar una de estas marcas en una llamada de creación determinada.
