# <a name="preprocessor-operators"></a>Operadores de preprocesador

RC admite el uso de los operadores de preprocesamiento estándar de C en definiciones de macro. Estos operadores se describen en la tabla siguiente.



| Operador | Descripción                                                 |
|----------|-------------------------------------------------------------|
| \#       | Incluye el argumento entre comillas.                            |
| \#@      | Incluye el argumento entre comillas simples.                     |
| \#\#     | Concatena los tokens usados como argumentos para formar otros tokens. |
