# <a name="tapi-object"></a>Objeto TAPI

El objeto TAPI representa todos los recursos de telefonía a los que tiene acceso el equipo local, lo que permite a una aplicación enumerar todas las direcciones locales y remotas. Una aplicación TAPI 3 debe crear una instancia del objeto TAPI y, a continuación, inicializarse.

El [ejemplo de código Initialize TAPI](initialize-tapi.md) muestra un ejemplo de creación de objetos TAPI.

Vea [Interfaces de objetos TAPI](tapi-object-interfaces.md) para obtener una lista de todas las interfaces asociadas al objeto TAPI.
