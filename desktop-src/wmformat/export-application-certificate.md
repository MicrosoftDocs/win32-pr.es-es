# <a name="export-application-certificate"></a>Exportar certificado de aplicación

Para exportar Windows contenido protegido con DRM multimedia, debe obtener un certificado de aplicación de exportación de DRM multimedia de Windows, que recibirá después de entrar en el contrato de exportación de DRM con Microsoft.
