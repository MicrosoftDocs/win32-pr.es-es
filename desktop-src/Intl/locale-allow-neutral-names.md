# <a name="locale_allow_neutral_names"></a>la configuración \_ regional \_ permite \_ nombres neutros

**Windows 7 y versiones posteriores:** Permite la devolución de nombres neutros o LCID al convertir entre los nombres de configuración regional y los identificadores de configuración regional.
