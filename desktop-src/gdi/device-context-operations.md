# <a name="device-context-operations"></a>Operaciones de contexto de dispositivo

Una aplicación puede realizar las siguientes operaciones en un contexto de dispositivo:

-   Enumerar objetos gráficos existentes.
-   Seleccione nuevos objetos gráficos.
-   Eliminar objetos gráficos existentes.
-   Guarde los objetos gráficos actuales, sus atributos y los modos gráficos.
-   Restaure los objetos gráficos previamente guardados, sus atributos y los modos gráficos.

Además, una aplicación puede usar un contexto de dispositivo para:

-   Determine cómo se traduce la salida de gráficos.
-   Cancelar operaciones de dibujo largas (iniciadas por un subproceso en una aplicación multiproceso).
-   Restablezca una impresora a un estado determinado.
