# <a name="win32_tpmisreadyinformation-method"></a>Método \_ Tpm::IsReadyInformation de Win32

Indica si el TPM está listo y proporciona información adicional sobre el estado del TPM. El parámetro information devuelve una máscara de bits de información de lo que se necesita para aprovisionar completamente el TPM.

Los administradores locales solo pueden acceder a este método.

## <a name="syntax"></a>Sintaxis


```C++
uint32 IsReadyInformation(
  [out] BOOL   IsReady,
  [out] uint32 Information
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*IsReady* \[ out\]
</dt> <dd>

Establezca en **TRUE** si el TPM y el sistema están totalmente aprovisionados para su uso.

</dd> <dt>

*Información* \[ out\]
</dt> <dd>

Devuelve una máscara de bits de tanta información como esté disponible de lo que se necesita para aprovisionar completamente el TPM.
