# <a name="wmauthorurl"></a>WM/AuthorURL

El atributo **WM/AuthorURL** contiene la dirección del sitio web del autor.

## <a name="global-constant"></a>Constante global

g \_ wszWMAuthorURL

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

La dirección especificada por este atributo debe corresponderse con el artista o el autor en un atributo [**Author**](author.md) del archivo.
