# <a name="summary-information-stream"></a>Secuencia de información de resumen

El material de esta sección está destinado a desarrolladores que escriben sus propios programas de instalación que desean obtener más información sobre Windows de base de datos del instalador. Para obtener información general, vea [Acerca de Windows Installer](about-windows-installer.md).

El instalador usa el flujo de información de resumen para dos propósitos. En primer lugar, contiene información sobre el paquete que se puede ver a través de Microsoft Windows Explorer. Esta información es accesible a través de **la interfaz IStream.** En segundo lugar, contiene las propiedades que usa el instalador para instalar el paquete. En los temas siguientes se proporciona información sobre cómo usar el flujo de información de resumen con el instalador:

-   [Acerca del flujo de información de resumen](about-the-summary-information-stream.md)
-   [Uso del flujo de información de resumen](using-the-summary-information-stream.md)
-   [Referencia de flujo de información de resumen](summary-information-stream-reference.md)
