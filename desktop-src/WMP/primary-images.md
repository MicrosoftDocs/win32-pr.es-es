# <a name="primary-images"></a>Imágenes principales

Querrá crear imágenes principales para la máscara. La imagen principal se compone de una imagen de fondo definida por el **elemento VIEW** y cualquier otra imagen que vaya con elementos específicos. Por ejemplo, si crea un elemento **BUTTON,** debe incluir la imagen de ese botón y asegurarse de que tiene un orden Z superior al de la imagen de fondo.
