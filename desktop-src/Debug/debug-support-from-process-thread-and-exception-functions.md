# <a name="debug-support-from-process-thread-and-exception-functions"></a>Compatibilidad de depuración desde funciones de proceso, subproceso y excepción

Algunas funciones esenciales para la depuración se clasifican realmente como funciones de proceso, subproceso o control de excepciones. Para obtener una descripción de cómo iniciar un proceso que se va a depurar, vea [Funciones de proceso](process-functions-for-debugging.md). Para obtener información sobre cómo examinar y manipular el contexto y la ejecución de un subproceso, vea [Funciones de subproceso](thread-functions-for-debugging.md). Para obtener una descripción de cómo un depurador debe controlar las excepciones, vea [Funciones de control de excepciones](exception-handling-functions-for-debugging.md).
