# <a name="configuring-screen-capture-streams"></a>Configuración de la captura de pantalla Secuencias

Secuencias que usan el códec Windows media® Video 9 Screen están configurados por las aplicaciones de la misma manera que las secuencias de vídeo normales. Sin embargo, si establece el nivel de complejidad del vídeo en 0, el escritor omitirá cualquier valor de calidad de vídeo establecido con [**IWMVideoMediaProps::SetQuality**](/previous-versions/windows/desktop/api/Wmsdkidl/nf-wmsdkidl-iwmvideomediaprops-setquality). Para obtener más información, vea [Getting Good Results with the Windows Media Video 9 Screen Codec](getting-good-results-with-the-windows-media-video-9-screen-codec.md).
