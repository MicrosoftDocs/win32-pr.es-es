# <a name="codecapi_videoencoderdisplaycontenttype-property"></a>\_Propiedad VideoEncoderDisplayContentType de CODECAPI

Establece si el contenido está en pantalla completa, en lugar de en el contenido de pantalla que podría tener una ventana de vídeo más pequeña o no tener ningún vídeo.

## <a name="data-type"></a>Tipo de datos

**VT \_ UI4**

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ VideoEncoderDisplayContentType**

## <a name="property-value"></a>Valor de propiedad

Si el valor es distinto de cero, el contenido es un vídeo de pantalla completa. Si el valor es cero, el contenido no es un vídeo de pantalla completa.

## <a name="remarks"></a>Observaciones

En Windows 10, no se utiliza el valor de esta propiedad.
