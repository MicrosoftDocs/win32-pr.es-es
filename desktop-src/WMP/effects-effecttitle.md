# <a name="effectseffecttitle"></a>EFFECTs. effectTitle

El método **effectTitle** recupera el título de presentación de la visualización con el índice de registro especificado.

``` syntax
        elementID.effectTitle(index)
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

<span id="index"></span><span id="INDEX"></span>*ajustar*
</dt> <dd>

**Número** (**largo**) que contiene el índice del registro de una visualización.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Este método devuelve una **cadena**.

## <a name="remarks"></a>Observaciones

Este método se usa para mostrar los títulos de visualización en una interfaz de usuario.
