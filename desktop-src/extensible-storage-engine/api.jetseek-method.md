# <a name="apijetseek-method"></a>Método Api.JetSeek

Coloca eficazmente un cursor en una entrada de índice que coincide con los criterios de búsqueda especificados por la clave de búsqueda en ese cursor y la desigualdad especificada. Una clave de búsqueda se debe haber construido previamente mediante [JetMakeKey(JET_SESID, JET_TABLEID, \[ \] , Int32, MakeKeyGrbit).](./api.jetmakekey-method.md) Consulte también [TrySeek(JET_SESID, JET_TABLEID, SeekGrbit).](./api.tryseek-method.md)

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop](./microsoft.isam.esent.interop-namespace.md)  
**Ensamblado:**  Microsoft.Isam.Esent.Interop (en Microsoft.Isam.Esent.Interop.dll)
