# <a name="setup-applications"></a>Instalación de aplicaciones

Normalmente, se crea un *archivo INF* antes de crear una aplicación de instalación. Un archivo INF es un archivo de texto que contiene información usada por la aplicación de instalación durante una instalación. Entre los ejemplos de la información que se puede almacenar en un archivo INF se incluyen INI y cambios del registro, nombres de archivo y ubicaciones de los archivos de origen en medios de origen.

Después de crear un archivo INF, escriba el código fuente de la aplicación de instalación. Puede usar las funciones de configuración de la aplicación de instalación de para tener acceso a la información del archivo INF y realizar las operaciones de instalación.

En las secciones siguientes se describe el uso de archivos INF y las funciones de procesamiento de la instalación:

-   [Acerca de los archivos INF](about-inf-files.md)
-   [Uso de archivos INF](using-inf-files.md)
-   [Funciones de configuración de archivos INF](inf-file-setup-functions.md)

Para obtener más información sobre el uso de las funciones de instalación en una aplicación, vea [crear aplicaciones de instalación](creating-setup-applications.md).
