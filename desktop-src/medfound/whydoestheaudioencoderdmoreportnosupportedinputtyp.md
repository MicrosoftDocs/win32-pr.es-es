# <a name="why-does-the-audio-encoder-report-no-supported-input-types"></a>¿Por qué el codificador de audio no informa de los tipos de entrada admitidos?

El codificador de audio no puede determinar los tipos de entrada admitidos hasta que establezca un tipo de salida. Enumere los tipos de salida y seleccione el que coincida con sus requisitos. Después de establecer el tipo de salida, el objeto devolverá los tipos de entrada admitidos.
