# <a name="d1197-wrong-d3d-device"></a>D1197: dispositivo D3D equivocado

La superficie de DXGI pasada a esta API es del dispositivo equivocado.



|             |         |
|-------------|---------|
| Nivel de error | Advertencia |



 

## <a name="possible-causes"></a>Causas posibles

La superficie de DXGI pasada a esta API es del dispositivo equivocado.

## <a name="possible-fixes"></a>Posibles correcciones

Compruebe que las superficies de DXGI tienen el mismo dispositivo D3D que el contexto de dispositivo D2D.
