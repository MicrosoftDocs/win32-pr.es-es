# <a name="network-share-management"></a>Administración de recursos compartidos de red

La administración de recursos compartidos de red permite a las aplicaciones administrar y supervisar las comunicaciones entre los clientes y servidores de Windows mediante el protocolo de bloque de mensajería del servidor (SMB).
