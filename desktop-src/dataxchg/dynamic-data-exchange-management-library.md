# <a name="dynamic-data-exchange-management-library"></a>Biblioteca de administración de Intercambio dinámico de datos

Intercambio dinámico de datos (DDE) es una forma de comunicación entre procesos que usa la memoria compartida para intercambiar datos entre aplicaciones. Las aplicaciones pueden utilizar DDE para las transferencias de datos de un solo uso y para intercambios y actualizaciones continuas de los datos.

|--------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|

La función de devolución de llamada del cliente DDE, [**DdeCallback**](/windows/win32/api/ddeml/nc-ddeml-pfncallback), recibe esta transacción después de establecer un bucle de notificaciones con un servidor. <br/>                                                                                                                                                                                                                                                                                              |
| [**XTYP \_ ADVREQ**](xtyp-advreq.md)                    | Informa al servidor de que una transacción de notificación está pendiente en el par de nombre de tema y nombre de elemento especificado y que han cambiado los datos correspondientes a los pares nombre de tema y nombre de elemento. El sistema envía esta transacción a la función de devolución de llamada DDE, [**DdeCallback**](/windows/win32/api/ddeml/nc-ddeml-pfncallback), una vez que el servidor llama a la función [**DdePostAdvise**](/windows/desktop/api/Ddeml/nf-ddeml-ddepostadvise) . <br/>                                                                                                                                     |
| [**XTYP \_ ADVSTART**](xtyp-advstart.md)                | Un cliente usa la transacción [**XTYP \_ ADVSTART**](xtyp-advstart.md) para establecer un bucle de notificación con un servidor. Una función de devolución de llamada del servidor DDE, [**DdeCallback**](/windows/win32/api/ddeml/nc-ddeml-pfncallback), recibe esta transacción cuando un cliente especifica **XTYP \_ ADVSTART** como el parámetro *wType* de la función [**DdeClientTransaction**](/windows/desktop/api/Ddeml/nf-ddeml-ddeclienttransaction) . <br/>                                                                                                                                                      |

Una función de devolución de llamada del servidor DDE, [**DdeCallback**](/windows/win32/api/ddeml/nc-ddeml-pfncallback), recibe esta transacción cuando un cliente especifica un nombre de servicio que admite el servidor (y un nombre de tema que no es **null**) en una llamada a la función [**DdeConnect**](/windows/desktop/api/Ddeml/nf-ddeml-ddeconnect) . <br/>                                                                                                                                            |
| [**\_confirmación de conexión de XTYP \_**](xtyp-connect-confirm.md) | Una función de devolución de llamada del servidor DDE, [*DdeCallback*](/windows/win32/api/ddeml/nc-ddeml-pfncallback), recibe la transacción de [**confirmación de XTYP \_ Connect \_**](xtyp-connect-confirm.md) para confirmar que se ha establecido una conversación con un cliente y para proporcionar al servidor el identificador de conversación. El sistema envía esta transacción como resultado de una transacción [**XTYP \_ Connect**](xtyp-connect.md) o [**XTYP \_ WILDCONNECT**](xtyp-wildconnect.md) anterior. <br/>                                                                   |
| [**desconexión de XTYP \_**](xtyp-disconnect.md)            | La función de devolución de llamada DDE de una aplicación, [*DdeCallback*](/windows/win32/api/ddeml/nc-ddeml-pfncallback), recibe la transacción [**XTYP \_ Disconnect**](xtyp-disconnect.md) cuando el asociado de la aplicación en una conversación usa la función [**DdeDisconnect**](/windows/desktop/api/Ddeml/nf-ddeml-ddedisconnect) para finalizar la conversación. <br/>                                                                                                                                                                                                                       |

| [**XTYP \_ anular registro**](xtyp-unregister.md)            | Una función de devolución de llamada DDE, [*DdeCallback*](/windows/win32/api/ddeml/nc-ddeml-pfncallback), recibe la transacción de [**\_ anulación del registro de XTYP**](xtyp-unregister.md) siempre que una aplicación de servidor ddeml usa la función [**DdeNameService**](/windows/desktop/api/Ddeml/nf-ddeml-ddenameservice) para anular el registro de un nombre de servicio o cuando se termina una aplicación que no es ddeml que admite el tema del sistema. <br/>                                                                                                                                                                |
| [**XTYP \_ WILDCONNECT**](xtyp-wildconnect.md)          | La transacción [**XTYP \_ WILDCONNECT**](xtyp-wildconnect.md) permite a un cliente establecer una conversación en cada uno de los pares nombre de servicio y nombre de tema del servidor que coinciden con el nombre de servicio y el nombre de tema especificados. | [**\_transacción XTYP \_ completada**](xtyp-xact-complete.md)     | Una función de devolución de llamada de cliente DDE, [*DdeCallback*](/windows/win32/api/ddeml/nc-ddeml-pfncallback), recibe la transacción de transacciones [**\_ \_ completadas de XTYP**](xtyp-xact-complete.md) cuando se completa una transacción asincrónica, iniciada por una llamada a la función [**DdeClientTransaction**](/windows/desktop/api/Ddeml/nf-ddeml-ddeclienttransaction) . <br/>                                                                                                                                                                                                                      |
