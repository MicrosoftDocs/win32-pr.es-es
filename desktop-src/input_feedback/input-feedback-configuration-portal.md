# <a name="input-feedback-configuration"></a>Configuración de comentarios de entrada

## <a name="purpose"></a>Propósito

Los temas de esta sección proporcionan información general sobre la configuración de comentarios de entrada en Windows 8. Configuración de comentarios de entrada proporciona acceso a la configuración de comentarios visuales para las interacciones del usuario.

En la colección de temas que se proporcionan aquí se describe cómo usar la configuración de comentarios de entrada para optimizar la experiencia del usuario de la aplicación. Por ejemplo, la configuración predeterminada de comentarios visuales podría no ser necesaria o deseable en una aplicación de juego debido a interferencias o conflictos con el juego en sí.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Las API de configuración de comentarios de entrada están diseñadas para desarrolladores que están creando marcos de interfaz de usuario que proporcionan una experiencia de usuario coherente optimizada para la función táctil en todas las aplicaciones de escritorio.
