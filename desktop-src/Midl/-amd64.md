# <a name="amd64-switch"></a>modificador/AMD64

El modificador **/AMD64** indica al compilador de MIDL que genere archivos de código auxiliar, o un archivo de biblioteca de tipos, para un entorno avanzado de micro devices 64 bits.

``` syntax
midl /amd64
```

## <a name="remarks"></a>Observaciones

> [!Note]  
> El modificador **/x64** descrito por la ayuda de la línea de comandos MIDL.exe está en desuso. en su lugar, se debe usar **/AMD64** .
