# <a name="raytracing-hlsl-structures-direct3d-12"></a>Raytracing estructuras HLSL (Direct3D 12)

Las siguientes estructuras HLSL admiten la canalización raytracing de Direct3D 12.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                                       | Descripción                                                                                                                                                                                                                                                                                             |
|-------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Estructura del parámetro de llamadas**](call-parameter-structure.md)<br/>                              | Una estructura definida por el usuario que se proporciona como un argumento INOUT a una llamada a CallShader y como un parámetro INOUT para el sombreador al que se puede llamar.<br/>                                                                                                                                                                                                                                              |
| [**Estructura de atributos de intersección**](intersection-attributes.md)<br/>                              | Una estructura definida por el usuario que se proporciona como un argumento INOUT en una llamada [**TraceRay**](traceray-function.md) y como un parámetro INOUT en los tipos de sombreador que pueden tener acceso a la carga de rayo.<br/>                                                                                                                                                                                                                                              |
| [**Estructura de la carga de rayos**](ray-payload.md)<br/>                              | Una estructura definida por el usuario que se proporciona como un argumento INOUT en una llamada [**TraceRay**](traceray-function.md) y como un parámetro INOUT en los tipos de sombreador que pueden tener acceso a la carga de rayo.<br/>                                                                                                                                                                                                                                              |
| [**Estructura RayDesc**](raydesc.md)<br/>                              | Marcas que se pasan a la función [**TraceRay**](traceray-function.md) para invalidar la transparencia, la selección y el comportamiento de tiempo de espera.<br/>                                                                                                                                                                                                                                              |
