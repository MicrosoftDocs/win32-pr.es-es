# <a name="swbemobjectpathauthority-property"></a>Propiedad SWbemObjectPath. Authority

La propiedad **Authority** del objeto [**SWbemObjectPath**](swbemobjectpath.md) contiene una cadena que define el componente de autoridad de la ruta de acceso del objeto. Para obtener más información, vea el parámetro *strAuthority* del método [**SWbemLocator. ConnectServer**](swbemlocator-connectserver.md) y [establezca la seguridad en IWbemServices y en otros servidores proxy](setting-the-security-on-iwbemservices-and-other-proxies.md).

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemObjectPath.Authority As String
```
