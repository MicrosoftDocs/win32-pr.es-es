# <a name="memory-management-memory-management"></a>Administración de memoria (administración de memoria)

El administrador de memoria implementa la memoria virtual, proporciona un conjunto básico de servicios, como archivos asignados a la memoria, memoria de copia en escritura, compatibilidad con memoria grande y compatibilidad subyacente para el administrador de caché.
