Para Windows 2000, Windows Server 2003 y sistemas operativos posteriores, use la Referencia de administración del servicio [de acceso remoto](remote-access-service-administration-reference.md).

-   [**RasAdminPortClearStatistics**](rasadminportclearstatistics.md)
-   [**RasAdminPortDisconnect**](rasadminportdisconnect.md)
-   [**RasAdminPortEnum**](rasadminportenum.md)
-   [**RasAdminPortGetInfo**](rasadminportgetinfo.md)
-   [**RasAdminServerGetInfo**](rasadminservergetinfo.md)
-   [**RasAdminUserGetInfo**](rasadminusergetinfo.md)
-   [**RasAdminUserSetInfo**](rasadminusersetinfo.md)
-   [Funciones DLL de administración de RAS](ras-administration-dll-functions.md)
