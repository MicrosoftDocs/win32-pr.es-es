# <a name="sensor-properties"></a>Propiedades del sensor

La plataforma de sensor y ubicación define constantes que identifican las propiedades de los sensores. Los fabricantes de sensores también pueden definir sus propias propiedades.

La plataforma define los siguientes valores de **PROPERTYKEY** para las propiedades del sensor. Estas propiedades son de solo lectura, a menos que se indique lo contrario.

Cada propiedad de sensor de la plataforma definida con **PROPERTYKEY** se basa en un **GUID** común denominado \_ GUID común de propiedad de sensor \_ \_ :

{7F8383EC-D3EC-495C-A8CF-B8BBE85C2920}.

> [!IMPORTANT]
> No utilice este valor base para definir sus propias claves de propiedad.

 

La aplicación cliente puede especificar los valores de las propiedades designadas como de lectura/escritura. Los valores de las propiedades designadas como static no deben cambiar con el tiempo. Las propiedades designadas como requeridas deben ser compatibles con el sensor.
