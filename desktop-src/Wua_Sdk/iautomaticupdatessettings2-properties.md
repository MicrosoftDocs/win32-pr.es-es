# <a name="iautomaticupdatessettings2-properties"></a>Propiedades IAutomaticUpdatesSettings2

La [**interfaz IAutomaticUpdatesSettings2**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings2) define la propiedad siguiente.



| Propiedad                                                                                  | Descripción                                                                                                                                                         |
|-------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IncludeRecommendedUpdates**](/windows/desktop/api/Wuapi/nf-wuapi-iautomaticupdatessettings2-get_includerecommendedupdates) | Obtiene y establece un valor booleano que indica si se deben incluir actualizaciones opcionales o recomendadas cuando se realiza una búsqueda de actualizaciones e instalación de actualizaciones. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz.

-   [**IAutomaticUpdatesSettings**](/windows/desktop/api/Wuapi/nn-wuapi-iautomaticupdatessettings)
