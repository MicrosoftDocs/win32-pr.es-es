# <a name="automenu-element"></a>Automenu (elemento)

El elemento de **menú automenu** proporciona una manera de mostrar el **Panel de acceso rápido** en una máscara. Se trata de un menú que aparece en el modo completo del reproductor al hacer clic en la flecha situada a la derecha del botón **reproducción en curso** . Proporciona acceso inmediato a archivos multimedia digitales organizados de varias maneras, como en listas de reproducción o por álbum, intérprete o género. Use los atributos **izquierdo** y **superior** para especificar la ubicación en la que debe aparecer el menú cuando se llama al método **Show** .

El elemento de **menú automenu** admite el método siguiente.



| Método                    | Descripción                          |
|---------------------------|--------------------------------------|
| [show](automenu-show.md) | Muestra el **Panel de acceso rápido**. |



 

El elemento de **menú automenu** admite los siguientes atributos de ambiente: left y Top.

> [!Note]  
> Este elemento requiere Windows Media Player 9 series o posterior.
