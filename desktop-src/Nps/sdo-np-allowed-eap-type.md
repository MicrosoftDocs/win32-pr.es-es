# <a name="np-allowed-eap-type"></a>NP-permitido-EAP-Type

Modos de cifrado EAP permitidos para una conexión.

**Nombre LDAP:** msNPAllowedEapType

**ID:** 4106 = [ **\_ atributo IAS \_ \_ tipo de \_ EAP \_ permitido**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintaxis:** Entero

**Valores  **

Definida por Internet Assigned Numbers Authority (IANA) en el documento [del registro del Protocolo de autenticación extensible (EAP)](https://www.iana.org/assignments/eap-numbers) basado en [RFC 3748](https://www.ietf.org/rfc/rfc3748.txt).

-   **MD5: desafío** (4)
-   **EAP \_ de TLS** (13)
-   **PEAP** (25)
-   **MS-EAP-autenticación** (26)
