# <a name="em_findtext-message"></a>Mensaje \_ EM FINDTEXT

Busca texto dentro de un control de edición enriquecido.

<dl> <dt>

## <a name="return-value"></a>Valor devuelto

Si se encuentra la cadena de destino, el valor devuelto es la posición de base cero del primer carácter de la coincidencia. Si no se encuentra el destino, el valor devuelto es -1.

## <a name="remarks"></a>Comentarios

El **miembro cpMin** de **FINDTEXT.chrg** siempre especifica el punto inicial de la búsqueda y **cpMax** especifica el punto final. Al buscar hacia atrás, **cpMin** debe ser igual o mayor que **cpMax.** Al buscar hacia delante, un valor de -1 en **cpMax** extiende el intervalo de búsqueda al final del texto.
