# <a name="type-baseeapparameters-element"></a>Elemento Type (BaseEapParameters)

El **elemento Type (BaseEapParameters)** define el elemento de marcador de posición para el tipo de método seleccionado y la configuración específica del método.

``` syntax
<xs:element name="Type"
    type="integer"
 />
```

El **tipo** complejo [**BaseEapParameters**](baseeapconnectionpropertiesv1schema-baseeapparameters-complextype.md) define el elemento Type.
