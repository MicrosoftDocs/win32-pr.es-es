# <a name="labeleduri-attribute"></a>labeledURI (atributo)

Un identificador uniforme de recursos seguido de una etiqueta. La etiqueta se usa para describir el recurso al que apunta el URI y está pensado como un nombre para mostrar que es legible para el usuario.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | labeledURI                                  |
| Nombre para mostrar de LDAP | labeledURI                                  |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.3.6.1.4.1.250.1.57                        |
| System-ID-GUID    | c569bb46-c680-44bc-a273-e6c227d71b45        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
