# <a name="iwmpplaylistarray-vb-and-c-interface"></a>Interfaz IWMPPlaylistArray (VB y C#)

Proporciona una propiedad y un método para obtener acceso a una colección de interfaces **IWMPPlaylist** por número de índice.

La **interfaz IWMPPlaylistArray** expone la siguiente propiedad.

## <a name="members"></a>Miembros

La **interfaz IWMPPlaylistArray (VB y C#)** tiene estos tipos de miembros:

-   [Métodos](#methods)
-   [Propiedades](#properties)

### <a name="methods"></a>Métodos

La **interfaz IWMPPlaylistArray (VB y C#)** tiene estos métodos.



| Método                                                                   | Descripción                                                                                        |
|:-------------------------------------------------------------------------|:---------------------------------------------------------------------------------------------------|
| [**Elemento**](wmplibiwmpplaylistarray-iwmpplaylistarray-item--vb-and-c.md) | Devuelve una **interfaz IWMPPlaylist que** representa la lista de reproducción en el índice especificado.<br/> |



 

### <a name="properties"></a>Propiedades

La **interfaz IWMPPlaylistArray (VB y C#)** tiene estas propiedades.



| Propiedad                                                                              | Tipo de acceso          | Descripción                                                   |
|:--------------------------------------------------------------------------------------|:---------------------|:--------------------------------------------------------------|
| [**Contar**](wmplibiwmpplaylistarray-iwmpplaylistarray-count--vb-and-c.md)<br/> | Solo lectura<br/> | Obtiene el número de listas de reproducción de la matriz de listas de reproducción.<br/> |



 

Obtenga una **interfaz IWMPPlaylistArray** mediante los métodos siguientes.



| Interfaz                                                      | Método                                                                                                                                                                               |
|----------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [IWMPPlaylistCollection](iwmpplaylistcollection--vb-and-c.md) | [**getAll**](wmplibiwmpplaylistcollection-iwmpplaylistcollection-getall--vb-and-c.md), [ **getByName**](wmplibiwmpplaylistcollection-iwmpplaylistcollection-getbyname--vb-and-c.md) |
