# <a name="account-object"></a>Objeto account

El **objeto Account** se usa para asignar [*privilegios,*](/windows/desktop/SecGloss/p-gly)acceso al sistema y cuotas especiales a usuarios individuales o a miembros de grupos o grupos locales.

La información almacenada en un **objeto Account** incluye:

-   SID de una cuenta de usuario, cuenta de grupo o cuenta de grupo local.
-   Lista de privilegios que se asignarán al usuario o a los miembros de un grupo. Si varios objetos **Account** asignan privilegios a un usuario, la lista incluye todos los privilegios asignados.
