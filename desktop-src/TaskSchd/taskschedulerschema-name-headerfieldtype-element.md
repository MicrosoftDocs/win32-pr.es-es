# <a name="name-headerfieldtype-element"></a>Elemento Name (headerFieldType)

Contiene el nombre de un campo de encabezado en un mensaje de correo electrónico.

``` syntax
<xs:element name="Name"
    type="nonEmptyString"
 />
```

El **elemento Name** se define mediante el tipo complejo [**headerFieldType.**](taskschedulerschema-headerfieldtype-complextype.md)

## <a name="parent-element"></a>Elemento primario



| Elemento                                                                         | Derivado de                                                               | Descripción                                                    |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------|----------------------------------------------------------------|
| [**HeaderField**](taskschedulerschema-headerfield-headerfieldstype-element.md) | [**headerFieldType**](taskschedulerschema-headerfieldtype-complextype.md) | Contiene un campo para un encabezado en un mensaje de correo electrónico. <br/> |
