# <a name="proof-of-possession-cookie-api"></a>API de prueba de posesión de cookies

Permite crear cookies de prueba de posesión, que admiten el inicio de sesión único para las aplicaciones web autenticadas por el protocolo de autenticación de Internet de OAuth. Puede usar la información devuelta por esta API para implementar cuentas de Microsoft de formulario de inicio de sesión único.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                         | Descripción                                                      |
|-----------------------------------------------------------------------------------------------|------------------------------------------------------------------|
| [**IProofOfPossessionCookieInfoManager**](/windows/desktop/api/ProofOfPossessionCookieInfo/nn-proofofpossessioncookieinfo-iproofofpossessioncookieinfomanager)<br/> | Admite la creación de las cookies de prueba de posesión.<br/> |
