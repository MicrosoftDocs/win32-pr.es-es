# <a name="native-wifi"></a>Wi-Fi nativa

## <a name="purpose"></a>Propósito

El componente de configuración automática WiFi nativa configura, se conecta a y desconecta de las redes inalámbricas. Wi-Fi nativo puede almacenar perfiles en las redes con las que interactúa en forma de documentos XML.

## <a name="developer-audience"></a>Audiencia de desarrolladores

La API WiFi nativa está diseñada para desarrolladores de C/C++. Los programadores deben estar familiarizados con los conceptos y la terminología de redes inalámbricas.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

El componente Wi-Fi nativo requiere clientes que ejecuten Windows Vista, Windows XP con Service Pack 3 (SP3) o la API de LAN inalámbrica para Windows XP con Service Pack 2 (SP2).
