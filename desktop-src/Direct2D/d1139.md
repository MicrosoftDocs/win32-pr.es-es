# <a name="d1139-unused-stroke-width"></a>D1139: ancho de trazo sin usar

Se especificó un tipo de transformación de trazo de fino, pero el ancho del trazo proporcionado no era 1. Se omitirá el ancho del trazo.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación dibujó un primitivo mediante ID2D1StrokeStyle con \_ el tipo de transformación de trazo D2D1 \_ \_ \_ fino, pero el ancho del trazo usado en esa llamada a Draw no era 1. Cuando se especifica el atributo fino, de hecho, reemplaza el ancho del trazo y lo corrige en unidades dependientes del dispositivo 1,0, independientemente de la transformación de escala o la opción de PPP.

## <a name="possible-fixes"></a>Posibles correcciones

Cuando se establece el atributo fino en un ID2D1StrokeStyle, la aplicación debe usar un ancho de trazo de 1,0 unidades.
