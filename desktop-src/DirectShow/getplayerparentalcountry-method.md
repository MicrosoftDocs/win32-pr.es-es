# <a name="getplayerparentalcountry-method"></a>Método GetPlayerParentalCountry

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

El `GetPlayerParentalCountry` método recupera el país o región actual establecido en el objeto **MSWebDVD** .

``` syntax
[ iCountryCode = ] MSWebDVD.GetPlayerParentalCountry()
```

## <a name="return-value"></a>Valor devuelto

Devuelve un valor entero con el código de país o región.
