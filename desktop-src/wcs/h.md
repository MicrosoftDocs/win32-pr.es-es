# <a name="h"></a>H

## <a name="hard-proofing"></a>prueba de hardware

Obtener una vista previa o probar una imagen imprimiendo una copia impresa de la misma.

## <a name="hls"></a>HLS

Un espacio de colores que suelen usar los artistas. Sus componentes son matiz, luminosidad y saturación (croma).

## <a name="hsv"></a>HSV

Un espacio de colores que suelen usar los artistas. Sus componentes son matiz, saturación (croma) y valor (luminosidad).

## <a name="hue"></a>altera

Color puro. Un matiz es una posición específica en la parte visible del espectro electromagnético.
