# <a name="winsat-samples"></a>Ejemplos de WinSAT

\[Los ejemplos de WinSAT pueden modificarse o no estar disponibles para las versiones después de Windows 8.1.\]

La herramienta de evaluación del sistema de Windows (WinSAT) proporciona los siguientes ejemplos en el [Windows SDK](https://msdn.microsoft.com/windows/bb980924.aspx).



| Muestra             | Descripción                                                                    |
|--------------------|--------------------------------------------------------------------------------|
| InitiateAssessment | Muestra cómo iniciar evaluaciones formales y personalizadas.                           |
| QueryWinSAT        | Muestra cómo consultar los almacenes de datos de WinSAT para obtener los resultados de una evaluación. |
