# <a name="wireless-zero-configuration-functions"></a>Funciones de configuración inalámbrica cero

\[La interfaz de programación de configuración de la red inalámbrica ya no es compatible con Windows Vista y Windows Server 2008. En su lugar, use la [API WiFi nativa](native-wifi-reference.md), que proporciona una funcionalidad similar. Para obtener más información, consulte [acerca de la API WiFi nativa](about-the-native-wifi-api.md).\]

Las siguientes funciones permiten a los desarrolladores de software usar la funcionalidad proporcionada por el módulo de configuración inalámbrica rápida.

-   [**WZCEapolGetInterfaceParams**](wzceapolgetinterfaceparams.md)
-   [**WZCEnumInterfaces**](wzcenuminterfaces.md)
-   [**WZCQueryInterface**](wzcqueryinterface.md)
-   [**WZCRefreshInterface**](wzcrefreshinterface.md)
