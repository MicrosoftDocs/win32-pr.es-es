# <a name="locale_igeoid"></a>\_IGEOID de CONFIGURACIÓN REGIONAL

Número de 32 bits con firma que identifica de forma única una [ubicación geográfica.](table-of-geographical-locations.md) La aplicación usa esta constante para proporcionar servicios específicos de la configuración regional a los clientes. Por ejemplo, se puede usar como clave para acceder a un registro de base de datos que contiene información específica sobre un país o región.
