# <a name="usage"></a>Uso

Hay dos casos generales de uso de la interfaz DLL de la interfaz de usuario de TSP: cuando la aplicación invoca una función diseñada para generar la interfaz de usuario y cuando el proveedor de servicios debe generar la interfaz de usuario de forma desa prueba durante el procesamiento de una función que no está definida como generación de la interfaz de usuario.
