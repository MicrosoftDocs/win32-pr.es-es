# <a name="solid-foundation"></a>Base sólida

Windows 7 proporciona una plataforma de desarrollo y herramientas muy productivas que ofrecen aspectos fundamentales sobre el sistema operativo básico. Windows 7 se basa en la plataforma de Windows Vista, lo que permite a los desarrolladores crear aplicaciones que son compatibles con ambas plataformas y limitar los problemas de compatibilidad de aplicaciones. Windows 7 mejora drásticamente el rendimiento y la administración de energía para que pueda crear aplicaciones que optimicen la experiencia móvil. Como resultado, las aplicaciones serán más atractivas visualmente, más fáciles de crear y admitir una amplia gama de normas internacionales. También se han realizado avances en el soporte global, la accesibilidad y la implementación de aplicaciones.
