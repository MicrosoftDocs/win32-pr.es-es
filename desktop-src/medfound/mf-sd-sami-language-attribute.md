# <a name="mf_sd_sami_language-attribute"></a>MF \_ SD \_ ( \_ atributo de lenguaje Sami)

Contiene el nombre del idioma de intercambio de medios accesibles (SAMI) sincronizado que se define para la secuencia.

Este atributo está presente en los descriptores de flujo devueltos desde el origen de medios SAMI.

## <a name="data-type"></a>Tipo de datos

Cadena de caracteres anchos

## <a name="remarks"></a>Observaciones

El nombre de lenguaje SAMI se especifica en el archivo SAMI.

La constante GUID para este atributo se exporta desde mfuuid. lib.
