# <a name="propertyname-control-attribute"></a>Atributo de control PropertyName

Es el nombre de la propiedad asociada a este control. Si no se establece el bit [indirecto](indirect-control-attribute.md) , el control muestra o cambia el valor de la propiedad que tiene este nombre. Este atributo se especifica en la columna de propiedades de la [tabla de control](control-table.md). Este atributo devuelve null para los controles inertes.

## <a name="valid-controls"></a>Controles v√°lidos

Todos los controles.

## <a name="associated-bit-flags"></a>Marcas de bits asociadas

Este atributo no utiliza marcas de bits.
