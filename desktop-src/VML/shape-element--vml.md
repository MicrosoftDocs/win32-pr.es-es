# <a name="shape-element-vml"></a>Elemento Shape (VML)

En este tema se describe VML, una característica que está en desuso a partir Windows Internet Explorer 9. Las páginas web y las aplicaciones que se basan en VML deben migrarse a SVG u otros estándares ampliamente admitidos.

> [!Note]  
> A partir de diciembre de 2011, este tema se archivó. Como resultado, ya no se mantiene activamente. Para obtener más información, vea [Contenido archivado.](/previous-versions/windows/internet-explorer/ie-developer/) Para obtener información, recomendaciones e instrucciones sobre la versión actual de Windows Internet Explorer, [vea Internet Explorer Developer Center](https://msdn.microsoft.com/ie/).

**Comentarios:**

Este elemento puede aparecer por sí solo o dentro de [un elemento Group.](msdn-online-vml-group-element.md)

A continuación se muestra el código mínimo necesario para generar una forma. Debe definir los atributos **FillColor**, **Position**, **Top**, **Left**, **Width**, **Height** y **Path** o la forma no se representará.


```HTML
   <v:shape
   fillcolor="green"
   style="position:relative;top:1;left:1;width:200;height:200"
   path = "m 1,1 l 1,200, 200,200, 200,1 x e">
   </v:shape>
```



Además, debe colocar el código siguiente en el elemento [**HTML**](https://msdn.microsoft.com/library/ms535255(v=VS.85).aspx) de la página web para obtener los esquemas correctos para VML y Microsoft Office Extensions. Si no usa extensiones de Office, puede omitir el segundo esquema.


```HTML
xmlns:v="urn:schemas-microsoft-com:vml"
xmlns:o="urn:schemas-microsoft-com:office:office"
```



También debe tener lo siguiente en el elemento [**STYLE**](https://developer.mozilla.org/docs/Web/HTML/Element/style) para registrar el comportamiento de VML y Microsoft Office Extensions. Si no usa extensiones de Office, puede omitir la segunda definición de comportamiento.


```HTML
v\:* { behavior: url(#default#VML); display:inline-block}
o\:* { behavior: url(#default#VML); }
```

**Ejemplo**

El código siguiente consta de una página web de VML de ejemplo:


```HTML
<HTML xmlns:v="urn:schemas-microsoft-com:vml"
xmlns:o="urn:schemas-microsoft-com:office:office">
<HEAD>
<STYLE>
v\:* { behavior: url(#default#VML);}
o\:* { behavior: url(#default#VML);}
</STYLE>
<TITLE>VML Sample</TITLE>
</HEAD>
<BODY>
   <v:shape
   fillcolor="green"
   style="position:relative;top:1;left:1;width:200;height:200"
   path = "m 1,1 l 1,200, 200,200, 200,1 x e">
   </v:shape>
</BODY>
</HTML>
```
