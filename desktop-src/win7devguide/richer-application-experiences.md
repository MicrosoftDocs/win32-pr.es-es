# <a name="richer-application-experiences"></a>Experiencias de aplicación más enriquecidas

Windows 7 permite a los desarrolladores crear aplicaciones distintivas e intuitivas que mejoran considerablemente la detectabilidad, la facilidad de uso y el disfrute. Los nuevos métodos de integración de escritorio ponen la funcionalidad de la aplicación al alcance de la mano del usuario y el explorador y las *bibliotecas* de Windows proporcionan un acceso fácil a la información de gran valor. El control de cinta de Windows y el marco de animación de Windows facilitan la creación de interfaces de usuario interactivas y atractivas. Las nuevas API de Touch permiten interacciones naturales a través de las API de múltiples funciones táctiles y de movimiento panorámico y de manipulación y inercia permiten efectos visuales impresionantes.

Los avances rápidos en tecnología de hardware y software también impulsan experiencias de usuario de mayor fidelidad. Windows 7 aporta estos avances en el control del desarrollador con API nuevas y flexibles que aprovechan al máximo la tecnología, a la vez que facilita aún más el desarrollo de aplicaciones atractivas.
