# <a name="translating-to-c"></a>Traducir a C++

Visual C++ proporciona varias herramientas que le ayudarán a agregar objetos COM al proyecto. Puede Agregar un control ActiveX al proyecto, utilizar el Asistente para clases MFC o ejecutar el compilador MIDL en un archivo IDL.

Si usa un entorno de desarrollo de C++ distinto de Visual C++, consulte la documentación del producto para obtener información sobre las herramientas de conversión de COM.

Una vez que haya agregado los archivos de inclusión o de código auxiliar del objeto COM al proyecto, la aplicación puede utilizar las clases del objeto COM.

Visual C++ también proporciona el visor de objetos OLE-COM, que puede usar para ver la información de la biblioteca de tipos compilada. El visor de objetos presenta las clases COM en la sintaxis de C++.
