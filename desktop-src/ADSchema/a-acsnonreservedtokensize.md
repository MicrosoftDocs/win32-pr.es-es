# <a name="acs-non-reserved-token-size-attribute"></a>Atributo ACS-no reservado-tamaño de token

El atributo **ACS-no reservado-token-size** solo es para uso interno. Basado en RFC2814.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | ACS-no reservado-tamaño del token          |
| Nombre para mostrar de LDAP | aCSNonReservedTokenSize              |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1319              |
| System-ID-GUID    | a916d7c9-3b90-11d2-90cc-00c04fd91ab1 |
| Sintaxis            | [**Interval**](s-interval.md)       |
