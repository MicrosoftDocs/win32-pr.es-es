# <a name="events-media-streaming-api"></a>Eventos (API de streaming de multimedia)

La [API de streaming de multimedia](media-streaming-api-portal.md) genera los siguientes eventos.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                           | Descripción                                                                                                                                              |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Evento ConnectionStatusChanged**](connectionstatuschanged.md)<br/>     | Se produce cuando cambia el estado de la conexión de red del dispositivo.<br/>                                                                                   |
| [**Evento DeviceArrival**](devicearrival.md)<br/>                         | Se produce cuando se agrega un nuevo dispositivo a la lista de dispositivos devueltos por el método [**CachedDevices**](idevicecontroller-cacheddevices.md) .<br/>     |
| [**Evento DeviceDeparture**](devicedeparture.md)<br/>                     | Se produce cuando se quita un nuevo dispositivo de la lista de dispositivos devueltos por el método [**CachedDevices**](idevicecontroller-cacheddevices.md) .<br/> |
| [**Evento RenderingParametersUpdate**](renderingparametersupdate.md)<br/> | Se produce siempre que se actualiza cualquier conjunto de parámetros de control de representación en el DMR.<br/>                                                          |
| [**Evento TransportParametersUpdate**](transportparametersupdate.md)<br/> | Se produce siempre que se actualiza cualquier conjunto de parámetros de transporte en el DMR.<br/>                                                                  |
