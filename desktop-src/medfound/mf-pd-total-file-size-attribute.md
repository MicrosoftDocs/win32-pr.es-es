# <a name="mf_pd_total_file_size-attribute"></a>\_Atributo de \_ Tamaño total de \_ archivo MF PD \_

Especifica el tamaño total del archivo de código fuente, en bytes. Este atributo se aplica a los descriptores de presentación. Opcionalmente, un origen de medios puede establecer este atributo.

## <a name="data-type"></a>Tipo de datos

**UINT64**

## <a name="remarks"></a>Observaciones

La constante GUID para este atributo se exporta desde mfuuid. lib.
