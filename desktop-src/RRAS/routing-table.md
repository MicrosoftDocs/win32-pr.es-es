# <a name="routing-table"></a>Tabla de enrutamiento

El administrador de tablas de enrutamiento mantiene una tabla de enrutamiento distinta para cada familia de direcciones (por ejemplo, IPX e IPv4). La familia de direcciones IPX es compatible con la API de RTMv1; IPv4 es compatible con la API de RTMv2.

La tabla de enrutamiento consta de destinos, rutas y próximos saltos. Estas entradas definen una ruta a una red de destino.
