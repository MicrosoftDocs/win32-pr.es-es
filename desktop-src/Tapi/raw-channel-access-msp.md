# <a name="raw-channel-access-msp"></a>MSP de acceso al canal sin formato

El MSP de acceso al canal sin formato (Rcsmsp.dll) realiza el streaming a través de ksproxy a RCA, lo que permite el streaming a través de dispositivos de especificación de interfaz de controlador de red (NDIS) orientados a la conexión, como tarjetas ATM e ISDN. El MSP de RCA se crea mediante ndptsp, que es el TSP proporcionado para los adaptadores WAN de NDIS. Con la Windows versión 2000, solo se implementa el streaming de audio.

El MSP de acceso al canal sin formato se instala Windows 2000.
