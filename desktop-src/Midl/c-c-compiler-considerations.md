# <a name="cc-compiler-considerations"></a>Consideraciones del compilador de C/C++

El compilador MIDL debe interoperar con el compilador de C y el preprocesador de C. MIDL no acepta la sintaxis de C++ en la entrada. El archivo. h generado tiene las definiciones de estilo de C y C++ para las interfaces. En los temas siguientes se describen los requisitos del compilador de C.

-   [Problemas de empaquetado del compilador de C](c-compiler-packing-issues.md)
-   [Definiciones de compilador de C para proxy/códigos auxiliares](c-compiler-definitions-for-proxy-stubs.md)
