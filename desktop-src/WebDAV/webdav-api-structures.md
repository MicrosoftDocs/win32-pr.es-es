# <a name="webdav-api-structures"></a>Estructuras de API de WebDAV

Las siguientes estructuras se utilizan en la API de WebDAV.



| Estructura                                                   | Descripción                                                                                                                                                                                         |
|-------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**BLOB de autenticación de devolución de \_ llamada DAV \_ \_**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_blob) | Esta estructura se usa para almacenar un [*BLOB*](/windows/desktop/SecGloss/b-gly) de autenticación recuperado por la función de devolución de llamada [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) . |
| [**Unp de autenticación de devolución de \_ llamada DAV \_ \_**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_auth_unp)   | Esta estructura se utiliza para almacenar la información de nombre de usuario y contraseña que se recuperó mediante la función de devolución de llamada [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) .                                               |
| [**\_devolución de llamada DAV \_ CRED**](/windows/desktop/api/Davclnt/ns-davclnt-dav_callback_cred)            | La función de devolución de llamada [*DavAuthCallback*](/windows/desktop/api/Davclnt/nc-davclnt-pfndavauthcallback) usa esta estructura para almacenar información de credenciales de usuario.                                                                               |
