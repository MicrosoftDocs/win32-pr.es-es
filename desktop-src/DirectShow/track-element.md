# <a name="track-element"></a>elemento track

> [!Note]  
> \[Obsoleto. Esta API puede quitarse de futuras versiones de Windows.\]

 

El `track` elemento define un objeto de seguimiento.

## <a name="attributes"></a>Atributos

[**lock**](lock-attribute.md), [**mute**](mute-attribute.md), [**userdata**](userdata-attribute.md), [**userid**](userid-attribute.md), [**username**](username-attribute.md)

## <a name="parentchild-information"></a>Información de elementos primarios y secundarios



| Etiqueta | Value |
|----------|----------------------------------------------------------------------------------------------------------|
| Parent   | [**compuesto**](composite-element.md), [ **grupo**](group-element.md)                                   |
| Children | [**clip**](clip-element.md), [**effect**](effect-element.md), [**transition**](transition-element.md) |
