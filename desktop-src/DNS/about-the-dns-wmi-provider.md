# <a name="about-the-dns-wmi-provider"></a>Acerca del proveedor WMI de DNS

Los servidores DNS asignan direcciones IP a nombres de equipo y nombres de equipo a direcciones IP. Al hacerlo, proporcionan el mecanismo para buscar recursos de red. El proveedor WMI de DNS permite a las aplicaciones interactuar con servidores DNS a través del marco de administración unificada de Instrumental de administración de Windows (WMI).
