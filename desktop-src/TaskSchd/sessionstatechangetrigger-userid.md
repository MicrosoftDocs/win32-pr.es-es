# <a name="sessionstatechangetriggeruserid-property"></a>Propiedad SessionStateChangeTrigger.UserId

Para el scripting, obtiene o establece el usuario para la sesión de Terminal Server. Cuando se detecta un cambio de estado de sesión para este usuario, se inicia una tarea.

## <a name="syntax"></a>Syntax


```VB
SessionStateChangeTrigger.UserId As String
```



## <a name="property-value"></a>Valor de propiedad

Usuario de la sesión de Terminal Server.
