# <a name="defaultfx-attribute"></a>Atributo defaultfx

> [!Note]  
> \[En desuso. Esta API se puede quitar de las versiones futuras de Windows.\]

 

El `defaultfx` atributo especifica el identificador de clase (CLSID) de un efecto predeterminado para la escala de tiempo.

## <a name="possible-values"></a>Valores posibles

Debe ser una cadena con el formato {00000000-0000-0000-0000-000000000000} .
