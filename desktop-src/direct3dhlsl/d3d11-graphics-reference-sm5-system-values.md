-   [ID. de \_ la SV](sv-groupid.md)
-   [VP \_ GroupIndex](sv-groupindex.md)
-   [VP \_ GroupThreadID](sv-groupthreadid.md)
-   [VP \_ GSInstanceID](sv-gsinstanceid.md)
-   [VP \_ InsideTessFactor](sv-insidetessfactor.md)
