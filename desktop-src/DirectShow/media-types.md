# <a name="media-types-directshow"></a>Tipos de medios (DirectShow)

Esta estructura incluye **miembros GUID** para el tipo principal, el subtipo y el tipo de formato. En esta sección se resumen los GUID de tipo principal y subtipo que se definen para DirectShow. Los filtros de terceros pueden definir sus propios GUID de tipo multimedia.

-   [**Tipos de medios de línea 21**](line-21-media-types.md)
