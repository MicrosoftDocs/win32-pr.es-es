# <a name="ms-tapi-conference-blob-attribute"></a>atributo MS-TAPI-Conference-BLOB

Un BLOB binario de datos que describe diversas propiedades de una conferencia de multidifusión de TAPI. El formato y el contenido vienen determinados por el valor del atributo Protocol-Id en el mismo objeto. Normalmente, los datos de este BLOB se ajustan a RFC2327.



| Entrada | Value |
|-------------------|--------------------------------------------------------------------------------------------------------------|
| CN                | MS-TAPI-Conference-BLOB                                                                                      |
| Nombre para mostrar de LDAP | msTAPI-ConferenceBlob                                                                                        |
| Tamaño              | Puede ser de longitud arbitraria.                                                                                  |
| Actualizar privilegio  | No se requieren privilegios especiales.                                                                              |
| Frecuencia de actualización  | Puede cambiar a discreción de un propietario de la Conferencia TAPI cuando algunos datos sobre la Conferencia deben cambiar. |
| Attribute-Id      | 1.2.840.113556.1.4.1700                                                                                      |
| System-ID-GUID    | 4cc4601e-7201-4141-abc8-3e529ae88863                                                                         |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md)                                                        |
