# <a name="remote-desktop-classes"></a>Escritorio remoto clases

Las Escritorio remoto wmi de virtualización se usan para administrar Escritorio remoto y RemoteFX recursos y máquinas virtuales.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                                            | Descripción                                                                                                                                                  |
|------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Msvm \_ InteractiveSessionACE**](msvm-interactivesessionace.md)<br/>                                     | Representa una *entrada de control de* acceso (ACE) que determina el acceso a la sesión interactiva de una máquina virtual.<br/>                                |
| [**Msvm \_ Physical3dGraphicsProcessor**](msvm-physical3dgraphicsprocessor.md)<br/>                         | Describe la unidad física de procesamiento de gráficos 3D (GPU).<br/>                                                                                        |
| [**Msvm \_ PhysicalGPUInfo**](msvm-physicalgpuinfo.md)<br/>                                                 | Contiene información sobre una RemoteFX de procesamiento de gráficos físicos (GPU).<br/>                                                                    |
| [**Msvm \_ RdvComponent**](msvm-rdvcomponent.md)<br/>                                                       | Representa el estado del componente RDV, que es responsable de proporcionar un transporte para el elemento primario al invitado con fines de configuración.<br/> |
| [**Msvm \_ RdvComponentSettingData**](msvm-rdvcomponentsettingdata.md)<br/>                                 | Representa el estado configurado del componente Escritorio remoto Virtualization (RDV). El estado predeterminado es Habilitado.<br/>                               |
| [**Msvm \_ Synth3dVideoPool**](msvm-synth3dvideopool.md)<br/>                                               | Contiene información sobre las unidades de procesamiento de gráficos de vídeo (GPU) sintéticas 3D disponibles en el sistema host.<br/>                                 |
| [**Msvm \_ Synthetic3DDisplayController**](msvm-synthetic3ddisplaycontroller.md)<br/>                       | Representa el controlador de pantalla 3D sintético que se asigna a una máquina virtual.<br/>                                                            |
| [**Msvm \_ Synthetic3DDisplayControllerSettingData**](msvm-synthetic3ddisplaycontrollersettingdata.md)<br/> | Representa la configuración de un controlador de pantalla 3D sintético para una máquina virtual.<br/>                                                                 |
| [**Msvm \_ Synthetic3DService**](msvm-synthetic3dservice.md)<br/>                                           | Describe el servicio de GPU 3D sintético.<br/>                                                                                                          |
| [**Msvm \_ Synthetic3DServiceSettingData**](msvm-synthetic3dservicesettingdata.md)<br/>                     | Representa la configuración del servicio 3D sintético presente en un único sistema host.<br/>                                                            |
| [**TerminalService de Msvm \_**](msvm-terminalservice.md)<br/>                                                 | Administra todas las conexiones de terminal remoto a un host determinado.<br/>                                                                                     |
| [**TerminalServiceSettingData de Msvm \_**](msvm-terminalservicesettingdata.md)<br/>                           | Representa la configuración de los servicios de terminal del equipo virtual en un host.<br/>                                                                     |
