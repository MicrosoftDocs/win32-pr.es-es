# <a name="released-versions-tools-and-redistributables"></a>Versiones publicadas, herramientas y redistribuibles

En los temas siguientes se proporciona más información acerca de las versiones publicadas, las herramientas y los redistribuibles de Windows Installer.

-   [Windows SDK componentes para desarrolladores Windows Installer](platform-sdk-components-for-windows-installer-developers.md) explica dónde obtener los componentes redistribuibles del instalador, la documentación, la herramienta de validación de base de datos, el editor de tablas de base de datos, el esquema de base de datos, las herramientas de desarrollo, las herramientas de VBScript, el producto de ejemplo y los ejemplos
-   [Windows Installer redistribuibles](windows-installer-redistributables.md) proporciona información sobre dónde obtener los componentes redistribuibles del instalador.
-   [Versiones publicadas de Windows Installer](released-versions-of-windows-installer.md) muestra las versiones publicadas del instalador que se han incluido con distintos sistemas operativos.
-   [Windows Installer herramientas de desarrollo](windows-installer-development-tools.md) contienen páginas de referencia para utilizar las herramientas de desarrollo del instalador.
-   [Instmsi.exe](instmsi-exe.md) describe qué Instmsi es compatible con qué sistema operativo. InstMsi es el paquete redistribuible para instalar el instalador.
