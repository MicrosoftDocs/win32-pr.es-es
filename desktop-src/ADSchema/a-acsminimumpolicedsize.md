# <a name="acs-minimum-policed-size-attribute"></a>Atributo ACS-Minimum-Policed-Size

El **atributo ACS-Minimum-Policed-Size** es solo para uso interno. Basado en RFC2210.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | Acs-Minimum-Policed-Size             |
| Ldap-Display-Name | aCSMinimumPolicedSize                |
| Size              | 8 bytes                              |
| Privilegio actualizar  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1315              |
| System-Id-Guid    | 8d0e7195-3b90-11d2-90cc-00c04fd91ab1 |
| Syntax            | [**Intervalo**](s-interval.md)       |
