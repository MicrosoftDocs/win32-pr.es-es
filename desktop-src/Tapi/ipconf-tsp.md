# <a name="ipconf-tsp"></a>IPConf TSP

\[IPConf TSP no está disponible para su uso en Windows Vista, Windows Server 2008 y las versiones posteriores del sistema operativo. La API de cliente de RTC proporciona una funcionalidad similar.\]

El TSP IPConf (ipconf. TSP), cuando se empareja con [IPCONF MSP](./ipconf-msp.md) (Confmsp.dll), proporciona acceso de conferencias a redes de protocolo de Internet, como Internet y muchas intranets corporativas.

Este TSP admite un valor de [**tipo de dirección**](./lineaddresstype--constants.md) de LINEADDRESSTYPE \_ SDP.

IPConf MSP se ha mejorado en Windows 2000 SP1 para habilitar la comunicación entre los clientes H323 y SDP.

El TSP de IPConf se instala con Windows 2000 y versiones posteriores.
