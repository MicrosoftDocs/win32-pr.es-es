# <a name="about-the-device-host-api"></a>Acerca de la API del host del dispositivo

La API de host de dispositivo con tecnología UPnP es un marco para implementar la funcionalidad de dispositivos basados en UPnP en la plataforma Windows. Los desarrolladores que crean dispositivos mediante la API de host de dispositivo con tecnología UPnP (a la que se hace referencia como [*dispositivos hospedados*](h-gly.md)) solo necesitan implementar la funcionalidad básica del dispositivo. Los desarrolladores pueden basarse en el host del dispositivo para controlar los detalles específicos de UPnP de detección, descripción, control, eventos y presentación. El host del dispositivo valida los datos entrantes de los clientes basados en UPnP y da formato a todos los datos salientes de los dispositivos hospedados según la arquitectura del dispositivo UPnP.

-   [Proveedores de dispositivos](device-providers.md)
-   [Eventos](eventing.md)
