# <a name="extending-language-resources"></a>Extensión de recursos de lenguaje

Windows La búsqueda usa recursos de lenguaje como separadores de palabras y lematizadores para interrumpir el texto en su configuración regional nativa durante la creación de índices y el procesamiento de consultas. Microsoft proporciona separadores de palabras y lematizadores para varios idiomas. En esta sección se describe cómo implementar y usar separadores de palabras y lematizadores personalizados para idiomas y configuraciones regionales más allá de los proporcionados por Microsoft.

-   [Descripción de los componentes de recursos de lenguaje](understanding-language-resource-components.md)
-   [Implementar un separador de palabras y un lematizador](implementing-a-word-breaker-and-stemmer.md)
-   [Consideraciones lingüísticas y Unicode](linguistic-and-unicode-considerations.md)
-   [Solución de problemas de recursos de lenguaje y procedimientos recomendados](troubleshooting-language-resources.md)

## <a name="additional-resources"></a>Recursos adicionales

-   Para obtener una lista de los idiomas admitidos por los separadores de palabras, vea [Idiomas admitidos por Windows Search](-search-3x-wds-language-support.md).
-   Si necesita identificar el idioma de un fragmento de texto, puede usar la detección automática de idioma (LAD), que está disponible en Windows 7 y versiones posteriores. Para obtener más información, vea [Extended Linguistic Services](../intl/extended-linguistic-services.md) (ELS).
-   Para obtener documentación de referencia aplicable, [vea Interfaces de complemento de datos](-search-data-addins-interfaces-entry-page.md).
