# <a name="windows-media-drm-implemented-interfaces"></a>Windows Interfaces DRM-Implemented multimedia

Las interfaces siguientes no están diseñadas para ser implementadas por proveedores de servicios de terceros, sino que se implementan y llaman internamente mediante drm de medios de Windows y Windows Media Administrador de dispositivos. La implementación de estas interfaces impedirá que se procese el contenido protegido por DRM. Se documentan simplemente por integridad.

-   [**Interfaz IWMDRMDevice**](iwmdrmdevice.md)
-   [**Interfaz IWMDRMDevice2**](iwmdrmdevice2.md)
