# <a name="acs-minimum-policed-size-attribute"></a>ACS-atributo de tamaño mínimo de directivas

El atributo **ACS-Minimum-policialed-size** solo es para uso interno. Basado en RFC2210.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | ACS: tamaño mínimo de directivas             |
| Nombre para mostrar de LDAP | aCSMinimumPolicedSize                |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1315              |
| System-ID-GUID    | 8d0e7195-3b90-11d2-90cc-00c04fd91ab1 |
| Sintaxis            | [**Interval**](s-interval.md)       |
