# <a name="mci"></a>MCI

La interfaz de control de medios (MCI) proporciona comandos estándar para reproducir dispositivos multimedia y grabar archivos de recursos multimedia. Estos comandos son una interfaz genérica para casi todos los tipos de dispositivos multimedia.
