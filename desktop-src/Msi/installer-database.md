# <a name="installer-database"></a>Base de datos del instalador

Este material es principalmente para los desarrolladores que quieren escribir herramientas que crean nuevos paquetes de instalador y que necesitan información detallada sobre la base de datos relacional del instalador. Los desarrolladores de aplicaciones adoptan los paquetes de instalador completos como un medio eficaz para administrar la instalación de sus aplicaciones. El conocimiento de la base de datos de instalación también es útil para los desarrolladores y administradores de aplicaciones que deseen modificar los paquetes de instalador existentes y los procesos de instalación mediante acciones personalizadas. Para obtener información general sobre Windows Installer y la tecnología de administración de componentes, consulte [acerca de Windows Installer](about-windows-installer.md).

La base de datos del instalador contiene toda la información necesaria para la instalación de una aplicación. Por lo tanto, los desarrolladores de herramientas para la creación de paquetes de instalación y desarrolladores que creen paquetes de instalación sin usar dichas herramientas deben comprender la base de datos del instalador. En los temas siguientes se presentan las áreas principales de funcionalidad de la base de datos del instalador:

[Usar la base de datos del instalador](using-the-installer-database.md)
