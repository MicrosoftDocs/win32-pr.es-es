# <a name="relation-to-other-services"></a>Relación con otros servicios

Otras partes de Microsoft Internet Security Framework usan el subsistema [*de tarjetas inteligentes*](../secgloss/s-gly.md), como se muestra en la ilustración siguiente. (Debido [*a*](../secgloss/c-gly.md) las restricciones de exportación de EE. UU., un proveedor de servicios criptográficos (CSP) que usa un SCSP para comunicar solicitudes relacionadas con la criptográfico debe firmar y comprobar el SCSP).

![relación del marco de seguridad de Internet de Microsoft con el subsistema de tarjetas inteligentes](images/smartovr3.png)

Para obtener más información sobre el rol del subsistema de tarjetas inteligentes en Microsoft Internet Security Framework, vea [https://www.microsoft.com/whdc/device/input/smartcard/default.mspx](https://www.microsoft.com/whdc/device/input/smartcard/default.mspx) .
