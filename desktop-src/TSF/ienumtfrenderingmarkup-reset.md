# <a name="ienumtfrenderingmarkupreset-method"></a>IEnumTfRenderingMarkup:: RESET (método)

El método **IEnumTfRenderingMarkup:: RESET** restablece el objeto de enumerador moviendo la posición actual al principio de la secuencia de enumeración.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT Reset();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método puede devolver uno de estos valores.



| Value                                                                                | Descripción                           |
|--------------------------------------------------------------------------------------|---------------------------------------|
| <dl> <dt>**S \_ correcto**</dt> </dl> | Método realizado correctamente.<br/> |
