# <a name="mf_capture_metadata_digitalwindow-attribute"></a>Atributo MF \_ CAPTURE \_ METADATA \_ DIGITALWINDOW

Valor que representa la configuración de límites de la ventana digital para la captura de vídeo.

## <a name="data-type"></a>Tipo de datos

**Blob**

## <a name="remarks"></a>Comentarios

El **valor BLOB** de esta propiedad es una estructura [DigitalWindowSetting.](/windows/win32/api/mfapi/ns-mfapi-digitalwindowsetting)
