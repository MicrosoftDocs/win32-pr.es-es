# <a name="idcompositionrotatetransform3dsetcenterx-methods"></a>Métodos IDCompositionRotateTransform3D::SetCenterX

Cambia o anima el valor de la propiedad CenterX de una transformación de rotación 3D. La propiedad CenterX especifica la coordenada x del punto sobre el que se realiza la rotación. El valor predeterminado es cero.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                                           | Descripción                                            |
|:-----------------------------------------------------------------------------------------------------------------|:-------------------------------------------------------|
| [**SetCenterX(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterx(float))                                     | Cambia el valor de la propiedad CenterX.<br/>  |
| [**SetCenterX(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setcenterx(idcompositionanimation)) | Anima el valor de la propiedad CenterX.<br/> |
