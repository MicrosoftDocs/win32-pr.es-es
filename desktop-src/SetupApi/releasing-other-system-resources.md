# <a name="releasing-other-system-resources"></a>Liberar otros recursos del sistema

Si se usó una lista de origen o un archivo de registro, los recursos deben liberarse llamando a [**SetupFreeSourceList**](/windows/desktop/api/Setupapi/nf-setupapi-setupfreesourcelista) o [**SetupTerminateFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog).
