# <a name="remoteapp-and-desktop-connection-management-service-api-reference"></a>Referencia de la API de servicio Administración de conexiones de RemoteApp y Escritorio

La API del servicio Administración de conexiones de RemoteApp y Escritorio admite una interfaz y una estructura que proporcionan información sobre los recursos disponibles para los usuarios de conexiones de RemoteApp y escritorio. La implementación de esta interfaz puede realizar un filtrado personalizado de los recursos y proporcionar compatibilidad con tipos de archivo que no se admiten actualmente. Para obtener más información, vea los siguientes temas.
