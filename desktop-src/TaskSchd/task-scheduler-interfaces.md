# <a name="task-scheduler-interfaces"></a>Interfaces de Programador de tareas

Las interfaces que se describen en los siguientes temas proporcionan acceso mediante programación a la funcionalidad que está disponible en el Programador de tareas.

Estos temas contienen una descripción de la interfaz, una lista de las propiedades y los métodos definidos por la interfaz, así como comentarios sobre las circunstancias especiales que se deben anotar al usar la interfaz.


Las siguientes interfaces se introdujeron en Programador de tareas 2,0, que se utiliza en el sistema operativo Windows Vista.



| Interfaz                                                        | Descripción                                                                                                                                                                                                                                                                                       |
|------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IAction**](/windows/desktop/api/taskschd/nn-taskschd-iaction)                                       | Proporciona las propiedades comunes que heredan todos los objetos de acción.                                                                                                                                                                                                                          |
| [**IActionCollection**](/windows/desktop/api/taskschd/nn-taskschd-iactioncollection)                   | Contiene las acciones que realiza la tarea. Sus métodos se pueden usar para agregar, quitar y recuperar las acciones de una tarea.                                                                                                                                                          |
| [**IBootTrigger**](/windows/desktop/api/taskschd/nn-taskschd-iboottrigger)                             | Representa un desencadenador que inicia una tarea cuando se inicia el sistema.                                                                                                                                                                                                                               |
| [**IComHandlerAction**](/windows/desktop/api/taskschd/nn-taskschd-icomhandleraction)                   | Representa una acción que activa un controlador.                                                                                                                                                                                                                                                        |
| [**IDailyTrigger**](/windows/desktop/api/taskschd/nn-taskschd-idailytrigger)                           | Representa un desencadenador que inicia una tarea en función de una programación diaria.                                                                                                                                                                                                                                |
| [**IEmailAction**](/windows/desktop/api/taskschd/nn-taskschd-iemailaction)                             | Representa una acción que envía un mensaje de correo electrónico.                                                                                                                                                                                                                                                 |
| [**IEventTrigger**](/windows/desktop/api/taskschd/nn-taskschd-ieventtrigger)                           | Representa un desencadenador que inicia una tarea cuando se produce un evento del sistema.                                                                                                                                                                                                                               |
| [**IExecAction**](/windows/desktop/api/taskschd/nn-taskschd-iexecaction)                               | Representa una acción que ejecuta una operación de la línea de comandos.                                                                                                                                                                                                                                      |
| [**IIdleSettings**](/windows/desktop/api/taskschd/nn-taskschd-iidlesettings)                           | Especifica cómo realiza el Programador de tareas las tareas cuando el equipo está en una condición de inactividad.                                                                                                                                                                                                        |
| [**IIdleTrigger**](/windows/win32/api/taskschd/nn-taskschd-iidletrigger)                             | Representa un desencadenador que inicia una tarea cuando el equipo entra en un estado de inactividad.                                                                                                                                                                                                                |
| [**ILogonTrigger**](/windows/desktop/api/taskschd/nn-taskschd-ilogontrigger)                           | Representa un desencadenador que inicia una tarea cuando un usuario inicia sesión.                                                                                                                                                                                                                                      |
| [**IMaintenanceSettings**](/windows/desktop/api/taskschd/nn-taskschd-imaintenancesettings)             | Proporciona la configuración que el Programador de tareas utiliza para realizar la tarea durante el mantenimiento automático.                                                                                                                                                                                                  |
| [**IMonthlyDOWTrigger**](/windows/desktop/api/taskschd/nn-taskschd-imonthlydowtrigger)                 | Representa un desencadenador que inicia una tarea en una programación mensual de día de la semana.                                                                                                                                                                                                                        |
| [**IMonthlyTrigger**](/windows/desktop/api/taskschd/nn-taskschd-imonthlytrigger)                       | Representa un desencadenador que inicia una tarea basada en una programación mensual.                                                                                                                                                                                                                              |
| [**INetworkSettings**](/windows/desktop/api/taskschd/nn-taskschd-inetworksettings)                     | Proporciona la configuración que usa el servicio de Programador de tareas para obtener un perfil de red.                                                                                                                                                                                                           |
| [**IPrincipal**](/windows/desktop/api/taskschd/nn-taskschd-iprincipal)                                 | Proporciona las credenciales de seguridad para una entidad de seguridad.                                                                                                                                                                                                                                                |
| [**IPrincipal2**](/windows/desktop/api/taskschd/nn-taskschd-iprincipal2)                               | Proporciona credenciales de seguridad adicionales para una entidad de seguridad.                                                                                                                                                                                                                                         |
| [**IRegisteredTask**](/windows/desktop/api/taskschd/nn-taskschd-iregisteredtask)                       | Proporciona los métodos que se usan para ejecutar la tarea inmediatamente, obtener las instancias en ejecución de la tarea, obtener o establecer las credenciales que se usan para registrar la tarea y las propiedades que describen la tarea.                                                                                  |
| [**IRegisteredTaskCollection**](/windows/desktop/api/taskschd/nn-taskschd-iregisteredtaskcollection)   | Contiene todas las tareas que están registradas.                                                                                                                                                                                                                                                       |
| [**IRegistrationInfo**](/windows/desktop/api/taskschd/nn-taskschd-iregistrationinfo)                   | Proporciona la información administrativa que se puede utilizar para describir la tarea. Esta información incluye detalles como, por ejemplo, una descripción de la tarea, el autor de la tarea, la fecha de registro de la tarea y el descriptor de seguridad de la tarea.                                             |
| [**IRegistrationTrigger**](/windows/desktop/api/taskschd/nn-taskschd-iregistrationtrigger)             | Representa un desencadenador que inicia una tarea cuando se registra la tarea.                                                                                                                                                                                                                              |
| [**IRepetitionPattern**](/windows/desktop/api/taskschd/nn-taskschd-irepetitionpattern)                 | Define la frecuencia con que se ejecuta la tarea y cuánto tiempo se repite el patrón de repetición una vez iniciada la tarea.                                                                                                                                                                                      |
| [**IRunningTask**](/windows/desktop/api/taskschd/nn-taskschd-irunningtask)                             | Proporciona los métodos para obtener información y controlar una tarea en ejecución.                                                                                                                                                                                                                          |
| [**IRunningTaskCollection**](/windows/desktop/api/taskschd/nn-taskschd-irunningtaskcollection)         | Proporciona una colección que se utiliza para controlar las tareas en ejecución.                                                                                                                                                                                                                                      |
| [**ISessionStateChangeTrigger**](/windows/desktop/api/taskschd/nn-taskschd-isessionstatechangetrigger) | Desencadena tareas de conexión o desconexión de la consola, conexión remota o desconexión, o notificaciones de bloqueo o desbloqueo de la estación de trabajo.                                                                                                                                                                      |
| [**IShowMessageAction**](/windows/desktop/api/taskschd/nn-taskschd-ishowmessageaction)                 | Representa una acción que muestra un cuadro de mensaje cuando se activa una tarea.                                                                                                                                                                                                                           |
| [**ITaskDefinition**](/windows/desktop/api/taskschd/nn-taskschd-itaskdefinition)                       | Define todos los componentes de una tarea, como la configuración de la tarea, los desencadenadores, las acciones y la información de registro.                                                                                                                                                                                 |
| [**ITaskFolder**](/windows/desktop/api/taskschd/nn-taskschd-itaskfolder)                               | Proporciona los métodos que se usan para registrar (crear) tareas en la carpeta, quitar tareas de la carpeta y crear o quitar subcarpetas de la carpeta.                                                                                                                                       |
| [**ITaskFolderCollection**](/windows/desktop/api/taskschd/nn-taskschd-itaskfoldercollection)           | Proporciona información y control para una colección de carpetas que contienen tareas.                                                                                                                                                                                                                  |
| [**ITaskHandler**](/windows/desktop/api/taskschd/nn-taskschd-itaskhandler)                             | Define los métodos a los que llama el servicio Programador de tareas para administrar un controlador COM.                                                                                                                                                                                                        |
| [**ITaskHandlerStatus**](/windows/desktop/api/taskschd/nn-taskschd-itaskhandlerstatus)                 | Proporciona los métodos que utilizan los controladores COM para notificar al Programador de tareas sobre el estado del controlador.                                                                                                                                                                                  |
| [**ITaskNamedValuePair**](/windows/desktop/api/taskschd/nn-taskschd-itasknamedvaluepair)               | Crea un par nombre-valor en el que el nombre está asociado con el valor.                                                                                                                                                                                                                         |
| [**ITaskNamedValueCollection**](/windows/desktop/api/taskschd/nn-taskschd-itasknamedvaluecollection)   | Contiene una colección de pares de nombre y valor de la interfaz [**ITaskNamedValuePair**](/windows/desktop/api/taskschd/nn-taskschd-itasknamedvaluepair) .                                                                                                                                                                                           |
| [**ITaskService**](/windows/desktop/api/taskschd/nn-taskschd-itaskservice)                             | Proporciona acceso al servicio de Programador de tareas para administrar tareas registradas.                                                                                                                                                                                                                      |
| [**ITaskSettings**](/windows/desktop/api/taskschd/nn-taskschd-itasksettings)                           | Proporciona la configuración que usa el servicio de Programador de tareas para realizar la tarea.                                                                                                                                                                                                                   |
| [**ITaskSettings2**](/windows/desktop/api/taskschd/nn-taskschd-itasksettings2)                         | Proporciona valores de configuración adicionales que el Programador de tareas utiliza para realizar la tarea.                                                                                                                                                                                                                    |
| [**ITaskVariables**](/windows/desktop/api/taskschd/nn-taskschd-itaskvariables)                         | Define las variables de tarea que se pueden pasar como parámetros a los controladores de tareas y los archivos ejecutables externos iniciados por las tareas. Los controladores de tareas que necesitan escribir y enviar datos a variables de trabajo deben realizar una interfaz de consulta en el puntero de servicios para [**ITaskVariables**](/windows/desktop/api/taskschd/nn-taskschd-itaskvariables). |
| [**ITimeTrigger**](/windows/desktop/api/taskschd/nn-taskschd-itimetrigger)                             | Representa un desencadenador que inicia una tarea cuando se activa el desencadenador.                                                                                                                                                                                                                            |
| [**ITrigger**](/windows/desktop/api/taskschd/nn-taskschd-itrigger)                                     | Proporciona las propiedades comunes que heredan todas las interfaces de desencadenador.                                                                                                                                                                                                                      |
| [**ITriggerCollection**](/windows/desktop/api/taskschd/nn-taskschd-itriggercollection)                 | Proporciona los métodos que se usan para agregar, quitar y obtener los desencadenadores de una tarea.                                                                                                                                                                                                        |
| [**IWeeklyTrigger**](/windows/desktop/api/taskschd/nn-taskschd-iweeklytrigger)                         | Representa un desencadenador que inicia una tarea basada en una programación semanal.                                                                                                                                                                                                                               |



 


> [!WARNING]
> Las interfaces Programador de tareas 1,0 solo están disponibles en los sistemas operativos Windows 2000, Windows XP y Windows Server 2003. Están en desuso en Windows Vista y pueden quitarse completamente en el futuro. En su lugar, use las interfaces Programador de tareas 2,0 indicadas anteriormente.
