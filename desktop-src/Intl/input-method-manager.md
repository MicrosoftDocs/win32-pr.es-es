# <a name="input-method-manager"></a>Administrador de métodos de entrada

El administrador de métodos de entrada (IMM) es una tecnología que una aplicación usa para comunicarse con un editor de métodos de entrada (IME), que se ejecuta como un servicio. El IME permite a los usuarios del equipo escribir caracteres y símbolos complejos, como caracteres Kanji japoneses, mediante un teclado estándar.

En esta sección se describe la API de IMM y se explica cómo utilizar la funcionalidad para crear y administrar las ventanas de IME. Se incluyen las secciones siguientes:

-   [Acerca del administrador de métodos de entrada](about-input-method-manager.md)
-   [Usar el administrador de métodos de entrada](using-input-method-manager.md)
-   [Referencia del administrador de métodos de entrada](input-method-manager-reference.md)
