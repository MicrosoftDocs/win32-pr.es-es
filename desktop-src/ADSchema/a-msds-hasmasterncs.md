# <a name="ms-ds-has-master-ncs-attribute"></a>atributo MS-DS-tiene-Master-NCS

Contiene los contextos de nomenclatura mantenidos por un controlador de dominio. Este atributo deja de ser-Master-NC.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | MS-DS-tiene-Master-NC                    |
| Nombre para mostrar de LDAP | msDS-hasMasterNCs                       |
| Tamaño              | \-                                      |
| Actualizar privilegio  | El sistema establece este valor.         |
| Frecuencia de actualización  | Cada vez que se crea un nuevo NC.          |
| Attribute-Id      | 1.2.840.113556.1.4.1836                 |
| System-ID-GUID    | ae2de0e2-59d7-4d47-8d47-ed4dfe4357ad    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
