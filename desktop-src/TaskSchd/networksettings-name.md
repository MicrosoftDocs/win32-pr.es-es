# <a name="networksettingsname-property"></a>Propiedad NetworkSettings.Name

En el caso de scripting, obtiene o establece el nombre de un perfil de red. El nombre se usa para fines de presentación.

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Sintaxis


```VB
NetworkSettings.Name As String
```



## <a name="property-value"></a>Valor de propiedad

El nombre de un perfil de red.
