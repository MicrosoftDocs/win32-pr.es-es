# <a name="about-the-wireless-ad-hoc-api"></a>Acerca de la API ad hoc inalámbrica

La API ad hoc inalámbrica expone interfaces para enumerar y conectarse a redes ad hoc 802.11. Estas interfaces proporcionan un enfoque simplificado orientado a objetos para la administración de redes ad hoc. Se proporcionan enumeradores estándar para redes e interfaces. Estos enumeradores se pueden filtrar para enumerar solo las redes ad hoc creadas por la aplicación.

> [!Note]  
> Es posible que el modo ad hoc no esté disponible en versiones futuras de Windows. A partir de Windows 8.1 y Windows Server 2012 R2, use Wi-Fi Direct en [su lugar.](about-the-wi-fi-direct-api.md)

 

Los desarrolladores que usan la API ad hoc deben tener en cuenta que las redes y los perfiles creados mediante la API ad hoc se pueden manipular mediante la API de Wi-Fi nativa sin notificar a los agentes de escucha de notificación de API ad hoc pertinentes.

La API ad hoc inalámbrica se describe en detalle en la [referencia ad hoc inalámbrica](wireless-ad-hoc-reference.md).

> [!Note]  
> La API ad hoc inalámbrica solo se admite en Windows Vista y versiones posteriores del sistema operativo.
