# <a name="getclipvideorect-method"></a>Método GetClipVideoRect

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

El `GetClipVideoRect` método recupera el rectángulo de recorte definido actualmente para la pantalla de vídeo.

``` syntax
[ oClipRect = ] MSWebDVD.GetClipVideoRect()
```

## <a name="return-value"></a>Valor devuelto

Devuelve un objeto [DVDRect](dvdrect-object.md) que define el rectángulo.
