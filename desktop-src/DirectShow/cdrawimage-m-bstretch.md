# <a name="cdrawimagem_bstretch-member"></a>Miembro bStretch CDrawImage:: m \_

La `m_bStretch` variable miembro indica si la imagen de vídeo debe ajustarse para ajustarse a la ventana de destino. El valor es **false** si el rectángulo de origen tiene el mismo tamaño que el rectángulo de destino, o bien **true** en caso contrario.

## <a name="syntax"></a>Sintaxis


```C++
BOOL m_bStretch;
```
