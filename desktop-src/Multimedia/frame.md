# <a name="frame-windows-multimedia"></a>Marco (Windows Multimedia)

Una ventana de captura usa mensajes de notificación de devolución de llamada de fotogramas para notificar a la aplicación cuando hay disponible un nuevo fotograma de vídeo. La ventana de captura habilita estas notificaciones de devolución de llamada solo si la velocidad de vista previa es distinta de cero y la captura de streaming no está en curso.
