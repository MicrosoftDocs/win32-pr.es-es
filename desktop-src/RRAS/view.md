# <a name="view"></a>Ver

Una vista es un subconjunto de la tabla de enrutamiento y contiene un grupo de rutas relacionadas (por ejemplo, rutas de multidifusión). Actualmente, solo se admiten las vistas unidifusión y multidifusión. Las vistas a veces se denominan bases de información de enrutamiento (costillas).
