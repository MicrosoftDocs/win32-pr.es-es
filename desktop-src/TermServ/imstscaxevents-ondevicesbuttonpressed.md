# <a name="imstscaxeventsondevicesbuttonpressed-method"></a>Método IMsTscAxEvents::OnDevicesButtonPressed

Se llama cuando se presiona el botón dispositivos de la barra de conexión.

## <a name="syntax"></a>Sintaxis


```C++
void OnDevicesButtonPressed();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.

## <a name="remarks"></a>Comentarios

El botón dispositivos se muestra en la barra de conexión cuando el redireccionamiento USB está habilitado tanto en el cliente como en el servidor.
