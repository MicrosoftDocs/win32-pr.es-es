# <a name="imapiv1-support"></a>Compatibilidad con IMAPIv1

La versión 1 de la API de masterización de imágenes (IMAPIv1) permite a una aplicación y grabar una imagen de audio o de datos simple en dispositivos CD-R y CD-RW. La API admite los formatos Joliet e ISO 9660 para los discos de datos y audio de Redbook.

> [!Note]  
> IMAPi se ha rediseñado completamente para la versión 2,0. La mayor parte de la funcionalidad de la versión 1,0 está disponible en la versión 2,0. Se recomienda que las aplicaciones de creación de patrones de imagen de escritura o el desarrollo de nuevos dispositivos y formatos usen la [versión 2,0](using-imapi.md) en lugar de la versión 1,0. Si ejecuta una aplicación de la versión 1,0 en Windows Vista, las interfaces de la versión 1,0 convertirán esas llamadas en las llamadas de la versión 2,0 adecuadas para lograr una funcionalidad idéntica.

 

Para obtener más información sobre el uso de la versión 1,0 en Windows XP, vea los temas siguientes:

-   [Acerca de la API de procesamiento de imágenes](about-the-image-mastering-api.md)
-   [Uso de la API de masterización de imágenes](using-the-image-mastering-api.md)
-   [Referencia de IMAPIv1](imapiv1-reference.md)
