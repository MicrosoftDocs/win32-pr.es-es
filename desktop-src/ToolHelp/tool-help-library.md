# <a name="tool-help-library"></a>Biblioteca de ayuda de herramientas

Las funciones proporcionadas por la biblioteca de ayuda de la herramienta facilitan la obtención de información acerca de las aplicaciones que se están ejecutando actualmente. Estas funciones están diseñadas para simplificar la creación de herramientas, en concreto depuradores.
