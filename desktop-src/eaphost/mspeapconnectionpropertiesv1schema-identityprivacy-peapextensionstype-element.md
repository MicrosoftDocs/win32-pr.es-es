# <a name="identityprivacy-peapextensionstype-element"></a>Elemento IdentityPrivacy (PeapExtensionsType)

El **elemento IdentityPrivacy (PeapExtensionsType)** indica si se envía la identidad verdadera de un usuario o una identidad anónima.

``` syntax
<xs:element
    minOccurs="0"
    ref="extendedPeap:IdentityPrivacy"
 />
```

El elemento [**PeapExtensionsType**](mspeapconnectionpropertiesv1schema-peapextensionstype-complextype.md) define el elemento .

## <a name="remarks"></a>Comentarios

El **elemento IdentityPrivacy** es opcional.
