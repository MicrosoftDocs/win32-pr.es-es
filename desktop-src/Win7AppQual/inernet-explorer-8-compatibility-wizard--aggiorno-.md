# <a name="internet-explorer-8-compatibility-wizard-aggiorno"></a>Asistente para la compatibilidad de Internet Explorer 8 (Aggiorno)

El Asistente para compatibilidad de Windows Internet Explorer 8 actualiza automáticamente los sitios web para que se representen correctamente en Internet Explorer 8 sin interrumpir su compatibilidad en exploradores más antiguos. El asistente automatiza la tarea de destino de la versión del explorador, manteniendo la apariencia y la funcionalidad de las páginas. Para obtener más información acerca de este asistente, consulte [Asistente para la compatibilidad de Internet Explorer 8 inicio rápido](https://www.mobilize.net/).
