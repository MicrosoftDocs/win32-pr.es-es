# <a name="windowsnumericsh-apis"></a>API de windowsnumerics. h

El archivo de encabezado windowsnumerics. h define los tipos de vector y matriz de C++ en el espacio de nombres [**Windows. Foundation. Numerics**](/uwp/api/Windows.Foundation.Numerics) . Extiende los Structs de **Windows. Foundation. Numerics** con un intervalo de funciones y operadores matemáticos.

Este espacio de nombres solo está disponible en C++. Su equivalente de .NET es [System. Numerics](/dotnet/api/system.numerics?view=netframework-4.8).

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|-|-|
| [**estructura float2**](float2-structure.md) | Vector con dos componentes. |
| [**estructura float3**](float3-structure.md) | Vector con tres componentes. |
| [**estructura float3x2**](float3x2-structure.md) | Matriz 3x2, que se usa para las transformaciones 2D. |
| [**FLOAT4 (estructura)**](float4-structure.md) | Vector con cuatro componentes. |
| [**estructura float4x4**](float4x4-structure.md) | Matriz 4x4 utilizada para las transformaciones 3D. |
| [**estructura del plano**](plane-structure.md) | Esta estructura representa un plano mediante un vector 3D normal y un valor de distancia. |
| [**cuaternión (estructura)**](quaternion-structure.md) | Un vector de cuatro dimensiones, que se usa para representar un giro. |
| [**Windows numéricos y API de interoperabilidad de DirectXMath**](windows-numerics-and-directxmath-interop-apis.md) | Estas funciones convierten los tipos [**Windows. Foundation. Numerics**](/uwp/api/Windows.Foundation.Numerics) a y desde los tipos SIMD de DirectXMath [XMVECTOR](../dxmath/xmvector-data-type.md) y [XMMATRIX](/windows/win32/api/directxmath/ns-directxmath-xmmatrix). |
