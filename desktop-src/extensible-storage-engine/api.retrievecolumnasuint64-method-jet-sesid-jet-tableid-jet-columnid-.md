# <a name="apiretrievecolumnasuint64-method-jet_sesid-jet_tableid-jet_columnid"></a>Método API. RetrieveColumnAsUInt64 (JET_SESID, JET_TABLEID, JET_COLUMNID)

Recupera un valor de columna UInt64 del registro actual. El registro es el registro asociado a la entrada de índice en la posición actual del cursor.

Esta API no es conforme a CLS. 

**Espacio de nombres:**  [Microsoft. ISAM. esent. Interop](./microsoft.isam.esent.interop-namespace.md)  
**Ensamblado:**  Microsoft. ISAM. esent. Interop (en Microsoft.Isam.Esent.Interop.dll)
