# <a name="system16folder-property"></a>Propiedad System16Folder

Windows Installer establece la propiedad **System16Folder** en la ruta de acceso completa a la carpeta para los archivos DLL del sistema de 16 bits. Tenga en cuenta que la propiedad **System16Folder** no está definida para Windows de 64 bits porque este sistema no admite aplicaciones o componentes de 16 bits.
