# <a name="u-security-glossary"></a>U (Glosario de seguridad)

<dl> <dt>

<span id="_security_ucs_gly"></span><span id="_SECURITY_UCS_GLY"></span>**UCS**
</dt> <dd>

Consulte *juego de caracteres universal*.

</dd> <dt>

<span id="_security_unicode_gly"></span><span id="_SECURITY_UNICODE_GLY"></span>**Unicode**
</dt> <dd>

Estándar de codificación de caracteres en todo el mundo que permite incluir más información en cada cadena mediante la definición de cadenas de caracteres de 16 bits en lugar de cadenas de caracteres de 8 bits estándar. Unicode permite el intercambio de datos universal y mejora el procesamiento de texto multilingüe. Las cadenas Unicode también se denominan *cadenas anchas*.

</dd> <dt>

<span id="_security_universal_character_set_gly"></span><span id="_SECURITY_UNIVERSAL_CHARACTER_SET_GLY"></span>**Juego de caracteres universal**
</dt> <dd>

UCS Juego de caracteres de 16 bits.

</dd> <dt>

<span id="_security_universal_well_known_security_identifier_gly"></span><span id="_SECURITY_UNIVERSAL_WELL_KNOWN_SECURITY_IDENTIFIER_GLY"></span>**identificador de seguridad conocido universal**
</dt> <dd>

Un [*identificador de seguridad*](s-gly.md) (SID) que se conoce en todos los sistemas que cumplen el modelo de seguridad basado en los usuarios autenticados y el control de acceso discrecional.

</dd> <dt>

<span id="_security_upn_gly"></span><span id="_SECURITY_UPN_GLY"></span>**UPN**
</dt> <dd>

Vea *nombre principal del usuario*.

</dd> <dt>

<span id="_security_user_gly"></span><span id="_SECURITY_USER_GLY"></span>**usuario**
</dt> <dd>

Un usuario de un sistema se identifica mediante un procedimiento de inicio de sesión, que establece un proceso por el que las aplicaciones pueden ejecutar y tener acceso a otros objetos relevantes para la seguridad. Un ejemplo de procedimiento de inicio de sesión es usar una tarjeta inteligente, posiblemente junto con una contraseña o un número de identificación personal (PIN).

</dd> <dt>

<span id="_security_user_interface_smart_card_gly"></span><span id="_SECURITY_USER_INTERFACE_SMART_CARD_GLY"></span>**interfaz de usuario (tarjeta inteligente)**
</dt> <dd>

Cuadro de diálogo común que permite al usuario conectarse a una tarjeta inteligente y usarla en una aplicación. Mediante el cuadro de diálogo, el usuario puede especificar una tarjeta específica o buscar la tarjeta inteligente que se va a abrir.

</dd> <dt>

<span id="_security_user_principal_name_gly"></span><span id="_SECURITY_USER_PRINCIPAL_NAME_GLY"></span>**nombre principal de usuario**
</dt> <dd>

UPN Un nombre de cuenta de usuario (a veces denominado nombre de inicio de sesión de usuario) y un nombre de dominio que identifica el dominio en el que se encuentra la cuenta de usuario. Éste es el uso estándar para iniciar sesión en un dominio de Windows. El formato es: someone@example.com (como para una dirección de correo electrónico).

</dd> </dl>
