# <a name="sfvm_size-message"></a>Mensaje SFVM \_ SIZE

Notifica al objeto de devolución de llamada que se ha cambiado el tamaño de la vista de carpeta. Usado por [**IShellFolderViewCB::MessageSFVCB**](/windows/win32/api/shlobj_core/nf-shlobj_core-ishellfolderviewcb-messagesfvcb).


```C++
SFVM_SIZE
```



## <a name="parameters"></a>Parámetros

Este mensaje no tiene parámetros.
