# <a name="supporting-system-language-settings"></a>Compatibilidad con la configuración de idioma del sistema

La mejor manera de que la aplicación de MUI establezca las preferencias de idioma de la aplicación es admitir el mismo conjunto de idiomas de la interfaz de usuario que el sistema operativo de destino. La aplicación simplemente depende del sistema operativo para seleccionar el idioma de la interfaz de usuario correcto para su presentación. En MUI, se proporciona un ejemplo asociado [: ejemplo de aplicación de configuración del sistema](mui-system-settings-application-sample.md).
