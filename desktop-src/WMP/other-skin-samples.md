# <a name="other-skin-samples"></a>Otros ejemplos de máscara

Para ver ejemplos de máscaras de trabajo completas, haga clic en el botón **Inicio** , seleccione **programas**, después **Windows Media**, **SDK de Windows Media** y, a continuación, haga clic en **SDK del reproductor**. Primero debe instalar el SDK de Windows Media Player para usar los ejemplos.
