# <a name="update-assessor-structures"></a>Actualizar las estructuras de asesor

El proveedor de descriptores de acceso de actualización usa las siguientes funciones.



| Función                                         | Descripción                                                           |
|--------------------------------------------------|-----------------------------------------------------------------------|
| [**OSUpdateAssessment**](/windows/win32/api/waasapitypes/ns-waasapitypes-osupdateassessment) | Define el grado de actualización del sistema operativo en un dispositivo de destino.                |
| [**UpdateAssessment**](/windows/win32/api/waasapitypes/ns-waasapitypes-osupdateassessment)   | Contiene información que evalúa el grado de actualización de un sistema operativo instalado. |
