# <a name="iupdateendpointauthprovider-methods"></a>Métodos IUpdateEndpointAuthProvider

La [**interfaz IUpdateEndpointAuthProvider**](iupdateendpointauthprovider.md) define los métodos siguientes.



| Método                                                                                             | Descripción                                                                         |
|----------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------|
| [**GetEndpointToken**](iupdateendpointauthprovider-getendpointtoken.md)                           | Solicite un token para el punto de conexión del servicio con las credenciales especificadas.    |
| [**GetPreferredEndpointTokenType**](iupdateendpointauthprovider-getpreferredendpointtokentype.md) | Devuelve el tipo preferido de token de autenticación para el punto de conexión del servicio. |
