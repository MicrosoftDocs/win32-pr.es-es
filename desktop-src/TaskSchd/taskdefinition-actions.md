# <a name="taskdefinitionactions-property"></a>Propiedad TaskDefinition. Actions

Para scripting, obtiene o establece una colección de acciones que realiza la tarea.

## <a name="syntax"></a>Sintaxis


```VB
TaskDefinition.Actions As ActionCollection
```



## <a name="property-value"></a>Valor de propiedad

Colección de acciones que realiza la tarea.
