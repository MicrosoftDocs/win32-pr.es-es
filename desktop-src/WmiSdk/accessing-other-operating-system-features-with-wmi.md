# <a name="accessing-other-operating-system-features-with-wmi"></a>Obtener acceso a otras características del sistema operativo con WMI

WMI tiene extensiones o proveedores que le permiten tener acceso a otras características del sistema operativo, como Active Directory, ODBC y SNMP. También puede tener acceso a los datos desde los controladores de dispositivos.

Las extensiones WMI no están disponibles en todos los sistemas operativos. Para obtener más información, vea [disponibilidad del sistema operativo de los componentes WMI](operating-system-availability-of-wmi-components.md).

En los temas siguientes se proporciona información acerca del acceso a otras características del sistema operativo:

-   [Acceder a los datos desde los controladores de dispositivos](accessing-data-from-device-drivers.md)
-   [Obtener acceso a Active Directory](accessing-active-directory.md)
-   [Acceso a dispositivos SNMP](accessing-snmp-devices.md)
