# <a name="prefetch-debug-apis-for-windows-store-apps"></a>Captura previa de las API de depuración para aplicaciones de la tienda Windows

Las siguientes API se usan para depurar y probar el comportamiento de captura previa de contenido en una aplicación de la tienda Windows que implementa la clase [**ContentPrefetcher**](/uwp/api/Windows.Networking.BackgroundTransfer.ContentPrefetcher) .



| Interfaz                                                              | Descripción                                                                                                                                                                      |
|------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IContentPrefetcherTaskTrigger**](/windows/desktop/api/IContentPrefetcherTaskTrigger/nn-icontentprefetchertasktrigger-icontentprefetchertasktrigger) | Use esta interfaz para comprobar que un paquete de aplicación de la tienda Windows instalado está registrado para esta tarea en segundo plano y desencadenar manualmente las operaciones de captura previa de contenido.<br/> |
