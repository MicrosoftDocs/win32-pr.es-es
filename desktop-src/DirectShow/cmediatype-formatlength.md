# <a name="cmediatypeformatlength-method"></a>CMediaType. FormatLength, método

El `FormatLength` método recupera la longitud del bloque de formato.

## <a name="syntax"></a>Sintaxis


```C++
ULONG FormatLength() const;
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve el miembro **cbFormat** .
