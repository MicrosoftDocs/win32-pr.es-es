# <a name="check-box-msaa-ui-element-reference"></a>Casilla (referencia de elementos de interfaz de usuario de MSAA)

> [!Note]  
> En este tema se describen los objetos de **casilla** para la referencia de elementos de la interfaz de usuario de MSAA. La forma de crear objetos de **casilla** en varios marcos de interfaz de usuario no se describe aquí. Consulte la documentación de referencia de la API del marco de interfaz de usuario que está usando.

 

Las casillas se usan para habilitar o deshabilitar una o varias características o opciones de un conjunto, normalmente en un cuadro de diálogo. Normalmente, una casilla contiene un cuadro pequeño con texto adyacente. Cuando se selecciona una opción, aparece una marca de verificación en el cuadro.

El nombre de la clase de ventana de una casilla es "BUTTON".
