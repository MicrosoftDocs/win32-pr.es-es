# <a name="boundingrectnothittestable"></a>BoundingRectNotHitTestable

## <a name="text"></a>Texto

El rectángulo notificado no parece contener un punto en el que este elemento se pueda recuperar con las pruebas de acceso.

## <a name="type"></a>Tipo

Advertencia

## <a name="description"></a>Descripción

Las pruebas de acceso no pueden acceder al rectángulo delimitador del elemento. Por ejemplo, se encuentra fuera de las coordenadas físicas de la pantalla (o región que se puede ver) del escritorio.
