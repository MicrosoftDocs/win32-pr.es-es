# <a name="xmshortn2-operators"></a>Operadores XMSHORTN2

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección

|--------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**operador =**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmshortn2-operator-assign)<br/> | Asigna los datos del componente de vector de una instancia de `XMSHORTN2` a la instancia actual de `XMSHORTN2` .<br/> |
