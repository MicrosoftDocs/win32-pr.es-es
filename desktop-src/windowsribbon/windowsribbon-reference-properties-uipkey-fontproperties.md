# <a name="ui_pkey_fontproperties"></a>UI \_ PKEY \_ FontProperties

Identifica la propiedad FontProperties de PKEY de la interfaz de usuario \_ \_ .

```
propertyDescription
   name = UI_PKEY_FontProperties
   shellPKey = UI_PKEY_FontProperties
   formatID = 00000300-7363-696e-8441798acf5aebb7
   propID = 300
   typeInfo
      type = IPropertyStore
```

## <a name="remarks"></a>Observaciones

UI \_ PKEY \_ FontProperties es un almacén de propiedades que una aplicación usa para consultar todas las propiedades de fuente.
