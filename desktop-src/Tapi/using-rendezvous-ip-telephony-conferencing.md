# <a name="using-rendezvous-ip-telephony-conferencing"></a>Uso de la conferencia de telefonía IP de Encuentro

\[Los controles e interfaces de conferencias de telefonía IP de encuentro no están disponibles para su uso en Windows Vista, Windows Server 2008 y versiones posteriores del sistema operativo. La API de cliente RTC proporciona una funcionalidad similar.\]

Rendezvous consta de tres conjuntos básicos de interfaces:

-   [Los controles de](directory-controls.md) directorio proporcionan un medio para detectar, crear o modificar la información de conferencias y usuarios en un servidor ILS o NTDS.
-   [Los controles de blobs en](conference-blob-controls.md) conferencias permiten modificaciones específicas del proveedor en una conferencia.
-   [Las interfaces COM de multidifusión](multicast-com-interfaces.md) proporcionan un medio para reservar direcciones para su uso por conferencias de multidifusión.
