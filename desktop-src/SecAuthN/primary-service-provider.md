# <a name="primary-service-provider"></a>Proveedor de servicios principal

Cada [*tarjeta inteligente*](../secgloss/s-gly.md) que conoce el sistema puede tener una interfaz com registrada como su proveedor de servicios principal. Cuando se tiene acceso a una *tarjeta inteligente* a través de com, este [*proveedor de servicios principal*](../secgloss/p-gly.md) proporciona las interfaces de control a la tarjeta. Esto permite que los servicios de tarjeta se expongan a una amplia gama de entornos de programación, incluidos Java y el sistema de desarrollo de Microsoft Visual Basic.
