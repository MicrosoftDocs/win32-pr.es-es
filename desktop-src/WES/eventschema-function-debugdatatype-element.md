# <a name="function-debugdatatype-element"></a>Function (DebugDataType), elemento

El nombre de la función que registró el mensaje de seguimiento.

``` syntax
<xs:element name="Function"
    type="string"
    minOccurs="0"
 />
```

El elemento de **función** se define mediante el tipo complejo de [**DebugDataType**](eventschema-debugdatatype-complextype.md) .
