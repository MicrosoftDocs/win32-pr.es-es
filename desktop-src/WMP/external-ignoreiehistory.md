# <a name="externalignoreiehistory"></a>External. ignoreIEHistory

> [!Note]  
> En este tema se describe la funcionalidad diseñada para su uso en tiendas en línea. No se admite el uso de esta funcionalidad fuera del contexto de una tienda en línea.

 

La propiedad **ignoreIEHistory** especifica si Windows Media Player debe omitir el historial de Internet Explorer.

``` syntax
window.external.ignoreIEHistory
      
```

## <a name="possible-values"></a>Valores posibles

Esta propiedad es un **booleano** de solo escritura. **True** indica que Windows Media Player debe omitir el historial.
