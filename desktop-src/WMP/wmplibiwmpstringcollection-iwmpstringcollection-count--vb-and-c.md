# <a name="iwmpstringcollectioncount-property"></a>IWMPStringCollection:: Count (propiedad)

La propiedad **Count** obtiene el número de elementos de la colección de cadenas.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```CSharp
public System.Int32 count {get;}
```


```VB

Public ReadOnly Property count As System.Int32
```





## <a name="property-value"></a>Valor de propiedad

**System. Int32** que es el número de elementos de la colección de cadenas.

## <a name="remarks"></a>Observaciones

Antes de usar esta propiedad, debe tener acceso de lectura a la biblioteca. Para obtener más información, vea [acceso a la biblioteca](library-access.md).
