# <a name="windows-networking-callbacks"></a>Windows Devoluciones de llamada de red

Los siguientes prototipos de función se usan en Windows Networking.



| Prototipo de función                                 | Descripción                                                                                                           |
|----------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------|
| [**PFNGETPROFILEPATH**](/previous-versions/windows/desktop/legacy/aa385358(v=vs.85))     | Prototipo de una función que obtiene una ruta de acceso de perfil de usuario.                                                            |
| [**PFNPROCESSPOLICIES**](/previous-versions/windows/desktop/legacy/aa385360(v=vs.85))   | Prototipo de una función que devuelve la información de directiva de cuenta de usuario asociada a un proceso de Windows específico. |
| [**PFNRECONCILEPROFILE**](/previous-versions/windows/desktop/legacy/aa385364(v=vs.85)) | Prototipo de una función que concilia un perfil local con su perfil global correspondiente.                        |
