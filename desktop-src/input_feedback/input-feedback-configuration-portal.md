# <a name="input-feedback-configuration"></a>Configuración de comentarios de entrada

## <a name="purpose"></a>Propósito

En los temas de esta sección se proporciona información general sobre la configuración de comentarios de entrada en Windows 8. La configuración de comentarios de entrada proporciona acceso a la configuración de comentarios visuales para las interacciones del usuario.

La colección de temas que se proporciona aquí describe cómo usar la configuración de comentarios de entrada para optimizar la experiencia del usuario de la aplicación. Por ejemplo, es posible que la configuración predeterminada de comentarios visuales no sea necesaria o deseable en una aplicación de juego debido a interferencias o conflictos con el propio juego.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Los ConfigurationAPIss de comentarios de entrada están diseñados para desarrolladores que compilan marcos de interfaz de usuario que proporcionan una experiencia de usuario coherente y optimizada para tactos en las aplicaciones de escritorio.
