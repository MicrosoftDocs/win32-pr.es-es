# <a name="windows-design-basics"></a>Windows (Conceptos básicos del diseño)

> [!NOTE]
> Esta guía de diseño se creó para Windows 7 y no se ha actualizado para las versiones más recientes de Windows. Gran parte de las instrucciones se siguen aplicando en principio, pero la presentación y los ejemplos no reflejan nuestra [guía de diseño actual.](/windows/uwp/design/)

Windows son los principales "lienzos" o superficies de interfaz de usuario de la aplicación de escritorio, incluidas las ventanas principales y los elementos emergentes, los diálogos y los asistentes. Siga estas directrices a la hora de decidir qué superficie usar y cómo usarlas mejor.
