# <a name="invocation-id-attribute"></a>Invocation-Id atributo)

Se utiliza para identificar de forma única cada directorio de Microsoft Exchange Server de la organización.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | Invocation-Id                                         |
| Nombre para mostrar de LDAP | Vocación                                          |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | Cuando se instala el servidor de Exchange.                |
| Attribute-Id      | 1.2.840.113556.1.2.115                                |
| System-ID-GUID    | bf96798e-0de6-11d0-a285-00aa003049e2                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
