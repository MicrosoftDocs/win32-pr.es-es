# <a name="hasborder-control-attribute"></a>Atributo HasBorder Control

Si se establece este bit, [RadioButtonGroup](radiobuttongroup-control.md) tiene texto y un borde que se muestra a su alrededor.

Si no se establece el bit de estilo, no se muestra el borde y no se muestra texto en el grupo.

## <a name="valid-controls"></a>Controles válidos

[RadioButtonGroup](radiobuttongroup-control.md)

## <a name="valid"></a>Válido



| Decimal  | Hexadecimal | Constante                            |
|----------|-------------|-------------------------------------|
| 16777216 | 0x01000000  | **msidbControlAttributesHasBorder** |



 

## <a name="remarks"></a>Comentarios

Para establecer este atributo en un control , incluya los bits HasBorder en la columna Atributos del registro del control en la [tabla Control](control-table.md).

Vea [Atributos de](control-attributes.md) control y el control que debe crear en [Controles](controls.md).
