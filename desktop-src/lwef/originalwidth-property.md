# <a name="originalwidth-property"></a>Propiedad OriginalWidth

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descripción**
</dt> <dd>

Devuelve un valor entero que especifica el ancho original del carácter.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxis**
</dt> <dd>

*agent.***Caracteres("**_CharacterID_*_"). OriginalWidth_*

</dd> </dl>

## <a name="remarks"></a>Comentarios

Esta propiedad devuelve el ancho del marco del carácter tal como se ha creado con el Editor de caracteres de Microsoft Agent.
