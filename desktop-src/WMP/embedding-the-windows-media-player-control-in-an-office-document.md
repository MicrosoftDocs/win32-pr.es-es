# <a name="embedding-the-windows-media-player-control-in-an-office-document"></a>Insertar el control Reproductor de Windows Media en un documento Office trabajo

Insertar el control Reproductor de Windows Media en un documento Office es una manera sencilla de agregar contenido multimedia digital dinámico e interactivo a un documento estático de otro modo. Por ejemplo, puede crear una hoja de cálculo en Microsoft Excel e insertar un vídeo de "cabeza de conversación" que resuma un informe, o bien puede crear un documento Microsoft Word e insertar una breve animación que ilustra un punto realizado en el texto. Si no le gusta la interfaz de usuario proporcionada por el control , puede usar Microsoft Visual Basic para Aplicaciones (VBA) para proporcionar una interfaz de usuario personalizada.

La inserción del control Reproductor de Windows Media en una presentación de diapositivas de Microsoft PowerPoint® permite complementar los efectos dinámicos ya proporcionados por ese programa. Al guardar la presentación como una página web, el control Reproductor de Windows Media se incrusta en el código HTML, tal como se describe en Uso del [control Reproductor de Windows Media en una página web](using-the-windows-media-player-control-in-a-web-page.md). Esto le permite agregar código de script adicional que puede haber desarrollado para otras páginas web.

Otra manera de generar fácilmente páginas web que insertan el control Reproductor de Windows Media es con Microsoft FrontPage®. FrontPage permite crear rápidamente una página, agregar el control Reproductor de Windows Media desde un menú y configurarlo en un cuadro de diálogo de propiedades. De nuevo, el CÓDIGO HTML resultante inserta el control Reproductor de Windows Media como se describió anteriormente.
