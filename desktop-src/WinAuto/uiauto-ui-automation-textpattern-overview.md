# <a name="ui-automation-support-for-textual-content"></a>Compatibilidad de UI Automation para el contenido textual

En esta sección se describe cómo usar la automatización de la interfaz de usuario de Microsoft para exponer el contenido textual, incluidos los atributos de estilo y formato, de los controles de texto en plataformas compatibles con la automatización de la interfaz de usuario. Los controles de texto incluyen, entre otros, los documentos, la información sobre herramientas, los controles de edición y los controles Rich Edit.
