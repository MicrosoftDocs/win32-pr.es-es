# <a name="peer-collaboration"></a>Colaboración del mismo nivel

La infraestructura de colaboración del mismo nivel proporciona un conjunto de interfaces de programación de aplicaciones que se pueden usar junto con la infraestructura del mismo nivel para habilitar aplicaciones de red sin servidor de colaboración, como mensajería instantánea y encuentros de juegos multijugador.
