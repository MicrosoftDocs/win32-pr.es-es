# <a name="taskfoldercollection-object"></a>Objeto TaskFolderCollection

Objeto de scripting que proporciona información y control para una colección de carpetas que contienen tareas.

## <a name="members"></a>Miembros

El **objeto TaskFolderCollection** tiene estos tipos de miembros:

-   [Propiedades](#properties)

### <a name="properties"></a>Propiedades

El **objeto TaskFolderCollection** tiene estas propiedades.



| Propiedad                                               | Descripción                                                |
|:-------------------------------------------------------|:-----------------------------------------------------------|
| [**Count**](taskfoldercollection-count.md)<br/> | Obtiene el número de carpetas de la colección.<br/>   |
| [**Elemento**](taskfoldercollection-item.md)<br/>   | Obtiene la carpeta especificada de la colección. <br/> |
