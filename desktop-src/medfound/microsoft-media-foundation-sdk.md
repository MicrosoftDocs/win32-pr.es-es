# <a name="microsoft-media-foundation"></a>Microsoft Media Foundation

Microsoft Media Foundation permite el desarrollo de aplicaciones y componentes para el uso de medios digitales en Windows Vista y versiones posteriores.

Media Foundation es la plataforma multimedia de próxima generación para Windows que permite a los desarrolladores, consumidores y proveedores de contenido adoptar la nueva ola de contenido Premium con una solidez mejorada, una calidad sin precedentes e interoperabilidad sin problemas.
