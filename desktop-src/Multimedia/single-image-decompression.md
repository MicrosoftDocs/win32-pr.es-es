# <a name="single-image-decompression"></a>Descompresión de Single-Image

Puede usar la función [**ICImageDecompress**](/windows/desktop/api/Vfw/nf-vfw-icimagedecompress) para descomprimir una sola imagen. Esta función devuelve un identificador del DIB descomprimido. El DIB descomprimido se almacena en \_ formato DIB de CF.
