# <a name="helper-interfaces-for-direct3d-12"></a>Interfaces auxiliares para Direct3D 12

Estas interfaces auxiliares ayudan especialmente a controlar los subrecursos y se declaran en `d3dx12.h` .

`d3dx12.h` está disponible por separado de los encabezados de Direct3D 12. Puede descargar desde `d3dx12.h` [la biblioteca auxiliar D3D12](https://github.com/microsoft/DirectX-Headers/blob/main/include/directx/d3dx12.h).

## <a name="in-this-section"></a>En esta sección

| Tema | Descripción |
|-|-|
| [**ID3DX12PipelineParserCallbacks**](id3dx12pipelineparsercallbacks.md) | Interfaz que representa una colección de devoluciones de llamada de error y analizador que usa la función auxiliar [**D3DX12parsePipelineStream.**](d3dx12parsepipelinestream.md) |
