# <a name="ms-ds-default-quota-attribute"></a>Atributo ms-DS-Default-Quota

Cuota predeterminada que se aplicará a una entidad de seguridad que crea un objeto en el NC si no existe ninguna especificación de cuota que a cubra la entidad de seguridad.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | ms-DS-Default-Quota                  |
| Ldap-Display-Name | msDS-DefaultQuota                    |
| Size              | \-                                   |
| Privilegio actualizar  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1846              |
| System-Id-Guid    | 6818f726-674b-441b-8a3a-f40596374 |
| Syntax            | [**Enumeración**](s-enumeration.md) |
