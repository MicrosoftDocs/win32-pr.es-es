# <a name="id2d1devicecontextfillopacitymask-methods"></a>Métodos ID2D1DeviceContext::FillOpacityMask

Rellene con el canal alfa del mapa de bits de máscara de opacidad proporcionado. La opacidad del pincel se modulará mediante la máscara. El modo de suavizado de contorno de destino de representación debe establecerse en alias.
