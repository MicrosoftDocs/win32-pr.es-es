# <a name="tb_setstyle-message"></a>TB \_ mensaje SETSTYLE

Establece el estilo de un control de barra de herramientas.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>Debe ser cero.</dd> <dt>

*lParam* 
</dt> <dd>

Valor que especifica los estilos que se van a establecer para el control. Este valor puede ser una combinación de [estilos de control](toolbar-control-and-button-styles.md)de la barra de herramientas.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

No de devuelve ningún valor.
