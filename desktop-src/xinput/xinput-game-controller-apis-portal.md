# <a name="xinput-game-controller-apis"></a>API de XInput Game Controller

## <a name="purpose"></a>Propósito

XInput Game Controller API permite a las aplicaciones recibir entradas del controlador de Xbox para Windows.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Las API de controlador de juegos XInput están diseñadas para que las usen los desarrolladores que quieran usar el controlador xbox para sus Windows aplicaciones.
