# <a name="locale_snegativesign"></a>LOCALE \_ SNEGATIVESIGN

Valor de cadena para el signo negativo, por ejemplo, "-" para la configuración regional de inglés (Estados Unidos). El número máximo de caracteres permitidos para esta cadena es cinco, incluido un carácter nulo de terminación.
