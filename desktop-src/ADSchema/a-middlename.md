# <a name="other-name-attribute"></a>Other-Name atributo

Nombres adicionales para un usuario. Por ejemplo, el nombre intermedio, el términoymic, matronímico u otros.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Other-Name                                  |
| Ldap-Display-Name | middleName                                  |
| Size              | \-                                          |
| Actualizar privilegios  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 2.16.840.1.113730.3.1.34                    |
| System-Id-Guid    | bf9679f2-0de6-11d0-a285-00aa003049e2        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
