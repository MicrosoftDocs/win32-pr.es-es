# <a name="msix64-property"></a>Propiedad Msix64

La **propiedad Msix64** solo se define si se ejecuta en un procesador x64. El instalador establece el valor en el nivel de procesador numérico. Los valores son los mismos que el **campo wProcessorLevel** de la [**estructura SYSTEM \_ INFO.**](/windows/win32/api/sysinfoapi/ns-sysinfoapi-system_info)

## <a name="remarks"></a>Comentarios

**Windows Installer 2.0 y Windows Installer 3.0:** No se admite. La **propiedad Msix64** está disponible a partir de Windows Installer 3.1.
