# <a name="imsrdpclientshelllaunch-method"></a>IMsRdpClientShell:: Launch (método)

Inicia el contenido del archivo remoto desde Escritorio remoto Web Access (acceso web de escritorio remoto) o desde otro portal web.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT Launch();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Si este método se ejecuta correctamente, devuelve **S \_ correcto**. De lo contrario, devuelve un código de error **HRESULT** .
