# <a name="directcomposition-interfaces"></a>Interfaces de DirectComposition

En esta sección se describen las interfaces proporcionadas por Microsoft DirectComposition API.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                               | Descripción                                                                                                                                                                                                                                                                                                                                                                                               |
|-----------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**IDCompositionAffineTransform2DEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionaffinetransform2deffect)<br/>     | El efecto compuesto aritmético se usa para combinar 2 imágenes mediante una suma ponderada de píxeles de las imágenes de entrada. <br/>                                                                                                                                                                                                                                                                             |
| [**IDCompositionAnimation**](/windows/desktop/api/DcompAnimation/nn-dcompanimation-idcompositionanimation)<br/>                                 | Representa una función para animar una o varias propiedades de uno o varios objetos DirectComposition.<br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionArithmeticCompositeEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionarithmeticcompositeeffect)<br/> | El efecto compuesto aritmético se usa para combinar 2 imágenes mediante una suma ponderada de píxeles de las imágenes de entrada.<br/>                                                                                                                                                                                                                                                                              |
| [**IDCompositionBlendEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionblendeffect)<br/>                             | El efecto Blend se usa para combinar 2 imágenes. <br/>                                                                                                                                                                                                                                                                                                                                                 |
| [**IDCompositionBrightnessEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionbrightnesseffect)<br/>                   | El efecto de brillo controla el brillo de la imagen.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionColorMatrixEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioncolormatrixeffect)<br/>                 | El efecto de matriz de colores modifica los valores RGBA de un mapa de bits.<br/>                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionCompositeEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioncompositeeffect)<br/>                     | El efecto compuesto se usa para combinar 2 o más imágenes. Este efecto tiene 13 modos compuestos diferentes. El efecto compuesto acepta 2 o más entradas. Cuando se especifican 2 imágenes, el destino es la primera entrada (índice 0) y el origen es la segunda entrada (índice 1). Si especifica más de 2 entradas, las imágenes se componen a partir de la primera entrada y la segunda, etc. <br/> |
| [**IDCompositionClip**](/windows/win32/api/dcomp/nn-dcomp-idcompositionclip)<br/>                                           | Representa un objeto clip que se usa para restringir la representación de un subárbol visual a un área rectangular. <br/>                                                                                                                                                                                                                                                                                    |
| [**IDCompositionDesktopDevice**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondesktopdevice)<br/>                         | Una aplicación debe usar la interfaz IDCompositionDesktopDevice para poder usar DirectComposition en una aplicación de escritorio Win32. Esta interfaz permite a la aplicación conectar un árbol visual a una ventana y hospedar ventanas secundarias en capas para la composición<br/>                                                                                                                                |
| [**IDCompositionDevice**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice)<br/>                                       | Actúa como generador para todos los demás objetos DirectComposition y proporciona métodos para controlar la composición transaccional.<br/>                                                                                                                                                                                                                                                                         |
| [**IDCompositionDevice2**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice2)<br/>                                     | Actúa como generador para todos los demás objetos DirectComposition y proporciona métodos para controlar la composición transaccional.<br/>                                                                                                                                                                                                                                                                         |
| [**IDCompositionDevice3**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevice3)<br/>                                     | Actúa como generador para todos los demás objetos DirectComposition y proporciona métodos para controlar la composición transaccional. <br/>                                                                                                                                                                                                                                                                        |
| [**IDCompositionDeviceDebug**](/windows/win32/api/dcomp/nn-dcomp-idcompositiondevicedebug)<br/>                             | Proporciona acceso a características de representación que ayudan con la depuración de aplicaciones y el ajuste del rendimiento. Esta interfaz se puede consultar desde la interfaz del dispositivo DirectComposition.<br/>                                                                                                                                                                                                                  |
| [**IDCompositionEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositioneffect)<br/>                                       | Representa un efecto de mapa de bits que modifica la rasterización del subárbol de un objeto visual. <br/>                                                                                                                                                                                                                                                                                                             |
| [**IDCompositionEffectGroup**](/windows/win32/api/dcomp/nn-dcomp-idcompositioneffectgroup)<br/>                             | Representa un grupo de efectos de mapa de bits que se aplican juntos para modificar la rasterización del subárbol de un objeto visual. <br/>                                                                                                                                                                                                                                                                             |
| [**IDCompositionFilterEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionfiltereffect)<br/>                           | Representa un efecto de filtro.<br/>                                                                                                                                                                                                                                                                                                                                                                    |
| [**IDCompositionFloodEffect**](/previous-versions/windows/desktop/legacy/dn919732(v=vs.85))<br/>                             | El efecto de desbordamiento se usa para generar un mapa de bits basado en el color y el valor alfa especificados. Puede usar este efecto cuando desee un color específico como entrada para un efecto, como un color de fondo. <br/>                                                                                                                                                                                        |
| [**IDCompositionGaussianBlurEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositiongaussianblureffect)<br/>               |                                                                                                                                                                                                                                                                                                                                                                                                           |
| [**IDCompositionHueRotationEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionhuerotationeffect)<br/>                 | El efecto de rotación de matiz modifica el matiz de una imagen aplicando una matriz de colores basada en el ángulo de rotación. <br/>                                                                                                                                                                                                                                                                                      |
| [**IDCompositionLinearTransferEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionlineartransfereffect)<br/>           | El efecto de transferencia lineal se usa para asignar las intensidades de color de una imagen mediante una función lineal creada a partir de una lista de valores que se proporcionan para cada canal. <br/>                                                                                                                                                                                                                               |
| [**IDCompositionMatrixTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionmatrixtransform)<br/>                     | Representa una transformación 2D afín arbitraria definida por una matriz 3 por 2.<br/>                                                                                                                                                                                                                                                                                                                   |
| [**IDCompositionMatrixTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionmatrixtransform3d)<br/>                 | Representa una transformación 3D arbitraria definida por una matriz 4 por 4.<br/>                                                                                                                                                                                                                                                                                                                          |
| [**IDCompositionRectangleClip**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrectangleclip)<br/>                         | Representa un objeto clip que restringe la representación de un subárbol visual a la región rectangular especificada. Opcionalmente, el objeto clip puede tener esquinas redondeadas especificadas.<br/>                                                                                                                                                                                                                 |
| [**IDCompositionRotateTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrotatetransform)<br/>                     | Representa una transformación 2D que afecta a la rotación de un objeto visual alrededor del eje Z. El sistema de coordenadas gira alrededor del punto central especificado. <br/>                                                                                                                                                                                                                                   |
| [**IDCompositionSaturationEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsaturationeffect)<br/>                   | Este efecto se usa para modificar la saturación de una imagen. El efecto de saturación es una especialización del efecto de matriz de colores. <br/>                                                                                                                                                                                                                                                                |
| [**IDCompositionRotateTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionrotatetransform3d)<br/>                 | Representa una transformación 3D que afecta a la rotación de un objeto visual a lo largo de un eje arbitrario en un espacio 3D. El sistema de coordenadas gira alrededor del punto central especificado. <br/>                                                                                                                                                                                                                 |
| [**IDCompositionScaleTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionscaletransform)<br/>                       | Representa una transformación 2D que afecta a la escala de un objeto visual a lo largo del eje X y el eje Y. El sistema de coordenadas se escala desde el punto central especificado. <br/>                                                                                                                                                                                                                               |
| [**IDCompositionScaleTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositionscaletransform3d)<br/>                   | Representa un efecto de transformación 3D que afecta a la escala de un objeto visual a lo largo del eje X, el eje Y y el eje Z. El sistema de coordenadas se escala desde el punto central especificado. <br/>                                                                                                                                                                                                               |
| [**IDCompositionSkewTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositionskewtransform)<br/>                         | Representa una transformación 2D que afecta al sesgo de un objeto visual a lo largo del eje X y el eje Y. El sistema de coordenadas está sesgado alrededor del punto central especificado. <br/>                                                                                                                                                                                                                              |
| [**IDCompositionShadowEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionshadoweffect)<br/>                           | El efecto de sombra se usa para generar una sombra a partir del canal alfa de una imagen. La sombra es más opaca para los valores alfa más altos y más transparente para los valores alfa inferiores. Puede establecer la cantidad de desenfoque y el color de la sombra. <br/>                                                                                                                                                 |
| [**IDCompositionSurface**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsurface)<br/>                                     | Representa un mapa de bits físico que se puede asociar a un objeto visual para la composición en un árbol visual. Esta interfaz también se puede usar para actualizar el contenido del mapa de bits. <br/>                                                                                                                                                                                                                            |
| [**IDCompositionTableTransferEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontabletransfereffect)<br/>             | El efecto de transferencia de tabla se usa para asignar las intensidades de color de una imagen mediante una función de transferencia creada a partir de la interpolación de una lista de valores que proporcione.<br/>                                                                                                                                                                                                                                  |
| [**IDCompositionSurfaceFactory**](/windows/win32/api/dcomp/nn-dcomp-idcompositionsurfacefactory)<br/>                       | Crea objetos de superficie y superficie virtual asociados a un dispositivo de representación proporcionado por la aplicación.<br/>                                                                                                                                                                                                                                                                                          |
| [**IDCompositionTarget**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontarget)<br/>                                       | Representa un enlace entre un árbol visual DirectComposition y un destino sobre el que se debe componer el árbol visual. <br/>                                                                                                                                                                                                                                                            |
| [**IDCompositionTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontransform)<br/>                                 | Representa una transformación 2D que se puede usar para modificar el espacio de coordenadas de un subárbol visual. <br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontransform3d)<br/>                             | Representa un efecto de transformación 3D que se puede usar para modificar la rasterización de un subárbol visual. <br/>                                                                                                                                                                                                                                                                                       |
| [**IDCompositionTranslateTransform**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontranslatetransform)<br/>               | Representa una transformación 2D que afecta solo al desplazamiento de un objeto visual a lo largo del eje X y el eje Y.<br/>                                                                                                                                                                                                                                                                                           |
| [**IDCompositionTurbulenceEffect**](/windows/win32/api/dcomp/nn-dcomp-idcompositionturbulenceeffect)<br/>                   | El efecto de rebulencia se usa para generar un mapa de bits basado en la función de ruido perlin. El efecto de refluencia no tiene ninguna imagen de entrada. <br/>                                                                                                                                                                                                                                                              |
| [**IDCompositionTranslateTransform3D**](/windows/win32/api/dcomp/nn-dcomp-idcompositiontranslatetransform3d)<br/>           | Representa una transformación 3D que afecta al desplazamiento de un objeto visual a lo largo del eje X, el eje Y y el eje Z. <br/>                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVirtualSurface**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvirtualsurface)<br/>                       | Representa un mapa de bits asignado dispersamente que se puede asociar a un objeto visual para la composición en un árbol visual.<br/>                                                                                                                                                                                                                                                                                  |
| [**IDCompositionVisual**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual)<br/>                                       | Representa un objeto visual DirectComposition. <br/>                                                                                                                                                                                                                                                                                                                                                        |
| [**IDCompositionVisual2**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual2)<br/>                                     | Representa un objeto visual DirectComposition en un árbol visual.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVisual3**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisual3)<br/>                                     | Representa un objeto visual DirectComposition en un árbol visual.<br/>                                                                                                                                                                                                                                                                                                                                      |
| [**IDCompositionVisualDebug**](/windows/win32/api/dcomp/nn-dcomp-idcompositionvisualdebug)<br/>                             | Representa un objeto visual de depuración.<br/>                                                                                                                                                                                                                                                                                                                                                                     |
