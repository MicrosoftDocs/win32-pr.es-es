# <a name="wmmediastationname"></a>WM/MediaStationName

El **atributo WM/MediaStationName** contiene el nombre de la estación que difunde el contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaStationName

## <a name="data-type"></a>Tipo de datos

**CADENA DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentarios

Este atributo se usa para la televisión de difusión.
