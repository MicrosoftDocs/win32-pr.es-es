# <a name="ms-ds-reveal-ondemand-group-attribute"></a>Atributo ms-DS-Reveal-OnDemand-Group

Se usa con RODC para definir qué usuarios, equipos y grupos pueden tener sus contraseñas almacenadas en caché en un RODC.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | ms-DS-Reveal-OnDemand-Group             |
| Ldap-Display-Name | msDS-RevealOnDemandGroup                |
| Size              | \-                                      |
| Actualizar privilegios  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1928                 |
| System-Id-Guid    | 303d9f4a-1dd6-4b38-8fc5-33afe8c988ad    |
| Syntax            | [**Object(DS-DN)**](s-object-ds-dn.md) |
