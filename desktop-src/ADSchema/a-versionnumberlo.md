# <a name="version-number-lo-attribute"></a>Versión: atributo Number

Un número de versión secundaria de uso general.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Versión: número-lo                    |
| Nombre para mostrar de LDAP | versionNumberLo                      |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.329               |
| System-ID-GUID    | 7d6c0e9b-7e20-11d0-afd6-00c04fd930c9 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
