# <a name="imsrdpclientshelllaunch-method"></a>IMsRdpClientShell::Launch (Método)

Inicia contenido de archivo remoto desde Escritorio remoto Web Access (Acceso web de Escritorio remoto) o desde algún otro portal web.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT Launch();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Si este método se realiza correctamente, devuelve **S \_ OK**. De lo contrario, devuelve un código de error **HRESULT.**
