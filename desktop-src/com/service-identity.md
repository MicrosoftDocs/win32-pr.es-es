# <a name="service-identity"></a>Identidad de servicio

La elección de esta identidad hace que la aplicación se ejecute como un servicio. Puede establecer las cuentas de servicio eligiendo la opción **servicios** del **Panel de control**. Para obtener más información, vea [instalar como aplicación de servicio](installing-as-a-service-application.md).
