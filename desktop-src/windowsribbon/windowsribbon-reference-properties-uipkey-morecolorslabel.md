# <a name="ui_pkey_morecolorslabel"></a>UI \_ PKEY \_ MoreColorsLabel

Identifica la propiedad MoreColorsLabel de PKEY de la interfaz de usuario \_ \_ .

```
propertyDescription
   name = UI_PKEY_MoreColorsLabel
   shellPKey = UI_PKEY_MoreColorsLabel
   formatID = 00000408-7363-696e-8441798acf5aebb7
   propID = 408
   typeInfo
      type = VT_LPWSTR
```

## <a name="remarks"></a>Observaciones

\_ \_ Una aplicación usa la interfaz de usuario PKEY MoreColorsLabel para consultar el valor de la etiqueta asociada al botón **más colores** de una [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md).

\_La interfaz de usuario PKEY \_ MoreColorsLabel solo es válida para [**DropDownColorPicker**](windowsribbon-element-dropdowncolorpicker.md) `ThemeColors` , donde o `StandardColors` se especifica como el valor del atributo **ColorTemplate** .
