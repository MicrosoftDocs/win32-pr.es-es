# <a name="using-other-symbol-stores"></a>Usar otros almacenes de símbolos

Es posible escribir su propio programa de creación del almacén de símbolos, en lugar de usar SymStore.

Dado que todas las transacciones de SymStore se registran en archivos de texto con formato CSV, puede aprovechar los archivos de registro de SymStore existentes para su uso en su propio programa de base de datos.

Si tiene previsto usar el programa SymSrv que se proporciona con las herramientas de depuración para Windows, se recomienda usar también SymStore. Las actualizaciones de estos dos programas siempre se publicarán juntas y, por lo tanto, sus versiones siempre coincidirán.
