# <a name="access-rights-for-active-directory-domain-services-objects"></a>Derechos de acceso para Active Directory Domain Services objetos

Todos los objetos de directorio usan el mismo conjunto de derechos de acceso predefinidos que corresponden a las operaciones de directorio comunes. Estos derechos se describen en el tipo [**\_ de enumeración ADS RIGHTS \_ ENUM.**](/windows/win32/api/iads/ne-iads-ads_rights_enum)

Además, puede definir derechos de acceso privado (derechos de acceso de control) para las operaciones que van más allá de las cubiertas por los derechos predefinidos. Consulte [Control de los derechos de acceso.](control-access-rights.md)
