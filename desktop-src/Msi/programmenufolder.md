# <a name="programmenufolder-property"></a>Propiedad ProgramMenuFolder

El instalador establece la **propiedad ProgramMenuFolder** en la ruta de acceso completa de la carpeta Menú del programa para el usuario actual. Si existe un perfil "Todos los usuarios" y se establece la propiedad [**ALLUSERS,**](allusers.md) esta propiedad se establece en la carpeta del perfil "Todos los usuarios".
