# <a name="the-output-page"></a>La página de salida

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

En esta página se incluyen las propiedades que controlan la salida de caracteres. Por ejemplo, el usuario puede determinar si se va a mostrar la salida en el globo de palabras, determinar cómo debe aparecer la salida del globo, reproducir el resultado hablado como audio, reproducir efectos sonoros de caracteres, mostrar el mensaje de reinicio y ajustar la velocidad de habla.
