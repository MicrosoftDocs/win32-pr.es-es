# <a name="msi-script-path-attribute"></a>Atributo MSI-script-path

Ruta de acceso del archivo de script de anuncios de Microsoft Installer para esta aplicación.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MSI-script-path                             |
| Nombre para mostrar de LDAP | msiScriptPath                               |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.15                       |
| System-ID-GUID    | bf967937-0de6-11d0-a285-00aa003049e2        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
