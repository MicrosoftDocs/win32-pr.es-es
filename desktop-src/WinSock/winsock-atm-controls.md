# <a name="winsock-atm-controls"></a>Controles ATM de Winsock

La especificación de Windows Sockets 2 es compatible de forma nativa con la configuración y el desmontaje de conexiones punto a punto y multipunto de ATM. De hecho, la especificación de QoS de Windows Sockets 2 y los mecanismos de multidifusión y multidifusión independientes del Protocolo se diseñaron pensando en ATM, junto con otros protocolos. Vea la sección 2,7 y el Apéndice D de la especificación de la API de Windows Sockets 2 para Windows Sockets 2, calidad de servicio y compatibilidad con Multipoint, respectivamente. Por lo tanto, no es necesario introducir ningún ioctl específico de ATM en este documento.
