# <a name="mfpkey_qpperframe-property"></a>\_Propiedad QPPERFRAME de MFPKEY

Especifica QP. Los valores posibles son de 1,0 a 31,0.

## <a name="constant-for-ipropertybag"></a>Constante para IPropertyBag

Solo está disponible mediante [**IPropertyStore**](/windows/win32/api/propsys/nn-propsys-ipropertystore).

## <a name="data-type"></a>Tipo de datos

**VT \_ R8**

## <a name="default-value"></a>Valor predeterminado

2.0

## <a name="remarks"></a>Observaciones

A medida que aumenta el valor de esta propiedad, aumenta la razón de compresión y disminuye la calidad.
