# <a name="importing-and-exporting-drm-protected-content"></a>Importación y exportación de contenido protegido con DRM

Uno de los roles principales de las nuevas API extendidas de cliente DRM de Windows Media es habilitar la importación y exportación de contenido protegido con DRM a sistemas de administración de derechos de terceros.

El principal desafío de importar y exportar datos protegidos en la aplicación es compartir sólidamente ejemplos de datos de ASF entre el sistema y el subsistema DRM. Cuando la aplicación controla ejemplos de datos, debe asegurarse de que se cumplen ciertos estándares de solidez para evitar la posibilidad de que se eluda el mecanismo de protección. Estos estándares se describen en las reglas de solidez y cumplimiento que acepta al licenciar DRM.
