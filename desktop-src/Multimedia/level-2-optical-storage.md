# <a name="level-2-optical-storage"></a>Almacenamiento óptico de nivel 2

El dispositivo de almacenamiento óptico de rendimiento mínimo es una unidad de CD-ROM de doble velocidad que cumple los siguientes criterios:

-   Una velocidad de transferencia de datos sostenida de 300 kilobytes por segundo
-   Un uso de ancho de banda de CPU del 40 por ciento o menos al mantener una velocidad de transferencia de datos sostenida de 150 kilobytes por segundo, o un ancho de banda de CPU del 60 por ciento o menos al mantener una velocidad de transferencia de datos sostenida de 300 kilobytes por segundo
-   Un tiempo medio de búsqueda de 400 milisegundos o menos
-   Una clasificación de tiempo medio entre errores de 10.000 hora
-   CD-ROM XA Ready (modo 1 compatible, modo 2 formulario 1 compatible, modo de modo 2 compatible)
-   Compatible con múltiples sesiones
-   Controlador MSCDEX-2,2 o equivalente que implementa las funciones de audio extendidas

Se debe alcanzar el ancho de banda de CPU recomendado mediante el uso de un tamaño de bloque de lectura de al menos 16 k y un tiempo de entrega superior al tiempo necesario para cargar un bloque de datos de lectura en el búfer de CD-ROM.
