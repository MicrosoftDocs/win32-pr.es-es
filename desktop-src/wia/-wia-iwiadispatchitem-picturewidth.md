# <a name="itempicturewidth-property"></a>Propiedad Item. PictureWidth

Ancho, en píxeles, de las imágenes generadas por esta cámara digital. Solo se aplica a cámaras digitales. Solo lectura.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Item.PictureWidth
```



## <a name="property-value"></a>Valor de propiedad

Variable que recibe el ancho.
