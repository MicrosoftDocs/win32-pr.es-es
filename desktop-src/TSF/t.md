# <a name="t-text-services-framework"></a>T (marco de trabajo de servicios de texto)

<dl> <dt>

<span id="_tsf_text_service_gly"></span><span id="_TSF_TEXT_SERVICE_GLY"></span>**servicio de texto**
</dt> <dd>

Tipo de aplicación que habilita la entrada de texto mediante voz, teclado multilingüe o dispositivo de lápiz. Un servicio de texto también puede habilitar herramientas de edición de texto como la revisión ortográfica y gramatical.

Vea también *procesador de texto*.

</dd> <dt>

<span id="_tsf_text_services_framework_gly"></span><span id="_TSF_TEXT_SERVICES_FRAMEWORK_GLY"></span>**Marco de trabajo de servicios de texto (TSF)**
</dt> <dd>

Un sistema autónomo independiente del dispositivo, independiente del lenguaje y extensible para la entrada de texto y tecnologías de lenguaje natural.

</dd> </dl>
