# <a name="direct3d-11-graphics"></a>Gráficos de Direct3D 11

Puede usar gráficos de Microsoft Direct3D 11 para crear gráficos 3D para juegos y aplicaciones científicas y de escritorio.

Esta sección contiene información sobre la programación con gráficos Direct3D 11.

Para obtener más información, consulte [características de Direct3D 11](direct3d-11-features.md).

Además de que Direct3D 11 sea compatible con Windows 7 y versiones posteriores, y con Windows Server 2008 R2 y versiones posteriores, Direct3D 11 está disponible para Windows Vista con Service Pack 2 (SP2) y Windows Server 2008 mediante la descarga de [kb 971644](https://support.microsoft.com/kb/971644) y [KB 971512](https://support.microsoft.com/kb/971512/).

Para obtener información sobre las nuevas características de Direct3D 11,1 que se incluyen con Windows 8, Windows Server 2012 y están parcialmente disponibles en Windows 7 y Windows Server 2008 R2 a través de la [actualización de la plataforma para Windows 7](https://support.microsoft.com/kb/2670838), consulte [características de Direct3D 11,1](direct3d-11-1-features.md) y la [actualización de la plataforma para Windows 7](/windows/desktop/direct3darticles/platform-update-for-windows-7).
