# <a name="rpc_ep_inq_handle"></a>\_ \_ controlador INQ de RPC EP \_

El tipo de datos de **\_ identificador de \_ INQ \_ de RPC EP** declara un identificador para un contexto de consulta. Las aplicaciones RPC lo usan para ver la información de dirección del servidor almacenada en la asignación de extremo.


```C++
typedef I_RPC_HANDLE* RPC_EP_INQ_HANDLE;
```
