# <a name="directxmath-3d-vector-transformation-functions"></a>Funciones de transformación de vector 3D de DirectXMath

Muestra las funciones de transformación vector 3D.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                               | Descripción                                                                                                                                      |
|-------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| [**XMVector3InverseRotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3inverserotate)<br/>                 | Gira un vector 3D con el inverso de un cuaternión.<br/>                                                                                |
| [**XMVector3Project**](/windows/win32/api/directxmath/nf-directxmath-xmvector3project)<br/>                             | Proyectar un vector 3D desde el espacio de objeto en el espacio de pantalla.<br/>                                                                              |
| [**XMVector3ProjectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3projectstream)<br/>                 | Proyecta un flujo de vectores 3D del espacio de objeto en el espacio de la pantalla.<br/>                                                                  |
| [**XMVector3Rotate**](/windows/win32/api/directxmath/nf-directxmath-xmvector3rotate)<br/>                               | Gira un vector 3D con un cuaternión.<br/>                                                                                               |
| [**XMVector3Transform**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transform)<br/>                         | Transforma un vector 3D mediante una matriz.<br/>                                                                                                   |
| [**XMVector3TransformCoord**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoord)<br/>               | Transforma un vector 3D por una matriz determinada y proyecta el resultado de nuevo en w = 1.<br/>                                                      |
| [**XMVector3TransformCoordStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformcoordstream)<br/>   | Transforma un flujo de vectores 3D por una matriz determinada, proyectando los vectores resultantes de forma que sus coordenadas w sean iguales a 1,0.<br/> |
| [**XMVector3TransformNormal**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormal)<br/>             | Transforma el vector 3D normal de la matriz especificada.<br/>                                                                                  |
| [**XMVector3TransformNormalStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformnormalstream)<br/> | Transforma una secuencia de vectores normales 3D por una matriz determinada.<br/>                                                                           |
| [**XMVector3TransformStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3transformstream)<br/>             | Transforma una secuencia de vectores 3D por una matriz determinada.<br/>                                                                                  |
| [**XMVector3Unproject**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unproject)<br/>                         | Proyecta un vector 3D a partir del espacio de la pantalla en el espacio de objeto.<br/>                                                                             |
| [**XMVector3UnprojectStream**](/windows/win32/api/directxmath/nf-directxmath-xmvector3unprojectstream)<br/>             | Transforma un flujo de vectores 3D del espacio de pantalla al espacio de objeto.<br/>                                                                  |
