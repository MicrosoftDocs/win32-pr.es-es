# <a name="certificate-enrollment-api-reference"></a>Referencia de API de inscripción de certificados

Las páginas de referencia de la API de inscripción de certificados contienen descripciones detalladas de las interfaces, las enumeraciones y otros elementos de programación que se pueden usar para inscribir un usuario o un equipo en una jerarquía de certificados. Las páginas de referencia se dividen en los siguientes grupos:

-   [Interfaces CertEnroll](certenroll-interfaces.md)
-   [Enumeraciones de CertEnroll](certenroll-enumerations.md)
