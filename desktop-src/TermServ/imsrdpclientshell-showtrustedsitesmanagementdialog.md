# <a name="imsrdpclientshellshowtrustedsitesmanagementdialog-method"></a>IMsRdpClientShell:: ShowTrustedSitesManagementDialog (método)

Muestra el cuadro de diálogo lista de sitios de confianza.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT ShowTrustedSitesManagementDialog();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Si este método se ejecuta correctamente, devuelve **S \_ correcto**. De lo contrario, devuelve un código de error **HRESULT** .
