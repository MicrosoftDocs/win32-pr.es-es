# <a name="processor-classes"></a>Clases de procesador

Las clases de procesador modelan un procesador para una máquina virtual.

A continuación se enumeran las clases WMI de virtualización relacionadas con el procesador.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                      | Descripción                                                                               |
|----------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**Procesador de MSVM \_**](msvm-processor.md)<br/>                       | Representa el procesador virtual en una máquina virtual.<br/>                         |
| [**MSVM \_ ProcessorPool**](msvm-processorpool.md)<br/>               | Agrega los recursos del procesador que se pueden asignar a una máquina virtual.<br/> |
| [**MSVM \_ ProcessorSettingData**](msvm-processorsettingdata.md)<br/> | Representa la configuración del procesador virtual para una máquina virtual.<br/>               |
