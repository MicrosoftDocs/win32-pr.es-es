# <a name="file-log-functions"></a>Funciones de registro de archivos

Las siguientes funciones se usan con registros de archivos.



| Función                                                   | Descripción                                 |
|------------------------------------------------------------|---------------------------------------------|
| [**SetupInitializeFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupinitializefileloga)   | Inicializa un archivo de registro para su uso.             |
| [**SetupLogFile**](/windows/desktop/api/Setupapi/nf-setupapi-setuplogfilea)                       | Agrega una entrada al archivo de registro.              |
| [**SetupQueryFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupqueryfileloga)             | Recupera información de un archivo de registro.      |
| [**SetupRemoveFileLogEntry**](/windows/desktop/api/Setupapi/nf-setupapi-setupremovefilelogentrya) | Quita una entrada de un archivo de registro.           |
| [**SetupTerminateFileLog**](/windows/desktop/api/Setupapi/nf-setupapi-setupterminatefilelog)     | Libera los recursos asignados a un archivo de registro. |
