# <a name="enabled-property-speech-input-object"></a>Propiedad Enabled (objeto de entrada de voz)

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descripción**
</dt> <dd>

Devuelve un valor booleano que indica si la entrada de voz está habilitada.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxis**
</dt> <dd>

*agent,. SpeechInput.Enabled**



| Value     | Descripción               |
|-----------|---------------------------|
| **True**  | La entrada de voz está habilitada.  |
| **False** | La entrada de voz está deshabilitada. |



 

</dd> </dl>

## <a name="remarks"></a>Comentarios

La [**propiedad Habilitado**](enabled-property.md) refleja la opción Escuchar entrada de caracteres en la página Entrada de voz de la hoja de propiedades del Agente (Opciones avanzadas de caracteres). El valor de propiedad afecta a todos los caracteres del Agente y es de solo lectura; solo el usuario puede cambiar esta propiedad.
