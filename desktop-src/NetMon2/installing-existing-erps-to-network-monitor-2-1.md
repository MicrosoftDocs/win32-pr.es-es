# <a name="installing-existing-erps-to-network-monitor-21"></a>Instalación de O ERP e existentes en Monitor de red 2,1

Después de escribir y asignar un nombre a una página de referencia de eventos (ERP), debe instalarla.

**Para instalar una página de referencia de eventos**

1.  Cree un directorio ERP. En estos ejemplos se muestran directorios para expertos:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Expert\
    ```

2.  Cree un subdirectorio para cada nuevo experto que mostrará el O ERP e. Las siguientes listas de subdirectorios son para un experto:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert
    ```

3.  Copie los archivos ERP en el subdirectorio asociado. En los siguientes ejemplos se proporcionan rutas de acceso completas a Expert O ERP e instalado:

    ``` syntax
    C:\Program Files\NetMon2\Bok\Experts\MyExpert\MyExpert1.htm
    ```
