# <a name="systemvideohorizontalaspectratio"></a>System. video. HorizontalAspectRatio

Indica la parte horizontal de la relación de aspecto de píxeles. La parte X de XX: YY. Por ejemplo, 10 es la parte X de 10:11.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Video.HorizontalAspectRatio
   shellPKey = PKEY_Video_HorizontalAspectRatio
   formatID = 64440491-4C8B-11D1-8B70-080036B11A03
   propID = 42
   SearchInfo
      InInvertedIndex = false
      IsColumn = true
   typeInfo
      type = UInt32
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
