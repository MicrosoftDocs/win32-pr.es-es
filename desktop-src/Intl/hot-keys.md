# <a name="hot-keys"></a>Hot Keys

Las teclas de acceso rápido proporcionan al usuario una manera de cambiar rápidamente el modo de entrada de un IME o cambiar a otro IME. Aunque las aplicaciones no pueden agregar teclas de acceso rápido al sistema operativo, pueden iniciar la misma acción que una tecla de acceso rápido mediante la función [**ImmSimulateHotKey**](/windows/desktop/api/Imm/nf-imm-immsimulatehotkey) .

El IME de HexToUnicode también permite la conversión entre caracteres hexadecimales y Unicode. Para obtener más información, consulte [HEXTOUNICODE IME](hextounicode-ime.md).
