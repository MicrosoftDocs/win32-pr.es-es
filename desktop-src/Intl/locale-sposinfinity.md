# <a name="locale_sposinfinity"></a>configuración regional \_ SPOSINFINITY

**Windows Vista y versiones posteriores:** Valor de cadena para "infinito positivo", por ejemplo, "infinito" para la configuración regional en inglés (Estados Unidos). No hay ningún límite en el número de caracteres permitidos para la cadena.
