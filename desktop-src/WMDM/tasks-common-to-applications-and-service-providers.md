# <a name="tasks-common-to-applications-and-service-providers"></a>Tareas comunes a aplicaciones y proveedores de servicios

En esta sección se describen las siguientes tareas que se aplican tanto a las aplicaciones como a los proveedores de servicios basados en Windows Media Administrador de dispositivos.

-   [Al compilar los](compiling-the-idl-files-supplied-with-the-sdk.md) archivos IDL proporcionados con el SDK, se describen los archivos IDL que se usan para crear los archivos de encabezado, si decide no usar los archivos de encabezado proporcionados con el SDK.
-   [Al habilitar el registro](enabling-logging.md) se describe un objeto de registro proporcionado por Windows Media Administrador de dispositivos para registrar datos de un objeto en ejecución.
-   [El uso de canales autenticados seguros](using-secure-authenticated-channels.md) describe los canales autenticados seguros, que se usan para el cifrado, el descifrado y la autenticación de componentes y mensajes. Todo lo que se Windows de Administrador de dispositivos al menos debe poder realizar la autenticación de componentes.
-   [Control del contenido](handling-protected-content.md) protegido describe cómo la aplicación puede controlar el contenido protegido por Windows Administración de derechos digitales multimedia (DRM).
-   [Obtener y establecer metadatos y atributos](getting-and-setting-metadata-and-attributes.md) describe cómo una aplicación puede recuperar y establecer atributos y metadatos para un almacenamiento o dispositivo.
