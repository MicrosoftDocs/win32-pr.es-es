# <a name="taskserviceconnecteduser-property"></a>Propiedad TaskService. ConnectedUser

En el caso de scripting, obtiene el nombre del usuario que está conectado al servicio Programador de tareas.

## <a name="syntax"></a>Sintaxis


```VB
TaskService.ConnectedUser As String
```



## <a name="property-value"></a>Valor de propiedad

Nombre del usuario que está conectado al servicio de Programador de tareas.
