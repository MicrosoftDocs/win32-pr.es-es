# <a name="version9x-property"></a>Propiedad Version9X

La propiedad **Version9X** proporciona el número de versión de las versiones de 9x de los sistemas operativos Windows.

El valor de esta propiedad es un entero: MajorVersion \* 100 + MinorVersion. La propiedad permanece sin definir si el sistema operativo no es una versión de 9x. Para obtener más información, vea [valores de propiedad del sistema operativo](operating-system-property-values.md).

## <a name="remarks"></a>Observaciones

Las expresiones de condición pueden probar la versión con el nombre de la propiedad o puede comprobar la versión mediante un operador de comparación.

Todos los nombres de propiedad distinguen mayúsculas de minúsculas. Tenga en cuenta que la X del nombre **Version9X** está en mayúsculas.
