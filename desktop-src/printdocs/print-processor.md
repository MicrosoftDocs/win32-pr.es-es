# <a name="print-processor"></a>Procesador de impresión

El colador de impresión supervisa los trabajos de impresión actuales y la impresora de destino para determinar la hora adecuada para imprimir un trabajo. Una vez que el colador determina que se debe imprimir un trabajo, llama al procesador de impresión. El procesador de impresión es un complemento que procesa los datos del trabajo de impresión.

Use las siguientes funciones para trabajar con procesadores de impresión.



| Función                                                           | Descripción                                                          |
|--------------------------------------------------------------------|----------------------------------------------------------------------|
| [**AddPrintProcessor**](addprintprocessor.md)                     | Instala un procesador de impresión en un servidor especificado.                    |
| [**DeletePrintProcessor**](deleteprintprocessor.md)               | Quita un procesador de impresora de un servidor especificado.                 |
| [**EnumPrintProcessorDatatypes**](enumprintprocessordatatypes.md) | Enumera los tipos de datos que admite un procesador de impresión especificado. |
| [**EnumPrintProcessors**](enumprintprocessors.md)                 | Enumera los procesadores de impresión instalados en un servidor especificado.     |
| [**GetPrintProcessorDirectory**](getprintprocessordirectory.md)   | Recupera la ruta de acceso del procesador de impresión en el servidor especificado.  |
