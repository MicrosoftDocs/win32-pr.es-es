# <a name="cgenericlistgetheadposition-method"></a>CGenericList. GetHeadPosition, método

El `GetHeadPosition` método recupera la posición del primer elemento de la lista.

## <a name="syntax"></a>Sintaxis


```C++
POSITION GetHeadPosition();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve un valor de posición o **null** si la lista está vacía.
