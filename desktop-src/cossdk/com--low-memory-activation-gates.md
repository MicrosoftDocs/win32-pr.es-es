# <a name="com-low-memory-activation-gates"></a>Puertas de activación de Low-Memory COM+

El servicio de las puertas de activación de memoria insuficiente de COM+ comprueba automáticamente la memoria antes de que se cree un servidor o un objeto COM+. Si el porcentaje de memoria virtual disponible para la aplicación cae por debajo de un umbral fijo, se produce un error en la activación antes de que se cree el objeto. De esta manera, el servicio de puertas de activación de memoria insuficiente mejora en gran medida la confiabilidad del sistema.

En los temas siguientes se proporciona información básica y de tareas sobre el servicio de puertas de activación de memoria insuficiente de COM+.
