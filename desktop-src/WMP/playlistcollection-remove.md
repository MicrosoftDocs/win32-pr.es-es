# <a name="playlistcollectionremove-method"></a>PlaylistCollection. Remove (método)

El método **Remove** quita una lista de reproducción de la biblioteca.

## <a name="syntax"></a>Sintaxis


```JScript
PlaylistCollection.remove(
  playlist
)
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*lista de reproducción* \[ de\]
</dt> <dd>

Objeto de la **lista de reproducción** que se va a quitar.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.

## <a name="remarks"></a>Observaciones

Para usar este método, se necesita acceso completo a la biblioteca. Para obtener más información, vea [acceso a la biblioteca](library-access.md).
