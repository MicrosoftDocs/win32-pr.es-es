# <a name="strict-type-checking"></a>Comprobación estricta de tipos

El archivo de encabezado Windows. h contiene definiciones, macros y estructuras que le ayudarán a escribir código fuente que sea portátil entre versiones de Windows. Algunas de estas características se habilitan al definir el símbolo STRICT al compilar la aplicación. En los temas siguientes se explican las ventajas de las características ESTRICTAs y el uso que afectan a la escritura de código.

-   [Habilitar STRICT](enabling-strict.md)
-   [Deshabilitar STRICT](disabling-strict.md)
-   [Cumplimiento estricto](strict-compliance.md)
