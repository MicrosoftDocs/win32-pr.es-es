# <a name="flow-specification-quality-of-service"></a>Flow Calidad de servicio de especificación

La calidad de servicio se implementa en Windows varios componentes de QoS del sistema. Para obtener detalles completos e instrucciones de implementación, vea [Quality of Service](/previous-versions/windows/desktop/qos/qos-start-page).

Para más información sobre las plantillas de QoS, consulte [Plantillas de QoS.](/previous-versions/aa374099(v=vs.80))

Para más información e instrucciones de implementación sobre la calidad de servicio, consulte la [**estructura FLOWSPEC.**](/windows/win32/api/qos/ns-qos-flowspec)
