# <a name="d1183-createwicbitmaprendertarget-with-incompatible-feature-level"></a>D1183: CreateWicBitmapRenderTarget con el nivel de característica incompatible

No es válido pasar un nivel de característica de Direct2D distinto del \_ valor predeterminado de nivel de característica D2D1 \_ \_ a CreateWicBitmapRenderTarget.






 

## <a name="possible-causes"></a>Causas posibles

Cuando una aplicación llamada ID2D1Factory:: CreateWicBitmapRenderTarget, el campo minLevel del \_ argumento propiedades de destino de representación D2D1 \_ \_ no era D2D1 de \_ nivel de característica \_ \_ predeterminado.

## <a name="possible-fixes"></a>Posibles correcciones

Compruebe que el nivel de característica que se usa al crear un destino de representación de WIC-interopped es DEFAULT. La interoperabilidad con WIC implica características de solo software; la semántica de nivel de característica es relevante para el comportamiento del hardware.
