# <a name="ms-pki-ra-signature-attribute"></a>atributo MS-PKI-RA-Signature

Especifica el número de firmas de autoridad de registro de inscripción que se requieren en una solicitud de inscripción.



| Entrada | Value |
|-------------------|---------------------------------------------------------------------------------------------------|
| CN                | MS-PKI-RA-Signature                                                                               |
| Nombre para mostrar de LDAP | Atributo mspki-RA-Signature                                                                                |
| Tamaño              | 4 bytes                                                                                           |
| Actualizar privilegio  | Administrador de dominio                                                                              |
| Frecuencia de actualización  | Cuando se edita, crea o clona el objeto plantilla de certificado (MS-PKI-Certificate-template). |
| Attribute-Id      | 1.2.840.113556.1.4.1429                                                                           |
| System-ID-GUID    | fe17e04b-937d-4f7e-8e0e-9292c8d5683e                                                              |
| Sintaxis            | [**Enumeración**](s-enumeration.md)                                                              |
