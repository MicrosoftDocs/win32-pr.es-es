# <a name="ms-sql-informationdirectory-attribute"></a>Atributo MS-SQL-InformationDirectory

Directorio informativo de la instancia actual de SQL Server.



| Entrada | Value |
|-------------------|------------------------------------------|
| CN                | MS-SQL-InformationDirectory              |
| Nombre para mostrar de LDAP | mS-SQL-InformationDirectory              |
| Tamaño              | \-                                       |
| Actualizar privilegio  | Administrador de dominio                     |
| Frecuencia de actualización  | Al iniciar el sistema o al actualizar AD. |
| Attribute-Id      | 1.2.840.113556.1.4.1392                  |
| System-ID-GUID    | d0aedb2e-ccee-11d2-9993-0000f87a57d4     |
| Sintaxis            | [**Booleano**](s-boolean.md)             |
