# <a name="displaying-web-pages-in-windows-media-player"></a>Mostrar páginas web en Windows Media Player

La característica HTMLView de Windows Media Player permite a los proveedores de contenido de Internet (ICP) usar la característica de **reproducción en curso** del reproductor en modo completo para mostrar contenido basado en web mientras se reproduce el contenido multimedia digital. (HTMLView no es compatible con el control Media Player de Windows). En el pasado, ICP tenía que mostrar anuncios en ventanas de explorador web independientes (denominadas "publicidad emergente") o tenía que agregar reproductores multimedia digitales personalizados a sus páginas Web. Con HTMLView, ICP puede proporcionar una experiencia personalizada e integrada creada específicamente para Windows Media Player 9 series y Windows Media Player 10, a la vez que garantiza que el contenido multimedia digital se vuelve a reproducir en versiones anteriores de Windows Media Player.

En esta sección se proporciona información acerca del uso de HTMLView, la creación de contenido de HTMLView y la resolución de posibles problemas que pueden surgir al usar la característica HTMLView en los temas siguientes:

-   [Ventajas del uso de HTMLView](advantages-of-using-htmlview.md)
-   [Crear una presentación de HTMLView](creating-an-htmlview-presentation.md)
-   [Agregar un control de Media Player de Windows incrustado](adding-an-embedded-windows-media-player-control.md)
-   [Asegurarse de que Windows Media Player abre el contenido de HTMLView](ensuring-that-windows-media-player-opens-the-htmlview-content.md)
-   [Problemas de páginas web](web-page-issues.md)
