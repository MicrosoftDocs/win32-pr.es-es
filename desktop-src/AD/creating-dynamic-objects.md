# <a name="creating-dynamic-objects"></a>Crear objetos dinámicos

Especifique la clase auxiliar **dynamicObject** como valor para el atributo **objectClass** en el momento de crear una instancia de un objeto. En este caso, la clase auxiliar especial se mostrará en la lista de clases que es el valor del atributo **objectClass** del objeto. Si no se proporciona un valor para el atributo **entryTTL** en el momento de agregar la clase auxiliar especial, Active Directory Domain Services proporcionará un valor predeterminado, tal y como se especificó más adelante.

Para obtener más información sobre cómo agregar una clase auxiliar a un objeto, vea [clases auxiliares dinámicas](dynamic-auxiliary-classes.md).
