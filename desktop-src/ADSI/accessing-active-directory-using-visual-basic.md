# <a name="accessing-active-directory-using-visual-basic"></a>Obtener acceso a Active Directory mediante Visual Basic

Una de las características más eficaces que estaban disponibles con el sistema operativo Microsoft Windows 2000 es el servicio de directorio de Microsoft Active Directory. Cuando se inicia sesión en un dominio de Windows 2000, Active Directory se pone en acción; para buscar la impresora de color más cercana del edificio, puede usar Active Directory. Se puede usar para una búsqueda en la libreta de direcciones o para buscar usuarios que trabajan en la compilación 40. Con Active Directory, puede usar una tarjeta inteligente para iniciar sesión en un dominio de Windows 2000. Puede incluso combinar datos de Microsoft SQL Server software y Active Directory de base de datos.

Muchas otras tecnologías de Microsoft, como Microsoft Message Queue Server (MSMQ), COM (almacén de clases), el protocolo de seguridad de Internet (IPsec), los objetos de directiva de grupo (GPO) y Exchange se integran con Active Directory.

En esta sección se describe cómo usar las interfaces de servicio de Active Directory (ADSI) para tener acceso a Active Directory. El uso de un escenario para una empresa ficticia, Fabrikam Corporation, aprenderá a realizar algunas tareas administrativas básicas.

A medida que avanza por este escenario, los ejemplos de código de cada sección se basan en ellos mismos.

Por motivos de brevedad, todos los ejemplos de código se escriben con el sistema de desarrollo de Microsoft Visual Basic. Para obtener más información sobre la conversión de C++, vea [asignar código ADSI Visual Basic a código de c++](mapping-adsi-visual-basic-code-to-c-code.md).
