# <a name="istringcollection-methods"></a>Métodos IStringCollection

La [**interfaz IStringCollection**](/windows/desktop/api/Wuapi/nn-wuapi-istringcollection) define los métodos siguientes.



| Método                                         | Descripción                                                    |
|------------------------------------------------|----------------------------------------------------------------|
| [**Añadir**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-add)           | Agrega un elemento a la colección.                                |
| [**Borrar**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-clear)       | Quita todos los elementos de la colección.                  |
| [**Copiar**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-copy)         | Crea una copia de lectura y escritura en profundidad de la colección.              |
| [**insertar**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-insert)     | Inserta un elemento en la colección situada en la posición especificada. |
| [**RemoveAt**](/windows/desktop/api/Wuapi/nf-wuapi-istringcollection-removeat) | Quita de la colección un elemento situado en el índice especificado.   |
