# <a name="metadata-area-plug-ins-deprecated"></a>Complementos de área de metadatos (desusado)

En esta página se documenta una característica que puede no estar disponible en versiones futuras de Windows Media Player y el SDK de Windows Media Player.

Los complementos de la interfaz de usuario del área de metadatos se muestran en el área de metadatos del panel **reproducción en curso** a la derecha del área visual principal y del panel herramientas, y debajo de la vista de lista de reproducción (si está habilitada). Solo se puede habilitar un complemento de área de metadatos a la vez.

Los complementos de área de metadatos son útiles para mostrar información concisa sobre la reproducción de elementos multimedia, controles simples o hipervínculos a información adicional.
