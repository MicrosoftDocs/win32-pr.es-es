# <a name="version-windows-installer"></a>Versión (Windows Installer)

El tipo de datos version es una cadena de texto que contiene una cadena de versión válida. Una cadena de versión tiene el formato

``` syntax
xxxxx.xxxxx.xxxxx.xxxxx
```

donde *x* es un dígito.

La cadena de versión máxima aceptable es 65535.65535.65535.65535.

A continuación se muestran ejemplos de cadenas de versión válidas:

-   1
-   1,0
-   1.00
-   10,00
-   1.00.1
-   1.0.1
-   1.00.10
-   1.00.100
-   1.0.1000.0
