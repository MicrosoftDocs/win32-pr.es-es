# <a name="namespacedefinitions-element"></a>elemento namespaceDefinitions

Genera definiciones de C para las tablas de espacios de nombres.

## <a name="usage"></a>Uso

``` syntax
<namespaceDefinitions/>
```

## <a name="attributes"></a>Atributos

No hay atributos.

## <a name="child-elements"></a>Elementos secundarios

No hay elementos secundarios.

## <a name="parent-elements"></a>Elementos primarios



| Elemento                         | Descripción                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**archivo**](file.md)<br/> | Genera un archivo desde el generador de código.<br/> <br/> |



## <a name="remarks"></a>Observaciones

Este elemento se utiliza generalmente en archivos de código fuente de C para proporcionar las tablas de espacio de nombres que se declararon mediante [**namespaceDeclarations**](namespacedeclarations.md).
