# <a name="xmu555-operators"></a>Operadores XMU555

La estructura [**XMU555**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmu555) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                           | Descripción                                                                                                          |
|-------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------|
| [**Operator UInt16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmu555-operatoruint16_t)<br/> | Devuelve una instancia de `uint16_t` que contiene los componentes de la `XMU555` instancia en un formato empaquetado. <br/> |
| [**operador =**](xmu555-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMU555` . <br/>                                       |
