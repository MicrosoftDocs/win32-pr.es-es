# <a name="installing-the-windows-region-change-support-components"></a>Instalar los componentes de compatibilidad de cambio de la región de Windows

Los desarrolladores de aplicaciones pueden utilizar la compatibilidad con el cambio de la región de DVD integrada en Windows para cambiar la región dentro de las aplicaciones de reproducción de DVD.

En Windows 2000 y versiones posteriores, el componente que proporciona la compatibilidad con el cambio de la región de DVD está en el storprop.dll DLL (disponible en% WINDIR% \\ system32). Este archivo DLL se instala de forma predeterminada y no se requieren pasos adicionales para asegurarse de que este archivo DLL está disponible.
