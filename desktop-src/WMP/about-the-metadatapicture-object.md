# <a name="about-the-metadatapicture-object"></a>Acerca del objeto MetadataPicture

El atributo de metadatos de **WM/imagen** proporciona información detallada sobre las imágenes de metadatos, como carátulas de álbum. El objeto **MetadataPicture** proporciona una manera de recuperar los datos individuales relacionados con la imagen. Puede pensar en el atributo de metadatos de **WM/imagen** como un atributo compuesto. Cuando se utiliza un *medio*. **getItemInfoByType** para recuperar información acerca de una imagen de metadatos, se recupera un objeto **MetadataPicture** que tiene propiedades que contienen los datos individuales.
