# <a name="statements"></a>Instrucciones

Las instrucciones de HLSL son combinaciones de variables y funciones que componen expresiones. al igual que una oración, una instrucción HLSL requiere operadores que determinen cómo se evaluará una expresión. Puede crear instrucciones individuales en un sombreador o crear varias instrucciones en un grupo denominado bloque de instrucciones. Para crear instrucciones de HLSL, deberá conocer la sintaxis de escritura de expresiones y los operadores que usará para combinar expresiones.

-   [Expresiones](dx-graphics-hlsl-expressions.md)
-   [Operadores](dx-graphics-hlsl-operators.md)
-   [Bloques de instrucciones](dx-graphics-hlsl-statement-blocks.md)
