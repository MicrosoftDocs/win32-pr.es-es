# <a name="domain-other-parameters-property-set"></a>Conjunto de propiedades Domain-Other-Parameters

Conjunto de propiedades que permite el control en una lista de atributos de dominio.



| Entrada | Valor |
|--------------|------------------------------------------|
| CN           | Domain-Other-Parameters                  |
| Display-Name | Otros parámetros de dominio (para su uso por SAM) |
| Rights-GUID  | b8119fd0-04f6-4762-ab7a-4986c76b3f9a     |
