# <a name="media-properties"></a>Propiedades de medios

Los dispositivos portátiles de Windows admiten las siguientes propiedades de medios.



| Propiedad                                                                                                                 | VarType        | Descripción                                                                                                                                                                                                                                                                                                    |
|--------------------------------------------------------------------------------------------------------------------------|----------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **\_intérprete del \_ álbum \_ multimedia de WPD**                                                                                            | **VT \_ LPWStr** | Cadena que identifica el artista que realiza el medio.                                                                                                                                                                                                                                                      |
| **\_Perfil de \_ codificación de audio multimedia \_ WPD \_**                                                                                 | VT \_ LPWStr     | Cadena que especifica un perfil de codificación. Los códecs multimedia se pueden codificar de acuerdo con un perfil, que define un algoritmo de codificación determinado o un proceso de optimización.                                                                                                                                  |
| **\_tipo de \_ velocidad de bits de medios de WPD \_**                                                                                            | **VT \_ UI4**    | Una enumeración de [**\_ \_ tipos de velocidad**](wpd-bitrate-types.md) de bits de WPD que califica la velocidad de bits de los datos de audio o de vídeo.                                                                                                                                                                                     |
| **\_compra multimedia \_ WPD \_ Now**                                                                                                 | **VT \_ bool**   | Valor booleano que especifica si el usuario desea comprar el contenido la próxima vez que se realiza una conexión a Internet.                                                                                                                                                                                         |
| **\_marcador de \_ bytes multimedia de WPD \_**                                                                                           | **VT \_ UI8**    | Especifica un marcador (un desplazamiento de bytes basado en cero) que corresponde a la última posición reproducida o visualizada en el medio determinado.                                                                                                                                                                                     |
| **\_compositor de medios de WPD \_**                                                                                                 | **VT \_ LPWStr** | Compositor, si el compositor y el rendimiento son diferentes.                                                                                                                                                                                                                                                     |
| **COPYRIGHT de los medios de WPD \_ \_**                                                                                                | **VT \_ LPWStr** | Cadena que proporciona información de copyright sobre el objeto.                                                                                                                                                                                                                                                 |
| <span id="wpd_media_description"></span><span id="WPD_MEDIA_DESCRIPTION"></span>**\_Descripción de medios de WPD \_**              | **VT \_ LPWStr** | Descripción del objeto especificado. En los casos en los que la propiedad de [ \_ \_ \_ subdescripción](/windows) de los medios de WPD puede describir un solo elemento de contenido, esta propiedad describirá una colección de medios relacionados.<br/>                                                                                 |
| **\_URL de \_ destino de medios de WPD \_**                                                                                         | **VT \_ LPWStr** | Cadena terminada en un valor nulo que indica la dirección URL de un objeto. Esta dirección URL identifica el contenido que aparecerá si el usuario hace clic en el objeto. Los objetos **Mediacast** suelen usar esta propiedad para especificar la ubicación de la información relacionada.<br/>                                          |
| **duración de los medios de WPD \_ \_**                                                                                                 | **VT \_ UI8**    | La duración del contenido, en milisegundos.                                                                                                                                                                                                                                                                  |
| **\_ \_ clasificación efectiva de medios de WPD \_**                                                                                        | **VT \_ UI4**    | Número comprendido entre 1 y 100, ambos inclusive, que clasifica el contenido, donde 1 es el valor más bajo. Este valor se basa en las estadísticas de uso que el usuario no establece explícitamente.                                                                                                                                   |
| **\_Perfil de \_ codificación multimedia \_ WPD**                                                                                        | **VT \_ LPWStr** | Un nombre de perfil, tal y como lo define el códec determinado que se utiliza. Si el códec no requiere un nombre de perfil específico, en su lugar debe especificar la información de audio en las propiedades adecuadas.                                                                                                            |
| **\_género multimedia de WPD \_**                                                                                                    | **VT \_ LPWStr** | Cadena terminada en un valor nulo que indica el género al que pertenece este medio.                                                                                                                                                                                                                     |
| **\_GUID de medios de WPD \_**                                                                                                     | **VT \_ LPWStr** | Especifica un identificador único global (GUID) de 128 bits.                                                                                                                                                                                                                                                         |
| <span id="wpd_media_height"></span><span id="WPD_MEDIA_HEIGHT"></span>**altura de los medios de WPD \_ \_**                             | **VT \_ UI4**    | Valor que especifica el alto del recurso principal (normalmente, un vídeo o una imagen), en píxeles. Vea [el \_ \_ ancho del medio de WPD](/windows).                                                                                                                                                                    |
| **\_fecha de \_ última \_ compilación \_ de los medios de WPD**                                                                                        | **fecha de VT \_**   | Fecha que especifica la última vez que se modificó o editó una serie de un mediacast.                                                                                                                                                                                                                                 |
| **\_ \_ última hora de \_ acceso a los medios \_ de WPD**                                                                                     | **fecha de VT \_**   | Valor que especifica la fecha y la hora en que se produjo el último acceso al elemento multimedia en el dispositivo.                                                                                                                                                                                                                       |
| **\_Editor de \_ Administración de medios de WPD \_**                                                                                         | **VT \_ LPWStr** | Cadena terminada en un valor nulo que se puede leer y que indica la dirección de correo electrónico del editor de administración del medio. Los objetos **Mediacast** suelen usar esta propiedad para especificar el editor de administración de una serie o canal RSS.<br/>                                                                                 |
| **metagénero de multimedia de WPD \_ \_ \_**                                                                                              | **VT \_ UI4**    | Una enumeración de [**\_ \_ metagéneros de WPD**](wpd-meta-genres.md) que describe el tipo de medio.                                                                                                                                                                                                                   |
| **\_marcador de \_ objeto \_ multimedia WPD**                                                                                         | **VT \_ LPWStr** | Especifica el \_ identificador del objeto WPD \_ para el último objeto que se ha visualizado o reproducido. Esta propiedad se aplica a las listas de reproducción y mediacasts.<br/>                                                                                                                                                                               |
| **\_propietario de medios de WPD \_**                                                                                                    | **VT \_ LPWStr** | Cadena terminada en un valor nulo que se puede leer y que indica la dirección de correo electrónico del propietario del medio. Los objetos **Mediacast** suelen usar esta propiedad para especificar el propietario de una serie o canal RSS.<br/>                                                                                                     |
| **\_ \_ clasificación parental de medios de \_ WPD**                                                                                         | **VT \_ LPWStr** | Cadena terminada en un valor nulo que se puede leer y que indica la clasificación parental del archivo multimedia. Normalmente, se trata de una clasificación de la Asociación de fotografías de América (MPAA).                                                                                                                                        |
| **\_fecha de \_ lanzamiento de medios de WPD \_**                                                                                            | **fecha de VT \_**   | Fecha en que el contenido se liberó originalmente.                                                                                                                                                                                                                                                             |
| **\_velocidad de \_ muestra \_ multimedia de WPD**                                                                                             | **VT \_ UI4**    | La velocidad de muestra de codificación, en muestras por segundo, para el contenido.                                                                                                                                                                                                                                              |
| **recuento de omisiones de medios de WPD \_ \_ \_**                                                                                              | **VT \_ UI4**    | Número total de veces que este medio se puso en cola para la reproducción, pero el usuario lo omitió explícitamente. Se puede usar para calcular automáticamente una clasificación de usuario o modificar las reglas de actualización de contenido.                                                                                                                  |
| **\_URL de \_ origen de medios de WPD \_**                                                                                              | **VT \_ LPWStr** | Cadena terminada en un valor nulo que se puede leer y que indica la dirección URL de origen de los medios especificados. Los objetos **Mediacast** suelen usar esta propiedad para especificar dónde se originó el contenido.<br/>                                                                                                                   |
| **\_clasificación de media \_ Star \_ de WPD**                                                                                             | **VT \_ UI4**    | Número comprendido entre 0 y 5 que califica el contenido, donde 0 es sin puntuación, 1 es la clasificación más baja y 5 es la clasificación más alta. El usuario escribe explícitamente este valor.                                                                                                                                        |
| <span id="wpd_media_sub_description"></span><span id="WPD_MEDIA_SUB_DESCRIPTION"></span>**subdescripción de los medios de WPD \_ \_ \_** | **VT \_ LPWStr** | Una descripción subordinada para el objeto especificado. Esta propiedad se utiliza a menudo para proporcionar información adicional sobre el contenido. En el caso de que la propiedad [ \_ \_ Descripción](/windows) de los medios de WPD pueda describir una colección de medios relacionados, esta propiedad desscribirá un único fragmento de contenido.<br/> |
| <span id="wpd_media_sub_title"></span><span id="WPD_MEDIA_SUB_TITLE"></span>**subtítulo de medios de WPD \_ \_ \_**                   | **VT \_ LPWStr** | Un subtítulo para el objeto. Si el objeto es un episodio de una serie, sería el título del episodio y el [ \_ \_ título del medio de WPD](/windows) sería el nombre de la serie.                                                                                                                                 |
| **\_ \_ \_ ID. de contenido de suscripción de medios de WPD \_**                                                                                | **VT \_ LPWStr** | IDENTIFICADOR que se utiliza para identificar un elemento de contenido en un servicio de suscripción en línea. Normalmente, el cliente establece este valor en lugar del controlador o dispositivo.                                                                                                                                                    |
| **\_marcador de \_ tiempo de medio de WPD \_**                                                                                           | **VT \_ UI8**    | Especifica un marcador (en milisegundos) que corresponde a la última posición reproducida o visualizada en el medio determinado.                                                                                                                                                                                              |
| **\_ \_ tiempo \_ de vida de los medios de WPD \_**                                                                                           | **VT \_ UI8**    | Especifica el tiempo (en minutos) hasta la siguiente actualización de contenido.                                                                                                                                                                                                                                                 |
| <span id="wpd_media_title"></span><span id="WPD_MEDIA_TITLE"></span>**título de los medios de WPD \_ \_**                                | **VT \_ LPWStr** | Título del contenido. Si el objeto es un episodio de una serie, sería el título de la serie y el [ \_ \_ \_ subtítulo](/windows) de los medios de WPD sería el nombre del episodio.                                                                                                                               |
| **\_velocidad de \_ bits total de medios de WPD \_**                                                                                           | **VT \_ UI4**    | Velocidad de bits del objeto.                                                                                                                                                                                                                                                                                    |
| **recuento de uso de medios de WPD \_ \_ \_**                                                                                               | **VT \_ UI4**    | Número total de veces que este medio se ha reproducido o visualizado en el dispositivo.                                                                                                                                                                                                                                  |
| **\_ \_ \_ clasificación efectiva del usuario \_ de los medios de WPD**                                                                                  | **VT \_ UI4**    | Un número entre 1 y 100 que califica este contenido, siendo 1 el más bajo. El usuario establece este valor explícitamente. El valor predeterminado es 0, que indica que no se ha establecido este valor.                                                                                                                    |
| **webmaster de multimedia de WPD \_ \_**                                                                                                | **VT \_ LPWStr** | Cadena terminada en un valor nulo que se puede leer y que indica la dirección de correo electrónico del webmaster del medio. Los objetos **Mediacast** suelen usar esta propiedad para especificar el webmaster de una serie o canal RSS.<br/>                                                                                             |
| <span id="wpd_media_width"></span><span id="WPD_MEDIA_WIDTH"></span>**\_ancho de medios de WPD \_**                                | **VT \_ UI4**    | Valor que especifica el ancho del recurso principal (normalmente, un vídeo o una imagen), en píxeles. Consulte [ \_ \_ altura](/windows)de los medios de WPD.                                                                                                                                                                   |
| **\_intérprete multimedia de WPD \_**                                                                                                   | **VT \_ LPWStr** | Valor que identifica el intérprete multimedia.                                                                                                                                                                                                                                                                      |
