# <a name="iwmpclosedcaption-vb-and-c-interface"></a>Interfaz IWMPClosedCaption (VB y C#)

Proporciona una manera de incluir leyendas con un archivo multimedia digital. El texto de la leyenda se encuentra en un archivo de intercambio de medios accesibles (SAMI) sincronizado.

## <a name="members"></a>Miembros

La interfaz **IWMPClosedCaption (VB y C#)** no define ningún miembro.

La interfaz **IWMPClosedCaption** expone las siguientes propiedades.



| Propiedad                                                                             | Descripción                                                                                |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------|
| [captioningId](wmplibiwmpclosedcaption-iwmpclosedcaption-captioningid--vb-and-c.md) | Obtiene o establece el nombre del elemento HTML que muestra los subtítulos.                       |
| [SAMIFileName](wmplibiwmpclosedcaption-iwmpclosedcaption-samifilename--vb-and-c.md) | Obtiene o establece el nombre del archivo que contiene la información necesaria para los subtítulos (CC). |
| [SAMILang](wmplibiwmpclosedcaption-iwmpclosedcaption-samilang--vb-and-c.md)         | Obtiene o establece el idioma que se muestra para los subtítulos (CC).                                 |
| [SAMIStyle](wmplibiwmpclosedcaption-iwmpclosedcaption-samistyle--vb-and-c.md)       | Obtiene o establece el estilo de subtítulos (CC).                                                  |



 

Obtenga una interfaz **IWMPClosedCaption** con la siguiente propiedad.



| Object                                                            | Propiedad                                                                   |
|-------------------------------------------------------------------|----------------------------------------------------------------------------|
| [AxWindowsMediaPlayer](axwindowsmediaplayer-object--vb-and-c.md) | [closedCaption](axwmplib-axwindowsmediaplayer-closedcaption--vb-and-c.md) |
