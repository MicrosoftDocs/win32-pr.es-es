# <a name="next-hop"></a>Próximo salto

Un próximo salto es el siguiente enrutador en la ruta de acceso hacia un destino. Los paquetes de un origen se reenvía a un destino salto a salto.

La dirección del enrutador que es el próximo salto de una ruta se usa para identificar de forma única los vínculos en una interfaz de punto a multipunto, donde todos los vínculos comparten el mismo índice de interfaz.
