# <a name="inheritance-ad-ds"></a>Herencia (AD DS)

[*La*](/previous-versions/windows/desktop/legacy/ms681910(v=vs.85)) herencia permite propagar una ACE determinada desde el contenedor donde se aplicó a todos los secundarios del contenedor. La herencia se puede combinar con la delegación para conceder derechos administrativos a un subárbol completo del directorio en una sola operación.
