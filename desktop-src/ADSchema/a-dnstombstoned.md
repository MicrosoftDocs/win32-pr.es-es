# <a name="dns-tombstoned-attribute"></a>DNS-Tombstoned atributo)

True si este objeto se ha desechado. Este atributo existe para facilitar y agilizar la búsqueda de registros de marcadores de exclusión. Los objetos de desecho son objetos que se han eliminado pero que todavía no se han quitado del directorio.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | DNS-Tombstoned                       |
| Nombre para mostrar de LDAP | dNSTombstoned                        |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | Cada vez que se elimina un objeto.       |
| Attribute-Id      | 1.2.840.113556.1.4.1414              |
| System-ID-GUID    | d5eb2eb7-be4e-463b-a214-634a44d7392e |
| Sintaxis            | [**Booleano**](s-boolean.md)         |
