# <a name="application-gestures"></a>Gestos de aplicación

Los gestos de aplicación son gestos que elija para que la aplicación sea compatible. Son más comunes en las aplicaciones diseñadas específicamente para trabajar con un lápiz de Tablet PC. Una aplicación heredada puede usar gestos, pero los modelos de selección existentes dentro de la aplicación pueden complicar la compatibilidad con los gestos. Solo los gestos admitidos en el panel de entrada de Tablet PC están disponibles para las aplicaciones heredadas.

En las secciones siguientes se describen los gestos de aplicación y su uso.

-   [Gestos de aplicación y comportamiento semántico](application-gestures-and-semantic-behavior.md)
-   [Usar el reconocedor de gestos de Microsoft únicamente](using-the-microsoft-gesture-recognizer-only.md)
-   [Usar un reconocedor de gestos personalizado](using-a-custom-gesture-recognizer.md)
-   [Posibles conflictos con los gestos de aplicación](potential-conflicts-with-application-gestures.md)
-   [Documentar los gestos usados por las aplicaciones](documenting-the-gestures-used-by-your-applications.md)
-   [Glifos no implementados](unimplemented-glyphs.md)
