<span data-ttu-id="2ae43-122">Windows Media Player 11 presenta los conceptos de varias bibliotecas remotas y.</span><span class="sxs-lookup"><span data-stu-id="2ae43-122">Windows Media Player 11 <span data-ttu-id="2ae43-135">Después de recuperar el número de bibliotecas, puede recorrer en iteración la colección de bibliotecas disponibles mediante una llamada repetida a [IWMPLibraryServices:: getLibraryByType](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmplibraryservices-getlibrarybytype) en un bucle, pasando un nuevo valor de índice con cada iteración.</span><span class="sxs-lookup"><span data-stu-id="2ae43-135">After you retrieve the library count, you can iterate the collection of available libraries by repeatedly calling [IWMPLibraryServices::getLibraryByType](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmplibraryservices-getlibrarybytype) in a loop, passing a new index value with each iteration.</span></span> <span data-ttu-id="2ae43-136">Este método recupera un puntero a una interfaz [IWMPLibrary](/previous-versions/windows/desktop/api/wmp/nn-wmp-iwmplibrary) , que representa la biblioteca individual.</span><span class="sxs-lookup"><span data-stu-id="2ae43-136">This method retrieves a pointer to an [IWMPLibrary](/previous-versions/windows/desktop/api/wmp/nn-wmp-iwmplibrary) interface, which represents the individual library.</span></span>

<span data-ttu-id="2ae43-137">Después de recuperar un puntero a **IWMPLibrary**, puede llamar a [IWMPLibrary:: get \_ <span data-ttu-id="2ae43-138">Con este puntero de interfaz, puede trabajar con una biblioteca individual de forma muy similar a como usaría la biblioteca local.</span><span class="sxs-lookup"><span necesita mostrar el nombre de la biblioteca al usuario.</span><span class="sxs-lookup"><span data-stu-id="2ae43-139">You can also retrieve a string containing the name of the library by calling [IWMPLibrary::get\_name](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmplibrary-get_name), which is useful if you need to display the library name to the user.</span></span> <span data-ttu-id="2ae43-140">Puede llamar a [IWMPLibrary:: get \_ Type](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmplibrary-get_type) para obtener el **WMPLibraryType** de una biblioteca determinada.</span><span class="sxs-lookup"><span data-stu-id="2ae43-140">You can call [IWMPLibrary::get\_type](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmplibrary-get_type) to get the **WMPLibraryType** of a particular library.</span></span>

<span data-ttu-id="2ae43-141">Puede controlar el evento [IWMPEvents3:: LibraryConnect](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmpevents3-libraryconnect) para saber cuándo está disponible una biblioteca y el evento [IWMPEvents3:: LibraryDisconnect](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmpevents3-librarydisconnect) para saber cuándo se desconecta una biblioteca.</span><span class="sxs-lookup"><span data-stu-id="2ae43-141">You can handle the [IWMPEvents3::LibraryConnect](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmpevents3-libraryconnect) event to know when a library becomes available and the [IWMPEvents3::LibraryDisconnect](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmpevents3-librarydisconnect) event to know when a library disconnects.</span></span> <span data-ttu-id="2ae43-142">Cada uno de estos eventos proporciona un puntero **IWMPLibrary** que representa la biblioteca que se conectó o se desconectó.</span><span class="sxs-lookup"><span data-stu-id="2ae43-142">Each of these events provides an **IWMPLibrary** pointer that represents the library that connected or disconnected.</span></span> <span data-ttu-id="2ae43-143">Puede llamar a [IWMPLibrary:: isIdentical](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmplibrary-isidentical) para determinar si la biblioteca que se conectó o desconectó es una biblioteca que está usando actualmente.</span><span class="sxs-lookup"><span data-stu-id="2ae43-143">You can call [IWMPLibrary::isIdentical](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmplibrary-isidentical) to determine whether the library that connected or disconnected is a library that you are currently using.</span></span>

-   <span data-ttu-id="2ae43-149">Si recupera la biblioteca local a través de **IWMPLibraryServices**, todos los mismos atributos están disponibles para su uso como los que están disponibles al recuperar la biblioteca local a través de **IWMPCore**.</span><span class="sxs-lookup"><span data-stu-id="2ae43-149">If you retrieve the local library through **IWMPLibraryServices**, all of the same attributes are available for your use as those available when you retrieve the local library through **IWMPCore**.</span></span> <span data-ttu-id="2ae43-150">Sin embargo, podrá recuperar los resultados de la consulta mucho más rápido para los atributos de uso más frecuente.</span><span class="sxs-lookup"><span data-stu-id="2ae43-150">However, you will be able to retrieve query results much faster for the most commonly-used attributes.</span></span>
-   <span data-ttu-id="2ae43-151">Debe usar colecciones de cadenas para crear elementos de la interfaz de usuario al trabajar con las bibliotecas recuperadas a través de **IWMPLibraryServices**.</span><span class="sxs-lookup"><span data-stu-id="2ae43-151">You should use string collections to create user interface elements when you work with libraries retrieved through **IWMPLibraryServices**.</span></span> <span data-ttu-id="2ae43-152">Por ejemplo, [IWMPMediaCollection2:: getStringCollectionByQuery](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmpmediacollection2-getstringcollectionbyquery) permite usar una consulta personalizada (representada por la interfaz **IWMPQuery** ) para recuperar una colección de cadenas.</span><span class="sxs-lookup"><span data-stu-id="2ae43-152">For example, [IWMPMediaCollection2::getStringCollectionByQuery](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmpmediacollection2-getstringcollectionbyquery) enables you to use a custom query (represented by the **IWMPQuery** interface) to retrieve a string collection.</span></span>
-   <span data-ttu-id="2ae43-153">Es posible que las bibliotecas remotas recuperadas a través de la interfaz **IWMPLibraryServices** no siempre estén conectadas a la instancia actual del reproductor.</span><span class="sxs-lookup"><span data-stu-id="2ae43-153">Remote libraries retrieved through the **IWMPLibraryServices** interface might not always be connected to the current Player instance.</span></span> <span data-ttu-id="2ae43-154">Esto significa que es importante controlar los eventos **LibraryConnect** y **LibraryDisconnect** para saber cuándo cambia la disponibilidad de la biblioteca y controlar el evento [IWMPEvents3:: StringCollectionChange](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmpevents3-stringcollectionchange) para saber cuándo se agregan o quitan cadenas de las colecciones de cadenas que se usan para mostrar los elementos de la interfaz de usuario.</span><span class="sxs-lookup"><span data-stu-id="2ae43-154">This means that it is important to handle the **LibraryConnect** and **LibraryDisconnect** events to know when library availability changes, and to handle the [IWMPEvents3::StringCollectionChange](/previous-versions/windows/desktop/api/wmp/nf-wmp-iwmpevents3-stringcollectionchange) event to know when strings are added to or removed from the string collections you use to display user interface elements.</span></span> <span data-ttu-id="2ae43-155">También debe controlar los eventos de **StringCollectionChange** para la biblioteca local por motivos similares.</span><span class="sxs-lookup"><span data-stu-id="2ae43-155">You should also handle **StringCollectionChange** events for the local library for similar reasons.</span></span>

<span data-ttu-id="2ae43-156">Es importante comprender que las colecciones de cadenas de una biblioteca determinada pueden cambiar en cualquier momento.</span><span class="sxs-lookup"><span data-stu-id="2ae43-156">It is important to understand that string collections for a particular library can change at any time.</span></span> <span data-ttu-id="2ae43-157">Una colección de cadenas puede estar vacía cuando se recupera por primera vez, como en el caso de una biblioteca que se ha conectado recientemente pero el reproductor todavía no ha enumerado su contenido.</span><span class="sxs-lookup"><span data-stu-id="2ae43-157">A string collection could be empty when you first retrieve it, as in the case of a library that has recently connected but the Player has not yet enumerated its contents.</span></span> <span data-ttu-id="2ae43-158">Por el contrario, es posible que una colección de cadenas ya contenga toda la información que necesita.</span><span class="sxs-lookup"><span data-stu-id="2ae43-158">Conversely, a string collection might already contain all the information you need.</span></span> <span data-ttu-id="2ae43-159">En este caso, es posible que nunca reciba un evento **StringCollectionChange** porque el contenido de la biblioteca se ha enumerado completamente antes de crear la consulta y no está cambiando nada más.</span><span class="sxs-lookup"><span data-stu-id="2ae43-159">In this case, you might never receive a **StringCollectionChange** event because the library's contents had been completely enumerated before you created your query and nothing else is changing.</span></span>

<span data-ttu-id="2ae43-160">Por lo tanto, para mantener la interfaz de usuario actual, debe enumerar el contenido de la colección de cadenas al recuperar la interfaz **IWMPStringCollection** y controlar el evento **StringCollectionChange** para conocer las actualizaciones cuando se producen.</span><span class="sxs-lookup"><span data-stu-id="2ae43-160">Therefore, to keep your user interface current, you must both enumerate the string collection contents when you retrieve the **IWMPStringCollection** interface and handle the **StringCollectionChange** event to know about updates when they happen.</span></span>
