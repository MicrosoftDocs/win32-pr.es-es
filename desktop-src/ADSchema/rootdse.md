# <a name="rootdse-ad-schema"></a>RootDSE (esquema de AD)

En LDAP 3,0, rootDSE se define como la raíz del árbol de datos de directorio en un servidor de directorio. RootDSE no forma parte de ningún espacio de nombres. El propósito de rootDSE es proporcionar datos sobre el servidor de directorio. Para obtener más información acerca de rootDSE, vea [enlace sin servidor y RootDSE](/windows/desktop/AD/serverless-binding-and-rootdse) en la documentación del SDK de Active Directory.

rootDSE contiene los siguientes atributos. Todos los atributos tienen un solo valor, a menos que se indique lo contrario.



| Atributo                                    | Sintaxis                                                 | Descripción                                                                                                                                                                                                                                                                                                                                                                                                            |
|----------------------------------------------|--------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **configurationNamingContext**<br/>    | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del contenedor de configuración.<br/>                                                                                                                                                                                                                                                                                                                                            |
| **currentTime**<br/>                   | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene la hora actual establecida en este servidor de directorio en formato de hora universal coordinada.<br/>                                                                                                                                                                                                                                                                                                                |
| **defaultNamingContext**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del dominio del que es miembro este servidor de directorio.<br/>                                                                                                                                                                                                                                                                                                                  |
| **servicePrincipalName**<br/>                   | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene la dirección DNS de este servidor de directorio.<br/>                                                                                                                                                                                                                                                                                                                                                         |
| **domainControllerFunctionality**<br/> | [**String(Teletex)**](s-string-teletex.md)<br/> | Indica el nivel funcional del controlador de dominio. Puede ser uno de los valores siguientes.<br/> "0": modo Windows 2000<br/> "2": modo de Windows Server 2003<br/> "3": modo Windows Server 2008<br/>                                                                                                                                                                                    |

| **dsServiceName**<br/>                 | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del objeto de configuración NTDS para este servidor de directorio.<br/>                                                                                                                                                                                                                                                                                                                      |
| **forestFunctionality**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Indica el nivel funcional del bosque. | **highestCommittedUSN,**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el número de secuencia de actualización (USN) más alto en este servidor de directorio. | **isGlobalCatalogReady**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Indica si el catálogo global está totalmente operativo. | **isSynchronized**<br/>                | [**String(Teletex)**](s-string-teletex.md)<br/> | Indica si el servidor de directorio está totalmente sincronizado. Contiene "TRUE" o "FALSE".<br/>                                                                                                                                                                                                                                                                                                                 |
| **ldapServiceName**<br/>               | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre de entidad de seguridad de servicio (SPN) para el servidor LDAP. Se utiliza para la autenticación mutua.<br/>                                                                                                                                                                                                                                                                                                              |
| **namingContexts**<br/>                | [**String(Teletex)**](s-string-teletex.md)<br/> | Un atributo de varios valores que contiene los nombres distintivos de todos los contextos de nomenclatura almacenados en este servidor de directorio. De forma predeterminada, un controlador de dominio de Windows 2000 contiene al menos tres contextos de nomenclatura: esquema, configuración y uno para el dominio del que el servidor es miembro.<br/>                                                                                                             |
| **rootDomainNamingContext**<br/>       | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del primer dominio del bosque que contiene el dominio del que es miembro este servidor de directorio.<br/>                                                                                                                                                                                                                                                                     |
| **schemaNamingContext**<br/>           | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del contenedor de esquemas.<br/>                                                                                                                                                                                                                                                                                                                                                   |
| **serverName**<br/>                    | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del objeto de servidor para este servidor de directorio en el contenedor de configuración.<br/>                                                                                                                                                                                                                                                                                             |
| **subschemaSubentry**<br/>             | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene el nombre distintivo del objeto de [**subesquema**](c-subschema.md) . | **supportedCapabilities**<br/>         | [**String(Teletex)**](s-string-teletex.md)<br/> | Un atributo de varios valores que contiene las funciones admitidas por este servidor de directorio.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedControl**<br/>              | [**String(Teletex)**](s-string-teletex.md)<br/> | Un atributo de varios valores que contiene los OID para los controles de extensión admitidos por este servidor de directorio. Vea la tabla siguiente para obtener una lista de los posibles OID de control.<br/>                                                                                                                                                                                                                                  |
| **supportedLDAPPolicies**<br/>         | [**String(Teletex)**](s-string-teletex.md)<br/> | Un atributo de varios valores que contiene los nombres de las directivas de administración de LDAP admitidas.<br/>                                                                                                                                                                                                                                                                                                              |
| **supportedLDAPVersion**<br/>          | [**String(Teletex)**](s-string-teletex.md)<br/> | Un atributo de varios valores que contiene las versiones LDAP (especificadas por el número de versión principal) admitidas por este servidor de directorio.<br/>                                                                                                                                                                                                                                                                         |
| **supportedSASLMechanisms**<br/>       | [**String(Teletex)**](s-string-teletex.md)<br/> | Contiene los mecanismos de seguridad admitidos para la negociación SASL (consulte RFC de LDAP). De forma predeterminada, se admite GSSAPI.<br/>                                                                                                                                                                                                                                                                                           |
