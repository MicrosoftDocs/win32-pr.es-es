# <a name="system-restore-api"></a>Restaurar sistema API

La API Restaurar sistema permite a una aplicación administrar puntos de restauración. También hay una interfaz que admite scripts disponible a través [de Windows Management Instrumentation](/windows/desktop/WmiSdk/wmi-start-page) (WMI).

Para obtener más información sobre Restaurar sistema API, consulte los temas siguientes:

-   [Llamada a SRSetRestorePoint](calling-srsetrestorepoint.md)
-   [Texto de descripción del punto de restauración](restore-point-description-text.md)
-   [Llamadas anidadas a SRSetRestorePoint](nested-calls-to-srsetrestorepoint.md)

Para obtener más información sobre las Restaurar sistema WMI, vea [Configuring Restaurar sistema](configuring-system-restore.md).
