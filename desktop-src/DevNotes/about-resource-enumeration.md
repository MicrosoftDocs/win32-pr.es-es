# <a name="about-resource-enumeration"></a>Acerca de la enumeración de recursos

La enumeración de recursos proporciona una vista cercana y precisa de la actividad de instrumentación que tiene lugar cuando se ejecuta una aplicación. La información de instrumentación se puede abstraer y clasificar como un conjunto de elementos específicos del proceso como parte del proceso de depuración.

Las herramientas, como los depuradores, suelen requerir el acceso a la información de instrumentación, pero pueden encontrar que no está disponible. Para solucionar este problema, la función [**VerifierEnumerateResource**](/windows/desktop/api/Avrfsdk/nf-avrfsdk-verifierenumerateresource) extrae este tipo de información de las aplicaciones para proporcionar una mejor información de contexto para el problema que se está depurando.
