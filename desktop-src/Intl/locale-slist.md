# <a name="locale_slist"></a>SLIST de configuración regional \_

Los caracteres usados para separar los elementos de la lista, por ejemplo, una coma se usa en muchas configuraciones regionales. El número máximo de caracteres permitido para esta cadena es cuatro, incluido un carácter nulo de terminación.
