# <a name="sample-trackbars-section"></a>Sección Trackbars de ejemplo

Las siguientes líneas muestran una sección típica de Trackbars de un archivo de definición de máscara:


```C++
[ Trackbars ]

//  <Type>      <Location>     <Dis Image Src>  <Thumb Image Src>  <Thumb Size>
//  ----------  ----------     ---------------  -----------------  ------------
    Seek      32,220,172,23  Disabled @ 32,27    SeekThumb.bmp    23,22

```



En el ejemplo anterior se define una barra de contenido que permite al usuario buscar en un elemento multimedia punteando y arrastrando la imagen SeekThumb.
