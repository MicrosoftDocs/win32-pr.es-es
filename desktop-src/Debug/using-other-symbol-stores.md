# <a name="using-other-symbol-stores"></a>Usar otros almacenes de símbolos

Es posible escribir su propio programa de creación de almacén de símbolos, en lugar de usar SymStore.

Dado que todas las transacciones de SymStore se registran en archivos de texto en formato CSV, puede aprovechar los archivos de registro de SymStore existentes para usarlos en su propio programa de base de datos.

Si tiene previsto usar el programa SymSrv proporcionado con Herramientas de depuración para Windows, se recomienda que también use SymStore. Las actualizaciones de estos dos programas siempre se lanzarán juntas y, por tanto, sus versiones siempre coincidirán.
