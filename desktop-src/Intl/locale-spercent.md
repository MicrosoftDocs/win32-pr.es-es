# <a name="locale_spercent"></a>configuración regional \_ SPERCENT

**Windows 7 y versiones posteriores:** Símbolo que se usa para indicar el porcentaje, por ejemplo, "%". El valor es una cadena única de 0 a 3 caracteres.
