# <a name="using-event-logging"></a>Uso del registro de eventos

Las aplicaciones pueden usar event logging API para notificar y ver eventos. Para obtener más información sobre los eventos de informes, vea [Reporting Events](reporting-an-event.md).

Para ver los eventos que se han escrito en archivos de registro, vea Consulta de mensajes de [origen](querying-for-event-source-messages.md) de eventos y [Recepción de notificaciones de eventos.](receiving-event-notification.md) También puede usar el Visor de eventos para ver eventos.
