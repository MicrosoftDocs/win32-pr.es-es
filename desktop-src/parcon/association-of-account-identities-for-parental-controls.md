# <a name="association-of-account-identities-for-parental-controls"></a>Asociación de identidades de cuenta para controles parentales

Como se describe en la introducción, el uso efectivo de la funcionalidad de configuración, registro y restricciones que proporciona la infraestructura de controles parentales de Windows Vista depende de las cuentas de Windows que se están configurando como usuarios estándar. Se recomienda que los proveedores que proporcionan acceso en línea (como los conjuntos de servicios de Internet) creen y mantengan una asociación de nombres de usuario en línea que el primario o el tutor permitan para cada cuenta de Windows controlada. De esta manera, se reduce la omisión de controles parentales basados en la nube simplemente mediante identidades en línea alternativas.
