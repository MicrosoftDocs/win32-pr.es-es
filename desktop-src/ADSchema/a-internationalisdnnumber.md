# <a name="international-isdn-number-attribute"></a>International-ISDN-Number (atributo)

Especifica un número de ISDN internacional asociado a un objeto.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | International-ISDN (número)                   |
| Nombre para mostrar de LDAP | internationalISDNNumber                     |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 2.5.4.25                                    |
| System-ID-GUID    | bf96798d-0de6-11d0-a285-00aa003049e2        |
| Sintaxis            | [**String(Numeric)**](s-string-numeric.md) |
