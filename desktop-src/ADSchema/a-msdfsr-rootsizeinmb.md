Contiene el tamaño, en megabytes, del directorio raíz.

| CN                | MS-DFSR-RootSizeInMb                 |
| Nombre para mostrar de LDAP | msDFSR-RootSizeInMb                  |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.4            |
| System-ID-GUID    | 90b769ac-4413-43cf-ad7a-867142e740a3 |
