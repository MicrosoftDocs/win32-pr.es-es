# <a name="imagebutton"></a>Imagebutton

Se trata de un **button predefinido** con el siguiente valor predeterminado.

``` syntax
cursor="Hand"
```

## <a name="remarks"></a>Comentarios

Todos los botones pueden contener imágenes. Este elemento es útil con fines de aclaración de código para indicar un botón que se usa principalmente como contenedor de imágenes.

Todas las propiedades de **este BOTÓN** se pueden invalidar especificándolos explícitamente.

## <a name="requirements"></a>Requisitos



| Requisito | Valor |
|--------------------|------------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media versión 7.0 o posterior<br/> |
