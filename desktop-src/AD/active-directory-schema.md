# <a name="active-directory-schema-ad-ds"></a>Esquema de Active Directory (AD DS)

El esquema de Active Directory contiene definiciones formales de cada clase de objeto que se puede crear en un bosque Active Directory. El esquema también contiene definiciones formales de cada atributo que puede existir en un objeto de Active Directory.

Entre los temas que proporcionan información general sobre el esquema de Active Directory se incluyen:

-   [Implementación de esquema](schema-implementation.md)
-   [Características de las clases de objeto](characteristics-of-object-classes.md)
-   [Características de los atributos](characteristics-of-attributes.md)
-   [El esquema abstracto](the-abstract-schema.md)

Para obtener más información sobre la programación de esquemas, incluida la lectura del esquema y la definición de nuevas clases y atributos en el esquema, vea [extender el esquema](extending-the-schema.md).

Para obtener más información, consulte la [referencia de Active Directory Domain Services](active-directory-domain-services-reference.md). Para ver las páginas de referencia de las clases de esquema predefinidas, atributos y sintaxis de atributos, vea la [referencia de esquemas de Active Directory](/windows/desktop/ADSchema/active-directory-schema).
