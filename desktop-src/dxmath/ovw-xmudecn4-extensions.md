# <a name="xmudecn4-extensions"></a>Extensiones de XMUDECN4

[**XMUDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudecn4) Un vector 4D para almacenar valores normalizados y sin signo como componentes x-, y-y z-unsigned de 10 bits y un componente w-unsigned de 2 bits.

> [!Note]  
> Las características que se enumeran aquí solo están disponibles cuando se desarrollan con C++.
