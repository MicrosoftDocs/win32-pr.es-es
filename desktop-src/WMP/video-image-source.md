# <a name="video-image-source"></a>Origen de imagen de vídeo

No se requiere ninguna imagen independiente para el área de presentación de vídeo. Es posible que desee crear un área en la imagen de fondo, que se mostrará de forma predeterminada, para que actúe como un marcador de posición. Haga que el área sea rectangular en forma, con el mismo ancho y alto que desee especificar en el archivo de definición de máscara. Cuando se reproduzca el vídeo, el vídeo se superpondrá en el área del archivo de fondo.
