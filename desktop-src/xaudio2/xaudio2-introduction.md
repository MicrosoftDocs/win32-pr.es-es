    <span data-ttu-id="f3d55-118">La submezcla combina varios sonidos en una única secuencia de audio, por ejemplo, un sonido de motor compuesto por partes compuestas, que se reproducen simultáneamente.</span><span class="sxs-lookup"><span data-stu-id="f3d55-118">Submixing combines several sounds into a single audio stream—for example, an engine sound made up of composite parts, all of which are playing simultaneously.</span></span> <span data-ttu-id="f3d55-119">Además, puede usar la submezcla para procesar y combinar partes similares de un juego.</span><span class="sxs-lookup"><span data-stu-id="f3d55-119">Also, you can use submixing to process and combine similar parts of a game.</span></span> <span data-ttu-id="f3d55-120">Por ejemplo, puede combinar todos los efectos sonoros de juego para permitir que se aplique una configuración de volumen de usuario mientras una configuración independiente controla el volumen de música.</span><span class="sxs-lookup"><span data-stu-id="f3d55-120">For example, you could combine all game sound effects to allow a user volume setting to be applied while a separate setting controls music volume.</span></span> <span data-ttu-id="f3d55-121">Combinado con DSP, la submezcla proporciona el tipo de enrutamiento de datos y el procesamiento necesario para los juegos de hoy en día.</span><span class="sxs-lookup"><span data-stu-id="f3d55-121">Combined with DSP, submixing provides the type of data routing and processing necessary for today's games.</span></span> <span data-ttu-id="f3d55-122">XAudio2 permite niveles arbitrarios de submezcla, lo que permite la creación de sonidos y mezclas de juegos complejos.</span><span class="sxs-lookup"><span data-stu-id="f3d55-122">XAudio2 allows for arbitrary levels of submixing, enabling the creation of complex sounds and game mixes.</span></span>

-   <span data-ttu-id="f3d55-124">Compatibilidad con audio comprimido</span><span class="sxs-lookup"><span data-stu-id="f3d55-124">Compressed Audio Support</span></span>

-   <span data-ttu-id="f3d55-127">Compatibilidad con multicanal y sonido envolvente mejorado</span><span class="sxs-lookup"><span data-stu-id="f3d55-127">Enhanced Multichannel and Surround Sound Support</span></span>

    <span data-ttu-id="f3d55-128">Se amplía la compatibilidad con multicanal, 3D y sonido envolvente.</span><span class="sxs-lookup"><span data-stu-id="f3d55-128">Multichannel, 3D, and surround sound support is expanded.</span></span> <span data-ttu-id="f3d55-129">3D y el sonido envolvente son ahora mucho más flexibles y transparentes.</span><span class="sxs-lookup"><span data-stu-id="f3d55-129">3D and surround sound are now much more flexible and transparent.</span></span> <span data-ttu-id="f3d55-130">XAudio2 quita el límite de 6 canales de los sonidos multicanal y admite audio multicanal en cualquier tarjeta de audio compatible con multicanal.</span><span class="sxs-lookup"><span data-stu-id="f3d55-130">XAudio2 removes the 6-channel limit on multichannel sounds, and supports multichannel audio on any multichannel-capable audio card.</span></span> <span data-ttu-id="f3d55-131">No es necesario que la tarjeta se acelere por hardware.</span><span class="sxs-lookup"><span data-stu-id="f3d55-131">The card does not need to be hardware-accelerated.</span></span>

-   <span data-ttu-id="f3d55-132">Procesamiento de velocidad</span><span class="sxs-lookup"><span data-stu-id="f3d55-132">Multirate Processing</span></span>
