# <a name="registeredtaskcollectioncount-property"></a>RegisteredTaskCollection. Count (propiedad)

En el caso de scripting, obtiene el número de tareas registradas en la colección.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
RegisteredTaskCollection.Count As long
```



## <a name="property-value"></a>Valor de propiedad

El número de tareas registradas en la colección.
