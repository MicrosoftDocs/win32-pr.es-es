# <a name="originalheight-property"></a>Propiedad Altooriginal

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Devuelve un valor entero que especifica el alto original del carácter.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintáctica**
</dt> <dd>

*directivas. ***Caracteres ("*** CharacterID * *"). Altooriginal**

</dd> </dl>

## <a name="remarks"></a>Observaciones

Esta propiedad devuelve el alto del marco del carácter tal y como se ha creado con el editor de caracteres del agente de Microsoft.
