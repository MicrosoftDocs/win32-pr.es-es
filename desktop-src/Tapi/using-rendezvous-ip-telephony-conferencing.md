# <a name="using-rendezvous-ip-telephony-conferencing"></a>Usar conferencias de telefonía IP de Rendezvous

\[ Las interfaces y controles de conferencias de telefonía IP de encuentro no están disponibles para su uso en Windows Vista, Windows Server 2008 y las versiones posteriores del sistema operativo. La API de cliente de RTC proporciona una funcionalidad similar.\]

Rendezvous consta de tres conjuntos básicos de interfaces:

-   [Los controles de directorio](directory-controls.md) proporcionan un medio para detectar, crear o modificar la información de la Conferencia y del usuario en un servidor ILS o Ntds.
-   [Los controles de BLOB de conferencia](conference-blob-controls.md) permiten realizar modificaciones específicas del proveedor en una conferencia.
-   Las [interfaces com multidifusión](multicast-com-interfaces.md) proporcionan un medio para reservar direcciones para su uso en conferencias multidifusión.
