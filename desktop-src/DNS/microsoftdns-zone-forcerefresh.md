# <a name="forcerefresh-method-of-the-microsoftdns_zone-class"></a>Método ForceRefresh de la clase Zone de MicrosoftDNS \_

> [!NOTE]
> Este artículo contiene referencias al término servidor maestro, un término que Microsoft ya no usa. Cuando se quite el término del software, se quitará también del artículo.

El **método ForceRefresh** fuerza una actualización del servidor DNS secundario desde el servidor maestro.

## <a name="syntax"></a>Sintaxis


```mof
void ForceRefresh();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
