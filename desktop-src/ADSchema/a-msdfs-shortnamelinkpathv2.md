# <a name="ms-dfs-short-name-link-path-v2-attribute"></a>atributo MS-DFS-Short-Name-Link-path-V2

Ruta de acceso de vínculo DFS de nombre corto relativa al recurso compartido de destino raíz DFS (es decir, sin los componentes servidor/dominio y nombre del espacio de nombres DFS). Utilice barras diagonales (/) en lugar de barras diagonales inversas ( \) , de modo que las búsquedas LDAP se puedan realizar sin tener que usar escapes.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-DFS-Short-Name-Link-path-V2              |
| Nombre para mostrar de LDAP | msDFS-ShortNameLinkPathv2                   |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2042                     |
| System-ID-GUID    | 2d7826f0-4cf7-42e9-a039-1110e0d9ca99        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
