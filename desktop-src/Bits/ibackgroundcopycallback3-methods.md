# <a name="ibackgroundcopycallback3-methods"></a>Métodos IBackgroundCopyCallback3

La interfaz [**IBackgroundCopyCallback3**](/windows/desktop/api/Bits10_1/nn-bits10_1-ibackgroundcopycallback3) expone los métodos siguientes.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                             | Descripción                                                                                                                                                                                                                                                                                                                             |
|---------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Método FileRangesTransferred**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopycallback3-filerangestransferred)<br/> | BITS llama a la implementación del método [**FileRangesTransferred**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopycallback3-filerangestransferred) cuando se han descargado uno o varios intervalos de archivos. Los intervalos de archivos se agregan al trabajo mediante el método [**IBackgroundCopyFile6:: RequestFileRanges**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-requestfileranges) .<br/> |
