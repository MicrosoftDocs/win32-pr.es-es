# <a name="dsp-plug-in-developer-overview"></a>Introducción al desarrollador de complementos de DSP

Desde la perspectiva del desarrollador, los complementos DSP son programas de software que reciben datos de audio o vídeo proporcionados por Windows Media Player justo antes de la fase de representación y, a continuación, procesan los datos y devuelven los datos a Media Player de Windows para su representación.

En las secciones siguientes se proporciona información general sobre los complementos DSP de Windows Media Player:

-   [Empaquetado de complementos de DSP](dsp-plug-in-packaging.md)
-   [Conectarse a Media Player de Windows](connecting-to-windows-media-player.md)
-   [Entrada y salida de datos](data-input-and-output.md)
-   [Negociación de formato](format-negotiation.md)
-   [Proporcionar una interfaz de usuario](providing-a-user-interface.md)
-   [Asistente para complementos de DSP](dsp-plug-in-wizard.md)
-   [Interfaces necesarias](required-interfaces.md)
-   [Registro de complementos de DSP](registering-dsp-plug-ins.md)
-   [Actualización de complementos de DSP existentes](updating-existing-dsp-plug-ins.md)
