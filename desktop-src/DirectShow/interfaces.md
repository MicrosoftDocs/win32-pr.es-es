# <a name="directshow-interfaces"></a>Interfaces de DirectShow

Este tema contiene información de referencia para las interfaces principales de Microsoft® DirectShow®. Para proporcionar una amplia compatibilidad con diversos tipos de hardware, formatos de archivo y operaciones de streaming, implementa un gran número de interfaces. Algunas son usadas por las aplicaciones; otros son utilizados internamente por los filtros y otros componentes de DirectShow.

El tema siguiente contiene una lista alfabética de todas las interfaces de DirectShow de código.

-   [Lista alfabética de interfaces de DirectShow](alphabetical-list-of-directshow-interfaces.md)

Cualquier aplicación determinada usará solo un pequeño subconjunto de estas interfaces. En los temas siguientes se enumeran las interfaces por categoría funcional. No todas las interfaces se muestran en estas páginas y algunas interfaces aparecen en más de una página.

-   [Interfaces para generar gráficos de filtros](interfaces-for-building-filter-graphs.md)
-   [Interfaces para controlar un gráfico de filtros](interfaces-for-controlling-a-filter-graph.md)
-   [Interfaces de representación y captura de audio](audio-capture-and-rendering-interfaces.md)
-   [Interfaces de DVD](dvd-interfaces.md)
-   [Interfaces de captura de vídeo](video-capture-interfaces.md)
-   [Interfaces de codificación y descodificación de archivos](file-encoding-and-decoding-interfaces.md)
-   [Interfaces para la representación y superposición de vídeo](interfaces-for-video-rendering-and-overlay.md)
-   [Interfaces de control de dispositivo externo](external-device-control-interfaces.md)
