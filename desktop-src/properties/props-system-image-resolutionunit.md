# <a name="systemimageresolutionunit"></a>System. Image. ResolutionUnit

Indica las unidades de resolución. Se usa para las imágenes con una relación de aspecto no cuadrada, pero sin dimensiones absolutas significativas. 1 = ninguna unidad de medida absoluta. 2 = pulgadas. 3 = centímetros. El valor predeterminado es 2 (pulgadas).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.Image.ResolutionUnit
   shellPKey = PKEY_Image_ResolutionUnit
   formatID = 19B51FA6-1F92-4A5C-AB48-7DF0ABD67444
   propID = 100
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Int16
      IsInnate = true
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
