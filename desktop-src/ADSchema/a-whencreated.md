# <a name="when-created-attribute"></a>When-Created atributo

Fecha en que se creó este objeto. Este valor se replica y se encuentra en el catálogo global.



| Entrada | Valor |
|-------------------|---------------------------------------------------------------|
| CN                | When-Created                                                  |
| Ldap-Display-Name | whenCreated                                                   |
| Size              | 8 bytes                                                       |
| Privilegio actualizar  | El sistema establece este valor.                              |
| Frecuencia de actualización  | Cuando se crea el objeto .                                   |
| Attribute-Id      | 1.2.840.113556.1.2.2                                          |
| System-Id-Guid    | bf967a78-0de6-11d0-a285-00aa003049e2                          |
| Syntax            | [**String(Generalized-Time)**](s-string-generalized-time.md) |
