# <a name="xmudecn4-operators"></a>Operadores XMUDECN4

La estructura [**XMUDECN4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudecn4) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                             | Descripción                                                                                                            |
|---------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------------|
| [**operador UInt32 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmudecn4-operatoruint32_t)<br/> | Devuelve una instancia de `uint32_t` que contiene los componentes de la `XMUDECN4` instancia en un formato empaquetado. <br/> |
| [**operador =**](xmudecn4-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMUDECN4` . <br/>                                       |
