# <a name="omt-indx-guid-attribute"></a>OMT-indx-atributo GUID

Identificador de índice de una entrada de la tabla Link-Track-Object-Move.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | OMT-indx-GUID                                         |
| Nombre para mostrar de LDAP | oMTIndxGuid                                           |
| Tamaño              | 16 bytes                                              |
| Actualizar privilegio  | El sistema establece este valor.                      |
| Frecuencia de actualización  | Siempre que el vínculo de un archivo cambie.                 |
| Attribute-Id      | 1.2.840.113556.1.4.333                                |
| System-ID-GUID    | 1f0075fa-7e40-11d0-afd6-00c04fd930c9                  |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md) |
