# <a name="print-min-x-extent-attribute"></a>Print-min-X-extensión (atributo)

La región de impresión horizontal mínima.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Print-min-X-extensión                   |
| Nombre para mostrar de LDAP | printMinXExtent                      |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.279               |
| System-ID-GUID    | ba305f71-47e3-11d0-a1a6-00c04fd930c9 |
