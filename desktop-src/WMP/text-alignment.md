# <a name="text-alignment"></a>Alineación del texto

Debe definir la alineación del cuadro de presentación de texto que desea usar. Los valores que se muestran en la tabla siguiente son válidos.



| Value  | Descripción                |
|--------|----------------------------|
| Left   | El texto está alineado a la izquierda.  |
| Center | El texto se centra.      |
| Right  | El texto está alineado a la derecha. |
