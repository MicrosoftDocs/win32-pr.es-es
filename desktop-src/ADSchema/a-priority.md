# <a name="priority-attribute"></a>Atributo de prioridad

La prioridad actual (de un proceso, un trabajo de impresión, etc.).



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Prioridad                             |
| Nombre para mostrar de LDAP | priority                             |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.231               |
| System-ID-GUID    | 281416c7-1968-11d0-a28f-00aa003049e2 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
