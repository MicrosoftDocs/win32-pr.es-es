# <a name="the-application-configuration-file-acf"></a>El archivo de configuración de la aplicación (ACF)

El archivo de configuración de la aplicación (ACF) tiene dos partes: un [encabezado de interfaz](the-acf-header.md), similar al encabezado de la interfaz en el archivo IDL, y un [cuerpo](the-acf-body.md), que contiene los atributos de configuración que se aplican a los tipos y funciones definidos en el cuerpo de la interfaz del archivo IDL. Se describen en los temas siguientes:

-   [El encabezado ACF](the-acf-header.md)
-   [El cuerpo ACF](the-acf-body.md)
