# <a name="internet-explorer-8"></a>Internet Explorer 8

La evolución de *Internet* ha introducido nuevas fuentes de información enriquecida y otras formas de acceder a ellas. Este crecimiento ha creado nuevas oportunidades, experiencias, servicios en línea y estándares en la Web. Con esta intensidad y dependencia, los desarrolladores web se enfrentarán a un conjunto de necesidades en constante evolución, que es la dirección de Windows Internet Explorer 8 de tres maneras clave.

En primer lugar, Internet Explorer 8 proporciona interoperabilidad real con otros exploradores y compatibilidad con los sitios existentes. Incluye mejoras en la plataforma y arquitectura principales, que ofrecen un rendimiento mejorado, seguridad, confiabilidad y compatibilidad. En las versiones anteriores de Windows Internet Explorer, los desarrolladores y diseñadores han observado en ocasiones que Internet Explorer ha tenido su propia interpretación de estándares web y la forma en que el explorador controla HTML, Hojas de estilo CSS (CSS), scripting, etc. En algunos casos, se decidieron las interpretaciones porque Internet Explorer admitía ciertas características antes de que se finalizaran los estándares correspondientes. Si esos estándares cambian a medida que finalizan, la implementación de Internet Explorer puede variar de lo que especifica el estándar. Con Internet Explorer 8, la decisión de admitir comportamientos heredados frente a estándares estrictos se pondrán en manos de los desarrolladores al permitirle seleccionar el modo de representación página por página.

En segundo lugar, Internet Explorer 8 agiliza y facilita el desarrollo web con las herramientas de desarrollo integradas. Gracias a la compatibilidad con las aplicaciones asincrónicas de JavaScript y XML (AJAX), así como con mejoras que simplifican el proceso de creación de aplicaciones para varios exploradores, Internet Explorer 8 permite que los desarrolladores sean más productivos cuando se crean las aplicaciones web más sólidas posible.

Por último, Internet Explorer 8 permite experiencias que llegan más allá de la página a través de nuevas características del explorador que conectan a los usuarios con los servicios web innovadores. Internet Explorer 8 incluye un conjunto de herramientas de desarrollo integradas que son un componente del explorador. Como resultado, se mejora el rendimiento y no se usa ninguna memoria cuando las herramientas no se están ejecutando. Las herramientas de desarrollo integradas facilitan y facilitan a los desarrolladores el desarrollo y la solución de problemas de sitios de contenido enriquecido. El motor de diseño de modo estándar de Internet Explorer 8 también se ha creado con la especificación de la Hojas de estilo CSS, el nivel 2 de la revisión 1 (CSS 2.1) a mano y el cumplimiento completo en mente, lo que ayuda a los desarrolladores a admitir fácilmente los estándares más recientes, a la vez que tienen la opción de revertir a un comportamiento compatible con Windows Internet Explorer 7 (o anterior) si Para ayudar a los desarrolladores a sacar el máximo partido de los elementos que ofrece HTML 4, Internet Explorer 8 proporciona compatibilidad actualizada para varios elementos de presentación. Por ejemplo, el elemento Q representa una cadena entrecomillada insertada y el elemento Object puede representar ahora cualquier "objeto", incluidas las imágenes. Gracias a la compatibilidad mejorada con estos y otros elementos HTML, los desarrolladores web pueden ofrecer un marcado HTML más expresivo y accesible.

Además de las mejoras realizadas en la plataforma subyacente y la experiencia de desarrollador, Internet Explorer 8 presenta nuevas características que proporcionan nuevas oportunidades para que los desarrolladores web amplíen su experiencia más allá de la Página Web. Estas nuevas características se habilitan a través de los estándares de la comunidad en lugar de crear código de cliente personalizado que no se pueda interoperar entre los exploradores.
