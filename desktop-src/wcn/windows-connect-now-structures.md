# <a name="windows-connect-now-structures"></a>Windows Connect Now Estructuras

Las estructuras siguientes admiten las funciones Windows Connect Now API.



| Estructura                                                                             | Descripción                                                                                                                                            |
|---------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**TIPO DE VALOR \_ WCN \_ TIPO DE DISPOSITIVO \_ \_ \_ PRINCIPAL**](/windows/win32/api/wcntypes/ns-wcntypes-wcn_value_type_primary_device_type) | Contiene datos enumerados que identifican el tipo de un dispositivo por categoría, subcategoría y una OUI específica del fabricante (id. de organización).<br/> |
| [**ESPECIFICACIÓN DE \_ EXTENSIÓN \_ DE PROVEEDOR DE WCN \_**](/windows/win32/api/wcndevice/ns-wcndevice-wcn_vendor_extension_spec)                     | Contiene datos que definen una extensión de proveedor. <br/>                                                                                             |
