# <a name="builtin-creation-time-attribute"></a>Atributo Builtin-Creation-Time

El **atributo Builtin-Creation-Time** se usa para admitir la replicación en dominios de Windows NT 4.0.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Builtin-Creation-Time                |
| Ldap-Display-Name | builtinCreationTime                  |
| Size              | 8 bytes                              |
| Actualizar privilegios  | El sistema establece este valor.     |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.13                |
| System-Id-Guid    | bf96792f-0de6-11d0-a285-00aa003049e2 |
| Syntax            | [**Intervalo**](s-interval.md)       |
