# <a name="productid-property"></a>ProductID, propiedad

La **propiedad ProductID** se establece en el **ProductID** completo después de una validación correcta.

La interfaz de usuario muestra este valor y lo usa la [acción RegisterProduct](registerproduct-action.md).

Después de una validación correcta, esta propiedad se establece mediante la [acción ValidateProductID](validateproductid-action.md) o [validateProductID ControlEvent](validateproductid-controlevent.md).

Tenga en cuenta que la interfaz de usuario de ejemplo determinada proporcionada con el SDK como Uisample.msi requiere un valor para **ProductID**. Si usa esta interfaz de usuario, pero no quiere usar **ProductID** para validar los identificadores de producto, establezca la propiedad **ProductID** en "none" en la [tabla Property](property-table.md).
