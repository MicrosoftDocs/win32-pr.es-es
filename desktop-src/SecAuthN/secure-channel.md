# <a name="secure-channel"></a>Canal seguro

El canal seguro, también conocido como [*Schannel*](../secgloss/s-gly.md), es un [*proveedor de compatibilidad para seguridad*](../secgloss/s-gly.md) (SSP) que contiene un conjunto de protocolos de seguridad que proporcionan autenticación de identidad y comunicación privada segura a través del cifrado.

Schannel se usa principalmente para aplicaciones de Internet que requieren comunicaciones del protocolo seguro de transferencia de hipertexto (HTTP).
