# <a name="super-scopes-attribute"></a>Super-Scopes atributo)

Este atributo se utiliza para agrupar todos los diferentes ámbitos utilizados en la clase DHCP en una sola entidad.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Super-Scopes                         |
| Nombre para mostrar de LDAP | Superámbitos                          |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.710               |
| System-ID-GUID    | 963d274b-48be-11d1-a9c3-0000f80367c1 |
| Sintaxis            | [**String(IA5)**](s-string-ia5.md)  |
