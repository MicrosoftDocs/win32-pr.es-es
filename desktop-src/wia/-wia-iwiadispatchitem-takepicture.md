# <a name="itemtakepicture-method"></a>Método Item.TakePicture

El **método TakePicture** del objeto [**Item**](-wia-item.md) hace que un dispositivo de cámara digital tome una imagen y devuelve un objeto **Item** que representa la imagen resultante. Este método solo se aplica a dispositivos de cámara digital.

## <a name="syntax"></a>Sintaxis


```JScript
retVal = Item.TakePicture()
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **IWiaDispatchItem**

Objeto [**Item**](-wia-item.md) que representa la imagen que crea este método.
