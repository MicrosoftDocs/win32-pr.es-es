# <a name="iupdateexception-properties"></a>Propiedades de IUpdateException

La interfaz [**IUpdateException**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateexception) define las siguientes propiedades.



| Propiedad                                    | Descripción                                                |
|---------------------------------------------|------------------------------------------------------------|
| [**Context**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_context) | Obtiene el contexto de la excepción.                         |
| [**HResult**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_hresult) | Obtiene el código **HRESULT** basado en Windows para la excepción. |
| [**Message**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateexception-get_message) | Obtiene un mensaje que describe la excepción actual.       |
