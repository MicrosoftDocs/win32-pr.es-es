# <a name="active-directory-service-interfaces-scripting"></a>Scripting de interfaces de servicio Active Directory

Las interfaces de servicio de Microsoft Active Directory (ADSI), cuando se combinan con un lenguaje de scripting como Microsoft Visual Basic Scripting Edition o el software de desarrollo Microsoft JScript, proporcionan a un administrador del sistema herramientas eficaces para administrar usuarios, servicios, colas de impresión y otras partes de una red de equipos que ejecutan sistemas operativos Microsoft Windows NT o Windows 2000. Los scripts que usan el modelo de objetos ADSI pueden reemplazar esfuerzos manuales repetitivos que normalmente se realizarían mediante herramientas administrativas o Microsoft Management Console. El scripting ADSI también puede automatizar las tareas de mantenimiento habituales del sistema para una operación gratuita y proporciona una interfaz sencilla para escribir herramientas de administración del sistema basadas en Web mediante Active Server páginas.

Para obtener más información sobre el scripting con ADSI, con tutoriales de scripting, ejemplos y herramientas, vea:

-   [Manual de scripts ADSI](https://www.microsoft.com/resources/documentation/windows/2000/server/scriptguide/sas_ads_overview.mspx) en el sitio web de Microsoft TechNet
-   [Centro de scripts](https://www.microsoft.com/technet/community/scriptcenter/default.mspx) en el sitio web de Microsoft TechNet
-   [Errores y interceptación de errores](errors-and-error-trapping.md)
