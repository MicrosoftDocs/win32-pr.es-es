# <a name="swbempropertyname-property"></a>Propiedad SWbemProperty.Name

La propiedad **Name** del objeto [**SWbemProperty**](swbemproperty.md) devuelve el nombre de esta propiedad WMI. Esta propiedad es de solo lectura.

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemProperty.Name As String
```
