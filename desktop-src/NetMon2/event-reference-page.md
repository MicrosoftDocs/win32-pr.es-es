# <a name="event-reference-page"></a>Página referencia de eventos

En esta sección se describe la página de referencia de eventos (ERP), un documento basado en HTML que se puede escribir para mostrar los eventos que se producen durante el funcionamiento de un experto en Monitor de red. Después de definir los requisitos de creación de expertos, puede usar O ERP e para ayudar a Monitor de red resolver problemas de red.
