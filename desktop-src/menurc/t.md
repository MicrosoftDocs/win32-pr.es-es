<span id="tools.t_1_gly"></span><span id="TOOLS.T_1_GLY"></span>**Se esperaba una cadena de texto o un ordinal en el control**
</dt> <dd>

El parámetro de *texto* de una instrucción de [**control**](control-control.md) en la instrucción [**Dialog**](dialog-resource.md) debe ser una cadena de texto o una referencia ordinal al tipo de control que se espera. Si usa un ordinal, asegúrese de que tiene una instrucción de [**\# definición**](-define.md) para el control.
