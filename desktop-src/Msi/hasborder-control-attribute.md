# <a name="hasborder-control-attribute"></a>Atributo de control HasBorder

Si se establece este bit, [RadioButtonGroup](radiobuttongroup-control.md) tiene texto y se muestra un borde alrededor.

Si no se establece el bit de estilo, no se muestra el borde y no se muestra ningún texto en el grupo.

## <a name="valid-controls"></a>Controles válidos

[RadioButtonGroup](radiobuttongroup-control.md)

## <a name="valid"></a>Válido



| Decimal  | Hexadecimal | Constante                            |
|----------|-------------|-------------------------------------|
| 16777216 | 0x01000000  | **msidbControlAttributesHasBorder** |



 

## <a name="remarks"></a>Observaciones

Para establecer este atributo en un control, incluya los bits HasBorder en la columna Attributes del registro del control en la [tabla de control](control-table.md).

Vea [atributos de control](control-attributes.md) y el control que debe crear en [controles](controls.md).
