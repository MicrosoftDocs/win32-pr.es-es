# <a name="avencmpvaddseqendcode-property"></a>Propiedad AVEncMPVAddSeqEndCode

Especifica si el codificador agrega un código final de secuencia al final de la secuencia. Esta propiedad se aplica a los codificadores de vídeo MPEG.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncMPVAddSeqEndCode**

## <a name="remarks"></a>Comentarios

Si el valor es **VARIANT \_ TRUE,** el codificador agrega un código final de secuencia al final de la secuencia.
