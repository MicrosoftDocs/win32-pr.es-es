# <a name="mft_enum_video_renderer_extension_profile-attribute"></a>Atributo MFT \_ ENUM \_ VIDEO \_ RENDERER \_ EXTENSION \_ PROFILE

Contiene una lista de todas las **entradas de VideoRendererExtensionProfile** en la etiqueta **VideoRendererExtensionProfiles** de un archivo de manifiesto de aplicación para UWP. [**MFTEnumEx**](/windows/desktop/api/mfapi/nf-mfapi-mftenumex) almacena esto en el almacén de atributos del objeto [**IMFActivate**](/windows/desktop/api/mfobjects/nn-mfobjects-imfactivate) que **MFTEnumEx** crea para las MFTEnumEx que tienen un manifiesto de UWP asociado que contiene la etiqueta **VideoRendererExtensionProfiles.**
