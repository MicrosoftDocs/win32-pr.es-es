# <a name="painting-and-drawing"></a>Dibujo y dibujo

En esta introducción se describe cómo el sistema administra la salida en la pantalla y se explica qué deben hacer las aplicaciones para dibujar en una ventana. En concreto, esta información general describe cómo mostrar *contextos* de dispositivo (o, más sencillamente, mostrar los *DCs)* y cómo prepararlos y usarlos. En la información general no se explica cómo usar las funciones de interfaz de dispositivo gráfico (GDI) para generar la salida, ni se explica cómo imprimir.
