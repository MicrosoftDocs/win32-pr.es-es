# <a name="winsock-spi-structures"></a>Estructuras de Winsock SPI

En la lista siguiente se proporcionan descripciones concisas de cada estructura de Winsock SPI. Para obtener información adicional sobre cualquier estructura de Winsock SPI, haga clic en el nombre de la estructura.



| Estructura de Winsock SPI                                         | Descripción                                                                                                   |
|---------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------|
| [**\_rutina NSP**](/windows/desktop/api/Ws2spi/ns-ws2spi-nsp_routine)                           | Contiene información sobre las funciones implementadas por un proveedor de proveedor de servicios de espacio de nombres versión 1 (NSPv1). |
| [**\_Rutina NSPV2**](/windows/desktop/api/Ws2spi/ns-ws2spi-nspv2_routine)                       | Contiene información sobre las funciones implementadas por un proveedor de proveedor de servicios de espacio de nombres versión 2 (NSPv2). |
| [**WSATHREADID**](/windows/desktop/api/Ws2spi/ns-ws2spi-wsathreadid)                          | Permite a un proveedor identificar un subproceso en el que se pueden poner en cola llamadas a procedimientos asincrónicos (APC).           |
| [**\_información de \_ Auditoría del proveedor de WSC \_**](/windows/desktop/api/Ws2spi/ns-ws2spi-wsc_provider_audit_info) | Contiene información de auditoría para una entrada de proveedor de servicios por capas (LSP).                                        |
| [**WSPDATA**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspdata)                                  | Contiene información del proveedor de servicios.                                                                        |
| [**\_tabla WSPPROC**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspproc_table)                       | Contiene una tabla de punteros a las funciones del proveedor de servicios.                                                   |
| [**WSPUPCALLTABLE**](/windows/desktop/api/Ws2spi/ns-ws2spi-wspupcalltable)                      | Contiene una tabla de punteros a funciones de llamada de proveedor de servicios.                                            |
