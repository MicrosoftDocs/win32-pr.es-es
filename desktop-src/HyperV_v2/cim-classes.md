# <a name="cim-classes-hyper-v-management"></a>Clases CIM (administración de Hyper-V)

Las clases CIM son clases base de las que derivan todos los objetos WMI. Cada proveedor puede usar posiblemente versiones ligeramente diferentes de las clases CIM; Por lo tanto, la mayoría de los proveedores WMI v2 contienen su propia documentación de clase CIM.

Las siguientes clases CIM se encuentran en el espacio de \\ nombres de \\ virtualización raíz v2.
