# <a name="xmmatrix-operators"></a>Operadores XMMATRIX

La estructura [**XMMATRIX**](/windows/win32/api/directxmath/ns-directxmath-xmmatrix) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                    | Descripción                                                                                                                                                                                   |
|------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**operador \** de _](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult)<br/>    | Realiza una multiplicación de matriz de la instancia actual de `XMMATRIX` por otra instancia de `XMMATRIX` .<br/>                                                                          |
| [_ *( \* = operador* )*](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-mult-assign)<br/> | Realiza una multiplicación de matriz de la instancia actual de `XMMATRIX` por otra instancia de `XMMATRIX` y devuelve una referencia a la instancia actual, que se ha actualizado. <br/> |
| [**operador =**](/windows/win32/api/directxmath/nf-directxmath-xmmatrix-operator-assign)<br/>      | Asigna los datos de la matriz de una instancia de `XMMATRIX` a la instancia actual de `XMMATRIX` y devuelve un `reference` a la instancia actual. <br/>                                    |
| [**operador ()**](xmmatrix-operator-parens.md)<br/> | Obtiene acceso a elementos de matriz concretos a los que hace referencia la fila y la columna de la instancia actual de `XMMATRIX` . <br/>                                                                           |
