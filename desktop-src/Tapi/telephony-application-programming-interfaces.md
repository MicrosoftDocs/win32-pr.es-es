# <a name="telephony-application-programming-interfaces"></a>Interfaces de programación de aplicaciones de telefonía

Las interfaces de programación de aplicaciones de telefonía de Microsoft admiten el desarrollo de aplicaciones de comunicaciones para Microsoft Windows. En la tabla siguiente se enumeran las interfaces de telefonía.



| Interfaz                                                  | Descripción                                                                                                                                                                                                                                                                                                                                             |
|------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [TAPI 2. x](./tapi-2-2-start-page.md)               | Una API basada en lenguaje de programación C que le permite implementar aplicaciones de comunicaciones que van desde el control básico de módem hasta centros de llamadas con varios agentes y conmutadores.                                                                                                                                                                        |
| [TAPI 3. x](tapi-3-1-start-page.md)                        | Una API basada en COM que combina la telefonía clásica y la IP.                                                                                                                                                                                                                                                                                                   |
| [TSPI](./telephony-service-providers-start-page.md) | Un proveedor de servicios de telefonía (TSP) es una biblioteca de vínculos dinámicos (DLL) que admite el control de los dispositivos de comunicaciones a través de un conjunto de funciones de servicio exportadas. Una aplicación TAPI usa comandos estandarizados, TAPI pasa información al proveedor de servicios de telefonía y el TSP controla los comandos específicos que se deben intercambiar con el dispositivo. |
| [MSPI](media-service-providers-start-page.md)             | Un proveedor de servicios multimedia (MSP) permite que una aplicación tenga un control considerable sobre los medios de un mecanismo de transporte determinado. Un MSP siempre se empareja con un proveedor de servicios de telefonía (TSP).                                                                                                                                                         |
