# <a name="ole-moniker-implementations"></a>Implementaciones de moniker OLE

OLE proporciona implementaciones de varios monikers para diferentes situaciones, tal y como se describe en los temas siguientes:

-   [Monikers de archivo](file-monikers.md)
-   [Monikers compuestos](composite-monikers.md)
-   [Monikers de elemento](item-monikers.md)
-   [Anti-monikers](anti-monikers.md)
-   [Monikers de puntero](pointer-monikers.md)
-   [Monikers de clase](class-monikers.md)
-   [Monikers asincrónicos](asynchronous-monikers.md)
-   [Monikers de URL](url-monikers.md)

Los monikers de archivo, compuesto y elemento son los monikers usados con mayor frecuencia, ya que se pueden usar para hacer prácticamente cualquier objeto en cualquier ubicación. Los moniker y moniker de puntero se utilizan principalmente dentro de OLE, pero tienen alguna aplicación en la implementación de monikers personalizados.
