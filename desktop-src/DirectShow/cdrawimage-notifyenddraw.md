# <a name="cdrawimagenotifyenddraw-method"></a>Método CDrawImage.NotifyEndDraw

No `NotifyEndDraw` se admite el método .

## <a name="syntax"></a>Sintaxis


```C++
void NotifyEndDraw();
```



## <a name="parameters"></a>Parámetros
