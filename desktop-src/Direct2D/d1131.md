# <a name="d1131-dxgi-format-not-gdi-compatible"></a>D1131: formato de DXGI no compatible con GDI

Se pasó una opción compatible con GDI a esta API, pero el formato de DXGI no es compatible con GDI.






 

## <a name="possible-causes"></a>Causas posibles

El uso de la \_ marca GDI compatible con las opciones de mapa de bits D2D1 \_ durante la creación de mapas de \_ \_ bits habilita la compatibilidad del mapa de bits con la representación de GDI, pero introduce restricciones adicionales en el formato de superficie del mapa de bits. En general, la representación de GDI es compatible con los formatos de DXGI \_ format \_ B8G8R8A8 \_ UNORM y dxgi \_ format \_ B8G8R8A8 \_ UNORM \_ sRGB.

## <a name="possible-fixes"></a>Posibles correcciones

Para solucionar este problema, compruebe que el formato especificado para un mapa de bits compatible con GDI sea un formato de BGRA de cuatro canales como se especificó anteriormente. Si el mapa de bits no requiere interoperabilidad con GDI, omita el \_ especificador compatible con GDI.
