# <a name="id3dxkeyframedanimationsetgetnumtranslationkeys-method"></a>Método ID3DXKeyframedAnimationSet::GetNumTranslationKeys

Obtiene el número de claves de traducción en la animación de fotograma clave especificada.

## <a name="syntax"></a>Sintaxis


```C++
UINT GetNumTranslationKeys(
  [in] UINT Animation
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*Animación* \[ En\]
</dt> <dd>

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

Índice de animación.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **UINT**](../winprog/windows-data-types.md)**

Número de claves de traducción en la animación de fotograma clave especificada.
