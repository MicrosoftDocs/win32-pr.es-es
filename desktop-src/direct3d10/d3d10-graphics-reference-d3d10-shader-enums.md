# <a name="shader-enumerations-direct3d-10-graphics"></a>Enumeraciones de sombreador (Direct3D 10 Graphics)

Esta sección contiene información sobre las siguientes enumeraciones de sombreador:



| Enumeración                                                             | Descripción                                                                               |
|-------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [**D3D10 \_ CBUFFER \_ TYPE**](/previous-versions/windows/desktop/legacy/bb204899(v=vs.85))                      | Estas marcas identifican los tipos de datos constant-buffer-data.                                          |
| [**D3D \_ INCLUDE \_ TYPE**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_include_type)                      | Estas marcas identifican la ubicación de un archivo de include.                                     |
| [**PRIMITIVA \_ D3D**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_primitive)                             | Estas marcas identifican los tipos primitivos para un dispositivo Direct3D 10.1.                          |
| [**D3D10 \_ SHADER \_ DEBUG \_ REGTYPE**](/windows/win32/api/D3D10_1Shader/ne-d3d10_1shader-d3d10_shader_debug_regtype)     | Tipos de registro de sombreador.                                                                    |
| [**D3D10 \_ SHADER \_ DEBUG \_ SCOPETYPE**](/windows/win32/api/d3d10_1shader/ne-d3d10_1shader-d3d10_shader_debug_scopetype) | Tipos de ámbito.                                                                              |
| [**D3D \_ NAME**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_name)                                       | Estas marcas identifican los parámetros del sombreador que usan semántica de valores del sistema.                   |
| [**MARCAS DE \_ \_ CBUFFER DEL SOMBREADOR D3D \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_cbuffer_flags)     | Estas marcas identifican las propiedades de búfer constante.                                          |
| [**MARCAS DE ENTRADA DEL \_ \_ SOMBREADOR D3D \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_input_flags)         | Estas marcas identifican las propiedades de entrada del sombreador.                                             |
| [**TIPO DE ENTRADA DEL SOMBREADOR D3D \_ \_ \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_input_type)           | Estas marcas identifican un tipo de recurso de sombreador.                                              |
| [**D3D \_ SHADER \_ VARIABLE \_ (CLASE)**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_variable_class)   | Estas marcas identifican la clase shader-variable.                                           |
| [**MARCAS DE VARIABLES DE \_ \_ SOMBREADOR D3D \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_variable_flags)   | Estas marcas identifican una propiedad de variable de sombreador.                                          |
| [**TIPO DE VARIABLE DE \_ \_ SOMBREADOR D3D \_**](/windows/win32/api/d3dcommon/ne-d3dcommon-d3d_shader_variable_type)     | Estas marcas identifican el tipo de variable de sombreador.                                            |
| [**D3D10 \_ SRV \_ DIMENSION**](/previous-versions/windows/desktop/legacy/bb172452(v=vs.85))                    | Estas marcas identifican el tipo de recurso que se verá para un dispositivo Direct3D 10.0. |
| [**D3D10 \_ SRV \_ DIMENSION1**](/previous-versions/windows/desktop/legacy/bb694535(v=vs.85))                  | Estas marcas identifican el tipo de recurso que se verá para un dispositivo Direct3D 10.1. |
