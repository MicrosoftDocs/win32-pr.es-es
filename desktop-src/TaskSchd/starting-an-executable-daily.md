# <a name="starting-an-executable-daily"></a>Iniciar un archivo ejecutable diariamente

La escritura de una tarea que inicia un ejecutable diariamente se realiza definiendo un desencadenador diario y una acción ejecutable.

## <a name="daily-triggers"></a>Desencadenadores diarios

Los desencadenadores diarios usan su límite de inicio para activar el desencadenador y especificar la hora del día en que se ejecuta la tarea. Una vez que se activa el desencadenador, el intervalo de desencadenadores se usa para indicar si la tarea se ejecuta diariamente, cada día, cada tres días, etc.
