# <a name="debug-help-library"></a>Depurar biblioteca de ayuda

En esta información general se describe el conjunto de funciones proporcionado por la biblioteca de ayuda de depuración, DbgHelp. Contiene un conjunto de rutinas de soporte de depuración que permiten trabajar con imágenes ejecutables en el formato portable ejecutable (PE).

La documentación de DbgHelp es la siguiente:

- [Acerca de DbgHelp](about-dbghelp.md)
- [Usar DbgHelp](using-dbghelp.md)
- [Referencia de DbgHelp](dbghelp-reference.md)

Para obtener la versión más reciente de DbgHelp.dll, vaya a [https://developer.microsoft.com/windows/downloads/windows-10-sdk](https://developer.microsoft.com/windows/downloads/windows-10-sdk) y descargue herramientas de depuración para Windows.

Para obtener una descripción del formato PE, descargue la especificación de la siguiente ubicación: [https://www.microsoft.com/whdc/system/platform/firmware/PECOFF.mspx](https://www.microsoft.com/whdc/system/platform/firmware/PECOFF.mspx) .

Para obtener información sobre cómo examinar la información encontrada en los archivos. pdb, vea el [Debug interface Access SDK](/visualstudio/debugger/debug-interface-access/debug-interface-access-sdk?view=vs-2015).
