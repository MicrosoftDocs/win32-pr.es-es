# <a name="creating-the-skin-definition-file"></a>Crear el archivo de definición de máscara

El archivo de definición de máscara es un archivo de texto simple. Puede crear archivos de definición de máscara en cualquier editor de texto que guarde archivos de texto sin formato ASCII, como Microsoft Notepad, Microsoft Visual InterDev® o Microsoft Visual Studio. Dado que va a escribir XML, no HTML, es posible que no desee usar un editor para HTML porque puede Agregar códigos adicionales que no desea o puede indicarle los errores que no tiene. Probablemente querrá evitar procesadores de texto como Microsoft Word; Aunque Word puede guardar archivos de texto sin formato, es posible que no desee recordar guardar como texto cada vez que guarde el archivo.

Las secciones siguientes le guiarán en la creación de un archivo de definición de máscara:

-   [Empezar con el tema y la vista](start-with-theme-and-view.md)
-   [Agregar BUTTONGROUP](adding-buttongroup.md)
-   [Agregar el BUTTONELEMENT de juego](adding-the-play-buttonelement.md)
-   [Agregando el cerrar BUTTONELEMENT](adding-the-close-buttonelement.md)
-   [Código completo para la máscara simple](complete-code-for-simple-skin.md)
