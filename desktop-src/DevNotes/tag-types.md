# <a name="tag-types"></a>Tipos de etiquetas

Describe el tipo de datos del valor asociado a una etiqueta.



| Constante o valor                                                                                                                                                                                                                            | Descripción                                           |
|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:------------------------------------------------------|
| <span id="TAG_TYPE_NULL"></span><span id="tag_type_null"></span><dl> <dt>**Etiqueta \_ de TIPO \_ null**</dt> <dt>0x1000</dt> </dl>                | No hay ningún valor asociado a la etiqueta.<br/> |
| <span id="TAG_TYPE_BYTE"></span><span id="tag_type_byte"></span><dl> <dt>**Etiqueta \_ de TIPO \_ byte**</dt> <dt>0x2000</dt> </dl>                | Valor de **byte** .<br/>                          |
| <span id="TAG_TYPE_WORD"></span><span id="tag_type_word"></span><dl> <dt>**Etiqueta \_ de ESCRIBIR \_ palabra**</dt> <dt>0x3000</dt> </dl>                | Valor de **palabra** .<br/>                          |
| <span id="TAG_TYPE_DWORD"></span><span id="tag_type_dword"></span><dl> <dt>**Etiqueta \_ de Escriba \_ DWORD**</dt> <dt>0x4000</dt> </dl>             | Valor **DWORD** .<br/>                         |
| <span id="TAG_TYPE_QWORD"></span><span id="tag_type_qword"></span><dl> <dt>**Etiqueta \_ de Escriba \_ QWord**</dt> <dt>0x5000</dt> </dl>             | Valor de **ULONGLONG** .<br/>                     |
| <span id="TAG_TYPE_STRINGREF"></span><span id="tag_type_stringref"></span><dl> <dt>**Etiqueta \_ de Escriba \_ STRINGREF**</dt> <dt>0x6000</dt> </dl> | Valor de cadena con tokens.<br/>                  |
| <span id="TAG_TYPE_LIST"></span><span id="tag_type_list"></span><dl> <dt>**Etiqueta \_ de TYPE \_ List**</dt> <dt>0x7000</dt> </dl>                | Una lista de valores de etiqueta.<br/>                      |
| <span id="TAG_TYPE_STRING"></span><span id="tag_type_string"></span><dl> <dt>**Etiqueta \_ de TIPO \_ String**</dt> <dt>0x8000</dt> </dl>          | Valor de cadena.<br/>                            |
| <span id="TAG_TYPE_BINARY"></span><span id="tag_type_binary"></span><dl> <dt>**Etiqueta \_ de TIPO \_ binario**</dt> <dt>0x9000</dt> </dl>          | Valor binario.<br/>                            |
