# <a name="vml-h-element"></a>Elemento VML H

En este tema se describe VML, una característica que está en desuso a partir Windows Internet Explorer 9. Las páginas web y las aplicaciones que se basan en VML deben migrarse a SVG u otros estándares ampliamente admitidos.

> [!Note]  
> A partir de diciembre de 2011, este tema se archivó. Como resultado, ya no se mantiene activamente. Para obtener más información, vea [Contenido archivado.](/previous-versions/windows/internet-explorer/ie-developer/) Para obtener información, recomendaciones e instrucciones sobre la versión actual de Windows Internet Explorer, [vea Internet Explorer Developer Center](https://msdn.microsoft.com/ie/).

 

Define un identificador para una forma.

Los atributos siguientes modifican un identificador.



| Atributo                                                | Descripción                                                    |
|----------------------------------------------------------|----------------------------------------------------------------|
| [InvX](msdn-online-vml-invx-attribute.md)               | Determina si se invierte la posición *x* del identificador. |
| [InvY](msdn-online-vml-invy-attribute.md)               | Determina si se invierte la posición *y* del identificador. |
| [Map](msdn-online-vml-map-attribute.md)                 | Define el intervalo de asignación de un identificador.                         |
| [Polar](msdn-online-vml-polar-attribute.md)             | Define la posición central de los controladores polares.                 |
| [Position](position-attribute--h--vml.md)               | Especifica los valores *x* *e y* del identificador.                |
| [RadiusRange](msdn-online-vml-radiusrange-attribute.md) | Define el intervalo de un controlador polar.                           |
| [Switch](msdn-online-vml-switch-attribute.md)           | Determina si se intercambian las direcciones del identificador.          |
| [XRange](msdn-online-vml-xrange-attribute.md)           | Define el *intervalo x* de un identificador.                             |
| [YRange](msdn-online-vml-yrange-attribute.md)           | Define el *intervalo y* de un identificador.                             |



 

**Comentarios:**

Este elemento debe definirse dentro de un [elemento Shape.](shape-element--vml.md)
