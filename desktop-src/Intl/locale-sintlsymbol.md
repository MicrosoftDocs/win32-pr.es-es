# <a name="locale_sintlsymbol"></a>CONFIGURACIÓN REGIONAL \_ SINTLSYMBOL

Tres caracteres del símbolo monetario internacional especificado en ISO 4217, seguidos del carácter que separa esta cadena de la cantidad. El número máximo de caracteres permitido para esta cadena es de nueve, incluido un carácter nulo de terminación.
