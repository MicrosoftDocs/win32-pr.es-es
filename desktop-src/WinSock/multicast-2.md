# <a name="multicast"></a>Multidifusión

Las funciones multipunto de Winsock genéricas admiten multidifusión IP. Sin embargo, los proveedores de transporte TCP/IP que admiten multidifusión también deben proporcionar compatibilidad con multidifusión de estilo de distribución de software de Berkeley (BSD) al admitir las opciones de multidifusión correspondientes. Esto simplifica la porción de las aplicaciones de multidifusión existentes a Windows Sockets 2.
