# <a name="department-attribute"></a>Department (atributo)

Contiene el nombre del Departamento en el que trabaja el usuario.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | department                                  |
| Nombre para mostrar de LDAP | department                                  |
| Tamaño              | \-                                          |
| Actualizar privilegio  | Administrador de dominio o propietario de la cuenta.      |
| Frecuencia de actualización  | Cada vez que el Departamento debe cambiar.    |
| Attribute-Id      | 1.2.840.113556.1.2.141                      |
| System-ID-GUID    | bf96794f-0de6-11d0-a285-00aa003049e2        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
