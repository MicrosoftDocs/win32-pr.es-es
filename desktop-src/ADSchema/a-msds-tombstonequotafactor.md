# <a name="ms-ds-tombstone-quota-factor-attribute"></a>atributo de factor de cuota MS-DS-Extinguite

El factor de porcentaje por el que se debe reducir el recuento de objetos de marcadores de exclusión con el fin de tener en cuenta la cuota.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-DS-desecho-factor de cuota         |
| Nombre para mostrar de LDAP | msDS-TombstoneQuotaFactor            |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1847              |
| System-ID-GUID    | 461744d7-f3b6-45ba-8753-fb9552a5df32 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
