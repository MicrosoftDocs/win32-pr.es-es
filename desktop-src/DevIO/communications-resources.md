# <a name="communications-resources"></a>Recursos de comunicaciones

Un *recurso de comunicaciones* es un dispositivo físico o lógico que proporciona un único flujo de datos asincrónico y bidireccional. Los puertos serie, los puertos paralelos, los equipos de fax y los módems son ejemplos de recursos de comunicaciones. Para cada recurso de comunicaciones, hay un proveedor de servicios, que consta de una biblioteca o un controlador, que permite que las aplicaciones tengan acceso al recurso.

-   [Uso de recursos de comunicaciones](using-communications-resources.md)
-   [Referencia de comunicaciones](communications-reference.md)
