# <a name="ms-authz-resource-condition-attribute"></a>atributo MS-authz-Resource-Condition

En el caso de una regla de acceso central, este atributo es una expresión que identifica el ámbito del recurso de destino al que se aplica la Directiva.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-authz-Resource-Condition                 |
| Nombre para mostrar de LDAP | msAuthz-ResourceCondition                   |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.2153                     |
| System-ID-GUID    | 80997877-f874-4c68-864d-6e508a83bdbd        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
