# <a name="corecpp"></a>Core. CPP

En el componente de proveedor de ejemplo, el ejemplo de código, que se usa para inicializar las propiedades básicas de un objeto genérico, se encuentra en Core. cpp. En la tabla siguiente se enumeran los métodos admitidos.



| Método                                   | Descripción                                                                                              |
|------------------------------------------|----------------------------------------------------------------------------------------------------------|
| **CCoreADsObject::InitializeCoreObject** | Inicialice las propiedades básicas de [**IADs**](/windows/desktop/api/Iads/nn-iads-iads) para este objeto, una comodidad para la creación de objetos. |
| **CCoreADsObject::CCoreADsObject**       | Creador estándar.                                                                                        |
| **CCoreADsObject:: ~ CCoreADsObject**      | Destructor estándar.                                                                                     |
