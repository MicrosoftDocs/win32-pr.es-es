# <a name="windows-media-drm-implemented-interfaces"></a>Interfaces de DRM-Implemented de Windows Media

Las siguientes interfaces no están diseñadas para ser implementadas por proveedores de servicios de terceros, pero se implementan y se llaman internamente mediante Windows Media DRM y Windows Media Administrador de dispositivos. La implementación de estas interfaces impedirá que se procese el contenido protegido con DRM. Están documentados solo por integridad.

-   [**Interfaz IWMDRMDevice**](iwmdrmdevice.md)
-   [**Interfaz IWMDRMDevice2**](iwmdrmdevice2.md)
