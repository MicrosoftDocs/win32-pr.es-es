# <a name="ms-ds-machine-account-quota-attribute"></a>Atributo MS-DS-Machine-Account-Quota

Número de cuentas de equipo que un usuario puede crear en un dominio.



| Entrada | Valor |
|-------------------|--------------------------------------------------|
| CN                | MS-DS-Machine-Account-Quota                      |
| Ldap-Display-Name | ms-DS-MachineAccountQuota                        |
| Size              | 4 bytes                                          |
| Privilegio actualizar  | Administrador de dominio                             |
| Frecuencia de actualización  | Siempre que sea necesario cambiar la cuota de un dominio. |
| Attribute-Id      | 1.2.840.113556.1.4.1411                          |
| System-Id-Guid    | d064fb68-1480-11d3-91c1-0000f87a57d4             |
| Syntax            | [**Enumeración**](s-enumeration.md)             |
