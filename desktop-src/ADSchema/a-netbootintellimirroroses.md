# <a name="netboot-intellimirror-oses-attribute"></a>netboot-IntelliMirror-os (atributo)

El atributo **netboot-IntelliMirror-os** está reservado para uso interno.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | netboot-IntelliMirror-os                  |
| Nombre para mostrar de LDAP | netbootIntelliMirrorOSes                    |
| Tamaño              | \-                                          |
| Actualizar privilegio  | El sistema establece este valor.            |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.857                      |
| System-ID-GUID    | 0738307e-91df-11d1-aebc-0000f80367c1        |
