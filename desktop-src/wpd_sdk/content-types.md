# <a name="content-types"></a>Tipos de contenido

Los dispositivos portátiles de Windows clasifican todos los objetos por tipo de contenido. En las secciones siguientes se enumeran los tipos admitidos.



| Tipo de contenido                                                                              | Descripción                                                                                 |
|-------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------|
| [**\_álbum de \_ imagen del tipo de contenido de WPD \_ \_**](wpd-content-type-image-album.md)                  | Representa una colección de imágenes fijas.                                                    |
| [**tipo de contenido de WPD, \_ \_ \_ memorando**](wpd-content-type-memo.md)                                 | Representa un memorando.                                                                          |
| [**\_álbum de \_ \_ contenido mixto \_ de tipo de contenido de \_ WPD**](wpd-content-type-mixed-content-album.md) | Representa una colección de objetos multimedia mixtos, por ejemplo, archivos de audio, imagen y vídeo. |
| [**\_lista de \_ reproducción de tipo de contenido de WPD \_**](wpd-content-type-playlist.md)                         | Representa una lista de reproducción.                                                                      |
| [**tipo de contenido de WPD \_ \_ \_ todo**](wpd-content-type-all.md)                                   | Representa todos los tipos de objeto.                                                                |
| [**\_cita de \_ tipo de contenido de WPD \_**](wpd-content-type-appointment.md)                   | Representa una cita en un calendario.                                                    |
| [**tipo de contenido de WPD \_ \_ \_ álbum de audio \_**](wpd-content-type-audio-album.md)                  | Representa una colección de archivos de audio.                                                     |
| [**tipo de contenido de WPD ( \_ \_ \_ audio)**](wpd-content-type-audio.md)                               | Representa un archivo de audio, como un archivo Windows Media Audio (WMA) o MP3.                  |
| [**\_calendario de \_ tipo de contenido de WPD \_**](wpd-content-type-calendar.md)                         | Representa un calendario.                                                                      |
| [**\_certificado de \_ tipo de contenido de WPD \_**](wpd-content-type-certificate.md)                   | Representa un certificado utilizado para la autenticación.                                           |
| [**\_contacto del \_ tipo de contenido de WPD \_**](wpd-content-type-contact.md)                           | Representa datos de contacto personales.                                                           |
| [**\_grupo de \_ contactos de tipo de contenido de WPD \_ \_**](wpd-content-type-contact-group.md)              | Representa un grupo de contactos.                                                             |
| [**\_documento de \_ tipo de contenido de WPD \_**](wpd-content-type-document.md)                         | Representa un documento.                                                                      |
| [**\_ \_ correo electrónico de tipo de contenido de WPD \_**](wpd-content-type-email.md)                               | Representa un correo electrónico.                                                                       |
| [**\_carpeta de \_ tipo de contenido de WPD \_**](wpd-content-type-folder.md)                             | Representa una carpeta.                                                                        |
| [**\_ \_ objeto funcional de tipo de contenido de \_ WPD \_**](wpd-content-type-functional-object.md)      | Representa un objeto funcional.                                                             |

| [**\_ \_ mensaje genérico de tipo de contenido de \_ WPD \_**](wpd-content-type-generic-message.md)          | Representa un mensaje (SMS, correo electrónico, etc.).                                              |
| [**\_conversión de \_ multimedia de tipo de contenido de WPD \_ \_**](wpd-content-type-media-cast.md)                    | Representa una conversión de multimedia.                                                                    |
| [**\_imagen de \_ tipo de contenido de WPD \_**](wpd-content-type-image.md)                               | Representa una imagen.                                                                        |
| [**\_Asociación de \_ red del tipo de contenido de WPD \_ \_**](wpd-content-type-network-association.md)  | Representa una asociación entre un host y un dispositivo.                                      |

| [**tipo de contenido de WPD \_ \_ \_ álbum de vídeo \_**](wpd-content-type-video-album.md)                  | Representa una colección de archivos de vídeo.                                                     |
| [**vídeo sobre el tipo de contenido de WPD \_ \_ \_**](wpd-content-type-video.md)                               | Representa un vídeo.                                                                         |
| [**\_ \_ perfil inalámbrico de tipo de contenido de \_ WPD \_**](wpd-content-type-wireless-profile.md)        | Representa la información que se usa para tener acceso a una red inalámbrica.                                   |
