# <a name="windows-media-format-sdk-programming-guide"></a>Guía de programación del SDK de Windows Media Format

La guía de programación está pensada para ayudarle a aprender a usar el kit de desarrollo de software (SDK) de Microsoft Windows Media Format para crear aplicaciones que funcionen con archivos con el formato de sistemas avanzados (ASF).

Puede usar el SDK de Windows Media Format para crear aplicaciones que escriban secuencias multimedia digitales y lean secuencias de archivos ASF. Este SDK también admite la edición de metadatos en archivos ASF. Además, este SDK se puede usar para leer y manipular metadatos en archivos MP3.

En las secciones siguientes se explica con detalle los pasos necesarios para escribir, leer y editar archivos ASF mediante este SDK.
