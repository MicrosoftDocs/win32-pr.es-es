# <a name="supporting-schema-versions"></a>Versiones de esquema compatibles

Este tema no está actualizado. Para obtener la información más reciente, consulte la [especificación del esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

El marco de trabajo del esquema de impresión está sujeto a cambios en el futuro a medida que surgen nuevas necesidades. Se espera que estos cambios sean muy poco frecuentes, ya que el cambio más común es la introducción de nuevos nombres de instancia. Este cambio no afecta a la estructura del marco de trabajo y debe ser transparente para los clientes y los proveedores. Cualquier cambio en la estructura y la organización de la plataforma del esquema de impresión se identificará mediante incrementos hasta su número de versión. No se identificarán las adiciones a las palabras clave del esquema de impresión. Los proveedores de PrintTicket deben ser capaces de admitir la versión actual del marco del esquema de impresión, así como cualquier versión anterior. La versión del marco de esquema de impresión se identifica mediante el atributo XML: versión que aparece en el elemento raíz del PrintTicket.
