# <a name="equalizersettingssplinetension"></a>EQUALIZERSETTINGS.splineTension

El atributo **splineTension** especifica o recupera la tensión spline para el control de ecualizador.

``` syntax
        elementID.splineTension
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **número** de lectura/escritura (**float**) que va desde 0,0 (sin tensión) hasta 10,0 (tensión completa) con un valor predeterminado de 3,0.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------|
| Versión<br/> | Windows Media Player 9 series o posterior<br/> |
