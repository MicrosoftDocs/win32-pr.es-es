# <a name="benefits-of-using-active-directory-service-interfaces"></a>Ventajas del uso de interfaces de servicio de Active Directory

ADSI ofrece muchas ventajas a los administradores del sistema, como se describe en la tabla siguiente.



| Característica                   | Ventaja                                                                                                                                                                                                                                                                                                                             |
|---------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Abrir                      | Cualquier proveedor de directorio puede implementar un proveedor ADSI; los usuarios obtienen libertad de elección en los servicios de directorio sin sacrificar la capacidad de administración.                                                                                                                                                                                        |
| Independiente de DS            | Las aplicaciones administrativas no están estrechamente enlazadas a un servicio de directorio de un proveedor determinado. Las aplicaciones administrativas y otras aplicaciones habilitadas para el directorio pueden desarrollarse sin necesidad de comprender las API de directorio específicas del proveedor. La misma aplicación puede trabajar en varios directorios. Se reduce el tiempo de desarrollo y los costos de soporte técnico. |
| Compatibilidad con varios idiomas | Los objetos ADSI proporcionan acceso sencillo a los servicios de directorio a través del modelo de objetos componentes. Las aplicaciones COM se pueden escribir en lenguajes como Microsoft Visual Basic, Microsoft Visual Basic Scripting Edition (VBScript) y C/C++.                                                                                             |
| Modelo de programación simple  | ADSI consta de un conjunto de interfaces pequeño y fácil de aprender.                                                                                                                                                                                                                                                                          |
| Scripts                | Cualquier lenguaje compatible con automatización (por ejemplo, Visual Basic, VBScript, Perl, REXX y otros) se puede usar para desarrollar aplicaciones de servicios de directorio. Los administradores y desarrolladores pueden usar las herramientas que ya conocen. Se ha mejorado la productividad: se reducen el tiempo de desarrollo y los costos de soporte técnico.                               |
| Funcionalmente enriquecida         | Los ISV y los usuarios finales sofisticados pueden desarrollar aplicaciones serias con los mismos modelos ADSI que se usan para las aplicaciones administrativas sencillas con scripts.                                                                                                                                                                         |
| Extensible                | Los proveedores de directorios, los ISV y los usuarios finales pueden extender ADSI con nuevos objetos y funciones para agregar valores o satisfacer necesidades únicas.                                                                                                                                                                                                          |
| Reconocimiento de OLE DB              | ADSI proporciona una interfaz OLE DB para que los programadores familiarizados con la programación de bases de datos a través de OLE DB puedan ser productivos rápidamente.                                                                                                                                                                                                  |
