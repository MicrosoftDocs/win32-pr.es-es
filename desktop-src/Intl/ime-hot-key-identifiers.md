# <a name="ime-hot-key-identifiers"></a>Identificadores de tecla de acceso rápido de IME

Estos identificadores se usan con la función [**ImmSimulateHotKey**](/windows/desktop/api/Imm/nf-imm-immsimulatehotkey) .



| Identificadores                       | Significado                                                                                                                                                                                                 |
|-----------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| \_ \_ alternar IME CHOTKEY IME \_ NONIME \_ | (Chino Simplificado) Alterne entre la operación de IME y no IME cuando el idioma sea chino simplificado.                                                                                                  |
| \_alternancia de \_ forma de CHOTKEY de IME \_       | (Chino Simplificado) Alternar el modo de conversión de forma de IME.                                                                                                                                           |
| \_alternancia de \_ símbolo de CHOTKEY de IME \_      | (Chino Simplificado) Alterne el modo de conversión de símbolos de IME. El modo de símbolo indica que el usuario puede escribir símbolos y signos de puntuación de chino mediante la asignación a la puntuación y los símbolos del teclado. |
| IME \_ ITHOTKEY \_ RECONVERTSTRING    | **Windows Me/98, windows 2000, Windows XP:** (chino tradicional) reconversión de desencadenador.                                                                                                                |
| \_cerrar JHOTKEY de IME \_ \_         | Textos Abra y cierre el IME de forma alternativa.                                                                                                                                                          |
| IME \_ KHOTKEY \_ Inglés             | Idiomas Cambie a Inglés.                                                                                                                                                                             |
| \_alternancia de \_ forma de KHOTKEY de IME \_       | Idiomas Alternar el modo de conversión de forma de IME.                                                                                                                                                       |
| IME \_ KHOTKEY \_ HANJACONVERT        | Idiomas Cambiar a conversión hanja.                                                                                                                                                                    |
| \_ \_ alternar IME THOTKEY IME \_ NONIME \_ | (Chino tradicional) Alterne entre la operación de IME y no IME cuando el idioma sea el chino tradicional.                                                                                                |
| \_alternancia de \_ forma de THOTKEY de IME \_       | (Chino tradicional) Alternar el modo de conversión de forma de IME.                                                                                                                                          |
| \_alternancia de \_ símbolo de THOTKEY de IME \_      | (Chino tradicional) Alterne el modo de conversión de símbolos de IME.                                                                                                                                         |
