# <a name="ds-query-self-quota-extended-right"></a>Derecho extendido DS-Query-Self-Quota

| Entrada | Valor |
|--------------|--------------------------------------|
| CN           | DS-Query-Self-Quota                  |
| Display-Name | Cuota de consulta propia                     |
| Rights-GUID  | 4ecc03fe-ffc0-4947-b630-eb672a8a9dbc |
