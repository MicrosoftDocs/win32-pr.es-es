# <a name="boundingrectisincorrect"></a>BoundingRectIsIncorrect

## <a name="text"></a>Texto

El {0} borde del rectángulo indicado es incoherente con la prueba de posicionamiento {1} del Rect en el punto {2} .

## <a name="type"></a>Tipo

Error

## <a name="description"></a>Descripción

El rectángulo delimitador del elemento no es coherente con los resultados esperados de la prueba de posicionamiento. Por ejemplo, la prueba de posicionamiento de un punto fuera del rectángulo delimitador del elemento no devuelve un elemento diferente.
