# <a name="wmvideoframerate"></a>WM/VideoFrameRate

El atributo **WM/VideoFrameRate** contiene la velocidad, en fotogramas por segundo, del vídeo de una secuencia.

## <a name="global-constant"></a>Constante global

g \_ wszWMVideoFrameRate

## <a name="data-type"></a>Tipo de datos

**tipo de WMT \_ \_ DWORD**
