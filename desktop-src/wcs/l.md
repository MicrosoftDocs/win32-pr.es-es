# <a name="l"></a>L

## <a name="lightness"></a>luminosidad

Un canal de color en el espacio de colores HLS. Es una medida del brillo relativo del color. Aumentar la luminosidad produce matices de un matiz. La disminución de la luminosidad produce tonalidades de un matiz.
