# <a name="left-property-commandswindow-object"></a>Propiedad Left (objeto CommandsWindow)

\[Microsoft Agent está en desuso a partir Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Descripción**
</dt> <dd>

Devuelve un valor entero que especifica el borde izquierdo, en píxeles, de la ventana Comandos de voz.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintaxis**
</dt> <dd>

*agent!". ComandosWindow.Left**

</dd> </dl>

## <a name="remarks"></a>Comentarios

El servidor muestra la ventana Comandos de voz en función de la posición y el tamaño establecidos por el usuario.
