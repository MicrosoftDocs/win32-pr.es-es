# <a name="prov_fortezza"></a>Fortezza de PROV \_

El \_ tipo de proveedor de Fortezza de Prov contiene un conjunto de algoritmos y protocolos criptográficos que pertenecen al [*National Institute of Standards and Technology*](../secgloss/n-gly.md) (NIST).

## <a name="algorithms-supported"></a>Algoritmos admitidos

Para obtener descripciones de cada uno de estos algoritmos, vea el glosario.



| Propósito      | Algoritmos admitidos                                                                                      |
|--------------|-----------------------------------------------------------------------------------------------------------|
| Intercambio de claves | [*KEA*](../secgloss/k-gly.md)         |
| Firma    | [*DSS*](../secgloss/d-gly.md) |
| Cifrado   | [*Skipjack*](../secgloss/s-gly.md)                                |
| Aplicación de algoritmo hash      | [*SHA*](../secgloss/s-gly.md)           |

Hay más información disponible directamente del [*National Institute of Standards and Technology*](../secgloss/n-gly.md) (NIST).
