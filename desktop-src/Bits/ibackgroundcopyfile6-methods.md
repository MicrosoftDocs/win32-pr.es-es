# <a name="ibackgroundcopyfile6-methods"></a>Métodos IBackgroundCopyFile6

La interfaz [**IBackgroundCopyFile6**](/windows/desktop/api/bits10_1/nn-bits10_1-ibackgroundcopyfile6) expone los métodos siguientes.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                           | Descripción                                                                   |
|-------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------|
| [**Método UpdateDownloadPosition**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-updatedownloadposition)<br/> | Especifica una posición para dar prioridad a la descarga de los datos que faltan de. <br/> |
| [**Método RequestFileRanges**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-requestfileranges)<br/>           | Agrega un nuevo conjunto de intervalos de archivos a los que se va a dar prioridad para su descarga. <br/>     |
| [**Método GetFilledFileRanges**](/windows/desktop/api/Bits10_1/nf-bits10_1-ibackgroundcopyfile6-getfilledfileranges)<br/>       | Devuelve el conjunto de intervalos de archivos que se han descargado.<br/>          |
