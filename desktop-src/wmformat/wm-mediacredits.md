# <a name="wmmediacredits"></a>WM/MediaCredits

El atributo **WM/MediaCredits** contiene los créditos de los medios de difusión.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaCredits

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Este atributo se usa para difundir televisión o radio.
