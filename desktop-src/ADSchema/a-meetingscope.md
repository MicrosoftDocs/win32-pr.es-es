# <a name="meetingscope-attribute"></a>atributo meetingScope

Puede ser global, local o cualquier otro valor. Su interpretación es específica de la aplicación.

| CN                | meetingScope                                |
| Ldap-Display-Name | meetingScope                                |
| Size              | \-                                          |
| Privilegio actualizar  | \-                                          |
