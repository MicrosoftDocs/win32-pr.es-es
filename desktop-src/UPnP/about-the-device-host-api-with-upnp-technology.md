# <a name="about-the-device-host-api"></a>Acerca de device host API

Device Host API con tecnología UPnP es un marco para implementar la funcionalidad de dispositivo basada en UPnP en Windows plataforma. Los desarrolladores que crean dispositivos mediante device host API con [](h-gly.md)la tecnología UPnP (denominados dispositivos hospedados) solo necesitan implementar la funcionalidad principal del dispositivo. Los desarrolladores pueden confiar en el host del dispositivo para controlar los detalles específicos de UPnP de detección, descripción, control, eventos y presentación. El host de dispositivo valida los datos entrantes de los clientes basados en UPnP y da formato a todos los datos salientes de los dispositivos hospedados según la arquitectura del dispositivo UPnP.

En las secciones siguientes se explica, en general, cómo funciona el host de dispositivos UPnP:

-   [Registro de un dispositivo hospedado con el host de dispositivo](registering-a-hosted-device-with-the-device-host.md)
-   [Proveedores de dispositivos](device-providers.md)
-   [Eventos](eventing.md)
-   [Presentación](presentation.md)
