# <a name="ms-sql-memory-attribute"></a>Atributo MS-SQL-Memory

Cantidad de memoria del equipo.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-SQL-memoria                        |
| Nombre para mostrar de LDAP | mS-SQL-memoria                        |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | Al iniciar el sistema.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1367              |
| System-ID-GUID    | 5b5d448c-ccee-11d2-9993-0000f87a57d4 |
| Sintaxis            | [**Interval**](s-interval.md)       |
