# <a name="id3dx10dataprocessordestroy-method"></a>ID3DX10DataProcessor::D método estroy

Lo usa una [**interfaz ID3DX10ThreadPump**](id3dx10threadpump.md) para destruir el procesador después de que se complete un elemento de trabajo.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT Destroy();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **HRESULT**](https://msdn.microsoft.com/library/Bb401631(v=MSDN.10).aspx)**

El valor devuelto es uno de los valores que aparecen en los [códigos de retorno de Direct3D 10](d3d10-graphics-reference-returnvalues.md).
