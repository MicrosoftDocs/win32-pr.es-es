# <a name="programming-the-microsoft-agent-control"></a>Programar el control de Microsoft Agent

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

Aunque las aplicaciones pueden escribir directamente en los servicios del agente de Microsoft mediante sus interfaces de servidor de automatización, Microsoft Agent también incluye un control ActiveX (OLE). El control admite la programación sencilla mediante un lenguaje de scripting como Microsoft Visual Basic Scripting Edition (VBScript) u otros lenguajes que admiten la interfaz de control ActiveX.
