# <a name="third-party-msp"></a>MSP de terceros

Normalmente, el hardware de comunicaciones especializado llega con uno o varios TSP y MSP específicos del dispositivo. La documentación del dispositivo debe indicar las funcionalidades y proporcionar directrices de programación.
