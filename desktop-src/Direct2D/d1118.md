# <a name="d1118-target-not-a-bitmap"></a>D1118: el destino no es un mapa de bits

No se pudo realizar una operación de dibujo porque el destino es NULL o una lista de comandos.



|             |         |
|-------------|---------|
| Nivel de error | Advertencia |



 

## <a name="possible-causes"></a>Causas posibles

Una causa probable de este mensaje es que la aplicación llamó a GetDC o ReleaseDC mientras el destino del contexto del dispositivo se estableció en NULL, o en una lista de comandos.

## <a name="possible-fixes"></a>Posibles correcciones

Si una aplicación usa GetDC o ReleaseDC expuesto por IGdiInteropRenderTarget, asegúrese de que la ID2D1DeviceContext correspondiente tiene un conjunto de mapas de bits válido como destino.
