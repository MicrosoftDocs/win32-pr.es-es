# <a name="reset-method-of-the-msvm_synthetickeyboard-class"></a>Método Reset de la \_ clase MSVM SyntheticKeyboard

Restablece el teclado virtual.

## <a name="syntax"></a>Sintaxis


```mof
uint32 Reset();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Si se ejecuta correctamente, devuelve 0; de lo contrario, devuelve un error.

<dl> <dt>

**Completado sin error** (0)
</dt> <dt>

**No compatible** (1)
</dt> </dl>
