# <a name="error-checking-cryptography"></a>Comprobación de errores (criptografía)

En el caso de C++, el valor devuelto siempre es **HRESULT** y indica si la llamada al método se realizó correctamente o no. Si se produce un error en la llamada, el valor devuelto indica por qué se produjo un error.
