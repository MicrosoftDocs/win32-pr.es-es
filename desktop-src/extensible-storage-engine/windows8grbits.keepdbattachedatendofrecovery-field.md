# <a name="windows8grbitskeepdbattachedatendofrecovery-field"></a>Campo Windows8Grbits.KeepDbAttachedAtEndOfRecovery

Permite que la base de datos permanezca adjunta al final de la recuperación (para una transición más rápida al estado de ejecución).

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop.Windows8](./microsoft.isam.esent.interop.windows8-namespace.md)  
**Ensamblado:**  Microsoft.Isam.Esent.Interop (en Microsoft.Isam.Esent.Interop.dll)
