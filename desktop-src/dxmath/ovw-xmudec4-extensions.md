# <a name="xmudec4-extensions"></a>Extensiones de XMUDEC4

[**XMUDEC4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmudec4)describe un vector 4D con x-, y-y los componentes z representados como valores enteros sin signo de 10 bits y el componente w-como un valor entero sin signo de 2 bits.

> [!Note]  
> Las características que se enumeran aquí solo están disponibles cuando se desarrollan con C++.
