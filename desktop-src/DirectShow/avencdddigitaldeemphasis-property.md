# <a name="avencdddigitaldeemphasis-property"></a>Propiedad AVEncDDDigitalDeemphasis

Especifica si la desacentinación digital está habilitada en una secuencia de audio de Dolby Digital. Esta propiedad se aplica a los codificadores de audio Dolby Digital.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncDDigitalDeemphasis**
