# <a name="user-configurable-elements"></a>Elementos configurables por el usuario

Este tema no es actual. Para obtener la información más reciente, vea [La especificación de esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

En esta sección se describen las palabras clave públicas que pueden aparecer en un documento PrintCapabilities o PrintTicket.

No es necesario que se admiten todas las palabras clave y elementos de las palabras clave. Los implementadores de PrintTicket e PrintCapabilities solo deben admitir la funcionalidad si es aplicable al dispositivo o controlador de destino.

## <a name="element-type-implementations-for-user-configurable-elements"></a>Implementaciones de tipos de elemento para elementos configurables por el usuario

Es importante prestar atención a los elementos Primario y Secundario permitidos en el contenido estructural de cada palabra clave. Para obtener más información, consulte la [sección Resumen de tipos de](./summary-of-element-types.md) elementos.
