# <a name="avencmpaemphasistype-property"></a>Propiedad AVEncMPAEmphasisType

Especifica el tipo de filtro de desacentración que se debe usar alcoding. Esta propiedad se aplica a los codificadores de audio MPEG.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncMPAEmphasisType**

## <a name="property-value"></a>Valor de propiedad

El valor de esta propiedad es miembro de la [**enumeración eAVEncMPAEmphasisType.**](/windows/win32/api/codecapi/ne-codecapi-eavencmpaemphasistype)

## <a name="remarks"></a>Comentarios

Esta propiedad establece los bits de énfasis en el encabezado del marco.
