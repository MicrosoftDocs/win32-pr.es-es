# <a name="span-idvspixenginepackedcallpkgspanpackedcallpkg-structure"></a><span id="vspixengine.packedcallpkg"></span>PackedCallPkg (estructura)

Representa un paquete de cuatro llamadas.

## <a name="syntax"></a>Sintaxis


```C++
typedef struct PackedCallPkg {
  UINT pcp1;
  UINT pcp2;
  UINT pcp3;
  UINT pcp4;
} PackedCallPkg;
```

## <a name="members"></a>Miembros

**año 1**  
Primera llamada en el paquete.

**lugar2**  
Segunda llamada del paquete.

**gp3**  
Tercera llamada del paquete.

**año 4**  
Cuarta llamada del paquete.
