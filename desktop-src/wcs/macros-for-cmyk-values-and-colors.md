# <a name="macros-for-cmyk-values-and-colors"></a>Macros para valores y colores CMYK

Las macros siguientes son útiles para manipular los valores MANIPULACIÓN.



| Macro                          | Descripción                                                                  |
|--------------------------------|------------------------------------------------------------------------------|
| [**Cmyk**](/windows/desktop/api/Wingdi/nf-wingdi-cmyk)           | Crea un valor CIAN a partir de valores individuales de cian,rillón, amarillo y negro. |
| [**GetCValue**](/windows/desktop/api/Wingdi/nf-wingdi-getcvalue) | Recupera el valor de color de cian de un valor de color CIAN.                      |
| [**GetKValue**](/windows/desktop/api/Wingdi/nf-wingdi-getkvalue) | Recupera el valor de color negro de un valor de color COLOR.                     |
| [**GetMValue**](/windows/desktop/api/Wingdi/nf-wingdi-getmvalue) | Recupera el valor de color después de un valor de color CIAN.                         |
| [**GetYValue**](/windows/desktop/api/Wingdi/nf-wingdi-getyvalue) | Recupera el valor de color amarillo de un valor de color YELLOW.                    |



 

Las macros siguientes se usan con color.



| Macro                                | Descripción                                                                                                                                           |
|--------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**GetBValue**](/windows/win32/api/wingdi/nf-wingdi-getbvalue)       | Obtiene un valor azul RGB.                                                                                                                               |
| [**GetGValue**](/windows/win32/api/wingdi/nf-wingdi-getgvalue)       | Obtiene un valor verde RGB.                                                                                                                              |
| [**GetRValue**](/windows/win32/api/wingdi/nf-wingdi-getrvalue)       | Obtiene un valor rojo RGB.                                                                                                                                |
| [**PALETTEINDEX**](/previous-versions//dd162770(v=vs.85)) | Acepta un índice en una entrada de paleta de colores lógicos y devuelve un especificador de entrada de paleta.                                                              |
| [**PALETTERGB**](/windows/win32/api/wingdi/nf-wingdi-palettergb)     | Acepta tres valores que representan las intensidades relativas de rojo, verde y azul y devuelve un especificador rojo, verde, azul (RGB) relativo a la paleta. |
| [RGB](/windows/win32/api/wingdi/nf-wingdi-rgb)                       | Selecciona un color rojo, verde, azul (RGB) en función de los argumentos proporcionados y las funcionalidades de color del dispositivo de salida.                               |
