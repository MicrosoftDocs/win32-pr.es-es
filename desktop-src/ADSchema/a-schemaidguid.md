# <a name="schema-id-guid-attribute"></a>Atributo Schema-ID-GUID

Identificador único de un atributo.



| Entrada | Value |
|-------------------|---------------------------------------------------------------------|
| CN                | Schema-ID-GUID                                                      |
| Ldap-Display-Name | schemaIDGUID                                                        |
| Size              | 16 bytes                                                            |
| Actualizar privilegios  | El sistema establece este valor.                                    |
| Frecuencia de actualización  | Este valor se establece cuando se crea el objeto y no se puede cambiar. |
| Attribute-Id      | 1.2.840.113556.1.4.148                                              |
| System-Id-Guid    | bf967923-0de6-11d0-a285-00aa003049e2                                |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md)               |
