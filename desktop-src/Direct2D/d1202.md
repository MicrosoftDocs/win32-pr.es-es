# <a name="d1202-copyfromrendertarget-with-multisampled-source"></a>D1202: CopyfromRenderTarget con origen multimuestreado

CopyFromRenderTarget se usó con un rendertarget de muestreo multimuestreado. La llamada se seguirá emitiendo, pero es posible que el resultado copiado no represente el patrón de Multimuestra esperado.






 

## <a name="possible-causes"></a>Causas posibles

La aplicación interopera una textura de Direct3D multimuestreada como un destino de Direct2D y se denomina CopyFromRenderTarget en el destino.

## <a name="possible-fixes"></a>Posibles correcciones

Evite la interoperación de texturas multimuestreadas como destinos si es importante la funcionalidad de CopyFromRenderTarget. Es posible que el resultado copiado no represente el patrón de Multimuestra esperado.
