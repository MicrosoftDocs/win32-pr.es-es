# <a name="ms-dfsr-flags-attribute"></a>atributo MS-DFSR-Flags

Contiene las marcas de objeto del servicio de replicación de Sistema de archivos distribuido (DFS).



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-DFSR-marcas                        |
| Nombre para mostrar de LDAP | msDFSR-marcas                         |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.6.13.3.16           |
| System-ID-GUID    | fe515695-3f61-45c8-9bfa-19c148c57b09 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
