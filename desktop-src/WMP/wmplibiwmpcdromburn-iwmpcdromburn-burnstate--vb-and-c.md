# <a name="iwmpcdromburnburnstate-property"></a>IWMPCdromBurn:: burnState (propiedad)

La propiedad **burnState** obtiene un valor de enumeración que indica el estado actual de la grabación.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```CSharp
public WMPBurnState burnState {get;}
```


```VB

Public ReadOnly Property burnState As WMPBurnState
```





## <a name="property-value"></a>Valor de propiedad

**WMPLib. WMPBurnState** que es un valor de la enumeración **WMPBurnState** que indica el estado actual.
