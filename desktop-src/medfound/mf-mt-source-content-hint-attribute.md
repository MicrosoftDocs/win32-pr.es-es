# <a name="mf_mt_source_content_hint-attribute"></a>\_Atributo de \_ sugerencia de contenido de origen MF MT \_ \_

Describe la relación de aspecto prevista para un tipo de medio de vídeo.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

El valor de este atributo es un miembro de la enumeración [**MFVideoSrcContentHintFlags**](/windows/desktop/api/mfapi/ne-mfapi-mfvideosrccontenthintflags) .

La constante GUID para este atributo se exporta desde mfuuid. lib.
