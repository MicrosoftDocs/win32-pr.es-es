# <a name="print-duplex-supported-attribute"></a>Atributo Print-Duplex-Supported

Indica el tipo de compatibilidad dúplex que tiene una impresora.



| Entrada | Valor |
|-------------------|-------------------------------------------------------|
| CN                | Compatible con dúplex de impresión                                |
| Ldap-Display-Name | printDuplexSupported                                  |
| Size              | 4 bytes. Dúplex admitido = 2. Solo Simplex = 1 o 0. |
| Privilegio actualizar  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1311                               |
| System-Id-Guid    | 281416cc-1968-11d0-a28f-00aa003049e2                  |
| Syntax            | [**Booleana**](s-boolean.md)                          |
