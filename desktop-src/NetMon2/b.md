# <a name="b-network-monitor"></a>B (Monitor de red)

<dl> <dt>

<span id="_netmon_binary_large_object_gly"></span><span id="_NETMON_BINARY_LARGE_OBJECT_GLY"></span>**objeto binario grande (BLOB)**
</dt> <dd>

La estructura de datos genérica de Monitor de red. Puede usar un BLOB para comunicar y almacenar información de configuración y ubicación, y para almacenar información específica de la aplicación, sin que afecte a otros datos en un BLOB.

</dd> <dt>

<span id="_netmon_blob_gly"></span><span id="_NETMON_BLOB_GLY"></span>**BLOB**
</dt> <dd>

Vea objeto binario grande.

</dd> <dt>

<span id="_netmon_buffer_gly"></span><span id="_NETMON_BUFFER_GLY"></span>**búfer**
</dt> <dd>

Una parte reservada de la memoria en la que los datos se conservan temporalmente a la espera de completar una transferencia hacia o desde un dispositivo de almacenamiento u otra ubicación en la memoria.

</dd> </dl>
