|-------------------|---------------------------------------------|
| CN                | RPC-NS-Group                                |
| Nombre para mostrar de LDAP | rpcNsGroup                                  |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.114                      |
| System-ID-GUID    | bf967a24-0de6-11d0-a285-00aa003049e2        |
