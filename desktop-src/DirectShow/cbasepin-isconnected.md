# <a name="cbasepinisconnected-method"></a>CBasePin. IsConnected (método)

El `IsConnected` método determina si el PIN está conectado a otro PIN.

## <a name="syntax"></a>Sintaxis


```C++
BOOL IsConnected();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve **true** si el PIN está conectado. De lo contrario, devuelve **false**.
