# <a name="device-properties-wia"></a>Propiedades del dispositivo (WIA)

Las propiedades del dispositivo son propiedades que controlan la función de todo el dispositivo. Solo se puede tener acceso a estas propiedades a través del elemento raíz del árbol que representa un dispositivo. Hay tres tipos de propiedades de dispositivo: común, escáner y cámara.

Para obtener información de referencia, vea:

-   [**Constantes de propiedad de dispositivo comunes**](-wia-wiaitempropcommondevice.md)
-   [**Constantes de propiedades de dispositivo de cámara**](-wia-wiaitempropcameradevice.md)
-   [**Constantes de propiedades de dispositivo de escáner**](-wia-wiaitempropscannerdevice.md)
