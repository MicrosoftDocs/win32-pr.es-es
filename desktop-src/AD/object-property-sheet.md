# <a name="object-property-sheet"></a>Hoja de propiedades de objeto

En este tema se describe la hoja de propiedades Object, que es común a todas las clases de objeto del Usuarios y equipos de Active Directory complemento.

## <a name="object-property-sheet"></a>Hoja de propiedades de objeto

En la tabla siguiente se muestran las etiquetas de interfaz de usuario de la **hoja de** propiedades Object.



| Etiqueta de interfaz de usuario                              | Atributo en Active Directory Domain Services | Comentarios                                                   |
|---------------------------------------|-----------------------------------------------|------------------------------------------------------------|
| Nombre de dominio completo del objeto |                                               | Este es el nombre distintivo del objeto en forma canónica. |
| Object (clase)                          | objectClass                                   |                                                            |
| Creado                               | whenCreated                                   |                                                            |
| Modificado                              | whenChanged                                   |                                                            |
| Actualizar números de secuencia: Actual      | uSNChanged                                    |                                                            |
| Actualizar números de secuencia: Original     | uSNCreated                                    |                                                            |
