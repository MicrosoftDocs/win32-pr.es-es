# <a name="ms-ts-allow-logon-attribute"></a>atributo MS-TS-allow-Logon

Especifica si el usuario puede iniciar sesión en el Terminal Server. El valor es 1 si se permite el inicio de sesión y 0 si no se permite el inicio de sesión.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-TS-allow-Logon                    |
| Nombre para mostrar de LDAP | msTSAllowLogon                       |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1979              |
| System-ID-GUID    | 3a0cd464-bc54-40e7-93ae-a646a6ecc4b4 |
| Sintaxis            | [**Booleano**](s-boolean.md)         |
