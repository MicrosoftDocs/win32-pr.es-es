# <a name="locale_ipapersize"></a>configuración regional \_ IPAPERSIZE

**Windows 2000:** Tamaño de papel predeterminado asociado a la configuración regional. Normalmente, el tamaño tiene uno de los valores siguientes, aunque se puede establecer en cualquiera de los tamaños de papel definidos por el administrador de trabajos de impresión.



| Value | Significado   |
|-------|-----------|
| 1     | Carta estadounidense |
| 5     | Legal de EE. UU.  |
| 8     | A3        |
| 9     | A4        |
