# <a name="character-method"></a>Character (método)

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Devuelve un objeto de [**carácter**](/windows/desktop/lwef/the-characters-object) en una colección de [**caracteres**](/windows/desktop/lwef/the-characters-object) .

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintáctica**
</dt> <dd>

*agente ***. Characters. Character "*** CharacterID * *"**

</dd> </dl>

## <a name="remarks"></a>Observaciones

Puede utilizar este método para tener acceso a los métodos y propiedades de un objeto de [**carácter**](/windows/desktop/lwef/the-characters-object) .

> [!Note]  
> Este método puede ser necesario para algunos lenguajes de programación que no admiten colecciones. No es necesario para VBScript o Visual Basic. Para obtener más información sobre cómo especificar métodos de [**caracteres**](/windows/desktop/lwef/the-characters-object) , vea métodos de objetos de carácter.
