# <a name="partition-properties"></a>Propiedades de partición

Las particiones tienen cinco propiedades disponibles. Estas propiedades y sus descripciones son las siguientes:

<dl> <dt>

<span id="Partition_Name"></span><span id="partition_name"></span><span id="PARTITION_NAME"></span>Nombre de partición
</dt> <dd>

Cadena de texto asociada a la partición.

</dd> <dt>

<span id="Partition_ID"></span><span id="partition_id"></span><span id="PARTITION_ID"></span>IDENTIFICADOR de partición
</dt> <dd>

GUID que el sistema utiliza para correlacionar un usuario con una partición.

</dd> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>Denominación
</dt> <dd>

Una descripción textual de la partición.

</dd> <dt>

<span id="Disable_Deletion"></span><span id="disable_deletion"></span><span id="DISABLE_DELETION"></span>Deshabilitar eliminación
</dt> <dd>

Mecanismo para prohibir a un usuario la eliminación de una partición.

</dd> <dt>

<span id="Disable_Changes"></span><span id="disable_changes"></span><span id="DISABLE_CHANGES"></span>Deshabilitar cambios
</dt> <dd>

Mecanismo para impedir que un usuario realice cambios en una partición.

</dd> </dl>

Estas propiedades se pueden ver haciendo clic con el botón secundario en el nombre de una partición dentro de la herramienta administrativa Servicios de componentes y eligiendo **propiedades**.

Para obtener más información sobre la propiedad ID. de partición, vea [registrar y activar componentes en particiones](registering-and-activating-components-in-partitions.md).
