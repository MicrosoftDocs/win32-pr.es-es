# <a name="media-locator"></a>Localizador de medios

> [!Note]  
> \[Obsoleto. Esta API puede quitarse de futuras versiones de Windows.\]

 

El objeto Localizador de medios busca los archivos asociados a objetos de origen. Use este objeto para validar orígenes en la escala de tiempo. Para crear este objeto, llame a **CoCreateInstance**. El identificador de clase es CLSID \_ MediaLocator.

El objeto Media Locator expone la siguiente interfaz:

-   [**IMediaLocator**](imedialocator.md)
