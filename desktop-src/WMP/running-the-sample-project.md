# <a name="running-the-sample-project"></a>Ejecución del ejemplo Project

El componente COM de ejemplo muestra la funcionalidad proporcionada por los métodos de **IWMPSubscriptionServices** mostrando un cuadro de diálogo cada vez que Reproductor de Windows Media llama a uno de los métodos. Para ver esto, primero debe agregar contenido a la biblioteca que contiene un atributo **ContentDistributor** que coincida con el servicio. Vea [Agregar el atributo ContentDistributor](adding-the-contentdistributor-attribute.md). A continuación, Reproductor de Windows Media 10 o posterior para abrir la tienda en línea, reproducir el contenido Premium y copiar el contenido Premium en un dispositivo. Reproductor de Windows Media crear una instancia del componente COM de ejemplo y llamar a métodos cuando sea necesario.
