# <a name="mciwnd-window-class"></a>Clase de ventana MCIWnd

MCIWnd es una clase de ventana para controlar los dispositivos multimedia. Una biblioteca de funciones, mensajes y macros asociadas a MCIWnd proporciona un método sencillo para agregar capacidades de reproducción o grabación multimedia a las aplicaciones.
