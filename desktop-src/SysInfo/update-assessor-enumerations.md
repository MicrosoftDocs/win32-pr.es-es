# <a name="update-assessor-enumerations"></a>Actualizar enumeraciones de asesor

El proveedor de descriptores de acceso de actualización usa las siguientes enumeraciones.



| Función                                                 | Descripción                                                                    |
|----------------------------------------------------------|--------------------------------------------------------------------------------|
| [**UpdateAssessmentStatus**](updateassessmentstatus.md) | Describe la actualización del sistema operativo en un dispositivo.                                |
| [**UpdateImpactLevel**](updateimpactlevel.md)           | Indica un impacto alto, medio o bajo de un dispositivo que ejecuta un sistema operativo obsoleto. |
