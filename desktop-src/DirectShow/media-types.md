# <a name="media-types-directshow"></a>Tipos de medios (DirectShow)

Microsoft® DirectShow® usa la estructura de [**\_ \_ tipo de medio am**](/windows/win32/api/strmif/ns-strmif-am_media_type) para describir los ejemplos de medios. Esta estructura incluye miembros **GUID** para el tipo principal, el subtipo y el tipo de formato. En esta sección se resumen los GUID de tipo y subtipo principales definidos para DirectShow. Los filtros de terceros pueden definir sus propios GUID de tipo de medio.

-   [**Tipos principales**](major-types.md)
-   [**Subtipos de audio**](audio-subtypes.md)
-   [**Tipos de medios BDA**](bda-media-types.md)
-   [**Tipos de medios de DVD**](dvd-media-types.md)
-   [**Tipos de medios de línea 21**](line-21-media-types.md)
-   [**Tipos de medios MPEG-1**](mpeg-1-media-types.md)
-   [Tipos de medios MPEG-2](mpeg-2-media-types.md)
-   [**Subtipos de secuencia**](stream-subtypes.md)
-   [Tipos de medios VBI](vbi-media-types.md)
-   [Subtipos de vídeo](video-subtypes.md)
