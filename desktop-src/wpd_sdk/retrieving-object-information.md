# <a name="retrieving-object-information"></a>Recuperar información de objetos

Además de recuperar la información del dispositivo, la API de WPD proporciona un conjunto de interfaces y métodos que se pueden usar para recuperar información de un objeto determinado. En general, la recuperación de la información de objetos se puede dividir en dos categorías, como se describe en la tabla siguiente.



| Datos del dispositivo                                                                                    | Descripción                                                                               |
|------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Identificador de objeto](retrieving-an-object-identifier-from-a-persistent-unique-identifier.md)   | Puede recuperar el identificador de objeto que se asigna a un identificador único persistente determinado. |
| [Identificadores de objetos funcionales](retrieving-the-functional-object-identifiers-for-a-device.md) | Cada categoría funcional admite uno o varios objetos funcionales.                         |
