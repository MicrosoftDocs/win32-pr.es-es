# <a name="msmq-site-name-attribute"></a>Atributo MSMQ-Site-Name

El atributo **MSMQ-Site-Name** se ha reemplazado por el atributo [**MSMQ-Site-Name-ex**](a-msmqsitenameex.md) .



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Nombre de sitio de MSMQ                              |
| Nombre para mostrar de LDAP | mSMQSiteName                                |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.965                      |
| System-ID-GUID    | ffadb4b2-de39-11d1-90a5-00c04fd91ab1        |
| Sintaxis            | [**String(Teletex)**](s-string-teletex.md) |
