# <a name="about-the-audio-compression-manager"></a>Acerca del administrador de compresión de audio

El administrador de compresión de audio agrega compatibilidad de nivel de sistema para los siguientes servicios:

-   Compresión y descompresión de audio en tiempo de ejecución transparente
-   Forma de onda: selección de formato de datos de audio
-   Onda-selección de filtro de datos de audio
-   Forma de onda: conversión de formato de datos de audio
-   Onda: filtrado de datos de audio
