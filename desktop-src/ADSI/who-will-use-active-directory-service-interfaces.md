# <a name="who-will-use-active-directory-service-interfaces"></a>¿Quién usará Active Directory interfaces de servicio?

Los administradores de red usarán ADSI para automatizar las tareas administrativas comunes, como agregar usuarios y grupos, administrar impresoras y establecer permisos en los recursos de red.

Los fabricantes de software independientes (ISV) y los desarrolladores de usuarios finales usarán ADSI para "habilitar el directorio" sus productos y aplicaciones. Los servicios pueden publicarse en un directorio, los clientes pueden usar el directorio para encontrar los servicios y ambos pueden usar el directorio para buscar y manipular otros objetos de interés. Dado que Active Directory interfaces de servicio son independientes de los servicios de directorio subyacentes, los productos y aplicaciones habilitados para el directorio funcionarán correctamente en varios entornos de red y de directorio.
