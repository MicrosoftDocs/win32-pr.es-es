# <a name="xmuint4-extensions"></a>Extensiones de XMUINT4

[**XMUINT4**](/windows/win32/api/directxmath/ns-directxmath-xmuint4) describe una estructura que contiene un vector 4D que consta de cuatro valores enteros sin signo.

> [!Note]  
> Las características que se enumeran aquí solo están disponibles cuando se desarrollan con C++.
