# <a name="vml-mso-fit-shape-to-text-attribute"></a>VML MSO-Fit: atributo de forma a texto

En este tema se describe VML, una característica que está desusada en Windows Internet Explorer 9. Las páginas web y las aplicaciones que se basan en VML se deben migrar a SVG u otros estándares ampliamente admitidos.

> [!Note]  
> A partir del 2011 de diciembre, este tema se ha archivado. Como resultado, ya no se mantiene de forma activa. Para obtener más información, vea [contenido archivado](/previous-versions/windows/internet-explorer/ie-developer/). Para obtener información, recomendaciones e instrucciones sobre la versión actual de Windows Internet Explorer, consulte [Centro para desarrolladores de Internet Explorer](https://msdn.microsoft.com/ie/).

 

Define si la forma se ajustará para ajustarse al texto en el cuadro de texto. Lectura/escritura **VgTriState**.

**Se aplica a**

[TextBox](msdn-online-vml-textbox-element.md)

**Sintaxis de etiquetas**

<v: *Element* style = "MSO-Fit-Shape-to-Text: expresión" >

**Comentarios:**

Si es **true**, la forma se ajusta para ajustarse al texto. El valor predeterminado es **false**.
