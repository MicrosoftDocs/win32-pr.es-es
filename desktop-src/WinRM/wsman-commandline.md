# <a name="wsmancommandline-property"></a>Propiedad WSMan.CommandLine

Obtiene la línea de comandos del proceso que carga el componente de automatización.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```VB
WSMan.CommandLine As BSTR
```



## <a name="property-value"></a>Valor de propiedad

Cadena que contiene la línea de comandos.
