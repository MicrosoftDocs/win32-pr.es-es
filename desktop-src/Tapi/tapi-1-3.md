# <a name="tapi-13"></a>TAPI 1.3

TAPI 1.3 (a veces conocido como TAPI 1.03) era la primera versión publicada de la especificación TAPI. Los archivos binarios del sistema TAPI 1.3 (Tapi.dll y otros) se encuentran en el SDK de TAPI 1.0, que solo se puede encontrar en la suscripción de nivel II de Desarrollador de Microsoft Network (MSDN). Los archivos binarios del sistema (que se encuentran en el directorio REDIST) son redistribuibles, pero están diseñados para ejecutarse solo en Microsoft Windows 3.1 y Windows for Workgroups 3.11. No se admiten y no funcionarán en ninguna versión de 32 bits Windows.

Los encabezados y bibliotecas que se usan para desarrollar una aplicación TAPI 1.3 solo están disponibles en el SDK de TAPI 1.0 y solo admiten el desarrollo de 16 bits. De hecho, las bibliotecas TAPI 1.3 son las únicas bibliotecas TAPI de 16 bits disponibles y deben usarse para todas las aplicaciones de 16 bits, independientemente de la plataforma de destino o la versión de destino.
