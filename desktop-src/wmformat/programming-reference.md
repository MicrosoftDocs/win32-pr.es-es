# <a name="windows-media-format-sdk-programming-reference"></a>Referencia de programación del SDK de Windows Media Format

El kit de desarrollo de software (SDK) de Windows Media® Format de Microsoft® admite una variedad de objetos, interfaces, funciones independientes, estructuras, tipos de enumeración, atributos, etc. En las secciones siguientes se documentan en detalle.



| Sección                                                                              | Descripción                                                                                                  |
|--------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------|
| [Objects](objects.md)                                                               | Describe los objetos y las interfaces compatibles con cada objeto en el SDK de formato de Windows Media.           |
| [Funciones](functions.md)                                                           | Describe todas las funciones independientes, que suelen usarse para crear e inicializar varios objetos.      |
| [Interfaces](interfaces.md)                                                         | Describe todas las interfaces y los métodos de este SDK.                                                        |
| [Estructuras](structures.md)                                                         | Describe las estructuras que admite este SDK.                                                              |
| [Tipos de enumeración](enumeration-types.md)                                           | Describe los tipos de enumeración admitidos por este SDK.                                                       |
| [Atributos](attributes.md)                                                         | Describe los atributos que se pueden especificar en los encabezados de los archivos ASF.                                  |
| [Tipos de medios](media-types.md)                                                       | Describe los identificadores de tipo de medio utilizados por este SDK.                                                       |
| [Configuración de salida](output-settings.md)                                               | Describe las constantes globales que se usan para identificar la configuración de salida del lector.                                      |
| [Cadenas de idioma](language-strings.md)                                             | Describe los identificadores de idioma compatibles con RFC 1766.                                         |
| [Parámetros de plantilla de conformidad de dispositivos](device-conformance-template-parameters.md) | Describe las plantillas de conformidad del dispositivo, que describen los intervalos de valores adecuados para varios dispositivos. |
| [Perfiles del sistema](system-profiles.md)                                               | Enumera todos los perfiles del sistema admitidos.                                                                         |
| [Perfiles del sistema localizados](localized-system-profiles.md)                           | Enumera los archivos de Perfil de sistema localizados que se incluyen con el SDK y los idiomas asociados a cada uno de ellos.       |
| [Valores GUID](guid-values.md)                                                       | Describe los valores de GUID utilizados por este SDK.                                                                  |
| [Códigos de error](error-codes.md)                                                       | Describe los códigos de error comunes devueltos por los métodos y las funciones del SDK.                                   |
| [Complementos de origen](source-plug-ins.md)                                               | Describe los requisitos para los orígenes implementados por el usuario.                                                     |
