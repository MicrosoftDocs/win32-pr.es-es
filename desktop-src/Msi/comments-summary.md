# <a name="comments-summary-property"></a>Propiedad de Resumen de comentarios

La propiedad **Resumen de comentarios** transmite el uso general del paquete de instalación, transformación o paquete de revisión.

El autor de un paquete de instalación, transformación o paquete de revisión debe establecer el valor de la propiedad **comentarios Summary** en uno de los siguientes valores:

<dl> "Esta base de datos del instalador contiene la lógica y los datos necesarios para instalar <> del *producto* ".  
"Esta transformación contiene la lógica y los datos necesarios para instalar <> de *producto* ".  
"Esta revisión contiene la lógica y los datos necesarios para instalar <> de *producto* ".  
</dl>

donde <> de *producto* es el nombre del producto.
