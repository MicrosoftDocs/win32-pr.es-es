# <a name="ms-sql-registeredowner-attribute"></a>Atributo MS-SQL-RegisteredOwner

Propietario registrado de la instancia actual de SQL Server.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-RegisteredOwner                      |
| Nombre para mostrar de LDAP | mS-SQL-RegisteredOwner                      |
| Tamaño              | \-                                          |
| Actualizar privilegio  | Administrador de dominio                        |
| Frecuencia de actualización  | En el programa de instalación del sistema.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1364                     |
| System-ID-GUID    | 48fd44ea-ccee-11d2-9993-0000f87a57d4        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
