# <a name="tools-to-sign-files-and-check-signatures"></a>Herramientas para firmar archivos y comprobar firmas

CryptoAPI Tools son las siguientes herramientas que puede usar para firmar archivos digitalmente, confirmar firmas en archivos y crear archivos de catálogo.



| Herramienta                     | Comentarios                                                                                                                                                                                                                                                                            |
|--------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [MakeCat](makecat.md)   | Crea un archivo de catálogo sin signo. Este archivo de catálogo se usa posteriormente para comparar los hashes dentro del archivo de catálogo con los hash de archivos [*individuales,*](../secgloss/h-gly.md) lo que ayuda a demostrar que los archivos originales están libres de alteraciones. |
| [SetReg](setreg.md)     | Establece las claves del Registro que controlan el proceso de comprobación del certificado.                                                                                                                                                                                                              |
| [SignTool](signtool.md) | Firma digitalmente los archivos y confirma las firmas digitales de los archivos.                                                                                                                                                                                                                    |
