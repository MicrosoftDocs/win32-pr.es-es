# <a name="avencddlorosurroundmixlvl_x10-property"></a>\_Propiedad AVEncDDLoRoSurroundMixLvl x10

Especifica el cambio de nivel que se aplica a los canales de envolvente para el downmixing de carga. Esta propiedad se aplica a los codificadores de audio Dolby digital.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT32** (**VT \_ UI4**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncDDLoRoSurroundMixLvl \_ x10**

## <a name="property-value"></a>Valor de propiedad

El valor se especifica como decibelios (dB) x 10.
