# <a name="systemproplistquicktip"></a>System. proplist. QuickTip

Lista de propiedades que se muestran en el recuadro informativo cuando el elemento está en una red lenta. No se mostrarán las propiedades con valores vacíos. Regístrese en el regvalue de "QuickTip".

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.PropList.QuickTip
   shellPKey = PKEY_PropList_QuickTip
   formatID = C9944A21-A406-48FE-8225-AEC7E24C211B
   propID = 5
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = String
```

## <a name="remarks"></a>Observaciones

Los valores PKEY se definen en Propkey. h.
