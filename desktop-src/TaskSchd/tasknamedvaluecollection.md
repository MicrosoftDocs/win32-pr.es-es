# <a name="tasknamedvaluecollection-object"></a>Objeto TaskNamedValueCollection

Objeto de scripting que contiene una colección de pares nombre-valor de objeto [**TaskNamedValuePair.**](tasknamedvaluepair.md)

## <a name="members"></a>Miembros

El **objeto TaskNamedValueCollection** tiene estos tipos de miembros:

-   [Métodos](#methods)
-   [Propiedades](#properties)

### <a name="methods"></a>Métodos

El **objeto TaskNamedValueCollection** tiene estos métodos.



| Método                                            | Descripción                                                        |
|:--------------------------------------------------|:-------------------------------------------------------------------|
| [**Borrar**](tasknamedvaluecollection-clear.md)   | Borra toda la colección de pares nombre-valor.<br/>       |
| [**Crear**](tasknamedvaluecollection-create.md) | Crea un par nombre-valor en la colección.<br/>            |
| [**Quitar**](tasknamedvaluecollection-remove.md) | Quita un par nombre-valor seleccionado de la colección.<br/> |



 

### <a name="properties"></a>Propiedades

El **objeto TaskNamedValueCollection** tiene estas propiedades.



| Propiedad                                                   | Descripción                                                        |
|:-----------------------------------------------------------|:-------------------------------------------------------------------|
| [**Count**](tasknamedvaluecollection-count.md)<br/> | Obtiene el número de pares nombre-valor de la colección.<br/>  |
| [**Elemento**](tasknamedvaluecollection-item.md)<br/>   | Obtiene el par nombre-valor especificado de la colección.<br/> |
