# <a name="signing-data-and-verifying-digital-signatures"></a>Firmar datos y comprobar firmas digitales

\[CAPICOM es un componente de solo bits de 32 que está disponible para su uso en los siguientes sistemas operativos: Windows Server 2008, Windows Vista y Windows XP. En su lugar, use la .NET Framework para implementar características de seguridad. Para obtener más información, vea [alternativas al uso de CAPICOM](alternatives-to-using-capicom.md).\]

La firma de datos y la comprobación de firmas en los datos se realizan entre las tareas más importantes realizadas con la [*Criptografía*](../secgloss/c-gly.md). En las secciones siguientes se muestran ejemplos de tareas de firma digital:

-   [Firmar un documento](signing-a-document.md)
-   [Firmar un documento](cosigning-a-document.md)
-   [Comprobar la firma de un documento](verifying-the-signature-on-a-document.md)
