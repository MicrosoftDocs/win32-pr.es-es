# <a name="setting-permissions-on-virtual-directories"></a><span data-ttu-id="b590b-103">Establecimiento de permisos en directorios virtuales</span><span class="sxs-lookup"><span data-stu-id="b590b-103">Setting Permissions on Virtual Directories</span></span>

<span data-ttu-id="b590b-104">Por motivos de seguridad, Servicio de transferencia inteligente en segundo plano (BITS) no carga los archivos en un directorio virtual que tenga habilitados los permisos de ejecución y scripting.</span><span class="sxs-lookup"><span data-stu-id="b590b-104">For security reasons, Background Intelligent Transfer Service (BITS) does not upload files to a virtual directory that has scripting and execute permissions enabled.</span></span> <span data-ttu-id="b590b-105">Si carga un archivo en un directorio virtual que tiene estos permisos habilitados, se produce un error en el trabajo con un código de error de BG \_ E \_ Server \_ Execute \_ enabled.</span><span class="sxs-lookup"><span data-stu-id="b590b-105">If you upload a file to a virtual directory that has these permissions enabled, the job fails with an error code of BG\_E\_SERVER\_EXECUTE\_ENABLED.</span></span>
