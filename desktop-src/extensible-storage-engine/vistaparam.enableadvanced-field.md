# <a name="vistaparamenableadvanced-field"></a>VistaParam.EnableAdvanced field (VistaParam.Enable)Vanced field (VistaParam.EnableAdvanced)

Este parámetro se usa para controlar cuándo el motor de base de datos acepta o rechaza los cambios en un subconjunto de los parámetros del sistema. Este parámetro se usa junto con [Configuration](./vistaparam.configuration-field.md) para evitar que algunos parámetros del sistema se establezcan fuera de los valores predeterminados de la configuración seleccionada.

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop.Vista](./microsoft.isam.esent.interop.vista-namespace.md)  
**Ensamblado:**  Microsoft.Isam.Esent.Interop (en Microsoft.Isam.Esent.Interop.dll)
