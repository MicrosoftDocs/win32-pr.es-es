# <a name="matrix5x4f-class"></a>Clase Matrix5x4F

La clase **Matrix5x4F** representa una matriz de 5 por 4 y proporciona métodos útiles para crear matrices.

## <a name="members"></a>Miembros

La clase **Matrix5x4F** hereda de [**D2D1 \_ Matrix \_ 5x4 \_ F**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_5x4_f). **Matrix5x4F** también tiene estos tipos de miembros:

-   [Constructores](#constructors)

### <a name="constructors"></a>Constructores

La clase **Matrix5x4F** tiene estos constructores.



| Constructor                                                                                                                                                                                                                                                                                           | Descripción                                                                                                                        |

| [**Matrix5x4F (FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, FLOAT, Float, FLOAT) (FLOAT, Float, Float, Float, Float, Float, Float, Float, Float, Float, Float, FLOAT, Float, Float, Float, FLOAT)**](matrix5x4f-matrix5x4f-floats-.md) | Crea una nueva instancia de una clase **Matrix5x4F** que se inicializa con todos los valores de la matriz de punto flotante.<br/> |
