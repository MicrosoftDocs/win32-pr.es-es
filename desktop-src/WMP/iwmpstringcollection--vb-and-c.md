# <a name="iwmpstringcollection-vb-and-c-interface"></a>Interfaz IWMPStringCollection (VB y C#)

Proporciona una propiedad y un método para tener acceso a una colección de cadenas por número de índice.

La interfaz **IWMPStringCollection** expone la propiedad siguiente.

## <a name="members"></a>Miembros

La interfaz **IWMPStringCollection (VB y C#)** tiene estos tipos de miembros:

-   [Métodos](#methods)
-   [Propiedades](#properties)

### <a name="methods"></a>Métodos

La interfaz **IWMPStringCollection (VB y C#)** tiene estos métodos.



| Método                                                                         | Descripción                                           |
|:-------------------------------------------------------------------------------|:------------------------------------------------------|
| [**Elemento**](wmplibiwmpstringcollection-iwmpstringcollection-item--vb-and-c.md) | Devuelve la cadena que se encuentra en el índice especificado.<br/> |



 

### <a name="properties"></a>Propiedades

La interfaz **IWMPStringCollection (VB y C#)** tiene estas propiedades.



| Propiedad                                                                                    | Tipo de acceso          | Descripción                                                   |
|:--------------------------------------------------------------------------------------------|:---------------------|:--------------------------------------------------------------|
| [**contabiliza**](wmplibiwmpstringcollection-iwmpstringcollection-count--vb-and-c.md)<br/> | Solo lectura<br/> | Obtiene el número de elementos de la colección de cadenas.<br/> |
