# <a name="event-tracing-tools"></a>Herramientas de seguimiento de eventos

Las siguientes herramientas de seguimiento de eventos están disponibles para su uso.



| Herramienta     | Descripción                                                                                                                                                                                                                                                                                                                                |
|----------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Logman   | Administra y programa las colecciones de registros de seguimiento de eventos y contadores de rendimiento en sistemas locales y remotos. Puede usar esta herramienta para enumerar los proveedores que han [publicado el diseño de los datos de eventos](publishing-your-event-schema.md) en el \\ espacio de nombres WMI raíz. Para obtener información detallada sobre el uso de esta herramienta, vea el **centro de ayuda y soporte técnico**. |
| Tracefmt | Da formato a la información de un archivo de registro de seguimiento en un formato legible. Para obtener más información, consulte Herramientas para el seguimiento de software en el kit de desarrollo de controladores (DDK) de Microsoft Windows. Solo lo usa el seguimiento de software WPP.                                                                                                                                      |
| Tracelog | Controla una sesión de seguimiento de eventos. Para obtener más información, consulte Herramientas para el seguimiento de software en el DDK.                                                                                                                                                                                                                                                   |
| Tracepdb | Crea un archivo de formato de mensaje de seguimiento que utiliza la herramienta Tracefmt para convertir los mensajes registrados en un formato legible para el usuario. Para obtener más información, consulte Herramientas para el seguimiento de software en el DDK. Solo lo usa el seguimiento de software WPP.                                                                                                                            |
| Tracerpt | Procesa los registros de seguimiento de eventos o eventos en tiempo real de los proveedores de seguimiento de eventos instrumentados y permite generar informes de análisis de seguimiento y archivos CSV (delimitados por comas) para los eventos generados. Para obtener más información, vea el **centro de ayuda y soporte técnico**.                                                                                       |
