# <a name="destroying-event-objects"></a>Destruir objetos de evento

La entidad que crea un objeto de evento (aplicación o proveedor de servicios) es responsable de destruirlo cuando ya no es necesario. Los proveedores de servicios lo hacen a través de **WPUCloseEvent**.
