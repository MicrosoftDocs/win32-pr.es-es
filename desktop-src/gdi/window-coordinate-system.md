# <a name="window-coordinate-system"></a><span data-ttu-id="5ef5a-103">Sistema de coordenadas de ventana</span><span class="sxs-lookup"><span data-stu-id="5ef5a-103">Window Coordinate System</span></span>

<span data-ttu-id="5ef5a-104">El sistema de coordenadas de una ventana se basa en el sistema de coordenadas del dispositivo de pantalla.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-104">The coordinate system for a window is based on the coordinate system of the display device.</span></span> <span data-ttu-id="5ef5a-105">La unidad de medida básica es la unidad del dispositivo (normalmente, el píxel).</span><span class="sxs-lookup"><span data-stu-id="5ef5a-105">The basic unit of measure is the device unit (typically, the pixel).</span></span> <span data-ttu-id="5ef5a-106">Los puntos de la pantalla se describen mediante los pares de coordenadas x e y.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-106">Points on the screen are described by x- and y-coordinate pairs.</span></span> <span data-ttu-id="5ef5a-107">Las coordenadas x aumentan a la derecha; las coordenadas y aumentan de arriba a abajo.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-107">The x-coordinates increase to the right; y-coordinates increase from top to bottom.</span></span> <span data-ttu-id="5ef5a-108">El origen (0,0) del sistema depende del tipo de coordenadas que se estén usando.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-108">The origin (0,0) for the system depends on the type of coordinates being used.</span></span>

<span data-ttu-id="5ef5a-109">El sistema y las aplicaciones especifican la posición de una ventana en la pantalla en coordenadas de la *pantalla*.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-109">The system and applications specify the position of a window on the screen in *screen coordinates*.</span></span> <span data-ttu-id="5ef5a-110">En el caso de las coordenadas de pantalla, el origen es la esquina superior izquierda de la pantalla.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-110">For screen coordinates, the origin is the upper-left corner of the screen.</span></span> <span data-ttu-id="5ef5a-111">La posición completa de una ventana suele describirse mediante una estructura [**Rect**](/previous-versions//dd162897(v=vs.85)) que contiene las coordenadas de pantalla de dos puntos que definen las esquinas superior izquierda e inferior derecha de la ventana.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-111">The full position of a window is often described by a [**RECT**](/previous-versions//dd162897(v=vs.85)) structure containing the screen coordinates of two points that define the upper-left and lower-right corners of the window.</span></span>

<span data-ttu-id="5ef5a-112">El sistema y las aplicaciones especifican la posición de los puntos en una ventana mediante las *coordenadas de cliente*.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-112">The system and applications specify the position of points in a window by using *client coordinates*.</span></span> <span data-ttu-id="5ef5a-113">El origen en este caso es la esquina superior izquierda del área de la ventana o cliente.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-113">The origin in this case is the upper-left corner of the window or client area.</span></span> <span data-ttu-id="5ef5a-114">Las coordenadas de cliente garantizan que una aplicación puede usar valores de coordenadas coherentes mientras se dibuja en la ventana, independientemente de la posición de la ventana en la pantalla.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-114">Client coordinates ensure that an application can use consistent coordinate values while drawing in the window, regardless of the position of the window on the screen.</span></span>

<span data-ttu-id="5ef5a-115">Las dimensiones del área cliente también se describen mediante una estructura [**Rect**](/previous-versions//dd162897(v=vs.85)) que contiene coordenadas de cliente para el área.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-115">The dimensions of the client area are also described by a [**RECT**](/previous-versions//dd162897(v=vs.85)) structure that contains client coordinates for the area.</span></span> <span data-ttu-id="5ef5a-116">En todos los casos, la coordenada superior izquierda del rectángulo se incluye en el área de ventana o cliente, mientras que la coordenada inferior derecha se excluye.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-116">In all cases, the upper-left coordinate of the rectangle is included in the window or client area, while the lower-right coordinate is excluded.</span></span> <span data-ttu-id="5ef5a-117">Las operaciones de gráficos en una ventana o área cliente se excluyen de los bordes derecho e inferior del rectángulo envolvente.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-117">Graphics operations in a window or client area are excluded from the right and lower edges of the enclosing rectangle.</span></span>

<span data-ttu-id="5ef5a-118">En ocasiones, es posible que las aplicaciones necesiten asignar coordenadas en una ventana a las de otra ventana.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-118">Occasionally, applications may be required to map coordinates in one window to those of another window.</span></span> <span data-ttu-id="5ef5a-119">Una aplicación puede asignar coordenadas mediante la función [**MapWindowPoints**](/windows/desktop/api/Winuser/nf-winuser-mapwindowpoints) .</span><span class="sxs-lookup"><span data-stu-id="5ef5a-119">An application can map coordinates by using the [**MapWindowPoints**](/windows/desktop/api/Winuser/nf-winuser-mapwindowpoints) function.</span></span> <span data-ttu-id="5ef5a-120">Si una de las ventanas es la ventana del escritorio, la función convierte eficazmente las coordenadas de la pantalla en coordenadas de cliente y viceversa. la ventana de escritorio siempre se especifica en coordenadas de pantalla.</span><span class="sxs-lookup"><span data-stu-id="5ef5a-120">If one of the windows is the desktop window, the function effectively converts screen coordinates to client coordinates and vice versa; the desktop window is always specified in screen coordinates.</span></span>
