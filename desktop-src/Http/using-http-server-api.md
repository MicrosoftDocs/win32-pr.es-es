# <a name="using-http-server-api"></a>Uso de LA API del servidor HTTP

Los archivos de origen del programa de aplicación incluyen el archivo de encabezado Http.h para acceder a prototipos de función y definiciones de estructura para la API de servidor HTTP. Los desarrolladores pueden usar el archivo de biblioteca Httpapi.lib para compilar aplicaciones que usan la API del servidor HTTP. En tiempo de ejecución, las aplicaciones se vinculan al Httpapi.dll.
