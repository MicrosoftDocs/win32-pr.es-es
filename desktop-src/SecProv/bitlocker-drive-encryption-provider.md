# <a name="bitlocker-drive-encryption-provider"></a>Proveedor de cifrado de unidad BitLocker

La interfaz del proveedor WMI es para administrar y configurar Cifrado de unidad BitLocker (BDE) en Windows Server 2008 R2, Windows Server 2008 y solo las versiones específicas de Windows 7, Windows Vista Enterprise y Windows Vista Ultimate. Está diseñado para que lo lean los scripts de escritura, los elementos de la interfaz de usuario u otras herramientas administrativas para BDE, así como para las personas que desean comprender mejor la funcionalidad de BDE.
