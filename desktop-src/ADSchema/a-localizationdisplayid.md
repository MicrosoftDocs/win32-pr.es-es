# <a name="localization-display-id-attribute"></a>Atributo Localization-Display-Id

Se usa para indexar en el archivo Extrts.mc para obtener el displayName localizado para los objetos con fines de interfaz de usuario.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | Localization-Display-Id              |
| Ldap-Display-Name | localizationDisplayId                |
| Size              | 4 bytes                              |
| Actualizar privilegios  | Administrador de esquemas                 |
| Frecuencia de actualización  | Nunca se debe cambiar un identificador.       |
| Attribute-Id      | 1.2.840.113556.1.4.1353              |
| System-Id-Guid    | a746f0d1-78d0-11d2-9916-0000f87a57d4 |
| Syntax            | [**Enumeración**](s-enumeration.md) |
