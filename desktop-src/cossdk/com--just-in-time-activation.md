# <a name="com-just-in-time-activation"></a>Activación Just-In-Time de COM+

La activación Just-In-Time (JIT) es un servicio COM+ automático que puede ayudarle a usar los recursos del servidor de forma más eficaz, especialmente al escalar verticalmente la aplicación para realizar transacciones de gran volumen.

En los temas siguientes se proporciona información general y de tareas sobre la activación Just-In-Time de COM+.

|------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------|
| [Conceptos de activación Just-In-Time de COM+](com--just-in-time-activation-concepts.md) | Proporciona información general sobre las características y ventajas de habilitar el servicio de activación JIT para los componentes com+. |
| [Tareas de activación Just-In-Time de COM+](com--just-in-time-activation-tasks.md)       | Proporciona instrucciones para configurar la activación JIT para las aplicaciones COM+.                                       |
