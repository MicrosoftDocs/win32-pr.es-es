# <a name="deciding-which-attributes-to-retrieve-for-each-object-found"></a>Decidir qué atributos recuperar para cada objeto encontrado

Si busca objetos de una clase determinada, parece lógico que recupere los atributos que existen realmente en los objetos de esa clase. Siga las mismas reglas que se de [abordan en Búsqueda de una lista de atributos para consultar](finding-a-list-of-attributes-to-query.md). Tenga en cuenta que los atributos construidos se pueden recuperar en la consulta, pero los atributos construidos no se pueden usar en filtros.
