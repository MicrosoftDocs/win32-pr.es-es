# <a name="version-property"></a>Propiedad Version

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Devuelve la versión del carácter.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintáctica**
</dt> <dd>

*directivas. ***Caracteres ("*** CharacterID * *"). Versión**

</dd> </dl>

## <a name="remarks"></a>Observaciones

La propiedad **version** devuelve una cadena que corresponde a la versión de la definición del conjunto de animación estándar para la que se compiló el carácter. El número de versión del carácter se establece automáticamente al compilarlo con el editor de caracteres del agente de Microsoft.
