# <a name="user-name-functions"></a>Funciones de nombre de usuario

Las funciones de nombre de usuario incluyen una sola función, [**NPGetUser**](/windows/desktop/api/Npapi/nf-npapi-npgetuser). Esta función recupera el nombre de usuario predeterminado actual o el nombre de usuario usado para establecer la conexión de red.
