# <a name="systemdevicesnotificationsnewvoicemail"></a>System. Devices. notifications. NewVoicemail

Notificación de correo de voz de dispositivo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Notifications.NewVoicemail
   shellPKey = PKEY_Devices_Notifications_NewVoicemail
   formatID = 59569556-0A08-4212-95B9-FAE2AD6413DB
   propID = 2
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Byte
      IsInnate = true
      EnumeratedList
         UseValueForDefault = True
         enumRange
            name = NewVoicemailNotification
            minValue = 0
            setValue = 0
            text = You have a new voicemail message.
```
