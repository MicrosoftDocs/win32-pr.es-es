# <a name="transferring-data"></a>Transferir datos

> [!Note]  
> Este sistema de scripting se ha reemplazado por Windows de automatización de adquisición de imágenes (WIA). Consulte [Windows capa de automatización de adquisición de imágenes.](/previous-versions/windows/desktop/wiaaut/-wiaaut-startpage)

 

Use el [**método Transfer**](-wia-iwiadispatchitem-transfer.md) de un [**objeto Item**](-wia-item.md) para transferir datos de imagen o audio desde un dispositivo a un archivo o al Portapapeles.

Pase una ruta de acceso y un nombre de archivo como primer parámetro para guardar los datos en el disco o pase la cadena "clipboard" para transferir el elemento al Portapapeles.
