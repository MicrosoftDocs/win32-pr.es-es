# <a name="cryptography-constants"></a>Constantes de criptografía

Las siguientes constantes las utilizan las funciones de criptografía y se definen en Wincrypt. h. Las constantes de criptografía se clasifican en función del uso como se indica a continuación.



| Constante                                                                                                           | Descripción                                                                                                                                                                                                 |
|--------------------------------------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Constantes para CryptEncodeObject y CryptDecodeObject](constants-for-cryptencodeobject-and-cryptdecodeobject.md) | Constantes predefinidas usadas con operaciones de codificación y descodificación de [*notación de sintaxis abstracta One*](../secgloss/a-gly.md) (ASN. 1). |
| [Constantes para extensiones de Netscape](constants-for-netscape-extensions.md)                                         | Constantes predefinidas utilizadas para las extensiones de certificado de Netscape. En esta sección se proporcionan detalles sobre el uso de estas extensiones con operaciones de codificación y descodificación ASN. 1.                                      |
| [Nombres de proveedor de servicios criptográficos](cryptographic-provider-names.md)                                                   | Constantes de cadena predefinidas para los nombres de [*proveedores de servicios criptográficos*](../secgloss/c-gly.md) (CSP) comunes.   |
| [**MAXUIDLEN**](maxuidlen.md)                                                                                     | Constante numérica que especifica el número máximo de caracteres que utilizarán algunos de los proveedores de servicios criptográficos de Microsoft al obtener el identificador de usuario.                                                  |
| [**Constantes de almacenamiento en caché de claves privadas**](private-key-caching-constants.md)                                             | Constantes que representan entradas del registro que controlan el almacenamiento en caché de la [*clave privada*](../secgloss/p-gly.md) por parte de un CSP.                                                |
| [Marcas de protocolo](protocol-flags.md)                                                                               | Constantes predefinidas para los protocolos de criptografía.                                                                                                                                                            |
