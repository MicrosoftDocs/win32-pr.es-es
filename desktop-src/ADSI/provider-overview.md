Además de los [requisitos mínimos del proveedor](provider-minimum-requirements.md), otros datos de uso incluyen:

-   Las [interfaces de esquema](schema-interfaces.md) describen las interfaces de administración de esquemas desde el punto de vista del proveedor.
-   [Enumerar objetos de contenedor](enumerating-container-objects.md) describe los requisitos para admitir enumeraciones.
-   [Información del registro del proveedor](provider-registry-information.md) muestra las entradas del registro que usa un proveedor ADSI.
-   La [compatibilidad con consultas](support-for-queries.md) describe los requisitos para admitir la búsqueda del directorio.
-   Los [problemas de implementación de los proveedores ADSI](implementation-issues-for-adsi-providers.md) incluyen notas a proveedores sobre diversos problemas de software.
