# <a name="indespolicy-methods"></a>Métodos INDESPolicy

La [**interfaz INDESPolicy**](/windows/desktop/api/certpol/nn-certpol-indespolicy) admite los métodos siguientes.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                        | Descripción                                                                                       |
|------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| [**Método GenerateChallenge**](/windows/desktop/api/certpol/nf-certpol-indespolicy-generatechallenge)<br/> | Realiza la decisión de directiva de si se debe emitir una contraseña de desafío al cliente SCEP.<br/> |
| [**Inicializar método**](/windows/desktop/api/certpol/nf-certpol-indespolicy-initialize)<br/>               | Inicializa el módulo de directivas NDES.<br/>                                                    |
| [**Método Notify**](/windows/desktop/api/certpol/nf-certpol-indespolicy-notify)<br/>                       | Notifica al complemento el estado de transacción de la solicitud de certificado SCEP.<br/>        |
| [**Uninitialize (método)**](/windows/desktop/api/certpol/nf-certpol-indespolicy-uninitialize)<br/>           | Desinicializa el módulo de directivas NDES.<br/>                                                  |
| [**Método VerifyRequest**](/windows/desktop/api/certpol/nf-certpol-indespolicy-verifyrequest)<br/>         | Comprueba la solicitud de certificado NDES para su envío a la entidad de certificación.<br/>                        |
