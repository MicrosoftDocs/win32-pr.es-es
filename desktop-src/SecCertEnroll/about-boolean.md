# <a name="boolean-certificate-enrollment-api"></a>BOOLEANO (API de inscripción de certificados)

Un valor booleano puede ser **true** o **false**. **False** se codifica como un tripledo TLV en el que el campo de **valor** es cero (0). **True** se codifica como un valor distinto de cero. La **etiqueta** para el tipo de datos **Boolean** es 0x01.
