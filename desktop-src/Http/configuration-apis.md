# <a name="configuration-apis"></a>API de configuración

Las API de servidor HTTP proporcionan funcionalidad para que los administradores del sistema establezcan, consulten y eliminen parámetros de configuración. Las API de configuración controlan la configuración de todas las aplicaciones que usan la API del servidor HTTP en el sistema.

Las secciones que se enumeran a continuación incluyen información sobre las opciones de configuración.

> [!Note]  
> Todos estos son pasos de configuración opcionales que se aplicarán en función de los requisitos del sistema o de la aplicación.

 

Lista de escucha de [IP](ip-listen-list.md)   : describe cómo configurar HTTP para que opere sobre interfaces IP específicas.

[Certificados SSL](ssl-certificates.md)   : detalla los procedimientos para configurar certificados con la API del servidor HTTP.

[Reservas de espacio de nombres, registros y enrutamiento](namespace-reservations-registrations-and-routing.md)   : describe el proceso de reserva de partes del espacio de nombres para todas las aplicaciones del equipo mediante la API de servidor HTTP. También se describe el proceso para registrar y enrutar solicitudes a aplicaciones concretas.

[HttpCfg.exe](httpcfg-exe.md)   : un programa de ejemplo que usa las API de configuración de. Una versión compilada de esta herramienta también se incluye con el sistema operativo entre las herramientas de soporte, junto con un archivo de ayuda que documenta cómo utilizarla.
