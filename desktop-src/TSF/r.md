# <a name="r-text-services-framework"></a>R (marco de trabajo de servicios de texto)

<dl> <dt>

<span id="_tsf_range_gly"></span><span id="_TSF_RANGE_GLY"></span>**variedad**
</dt> <dd>

Objeto que describe un intervalo de texto que existe dentro de una secuencia de texto. Un intervalo se usa para leer, escribir y manipular texto. También se conoce como *intervalo de texto*.

Vea también, *región*.

</dd> <dt>

<span id="_tsf_reconversion_gly"></span><span id="_TSF_RECONVERSION_GLY"></span>**reconversión**
</dt> <dd>

Proceso que permite al usuario seleccionar texto alternativo para el texto insertado.

</dd> <dt>

<span id="_tsf_region_gly"></span><span id="_TSF_REGION_GLY"></span>**Region**
</dt> <dd>

Intervalo de texto que se administra como un intervalo de edición único, basado en el formato.

</dd> </dl>
