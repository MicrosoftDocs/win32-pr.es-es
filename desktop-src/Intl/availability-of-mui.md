# <a name="availability-of-mui"></a>Disponibilidad de MUI

Las siguientes versiones de Windows incluyen API y construcciones para proporcionar compatibilidad nativa para desarrollar aplicaciones multilingües mediante la tecnología DEIS:

-   Windows 7 y versiones posteriores.
-   Windows Vista.

Se proporciona una biblioteca de nivel inferior, [**LoadMUILibrary,**](/windows/desktop/api/Muiload/nf-muiload-loadmuilibrarya)para permitir la escritura de aplicaciones multilingües para versiones de Windows antes de Windows Vista, a la vez que se proporciona un comportamiento compatible con el de las aplicaciones DE NIS para Windows Vista y versiones posteriores de Windows.
