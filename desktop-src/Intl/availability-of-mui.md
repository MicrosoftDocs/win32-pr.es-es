# <a name="availability-of-mui"></a>Disponibilidad de MUI

Las siguientes versiones de Windows incluyen API y construcciones para proporcionar compatibilidad nativa para desarrollar aplicaciones multilingües mediante la tecnología MUI:

-   Windows 7 y versiones posteriores.
-   Windows Vista.

Una biblioteca de nivel inferior, [**LoadMUILibrary**](/windows/desktop/api/Muiload/nf-muiload-loadmuilibrarya), se proporciona para permitir la escritura de aplicaciones multilingües para versiones de Windows anteriores a Windows Vista, a la vez que proporciona un comportamiento compatible con el de las aplicaciones de MUI escritas para Windows Vista y versiones posteriores de Windows.
