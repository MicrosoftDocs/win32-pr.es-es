# <a name="microsoft-windows-media-drm-client-sample-applications"></a>Aplicaciones de ejemplo del cliente DRM de Microsoft Windows Media

Las aplicaciones de ejemplo que usan las API extendidas del cliente DRM de Windows Media se proporcionan como proyectos para Microsoft Visual Studio 2005.

En la tabla siguiente se resumen los ejemplos. Puede encontrar información adicional sobre cada ejemplo, incluida la información de uso, en el archivo de readme.txt de cada directorio de ejemplo.



| Muestra      | Descripción                                                                                                                                 |
|-------------|---------------------------------------------------------------------------------------------------------------------------------------------|
| QueryRights | Notifica los derechos concedidos por licencias en el equipo cliente para obtener una lista de niños. Muestra las consultas rápidas y las consultas de estado de licencia. |
| DRMUpdate   | Realiza la individualización y crea un registro de todos los eventos generados durante el proceso.                                                   |
