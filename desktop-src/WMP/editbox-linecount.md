# <a name="editboxlinecount"></a>EDITBOX. lineCount

El atributo **lineCount** recupera el número de líneas en el control de cuadro de edición.

``` syntax
        elementID.lineCount
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un **número** de solo lectura (**Long**).

## <a name="remarks"></a>Observaciones

Este atributo solo es útil cuando **editStyle** se establece en "Multiline".

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------------|
| Versión<br/> | Windows Media Player para Windows XP o posterior<br/> |
