# <a name="limited-user-access-support"></a>Compatibilidad con acceso de usuario limitado

Solo el administrador del equipo o los usuarios del grupo de usuarios registros de rendimiento pueden registrar los datos del contador. Los usuarios del grupo Administrador solo pueden registrar datos de contador si la herramienta que usan para registrar los datos del contador se inicia desde una ventana del símbolo del sistema que se abre con Ejecutar **como administrador...**. Los usuarios de sesiones de inicio de sesión interactivas pueden ver los datos del contador. Sin embargo, los usuarios de sesiones de inicio de sesión no interactivas deben estar en el grupo Usuarios de supervisión de rendimiento para ver los datos del contador.

**Windows XP:** El administrador o los usuarios del grupo Administrador pueden registrar y ver los datos del contador sin restricciones.
