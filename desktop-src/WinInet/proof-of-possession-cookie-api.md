# <a name="proof-of-possession-cookie-api"></a>API de cookie de prueba de posesión

Permite crear cookies de prueba de posesión, que admiten el inicio de sesión único para aplicaciones web autenticadas mediante el protocolo de autenticación de Internet OAuth. Puede usar la información devuelta por esta API para implementar el inicio de sesión único en cuentas Microsoft.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                         | Descripción                                                      |
|-----------------------------------------------------------------------------------------------|------------------------------------------------------------------|
| [**ISessionOfPossessionCookieInfoManager**](/windows/desktop/api/ProofOfPossessionCookieInfo/nn-proofofpossessioncookieinfo-iproofofpossessioncookieinfomanager)<br/> | Admite la creación de cookies de prueba de posesión.<br/> |
