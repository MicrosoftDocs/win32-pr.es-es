# <a name="ms-wmi-intmin-attribute"></a>atributo MS-WMI-intMin

El valor mínimo de un objeto de parámetro de entero de 32 bits de WMI.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-WMI-intMin                        |
| Nombre para mostrar de LDAP | msWMI-IntMin                         |
| Tamaño              | 4 bytes                              |
| Actualizar privilegio  | Administrador de directiva de grupo           |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.1630              |
| System-ID-GUID    | 68c2e3ba-9837-4c70-98e0-f0c33695d023 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
