# <a name="pen-functions-windows-gdi"></a>Funciones de lápiz (Windows GDI)

Las funciones siguientes se usan con lápices.



| Función                                       | Descripción                                                                                            |
|------------------------------------------------|--------------------------------------------------------------------------------------------------------|
| [**CreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-createpen)                 | Crea un lápiz lógico que tiene el estilo, ancho y color especificados.                                  |
| [**CreatePenIndirect**](/windows/desktop/api/Wingdi/nf-wingdi-createpenindirect) | Crea un lápiz lógico que tiene el estilo, el ancho y el color especificados en una estructura.          |
| [**ExtCreatePen**](/windows/desktop/api/Wingdi/nf-wingdi-extcreatepen)           | Crea un lápiz lógico geométrico o lógico que tiene los atributos de estilo, ancho y pincel especificados. |
| [**SetDCPenColor**](/windows/desktop/api/Wingdi/nf-wingdi-setdcpencolor)         | Establece el color del lápiz de contexto del dispositivo actual.                                                             |
