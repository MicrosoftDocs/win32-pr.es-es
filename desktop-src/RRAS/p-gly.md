<dl> <dt>

<span id="_rras_point_to_multipoint_interface_gly"></span><span id="_RRAS_POINT_TO_MULTIPOINT_INTERFACE_GLY"></span>**interfaz punto a multipunto**
</dt> <dd>

La tecnología de administrador de grupo de multidifusión utiliza este término para indicar una interfaz que proporciona comunicación entre un solo origen y un destino único, como un vínculo PPP.

</dd> <dt>

<span id="_rras_point_to_point_interface_gly"></span><span id="_RRAS_POINT_TO_POINT_INTERFACE_GLY"></span>**interfaz punto a punto**
</dt> <dd>

La tecnología de administrador de grupo de multidifusión utiliza este término para indicar una interfaz que proporciona comunicación entre un solo host y varios destinos. Las interfaces punto a multipunto se pueden considerar como una colección de vínculos punto a punto con una sola terminación, como un vínculo ATM.

</dd> </dl>
