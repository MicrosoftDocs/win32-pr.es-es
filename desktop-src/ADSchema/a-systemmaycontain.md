# <a name="system-may-contain-attribute"></a>Sistema: atributo puede contener

Lista de atributos opcionales para una clase. El sistema solo puede modificar la lista de atributos.



| Entrada | Value |
|-------------------|------------------------------------------------------------------------------|
| CN                | Sistema: puede contener                                                           |
| Nombre para mostrar de LDAP | systemMayContain                                                             |
| Tamaño              | \-                                                                           |
| Actualizar privilegio  | Administrador de esquema                                                         |
| Frecuencia de actualización  | Cuando se crea la clase o se agrega un nuevo atributo opcional a la clase. |
| Attribute-Id      | 1.2.840.113556.1.4.196                                                       |
| System-ID-GUID    | bf967a44-0de6-11d0-a285-00aa003049e2                                         |
| Sintaxis            | [**String(Object-Identifier)**](s-string-object-identifier.md)              |
