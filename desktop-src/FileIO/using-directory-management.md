# <a name="using-directory-management"></a>Uso de la administración de directorios

En los temas siguientes se muestra el uso de las funciones de administración de directorios.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                               | Descripción                                                                                                                                                                                                                                                                     |
|-----------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Cambiar el directorio actual](changing-the-current-directory.md)<br/>                     | Una aplicación puede cambiar el directorio actual llamando a la función SetCurrentDirectory.<br/>                                                                                                                                                                         |
| [Enumerar los archivos de un directorio](listing-the-files-in-a-directory.md)<br/>                 | Para enumerar los archivos de un directorio especificado, llame a FindFirstFile, FindNextFile y FindClose.<br/>                                                                                                                                                                             |
| [Mover directorios](moving-directories.md)<br/>                                             | Para trasladar un directorio a otra ubicación, junto con los archivos y subdirectorios contenidos en él, llame a la función [**MoveFileEx**](/windows/desktop/api/WinBase/nf-winbase-movefileexa), [**MoveFileWithProgress**](/windows/desktop/api/WinBase/nf-winbase-movefilewithprogressa)o [**MoveFileTransacted**](/windows/desktop/api/WinBase/nf-winbase-movefiletransacteda) .<br/> |
| [Obtención de notificaciones de cambio de directorio](obtaining-directory-change-notifications.md)<br/> | Una aplicación puede supervisar el contenido de un directorio y sus subdirectorios mediante notificaciones de cambios.<br/>                                                                                                                                                         |
