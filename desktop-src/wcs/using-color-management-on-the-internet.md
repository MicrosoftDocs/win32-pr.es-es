# <a name="using-color-management-on-the-internet"></a>Uso de la administración del color en Internet

Las imágenes de color y los gráficos son una forma cada vez más importante de comunicar información a los usuarios de Internet. La información de conversión de color se puede insertar en algunos formatos de archivo. Las técnicas de incrustación utilizadas actualmente no son compactas. Las consideraciones sobre el ancho de banda de Internet suelen hacer que sean poco prácticas. Las consideraciones de Internet incluyen los siguientes temas:

-   [sRGB: un espacio de colores estándar](srgb--a-standard-color-space.md)
-   [Valores predeterminados de WCS 1,0 para el espacio de colores de entrada y el perfil de salida](srgb--a-standard-color-space.md)
-   [Perfiles de sRGB y Embedded](srgb--a-standard-color-space.md)
