# <a name="list-box-styles"></a>Estilos de cuadro de lista

Para crear un cuadro de lista mediante las funciones [**CreateWindow**](/windows/desktop/api/winuser/nf-winuser-createwindowa) o [**CreateWindowEx,**](/windows/desktop/api/winuser/nf-winuser-createwindowexa) use la clase LISTBOX, las constantes de estilo de ventana adecuadas y las siguientes constantes de estilo para definir el cuadro de lista. Una vez creado el control, estos estilos no se pueden modificar, excepto como se indicó.



| Constante                                                                                                                                                                              | Descripción                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <span id="LBS_COMBOBOX"></span><span id="lbs_combobox"></span><dl> <dt>**LBS \_ COMBOBOX**</dt> </dl>                            | Notifica a un cuadro de lista que forma parte de un cuadro combinado. Esto permite la coordinación entre los dos controles para que presenten una interfaz de usuario unificada. El propio cuadro combinado debe establecer este estilo. Si el estilo se establece por cualquier elemento menos el cuadro combinado, el cuadro de lista se considerará incorrectamente como un elemento secundario de un cuadro combinado y se produce un error.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_DISABLENOSCROLL"></span><span id="lbs_disablenoscroll"></span><dl> <dt>**DISABLENOSCROLL DE LB \_**</dt> </dl>       | Muestra una barra de desplazamiento horizontal o vertical deshabilitada cuando el cuadro de lista no contiene suficientes elementos para desplazarse. Si no especifica este estilo, la barra de desplazamiento se oculta cuando el cuadro de lista no contiene suficientes elementos. Este estilo debe usarse con el [**estilo \_ WS VSCROLL**](/windows/desktop/winmsg/window-styles) o [**\_ WS HSCROLL.**](/windows/desktop/winmsg/window-styles)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_EXTENDEDSEL"></span><span id="lbs_extendedsel"></span><dl> <dt>**LBS \_ EXTENDEDSEL**</dt> </dl>                   | Permite seleccionar varios elementos mediante la tecla MAYÚS y las combinaciones de teclas especiales o del mouse.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| <span id="LBS_HASSTRINGS"></span><span id="lbs_hasstrings"></span><dl> <dt>**LBS \_ HASSTRINGS**</dt> </dl>                      | Especifica que un cuadro de lista contiene elementos que constan de cadenas. El cuadro de lista mantiene la memoria y las direcciones de las cadenas para que la aplicación pueda usar el mensaje [**\_ LB GETTEXT**](lb-gettext.md) para recuperar el texto de un elemento determinado. De forma predeterminada, todos los cuadros de lista excepto los cuadros de lista dibujados por el propietario tienen este estilo. Puede crear un cuadro de lista dibujado por el propietario con o sin este estilo.<br/> Para los cuadros de lista dibujados por el propietario sin este estilo, el mensaje [**\_ GETTEXT**](lb-gettext.md) de LB recupera el valor asociado a un elemento (los datos del elemento). <br/>                                                                                                                                                                                                                                                                                                                                    |
| <span id="LBS_MULTICOLUMN"></span><span id="lbs_multicolumn"></span><dl> <dt>**LBS \_ MULTICOLUMN**</dt> </dl>                   | Especifica un cuadro de lista de varias columnas que se desplaza horizontalmente. El cuadro de lista calcula automáticamente el ancho de las columnas o una aplicación puede establecer el ancho mediante el mensaje [**\_ LB SETCOLUMNWIDTH.**](lb-setcolumnwidth.md) Si un cuadro de lista tiene el estilo **LBS \_ OWNERDRAWFIXED,** una aplicación puede establecer el ancho cuando el cuadro de lista envía el [**mensaje \_ MEASUREITEM de WM.**](wm-measureitem.md) <br/> Un cuadro de lista con el **estilo \_ MULTICOLUMN de LBS** no puede desplazarse verticalmente, omite los mensajes [**\_ VSCROLL**](wm-vscroll.md) de WM que recibe. <br/> Los **estilos LBS \_ MULTICOLUMN** **y LBS \_ OWNERDRAWVARIABLE** no se pueden combinar. Si se especifican ambos, **se omite LBS \_ OWNERDRAWVARIABLE.**<br/>                                                                                                                                                                                |
| <span id="LBS_MULTIPLESEL"></span><span id="lbs_multiplesel"></span><dl> <dt>**MULTIPLESEL DE LB \_**</dt> </dl>                   | Activa o desactiva la selección de cadenas cada vez que el usuario hace clic o hace doble clic en una cadena en el cuadro de lista. El usuario puede seleccionar cualquier número de cadenas.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NODATA"></span><span id="lbs_nodata"></span><dl> <dt>**LBS \_ NODATA**</dt> </dl>                                  | Especifica un cuadro de lista sin datos. Especifique este estilo cuando el número de elementos del cuadro de lista supere los mil. Un cuadro de lista sin datos también debe tener el estilo **LBS \_ OWNERDRAWFIXED,** pero no debe tener el estilo **LBS \_ SORT** o **LBS \_ HASSTRINGS.** <br/> Un cuadro de lista sin datos es similar a un cuadro de lista dibujado por el propietario, salvo que no contiene datos de cadena o mapa de bits para un elemento. Los comandos para agregar, insertar o eliminar un elemento siempre omiten los datos de elementos especificados; Las solicitudes para buscar una cadena en el cuadro de lista siempre producirán un error. El sistema envía el [**mensaje \_ DRAWITEM de WM**](wm-drawitem.md) a la ventana de propietario cuando se debe dibujar un elemento. El **miembro itemID** de la [**estructura DRAWITEMSTRUCT**](/windows/win32/api/winuser/ns-winuser-drawitemstruct) pasada con el mensaje **\_ DRAWITEM de WM** especifica el número de línea del elemento que se va a dibujar. Un cuadro de lista sin datos no envía un [**mensaje \_ DELETEITEM de WM.**](wm-deleteitem.md)<br/> |
| <span id="LBS_NOINTEGRALHEIGHT"></span><span id="lbs_nointegralheight"></span><dl> <dt>**LBS \_ NOINTEGRALHEIGHT**</dt> </dl>    | Especifica que el tamaño del cuadro de lista es exactamente el tamaño especificado por la aplicación cuando creó el cuadro de lista. Normalmente, el sistema da tamaño a un cuadro de lista para que el cuadro de lista no muestre elementos parciales.<br/> Para los cuadros de lista **con el estilo LBS \_ OWNERDRAWVARIABLE,** siempre se aplica el estilo **LBS \_ NOINTEGRALHEIGHT.**<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| <span id="LBS_NOREDRAW"></span><span id="lbs_noredraw"></span><dl> <dt>**LBS \_ NOREDRAW**</dt> </dl>                            | Especifica que la apariencia del cuadro de lista no se actualiza cuando se realizan cambios.<br/> Para cambiar el estado de volver a dibujar del control, use el [**mensaje \_ WM SETREDRAW.**](/windows/desktop/gdi/wm-setredraw)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <span id="LBS_NOSEL"></span><span id="lbs_nosel"></span><dl> <dt>**LBS \_ NOSEL**</dt> </dl>                                     | Especifica que el cuadro de lista contiene elementos que se pueden ver pero no seleccionar. <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_NOTIFY"></span><span id="lbs_notify"></span><dl> <dt>**NOTIFICACIÓN DE \_ LB**</dt> </dl>                                  | Hace que el cuadro de lista envíe un código de notificación a la ventana primaria cada vez que el usuario haga clic en un elemento del cuadro de lista[(LBN \_ SELCHANGE),](lbn-selchange.md)haga doble clic en un elemento[(LBN \_ DBLCLK)](lbn-dblclk.md)o cancele la selección[(LBN \_ SELCANCEL).](lbn-selcancel.md) <br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| <span id="LBS_OWNERDRAWFIXED"></span><span id="lbs_ownerdrawfixed"></span><dl> <dt>**LBS \_ OWNERDRAWFIXED**</dt> </dl>          | Especifica que el propietario del cuadro de lista es responsable de dibujar su contenido y que los elementos del cuadro de lista tienen el mismo alto. La ventana de propietario recibe un [**mensaje \_ MEASUREITEM de WM**](wm-measureitem.md) cuando se crea el cuadro de lista y un mensaje [**\_ DRAWITEM**](wm-drawitem.md) de WM cuando ha cambiado un aspecto visual del cuadro de lista.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| <span id="LBS_OWNERDRAWVARIABLE"></span><span id="lbs_ownerdrawvariable"></span><dl> <dt>**LBS \_ OWNERDRAWVARIABLE**</dt> </dl> | Especifica que el propietario del cuadro de lista es responsable de dibujar su contenido y que los elementos del cuadro de lista tienen un alto variable. La ventana de propietario recibe un [**mensaje \_ MEASUREITEM**](wm-measureitem.md) de WM para cada elemento del cuadro cuando se crea el cuadro de lista y un mensaje [**\_ DRAWITEM**](wm-drawitem.md) de WM cuando ha cambiado un aspecto visual del cuadro de lista.<br/> Este estilo hace que el **\_ estilo LBS NOINTEGRALHEIGHT** esté habilitado.<br/> Este estilo se omite si se especifica el estilo **\_ MULTICOLUMN** de LBS. <br/>                                                                                                                                                                                                                                                                                                                                                                  |
| <span id="LBS_SORT"></span><span id="lbs_sort"></span><dl> <dt>**SORT de \_ LB**</dt> </dl>                                        | Ordena las cadenas en el cuadro de lista alfabéticamente.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| <span id="LBS_STANDARD"></span><span id="lbs_standard"></span><dl> <dt>**ESTÁNDAR DE LB \_**</dt> </dl>                            | Ordena las cadenas en el cuadro de lista alfabéticamente. La ventana primaria recibe un código de notificación cada vez que el usuario hace clic en un elemento del cuadro de lista, hace doble clic en un elemento o cancela la selección. El cuadro de lista tiene una barra de desplazamiento vertical y tiene bordes en todos los lados. Este estilo combina los estilos **LBS \_ NOTIFY**, **LBS \_ SORT,** [**WS \_ VSCROLL**](/windows/desktop/winmsg/window-styles)y [**WS \_ BORDER.**](/windows/desktop/winmsg/window-styles)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| <span id="LBS_USETABSTOPS"></span><span id="lbs_usetabstops"></span><dl> <dt>**LBS \_ USETABSTOPS**</dt> </dl>                   | Permite que un cuadro de lista reconozca y expanda caracteres de tabulación al dibujar sus cadenas. Puede usar el mensaje [**\_ LB SETTABSTOPS para**](lb-settabstops.md) especificar las posiciones de tabulación. Las posiciones de tabulación predeterminadas están separadas por 32 unidades de plantilla de cuadro de diálogo. Las unidades de plantilla de diálogo son las unidades independientes del dispositivo que se usan en las plantillas de cuadro de diálogo. Para convertir medidas de unidades de plantilla de diálogo en unidades de pantalla (píxeles), use la [**función MapDialogRect.**](/windows/desktop/api/winuser/nf-winuser-mapdialogrect)<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| <span id="LBS_WANTKEYBOARDINPUT"></span><span id="lbs_wantkeyboardinput"></span><dl> <dt>**LBS \_ WANTKEYBOARDINPUT**</dt> </dl> | Especifica que el propietario del cuadro de lista recibe mensajes [**\_ WM VKEYTOITEM**](wm-vkeytoitem.md) cada vez que el usuario presiona una tecla y el cuadro de lista tiene el foco de entrada. Esto permite que una aplicación realice un procesamiento especial en la entrada del teclado.<br/>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
