# <a name="bits-reference"></a>Referencia de BITS

La Servicio de transferencia inteligente en segundo plano de referencia (BITS) contiene información detallada sobre los siguientes elementos de API:

-   [Enumeraciones](bits-enumerations.md)
-   [Interfaces](bits-interfaces.md)
-   [Valores devueltos](bits-return-values.md)
-   [Estructuras y uniones](bits-c---structures-and-unions.md)

Estos elementos de API se usan para crear aplicaciones que transfieren archivos entre un cliente y un servidor. Para obtener código de ejemplo que usa las interfaces bits, vea [Ejemplos y herramientas de BITS.](bits-samples-and-tools.md)

Para obtener información general sobre BITS, vea la [página de inicio de BITS](background-intelligent-transfer-service-portal.md).
