# <a name="tasknamedvaluecollectionremove-method"></a>Método TaskNamedValueCollection.Remove

Para el scripting, quita un par nombre-valor seleccionado de la colección.

## <a name="syntax"></a>Sintaxis


```VB
TaskNamedValueCollection.Remove( _
  ByVal index _
)
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*index* \[ En\]
</dt> <dd>

Índice del par nombre-valor que se va a quitar.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Este método no devuelve ningún valor.
