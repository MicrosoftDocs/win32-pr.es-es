# <a name="initial-medial-and-final-search-strings"></a>Cadenas de búsqueda inicial, medial y final

Hay tres tipos de búsquedas con caracteres comodín a los que se hace referencia en la documentación sobre la construcción de consultas de cadenas de búsqueda. Estas búsquedas de caracteres comodín son: iniciales-, medial-y las cadenas de búsqueda final. En este tema se describen estas búsquedas.

## <a name="initial-search-strings"></a>Cadenas de búsqueda iniciales

Las cadenas de búsqueda inicial coinciden con un conjunto determinado de caracteres al principio de una cadena, seguido de un carácter comodín. Por ejemplo, la cadena de búsqueda inicial `Act*` coincidiría con "Active Directory".

## <a name="medial-search-strings"></a>Cadenas de Medial-Search

Las cadenas de búsqueda medial coinciden con un conjunto determinado de caracteres en el medio de una cadena, precedidos por y/o seguidos de un carácter comodín. Las cadenas de búsqueda medial `*Dir*` , `*ive*Dir*` y `*ve*Dir*tor*` coincidirían con "Active Directory".

## <a name="final-search-strings"></a>Cadenas de búsqueda final

Las cadenas de búsqueda final coinciden con un conjunto determinado de caracteres al final de una cadena, precedidos por un carácter comodín. Por ejemplo, la cadena de búsqueda final `*ory` coincidiría con "Active Directory".
