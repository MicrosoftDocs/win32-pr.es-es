# <a name="summary-of-namespace-provider-functions"></a>Resumen de las funciones del proveedor de espacios de nombres

Las funciones del proveedor de servicios de espacio de nombres se pueden agrupar en cinco categorías:

-   Configuración del proveedor de espacio de nombres (e instalación)
-   Inicialización del proveedor
-   Instalación del servicio
-   Consultas de cliente
-   Funciones auxiliares (y macros)

En las secciones siguientes se identifican las funciones de cada categoría y se describe brevemente su uso previsto. También se describen las estructuras de datos clave.

-   [Inicialización y limpieza del proveedor de espacios de nombres](name-space-provider-initialization-and-cleanup-2.md)
-   [Instalación del servicio en Windows Sockets 2 SPI](service-installation-in-the-windows-sockets-2-spi-2.md)
-   [Consulta de servicio](service-query-2.md)
-   [Funciones auxiliares en el SPI](helper-functions-in-the-spi-2.md)
