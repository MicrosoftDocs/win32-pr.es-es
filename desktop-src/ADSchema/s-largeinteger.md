Representa un valor entero de 64 bits con signo.

|--------------|------------------------------------------------------------------------------------|
| Nombre         | LargeInteger                                                                       |
| Identificador de sintaxis    | 2.5.5.16                                                                           |
| Id. de OM        | 65                                                                                 |
| Tipo DE ARCHIVO    | \-                                                                                 |
| ADS Type     | ADSTYPE \_ LARGE \_ INTEGER                                                            |
| Tipo de variante | VT \_ DISPATCH                                                                       |
