# <a name="pathgradientbrushsetcenterpoint-methods"></a>Métodos PathGradientBrush.SetCenterPoint

En este tema se enumeran los métodos SetCenterPoint de [**la clase PathGradientBrush.**](/windows/win32/api/gdipluspath/nl-gdipluspath-pathgradientbrush) Para obtener una lista completa de los métodos **de la clase PathGradientBrush,** vea [Métodos PathGradientBrush](-gdiplus-class-pathgradientbrush-methods.md).

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                           | Descripción                                                                                                                                                                                                                                                                                                                                    |
|:-------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**SetCenterPoint(Point&)**](/windows/win32/api/gdipluspath/nf-gdipluspath-pathgradientbrush-setcenterpoint(inconstpoint_))   | El [**método PathGradientBrush::SetCenterPoint**](/windows/win32/api/gdipluspath/nf-gdipluspath-pathgradientbrush-setcenterpoint(inconstpoint_)) establece el punto central de este pincel de degradado de ruta de acceso. De forma predeterminada, el punto central está en el centroide de la ruta de acceso de límite del pincel, pero puede establecer el punto central en cualquier ubicación dentro o fuera de la ruta de acceso.<br/>  |
| [**SetCenterPoint(PointF&)**](/previous-versions//ms535095(v=vs.85)) | El [**método PathGradientBrush::SetCenterPoint**](/previous-versions//ms535095(v=vs.85)) establece el punto central de este pincel de degradado de ruta de acceso. De forma predeterminada, el punto central está en el centroide de la ruta de acceso de límite del pincel, pero puede establecer el punto central en cualquier ubicación dentro o fuera de la ruta de acceso.<br/> |
