# <a name="overview-of-mgm-callbacks"></a>Introducción a las devoluciones de llamada de MGM

Hay dos tipos de devoluciones de llamada en la API de MGM:

-   [Devoluciones de llamada del protocolo de enrutamiento](routing-protocol-callbacks.md)
-   [Habilitar y deshabilitar devoluciones de llamada de IGMP](igmp-enable-and-disable-callbacks.md)

Estas devoluciones de llamada, combinadas con las API de MGM, permiten la comunicación continua entre los protocolos de enrutamiento, IGMP y el administrador de grupos de multidifusión.
