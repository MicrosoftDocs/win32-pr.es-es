# <a name="rights-guid-attribute"></a>Rights-Guid atributo

GUID que se usa para representar un derecho extendido dentro de una entrada de control de acceso.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | Rights-Guid                                 |
| Ldap-Display-Name | rightsGuid                                  |
| Size              | 16 bytes                                    |
| Privilegio actualizar  | El sistema establece este valor.            |
| Frecuencia de actualización  | Cuando se crea un nuevo derecho extendido.       |
| Attribute-Id      | 1.2.840.113556.1.4.340                      |
| System-Id-Guid    | 8297931c-86d3-11d0-afda-00c04fd930c9        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
