# <a name="cryptography-essentials"></a>Cryptography Essentials

Microsoft CryptoAPI, la interfaz de programación de aplicaciones (API) y CAPICOM proporcionan servicios que permiten a los desarrolladores agregar seguridad basada en [*criptografía a*](../secgloss/c-gly.md) las aplicaciones. CryptoAPI incluye funcionalidad para codificar y descodificar desde La notación de sintaxis abstracta [*1*](../secgloss/a-gly.md) (ASN.1), cifrar y descifrar datos, autenticación mediante certificados [*digitales*](../secgloss/d-gly.md)y administración de certificados digitales mediante almacenes de certificados. CryptoAPI y CAPICOM admiten criptografía de clave simétrica y PKI.

-   [Datos codificados y descodificados](encoded-and-decoded-data.md)
-   [Cifrado y descifrado de datos](data-encryption-and-decryption.md)
-   [Hashes y firmas digitales](hashes-and-digital-signatures.md)
-   [Firmas de authenticode de marca de tiempo](time-stamping-authenticode-signatures.md)
-   [Certificados digitales](digital-certificates.md)
-   [Mensajes digitales](digital-messages.md)
