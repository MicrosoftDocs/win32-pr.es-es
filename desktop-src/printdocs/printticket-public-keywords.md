# <a name="printticket-public-keywords"></a>Palabras clave públicas de PrintTicket

Este tema no está actualizado. Para obtener la información más reciente, consulte la [especificación del esquema de impresión](https://download.microsoft.com/download/D/E/C/DECA6E6B-3E81-48E7-B7EF-6D92A547D03C/print-schema-spec-2-0.zip).

En la sección siguiente se especifican las descripciones de propiedades específicas de PrintTicket y los inicializadores de parámetros que pueden ser aplicables a un PrintTicket.

## <a name="printticket-specific-property-descriptions-usage"></a>Uso de descripciones de propiedades específicas de PrintTicket

Los descriptores de propiedad específicos de PrintTicket representan palabras clave públicas del esquema de impresión que pueden aparecer en un PrintTicket. Contienen información específica que una aplicación o un usuario pueden desear especificar en un PrintTicket.

## <a name="parameter-initializers-usage"></a>Uso de inicializadores de parámetros

Los inicializadores de parámetro adoptan la forma de los tipos de elementos ParameterInit de un PrintTicket. Los tipos de elementos ParameterInit contienen el valor del parámetro y se inicializan dentro de un PrintTicket. Para obtener más información, consulte la sección [construcciones de parámetros](parameter-constructs.md) .
