# <a name="windows-media-drm-10-for-network-devices"></a>Windows DRM multimedia 10 para dispositivos de red

La aplicación puede transmitir contenido protegido a receptores de medios digitales en una red mediante Windows Drm multimedia 10 para dispositivos de red. Esta característica es un protocolo para comunicarse de forma segura con dispositivos de red.

El SDK de formato multimedia de Windows admite Windows Media DRM 10 para dispositivos de red mediante el mantenimiento de una base de datos de dispositivos de red, la validación de la proximidad de red de esos dispositivos y la conversión de contenido protegido por DRM a un formato de transporte seguro.

> [!Note]  
> DRM no es compatible con la versión basada en x64 de este SDK.
