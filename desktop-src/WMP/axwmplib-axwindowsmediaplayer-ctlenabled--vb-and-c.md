# <a name="axwindowsmediaplayerctlenabled-property"></a>Propiedad AxWindowsMediaPlayer. Ctlenabled

La propiedad Ctlenabled obtiene o establece un valor que indica si el control Media Player de Windows está habilitado.

## <a name="syntax"></a>Sintaxis


```CSharp
public System.Boolean Ctlenabled {get; set;}
```


```VB

Public Property Ctlenabled As System.Boolean
```





## <a name="property-value"></a>Valor de propiedad

Valor System. Boolean que indica si el control de Media Player de Windows está habilitado. El valor predeterminado es true.

## <a name="remarks"></a>Observaciones

Si la propiedad **Ctlenabled** se establece en false, Windows Media Player oculta los controles de usuario durante la reproducción a pantalla completa.
