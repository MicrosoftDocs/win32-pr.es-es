# <a name="link-id-attribute"></a>Atributo Link-ID

Un entero que indica que el atributo es un atributo vinculado. Un entero par es un vínculo hacia delante y un entero impar es un vínculo hacia atrás.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | IDENTIFICADOR de vínculo                              |
| Nombre para mostrar de LDAP | linkID                               |
| Tamaño              | \-                                   |
| Actualizar privilegio  | Administrador de esquema                 |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.2.50                |
| System-ID-GUID    | bf96799b-0de6-11d0-a285-00aa003049e2 |
| Sintaxis            | [**Enumeración**](s-enumeration.md) |
