# <a name="retrieving-wmi-class-or-instance-data"></a>Recuperación de datos de clase o instancia de WMI

Los principales objetos de datos en WMI son clases e instancias. En la tabla siguiente se enumeran las distintas técnicas disponibles para recuperar datos de una clase o una instancia de WMI.



| Tema                                                                   | Descripción                                                                       |
|-------------------------------------------------------------------------|-----------------------------------------------------------------------------------|
| [Recuperación de una clase WMI](retrieving-a-class.md)                        | Recuperación de información sobre los parámetros generales de un objeto.                    |
| [Recuperación de una instancia de WMI](retrieving-an-instance.md)                 | Recuperación de la información que describe un objeto individual único de la empresa. |
| [Recuperar parte de una instancia de WMI](retrieving-part-of-an-instance.md) | Recuperar una o más partes de una instancia, pero no toda la instancia.         |
| [Obtener acceso a un calificador de WMI](accessing-a-qualifier.md)                  | Obtener acceso a más información contenida en un calificador.                              |



 

Para obtener más información, vea [manipular la información de clase e instancia](manipulating-class-and-instance-information.md).
