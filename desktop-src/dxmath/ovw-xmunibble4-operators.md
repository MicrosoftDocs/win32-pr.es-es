# <a name="xmunibble4-operators"></a>Operadores XMUNIBBLE4

La estructura [**XMUNIBBLE4**](/windows/win32/api/directxpackedvector/ns-directxpackedvector-xmunibble4) expone los siguientes operadores.

> [!Note]  
> Los operadores que se enumeran aquí solo están disponibles en C++.

 

## <a name="in-this-section"></a>En esta sección



| Métodos                                                               | Descripción                                                                                                              |
|-----------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------|
| [**Operator UInt16 \_ t**](/windows/win32/api/directxpackedvector/nf-directxpackedvector-xmunibble4-operatoruint16_t)<br/> | Devuelve una instancia de `uint16_t` que contiene los componentes de la `XMUNIBBLE4` instancia en un formato empaquetado. <br/> |
| [**operador =**](xmunibble4-operator-eq.md)<br/>               | Asigna datos de componentes vectoriales a la instancia actual de `XMUNIBBLE4` . <br/>                                       |
