# <a name="iscanprofilemgrdeleteallprofilesforuser-method"></a>IScanProfileMgr::D método eleteAllProfilesForUser

Elimina todos los perfiles de digitalización disponibles para el usuario en el sistema en el que se ejecuta la aplicación.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT DeleteAllProfilesForUser();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **HRESULT**

Si este método se ejecuta correctamente, devuelve **S \_ correcto**. De lo contrario, devuelve un código de error **HRESULT** .
