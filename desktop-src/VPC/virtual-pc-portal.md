# <a name="windows-virtual-pc"></a>Windows Virtual PC

\[Windows El equipo virtual ya no está disponible para su uso a Windows 8. En su lugar, use [el proveedor WMI de Hyper-V (V2).](../hyperv_v2/windows-virtualization-portal.md)\]

## <a name="purpose"></a>Propósito

En esta documentación se proporciona información sobre Windows interfaz COM de PC virtual.

## <a name="where-applicable"></a>Donde sea aplicable

Windows Virtual PC es la tecnología de virtualización de Microsoft más reciente; permite ejecutar muchas aplicaciones de productividad en un entorno Windows virtual, con un solo clic, directamente desde Windows 7. Puede crear máquinas virtuales independientes sobre el escritorio Windows 7. Cada máquina virtual emula un sistema de hardware completo (desde el procesador hasta la tarjeta de red) en un entorno de software aislado independiente, lo que permite el funcionamiento simultáneo de sistemas incompatibles de otro modo.

## <a name="developer-audience"></a>Audiencia de desarrolladores

Las Windows COM de Pc virtual son para desarrolladores que crean aplicaciones cliente que automatizan la implementación y el funcionamiento de las máquinas virtuales.

## <a name="run-time-requirements"></a>Requisitos de tiempo de ejecución

Windows Virtual PC requiere una de las siguientes ediciones Windows 7: Home Basic, Home Premium, Professional, Ultimate o Enterprise edition.

Para obtener los archivos de encabezado y biblioteca relacionados, instale el SDK de Windows para Windows 7 desde el Centro [de descarga de Microsoft](https://www.microsoft.com/download/details.aspx?id=8279).
