# <a name="rating"></a>Rating

El atributo Rating contiene la clasificación del contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMRating

## <a name="data-type"></a>Tipo de datos

**CADENA DE \_ TIPO \_ WMT**

## <a name="remarks"></a>Comentarios

Se trata de un atributo de nivel de archivo.

Las implementaciones individuales de los archivos existentes usan diferentes escalas y notaciones. Por este problema, debe evitar el uso de este atributo; no hay ninguna normalización a partir de la cual desarrollar interfaces de usuario coherentes.
