# <a name="active-directory-schema-ad-ds"></a>Active Directory esquema (AD DS)

El esquema de Active Directory contiene definiciones formales de cada clase de objeto que se puede crear en un bosque de Active Directory. El esquema también contiene definiciones formales de cada atributo que puede existir en un Active Directory objeto .

Entre los temas que proporcionan información general sobre Active Directory esquema se incluyen:

-   [Implementación de esquema](schema-implementation.md)
-   [Características de las clases Object](characteristics-of-object-classes.md)
-   [Características de los atributos](characteristics-of-attributes.md)
-   [Esquema abstracto](the-abstract-schema.md)

Para obtener más información sobre la programación de esquemas, incluida la lectura del esquema y la definición de nuevas clases y atributos en el esquema, vea [Extender el esquema](extending-the-schema.md).

Para obtener más información, vea la [Active Directory Domain Services referencia](active-directory-domain-services-reference.md). Para obtener las páginas de referencia de las clases de esquema predefinidas, los atributos y las [sintaxis](/windows/desktop/ADSchema/active-directory-schema)de atributo, vea la referencia de Active Directory esquema .
