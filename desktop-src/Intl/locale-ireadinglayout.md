# <a name="locale_ireadinglayout"></a>CONFIGURACIÓN REGIONAL \_ IREADINGLAYOUT

**Windows 7 y versiones posteriores:** Diseño de lectura para texto. Los valores posibles se definen en la tabla siguiente.



| Value | Significado                                                                                                                                                             |
|-------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 0     | Lea de izquierda a derecha, como en la configuración regional inglés (Estados Unidos).                                                                                                 |
| 1     | Lea de derecha a izquierda, como en el caso de las configuraciones regionales en árabe.                                                                                                                     |
| 2     | Lea verticalmente de arriba abajo con columnas que van de derecha a izquierda, o bien lea en filas horizontales de izquierda a derecha, como en la configuración regional japonesa (Japón). |
| 3     | Lea verticalmente de arriba a abajo con columnas que van de izquierda a derecha, como para la configuración regional de Los arácn (arácn).                                                  |
