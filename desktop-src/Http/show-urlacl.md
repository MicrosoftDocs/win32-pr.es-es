# <a name="show-urlacl"></a>show urlacl

Muestra las listas DACL de la dirección URL reservada especificada o de todas las direcciones URL reservadas.

``` syntax
show urlacl [url=]string
 
```

## <a name="parameters"></a>Parámetros

<dl> <dt>

<span id="_url__string"></span><span id="_URL__STRING"></span>**\[URL = \] * * * cadena*
</dt> <dd>

Especifica la dirección URL completa. Si no se especifica, implica todas las direcciones URL.

</dd> </dl>

## <a name="examples"></a>Ejemplos

**Mostrar dirección URL de urlacl =https://+:80/MyUrl**

**Mostrar dirección URL de urlacl =https://www.contoso.com:80/MyUrl**
