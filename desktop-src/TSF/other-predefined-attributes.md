# <a name="other-predefined-attributes"></a>Otros atributos predefinidos

Los valores siguientes identifican los atributos varios obtenidos con el método [ITfContext:: GetAppProperty](/windows/desktop/api/msctf/nf-msctf-itfcontext-getappproperty) . Se incluyen el formato de datos y el contenido de cada tipo de propiedad.

## <a name="properties"></a>Propiedades



| Propiedad                             | Descripción                                                                       |
|--------------------------------------|-----------------------------------------------------------------------------------|
| **\_Aplicación TSATTRID**                    | No se utiliza.                                                                         |
| **IncorrectGrammar de la \_ aplicación TSATTRID \_**  | Contiene un valor distinto de cero si el texto contiene un error gramatical o cero de lo contrario.  |
| **IncorrectSpelling de la \_ aplicación TSATTRID \_** | Contiene un valor distinto de cero si el texto contiene un error ortográfico o cero de lo contrario. |
| **TSATTRID \_ otros**                 | No se utiliza.                                                                         |
