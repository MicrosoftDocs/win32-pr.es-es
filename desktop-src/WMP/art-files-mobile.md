# <a name="art-files-mobile"></a>Archivos de imagen (móviles)

Tendrá que crear archivos de imagen para la máscara. Hay varios tipos de archivos de imagen que se pueden crear y deben estar en un formato de archivo de arte específico.

En la tabla siguiente se describen siete tipos de archivos de imagen que se pueden crear para la máscara.



| Tipo de archivo de imagen                            | Descripción                                                                                                |
|------------------------------------------|------------------------------------------------------------------------------------------------------------|
| [Archivo en segundo plano](background-file.md)   | Obligatorio. El fondo es la imagen en la que se superponen todas las imágenes de botón                        |
| [Archivo deshabilitado](disabled-file.md)       | Obligatorio. Las imágenes que se usan para indicar que los botones de inserción no tendrán ningún efecto.                                |
| [Archivo insertado](pushed-file.md)           | Obligatorio. Define las imágenes que se muestran cuando el usuario presiona los botones.                              |
| [Archivos de regiones](region-files.md)         | Define mapas de bits que usan áreas de color para representar las ubicaciones de la respuesta de punteo de los botones de tipo de posicionamiento      |
| [Súper archivos](super-files.md)           | Define las imágenes deshabilitadas para trackbars.                                                                 |
| [Archivo SeekThumb](seekthumb-file.md)     | Define una imagen que se va a usar junto con una barra de contenido para indicar la posición actual en el elemento multimedia. |
| [Archivo VolumeThumb](volumethumb-file.md) | Define una imagen que se va a usar junto con una barra de control para indicar la posición del control de volumen.            |



 

En las secciones siguientes se proporciona más información acerca de los archivos de imagen.

-   [Formatos de archivo de imagen](art-file-formats-mobile.md)
-   [Paletas de colores](color-palettes.md)
-   [Tamaños de imagen](image-sizes.md)
-   [Ubicación de los botones](button-placement.md)

Para obtener más información sobre los tipos de archivos de imagen y cómo identificarlos en el archivo de definición de máscara, vea la sección [mapas de bits](bitmaps.md) de la referencia de máscara.
