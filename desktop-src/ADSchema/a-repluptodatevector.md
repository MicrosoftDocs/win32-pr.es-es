# <a name="repl-uptodate-vector-attribute"></a>REPL-UpToDate-vector (atributo)

Realiza el seguimiento de la información de estado de replicación interna para un NC completo. La información aquí se puede extraer de forma pública a través de la API DsReplicaGetInfo (). Presente en todos los objetos raíz de NC.



| Entrada | Value |
|-------------------|--------------------------------------------------------------------------------|
| CN                | REPL-UpToDate-Vector                                                           |
| Nombre para mostrar de LDAP | Del repluptodatevector                                                             |
| Tamaño              | La longitud es proporcional al número de réplicas (pasadas y presentes) del NC. |
| Actualizar privilegio  | El sistema establece este valor.                                               |
| Frecuencia de actualización  | Cambios en respuesta a los ciclos de replicación de entrada completados.                   |
| Attribute-Id      | 1.2.840.113556.1.4.4                                                           |
| System-ID-GUID    | bf967a16-0de6-11d0-a285-00aa003049e2                                           |
| Sintaxis            | [**Object(Replica-Link)**](s-object-replica-link.md)                          |
