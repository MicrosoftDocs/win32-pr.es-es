# <a name="jet_recsizeclongvalues-property"></a>JET_RECSIZE.cLongValues, propiedad

Obtiene el número total de valores long almacenados en el árbol de valores largos para este registro. Esto no incluye valores long intrínsecos.

**Espacio de nombres:**  [Microsoft.Isam.Esent.Interop.Vista](./microsoft.isam.esent.interop.vista-namespace.md)
