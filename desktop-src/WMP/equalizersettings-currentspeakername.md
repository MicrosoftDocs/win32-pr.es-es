# <a name="equalizersettingscurrentspeakername"></a>EQUALIZERSETTINGS.currentSpeakerName

El atributo **currentSpeakerName** recupera el nombre de la configuración del orador actual.

``` syntax
        elementID.currentSpeakerName
```

## <a name="possible-values"></a>Valores posibles

Este atributo es una **cadena** de solo lectura que contiene uno de los valores siguientes.



| Value           | Descripción                              |
|-----------------|------------------------------------------|
| Auriculares      | Los altavoces actuales son auriculares.     |
| Altavoces normales | Los altavoces actuales tienen un tamaño normal. |
| Altavoces grandes  | Los altavoces actuales son grandes.          |



 

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------|
| Versión<br/> | Windows Media Player 9 series o posterior<br/> |
