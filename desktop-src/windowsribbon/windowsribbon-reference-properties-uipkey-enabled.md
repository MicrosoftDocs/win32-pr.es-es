# <a name="ui_pkey_enabled"></a>Ui \_ PKEY \_ habilitado

Identifica la propiedad ui \_ PKEY \_ Enabled.

```
propertyDescription
   name = UI_PKEY_Enabled
   shellPKey = UI_PKEY_Enabled
   formatID = 00000004-7363-696e-8441798acf5aebb7
   propID = 1
   typeInfo
      type = VT_BOOL
   booleanFormat
      formatAs = VARIANT_TRUE=-1, VARIANT_FALSE=0
```

## <a name="remarks"></a>Observaciones

Una aplicación usa ui PKEY Enabled para consultar si un control está habilitado y \_ puede responder a la interacción del \_ usuario.

> [!Note]  
> El valor no se establece automáticamente en 0 cuando se cambia el modo de aplicación y se desactiva la visibilidad de un comando.
