# <a name="property-pages-and-property-sheets"></a>Páginas de propiedades y hojas de propiedades

Las páginas de propiedades OLE permiten que un objeto muestre sus propiedades en un cuadro de diálogo con pestañas conocido como hoja de propiedades. A continuación, un usuario final puede ver y cambiar las propiedades del objeto. Un objeto puede mostrar sus páginas de propiedades independientemente de su cliente o el cliente puede administrar la presentación de páginas de propiedades a partir de varios objetos contenidos en una sola hoja de propiedades. Las páginas de propiedades también proporcionan un medio para notificar a un cliente los cambios en las propiedades de un objeto.

Cualquier objeto que quiera proporcionar una interfaz de usuario para sus propiedades cambiantes puede usar esta tecnología.
