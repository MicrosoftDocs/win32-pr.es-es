# <a name="remoteapp-classes"></a>Clases RemoteApp

El proveedor WMI de RemoteApp proporciona las clases siguientes.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**\_RdAllowListFileAssociation de Win32**](win32-rdallowlistfileassociation.md)
</dt> <dd>

Describe una asociación de tipo de archivo publicado con remoteApp.

</dd> <dt>

[**RdFileAssociation de Win32 \_**](win32-rdfileassociation.md)
</dt> <dd>

Representa una asociación de tipo de archivo para remoteApp.

</dd> <dt>

[**Win32 \_ RemoteAppChangeEvent**](win32-remoteappchangeevent.md)
</dt> <dd>

Representa un cambio en Windows configuración de RemoteApp de Server 2008 R2 en el servidor Escritorio remoto Host de sesión de Escritorio remoto.

</dd> <dt>

[**TSApplicationFileExtensions de Win32 \_**](win32-tsapplicationfileextensions.md)
</dt> <dd>

las extensiones de nombre de archivo que administra una aplicación en un servidor host de sesión de Escritorio remoto.

</dd> <dt>

[**Win32 \_ TSDeploymentLicensing**](win32-tsdeploymentlicensing.md)
</dt> <dd>

Licencias Configuración para la implementación Escritorio remoto Virtualización de aplicaciones (RDV).

</dd> <dt>

[**Win32 \_ TSDeploymentSettings**](win32-tsdeploymentsettings.md)
</dt> <dd>

Define la configuración predeterminada que el Administrador de RemoteApp usa al crear archivos Protocolo de escritorio remoto (RDP).

</dd> <dt>

[**Win32 \_ TSExpandEnvironmentVariables**](win32-tsexpandenvironmentvariables.md)
</dt> <dd>

Expande las variables de entorno definidas por el sistema.

</dd> <dt>

[**TSGetIcon de Win32 \_**](win32-tsgeticon.md)
</dt> <dd>

Devuelve el contenido del icono especificado por la ruta de acceso del archivo y el índice.

</dd> <dt>

[**TSPublishedApplication de Win32 \_**](win32-tspublishedapplication.md)
</dt> <dd>

Define las aplicaciones que están disponibles para su uso remoto mediante Windows Server 2008 R2 RemoteApp.

</dd> <dt>

[**Win32 \_ TSPublishedApplicationList**](win32-tspublishedapplicationlist.md)
</dt> <dd>

Representa la lista de programas que se encuentran en la lista Programas de RemoteApp en un servidor host de sesión de Escritorio remoto.

</dd> <dt>

[**Win32 \_ TSRemoteDesktop**](win32-tsremotedesktop.md)
</dt> <dd>

Describe una conexión a Escritorio remoto que está disponible a través Escritorio remoto Web Access (Acceso web de Escritorio remoto).

</dd> <dt>

[**TSStartMenuApplication de Win32 \_**](win32-tsstartmenuapplication.md)
</dt> <dd>

Describe las aplicaciones que se encuentran en la menú Inicio de un servidor host de sesión de Escritorio remoto.

</dd> <dt>

[**TSSystemInfo de Win32 \_**](win32-tssysteminfo.md)
</dt> <dd>

Representa el servicio de información del servidor de publicación centralizado

</dd> </dl>
