# <a name="enumerating-routing-table-entries"></a>Enumerar entradas de tabla de enrutamiento

Las funciones de enumeración permiten a un cliente recuperar información sobre un tipo específico de objeto de tabla de enrutamiento (rutas, destinos y próximo saltos). Tanto los protocolos de enrutamiento como los programas de administración pueden usar estas funciones para buscar datos específicos.

Para obtener código de ejemplo que muestra cómo usar estas funciones, vea [Enumerar todos](enumerate-all-destinations.md) los destinos y [Enumerar todas las rutas.](enumerate-all-routes.md)
