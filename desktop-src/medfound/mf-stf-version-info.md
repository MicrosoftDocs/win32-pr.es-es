# <a name="mf_stf_version_info-attribute"></a>\_Atributo de \_ información de versión MF STF \_

Este atributo representa el generador de transformación del sensor de información de versión. Este es el único atributo que se requiere para las transformaciones del sensor. El desarrollador de transformación del sensor define el valor y se trata como opaco por la canalización multimedia.

## <a name="data-type"></a>Tipo de datos

**WCHAR\***
