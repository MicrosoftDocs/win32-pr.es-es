# <a name="bits-samples-and-tools"></a>Herramientas y ejemplos de BITS

La sección siguiente contiene ejemplos de C++ paso a paso para Servicio de transferencia inteligente en segundo plano (BITS).



| Sección                                                            | Propósito                                                                                      |
|--------------------------------------------------------------------|----------------------------------------------------------------------------------------------|
| [Ejemplos de aplicaciones de BITS C++](bits-c---application-examples.md) | Ejemplos de C++ que muestran una serie de tareas que se pueden completar mediante características de BITS. |



 

Además, BITS incluye los ejemplos siguientes en Windows SDK. Los ejemplos se encuentran en *InstallDirectory* Samples Web (Web de ejemplos de \\ \\ InstallDirectory). Cada ejemplo incluye un Readme.txt que explica cómo instalar y ejecutar el ejemplo. Puede descargar el SDK Windows de [Microsoft Windows Software Development Kit](https://msdn.microsoft.com/windowsserver/bb980924.aspx). BITS se incluye en core SDK.

> [!Note]  
> Windows Vista: los ejemplos de esta página se incluyen en [el SDK de Microsoft Windows para Windows Vista](https://www.microsoft.com/download/details.aspx?id=30998). Sin embargo, no se incluye el ejemplo ClientCert.

 



| Muestra       | Propósito                                                                                                                                                                                                                                  |
|--------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| BITS \_ IE     | Muestra cómo usar las [interfaces](bits-interfaces.md) BITS para descargar archivos de un servidor.                                                                                                                                             |
| Controles     | Muestra cómo usar las interfaces bits para controlar y configurar opciones de BITS, incluido el uso de algunos de los [métodos](bits-interfaces.md) de administración de caché del mismo nivel.<br/> **BITS 4.0:** El ejemplo Controls está en desuso.<br/> |
| Descargas    | Muestra cómo usar las [interfaces](bits-interfaces.md) BITS para descargar archivos de un servidor.                                                                                                                                             |
| HTTPHeaders  | Muestra cómo usar las interfaces BITS para modificar [encabezados](bits-interfaces.md) HTTP.                                                                                                                                                      |
| Almacenamiento en caché del mismo nivel  | Muestra cómo usar las [interfaces](bits-interfaces.md) BITS para permitir que un trabajo se descargue desde un mismo nivel.<br/> **BITS 4.0:** El ejemplo PeerCaching está en desuso.<br/>                                                           |
| Fotos      | Muestra cómo usar las [interfaces](bits-interfaces.md) BITS para cargar un archivo.                                                                                                                                                            |
| UploadSample | Muestra cómo usar las [interfaces](bits-interfaces.md) BITS para cargar un archivo en un servidor y recibir una respuesta.                                                                                                                            |



 

BITS proporciona la siguiente herramienta.



| Herramienta                            | Propósito                                                                         |
|---------------------------------|---------------------------------------------------------------------------------|
| [BITSAdmin](bitsadmin-tool.md) | Herramienta de línea de comandos para crear trabajos de descarga o carga y supervisar su progreso. |
