# <a name="name-property-agent-control"></a>Propiedad Name (control de agente)

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

<dl> <dt>

<span id="Description"></span><span id="description"></span><span id="DESCRIPTION"></span>**Denominación**
</dt> <dd>

Devuelve el nombre utilizado en el código para identificar el control. Esta propiedad es de solo lectura en tiempo de ejecución.

</dd> <dt>

<span id="Syntax"></span><span id="syntax"></span><span id="SYNTAX"></span>**Sintáctica**
</dt> <dd>

*agente * * *. Name**

</dd> </dl>

## <a name="remarks"></a>Observaciones

En algunos entornos de programación como Visual Basic, al agregar el control se genera automáticamente un nombre predeterminado para el control que se puede cambiar en tiempo de diseño. En el caso de los scripts HTML, puede definir el nombre en la <OBJECT> etiqueta. Si define el nombre, siga las convenciones del lenguaje de programación para definir los nombres de objeto.
