# <a name="title-summary-property"></a>Propiedad Title Summary

La **propiedad Resumen del** título describe brevemente el tipo del paquete del instalador. Se pueden usar frases como "Base de datos de instalación" o "Transformar" o "Revisión" para esta propiedad.
