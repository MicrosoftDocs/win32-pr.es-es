# <a name="changes-to-com-in-windows-vista"></a>Cambios en COM en Windows Vista

Se realizaron los siguientes cambios en COM en Windows Vista:

-   El moniker de elevación de COM permite a las aplicaciones que se ejecutan en una cuenta de usuario limitada (LUA) activar las clases COM con funcionalidad elevada. Para obtener más información, vea [el moniker de elevación de com](the-com-elevation-moniker.md).
-   Al instalar una aplicación que realiza llamadas COM en un equipo con Windows Vista y tiene previsto ejecutar la aplicación como administrador, debe instalar la aplicación por equipo, en lugar de por usuario, o no funcionará correctamente.
