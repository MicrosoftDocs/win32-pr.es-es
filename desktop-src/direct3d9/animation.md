# <a name="animation-direct3d-9-graphics"></a>Animación (gráficos de Direct3D 9)

Contiene animaciones que hacen referencia a un fotograma anterior. Debe contener una referencia a un marco y al menos un conjunto de plantillas [**AnimationKey**](animationkey.md) . También puede contener un objeto de datos [**AnimationOptions**](animationoptions.md) .

``` syntax
template Animation
{
    < 3D82AB4F-62DA-11cf-AB39-0020AF71E433 >
    [...]
} 
```

Donde:

-   \[ ... \] -Aquí puede usarse cualquier plantilla de archivo. x. Esto hace que la arquitectura sea extensible.
