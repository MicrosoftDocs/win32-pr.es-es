# <a name="avencvideosourceisbw-property"></a>Propiedad AVEncVideoSourceIsBW

Especifica si el vídeo es monocromo (blanco y negro) o contiene color.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**Variante \_ BOOL** (**VT \_ bool**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncVideoSourceIsBW**

## <a name="property-value"></a>Valor de propiedad

Si el valor es **Variant \_ true**, el vídeo es monocromo. De lo contrario, el vídeo contiene color.
