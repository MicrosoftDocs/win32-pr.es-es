# <a name="itsremoteprogram2-interface"></a>Interfaz ITSRemoteProgram2

Define las propiedades que se van a usar con RemoteApp.

## <a name="members"></a>Miembros

La interfaz **ITSRemoteProgram2** hereda de [**ITSRemoteProgram**](itsremoteprogram.md). **ITSRemoteProgram2** también tiene estos tipos de miembros:

-   [Propiedades](#properties)

### <a name="properties"></a>Propiedades

La interfaz **ITSRemoteProgram2** tiene estas propiedades.



| Propiedad                                                                                  | Tipo de acceso           | Descripción                                                 |
|:------------------------------------------------------------------------------------------|:----------------------|:------------------------------------------------------------|
| [**RemoteApplicationArgs**](itsremoteprogram2-remoteapplicationargs.md)<br/>       | Solo escritura<br/> | Argumentos de la línea de comandos para RemoteApp.<br/>    |
| [**RemoteApplicationName**](itsremoteprogram2-remoteapplicationname.md)<br/>       | Solo escritura<br/> | Nombre del RemoteApp.<br/>                       |
| [**RemoteApplicationProgram**](itsremoteprogram2-remoteapplicationprogram.md)<br/> | Solo escritura<br/> | Ruta de acceso y nombre de archivo del programa RemoteApp.<br/> |
