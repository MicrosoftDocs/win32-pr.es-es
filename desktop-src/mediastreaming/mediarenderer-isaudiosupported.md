# <a name="mediarendererisaudiosupported-property"></a>Propiedad MediaRenderer. IsAudioSupported

Obtiene un valor que indica si el DMR es capaz de reproducir contenido de audio.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```C++
HRESULT get_IsAudioSupported(
  [out] boolean *value
);
```



## <a name="property-value"></a>Valor de propiedad

Valor booleano que es **true** si el DMR es capaz de reproducir contenido de audio y **false** en caso contrario.
