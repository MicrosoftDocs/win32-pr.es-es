# <a name="tapi-quick-function-reference"></a>Referencia rápida de funciones de TAPI

El siguiente Resumen de hipervínculo de funciones TAPI se organiza por categoría de operación. Para obtener más información, consulte la información general de los [niveles de servicio](./tapi-levels-of-service.md)y [telefonía asistida](./assisted-telephony-overview.md) .



| Sección de referencia                                                                            | Descripción                                                                                         |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------|
| [Referencia de servicios de telefonía básicos](basic-telephony-services-reference.md)                 | Funciones asociadas a las tareas de telefonía de nivel de POTS, como [**lineMakeCall**](/windows/desktop/api/Tapi/nf-tapi-linemakecall). |
| [Referencia de servicios de telefonía adicionales](supplementary-telephony-services-reference.md) | Funciones con características más avanzadas, como la redirección de llamadas.                                       |
| [Referencia de servicios de telefonía extendidos](extended-telephony-services-reference.md)           | Funciones que permiten el acceso a funciones específicas del proveedor de servicios no definidas directamente por TAPI.   |
| [Referencia de servicios de telefonía asistidos](assisted-telephony-services-reference.md)           | Funciones que establecen llamadas de voz y llamadas multimedia, como en el uso de un marcador de teléfono.              |
