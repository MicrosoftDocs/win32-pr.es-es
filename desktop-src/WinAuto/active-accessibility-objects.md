# <a name="active-accessibility-objects"></a>Objetos Active Accessibility

En la terminología de Microsoft Active Accessibility, hay *objetos accesibles* y *elementos simples*. Aunque la mayoría de las aplicaciones contienen ambos objetos accesibles, son más comunes que los elementos simples. En esta sección se definen y describen los objetos accesibles y los elementos simples, y se proporcionan ejemplos adecuados de la interfaz de usuario.
