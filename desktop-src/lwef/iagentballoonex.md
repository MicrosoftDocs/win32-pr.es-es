# <a name="iagentballoonex"></a>IAgentBalloonEx

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

[**IAgentBalloonEx**](/windows/desktop/lwef/iagentballoon) se deriva de la interfaz **IAgentBalloon** . Incluye todos los métodos **IAgentBalloon** y proporciona acceso a funciones adicionales.

**Métodos en orden de Vtable**



| Métodos IAgentBalloonEx                                  | Descripción                                                        |
|----------------------------------------------------------|--------------------------------------------------------------------|
| [**GetStyle**](https://www.bing.com/search?q=**GetStyle**)                     | Devuelve el estilo de salida del globo de palabra.                           |
| [**SetStyle**](https://www.bing.com/search?q=**SetStyle**)                     | Establece el estilo de salida del globo de palabras.                              |
| [**SetNumLines**](https://www.bing.com/search?q=**SetNumLines**)               | Establece la salida de las líneas numéricas en el globo de palabras.                  |
| [**SetNumCharsPerLine**](https://www.bing.com/search?q=**SetNumCharsPerLine**) | Establece el número de caracteres por salida de línea en el globo de palabras. |
