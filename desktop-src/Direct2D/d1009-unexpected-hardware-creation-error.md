# <a name="d1009-unexpected-hardware-creation-error"></a>D1009: error de creación de hardware inesperado

Error inesperado \[  \] al intentar crear un destino de Direct3D.

<span id="error"></span><span id="ERROR"></span>*error*

|             |         |
|-------------|---------|
| Nivel de error | Advertencia |



 

## <a name="possible-causes"></a>Causas posibles

-   El destino era mayor que el tamaño máximo del mapa de bits.
-   Memoria de vídeo agotada cuando se crea el destino de representación.
-   No se pudo crear ningún dispositivo Direct3D.
-   La aplicación se ejecuta en IA64.
