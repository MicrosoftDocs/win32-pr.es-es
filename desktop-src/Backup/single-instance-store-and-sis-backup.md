# <a name="single-instance-store-and-sis-backup"></a>Almacenamiento de Single-Instance y copia de seguridad de SIS

*El almacén de instancia única*, o SIS, es una arquitectura diseñada para mantener archivos duplicados con un mínimo de disco, caché y sobrecarga de medios de copia de seguridad. *La copia de seguridad de un almacén de instancia única* o una copia de seguridad de SIS es una API que las aplicaciones de copia de seguridad llaman para acceder a la arquitectura de SIS.
