# <a name="tspi_linedroponclose"></a>TSPI \_ lineDropOnClose

La función **TSPI \_ lineDropOnClose** está obsoleta. TAPI ya no quita automáticamente una llamada cuando el propietario único cierra su identificador al dispositivo de línea, con el fin de dar cabida al control distribuido de llamadas entre varios equipos cliente en un entorno cliente/servidor.
