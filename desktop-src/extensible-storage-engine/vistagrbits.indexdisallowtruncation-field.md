# <a name="vistagrbitsindexdisallowtruncation-field"></a>VistaGrbits. IndexDisallowTruncation, campo

Si se especifica este marcador, se producirá un error en cualquier actualización del índice que dé lugar a una clave truncada con [KeyTruncated](./jet-err-enumeration.md). De lo contrario, las claves se truncarán de forma silenciosa.
