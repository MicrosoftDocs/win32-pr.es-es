# <a name="systemdevicespaired"></a>System.Devices.Paired

Estado emparejado del dispositivo. Si es true, indica que el dispositivo no está emparejado con el equipo.

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8

```
propertyDescription
   name = System.Devices.Paired
   shellPKey = PKEY_Devices_Paired
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 56
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="windows-7"></a>Windows 7

```
propertyDescription
   name = System.Devices.Paired
   shellPKey = PKEY_Devices_IsPaired
   formatID = 78C34FC8-104A-4ACA-9EA4-524D52996E57
   propID = 56
   SearchInfo
      InInvertedIndex = false
      IsColumn = false
   typeInfo
      type = Boolean
      IsInnate = true
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.
