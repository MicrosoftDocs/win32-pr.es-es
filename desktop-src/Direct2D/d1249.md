# <a name="d1249-inaccessible-target"></a>D1249: Destino inaccesible

Se ha detectado un intento de dibujar en un destino inaccesible.






 

## <a name="possible-causes"></a>Causas posibles

Se ha detectado un intento de dibujar en un destino inaccesible. Una razón por la que un destino puede ser inaccesible es si se creó a partir de una textura compartida y no se adquirió la textura.

## <a name="possible-fixes"></a>Posibles correcciones

Asegúrese de que todos los destinos son accesibles antes de representarlos con Direct2D.
