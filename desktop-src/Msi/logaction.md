# <a name="logaction-property"></a>Propiedad LOGACTION

La propiedad **LOGACTION** es una lista de nombres de acción separados solo por punto y coma y sin espacios. Los nombres de acción de esta propiedad no distinguen mayúsculas de minúsculas y la lista puede acabar con un punto y coma.

## <a name="remarks"></a>Observaciones

El instalador registra los mensajes de datos de acción para las acciones que se encuentran en la lista. La propiedad no tiene ningún efecto en otros tipos de registro.
