# <a name="dds"></a>DDS

El formato de archivo de superficie de DirectDraw (. DDS) se incorporó con DirectX 7 para almacenar texturas sin comprimir y comprimidas (DXTn). El formato de archivo admite mapas MIP, mapas de cubo y mapas de volumen. El formato de archivo DDS es compatible con [DirectXTex](https://github.com/Microsoft/DirectXTex), [DirectXTK](https://github.com/Microsoft/DirectXTK), la herencia heredada y otras herramientas de DirectX. A partir de Direct3D 10, los archivos DDS admiten matrices de texturas.

> [!Note]  
> La biblioteca de utilidades de D3DX (D3DX 9, D3DX 10 y [d3dx 11](/windows/desktop/direct3d11/d3d11-graphics-reference-d3dx11)) está en desuso para Windows 8 y no se admite para aplicaciones de la tienda Windows. Se recomienda usar [DirectXTex](https://github.com/Microsoft/DirectXTex), [DirectXTK](https://github.com/Microsoft/DirectXTK)o ambos.
