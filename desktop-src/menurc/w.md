<dl> <dt>

<span id="tools.w_1_gly"></span><span id="TOOLS.W_1_GLY"></span>**ADVERTENCIA: carácter ASCII no equivalente a código de tecla virtual**
</dt> <dd>

Existe un código de clave virtual no válido en la instrucción [**Accelerators**](accelerators-resource.md) . Los valores ASCII para algunos caracteres como \* , ^ o & no son equivalentes a los códigos de tecla virtual de las claves correspondientes. En el caso del asterisco ( \* ), el código de tecla virtual es equivalente al valor ASCII de 8, el carácter numérico de la misma clave.

</dd> <dt>

<span id="tools.w_2_gly"></span><span id="TOOLS.W_2_GLY"></span>**ADVERTENCIA: se usa SHIFT o CONTROL sin VIRTKEY**
</dt> <dd>

Las opciones **Alt**, **Shift** y [**control**](control-control.md) solo se aplican a las teclas virtuales de la instrucción [**Accelerators**](accelerators-resource.md) . Asegúrese de que la opción **VIRTKEY** se usa con una de estas otras opciones.

</dd> <dt>

<span id="tools.w_3_gly"></span><span id="TOOLS.W_3_GLY"></span>**Escribiendo resourcetypeimagelang: languageing: Size**
</dt> <dd>

(V) el compilador de recursos de Microsoft Windows (RC) está escribiendo el recurso especificado. El *tipo* es el tipo de recurso. Puede ser un número o una cadena. La *imagen* puede ser un número o una cadena. El *idioma* es el idioma nacional del recurso. El *tamaño* es el tamaño decimal del recurso en bytes.

</dd> </dl>
