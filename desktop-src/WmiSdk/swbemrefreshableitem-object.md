# <a name="swbemrefreshableitemobject-property"></a>Propiedad SWbemRefreshableItem. Object

La propiedad **SWbemRefreshableItem. Object** representa una instancia de [**SWbemObject**](swbemobject.md) única que se actualiza. El elemento está contenido en un objeto [**SWbemRefresher**](swbemrefresher.md) .

Para obtener una explicación de esta sintaxis, vea [convenciones de documentos para la API de scripting](document-conventions-for-the-scripting-api.md).

Esta propiedad es de lectura y escritura.

## <a name="syntax"></a>Sintaxis


```VB
SWbemRefreshableItem.Object As Object
```



## <a name="property-value"></a>Valor de propiedad

## <a name="remarks"></a>Observaciones

Esta propiedad es **null** a menos que [**SWbemRefreshableItem. IsSet**](swbemrefreshableitem-isset.md) sea **false**.
