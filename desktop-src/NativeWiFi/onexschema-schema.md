# <a name="onex-schema"></a>Esquema OneX

El esquema OneX define los elementos de configuración de 802.1 X. Todos los elementos de esquema OneX se aplican tanto a los perfiles con cable como a los inalámbricos. Para obtener una lista de los elementos definidos, consulte [elementos de esquema Onex](onexschema-elements.md).

El elemento raíz de un perfil de 802.1 X es el elemento [**Onex**](onexschema-onex-element.md) . Cada perfil tendrá exactamente un elemento raíz. El elemento **Onex** está en el `https://www.microsoft.com/networking/OneX/v1` espacio de nombres.

-   [Ejemplo de perfil WPA2-Enterprise con TLS](wpa2-enterprise-with-tls-profile-sample.md)
