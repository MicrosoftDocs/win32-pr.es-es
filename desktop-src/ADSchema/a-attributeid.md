# <a name="attribute-id-attribute"></a>Atributo Attribute-ID

El OID X. 500 único para identificar un atributo.



| Entrada | Value |
|-------------------|-----------------------------------------------------------------|
| CN                | IDENTIFICADOR de atributo                                                    |
| Nombre para mostrar de LDAP | attributeID                                                     |
| Tamaño              | \-                                                              |
| Actualizar privilegio  | Administrador de esquema                                            |
| Frecuencia de actualización  | \-                                                              |
| Attribute-Id      | 1.2.840.113556.1.2.30                                           |
| System-ID-GUID    | bf967922-0de6-11d0-a285-00aa003049e2                            |
| Sintaxis            | [**String(Object-Identifier)**](s-string-object-identifier.md) |
