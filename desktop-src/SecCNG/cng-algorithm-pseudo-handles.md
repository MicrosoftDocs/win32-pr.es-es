# <a name="cng-algorithm-pseudo-handles"></a>Manipuladores del algoritmo CNG

## <a name="remarks"></a>Observaciones

A partir de Windows 10, CNG proporciona controladores de algoritmo predefinidos para muchos algoritmos. Puede utilizar estos identificadores en cualquier situación que requiera un identificador de algoritmo. Sin embargo, se produce un error en cualquier llamada a [**BCryptSetProperty**](/windows/desktop/api/Bcrypt/nf-bcrypt-bcryptsetproperty) , ya que el identificador del algoritmo está compartido y no se puede modificar. Además, estos identificadores no se pueden usar en IRQL = DISPATCH en modo kernel.
