# <a name="adsi-extension-for-remote-desktop-services-user-configuration"></a>Extensión ADSI para Servicios de Escritorio remoto configuración de usuario

Una extensión de la interfaz de servicio de Active Directory (ADSI) es un objeto COM especial que permite la extensión de un objeto COM. Las extensiones ADSI pueden agregar funcionalidad a un objeto de directorio y simplificar las tareas de administración.

La extensión ADSI para Servicios de Escritorio remoto configuración de usuario extiende un objeto ADSI que permite a los administradores de dominio escribir scripts que automaticen el mantenimiento de las propiedades de usuario específicas de Servicios de Escritorio remoto.

La extensión ADSI para Servicios de Escritorio remoto configuración de usuario se incluye en Windows Server.

Para obtener más información, vea los temas siguientes:

-   [Uso de la extensión ADSI para Servicios de Escritorio remoto la configuración de usuario](using-the-adsi-extension-for-terminal-services-user-configuration.md)
-   [Referencia de la extensión ADSI para Servicios de Escritorio remoto configuración de usuario](reference-for-the-adsi-extension-for-terminal-services-user-configuration.md)

Para obtener información general sobre ADSI, consulte [ADSI Extensions](/windows/desktop/ADSI/adsi-extensions), [la memoria caché de atributos ADSI](/windows/desktop/ADSI/the-adsi-attribute-cache)y [Active Directory interfaces de servicio](/windows/desktop/ADSI/active-directory-service-interfaces-adsi).
