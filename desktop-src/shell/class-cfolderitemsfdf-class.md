# <a name="cfolderitemsfdf-class"></a>Clase CFolderItemsFDF

**CFolderItemsFDF** es una colección de [**FolderItems**](folderitems.md). Implementa la interfaz [**IInsertItem**](/windows/desktop/api/Shobjidl/nn-shobjidl-iinsertitem) .

## <a name="members"></a>Miembros

La clase **CFolderItemsFDF** hereda de [**CFolderItems**](class-cfolderitems-class.md) pero no tiene miembros adicionales.

## <a name="remarks"></a>Observaciones

FDF significa *de distintas carpetas*.

La clase [**CFolderItems**](class-cfolderitems-class.md) enumera los elementos de varias carpetas y usa la interfaz [**IInsertItem**](/windows/desktop/api/Shobjidl/nn-shobjidl-iinsertitem) para insertar los elementos en la colección.
