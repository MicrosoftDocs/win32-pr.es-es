# <a name="using-http-server-api"></a>Uso de la API del servidor HTTP

Los archivos de origen del programa de aplicación incluyen el archivo de encabezado HTTP. h para tener acceso a los prototipos de función y las definiciones de estructura de la API del servidor HTTP. Los desarrolladores pueden utilizar el archivo de biblioteca httpapi. lib para compilar aplicaciones que usan la API del servidor HTTP. En tiempo de ejecución, las aplicaciones se vinculan a la Httpapi.dll.
