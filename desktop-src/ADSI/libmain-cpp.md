# <a name="libmaincpp"></a>LIBMAIN. Cpp

En el componente de proveedor de ejemplo, un ejemplo de código del punto de entrada dll estándar para Adssmp.dll está en Libmain.cpp. Las rutinas admitidas se enumeran en la tabla siguiente.



| Elemento                                  | Descripción                                                              |
|---------------------------------------|--------------------------------------------------------------------------|
| **DllGetClassObject**<br/>      | Punto de entrada dll estándar para buscar generadores de clases.<br/>        |
| **DllCanUnloadNow**<br/>        | Punto de entrada dll estándar para determinar si se puede descargar el archivo DLL.<br/> |
| **LibMain**<br/>                | Punto de entrada de inicialización de DLL estándar.<br/>                      |
| **IsCompatibleOleVersion**<br/> | Compruebe la compatibilidad con el tiempo de ejecución ole.<br/>                   |
| **DllMain**<br/>                | Punto de entrada para la mayoría Windows versiones 2000 o Windows NT.<br/>     |
