# <a name="mobile-computing-configurations"></a>Configuraciones de informática móvil

El administrador de sincronización es útil para los equipos configurados de la siguiente manera:

-   Equipo móvil usado en una estación de acoplamiento en una red de ancho de banda alto, pero que se usa ocasionalmente a través de una conexión de acceso telefónico.
-   El equipo móvil se usa principalmente a través de una conexión de acceso telefónico.
-   Equipo de escritorio que se usa estrictamente a través de una conexión de acceso telefónico, como un equipo usado por un teletrabajo.
-   Equipo de escritorio usado exclusivamente a través de una red de ancho de banda alto.

Aunque el último caso no es un escenario típico móvil, los problemas de latencia con el acceso a la red pueden hacer que sea conveniente almacenar en caché los recursos de forma local. En todas estas configuraciones, la aplicación puede usar el SyncMgr para mantener los archivos y otros recursos almacenados en caché localmente y sincronizados entre el uso en línea y sin conexión.
