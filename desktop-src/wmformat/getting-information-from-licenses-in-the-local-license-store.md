# <a name="getting-information-from-licenses-in-the-local-license-store"></a>Obtención de información de licencias en el almacén de licencias local

Las API extendidas del cliente DRM de Windows Media proporcionan varias maneras de obtener información acerca de los derechos concedidos por las licencias en el almacén de licencias local. En los temas siguientes se describe cómo obtener estos diversos tipos de información.



| Tema                                                                                                  | Descripción                                                                                                                                                                     |
|--------------------------------------------------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [Consulta de información de derechos simples](querying-for-simple-rights-information.md)                   | Describe cómo obtener información simple sobre si se permiten acciones específicas para el contenido protegido. Por ejemplo, puede que una aplicación reproduzca un archivo multimedia.                    |
| [Consulta para obtener información detallada sobre los derechos](querying-for-detailed-rights-information.md)               | Describe cómo obtener información detallada agregada sobre los derechos concedidos por todas las licencias en el almacén de licencias local para un elemento de contenido protegido con el identificador de clave. |
| [Enumerar licencias en el almacén de licencias local](enumerating-licenses-in-the-local-license-store.md) | Describe cómo obtener licencias individuales del almacén de licencias local.                                                                                                          |
