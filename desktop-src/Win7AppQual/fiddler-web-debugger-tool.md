# <a name="fiddler-web-debugger-tool"></a>Herramienta Fiddler Web Debugger

La herramienta Fiddler le ayuda a depurar aplicaciones web mediante la captura del tráfico de red entre Internet y los equipos de prueba. La herramienta permite inspeccionar los datos entrantes y salientes para supervisar y modificar solicitudes y respuestas antes de que el explorador las reciba. Fiddler también incluye un eficaz subsistema de scripting basado en eventos, que se puede extender mediante cualquier lenguaje .NET Framework eventos.

Fiddler y las opciones de reproducción HTTP pueden ayudarle a solucionar problemas del lado cliente con aplicaciones web mediante la realización de una copia sin conexión del sitio de prueba. Con estas herramientas, puede crear imágenes sin conexión de la experiencia de exploración y, a continuación, empaquetar y analizar los resultados para obtener información de depuración más detallada.

Para descargar el complemento Fiddler, vaya a la [página Internet Explorer complementos.](https://www.microsoft.com/iegallery) Para obtener más información sobre cómo solucionar problemas mediante Fiddler y las herramientas relacionadas, consulte la entrada de blog How [to use fiddler and HTTP replay to have an offline copy of your site](https://blogs.msdn.com/b/ricardo/archive/2010/06/22/how-to-user-fiddler-and-http-replay-to-have-an-offline-copy-of-your-site.aspx) (Cómo usar fiddler y la reproducción HTTP para tener una copia sin conexión de su sitio) y la entrada de blog Troubleshooting Authentication with Fiddler (Solución de problemas de autenticación con [Fiddler).](https://blogs.msdn.com/b/ieinternals/archive/2009/11/22/clipboard-paste-bug-in-windows-7-password-prompt.aspx)
