# <a name="directory-manager-errors"></a>Errores del Administrador de directorios

Este tema contiene los valores devueltos para los errores del Administrador de directorios de las funciones Active Directory Domain Services.



| Valor                    | Descripción                                              |
|--------------------------|----------------------------------------------------------|
| **hrPMRecDeleted**       | Se eliminó el registro.                                  |
| **hrRemainingVersions**  | Queda trabajo inactivo.                            |
| **hrFLDKeyTooBig**       | La clave se trunó porque tiene más de 255 bytes. |
| **hrFLDTooManySegments** | Hay demasiados segmentos clave.                         |
| **hrFLDNullKey**         | La clave es **NULL.**                                     |
