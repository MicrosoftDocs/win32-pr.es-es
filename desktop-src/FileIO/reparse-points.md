data-stu-id="d3ead-105">The format of this data is understood by the application which stores the data, and a file system filter, which you install to interpret the data and process the file.</span></span> <span data-ttu-id="d3ead-106">Cuando una aplicación establece un punto de reanálisis, almacena estos datos, además de una *etiqueta de reanálisis*, que identifica de forma única     <span data-ttu-id="d3ead-126">**Windows Server 2003 y Windows XP:** Hay un límite de 31 puntos de análisis en una ruta de acceso dada.</span><span class="sxs-lookup"><span data-stu-id="d3ead-126">**Windows Server 2003 and Windows XP:** There is a limit of 31 reparse points on any given path.</span></span>
