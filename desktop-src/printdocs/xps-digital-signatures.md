# <a name="xps-digital-signature-api"></a>API de firma digital XPS

La API de firma digital XPS permite a un usuario firmar un documento, comprobar la identidad del firmante e indicar si un documento XPS ha cambiado desde que se firmó. La API de firma digital XPS se basa en la tecnología de firma digital que se usa en las convenciones de empaquetado abierto, que se especifican en la primera edición, parte 2, "convenciones de empaquetado abierto", de los [formatos de archivo estándar ECMA-376, Office Open XML](https://www.ecma-international.org/publications/standards/Ecma-376.htm).

## <a name="in-this-section"></a>En esta sección

Esta sección contiene los temas siguientes.

### <a name="about-xps-digital-signature-api"></a>Acerca de la API de firma digital XPS

[Acerca de XPS Digital Signature API](about-xps-digital-signatures.md) describe la API de firma digital XPS en un nivel alto.

### <a name="using-xps-digital-signature-api"></a>Uso de la API de firma digital XPS

El uso de la [API de firma digital XPS](using-digital-signatures-in-xps-documents.md) describe cómo usar la API de firma digital XPS.

### <a name="xps-digital-signatures-reference"></a>Referencia de firmas digitales XPS

La [referencia de la API de firma digital XPS](xps-digital-signatures-programming-reference.md) contiene una lista completa de las interfaces, los métodos y los enumeradores que se implementan mediante la API de firmas digitales XPS.

## <a name="platform-update-for-windows-vista"></a>Actualización de la plataforma para Windows Vista

Las interfaces de firma digital XPS que se describen en esta sección no son compatibles con la actualización de la plataforma para Windows Vista o la actualización de la plataforma para Windows Server 2008. Una aplicación que requiera estas interfaces se debe ejecutar en Windows 7 o posterior, o en Windows Server 2008 R2 o posterior. De lo contrario, es posible que la aplicación no proporcione al usuario la funcionalidad completa de la aplicación.
