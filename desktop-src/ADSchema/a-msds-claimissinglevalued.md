# <a name="ms-ds-claim-is-single-valued-attribute"></a>atributo de valor único de MS-DS-Claim-is-Value

Para un objeto de tipo de demanda, este atributo identifica si el tipo de demanda o la propiedad de recurso solo pueden contener un valor único.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-DS-Claim-is-Single-valued         |
| Nombre para mostrar de LDAP | msDS-ClaimIsSingleValued             |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
| Frecuencia de actualización  | \-                                   |
| Attribute-Id      | 1.2.840.113556.1.4.2160              |
| System-ID-GUID    | cd789fb9-96b4-4648-8219-ca378161af38 |
| Sintaxis            | [**Booleano**](s-boolean.md)         |
