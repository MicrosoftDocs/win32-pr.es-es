# <a name="extensibility-cryptography"></a>Extensibilidad (criptografía)

Certificate Services admite [*extensiones de certificado X.509.*](../secgloss/x-gly.md) Los módulos de directiva pueden agregar o modificar extensiones, y un módulo de salida puede recuperar estas extensiones.

Vea [Controladores de extensión para](extension-handlers.md) obtener más información.
