# <a name="supplemental-credentials-attribute"></a>Supplemental-Credentials atributo

Credenciales almacenadas para su uso en la autenticación. La versión cifrada de la contraseña del usuario. Este atributo no es legible ni se puede escribir.



| Entrada | Value |
|-------------------|-------------------------------------------------------|
| CN                | Supplemental-Credentials                              |
| Ldap-Display-Name | supplementalCredentials                               |
| Size              | \-                                                    |
| Actualizar privilegios  | El sistema establece este valor.                      |
| Frecuencia de actualización  | Cuando cambia la contraseña del usuario.                     |
| Attribute-Id      | 1.2.840.113556.1.4.125                                |
| System-Id-Guid    | bf967a3f-0de6-11d0-a285-00aa003049e2                  |
| Syntax            | [**Object(Replica-Link)**](s-object-replica-link.md) |
