# <a name="windows-media-streaming-api"></a>API de streaming de Windows Media

## <a name="purpose"></a>Propósito

La API de streaming de multimedia modela los dispositivos del representador de medios digitales (DMR) y del servidor de medios digitales (DMS) tal y como se define en las instrucciones de la DLNA para facilitar el uso mediante programación, lo que permite a los desarrolladores crear aplicaciones con capacidades de controlador de medios digitales (DMC), Digital Media Player (DMP) y controlador de extracción (+ PU +)

La API de streaming de multimedia está diseñada para que la usen los desarrolladores de C++ que crean aplicaciones multimedia digitales que interactúan con dispositivos DLNA.
