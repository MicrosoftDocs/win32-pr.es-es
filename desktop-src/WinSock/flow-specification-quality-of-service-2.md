# <a name="flow-specification-quality-of-service"></a>Calidad de servicio de especificación de flujo

La calidad de servicio se implementa en Windows a través de varios componentes de QoS del sistema. Para obtener detalles completos e instrucciones de implementación, consulte [calidad de servicio](/previous-versions/windows/desktop/qos/qos-start-page).

Para obtener más información acerca de las plantillas de QoS, consulte [plantillas de QoS](/previous-versions/aa374099(v=vs.80)).

Para obtener información detallada e instrucciones de implementación sobre calidad de servicio, vea la estructura [**FLOWSPEC**](/windows/win32/api/qos/ns-qos-flowspec) .
