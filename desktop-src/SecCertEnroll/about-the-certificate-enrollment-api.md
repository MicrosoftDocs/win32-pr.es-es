# <a name="about-the-certificate-enrollment-api"></a>Acerca de la API de inscripción de certificados

La API de inscripción de certificados permite crear aplicaciones cliente que se pueden usar para solicitar e instalar certificados. Las solicitudes de certificado se describen en las secciones siguientes. Para obtener la información más reciente, consulte la biblioteca de Microsoft Developer Network (MSDN) ( [https://msdn.microsoft.com/library/default.asp](/documentation/) ).

-   [Infraestructura de clave pública](public-key-infrastructure.md)
-   [Codificación de solicitud de certificado](about-certificate-request-encoding.md)
-   [Solicitudes de certificado](certificate-requests.md)
-   [Solicitudes de ejemplo](sample-requests.md)
