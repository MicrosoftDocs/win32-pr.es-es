# <a name="about-this-guide"></a>Acerca de esta guía

Esta sección es una referencia para el Lenguaje de definición de interfaz de Microsoft (MIDL) y una guía de usuario para el compilador MIDL, que genera archivos de lenguaje C a partir de archivos IDL.

En la sección [uso del compilador de MIDL](using-the-midl-compiler-2.md) se enumeran los requisitos del compilador de C y el preprocesador con el que debe interoperar MIDL. En esta sección también se describen los archivos que genera el compilador MIDL para los códigos auxiliares RPC, los códigos auxiliares DCOM y las bibliotecas de tipos de automatización OLE.

En la sección [definiciones de interfaz y bibliotecas de tipos](interface-definitions-and-type-libraries.md) se proporciona información sobre cómo crear archivos de definición de interfaz (IDL) y de configuración de la aplicación (ACF) para MIDL, incluidas las páginas de referencia para los atributos IDL y ACF. También hay información sobre cómo importar archivos y bibliotecas de tipos, e importar archivos de encabezado del sistema.

La sección de [referencia de Command-Line de MIDL](midl-command-line-reference.md) contiene información de referencia para cada modificador de línea de comandos y opción de modificador que reconoce el compilador de MIDL.

La sección [Referencia del lenguaje MIDL](midl-language-reference.md) contiene una entrada de referencia para cada palabra clave en el lenguaje de definición de interfaz de Microsoft.

En la sección [errores y advertencias del compilador de MIDL](midl-compiler-errors-and-warnings.md) se enumeran los mensajes de error y de advertencia que el compilador MIDL puede generar.
