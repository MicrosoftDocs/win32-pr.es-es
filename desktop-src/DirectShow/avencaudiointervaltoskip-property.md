# <a name="avencaudiointervaltoskip-property"></a>Propiedad AVEncAudioIntervalToSkip

Especifica el número de muestras de audio que va a omitir el codificador.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**UINT64** (**VT \_ UI8**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncAudioIntervalToSkip**
