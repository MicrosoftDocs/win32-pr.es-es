# <a name="data-model"></a>Modelo de datos

El modelo de datos Active Directory Domain Services se deriva del modelo de datos X.500. El directorio contiene objetos que representan elementos de varios tipos, descritos por atributos. El universo de objetos que se pueden almacenar en el directorio se define en el esquema. Para cada clase de objeto, el esquema define qué atributos debe tener una instancia de la clase, qué atributos adicionales puede tener y qué clase de objeto puede ser un elemento primario de la clase de objeto actual.
