# <a name="id3dxkeyframedanimationsetgetnumrotationkeys-method"></a>ID3DXKeyframedAnimationSet:: GetNumRotationKeys (método)

Obtiene el número de claves de rotación en la animación de fotogramas clave especificada.

## <a name="syntax"></a>Sintaxis


```C++
UINT GetNumRotationKeys(
  [in] UINT Animation
);
```



## <a name="parameters"></a>Parámetros

<dl> <dt>

*Animación* \[ de de\]
</dt> <dd>

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Índice de animación.

</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **uint**](../winprog/windows-data-types.md)**

Número de claves de rotación en la animación de fotogramas clave especificada.
