# <a name="vss-security-issues"></a>Problemas de seguridad de VSS

La infraestructura de VSS se basa internamente en COM y, por tanto, permitir que los escritores y solicitantes se comuniquen requiere seguridad basada en COM. La seguridad COM proporciona mecanismos e instrucciones que un desarrollador de VSS debe tener en cuenta:

-   [Consideraciones de seguridad para escritores](security-considerations-for-writers.md)
-   [Consideraciones de seguridad para los solicitantes](security-considerations-for-requestors.md)
