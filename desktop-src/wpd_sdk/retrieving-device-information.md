# <a name="retrieving-device-information"></a>Recuperando información del dispositivo

La API de WPD proporciona un amplio conjunto de interfaces y métodos que se pueden usar para recuperar información de un dispositivo determinado. En general, los datos del dispositivo se pueden dividir en tres categorías, como se describe en la tabla siguiente.



| Datos del dispositivo                                                                             | Descripción                                                                               |
|-----------------------------------------------------------------------------------------|-------------------------------------------------------------------------------------------|
| [Categorías funcionales](retrieving-the-functional-categories-supported-by-a-device.md) | Las categorías funcionales describen las operaciones que un dispositivo puede realizar. |
| [Tipos de contenido](retrieving-the-content-types-supported-by-a-device.md)                 | Los tipos de contenido denotan qué tipos de contenido puede representar un dispositivo.         |
| [Eventos](retrieving-the-events-supported-by-a-device.md)                               | Algunos dispositivos son capaces de desencadenar eventos a los que la aplicación puede realizar un seguimiento y responder. |
