# <a name="trap-type-macro"></a>TRAP-TYPE (macro)

Para la uniformidad, cada instancia de la macro de tipo de captura se asigna primero a una instancia de la macro de [tipo de notificación](notification-type-macro.md) mediante la asignación definida en [RFC 1908](https://www.ietf.org/rfc/rfc1908.txt).

> [!Note]  
> Para obtener más información acerca de cómo instalar el proveedor, consulte [configuración del entorno WMI SNMP](setting-up-the-wmi-snmp-environment.md).
