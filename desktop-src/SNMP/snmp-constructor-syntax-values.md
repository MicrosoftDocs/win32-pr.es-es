# <a name="snmp-constructor-syntax-values"></a>Valores de sintaxis del constructor SNMP

\[SNMP está disponible para su uso en los sistemas operativos especificados en la sección de requisitos. En versiones posteriores podría modificarse o no estar disponible. En su lugar, use [administración remota de Windows](/windows/desktop/WinRM/portal), que es la implementación de Microsoft de WS-MAN.\]

Los valores de sintaxis del constructor SNMP describen tipos que son compatibles con el estándar de codificación de la notación de sintaxis abstracta uno (ASN. 1).



| Constante                                                                                                                                                         | Descripción                                               |
|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------|:----------------------------------------------------------|
| <span id="ASN_SEQUENCE"></span><span id="asn_sequence"></span><dl> <dt>**\_secuencia ASN**</dt> </dl>       | Indica que el mensaje es una secuencia ASN.<br/> |
| <span id="ASN_SEQUENCEOF"></span><span id="asn_sequenceof"></span><dl> <dt>**secuencia de ASN \_**</dt> </dl> | Consulte secuencia de ASN \_ .<br/>                             |
