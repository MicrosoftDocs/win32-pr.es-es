# <a name="registeredtaskname-property"></a>Propiedad RegisteredTask.Name

En el caso de scripting, obtiene el nombre de la tarea registrada.

## <a name="syntax"></a>Sintaxis


```VB
RegisteredTask.Name As String
```



## <a name="property-value"></a>Valor de propiedad

Nombre de la tarea registrada.
