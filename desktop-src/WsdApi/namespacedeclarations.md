# <a name="namespacedeclarations-element"></a>elemento namespaceDeclarations

Genera declaraciones de C para las tablas de espacio de nombres.

## <a name="usage"></a>Uso

``` syntax
<namespaceDeclarations/>
```

## <a name="attributes"></a>Atributos

No hay atributos.

## <a name="child-elements"></a>Elementos secundarios

No hay elementos secundarios.

## <a name="parent-elements"></a>Elementos primarios



| Elemento                         | Descripción                                                    |
|---------------------------------|----------------------------------------------------------------|
| [**archivo**](file.md)<br/> | Genera un archivo desde el generador de código.<br/> <br/> |



## <a name="remarks"></a>Observaciones

Se hace referencia a las tablas de espacio de nombres mediante tablas de tipos y otro código generado, por lo que este elemento se usa para generar archivos de inclusión.
