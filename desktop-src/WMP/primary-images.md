# <a name="primary-images"></a>Imágenes principales

Querrá crear imágenes principales para la máscara. La imagen principal se compone de una imagen de fondo definida por el elemento de **vista** , así como de otras imágenes que van acompañadas de elementos concretos. Por ejemplo, si crea un elemento de **botón** , debe incluir el material gráfico de ese botón y asegurarse de que tiene un orden z superior al de la imagen de fondo.
