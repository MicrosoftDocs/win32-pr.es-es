# <a name="systemparentalrating"></a>System.ParentalRating

Clasificación parental almacenada en un formato normalmente determinado por la organización denominada [en System.ParentalRatingsOrganization](./props-system-parentalratingsorganization.md).

## <a name="windows-10-version-1703-windows-10-version-1607-windows-10-version-1511-windows-10-version-1507-windows-81-windows-8-windows-7-windows-vista"></a>Windows 10, versión 1703, Windows 10, versión 1607, Windows 10, versión 1511, Windows 10, versión 1507, Windows 8.1, Windows 8, Windows 7, Windows Vista

```
propertyDescription
   name = System.ParentalRating
   shellPKey = PKEY_ParentalRating
   formatID = 64440492-4C8B-11D1-8B70-080036B11A03
   propID = 21
   SearchInfo
      InInvertedIndex = true
      IsColumn = true
   typeInfo
      type = String
```

## <a name="remarks"></a>Comentarios

Los valores PKEY se definen en Propkey.h.

Esta cadena se muestra directamente al usuario, por lo que se deben usar palabras fáciles de usar. Por ejemplo, "R-Restricted" al basar las clasificaciones en el sistema de clasificación de Motion Picture Association of America (MPAA).
