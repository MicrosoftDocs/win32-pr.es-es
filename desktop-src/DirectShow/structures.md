# <a name="directshow-structures"></a>DirectShow estructuras

> [!Note]  
> \[Obsoleto. Esta API puede quitarse de futuras versiones de Windows.\]

 

DirectShow Los servicios de edición definen las estructuras siguientes.



| Estructura                             | Descripción                                          |
|---------------------------------------|------------------------------------------------------|
| [**PARAM DE \_**](dexter-param.md) | Especifica una propiedad en una transición o efecto.      |
| [**VALUE DE LA \_ PROPIEDAD**](dexter-value.md) | Especifica el valor de una propiedad.                   |
| [**SCompFmt0**](scompfmt0.md)        | Contiene información de formato para la recompresión inteligente. |
