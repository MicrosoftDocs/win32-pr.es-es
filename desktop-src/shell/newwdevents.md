# <a name="newwdevents-object"></a>Objeto NewWDEvents

Extiende el objeto [**WebWizardHost**](webwizardhost.md) habilitando las páginas del lado servidor hospedadas en un asistente para comprobar que el usuario se ha autenticado a través de un cuenta Microsoft.

## <a name="members"></a>Miembros

El objeto **NewWDEvents** tiene estos tipos de miembros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

El objeto **NewWDEvents** tiene estos métodos.



| Método                                                            | Descripción                                                                                                                         |
|:------------------------------------------------------------------|:------------------------------------------------------------------------------------------------------------------------------------|
| [**PassportAuthenticate**](inewwdevents-passportauthenticate.md) | Habilita las páginas del lado servidor hospedadas en un asistente para comprobar que el usuario se ha autenticado a través de un cuenta Microsoft.<br/> |
