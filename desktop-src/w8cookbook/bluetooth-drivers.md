# <a name="bluetooth-drivers"></a>Controladores Bluetooth

Los clientes de IHV y Enterprise deben tener en cuenta que los controladores Bluetooth y los paquetes de Perfil de terceros no se migrarán a Windows 10 como parte de la actualización de Windows 7, Windows 8 o Windows 8.1. Consulte [este artículo](/windows-hardware/design/component-guidelines/bluetooth) para obtener más información sobre los cambios de Bluetooth para Windows 10.
