# <a name="using-xps-document-api"></a>Uso de la API de documento XPS

En esta sección se describe cómo usar la [API de documentos XPS](documents-xps.md) para realizar tareas de programación.

Para obtener ejemplos de cómo usar la [API de documento XPS](documents-xps.md) en un programa, consulte la sección tareas de programación de la [API de documentos XPS](#xps-document-api-programming-tasks) .

Para obtener información acerca de cómo usar el modelo de objetos XPS y cómo lo implementa la [API de documento XPS](documents-xps.md), consulte [acerca de la API de documentos XPS](about-xps-document-api.md).

## <a name="getting-started-with-the-xps-document-api"></a>Introducción con la API de documento XPS

Antes de empezar a usar la API de documento XPS, asegúrese de que está familiarizado con los siguientes temas de programación:<dl>

[Programación COM](/windows/desktop/com/component-object-model--com--portal)  
[Control de errores en COM](/windows/desktop/com/error-handling-in-com)  
</dl>

Al usar la API de documento XPS, es posible que también desee usar las siguientes tecnologías:<dl>

[DirectWrite](/windows/desktop/DirectWrite/direct-write-portal)  
[API de impresión XPS](./printing-with-the-xpsprint-api.md)  
[API de firma digital XPS](xps-digital-signatures.md)  
</dl>
