# <a name="events-media-streaming-api"></a>Eventos (Media Streaming API)

Media [Streaming API genera](media-streaming-api-portal.md) los siguientes eventos.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                           | Descripción                                                                                                                                              |
|---------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Evento ConnectionStatusChanged**](connectionstatuschanged.md)<br/>     | Se produce cuando cambia el estado de conexión de red del dispositivo.<br/>                                                                                   |
| [**Evento DeviceArrival**](devicearrival.md)<br/>                         | Se produce cuando se agrega un nuevo dispositivo a la lista de dispositivos devueltos por el [**método CachedDevices.**](idevicecontroller-cacheddevices.md)<br/>     |
| [**Evento DeviceDeparture**](devicedeparture.md)<br/>                     | Se produce cuando se quita un nuevo dispositivo de la lista de dispositivos devueltos por el [**método CachedDevices.**](idevicecontroller-cacheddevices.md)<br/> |
| [**Evento RenderingParametersUpdate**](renderingparametersupdate.md)<br/> | Se produce cada vez que se actualiza cualquiera de los parámetros de control de representación en la DMR.<br/>                                                          |
| [**Evento TransportParametersUpdate**](transportparametersupdate.md)<br/> | Se produce cada vez que se actualiza cualquiera de un conjunto de parámetros de transporte en la DMR.<br/>                                                                  |
