# <a name="mft_enum_adapter_luid-attribute"></a>\_ \_ Atributo LUID del adaptador de enumeración de MFT \_

Especifica el identificador único de un adaptador de vídeo. Use este atributo al llamar a [**MFTEnum2**](/windows/desktop/api/mfapi/nf-mfapi-mftenum2) para enumerar MFTs asociados a un adaptador específico.

## <a name="data-type"></a>Tipo de datos

**[**LUID**](/windows/win32/api/devicetopology/ns-devicetopology-luid)**

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|-------------------------------------|------------------------------------------------------------------------------------------|
| Cliente mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows 10, versión 1703 \[\]<br/>                                |
| Servidor mínimo compatible<br/> | Solo aplicaciones de escritorio de Windows Server 2016 \[\]<br/>                                     |
