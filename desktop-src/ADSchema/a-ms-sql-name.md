# <a name="ms-sql-name-attribute"></a>Atributo MS-SQL-Name

Nombre de una SQL Server de datos. El valor predeterminado es MSSQLSERVER.



| Entrada | Valor |
|-------------------|---------------------------------------------|
| CN                | MS-SQL-Name                                 |
| Ldap-Display-Name | mS-SQL-Name                                 |
| Size              | \-                                          |
| Actualizar privilegios  | El sistema establece este valor.            |
| Frecuencia de actualización  | En la configuración del sistema.                            |
| Attribute-Id      | 1.2.840.113556.1.4.1363                     |
| System-Id-Guid    | 3532dfd8-ccee-11d2-9993-0000f87a57d4        |
| Syntax            | [**String(Unicode)**](s-string-unicode.md) |
