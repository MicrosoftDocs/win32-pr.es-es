# <a name="getdvdtextnumberoflanguages-method"></a>Método GetDVDTextNumberOfLanguages

> [!Note]  
> Este componente está disponible para su uso en los sistemas operativos Microsoft Windows 2000, Windows XP y Windows Server 2003. En versiones posteriores podría modificarse o no estar disponible.

 

El `GetDVDTextNumberOfLanguages` método recupera el número de idiomas de texto disponibles en el disco.

``` syntax
[ iLanguages = ] MSWebDVD.GetDVDTextNumberOfLanguages()
```

## <a name="return-value"></a>Valor devuelto

Devuelve un valor entero que especifica el número de idiomas de texto disponibles.
