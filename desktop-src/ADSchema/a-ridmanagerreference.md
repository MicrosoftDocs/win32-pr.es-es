# <a name="rid-manager-reference-attribute"></a>Atributo RID-Manager-Reference

Nombre distintivo para el administrador de RID de un objeto.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | RID-administrador-referencia                   |
| Nombre para mostrar de LDAP | rIDManagerReference                     |
| Tamaño              | \-                                      |
| Actualizar privilegio  | El sistema establece este valor.        |
| Frecuencia de actualización  | Cada vez que cambia el administrador de RID.       |
| Attribute-Id      | 1.2.840.113556.1.4.368                  |
| System-ID-GUID    | 66171886-8f3c-11d0-afda-00c04fd930c9    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
