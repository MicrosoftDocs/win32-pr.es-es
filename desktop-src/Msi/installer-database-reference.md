# <a name="installer-database-reference"></a>Referencia de base de datos del instalador

Para agregar funcionalidad de instalador a la aplicación, puede usar las tablas de base de datos de Windows Installer y las funciones de base de datos del instalador. En los temas de esta sección se describe cada una de las funciones de base de datos y las tablas de base de datos.

Para ver una lista de todas las tablas de base de datos de instalación, vea [tablas de base de datos](database-tables.md).

Para ver una lista de las funciones que se usan para trabajar con información en las bases de datos de instalación, consulte [funciones de base de datos](database-functions.md).
