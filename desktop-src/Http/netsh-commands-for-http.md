# <a name="netsh-commands-for-http"></a>Comandos Netsh para HTTP

Puede usar los comandos Netsh para el contexto HTTP para consultar y configurar la configuración y los parámetros de HTTP.sys.

Puede ejecutar estos comandos en el símbolo del sistema en el sistema operativo Windows Vista o en el símbolo del sistema para el contexto **http de Netsh** . Para que estos comandos funcionen en el símbolo del sistema de Windows Vista, debe escribir **netsh http** antes de escribir comandos y parámetros tal y como aparecen en la sintaxis siguiente. Para ver la ayuda de un comando en el símbolo del sistema, escriba * CommandName ***/?**, donde *CommandName* es el nombre del comando.

Para ver la sintaxis del comando, vea los comandos:

-   [**add iplisten**](add-iplisten.md)
-   [**add sslcert**](add-sslcert.md)
-   [**add timeout**](add-timeout.md)
-   [**add urlacl**](add-urlacl.md)
-   [**eliminar caché**](delete-cache.md)
-   [**delete iplisten**](delete-iplisten.md)
-   [**delete sslcert**](delete-sslcert.md)
-   [**delete timeout**](delete-timeout.md)
-   [**delete urlacl**](delete-urlacl.md)
-   [**flush logbuffer**](flush-logbuffer.md)
-   [**show cachestate**](show-cachestate.md)
-   [**show iplisten**](show-iplisten.md)
-   [**show servicestate**](show-servicestate.md)
-   [**show sslcert**](show-sslcert.md)
-   [**show timeout**](show-timeout.md)
-   [**show urlacl**](show-urlacl.md)
