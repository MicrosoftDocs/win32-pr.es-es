# <a name="component-categories"></a>Categorías del componente

Las categorías de componentes de OLE permiten identificar las capacidades y los requisitos de un componente de software con las entradas del registro. En un escenario en el que un contenedor puede no ser capaz de admitir o no ser compatible con un área de funcionalidad, como el enlace de enlaces por ejemplo, el contenedor no querrá hospedar controles que requieran enlace de los enlaces para realizar su función de trabajo. Las categorías de componentes permiten identificar áreas de funcionalidad como el enlace de los enlaces, de modo que el contenedor de control pueda evitar los controles que lo indican como un requisito. Las categorías de componentes se especifican por separado como parte de OLE y no son específicas de la arquitectura de controles ActiveX. la especificación de las categorías de componentes incluye las API para la manipulación de las claves del registro de la categoría de componentes.
