# <a name="iupdate2-properties"></a>Propiedades de IUpdate2

La interfaz [**IUpdate2**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate2) define las siguientes propiedades.



| Propiedad                                          | Descripción                                                                                                                                             |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**CveIDs**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_cveids)                 | Obtiene una colección de identificadores de vulnerabilidades y exposiciones comunes (CVE) que están asociados a la actualización.                                                |
| [**IsPresent**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_ispresent)           | Obtiene un valor booleano que indica si hay una actualización en un equipo.                                                                         |
| [**RebootRequired**](/windows/desktop/api/Wuapi/nf-wuapi-iupdate2-get_rebootrequired) | Obtiene un valor booleano que indica si se requiere un reinicio del sistema en un equipo para completar la instalación o la desinstalación de una actualización. |



 

Para obtener información sobre los miembros heredados por esta interfaz, vea la siguiente interfaz:

-   [**IUpdate**](/windows/desktop/api/Wuapi/nn-wuapi-iupdate)
