# <a name="programming-the-microsoft-agent-server-interface"></a>Programación de la interfaz del servidor de Microsoft Agent

\[Microsoft Agent está en desuso a partir de Windows 7 y puede que no esté disponible en versiones posteriores de Windows.\]

El agente de Microsoft proporciona servicios que permiten programar caracteres animados desde una aplicación. Estos servicios se implementan como un servidor de automatización OLE. OLE Automation permite a una aplicación controlar el objeto de otra aplicación mediante programación. En este documento se supone que se conoce el modelo de objetos componentes (COM) y OLE. Para obtener una introducción de estos servicios, consulte la [información general sobre la interfaz de programación](microsoft-agent-programming-interface-overview.md).
