# <a name="extending-data-structures-and-types"></a>Extender estructuras y tipos de datos

Se reserva un intervalo de valores para dar cabida a futuras extensiones para el conjunto de funciones TAPI básico y complementario. Para obtener información sobre la cantidad por la que se puede ampliar una estructura de datos, vea [Extensibilidad.](extensibility.md) Para obtener una lista de las funciones que se usan para extender la telefonía, vea [Referencia de función rápida de TAPI.](./tapi-quick-function-reference.md)
