# <a name="events-in-controls"></a>Eventos en controles

Aunque la mayoría de los controles exponen y activen varios eventos, no es necesario que exponan ni activen ningún evento y, por tanto, el control no requiere [**IConnectionPointContainer**](/windows/desktop/api/OCIdl/nn-ocidl-iconnectionpointcontainer). Si el control tiene algún evento, no hay ningún requisito para los eventos que un control debe exponer.
