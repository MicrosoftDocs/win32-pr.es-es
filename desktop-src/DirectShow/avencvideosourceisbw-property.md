# <a name="avencvideosourceisbw-property"></a>AvEncVideoSourceIsBW, propiedad

Especifica si el vídeo es monocromo (blanco y negro) o contiene color.

Esta propiedad es de lectura y escritura.

## <a name="data-type"></a>Tipo de datos

**VARIANT \_ BOOL** (**VT \_ BOOL**)

## <a name="property-guid"></a>GUID de propiedad

**CODECAPI \_ AVEncVideoSourceIsBW**

## <a name="property-value"></a>Valor de propiedad

Si el valor es **VARIANT \_ TRUE,** el vídeo es monocromo. De lo contrario, el vídeo contiene color.
