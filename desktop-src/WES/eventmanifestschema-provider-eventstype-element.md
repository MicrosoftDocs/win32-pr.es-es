# <a name="provider-eventstype-element"></a>Elemento Provider (EventsType)

Define un proveedor y los metadatos que usa para definir sus eventos.

``` syntax
<xs:element name="provider"
    type="ProviderType"
 />
```

El elemento de **proveedor** se define mediante el tipo complejo de [**EventsType**](eventmanifestschema-eventstype-complextype.md) .
