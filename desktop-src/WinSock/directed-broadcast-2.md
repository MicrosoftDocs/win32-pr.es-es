# <a name="directed-broadcast"></a>Difusión dirigida

Generalmente se considera más fácil de usar en la red que una difusión de todas las rutas, una difusión dirigida se limita a la red local que especifique. Rellene **sa \_ netnum con** el número de red de destino y sa **\_ nodenum** con binarios. Los enrutadores reenván esta solicitud a la red de destino donde se convierte en una difusión local. Las redes intermedias no ven este paquete como una difusión.
