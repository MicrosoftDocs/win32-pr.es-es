# <a name="watermark-category-types"></a>Tipos de categoría de marca de agua

Puede usar la función de enumeración de DMO, **DMOEnum**, para obtener información acerca de los DMOs de marca de agua que están instalados en un equipo. El SDK de Windows Media Format define los GUID que identifican las categorías de marcas de agua de DMO. Para obtener más información sobre **DMOEnum**, consulte la documentación de DIRECTSHOW en MSDN.

En la tabla siguiente se enumeran los identificadores de las categorías de marca de agua.



| Marca de agua (constante de categoría DMO) | GUID                                 |
|------------------------------------|--------------------------------------|
| \_marca de \_ agua de audio DMOCATEGORY WMT \_ | 65221c5a-fa75-4b39-b50c-06c336b6a3ef |
| \_límite de \_ vídeo de DMOCATEGORY WMT \_ | 187cc922-8efc-4404-9daf-63f4830df1bc |
