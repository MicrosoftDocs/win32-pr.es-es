# <a name="iupdateinstaller-methods"></a>Métodos IUpdateInstaller

La interfaz [**IUpdateInstaller**](/windows/desktop/api/Wuapi/nn-wuapi-iupdateinstaller) define los siguientes métodos.



| Método                                                    | Descripción                                                                          |
|-----------------------------------------------------------|--------------------------------------------------------------------------------------|
| [**BeginInstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-begininstall)     | Inicia una instalación asincrónica de las actualizaciones.                                  |
| [**BeginUninstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-beginuninstall) | Inicia una desinstalación asincrónica de las actualizaciones.                                |
| [**EndInstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-endinstall)         | Completa una instalación asincrónica de las actualizaciones.                               |
| [**EndUninstall**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-enduninstall)     | Completa una desinstalación asincrónica de las actualizaciones.                             |
| [**Instalar**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-install)               | Inicia una instalación sincrónica de las actualizaciones.                                    |
| [**RunWizard**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-runwizard)           | Inicia un asistente que guía al usuario local a través de los pasos necesarios para instalar las actualizaciones. |
| [**Desinstalar**](/windows/desktop/api/Wuapi/nf-wuapi-iupdateinstaller-uninstall)           | Inicia una desinstalación sincrónica de las actualizaciones.                                  |
