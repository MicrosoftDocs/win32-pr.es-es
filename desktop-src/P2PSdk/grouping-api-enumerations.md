# <a name="grouping-api-enumerations"></a>Agrupar enumeraciones de API

La API de agrupación utiliza las enumeraciones siguientes:



| Enumeración                                                                        | Descripción                                                                                                                                                                                                                                               |
|------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**\_esquema de \_ autenticación de grupo del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_authentication_scheme)    | Define el conjunto de posibles esquemas de autenticación que se pueden usar para autenticar a un grupo del mismo nivel.                                                                                                                                           |
| [**\_tipo de \_ evento de grupo del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_event_type)                          | Contiene los tipos de evento del mismo nivel específicos que pueden producirse en un grupo del mismo nivel.                                                                                                                                                                                |
| [**\_marcas de \_ \_ credenciales de problema de grupo del mismo nivel \_**](/windows/win32/api/p2p/ne-p2p-peer_group_issue_credential_flags) | Especifica si las credenciales del usuario se almacenan dentro de un grupo.                                                                                                                                                                                                  |
| [**\_marcas de \_ propiedad del grupo del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_property_flags)                  | Especifica la configuración de pertenencia a grupos del mismo nivel.                                                                                                                                                                                                                 |
| [**\_Estado de grupo del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_group_status)                                   | Indica si el grupo del mismo nivel tiene o no conexiones presentes.                                                                                                                                                                                          |
| [**\_tipo de \_ cambio de miembro del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_member_change_type)                      | Define el conjunto de posibles estados de presencia y pertenencia a grupos del mismo nivel para un elemento del mismo nivel.                                                                                                                                                                         |
| [**\_marcas de miembro del mismo nivel \_**](/windows/desktop/api/P2P/ne-p2p-peer_member_flags)                                   | Permite a una aplicación especificar si todos los miembros o solo los presentes deben enumerarse cuando se llama a la función [**PeerGroupEnumMembers**](/windows/desktop/api/P2P/nf-p2p-peergroupenummembers) o para indicar si un miembro está presente en el grupo del mismo nivel. |
