# <a name="shortcuts-for-exposing-custom-user-interface-elements"></a>Métodos abreviados para exponer elementos de la interfaz de usuario personalizados

En esta sección se describen los métodos abreviados que los desarrolladores de aplicaciones pueden utilizar para exponer determinados tipos de elementos de la interfaz de usuario personalizados. Las técnicas descritas en esta sección simplifican la implementación de la interfaz [**IAccessible**](/windows/desktop/api/oleacc/nn-oleacc-iaccessible) o proporcionan una alternativa a la exposición de información sin implementar **IAccessible**.

> [!Note]  
> Debe considerar el uso de alguna forma de anotación dinámica (directa, asignación de valores o servidor) antes de intentar las técnicas descritas en esta sección. Para obtener más información, consulte [API de anotación dinámica](dynamic-annotation-api.md).
