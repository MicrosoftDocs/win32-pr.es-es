# <a name="ms-sql-status-attribute"></a>Atributo MS-SQL-Status

Estado del servidor SQL servidor. El estado es **TRUE si** el servidor se está ejecutando.



| Entrada | Valor |
|-------------------|--------------------------------------|
| CN                | MS-SQL-Status                        |
| Ldap-Display-Name | mS-SQL-Status                        |
| Size              | 8 bytes                              |
| Actualizar privilegios  | El sistema establece este valor.     |
| Frecuencia de actualización  | Al iniciar el sistema.                   |
| Attribute-Id      | 1.2.840.113556.1.4.1380              |
| System-Id-Guid    | 9a7d4770-ccee-11d2-9993-0000f87a57d4 |
| Syntax            | [**Intervalo**](s-interval.md)       |
