# <a name="remote-desktop-services-api-data-types"></a>Tipos de datos de Servicios de Escritorio remoto API

Servicios de Escritorio remoto define los siguientes tipos de datos.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**identificador de TLS \_**](tls-handle.md)
</dt> <dd>

Representa un identificador de un servidor de licencias de Escritorio remoto.

</dd> <dt>

[**WTSLISTENERNAME**](wtslistenername.md)
</dt> <dd>

Representa el nombre de una Servicios de Escritorio remoto agentes de escucha en un servidor host de sesión de Escritorio remoto (host de sesión de escritorio remoto).

</dd> </dl>
