# <a name="using-htmlview-with-online-stores"></a>Uso de HTMLView con tiendas en línea

Reproductor de Windows Media solicita permiso a los usuarios para mostrar contenido HTMLView en **Reproducción ahora.** Las tiendas en línea pueden deshabilitar este mensaje especificando un valor de dirección URL base en el **elemento HTMLView** del documento ServiceInfo. Cuando Reproductor de Windows Media una lista de reproducción que especifica el contenido HTMLView que se mostrará, el reproductor compara la dirección URL base del documento ServiceInfo del almacén en línea activo actual con la dirección URL base del contenido HTMLView. Si coinciden, Reproductor de Windows Media muestra el contenido HTMLView sin preguntar al usuario.
