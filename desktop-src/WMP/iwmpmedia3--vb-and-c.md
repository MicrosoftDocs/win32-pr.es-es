# <a name="iwmpmedia3-vb-and-c-interface"></a>Interfaz IWMPMedia3 (VB y C#)

Proporciona métodos adicionales para tener acceso a las propiedades de un elemento multimedia.

## <a name="members"></a>Miembros

La interfaz **IWMPMedia3 (VB y C#)** tiene estos tipos de miembros:

-   [Métodos](#methods)

### <a name="methods"></a>Métodos

La interfaz **IWMPMedia3 (VB y C#)** tiene estos métodos.



| Método                                                                                           | Descripción                                                                                            |
|:-------------------------------------------------------------------------------------------------|:-------------------------------------------------------------------------------------------------------|
| [**getAttributeCountByType**](wmplibiwmpmedia3-iwmpmedia3-getattributecountbytype--vb-and-c.md) | Devuelve el número de atributos asociados al tipo de atributo especificado.<br/>              |
| [**getItemInfoByType**](wmplibiwmpmedia3-iwmpmedia3-getiteminfobytype--vb-and-c.md)             | Devuelve el valor del atributo correspondiente al tipo de atributo y el índice especificados.<br/> |



 

Obtenga una interfaz **IWMPMedia3** convirtiendo el valor devuelto por una de las siguientes propiedades o métodos a los que se tiene acceso a través del objeto o la interfaz siguientes.



| Objeto o interfaz                                               | Propiedad o método                                                                                                                |
|-------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------|
| [IWMPControls](iwmpcontrols--vb-and-c.md)                        | [currentItem](wmplibiwmpcontrols-iwmpcontrols-currentitem--vb-and-c.md)                                                          |
| [AxWindowsMediaPlayer](axwindowsmediaplayer-object--vb-and-c.md) | [currentMedia](axwmplib-axwindowsmediaplayer-currentmedia--vb-and-c.md) , [newMedia](axwmplib-axwindowsmediaplayer-newmedia.md) |
| [IWMPPlaylist](iwmpplaylist--vb-and-c.md)                        | [Elemento](iwmpplaylist-item--vb-and-c.md) ( **obtener \_ elemento** en C#)                                                                   |
