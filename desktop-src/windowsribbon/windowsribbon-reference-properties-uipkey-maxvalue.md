# <a name="ui_pkey_maxvalue"></a>UI \_ PKEY \_ MaxValue

Identifica la propiedad PKEY de la interfaz de usuario \_ \_ .

```
propertyDescription
   name = UI_PKEY_MaxValue
   shellPKey = UI_PKEY_MaxValue
   formatID = 00000203-7363-696e-8441798acf5aebb7
   propID = 203
   typeInfo
      type = VT_DECIMAL
```

## <a name="remarks"></a>Observaciones

\_ \_ Una aplicación usa la interfaz de usuario PKEY MaxValue para consultar el valor máximo del control de [número](windowsribbon-controls-spinner.md) .
