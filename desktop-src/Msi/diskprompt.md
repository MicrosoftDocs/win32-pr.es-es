# <a name="diskprompt-property"></a>Propiedad DiskPrompt

El **valor de la propiedad DiskPrompt** contiene una cadena mostrada por un cuadro de diálogo que solicita un disco. Esta cadena debe incluir el nombre del producto y un marcador de posición para el texto impreso en la superficie de CD-ROM.

## <a name="remarks"></a>Comentarios

Se anuncia como una propiedad de producto.

Windows Instalador en Windows Server 2003 o Windows XP.
