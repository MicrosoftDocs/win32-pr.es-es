# <a name="examples-of-light-aware-user-interfaces"></a>Ejemplos de interfaces de usuario Light-Aware

Ahora que comprende algunos de los principios básicos para optimizar la interfaz de usuario para las distintas condiciones de iluminación, puede ver algunos ejemplos de cómo esta optimización tiene una diferencia al ver el contenido en el exterior. Las siguientes imágenes son comparaciones en paralelo de equipos portátiles en la luz solar directa.

Las siguientes imágenes comparan la interfaz de usuario con reconocimiento ligero con la interfaz de usuario normal en un equipo portátil en la luz solar directa. En la primera fotografía se muestra el contenido que suele tener un aspecto más exterior cuando un equipo portátil usa la configuración de pantalla de uso de la batería. La segunda fotografía muestra la combinación de brillo adaptable y la interfaz de usuario con reconocimiento de luz, y cómo estas características pueden trabajar conjuntamente para aumentar la legibilidad de la pantalla.

![interfaz de usuario normal con un brillo del 40%](images/ui-unaware.png)![interfaz de usuario con reconocimiento luminoso en el brillo del 100%](images/ui-lightaware.png)

Las siguientes fotografías muestran un programa de navegación, tal como se muestra en el exterior con el reconocimiento ligero desactivado y encendido. El contenido interior no se puede usar en el exterior, mientras que el contenido exterior es fácilmente legible. Tenga en cuenta también cómo se minimizan los reflejos cuando se usa una combinación de colores de color negro en blanco, como se muestra en la segunda fotografía.

![interfaz de usuario de navegación normal](images/nav-photo-normal.png)![interfaz de usuario de navegación compatible con Light](images/nav-photo-contrast.png)
