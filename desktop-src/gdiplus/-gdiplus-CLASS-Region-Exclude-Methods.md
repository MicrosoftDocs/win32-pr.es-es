# <a name="regionexclude-methods"></a>Region. Exclude (métodos)

En este tema se enumeran los métodos exclude de la clase [**Region**](/windows/win32/api/gdiplusheaders/nl-gdiplusheaders-region) . Para obtener una lista completa de los métodos de la clase **Region** , consulte [métodos de región](-gdiplus-class-region-methods.md).

| Método                                                                 | Descripción                                                                                                                                                                                         |
|:-----------------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Excluir (rectángulo&)**](/previous-versions//ms534825(v=vs.85))     | El método [**Region:: Exclude**](/previous-versions//ms534825(v=vs.85)) actualiza esta región a la parte de sí mismo que no forma una intersección con el interior del rectángulo especificado.<br/>  |

| [**Excluir (región \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstregion))     | El método [**Region:: Exclude**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstregion)) actualiza esta región a la parte de sí mismo que no forma una intersección con otra región.<br/>                         |
| [**Exclude (GraphicsPath \* )**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstgraphicspath)) | El método [**Region:: Exclude**](/windows/win32/api/gdiplusheaders/nf-gdiplusheaders-region-exclude(inconstgraphicspath)) actualiza esta región a la parte de sí mismo que no forma una intersección con el interior de la ruta de acceso especificada.<br/>            |
