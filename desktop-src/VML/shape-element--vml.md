# <a name="shape-element-vml"></a>Elemento Shape (VML)

En este tema se describe VML, una característica que está desusada en Windows Internet Explorer 9. Las páginas web y las aplicaciones que se basan en VML se deben migrar a SVG u otros estándares ampliamente admitidos.

> [!Note]  
> A partir del 2011 de diciembre, este tema se ha archivado. Como resultado, ya no se mantiene de forma activa. Para obtener más información, vea [contenido archivado](/previous-versions/windows/internet-explorer/ie-developer/). Para obtener información, recomendaciones e instrucciones sobre la versión actual de Windows Internet Explorer, consulte [Centro para desarrolladores de Internet Explorer](https://msdn.microsoft.com/ie/).

**Comentarios:**

Este elemento puede aparecer por sí solo o dentro de un elemento de [Grupo](msdn-online-vml-group-element.md) .

A continuación se encuentra el código mínimo necesario para producir una forma. Debe definir los atributos **colorderelleno**, **posición**, **superior**, **izquierda**, **ancho**, **alto** y **ruta de acceso** , o la forma no se representará.


```HTML
   <v:shape
   fillcolor="green"
   style="position:relative;top:1;left:1;width:200;height:200"
   path = "m 1,1 l 1,200, 200,200, 200,1 x e">
   </v:shape>
```



Además, debe colocar el código siguiente en el elemento [**HTML**](https://msdn.microsoft.com/library/ms535255(v=VS.85).aspx) de la página web para obtener los esquemas correctos para las extensiones de VML y Microsoft Office. Si no usa extensiones de Office, puede omitir el segundo esquema.


```HTML
xmlns:v="urn:schemas-microsoft-com:vml"
xmlns:o="urn:schemas-microsoft-com:office:office"
```



También debe tener lo siguiente en el elemento de [**estilo**](https://developer.mozilla.org/docs/Web/HTML/Element/style) para registrar el comportamiento de las extensiones VML y Microsoft Office. Si no usa extensiones de Office, puede omitir la segunda definición de comportamiento.


```HTML
v\:* { behavior: url(#default#VML); display:inline-block}
o\:* { behavior: url(#default#VML); }
```

**Ejemplo**

El código siguiente incluye una página web VML de ejemplo:


```HTML
<HTML xmlns:v="urn:schemas-microsoft-com:vml"
xmlns:o="urn:schemas-microsoft-com:office:office">
<HEAD>
<STYLE>
v\:* { behavior: url(#default#VML);}
o\:* { behavior: url(#default#VML);}
</STYLE>
<TITLE>VML Sample</TITLE>
</HEAD>
<BODY>
   <v:shape
   fillcolor="green"
   style="position:relative;top:1;left:1;width:200;height:200"
   path = "m 1,1 l 1,200, 200,200, 200,1 x e">
   </v:shape>
</BODY>
</HTML>
```
