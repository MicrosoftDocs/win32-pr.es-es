# <a name="global-address-list-attribute"></a>Atributo global-Address-List

Este atributo se usa en un contenedor de Microsoft Exchange para almacenar el nombre distintivo de una lista global de direcciones (GAL) recién creada. Este atributo debe tener una entrada antes de poder habilitar los clientes de la interfaz de programación de aplicaciones (MAPI) de mensajería para usar una GAL.



| Entrada | Value |
|-------------------|-----------------------------------------|
| CN                | Lista global de direcciones                     |
| Nombre para mostrar de LDAP | globalAddressList                       |
| Tamaño              | \-                                      |
| Actualizar privilegio  | \-                                      |
| Frecuencia de actualización  | \-                                      |
| Attribute-Id      | 1.2.840.113556.1.4.1245                 |
| System-ID-GUID    | f754c748-06f4-11d2-aa53-00c04fd7d83a    |
| Sintaxis            | [**Object(DS-DN)**](s-object-ds-dn.md) |
