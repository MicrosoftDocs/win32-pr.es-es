# <a name="compiling-an-instrumentation-manifest"></a>Compilar un manifiesto de instrumentación

Después [de escribir el manifiesto](writing-an-instrumentation-manifest.md), use el compilador de mensajes para validar el manifiesto y generar los archivos de recursos y encabezados que incluya en el proveedor. Para obtener más información sobre el uso del compilador, vea [**Compilador de mensajes.**](message-compiler--mc-exe-.md) Incluya los archivos de encabezado y de recursos que genera el compilador en el proyecto que contiene el resto del código fuente del proveedor. Para obtener información sobre cómo escribir los eventos definidos en el manifiesto, vea [Developing a Provider](developing-a-provider.md).
