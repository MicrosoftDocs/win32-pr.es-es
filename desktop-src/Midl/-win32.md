# <a name="win32-switch"></a>modificador/Win32

El modificador **/Win32** indica al compilador de MIDL que genere archivos de código auxiliar, o un archivo de biblioteca de tipos, para un entorno de 32 bits.

``` syntax
midl /win32
```

## <a name="switch-options"></a>Opciones de conmutador

Este modificador no tiene parámetros.

## <a name="remarks"></a>Observaciones

El modificador **/Win32** es equivalente a establecer el modificador [**/env**](-env.md) en Win32.
