# <a name="iwmpdvd-vb-and-c-interface"></a>Interfaz IWMPDVD (VB y C#)

La interfaz **IWMPDVD** proporciona propiedades y métodos para trabajar con DVDs.

La interfaz **IWMPDVD** expone las siguientes propiedades.

## <a name="members"></a>Miembros

La interfaz **IWMPDVD (VB y C#)** tiene estos tipos de miembros:

-   [Métodos](#methods)
-   [Propiedades](#properties)

### <a name="methods"></a>Métodos

La interfaz **IWMPDVD (VB y C#)** tiene estos métodos.



| Método                                                         | Descripción                                                                                                     |
|:---------------------------------------------------------------|:----------------------------------------------------------------------------------------------------------------|
| [**Atrás**](wmplibiwmpdvd-iwmpdvd-back--vb-and-c.md)           | Cambia la presentación de un submenú a su menú primario.<br/>                                               |
| [**Recuper**](wmplibiwmpdvd-iwmpdvd-resume--vb-and-c.md)       | Cambia el modo de reproducción del modo de menú, reanudando en la misma posición que cuando se invocó el menú.<br/> |
| [**titleMenu**](wmplibiwmpdvd-iwmpdvd-titlemenu--vb-and-c.md) | Detiene la reproducción y muestra el menú de título.<br/>                                                          |
| [**Menú de menús**](wmplibiwmpdvd-iwmpdvd-topmenu--vb-and-c.md)     | Detiene la reproducción y muestra el menú raíz.<br/>                                                           |



 

### <a name="properties"></a>Propiedades

La interfaz **IWMPDVD (VB y C#)** tiene estas propiedades.



| Propiedad                                                            | Tipo de acceso          | Descripción                                                                                                                                                                          |
|:--------------------------------------------------------------------|:---------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**dominio**](wmplibiwmpdvd-iwmpdvd-domain--vb-and-c.md)<br/> | Solo lectura<br/> | Obtiene el dominio actual del DVD.<br/>                                                                                                                                       |
| [**isAvailable**](iwmpdvd-isavailable--vb-and-c.md)<br/>     | Solo lectura<br/> | Obtiene un valor que indica si un tipo especificado de información está disponible o se puede realizar una acción especificada. En C#, este es el método **Get \_ isavailable** .<br/> |



 

Obtenga una interfaz **IWMPDVD** con la siguiente propiedad.



| Object                                                                   | Propiedad                                                   |
|--------------------------------------------------------------------------|------------------------------------------------------------|
| [Objeto AxWindowsMediaPlayer](axwindowsmediaplayer-object--vb-and-c.md) | [**discos**](axwmplib-axwindowsmediaplayer-dvd--vb-and-c.md) |
