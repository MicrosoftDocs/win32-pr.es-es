# <a name="controls-menu"></a>Menú Controles

El **menú** Controles proporciona comandos de control de transporte para una topología establecida en la sesión multimedia. Este menú solo se habilita después de que una topología se resuelva correctamente. Para acceder rápidamente a estos comandos, puede usar la barra de [herramientas TopoEdit](topoedit-toolbar.md).

El **menú** Controles proporciona los siguientes comandos:



| Comando   | Descripción                                                              |
|-----------|--------------------------------------------------------------------------|
| **Reproducir**  | Inicia la reproducción de la última topología resuelta.                           |
| **Detención**  | Detiene la reproducción de la última topología resuelta que se está reproduciendo actualmente.  |
| **Pausar** | Pausa la reproducción de la última topología resuelta que se está reproduciendo actualmente. |
