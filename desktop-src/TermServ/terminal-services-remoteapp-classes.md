# <a name="remoteapp-classes"></a>Clases RemoteApp

El proveedor de WMI de RemoteApp proporciona las siguientes clases.

## <a name="in-this-section"></a>En esta sección

<dl> <dt>

[**Win32 \_ RDAllowListFileAssociation**](win32-rdallowlistfileassociation.md)
</dt> <dd>

Describe una asociación de tipo de archivo publicada con RemoteApp.

</dd> <dt>

[**Win32 \_ RDFileAssociation**](win32-rdfileassociation.md)
</dt> <dd>

Representa una asociación de tipo de archivo para RemoteApp.

</dd> <dt>

[**Win32 \_ RemoteAppChangeEvent**](win32-remoteappchangeevent.md)
</dt> <dd>

Representa un cambio en la configuración de RemoteApp de Windows Server 2008 R2 en el servidor host de sesión de Escritorio remoto (host de sesión de escritorio remoto).

</dd> <dt>

[**Win32 \_ TSApplicationFileExtensions**](win32-tsapplicationfileextensions.md)
</dt> <dd>

extensiones de nombre de archivo que se administran mediante una aplicación en un servidor host de sesión de escritorio remoto.

</dd> <dt>

[**Win32 \_ TSDeploymentLicensing**](win32-tsdeploymentlicensing.md)
</dt> <dd>

Configuración de licencias para la implementación de Escritorio remoto Virtualization (RDV).

</dd> <dt>

[**Win32 \_ TSDeploymentSettings**](win32-tsdeploymentsettings.md)
</dt> <dd>

Define la configuración predeterminada que utiliza el Administrador de RemoteApp al crear archivos de Protocolo de escritorio remoto (RDP).

</dd> <dt>

[**Win32 \_ TSExpandEnvironmentVariables**](win32-tsexpandenvironmentvariables.md)
</dt> <dd>

Expande las variables de entorno definidas por el sistema.

</dd> <dt>

[**Win32 \_ TSGetIcon**](win32-tsgeticon.md)
</dt> <dd>

Devuelve el contenido del icono especificado por la ruta de acceso y el índice del archivo.

</dd> <dt>

[**Win32 \_ TSPublishedApplication**](win32-tspublishedapplication.md)
</dt> <dd>

Define las aplicaciones que se ponen a disposición del uso remoto a través de RemoteApp de Windows Server 2008 R2.

</dd> <dt>

[**Win32 \_ TSPublishedApplicationList**](win32-tspublishedapplicationlist.md)
</dt> <dd>

Representa la lista de programas que se encuentran en la lista Programas RemoteApp en un servidor host de sesión de escritorio remoto.

</dd> <dt>

[**Win32 \_ TSRemoteDesktop**](win32-tsremotedesktop.md)
</dt> <dd>

Describe una conexión a escritorio remoto que está disponible a través de Escritorio remoto acceso web (acceso web de escritorio remoto).

</dd> <dt>

[**Win32 \_ TSStartMenuApplication**](win32-tsstartmenuapplication.md)
</dt> <dd>

Describe las aplicaciones que se encuentran en el menú Inicio de un servidor host de sesión de escritorio remoto.

</dd> <dt>

[**Win32 \_ TSSystemInfo**](win32-tssysteminfo.md)
</dt> <dd>

Representa el servicio de información del servidor de publicación centralizado

</dd> </dl>
