# <a name="itemthumbheight-property"></a>Propiedad Item. ThumbHeight

Alto, en píxeles, de la imagen en miniatura. Esta propiedad devuelve-1 si este elemento no admite miniaturas. Solo lectura.

Esta propiedad es de solo lectura.

## <a name="syntax"></a>Sintaxis


```JScript
propVal = Item.ThumbHeight
```



## <a name="property-value"></a>Valor de propiedad

Variable que recibe el alto de la miniatura.
