# <a name="using-lsa-policy"></a>Uso de la Directiva LSA

En los temas siguientes se describe cómo usar las funciones de directiva de LSA.

-   El [uso de cadenas Unicode LSA](using-lsa-unicode-strings.md) explica cómo convertir cadenas en el formato de estructura que usan algunas funciones de directiva de LSA.
-   Al [abrir un identificador de objeto de directiva](opening-a-policy-object-handle.md) se describe cómo abrir un identificador para el objeto de [**Directiva**](policy-object.md) local. Muchas de las funciones de directiva de LSA requieren este identificador como parámetro de entrada.
-   [Administrar información de directivas](managing-policy-information.md) detalla cómo establecer y consultar información global de directivas para el sistema local y el dominio.
-   [Recepción de eventos de cambio de directiva](receiving-policy-change-events.md) describe cómo crear y registrar un evento para recibir notificaciones cuando cambia la información de la Directiva.
-   La [Administración de permisos de cuenta](managing-account-permissions.md) explica cómo crear, eliminar y enumerar cuentas, y cómo agregar y quitar privilegios para esas cuentas.
-   [Administrar la información de dominio de confianza](managing-trusted-domain-information.md) describe cómo crear, enumerar y eliminar dominios de confianza, y cómo establecer y recuperar información de dominios de confianza.
-   La [conversión entre nombres y Sid](translating-between-names-and-sids.md) explica cómo asignar los SID a los nombres de cuenta y viceversa.
-   La [asignación de identificadores POSIX](mapping-posix-identifiers.md) describe cómo asignar SID a identificadores posix de 32 bits.
-   [Almacenar datos privados](storing-private-data.md) explica cómo almacenar y recuperar cadenas de datos privados.

Para obtener información acerca de cómo usar la LSA para iniciar sesión y autenticar a los usuarios, consulte [LSA Authentication](/windows/desktop/SecAuthN/lsa-authentication).
