# <a name="d1178-custom-dash-style-requires-dash-array"></a>D1178: El estilo de guión personalizado requiere una matriz dash

La creación de un estilo de trazo con \_ DASH STYLE CUSTOM requiere una \_ matriz de guiones.






 

## <a name="possible-causes"></a>Causas posibles

Cuando la aplicación intentó crear un ID2D1StrokeStyle, usó el atributo D2D1 DASH STYLE CUSTOM, pero proporcionó una \_ \_ \_ matriz de guiones NULL.

## <a name="possible-fixes"></a>Posibles correcciones

Si la aplicación requiere guiones personalizados, asegúrese de que se especifica una matriz de guiones válida. Vea CreateStrokeStyle para obtener un ejemplo. Si un estilo de guion integrado es aceptable, deje la matriz de guiones como NULL y use un especificador como DASH STYLE DASH de D2D1. \_ \_ \_
