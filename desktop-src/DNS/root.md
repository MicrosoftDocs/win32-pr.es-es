# <a name="root"></a>Root

El dominio superior de un espacio de nombres jerárquico, como el espacio de nombres DNS, se denomina [*raíz*](r-gly.md). La raíz, que se define como "" (Nothing) o con más precisión "". (un punto) es la base en la que se debe iniciar cualquier búsqueda de árbol de dominio completo.
