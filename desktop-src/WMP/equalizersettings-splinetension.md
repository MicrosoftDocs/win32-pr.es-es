# <a name="equalizersettingssplinetension"></a>EQUALIZERSETTINGS.splineTension

El **atributo splineTension** especifica o recupera la curva spline para el control de igualador.

``` syntax
        elementID.splineTension
```

## <a name="possible-values"></a>Valores posibles

Este atributo es un número de lectura y **escritura** **(float)** que va de 0,0 (sin tensión) a 10,0 (tensión completa) con un valor predeterminado de 3,0.

## <a name="requirements"></a>Requisitos



| Requisito | Value |
|--------------------|---------------------------------------------------|
| Versión<br/> | Reproductor de Windows Media serie 9 o posterior<br/> |
