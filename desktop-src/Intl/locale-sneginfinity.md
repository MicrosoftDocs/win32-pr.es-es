# <a name="locale_sneginfinity"></a>LOCALE \_ SNEGINFINITY

**Windows Vista y versiones posteriores:** Valor de cadena para "infinito negativo", por ejemplo, "-Infinity" para la configuración regional de inglés (Estados Unidos). No hay ningún límite en el número de caracteres permitidos para esta cadena.
