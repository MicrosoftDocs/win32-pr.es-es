# <a name="context-menu-attribute"></a>Context-Menu atributo)

El número de orden y el GUID del menú contextual que se va a utilizar para un objeto.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | Context-Menu                                |
| Nombre para mostrar de LDAP | contextMenu                                 |
| Tamaño              | \-                                          |
| Actualizar privilegio  | Administrador de esquema                        |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.499                      |
| System-ID-GUID    | 4d8601ee-ac85-11d0-afe3-00c04fd930c9        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
