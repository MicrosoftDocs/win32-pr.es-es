# <a name="rid-used-pool-attribute"></a>Atributo RID-Used-Pool

Los grupos de RID utilizados por un controlador de dominio.



| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | RID: grupo usado                        |
| Nombre para mostrar de LDAP | rIDUsedPool                          |
| Tamaño              | 8 bytes                              |
| Actualizar privilegio  | El sistema establece este valor.     |
| Frecuencia de actualización  | Cada vez que se vacía un grupo de RID.      |
| Attribute-Id      | 1.2.840.113556.1.4.373               |
| System-ID-GUID    | 6617188b-8f3c-11d0-afda-00c04fd930c9 |
| Sintaxis            | [**Interval**](s-interval.md)       |
