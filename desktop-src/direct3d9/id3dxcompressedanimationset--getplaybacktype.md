# <a name="id3dxcompressedanimationsetgetplaybacktype-method"></a>ID3DXCompressedAnimationSet:: GetPlaybackType (método)

Obtiene el tipo del bucle de reproducción establecido de la animación.

## <a name="syntax"></a>Sintaxis


```C++
D3DXPLAYBACK_TYPE GetPlaybackType();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Tipo: **[ **D3DXPLAYBACK \_ Type**](./d3dxplayback-type.md)**

Tipo del bucle de reproducción establecido de la animación. Consulte [**\_ tipo de D3DXPLAYBACK**](./d3dxplayback-type.md).
