# <a name="dynamic-annotation"></a>Anotación dinámica

La anotación dinámica solo estaba disponible anteriormente para servidores y clientes de Microsoft Active Accessibility. Ahora la técnica está disponible para todos los clientes de automatización de la interfaz de usuario, así como algunas implementaciones del proveedor de automatización de la interfaz de usuario y del servidor de MSAA. Además, los clientes de automatización de la interfaz de usuario pueden acceder a las anotaciones realizadas en controles personalizados mediante la anotación dinámica (las tres técnicas).

Para obtener más información, consulte [API de anotación dinámica](dynamic-annotation-api.md).
