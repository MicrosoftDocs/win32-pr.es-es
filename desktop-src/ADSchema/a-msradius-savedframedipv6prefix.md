| Entrada | Value |
|-------------------|--------------------------------------|
| CN                | MS-RADIUS-SavedFramedIpv6Prefix      |
| Nombre para mostrar de LDAP | msRADIUS-SavedFramedIpv6Prefix       |
| Tamaño              | \-                                   |
| Actualizar privilegio  | \-                                   |
