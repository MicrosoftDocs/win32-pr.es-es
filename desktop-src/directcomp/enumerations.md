# <a name="directcomposition-enumerations"></a>Enumeraciones de DirectComposition

En esta sección se describen las enumeraciones proporcionadas por Microsoft DirectComposition API.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                                  | Descripción                                                                                                                                                                                         |
|--------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**VISIBILIDAD DEL FONDO DE \_ LA DCOMPOSITION \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_backface_visibility)<br/>              | Especifica la visibilidad del aspecto posterior que se va a aplicar a un objeto visual. <br/>                                                                                                                            |
| [**MODO DE \_ INTERPOLACIÓN DE MAPA \_ DE BITS DCOMPOSITION \_**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_bitmap_interpolation_mode)<br/> | Especifica el modo de interpolación que se va a usar cuando un mapa de bits se compone de cualquier transformación en la que los píxeles del mapa de bits no se alinean exactamente uno a uno con píxeles en pantalla. <br/>         |
| [**DCOMPOSITION \_ BORDER \_ MODE**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_border_mode)<br/>                              | Especifica el modo de borde que se va a usar al crear un mapa de bits o aplicar un clip con cualquier transformación, de modo que los bordes del mapa de bits o del clip no estén alineados en el eje con coordenadas enteras. <br/> |
| [**DCOMPOSITION \_ COMPOSITE \_ MODE**](/windows/desktop/api/DcompTypes/ne-dcomptypes-dcomposition_composite_mode)<br/>                        | Modo que se usa para combinar el contenido de mapa de bits de un objeto visual con el destino de representación.<br/>                                                                                                          |
| [**DCOMPOSITION \_ OPACITY \_ MODE**](/windows/desktop/api/dcomptypes/ne-dcomptypes-dcomposition_opacity_mode)<br/>                            | Especifica cómo se aplica el valor de opacidad efectivo de un objeto visual al contenido y los elementos secundarios de ese objeto visual.<br/>                                                                                  |
| [**MODO DE TRANSFORMACIÓN DCOMPOSITION \_ \_**](/previous-versions/windows/desktop/legacy/dn904487(v=vs.85))<br/>                        | Especifica el estilo de transformación que se va a aplicar a un objeto visual. <br/>                                                                                                                                |
