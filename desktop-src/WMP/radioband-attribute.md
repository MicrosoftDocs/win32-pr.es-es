# <a name="radioband-attribute"></a>Atributo RadioBand

El **atributo RadioBand** es la banda de frecuencia de la estación de radio.

## <a name="applies-to"></a>Se aplica a

-   [Elementos de radio](radio-item-attributes.md)

## <a name="remarks"></a>Comentarios

Este atributo solo se almacena en la biblioteca.

Este atributo tiene uno de los siguientes valores: "FM", "AM" o "Net".

Para determinar si puede cambiar el valor de este atributo, use el [método Media.isReadOnlyItem.](media-isreadonlyitem.md)

|--------------------|-----------------------------------------------|
