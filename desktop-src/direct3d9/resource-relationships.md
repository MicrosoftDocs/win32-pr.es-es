# <a name="resource-relationships-direct3d-9"></a>Relaciones de recursos (Direct3D 9)

En el diagrama siguiente se muestran las operaciones que son sintácticamente posibles en recursos específicos y su contenido. Para obtener más información sobre cómo usar los recursos, vea [manipular recursos (Direct3D 9)](manipulating-resources.md).

![diagrama de relaciones de recursos](images/resourcerelationship.png)

Las flechas que se ejecutan de izquierda a derecha indican que los tipos de destino se crean mediante los métodos especificados, mientras que las flechas que se ejecutan de derecha a izquierda indican que esos tipos de recursos se pueden pasar como argumentos a los métodos especificados.
