# <a name="glossary-imapi"></a>Glosario (IMAPI)

<dl> <dt>

<span id="imapi.imapi2_glossary_ANSI"></span><span id="imapi.imapi2_glossary_ansi"></span><span id="IMAPI.IMAPI2_GLOSSARY_ANSI"></span>**Ansi**
</dt> <dd>

El American National Standards Institute.

</dd> <dt>

<span id="imapi.imapi2_glossary_BurnEngine"></span><span id="imapi.imapi2_glossary_burnengine"></span><span id="IMAPI.IMAPI2_GLOSSARY_BURNENGINE"></span>**Motor de grabación**
</dt> <dd>

Componente que lee datos de un flujo de datos proporcionado (por ejemplo, IStream o E/S. Stream) y escribe los datos en un dispositivo CD/DVD. El componente debe tener en cuenta el búfer disponible en la unidad, pero no contiene ningún conocimiento del formato de los datos, preparar la unidad para la escritura o informar a la unidad de que el proceso de escritura se ha completado.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDDVDDevice"></span><span id="imapi.imapi2_glossary_cddvddevice"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDDVDDEVICE"></span>**Dispositivo CD/DVD**
</dt> <dd>

Un dispositivo que se notifica a sí mismo como un dispositivo de tipo 5. Esto incluye CD-ROM, DVD-ROM y la mayoría de las unidades de escritura de CD y DVD. Además, algunas unidades con medios magnéticos (como IOMEGA REV) se informan ellos mismos como dispositivos de tipo 5 y se muestran como dispositivos de CD-ROM en Windows.

</dd> <dt>

<span id="imapi.imapi2_glossary_CDG"></span><span id="imapi.imapi2_glossary_cdg"></span><span id="IMAPI.IMAPI2_GLOSSARY_CDG"></span>**CD-G** 
</dt> <dd>

Acrónimo de GRÁFICOS DE CD, tecnología exclusiva de los CD de audio que permite agregar gráficos de baja velocidad de bits a una pista de audio. Esta tecnología se usa con más frecuencia en los CD usados para Asíns.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_R"></span><span id="imapi.imapi2_glossary_cd_r"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_R"></span>**CD-R**
</dt> <dd>

Un disco compacto de una escritura.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RandRW"></span><span id="imapi.imapi2_glossary_cd_randrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RANDRW"></span>**CD-R/RW**
</dt> <dd>

Un disco compacto de CD-R o CD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_ROM"></span><span id="imapi.imapi2_glossary_cd_rom"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_ROM"></span>**CD-ROM**
</dt> <dd>

Disco compacto de solo lectura.

</dd> <dt>

<span id="imapi.imapi2_glossary_CD_RW"></span><span id="imapi.imapi2_glossary_cd_rw"></span><span id="IMAPI.IMAPI2_GLOSSARY_CD_RW"></span>**CD-RW**
</dt> <dd>

Disco compacto reescribible.

</dd> <dt>

<span id="imapi.imapi2_glossaryCDText"></span><span id="imapi.imapi2_glossarycdtext"></span><span id="IMAPI.IMAPI2_GLOSSARYCDTEXT"></span>**CD-Text** 
</dt> <dd>

Tecnología exclusiva de los CD de audio que permite almacenar información de velocidad de bits baja, como el nombre del intérprete y el nombre de la canción, en los medios físicos. Las aplicaciones o dispositivos que reproducen el CD resultante pueden recuperar esta información, si existe, para mostrarla al usuario.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disc"></span><span id="imapi.imapi2_glossary_disc"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISC"></span>**Disco**
</dt> <dd>

Término genérico para medios ópticos de redondeo, como un CD-ROM o un DVD.

</dd> <dt>

<span id="imapi.imapi2_glossary_DiscAtOnce"></span><span id="imapi.imapi2_glossary_discatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISCATONCE"></span>**Disco a la vez** 
</dt> <dd>

Método de escritura en un medio de CD donde todo el contenido de la imagen de grabación se escribe en el disco en una secuencia continua. Además, los datos no accesibles, como el cliente potencial y el cliente potencial, se incluyen en la imagen de grabación.

</dd> <dt>

<span id="imapi.imapi2_glossary_Disk"></span><span id="imapi.imapi2_glossary_disk"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISK"></span>**Disco**
</dt> <dd>

Término genérico para medios magnéticos round, como una disquete o una unidad de disco duro.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dispatch"></span><span id="imapi.imapi2_glossary_dispatch"></span><span id="IMAPI.IMAPI2_GLOSSARY_DISPATCH"></span>**Envío** 
</dt> <dd>

Designación de interfaz COM donde una interfaz determinada admite el **método IDispatch** para llamar a una función. Consulte MSDN para obtener más información sobre este tema de COM avanzado.

</dd> <dt>

<span id="imapi.imapi2_glossary_Dual"></span><span id="imapi.imapi2_glossary_dual"></span><span id="IMAPI.IMAPI2_GLOSSARY_DUAL"></span>**Doble** 
</dt> <dd>

Designación de interfaz COM donde una interfaz determinada admite los métodos VTable e **IDispatch** para llamar a una función. Consulte MSDN para obtener más información sobre estos temas avanzados de COM.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVD"></span><span id="imapi.imapi2_glossary_dvd"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVD"></span>**Dvd**
</dt> <dd>

Un disco versátil digital. Originalmente, esta abreviatura significaba "Disco de vídeo digital".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusR"></span><span id="imapi.imapi2_glossary_dvdplusr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSR"></span>**DVD+R**

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRDL"></span><span id="imapi.imapi2_glossary_dvdplusrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRDL"></span>**DVD+R DL**
</dt> <dd>

Disco DVD+R de doble capa. Este tipo de disco también se conoce como "capa doble de DVD+R".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRRW"></span><span id="imapi.imapi2_glossary_dvdplusrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRRW"></span>**DVD+R/RW**
</dt> <dd>

Un disco DVD+R y un disco DVD+RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusRW"></span><span id="imapi.imapi2_glossary_dvdplusrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSRW"></span>**DVD+RW**
</dt> <dd>

UN DVD que se puede volver a escribir. El acrónimo se pronuncia "DVD Plus RW".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashR"></span><span id="imapi.imapi2_glossary_dvdplusdashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHR"></span>**DVD±R**
</dt> <dd>

Un disco DVD+R y un disco DVD-R.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRRW"></span>**DVD±R/RW**
</dt> <dd>

Un disco DVD+R, un disco DVD-R, un disco DVD+RW y un disco DVD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDPlusDashRW"></span><span id="imapi.imapi2_glossary_dvdplusdashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDPLUSDASHRW"></span>**DVD±RW**
</dt> <dd>

Un disco DVD+RW y un medio DE DVD+R

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashR"></span><span id="imapi.imapi2_glossary_dvddashr"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHR"></span>**DVD-R**
</dt> <dd>

UN DVD de una escritura. El acrónimo se pronuncia "DVD Dash R".

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRDL"></span><span id="imapi.imapi2_glossary_dvddashrdl"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRDL"></span>**DVD-R DL**
</dt> <dd>

Un medio DVD-R de doble capa.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDRAM"></span><span id="imapi.imapi2_glossary_dvdram"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDRAM"></span>**DVD-RAM**
</dt> <dd>

DVD que se puede escribir aleatoriamente. Este tipo de medio admite la administración de defectos de hardware que identifica y evita el uso de pistas dañadas. Las unidades que admiten este formato son más frecuentes en Japón que en Estados Unidos.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDROM"></span><span id="imapi.imapi2_glossary_dvdrom"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDROM"></span>**DVD-ROM**
</dt> <dd>

UN DVD de solo lectura.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRRW"></span><span id="imapi.imapi2_glossary_dvddashrrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRRW"></span>**DVD-R/RW**
</dt> <dd>

Un disco DVD-R y un disco DVD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_DVDDashRW"></span><span id="imapi.imapi2_glossary_dvddashrw"></span><span id="IMAPI.IMAPI2_GLOSSARY_DVDDASHRW"></span>**DVD-RW**
</dt> <dd>

UN DVD que se puede volver a escribir. El acrónimo se pronuncia "DVD Dash RW".

</dd> <dt>

<span id="imapi.imapi2_glossary_ECMA"></span><span id="imapi.imapi2_glossary_ecma"></span><span id="IMAPI.IMAPI2_GLOSSARY_ECMA"></span>**Ecma** 
</dt> <dd>

La Asociación Europea de Fabricantes de Equipos es una asociación del sector creada en 1961 dedicada a la normalización de los sistemas de información y comunicación. ECMA edita estándares e informes técnicos. Todas las publicaciones ecma están disponibles de forma gratuita.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPI"></span><span id="imapi.imapi2_glossary_imapi"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPI"></span>**IMAPI** 
</dt> <dd>

Image Mastering API que se usa para crear y escribir imágenes maestras (en lugar de una FS en directo) en dispositivos CD/DVD. Este término se aplica genéricamente a todas las versiones de Image Mastering API.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv1"></span><span id="imapi.imapi2_glossary_imapiv1"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV1"></span>**IMAPIv1** 
</dt> <dd>

La versión de IMAPI expuesta por primera vez en Windows XP. IMAPI 1.0 admite medios CD-R y CD-RW.

</dd> <dt>

<span id="imapi.imapi2_glossary_IMAPIv2"></span><span id="imapi.imapi2_glossary_imapiv2"></span><span id="IMAPI.IMAPI2_GLOSSARY_IMAPIV2"></span>**IMAPIv2** 
</dt> <dd>

La segunda versión de IMAPI, publicada con Windows Vista.

</dd> <dt>

<span id="imapi.imapi2_glossary_INCITS"></span><span id="imapi.imapi2_glossary_incits"></span><span id="IMAPI.IMAPI2_GLOSSARY_INCITS"></span>**Incits**
</dt> <dd>

El Comité Inter international para estándares de tecnologías de la información, una organización autorizada y aprobada por ANSI.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISO9660"></span><span id="imapi.imapi2_glossary_iso9660"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISO9660"></span>**ISO9660**
</dt> <dd>

Un sistema de archivos diseñado para la primera generación de medios de CD. Este sistema de archivos tiene importantes limitaciones de ordenación y nombre de archivo.

</dd> <dt>

<span id="imapi.imapi2_glossary_ISOImage"></span><span id="imapi.imapi2_glossary_isoimage"></span><span id="IMAPI.IMAPI2_GLOSSARY_ISOIMAGE"></span>**Imagen ISO**
</dt> <dd>

Un nombre común para una imagen sector a sector de todo el área de datos de un disco de CD o DVD de datos. El nombre ha crecido a partir del uso común de ISO9660 como sistema de archivos para medios de CD.

</dd> <dt>

<span id="imapi.imapi2_glossary_Joliet"></span><span id="imapi.imapi2_glossary_joliet"></span><span id="IMAPI.IMAPI2_GLOSSARY_JOLIET"></span>**Joliet**
</dt> <dd>

Un conjunto de extensiones estandarizadas para ISO9660 que admite nombres de archivo UNICODE de hasta 64 caracteres de longitud. Consulte el número de especificación 119 de ECMA para obtener más información.

</dd> <dt>

<span id="imapi.imapi2_glossary_MMC"></span><span id="imapi.imapi2_glossary_mmc"></span><span id="IMAPI.IMAPI2_GLOSSARY_MMC"></span>**Mmc**
</dt> <dd>

Conjunto de comandos multimedia definido por el grupo de trabajo T10 de INCITS. MMC define comandos de paquetes para dispositivos CD y DVD. Los borradores de trabajo actuales de MMC se pueden encontrar en el sitio web de T10.

</dd> <dt>

<span id="imapi.imapi2_glossary_MtFuji"></span><span id="imapi.imapi2_glossary_mtfuji"></span><span id="IMAPI.IMAPI2_GLOSSARY_MTFUJI"></span>**Mt. Fuji**
</dt> <dd>

Conjunto de comandos para controlar dispositivos multimedia, similar al conjunto de comandos mmc. Algunos elementos (como características o tipos de medios) se describen en mt. Los documentos de Momento, mientras que otros se describen primero en los documentos mmc. Los borradores de trabajo actuales del conjunto de comandos se pueden encontrar [en ftp://ftp.avc-pioneer.com](ftp://ftp.avc-pioneer.com/).

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSI"></span><span id="imapi.imapi2_glossary_scsi"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSI"></span>**Scsi**
</dt> <dd>

Interfaz del sistema de equipos pequeños.

</dd> <dt>

<span id="imapi.imapi2_glossary_SCSICommand"></span><span id="imapi.imapi2_glossary_scsicommand"></span><span id="IMAPI.IMAPI2_GLOSSARY_SCSICOMMAND"></span>**Comando SCSI**
</dt> <dd>

Cualquiera de varios comandos de paquetes, como los definidos en el documento mmc. Estos comandos se pueden enviar mediante cualquier mecanismo de transporte adecuado, como SCSI, ATAPI, USB o IEEE1394.

</dd> <dt>

<span id="imapi.imapi2_glossary_SessonAtOnce"></span><span id="imapi.imapi2_glossary_sessonatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_SESSONATONCE"></span>**Sesión a la vez**
</dt> <dd>

Un método para crear un disco de CD Plus. Una sesión de escritura registra pistas de audio en el disco y una segunda sesión escribe una sola pista CD-XA en el disco y cierra el disco.

</dd> <dt>

<span id="imapi.imapi2_glossary_TrackAtOnce"></span><span id="imapi.imapi2_glossary_trackatonce"></span><span id="IMAPI.IMAPI2_GLOSSARY_TRACKATONCE"></span>**Seguimiento a la vez**
</dt> <dd>

Un método de escritura en un medio de CD en el que los datos de una sola pista se escriben al mismo tiempo. Cuando se escriben todas las pistas, la sesión se cierra como un paso final que implica escribir las áreas de clientes potenciales y de clientes potenciales del disco.

</dd> <dt>

<span id="imapi.imapi2_glossary_UDF"></span><span id="imapi.imapi2_glossary_udf"></span><span id="IMAPI.IMAPI2_GLOSSARY_UDF"></span>**Formato de disco universal (UDF)**
</dt> <dd>

Este sistema de archivos se define mediante LED (asociación de tecnología de Storage óptico) que admite discos de sesión única y de sesión múltiple.

</dd> <dt>

<span id="imapi.imapi2_glossary_VTable"></span><span id="imapi.imapi2_glossary_vtable"></span><span id="IMAPI.IMAPI2_GLOSSARY_VTABLE"></span>**Vtable** 
</dt> <dd>

Designación de interfaz COM donde una interfaz determinada admite el método vtable de llamar a una función. Consulte MSDN para obtener más información sobre este tema com avanzado.

</dd> </dl>
