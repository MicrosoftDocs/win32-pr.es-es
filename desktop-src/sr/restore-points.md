### <a name="cause"></a><span data-ttu-id="6619f-157">Causa</span><span class="sxs-lookup"><span data-stu-id="6619f-157">Cause</span></span>

<span data-ttu-id="6619f-158">Se trata de un problema conocido en Windows 10.</span><span class="sxs-lookup"><span data-stu-id="6619f-158">This is a known issue in Windows 10.</span></span>
