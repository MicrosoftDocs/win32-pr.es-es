# <a name="locale_syearmonth"></a>LOCALE \_ SYEARMONTH

**Windows Me/98, Windows 2000:** Cadena de formato de mes de año para la configuración regional. El número máximo de caracteres permitido para esta cadena es 80, incluido un carácter nulo de terminación. Esta cadena muestra el formato adecuado para una cadena de fecha que contiene solo el año y el mes, usando imágenes de formato como se define en Imágenes de formato de [día, mes, año y era.](day--month--year--and-era-format-pictures.md)
