# <a name="d2d1_matrix_4x4_f"></a>\_Matriz D2D1 \_ 4x4 \_ F

Representa una matriz de 4 por 4.


```C++
typedef D2D_MATRIX_4X4_F D2D1_MATRIX_4X4_F;
```



## <a name="remarks"></a>Observaciones

**D2D1 \_ La matriz \_ 4x4** es un nombre nuevo para la estructura de D2D de la [**\_ matriz \_ 4x4 \_**](/windows/desktop/api/dcommon/ns-dcommon-d2d_matrix_4x4_f) . Para obtener una lista de los campos proporcionados por la matriz, vea **D2D \_ Matrix \_ 4x4 \_ F**.
