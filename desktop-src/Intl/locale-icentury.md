# <a name="locale_icentury"></a>\_ICENTURY DE CONFIGURACIÓN REGIONAL

Un siglo de 2 o 4 dígitos solo para la fecha corta. El siglo puede tener uno de los valores siguientes. Es preferible que la aplicación use [LOCALE \_ SSHORTDATE en](locale-sshortdate.md) lugar de LOCALE \_ ICENTURY.



| Value | Significado                     |
|-------|-----------------------------|
| 0     | Siglo abreviado de 2 dígitos |
| 1     | Siglo completo de 4 dígitos        |
