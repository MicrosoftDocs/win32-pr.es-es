# <a name="idcompositionrotatetransform3dsetaxisy-methods"></a>Métodos IDCompositionRotateTransform3D::SetAxisY

Cambia o anima el valor de la propiedad AxisY de una transformación de rotación. La propiedad AxisY especifica la coordenada y para el vector de rotación del eje. El valor predeterminado es cero.

### <a name="overload-list"></a>Lista de sobrecarga



| Método                                                                                                       | Descripción                                          |
|:-------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------|
| [**SetAxisY(float)**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(float))                                     | Cambia el valor de la propiedad AxisY.<br/>  |
| [**SetAxisY(IDCompositionAnimation \* )**](/windows/win32/api/dcomp/nf-dcomp-idcompositionrotatetransform3d-setaxisy(idcompositionanimation)) | Anima el valor de la propiedad AxisY.<br/> |
