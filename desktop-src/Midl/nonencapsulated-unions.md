# <a name="nonencapsulated-unions"></a>Uniones no encapsuladas

Una Unión no encapsulada es una Unión que no está contenida en una estructura con su discriminante. La Unión no encapsulada se indica mediante la presencia del **\[ \_ tipo \] de modificador** de atributo de tipo y el **\[ modificador \_ \]** de atributo de campo es.

Para obtener una descripción del formulario de una Unión no encapsulada, vea [**Union**](union.md).

La forma de las uniones debe ser la misma en todas las plataformas para garantizar la interconectividad.

Para obtener más información, vea [uniones encapsuladas](encapsulated-unions.md), [**Union**](union.md), **\[** [**Switch \_ is**](switch-is.md) **\]** y **\[** [**\_ Type switch**](switch-type.md) **\]** .
