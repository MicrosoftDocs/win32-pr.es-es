# <a name="cryptography-essentials"></a>Cryptography Essentials

Microsoft CryptoAPI, la interfaz de programación de aplicaciones (API) y CAPICOM proporcionan servicios que permiten a los desarrolladores agregar seguridad basada en la [*Criptografía*](../secgloss/c-gly.md) a las aplicaciones. CryptoAPI incluye funcionalidad para codificar y descodificar a partir de la [*notación de sintaxis abstracta uno*](../secgloss/a-gly.md) (ASN. 1), el cifrado y descifrado de datos, la autenticación mediante [*certificados digitales*](../secgloss/d-gly.md)y la administración de certificados digitales con almacenes de certificados. CryptoAPI y CAPICOM admiten la criptografía de clave simétrica y PKI.

Cryptography Essentials contiene los siguientes temas:

-   [Criptografía, CryptoAPI y CAPICOM](cryptography--cryptoapi--and-capicom.md)
-   [Claves criptográficas](cryptographic-keys.md)
-   [Datos codificados y descodificados](encoded-and-decoded-data.md)
-   [Cifrado y descifrado de datos](data-encryption-and-decryption.md)
-   [Hashes y firmas digitales](hashes-and-digital-signatures.md)
-   [Marcas de tiempo de las firmas de Authenticode](time-stamping-authenticode-signatures.md)
-   [Certificados digitales](digital-certificates.md)
-   [Mensajes digitales](digital-messages.md)
