# <a name="boundingfrustumcontains-methods"></a>Métodos BoundingFrustum.Contains

Comprueba si [**BoundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contiene un objeto especificado.

### <a name="overload-list"></a>Lista de sobrecarga

| Método | Descripción |
|-|-|
| [**BoundingFrustum::Contains (const BoundingBox&)**](/windows/desktop/api/directxcollision/nf-directxcollision-boundingfrustum-contains(constboundingbox_)) | Comprueba si [**boundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contiene el [**valor de BoundingBox especificado.**](/windows/desktop/api/DirectXCollision/ns-directxcollision-boundingbox) |
| [**BoundingFrustum::Contains (const BoundingFrustum&)**](/windows/desktop/api/directxcollision/nf-directxcollision-boundingfrustum-contains(constboundingfrustum_)) | Comprueba si [**boundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contiene el **valor de BoundingFrustum especificado.** |
| [**BoundingFrustum::Contains (const BoundingOrientedBox&)**](/windows/desktop/api/directxcollision/nf-directxcollision-boundingfrustum-contains(constboundingorientedbox_)) | Comprueba si [**boundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contiene el [**valor de BoundingOrientedBox especificado.**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingorientedbox) |
| [**BoundingFrustum::Contains (const BoundingSphere&)**](/windows/desktop/api/directxcollision/nf-directxcollision-boundingfrustum-contains(constboundingsphere_)) | Comprueba si [**BoundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contiene el [**valor de BoundingSphere especificado.**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingsphere) |
| [**BoundingFrustum::Contains (FXMVECTOR)**](/windows/desktop/api/directxcollision/nf-directxcollision-boundingfrustum-contains) | Comprueba si [**BoundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contiene el punto especificado. |
| [**BoundingFrustum::Contains (FXMVECTOR, FXMVECTOR, FXMVECTOR)**](/windows/desktop/api/directxcollision/nf-directxcollision-boundingfrustum-contains(fxmvector_fxmvector_fxmvector)) | Comprueba si [**boundingFrustum**](/windows/desktop/api/directxcollision/ns-directxcollision-boundingfrustum) contiene el triángulo especificado. |
