# <a name="dynamicui-element"></a>Elemento DynamicUI

Representa un contenedor para el marcado de la cinta de opciones. Este contenedor y su contenido se declaran en tiempo de ejecución.

## <a name="usage"></a>Uso

``` syntax
<DynamicUI>
  child elements
</DynamicUI>
```

## <a name="attributes"></a>Atributos

No hay atributos.

## <a name="child-elements"></a>Elementos secundarios



| Elemento                                                               | Descripción                                     |
|-----------------------------------------------------------------------|-------------------------------------------------|
| [**ContextPopup**](windowsribbon-element-contextpopup.md)<br/> | Debe aparecer exactamente una vez <br/> <br/> |
| [**Tabulación**](windowsribbon-element-tab.md)<br/>                   | Debe aparecer exactamente una vez <br/> <br/> |
| [**TabGroup**](windowsribbon-element-tabgroup.md)<br/>         | Debe aparecer exactamente una vez <br/> <br/> |



## <a name="parent-elements"></a>Elementos primarios

No hay elementos primarios.

## <a name="element-information"></a>Información de elemento



|                                     |                                                                         |
|-------------------------------------|-------------------------------------------------------------------------|
| Sistema mínimo compatible<br/> | Windows 7, Windows Vista con SP2 y la actualización de la plataforma para Windows Vista |
| Puede estar vacío                        | No                                                                      |
