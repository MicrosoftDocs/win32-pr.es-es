# <a name="tasknamedvaluecollectionitem-property"></a>TaskNamedValueCollection. Item (propiedad)

En el caso de scripting, obtiene el par nombre-valor especificado de la colección.

## <a name="syntax"></a>Sintaxis


```VB
TaskNamedValueCollection.Item( _
  ByVal index, _
  ByRef pair _
) As long
```



## <a name="property-value"></a>Valor de propiedad

Objeto [**TaskNamedValuePair**](tasknamedvaluepair.md) que representa el par solicitado.
