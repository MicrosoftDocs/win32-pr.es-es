# <a name="the-private-data-object-type"></a>El tipo de objeto de datos privado

Puede usar objetos de datos privados para almacenar cadenas de datos en un formato cifrado. Un sistema puede contener varios objetos de datos privados. Los objetos de datos privados se proporcionan principalmente para admitir el almacenamiento de contraseñas de cuentas de servidor. La información almacenada en objetos de datos privados se cifra con una clave específica del sistema generada.
