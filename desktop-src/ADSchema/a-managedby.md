# <a name="managed-by-attribute"></a>Managed-By atributo

Nombre distintivo del usuario asignado para administrar este objeto.



| Entrada | Value |
|-------------------|-------------------------------------------------------------------------------------|
| CN                | Managed-By                                                                          |
| Ldap-Display-Name | managedBy                                                                           |
| Size              | \-                                                                                  |
| Actualizar privilegios  | Administrador de dominio o propietario de la cuenta.                                              |
| Frecuencia de actualización  | Cuando se crea el registro del usuario y siempre que los objetos administrados necesitan cambiar. |
| Attribute-Id      | 1.2.840.113556.1.4.653                                                              |
| System-Id-Guid    | 0296c120-40da-11d1-a9c0-0000f80367c1                                                |
