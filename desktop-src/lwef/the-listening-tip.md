# <a name="the-listening-tip"></a>Sugerencia de escucha

\[Microsoft Agent está en desuso a partir de Windows 7 y puede no estar disponible en versiones posteriores de Windows.\]

La sugerencia de escucha es otro servicio de entrada de voz proporcionado por Microsoft Agent. Cuando se instala la entrada de voz, el Agente incluye una ventana de información sobre herramientas especial que aparece cuando el usuario presiona la tecla de acceso rápido Listening o llama al método Listen. La sugerencia de escucha solo aparece cuando los servicios de voz están disponibles. Si ningún cliente ha creado un comando de voz o carga correctamente un motor de voz, la sugerencia de escucha no aparece. Además, tanto la entrada de voz como la opción Mostrar Sugerencias en Opciones avanzadas de caracteres deben estar habilitadas para que aparezca la sugerencia.

La sugerencia de escucha se tiempos de espera automáticamente después de presentarse. Si el tiempo de espera del texto "Heard" se completa mientras el usuario sigue manteniendo presionada la tecla de acceso rápido, la sugerencia se revierte al texto de "escucha" a menos que el servidor reciba otra expresión correspondiente. En este caso, la sugerencia muestra el nuevo texto "Heard" y comienza el tiempo de espera para ese texto de sugerencia. Si el usuario libera la tecla de acceso rápido y el servidor muestra el texto "Heard", el tiempo de espera continúa y la ventana Sugerencia de escucha se oculta cuando transcurre el intervalo de tiempo de espera.

Si el servidor aún no ha intentado cargar un motor de reconocimiento de voz, no se mostrará la sugerencia de escucha. De forma similar, si el usuario ha deshabilitado la presentación de la sugerencia de escucha o ha deshabilitado la entrada de voz en Opciones avanzadas de caracteres, no se mostrará la sugerencia de escucha.

La sugerencia de escucha no aparece cuando el puntero está sobre el icono de la barra de tareas del carácter. En su lugar, aparece la ventana de sugerencias de notificación estándar y muestra el nombre del carácter.

Las aplicaciones cliente no pueden escribir directamente en la sugerencia de escucha, pero puede especificar texto alternativo que el servidor muestra al reconocer un comando de voz correspondiente. Para ello, establezca la [**propiedad Confidence**](confidence-property.md) y la nueva [**propiedad ConfidenceText**](confidencetext-property.md)para el comando. Si la entrada hablada coincide con el comando , pero la mejor coincidencia no supera la configuración de confianza, el servidor usa el conjunto de texto en la propiedad **ConfidenceText** en la ventana de sugerencias. Si el cliente no proporciona este valor, el servidor muestra el texto (gramática) con el que coincidió.

El texto De sugerencia de escucha aparece en el idioma en función de la configuración del identificador de idioma de caracteres del cliente activo de entrada, independientemente de si hay un motor de reconocimiento de voz compatible con el idioma disponible.
