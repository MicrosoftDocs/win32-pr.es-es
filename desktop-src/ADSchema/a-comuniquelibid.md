# <a name="com-unique-libid-attribute"></a>Atributo de identificador único de COM

IDENTIFICADOR único GUID de cadena de un solo valor para una biblioteca de tipos.



| Entrada | Value |
|-------------------|---------------------------------------------|
| CN                | IDENTIFICADOR único de COM                            |
| Nombre para mostrar de LDAP | cOMUniqueLIBID                              |
| Tamaño              | \-                                          |
| Actualizar privilegio  | \-                                          |
| Frecuencia de actualización  | \-                                          |
| Attribute-Id      | 1.2.840.113556.1.4.250                      |
| System-ID-GUID    | 281416da-1968-11d0-a28f-00aa003049e2        |
| Sintaxis            | [**String(Unicode)**](s-string-unicode.md) |
