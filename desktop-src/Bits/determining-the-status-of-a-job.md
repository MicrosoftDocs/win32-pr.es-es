# <a name="determining-the-status-of-a-job"></a>Determinar el estado de un trabajo

El estado de un trabajo hace referencia a los cambios en el estado de un trabajo (por ejemplo, el trabajo cambia el estado de BG JOB STATE TRANSFERRING a BG JOB STATE TRANSFERRED) y al progreso en la transferencia de los \_ \_ \_ \_ \_ \_ archivos. Para capturar cambios en el estado de un trabajo, use las siguientes opciones:

-   [Sondee el estado del trabajo.](polling-for-the-status-of-the-job.md)
-   [Regístrese para recibir la notificación de eventos](registering-a-com-callback.md).
-   [Regístrese para ejecutar un programa](registering-to-execute-a-program.md) basado en un evento.

Para obtener información sobre el progreso de un trabajo, vea [Determinar el progreso de un trabajo.](determining-the-progress-of-a-job.md)
