# <a name="hyper-v-application-health-monitoring-wmi-classes"></a>Clases WMI de supervisión de estado de aplicaciones de Hyper-V

La API de supervisión de estado de aplicaciones de Hyper-V define las siguientes clases WMI.

## <a name="in-this-section"></a>En esta sección



| Tema                                                                                        | Descripción                                                                                                                                                                 |
|----------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**MSVM \_ HeartbeatComponent**](msvm-heartbeatcomponent.md)<br/>                       | Representa el estado del servicio de latido, que es responsable de supervisar el estado de una máquina virtual mediante la notificación de un latido a intervalos regulares.<br/> |
| [**MSVM \_ HeartbeatComponentSettingData**](msvm-heartbeatcomponentsettingdata.md)<br/> | Representa el estado configurado del servicio de latido.<br/>                                                                                                        |
