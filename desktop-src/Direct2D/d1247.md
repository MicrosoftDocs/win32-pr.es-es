# <a name="d1247-dynamic-copy-from-memory"></a>D1247: Copia dinámica desde la memoria

CopyFromMemory no puede copiar memoria en un mapa de bits creado a partir de una textura dinámica.





 

## <a name="possible-causes"></a>Causas posibles

Se llamó a CopyFromMemory en un mapa de bits que se creó como D3D11 \_ USAGE \_ DYNAMIC.

## <a name="possible-fixes"></a>Posibles correcciones

Use Map y Unmap en su lugar para este tipo de mapa de bits.
