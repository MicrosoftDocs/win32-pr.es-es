# <a name="mf_mt_dv_aaux_ctrl_pack_1-attribute"></a>Atributo MF \_ MT DV A CTRL PACK \_ \_ \_ \_ \_ 1

Paquete de control de código fuente auxiliar de audio (A ODBC) para el segundo bloque de audio en un tipo de medio de vídeo digital (DV).

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Comentarios

Este atributo corresponde al miembro **dwDVACtlCtl1** de la estructura DirectShow [**DVINFO.**](/windows/win32/api/strmif/ns-strmif-dvinfo)

La constante GUID para este atributo se exporta desde mfuuid.lib.
