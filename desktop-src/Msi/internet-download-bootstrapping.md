# <a name="internet-download-bootstrapping"></a>Arranque de descarga de Internet

Con Windows Installer, se incluye un ejecutable de bootstrap configurable (Setup.exe) y una herramienta de configuración ([Msistuff.exe](msistuff-exe.md)) en los [componentes Windows SDK para Windows Installer desarrolladores](platform-sdk-components-for-windows-installer-developers.md). Mediante el uso de Msistuff.exe para configurar los recursos en Setup.exe, los desarrolladores pueden crear fácilmente una instalación Web de un paquete de Windows Installer.

La versión de instalador mínima requerida por el ejecutable de bootstrap es Windows Installer versión 2,0. Las aplicaciones pueden obtener la versión del instalador mediante [**DllGetVersion**](/windows/win32/api/shlwapi/nc-shlwapi-dllgetversionproc). Para obtener más información, vea [determinar la versión de Windows Installer](determining-the-windows-installer-version.md)

El ejecutable de bootstrap proporcionado con el SDK de Windows Installer hace lo siguiente:

-   Llama a [**WinVerifyTrust**](/windows/win32/api/wintrust/nf-wintrust-winverifytrust) para comprobar la firma digital del archivo. msi. Windows Installer versión 2,0 y versiones posteriores proporciona un paquete de interfaz de asunto (SIP) para habilitar la firma de paquetes de Windows Installer.
-   Si es necesario, actualiza la versión de la Windows Installer en la máquina.

## <a name="setupexe-command-line-parameters"></a>Parámetros de la línea de comandos Setup.exe

En la tabla siguiente se enumeran las opciones de comando que se pueden usar con la Setup.exe que se incluye con los [componentes de Windows SDK para Windows Installer desarrolladores](platform-sdk-components-for-windows-installer-developers.md). Cuando se invoca sin ninguna opción de línea de comandos, Setup.exe toma como valor predeterminado el comportamiento especificado a través de los recursos configurados. Las opciones de la línea de comandos se excluyen mutuamente; solo se puede usar una de ellas a la vez.



| Opción | Uso                                                                      | Significado                                                                                                                                                                                                                                                                                                         |
|--------|----------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **/a** | setup.exe **/a**                                                           | Inicia una instalación administrativa. Si la \_ propiedad de base de datos ISETUPPROPNAME está configurada en Setup.exe, esta opción se puede usar para realizar una instalación administrativa del producto. Esta opción de línea de comandos no se admite si el \_ recurso de revisión ISETUPPROPNAME está configurado.<br/> |
| **/a** | setup.exe **/a** *<ruta de acceso completa a una instalación administrativa existente>* | Si la \_ propiedad revisión ISETUPPROPNAME está configurada en Setup.exe, esta opción se puede usar para aplicar una revisión a una instalación administrativa existente del producto. Esta opción de línea de comandos no se admite si el \_ recurso de base de datos ISETUPPROPNAME está configurado.                                               |
| **/v** | setup.exe **/v***<full path to a file>*                             | Comprueba la firma del archivo mediante WinVerifyTrust. No se muestra ninguna interfaz de usuario cuando se usa esta opción. En este caso, el resultado de la comprobación de confianza es el valor devuelto Obtenido de la ejecución de Setup.exe.                                                                                                 |
| **/?** |                                                                            | Muestra un cuadro de diálogo de ayuda que indica los argumentos válidos. Nota: el cuadro de diálogo ayuda también se muestra si se proporcionan argumentos no válidos.                                                                                                                                                                               |
