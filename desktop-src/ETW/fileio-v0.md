# <a name="fileio_v0-class"></a>Clase FileIo \_ V0

Esta clase es la clase primaria para los eventos de E/S de archivo.

La sintaxis siguiente se simplifica a partir del código MOF.

## <a name="syntax"></a>Sintaxis

``` syntax
[Guid("{90cbdc39-4a3e-11d1-84f4-0000f80464e3}"), EventVersion(0)]
class FileIo_V0 : MSNT_SystemTrace
{
};
```

## <a name="members"></a>Miembros

La **clase FileIo \_ V0** no define ningún miembro.
