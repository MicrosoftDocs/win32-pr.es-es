# <a name="user-cannot-change-password-winnt-provider"></a>El usuario no puede cambiar la contraseña (proveedor winNT)

La capacidad de un usuario para cambiar su contraseña es un permiso que se puede conceder o denegar. Para obtener más información sobre cómo leer y modificar este permiso mediante programación mediante el proveedor WinNT, consulte:

-   [Leer que el usuario no puede cambiar la contraseña (proveedor winNT)](reading-user-cannot-change-password-winnt-provider.md)
-   [Modificar el usuario no puede cambiar la contraseña (proveedor winNT)](modifying-user-cannot-change-password-winnt-provider.md)

El **atributo userFlags** del objeto de usuario solo está disponible cuando se usa el proveedor WinNT.

También es posible leer y modificar este permiso mediante el proveedor LDAP. Para obtener más información, vea [El usuario no puede cambiar la contraseña (proveedor LDAP).](user-cannot-change-password.md)
