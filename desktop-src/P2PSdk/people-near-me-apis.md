# <a name="people-near-me-apis"></a>API de equipos a mi alrededor

Cualquier aplicación que pueda usar [People Near me](about-people-near-me.md) (PNM) puede utilizar las siguientes API de Win32:

### <a name="contacts-api"></a>API de contactos

La API de contactos se usa para obtener acceso a la información de contacto del usuario que ha iniciado sesión, recuperar información acerca de los contactos previamente almacenados o almacenar la información de contacto y el certificado digital de un nuevo contacto.

### <a name="people-near-me-api"></a>API de equipos a mi alrededor

La API PNM se usa para buscar usuarios cercanos, sus intereses anunciados, cualquier objeto arbitrario que decida publicar y aplicaciones anunciadas.

### <a name="publication-api"></a>API de publicación

La API de publicación se usa para interactuar con los contactos remotos. El módulo PNM también usa la capa de [señal del mismo nivel](windows-peer-components-for-people-near-me.md) usada por el administrador de publicación para establecer conexiones.

### <a name="invitation-api"></a>API de invitación

Una aplicación de colaboración usa la API de invitación para invitar a un par específico a una actividad de colaboración.

### <a name="application-registration-api"></a>API de registro de aplicaciones

El instalador de una aplicación de colaboración usa la API de registro de aplicaciones para registrar la aplicación. Una aplicación compatible con PNM le pedirá al usuario durante el proceso de instalación para determinar si la aplicación debe estar disponible para los usuarios en la subred.
