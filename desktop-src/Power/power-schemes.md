# <a name="power-schemes"></a>Planes de energía

Un plan de energía es una colección de valores de configuración que controla el uso de energía del equipo. Puede usar los esquemas de energía para reducir el consumo de energía de dispositivos individuales o todo el sistema. Puede configurar características opcionales, como la compatibilidad con la hibernación, la cantidad de tiempo que el sistema debe estar inactivo antes de desactivar la pantalla o el estado de suspensión que se usa cuando el temporizador de inactividad del sistema expira.

Hay varias maneras de crear esquemas de energía: la aplicación opciones de energía del panel de control, la herramienta de línea de comandos PowerCfg.exe y la API de plan de energía. Para obtener más información acerca de las opciones de energía o PowerCfg.exe, consulte la ayuda incluida en el sistema operativo.

-   [Administración de combinaciones de energía](managing-power-schemes.md)
-   [Invalidaciones de administrador](administrator-overrides.md)
