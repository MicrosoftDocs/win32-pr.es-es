# <a name="about-pipes"></a>Acerca de las canalizaciones

Hay dos tipos de canalizaciones: [canalizaciones anónimas](anonymous-pipes.md) y [con nombre](named-pipes.md). Las canalizaciones anónimas requieren menos sobrecarga que las canalizaciones con nombre, pero ofrecen servicios limitados.

El término canalización, tal y como se usa aquí, implica que se usa una canalización como conducto de información. Conceptualmente, una canalización tiene dos extremos. Una canalización unidireccional permite que el proceso de un extremo escriba en la canalización y permite que el otro extremo Lea de la canalización. Una canalización bidireccionales (o dúplex) permite que un proceso Lea y escriba desde su extremo de la canalización.

-   [Canalizaciones anónimas](anonymous-pipes.md)
-   [Canalizaciones con nombre](named-pipes.md)
