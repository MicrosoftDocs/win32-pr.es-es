# <a name="sv_stencilref"></a>SV \_ StencilRef

SV StencilRef representa el valor de referencia de la galería de símbolos del \_ sombreador de píxeles actual.

## <a name="type"></a>Tipo



|      |
|------|
| Tipo |
| uint |
