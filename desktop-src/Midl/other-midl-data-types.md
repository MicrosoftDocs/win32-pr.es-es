# <a name="other-midl-data-types"></a>Otros tipos de datos MIDL

MIDL reconoce una serie de tipos de datos COM y de automatización, **como BSTR**, **Variant** y **SAFEARRAY**, y los identificadores com, **como HMETAFILE \_ PICT**, **HENHMETAFILE**, **HMETAFILE**, **hBitmap**, **HPALETTE** y **HGLOBAL**. Para obtener más información, incluidos los nombres de los archivos de importación necesarios para usar estos tipos, vea [serialización de tipos de datos OLE](marshaling-ole-data-types.md).
