# <a name="tb_isbuttonhighlighted-message"></a>\_Mensaje ISBUTTONHIGHLIGHTED TB

Comprueba el estado de resaltado de un botón de la barra de herramientas.

## <a name="parameters"></a>Parámetros

<dl> <dt>

*wParam* 
</dt> <dd>

Identificador de comando para un botón de la barra de herramientas.

</dd> <dt>

*lParam* 
</dt> <dd>Debe ser cero.</dd> </dl>

## <a name="return-value"></a>Valor devuelto

Devuelve un valor distinto de cero si el botón está resaltado o cero en caso contrario.
