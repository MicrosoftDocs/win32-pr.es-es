# <a name="kernel-mode-device-driver-tsp"></a>Kernel-Mode Device Driver TSP

El TSP en modo kernel (Kmddsp.tsp) permite que una aplicación controle miniports WAN heredados (por ejemplo, controladores NDIS 4.0) como dispositivos TAPI.

El TSP en modo kernel se instala con los sistemas operativos Windows Server 2003, Windows XP, Windows 2000, Windows NT, Windows Millennium Edition, Windows 98 y Windows 95.
