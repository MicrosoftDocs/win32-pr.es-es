# <a name="g-network-monitor"></a>G (Monitor de red)

<dl> <dt>

<span id="_netmon_generic_formatter_gly"></span><span id="_NETMON_GENERIC_FORMATTER_GLY"></span>**formateador genérico**
</dt> <dd>

Formateador de datos de propiedad que Monitor de red proporciona. El formateador genérico puede dar formato a los datos que se muestran en el panel de detalles de la interfaz de usuario de Monitor de red. Cuando desee Monitor de red usar el formateador genérico, debe especificar la función **FormatPropertyInstance** en **InstanceData** miembro de la estructura **PROPERTYINFO** de una propiedad. Vea también [salida de formateador genérico](generic-formatter-output.md)

</dd> </dl>
