# <a name="mf_sa_required_sample_count_progressive-attribute"></a>MF \_ SA \_ obligatorio \_ \_ recuento de muestra \_ atributo progresivo

Indica el número de muestras que requiere la asignación de una Microsoft Media Foundation (MFT) para el contenido progresivo.

## <a name="data-type"></a>Tipo de datos

**UINT32**

## <a name="remarks"></a>Observaciones

Este valor se usa si el siguiente nodo de nivel inferior tiene un [**IMFVideoSampleAllocator**](/windows/desktop/api/mfidl/nn-mfidl-imfvideosampleallocator).
