# <a name="wmmediacredits"></a>WM/MediaCredits

El **atributo WM/MediaCredits** contiene los créditos para los medios de difusión.

## <a name="global-constant"></a>Constante global

g \_ wszWMMediaCredits

## <a name="data-type"></a>Tipo de datos

**CADENA DE TIPO WMT \_ \_**

## <a name="remarks"></a>Comentarios

Este atributo se usa para la difusión de televisión o radio.
