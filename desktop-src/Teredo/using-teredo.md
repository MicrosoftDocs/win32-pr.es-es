# <a name="using-teredo"></a>Uso de Teredo

Al desarrollar una aplicación que utiliza [Teredo](about-teredo.md), el objetivo es lograr la conectividad IPv6 y no centrarse en el uso de Teredo. Por ejemplo, el área de encuentro de Windows y la asistencia remota son dos aplicaciones que aprovechan la conectividad de un extremo a otro que proporciona IPv6. Estas aplicaciones pueden obtener conectividad mediante direcciones IPv6, 6to4 o Teredo nativas. Para estas aplicaciones y cualquier otra que se pueda desarrollar para IPv6, Teredo se trata como cualquier otro protocolo que permita la conectividad.
