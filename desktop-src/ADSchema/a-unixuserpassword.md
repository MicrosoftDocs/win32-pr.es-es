# <a name="unixuserpassword-attribute"></a>atributo unixUserPassword

Contiene una contraseña de usuario que es compatible con un sistema UNIX.

|-------------------|-------------------------------------------------------|
| CN                | unixUserPassword                                      |
| Nombre para mostrar de LDAP | unixUserPassword                                      |
| Tamaño              | \-                                                    |
| Actualizar privilegio  | \-                                                    |
| Frecuencia de actualización  | \-                                                    |
| Attribute-Id      | 1.2.840.113556.1.4.1910                               |
