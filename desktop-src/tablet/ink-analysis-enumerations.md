# <a name="ink-analysis-enumerations"></a>Enumeraciones de análisis de tinta

Esta sección contiene información sobre las enumeraciones utilizadas en el análisis de tinta.

## <a name="enumerations"></a>Enumeraciones



| Enumeración                                              | Descripción                                                                                                                               |
|----------------------------------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------|
| [**AnalysisModes**](analysismodes.md)                   | Especifica cómo el [**IInkAnalyzer**](iinkanalyzer.md) realiza el análisis de tinta.<br/>                                                  |
| [**AnalysisWarningCode**](analysiswarningcode.md)       | Especifica el conjunto de advertencias disponibles que se pueden producir durante el análisis de tinta.<br/>                                                    |
| [**ConfirmationType**](confirmationtype.md)             | Especifica el tipo de confirmación que se puede producir en un objeto [**IContextNode**](icontextnode.md) .<br/>                           |
| [**ContextLinkDirection**](contextlinkdirection.md)     | Especifica los tipos de objetos [**IContextLink**](icontextlink.md) .<br/>                                                           |
| [**RecognitionConfidence**](recognitionconfidence.md)   | Especifica el nivel de confianza que [**IInkAnalyzer**](iinkanalyzer.md) tiene en la precisión del resultado del reconocimiento.<br/> |
| [**RecognizerCapabilities**](recognizercapabilities.md) | Especifica los atributos de un [**IInkAnalysisRecognizer**](iinkanalysisrecognizer.md).<br/>                                       |
| [**StrokeType**](stroketype.md)                         | Especifica cómo el [**IInkAnalyzer**](iinkanalyzer.md) debe clasificar un trazo durante el análisis.<br/>                               |
