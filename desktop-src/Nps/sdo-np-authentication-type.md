# <a name="np-authentication-type"></a>NP-Authentication-Type

Tipos de autenticación permitidos para una conexión.

**Nombre LDAP:** msNPAuthenticationType2

**IDENTIFICADOR:** 4105 = TIPO [ **DE AUTENTICACIÓN \_ NP DE ATRIBUTO \_ \_ IAS \_**](/windows/desktop/api/sdoias/ne-sdoias-attributeid)

**Sintaxis:** Entero

**Valores: **

Se define en el [**tipo enumerado AUTHENTICATION \_ TYPE**](/windows/desktop/api/SdoIas/ne-sdoias-authentication_type) declarado en el archivo de encabezado público SdoIas.h.

-   **IAS \_ AUTENTICACIÓN \_ NO VÁLIDA** (0)
-   **IAS \_ AUTH \_ PAP** (1)
-   **IAS \_ AUTH \_ MD5CHAP** (2)
-   **IAS \_ AUTH \_ MSCHAP** (3)
-   **IAS \_ AUTH \_ MSCHAP2** (4)
-   **IAS \_ AUTH \_ EAP** (5)
-   **IAS \_ AUTH \_ ARAP** (6)
-   **IAS \_ AUTH \_ NONE** (7)
-   **IAS \_ AUTENTICACIÓN \_ PERSONALIZADA** (8)
-   **IAS \_ AUTH \_ MSCHAP \_ CPW** (9)
-   **IAS \_ AUTH \_ MSCHAP2 \_ CPW** (10)
-   **IAS \_ AUTH \_ PEAP** (11)
