# <a name="isearchcompletedcallback-methods"></a>Métodos ISearchCompletedCallback

La [**interfaz ISearchCompletedCallback**](/windows/desktop/api/Wuapi/nn-wuapi-isearchcompletedcallback) define el método siguiente.



| Método                                            | Descripción                                                                                                                                                                     |
|---------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| [**Invocar**](/windows/desktop/api/Wuapi/nf-wuapi-isearchcompletedcallback-invoke) | Controla la notificación de la finalización de una búsqueda asincrónica que se inicia mediante una llamada al [**método IUpdateSearcher.BeginSearch.**](/windows/desktop/api/Wuapi/nf-wuapi-iupdatesearcher-beginsearch) |
