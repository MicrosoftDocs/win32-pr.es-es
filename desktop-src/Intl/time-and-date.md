# <a name="time-and-date"></a>Fecha y hora

NLS admite la recuperación de cadenas para cualquier hora o fecha en un formato adecuado para la configuración regional actual. En Windows Vista y versiones posteriores, NLS también admite la recuperación de cadenas de duración para cualquier configuración regional, incluidas las configuraciones regionales personalizadas. Para obtener instrucciones de desarrollo, vea [recuperar información de fecha y hora](retrieving-time-and-date-information.md).
