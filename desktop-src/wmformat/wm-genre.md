# <a name="wmgenre"></a>WM/género

El atributo **WM/Genre** contiene el género del contenido.

## <a name="global-constant"></a>Constante global

g \_ wszWMGenre

## <a name="data-type"></a>Tipo de datos

**\_cadena de tipo WMT \_**

## <a name="remarks"></a>Observaciones

Este es el atributo preferido para especificar el género del contenido.

Si cambia **WM/Genre** o **WM/GenreID** en un archivo mp3, se cambiará el otro atributo para que coincida.

### <a name="example"></a>Ejemplo



| Tipo de archivo | Valor de ejemplo |
|-----------|---------------|
| Audio     | Espada        |
| Vídeo     | Dramático       |
