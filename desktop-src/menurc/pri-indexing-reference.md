# <a name="package-resource-indexing-pri-reference"></a>Referencia de indexación de recursos de paquetes (PRI)

Un conjunto de API para trabajar con un indizador de recursos. Un indizador de recursos se usa para generar los archivos de índice de recursos del paquete (PRI) para una aplicación para UWP. Para obtener más información y tutoriales basados en escenarios sobre cómo usar estas API, vea [API de indexación de recursos de paquetes (PRI) y sistemas de compilación personalizados](/windows/uwp/app-resources/pri-apis-custom-build-systems).
