# <a name="cdispparams-class"></a>Clase CDispParams

![jerarquía de clases cdispparams](images/cutil11.png)

La `CDispParams` clase implementa la estructura **DISPPARAMS** utilizada en la automatización como una clase base de C++. El método **IDispatch:: Invoke** usa la estructura **DISPPARAMS** de OLE para contener los argumentos pasados a cualquier método o propiedad.

|------------------------------------------------|--------------------------------------|
| [**CDispParams**](cdispparams-cdispparams.md) | Construye un objeto **CDispParams** . |
