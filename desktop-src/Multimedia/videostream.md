# <a name="videostream"></a>Videostream

Las aplicaciones pueden usar funciones de devolución de llamada de Videostream durante la captura de streaming para procesar un fotograma de vídeo capturado. La ventana captura llama a una función de devolución de llamada de Videostream justo antes de escribir cada fotograma capturado en el disco.
