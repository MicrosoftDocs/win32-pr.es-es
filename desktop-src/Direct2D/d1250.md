# <a name="d1250-inaccessible-source"></a>D1250: origen inaccesible

Se ha detectado un intento de dibujar desde un mapa de bits inaccesible.






 

## <a name="possible-causes"></a>Causas posibles

Se ha detectado un intento de dibujar desde un origen inaccesible. Una razón por la que un origen puede ser inacessible es si se creó a partir de una textura compartida y no se adquirió la textura.

## <a name="possible-fixes"></a>Posibles correcciones

Asegúrese de que todos los mapas de bits son accesibles antes de dibujarlos con Direct2D.
