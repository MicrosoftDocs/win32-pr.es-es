# <a name="imagebutton"></a>IMAGEBUTTON

Este es un **botón** predefinido con el siguiente valor predeterminado.

``` syntax
cursor="Hand"
```

## <a name="remarks"></a>Observaciones

Todos los botones pueden contener imágenes. Este elemento es útil con fines de aclaración de código para indicar un botón que se usa principalmente como un contenedor de imágenes.

Todas las propiedades de este **botón** se pueden invalidar si se especifican explícitamente.
