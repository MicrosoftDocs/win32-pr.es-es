# <a name="exposing-the-contents-of-a-window"></a>Exponer el contenido de una ventana

Una aplicación debe exponer el contenido de las clases de ventana personalizadas que utilizan Microsoft Active Accessibility. Toda la ventana se debe tratar como un control personalizado, que es un contenedor para otros controles estándar, personalizados o personalizados.

Para obtener más información sobre cómo exponer el contenido de ventanas personalizadas mediante Active Accessibility, vea la sección [accesibilidad](/windows/desktop/accessibility) .
