# <a name="cbaserendererisstreaming-method"></a>CBaseRenderer. IsStreaming, método

El `IsStreaming` método consulta si el filtro es un flujo de datos.

## <a name="syntax"></a>Sintaxis


```C++
BOOL IsStreaming();
```



## <a name="parameters"></a>Parámetros

Este método no tiene parámetros.

## <a name="return-value"></a>Valor devuelto

Devuelve la marca [**CBaseRenderer:: m \_ bStreaming**](cbaserenderer-m-bstreaming.md) .
